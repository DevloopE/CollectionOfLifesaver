# 普林斯顿读本系列 · 持久化写作规则

你正在为这个 7 本（即将 14 本）的简体中文 Banner 风格数学教科书系列贡献一章或几章。
开工前请把这份规则**完整读完**，整个写作过程不得违反任何一条。

仓库：github.com/DevloopE/CollectionOfLifesaver

## 1. 开工前必须读

- 风格样板：随机挑两章读完
  - 偏微分方程/ch5.html （热方程，物理推导 + 算例 + 三句话总结）
  - 实分析/ch17.html （函数列，反例驱动 + Cauchy 准则 + Dini）
  - 线性代数/ch9.html （四个基本子空间，Banner 抽象核心范本）

读完之后你应该能复述：
- "扩散就是削峰填谷" 这种贴近物理的句法
- "你可能会想…但其实…" 的反例接种
- 第二人称"你"、"我们"
- 章末 3 句话总结

## 2. 不可违反的硬规则

▸ **零脚手架标签**。下列字眼不出现：
  "**方法**:" "**例题**:" "**内心独白**:" "**理智检验**:" "**常见错误**:"
  "**注意**:" "**判别**:" "**充分条件**:" "*试一试*:" "本章我们要讨论：" 这种带枚举的开篇
  改写成自然散文。

▸ **body 片段，不要包壳**。每个文件第一行就是 `<h1 id="chN">第 N 章 标题</h1>`。
  不要写 `<!DOCTYPE>`、`<html>`、`<head>`、`<body>`、`<main>`。

▸ **不要 push GitHub**。

▸ **保持简体中文**。

▸ **不要用 `<blockquote>` 当 callout box**。

## 3. 深度校准（防"看似对其实单薄"）

每章必须达到：

| 指标 | 最低 | 目标 |
|---|---|---|
| 中文字符数 | ≥ 8000 | 10000–13000 |
| `<h2>` 节数 | ≥ 6（**禁止全章固定 5 节**） | 6–10 |
| 完整推演的例题 | ≥ 4 道（每步算到底） | 4–6 |
| display 数学 `\[...\]` | ≥ 20 | 20–40 |
| `\boxed{}` 关键公式 | ≥ 4 | 4–8 |
| 跨卷引用（用 `../别书/chN.html` 路径） | ≥ 2 | 3–6 |

## 4. 之前出过的具体错误

【错误 1】每章硬性 5 节：让章节按"动机 → 定义 → 主结果 → 反例 → 应用 → 延伸"自然展开

【错误 2】例题只 1 道：每节至少 1 道，每章至少 4 道，每道完整算到底

【错误 3】关键反例只名字出现没真算：每个反例都全程推演

【错误 4】定理只陈述不证：核心定理至少给推导思路

【错误 5】把 `\[ ... \]` 包在 `<p>` 里：display 数学独占一行，前后空行，**不要**用 `<p>`

【错误 6】`\boxed{}` 用得太节省：每章 4-8 个

【错误 7】跨卷引用稀少：先打开其他书 index.html 看目录，引到具体章节

## 5. HTML 与 LaTeX 格式

- `<h1 id="chN">第 N 章 标题</h1>` —— 第一行
- `<h2 id="secN-K">N.K 标题</h2>` —— 节
- `<p>...</p>` —— 段落
- 内联数学 `\(...\)`，行间数学 `\[...\]` 独占一行前后空行
- 关键公式包 `\boxed{}`
- 矩阵 `\begin{pmatrix}...\end{pmatrix}` 或 `bmatrix`
- 不要用 `<blockquote>`
- HTML 标签全部对齐闭合

## 7. 工作流

1. 读这份规则
2. 读 2-3 章样板（上面列的）
3. 看你要写的章节的内容规格（在 prompt 里给）
4. 写 body 片段到 `<book>/_bodies/chN.html`
5. **不要**跑 wrap_book.py（编排者负责）
6. **不要** push（编排者负责）
7. 报告完成
