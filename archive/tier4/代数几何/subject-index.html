索引

这份索引按拼音首字母分组，但写法尽量面向“你现在想做什么”，而不是把术语孤零零排成词表。条目后面的数字默认指本卷的小节；若顺手需要跨到附录，我会直接写“附录 A/B”。最推荐的用法不是从头读到尾，而是先把你脑子里的问题翻成一个动作，再来找对应条目。

A

  • 阿贝尔群 想知道椭圆曲线上的点为什么会自己长成交换群，先看 15.2-15.4；想看这个群在 \(\mathbb Q\) 上为什么还会携带深刻数论信息，再看 20.2-20.5。
  • 仿射簇 想从“多项式零点集就是几何对象”这句总纲真正入门，先看 1.2-2.6；想把仿射对象升级成概形语言，看 11.1-12.5。

B

  • Bezout 定理 想数两条投影曲线为什么总共有 \(de\) 个交点，先看 5.1-5.5；想看它怎样变成椭圆曲线群律的来源，再回 15.2。
  • blow-up 想看奇点附近怎样“拉开方向”而不是硬算偏导，先看 9.5-9.6。

C

  • 层 想把“局部数据如何粘成全局对象”说成正式语言，先看 10.1-10.5；想看层为什么会自然带出上同调，再接 16.1-16.4。
  • 除子 想把零点、极点和函数的消失阶数统一记账，先看 14.1-14.4；想知道这为什么会进入 Riemann-Roch，看 14.5-14.6。

D

  • 对偶 想知道标准层、Serre 对偶和曲线上的微分为何会重新相遇，先看 17.1-17.5，再看 18.7。
  • 代数闭域 想知道为什么零点理论最好先在 \(\bar k\) 上说，再回 1.5 与 3.1-3.4。

G

  • 概形 想从“环的几何”进入 \(\operatorname{Spec}\) 与 \(\operatorname{Proj}\)，先看 11.1-11.5、12.1-12.5、13.1-13.5。
  • 光滑性 想判断一个点是不是奇点，先看 8.4、9.1-9.4；想知道一阶微分为何也能探测光滑与奇异，再看 18.4-18.5。
  • 归一化 想知道为什么尖点曲线的坐标环要靠整闭包来修补，先看附录 A.2，再对照 9.2-9.5。

H

  • Hilbert 零点定理 想把理想与零点集之间的反演关系真正钉死，直接看 3.1-3.5。

J

  • \(j\)-不变量 想知道椭圆曲线为什么能被一个参数分类，先看 15.4-15.6；想看它怎样直接长成模空间坐标，再看 19.5。
  • 局部环 想把“在一点附近看几何”说成环论，先看 7.5 与 8.1-8.3；想看一维光滑局部模型为何是 DVR，再看附录 A.3。
  • 交点重数 想把“曲线相交几次”从画图直觉升级成可计数对象，先看 5.2-5.5。

K

  • Kähler 微分 想知道代数几何里怎样不靠极限也定义 \(df\)，直接看 18.1-18.8。
  • Krull 维数 想把“一个局部环到底有几维”从链条件角度说准，先看 8.1-8.3，再看附录 A.5。

L

  • \(L\)-函数 想知道模 \(p\) 点数怎样拼成椭圆曲线的解析对象，先看 20.4；想看它为何会和 rank 碰头，再接 20.5-20.6。
  • 线丛 想把除子翻成可几何操作的层对象，先看 14.1-14.6；想看它怎样进入上同调与对偶，再看 16.1-16.4、17.1-17.5。

M

  • 模空间 想知道“所有曲线的同构类”为什么自己也该形成几何对象，先看 19.1-19.3；想看最经典例子 \(M_{1,1}\)，接着看 19.5。
  • Mordell-Weil 想知道椭圆曲线的有理点群为什么总是有限生成，直接看 20.3。

N

  • Noether 性 想知道为什么代数几何里老强调“有限生成”，先看附录 A.1。

P

  • Proj 想把投影空间和齐次坐标环连起来，先看 4.1-4.5 与 13.1-13.5。
  • 平展 想知道一个家族在换基底后会不会突然撕裂或粘连，先看附录 A.4；想把这件事放回模空间语言，再对照 19.2-19.3。

Q

  • 切空间 想把“一个点附近的一阶方向”算出来，先看 8.4；想看它为何等于余切空间对偶，再看 18.5 与附录 B.4。
  • 曲线亏格 想知道亏格为什么控制 Riemann-Roch、椭圆曲线与模空间维数，先看 14.5、15.1、19.1、19.4。

S

  • Serre 对偶 想知道上同调为何会在互补次数之间彼此对偶，先看 17.1-17.5；想看它怎样直接给出模空间维数 \(3g-3\)，再看 19.4。
  • Spec 想从素理想长出几何对象，先看 11.1-11.4 与 12.1-12.5。
  • 稳定曲线 想知道模空间边界上该添进哪些退化对象，先看 19.6。

T

  • 切丛与余切丛 想把微分、切向量和线性代数里的对偶空间真正接上，先看 18.5-18.7。
  • Taniyama-Shimura / 模性定理 想知道 Wiles 工作在这本书的大图里究竟处在哪，直接看 20.6。
  • Tate-Shafarevich 群 想知道 BSD 公式里那个最神秘的有限群在什么位置，先看 20.5。

W

  • 微分模 想把“形式微分”从多项式环推进到任意环同态，先看 18.2-18.4。
  • 无穷远点 想知道为什么仿射对象必须补到投影里才能把交点和群律说完整，先看 4.1-4.4、5.1-5.3、15.2。

X

  • 想把一个仿射方程补成投影对象 先看 4.3-4.5，再看 13.1-13.5。
  • 想判断一个点是不是奇点 先看 8.4 与 9.1-9.4；若想从微分关系角度再看一遍，接 18.4-18.5。
  • 想知道某条椭圆曲线在 \(\mathbb Q\) 上有没有很多点 先看 20.2-20.3；想知道为什么这个问题会转去看 \(L(E,s)\)，再看 20.4-20.5。
  • 想把一个模问题写成现代语言 先看 19.2，再看附录 B.2-B.5。

Y

  • Yoneda 想知道为什么“对象由它对所有测试对象的反应决定”，直接看附录 B.4；想知道这为什么会逼出模空间与 stack 语言，再接附录 B.5-B.6。
  • 有理点 想从几何方程真正走向数论问题，先看 20.1-20.3。
  • 雅可比判别法 想快速检查光滑与奇异，先看 9.1-9.3；想知道它和 Kähler 微分的关系，再看 18.4-18.5。

Z

  • Zariski 拓扑 想看为什么闭集不是用距离，而是用共同零点来定义，先看 2.1-2.4；想看它怎样在 \(\operatorname{Spec}\) 上再次出现，接 11.2、12.1。
  • 整闭包 想知道一个环为什么还需要“把被首一方程逼出来的元素补回来”，先看附录 A.2。
  • 主理想定理 想把“一个方程最多切掉一维”说成真正的代数定理，先看附录 A.5。
  • 正则微分 想知道椭圆曲线上的 \(\dfrac{dx}{2y}\) 为什么是全局 1-form，直接看 18.8。
