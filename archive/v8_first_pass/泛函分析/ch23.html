第 23 章 应用：Sobolev 空间与变分入门

这一章把前面 22 章的工具组合在一起，给经典椭圆偏微分方程一个干净的"弱解"理论。具体说，我们要解

\[ -\Delta u + u = f \text{ 在 } \Omega, \quad u|_{\partial \Omega} = 0, \]

其中 \(\Omega \subset \mathbb{R}^n\) 是有界域，\(f\) 给定。

这是偏微分方程课的标准经典问题。古典方法（分离变量、Green 函数）对几何形状有限制。变分 / 弱解方法靠的是泛函分析：

  1. 定义合适的"解所在空间"——Sobolev 空间 \(H_0^1(\Omega)\)；
  2. 把 PDE 写成"双线性形式 + 线性泛函"形式；
  3. 用 Lax–Milgram 定理（第 16 章应用）+ Riesz 表示得到唯一弱解；
  4. 由 Sobolev 嵌入 + Rellich 紧嵌入研究弱解的正则性。
  • Sobolev 空间 \(W^{k,p}(\Omega)\) 与 \(H^k(\Omega) = W^{k,2}(\Omega)\) 的定义。
  • \(H^1\) 上的内积、范数、完备性。
  • "零边值"子空间 \(H_0^1(\Omega)\) 的定义。
  • Poincaré 不等式与 \(H_0^1\) 上的等价范数。
  • Lax–Milgram 在 \(-\Delta + 1\) 上的应用：弱解存在唯一。
  • Sobolev 嵌入与 Rellich 紧嵌入（陈述 + 应用到正则性）。
  • 变分法的标准模板：极小化能量泛函 \(\Rightarrow\) 弱解。

23.1 Sobolev 空间的定义

设 \(\Omega \subset \mathbb{R}^n\) 是开集，\(k \in \mathbb{N}\)，\(1 \le p \le \infty\)。Sobolev 空间

\[ W^{k,p}(\Omega) = \{u \in L^p(\Omega) : D^\alpha u \in L^p(\Omega) \text{ 对 } |\alpha| \le k \text{（弱意义）}\}. \]

"弱导数" \(D^\alpha u\) 的定义：\(v = D^\alpha u\) 弱意义是说

\[ \int_\Omega u D^\alpha \varphi \, dx = (-1)^{|\alpha|} \int_\Omega v \varphi \, dx, \quad \forall \varphi \in C_c^\infty(\Omega). \]

这是把分部积分反向定义"导数"——经典 \(C^k\) 的导数自动满足这条等式（正向分部积分），所以弱导数在 \(C^k\) 上和经典导数一致。

\(W^{k,p}\) 配范数

\[ \|u\|_{W^{k,p}} = \left( \sum_{|\alpha| \le k} \|D^\alpha u\|_{L^p}^p \right)^{1/p} \]

是 Banach 空间。当 \(p = 2\) 时记 \(H^k(\Omega) = W^{k,2}(\Omega)\)，是 Hilbert 空间，配内积

\[ \langle u, v \rangle_{H^k} = \sum_{|\alpha| \le k} \int_\Omega D^\alpha u \cdot D^\alpha v \, dx. \]

本章主要用 \(H^1(\Omega)\)。它的内积是

\[ \boxed{\ \langle u, v \rangle_{H^1} = \int_\Omega (u v + \nabla u \cdot \nabla v) \, dx.\ } \]

对应范数 \(\|u\|_{H^1}^2 = \|u\|_{L^2}^2 + \|\nabla u\|_{L^2}^2\)。

23.2 \(H_0^1(\Omega)\)：零边值子空间

"\(u|_{\partial \Omega} = 0\)" 这个边界条件在 \(H^1\) 上不直接有意义——\(H^1\) 函数只有"几乎处处"定义，没有"边界值"。要让边界条件可表达，定义

\[ H_0^1(\Omega) := \overline{C_c^\infty(\Omega)}^{H^1}. \]

即 "紧支撑光滑函数"在 \(H^1\) 范数下的闭包。\(H_0^1\) 是 \(H^1\) 的闭子空间，因此自身是 Hilbert 空间。

"\(u \in H_0^1\)"是把"零边值"几何化的方式——你不能直接谈"边界值"，但可以谈"是否能被零延拓的紧支撑光滑函数 \(H^1\) 逼近"。

对足够"光滑"的边界（比如 Lipschitz 边界），有迹算子 \(T : H^1(\Omega) \to L^2(\partial \Omega)\) 让"边界值"有意义；这时 \(H_0^1 = \ker T\)。详细在 Adams《Sobolev Spaces》。

23.3 Poincaré 不等式

定理（Poincaré 不等式）。设 \(\Omega \subset \mathbb{R}^n\) 是有界域。则存在常数 \(C = C(\Omega) > 0\) 使

\[ \boxed{\ \|u\|_{L^2} \le C \|\nabla u\|_{L^2}, \quad \forall u \in H_0^1(\Omega).\ } \]

"零边值"+"有界域"是必须的——在 \(H^1(\mathbb{R}^n)\) 或非零边值下不成立（取 \(u \equiv 1\) 给反例：\(\nabla u = 0\) 但 \(\|u\|_{L^2}\) 大）。

证明思路（一维情形）：\(u(x) = u(0) + \int_0^x u'(t) dt\)。\(u(0) = 0\)（零边值），所以 \(|u(x)| \le |\Omega|^{1/2} \|u'\|_{L^2}\)（Cauchy–Schwarz）。两边平方积分得 \(\|u\|_{L^2}^2 \le C \|u'\|_{L^2}^2\)。一般维度论证类似但需要 Friedrichs 引理或 Faber–Krahn。

Poincaré 不等式给一个非常有用的等价范数：在 \(H_0^1(\Omega)\) 上，

\[ \|u\|_* := \|\nabla u\|_{L^2} \]

是范数（正定来自 Poincaré：\(\|u\|_* = 0 \Rightarrow \|u\|_{L^2} = 0 \Rightarrow u = 0\)），且与 \(H^1\) 范数等价：

\[ \|u\|_{H^1}^2 = \|u\|_{L^2}^2 + \|\nabla u\|_{L^2}^2 \le (C^2 + 1) \|\nabla u\|_{L^2}^2, \]

反向 \(\|\nabla u\|_{L^2} \le \|u\|_{H^1}\)。所以 \(H_0^1\) 配 \(\|\cdot\|_*\) 也是 Hilbert 空间。这个范数在变分法里更自然。

23.4 弱解的定义与 Lax–Milgram 应用

问题：解 \(-\Delta u + u = f\) 在 \(\Omega\)，\(u \in H_0^1(\Omega)\)，\(f \in L^2(\Omega)\) 给定。

"弱解"的来源：用测试函数 \(\varphi \in H_0^1\) 乘 PDE 两边并积分：

\[ \int_\Omega (-\Delta u) \varphi \, dx + \int_\Omega u \varphi \, dx = \int_\Omega f \varphi \, dx. \]

第一项用分部积分（"零边值"让边界项消失）：\(\int_\Omega (-\Delta u) \varphi = \int_\Omega \nabla u \cdot \nabla \varphi\)。所以

\[ \int_\Omega \nabla u \cdot \nabla \varphi \, dx + \int_\Omega u \varphi \, dx = \int_\Omega f \varphi \, dx. \]

左边的第一项 + 第二项就是 \(H^1\) 内积 \(\langle u, \varphi \rangle_{H^1}\)。

定义。\(u \in H_0^1(\Omega)\) 是 PDE \(-\Delta u + u = f\)（带零边值）的弱解，是说

\[ \boxed{\ \langle u, \varphi \rangle_{H^1} = \int_\Omega f \varphi \, dx, \quad \forall \varphi \in H_0^1(\Omega).\ } \]

右边定义了 \(H_0^1\) 上的线性泛函 \(L_f(\varphi) = \int f \varphi\)。验证它有界：

\[ |L_f(\varphi)| = \left| \int f \varphi \right| \le \|f\|_{L^2} \|\varphi\|_{L^2} \le \|f\|_{L^2} \|\varphi\|_{H^1}. \]

所以 \(L_f \in (H_0^1)^*\)，\(\|L_f\| \le \|f\|_{L^2}\)。

由 Riesz 表示定理（第 16 章），存在唯一 \(u \in H_0^1\) 使

\[ \langle u, \varphi \rangle_{H^1} = L_f(\varphi), \quad \forall \varphi. \]

这个 \(u\) 就是弱解。存在唯一 由 Riesz 一行字给。

\[ \boxed{\ \text{对每 } f \in L^2(\Omega), \text{ PDE } -\Delta u + u = f \text{（零边值）有唯一弱解 } u \in H_0^1.\ } \]

23.5 Lax–Milgram：更一般的椭圆方程

上一节的方法直接用 Riesz 表示是因为左边恰好就是 \(H^1\) 内积。对更一般的椭圆 PDE 比如 \(-\nabla \cdot (A \nabla u) + b \cdot \nabla u + c u = f\)，左边变成非对称双线性形式，不直接是内积。这时用Lax–Milgram 定理。

定理（Lax–Milgram）。设 \(H\) Hilbert 空间，\(a : H \times H \to \mathbb{R}\) 双线性满足：

  • 有界：\(|a(u, v)| \le M \|u\|_H \|v\|_H\)；
  • 强制（coercive）：\(a(u, u) \ge \alpha \|u\|_H^2\) 某 \(\alpha > 0\)。

对每个 \(L \in H^*\)，存在唯一 \(u \in H\) 使 \(a(u, v) = L(v), \forall v\)。

证明思路：固定 \(u\)，\(v \mapsto a(u, v)\) 是 \(H^*\) 元素，由 Riesz 对应 \(A u \in H\)。即 \(a(u, v) = \langle A u, v \rangle\)。\(A : H \to H\) 有界 + 强制 \(\Rightarrow\) 双射（强制给单射 + 像闭，再用对偶论证给满射）。再用 Riesz 表示 \(L(v) = \langle \ell, v \rangle\) 某 \(\ell \in H\)。由 \(A\) 双射 \(\Rightarrow\) \(u = A^{-1} \ell\) 是 \(a(u, v) = L(v)\) 的唯一解。

这条定理把上一节的"内积形式"推广到"强制双线性形式"。它是偏微分方程椭圆问题的核心存在性工具。具体应用：

  • 变系数椭圆：\(-\nabla \cdot (A(x) \nabla u) = f\)，\(A(x)\) 一致椭圆 \(\Rightarrow\) 强制 \(\Rightarrow\) Lax–Milgram 给唯一解。
  • 带漂移项：\(-\Delta u + b \cdot \nabla u + c u = f\)，\(c \ge 0\) + 适当 \(b\) 控制 \(\Rightarrow\) 强制（用 Poincaré 控漂移）。

23.6 Sobolev 嵌入与 Rellich 紧嵌入

有了"弱解存在"，下一个问题是：弱解多正则？它是否在某个 \(L^p\) 中、是否连续、是否光滑？

定理（Sobolev 嵌入）。设 \(\Omega \subset \mathbb{R}^n\) 有界 + Lipschitz 边界。对 \(k \in \mathbb{N}\) 与 \(1 \le p \le \infty\)：

  • 若 \(k p < n\)：\(W^{k,p}(\Omega) \hookrightarrow L^q(\Omega)\)，其中 \(\frac{1}{q} = \frac{1}{p} - \frac{k}{n}\)；
  • 若 \(k p = n\)：\(W^{k,p}(\Omega) \hookrightarrow L^q(\Omega)\) 对 \(p \le q < \infty\)；
  • 若 \(k p > n\)：\(W^{k,p}(\Omega) \hookrightarrow C^{k - \lceil n/p \rceil, \alpha}(\overline\Omega)\) 某 \(\alpha\)。

第三条最有用——它说"导数阶数 \(k\) 比 \(n/p\) 大时，函数自动连续"。例如 \(H^1(\Omega)\) 在 \(n = 1\) 时（\(k = 1, p = 2, k p = 2 > 1 = n\)）嵌入 \(C^{0, 1/2}\)；在 \(n = 2\) 时不嵌入到连续函数（\(k p = n\) 临界）；在 \(n = 3\) 时嵌入到 \(L^6\)（\(\frac{1}{q} = \frac{1}{2} - \frac{1}{3} = \frac{1}{6}\)）。

定理（Rellich–Kondrachov 紧嵌入）。在 Sobolev 嵌入的"严格"情形下（即 \(q\) 严格小于嵌入定理给的临界 \(q^*\)），嵌入还是紧的。最常用版本：

\[ \boxed{\ H^1(\Omega) \hookrightarrow\hookrightarrow L^2(\Omega), \quad \Omega \text{ 有界 + Lipschitz 边界}.\ } \]

"\(\hookrightarrow\hookrightarrow\)" 表示紧嵌入——\(H^1\) 中的有界集在 \(L^2\) 中相对紧。

这条紧嵌入是 PDE 弱解理论里最重要的一条事实。它的实际意义：你拿到 \(H^1\) 弱收敛序列 \(u_n \rightharpoonup u\)，自动得到 \(L^2\) 强收敛 \(u_n \to u\)。这就是第 20 章 20.5 节"紧算子把弱收敛变强收敛"的具体化——嵌入 \(H^1 \hookrightarrow L^2\) 紧 \(\Rightarrow\) 嵌入算子是紧算子。

用紧嵌入证明非线性 PDE 弱解存在性的"极小化方法"模板：

  1. 定义能量 \(F(u) = \int (\frac{1}{2} |\nabla u|^2 + G(u))\)；
  2. 取极小化序列 \(u_n\) in \(H_0^1\)；
  3. 能量有界 \(\Rightarrow \|\nabla u_n\|_{L^2}\) 有界 \(\Rightarrow \{u_n\}\) 在 \(H_0^1\) 中有界；
  4. \(H_0^1\) 自反 \(\Rightarrow u_{n_k} \rightharpoonup u^*\) 弱收敛；
  5. Rellich \(\Rightarrow u_{n_k} \to u^*\) 在 \(L^2\) 中强收敛；
  6. \(F\) 弱下半连续 \(\Rightarrow F(u^*) \le \liminf F(u_{n_k}) = \inf F\)；
  7. \(u^*\) 是极小元 \(\Rightarrow\) Euler–Lagrange 给 \(u^*\) 是 PDE 弱解。

这条模板是 Calderón、De Giorgi、Stampacchia 等在 1950-60 年代标准化的。它把 PDE 弱解存在性问题完全代数化——只要验证 7 步里每一步的假设，存在性自动有。

本章把"泛函分析 \(\to\) PDE"这条桥梁的入口处展示完整。深入下去就是 Brezis、Evans、Gilbarg–Trudinger 这种 PDE 教材的内容。本书止步于此——下一章我们去看另一个"应用"：量子力学，这是泛函分析最受人欢迎的物理化身。
