第 4 章 经典空间例子库

前三章我们把 Banach 空间和 Hilbert 空间的"框架"立起来了：完备的赋范空间、完备的内积空间。但泛函分析不是抽象自娱自乐——这些框架要被实际使用。"实际使用"靠的就是几个标准空间，所有定理你都得能把它们具体落到这些空间上。本章把这一组"硬件"过一遍，让你以后看到 \(\ell^p\)、\(L^p\)、\(C[a,b]\)、\(C^k[a,b]\)、\(c_0\) 时，不需要任何再回忆——它们就该像你脑子里的 \(\mathbb{R}^n\) 一样自然。

这一章不堆证明，把每个空间讲到三件事的程度：它装的是什么、范数是什么、它的几个最关键性质是什么。证明工具（特别是 Hölder、Minkowski、嵌入关系）放进下一章和附录。本章的目的是让你建立"看见名字就立刻有图像"的反射。

  • 序列空间 \(\ell^p\)（\(1 \le p \le \infty\)）+ 子空间 \(c_0\)、\(c\)、\(\ell_0\)。
  • 函数空间 \(L^p(\Omega, \mu)\) 的定义、几个关键事实、与 \(\ell^p\) 的对应。
  • 连续函数空间 \(C[a,b]\)、\(C(K)\)（\(K\) 紧致 Hausdorff）、\(C_0(\mathbb{R}^n)\)、\(C_c(\mathbb{R}^n)\)。
  • 光滑函数空间 \(C^k[a,b]\)、\(C^\infty[a,b]\) 的范数与完备性。
  • Hilbert 与 Banach 之间的"光谱"：哪些是 Hilbert、哪些只是 Banach、哪些既不可分又不自反。
  • 嵌入与对偶速览（细节延到第 5、15、19 章）。

4.1 序列空间 \(\ell^p\)

对 \(1 \le p < \infty\)，

\[ \ell^p = \left\{ x = (x_1, x_2, \ldots) \in \mathbb{K}^{\mathbb{N}} : \|x\|_p := \left( \sum_{n=1}^{\infty} |x_n|^p \right)^{1/p} < \infty \right\}. \]

对 \(p = \infty\)，

\[ \ell^\infty = \left\{ x : \|x\|_\infty := \sup_n |x_n| < \infty \right\}. \]

每个 \(\ell^p\)（\(1 \le p \le \infty\)）都是 Banach 空间。\(\ell^2\) 是 Hilbert 空间，配内积 \(\langle x, y \rangle = \sum x_n \bar y_n\)。其它 \(\ell^p\)（\(p \neq 2\)）不是 Hilbert（由第 3 章 3.3 节平行四边形反例）。

关于 \(\ell^p\) 的几条要随手记住的事实：

(L1) 嵌入。对 \(1 \le p < q \le \infty\)，

\[ \ell^p \subset \ell^q, \qquad \|x\|_q \le \|x\|_p. \]

这条嵌入和 \(L^p\) 上的"\(p\) 越大空间越大"方向相反——这是序列空间和函数空间最常被弄反的一处。原因是序列长度无界但每位独立，逐位 \(|x_n| \le \|x\|_p\) 给出 \(\sup\) 控制。

(L2) 可分性。对 \(1 \le p < \infty\)，\(\ell^p\) 是可分的（有可数稠子集，比如有理坐标的有限非零序列）；但 \(\ell^\infty\) 不可分（取所有 0/1 序列就有不可数互相距离 \(= 1\) 的元素，可数稠子集放不下）。

(L3) 对偶。对 \(1 \le p < \infty\) 与 \(1/p + 1/q = 1\)，

\[ (\ell^p)^* \cong \ell^q. \]

具体的对偶配对是 \(\langle x, y \rangle = \sum x_n y_n\)。这是第 15 章会反复用的标准结论。注意 \(p = 1, q = \infty\) 两个端点的不对称：\((\ell^1)^* = \ell^\infty\) 但 \((\ell^\infty)^* \neq \ell^1\)（\(\ell^\infty\) 的对偶比 \(\ell^1\) 大很多）。这是 \(\ell^1\) 不自反的根源。

(L4) 子空间 \(c_0\)。

\[ c_0 = \{x \in \ell^\infty : x_n \to 0\} \]

是 \(\ell^\infty\) 的闭子空间（"趋于 0" 在 sup 范数下闭合）。它本身配 sup-范数也是 Banach 空间，并且可分。它的对偶是 \((c_0)^* = \ell^1\)——这是为什么 \(c_0\) 在自反性话题里非常重要的对照例。

另一个常用子空间是 \(c = \{x : \lim_n x_n \text{ 存在}\}\)，包含 \(c_0\) 作为其闭子空间，差一个一维"常值序列"方向。

4.2 Lebesgue 空间 \(L^p(\Omega, \mu)\)

设 \((\Omega, \mathcal{F}, \mu)\) 是测度空间。对 \(1 \le p < \infty\)，

\[ L^p(\Omega, \mu) = \left\{ f : \Omega \to \mathbb{K} \text{ 可测}, \int |f|^p \, d\mu < \infty \right\} / \sim, \]

其中 \(\sim\) 是"几乎处处相等"的等价关系。范数

\[ \|f\|_p = \left( \int |f|^p \, d\mu \right)^{1/p}. \]

对 \(p = \infty\)，\(L^\infty\) 是"几乎处处有界函数"的等价类空间，配范数 \(\|f\|_\infty = \operatorname{ess\,sup} |f|\)。

关键事实（详见实分析读本第 24 章）：

(LP1) 完备（Riesz–Fischer）。每个 \(L^p\)（\(1 \le p \le \infty\)）都是 Banach 空间。\(L^2\) 还是 Hilbert 空间，配内积 \(\langle f, g \rangle = \int f \bar g \, d\mu\)。

(LP2) 嵌入（注意方向！）。当 \(\mu\) 是有限测度（\(\mu(\Omega) < \infty\)，比如 \(\Omega\) 是有界区域配 Lebesgue 测度），对 \(1 \le p \le q \le \infty\)，

\[ L^q(\Omega) \subset L^p(\Omega), \qquad \|f\|_p \le \mu(\Omega)^{1/p - 1/q} \|f\|_q. \]

这是 \(L^p\) 嵌入的"有限测度"版本。\(p\) 越大空间越小——和 \(\ell^p\) 完全相反。当 \(\mu\) 不是有限测度（比如 \(\Omega = \mathbb{R}^n\) 配 Lebesgue 测度），没有嵌入关系——\(L^1(\mathbb{R})\) 与 \(L^\infty(\mathbb{R})\) 互不包含。

(LP3) 对偶。对 \(1 \le p < \infty\) 与 \(1/p + 1/q = 1\)，

\[ (L^p)^* \cong L^q, \qquad \langle f, g \rangle = \int f g \, d\mu. \]

这是第 16 章里 Riesz 表示在 \(L^p\) 形式的结论。\(p = 1, q = \infty\) 端点同样不对称——\((L^1)^* = L^\infty\) 但 \((L^\infty)^* \neq L^1\)。

(LP4) 可分性。对 \(\sigma\)-有限测度空间与 \(1 \le p < \infty\)，\(L^p\) 可分。但 \(L^\infty\) 不可分。

\(L^p\) 是泛函分析里出现最频繁的"函数版"例子。本书后面要反复用到的具体场景包括：第 7 章 Fourier 在 \(L^2\)；第 16 章 Riesz 在 \(L^p\)；第 17 章弱收敛在 \(L^p\)；第 23 章 Sobolev 嵌入到 \(L^p\)。

4.3 连续函数空间 \(C[a,b]\) 与 \(C(K)\)

设 \(K\) 是紧致 Hausdorff 空间。

\[ C(K) = \{f : K \to \mathbb{K} \text{ 连续}\}, \qquad \|f\|_\infty = \sup_{x \in K} |f(x)|. \]

这是 Banach 空间。最常见的特例是 \(K = [a,b]\)，得到 \(C[a,b]\)。

关键事实：

(C1) 完备。由 \(K\) 紧 + 连续函数 \(\Rightarrow\) 有界，再由"一致收敛极限连续"得完备。

(C2) Stone–Weierstrass。如果 \(\mathcal{A} \subset C(K)\) 是含常数、点分离、闭对乘法与加法的代数，那么 \(\mathcal{A}\) 在 \(C(K)\) 中稠。它的两个最常用推论是：多项式在 \(C[a,b]\) 中稠（Weierstrass）、三角多项式在 \(C(\mathbb{T})\) 中稠（Fourier 部分基础）。

(C3) 对偶。\(C(K)^*\) 是 \(K\) 上的正则有限有符号 Borel 测度空间，配总变差范数。这是 Riesz–Markov–Kakutani 表示定理（第 16 章）。这条让你看到"线性泛函"和"测度"在 \(C(K)\) 上是同一回事——这是测度论里最美的一座桥。

另外两个常用变体：

\(C_0(\mathbb{R}^n)\)："无穷远处趋零的连续函数"，配 sup-范数。是 Banach 空间。它的对偶是 \(\mathbb{R}^n\) 上的有限有符号 Borel 测度。

\(C_c(\mathbb{R}^n)\)："紧支撑的连续函数"。不是 Banach 空间——它在 sup-范数下不完备，闭包就是 \(C_0(\mathbb{R}^n)\)。在分布理论和测度论里它常作为"测试函数"使用。

注意 \(C[a,b]\)、\(C(K)\) 都是不可分是错的——\(K\) 是紧致度量空间时它们可分（用多项式或 Lipschitz 函数稠子集）。但 \(K\) 是更一般的紧致 Hausdorff 时不一定。本书绝大部分例子取 \(K\) 是闭区间或紧致欧氏域，所以可分性默认成立。

4.4 光滑函数空间 \(C^k[a,b]\) 与 \(C^\infty\)

对正整数 \(k\)，

\[ C^k[a,b] = \{f : [a,b] \to \mathbb{K}, f \text{ 是 } k \text{ 次连续可微}\}, \]

配范数

\[ \|f\|_{C^k} = \sum_{j=0}^{k} \|f^{(j)}\|_\infty. \]

这是 Banach 空间。完备性用归纳：\(C^0\) 完备 + 导数列一致收敛 \(\Rightarrow\) 极限可微且导数为极限，逐阶上推。

注意 sup-范数 \(\|f\|_\infty\) 单独不能让 \(C^k\) 完备——一致收敛极限的导数不一定是导数列的极限（典型反例 \(f_n(x) = \sin(n^2 x)/n\)）。这就是为什么 \(C^k\)-范数把所有阶导数打包在一起。

\(C^\infty[a,b] = \bigcap_k C^k[a,b]\) 不是 Banach 空间——它没有自然的单一范数让它完备。它是 Fréchet 空间（由可数族半范数定义的完备拓扑向量空间）。这是本书不深入讨论的"广义对象"，提一下作为方向标。

4.5 谁是 Hilbert，谁只是 Banach

把上面的空间按"是否 Hilbert"分类，对你以后看待这些空间会非常有用。

Hilbert 空间。\(\mathbb{R}^n\)（欧氏）、\(\mathbb{C}^n\)（标准内积）、\(\ell^2\)、\(L^2(\Omega, \mu)\)、\(H^1(\Omega)\)、Sobolev \(H^k\)。这是几何最干净的一类。

非 Hilbert 但自反的 Banach。\(\ell^p\)（\(1 < p < \infty\), \(p \neq 2\)）、\(L^p(\Omega, \mu)\)（\(1 < p < \infty\), \(p \neq 2\)）。它们不是 Hilbert，但还有"对偶的对偶等于自身"这个好性质，详见第 19 章。

Banach，不自反。\(\ell^1\)、\(L^1\)、\(c_0\)、\(C(K)\)、\(\ell^\infty\)、\(L^\infty\)。这些空间没有"投影到任意闭子空间"，对偶比自身大或者干脆很奇怪。它们仍然是非常重要的工具，但你做分析时要小心——很多在 \(L^2\) 下白送的论证在 \(L^1\) 或 \(L^\infty\) 上要换工具。

这个分类不是为了背名字，而是为了让你形成"防卫意识"。每次有人给你一个新结论，你应该自动反问：它要的是 Hilbert、还是自反、还是只要 Banach？这就是泛函分析的"假设审计"。

4.6 嵌入与对偶速览

把上面散落的嵌入和对偶关系整理成一张速查表，本书后面会反复回看。

嵌入：

  • \(\ell^p \subset \ell^q\)（\(p < q\)）；
  • \(L^q \subset L^p\)（\(p < q\)，有限测度时）；
  • \(C^{k+1} \subset C^k\)；
  • \(H^{k+1} \subset H^k\)；
  • \(C_c \subset C_0 \subset C_b\)（紧支撑 \(\subset\) 趋零 \(\subset\) 有界连续）；
  • Sobolev 嵌入 \(H^1(\Omega) \hookrightarrow L^p(\Omega)\)（视维数与边界条件而定，第 23 章）。

对偶：

  • \((\ell^p)^* = \ell^q\)（\(1 < p < \infty\), \(1/p + 1/q = 1\)）；
  • \((\ell^1)^* = \ell^\infty\)；
  • \((c_0)^* = \ell^1\)；
  • \((L^p)^* = L^q\)（\(1 \le p < \infty\), \(1/p + 1/q = 1\)）；
  • \((C(K))^* = M(K)\)（\(K\) 上 Radon 测度）；
  • \(H^* = H\)（Hilbert 空间，第 16 章 Riesz 表示）。

这张表你不必背，但应该知道在哪查。等做完第 13、15、16、19 章你就会自然记住。

下一章我们要把这些范数之间的不等式（Hölder、Minkowski、Cauchy–Schwarz、Bessel、Jensen）一次性证清。这些不等式不是技术装饰——上面这一长串"嵌入""对偶""完备" 几乎全部最终都归约到这几条不等式上。
