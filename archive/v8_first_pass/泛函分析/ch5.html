第 5 章 经典不等式

这一章把分析里几条最常见的不等式集中讲明白。它们看起来是"工具章"，但每一条不等式背后都有一个强的几何或概率含义。Cauchy–Schwarz 来自内积；Hölder 是 Cauchy–Schwarz 在 \(L^p\) 里的推广，本质是凸性；Minkowski 是 \(L^p\) 范数三角不等式的全部内容；Bessel 控制 Fourier 系数；Jensen 把所有"凸函数 + 平均"的关系统一成一行。把这五条都看清，你后面读 PDE、概率、调和分析时就再也不会被"为什么这一行不等式成立"卡住。

  • 从 Young 不等式（\(ab \le a^p/p + b^q/q\)）这条最朴素的凸性开始。
  • 用 Young 推出 Hölder 不等式，这是 \(L^p\) 配对的基本工具。
  • 用 Hölder 推出 Minkowski 不等式，这是 \(L^p\) 范数本身的三角不等式。
  • 把 Cauchy–Schwarz 重新放回这条链条，看它是 Hölder 的 \(p = q = 2\) 特例。
  • Bessel 不等式：把"投影到正交规范集"的几何写成一行控制。
  • Jensen 不等式：把所有凸函数 + 平均的关系一并打包，是 Hölder 的"概率版"。

5.1 Young 不等式：凸性的最朴素形式

Young 不等式说：对 \(1 < p, q < \infty\) 满足 \(1/p + 1/q = 1\)，以及任意 \(a, b \ge 0\)，

\[ \boxed{\ ab \le \frac{a^p}{p} + \frac{b^q}{q}.\ } \]

等号成立 \(\iff a^p = b^q\)。

证明用 \(\log\) 凹性。\(\log\) 是凹函数，于是对 \(1/p + 1/q = 1\)：

\[ \log\left( \frac{a^p}{p} + \frac{b^q}{q} \right) \ge \frac{1}{p} \log a^p + \frac{1}{q} \log b^q = \log a + \log b = \log(ab). \]

取指数即得 Young。

这条小不等式是后面所有 \(L^p\) 不等式的原子。它的对称版（任何凸函数 + 共轭凸函数都有类似估计，叫做 Fenchel–Young 不等式）是凸优化、对偶分析整个分支的源头。本书暂时只用最简单的 \(a^p/p + b^q/q\) 这一条。

5.2 Hölder 不等式

对 \(1 \le p, q \le \infty\) 满足 \(1/p + 1/q = 1\)（约定 \(1/\infty = 0\)），与任意 \(f \in L^p\)、\(g \in L^q\)：

\[ \boxed{\ \|fg\|_1 = \int |fg| \, d\mu \le \|f\|_p \cdot \|g\|_q.\ } \]

序列版本：\(\sum_n |x_n y_n| \le \|x\|_p \|y\|_q\)。

证明（\(1 < p < \infty\) 情形）：先齐次化，假设 \(\|f\|_p = \|g\|_q = 1\)。对每个 \(x\) 应用 Young：

\[ |f(x) g(x)| \le \frac{|f(x)|^p}{p} + \frac{|g(x)|^q}{q}. \]

两边对 \(x\) 积分：

\[ \int |fg| \, d\mu \le \frac{1}{p} \int |f|^p + \frac{1}{q} \int |g|^q = \frac{1}{p} + \frac{1}{q} = 1 = \|f\|_p \|g\|_q. \]

结束。\(p = 1, q = \infty\) 端点情形直接 \(|fg| \le \|g\|_\infty |f|\) 后积分。

Hölder 不等式可以理解成"\(L^p\) 和它的对偶 \(L^q\) 之间的配对受范数控制"。这是为什么对偶配对 \(\langle f, g \rangle = \int fg\) 在 \(L^p \times L^q\) 上良定义，并且

\[ |\langle f, g \rangle| \le \|f\|_p \|g\|_q. \]

这是第 15 章对偶空间 \((L^p)^* = L^q\) 等距同构的核心估计。

Hölder 的几个常用变体：

  • 三因子 Hölder：\(1/p + 1/q + 1/r = 1\) 时 \(\int |fgh| \le \|f\|_p \|g\|_q \|h\|_r\)；
  • 插值 Hölder：对 \(1/p = \theta/p_0 + (1-\theta)/p_1\)（\(0 < \theta < 1\)），\(\|f\|_p \le \|f\|_{p_0}^\theta \|f\|_{p_1}^{1-\theta}\)；
  • 反向 Hölder（在某些场景下成立）：需要额外控制条件，本书不深入。

5.3 Minkowski 不等式

Minkowski 不等式就是 \(L^p\) 范数的三角不等式：

\[ \boxed{\ \|f + g\|_p \le \|f\|_p + \|g\|_p, \quad 1 \le p \le \infty.\ } \]

证明（\(1 < p < \infty\)）：令 \(h = f + g\)。

\[ |h|^p = |h| \cdot |h|^{p-1} \le (|f| + |g|) |h|^{p-1}. \]

对两边积分，对每一项用 Hölder（\(p\) 与 \(q = p/(p-1)\)）：

\[ \int |f| |h|^{p-1} \le \|f\|_p \cdot \||h|^{p-1}\|_q = \|f\|_p \cdot \|h\|_p^{p-1}. \]

同样估计 \(\int |g| |h|^{p-1}\)。合起来

\[ \|h\|_p^p \le (\|f\|_p + \|g\|_p) \|h\|_p^{p-1}. \]

两边除以 \(\|h\|_p^{p-1}\)（若为 0 则 \(h = 0\) 已立）即得 Minkowski。

有了 Minkowski，\(L^p\) 真的是赋范空间这件事才落实。换句话说，\(\ell^p\) 与 \(L^p\) 是 Banach 空间的全部根基都在这条 Minkowski。

序列版本同理：\(\|x + y\|_p \le \|x\|_p + \|y\|_p\)。

5.4 Cauchy–Schwarz 在这条链条里的位置

第 3 章证过 Cauchy–Schwarz：\(|\langle x, y \rangle| \le \|x\| \|y\|\)。它其实就是 Hölder 在 \(p = q = 2\) 的特例（在 \(L^2\) 上）：

\[ \left| \int f \bar g \right| \le \int |f \bar g| \le \|f\|_2 \|g\|_2. \]

所以这条不等式有两个来源：从内积公理（第 3 章用判别式法）或从 Hölder（本章）。两种证法各有意义——第一种说"Cauchy–Schwarz 是 Hilbert 空间的内禀性质"，第二种说"它是 Hölder 链条的一环"。两种眼光都要会。

5.5 Bessel 不等式：把"部分和投影"控制住

设 \(H\) 是 Hilbert 空间，\(\{e_n\}_{n=1}^N\) 是有限正交规范集。对任意 \(x \in H\)，令

\[ \hat x_n = \langle x, e_n \rangle, \qquad y_N = \sum_{n=1}^N \hat x_n e_n. \]

这个 \(y_N\) 是"\(x\) 在 \(\operatorname{span}\{e_1, \ldots, e_N\}\) 上的投影"（具体证明在第 6 章）。Pythagoras：

\[ \|x\|^2 = \|x - y_N\|^2 + \|y_N\|^2 = \|x - y_N\|^2 + \sum_{n=1}^N |\hat x_n|^2. \]

从中立刻得

\[ \boxed{\ \sum_{n=1}^N |\langle x, e_n \rangle|^2 \le \|x\|^2.\ } \]

这就是 Bessel 不等式。它对任何（可数甚至不可数）正交规范集都成立——只取那些 \(\langle x, e_\alpha \rangle \neq 0\) 的项即可，自动只剩可数个非零（这是 Bessel 自带的"可数支撑"奇迹）。

Bessel 不等式的应用：

  • Fourier 系数列在 \(\ell^2\) 中。把 \(\{e^{inx}/\sqrt{2\pi}\}\) 当成 \(L^2[-\pi, \pi]\) 上的正交规范集（不一定是基），Bessel 立即给 \(\sum |\hat f(n)|^2 \le \|f\|_2^2 / (2\pi)\)。
  • 正交规范集的"投影列"控制。第 7 章会用 Bessel 推出 Parseval：当 \(\{e_n\}\) 是完备正交基时，Bessel 退化成等式。

Bessel 是不等式，Parseval 是等式。两者的差距 \(\|x\|^2 - \sum |\langle x, e_n \rangle|^2\) 就是"投影外的剩余"，等于 0 当且仅当 \(\{e_n\}\) 真的够用——也就是它是完备正交基。这正是第 7 章要说穿的事。

5.6 Jensen 不等式：凸函数 + 平均

设 \(\varphi : \mathbb{R} \to \mathbb{R}\) 是凸函数，\(X\) 是概率空间上的随机变量（或测度空间上的可积函数），\(\mathbb{E}\) 是期望（积分）。Jensen 不等式说：

\[ \boxed{\ \varphi(\mathbb{E} X) \le \mathbb{E}[\varphi(X)].\ } \]

等号当 \(\varphi\) 严格凸时成立 \(\iff X\) 几乎必然为常数。

证明用支撑线（subgradient）。在 \(\mathbb{E} X\) 处取 \(\varphi\) 的支撑线 \(L(t) = \varphi(\mathbb{E} X) + c (t - \mathbb{E} X)\)，由凸性 \(\varphi(t) \ge L(t)\) 对所有 \(t\)。代入 \(t = X\) 取期望：

\[ \mathbb{E}[\varphi(X)] \ge \mathbb{E}[L(X)] = \varphi(\mathbb{E} X) + c (\mathbb{E} X - \mathbb{E} X) = \varphi(\mathbb{E} X). \]

结束。

Jensen 不等式有非常多的应用：

  • 取 \(\varphi(t) = t^2\)：\((\mathbb{E} X)^2 \le \mathbb{E} X^2\)，即 \(\operatorname{Var}(X) \ge 0\)。
  • 取 \(\varphi(t) = -\log t\)（在 \(t > 0\) 上凹反向应用）：得到 AM–GM 不等式 \(\sqrt[n]{x_1 \cdots x_n} \le (x_1 + \cdots + x_n)/n\)。
  • 取 \(\varphi(t) = e^t\)：得到 \(e^{\mathbb{E} X} \le \mathbb{E} e^X\)，是矩生成函数控制 cumulant 的根源。
  • Hölder 的概率证明：对 \(\varphi(t) = t^p\)（\(p \ge 1\)）应用 Jensen 即得 \(L^p\) 范数的几条插值不等式。

Jensen 不等式说穿了就是"凸函数图像在弦下方"这件事的积分版。这一句话在概率、信息论、统计、优化、PDE 里反复出现。

到这里你已经握住分析里最常用的几条不等式。它们之间的关系：Young 是凸性的微观形式 → Hölder 是它的"配对"形式 → Minkowski 是 Hölder 的三角推论。Cauchy–Schwarz 和 Bessel 在 Hilbert 上独立讲，但其实就是 \(p = q = 2\) 时的 Hölder 与 \(\ell^2\) 中的 Pythagoras。Jensen 把所有"凸函数和平均"的关系打包。

这五条工具备齐之后，下一章我们就可以把 Hilbert 空间的"几何"正式展开——投影到闭子空间、正交分解、最佳逼近。这些是泛函分析里最干净也最容易"看见"的几何，也是后面所有谱论、Fourier 展开、变分法的根。
