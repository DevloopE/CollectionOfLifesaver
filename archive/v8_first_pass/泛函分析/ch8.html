第 8 章 有界线性算子

到此为止我们一直在研究"空间"。从这一章开始转向算子——在赋范空间之间的线性映射。第 1 章里我们就提到过：在无穷维里，"线性"和"连续"不再自动等价，这是有限维和无穷维最大的差别之一。本章把这件事彻底说清，并把"算子范数 \(\|T\|\)"这个泛函分析里出现频率最高的对象之一引出来。

核心结论非常简单：线性映射连续 \(\iff\) 有界 \(\iff\) Lipschitz \(\iff\) 在原点连续。这四个条件互相等价，构成"我们想要的算子"的全部刻画。掌握之后，你看到任何线性算子都会立即想到"它有界吗？算子范数是多少？"——这就是泛函分析人的反射动作。

  • 线性算子的连续性等价于有界性，这是最基础也最重要的等价。
  • 算子范数 \(\|T\|\) 的三个等价定义，每个在不同场合都有用。
  • 典型例子：积分算子、乘子算子、移位算子、求导算子（再次！）。
  • 有限维下"所有线性算子都有界"的证明，并对比无穷维的失败。
  • BLT 定理（bounded linear extension）：在稠密子空间上定义的有界算子可唯一延拓到全空间。
  • "连续 + 线性"作为后面所有章的标准前提条件。

8.1 线性、有界、连续：四条等价

设 \(X, Y\) 是赋范空间，\(T : X \to Y\) 是线性映射。下列四条等价：

  • (C1) \(T\) 在某点 \(x_0\) 连续；
  • (C2) \(T\) 在 \(0\) 连续（即 \(x_n \to 0 \Rightarrow T x_n \to 0\)）；
  • (C3) \(T\) 全空间 Lipschitz：存在 \(L\) 使 \(\|T(x_1) - T(x_2)\| \le L \|x_1 - x_2\|\)；
  • (C4) \(T\) 有界：存在常数 \(C\) 使 \(\|Tx\|_Y \le C \|x\|_X\) 对所有 \(x \in X\)。

"有界"在线性算子语境下的意思和"实变函数"里完全不同——它不是说 "\(T\) 的值落在某个有界集"（这对线性算子只在 \(T = 0\) 时成立！）。它的意思是 "\(T\) 把单位球映成有界集"，等价于"\(T\) 把范数线性放大不超过常数倍"。

等价证明：

(C1) \(\Rightarrow\) (C2)：由线性，"在 \(x_0\) 处连续 + \(x_n \to 0\)"\(\iff\) "\(x_n + x_0 \to x_0 \Rightarrow T(x_n + x_0) \to T(x_0)\)"\(\iff\) "\(T x_n + T x_0 \to T x_0\)"\(\iff\) "\(T x_n \to 0\)"。

(C2) \(\Rightarrow\) (C4)：反证。假设 \(T\) 不有界。则存在 \(x_n \neq 0\) 使 \(\|T x_n\| / \|x_n\| \to \infty\)。令 \(y_n = x_n / (n \|x_n\|)\)，则 \(\|y_n\| = 1/n \to 0\)，但 \(\|T y_n\| = \|T x_n\|/(n \|x_n\|) \to \infty\)，矛盾 (C2)。

(C4) \(\Rightarrow\) (C3)：\(\|T(x_1) - T(x_2)\| = \|T(x_1 - x_2)\| \le C \|x_1 - x_2\|\)。

(C3) \(\Rightarrow\) (C1)：Lipschitz \(\Rightarrow\) 全局连续。

这条四等价是泛函分析最常引用的事实之一。它的实际意义是：以后你证一个线性算子连续，只需要证它在原点连续，或者直接证 \(\|Tx\| \le C \|x\|\)；一旦其中之一成立，全部别的都自动有。

8.2 算子范数：三种等价定义

对有界线性算子 \(T : X \to Y\)，定义

\[ \boxed{\ \|T\| := \sup_{x \neq 0} \frac{\|Tx\|_Y}{\|x\|_X} = \sup_{\|x\|_X \le 1} \|Tx\|_Y = \sup_{\|x\|_X = 1} \|Tx\|_Y.\ } \]

这三个定义彼此等价（齐次性立得），各有用处：第一个适合代数操作（直接给出 \(\|Tx\| \le \|T\| \|x\|\)），第二个适合涉及"闭单位球"的论证（紧致性、弱收敛），第三个适合具体计算。

立即得几条标准事实：

  • \(\|Tx\|_Y \le \|T\| \cdot \|x\|_X\) 对所有 \(x\)；
  • \(\|T\| = 0 \iff T = 0\)；
  • \(\|\alpha T\| = |\alpha| \|T\|\)；
  • \(\|S + T\| \le \|S\| + \|T\|\)（三角，用 \(\|(S+T)x\| \le \|Sx\| + \|Tx\| \le (\|S\|+\|T\|)\|x\|\)）；
  • 对复合 \(T_2 T_1\)（\(T_1 : X \to Y\)、\(T_2 : Y \to Z\)）：\(\|T_2 T_1\| \le \|T_2\| \cdot \|T_1\|\)。

最后一条会在第 9 章成为算子代数的"乘法范数性"。

8.3 标准例子库

下面把出现频率最高的几类算子讲透，让你以后看到"积分算子""乘子算子""移位算子""求导算子"立刻有图像。

例 1（恒等算子 \(I\)）。\(\|I\| = 1\)（除非空间是 \(\{0\}\)）。

例 2（数乘算子 \(\alpha I\)）。\(\|\alpha I\| = |\alpha|\)。

例 3（乘子算子）。设 \(\varphi \in L^\infty(\Omega)\)。映射

\[ M_\varphi : L^p(\Omega) \to L^p(\Omega), \qquad M_\varphi f = \varphi f. \]

是有界的，\(\|M_\varphi\| = \|\varphi\|_\infty\)（这是个值得自己核算的练习——上界来自 \(\|\varphi f\|_p \le \|\varphi\|_\infty \|f\|_p\)，下界由"取 \(f\) 集中在 \(|\varphi|\) 接近最大的点"实现）。乘子算子是后面"乘法 + Fourier 变换"很多技巧的核心。

例 4（积分算子）。设 \(K : [0,1]^2 \to \mathbb{R}\) 连续。定义

\[ T : C[0,1] \to C[0,1], \qquad (Tf)(x) = \int_0^1 K(x, y) f(y) \, dy. \]

有界，且 \(\|T\| \le \max_{x} \int_0^1 |K(x,y)| \, dy =: \|K\|_{L^\infty L^1}\)（一般还小于这个上界）。这种 Fredholm 型积分算子在第 22 章会重新登场。

例 5（左移与右移）。在 \(\ell^2\) 上：

  • 右移 \(R(x_1, x_2, \ldots) = (0, x_1, x_2, \ldots)\)：保范数，\(\|R\| = 1\)，单射不满射；
  • 左移 \(L(x_1, x_2, \ldots) = (x_2, x_3, \ldots)\)：\(\|L\| = 1\)，满射但有非平凡核（\(L e_1 = 0\)）。

注意 \(L R = I\) 但 \(R L \neq I\)（\(R L\) 把第一位压成 0）。无穷维"左逆"和"右逆"不一致——这是有限维直觉的关键失效之一。

例 6（求导算子的"重伤")。定义在 \((C^1[0,1], \|\cdot\|_\infty)\) 到 \((C[0,1], \|\cdot\|_\infty)\) 上，求导不有界（第 1 章 1.2 节已证）。但若把定义域换成 \((C^1[0,1], \|\cdot\|_{C^1})\)，求导就有界，且 \(\|D\| \le 1\)。这再次说明"在哪个范数下看"是泛函分析里要永远先问的事。

\[ \boxed{\ \text{算子是否有界，由"输入空间范数 + 输出空间范数"共同决定，与算子的代数表达式无关。}\ } \]

8.4 有限维：一切都自动有界

定理。若 \(X\) 有限维，则任意线性映射 \(T : X \to Y\) 都有界。

证明：取 \(X\) 的一组基 \(\{u_1, \ldots, u_n\}\)。\(X\) 上所有范数等价（这是有限维"任性"事实，附录 A给完整证明），所以等价于在"基坐标范数" \(\|x\| = \max_i |c_i|\)（\(x = \sum c_i u_i\)）下证。

\[ \|T x\|_Y = \left\| \sum c_i T u_i \right\|_Y \le \sum |c_i| \cdot \|T u_i\|_Y \le n \cdot \max_i |c_i| \cdot \max_i \|T u_i\|_Y = n \cdot \|x\| \cdot M, \]

其中 \(M = \max_i \|T u_i\|_Y\)。所以 \(T\) 有界，\(\|T\| \le n M\)。

这条定理就是为什么有限维线性代数从来不谈连续性。所有事情自动连续。一旦走出有限维，这件事不再自动——这是泛函分析"分析"那一半的根源。

8.5 BLT 定理：稠密子空间上定义后延拓

在实际工作中，你常常先在一个稠密子空间上定义算子（因为它在那里有简单公式），然后想把它延拓到整个空间。BLT 定理（bounded linear transformation extension）告诉你这件事什么时候能做，怎么做。

定理（BLT / 唯一连续延拓）。设 \(X\) 是赋范空间，\(D \subset X\) 是稠子空间（线性子集，张成 \(X\)）。设 \(Y\) 是 Banach 空间，\(T : D \to Y\) 是有界线性算子。则 \(T\) 唯一延拓为 \(\widetilde T : X \to Y\) 上的有界线性算子，且

\[ \boxed{\ \|\widetilde T\| = \|T\|.\ } \]

证明思路：对 \(x \in X\)，取 \(D\) 中序列 \(d_n \to x\)。则 \(\{d_n\}\) 是 Cauchy 序列，\(T\) 有界 \(\Rightarrow\) \(\{T d_n\}\) 在 \(Y\) 中也是 Cauchy；\(Y\) 完备 \(\Rightarrow\) \(T d_n \to y\) 某个 \(y \in Y\)。定义 \(\widetilde T(x) = y\)。可以验证这个定义不依赖 \(d_n\) 的选择（用两个序列差的 Cauchy 性），且线性、有界、范数保持。

BLT 定理是泛函分析里最常用的"延拓"工具。它的几个典型应用：

  • Fourier 变换最先在 Schwartz 类上有简单定义，然后用 BLT 延拓到 \(L^2(\mathbb{R})\)；
  • Lebesgue 积分最先对简单函数有定义，然后用 BLT 延拓到 \(L^1\)（更准确的说法是单调收敛 + 控制收敛，但思路一致）；
  • 第 13 章 Hahn–Banach 在某些情形下也是"延拓"思路的几何版本。

注意：\(Y\) 必须完备。BLT 这条结论的全部技术核心就是"\(Y\) 完备 \(\Rightarrow\) 极限存在"。如果 \(Y\) 不完备，序列 \(\{T d_n\}\) 是 Cauchy 但可能漏出去，延拓就做不出。

8.6 "连续 + 线性"是后面所有章的标准前提

从这一章开始，本书提到"算子" \(T : X \to Y\) 时，默认 它是有界的（即连续的、线性的）。这种约定在文献里也很普遍，但你应该意识到这个约定是个假设，而不是定理给你的。

当我们偶尔需要研究无界算子（比如求导算子 \(D\) 在 \(L^2\) 上、量子力学里的位置算子 \(x\)、动量算子 \(-i \partial_x\)），我们会专门标注。这是第 21 章末尾和第 24 章的话题——无界算子需要专门的"定义域"管理工具，比如自伴扩张、谱测度。这些是泛函分析"硬核"那一面，本书会给一个入门级介绍。

本章的整套语言可以总结成一句话："我们要研究的算子"\(=\) \(\mathcal{B}(X, Y) := \{T : X \to Y \text{ 线性 + 有界}\}\)。下一章我们会发现 \(\mathcal{B}(X, Y)\) 自己也是一个 Banach 空间，且 \(\mathcal{B}(X) := \mathcal{B}(X, X)\) 是一个"Banach 代数"——它有完备范数 + 良好乘法。这个代数结构是后面谱论、紧算子、解析函数演算的根基。
