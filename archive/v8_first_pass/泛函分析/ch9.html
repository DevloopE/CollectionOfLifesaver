第 9 章 算子范数与算子代数

第 8 章我们把"什么算子值得研究"挑出来：有界线性算子。这一章把"所有有界算子"作为一个整体研究——它本身是一个赋范空间，事实上是一个 Banach 空间，并且当 \(X = Y\) 时还是一个Banach 代数。这件事的实际意义是：算子也是分析对象，也可以"取极限""做级数展开""谈乘法"——这些操作组成了后面谱论、解析函数演算、紧算子的全部技术基础。

  • 把 \(\mathcal{B}(X, Y) = \{T : X \to Y \text{ 有界线性}\}\) 作为赋范空间组织起来。
  • 证 \(\mathcal{B}(X, Y)\) 是 Banach 空间——只要 \(Y\) 是。这条结论看似温和，却是后面所有"算子序列收敛"的根基。
  • 三种算子序列收敛：算子范数收敛、强收敛、弱收敛。三者在无穷维彼此不等价。
  • \(\mathcal{B}(X)\) 作为 Banach 代数：有结合乘法、有单位元 \(I\)、范数与乘法相容 \(\|ST\| \le \|S\| \|T\|\)。
  • Neumann 级数：当 \(\|T\| < 1\) 时 \(I - T\) 可逆，且 \((I - T)^{-1} = \sum T^n\)。这是分析里"几何级数"的算子版。
  • 谱半径公式预告 \(\rho(T) = \lim \|T^n\|^{1/n}\)——第 21 章会真正用到，这里先把它写在视野里。

9.1 \(\mathcal{B}(X, Y)\) 作为赋范空间

设 \(X, Y\) 是赋范空间。集合

\[ \mathcal{B}(X, Y) = \{T : X \to Y \text{ 线性 + 有界}\} \]

在加法 \((S + T)(x) = Sx + Tx\) 与数乘 \((\alpha T)(x) = \alpha (Tx)\) 下是向量空间。算子范数 \(\|T\|\)（第 8 章定义）让它成为赋范空间。这一步几乎是字面验证，但不能跳过——你需要确认：

  • \(\|T\| \ge 0\)；\(\|T\| = 0 \iff T = 0\)；
  • \(\|\alpha T\| = |\alpha| \|T\|\)；
  • \(\|S + T\| \le \|S\| + \|T\|\)。

这三条已经在第 8 章 8.2 节验过。所以 \(\mathcal{B}(X, Y)\) 配算子范数是赋范空间。

9.2 \(\mathcal{B}(X, Y)\) 是 Banach 空间

这是本章的第一条结构定理：

定理。若 \(Y\) 是 Banach 空间（\(X\) 只需赋范），则 \(\mathcal{B}(X, Y)\) 在算子范数下是 Banach 空间。

证明：取 Cauchy 序列 \(\{T_n\} \subset \mathcal{B}(X, Y)\)。对每个固定 \(x \in X\)：

\[ \|T_n x - T_m x\|_Y \le \|T_n - T_m\| \cdot \|x\|_X \to 0, \]

所以 \(\{T_n x\}\) 是 \(Y\) 中的 Cauchy 序列。\(Y\) 完备 \(\Rightarrow\) 存在 \(T x := \lim T_n x \in Y\)。

线性：由极限算子保线性性。

有界：由 \(\{T_n\}\) Cauchy \(\Rightarrow\) \(\|T_n\|\) 有界（设 \(\le M\)）；取极限 \(\|T x\|_Y \le M \|x\|_X\)。所以 \(T \in \mathcal{B}(X, Y)\)。

范数收敛：对任意 \(\varepsilon > 0\)，取 \(N\) 使 \(\|T_n - T_m\| < \varepsilon\) 对 \(n, m \ge N\)。固定 \(x\) 与 \(n \ge N\)，让 \(m \to \infty\)：

\[ \|T_n x - T x\|_Y \le \varepsilon \|x\|_X. \]

除以 \(\|x\|_X\) 取上确界：\(\|T_n - T\| \le \varepsilon\) 对 \(n \ge N\)。即 \(T_n \to T\) 在算子范数下。结束。

这条定理至关重要：它让"算子的极限是算子"成为可靠操作。后面所有用 Neumann 级数、谱投影、半群指数函数 \(e^{tA}\) 的论证都要靠它。

9.3 算子序列的三种收敛

无穷维里"算子序列收敛"有三种自然定义，彼此不等价：

(U) 算子范数收敛 / 一致收敛：\(T_n \to T\) 在 \(\mathcal{B}(X, Y)\) 中，即 \(\|T_n - T\| \to 0\)。

(S) 强收敛 / 逐点收敛：对每个 \(x \in X\)，\(T_n x \to T x\) 在 \(Y\) 中。

(W) 弱收敛：对每个 \(x \in X\) 与每个 \(y^* \in Y^*\)，\(\langle y^*, T_n x \rangle \to \langle y^*, T x \rangle\)。

三种收敛之间的强弱关系是 \(\text{(U)} \Rightarrow \text{(S)} \Rightarrow \text{(W)}\)。反向蕴含都不成立——本节给反例。

反例 (S) 不蕴含 (U)：投影到前 \(n\) 个坐标。在 \(\ell^2\) 上定义 \(P_n x = (x_1, x_2, \ldots, x_n, 0, 0, \ldots)\)。每个 \(x \in \ell^2\) 满足 \(P_n x \to x\)（\(L^2\) 收敛 = 余尾趋零），所以 \(P_n \to I\) 强收敛。但 \(\|I - P_n\| = 1\)（取 \(x = e_{n+1}\)），所以不是算子范数收敛。

反例 (W) 不蕴含 (S)：移位算子。在 \(\ell^2\) 上 \(R\) 是右移。考虑 \(R^n e_1 = e_{n+1}\)。这个序列不强收敛（\(\|R^n e_1\| = 1\) 不趋零；事实上不存在 \(\ell^2\) 中的极限）。但弱收敛：对任意 \(y \in \ell^2\)，\(\langle y, R^n e_1 \rangle = \overline{y_{n+1}} \to 0\)（因为 \(\sum |y_k|^2 < \infty\) 蕴含 \(y_k \to 0\)）。所以 \(R^n e_1 \to 0\) 弱收敛但不强收敛。

这三种收敛各有用处：

  • 算子范数收敛是最强的，分析里非常稀有，常用在 Neumann 级数、解析函数演算；
  • 强收敛是默认收敛模式，定义"算子序列趋近极限算子"的最自然方式；
  • 弱收敛常出现在变分法、紧算子、Banach–Alaoglu 等话题（第 17 章会专门展开）。

9.4 \(\mathcal{B}(X)\) 作为 Banach 代数

当 \(X = Y\) 时，\(\mathcal{B}(X) := \mathcal{B}(X, X)\) 上多了一个自然的乘法——算子复合：

\[ (ST)(x) = S(T x). \]

这个乘法满足：

  • 结合：\((ST)U = S(TU)\)；
  • 左/右分配：\(S(T_1 + T_2) = S T_1 + S T_2\)、\((S_1 + S_2) T = S_1 T + S_2 T\)；
  • 数乘相容：\(\alpha (ST) = (\alpha S) T = S(\alpha T)\)；
  • 单位元 \(I\)：\(IT = TI = T\)；
  • 子乘法范数性：\(\|ST\| \le \|S\| \cdot \|T\|\)。

最后一条值得专门看一眼。它的证明就一行：\(\|(ST)x\| = \|S(Tx)\| \le \|S\| \|Tx\| \le \|S\| \|T\| \|x\|\)。但这条不等式有非常深的后果——它就是"Banach 代数"这个概念的唯一定义性条件之外的内容。

定义。一个Banach 代数 \(A\) 是一个 Banach 空间 + 一个结合乘法，满足子乘法范数性 \(\|ab\| \le \|a\| \|b\|\)。如果还有单位元 \(1\) 满足 \(\|1\| = 1\)，叫带单位 Banach 代数。

\[ \boxed{\ \mathcal{B}(X) \text{（} X \text{ Banach）是带单位 Banach 代数。}\ } \]

这条结构性结论就是谱论的舞台。算子的"代数运算"和"分析运算"被打包在一起，使得我们能给一个算子写下它的"幂级数函数"\(f(T)\)、"指数" \(e^T\)、"逆" \(T^{-1}\)、"谱" \(\sigma(T)\) 等等。这些工具都需要 Banach 代数结构。

9.5 Neumann 级数：算子版的几何级数

这是本章最实用的一条结论：

定理（Neumann 级数）。设 \(X\) 是 Banach 空间，\(T \in \mathcal{B}(X)\)，\(\|T\| < 1\)。则 \(I - T\) 可逆，且

\[ \boxed{\ (I - T)^{-1} = \sum_{n=0}^\infty T^n, \qquad \|(I - T)^{-1}\| \le \frac{1}{1 - \|T\|}.\ } \]

证明几乎是字面把"\(1/(1 - r) = \sum r^n\)"搬到算子上。子乘法范数性给 \(\|T^n\| \le \|T\|^n\)，所以 \(\sum \|T^n\| \le \sum \|T\|^n = 1/(1 - \|T\|) < \infty\)（因为 \(\|T\| < 1\)）。\(\mathcal{B}(X)\) 完备 \(\Rightarrow\) 绝对收敛级数收敛（第 2 章 2.5 节判据），所以 \(S := \sum_{n=0}^\infty T^n \in \mathcal{B}(X)\)。再核算 \(S(I - T) = (I - T) S = I\)。范数估计直接由级数。

Neumann 级数的几个典型应用：

  • 积分方程的逐次逼近 / Picard 迭代。方程 \(f - K f = g\)（\(K\) 是 \(\|K\| < 1\) 的积分算子）有唯一解 \(f = (I - K)^{-1} g = \sum K^n g\)。这是微分方程读本里 Picard 存在唯一性的几何重写——它的核心就是 Banach 代数 + Neumann 级数。
  • "小扰动"理论。若 \(A\) 可逆且 \(B\) 是小扰动（\(\|A^{-1} B\| < 1\)），则 \(A + B = A(I + A^{-1} B)\) 仍可逆，因为 \(I + A^{-1} B = I - (-A^{-1} B)\) 可逆。这给出"可逆算子集是开集"的关键事实——在第 21 章谱论里这条会反复用。
  • 谱半径估计。Neumann 级数对 \(\|T\| < 1\) 起作用，但实际真正起作用的常数是谱半径：\(I - T\) 可逆当且仅当 \(1 \notin \sigma(T)\)，等价于 \(\rho(T) < 1\)。下一节先把谱半径的"具体公式"指出来。

9.6 谱半径公式预告

对 \(T \in \mathcal{B}(X)\)，定义谱半径

\[ \rho(T) := \inf_{n \ge 1} \|T^n\|^{1/n} = \lim_{n \to \infty} \|T^n\|^{1/n}. \]

下确界与极限相等是个不平凡的事实（用次乘法序列的"Fekete 引理"，详细在附录 A）。

\[ \boxed{\ \rho(T) = \lim_{n \to \infty} \|T^n\|^{1/n} \le \|T\|.\ } \]

不等号 \(\rho(T) \le \|T\|\) 来自次乘法（\(\|T^n\|^{1/n} \le \|T\|^{n/n} = \|T\|\)）。值得知道的是 \(\rho(T)\) 可以严格小于 \(\|T\|\)：比如幂零矩阵的谱半径为 0 但范数可能很大。这种情形在无穷维同样出现。

谱半径的几何含义在第 21 章会被正式给出：它就是"复平面上 \(\sigma(T)\) 的最大模"——也就是说

\[ \rho(T) = \max\{|\lambda| : \lambda \in \sigma(T)\}. \]

这条等式叫Gelfand 谱半径公式，它的证明本质上是给 \((\lambda I - T)^{-1}\) 当成 \(\lambda\) 的解析函数处理（用复分析里 Liouville 定理 \(\Rightarrow\) 谱非空）。这是泛函分析与复分析最优雅的交叉点之一。

本章到此把"算子代数"的语言备齐：算子范数、三种收敛、子乘法范数性、Neumann 级数、谱半径。这些工具会在第 20 章紧算子、第 21 章谱定理、第 22 章 Fredholm、第 24 章量子力学算子里反复使用。
