第 12 章 理想与商环

群论里我们花了整整第 5 章把"正规子群 → 商群"这个故事讲完：正规子群恰好是那种能让"陪集集合配诱导运算"也成为群的子群。环论里有一个完全平行的故事——"理想 → 商环"。理想恰好是那种能让"陪集集合配诱导加法和乘法"也成为环的子集。

但环比群多一种运算，理想比正规子群多一条要求："吸收乘法"——理想里的元素乘上环里任何东西都还要落回理想里。这条多出来的"吸收"才是环的商能存在的真正原因。把这件事弄明白，整个环论的同构定理 / 极大理想 / 素理想 / 谱（spectrum）的语言就一气呵成。

  • 理想的定义（左、右、双侧）；交换环里三者一致。
  • 第一组例子：\((n) \subset \mathbb{Z}\)、\((x) \subset R[x]\)、矩阵环里的"行 / 列零理想"。
  • 主理想 \((a) = aR\)；理想的和、积、交。
  • 商环 \(R/I\) 的构造：陪集 \(r + I\) 配良定义的加 / 乘。
  • 第一同构定理（环版）；其它两个同构定理同样成立。
  • 极大理想 \(\mathfrak{m}\)：\(R/\mathfrak{m}\) 是域 \(\iff \mathfrak{m}\) 极大。
  • 素理想 \(\mathfrak{p}\)：\(R/\mathfrak{p}\) 是整环 \(\iff \mathfrak{p}\) 素。
  • 实例：\(\mathbb{Z}\) 中 \((p)\) 极大；\(\mathbb{Z}[i]\) 中 \((2) = (1+i)^2\) 不素。

12.1 理想的定义

设 \(R\) 是环。子集 \(I \subset R\) 称为左理想，如果：

  • (I1) \((I, +)\) 是 \((R, +)\) 的子群（即对加、减封闭、含 0）；
  • (I2L) 左吸收：对任意 \(r \in R\) 与 \(a \in I\)，\(ra \in I\)。

对称地，如果 (I2L) 替换为 \(ar \in I\)，就叫右理想。如果两条吸收同时成立，就叫双侧理想，简称理想。在交换环里左 = 右 = 双侧，三者完全是一回事——所以本书的核心场景里讲"理想"就是"双侧理想"，毫无歧义。

把这条写成一个 boxed 公式：

\[ \boxed{\ I \text{ 是 } R \text{ 的理想} \iff I \le (R, +) \text{ 且 } RI \subset I, IR \subset I.\ } \]

第一条是"加法子结构封闭"，第二条是"乘法吸收"。两者缺一不可。注意这条 (I2) 比"对乘法封闭"强得多——一个子环只要求 \(I \cdot I \subset I\)，理想要求 \(R \cdot I \subset I\)（外面整个环都吃不动它）。这就是为什么大多数子环不是理想：比如 \(\mathbb{Z} \subset \mathbb{Q}\) 是子环，但 \(\mathbb{Z}\) 不是 \(\mathbb{Q}\) 的理想（\(\frac{1}{2} \cdot 1 = \frac{1}{2} \notin \mathbb{Z}\)）。

顺便：\(\{0\}\) 是任何环的理想，叫零理想；\(R\) 本身也是 \(R\) 的理想，叫单位理想。这两个叫"平凡理想"。

关于"理想里有没有 1"——从不。因为如果 \(1 \in I\)，则对任意 \(r \in R\)，\(r = r \cdot 1 \in I\)，故 \(I = R\)。所以 "\(I\) 是真理想（\(I \neq R\)）" \(\iff 1 \notin I\) \(\iff I\) 不含任何单位（\(I\) 含单位 \(\Rightarrow I\) 含 1）。

12.2 第一组例子

\((n) \subset \mathbb{Z}\)。对每个 \(n \ge 0\)，\(n \mathbb{Z} = \{0, \pm n, \pm 2n, \ldots\}\) 是 \(\mathbb{Z}\) 的理想：对加减封闭显然，吸收 \(\mathbb{Z} \cdot (n\mathbb{Z}) \subset n\mathbb{Z}\) 也直接。事实上所有 \(\mathbb{Z}\) 的理想都长这样——这是 \(\mathbb{Z}\) 的"主理想环"性质，详见第 14 章。具体证明：取 \(\mathbb{Z}\) 的非零理想 \(I\)，令 \(n = \min\{k > 0 : k \in I\}\)，欧几里得算法给 \(I = n\mathbb{Z}\)。所以 \(\mathbb{Z}\) 的理想全集是

\[ \{(0)\} \cup \{(n) : n \ge 1\} = \{n\mathbb{Z} : n \ge 0\}. \]

\((x) \subset R[x]\)。多项式环里"以 \(x\) 为系数因子"的多项式 \(\{x \cdot g(x) : g \in R[x]\} = \{f \in R[x] : f(0) = 0\}\) 构成理想。验证：吸收 \(R[x] \cdot (x g) = (x)(R[x] \cdot g) \subset (x)\)。这个理想也常写 \(xR[x]\) 或 \((x)\)，是主理想的另一个例子。

更一般地，\((x_1, x_2, \ldots, x_n) \subset R[x_1, \ldots, x_n]\) 是"在原点取零"的多元多项式构成的理想。这是代数几何里"原点的理想"——当 \(R = k\) 是域时，这个理想极大（12.6 节会看到）。

\(M_n(R)\) 里的理想。有件事让初学者意外：\(M_n(R)\) 的双侧理想和 \(R\) 的双侧理想一一对应。具体来说，\(M_n(R)\) 的所有双侧理想恰好形如 \(M_n(I)\)（所有元素都在 \(I\) 内的矩阵），其中 \(I\) 跑遍 \(R\) 的双侧理想。当 \(R\) 是域时 \(R\) 的真理想只有 \(\{0\}\)，所以 \(M_n(R)\) 的真双侧理想只有 \(\{0\}\)——一个非平凡环却没有真双侧理想，这种环叫单环，是非交换代数的基本对象。

这件事告诉你"非交换环里的理想数量比 \(R\) 上看起来少"。但左理想很多——\(M_n(R)\) 里"前 \(k\) 列固定为零"的矩阵就是大量左理想。所以做非交换代数时一定要分清左 / 右 / 双侧。

12.3 主理想；和、积、交

设 \(R\) 是交换环，\(a \in R\)。把

\[ (a) := aR = \{ar : r \in R\} \]

叫做 \(a\) 生成的主理想。验证：对加封闭 \(ar_1 + ar_2 = a(r_1 + r_2)\)，吸收 \(s \cdot (ar) = a(sr) \in (a)\)。所以 \((a)\) 是理想。

更一般地，\(a_1, \ldots, a_n\) 生成的理想

\[ (a_1, \ldots, a_n) := a_1 R + a_2 R + \cdots + a_n R = \left\{\sum_i a_i r_i : r_i \in R\right\} \]

是包含 \(\{a_1, \ldots, a_n\}\) 的最小理想。再一般一点，对任意子集 \(S \subset R\)，

\[ (S) := \left\{\sum_{\text{有限}} s_i r_i : s_i \in S, r_i \in R\right\} \]

是 \(S\) 生成的理想。

有了"取理想"的方法，理想之间还有几个标准运算。

和。 \(I + J := \{i + j : i \in I, j \in J\}\)。这是包含 \(I \cup J\) 的最小理想。比如 \(\mathbb{Z}\) 里 \((6) + (10) = (\gcd(6, 10)) = (2)\)（这是欧几里得算法的代数表述）。

交。 \(I \cap J\) 是理想（直接验证）。比如 \(\mathbb{Z}\) 里 \((6) \cap (10) = (\text{lcm}(6, 10)) = (30)\)。

积。 \(IJ := \{\text{有限和 } \sum i_k j_k : i_k \in I, j_k \in J\}\)。注意"积"不是"\(\{ij : i \in I, j \in J\}\)" —— 后者一般不对加封闭，所以要取有限和。比如 \(\mathbb{Z}\) 里 \((6)(10) = (60)\)。

三者满足 \(IJ \subset I \cap J\)（每个 \(ij\) 在两个理想里都呆着），但反过来不一定相等。在 PID 里 \(I \cap J = (\text{lcm})\), \(IJ = (\text{乘积})\)，所以两者相等当且仅当 \(\gcd = 1\)（互素）。这句话是中国剩余定理的代数版核心。

12.4 商环 \(R/I\)

给定环 \(R\) 和理想 \(I\)。考虑加法陪集

\[ r + I = \{r + i : i \in I\}, \quad r \in R. \]

因为 \(I\) 是 \((R, +)\) 的子群（且 \((R, +)\) 交换，所以子群自动正规），所有陪集构成集合 \(R/I\)（这个商在第 5 章 5.2 节已经做过——但那时只是加法商群）。在这上面已经有了诱导加法

\[ (r + I) + (s + I) := (r + s) + I. \]

这给了 \(R/I\) 一个加法交换群结构。

关键问题来了：能不能把乘法也按陪集定义

\[ (r + I) \cdot (s + I) := rs + I \quad ? \]

这条要良定义——必须是说"代表元的不同选取给同一个结果"。设 \(r' + I = r + I\) 与 \(s' + I = s + I\)，即 \(r' - r \in I\) 与 \(s' - s \in I\)。那么

\[ r' s' - rs = r' s' - r s' + r s' - rs = (r' - r) s' + r (s' - s). \]

第一项 \((r' - r) s' \in I\) 因为 \(I\) 吸收右乘 \(s'\)；第二项 \(r(s' - s) \in I\) 因为 \(I\) 吸收左乘 \(r\)。两项都在 \(I\) 内，所以 \(r's' - rs \in I\)，即 \(r's' + I = rs + I\)。乘法良定义。

请你看一眼这段验证——它正好用了 \(I\) 的左吸收和右吸收。如果 \(I\) 只是子环没有吸收，乘法就不良定义；如果只是单侧理想，乘法在交换环外也可能不良定义。这件事就是为什么"理想"被定义成"加法子群 + 双侧吸收"——不为别的，就为了让商环存在。

把这条写成核心 boxed 公式：

\[ \boxed{\ R/I \text{ 是环} \iff I \text{ 是 } R \text{ 的（双侧）理想}.\ } \]

右推左是上面的良定义验证；左推右是反过来——如果 \(I\) 不吸收，可以构造代表元让乘法不良定义。

具体例子：

  • \(\mathbb{Z}/n\mathbb{Z} = \mathbb{Z}_n\) 就是第 11 章 11.3 节的剩余环。
  • \(R[x]/(x) \cong R\) —— 商掉 \(x\) 等于"把 \(x\) 设为 0"，所以剩下的就是常多项式 \(R\)。
  • \(\mathbb{R}[x]/(x^2 + 1) \cong \mathbb{C}\) —— 商掉 \(x^2 + 1\) 等于"强制 \(x^2 = -1\)"，于是 \(x\) 就行为像 \(i\)，整个商环就是复数。这是把 \(\mathbb{C}\) 用"代数手段"建造出来的标准方法（见第 15 章）。

12.5 三大同构定理（环版）

群论里的三大同构定理（第 5 章）在环论里有完全平行的版本。设 \(\varphi : R \to S\) 是环同态——即 \(\varphi\) 同时尊重加法、乘法、并且 \(\varphi(1_R) = 1_S\)。它的核定义为

\[ \ker \varphi = \{r \in R : \varphi(r) = 0\}. \]

断言：\(\ker \varphi\) 是 \(R\) 的双侧理想。证明：加法子群是已知；吸收因为 \(\varphi(rk) = \varphi(r) \varphi(k) = \varphi(r) \cdot 0 = 0\)，故 \(rk \in \ker \varphi\)；右吸收对称。

反过来，每个理想 \(I \subset R\) 都是某个环同态 \(\pi : R \to R/I\) 的核，即商映射 \(\pi(r) = r + I\)。验证 \(\pi\) 是环同态、\(\ker \pi = I\) 都直接。

所以"理想 = 环同态的核"——这个对应是双向的、完美的。

第一同构定理。设 \(\varphi : R \to S\) 是环同态。则

\[ \boxed{\ R / \ker \varphi \;\cong\; \mathrm{Im}\, \varphi.\ } \]

证：定义 \(\bar \varphi : R/\ker \varphi \to \mathrm{Im}\,\varphi\) by \(\bar \varphi(r + \ker \varphi) = \varphi(r)\)。良定义、单同态、满同态——和群论里完全一样。

这是最常用的环同态分解技术。比如要证 \(\mathbb{R}[x]/(x^2 + 1) \cong \mathbb{C}\)，就构造 \(\varphi : \mathbb{R}[x] \to \mathbb{C}\) by \(\varphi(f) = f(i)\)。这是环同态（求值映射总是同态）；像是整个 \(\mathbb{C}\)（任意 \(a + bi \in \mathbb{C}\) 是 \(\varphi(a + bx)\)）；核恰好是 \((x^2 + 1)\)（因为 \(f(i) = 0 \iff (x^2 + 1) \mid f\)，因 \(x^2 + 1\) 是 \(i\) 的最小多项式）。第一同构定理立刻给 \(\mathbb{R}[x]/(x^2 + 1) \cong \mathbb{C}\)。

第二同构定理。设 \(S \subset R\) 是子环（含 1），\(I \subset R\) 是理想。则 \(S + I\) 是 \(R\) 的子环、\(S \cap I\) 是 \(S\) 的理想，且

\[ (S + I)/I \;\cong\; S/(S \cap I). \]

第三同构定理。设 \(I \subset J\) 都是 \(R\) 的理想。则 \(J/I\) 是 \(R/I\) 的理想，并且

\[ (R/I) / (J/I) \;\cong\; R/J. \]

这两条的证法和群论里完全平行，本节就不再重复——你只需要记住"理想 ↔ 正规子群" 这个对应让全部三条定理移植过来。

12.6 极大理想与域

真理想 \(\mathfrak{m} \subset R\)（即 \(\mathfrak{m} \neq R\)）称为极大，如果不存在中间理想 \(\mathfrak{m} \subsetneq I \subsetneq R\)。也就是说"\(\mathfrak{m}\) 上面除了 \(R\) 没有别的真理想"。

关键定理（本章最重要的结论之一）：

\[ \boxed{\ R/\mathfrak{m} \text{ 是域} \iff \mathfrak{m} \text{ 是 } R \text{ 的极大理想}.\ } \]

这里假设 \(R\) 是交换环。证明大意：

  • (\(\Rightarrow\)) 设 \(R/\mathfrak{m}\) 是域。要证 \(\mathfrak{m}\) 极大，反设有中间理想 \(\mathfrak{m} \subsetneq I \subsetneq R\)。取 \(a \in I \setminus \mathfrak{m}\)。则 \(a + \mathfrak{m} \neq 0\) 在 \(R/\mathfrak{m}\) 内，由域性质 \(a + \mathfrak{m}\) 可逆，存在 \(b\) 使 \((a + \mathfrak{m})(b + \mathfrak{m}) = 1 + \mathfrak{m}\)，即 \(ab - 1 \in \mathfrak{m} \subset I\)。又 \(ab \in I\) 由吸收，故 \(1 = ab - (ab - 1) \in I\)，与 \(I \neq R\) 矛盾。
  • (\(\Leftarrow\)) 设 \(\mathfrak{m}\) 极大。取 \(0 \neq \bar a \in R/\mathfrak{m}\)，要造逆。\(a \notin \mathfrak{m}\)，所以理想 \(\mathfrak{m} + (a) \supsetneq \mathfrak{m}\)，由极大性 \(\mathfrak{m} + (a) = R\)，故 \(1 = m + ab\) 对某 \(m \in \mathfrak{m}, b \in R\)，即 \(ab \equiv 1 \pmod \mathfrak{m}\)，\(\bar a \cdot \bar b = 1\)。

具体：

  • \(\mathbb{Z}\) 中 \(p\mathbb{Z}\) 极大 \(\iff p\) 素，所以 \(\mathbb{Z}/p\mathbb{Z} = \mathbb{F}_p\) 是有限域。这就是 \(\mathbb{Z}_p\)（\(p\) 素）变成域的原因。
  • \(\mathbb{R}[x]\) 中 \((x^2 + 1)\) 极大（因为 \(x^2 + 1\) 在 \(\mathbb{R}[x]\) 上不可约），故 \(\mathbb{R}[x]/(x^2 + 1) = \mathbb{C}\) 是域。
  • \(\mathbb{C}[x_1, \ldots, x_n]\) 中 \((x_1 - a_1, \ldots, x_n - a_n)\) 极大，商等于 \(\mathbb{C}\)（求值在 \((a_1, \ldots, a_n)\)）。这是 Hilbert Nullstellensatz 的核心：\(\mathbb{C}^n\) 的点 \(\leftrightarrow\) \(\mathbb{C}[x_1, \ldots, x_n]\) 的极大理想。

12.7 素理想与整环

真理想 \(\mathfrak{p} \subset R\) 称为素，如果对任意 \(a, b \in R\)：\(ab \in \mathfrak{p} \Rightarrow a \in \mathfrak{p}\) 或 \(b \in \mathfrak{p}\)。这是把"素数性质" \(p \mid ab \Rightarrow p \mid a\) 或 \(p \mid b\) 抽象到一般环上。

对应定理：

\[ \boxed{\ R/\mathfrak{p} \text{ 是整环} \iff \mathfrak{p} \text{ 是 } R \text{ 的素理想}.\ } \]

证明：\(R/\mathfrak{p}\) 整环 \(\iff\) 无零因子 \(\iff (\bar a)(\bar b) = 0 \Rightarrow \bar a = 0\) 或 \(\bar b = 0\) \(\iff (ab \in \mathfrak{p} \Rightarrow a \in \mathfrak{p}\) 或 \(b \in \mathfrak{p})\) \(\iff \mathfrak{p}\) 素。直接的等价改写。

由于"域 \(\Rightarrow\) 整环"是显然的（域里非零都可逆，故无零因子），可以推出"极大 \(\Rightarrow\) 素"。在交换环里这是常用的层级关系：

\[ \text{极大理想} \subset \text{素理想} \subset \text{真理想}. \]

反过来"素 \(\Rightarrow\) 极大"不对。最小反例：\(\mathbb{Z}\) 里 \((0)\) 是素理想（\(\mathbb{Z}\) 是整环 \(\Leftrightarrow (0)\) 素），但 \((0)\) 不极大（\((0) \subsetneq (2) \subsetneq \mathbb{Z}\)）。一般地，主理想环里 \((0)\) 永远素但通常不极大；非零素理想 = 极大理想。

"\(R\) 的所有素理想构成的集合"叫做 \(R\) 的谱 \(\mathrm{Spec}\, R\)，配上合适的拓扑（Zariski 拓扑）就是代数几何里"概形" 的基本对象。本书不深入，但你应该知道这个名字以后看代数几何文献时不陌生。

12.8 实例：\(\mathbb{Z}\) 与 \(\mathbb{Z}[i]\) 中的极大 / 素理想

把上面两节的理论在 \(\mathbb{Z}\) 和高斯整数 \(\mathbb{Z}[i]\) 上跑一遍。

\(\mathbb{Z}\) 的所有理想分类。所有理想都是 \((n) = n\mathbb{Z}\)（\(n \ge 0\)）。

  • \((n)\) 极大 \(\iff \mathbb{Z}/n\mathbb{Z}\) 是域 \(\iff n\) 是素数。所以 \(\mathbb{Z}\) 的极大理想恰好是 \(\{(p) : p \text{ 素}\}\)。
  • \((n)\) 素 \(\iff \mathbb{Z}/n\mathbb{Z}\) 是整环 \(\iff n = 0\) 或 \(n\) 是素数。
  • 所以 \(\mathrm{Spec}\, \mathbb{Z} = \{(0)\} \cup \{(p) : p \text{ 素}\}\)。

这是数论里最干净的"理想 ↔ 素数"对应。

\(\mathbb{Z}[i]\) 中的素理想。对 \(\mathbb{Z}[i]\) 来说，"哪些理想素 / 极大"涉及"\(\mathbb{Z}\) 的素数 \(p\) 在 \(\mathbb{Z}[i]\) 中如何分解"。结论：

  • \(p = 2\)：\((2) = (1 + i)^2\)（"分歧"）。所以 \((2)\) 不素，因为 \((1+i)^2 \in (2)\) 但 \((1+i) \notin (2)\)（\(1 + i\) 不能写成 \(2 \cdot z\) 因 \(z\) 必非高斯整数）。
  • \(p \equiv 1 \pmod 4\)：\(p = a^2 + b^2 = (a + bi)(a - bi)\) 在 \(\mathbb{Z}[i]\) 中分裂为两个共轭素元的乘积。所以 \((p)\) 不素，但 \((a + bi)\) 与 \((a - bi)\) 都极大。
  • \(p \equiv 3 \pmod 4\)：\(p\) 在 \(\mathbb{Z}[i]\) 中仍然是素元（"惰性"）。所以 \((p)\) 在 \(\mathbb{Z}[i]\) 中仍是极大理想。

三种情况对应数论里素数的"分歧 / 分裂 / 惰性"。本书不展开（这是代数数论的入门内容），但你应该看到："把 \(\mathbb{Z}\) 嵌入更大的整数环 \(\mathbb{Z}[i]\) 之后，原来的素理想要么分裂、要么不变、要么平方化" —— 这种现象是经典数论里"二次互反律"和"类域论"的源头。

验算 \((1 + i)^2 = 1 + 2i + i^2 = 1 + 2i - 1 = 2i\)，故 \((1+i)^2 = (2i) = (2)\)（因 \(i\) 是单位）。所以的确 \((2) = (1+i)^2\)。

这一章的核心收获是把"群论里正规子群 → 商群"这个故事在环论里完整复制，并加上极大 / 素理想这两个环论独有的精细结构。下一章我们要把"商环什么时候是域 / 整环"这个问题反过来研究——直接定义域、整环、商域这三个标准对象，把整个环的"层级"从最干净的一端立起来。
