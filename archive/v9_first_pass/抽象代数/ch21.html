第 21 章 Galois 理论基本定理

这是抽象代数最深的结构定理。它说："Galois 扩张 \(K/F\) 的中间域"和"Galois 群 \(\text{Gal}(K/F)\) 的子群"反向一一对应。具体说，给定 Galois 扩张 \(K/F\)，每个介于 \(F\) 与 \(K\) 之间的中间域 \(M\) 对应一个子群 \(\text{Gal}(K/M) \le \text{Gal}(K/F)\)；反之每个子群 \(H\) 对应一个不动域 \(\text{Fix}(H)\)；两者互为逆映射。

这种"几何 / 代数对应"在 19 世纪让 Galois 把"五次以上方程没有求根公式"这个困扰了数学家三百年的难题完整解决。它的力量在于：把"找中间域"这种连续型几何问题翻译成"找子群"这种有限离散问题。一个 \(n\) 阶群的子群只有有限多个，可以用 Lagrange + Sylow 系统列出来——你立刻看到 \(K\) 与 \(F\) 之间所有中间域的完整图谱。

本章把 Galois 群的定义、计算、不动域、主定理及其证明思路统统讲清，并用三个核心例子让你看到对应是怎么具体工作的。下一章会用这套机器解决尺规作图与五次方程不可解。

  • Galois 扩张：有限正规可分；自动 \([K:F] = |\text{Aut}(K/F)|\)。
  • Galois 群 \(\text{Gal}(K/F)\) 的定义与基本性质。
  • 三个标志性例子：\(\mathbb{C}/\mathbb{R}\)、\(\mathbb{Q}(\sqrt 2)/\mathbb{Q}\)、\(\mathbb{Q}(\sqrt 2, \sqrt 3)/\mathbb{Q}\)。
  • 分圆扩张：\(\text{Gal}(\mathbb{Q}(\zeta_p)/\mathbb{Q}) \cong (\mathbb{Z}/p)^*\)。
  • 不动域 \(\text{Fix}(H)\) 的定义与基本事实。
  • Galois 主定理（完整陈述）。
  • 主定理的证明结构——为什么 \(\text{Fix}\) 与 \(\text{Gal}\) 互为逆。
  • 应用：用主定理列出 \(\mathbb{Q}(\sqrt 2, \sqrt 3)\) 的所有中间域。

21.1 Galois 扩张与 \([K:F] = |\text{Aut}(K/F)|\)

把第 20 章的两个条件——正规性与可分性——合在一起，就得到本章的核心对象。

定义。有限扩张 \(K/F\) 称为Galois 扩张，如果它既正规又可分。等价地（由第 20 章 20.5 节）：\(K\) 是 \(F\) 上某个可分多项式的分裂域。

在特征 0（如 \(\mathbb{Q}\)）上，"可分"自动满足，所以 Galois 扩张就是"分裂域扩张"。这是为什么本章的所有例子都来自 \(\mathbb{Q}\)。

记 \(\text{Aut}(K/F) = \{\sigma : K \to K \mid \sigma \text{ 是域同构}, \sigma|_F = \text{id}\}\) 为 \(K\) 在合成下的"\(F\)-自同构群"。它是一个群（自同构合成 + 求逆）。

定理（基本恒等式）。设 \(K/F\) 是有限扩张。则

\[ |\text{Aut}(K/F)| \le [K : F], \]

等号成立 \(\Leftrightarrow K/F\) 是 Galois 扩张：

\[ \boxed{\ K/F \text{ Galois}\ \Leftrightarrow\ |\text{Aut}(K/F)| = [K:F].\ } \]

证明思路：上界来自第 20 章同构延拓引理——任何 \(F\)-自同构 \(\sigma\) 完全由它在生成元 \(\alpha_i\) 上的取值决定，\(\sigma(\alpha_i)\) 必须是 \(\alpha_i\) 的最小多项式 \(p_i\) 在 \(K\) 中的另一个根，所以候选数最多是 \(\prod \deg p_i\) 受 \([K : F]\) 控制。等号当且仅当每个 \(p_i\) 在 \(K\) 中"满根分裂"（\(\Rightarrow\) 正规）且"无重根"（\(\Rightarrow\) 可分）。详细论证用 Artin 引理（21.7 节再回到这一点）。

这条恒等式是 Galois 理论的入口——它把"扩张次数"这个线性代数量与"自同构数"这个群论量挂上了钩。一旦 \(K/F\) 是 Galois，整个故事就铺开了。

21.2 Galois 群的定义

定义。设 \(K/F\) 是 Galois 扩张。它的Galois 群是

\[ \text{Gal}(K/F) := \text{Aut}(K/F) = \{\sigma : K \xrightarrow{\sim} K \mid \sigma|_F = \text{id}\}. \]

群运算是函数合成。由 21.1 节，\(|\text{Gal}(K/F)| = [K : F]\)。

"Galois 自同构 \(\sigma\) 怎么作用"的几何图景：\(K\) 是 \(F\) 上某分裂域，\(\sigma\) 把 \(F\) 留在原地、把 \(K\) 中的元素重新排列——但必须把根送到根。具体说，如果 \(\alpha \in K\) 是 \(F\) 上某不可约 \(p \in F[x]\) 的根，那 \(\sigma(\alpha)\) 必须也是 \(p\) 的根（因为 \(p(\sigma(\alpha)) = \sigma(p(\alpha)) = \sigma(0) = 0\)，第二步用了 \(\sigma\) 保 \(F\) 中系数）。

\[ \boxed{\ \sigma \in \text{Gal}(K/F)\ \Rightarrow\ \forall \alpha \in K,\ \sigma(\alpha) \text{ 与 } \alpha \text{ 同满足同一最小多项式}.\ } \]

这条原理叫"Galois 自同构置换共轭根"——它把 \(K/F\) 上的对称完全归结为"\(F\) 上不可约多项式的根之间的置换"。这是为什么 Galois 群本质上是一个置换群。

更精确：取 \(K\) 的某个 \(F\)-生成集 \(\{\alpha_1, \ldots, \alpha_n\}\)（可以是单一生成元——本原元定理保证 \(K = F(\theta)\) 对某 \(\theta\)，但暂不需要）。\(\sigma\) 由 \(\sigma(\alpha_i)\) 完全决定；每个 \(\sigma(\alpha_i)\) 必须是 \(\alpha_i\) 在 \(F\) 上的某个共轭根。所以 \(\text{Gal}(K/F)\) 嵌入到"根的置换"群里——具体怎么嵌入由这些根之间的代数关系决定。

21.3 例 1：\(\text{Gal}(\mathbb{C}/\mathbb{R}) \cong \mathbb{Z}/2\)

\(\mathbb{C}/\mathbb{R}\) 是 Galois 扩张：\(\mathbb{C} = \mathbb{R}(i)\) 是 \(x^2 + 1\) 的分裂域，特征 0 自动可分。次数 \([\mathbb{C} : \mathbb{R}] = 2\)，所以 \(|\text{Gal}(\mathbb{C}/\mathbb{R})| = 2\)。

那两个自同构是什么？任何 \(\sigma \in \text{Gal}(\mathbb{C}/\mathbb{R})\) 必须把 \(i\) 送到 \(x^2 + 1\) 的某根，即 \(\pm i\)。两种选择：

  • \(\sigma(i) = i\)：\(\sigma\) 是恒等映射 \(\text{id}\)。
  • \(\sigma(i) = -i\)：\(\sigma\) 是复共轭 \(z \mapsto \bar z\)。

验证两者都是良定义的域自同构（线性 + 乘法保 + 保 \(\mathbb{R}\)）。所以

\[ \text{Gal}(\mathbb{C}/\mathbb{R}) = \{\text{id}, \text{共轭}\} \cong \mathbb{Z}/2. \]

这个例子虽然小但极端重要——它说复共轭就是 \(\mathbb{R}\) 上的全部 Galois 对称。复分析里反复出现的"实部 = \((z + \bar z)/2\)"、"模平方 = \(z \bar z\)"、"实多项式的复根成对出现"，背后都是这个 2 阶 Galois 群在起作用。

21.4 例 2：\(\mathbb{Q}\) 上的二次扩张与双二次扩张

例 2a：\(\text{Gal}(\mathbb{Q}(\sqrt 2)/\mathbb{Q}) \cong \mathbb{Z}/2\)。

\([\mathbb{Q}(\sqrt 2) : \mathbb{Q}] = 2\) 已计算。\(\sqrt 2\) 的最小多项式 \(x^2 - 2\) 两根 \(\pm \sqrt 2\) 都在 \(\mathbb{Q}(\sqrt 2)\) 里，所以正规——加上特征 0 可分，是 Galois。

两个自同构：\(\text{id}\)；以及"实数共轭" \(\tau : a + b\sqrt 2 \mapsto a - b\sqrt 2\)。验证 \(\tau\) 是域自同构：\(\tau((a + b\sqrt 2)(c + d\sqrt 2)) = \tau(ac + 2bd + (ad + bc)\sqrt 2) = ac + 2bd - (ad + bc)\sqrt 2 = (a - b\sqrt 2)(c - d\sqrt 2) = \tau(a + b\sqrt 2) \tau(c + d\sqrt 2)\)。 \(\tau^2 = \text{id}\)。

所以 \(\text{Gal}(\mathbb{Q}(\sqrt 2)/\mathbb{Q}) = \{\text{id}, \tau\} \cong \mathbb{Z}/2\)。和 21.3 节几乎一模一样的故事。

例 2b：\(\text{Gal}(\mathbb{Q}(\sqrt 2, \sqrt 3)/\mathbb{Q}) \cong \mathbb{Z}/2 \times \mathbb{Z}/2\)（Klein 4 群）。

这是 Galois 群第一次"具备结构"的例子。\([\mathbb{Q}(\sqrt 2, \sqrt 3) : \mathbb{Q}] = 4\)：\(\mathbb{Q}(\sqrt 2)\) 是 2 次扩张，再加 \(\sqrt 3\) 得到 \(\mathbb{Q}(\sqrt 2)(\sqrt 3)\)，\(\sqrt 3\) 在 \(\mathbb{Q}(\sqrt 2)\) 上的最小多项式仍是 \(x^2 - 3\)（要证 \(\sqrt 3 \notin \mathbb{Q}(\sqrt 2)\)：若 \(\sqrt 3 = a + b \sqrt 2\) 平方 \(\Rightarrow 3 = a^2 + 2b^2 + 2ab \sqrt 2\)，必 \(ab = 0\)；\(a = 0 \Rightarrow 2b^2 = 3\)、\(b = 0 \Rightarrow a^2 = 3\)，都不可能在 \(\mathbb{Q}\) 中）。所以 \([\mathbb{Q}(\sqrt 2, \sqrt 3) : \mathbb{Q}(\sqrt 2)] = 2\)，乘起来 \(= 4\)。

\(\mathbb{Q}(\sqrt 2, \sqrt 3)\) 是 \((x^2 - 2)(x^2 - 3)\) 的分裂域 \(\Rightarrow\) Galois。\(|\text{Gal}| = 4\)。

4 个自同构由 \(\sigma(\sqrt 2), \sigma(\sqrt 3)\) 的取值决定。\(\sigma(\sqrt 2) \in \{\pm \sqrt 2\}\)、\(\sigma(\sqrt 3) \in \{\pm \sqrt 3\}\)，2 × 2 = 4 种组合，全部都是良定义的自同构。列表：

  • \(\text{id}\)：\(\sqrt 2 \mapsto \sqrt 2\), \(\sqrt 3 \mapsto \sqrt 3\)；
  • \(\sigma\)：\(\sqrt 2 \mapsto -\sqrt 2\), \(\sqrt 3 \mapsto \sqrt 3\)；
  • \(\tau\)：\(\sqrt 2 \mapsto \sqrt 2\), \(\sqrt 3 \mapsto -\sqrt 3\)；
  • \(\sigma \tau\)：\(\sqrt 2 \mapsto -\sqrt 2\), \(\sqrt 3 \mapsto -\sqrt 3\)。

\(\sigma^2 = \tau^2 = (\sigma\tau)^2 = \text{id}\)，\(\sigma \tau = \tau \sigma\)。这是 Klein 4 群 \(V = \mathbb{Z}/2 \times \mathbb{Z}/2\)。

这个 4 阶 Galois 群有三个 2 阶子群：\(\{\text{id}, \sigma\}\)、\(\{\text{id}, \tau\}\)、\(\{\text{id}, \sigma\tau\}\)，外加平凡子群 \(\{\text{id}\}\) 与全群本身——共 5 个子群。Galois 主定理预言这对应 5 个中间域。21.8 节会把它们一个个列出来。

21.5 例 3：分圆扩张 \(\text{Gal}(\mathbb{Q}(\zeta_p)/\mathbb{Q}) \cong (\mathbb{Z}/p)^*\)

取奇素数 \(p\)，\(\zeta_p = e^{2\pi i / p}\) 是 \(p\) 次本原单位根。多项式 \(x^p - 1 = (x - 1)(x^{p-1} + x^{p-2} + \cdots + 1)\)，第二因子叫第 \(p\) 个分圆多项式记作 \(\Phi_p(x)\)，它在 \(\mathbb{Q}\) 上不可约（用 Eisenstein 在 \(\Phi_p(x + 1)\) 上）。\(\zeta_p\) 是 \(\Phi_p\) 的根，所以 \([\mathbb{Q}(\zeta_p) : \mathbb{Q}] = \deg \Phi_p = p - 1\)。

\(\Phi_p\) 的全部根是 \(\zeta_p, \zeta_p^2, \ldots, \zeta_p^{p-1}\)（即所有 \(p\) 次本原单位根），都是 \(\zeta_p\) 的幂，所以都在 \(\mathbb{Q}(\zeta_p)\) 里 \(\Rightarrow\) 正规 \(\Rightarrow\) Galois 扩张。\(|\text{Gal}| = p - 1\)。

每个 \(\sigma \in \text{Gal}\) 由 \(\sigma(\zeta_p)\) 决定，且 \(\sigma(\zeta_p)\) 必须是 \(\Phi_p\) 的根 = 某 \(\zeta_p^k\)，\(k \in \{1, 2, \ldots, p - 1\}\)。记 \(\sigma_k(\zeta_p) = \zeta_p^k\)。验证 \(\sigma_k\) 是良定义自同构（\(\zeta_p\) 在 \(\mathbb{Q}\) 上由 \(\Phi_p = 0\) 完全决定，把它送到另一个根给出域同构）。

合成法则：\(\sigma_k \circ \sigma_l(\zeta_p) = \sigma_k(\zeta_p^l) = (\sigma_k(\zeta_p))^l = \zeta_p^{kl}\)，所以 \(\sigma_k \sigma_l = \sigma_{kl}\)（指标乘法 mod \(p\)）。所以

\[ \text{Gal}(\mathbb{Q}(\zeta_p)/\mathbb{Q}) \cong (\mathbb{Z}/p)^* \cong \mathbb{Z}/(p - 1) \]

（\((\mathbb{Z}/p)^*\) 是循环群——这是第 7 章的结论：素数模乘法群循环）。

这个例子结构最透——Galois 群就是\((\mathbb{Z}/p)^*\)，它的子群对应循环群的子群（每个 \(d \mid p - 1\) 一个），相应中间域恰好是下一章 22.5 节"正多边形作图"问题的关键。

21.6 不动域

Galois 主定理需要"群 \(\to\) 域"方向的对应。这就是不动域。

定义。设 \(H \le \text{Aut}(K/F)\) 是子群（或更一般，\(K\) 的任意自同构子群）。它的不动域是

\[ \text{Fix}(H) := \{\alpha \in K : \sigma(\alpha) = \alpha,\ \forall \sigma \in H\}. \]

这是 \(K\) 的子域（\(\text{id} \in H\) 让 \(\text{Fix}(H) \ne \emptyset\)；\(\sigma\) 是同构 \(\Rightarrow\) 闭于加减乘除）。它包含 \(F\)（因为 \(\sigma|_F = \text{id}\)）。

关键事实（Artin 引理）：

\[ \boxed{\ H \text{ 有限子群} \Rightarrow [K : \text{Fix}(H)] = |H|.\ } \]

这条引理是 Galois 主定理的核心引擎。它的意思是："不动域的'尺寸'恰好等于群的'尺寸'"——你不能有更多自同构去固定一个更大的子域。证明用线性代数 + 鸽笼原理（详见 Artin《Galois Theory》）。

结合 21.1 节的恒等式 \(|\text{Gal}(K/L)| = [K : L]\) （\(L = \text{Fix}(H)\) 上 \(K\) 是 Galois——因为继承正规可分），我们得到

\[ |\text{Gal}(K/\text{Fix}(H))| = [K : \text{Fix}(H)] = |H|. \]

而 \(H \subset \text{Gal}(K/\text{Fix}(H))\)（每个 \(H\) 中元素显然固定 \(\text{Fix}(H)\)），尺寸相等 \(\Rightarrow H = \text{Gal}(K/\text{Fix}(H))\)。这就是"\(H \mapsto \text{Fix}(H) \mapsto \text{Gal}(K/\text{Fix}(H))\) 复合 = \(H\)"的核心步骤。

类似的"另一边"恒等式 \(\text{Fix}(\text{Gal}(K/M)) = M\)（对每个中间域 \(M\)）由 \(K/M\) Galois 直接给：\(|\text{Gal}(K/M)| = [K : M]\) \(\Rightarrow\) \(\text{Fix}(\text{Gal}(K/M))\) 也满足 \([K : \text{Fix}] = [K : M]\)，再加 \(\text{Fix} \supset M\) 给等。

\[ \boxed{\ \text{Fix}(\text{Gal}(K/F)) = F \quad \text{（} K/F \text{ Galois 时）}.\ } \]

21.7 Galois 主定理

把所有上述材料合起来。

定理（Galois 主定理）。设 \(K/F\) 是有限 Galois 扩张，\(G = \text{Gal}(K/F)\)。则下面是事实清单：

(GFT1) 反序对应。映射

\[ \begin{aligned} \Phi : \{F \subset M \subset K\} &\longrightarrow \{H \le G\}, \quad M \mapsto \text{Gal}(K/M); \\ \Psi : \{H \le G\} &\longrightarrow \{F \subset M \subset K\}, \quad H \mapsto \text{Fix}(H); \end{aligned} \]

互为反函数（\(\Phi \Psi = \text{id}\)，\(\Psi \Phi = \text{id}\)），且都是反序的：

\[ \boxed{\ M_1 \subset M_2\ \Leftrightarrow\ \text{Gal}(K/M_1) \supset \text{Gal}(K/M_2).\ } \]

(GFT2) 次数与指标。对每个中间域 \(M\)：

\[ [K : M] = |\text{Gal}(K/M)|, \qquad [M : F] = [G : \text{Gal}(K/M)]. \]

(GFT3) 正规性 \(\Leftrightarrow\) 正规子群。中间域 \(M\) 满足 \(M/F\) 正规 \(\Leftrightarrow \text{Gal}(K/M) \trianglelefteq G\)（正规子群）。在这种情形：

\[ \boxed{\ \text{Gal}(M/F) \cong G / \text{Gal}(K/M).\ } \]

这条主定理把"中间域的整个格子"和"Galois 群子群的整个格子"反序同构地对接起来。它是抽象代数最深的结构定理，没有之一。

证明思路（不写每步严格证明，但要说清"为什么互为逆"）：

  1. (GFT1) 互为逆。\(\Phi \Psi(H) = \text{Gal}(K/\text{Fix}(H))\)，由 21.6 节 Artin 引理 \(= H\)。\(\Psi \Phi(M) = \text{Fix}(\text{Gal}(K/M))\)，由 \(K/M\) 是 Galois \(\Rightarrow\) \(\text{Fix}(\text{Gal}(K/M)) = M\)（21.6 节末尾）。反序性显然——更大子群固定更少元素。
  2. (GFT2) 次数。\([K : M] = |\text{Gal}(K/M)|\) 是 21.1 节恒等式。\([M : F] = [K : F]/[K : M] = |G|/|\text{Gal}(K/M)| = [G : \text{Gal}(K/M)]\)（Lagrange 指标）。
  3. (GFT3) 正规性 \(\Leftrightarrow\) 正规子群。\(M/F\) 正规 \(\Leftrightarrow\) 每个 \(\sigma \in G\) 把 \(M\) 映到自身（\(\sigma\) 在 \(M\) 中的像满足同样最小多项式，\(M/F\) 正规让根全在 \(M\) 里 \(\Rightarrow \sigma(M) \subset M\)；反之类似）\(\Leftrightarrow\) 对每个 \(\sigma \in G\)，\(\sigma \text{Gal}(K/M) \sigma^{-1} = \text{Gal}(K/\sigma(M)) = \text{Gal}(K/M) \Leftrightarrow \text{Gal}(K/M)\) 正规子群。商群同构通过限制映 \(G \to \text{Gal}(M/F)\)，\(\sigma \mapsto \sigma|_M\)（良定义于 \(M/F\) 正规）；核 = \(\text{Gal}(K/M)\)；满射由 21.1 让 \([G : \text{Gal}(K/M)] = [M : F] = |\text{Gal}(M/F)|\)。

整个证明的"心脏"是 21.6 节 Artin 引理——"群尺寸 = 不动域指标"。一旦有了这条，所有别的都几乎自动。

21.8 应用：列出 \(\mathbb{Q}(\sqrt 2, \sqrt 3)\) 的所有中间域

用 21.4 节的 Klein 4 群 Galois 群把这件事做透。\(G = \text{Gal}(\mathbb{Q}(\sqrt 2, \sqrt 3)/\mathbb{Q}) = \{e, \sigma, \tau, \sigma\tau\} \cong V_4\)。

\(V_4\) 的子群一共 5 个：\(\{e\}, \{e, \sigma\}, \{e, \tau\}, \{e, \sigma\tau\}, V_4\)。由主定理，中间域也恰好 5 个，对应表：

  • 子群 \(\{e\}\) \(\leftrightarrow\) 中间域 \(K = \mathbb{Q}(\sqrt 2, \sqrt 3)\)（最大）；
  • 子群 \(\{e, \sigma\}\) \(\leftrightarrow\) \(\text{Fix}(\{e, \sigma\}) = ?\)；
  • 子群 \(\{e, \tau\}\) \(\leftrightarrow\) \(\text{Fix}(\{e, \tau\}) = ?\)；
  • 子群 \(\{e, \sigma\tau\}\) \(\leftrightarrow\) \(\text{Fix}(\{e, \sigma\tau\}) = ?\)；
  • 子群 \(V_4\) \(\leftrightarrow\) \(\text{Fix}(V_4) = \mathbb{Q}\)（最小）。

计算三个不动域。回忆 \(\sigma : \sqrt 2 \mapsto -\sqrt 2, \sqrt 3 \mapsto \sqrt 3\)；\(\tau : \sqrt 2 \mapsto \sqrt 2, \sqrt 3 \mapsto -\sqrt 3\)；\(\sigma\tau\) 反两个号。

(a) \(\text{Fix}(\{e, \sigma\}) = \mathbb{Q}(\sqrt 3)\)。\(\sigma\) 不动 \(\Leftrightarrow\) 系数中的 \(\sqrt 2\) 系数为 0 \(\Rightarrow\) 元素形如 \(a + b \sqrt 3\)。

(b) \(\text{Fix}(\{e, \tau\}) = \mathbb{Q}(\sqrt 2)\)。同理。

(c) \(\text{Fix}(\{e, \sigma\tau\}) = \mathbb{Q}(\sqrt 6)\)。\(\sigma\tau(\sqrt 6) = \sigma\tau(\sqrt 2 \cdot \sqrt 3) = (-\sqrt 2)(-\sqrt 3) = \sqrt 6\)（不动），所以 \(\sqrt 6 \in \text{Fix}\)。\([\text{Fix} : \mathbb{Q}] = [G : \{e, \sigma\tau\}] = 2\)，所以 \(\text{Fix} = \mathbb{Q}(\sqrt 6)\)。

所以中间域格子完整图是：

\[ \mathbb{Q} \subset \mathbb{Q}(\sqrt 2),\ \mathbb{Q}(\sqrt 3),\ \mathbb{Q}(\sqrt 6) \subset \mathbb{Q}(\sqrt 2, \sqrt 3). \]

三个中间二次域不可遗漏，也不可多——主定理保证一一对应。这种"完整列表"的能力是 Galois 理论最实用的一面。

顺便 (GFT3)：\(V_4\) 是 Abel 群，所有子群正规 \(\Rightarrow\) 所有中间域 \(M/\mathbb{Q}\) 正规 (对 \(\mathbb{Q}\))，相应商群 \(V_4 / H \cong \mathbb{Z}/2\) 给 \(\text{Gal}(M/\mathbb{Q}) \cong \mathbb{Z}/2\) ——和"\(M\) 是 \(\mathbb{Q}\) 上二次扩张" 直接吻合。一切自洽。

下一章把 Galois 主定理用到两个 19 世纪经典难题：(1) 用尺规能不能三等分任意角 / 倍立方 / 化圆为方？(2) 五次以上多项式方程能不能有"求根公式"？两个答案都是不，工具都是 Galois。这是本书的真正最高潮——把整套抽象代数机器用在一个能让外行也理解的问题上，给出完整答案。
