第 13 章 Laurent 级数

第 11 章我们看到一件惊人的事：解析函数在它的解析圆盘里等于自己的 Taylor 级数。但 Taylor 级数有一个限制——它只能在整个圆盘内都解析的函数上展开。一旦圆盘的中心是奇点（哪怕是孤立奇点），Taylor 级数就完全用不上了。可是直觉告诉我们，奇点附近的函数也应该有"某种级数表示"。比如 \(\dfrac{1}{z}\) 在 \(z = 0\) 附近的"级数"应该是它本身——一个 \(z^{-1}\) 项；\(e^{1/z}\) 在 \(z = 0\) 附近应该是 \(1 + \dfrac{1}{z} + \dfrac{1}{2! z^2} + \dfrac{1}{3! z^3} + \cdots\)。这两种"级数"都包含负次幂。

这正是Laurent 级数要做的事——把幂级数推广到允许负次幂的"双向级数"，使得任何在去心邻域（更一般地，在环域）上解析的函数都有一个唯一的级数表示。这个推广不仅给奇点附近的函数提供了代数表示，还提供了一个判别奇点类型的"立刻读出"工具：看负次幂项有多少（零、有限、无限），就知道奇点是哪一类（可去、极点、本性）。

Laurent 级数中有一个特殊的系数——\(z^{-1}\) 的系数 \(a_{-1}\)——它有一个专门的名字叫做留数。这个看似不起眼的系数承担着复分析里最实用工具——留数定理——的全部魔力。本章末尾我们会先把留数的概念引入，下一章再展开它的威力。

本章覆盖：

  • 动机：奇点附近的"双向"幂级数。
  • Laurent 定理：环域上解析 ⇒ 唯一 Laurent 展开。
  • 系数公式 \(a_n = \tfrac{1}{2\pi i} \oint \tfrac{f(z)}{(z - z_0)^{n+1}}\, dz\)。
  • 推导：Cauchy 积分公式应用于环。
  • 主要部分与解析部分。
  • 由 Laurent 展开判别奇点类型。
  • 实操技巧：几何级数、部分分式、不同环域不同展开。
  • 详细例题：\(\dfrac{1}{(z-1)(z-2)}\) 在三个区域。
  • 留数的引入。

13.1 动机：双向幂级数

来看一个具体例子。函数 \(\dfrac{1}{1 - z}\) 在 \(z = 0\) 附近（也就是在 \(|z| < 1\) 内）有 Taylor 展开：

\[ \dfrac{1}{1 - z} = \sum_{n=0}^{\infty} z^n = 1 + z + z^2 + z^3 + \cdots, \quad |z| < 1. \]

这个表示只在 \(|z| < 1\) 上有效。在 \(|z| > 1\) 上呢？这个函数在那里也是解析的（除 \(z = 1\) 外）；它有什么级数表示？

试一个小技巧。把分母重写：

\[ \dfrac{1}{1 - z} = \dfrac{1}{-z(1 - 1/z)} = -\dfrac{1}{z} \cdot \dfrac{1}{1 - 1/z}. \]

当 \(|z| > 1\) 时，\(|1/z| < 1\)，可用几何级数：

\[ \dfrac{1}{1 - 1/z} = \sum_{n=0}^{\infty} (1/z)^n = 1 + \dfrac{1}{z} + \dfrac{1}{z^2} + \cdots. \]

所以

\[ \dfrac{1}{1 - z} = -\dfrac{1}{z} \left(1 + \dfrac{1}{z} + \dfrac{1}{z^2} + \cdots\right) = -\dfrac{1}{z} - \dfrac{1}{z^2} - \dfrac{1}{z^3} - \cdots, \quad |z| > 1. \]

这是一个完全由负次幂组成的"级数"。它在 \(|z| > 1\) 时收敛，给的就是 \(\dfrac{1}{1 - z}\) 这同一个函数在外侧区域的级数表示。

所以同一个函数 \(\dfrac{1}{1 - z}\)，在两个不同区域 \(|z| < 1\) 与 \(|z| > 1\)，有两个不同的级数表示——一个全是非负次幂、一个全是负次幂。这两个都是合法的"幂级数表示"，只不过形式上要允许负次幂。

这个例子告诉我们，容许负次幂的幂级数是研究在不同区域上解析行为的自然语言。一般地，我们考虑形如

\[ \sum_{n = -\infty}^{\infty} a_n (z - z_0)^n \]

的"双向"无穷级数，称之为Laurent 级数。它的收敛区域将不再是圆盘，而是环域（annulus）——形如 \(\{z : r < |z - z_0| < R\}\) 的"圆环"。

13.2 Laurent 定理

Laurent 定理　设 \(f\) 在环域 \(A = \{z : r < |z - z_0| < R\}\)（其中 \(0 \leq r < R \leq +\infty\)）上解析。则在 \(A\) 上，\(f\) 有唯一的Laurent 展开

\[ \boxed{f(z) = \sum_{n = -\infty}^{\infty} a_n (z - z_0)^n,} \]

且这个级数在 \(A\) 上绝对、内部一致收敛。

系数由公式

\[ a_n = \dfrac{1}{2\pi i} \oint_C \dfrac{f(z)}{(z - z_0)^{n+1}}\, dz \]

给出，其中 \(C\) 是 \(A\) 内任何一条逆时针围绕 \(z_0\) 的简单闭曲线（特别地，可以选 \(|z - z_0| = \rho\) 这样的圆，对任何 \(\rho \in (r, R)\)）。

几个要点：

  • 环域的"内半径" \(r\) 可以为 0（这时是去心圆盘 \(0 < |z - z_0| < R\)），外半径 \(R\) 可以为 \(+\infty\)（这时是 \(|z - z_0| > r\) 的外部）。所以"环域"包括了"去心邻域"和"无穷远附近"等极端情况。
  • "唯一"是关键——给定函数 \(f\) 与环域 \(A\)，Laurent 展开恰好只有一个；不存在两套不同的系数都能展开 \(f\)。这类似 Taylor 展开的唯一性。
  • 在不同的环域上，同一个函数可能有不同的 Laurent 展开（13.1 节的例子！）。这一点尤其要小心——下面 13.7、13.8 节专门展开。

注意 Laurent 级数包含了 Taylor 级数作为特例：当 \(r = 0\) 且 \(f\) 在整个圆盘 \(|z - z_0| < R\) 上解析（不只是去心圆盘）时，所有 \(a_n\)（\(n < 0\)）必然为 0（因为可以证明负次幂系数 \(a_{-k}\) 的积分可以拉到 \(z_0\) 而消失），剩下的就是 Taylor 级数。所以 Laurent 是 Taylor 的真推广。

13.3 系数公式

Laurent 系数公式

\[ a_n = \dfrac{1}{2\pi i} \oint_C \dfrac{f(z)}{(z - z_0)^{n+1}}\, dz \]

对所有整数 \(n\)（包括负数）一致地成立。这个公式与 Taylor 系数公式 \(a_n = \dfrac{f^{(n)}(z_0)}{n!} = \dfrac{1}{2\pi i} \oint \dfrac{f(z)}{(z - z_0)^{n+1}}\, dz\) 形式上完全相同——但含义不同：

  • 对 Taylor 系数，仅 \(n \geq 0\) 有意义；\(a_n\) 用 \(f^{(n)}(z_0)/n!\) 表示。
  • 对 Laurent 系数，\(n\) 可以是任何整数；当 \(n < 0\) 时，\(f^{(n)}\) 没有意义（因为 \(z_0\) 是奇点），所以只能用积分公式。

这是很重要的：Laurent 系数没有"用导数表达"的简洁形式；要计算它们一般要靠"凑出"展开（用几何级数等技巧），而不是直接套积分公式。直接套积分公式反过来需要先知道 \(f\) 的具体形式才能算积分；通常更省事的是用其他方法把 Laurent 展开"猜出来"。

话说回来，系数公式还是有用——它告诉我们：

\[ a_{-1} = \dfrac{1}{2\pi i} \oint_C f(z)\, dz. \]

这就是说，\(z^{-1}\) 项的系数 \(a_{-1}\) 等于 \(\oint_C f(z)\, dz\) 除以 \(2\pi i\)。换句话说，\(\oint_C f(z)\, dz = 2\pi i \cdot a_{-1}\)。这个公式是留数定理的核心——它说环路积分被一个单一的系数 \(a_{-1}\) 完全决定。这个 \(a_{-1}\) 就是所谓的"留数"。我们 13.9 节会专门讨论。

13.4 推导：在环上应用 Cauchy 积分公式

Laurent 定理的证明思路与 Taylor 定理非常相似——都是"Cauchy 积分公式 + 几何级数"的组合，只不过这次"绕一圈"的曲线是环域的两条边界（内圈和外圈）。

设 \(f\) 在环域 \(A = \{r < |z - z_0| < R\}\) 上解析，固定 \(z \in A\)。选两个圆周 \(C_1 : |w - z_0| = r_1\)、\(C_2 : |w - z_0| = r_2\)，半径满足 \(r < r_1 < |z - z_0| < r_2 < R\)（\(C_1\) 在内，\(C_2\) 在外，\(z\) 在中间）。

对环域 \(\{r_1 < |w - z_0| < r_2\}\) 内的点 \(z\) 应用 Cauchy 积分公式（在多连通区域上的版本）：

\[ f(z) = \dfrac{1}{2\pi i} \oint_{C_2} \dfrac{f(w)}{w - z}\, dw - \dfrac{1}{2\pi i} \oint_{C_1} \dfrac{f(w)}{w - z}\, dw. \]

注意 \(C_1\) 前的负号——这是因为对于这个多连通的环，外圈逆时针、内圈应"从外侧看也是逆时针"，这意味着内圈在自己的视角下是顺时针的；改成逆时针就要加负号。

对外圈 \(C_2\) 上的积分：因为 \(z\) 在 \(C_2\) 内（\(|z - z_0| < r_2 = |w - z_0|\)），所以 \(\left|\dfrac{z - z_0}{w - z_0}\right| < 1\)，可用几何级数展开（与 Taylor 定理证明完全一样）：

\[ \dfrac{1}{w - z} = \sum_{n=0}^{\infty} \dfrac{(z - z_0)^n}{(w - z_0)^{n+1}}, \]

代入并交换求和与积分：

\[ \dfrac{1}{2\pi i} \oint_{C_2} \dfrac{f(w)}{w - z}\, dw = \sum_{n=0}^{\infty} \left(\dfrac{1}{2\pi i} \oint_{C_2} \dfrac{f(w)}{(w - z_0)^{n+1}}\, dw\right) (z - z_0)^n. \]

这就是 Laurent 展开的非负次幂部分，对应 \(n \geq 0\) 的系数。

对内圈 \(C_1\) 上的积分：现在 \(z\) 在 \(C_1\) 外（\(|w - z_0| = r_1 < |z - z_0|\)），所以 \(\left|\dfrac{w - z_0}{z - z_0}\right| < 1\)。这次要把分子分母颠倒一下，用不同形式的几何级数：

\[ \dfrac{1}{w - z} = \dfrac{-1}{(z - z_0) - (w - z_0)} = -\dfrac{1}{z - z_0} \cdot \dfrac{1}{1 - \dfrac{w - z_0}{z - z_0}} = -\sum_{m=0}^{\infty} \dfrac{(w - z_0)^m}{(z - z_0)^{m+1}}. \]

代入：

\[ -\dfrac{1}{2\pi i} \oint_{C_1} \dfrac{f(w)}{w - z}\, dw = \sum_{m=0}^{\infty} \left(\dfrac{1}{2\pi i} \oint_{C_1} f(w) (w - z_0)^m\, dw\right) \dfrac{1}{(z - z_0)^{m+1}}. \]

令 \(n = -(m + 1)\)，即 \(m = -n - 1\)（注意此时 \(n \leq -1\)），可改写为

\[ = \sum_{n = -\infty}^{-1} \left(\dfrac{1}{2\pi i} \oint_{C_1} \dfrac{f(w)}{(w - z_0)^{n+1}}\, dw\right) (z - z_0)^n. \]

这就是 Laurent 展开的负次幂部分，对应 \(n \leq -1\) 的系数。

把两部分合并：

\[ f(z) = \sum_{n = -\infty}^{\infty} a_n (z - z_0)^n, \]

其中

\[ a_n = \dfrac{1}{2\pi i} \oint_{C} \dfrac{f(w)}{(w - z_0)^{n+1}}\, dw, \]

对 \(n \geq 0\) 用 \(C = C_2\)，对 \(n \leq -1\) 用 \(C = C_1\)。事实上，由于被积函数 \(\dfrac{f(w)}{(w - z_0)^{n+1}}\) 在环域上解析，根据 Cauchy 定理（多连通版）这个积分对环内任何"绕 \(z_0\) 一圈"的简单闭曲线都给同样的结果——所以 \(C\) 可以选环内任意这样的曲线。证毕。

这个证明的精彩之处在于：它清楚地揭示了 Laurent 展开的两部分（非负次幂、负次幂）分别来自环的两条边界，而每条边界的几何级数展开方向不同（中心还是边缘？），这正好对应 \(z\) 与 \(w\) 模的大小关系。

13.5 主要部分与解析部分

把 Laurent 展开按非负与负次幂分开：

\[ f(z) = \underbrace{\sum_{n=0}^{\infty} a_n (z - z_0)^n}_{\text{解析部分}} + \underbrace{\sum_{n = -\infty}^{-1} a_n (z - z_0)^n}_{\text{主要部分}}. \]

这两部分分别叫做 Laurent 展开的解析部分 (analytic part) 与主要部分 (principal part)。这两个名字都富有提示：

  • 解析部分是一个普通的（向 0 累加的）幂级数；它在整个圆盘 \(|z - z_0| < R\) 内（不只是环里）都收敛、都解析。
  • 主要部分包含所有负次幂；它就是奇点处"奇异行为"的具体表达——它在外面 \(|z - z_0| > r\) 上收敛，在 \(z_0\) 处发散（除非全为零）。它"主导"了奇点附近的局部行为，所以叫"主要部分"。

这种分解非常实用：当我们关心 \(z_0\) 处奇点的"性质"时，只看主要部分就够；解析部分只是"背景"。下一节就把这件事用到判别奇点类型上。

13.6 由 Laurent 展开判别奇点类型

设 \(z_0\) 是 \(f\) 的孤立奇点，所以 \(f\) 在去心邻域 \(0 < |z - z_0| < R\)（也就是环域 \(r = 0\)）上解析。在这个去心圆盘上，\(f\) 有 Laurent 展开

\[ f(z) = \sum_{n = -\infty}^{\infty} a_n (z - z_0)^n. \]

奇点的类型完全由主要部分（即 \(n < 0\) 的系数）决定：

奇点类型判别（Laurent 视角）

  • 所有 \(a_n = 0\)（\(n < 0\)）⟺ \(z_0\) 是可去奇点。
  • 有限多个 \(a_n \neq 0\)（\(n < 0\)），最负的非零下标是 \(n = -m\)（\(m \geq 1\)）⟺ \(z_0\) 是 \(\boldsymbol{m}\) 阶极点。
  • 有无限多个 \(a_n \neq 0\)（\(n < 0\)）⟺ \(z_0\) 是本性奇点。

这个判别准则的好处是一目了然——你不需要计算极限、不需要绕一圈论证，只需要看主要部分有几项。这与上一章基于"\(\lim f\) 的行为"的判别是等价的，但更具操作性。

看几个例子：

例 13.6.1　\(f(z) = \dfrac{\sin z}{z}\) 在 \(z = 0\) 附近的 Laurent 展开。\(\sin z = z - z^3/6 + z^5/120 - \cdots\)，所以

\[ \dfrac{\sin z}{z} = 1 - \dfrac{z^2}{6} + \dfrac{z^4}{120} - \cdots. \]

所有负次幂系数都为 0。所以 \(z = 0\) 是可去奇点。这与上一章的判别（\(\lim_{z \to 0} \sin z / z = 1\) 有限）一致。

例 13.6.2　\(f(z) = \dfrac{\cos z - 1}{z^4}\) 在 \(z = 0\) 附近。\(\cos z - 1 = -z^2/2 + z^4/24 - z^6/720 + \cdots\)，所以

\[ \dfrac{\cos z - 1}{z^4} = -\dfrac{1}{2 z^2} + \dfrac{1}{24} - \dfrac{z^2}{720} + \cdots. \]

主要部分只有一项 \(-\dfrac{1}{2 z^2}\)；最负的下标是 \(-2\)。所以 \(z = 0\) 是二阶极点。

例 13.6.3　\(f(z) = e^{1/z}\) 在 \(z = 0\) 附近。\(e^w = \sum w^n / n!\)，代 \(w = 1/z\)：

\[ e^{1/z} = \sum_{n=0}^{\infty} \dfrac{1}{n! \, z^n} = 1 + \dfrac{1}{z} + \dfrac{1}{2! z^2} + \dfrac{1}{3! z^3} + \cdots. \]

主要部分有无穷多项非零（每一项 \(\dfrac{1}{n! z^n}\)，\(n \geq 1\)，系数 \(1/n!\) 都不为 0）。所以 \(z = 0\) 是本性奇点。这与上一章的判别（沿不同方向极限不同）一致，但用 Laurent 一眼就看出来。

例 13.6.4　\(f(z) = z^2 e^{1/z}\) 在 \(z = 0\) 附近。从上面算得 \(e^{1/z} = 1 + 1/z + 1/(2z^2) + \cdots\)，乘 \(z^2\)：

\[ z^2 e^{1/z} = z^2 + z + \dfrac{1}{2} + \dfrac{1}{3! z} + \dfrac{1}{4! z^2} + \cdots. \]

主要部分仍有无穷多项：\(\dfrac{1}{3! z} + \dfrac{1}{4! z^2} + \cdots\)。所以 \(z = 0\) 仍是本性奇点——乘 \(z^2\) 不能"消除"本性奇点。这是一个一般原则：本性奇点不能被乘任何 \((z - z_0)^k\) 消除。

对极点恰好相反：如果 \(z_0\) 是 \(m\) 阶极点，则 \((z - z_0)^m f(z)\) 是可去的。这正是 12.7 节"极点等价刻画"中 (c) 条款的内容。

13.7 实操技巧：求 Laurent 展开

实际计算 Laurent 展开几乎从来不用积分公式 \(a_n = \tfrac{1}{2\pi i} \oint \cdots\)——那个公式更多是理论价值。实操时主要靠两个工具：

工具 1：几何级数

这是最重要的工具。两种形式：

\[ \dfrac{1}{1 - w} = \sum_{n=0}^{\infty} w^n, \quad \text{当 } |w| < 1, \] \[ \dfrac{1}{1 - w} = -\dfrac{1/w}{1 - 1/w} = -\sum_{n=0}^{\infty} \dfrac{1}{w^{n+1}}, \quad \text{当 } |w| > 1. \]

第一个给出常规幂级数展开；第二个给出"负次幂"展开。判别条件是 \(|w|\) 与 1 的大小关系。

面对 \(\dfrac{1}{a - z}\) 形式的表达式，选哪个公式取决于环域：

  • 在环域内圈外、奇点 \(a\) 内（即 \(|z| < |a|\)），用第一种：\(\dfrac{1}{a - z} = \dfrac{1/a}{1 - z/a} = \dfrac{1}{a} \sum_{n=0}^{\infty} \left(\dfrac{z}{a}\right)^n\)，是 \(z\) 的常规幂级数。
  • 在环域奇点 \(a\) 外（即 \(|z| > |a|\)），用第二种：\(\dfrac{1}{a - z} = -\dfrac{1/z}{1 - a/z} = -\dfrac{1}{z} \sum_{n=0}^{\infty} \left(\dfrac{a}{z}\right)^n = -\sum_{n=0}^{\infty} \dfrac{a^n}{z^{n+1}}\)，是 \(z\) 的负次幂级数。

这两个套路一定要熟。

工具 2：部分分式分解

对有理函数 \(\dfrac{P(z)}{Q(z)}\)，先做部分分式分解，把它化为若干个 \(\dfrac{C}{(z - a)^k}\) 之和，然后对每一项分别用工具 1。这套程序是处理有理函数 Laurent 展开的标准化流水线，每道题都按这个走。

下面看一个非常彻底的例子。

13.8 详细例题：\(\dfrac{1}{(z - 1)(z - 2)}\) 在三个区域

这是 Laurent 展开的"标志性"例题，几乎每本复分析书都讲它。函数

\[ f(z) = \dfrac{1}{(z - 1)(z - 2)} \]

有两个奇点 \(z = 1, z = 2\)（都是简单极点）。在 \(\mathbb{C}\) 上去掉这两个点剩下的部分，被这两个奇点分成三个同心环域（以 \(z = 0\) 为中心）：

  • 区域 \(I\)：\(|z| < 1\)（不含任何奇点的"内圈"）。
  • 区域 \(II\)：\(1 < |z| < 2\)（中间环）。
  • 区域 \(III\)：\(|z| > 2\)（最外圈）。

函数 \(f\) 在每个区域上解析；在每个区域上有一个（且唯一一个）以 \(z_0 = 0\) 为中心的 Laurent 展开。我们要算这三个不同的展开。

第一步：部分分式分解

\[ \dfrac{1}{(z - 1)(z - 2)} = \dfrac{A}{z - 1} + \dfrac{B}{z - 2}. \]

通分：\(A(z - 2) + B(z - 1) = 1\)。代 \(z = 1\)：\(-A = 1 \Rightarrow A = -1\)。代 \(z = 2\)：\(B = 1\)。所以

\[ f(z) = -\dfrac{1}{z - 1} + \dfrac{1}{z - 2}. \]

第二步：在每个区域分别展开

区域 I（\(|z| < 1\)）：在这个区域里 \(|z| < 1 < 2\)，所以两个奇点 \(z = 1, z = 2\) 都"在外面"。两个分式都用第一种几何级数展开：

\[ -\dfrac{1}{z - 1} = \dfrac{1}{1 - z} = \sum_{n=0}^{\infty} z^n. \] \[ \dfrac{1}{z - 2} = -\dfrac{1}{2 - z} = -\dfrac{1/2}{1 - z/2} = -\dfrac{1}{2} \sum_{n=0}^{\infty} \left(\dfrac{z}{2}\right)^n = -\sum_{n=0}^{\infty} \dfrac{z^n}{2^{n+1}}. \]

合并：

\[ f(z) = \sum_{n=0}^{\infty} z^n - \sum_{n=0}^{\infty} \dfrac{z^n}{2^{n+1}} = \sum_{n=0}^{\infty} \left(1 - \dfrac{1}{2^{n+1}}\right) z^n. \]

注意：在区域 I 上，Laurent 展开实际上就是 Taylor 展开（没有负次幂）——因为 \(f\) 在整个 \(|z| < 1\) 上解析，所以 0 处可以做 Taylor 展开。Laurent 退化为 Taylor。

区域 II（\(1 < |z| < 2\)）：现在 \(|z| > 1\) 但 \(|z| < 2\)。第一个奇点 \(z = 1\) 在内圈（\(|z| > 1\)），第二个奇点 \(z = 2\) 还在外（\(|z| < 2\)）。两个分式分别用不同公式：

对 \(-\dfrac{1}{z - 1}\)：此时 \(|z| > 1\)，要用第二种几何级数：

\[ -\dfrac{1}{z - 1} = -\dfrac{1/z}{1 - 1/z} = -\dfrac{1}{z} \sum_{n=0}^{\infty} \dfrac{1}{z^n} = -\sum_{n=0}^{\infty} \dfrac{1}{z^{n+1}} = -\sum_{n=1}^{\infty} z^{-n}. \]

对 \(\dfrac{1}{z - 2}\)：此时 \(|z| < 2\)，用第一种（与区域 I 时相同）：

\[ \dfrac{1}{z - 2} = -\sum_{n=0}^{\infty} \dfrac{z^n}{2^{n+1}}. \]

合并：

\[ f(z) = -\sum_{n=1}^{\infty} z^{-n} - \sum_{n=0}^{\infty} \dfrac{z^n}{2^{n+1}}. \]

这个展开同时含有正次幂和负次幂——这是真正的 Laurent 展开（不是 Taylor 的退化形式）。

区域 III（\(|z| > 2\)）：现在 \(|z| > 2 > 1\)，两个奇点都"在内圈"。两个分式都用第二种几何级数：

\[ -\dfrac{1}{z - 1} = -\sum_{n=1}^{\infty} z^{-n} \quad \text{(同区域 II)}. \] \[ \dfrac{1}{z - 2} = \dfrac{1/z}{1 - 2/z} = \dfrac{1}{z} \sum_{n=0}^{\infty} \left(\dfrac{2}{z}\right)^n = \sum_{n=0}^{\infty} \dfrac{2^n}{z^{n+1}} = \sum_{n=1}^{\infty} \dfrac{2^{n-1}}{z^n}. \]

合并：

\[ f(z) = -\sum_{n=1}^{\infty} z^{-n} + \sum_{n=1}^{\infty} \dfrac{2^{n-1}}{z^n} = \sum_{n=1}^{\infty} (2^{n-1} - 1) z^{-n}. \]

这个展开只含负次幂——因为我们在 \(\infty\) 邻域内，\(f\) 在 \(z \to \infty\) 时趋于 0（\(f \sim 1/z^2\)），所以正次幂全为 0。

三个区域的结论汇总：

区域Laurent 展开性质
\(|z| < 1\)\(\sum_{n=0}^{\infty} \left(1 - \dfrac{1}{2^{n+1}}\right) z^n\)纯非负次幂（=Taylor）
\(1 < |z| < 2\)\(-\sum_{n=1}^{\infty} z^{-n} - \sum_{n=0}^{\infty} \dfrac{z^n}{2^{n+1}}\)正负次幂兼有
\(|z| > 2\)\(\sum_{n=1}^{\infty} (2^{n-1} - 1) z^{-n}\)纯负次幂

三个完全不同的级数，对应同一个函数 \(f(z) = \dfrac{1}{(z - 1)(z - 2)}\)！这一开始让人难以接受，但仔细想一想就明白：每个级数只在自己对应的区域上收敛；三个级数是同一个函数在三个不同区域上的表示。它们不是"相互冲突"的，而是"互补的"——正是 Laurent 展开"在不同环域不同"这一点的具体体现。

这件事的实用含义是：当你要展开一个有理函数为 Laurent 级数时，务必首先明确"以哪一点为中心"和"在哪个环域上"——否则连"哪个 Laurent 展开"都说不清，更别说算了。

来一个稍微变形的例题。

例 13.8.2　把 \(f(z) = \dfrac{1}{(z - 1)(z - 2)}\) 在 \(z_0 = 1\) 附近展开（即在 \(z = 1\) 的某个去心邻域内）。

这次的中心是 \(z_0 = 1\)，不是 0。我们要把 \(f\) 写成 \((z - 1)\) 的幂的级数。技巧：把 \((z - 2)\) 也用 \((z - 1)\) 表示——\((z - 2) = (z - 1) - 1\)。所以

\[ f(z) = \dfrac{1}{(z - 1) \cdot ((z - 1) - 1)} = \dfrac{1}{z - 1} \cdot \dfrac{1}{(z - 1) - 1} = -\dfrac{1}{z - 1} \cdot \dfrac{1}{1 - (z - 1)}. \]

当 \(|z - 1| < 1\) 时（即 \(z\) 在 \(z = 1\) 与 \(z = 2\) 之间的去心邻域），可以把后一个分式用几何级数展开：

\[ \dfrac{1}{1 - (z - 1)} = \sum_{n=0}^{\infty} (z - 1)^n. \]

所以

\[ f(z) = -\dfrac{1}{z - 1} \sum_{n=0}^{\infty} (z - 1)^n = -\sum_{n=0}^{\infty} (z - 1)^{n - 1}. \]

把指数挪一挪：令 \(k = n - 1\)，即 \(n = k + 1\)：

\[ f(z) = -\sum_{k = -1}^{\infty} (z - 1)^k = -\dfrac{1}{z - 1} - 1 - (z - 1) - (z - 1)^2 - \cdots. \]

这是在 \(z_0 = 1\) 附近、环域 \(0 < |z - 1| < 1\) 上的 Laurent 展开。注意主要部分只有一项 \(-\dfrac{1}{z - 1}\)，即 \(a_{-1} = -1\)，更高负次幂系数都为 0——这告诉我们 \(z = 1\) 是 \(f\) 的简单极点，而且留数（即 \(a_{-1}\) 系数）等于 \(-1\)。

这个例子完美呈现了"Laurent 展开 + 主要部分系数读数 = 直接给出极点阶 + 留数"——这就是下一章留数微积分的全部思想雏形。

13.9 留数：\(a_{-1}\) 的特殊地位

在 Laurent 展开

\[ f(z) = \sum_{n = -\infty}^{\infty} a_n (z - z_0)^n \]

的所有系数中，\(a_{-1}\) 这一个系数有一个独特的、深刻的地位。回看系数公式：

\[ a_{-1} = \dfrac{1}{2\pi i} \oint_C f(z)\, dz. \]

所以

\[ \boxed{\oint_C f(z)\, dz = 2\pi i \cdot a_{-1}.} \]

用大白话翻译：解析函数沿着包围孤立奇点 \(z_0\) 的简单闭曲线的积分，等于 \(2\pi i\) 乘以 \(z_0\) 处 Laurent 展开中 \(z^{-1}\) 项的系数。

这个事实本身已经足够惊人——一个看似难算的环路积分，被一个单一的 Laurent 系数完全决定。它给"环路积分的计算"提供了一个极其有效的"代数"方法：你不必直接算积分，只需要算出 Laurent 展开的一个系数即可。

这个 \(a_{-1}\) 系数被起了个特殊的名字——留数 (residue)，记作

\[ \operatorname{Res}_{z = z_0} f(z) := a_{-1}. \]

"留数"这个名字暗示着"剩下的、留下的部分"——它是积分中"实际留下"的那一项。其他所有 Laurent 系数 \(a_n\)（\(n \neq -1\)）的积分都为 0（因为 \(\oint (z - z_0)^n\, dz = 0\) 对一切 \(n \neq -1\)），唯一"留下"贡献的就是 \(n = -1\) 这一项。

第 14 章我们将把这件事推广到"被多个奇点包围的曲线"，得到完整的留数定理：

\[ \oint_C f(z)\, dz = 2\pi i \sum_k \operatorname{Res}_{z = z_k} f(z), \]

对 \(C\) 围住的所有孤立奇点 \(z_k\) 求和。这个定理把"算积分"的工作转化为"算留数"的工作，而留数往往可以从 Laurent 展开瞬间读出（特别是对极点）。它会成为复分析里最强大、最实用的工具之一——能用它算出大量在实分析里几乎不可能算的实积分（比如 \(\int_{-\infty}^{\infty} \dfrac{dx}{1 + x^2}\)、\(\int_0^{2\pi} \dfrac{d\theta}{a + b \cos \theta}\) 等等）。

本章结束，我们已经为下一章铺好了路——所有需要的概念（孤立奇点、Laurent 展开、留数）都在台面上了。剩下的就是把它们串起来变成一个能"动手算积分"的强大机器。继续加油。
