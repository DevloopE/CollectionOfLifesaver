第 3 章 复微分与解析性

第 1、2 两章里我们做的全是"清点门面"的工作：把复数当对象认识，把它的几何、极坐标、辐角搞清楚，把指数 \(e^z\)、对数 \(\log z\)、幂 \(z^\alpha\) 这几个最常用的初等函数定义出来。这些都是静态的认识。从本章开始我们要做动态的事——研究复函数怎么"变化"。换句话说，我们要把单变量微积分里的"导数"那一套，搬到复平面上。

表面上看，这件事好像没什么新意。我们在实数里有 \(f'(x) = \lim_{h \to 0} \dfrac{f(x+h) - f(x)}{h}\)；只要把里面所有的"实变量"换成"复变量"，定义就抄过来了。是的，定义确实可以抄过来——但是它的含义会发生你完全意想不到的变化。这是本章你要反复体会的核心反差：形式一样，威力悬殊。我们将看到，"复可导"这件事远比"实可导"苛刻得多；但代价是值得的——一旦一个复函数在某个区域里处处可导（这种函数将被称为解析函数 (analytic function)，或者等价地全纯函数 (holomorphic function)），它就会拥有一连串极其漂亮的性质：它自动可以求任意阶导数、它可以展开成幂级数、它的积分可以围着回路缩为零、它的零点是孤立的、它由任何一个开子集上的取值就被唯一决定……这些"惊喜"都会在第 5、6 章接连出现。本章只先把第一个台阶——可导性的定义——彻底讲清。

本章的具体内容是：

  • 把"复导数"按定义写出来，并立刻强调"\(h\) 是复数"的含义。
  • 用两个反复出现的具体例子 (\(f(z) = z^2\)，\(f(z) = \bar z\)) 演示"什么时候定义是好的、什么时候不是"。
  • 一个特别狡猾的例子 \(f(z) = |z|^2\)：它仅在原点这一点可导。
  • 从"方向性"的失败/成功里，预告即将出现的 Cauchy-Riemann 方程。
  • 正式定义"解析函数"和"整函数 (entire function)"，列出几个最常见的整函数。
  • 把单变量微分学里的求导规则——加、减、乘、除、链式法则、商法则——确认它们在复变量下原封不动成立。

3.1 复导数的定义

设 \(\Omega \subset \mathbb{C}\) 是一个开集，\(f: \Omega \to \mathbb{C}\) 是定义在 \(\Omega\) 上的一个函数，\(z_0 \in \Omega\) 是 \(\Omega\) 内的一个点。我们说 \(f\) 在 \(z_0\) 处复可导 (complex differentiable)，如果下面这个极限存在并且是一个复数：

\[ f'(z_0) = \lim_{h \to 0} \frac{f(z_0 + h) - f(z_0)}{h}. \]

这里 \(h\) 是一个非零的复数，且当 \(|h|\) 充分小时 \(z_0 + h\) 仍在 \(\Omega\) 里（这是为什么我们要求 \(\Omega\) 是开集——保证 \(z_0\) 周围有一整个圆盘可供 \(h\) 取值）。极限"存在"的含义是：存在某个复数 \(L\)，使得对任意 \(\varepsilon > 0\)，存在 \(\delta > 0\)，只要 \(0 < |h| < \delta\)，就有

\[ \left| \frac{f(z_0 + h) - f(z_0)}{h} - L \right| < \varepsilon. \]

这时候我们就把这个 \(L\) 称作 \(f\) 在 \(z_0\) 处的复导数，记作 \(f'(z_0)\) 或 \(\dfrac{df}{dz}\Big|_{z_0}\)。

把这件事跟实数情形并排放一下：

\[ \text{实情形:} \quad f'(x_0) = \lim_{h \to 0,\ h \in \mathbb{R}} \frac{f(x_0 + h) - f(x_0)}{h}. \] \[ \text{复情形:} \quad f'(z_0) = \lim_{h \to 0,\ h \in \mathbb{C}} \frac{f(z_0 + h) - f(z_0)}{h}. \]

表面上唯一的区别就是 \(h\) 取值范围：在实情形里 \(h\) 是一个实数，沿着一根实轴（一维）逼近 0；在复情形里 \(h\) 是一个复数，从复平面（二维）的任何方向逼近 0。这个看似不起眼的差别，是复分析与实分析所有差异的根源。下一节我们就来把它的含义掰开揉碎。

3.2 "无论从哪个方向"——一个非常苛刻的条件

让我们停下来仔细想一下"\(h \to 0\) 在复平面上"到底意味着什么。在实直线上，\(h\) 趋于 0 只有两种"路径"：从右边逼近 (\(h \to 0^+\))，或者从左边逼近 (\(h \to 0^-\))。我们在实分析里已经知道：左极限和右极限必须相等，否则函数在这一点不可导。

到了复平面，\(h\) 趋于 0 的"路径"瞬间膨胀为无穷多种。\(h\) 可以沿着实轴正方向逼近 (\(h = t\)，\(t \to 0^+\))，可以沿着实轴负方向，可以沿着虚轴正方向 (\(h = it\)，\(t \to 0^+\))，可以沿着虚轴负方向，可以沿着任何斜率的射线 (\(h = t e^{i\theta}\)，\(t \to 0^+\)，\(\theta\) 任选)，甚至可以沿着一条螺旋线、一条折线、一条余弦曲线……只要这条路径最终把 \(|h|\) 压到 0。

而极限存在这个要求，意味着上面那个差商 \(\dfrac{f(z_0 + h) - f(z_0)}{h}\) 沿所有这些路径都得趋于同一个复数 \(L\)。任何两条路径如果给出不一样的极限，函数就不可导。

注意这里"差商"本身也是个复数。在实情形下差商是个实数，"两个实数相等"已经足够苛刻了；在复情形下差商是复数——要让所有方向的差商都趋于同一个复数，得让它的实部和虚部同时统一。这是一个二维约束：实部相等、虚部相等。

因此，"复可导"这个要求实际上是同时塞进了无穷多个条件——每一条逼近 \(z_0\) 的路径都给出一个差商极限，所有这些极限值必须重合。这跟"实可导"只需协调"左极限 = 右极限"两条路径形成了天壤之别。我们立刻就会看到，这个苛刻性如何把"几乎所有人能想到的初等函数"拦在门外，又如何把那些幸运通过的函数一下子赋予非凡的性质。

下面三节我们做三个具体例子来感受这种差别。

3.3 第一个好例子：\(f(z) = z^2\)

来一个最简单的多项式 \(f(z) = z^2\)。它在任意一点 \(z\) 处可导吗？按定义算：

\[ \frac{f(z + h) - f(z)}{h} = \frac{(z + h)^2 - z^2}{h} = \frac{z^2 + 2 z h + h^2 - z^2}{h} = \frac{2 z h + h^2}{h} = 2 z + h. \]

当 \(h \to 0\) 时（无论 \(h\) 沿什么路径趋于 0），右边 \(2 z + h\) 都趋于 \(2 z\)。所以极限存在并等于 \(2 z\)：

\[ \boxed{f'(z) = 2 z.} \]

结论跟实情形 \((x^2)' = 2 x\) 一字不差。这并不奇怪——刚才整个推导过程里，我们既没有用到 \(z\) 是复数这件事，也没有用到 \(h\) 是复数这件事。整个代数操作 \((z + h)^2 = z^2 + 2 z h + h^2\) 在任何一个交换的代数结构里都成立。所以多项式（以及它们的代数组合）在复变量下导数照样存在，公式照样不变。

更一般地，对任意正整数 \(n\)，可以推广得

\[ (z^n)' = n z^{n-1}, \]

推导跟二项式展开 \((z+h)^n = z^n + n z^{n-1} h + O(h^2)\) 完全一样。这里 \(O(h^2)\) 表示一堆带 \(h^2, h^3, \ldots\) 因子的项，除以 \(h\) 后仍含 \(h\)，故当 \(h \to 0\) 时整体趋零。

所以从结论看来，"\(z^2\) 处处可导"跟"\(x^2\) 处处可导"是一样的——一切如常。但请记住：一切如常的真正原因是，幂运算具有内在的多项式代数结构，跟"是实数还是复数"无关。一旦我们离开多项式，麻烦立刻浮现。

3.4 第一个反例：\(f(z) = \bar z\) 处处不可导

下面看一个看似无害、其实致命的函数：复共轭函数

\[ f(z) = \bar z. \]

如果 \(z = x + iy\)，那么 \(f(z) = x - iy\)。这个函数把上半平面翻折到下半平面，是一个实平面上看起来非常"光滑"的对称变换。如果你告诉一个不学复变的物理同学"这个函数是不是可微的呢"，他八成会回答"显然可微"——因为它的实部 \(u(x,y) = x\) 和虚部 \(v(x,y) = -y\) 都是漂亮的多项式，分别对 \(x, y\) 求偏导都给常数。从二元实分析的角度看，这是一个 \(C^\infty\)、甚至解析（实意义的）的二维映射。

但复可导是另外一回事。我们按定义算：

\[ \frac{f(z + h) - f(z)}{h} = \frac{\overline{z + h} - \bar z}{h} = \frac{\bar z + \bar h - \bar z}{h} = \frac{\bar h}{h}. \]

问题来了：当 \(h \to 0\) 时，比值 \(\bar h / h\) 趋于一个固定的复数吗？

沿实轴方向逼近。 设 \(h = t\)，其中 \(t\) 是非零实数。那么 \(\bar h = \bar t = t\)（实数的共轭就是它自己），所以

\[ \frac{\bar h}{h} = \frac{t}{t} = 1. \]

无论 \(t\) 多小，比值都精确等于 1。所以沿实轴方向，差商极限是 \(1\)。

沿虚轴方向逼近。 设 \(h = it\)，其中 \(t\) 是非零实数。那么 \(\bar h = \overline{it} = -it\)，所以

\[ \frac{\bar h}{h} = \frac{-it}{it} = -1. \]

沿虚轴方向，差商极限是 \(-1\)。

两条路径，两个截然不同的极限值（\(1 \ne -1\)），所以差商在 \(h \to 0\) 时根本没有公共极限。结论：\(f(z) = \bar z\) 在任何一点都不复可导。这不是哪一个特殊点的事——上面的推导对任意 \(z_0\) 都成立，因为 \(\bar h / h\) 的值跟 \(z_0\) 完全无关。

不妨更狠地补一刀。把 \(h = t e^{i\theta}\) 写成沿任意角度 \(\theta\) 的射线，那么

\[ \frac{\bar h}{h} = \frac{t e^{-i\theta}}{t e^{i\theta}} = e^{-2i\theta}. \]

这个值随着 \(\theta\) 在单位圆上转，每一个不同的方向 \(\theta\) 给出一个不同的极限——它分布在整个单位圆周上。极限不仅"沿不同方向不同"，而且所有可能的方向都给出不同的值。共轭函数复可导的失败是彻底的、致命的。

这个例子是整个复分析里最重要的"反例"之一，因为它揭穿了一种朴素的误解——"实平面上光滑的二元函数，搬到复平面上就是复可导的"。错。复可导比"两个分量都光滑"严苛得多，它要求两个分量之间满足一种耦合关系。这种耦合关系就是下一章要登场的 Cauchy-Riemann 方程。

3.5 一个更狡猾的例子：\(f(z) = |z|^2\) 仅在原点可导

再看一个更微妙的函数 \(f(z) = |z|^2 = z \bar z\)。如果 \(z = x + iy\)，那么 \(|z|^2 = x^2 + y^2\)，是一个完全实数值的函数（从复平面映到实数轴）。它处处不为负，在原点取最小值 0。这个函数在哪些点复可导？

按定义算差商：

\[ \frac{f(z + h) - f(z)}{h} = \frac{|z + h|^2 - |z|^2}{h} = \frac{(z + h)(\bar z + \bar h) - z \bar z}{h}. \]

把分子展开：

\[ (z + h)(\bar z + \bar h) = z \bar z + z \bar h + h \bar z + h \bar h. \]

减掉 \(z \bar z\) 后剩下 \(z \bar h + h \bar z + h \bar h\)，所以差商 = \(\dfrac{z \bar h + h \bar z + h \bar h}{h} = \dfrac{z \bar h}{h} + \bar z + \bar h\)。

当 \(h \to 0\) 时，最后一项 \(\bar h \to 0\)，没问题。中间一项 \(\bar z\) 是常数。但第一项 \(\dfrac{z \bar h}{h} = z \cdot \dfrac{\bar h}{h}\)——又是讨厌的 \(\bar h / h\)！我们在 3.4 里已经看到了，\(\bar h / h\) 在 \(h \to 0\) 时根本没有公共极限，它在单位圆上随方向变化。

所以差商一般有极限当且仅当 \(z \cdot \dfrac{\bar h}{h}\) 在 \(h \to 0\) 时有公共极限。如果 \(z \ne 0\)，把 \(z\) 看成非零常数提到外边，它就要求 \(\bar h / h\) 本身有极限，但我们知道这不行。所以当 \(z \ne 0\) 时函数不可导。

但当 \(z = 0\) 时呢？此时 \(z \cdot \bar h / h = 0 \cdot \bar h / h = 0\)（无论 \(\bar h / h\) 是什么）。差商化为

\[ \frac{f(0 + h) - f(0)}{h} = \frac{|h|^2 - 0}{h} = \frac{h \bar h}{h} = \bar h. \]

当 \(h \to 0\) 时，\(\bar h \to 0\)。所以差商极限存在并等于 0。结论：\(f(z) = |z|^2\) 仅在原点 \(z = 0\) 处复可导，且 \(f'(0) = 0\)；在其它任何点都不可导。

这个例子比 3.4 更耐人寻味。它告诉我们，复可导与点有关——一个函数可以在某些点可导而在另一些点不可导。但它在原点的"可导"是一种孤立、贫瘠的可导：它仅此一点可导。这种"孤岛式"的可导其实没什么用——后面我们将看到，复分析真正关心的是函数在整个开集上处处可导（也就是"解析"），那时候才会迸发出 Cauchy 公式、留数定理等令人惊叹的性质。一个仅在原点可导、在其它点都不可导的函数，本质上跟不可导没什么本质区别——它无法像解析函数那样获得"无穷阶可导"或"幂级数表示"等好处。

所以正确的态度是：把"在某一点可导"理解为定义层面的最小要求，把"在某个开集上处处可导（解析）"理解为我们真正关心的概念。这两件事在复分析里差异巨大，跟实分析里"可导"和"光滑"的差异完全不是一个数量级。

3.6 从方向性导出 Cauchy-Riemann 方程的雏形

让我们从前面三个例子里抽象出共性。无论是哪一个例子，我们都是把 \(h\) 沿不同方向逼近 0、然后看差商极限。\(z^2\) 沿任何方向都给同一个极限 \(2z\)；\(\bar z\) 沿不同方向给不同极限；\(|z|^2\) 在原点之外沿不同方向给不同极限。

把这个观察提炼一下，就得到一个非常实用的"必要条件"：

若 \(f\) 在 \(z_0\) 处复可导，则差商沿任意方向逼近的极限都必须相等。

让我们用最简单的两条方向——实方向 \(h = \Delta x\)（其中 \(\Delta x\) 是实数，\(\Delta x \to 0\)）和纯虚方向 \(h = i \Delta y\)（其中 \(\Delta y\) 是实数，\(\Delta y \to 0\)）——把这条必要条件挤出一个具体的方程。

设 \(f(z) = u(x, y) + i v(x, y)\)，其中 \(u, v\) 是 \(z = x + iy\) 的实部和虚部分别对应的实值函数。沿实方向：

\[ \frac{f(z_0 + \Delta x) - f(z_0)}{\Delta x} = \frac{[u(x_0 + \Delta x, y_0) - u(x_0, y_0)] + i [v(x_0 + \Delta x, y_0) - v(x_0, y_0)]}{\Delta x}. \]

当 \(\Delta x \to 0\) 时，这趋于 \(u_x(x_0, y_0) + i v_x(x_0, y_0)\)，即对 \(x\) 的偏导。

沿纯虚方向 \(h = i \Delta y\)：

\[ \frac{f(z_0 + i \Delta y) - f(z_0)}{i \Delta y} = \frac{[u(x_0, y_0 + \Delta y) - u(x_0, y_0)] + i [v(x_0, y_0 + \Delta y) - v(x_0, y_0)]}{i \Delta y}. \]

注意分母是 \(i \Delta y\) 不是 \(\Delta y\)。把 \(i\) 提到分母外面（即乘以 \(1/i = -i\)）后，当 \(\Delta y \to 0\) 时这趋于

\[ \frac{1}{i}\big[u_y(x_0, y_0) + i v_y(x_0, y_0)\big] = -i u_y + v_y = v_y - i u_y. \]

"复可导"要求这两个极限相等：

\[ u_x + i v_x = v_y - i u_y. \]

实部对实部、虚部对虚部分别相等，得

\[ u_x = v_y, \qquad v_x = -u_y. \]

这就是柯西—黎曼方程 (Cauchy-Riemann equations)，常常缩写为 CR。这是必要条件——任何在 \(z_0\) 处复可导的函数，它的实虚部 \(u, v\) 必满足上面这两个偏微分方程。

到这里我们只是"预告"了 CR——下一章会专门把它推导清楚、把它的充分性条件搞明白、并把它跟"解析"概念严密对接。本章先把这个雏形放在你脑里，让你下一节看"解析函数"定义时心里有底：所谓"解析"的等价描述就是 \(u, v\) 处处满足 CR（在适当连续性假设下）。

3.7 解析函数的定义

现在终于可以正式给出复分析里最重要的概念。

定义。 设 \(\Omega\) 是 \(\mathbb{C}\) 的一个开集。函数 \(f: \Omega \to \mathbb{C}\) 称为在 \(\Omega\) 上解析 (analytic) 或全纯 (holomorphic)，如果它在 \(\Omega\) 内的每一点都复可导。

"解析"和"全纯"在复分析里是同义词，指同一件事。"解析"这个词更老，强调它能展开为幂级数（这件事我们到第 5 章证明）；"全纯"是后来 Cauchy/Weierstrass 之后逐渐流行的术语，强调"处处复可导"这一基础定义。本书中我们大体上用"解析"，偶尔用"全纯"，含义相同。

注意三件事：

第一，"解析"是开集上的性质，不是单点上的性质。我们不说 "\(f\) 在点 \(z_0\) 处解析"，而要说 "\(f\) 在 \(z_0\) 的某个邻域上解析"。原因是 3.5 节那个反例 \(f(z) = |z|^2\) 在原点这一孤立点可导，但它在原点周围任何小邻域内都不解析。仅一点可导而周围都不可导的函数，不会显示出解析函数的"漂亮性质"——所以"解析"被定义为开集性质，把这种孤岛式可导排除在外。

第二，开集 \(\Omega\) 不必连通。一般情况下 \(\Omega\) 可以是若干个互不相交的开集的并；只要在每个连通分量上 \(f\) 都处处可导，我们就说 \(f\) 在 \(\Omega\) 上解析。但绝大多数有意思的定理（例如恒等定理、最大模原理）都假设 \(\Omega\) 是区域 (domain)，即连通的开集。

第三，"解析"是由"处处可导"定义的；这个定义看起来非常基础，没用任何高级武器。但它的后果极其丰富——一旦 \(f\) 在 \(\Omega\) 上解析，我们将能证明：\(f\) 在 \(\Omega\) 内处处无穷次可导（不只是一次！），它能在任何一点周围展开为收敛的幂级数，它满足 Cauchy 积分公式，它的零点是孤立的，等等。这些事情现在听上去像广告语，但确实都是从这条朴素定义推出的。

3.8 整函数

有一类特别重要的解析函数：在整个 \(\mathbb{C}\) 上都解析的函数，叫做整函数 (entire function)。

已经知道哪些是整函数？凭目前的工具，可以确认下面这些：

  • 所有多项式 \(p(z) = a_n z^n + a_{n-1} z^{n-1} + \cdots + a_0\)。我们 3.3 节算了 \((z^n)' = n z^{n-1}\)，再加上即将证明的"加法、常数倍数也封闭"，所以多项式处处可导，整函数。
  • 指数函数 \(e^z\)。它的复可导性可以用幂级数 \(e^z = \sum_{n=0}^\infty z^n / n!\) 逐项求导（这件事的合法性属于幂级数理论，第 5 章证明），结果 \((e^z)' = e^z\)，处处成立。
  • 三角函数 \(\sin z, \cos z\)。它们由 \(\sin z = (e^{iz} - e^{-iz})/(2i)\)，\(\cos z = (e^{iz} + e^{-iz})/2\) 定义，自然继承了 \(e^{iz}\) 的处处可导性。求导规则也跟实情形一致：\((\sin z)' = \cos z\)，\((\cos z)' = -\sin z\)。
  • 双曲函数 \(\sinh z, \cosh z\)。同上理由，整函数。

反过来，下面这些不是整函数（虽然它们在某些区域内是解析的）：

  • 有理函数 \(R(z) = p(z) / q(z)\)。在 \(q(z) \ne 0\) 处解析，但在 \(q(z) = 0\) 的离散点处不定义（甚至发散），所以不是整函数。比如 \(1/z\) 不是整函数——它在原点不定义。
  • 对数函数 \(\log z\)。我们在第 2 章已经看到，\(\log z\) 是多值函数；要把它变成单值函数需要"切一刀"——比如在负实轴上切，得到主值 \(\mathrm{Log}\, z\)，定义在 \(\mathbb{C} \setminus (-\infty, 0]\) 上。它在这个开集上是解析的，但既然它不能定义为 \(\mathbb{C}\) 上的解析函数，自然不是整函数。
  • 幂函数 \(z^\alpha\)（\(\alpha\) 不是整数时）。同样涉及多值性问题，需要切支。
  • 共轭函数 \(\bar z\)。3.4 节已证明它处处不可导，远不是整函数。

整函数的"稀少"反映了复可导的苛刻。后面我们会看到一个让人震撼的结论（Liouville 定理，第 6 章）：有界的整函数必为常数。这条定理一句话就把"非常数的整函数"逼到必然在某些方向上爆掉。代数基本定理也将作为它的简短推论出现。

3.9 解析函数的代数运算

解析函数对常见的代数运算和复合运算都封闭。具体来说：

定理 (运算的封闭性)。设 \(f, g\) 在开集 \(\Omega\) 上解析。则：

  • 线性组合：对任意复常数 \(\alpha, \beta\)，\(\alpha f + \beta g\) 在 \(\Omega\) 上解析，\((\alpha f + \beta g)' = \alpha f' + \beta g'\)。
  • 乘积：\(f g\) 在 \(\Omega\) 上解析，\((f g)' = f' g + f g'\)（莱布尼茨乘积法则）。
  • 商：在 \(g(z) \ne 0\) 的所有点上 \(f / g\) 解析，\(\left(f / g\right)' = (f' g - f g') / g^2\)。
  • 复合：若 \(g\) 在 \(\Omega\) 上解析、\(f\) 在 \(g(\Omega)\) 的某个开集上解析，则 \(f \circ g\) 在 \(\Omega\) 上解析，\((f \circ g)'(z) = f'(g(z)) \cdot g'(z)\)（链式法则）。

这些规则证明跟实情形几乎一字不差，因为整个证明只依赖于：

  1. 极限的代数性质（极限保运算）。
  2. 差商的纯代数恒等式。

这两条在复极限里照样成立——只是"极限"的含义从"\(h \in \mathbb{R}\)"变成"\(h \in \mathbb{C}\)"。下面我们简单走一下乘积法则的推导，让你看到"代数照搬"的具体含义。

乘积法则的推导。设 \(f, g\) 在 \(z_0\) 复可导。考虑差商：

\[ \frac{(fg)(z_0 + h) - (fg)(z_0)}{h} = \frac{f(z_0 + h) g(z_0 + h) - f(z_0) g(z_0)}{h}. \]

用一个经典技巧：在分子里加减一项 \(f(z_0 + h) g(z_0)\)：

\[ = \frac{f(z_0 + h) g(z_0 + h) - f(z_0 + h) g(z_0) + f(z_0 + h) g(z_0) - f(z_0) g(z_0)}{h}. \]

分组：

\[ = f(z_0 + h) \cdot \frac{g(z_0 + h) - g(z_0)}{h} + g(z_0) \cdot \frac{f(z_0 + h) - f(z_0)}{h}. \]

当 \(h \to 0\) 时：第一项里 \(f(z_0 + h) \to f(z_0)\)（因为 \(f\) 复可导，特别地连续），\(\dfrac{g(z_0 + h) - g(z_0)}{h} \to g'(z_0)\)；第二项里 \(g(z_0)\) 是常数，差商极限是 \(f'(z_0)\)。所以总极限

\[ (fg)'(z_0) = f(z_0) g'(z_0) + f'(z_0) g(z_0), \]

这就是莱布尼茨乘积法则。整个推导跟实数情形完全一样——它从未用到 \(z_0, h\) 是实数还是复数，只用到了"差商极限存在"这件事的代数后果。

顺便注意一下我们用了"复可导推连续性"这个事实。这条事实跟实情形一样直接：若 \(f\) 在 \(z_0\) 复可导，写

\[ f(z_0 + h) - f(z_0) = h \cdot \frac{f(z_0 + h) - f(z_0)}{h}. \]

当 \(h \to 0\) 时，右边 \(h \to 0\)、差商极限是 \(f'(z_0)\)（有限），所以乘积 \(\to 0\)，故 \(f(z_0 + h) \to f(z_0)\)，连续。

商法则、链式法则的推导照搬实情形，不再赘述。

3.10 求导规则一览

把上面这些规则集中列出来，便于查阅：

\[ \begin{array}{ll} \text{常数}: & c' = 0. \\[4pt] \text{幂}: & (z^n)' = n z^{n-1}, \quad n \in \mathbb{Z}. \\[4pt] \text{指数}: & (e^z)' = e^z. \\[4pt] \text{三角}: & (\sin z)' = \cos z, \quad (\cos z)' = -\sin z. \\[4pt] \text{双曲}: & (\sinh z)' = \cosh z, \quad (\cosh z)' = \sinh z. \\[4pt] \text{对数 (主值)}: & (\mathrm{Log}\, z)' = \frac{1}{z}, \ \ z \in \mathbb{C} \setminus (-\infty, 0]. \\[4pt] \text{加法/数乘}: & (\alpha f + \beta g)' = \alpha f' + \beta g'. \\[4pt] \text{乘积}: & (f g)' = f' g + f g'. \\[4pt] \text{商}: & \left(\frac{f}{g}\right)' = \frac{f' g - f g'}{g^2}. \\[4pt] \text{链式}: & (f \circ g)'(z) = f'(g(z)) \cdot g'(z). \end{array} \]

每一条都和实情形对应公式形式上完全一致，只是变量从 \(x\) 换成了 \(z\)。这是复分析的一个反复出现的舒服之处：计算层面上你已经会做了，你的微积分肌肉记忆全都派得上用场。复分析的难处不在"算导数"，而在"哪些函数才允许这样算"——这正是 3.4、3.5 两个反例的教训。

3.11 几个具体例题

把上面的规则用几个例题练一遍，免得它们停留在抽象层面。

例 1。 求 \(f(z) = (z^2 + 3 z + 1)^5\) 的导数。

这是一个标准的链式法则问题。把外函数取作 \(w \mapsto w^5\)，内函数取作 \(z \mapsto z^2 + 3 z + 1\)。外函数的导数是 \(5 w^4\)，内函数的导数是 \(2 z + 3\)。代入链式法则：

\[ f'(z) = 5 (z^2 + 3 z + 1)^4 \cdot (2 z + 3). \]

例 2 看起来跟实情形一字不差，因为推导确实如此。

例 2。 求 \(f(z) = e^{z^2}\) 的导数。

外函数 \(w \mapsto e^w\)，导数 \(e^w\)；内函数 \(z \mapsto z^2\)，导数 \(2 z\)。链式法则：

\[ f'(z) = e^{z^2} \cdot 2 z = 2 z\, e^{z^2}. \]

例 3。 求 \(f(z) = \dfrac{\sin z}{z}\) 的导数（在 \(z \ne 0\) 处定义）。

商法则：分子导数 \(\cos z\)，分母导数 \(1\)，所以

\[ f'(z) = \frac{(\cos z) \cdot z - (\sin z) \cdot 1}{z^2} = \frac{z \cos z - \sin z}{z^2}. \]

注意这个函数 \(\sin z / z\) 在 \(z = 0\) 处看似有奇点，但其实可以"解析延拓"过去——令 \(g(0) = 1\)，\(g(z) = \sin z / z\) 否则，那么 \(g\) 是整函数。证明在第 5 章用 \(\sin z\) 的幂级数展开 \(\sin z = z - z^3/6 + z^5/120 - \cdots\) 立刻得到 \(\sin z / z = 1 - z^2/6 + z^4/120 - \cdots\)，这是一个处处收敛的幂级数，所以是整函数。

例 4。 验证 \(f(z) = \bar z\) 在某点处复可导是不可能的。

3.4 节我们已经做过这件事，但值得用"链式法则"的角度再看一下，因为有人会说"\(\bar z\) 是 \(z\) 的函数，能否用链式法则？"答：不能。链式法则要求外函数和内函数都要解析；而共轭操作 \(z \mapsto \bar z\) 就不是解析的（这正是我们在证明的事情！）。所以链式法则在这里根本启动不了。

这其实揭示了一个有用的判据：凡是写出来含有 \(\bar z\) 的"显式表达式"，都不是解析函数。例如 \(f(z) = z + \bar z = 2 x\)、\(f(z) = z \bar z = |z|^2\)、\(f(z) = z - \bar z = 2 i y\) 等等都不解析（第一个、第三个是实部和虚部的两倍，根本不是 \(z\) 的"函数"在通常意义上）。所以"看到 \(\bar z\) 出现，立刻警觉"是一个非常可靠的诊断习惯。

例 5。 求 \(f(z) = z^3 e^{2z}\) 的导数。

乘积法则：

\[ f'(z) = (z^3)' \cdot e^{2z} + z^3 \cdot (e^{2z})' = 3 z^2 \cdot e^{2z} + z^3 \cdot 2 e^{2z} = (3 z^2 + 2 z^3) e^{2z} = z^2 (3 + 2 z) e^{2z}. \]

注意 \((e^{2z})' = 2 e^{2z}\) 这里用到了链式法则——外函数 \(e^w\)、内函数 \(2z\)。

例 6。 已知 \(f(z) = (z^2 - 1)/(z + 1)\)，求 \(f'(z)\)。

这里有个小技巧。直接用商法则当然行：

\[ f'(z) = \frac{2z \cdot (z + 1) - (z^2 - 1) \cdot 1}{(z + 1)^2} = \frac{2 z^2 + 2 z - z^2 + 1}{(z + 1)^2} = \frac{z^2 + 2 z + 1}{(z + 1)^2} = \frac{(z + 1)^2}{(z + 1)^2} = 1. \]

更优雅的做法：先约分，\(\dfrac{z^2 - 1}{z + 1} = \dfrac{(z - 1)(z + 1)}{z + 1} = z - 1\)（在 \(z \ne -1\) 处）。所以 \(f'(z) = 1\)。两种方法答案吻合，但显然先约分省力。这是个好习惯：算导数前先化简函数，省下不少麻烦。

这六个例子覆盖了你之后会反复遇到的运算模式：链式法则、乘积法则、商法则、识别非解析函数。把它们消化掉，你的"计算肌肉"就建立好了。下一章我们调转方向——不再问"怎么算导数"，而是问"导数的结构告诉我们什么"，那就是 Cauchy-Riemann 方程的世界。
