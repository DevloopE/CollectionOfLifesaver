附录 B 拓扑预备

本附录把本课程反复用到的拓扑工具放在一起，让你在需要时有一个集中的查阅地。这些概念——开集、闭集、连通性、紧致性——在第一次接触时容易让人觉得"形式抽象"，但它们其实是对几何直觉的精确编码。一旦把它们和复分析里的具体场景对上号——"区域"是开集，"曲线"是道路连通的紧致集，"Cauchy 定理在单连通区域上成立"——它们就从抽象走向实用。

本附录的内容：

  • B.1 度量空间速通：距离、收敛、Cauchy 列。
  • B.2 开集、闭集、邻域：基本词汇。
  • B.3 连通、单连通、多连通：区域的"洞结构"。
  • B.4 道路连通、道路同伦：曲线之间的"连续变形"。
  • B.5 单连通区域的特征：每条闭曲线都可"收缩"。
  • B.6 紧致集；序列紧致；Heine–Borel。
  • B.7 这些为什么对 Cauchy 理论必不可少。

读这部分的方式：把每个定义先在脑里"画"一遍——哪种图形是开的、哪种是闭的、哪种是单连通的——再去理解形式定义。

B.1 度量空间速通

一个度量空间是一个集合 \(X\) 配上一个"距离函数" \(d : X \times X \to [0, \infty)\) 满足：

  1. \(d(x, y) = 0 \iff x = y\)（正定性）。
  2. \(d(x, y) = d(y, x)\)（对称性）。
  3. \(d(x, z) \leq d(x, y) + d(y, z)\)（三角不等式）。

本课程里 \(X\) 几乎总是 \(\mathbb{C}\) 或 \(\mathbb{C}\) 的子集，距离取标准的 \(d(z, w) = |z - w|\)（即把复数看成 \(\mathbb{R}^2\) 中的点的欧几里得距离）。

有了距离就能谈收敛：序列 \(\{z_n\} \subset X\) 收敛到 \(z \in X\)，记 \(z_n \to z\)，意思是 \(d(z_n, z) \to 0\)，等价于：对任意 \(\varepsilon > 0\)，存在 \(N\) 使 \(n \geq N\) 时 \(d(z_n, z) < \varepsilon\)。

序列 \(\{z_n\}\) 称为 Cauchy 列，若对任意 \(\varepsilon > 0\)，存在 \(N\) 使 \(m, n \geq N\) 时 \(d(z_m, z_n) < \varepsilon\)。

度量空间称为完备的，若每个 Cauchy 列都收敛（到该空间内的某点）。\(\mathbb{R}\) 完备（实数的完备性公理）；\(\mathbb{C}\) 完备（继承自 \(\mathbb{R}^2\)）；有理数 \(\mathbb{Q}\) 不完备（比如 Cauchy 列 \(3, 3.1, 3.14, 3.141, \ldots\) 不收敛到任何有理数）。

\(\mathbb{C}\) 完备这件事对复分析至关重要。下面所有的"嵌套定理"、"紧致性"、"一致收敛 \(\Rightarrow\) 连续"等结论，本质上都依赖完备性。

B.2 开集、闭集、邻域

对 \(z \in \mathbb{C}\) 和 \(r > 0\)，开圆盘 \(B_r(z) = \{w \in \mathbb{C} : |w - z| < r\}\)。闭圆盘 \(\overline{B_r(z)} = \{w : |w - z| \leq r\}\)。

子集 \(U \subset \mathbb{C}\) 称为开集，若对每个 \(z \in U\)，存在 \(r > 0\) 使 \(B_r(z) \subset U\)。换言之"在 \(U\) 里的每个点都'四面都还在 \(U\) 里'"。

子集 \(F \subset \mathbb{C}\) 称为闭集，若它的补集 \(\mathbb{C} \setminus F\) 是开的。等价地：若 \(z_n \in F\)、\(z_n \to z\)，则 \(z \in F\)（"包含所有自己的极限点"）。

注意"开"与"闭"不是对立的——一个集合可以既开又闭（如 \(\mathbb{C}\) 本身和空集 \(\emptyset\)），也可以既不开也不闭（如半开区间 \([0, 1) \subset \mathbb{R}\) 看作 \(\mathbb{C}\) 的子集）。"开"和"闭"只是补集关系，不是相反关系。

子集 \(N\) 称为点 \(z\) 的邻域，若 \(N\) 包含某 \(B_r(z)\)。一般我们用"开邻域"——即 \(N\) 本身就是开集。

对一个集合 \(A \subset \mathbb{C}\)：

  • 内部 \(A^\circ\) = "属于 \(A\) 且四面都在 \(A\) 里的点"全体 = \(A\) 中最大的开集。
  • 闭包 \(\overline{A}\) = \(A\) 加上所有它的极限点 = 包含 \(A\) 的最小闭集。
  • 边界 \(\partial A = \overline{A} \setminus A^\circ\) = "要么在 \(A\) 上要么不在 \(A\) 上的'边缘'点"。

例：\(A = \{z : 0 < |z| \leq 1\}\)（去心闭单位圆盘）。\(A^\circ = \{z : 0 < |z| < 1\}\)（开圆盘减去原点）；\(\overline{A} = \{z : |z| \leq 1\}\)（闭圆盘）；\(\partial A = \{z : |z| = 1\} \cup \{0\}\)（单位圆周加上原点）。

B.3 连通、单连通、多连通

子集 \(A \subset \mathbb{C}\) 称为不连通的，若存在两个开集 \(U, V\) 使 \(A \subset U \cup V\)、\(A \cap U \neq \emptyset\)、\(A \cap V \neq \emptyset\)、\(A \cap U \cap V = \emptyset\)。也就是说 \(A\) 可以"被分成两块各自包含在不相交开集里"。

反之，\(A\) 连通，若不存在这样的分割。

例：

  • 开圆盘 \(B_1(0)\) 连通。
  • 两个不相交开圆盘的并 \(B_1(0) \cup B_1(3)\) 不连通。
  • 圆环 \(\{1 < |z| < 2\}\) 连通（虽然中间挖了个洞，但仍是"一块"）。

本课程里我们最常说的"区域"指的就是连通的开集。这条术语是固定的——"区域" = "连通开集"。

对区域 \(\Omega\)，我们再加一个区分：

  • \(\Omega\) 是单连通的，若它"没有洞"——形式定义见 B.5 节。直观例子：开圆盘、上半平面、整个 \(\mathbb{C}\) 都是单连通的。
  • \(\Omega\) 是多连通的，若有"洞"。例：圆环 \(\{1 < |z| < 2\}\) 有一个洞（被挖掉的内圆盘 \(\{|z| \leq 1\}\)），多连通。挖掉两个圆盘的开圆盘"瑞士奶酪"，多连通，洞的个数 = 2。

"洞的个数"严格地说是第一同调群或第一基本群的秩，本课程不展开。在我们用得到的所有场景里，"洞"都是直观的。

B.4 道路连通；道路同伦

一个道路（或称路径）是连续映射 \(\gamma : [0, 1] \to \mathbb{C}\)。\(\gamma(0)\) 是起点，\(\gamma(1)\) 是终点。

子集 \(A \subset \mathbb{C}\) 称为道路连通的，若任意两点 \(z_1, z_2 \in A\) 都能用一条完全在 \(A\) 里的道路 \(\gamma\)（\(\gamma(0) = z_1\)、\(\gamma(1) = z_2\)、\(\gamma([0,1]) \subset A\)）相连。

道路连通 \(\Rightarrow\) 连通（这是简单证明，因为道路本身是 \([0, 1]\) 的连续像，连续像保持连通）。反过来一般不成立，但对开集反过来也成立——开集连通 \(\iff\) 开集道路连通。所以本课程里"区域"既是连通开集也是道路连通开集。

两条同起终点的道路 \(\gamma_0, \gamma_1 : [0, 1] \to \Omega\)（共起点 \(z_0\)、共终点 \(z_1\)，都在 \(\Omega\) 里）称为在 \(\Omega\) 中道路同伦，若存在连续映射

\[ H : [0, 1] \times [0, 1] \to \Omega \]

使得：

  • \(H(t, 0) = \gamma_0(t)\)、\(H(t, 1) = \gamma_1(t)\)（"\(s = 0\) 时是 \(\gamma_0\)、\(s = 1\) 时是 \(\gamma_1\)"）；
  • \(H(0, s) = z_0\)、\(H(1, s) = z_1\) 对所有 \(s\)（端点固定）。

直观地，\(H\) 是一族道路 \(\gamma_s(t) = H(t, s)\)（参数 \(s\) 从 0 到 1），它们把 \(\gamma_0\) 在 \(\Omega\) 内连续变形成 \(\gamma_1\)，整个变形过程不出 \(\Omega\) 的边界。

对闭曲线（\(\gamma(0) = \gamma(1)\)），相应的概念是自由同伦（端点不必固定，但每个 \(\gamma_s\) 都是闭曲线）和定基点同伦（基点 \(z_0\) 不动）。

同伦概念的物理意义：两条道路同伦 \(\iff\) 它们之间没有"洞"挡着。如果 \(\Omega\) 是带一个洞的圆环 \(\{1 < |z| < 2\}\)，那么"绕洞 0 圈"的道路与"绕洞 1 圈"的道路不同伦——洞挡住了变形。这是为什么"绕数"在多连通区域上是不变的拓扑量。

B.5 单连通区域的特征

有了同伦概念，我们可以正式定义单连通：

定义. 区域 \(\Omega\) 称为单连通的，若 \(\Omega\) 中每条闭曲线 \(\gamma\) 都同伦于一个常曲线（即可"收缩"到某个点）。

等价地：\(\Omega\) 中任意两条同起终点的道路都同伦。

等价的"洞洞"描述：\(\Omega\) 单连通 \(\iff\) \(\mathbb{C} \cup \{\infty\} \setminus \Omega\)（在 Riemann 球面上）是连通的。也就是说 \(\Omega\) 的"补集"在球面上是一整块（包括无穷远点的那一块），没有"洞洞"散落在 \(\Omega\) 内部。

一些例子：

  • 开圆盘 \(B_r(0)\) 单连通。
  • 上半平面 \(H = \{\mathrm{Im}\, z > 0\}\) 单连通。
  • 整个 \(\mathbb{C}\) 单连通。
  • "切口平面" \(\mathbb{C} \setminus (-\infty, 0]\)（去掉负实轴）单连通。这是为什么我们能在它上面定义 \(\log z\) 的主分支。
  • 去心平面 \(\mathbb{C} \setminus \{0\}\) 不单连通——原点是一个洞，绕原点的小圆不能收缩。
  • 圆环 \(\{1 < |z| < 2\}\) 不单连通。
  • "瑞士奶酪"（圆盘减去几个内部小圆盘）不单连通；洞的个数即不单连通的"程度"。

本课程里这条性质在两个地方被反复用到：

  1. Cauchy 定理：在单连通区域上 \(f\) 解析 \(\Rightarrow\) \(\oint_\gamma f\, dz = 0\) 对所有闭曲线 \(\gamma\)。证明思路就是用同伦把任意 \(\gamma\) 收缩到一点，整个收缩过程中积分值不变（此为同伦不变性），到一点时积分为 0。
  2. 解析对数与原函数：在单连通区域上每个不取零的解析函数都有解析的对数。这是 Cauchy 定理的直接应用——\(\int \frac{f'}{f}\) 在单连通上是路径无关的。

B.6 紧致集；序列紧致

子集 \(K \subset \mathbb{C}\) 称为紧致的，若它的每个开覆盖都有有限子覆盖。即：若 \(K \subset \bigcup_{i \in I} U_i\)（每个 \(U_i\) 开），则存在有限子集 \(I_0 \subset I\) 使 \(K \subset \bigcup_{i \in I_0} U_i\)。

这条定义看起来抽象，但有一个等价、更直观的版本：

序列紧致. \(K \subset \mathbb{C}\) 序列紧致，若 \(K\) 中每个序列都有收敛子列，且极限在 \(K\) 中。

对度量空间（包括 \(\mathbb{C}\)），紧致 \(\iff\) 序列紧致。所以两条性质可以互换使用。

对 \(\mathbb{C}\) 中的子集，还有一条非常方便的 Heine–Borel 定理：

\[\boxed{K \subset \mathbb{C} \text{ 紧致} \iff K \text{ 闭且有界}.}\]

"闭"指 B.2 意义；"有界"指存在 \(R\) 使 \(K \subset B_R(0)\)。这是判定紧致最实用的工具。

例：

  • 闭圆盘 \(\overline{B_1(0)}\) 紧致（闭且有界）。
  • 开圆盘 \(B_1(0)\) 不紧致（不闭——边界 \(\{|z| = 1\}\) 不在内）。
  • 整个 \(\mathbb{C}\) 不紧致（不有界）。
  • 有限点集 \(\{z_1, \ldots, z_n\}\) 紧致。
  • 圆 \(\{|z| = 1\}\) 紧致（闭且有界）——这是为什么我们的"围道积分"总是在紧曲线上。

紧致集的关键性质：

  1. 极值定理：紧致集上的实值连续函数取到最大值和最小值。
  2. 一致连续：紧致集上的连续函数自动一致连续。
  3. 嵌套：嵌套非空紧致闭集列的交不空。
  4. 有界覆盖：紧致集任何开覆盖都有有限子覆盖。

这四条性质在复分析里被反复用到。

B.7 这些为什么对 Cauchy 理论必不可少

把上面所有概念串起来，看它们在本课程里"打哪些工"：

开集 / 区域：解析函数必须定义在开集上（"在某点解析"意味着该点周围一个邻域上都可微）。所有的"\(f\) 在 \(\Omega\) 上解析"都默认 \(\Omega\) 是开的。本课的"区域"再加连通性，是为了能谈"恒等定理"——非平凡的解析函数不能在区域内部的非空开子集上恒为零。

紧致集与 ML-估计：第 13 章你学了"\(|\oint_\gamma f\, dz| \leq M \cdot L\)" 的 ML-估计，其中 \(M\) 是 \(|f|\) 在 \(\gamma\) 上的最大值。这个 \(M\) 的存在就用了 \(\gamma\) 是紧致集（紧致集上连续函数取最大值）。如果 \(\gamma\) 不紧致（比如是无界的射线），\(|f|\) 可能没有有限上界，ML-估计就失效。

紧致集与一致收敛：第 14 章 Laurent 级数的"在紧致环带上一致收敛"是把它逐项积分的依据。一致收敛允许 \(\sum \int = \int \sum\)。如果只是"在每点收敛"而非紧致一致收敛，就不能换序——这件事在留数定理证明（A.6）里被反复使用。

嵌套闭三角形：A.1 里 Cauchy–Goursat 定理证明的核心是"嵌套闭三角形列的交非空"。这是 \(\mathbb{C}\) 完备性的直接产物（紧致 + 嵌套 + 直径 \(\to 0\) \(\Rightarrow\) 交是单点）。没有完备性，整个 Cauchy 理论从根上塌掉。

单连通性：Cauchy 定理"在单连通区域上 \(\oint f\, dz = 0\)" 的"单连通"是不可省的——把同样的 \(f = 1/z\) 放到非单连通的 \(\mathbb{C} \setminus \{0\}\) 上，绕原点的圆周积分等于 \(2\pi i \neq 0\)。所以"区域是不是有洞"直接决定 Cauchy 理论的形式。

道路同伦：Cauchy 定理的"广义版本"是同伦不变性：若 \(\gamma_0\) 与 \(\gamma_1\) 在 \(\Omega\) 内同伦，则 \(\oint_{\gamma_0} f = \oint_{\gamma_1} f\)。这条是把"积分值是路径函数 \(\to\) 实际上只是同伦类的函数"翻译过去。在多连通区域上，路径同伦类形成一个群（基本群 \(\pi_1(\Omega)\)），积分变成这个群上的同态——这通向代数拓扑。

序列紧致与正规族：在更进阶的复分析（Montel 定理、Riemann 映射定理）里要用"解析函数族的紧致性"——一族一致有界的解析函数有局部一致收敛子列。这条 Montel 定理是序列紧致在解析函数空间上的版本。

把这一切总结成一句话：拓扑提供"几何形状的语言"，复分析在这门语言上谈"积分与导数"。脱开拓扑，复分析的每条定理的"假设栏"都没法说清；脱开复分析，拓扑的"几何性质"也找不到这么漂亮的应用对象。两门学科是天生伴侣。

本附录到此结束。下次你在正文里碰到"区域"、"紧致曲线"、"单连通"、"同伦"等词，就知道它们的精确含义和用处。建议在脑里建立一张"拓扑性质 \(\to\) 复分析定理"的对照表——例如"单连通 \(\to\) Cauchy 积分定理"、"紧致 \(\to\) ML-估计"、"完备性 \(\to\) Cauchy–Goursat"——读起来会顺畅得多。
