第 12 章 零点与孤立奇点

上一章我们讲了"在解析的地方，函数等于它的 Taylor 级数"。这是一个关于"良好行为"的故事——在 \(f\) 解析的圆盘内部，一切井然有序。本章把视角调过来：在 \(f\) 不解析的地方，发生了什么？

"不解析的点"可以分两大类。第一类是零点：\(f\) 在该点解析、值等于 0。零点本身并不是奇点，但它在分式 \(\dfrac{1}{f}\) 那里变成了奇点；而且解析函数零点的结构本身就有非常严格的特征——它们是孤立的，并且每个零点都有一个明确的"阶"。第二类是真正的奇点：\(f\) 在该点本身没定义（或者有定义但不解析），但在它的去心邻域内是解析的。这种点叫孤立奇点，本章后半段都在讲它。

奇点又分三种：可以"补"成解析函数的可去奇点、行为像 \(\dfrac{1}{(z - z_0)^m}\) 一样发散的极点，以及行为狂野到无法形容的本性奇点。这三种奇点有一组非常清晰、非常实用的判别准则；本章把它们一一列出，并辅以例题。

本章覆盖：

  • 解析函数的零点：定义、阶、孤立性。
  • 零点的局部结构 \(f(z) = (z - z_0)^m g(z)\)，\(g(z_0) \neq 0\)。
  • 孤立奇点的定义。
  • 三类孤立奇点：可去、极点、本性。
  • Riemann 可去奇点定理：去心邻域有界 ⇒ 可去。
  • 极点的等价刻画与阶的概念。
  • Casorati-Weierstrass 定理与 Picard 大定理（仅陈述）。
  • 经典例：\(e^{1/z}\) 在 \(z = 0\) 处的本性奇点。
  • 亚纯函数。
  • 无穷远处的奇性。

12.1 零点与零点的阶

设 \(f\) 在区域 \(\Omega\) 上解析，\(z_0 \in \Omega\)。若 \(f(z_0) = 0\)，我们就说 \(z_0\) 是 \(f\) 的一个零点 (zero)。

在 \(z_0\) 附近做 Taylor 展开：

\[ f(z) = a_0 + a_1 (z - z_0) + a_2 (z - z_0)^2 + \cdots, \]

其中 \(a_n = f^{(n)}(z_0)/n!\)。"\(z_0\) 是零点"就是 \(a_0 = f(z_0) = 0\)。但仅仅 \(a_0 = 0\) 还不够细致——我们想知道，从 \(a_0\) 开始，第几个系数才不为 0？这第几个的"几"，就是零点的"阶"。

定义　设 \(z_0\) 是 \(f\) 的零点，\(f \not\equiv 0\)。如果 Taylor 展开里 \(a_0 = a_1 = \cdots = a_{m-1} = 0\) 而 \(a_m \neq 0\)，我们就说 \(z_0\) 是 \(f\) 的一个 \(\boldsymbol{m}\) 阶零点（或重数 \(m\) 的零点）。等价地：\(f(z_0) = f'(z_0) = \cdots = f^{(m-1)}(z_0) = 0\)，而 \(f^{(m)}(z_0) \neq 0\)。

一阶零点（\(m = 1\)）也叫简单零点。

看几个例子立刻清楚：

例 12.1.1　\(f(z) = z\)。在 \(z_0 = 0\) 处，\(f(0) = 0\)，\(f'(0) = 1 \neq 0\)，所以是一阶零点。

例 12.1.2　\(f(z) = z^3\)。在 \(z_0 = 0\) 处，\(f(0) = f'(0) = f''(0) = 0\)，\(f'''(0) = 6 \neq 0\)，所以是三阶零点。一般 \(z^m\) 在 0 处是 \(m\) 阶零点。

例 12.1.3　\(\sin z\) 在 \(z = 0\) 处是几阶零点？由展开 \(\sin z = z - z^3/6 + \cdots\)，第一个非零系数是 \(a_1 = 1\)，所以是一阶零点（简单零点）。

例 12.1.4　\(1 - \cos z\) 在 \(z = 0\) 处是几阶？\(\cos z = 1 - z^2/2 + z^4/24 - \cdots\)，所以 \(1 - \cos z = z^2/2 - z^4/24 + \cdots\)，第一个非零系数是 \(a_2 = 1/2\)，这是二阶零点。

例 12.1.5　\(\sin z - z\) 在 \(z = 0\) 处？\(\sin z - z = -z^3/6 + z^5/120 - \cdots\)，第一个非零系数在 \(z^3\) 项，所以是三阶零点。

12.2 零点的孤立性

解析函数的零点有一个非常严格的几何特征：它们只能是孤立的。具体地：

零点孤立性定理　设 \(f\) 在连通区域 \(\Omega\) 上解析、\(f \not\equiv 0\)。若 \(z_0 \in \Omega\) 是 \(f\) 的零点，则存在 \(z_0\) 的某个邻域 \(U\)，使得 \(f\) 在 \(U \setminus \{z_0\}\) 上没有别的零点。

翻译成大白话：如果 \(f\) 不是恒为 0，那么 \(f\) 的零点之间彼此都隔着一段非零距离。它们不会"挤在一起"。

这件事的证明很短，并且本身就解释了"孤立性"。设 \(z_0\) 是 \(m\) 阶零点。则在 \(z_0\) 附近

\[ f(z) = (z - z_0)^m g(z), \quad \text{其中 } g(z_0) = a_m \neq 0. \]

这里 \(g(z) = a_m + a_{m+1} (z - z_0) + a_{m+2} (z - z_0)^2 + \cdots\)。由于 \(g\) 是这个幂级数的和，它在 \(z_0\) 解析，特别地连续；又 \(g(z_0) \neq 0\)，故由连续性，存在 \(z_0\) 的某个邻域 \(U\) 使得 \(g(z) \neq 0\) 在 \(U\) 中处处成立。在这个 \(U\) 里，\(f(z) = (z - z_0)^m g(z) = 0\) 只能由 \((z - z_0)^m = 0\) 即 \(z = z_0\) 给出。所以 \(z_0\) 是 \(f\) 在 \(U\) 中唯一的零点。

这个论证的第二个收获就是下一节要讲的局部因式分解。

"零点孤立性"另一个重要推论是唯一性定理（在第 11 章中我们用了它）：若 \(f, g\) 都解析、且它们的差 \(f - g\) 在某个有聚点的子集上恒为 0，则 \(f - g \equiv 0\)。证明思路：\(f - g\) 是解析的；它的零点集含有聚点；而解析函数的非平凡零点集不含聚点（每个零点都是孤立的）；所以 \(f - g\) 必定恒为 0。这件事是解析函数"刚性"的根源。

12.3 零点附近的局部因式分解

上一节论证里出现的

\[ \boxed{f(z) = (z - z_0)^m g(z), \quad g(z_0) \neq 0,} \]

本身就是一个很有用的结构定理：若 \(z_0\) 是 \(f\) 的 \(m\) 阶零点，则 \(f\) 在 \(z_0\) 附近恰好能这样分解，其中 \(g\) 在 \(z_0\) 处解析、\(g(z_0) \neq 0\)。

这与多项式 \(P(z)\) 的因式分解极其类似。一个多项式如果在 \(z_0\) 处有 \(m\) 重根，可以写成 \(P(z) = (z - z_0)^m Q(z)\)，\(Q(z_0) \neq 0\)。本节告诉你：所有解析函数的零点结构，都和多项式根的结构一样。这就是为什么把解析函数想象成"广义多项式"经常很奏效。

用这个结构定理，可以证明很多关于零点的事情。比如：

例 12.3.1　\(\sin z\) 的所有零点是 \(z = k\pi\)（\(k \in \mathbb{Z}\)）。每个都是简单零点（一阶）。原因：\(\sin'(z) = \cos z\)；在 \(z = k\pi\) 处 \(\cos(k\pi) = (-1)^k \neq 0\)。所以 \(\sin z\) 在每个零点处都满足 \(f(z_0) = 0, f'(z_0) \neq 0\)。在 \(z = k\pi\) 附近 \(\sin z = (z - k\pi) g(z)\)，\(g(k\pi) = (-1)^k \neq 0\)。

例 12.3.2　\(\sin^2 z\) 的零点在哪里？阶数？把 \(\sin^2 z = (\sin z)^2\) 看一下：\(z = k\pi\) 处 \(\sin z\) 是简单零点，所以 \(\sin^2 z = ((z - k\pi) g(z))^2 = (z - k\pi)^2 g(z)^2\)，是二阶零点。一般地：若 \(f\) 在 \(z_0\) 有 \(m\) 阶零点，则 \(f^k\) 在 \(z_0\) 有 \(km\) 阶零点。

例 12.3.3　\(\sin z \cdot (z^2 - \pi^2)\) 在 \(z = \pi\) 处是几阶零点？\(\sin z\) 在 \(z = \pi\) 是简单零点；\(z^2 - \pi^2 = (z - \pi)(z + \pi)\) 在 \(z = \pi\) 也是简单零点（\((z + \pi)\) 在 \(\pi\) 处不为零）。两者相乘：\(\sin z \cdot (z^2 - \pi^2) = (z - \pi)^2 \cdot (z + \pi) \cdot g(z)\)（其中 \(g\) 在 \(\pi\) 处不为零）——这是二阶零点。一般地：若 \(f, h\) 在 \(z_0\) 分别有 \(m, n\) 阶零点，则 \(fh\) 在 \(z_0\) 有 \(m + n\) 阶零点。

12.4 孤立奇点的定义

从这里开始进入本章的主战场：奇点。

定义　\(z_0 \in \mathbb{C}\) 称为 \(f\) 的孤立奇点 (isolated singularity)，若存在某个 \(r > 0\)，使得 \(f\) 在去心邻域 \(\{z : 0 < |z - z_0| < r\}\) 上解析，但 \(f\) 在 \(z_0\) 处不解析（要么没定义，要么有定义但不连续，要么连续但不可微）。

关键词："孤立"——意思是 \(z_0\) 周围一圈（去心邻域）都是好端端的解析点，只有 \(z_0\) 这一点出问题。所以 \(z_0\) 像一个被解析区域包围的"奇异孤岛"。

看几个例子：

例 12.4.1　\(f(z) = \dfrac{1}{z}\) 在 \(z = 0\) 处。它在 \(z = 0\) 没有定义；在 \(0 < |z| < r\)（任何 \(r > 0\)）上解析。所以 \(z = 0\) 是 \(f\) 的孤立奇点。

例 12.4.2　\(f(z) = \dfrac{\sin z}{z}\) 在 \(z = 0\) 处。同样在 \(z = 0\) 没定义（除非补 \(f(0) = 1\)），在 \(0 < |z| < r\) 上解析。所以 \(z = 0\) 是孤立奇点。但这个奇点其实可以"补掉"——下面 12.5 节会看到这就是"可去奇点"。

例 12.4.3　\(f(z) = \dfrac{1}{\sin(1/z)}\) 在 \(z = 0\) 处。\(\sin(1/z) = 0\) 等价于 \(1/z = k\pi\)，即 \(z = 1/(k\pi)\)（\(k \neq 0\)）。这一族零点积聚到 \(z = 0\)（当 \(k \to \infty\) 时 \(1/(k\pi) \to 0\)），所以 \(f\) 在 0 的任何去心邻域里都有无穷多个奇点（\(\sin\) 的零点处 \(f\) 有极点）。这种奇点不是孤立的，它叫做"非孤立的本性奇点"或"积聚奇点"，本章不予讨论。

所以"孤立奇点"是一个特定的、相对良好的奇点类型。本章只讨论这一类。

12.5 三类孤立奇点

设 \(z_0\) 是 \(f\) 的孤立奇点。根据 \(f\) 在 \(z_0\) 附近的行为，可分为三种：

(1) 可去奇点 (removable singularity)：

若 \(\lim_{z \to z_0} f(z)\) 存在且为有限值，则 \(z_0\) 称为可去奇点。这种情况下，定义 \(\tilde{f}(z_0) = \lim_{z \to z_0} f(z)\) 并保持 \(z \neq z_0\) 时 \(\tilde{f} = f\)，则 \(\tilde{f}\) 在 \(z_0\) 解析（这件事下面 12.6 节给的 Riemann 定理保证）。所以"奇点被补掉了"。

例：\(f(z) = \dfrac{\sin z}{z}\) 在 \(z = 0\) 处。\(\lim_{z \to 0} \dfrac{\sin z}{z} = 1\)（用 Taylor 展开 \(\sin z = z - z^3/6 + \cdots\)，所以 \(\dfrac{\sin z}{z} = 1 - z^2/6 + \cdots \to 1\)）。补 \(\tilde{f}(0) = 1\)，\(\tilde{f}\) 是整函数。可去。

(2) 极点 (pole)：

若 \(\lim_{z \to z_0} |f(z)| = +\infty\)（无论从哪个方向 \(z \to z_0\)），则 \(z_0\) 称为极点。等价的更具体描述：存在最小的正整数 \(k\)，使得 \((z - z_0)^k f(z)\) 在 \(z_0\) 有可去奇点（即极限存在有限非零）。这个 \(k\) 称为极点的阶。

例：\(f(z) = \dfrac{1}{z}\) 在 \(z = 0\) 处。\(|1/z| \to \infty\) 当 \(z \to 0\)。\((z - 0)^1 \cdot \dfrac{1}{z} = 1\)，是常数，可去（且不为 0）。所以 \(k = 1\)，是一阶极点（简单极点）。

例：\(f(z) = \dfrac{1}{(z - 2)^3}\) 在 \(z = 2\) 处是三阶极点。

(3) 本性奇点 (essential singularity)：

既非可去也非极点，称为本性奇点。等价地：\(\lim_{z \to z_0} f(z)\) 既不存在有限值，又不"统一地走向无穷"（不同方向逼近 \(z_0\) 给出不同极限或没有极限）。

例：\(e^{1/z}\) 在 \(z = 0\) 处。沿正实轴 \(z = x \to 0^+\)，\(e^{1/x} \to +\infty\)。沿负实轴 \(z = x \to 0^-\)，\(e^{1/x} \to 0\)。沿虚轴 \(z = iy \to 0\)，\(e^{1/(iy)} = e^{-i/y}\) 是模为 1 的复数，振荡而不收敛。三种方向给三种行为，所以非可去、非极点，是本性奇点。我们 12.9 节专门讨论它。

来看一张总结表：

类型判别（\(z \to z_0\) 时 \(f(z)\) 的行为）典型例
可去\(\lim f(z)\) 存在有限\(\dfrac{\sin z}{z}\) 在 0
极点（\(m\) 阶）\(\lim |f| = \infty\)\(\dfrac{1}{(z-z_0)^m}\)
本性极限不存在、也不到 \(\infty\)\(e^{1/z}\) 在 0

这张表是诊断奇点类型的"流程图"。下面几节我们逐个细讲。

12.6 Riemann 可去奇点定理

关于"可去奇点"，有一个极漂亮的判别准则——你甚至不需要知道 \(\lim_{z \to z_0} f(z)\) 的确切值，只需要知道 \(f\) 在去心邻域有界，就能判断它是可去的。

Riemann 可去奇点定理　设 \(z_0\) 是 \(f\) 的孤立奇点。若 \(f\) 在某个去心邻域 \(\{z : 0 < |z - z_0| < r\}\) 上有界，则 \(z_0\) 是 \(f\) 的可去奇点。

翻译：有界 ⇒ 可去。这是判别可去奇点最实用的工具——你不必费力计算极限，只需要看 \(f\) 在去心邻域里"会不会跑到无穷大"，不会就行。

这个定理的反向陈述当然也对：如果是可去的，则 \(f\) 在 \(z_0\) 附近趋向某个有限值，自然有界。所以"有界 ⟺ 可去"。

证明思路（不严格写）：在 \(z_0\) 附近做 Cauchy 积分公式 \(f(z) = \dfrac{1}{2\pi i} \oint_C \dfrac{f(w)}{w - z}\, dw\)，其中 \(C\) 是绕 \(z_0\) 的小圆。利用 \(f\) 有界，可以让积分围绕 \(z_0\) 的小圆趋于零，从而得到 \(f(z)\) 的一个解析延拓到 \(z_0\)。完整论证用的是 \(f\) 与 \((z - z_0)^2 f\) 之间的关系——后者在 \(z_0\) 上自动可解析地补成 0。

用这个定理立刻可以做：

例 12.6.1　判断 \(f(z) = \dfrac{\sin z}{z}\) 在 \(z = 0\) 是否可去。\(|f(z)| = |\sin z|/|z|\)。在 \(0 < |z| < 1\) 内，\(|\sin z| \leq C|z|\)（事实上 \(\sin z = z + O(z^3)\)，在小邻域里这个估计成立）。所以 \(|f(z)| \leq C\) 在去心邻域有界。由 Riemann 定理，可去。

例 12.6.2　判断 \(f(z) = \dfrac{1 - \cos z}{z^2}\) 在 \(z = 0\) 是否可去。展开 \(1 - \cos z = z^2/2 - z^4/24 + \cdots\)，所以 \(\dfrac{1 - \cos z}{z^2} = 1/2 - z^2/24 + \cdots\)，在去心邻域附近收敛、有界。可去，且补成 \(\tilde{f}(0) = 1/2\)。

例 12.6.3　判断 \(f(z) = z \cos(1/z)\) 在 \(z = 0\) 是否可去。\(|\cos(1/z)|\) 一般会取很大的值（沿虚轴时呈指数级增长），但请等等——其实在任何方向趋近 0 时 \(|\cos(1/z)|\) 都不一定有界（沿虚轴 \(z = iy\)，\(\cos(1/(iy)) = \cos(-i/y) = \cosh(1/y) \to \infty\)）。所以乍看不有界？但乘上 \(z\) 之后呢？需要更仔细的估计。事实上 \(z \cos(1/z)\) 在去心邻域不有界——同样沿 \(z = iy\)：\(|z \cos(1/z)| = |y| \cdot \cosh(1/y) \approx |y| \cdot \tfrac{1}{2} e^{1/|y|}\)，当 \(y \to 0\) 时分母 \(|y|\) 衰减到 0，但分子 \(e^{1/|y|}\) 暴涨；总的来说 \(|z \cos(1/z)|\) 趋于 \(\infty\)，不有界。所以 Riemann 定理不直接说它可去。事实上 \(z = 0\) 是 \(z \cos(1/z)\) 的本性奇点。

12.7 极点的等价刻画

极点是三种奇点中"最像有理函数"的一种——它的行为与 \(\dfrac{1}{(z - z_0)^m}\) 完全平行。下面给出几种等价说法：

极点的等价描述　以下命题彼此等价：

  • (a) \(z_0\) 是 \(f\) 的\(m\) 阶极点（按 12.5 节的定义）。
  • (b) 在 \(z_0\) 的某个邻域内，\(f\) 可以写成
\[ \boxed{f(z) = \dfrac{g(z)}{(z - z_0)^m},} \]

其中 \(g\) 在 \(z_0\) 处解析、\(g(z_0) \neq 0\)；并且 \(m\) 是这种表示中分母 \((z - z_0)^k\) 阶数 \(k\) 的最小可能值。

  • (c) \(\lim_{z \to z_0} (z - z_0)^m f(z)\) 存在、有限、非零。
  • (d) \(\dfrac{1}{f(z)}\) 在 \(z_0\) 有 \(m\) 阶零点（在补 \(\dfrac{1}{f}(z_0) = 0\) 之后）。

(b) 是最实用的描述：极点就是分母含 \((z - z_0)^m\) 的简单分式，且分子在 \(z_0\) 不为零。(d) 给了"零点 ⇄ 极点"的对偶关系——\(f\) 的 \(m\) 阶极点恰对应 \(\dfrac{1}{f}\) 的 \(m\) 阶零点；反过来，\(f\) 的 \(m\) 阶零点恰对应 \(\dfrac{1}{f}\) 的 \(m\) 阶极点。这个对偶非常方便：你处理过零点，就同样会处理极点。

看几个判极点阶的例题：

例 12.7.1　\(f(z) = \dfrac{1}{z^2 (z - 1)}\)。在 \(z = 0\) 处，分母含 \(z^2\)（且 \(\dfrac{1}{z - 1}\) 在 0 处解析非零），所以是二阶极点。在 \(z = 1\) 处，分母含 \((z - 1)\) 一次，\(\dfrac{1}{z^2}\) 在 1 处解析非零，所以是一阶极点（简单极点）。

例 12.7.2　\(f(z) = \dfrac{z^2 + 1}{(z - 1)^3}\)。分子在 \(z = 1\) 处的值是 \(1^2 + 1 = 2 \neq 0\)。所以分母 \((z - 1)^3\) 给出三阶极点。

例 12.7.3　\(f(z) = \dfrac{\sin z}{z^3}\) 在 \(z = 0\) 处。\(\sin z = z - z^3/6 + \cdots\)，所以 \(\dfrac{\sin z}{z^3} = \dfrac{1}{z^2} - \dfrac{1}{6} + \dfrac{z^2}{120} - \cdots\)。从这个展开看：\((z - 0)^2 \cdot f(z) = 1 - z^2/6 + \cdots \to 1 \neq 0\)。所以是二阶极点，不是三阶——因为 \(\sin z\) 在分子上"贡献了一次零点"，"抵消"了分母的一阶。

从这个例子总结：分式 \(f/g\) 在 \(z_0\) 的极点阶等于（\(g\) 在 \(z_0\) 的零点阶）\(-\)（\(f\) 在 \(z_0\) 的零点阶），前提是这个差为正；否则要看具体怎么相消（差为零给可去，差为负给零点）。

例 12.7.4　\(f(z) = \dfrac{1 - \cos z}{z^4}\) 在 \(z = 0\) 处。\(1 - \cos z\) 在 0 是二阶零点；\(z^4\) 是四阶零点。差 \(4 - 2 = 2\)，所以 \(f\) 在 0 是二阶极点。

12.8 本性奇点：Casorati-Weierstrass 与 Picard

本性奇点是三类中最"野"的一种。在它附近，函数完全没有"可控"的行为——既不靠近任何有限值，也不像极点那样统一地走向无穷。它是局部地"无所不为"。

这种"无所不为"的程度，被两条经典定理刻画。

Casorati-Weierstrass 定理　设 \(z_0\) 是 \(f\) 的本性奇点。则对 \(z_0\) 的任何去心邻域 \(U\)，\(f(U)\) 在 \(\mathbb{C}\) 中稠密。换句话说：对任何 \(w_0 \in \mathbb{C}\) 和任何 \(\varepsilon > 0\)，存在 \(z \in U\) 使得 \(|f(z) - w_0| < \varepsilon\)。

翻译：在本性奇点附近，\(f\) 取的值"挤遍"整个复平面——你随便指一个目标值，\(f\) 在 \(z_0\) 的小邻域里都能取到（任意接近的）那个值。

这已经够吓人了。但事实上还有更狠的版本：

Picard 大定理　设 \(z_0\) 是 \(f\) 的本性奇点。则对 \(z_0\) 的任何去心邻域 \(U\)，\(f(U)\) 含有 \(\mathbb{C}\) 中除最多一个值之外的所有值，并且每个被取到的值都被取到无限多次。

翻译：在本性奇点附近，\(f\) 不仅取的值稠密——它直接取遍 \(\mathbb{C}\) 中除至多一个例外的所有值，并且每个值都取无限多次。这是非常震撼的结论。它的证明远超本书范围，但你应该对它的陈述有印象。

"最多一个例外"这件事是必要的——下一节我们看的 \(e^{1/z}\) 例子里就能看到这唯一的例外值。

12.9 经典例：\(e^{1/z}\) 在 \(z = 0\) 处的本性奇点

这是本性奇点的标准教学例子，几乎每本复分析书都会讲它。让我们仔细分析。

函数 \(f(z) = e^{1/z}\) 在 \(z = 0\) 没有定义，在 \(0 < |z| < r\) 上解析（因为 \(1/z\) 在那里解析、\(e^w\) 是整函数）。所以 \(z = 0\) 是孤立奇点。

它是哪种？让我们沿不同方向逼近 0：

  • 沿正实轴 \(z = x \to 0^+\)：\(e^{1/x} \to +\infty\)。
  • 沿负实轴 \(z = x \to 0^-\)：\(e^{1/x} \to 0\)。
  • 沿虚轴 \(z = iy \to 0\)：\(e^{1/(iy)} = e^{-i/y} = \cos(1/y) - i \sin(1/y)\)，模为 1，振荡。

不同方向给的极限完全不同——所以 \(\lim_{z \to 0} f(z)\) 既不存在有限值，也不"统一地走向 \(\infty\)"。所以非可去、非极点，是本性奇点。

现在验证 Casorati-Weierstrass 与 Picard。让我们具体地问：在 \(z = 0\) 的小邻域里，\(e^{1/z}\) 能取到 \(w_0 = 5\) 这个值吗？方程 \(e^{1/z} = 5\) 等价于 \(1/z = \log 5 + 2k\pi i\)（任意整数 \(k\)），即 \(z = \dfrac{1}{\log 5 + 2k\pi i}\)。当 \(k\) 很大时，\(|z| = \dfrac{1}{\sqrt{(\log 5)^2 + (2k\pi)^2}} \to 0\)。所以确实有无穷多个 \(z\)（任意靠近 0）让 \(e^{1/z} = 5\)。验证了 Casorati-Weierstrass 与 Picard 大定理（"取每个值都无限多次"）。

同样的论证对任何非零的目标值 \(w_0 \neq 0\) 都成立——\(e^{1/z} = w_0\) 总能解出无穷多个 \(z\) 任意靠近 0。但 \(w_0 = 0\) 呢？\(e^{1/z} = 0\) 没有任何解（指数函数永远不为零）。所以 \(w_0 = 0\) 是 Picard 定理里"那个例外值"。

所以 \(e^{1/z}\) 在 0 的去心邻域里取到了 \(\mathbb{C} \setminus \{0\}\) 中的所有值，每个都无限多次——\(\{0\}\) 就是那个例外。这与 Picard 大定理完全吻合。

这种"在零点周围疯狂取遍几乎所有值"的行为，是本性奇点的基本特征。它解释了为什么本性奇点比极点"复杂得多"——极点的局部行为完全由 \(\dfrac{1}{(z - z_0)^m}\) 控制，而本性奇点的行为靠任何简单公式都说不清。

12.10 多极点：亚纯函数

到目前为止我们一直在讨论"\(f\) 在某一个孤立奇点 \(z_0\) 附近的行为"。如果 \(f\) 在一个区域 \(\Omega\) 上有多个奇点（但都是孤立的、且都是极点或可去），它有一个专门的名字：

定义　设 \(\Omega\) 是 \(\mathbb{C}\) 的开子集。\(f\) 称为 \(\Omega\) 上的亚纯函数 (meromorphic function)，若存在 \(\Omega\) 中的离散子集 \(S\)，使得 \(f\) 在 \(\Omega \setminus S\) 上解析，且 \(S\) 中每一点都是 \(f\) 的极点（或可去奇点）。

关键："离散"意味着 \(S\) 中每个点都是孤立的（不能积聚）。"亚纯函数"是"全纯函数"的轻微推广——前者允许有极点（但要求是孤立的），后者要求处处解析（无任何奇点）。亚纯函数是分析"有理函数及其推广"的天然框架。

例子：

  • 所有有理函数 \(\dfrac{P(z)}{Q(z)}\)（\(P, Q\) 是多项式，\(Q \not\equiv 0\)）都是 \(\mathbb{C}\) 上的亚纯函数。
  • \(\tan z = \dfrac{\sin z}{\cos z}\) 是 \(\mathbb{C}\) 上的亚纯函数，奇点（极点）在 \(z = \pi/2 + k\pi\)，每个都是简单极点。
  • \(\dfrac{1}{\sin z}\) 也是 \(\mathbb{C}\) 上的亚纯函数，极点在 \(z = k\pi\)，都是简单极点。
  • \(e^{1/z}\) 不是亚纯函数——它在 \(z = 0\) 处是本性奇点，不属于"极点 + 可去"的范畴。

亚纯函数的好处：它们继承了全纯函数的大部分良好性质（解、积分、零点结构等），同时允许处理"分式"。第 13、14 章的留数理论本质上就是关于亚纯函数的"积分微积分"。

例 12.10.1　\(f(z) = \dfrac{z + 1}{(z - 1)(z - 2)^2}\) 是 \(\mathbb{C}\) 上的亚纯函数。零点：\(z = -1\)（一阶）。极点：\(z = 1\)（一阶）和 \(z = 2\)（二阶）。

例 12.10.2　\(\Gamma(z)\)（伽马函数的解析延拓）是 \(\mathbb{C}\) 上的亚纯函数，极点在 \(z = 0, -1, -2, -3, \ldots\)，每个都是简单极点。这件事我们在 11.11 节已经提过。

12.11 在无穷远处的奇性

"无穷远"在复分析里也有一席之地。我们可以问：\(f\) 在 \(z = \infty\) 处的行为是怎样的？

正式定义需要一点技巧：把 \(\mathbb{C}\) 加上一个"无穷远点" \(\infty\)，得到所谓扩展复平面 \(\hat{\mathbb{C}} = \mathbb{C} \cup \{\infty\}\)（也叫"黎曼球面"，因为它拓扑上与 \(S^2\) 同胚）。要研究 \(f\) 在 \(\infty\) 处的行为，做变量替换 \(w = 1/z\)，定义

\[ \tilde{f}(w) := f(1/w), \]

这是 \(\tilde{f}\) 在 \(w\) 平面（除 \(w = 0\) 外）的函数。\(f\) 在 \(z = \infty\) 处的奇性定义为 \(\tilde{f}\) 在 \(w = 0\) 处的奇性。

看几个例子：

例 12.11.1　\(f(z) = z^2\) 在 \(z = \infty\) 处的行为。\(\tilde{f}(w) = (1/w)^2 = 1/w^2\)。在 \(w = 0\) 处，\(\tilde{f}\) 是二阶极点。所以我们说 \(f(z) = z^2\) 在 \(z = \infty\) 处是二阶极点。

一般地，多项式 \(P(z) = a_n z^n + \cdots + a_0\)（\(a_n \neq 0\)）在 \(z = \infty\) 处是 \(n\) 阶极点。这与"\(z \to \infty\) 时多项式跑到无穷大"的直觉一致。

例 12.11.2　\(f(z) = \dfrac{1}{z}\) 在 \(z = \infty\) 处。\(\tilde{f}(w) = w\)，在 \(w = 0\) 是简单零点（也即可去——补 \(\tilde{f}(0) = 0\) 后解析）。所以 \(f(z) = 1/z\) 在 \(z = \infty\) 处可去，并且补的值是 0。这与"\(z \to \infty\) 时 \(1/z \to 0\)"的直觉一致。

例 12.11.3　\(f(z) = e^z\) 在 \(z = \infty\) 处。\(\tilde{f}(w) = e^{1/w}\)，在 \(w = 0\) 是本性奇点（12.9 节）。所以 \(e^z\) 在 \(z = \infty\) 处是本性奇点。这与"\(e^z\) 在 \(z \to +\infty\) 沿正实轴时趋向 \(\infty\)，沿负实轴时趋向 0，沿虚轴振荡"的多重行为一致。

例 12.11.4　\(f(z) = \dfrac{1}{1 + z^2}\) 在 \(z = \infty\) 处。\(\tilde{f}(w) = \dfrac{1}{1 + 1/w^2} = \dfrac{w^2}{w^2 + 1}\)，在 \(w = 0\) 处 \(\tilde{f}(0) = 0/(0 + 1) = 0\)，是二阶零点（即可去并补 0）。所以 \(\dfrac{1}{1 + z^2}\) 在 \(\infty\) 处可去且趋于 0。

有了"在 \(\infty\) 处的奇性"概念，可以谈"\(\hat{\mathbb{C}}\) 上的亚纯函数"——这就是所有奇点（包括 \(\infty\)）都是极点或可去的函数。结论非常漂亮：\(\hat{\mathbb{C}}\) 上的亚纯函数恰好就是有理函数。这个事实一方面给了"有理函数"一个完整的内在描述，另一方面再次说明了"研究全平面（含无穷远）的解析行为"的几何威力。

本章我们围绕"零点"和"奇点"建立了完整的局部分类。下一章 Laurent 级数将给奇点研究一个更精细的代数工具——它把"在去心邻域里"的解析函数展成"双向幂级数"，从而能从级数表达直接看出奇点是哪一类。这种代数刻画到了 14 章变成了"留数微积分"，是复分析最具实战威力的工具之一。
