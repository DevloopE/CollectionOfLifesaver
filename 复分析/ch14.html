第 14 章 留数与留数定理

到上一章为止，你已经把全纯函数在一个孤立奇点附近的全部行为都"摊开"了：把它写成 Laurent 级数，只看主部里 \(1/(z-z_0)^k\) 的系数，就能立即判断这是可去奇点、极点还是本性奇点。这件事很漂亮，但你或许会问：把 Laurent 级数算出来到底有什么用？我们费这么大劲挑出每一个 \(a_k\)，难道只是为了"分类"奇点吗？

本章给出的回答会让你坐直一点：在这一长串 Laurent 系数 \(\ldots, a_{-3}, a_{-2}, a_{-1}, a_0, a_1, \ldots\) 里，只有一个系数特别——\(a_{-1}\)，也就是 \(1/(z-z_0)\) 这一项的系数。它特别在哪里？因为只有它在你沿着一条小圆周绕一圈做积分时留下来，其他所有项要么积出来直接是零，要么干脆变成解析函数沿闭曲线积分后为零。这个被"留下来"的系数，就叫做留数 (residue)。

把这件事推到底，就是留数定理：一条闭曲线 \(\gamma\) 围出来的区域里，被积函数 \(f\) 除有限个孤立奇点之外都解析，那么

\[ \oint_\gamma f(z)\, dz = 2\pi i \sum_k \mathrm{Res}(f, z_k). \]

这是全章的"金句"，也是整本复分析最有应用价值的公式之一——下一章我们会用它去算各种实积分，包括那些在实分析里看起来无从下手的怪东西。但先别急着应用，先把"留数"这个对象本身彻底搞懂。本章我们做四件事：

  • 定义留数（14.1），讲清它的几何意义（14.2）。
  • 陈述并证明留数定理（14.3、14.4）。
  • 把"求留数"做成一套标准化套路，覆盖一阶极点、\(m\) 阶极点、本性奇点（14.5）。
  • 用大量例题把这套机器跑熟（14.6、14.7）。

这一章的目标是：等你学完，看到任何一个孤立奇点你都能在脑子里几秒钟内给出留数；看到任何一条闭曲线积分你都能问"里头有几个奇点？每个的留数是多少？"——这就是留数法的核心心法。

14.1 留数的定义

设 \(z_0\) 是 \(f\) 的一个孤立奇点。也就是说，存在某个邻域 \(0 < |z - z_0| < R\)，\(f\) 在这个去心邻域里全纯。根据上一章我们已经知道，\(f\) 在这邻域里有 Laurent 展开

\[ f(z) = \sum_{n=-\infty}^{\infty} a_n (z - z_0)^n = \cdots + \frac{a_{-2}}{(z-z_0)^2} + \frac{a_{-1}}{z-z_0} + a_0 + a_1 (z - z_0) + \cdots.

定义如下。

定义。 \(f\) 在孤立奇点 \(z_0\) 处的留数，记作 \(\mathrm{Res}(f, z_0)\)，是 Laurent 展开里 \(1/(z-z_0)\) 的那个系数。也就是说

\[ \boxed{\mathrm{Res}(f, z_0) = a_{-1}.} \]

就是这一句话。短得让人怀疑自己漏看了什么——但确实就只是这一句。请注意：

  • 留数是一个复数，不是函数也不是曲线。
  • 它依赖于 \(f\) 和你选择的奇点 \(z_0\)，二者缺一不可。所以记号是两参的 \(\mathrm{Res}(f, z_0)\)，不是 \(\mathrm{Res}(f)\)。
  • 留数只关心 Laurent 展开里 \(1/(z-z_0)\) 那一项 的系数。其他所有项——包括 \(1/(z-z_0)^2\)、\(1/(z-z_0)^3\) 这些"看起来更糟"的高阶主部项，以及全部正幂项——一律不要。

来看一个最简单的例子，让定义不再是抽象的。考虑

\[ f(z) = \frac{1}{z}. \]

它在 \(z_0 = 0\) 处有一个一阶极点。它本身就已经是它自己的 Laurent 展开（系数 \(a_{-1} = 1\)，其他全为零）。所以

\[ \mathrm{Res}\!\left(\frac{1}{z},\, 0\right) = 1. \]

再来。\(g(z) = 1/z^2\)，\(z_0 = 0\) 处的留数是多少？这同样是它自己的 Laurent 展开：\(a_{-2} = 1\)，其他全为零，特别是 \(a_{-1} = 0\)。所以

\[ \mathrm{Res}\!\left(\frac{1}{z^2},\, 0\right) = 0. \]

注意这件事——\(1/z^2\) 在 0 处明明有一个二阶极点，但它的留数是零。这是初学者常常踩的坑："有奇点"和"留数非零"完全是两回事。下一节我们会看到这件事的几何含义。

再来一个稍微有趣一点的：

\[ h(z) = \frac{e^z}{z}. \]

把 \(e^z\) 在 \(z = 0\) 处展开成 Taylor 级数：\(e^z = 1 + z + \tfrac{z^2}{2!} + \tfrac{z^3}{3!} + \cdots\)。除以 \(z\) 得到

\[ h(z) = \frac{1}{z} + 1 + \frac{z}{2!} + \frac{z^2}{3!} + \cdots. \]

这就是它在 0 处的 Laurent 展开。\(a_{-1} = 1\)，所以 \(\mathrm{Res}(h, 0) = 1\)。看到了吗？这个例题的诀窍就一个：把分子的解析部分老老实实展开成 Taylor 级数，再除以分母，写成 Laurent 形式，最后看 \(1/z\) 的系数。

再换一个：

\[ \varphi(z) = \frac{e^z}{z^2}. \]

同样手法：\(\varphi(z) = \dfrac{1}{z^2} + \dfrac{1}{z} + \dfrac{1}{2!} + \dfrac{z}{3!} + \cdots\)。这一次 \(a_{-1} = 1\)（来自 \(e^z\) 展开里的 \(z^1\) 项除以 \(z^2\)），所以 \(\mathrm{Res}(\varphi, 0) = 1\) 仍然是 1。注意这里有个 \(1/z^2\) 项，但留数不把它算进去——留数永远只看 \(1/z\) 的那一项。

到这里你应该已经体会到：求留数本质上就是展开 Laurent 级数 + 挑出 \(1/(z-z_0)\) 的系数。逻辑上没什么深的——但实际操作中，每次都把整条 Laurent 级数展开是一件相当啰嗦的事情。所以 14.5 节会教你几个"不展开级数也能直接得到 \(a_{-1}\)" 的快捷公式。但首先，我们必须搞清楚为什么是 \(a_{-1}\) 这一项特别——而不是 \(a_{-2}\) 或 \(a_0\) 或别的什么。这就要看几何含义。

14.2 几何意义：小圆绕一圈"留下"什么

留数为什么叫"留数"（residue）？想象你有一长串 Laurent 项，让它们全都沿一条小闭曲线积一次分。绝大多数项被"吃掉"——积出来等于零；只有一项幸运地留下来。那一项的系数，就被冠以"留数"之名。

来仔细算一下。设 \(C\) 是一条以 \(z_0\) 为圆心、半径 \(\rho\) 充分小的圆周（\(\rho < R\)，让 Laurent 展开在 \(C\) 上仍然成立），方向取逆时针。我们想算

\[ \oint_C f(z)\, dz = \oint_C \sum_{n = -\infty}^{\infty} a_n (z - z_0)^n\, dz. \]

因为 Laurent 级数在 \(C\) 上一致收敛（这是它在去心圆环里成立的标配性质），可以把积分号和求和号交换：

\[ \oint_C f(z)\, dz = \sum_{n = -\infty}^{\infty} a_n \oint_C (z - z_0)^n\, dz. \]

所以问题归结为算每一个 \(\oint_C (z - z_0)^n\, dz\)。把 \(z - z_0 = \rho e^{i\theta}\) 参数化，\(\theta\) 从 0 跑到 \(2\pi\)，\(dz = i \rho e^{i\theta}\, d\theta\)：

\[ \oint_C (z - z_0)^n\, dz = \int_0^{2\pi} (\rho e^{i\theta})^n \cdot i \rho e^{i\theta}\, d\theta = i \rho^{n+1} \int_0^{2\pi} e^{i(n+1)\theta}\, d\theta. \]

来分两种情形看：

  • 若 \(n \neq -1\)：\(\int_0^{2\pi} e^{i(n+1)\theta}\, d\theta = \dfrac{e^{i(n+1)\theta}}{i(n+1)} \Big|_0^{2\pi} = 0\)，因为 \(e^{i 2\pi (n+1)} = 1\)。
  • 若 \(n = -1\)：被积函数变成 \(e^{0} = 1\)，积出来是 \(2\pi\)。前面 \(i \rho^{n+1} = i \rho^{0} = i\)，于是 \(\oint_C (z - z_0)^{-1}\, dz = i \cdot 2\pi = 2\pi i\)。

所以在那个无穷级数里，每一项都贡献零，唯独 \(n = -1\) 那一项贡献 \(2\pi i \cdot a_{-1}\)。换句话说

\[ \boxed{\oint_C f(z)\, dz = 2\pi i \cdot a_{-1} = 2\pi i \cdot \mathrm{Res}(f, z_0).} \]

这就是留数的几何意义：它是 \(f\) 沿任意一条紧紧绕住 \(z_0\) 的小圆周积分的值，除以 \(2\pi i\)。或者反过来说：把留数 \(a_{-1}\) 乘以 \(2\pi i\)，就得到积分。

这条结论很值得停下来咀嚼。你看这无穷多项 Laurent 展开，绝大多数项 \(a_n (z - z_0)^n\)（\(n \geq 0\)）是解析的，按 Cauchy-Goursat 定理沿闭曲线积分等于零，这没什么意外。但 \(a_{-2}/(z - z_0)^2\)、\(a_{-3}/(z - z_0)^3\) 这些主部项明明也有奇性，为什么它们也贡献零？因为它们都有原函数：\(\dfrac{1}{(z - z_0)^2}\) 的原函数是 \(-\dfrac{1}{z - z_0}\)，沿任何闭曲线都"回到原点"，差为零；类似地 \(1/(z - z_0)^k\) （\(k \geq 2\)）都有原函数 \(\dfrac{(z-z_0)^{1-k}}{1-k}\)，闭曲线积分为零。唯独 \(1/(z-z_0)\) 没有单值原函数——它的"原函数" \(\log(z - z_0)\) 在绕一圈后变化 \(2\pi i\)，造成那个唯一不为零的贡献。

所以"留数"这个名字是恰当的：在大屠杀般的相消之后，唯一残留下来的就是 \(a_{-1}\)。

顺便回头看 14.1 末尾那个让人困惑的例子：\(1/z^2\) 在 0 处有二阶极点，但留数为零。现在这件事完全说得通——\(1/z^2\) 有原函数 \(-1/z\)，沿任何不过原点的闭曲线积分都是零；既然积分是零，留数自然也是零。"奇性的存在"和"留下东西"是两件事。

14.2 的主结果可以独立成一句话：

\[ \boxed{\oint_C f(z)\, dz = 2\pi i \cdot \mathrm{Res}(f, z_0), \qquad C \text{ 紧绕 } z_0 \text{ 一圈，逆时针}.} \]

下一节我们把"一个奇点"换成"有限个奇点"，得到完整版的留数定理。

14.3 留数定理

考虑一条简单闭曲线 \(\gamma\)（不自交），逆时针定向。设 \(\gamma\) 围出的开区域为 \(D\)，并设 \(f\) 在 \(D\) 内除了有限多个孤立奇点 \(z_1, z_2, \ldots, z_n\) 之外全纯，且 \(f\) 在 \(\gamma\) 上也全纯（即奇点都严格落在 \(D\) 内部，不在边界上）。

留数定理。 在上述前提下，

\[ \boxed{\oint_\gamma f(z)\, dz = 2\pi i \sum_{k=1}^{n} \mathrm{Res}(f, z_k).} \]

这就是本章的主定理。把它和 14.2 比较一下：14.2 处理"一个奇点 + 紧绕的小圆"，14.3 处理"任意有限个奇点 + 任意闭曲线"。两者的关系，差不多就像"一根弹簧的力 = \(-kx\)"和"一堆耦合弹簧组成的振动系统"。前者是后者的基本单元。

来仔细品一下这条公式说了什么。它把一个路径上的积分（左边）变成了一个纯局部代数信息的求和（右边）——你只需要知道每个奇点附近的 Laurent 展开里 \(1/(z - z_k)\) 的系数，把它们加起来，乘以 \(2\pi i\)，积分就出来了。积分的"全局值"完全由"每个奇点的局部信息"决定。这是复分析最深的洞察之一，也是接下来一切应用的根。

注意几个常见的"细则"：

  1. 方向。 公式假设 \(\gamma\) 是逆时针。如果你的曲线是顺时针，整个积分变号——结论变成 \(\oint_\gamma f = -2\pi i \sum \mathrm{Res}\)。一个稳妥的做法是：永远默认逆时针，遇到顺时针就在最后乘个 \(-1\)。
  2. 只数 \(\gamma\) 内部的奇点。 若 \(f\) 还有别的奇点落在 \(\gamma\) 外部，它们一概不计入求和。求和只对"被 \(\gamma\) 围住"的奇点。
  3. 奇点必须严格在内部。 如果某个奇点恰好落在 \(\gamma\) 上，左边那个积分根本就不收敛，公式失效。下一章处理 \(\sin x / x\) 这种实积分时会遇到一个奇点正好"卡"在积分路径上的情况，那时要特殊处理（小半圆绕过去）。
  4. "简单"和"有限个"两个条件。 简单是说 \(\gamma\) 不自交，这是为了方便陈述；有限是说奇点个数是有限的，这通常自动成立（紧致区域里孤立奇点必有限）。

来一个具体例子热身。算

\[ \oint_{|z|=2} \frac{1}{z(z-1)}\, dz. \]

这是绕原点为圆心、半径 2 的圆周做的积分，逆时针。被积函数 \(f(z) = \dfrac{1}{z(z-1)}\) 的奇点在 \(z = 0\) 和 \(z = 1\) 两处。两个都在 \(|z| = 2\) 围出来的圆盘里。所以我们要算两个留数。

先算 \(\mathrm{Res}(f, 0)\)。在 0 附近，\(\dfrac{1}{z - 1} = -\dfrac{1}{1 - z} = -(1 + z + z^2 + \cdots)\)（几何级数，\(|z| < 1\) 时收敛）。所以

\[ f(z) = \frac{1}{z} \cdot \frac{1}{z - 1} = \frac{1}{z} \cdot \big[-(1 + z + z^2 + \cdots)\big] = -\frac{1}{z} - 1 - z - \cdots. \]

\(1/z\) 的系数是 \(-1\)，所以 \(\mathrm{Res}(f, 0) = -1\)。

再算 \(\mathrm{Res}(f, 1)\)。在 \(z = 1\) 附近，记 \(w = z - 1\)，\(z = 1 + w\)。则

\[ f = \frac{1}{(1 + w) \cdot w} = \frac{1}{w} \cdot \frac{1}{1 + w} = \frac{1}{w}(1 - w + w^2 - \cdots). \]

\(1/w\) 的系数是 \(1\)，所以 \(\mathrm{Res}(f, 1) = 1\)。

留数定理给出

\[ \oint_{|z|=2} \frac{dz}{z(z-1)} = 2\pi i (-1 + 1) = 0. \]

得零！是不是很神奇？你也可以用部分分式来看：\(\dfrac{1}{z(z-1)} = -\dfrac{1}{z} + \dfrac{1}{z-1}\)。两条分别是 \(\oint \dfrac{1}{z} dz = 2\pi i\) 和 \(\oint \dfrac{1}{z-1} dz = 2\pi i\)（后者也是绕住了 1），二者相减得 0，与留数定理一致。这种交叉验证非常重要——留数法的力量来源恰恰是它和"老办法"得到同样的答案，但程序更短。

14.4 推导：把每个奇点割掉一个小圆

留数定理的证明就一个想法："用 Cauchy-Goursat 把每个奇点单独拎出来，再用 14.2 的结论。"我们一步步走。

设 \(\gamma\) 围出区域 \(D\)，奇点 \(z_1, \ldots, z_n\) 都在 \(D\) 内。在每个 \(z_k\) 周围，画一条半径为 \(\rho_k\)（充分小，使所有小圆互不相交，且都完全位于 \(D\) 内）的小圆 \(C_k\)，方向取顺时针。这一段方向选反了不是为了为难自己，而是为了下一步的几何："沿 \(\gamma\) 逆时针走 + 沿每个 \(C_k\) 顺时针走"恰好构成挖了 \(n\) 个洞的区域 \(\widetilde{D} = D \setminus \bigcup_k \{|z - z_k| \leq \rho_k\}\) 的整条边界，且边界方向都"把内部留在左手边"。

关键观察：\(f\) 在 \(\widetilde{D}\) 上处处全纯。原因——\(f\) 在 \(D\) 上除奇点外全纯，而我们恰好把奇点附近的小圆盘抠掉了。所以在挖了洞的 \(\widetilde{D}\) 上，被积函数没有任何奇点。

对全纯函数，沿区域整条边界积分等于零（这是 Cauchy-Goursat 定理的"挖洞"版本，证明思路同样是把区域剖成小三角形然后逐个套基本 Cauchy-Goursat）。所以

\[ \oint_\gamma f(z)\, dz + \sum_{k=1}^{n} \oint_{C_k} f(z)\, dz = 0, \]

其中所有 \(C_k\) 是顺时针方向。把每个 \(\oint_{C_k}\) 翻一个方向变成逆时针，要加一个负号；记 \(C_k^{+}\) 为逆时针小圆，则 \(\oint_{C_k} = -\oint_{C_k^{+}}\)。代入：

\[ \oint_\gamma f(z)\, dz - \sum_{k=1}^{n} \oint_{C_k^{+}} f(z)\, dz = 0, \]

整理：

\[ \oint_\gamma f(z)\, dz = \sum_{k=1}^{n} \oint_{C_k^{+}} f(z)\, dz. \]

现在每个右边的小圆积分都正好是 14.2 处理的情形——一个孤立奇点、一条紧绕的逆时针小圆。所以

\[ \oint_{C_k^{+}} f(z)\, dz = 2\pi i \cdot \mathrm{Res}(f, z_k). \]

把 \(n\) 个加起来：

\[ \oint_\gamma f(z)\, dz = 2\pi i \sum_{k=1}^{n} \mathrm{Res}(f, z_k). \qquad \square \]

就这么一回事。证明的全部技术内容只有一招："在每个奇点附近抠一个小洞，让被积函数在剩下的区域里全纯，套 Cauchy-Goursat。"这是在前几章 Cauchy 积分公式里反复见过的"挖洞-缩圈"思路的一个直接应用。

注意一件常被忽略的事：定理与小圆的具体半径 \(\rho_k\) 无关。你随便挑半径，只要它们足够小让所有小圆不相交且都在 \(D\) 内，结果都一样。这跟 \(\rho_k\) 充当的"抠洞工具"角色是一致的——洞的大小与最终积分值无关，因为变化的部分（环形区域内的积分）本身就是零。

14.5 求留数的实用技巧

到此我们已经把"留数定理"安装好。它的左边——一个闭曲线积分——一旦你把右边的所有留数算出来就直接出结果。所以问题完全归结为：给定 \(f\) 和它的孤立奇点 \(z_0\)，怎么快速求出 \(\mathrm{Res}(f, z_0)\)？

原始定义"展开 Laurent 取 \(a_{-1}\)" 永远管用，但每次都展开一长串级数实在太啰嗦。本节给出三类常用的"极点情形快捷公式"，再加一类"本性奇点只能展级数"的覆盖。学会分类对症下药，95% 的实际问题都能在一行内出结果。

14.5.1 一阶极点的留数

设 \(z_0\) 是 \(f\) 的一阶极点。这意味着 \(f\) 在 \(z_0\) 附近的 Laurent 展开里只有一项主部，形如

\[ f(z) = \frac{a_{-1}}{z - z_0} + a_0 + a_1 (z - z_0) + a_2 (z - z_0)^2 + \cdots. \]

把两边乘以 \((z - z_0)\)：

\[ (z - z_0) f(z) = a_{-1} + a_0 (z - z_0) + a_1 (z - z_0)^2 + \cdots. \]

右边在 \(z \to z_0\) 时趋于 \(a_{-1}\)，因为后面所有项都至少含一个 \((z - z_0)\) 因子。所以

\[ \boxed{\mathrm{Res}(f, z_0) = \lim_{z \to z_0} (z - z_0) f(z), \qquad \text{若 } z_0 \text{ 是一阶极点}.} \]

这是所有留数公式中最常用的一个。它把"求留数"转化为"求一个简单极限"。绝大多数实际问题里你会看到的就是一阶极点，这条公式直接出结果。

例。 \(f(z) = \dfrac{z + 1}{z - 2}\)，求 \(\mathrm{Res}(f, 2)\)。直接套公式：

\[ \mathrm{Res}(f, 2) = \lim_{z \to 2} (z - 2) \cdot \frac{z + 1}{z - 2} = \lim_{z \to 2}(z + 1) = 3. \]

\((z - 2)\) 乘上去恰好把分母约掉，剩下分子在 \(z = 2\) 处的值。

例。 \(f(z) = \dfrac{1}{(z - 1)(z - 2)(z - 3)}\)，求三个留数。三处都是一阶极点：

\[ \mathrm{Res}(f, 1) = \lim_{z \to 1} \frac{1}{(z - 2)(z - 3)} = \frac{1}{(-1)(-2)} = \frac{1}{2}. \] \[ \mathrm{Res}(f, 2) = \lim_{z \to 2} \frac{1}{(z - 1)(z - 3)} = \frac{1}{(1)(-1)} = -1. \] \[ \mathrm{Res}(f, 3) = \lim_{z \to 3} \frac{1}{(z - 1)(z - 2)} = \frac{1}{(2)(1)} = \frac{1}{2}. \]

三个留数加起来是 \(\tfrac{1}{2} - 1 + \tfrac{1}{2} = 0\)。这件事不是巧合——下一节会看到，对一切"分母比分子高至少二阶"的有理函数，所有有限留数之和都是零。这是一个能用来交叉验证的好习惯。

14.5.2 一阶极点（\(f = g/h\) 形式）的快捷公式

很多时候 \(f\) 自然地写成两个解析函数的商 \(f = g/h\)，其中 \(g(z_0) \neq 0\) 而 \(h(z_0) = 0\) 但 \(h'(z_0) \neq 0\)（即 \(h\) 在 \(z_0\) 处有简单零点）。这样的 \(f\) 在 \(z_0\) 处自动是一阶极点。我们可以省掉 14.5.1 那个极限直接出结果。

用 14.5.1：

\[ \mathrm{Res}(f, z_0) = \lim_{z \to z_0} (z - z_0) \frac{g(z)}{h(z)} = g(z_0) \cdot \lim_{z \to z_0} \frac{z - z_0}{h(z)}. \]

而 \(\lim_{z \to z_0} \dfrac{h(z)}{z - z_0} = h'(z_0)\)（导数定义，因为 \(h(z_0) = 0\)）。所以

\[ \boxed{\mathrm{Res}\!\left(\frac{g}{h},\, z_0\right) = \frac{g(z_0)}{h'(z_0)}, \qquad h(z_0) = 0,\ h'(z_0) \neq 0.} \]

这条公式比 14.5.1 在很多问题里更顺手——你不用做"乘 \((z - z_0)\) 再约掉"的小动作，直接把分母求一次导，代入 \(z_0\) 就完事。

例。 求 \(f(z) = \dfrac{e^z}{z^2 + 1}\) 在 \(z = i\) 处的留数。分母 \(h(z) = z^2 + 1\) 在 \(z = i\) 处有简单零点，\(h'(z) = 2z\)，\(h'(i) = 2i\)。分子 \(g(z) = e^z\)，\(g(i) = e^i = \cos 1 + i \sin 1\)。所以

\[ \mathrm{Res}(f, i) = \frac{e^i}{2i} = \frac{\cos 1 + i \sin 1}{2i} = \frac{\sin 1 - i \cos 1}{2} \cdot (-1)\,? \]

等等，让我把代数算清楚：\(\dfrac{e^i}{2i} = \dfrac{e^i}{2i} \cdot \dfrac{-i}{-i} = \dfrac{-i e^i}{2}\)。所以

\[ \mathrm{Res}(f, i) = -\frac{i e^i}{2} = -\frac{i (\cos 1 + i \sin 1)}{2} = \frac{\sin 1 - i \cos 1}{2}. \]

整个计算两行就完。如果用 14.5.1 公式，你得先把 \((z - i)\) 乘进去，写成 \(\dfrac{(z - i) e^z}{(z - i)(z + i)} = \dfrac{e^z}{z + i}\)，再代 \(z = i\)，得 \(\dfrac{e^i}{2i}\)——结果当然一样，但中间多了一个步骤。一旦养成"分母有简单零点就直接 \(g(z_0)/h'(z_0)\)" 的反射，速度会快很多。

例。 求 \(f(z) = \dfrac{1}{\sin z}\) 在 \(z = 0\) 处的留数。\(\sin z\) 在 0 处有简单零点，\((\sin z)' = \cos z\)，\(\cos 0 = 1\)。所以

\[ \mathrm{Res}(f, 0) = \frac{1}{\cos 0} = 1. \]

更一般地，\(\dfrac{1}{\sin z}\) 在所有整数倍 \(\pi\) 处都有一阶极点，留数 \(\dfrac{1}{\cos n\pi} = (-1)^n\)。

14.5.3 \(m\) 阶极点的留数

设 \(z_0\) 是 \(f\) 的\(m\) 阶极点。这意味着 Laurent 展开里主部从 \((z - z_0)^{-m}\) 起步：

\[ f(z) = \frac{a_{-m}}{(z - z_0)^m} + \frac{a_{-m+1}}{(z - z_0)^{m-1}} + \cdots + \frac{a_{-1}}{z - z_0} + a_0 + a_1 (z - z_0) + \cdots. \]

\(a_{-m} \neq 0\)。把两边乘以 \((z - z_0)^m\)：

\[ (z - z_0)^m f(z) = a_{-m} + a_{-m+1}(z - z_0) + \cdots + a_{-1}(z - z_0)^{m-1} + a_0 (z - z_0)^m + \cdots. \]

这是一个解析函数（在 \(z_0\) 附近）。我们要的 \(a_{-1}\) 就是它的 \((m-1)\) 次方系数。怎么取？对一个解析函数，第 \(k\) 次方系数 = \(\dfrac{f^{(k)}(z_0)}{k!}\)（Taylor 展开的标准事实）。所以

\[ a_{-1} = \frac{1}{(m-1)!} \cdot \lim_{z \to z_0} \frac{d^{m-1}}{dz^{m-1}} \big[ (z - z_0)^m f(z) \big]. \]

装在框里：

\[ \boxed{\mathrm{Res}(f, z_0) = \frac{1}{(m-1)!}\, \lim_{z \to z_0} \frac{d^{m-1}}{dz^{m-1}} \big[ (z - z_0)^m f(z) \big], \qquad z_0 \text{ 是 } m \text{ 阶极点}.} \]

当 \(m = 1\) 时，\(d^0/dz^0\) 是恒等映射，\(0! = 1\)，公式退化成 14.5.1。所以这条是 14.5.1 的推广，不是替代——仍然首选 14.5.1，仅在阶 \(\geq 2\) 时才用本条。

例。 求 \(f(z) = \dfrac{1}{z^2 (z - 1)}\) 在 \(z = 0\) 处的留数。0 处是二阶极点（\(m = 2\)）。套公式：

\[ \mathrm{Res}(f, 0) = \frac{1}{1!} \lim_{z \to 0} \frac{d}{dz} \!\left[ z^2 \cdot \frac{1}{z^2 (z - 1)} \right] = \lim_{z \to 0} \frac{d}{dz} \!\left[\frac{1}{z - 1}\right] = \lim_{z \to 0} \!\left[-\frac{1}{(z-1)^2}\right] = -1. \]

核对一下：还有奇点 \(z = 1\)，是一阶极点。\(\mathrm{Res}(f, 1) = \lim_{z \to 1} \dfrac{(z - 1)}{z^2 (z - 1)} = \dfrac{1}{1} = 1\)。两个留数加起来是 \(-1 + 1 = 0\)，符合"分母比分子高 \(\geq 2\) 阶 ⇒ 留数和为 0" 的预期。

例。 求 \(f(z) = \dfrac{e^z}{z^3}\) 在 \(z = 0\) 处的留数。\(m = 3\)：

\[ \mathrm{Res}(f, 0) = \frac{1}{2!} \lim_{z \to 0} \frac{d^2}{dz^2} \!\left[z^3 \cdot \frac{e^z}{z^3}\right] = \frac{1}{2} \lim_{z \to 0} \frac{d^2}{dz^2} \big[e^z\big] = \frac{1}{2} \cdot e^0 = \frac{1}{2}. \]

验证：用直接 Laurent 展开，\(f = \dfrac{1}{z^3}(1 + z + \dfrac{z^2}{2} + \dfrac{z^3}{6} + \cdots) = \dfrac{1}{z^3} + \dfrac{1}{z^2} + \dfrac{1}{2z} + \dfrac{1}{6} + \cdots\)。\(1/z\) 的系数是 \(1/2\)，吻合。

例。 求 \(f(z) = \dfrac{\sin z}{z^4}\) 在 \(z = 0\) 处的留数。看上去像四阶极点，套 \(m = 4\) 公式应该没问题——但更聪明的做法是直接展开 \(\sin z\)。

\[ \sin z = z - \frac{z^3}{3!} + \frac{z^5}{5!} - \cdots. \] \[ f(z) = \frac{\sin z}{z^4} = \frac{1}{z^3} - \frac{1}{3!} \cdot \frac{1}{z} + \frac{z}{5!} - \cdots. \]

\(1/z\) 的系数是 \(-\dfrac{1}{3!} = -\dfrac{1}{6}\)。所以 \(\mathrm{Res}(f, 0) = -1/6\)。注意：用 \(m = 4\) 公式要算三次导数（\(d^3/dz^3 \sin z = -\cos z\)，代 0 得 \(-1\)，再除以 \(3! = 6\) 得 \(-1/6\)），结果一致，但展级数法显然更轻松。这件事告诉你：\(m\) 阶极点公式不是唯一选择，遇到分子是熟悉函数（\(\sin, \cos, e^z\)）就直接展级数，往往更快。

14.5.4 本性奇点：直接展开

对本性奇点（如 \(e^{1/z}\) 在 0 处、\(\sin(1/z)\) 在 0 处），没有套用极点公式的捷径——根本不存在 "\(m\) 阶"，主部有无穷多项。唯一的办法就是把 Laurent 级数显式展开，挑出 \(1/z\) 的系数。

例。 求 \(f(z) = e^{1/z}\) 在 \(z = 0\) 处的留数。把 \(e^w\) 在 \(w = 1/z\) 处展开：

\[ e^{1/z} = 1 + \frac{1}{z} + \frac{1}{2! z^2} + \frac{1}{3! z^3} + \cdots. \]

\(1/z\) 的系数显然是 1。所以 \(\mathrm{Res}(e^{1/z}, 0) = 1\)。

例。 求 \(f(z) = z^2 \sin(1/z)\) 在 \(z = 0\) 处的留数。展开：

\[ \sin(1/z) = \frac{1}{z} - \frac{1}{3! z^3} + \frac{1}{5! z^5} - \cdots. \] \[ f(z) = z^2 \sin(1/z) = z - \frac{1}{3! z} + \frac{1}{5! z^3} - \cdots. \]

\(1/z\) 的系数是 \(-\dfrac{1}{3!} = -\dfrac{1}{6}\)。所以 \(\mathrm{Res}(f, 0) = -1/6\)。

对本性奇点的留数题，关键就是把内部那个 \(1/z\) 替换出来后老老实实展开。展几项停下来？只要展到 \(1/z\) 这一项被算清楚就够。多展不影响，但浪费力气。

14.5.5 一张速查表

把上面四个情形列一张表，方便你做题时快速对症下药：

奇点类型判别留数公式
可去奇点所有 \(a_{-k} = 0\)\(\mathrm{Res} = 0\)
一阶极点\((z-z_0) f\) 在 \(z_0\) 处有非零有限极限\(\lim_{z \to z_0}(z - z_0) f(z)\)
一阶极点（\(f = g/h\)）\(g(z_0) \neq 0\)，\(h(z_0) = 0\)，\(h'(z_0) \neq 0\)\(g(z_0)/h'(z_0)\)
\(m\) 阶极点\((z-z_0)^m f\) 在 \(z_0\) 处有非零有限极限\(\dfrac{1}{(m-1)!} \lim \dfrac{d^{m-1}}{dz^{m-1}} [(z-z_0)^m f]\)
本性奇点主部有无穷多项展开 Laurent 取 \(a_{-1}\)

实际操作的优先级是这样：先看分母有没有简单零点——有就直接用 \(g/h'\)；没有就看是几阶极点——一阶用 \((z - z_0) f\) 极限，更高阶才动用 \(m\) 阶公式；如果分子是 \(e^z, \sin z, \cos z\) 这类有现成 Taylor 展开的，常常直接展级数比套公式更快；本性奇点必须老老实实展级数。

14.6 例题：把求留数练熟

纸上看公式不算会用，必须动手做。本节给出几道典型例题，逐一展示应该走哪条路径。

例 1：有理函数三个极点

求 \(f(z) = \dfrac{z^3}{(z - 1)(z - 2)(z - 3)}\) 在三个极点处的留数。

三处都是一阶极点（每个 \((z - k)\) 都是简单因子）。直接套 14.5.1：

\[ \mathrm{Res}(f, 1) = \lim_{z \to 1} \frac{z^3}{(z - 2)(z - 3)} = \frac{1}{(-1)(-2)} = \frac{1}{2}. \] \[ \mathrm{Res}(f, 2) = \lim_{z \to 2} \frac{z^3}{(z - 1)(z - 3)} = \frac{8}{(1)(-1)} = -8. \] \[ \mathrm{Res}(f, 3) = \lim_{z \to 3} \frac{z^3}{(z - 1)(z - 2)} = \frac{27}{(2)(1)} = \frac{27}{2}. \]

三个加起来：\(\tfrac{1}{2} - 8 + \tfrac{27}{2} = 14 - 8 = 6\)。不为零。为什么？因为这次分子 \(z^3\) 与分母 \((z-1)(z-2)(z-3)\) 同阶，"分母比分子高 \(\geq 2\)" 那条规则就不成立了——它需要至少高 2 阶才能保证留数和为零。"分子分母同阶"的情形里有一个"无穷远处的留数"贡献，本书暂时不展开讲。

例 2：分母有简单零点的快捷公式

求 \(f(z) = \dfrac{z}{e^z - 1}\) 在 \(z = 0\) 处的留数。

看似有点棘手——分子 \(z\) 在 0 处也是 0。先验证一下奇点类型：\(e^z - 1 = z + z^2/2 + z^3/6 + \cdots\) 在 0 处有简单零点，于是 \(\dfrac{z}{e^z - 1}\) 在 0 处有可去奇点，留数是 0。

验证：\(\dfrac{z}{e^z - 1} = \dfrac{1}{1 + z/2 + z^2/6 + \cdots} = 1 - \dfrac{z}{2} + \cdots\)（直接做长除法）。这是个解析函数（Bernoulli 数生成函数），主部全为零，所以 \(\mathrm{Res}(f, 0) = 0\)。

此例的教训：奇点不一定是极点，先判类型，再选公式。

例 3：\(g/h'\) 公式的优雅应用

求 \(f(z) = \dfrac{e^{az}}{1 + z^2}\) 在 \(z = i\) 和 \(z = -i\) 两个极点的留数（\(a\) 为实参数）。

分母 \(h(z) = 1 + z^2\)，\(h'(z) = 2z\)。两个零点 \(z = \pm i\) 都是简单零点，分子 \(g(z) = e^{az}\) 在两点处都不为零。所以

\[ \mathrm{Res}(f, i) = \frac{e^{a i}}{2i}, \qquad \mathrm{Res}(f, -i) = \frac{e^{-a i}}{-2i} = -\frac{e^{-ai}}{2i}. \]

注意 \(z = -i\) 的留数有那个负号——是因为 \(h'(-i) = -2i\)。这是一个常见的小失误：一不小心忘了 \(h'\) 在不同零点处取值不同，就会算错。

这个例子里我们故意保留 \(a\) 是为了方便下一章——下一章会用这两个留数算 \(\int_{-\infty}^{\infty} \dfrac{\cos(ax)}{1 + x^2}\, dx\)。

例 4：二阶极点

求 \(f(z) = \dfrac{e^z}{(z - 1)^2}\) 在 \(z = 1\) 处的留数。

二阶极点（\(m = 2\)）。套 14.5.3 公式：

\[ \mathrm{Res}(f, 1) = \frac{1}{1!} \lim_{z \to 1} \frac{d}{dz} \!\left[(z - 1)^2 \cdot \frac{e^z}{(z - 1)^2}\right] = \lim_{z \to 1} \frac{d}{dz} \big[e^z\big] = e^1 = e. \]

太干净了——乘 \((z - 1)^2\) 把分母彻底清掉，剩下 \(e^z\)，求一次导还是 \(e^z\)，代 \(z = 1\) 得 \(e\)。

例 5：复杂一点的二阶极点

求 \(f(z) = \dfrac{1}{(z^2 + 1)^2}\) 在 \(z = i\) 处的留数。

分母 \((z^2 + 1)^2 = (z - i)^2 (z + i)^2\)，所以 \(z = i\) 是二阶极点。套公式：

\[ \mathrm{Res}(f, i) = \lim_{z \to i} \frac{d}{dz} \!\left[(z - i)^2 \cdot \frac{1}{(z - i)^2 (z + i)^2}\right] = \lim_{z \to i} \frac{d}{dz} \!\left[\frac{1}{(z + i)^2}\right]. \]

求一次导：\(\dfrac{d}{dz}(z + i)^{-2} = -2 (z + i)^{-3}\)。代 \(z = i\)：\(-2 (2i)^{-3} = -\dfrac{2}{(2i)^3} = -\dfrac{2}{-8i} = \dfrac{1}{4i} = -\dfrac{i}{4}\)。所以

\[ \mathrm{Res}(f, i) = -\frac{i}{4}. \]

同理（或者注意 \(f\) 是实有理函数，留数关于实轴共轭对称）：\(\mathrm{Res}(f, -i) = \dfrac{i}{4}\)。两者之和是 0，下一章会看到这恰好对应于 \(\int_{-\infty}^{\infty} \dfrac{dx}{(x^2 + 1)^2}\) 的某个分项是纯实数。

例 6：本性奇点，老实展级数

求 \(f(z) = e^{1/z^2}\) 在 \(z = 0\) 处的留数。

展开：

\[ e^{1/z^2} = 1 + \frac{1}{z^2} + \frac{1}{2! z^4} + \frac{1}{3! z^6} + \cdots. \]

这一长串里只有偶数次方负幂。\(1/z\) 这一项的系数是 0（甚至根本不出现）。所以 \(\mathrm{Res}(e^{1/z^2}, 0) = 0\)。

注意——这是一个本性奇点（无穷多负幂项），但留数仍然为 0。本性奇点 ≠ 留数非零。再次提醒：留数只看 \(1/z\) 那一项。

例 7：分子有零点，看清是哪类奇点

求 \(f(z) = \dfrac{1 - \cos z}{z^4}\) 在 \(z = 0\) 处的留数。

判类型：\(1 - \cos z = \dfrac{z^2}{2!} - \dfrac{z^4}{4!} + \cdots\)，所以分子在 0 处是二阶零点；分母是四阶零点。商是二阶极点。套 14.5.3 \(m = 2\)：

但更直接：直接展开。

\[ \frac{1 - \cos z}{z^4} = \frac{1}{z^4}\!\left(\frac{z^2}{2} - \frac{z^4}{24} + \frac{z^6}{720} - \cdots\right) = \frac{1}{2 z^2} - \frac{1}{24} + \frac{z^2}{720} - \cdots. \]

\(1/z\) 的系数 = 0。所以 \(\mathrm{Res}(f, 0) = 0\)。

反思：奇点的阶数由"分母零点阶 - 分子零点阶"决定。分母 4 阶 - 分子 2 阶 = 2 阶极点。但留数由展开后看 \(1/z\) 的系数决定——这次是 0。两件事不要混。

例 8：综合，三个奇点都不一样

求 \(f(z) = \dfrac{\sin z}{z^2 (z - \pi)}\) 的全部留数。

奇点：\(z = 0\) 和 \(z = \pi\)。先看 \(z = 0\)：分子 \(\sin z\) 在 0 处是简单零点（\(\sin z \sim z\)），分母 \(z^2 (z - \pi)\) 在 0 处是二阶零点（\(z^2\) 因子）。所以 \(z = 0\) 处实际是一阶极点。

用 14.5.1（也可以用 \(m = 2\) 公式作为练习，但一阶极点解释更直接）：

\[ \mathrm{Res}(f, 0) = \lim_{z \to 0} z \cdot \frac{\sin z}{z^2 (z - \pi)} = \lim_{z \to 0} \frac{\sin z}{z (z - \pi)} = \lim_{z \to 0} \frac{\sin z}{z} \cdot \frac{1}{z - \pi} = 1 \cdot \frac{1}{-\pi} = -\frac{1}{\pi}. \]

用了 \(\sin z / z \to 1\)。再看 \(z = \pi\)：分子 \(\sin \pi = 0\)，分母 \(\pi^2 \cdot 0 = 0\)。但 \(\sin z\) 在 \(\pi\) 处也是简单零点（\(\sin' z = \cos z\)，\(\cos \pi = -1 \neq 0\)），\((z - \pi)\) 是简单零点。所以分子分母在 \(\pi\) 处都各有一个简单零点，二者相约后 \(z = \pi\) 是可去奇点，留数为 0。

验证：\(\lim_{z \to \pi} \dfrac{\sin z}{z - \pi} = \cos \pi = -1\)（\(L'\)Hôpital 或等价 Taylor 展开）。所以 \(f(z) = \dfrac{\sin z}{z^2 (z - \pi)} \to \dfrac{-1}{\pi^2}\) 在 \(z \to \pi\) 时有有限极限，是可去奇点。

结论：\(\mathrm{Res}(f, 0) = -1/\pi\)，\(\mathrm{Res}(f, \pi) = 0\)。

这道例题展示了一个常见陷阱：分母有零并不代表那里就是极点，要看分子是否同时也有零点，如果有要做约简。这件事本质上是在判极点的真实阶数，搞错了套公式就一定错。

14.7 用留数定理算闭曲线积分

把 14.5 学到的技巧装进 14.3 的公式里，立刻得到一种算闭曲线复积分的标准化方法：

  1. 把曲线 \(\gamma\) 围出的区域画清楚。
  2. 找出 \(f\) 在该区域内部的所有奇点。注意"内部"不含 "在 \(\gamma\) 上"。
  3. 对每个奇点，判类型，挑公式，算留数。
  4. 把所有留数相加，乘以 \(2\pi i\)。如果 \(\gamma\) 是顺时针，再乘 \(-1\)。

这一节我们做几道题把这套程序跑熟。

例 1：单极点

算 \(\displaystyle\oint_{|z| = 2} \dfrac{e^z}{z - 1}\, dz\)。

奇点 \(z = 1\)，落在 \(|z| < 2\) 内。一阶极点，留数 \(= e^1 = e\)。所以

\[ \oint_{|z|=2} \frac{e^z}{z - 1}\, dz = 2\pi i \cdot e = 2\pi i\, e. \]

注意：这就是 Cauchy 积分公式 \(\oint \dfrac{f(z)}{z - z_0} dz = 2\pi i f(z_0)\)（\(f(z) = e^z\)，\(z_0 = 1\)）的特殊情形！留数定理是 Cauchy 积分公式的真正推广——前者把"被 \((z - z_0)\) 一次方除"扩展到了"任意阶极点"乃至"任意有限多个奇点"。

例 2：两个极点都进圈

算 \(\displaystyle\oint_{|z| = 3} \dfrac{2z + 1}{z^2 - 4}\, dz\)。

分母 \(z^2 - 4 = (z - 2)(z + 2)\)，奇点 \(z = \pm 2\) 都在 \(|z| < 3\) 里。两个一阶极点。用 \(g/h'\) 公式：分子 \(g(z) = 2z + 1\)，分母 \(h(z) = z^2 - 4\)，\(h'(z) = 2z\)。

\[ \mathrm{Res}(f, 2) = \frac{g(2)}{h'(2)} = \frac{5}{4}, \qquad \mathrm{Res}(f, -2) = \frac{g(-2)}{h'(-2)} = \frac{-3}{-4} = \frac{3}{4}. \]

之和 \(= 5/4 + 3/4 = 2\)。所以

\[ \oint_{|z|=3} \frac{2z + 1}{z^2 - 4}\, dz = 2\pi i \cdot 2 = 4\pi i. \]

例 3：只有一个极点进圈

算 \(\displaystyle\oint_{|z - 1| = 1.5} \dfrac{2z + 1}{z^2 - 4}\, dz\)。

注意这次曲线是以 1 为圆心、半径 1.5 的圆。极点 \(z = 2\) 满足 \(|2 - 1| = 1 < 1.5\)，进圈。极点 \(z = -2\) 满足 \(|-2 - 1| = 3 > 1.5\)，没进圈。所以只算 \(z = 2\) 的留数：

\[ \oint_{|z - 1| = 1.5} \frac{2z + 1}{z^2 - 4}\, dz = 2\pi i \cdot \frac{5}{4} = \frac{5\pi i}{2}. \]

这个例子的关键是"曲线围出的区域不一定是以原点为心"，要逐个奇点判断在不在内。常见错误：套公式时把所有奇点的留数都加上去——错了。只加圈在里面的。

例 4：高阶极点

算 \(\displaystyle\oint_{|z| = 1} \dfrac{\sin z}{z^4}\, dz\)。

奇点 \(z = 0\) 在内。前面例 7 中我们算过 \(\mathrm{Res}(f, 0) = -1/6\)（用直接展级数）。所以

\[ \oint_{|z|=1} \frac{\sin z}{z^4}\, dz = 2\pi i \cdot \left(-\frac{1}{6}\right) = -\frac{\pi i}{3}. \]

这道题如果你坚持用 14.5.3 的 \(m = 4\) 公式，要算三阶导数；展级数显然舒服得多。对所有"分子是熟悉超越函数 / 分母是 \(z^k\)"的情形，先展级数，再求和系数。

例 5：本性奇点

算 \(\displaystyle\oint_{|z| = 1} e^{1/z}\, dz\)。

0 处本性奇点，留数 \(= 1\)（前面例题 14.5.4 算过）。所以

\[ \oint_{|z|=1} e^{1/z}\, dz = 2\pi i. \]

这条结果看起来不可思议：\(e^{1/z}\) 在原点的"行为"狂乱无比（取遍每个有限复数无穷多次），可它沿单位圆一圈积分却是 \(2\pi i\) 这么干净的数字。这就是留数把"局部疯狂"压缩成"一个数字"的威力。

例 6：方向反过来

算 \(\displaystyle\oint_{C} \dfrac{1}{z(z-1)}\, dz\)，其中 \(C\) 是 \(|z| = 2\) 但定向是顺时针。

14.3 的例题里我们用逆时针算过此积分，得到 0。顺时针就要乘 \(-1\)：仍然得 \(0 \cdot (-1) = 0\)。这次的"碰巧 0"让方向变没变都看不出区别——但如果留数和不为 0，方向就关键了。所以做题时务必先确认 \(\gamma\) 的方向。

例 7：组合多个奇点和阶数

算 \(\displaystyle\oint_{|z| = 4} \dfrac{1}{z^2 (z - 2)(z + 1)}\, dz\)。

奇点 \(z = 0, 2, -1\)，都在 \(|z| < 4\) 里。

\(z = -1\) 一阶极点：

\[ \mathrm{Res}(f, -1) = \lim_{z \to -1} \frac{1}{z^2 (z - 2)} = \frac{1}{1 \cdot (-3)} = -\frac{1}{3}. \]

\(z = 2\) 一阶极点：

\[ \mathrm{Res}(f, 2) = \lim_{z \to 2} \frac{1}{z^2 (z + 1)} = \frac{1}{4 \cdot 3} = \frac{1}{12}. \]

\(z = 0\) 二阶极点。\((z - 0)^2 f = \dfrac{1}{(z - 2)(z + 1)}\)。求一次导：

\[ \frac{d}{dz}\!\left[\frac{1}{(z-2)(z+1)}\right] = \frac{d}{dz}\!\left[\frac{1}{z^2 - z - 2}\right] = -\frac{2z - 1}{(z^2 - z - 2)^2}. \]

代 \(z = 0\)：\(-\dfrac{-1}{(-2)^2} = \dfrac{1}{4}\)。所以 \(\mathrm{Res}(f, 0) = \dfrac{1}{4}\)。

三个之和：\(-\dfrac{1}{3} + \dfrac{1}{12} + \dfrac{1}{4} = -\dfrac{4}{12} + \dfrac{1}{12} + \dfrac{3}{12} = 0\)。所以积分是 0。

再次验证那条规则：分子常数（0 阶）、分母 4 阶，差 4 \(\geq 2\)，留数和应为 0。✓

例 8：含有 \(\sin\) 和 \(\cos\) 的混合

算 \(\displaystyle\oint_{|z| = \pi/2} \tan z\, dz\)。

\(\tan z = \dfrac{\sin z}{\cos z}\)，奇点是 \(\cos z = 0\) 之处，即 \(z = \pi/2 + k\pi\)。在 \(|z| < \pi/2\) 内没有这些奇点（\(z = \pm \pi/2\) 恰好在边界上）。但题目要求 \(|z| = \pi/2\)——边界！这道题如所写其实有问题，因为奇点 \(\pm \pi/2\) 都恰好落在积分曲线上，积分根本不收敛。

修正一下：把曲线改成 \(|z| = 2\)。这时 \(z = \pm \pi/2\) 都进圈（\(\pi/2 \approx 1.57 < 2\)）。两个一阶极点，用 \(g/h'\) 公式：

\[ \mathrm{Res}(\tan z, \pi/2) = \frac{\sin(\pi/2)}{(\cos z)' |_{z = \pi/2}} = \frac{1}{-\sin(\pi/2)} = \frac{1}{-1} = -1. \] \[ \mathrm{Res}(\tan z, -\pi/2) = \frac{\sin(-\pi/2)}{-\sin(-\pi/2)} = \frac{-1}{1} = -1. \]

之和 \(-2\)，所以

\[ \oint_{|z| = 2} \tan z\, dz = 2\pi i \cdot (-2) = -4\pi i. \]

这道题的"陷阱"是：题目万一写错让奇点正好落在积分路径上，留数定理就不能直接用。下一章我们会处理这种"奇点恰在路径上"的情形——用小半圆绕过去，得到所谓的"主值积分"。

14.8 本章回顾

到这一章末尾你已经拿到了一把钥匙：留数。用一句话总结这把钥匙做什么——它把闭曲线复积分约化为局部代数运算。具体来说：

  • \(\mathrm{Res}(f, z_0) = a_{-1}\)：Laurent 展开里 \(1/(z - z_0)\) 的系数。
  • 留数定理：\(\oint_\gamma f = 2\pi i \sum \mathrm{Res}\)，求和取 \(\gamma\) 内部所有奇点。
  • 求留数有四类标配方法：一阶极点用 \((z - z_0) f\) 极限或 \(g/h'\)；\(m\) 阶极点用 \((m-1)\) 次导公式；本性奇点直接展级数。

"留数法算闭曲线积分"本身已经很有用——但它真正的舞台是实积分。下一章我们就来看：在实分析里有些怪积分（\(\int_0^{2\pi} \dfrac{d\theta}{a + b\cos\theta}\)、\(\int_{-\infty}^{\infty} \dfrac{dx}{1 + x^4}\)、\(\int_0^{\infty} \dfrac{\sin x}{x}\, dx\)、\(\int_0^{\infty} \dfrac{x^{a-1}}{1 + x}\, dx\)、……），用实分析的标准工具（换元、分部、部分分式）要么算不出来，要么算得让人想哭。但用留数法，每一道都能在三五行内给出干净的精确值。这是复分析最神奇、最实用的应用，也是为什么在工程、物理、概率论里复分析永远是标配工具的主因。准备好。
