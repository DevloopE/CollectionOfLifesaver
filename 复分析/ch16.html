第 16 章 共形映射基础

到目前为止，我们对解析函数的认识基本上是"分析的"：它有 Taylor 展开，它满足 Cauchy–Riemann 方程，它在闭曲线上的积分是零，等等。所有这些性质都是关于"函数值"的语句——告诉你 \(f(z)\) 该是什么数。但解析函数还有一面我们一直没有正经看过：它的几何面。一个解析函数 \(f\) 把 \(z\) 平面上的一片区域"搬"到 \(w\) 平面上的另一片区域。在这种"搬运"过程中，到底发生了什么几何变化？哪些角度被保留？哪些形状被扭曲？哪些区域能被映成什么样？

这一章的主题就是这件事。我们将看到一个朴素而惊人的事实：只要 \(f'(z_0) \ne 0\)，解析映射 \(f\) 在 \(z_0\) 附近保留任意两条曲线相交时的夹角——既保留大小，又保留方向。这就叫共形。这条性质是解析函数的"几何指纹"，它解释了为什么解析函数在物理学里如此有用：把一个形状奇怪的区域映成圆盘或上半平面，再把简单区域上解出来的物理量"搬"回来，原来的复杂边值问题就解决了。

本章我们做这几件事：先从一阶 Taylor 展开看出"为什么解析函数会保角"；再正式定义共形映射；然后看几个最经典的共形映射例子（\(z^2\)、\(\sin z\)、\(e^z\)）把哪些区域映成哪些区域；最后叙述黎曼映射定理——它告诉你一件极其漂亮的事：除了整个 \(\mathbb{C}\) 之外，所有的"单连通真子域"都共形等价于单位圆盘。这条定理是整个第 16–18 章的总纲，也是经典复分析最深的定理之一。

16.1 从 \(z\) 平面到 \(w\) 平面：几何动机

把一个解析函数 \(w = f(z)\) 看作一种"映射"：你给我 \(z\) 平面上一个点，我吐出 \(w\) 平面上一个点。如果你输入一条曲线 \(\gamma\)，我吐出来的是一条像曲线 \(f(\gamma)\)；如果你输入一片区域 \(D\)，我吐出来的是一片像区域 \(f(D)\)。

最朴素的例子是 \(w = z + 2\)：把每个点向右平移 2 个单位。整张地图被原样搬过去，距离、角度、形状全都不变。这是一个等距变换，也是一个解析函数。

稍微复杂一点的例子是 \(w = (1 + i)\, z\)。把 \(z = r e^{i\theta}\) 写出来，乘 \(1 + i = \sqrt{2}\, e^{i\pi/4}\) 得 \(w = \sqrt{2}\, r\, e^{i(\theta + \pi/4)}\)。所以这个映射干了两件事：把模长放大 \(\sqrt{2}\) 倍，再把幅角增加 \(\pi/4\)。也就是整张地图先放大、再绕原点逆时针转 45°。距离不再保持，但所有角度都被保持——两条曲线在 \(z\) 平面上交角多少，在 \(w\) 平面上还是多少。

这件事出乎意料吗？其实不。线性映射 \(z \mapsto (1+i) z\) 不过是"旋转加缩放"的复数表达，几何意义是显然的。真正出乎意料的是：任何解析函数，只要它的导数在某一点不为零，在那一点附近的局部行为都长得像"旋转加缩放"——即使它的全局表达式可能是 \(\sin z\)、\(e^z\) 那样高度非线性的东西。

下面我们来把这件事讲清楚。它的钥匙就在一阶 Taylor 展开里。

16.2 关键观察：\(f'(z_0) \ne 0\) 处保角

设 \(f\) 在 \(z_0\) 解析，且 \(f'(z_0) \ne 0\)。让两条光滑曲线 \(\gamma_1, \gamma_2\) 都过 \(z_0\)，记它们在 \(z_0\) 处的切线方向（用复数的辐角表示）分别为 \(\alpha_1, \alpha_2\)。它们在 \(z_0\) 处的交角就是

\[ \theta = \alpha_2 - \alpha_1. \]

把它们经过 \(f\) 映过去，得到两条像曲线 \(f(\gamma_1), f(\gamma_2)\)，都过 \(w_0 = f(z_0)\)。问：它们在 \(w_0\) 处的交角，是多少？

关键事实。 在上述假设下，

\[ \boxed{\arg\big(\text{切线}_2\text{在}\,w_0\big) - \arg\big(\text{切线}_1\text{在}\,w_0\big) = \alpha_2 - \alpha_1 = \theta.} \]

也就是说：两条像曲线在 \(w_0\) 处的交角，等于原曲线在 \(z_0\) 处的交角，连"方向"都一致。换句话说，这个角度既不被压扁、也不被翻转。我们把这种保留角度大小与方向的映射叫作共形 (conformal)。

这个事实在第一次见到时多少有点震撼。\(f\) 可能是 \(e^z\)、\(\sin z\)、甚至更怪的解析函数；它们在不同点附近的"形变"看上去千变万化。但只要 \(f'(z_0) \ne 0\)，无穷小尺度上它都只是"旋转加缩放"——所以角度被原原本本保留。下面这一节我们用一阶 Taylor 把这件事的"为什么"挖出来。

16.3 为什么会保角？一阶 Taylor 的几何含义

\(f\) 在 \(z_0\) 解析意味着它在 \(z_0\) 附近有 Taylor 展开

\[ f(z) = f(z_0) + f'(z_0) (z - z_0) + \tfrac{1}{2} f''(z_0) (z - z_0)^2 + \cdots \]

当 \(z\) 离 \(z_0\) 非常近时，\((z - z_0)^2\) 比 \((z - z_0)\) 小得多，更高阶的项更小。所以一阶近似（也就是把上式只留前两项）已经能抓住 \(f\) 在 \(z_0\) 附近的行为：

\[ f(z) - f(z_0) \approx f'(z_0) (z - z_0). \]

用 \(\Delta z := z - z_0\) 表示原图上的一小段位移，\(\Delta w := f(z) - f(z_0)\) 表示像图上对应的一小段位移。把 \(f'(z_0)\) 写成模与辐角形式 \(f'(z_0) = R e^{i\varphi}\)（其中 \(R = |f'(z_0)| > 0\)，\(\varphi = \arg f'(z_0)\)）。那么

\[ \Delta w \approx R e^{i\varphi} \cdot \Delta z. \]

这一行就是答案。乘 \(R e^{i\varphi}\) 在复平面上做了什么？两件事：

  • 缩放：把 \(|\Delta z|\) 放大 \(R = |f'(z_0)|\) 倍；
  • 旋转：把 \(\arg \Delta z\) 增加 \(\varphi = \arg f'(z_0)\)。

所以"无穷小"位移 \(\Delta z\) 在 \(f\) 下被变成了"旋转 \(\varphi\) 角，并放大 \(R\) 倍"的 \(\Delta w\)。注意一件至关重要的事：缩放因子 \(R\) 和旋转角 \(\varphi\) 对一切方向的 \(\Delta z\) 都一样！它们只取决于 \(f\) 在哪一点求值，与 \(\Delta z\) 沿什么方向无关。

这件事直接给出保角性。设两段无穷小位移 \(\Delta z_1, \Delta z_2\) 各代表两条曲线在 \(z_0\) 处的切向。它们的辐角差 \(\arg \Delta z_2 - \arg \Delta z_1\) 就是两曲线在 \(z_0\) 的交角 \(\theta\)。映过去后

\[ \Delta w_1 \approx R e^{i\varphi} \Delta z_1, \quad \Delta w_2 \approx R e^{i\varphi} \Delta z_2, \]

所以

\[ \arg \Delta w_2 - \arg \Delta w_1 = (\varphi + \arg \Delta z_2) - (\varphi + \arg \Delta z_1) = \arg \Delta z_2 - \arg \Delta z_1 = \theta. \]

左边是像曲线在 \(w_0\) 处的交角，右边是原曲线在 \(z_0\) 处的交角。二者相等。保角性证毕。

注意到的几个细节：

  1. 必须 \(f'(z_0) \ne 0\)。如果 \(f'(z_0) = 0\)，则 \(R = 0\)，"缩放因子"为零、"辐角"无定义，上面的论证整个崩溃。这时一阶近似 \(\Delta w \approx 0\) 不再是 \(\Delta z\) 的线性表达，必须看更高阶项才能算清局部几何（见 16.5 节）。
  2. 角度的方向也保持。\(\arg \Delta z_2 - \arg \Delta z_1\) 是一个有正负号的数，正号代表逆时针角度。映过去后这个号没变，所以"逆时针"还是"逆时针"，没翻转成"顺时针"。这与反共形映射（如 \(z \mapsto \bar z\)）不同——后者保角度的大小但翻转方向。
  3. 缩放因子 \(R\) 与方向无关，但与位置有关。在不同点 \(z_0\) 处，\(|f'(z_0)|\) 一般不同。所以解析映射"局部各向同性地缩放"，但缩放比例随位置变化。

这就是为什么解析函数的几何那么独特：它把"小圆"映成"小圆"（局部各向同性），尽管全局可以非常扭曲。

16.4 共形映射的定义

把上一节的几何性质提升为正式定义。

定义。 设 \(D \subset \mathbb{C}\) 是开集，\(f: D \to \mathbb{C}\) 是函数。如果 \(f\) 在每一点 \(z_0 \in D\) 处都保角（即保任意两条过 \(z_0\) 光滑曲线交角的大小与方向），就称 \(f\) 是 \(D\) 上的共形映射 (conformal map)。

16.3 节的论证立刻给出一个判别准则：

判别准则。 若 \(f\) 在 \(D\) 上解析且 \(f'(z) \ne 0\) 对所有 \(z \in D\) 成立，则 \(f\) 是 \(D\) 上的共形映射。

反过来其实也大致正确：若 \(f\) 是 \(D\) 上的共形映射（连同某些光滑性条件），则 \(f\) 必定解析且 \(f' \ne 0\)。我们暂不严格证明这个反向，只把它当作"共形 ⟺ 解析 + 导数非零"的口诀记住。

注意定义里对"双射"没有要求。一些教材把"共形映射"的定义直接绑上"双射"——这种定义下共形映射就是一种双向无歧义的几何变换，更有结构。本书采取较宽松的定义：共形只要求"局部保角"，不要求"全局一对一"。后面碰到的具体例子（如 \(w = z^2\)）会清楚地告诉你这里的区别。

下面立刻看一个反面例子，弄清"\(f'(z_0) = 0\)"时几何上发生了什么。

16.5 临界点：\(f'(z_0) = 0\) 处不保角

"\(f'(z_0) = 0\)"被称为 \(f\) 的临界点 (critical point)。在临界点处保角性失败，因为一阶 Taylor 展开退化为零。

看具体例子。取 \(f(z) = z^2\)，则 \(f'(z) = 2z\)，所以 \(z_0 = 0\) 是临界点。在 \(z_0 = 0\) 附近，\(f(z) - f(0) = z^2\)，所以 \(\Delta w = (\Delta z)^2\)。把 \(\Delta z = r e^{i\theta}\) 代入：

\[ \Delta w = r^2 e^{i \cdot 2\theta}. \]

看辐角的变化。\(\Delta z\) 的辐角是 \(\theta\)，\(\Delta w\) 的辐角是 \(2\theta\)——翻倍了。两段位移 \(\Delta z_1, \Delta z_2\) 的辐角差 \(\theta_2 - \theta_1\) 经映射后变成 \(2\theta_2 - 2\theta_1 = 2(\theta_2 - \theta_1)\)。角度被加倍了！

所以在 \(z_0 = 0\) 这个临界点，\(z \mapsto z^2\) 不保角而是把角放大两倍。原本两条互相垂直的小线段（夹角 \(\pi/2\)），映过去后夹角变成 \(\pi\)——变成共线了。

更一般地，若 \(f\) 在 \(z_0\) 处的最低阶非零导数是第 \(k\) 阶（即 \(f'(z_0) = \cdots = f^{(k-1)}(z_0) = 0\)，\(f^{(k)}(z_0) \ne 0\)），则在 \(z_0\) 附近

\[ f(z) - f(z_0) \approx \frac{f^{(k)}(z_0)}{k!} (z - z_0)^k. \]

同样的论证给出：在这种 \(k\) 阶临界点处，角度被乘以 \(k\)。\(k = 1\)（普通点）保角；\(k = 2\) 加倍；\(k = 3\) 三倍；以此类推。

这是一条非常有用的事实：临界点的"阶"（也就是导数最先在第几阶非零）直接决定了局部映射"扭曲"的程度。

16.6 例：\(w = z^2\) 的全局几何

把上一节的局部分析推到全局，来看 \(w = z^2\) 这个映射做了什么。

用极坐标。\(z = r e^{i\theta}\)（\(r \ge 0\)，\(\theta \in [0, 2\pi)\)），则 \(w = z^2 = r^2 e^{i \cdot 2\theta}\)。所以：

  • \(z\) 平面上 \(r =\) 常数的圆 \(|z| = a\)，被映为 \(w\) 平面上 \(|w| = a^2\) 的圆；
  • \(z\) 平面上 \(\theta =\) 常数的射线，被映为 \(w\) 平面上幅角为 \(2\theta\) 的射线；
  • 整个上半平面 \(\{0 < \theta < \pi\}\) 被映成整个除原点的 \(w\) 平面（因为 \(2\theta\) 跑遍 \((0, 2\pi)\)）。

换言之，\(w = z^2\) 把上半平面"展开"成整张 \(w\) 平面。具体地：

  • 正实轴 \(\theta = 0\) 映到正实轴 \(\arg w = 0\)；
  • 第一象限的边界射线 \(\theta = \pi/4\) 映到 \(\arg w = \pi/2\)，即正虚轴；
  • 正虚轴 \(\theta = \pi/2\) 映到 \(\arg w = \pi\)，即负实轴；
  • 第二象限的边界射线 \(\theta = 3\pi/4\) 映到 \(\arg w = 3\pi/2\)，即负虚轴；
  • 负实轴 \(\theta = \pi\) 映到 \(\arg w = 2\pi = 0\)，回到正实轴。

所以"上半平面"被这个映射展开成"完整的角度 \(2\pi\)"，恰好覆盖整张 \(w\) 平面（去掉原点）一次。

那如果你看的是整个 \(z\) 平面（不只是上半）呢？此时 \(\theta\) 跑遍 \([0, 2\pi)\)，\(2\theta\) 跑遍 \([0, 4\pi)\)，相当于把整个 \(w\) 平面绕了两圈——所以 \(w = z^2\) 在整个 \(\mathbb{C}\) 上是二对一的：每个 \(w \ne 0\) 都对应两个 \(z\)（互为相反数）。

这正好对应于"一元二次方程 \(z^2 = w\) 一般有两个根"。

例题 16.6.1。第一象限 \(\{x > 0, y > 0\}\) 在 \(w = z^2\) 下是什么样的区域？

解。 第一象限就是 \(\theta \in (0, \pi/2)\)、\(r > 0\)。映过去后 \(2\theta \in (0, \pi)\)，所以像区域就是上半平面 \(\{\operatorname{Im} w > 0\}\)。直观看就是"把第一象限张开成整个上半平面"——这把直角张开成了平角。

反过来说，函数 \(w = z^{1/2}\)（取主值，即"主平方根"）把上半平面映回到第一象限。这就是我们之后构造共形映射时的核心办法：通过幂次映射把"扇形角度"放大或缩小。

例题 16.6.2。直线 \(y = 1\) 在 \(w = z^2\) 下是什么曲线？

解。 直线 \(y = 1\) 上的点 \(z = x + i\)。算 \(w = z^2 = (x + i)^2 = x^2 - 1 + 2 i x\)。设 \(w = u + iv\)，则 \(u = x^2 - 1\)、\(v = 2x\)。消去 \(x\)：从第二式 \(x = v/2\)，代入第一式得 \(u = v^2/4 - 1\)，即 \(v^2 = 4(u + 1)\)——抛物线，焦点在原点附近，开口向右。

顺便观察一下这个例子的"保角性"。直线 \(y = 1\) 与垂直直线 \(x = 0\) 相交于 \(z = i\)，交角 \(\pi/2\)。在 \(z = i\) 处 \(f'(i) = 2i \ne 0\)，所以 \(f\) 保角。直线 \(x = 0\) 映过去是另一条曲线（具体而言：把 \(z = iy\) 代入得 \(w = -y^2\)，是负实轴）。这条像曲线（负实轴）在 \(w = -1\) 处与上面那条抛物线 \(v^2 = 4(u+1)\) 的切线方向是什么？算抛物线在 \(u = -1\) 即 \(v = 0\) 处的切线：对 \(v^2 = 4(u+1)\) 隐式求导得 \(2 v \, dv/du = 4\)，即 \(dv/du = 2/v\)；当 \(v = 0\) 时这"无穷大"，意味着切线垂直，方向是垂直方向。所以两条像曲线（负实轴是水平方向，抛物线在 \(w = -1\) 处是垂直方向）夹角 \(\pi/2\)，跟原图一样。保角性如约。

16.7 几个经典映射的几何

除了 \(z^2\)，还有几个映射在共形几何里反复出现。我们各看一下它们把哪些"标准区域"映成哪些"标准区域"。

16.7.1 \(w = e^z\)

把 \(z = x + iy\) 代入：\(w = e^z = e^x e^{iy}\)，所以 \(|w| = e^x\)、\(\arg w = y\)。这就是说：

  • \(z\) 平面上"水平直线" \(y =\) 常数 映为 \(w\) 平面上的"过原点射线" \(\arg w = y\)；
  • \(z\) 平面上"竖直直线" \(x =\) 常数 映为 \(w\) 平面上的"以原点为心的圆" \(|w| = e^x\)。

所以"竖直与水平的格点"被映成"圆与射线的极坐标格点"。这是 \(e^z\) 最漂亮的几何特性。

具体两个有用的例子：

  • 水平条带映为扇形。条带 \(\{0 < y < \pi\}\)（无穷长水平条带）映为上半平面。原因：\(y \in (0, \pi)\) 给出 \(\arg w \in (0, \pi)\)，而 \(x \in (-\infty, \infty)\) 给出 \(|w| \in (0, \infty)\)。所以像区域是 \(\{0 < \arg w < \pi\}\)，正是上半平面。
  • 半带映为半圆盘。半带 \(\{x < 0, 0 < y < \pi\}\) 映为上半圆盘 \(\{|w| < 1, \operatorname{Im} w > 0\}\)。原因：\(x < 0\) 给出 \(|w| < 1\)，再叠加 \(\arg w \in (0, \pi)\)。

这两个映射在物理问题里非常常用：把"无穷条带"或"半条带"上的 PDE 边值问题，通过 \(e^z\) 搬到上半平面或半圆盘上去解。

注意 \(f'(z) = e^z\) 在所有 \(z\) 处都不为零，所以 \(e^z\) 在整个 \(\mathbb{C}\) 上共形——但它不是双射，因为周期 \(2\pi i\) 让无穷多个 \(z\) 映到同一个 \(w\)。这告诉我们"共形"和"双射"是两个独立的概念。

16.7.2 \(w = \sin z\)

用恒等式 \(\sin z = \sin x \cosh y + i \cos x \sinh y\)，得 \(u = \sin x \cosh y\)、\(v = \cos x \sinh y\)。

看一个有趣的例子：把半带 \(\{-\pi/2 < x < \pi/2,\; y > 0\}\) 映成什么？

沿半带的左边界 \(x = -\pi/2\)，得 \(u = -\cosh y\)、\(v = 0\)，是负实轴上 \(u \le -1\) 的部分。

沿半带的右边界 \(x = \pi/2\)，得 \(u = \cosh y\)、\(v = 0\)，是正实轴上 \(u \ge 1\) 的部分。

沿半带的下边界 \(y = 0\)（即 \(x \in (-\pi/2, \pi/2)\)），得 \(u = \sin x\)、\(v = 0\)，是实轴上 \(-1 < u < 1\) 的部分。

所以半带的整条边界（三段直线）被映为 \(w\) 平面上的整条实轴。半带的内部呢？由保角性可以推出半带的内部被映为上半平面（这个推断需要稍多一点工作；这里你接受这个结论即可）。

所以 \(\sin\) 把半带 \(\{-\pi/2 < x < \pi/2,\; y > 0\}\) 共形地映到上半平面。这又是物理问题里常用的一个映射。\(f'(z) = \cos z\) 在上述半带内部不为零，所以确实保角。注意端点 \(\pm \pi/2\) 处 \(\cos = 0\) 是临界点，那里角度被加倍——而事实上半带的两个直角（在 \(z = \pm \pi/2\) 处）各被映为平角（沿实轴方向），符合"角度加倍"。

16.7.3 整理常用映射

把目前为止学到的全部映射做个汇总（也加上 17 章会讲的 Möbius 变换 \(\dfrac{z-i}{z+i}\)）：

映射把什么映成什么
\(w = z + b\)任何区域平移后的同形区域
\(w = az\)（\(a \ne 0\)）任何区域缩放并旋转后
\(w = z^2\)第一象限上半平面
\(w = z^{1/2}\)上半平面第一象限
\(w = z^a\)（\(0 < a\)）角 \(0 < \arg z < \pi/a\) 的扇形上半平面
\(w = e^z\)条带 \(0 < y < \pi\)上半平面
\(w = e^z\)半带 \(x < 0, 0 < y < \pi\)上半圆盘
\(w = \log z\)（主支）上半平面条带 \(0 < y < \pi\)
\(w = \sin z\)半带 \(|x| < \pi/2, y > 0\)上半平面
\(w = \dfrac{z-i}{z+i}\)（17 章）上半平面单位圆盘

有了这些"基本砖块"，再加上"复合"操作，几乎所有几何形状漂亮的区域，都能被映到标准区域（圆盘或上半平面）上去。下面的章节我们会反复使用这一点。

16.8 共形映射的物理力量

为什么我们花这么大力气研究共形映射？最直接的回答是：它能把复杂区域上的物理问题简化到简单区域上的问题。

很多重要的物理量——稳态温度分布、静电电势、不可压缩无旋流体的速度势、薄膜的位移——都满足同一类方程：Laplace 方程

\[ \Delta u := \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0. \]

这条方程的解叫做调和函数。我们之前已经看过：解析函数的实部和虚部都是调和函数。

典型的物理问题长这样：在一个区域 \(D\)（具体的形状）内，求满足 \(\Delta u = 0\) 的 \(u\)，使得它在边界 \(\partial D\) 上等于给定的函数 \(f\)。这就是Dirichlet 问题，第 18 章会专门讲。

问题的难度严重依赖区域的形状。如果区域是圆盘或上半平面，Dirichlet 问题有显式解（Poisson 公式）。如果区域是一般的扇形、矩形、或者带凹角的奇怪形状，直接解就难得多。

共形映射的妙处是：调和函数被解析映射"复合"后还是调和函数（这条事实将在 18.6 节详细推导）。所以你可以这么操作：

  1. 给定难解的区域 \(D\) 和它上面的 Dirichlet 问题；
  2. 找一个共形映射 \(\phi: D \to \mathbb{D}\)（\(\mathbb{D}\) 是单位圆盘）；
  3. 把 Dirichlet 问题的边界条件"搬"到 \(\partial \mathbb{D}\) 上——具体而言，\(\partial D\) 上的函数 \(f\) 通过 \(\phi^{-1}\) 翻译为 \(\partial \mathbb{D}\) 上的某个 \(\tilde f\)；
  4. 用 Poisson 公式在 \(\mathbb{D}\) 上解出 \(\tilde u\)；
  5. 把 \(\tilde u\) "搬"回到 \(D\) 上：\(u(z) = \tilde u (\phi(z))\)，这就是 \(D\) 上的解。

整个过程把"复杂区域上的难解问题"变成了"简单区域上的可解问题 + 一次共形映射"。下一节我们要看的黎曼映射定理就告诉你：这种"共形映射"几乎总是存在。

16.9 共形等价

给两个区域 \(D_1, D_2 \subset \mathbb{C}\)，如果存在一个双射共形映射 \(f: D_1 \to D_2\)（这意味着 \(f\) 解析、处处 \(f' \ne 0\)、且与逆函数互为反函数），就说 \(D_1\) 与 \(D_2\) 共形等价 (conformally equivalent)。

这是一种"等价关系"：自反（\(f = \mathrm{id}\)）、对称（\(f^{-1}\) 也是共形的）、传递（共形映射的复合还是共形）。所以"共形等价"把所有平面区域分成了等价类，每一类里所有的区域，从共形分析的角度看是"一样的"。

注意：共形等价比"拓扑等价"（同胚）严格得多。任何两个单连通区域在拓扑上都同胚（都同胚于圆盘），但它们不一定共形等价——共形等价多了"角度保留"的几何约束。

下面这个反例值得记住：整个 \(\mathbb{C}\) 与单位圆盘 \(\mathbb{D}\) 拓扑同胚，但不共形等价。原因是 Liouville 定理：从 \(\mathbb{C}\) 到 \(\mathbb{D}\) 的解析函数是有界的，而 \(\mathbb{C}\) 上有界解析函数必为常数——所以不可能存在 \(\mathbb{C}\) 到 \(\mathbb{D}\) 的双射解析映射。

这个唯一的"例外"令人有点难受：能不能找出"哪些"区域共形等价于圆盘？黎曼的回答非常令人吃惊——除了 \(\mathbb{C}\) 自己和"退化的"情形以外，几乎所有单连通区域都共形等价于圆盘。

16.10 黎曼映射定理

把这条惊人的事实叙述出来。

黎曼映射定理 (Riemann mapping theorem)。 设 \(D \subsetneq \mathbb{C}\) 是一个单连通开集（且 \(D \ne \mathbb{C}\)）。那么存在一个共形双射

\[ \phi: D \to \mathbb{D}, \]

其中 \(\mathbb{D} = \{w \in \mathbb{C}: |w| < 1\}\) 是单位开圆盘。此外，若再额外指定 \(z_0 \in D\) 与 \(\phi(z_0) = 0\) 以及 \(\arg \phi'(z_0) = 0\)，则这个 \(\phi\) 还是唯一的。

这条定理的力量在于"几乎不挑区域"：单连通、是 \(\mathbb{C}\) 的真子集（即排除 \(\mathbb{C}\) 自己）就够。形状无所谓——可以是漂亮的圆、椭圆、矩形，也可以是任意的没有"洞"的奇形怪状区域。它都共形等价于单位圆盘。

本书不证明这条定理。它的证明在 19 世纪后期由 Schwarz、Carathéodory、Koebe 等人陆续完成，需要相当多的预备知识（Montel 定理、Hurwitz 定理、紧致性论证等）。但理解定理的陈述就已经足够指导我们使用它。

对"单连通"和"\(D \ne \mathbb{C}\)"两个条件，注意：

  • 单连通：直观就是"区域里没有洞"。环形区域 \(1 < |z| < 2\) 不单连通，所以定理不适用。事实上环形区域共形等价于其它环形区域（且只共形等价于内外半径之比相同的环），这是另一类问题（多连通区域的共形分类）。
  • \(D \ne \mathbb{C}\)：上一节解释过为什么 \(\mathbb{C}\) 不能共形等价于 \(\mathbb{D}\)。

"唯一性"那一段（指定 \(z_0\) 映到 0、\(\arg \phi'(z_0) = 0\)）的物理含义是：把"中心"和"朝向"都钉死之后，共形映射就完全确定了。这是因为 \(\mathbb{D}\) 自己的共形自同构有 3 个实自由度（一个圆盘自同构形如 \(\dfrac{z - a}{1 - \bar a z} e^{i\theta}\)，有 \(a\) 的两个实自由度加 \(\theta\) 一个），刚好对应"\(\phi(z_0) = 0\)"两个实方程加 "\(\arg \phi'(z_0) = 0\)" 一个实方程。

16.11 黎曼映射定理的力量

用一句话总结这条定理的实用价值：理论上，几乎任何"形状"的单连通区域上的 PDE 边值问题，都可以"搬"到圆盘上去解。

定理的存在性意义和构造性意义有微妙的差别。"存在性"告诉你：这个共形映射 \(\phi\) 一定有。"构造性"是另一回事：实际找出 \(\phi\) 可能很难——黎曼映射定理的标准证明并不显式地构造 \(\phi\)，只是证明它存在（用紧致性论证）。所以在实际计算时，我们仍然要靠手工或数值地"拼"出 \(\phi\)：用之前 16.7 节列的几个标准映射（\(z^a, e^z, \sin z, \dfrac{z-i}{z+i}\) 等）的复合，把要研究的区域一步步变形过去。

这个过程像在解几何拼图。每一步用一个简单的映射把当前区域"扭"成一个稍简单的形状，最后到圆盘或上半平面。下面看一个具体例子。

例题 16.11.1。把上半圆盘 \(\Omega = \{|z| < 1, \operatorname{Im} z > 0\}\) 共形映到单位圆盘 \(\mathbb{D}\)。

解。分两步走。

第一步：把上半圆盘映到上半平面。先用 Möbius 变换 \(z_1 = \dfrac{1 + z}{1 - z}\)。这把 \(z = -1 \to z_1 = 0\)，\(z = 1 \to z_1 = \infty\)，\(z = i \to z_1 = i\)。它把上半圆盘映到第一象限（验证：边界圆 \(|z| = 1\) 映为虚轴的非负部分；边界直径 \([-1, 1]\) 映为正实轴；内部映为第一象限）。

第二步：把第一象限映到上半平面。用 \(z_2 = z_1^2\)：第一象限对应辐角 \(\theta \in (0, \pi/2)\)，平方后辐角 \(\in (0, \pi)\)，正是上半平面。

第三步：把上半平面映到单位圆盘。用 \(w = \dfrac{z_2 - i}{z_2 + i}\)（17 章我们会详细讲这个 Möbius 变换；现在你只要相信它把上半平面映到单位圆盘）。

把三步合成：

\[ w = \frac{z_2 - i}{z_2 + i} = \frac{z_1^2 - i}{z_1^2 + i} = \frac{\Big(\dfrac{1+z}{1-z}\Big)^2 - i}{\Big(\dfrac{1+z}{1-z}\Big)^2 + i}. \]

整理一下（两边都乘 \((1-z)^2\)）：

\[ \boxed{w = \frac{(1+z)^2 - i (1-z)^2}{(1+z)^2 + i (1-z)^2}.} \]

这就是从上半圆盘到单位圆盘的一个具体共形映射。三步分别是：Möbius 变换、平方、Möbius 变换。这种"找对一连串简单映射、串起来得到目标"的思路，是处理具体共形映射问题的标准套路。

例题 16.11.2。把扇形 \(\Omega = \{0 < \arg z < \pi/3\}\)（角度 60°的扇形）映到单位圆盘。

解。角度 \(\pi/3\) 不是 \(\pi\)（不是上半平面），也不是 \(\pi/2\)（不是第一象限）。怎么变？用幂次映射"放大"角度。具体地，取 \(z_1 = z^3\)：原扇形的角度 \(\pi/3\) 被乘以 3 后变成 \(\pi\)，正是上半平面。再用 \(w = \dfrac{z_1 - i}{z_1 + i}\) 把上半平面送到圆盘。最终：

\[ w = \frac{z^3 - i}{z^3 + i}. \]

这就是答案。注意这里 \(z = 0\) 处 \(f'(0) = 0\)（因为我们用了 \(z^3\)）——所以 0 是共形映射的临界点。但 0 不在扇形 \(\Omega\) 的内部（它在边界顶点上），所以这无伤大雅；映射在 \(\Omega\) 的内部仍然是处处共形的。

例题 16.11.3。把无穷条带 \(\Omega = \{0 < \operatorname{Im} z < \pi\}\) 映到单位圆盘。

解。16.7 节我们看到 \(z_1 = e^z\) 把这条带映到上半平面。再用 Möbius 把上半平面映到圆盘：

\[ w = \frac{e^z - i}{e^z + i}. \]

这条公式在物理学里常常出现，特别是处理沿一根无穷长棒的稳态温度问题。

例题 16.11.4。把上半平面去掉一段竖直线段 \(\{0 < y \le 1\}\)（即 \(\Omega = \{\operatorname{Im} z > 0\} \setminus \{x = 0, 0 < y \le 1\}\)）映到上半平面。

解。这是一个"切了一刀"的上半平面，用经典的 \(\sqrt{}\) 技巧。先把 \(z_1 = z^2\)。但要小心定义，因为 \(z^2\) 把整个上半平面映到全平面去原点不是双射。我们用 \(z_1 = z^2 + 1\)：算一下边界。

  • 实轴正半段 \(z = x > 0\)：\(z_1 = x^2 + 1 > 1\)，是 \([1, \infty)\)。
  • 实轴负半段 \(z = x < 0\)：\(z_1 = x^2 + 1 > 1\)，也是 \([1, \infty)\)（重叠了）。
  • 切口 \(z = iy, 0 < y \le 1\)：\(z_1 = -y^2 + 1 \in [0, 1)\)。

这有问题——边界塌缩。换一个思路：先用 \(\sqrt{z^2 + 1}\)，让切口的两边"展开"。

取 \(w = \sqrt{z^2 + 1}\)（取保正实部的支）。沿实轴 \(z = x\)：\(z^2 + 1 = x^2 + 1 > 0\)，\(w = \sqrt{x^2 + 1}\) 实正。沿切口右侧 \(z = 0^+ + iy\)：\(z^2 = -y^2 + 0\)，\(z^2 + 1 = 1 - y^2 \in [0, 1)\)，\(w \in [0, 1)\)。沿切口左侧 \(z = 0^- + iy\)：和右侧"差一个支"，\(w \in (-1, 0]\)。所以切口的两边被"撑开"成了实轴的 \([-1, 1]\) 段。整个上半平面（带切口）映成了上半平面。验证：边界变成实轴，内部映成内部。

所以 \(w = \sqrt{z^2 + 1}\) 把"被切了一道竖直缝的上半平面"映到上半平面。这种"\(\sqrt{}\) 把一条切口撑开"的技巧在物理学里叫做"打开支割线"，在流体绕翼型流动等问题里非常有用。

这一章我们看到，共形映射是把"几何"和"分析"挂钩的桥梁：解析函数的代数性质（Cauchy–Riemann、Taylor 展开）翻译成几何性质（保角、局部各向同性的缩放），而几何上的"共形等价"又给我们一个把复杂问题变简单问题的工具。下一章我们集中研究最重要的一类共形映射——Möbius 变换，它在共形几何里扮演着"基本砖块"的角色。
