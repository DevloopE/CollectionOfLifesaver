第 19 章 调和函数与边值问题

在第 17、18 章你看见过解析函数的实部和虚部各自满足 Cauchy–Riemann 方程，并隐隐约约听到一句"它们都是调和函数"。本章把这件事郑重其事地展开来讲。调和函数是物理里出现得最频繁的那一类函数——稳态温度场、不可压流体的速度势、静电势、引力势——基本上凡是"达到平衡、不再随时间变化"的二维场，背后都站着一个调和函数。本章我们会一一看到：

  • 调和函数到底是什么；它和解析函数是什么关系。
  • 给一个调和函数 \(u\)，怎么把它"补全"成一个解析函数 \(f = u + iv\)（求调和共轭）。
  • 调和函数继承自解析函数的两条核心性质：平均值性质与最大模原理。
  • 边界值唯一决定内部值——这就是 Dirichlet 问题的灵魂。
  • 圆盘上 Dirichlet 问题的封闭解：Poisson 核。
  • Schwarz 反射原理：对称性把"半域上的调和函数"延拓到整个平面。
  • 一个稳态二维热传导的具体例子，把所有理论收口到工程实用上。

读这一章的时候建议你随时回看第 18 章末尾的 Dirichlet 问题的引子，因为本章很多地方是在把那一节真正讲清楚。

19.1 调和函数复习

从最朴素的定义开始。一个定义在某区域 \(\Omega \subset \mathbb{R}^2\) 上的实值函数 \(u(x, y)\)，若它有连续二阶偏导数，并且满足

\[ \Delta u := u_{xx} + u_{yy} = 0 \]

就叫做调和函数。算符 \(\Delta = \partial_x^2 + \partial_y^2\) 叫做 Laplace 算符，所以这个方程也叫 Laplace 方程。简简单单两项相加等于零，这条方程为什么会在物理学里如此无所不在？因为它是"局部平衡"的数学形式——一个点上的值等于周围一小圈的平均值（这正是 19.3 节要证明的东西）。当一个量"既不流入也不流出"、"既不被生产也不被消耗"的时候，它必然如此。

来几个具体例子让定义不再是抽象符号。

例 19.1.1. 函数 \(u(x, y) = x\) 调和吗？算 \(u_{xx} = 0\)、\(u_{yy} = 0\)，加起来是 0，所以是。同理 \(u(x, y) = y\) 也调和；任何线性函数 \(ax + by + c\) 都调和。

例 19.1.2. \(u(x, y) = x^2 - y^2\)。\(u_{xx} = 2\)，\(u_{yy} = -2\)，相加是 0。调和。但请注意它不是 \(x^2 + y^2\) 的样子——\(u(x, y) = x^2 + y^2\) 算出来 \(u_{xx} + u_{yy} = 4 \neq 0\)，不调和。这是初学者常见误解：似乎"对称的"东西就是调和的。不是。区别就是那一个负号——它把方程从"两个二阶导都正"扭成"一正一负，恰好抵消"。

例 19.1.3. \(u(x, y) = e^x \cos y\)。\(u_{xx} = e^x \cos y\)，\(u_{yy} = -e^x \cos y\)，相加为 0。调和。换成 \(u = e^x \sin y\) 也调和（验证留给你）。这两位是 \(e^z = e^x(\cos y + i \sin y)\) 的实部和虚部——这里你应该已经嗅到调和函数和解析函数的紧密联系。

例 19.1.4. \(u(x, y) = \ln(x^2 + y^2) = 2 \ln|z|\)。它在原点之外有定义。算一算：

\[ u_x = \frac{2x}{x^2 + y^2}, \quad u_{xx} = \frac{2(y^2 - x^2)}{(x^2 + y^2)^2}. \]

由对称性 \(u_{yy} = \dfrac{2(x^2 - y^2)}{(x^2 + y^2)^2}\)，二者相加正好是 0。所以 \(\ln|z|\) 在 \(z \neq 0\) 处调和。它是物理上最重要的调和函数之一——二维静电荷在自由空间产生的电势就是它（差一个常数因子）。

注意到一个共同点：每次出现调和函数，似乎都能找到一个解析函数把它"罩"起来。\(x = \mathrm{Re}\, z\)、\(x^2 - y^2 = \mathrm{Re}\, z^2\)、\(e^x \cos y = \mathrm{Re}\, e^z\)、\(\ln|z| = \mathrm{Re}\, \log z\)。这不是巧合。下一节我们就把这件事一般地证明。

19.2 调和函数都是某个解析函数的实部

第 17 章我们证过：若 \(f(z) = u(x, y) + i v(x, y)\) 在某区域上解析，则 \(u\) 和 \(v\) 都是调和的。这件事来自 Cauchy–Riemann 方程 \(u_x = v_y, \, u_y = -v_x\) 的二次求导：

\[ u_{xx} = (u_x)_x = (v_y)_x = v_{xy}, \quad u_{yy} = (u_y)_y = (-v_x)_y = -v_{xy}. \]

相加得 \(u_{xx} + u_{yy} = 0\)，搞定。同理对 \(v\) 也成立。

本节问反方向的问题：给我一个调和函数 \(u\)，它一定是某个解析函数 \(f\) 的实部吗？答案是"在单连通区域上是"。下面我们一边给方法一边证明。

问题. 设 \(\Omega\) 是单连通区域，\(u : \Omega \to \mathbb{R}\) 调和。能否找到 \(v : \Omega \to \mathbb{R}\) 使得 \(f = u + iv\) 在 \(\Omega\) 上解析？这个 \(v\) 就叫做 \(u\) 的调和共轭 (harmonic conjugate)。

关键想法是这样：如果这样的 \(v\) 存在，根据 Cauchy–Riemann 方程，它必须满足

\[ v_x = -u_y, \qquad v_y = u_x. \]

所以 \(v\) 的两个偏导数完全由 \(u\) 决定。要构造 \(v\)，就是要解这个"梯度规定好"的两元一阶方程组。回忆向量微积分：一个向量场 \((-u_y, u_x)\) 是某个函数 \(v\) 的梯度，当且仅当它无旋（在单连通区域上）。检查无旋条件：

\[ \partial_y(-u_y) = -u_{yy}, \qquad \partial_x(u_x) = u_{xx}. \]

这两者相等当且仅当 \(u_{xx} + u_{yy} = 0\)，这正是 \(u\) 调和的假设！所以无旋成立，向量场是某函数的梯度，于是 \(v\) 存在。

具体怎么求？沿着任一条从某固定点 \((x_0, y_0)\) 到 \((x, y)\) 的曲线 \(\gamma\) 积分：

\[ v(x, y) = \int_\gamma (-u_y\, dx + u_x\, dy) + C. \]

在单连通区域上，由于无旋积分与路径无关，这个 \(v\) 良定义到一个相加常数。常数 \(C\) 反映了"\(v\) 整体加一个常数仍然是调和共轭"——这件事符合直觉：把 \(f\) 的虚部整体平移，\(f\) 还是解析的（差一个虚常数）。

实际计算时几乎不用线积分，而是这样操作：先用 \(v_y = u_x\) 把 \(v\) 对 \(y\) 积一次，得到含一个待定 \(x\) 的函数 \(g(x)\) 的中间表达式；再用 \(v_x = -u_y\) 把 \(g(x)\) 定下来。一个例子立刻看清这套程序。

例 19.2.1. 求 \(u(x, y) = x^2 - y^2\) 的调和共轭。

第一步算 \(u_x = 2x\)、\(u_y = -2y\)。

第二步用 \(v_y = u_x = 2x\) 对 \(y\) 积分：\(v(x, y) = 2xy + g(x)\)，其中 \(g(x)\) 是一个待定的纯 \(x\) 函数（积分常数对 \(y\) 来说是常数，但对 \(x\) 来说可以是任意函数）。

第三步用 \(v_x = -u_y = 2y\)。从我们刚得到的 \(v = 2xy + g(x)\) 算 \(v_x = 2y + g'(x)\)，要它等于 \(2y\)，所以 \(g'(x) = 0\)，\(g(x) = C\)（常数）。

结果：\(v(x, y) = 2xy + C\)。验证一下 \(f = u + iv = x^2 - y^2 + i(2xy + C) = z^2 + iC\)。果然解析！而且我们认出 \(z^2\) 来了：\((x + iy)^2 = x^2 - y^2 + 2ixy\)。

例 19.2.2. 求 \(u(x, y) = e^x \cos y\) 的调和共轭。

\(u_x = e^x \cos y\)，\(u_y = -e^x \sin y\)。

用 \(v_y = u_x = e^x \cos y\) 对 \(y\) 积分：\(v = e^x \sin y + g(x)\)。

用 \(v_x = -u_y = e^x \sin y\)。从 \(v = e^x \sin y + g(x)\) 算 \(v_x = e^x \sin y + g'(x)\)，要它等于 \(e^x \sin y\)，所以 \(g'(x) = 0\)。

结果：\(v = e^x \sin y + C\)。识别一下 \(f = u + iv = e^x \cos y + i e^x \sin y = e^z\)（取 \(C = 0\)）。再次完美。

关于"单连通"的提醒. 上面"积分与路径无关"用到了单连通性。如果区域不单连通，调和共轭可能不存在或多值。最经典的反例是 \(u = \ln|z| = \ln\sqrt{x^2 + y^2}\) 在 \(\Omega = \mathbb{C} \setminus \{0\}\) 上。它显然调和（19.1 节算过）。它的"形式上的"调和共轭是 \(v = \arg z\)——可这个东西多值！绕原点一圈，\(\arg z\) 增加 \(2\pi\)。所以在 \(\mathbb{C} \setminus \{0\}\) 上调和共轭不存在（在单值意义上）。要让它存在，就必须把区域剪开，比如剪掉负实轴使其单连通——这就是为什么对数函数总要选一个分支。

到这里我们应当郑重总结一下：

\[\boxed{\text{在单连通区域上, 一个调和函数等价于一个解析函数的实部.}}\]

这条等价关系是下面两节"继承性"的根。我们要把解析函数的好东西（积分性质、最值原理、唯一性）通过它取实部，搬到调和函数身上。

19.3 平均值性质

解析函数有一条惊人的性质：它在一点的值等于以那点为圆心的任何（包含在定义域里的）圆上的平均值。这是 Cauchy 积分公式的直接后果——把 Cauchy 公式

\[ f(z_0) = \frac{1}{2\pi i} \oint_{|z - z_0| = r} \frac{f(z)}{z - z_0}\, dz \]

用参数化 \(z = z_0 + r e^{i\theta}\)，\(dz = i r e^{i\theta}\, d\theta\) 写开，立刻得到

\[ f(z_0) = \frac{1}{2\pi} \int_0^{2\pi} f(z_0 + r e^{i\theta})\, d\theta. \]

这就是平均值性质。它说：解析函数在一点的值，恰好等于它在以该点为圆心的任意小圆上的平均。

把这条性质两边取实部，并把 \(f\) 写成 \(u + iv\)：

\[\boxed{u(z_0) = \frac{1}{2\pi} \int_0^{2\pi} u(z_0 + r e^{i\theta})\, d\theta.}\]

这就是调和函数的平均值性质。它说：调和函数在任一点的值，等于以该点为圆心的圆周上的平均。

注意我们其实只用到了"\(u\) 是某个解析函数的实部"，而没有用到 \(v\) 是谁。所以平均值性质只关于 \(u\) 本身，不依赖共轭的选取。那么对没有调和共轭的情况（多连通区域上的 \(u\)）平均值性质还成立吗？答：成立。因为平均值性质是一个局部性质——只要圆盘 \(\{|z - z_0| \leq r\}\) 包含在定义域内，那个圆盘本身就是单连通的，于是 \(u\) 在那个圆盘上有调和共轭，于是性质成立。结论：

\[ \boxed{\text{若 } u \text{ 在某开域上调和, 任取一个完全嵌入定义域的闭圆盘, 平均值性质就成立.}} \]

为什么这条性质如此重要？它给了我们一种"看穿内部"的能力：只要知道某圆周上的全部值，就知道圆心处的值。下面两节的最大值原理和唯一性，全是这条平均值性质的直接推论。

例 19.3.1（一道直接验算题）. 验证 \(u(x, y) = x^2 - y^2\) 在原点的平均值性质。\(u(0, 0) = 0\)。在半径 \(r\) 的圆上，\(u(r \cos\theta, r \sin\theta) = r^2 \cos^2\theta - r^2 \sin^2\theta = r^2 \cos 2\theta\)。它在 \([0, 2\pi]\) 上的平均

\[ \frac{1}{2\pi} \int_0^{2\pi} r^2 \cos 2\theta\, d\theta = 0. \]

等于 \(u(0, 0) = 0\)。完美。

例 19.3.2（更深的体味）. 验证 \(u = \ln|z|\) 在 \(z_0 = 1\)、半径 \(r = 1/2\) 的圆上的平均值性质。这个圆完全在 \(z \neq 0\) 的区域里，所以平均值应当成立。\(u(1) = 0\)。在圆 \(z = 1 + \tfrac{1}{2} e^{i\theta}\) 上，

\[ u = \ln\Big|1 + \tfrac{1}{2} e^{i\theta}\Big| = \tfrac{1}{2} \ln\Big(1 + \cos\theta + \tfrac{1}{4}\Big) = \tfrac{1}{2} \ln\Big(\tfrac{5}{4} + \cos\theta\Big). \]

它在 \([0, 2\pi]\) 上的平均要积出来当然不容易硬算，但一个聪明做法是：注意 \(\dfrac{1}{2\pi} \int_0^{2\pi} \ln|1 + a e^{i\theta}|\, d\theta = 0\) 对所有 \(|a| < 1\) 成立（这本身就是个调和函数平均值性质的应用——\(\ln|1 + a w|\) 是 \(w\) 的调和函数，在 \(w = 0\) 处值为 \(\ln 1 = 0\)）。这里 \(a = 1/2\) 满足要求。所以平均确实是 0。

19.4 最大模原理（调和版本）

解析函数的最大模原理告诉我们：非常值的解析函数 \(|f(z)|\) 在区域内部不取最大值。调和函数有一条更强、更对称的版本——不光不能取最大，连最小也不能取。

最大值原理（调和版）. 设 \(\Omega\) 是连通开区域，\(u\) 在 \(\Omega\) 上调和。如果 \(u\) 在 \(\Omega\) 内部某点 \(z_0\) 取到最大值，则 \(u\) 在整个 \(\Omega\) 上恒等于这个最大值。

证明几行就够。设 \(M = u(z_0)\) 是 \(\Omega\) 内的最大值。考虑集合 \(A = \{z \in \Omega : u(z) = M\}\)。我们要证 \(A = \Omega\)。

第一，\(A\) 是闭集（在 \(\Omega\) 的相对拓扑里）：因为 \(u\) 连续，\(\{u = M\}\) 是闭的。

第二，\(A\) 是开集：取 \(z_0 \in A\)。在 \(z_0\) 周围一个小圆盘 \(D\) 里用平均值性质——对任意小半径 \(r\)，\(M = u(z_0) = \dfrac{1}{2\pi} \int_0^{2\pi} u(z_0 + r e^{i\theta})\, d\theta\)。但 \(u \leq M\) 处处成立，所以这个平均要等于 \(M\)，唯一可能是 \(u\) 在整个圆周上恒等于 \(M\)。这对所有 \(0 < r \leq r_0\) 都成立，于是 \(u\) 在整个圆盘 \(D\) 上恒等于 \(M\)，于是 \(D \subset A\)。所以 \(A\) 开。

第三，\(A\) 非空（包含 \(z_0\)）。由 \(\Omega\) 的连通性，\(\Omega\) 不能被分成两个非空开闭集合的并，所以 \(A = \Omega\)。\(\square\)

把上面的论证用 \(-u\) 跑一遍，得到对称的最小值原理：调和函数也不能在内部取最小值（除非恒为常数）。

合在一起说：

\[\boxed{\text{非常值的调和函数, 在区域内部既不取最大值也不取最小值.}}\]

这条性质的物理意义十分直观。把 \(u\) 想成稳态温度场。如果某点温度严格高于周围所有点，那么热量必然从那点向外流——但稳态意味着不再有净热流，矛盾。所以"内部峰值"和"稳态"互不兼容。

例 19.4.1. 设 \(u\) 在闭单位圆盘 \(\overline{D}\) 上调和（含边界连续），且在边界 \(|z| = 1\) 上 \(0 \leq u \leq 5\)。问：\(u\) 在内部能取到 \(-1\) 吗？能取到 \(7\) 吗？

答：都不能。最大值在边界取，最大不超过 5；最小值也在边界取，最小不小于 0。所以 \(u\) 在整个闭圆盘上都满足 \(0 \leq u \leq 5\)。

19.5 唯一性

最大最小值原理立刻给出 Dirichlet 问题的唯一性。这件事说起来一句话：

唯一性定理. 设 \(\Omega\) 是有界连通开区域，\(u_1, u_2\) 在 \(\Omega\) 上调和、在 \(\overline{\Omega}\) 上连续，且在 \(\partial \Omega\) 上 \(u_1 = u_2\)。则 \(u_1 \equiv u_2\) 于 \(\overline{\Omega}\) 上。

证明用差函数 \(w = u_1 - u_2\)。\(w\) 也调和（Laplace 是线性算符），在边界上 \(w = 0\)，在 \(\overline{\Omega}\) 上连续。\(\overline{\Omega}\) 紧致，连续函数 \(w\) 在其上有最大值和最小值。由最大值原理，最大值要么在边界取，要么 \(w\) 是常数。由最小值原理同理。两种情况下都得到 \(w\) 的最大值与最小值都是 0（边界处取的），于是 \(w \equiv 0\)。\(\square\)

这条定理是 Dirichlet 问题"physically meaningful"的关键：你只要规定边界上的温度（势、流场……），内部的整个分布就唯一确定。这件事在数学上一点不显然——\(\Omega\) 内部有"无穷多个自由度"，怎么会被边界上"少一维的"信息完全决定？答案就是 Laplace 方程把内部和边界耦合得极紧。

注. "有界"假设非常关键。在无界区域上唯一性会失败。比如上半平面 \(y > 0\) 上，\(u_1 = 0\) 与 \(u_2 = y\) 都在实轴 \(y = 0\) 上为零，都调和，却显然不相等。要恢复唯一性，得在无穷远处加约束（如"\(u\) 有界"或"\(u\) 不增长太快"）。

19.6 圆盘上的 Dirichlet 问题：Poisson 核

有了平均值性质和唯一性，我们已经知道圆盘上 Dirichlet 问题的解（如果存在的话）是唯一的。现在剩一个问题：它存在吗？怎么把它写出来？对单位圆盘 \(D = \{|z| < 1\}\) 这个特殊情形，答案有一个非常漂亮的封闭式：Poisson 公式。

问题. 给定连续边界值 \(g : \partial D \to \mathbb{R}\)（可以认为 \(g\) 是变量 \(\theta\) 的函数 \(g(e^{i\theta})\)），求 \(u : \overline{D} \to \mathbb{R}\)，使得 \(u\) 在 \(D\) 内调和，在 \(\overline{D}\) 上连续，并且 \(u|_{\partial D} = g\)。

核心想法是这样的。如果我们找到一个"积分核" \(P_r(\theta)\) 使得：

\[ u(r e^{i\varphi}) = \frac{1}{2\pi} \int_0^{2\pi} P_r(\varphi - \theta)\, g(e^{i\theta})\, d\theta \]

对所有 \(r < 1\) 都给出 \(D\) 上的调和函数，且当 \(r \to 1\) 时上式收敛到 \(g(e^{i\varphi})\)，那问题就解决了。

怎么找出这个核？从 Cauchy 积分公式倒推。设 \(f\) 在 \(D\) 上解析、\(\overline{D}\) 上连续。Cauchy 公式给出

\[ f(z) = \frac{1}{2\pi i} \oint_{|w|=1} \frac{f(w)}{w - z}\, dw \qquad (|z| < 1). \]

这只用了 \(f\) 在边界上的值。但 Cauchy 公式给的是解析函数，我们想要调和函数。如果 \(g\) 是 \(f\) 的实部值，我们能恢复 \(\mathrm{Re}\, f\) 吗？

先做一个聪明的对称化。考虑"反演点" \(1/\bar z\)。它在 \(D\) 外（因为 \(|z| < 1\) 时 \(|1/\bar z| > 1\)），所以 \(\dfrac{f(w)}{w - 1/\bar z}\) 作为 \(w\) 的函数在 \(\overline{D}\) 上解析。由 Cauchy 定理：

\[ 0 = \frac{1}{2\pi i} \oint_{|w|=1} \frac{f(w)}{w - 1/\bar z}\, dw. \]

把这个 0 从 Cauchy 公式里减掉（不改变结果），得

\[ f(z) = \frac{1}{2\pi i} \oint_{|w|=1} f(w) \left[\frac{1}{w - z} - \frac{1}{w - 1/\bar z}\right] dw. \]

方括号里的两项做通分，可以化简成（计算细节留给读者，要点是利用 \(|w| = 1\) 即 \(\bar w = 1/w\)）一个对称、纯实的核。最终结果是：

\[ f(z) = \frac{1}{2\pi} \int_0^{2\pi} \frac{1 - |z|^2}{|w - z|^2}\, f(w)\, d\theta, \quad w = e^{i\theta}. \]

注意 \(\dfrac{1 - |z|^2}{|w - z|^2}\) 是实的、正的、与 \(z, w\) 都对称。两边取实部：

\[\boxed{u(z) = \frac{1}{2\pi} \int_0^{2\pi} \frac{1 - |z|^2}{|e^{i\theta} - z|^2}\, g(e^{i\theta})\, d\theta.}\]

这就是 Poisson 公式。中间那个核函数

\[ P(z, e^{i\theta}) = \frac{1 - |z|^2}{|e^{i\theta} - z|^2} \]

叫做 Poisson 核。设 \(z = r e^{i\varphi}\)，把分母展开（\(|e^{i\theta} - r e^{i\varphi}|^2 = 1 - 2r\cos(\theta - \varphi) + r^2\)）：

\[ P_r(\theta - \varphi) = \frac{1 - r^2}{1 - 2 r \cos(\theta - \varphi) + r^2}. \]

这个核的几个关键性质：

  1. \(P_r(\theta) > 0\)（因为 \(r < 1\) 使 \(1 - r^2 > 0\)，分母也正）。
  2. \(\dfrac{1}{2\pi} \int_0^{2\pi} P_r(\theta)\, d\theta = 1\)（取 \(g \equiv 1\)，常数函数 \(u \equiv 1\) 是合法解，回代得归一化）。
  3. 当 \(r \to 1\) 时，\(P_r(\theta)\) 集中到 \(\theta = 0\)：在 \(\theta \neq 0\) 的任何固定区间外它趋于 0，在 \(\theta = 0\) 处它趋于无穷。所以 \(P_r\) 是一个近似单位元 (approximate identity)，把卷积变成"边界值"。

这三条性质告诉你：Poisson 公式给出的 \(u\) 必定满足 \(u(r e^{i\varphi}) \to g(e^{i\varphi})\) 当 \(r \to 1\)（如果 \(g\) 连续）。这就完成了存在性证明。

例 19.6.1（最简单的边值）. 取 \(g(e^{i\theta}) = \cos\theta\)。求圆盘内的解。

聪明做法：注意 \(\cos\theta = \mathrm{Re}\, e^{i\theta}\)，所以 \(g\) 是 \(z = e^{i\theta}\) 在边界上的"\(x\) 坐标"。但 \(u(x, y) = x\) 是调和的（线性函数），且在 \(\partial D\) 上等于 \(\cos\theta\)。由唯一性，这就是解。所以

\[ u(x, y) = x = r \cos\varphi, \qquad z = r e^{i\varphi}. \]

用 Poisson 积分硬算也能得同样结果，但显然这条捷径更优雅。教训：对多项式或低阶谐波边值，直接猜调和函数比积分快得多。

例 19.6.2（不连续边值）. 取 \(g(e^{i\theta}) = 1\) 当 \(0 < \theta < \pi\)，\(g = 0\) 当 \(\pi < \theta < 2\pi\)（上半圆为 1、下半圆为 0）。求圆盘内的解。

用 Poisson 公式：

\[ u(z) = \frac{1}{2\pi} \int_0^{\pi} \frac{1 - r^2}{1 - 2r \cos(\theta - \varphi) + r^2}\, d\theta. \]

这个积分实际上能求出来。一个常用的几何技巧：Poisson 积分

\[ \frac{1}{2\pi} \int_a^b P_r(\theta - \varphi)\, d\theta = \frac{\omega(b) - \omega(a)}{2\pi}, \]

其中 \(\omega(\theta)\) 是从内部点 \(z = r e^{i\varphi}\) 看到边界点 \(e^{i\theta}\) 的辐角（"视角"）。所以

\[ u(z) = \frac{\omega(\pi) - \omega(0)}{2\pi}. \]

这就是著名的"夹角公式"——在圆盘内部任一点 \(z\)，把上半圆"看成"一段弧，从 \(z\) 看这段弧所张的角度除以 \(2\pi\)，就是该点的解。这件事的物理意义美妙：解的等高线就是从两端点 \(e^{i \cdot 0} = 1\) 与 \(e^{i\pi} = -1\) 张成等角的曲线，也就是过 \(\pm 1\) 的圆弧族。

本节最后我们指出一句话：Poisson 公式只对单位圆盘直接写出。对其他形状的区域，原则上可以用共形映射（这也是为什么 Riemann 映射定理那么重要——它说任何单连通区域都共形等价于单位圆盘）把问题化到圆盘上。但这是 21、22 章的事；本章止步于圆盘。

19.7 Schwarz 反射原理

本节讲一条把"半域上的调和函数"延拓为"全域上"的对称性原理。它最初来自 H. A. Schwarz 在 19 世纪研究 Riemann 映射时的发现。

反射原理（调和版）. 设 \(u\) 在上半平面 \(H = \{z : \mathrm{Im}\, z > 0\}\) 上调和、在 \(\overline{H}\) 上连续，且在实轴上 \(u(x, 0) = 0\) 处处成立。把它"反射"到下半平面：定义

\[ \tilde u(x, y) = \begin{cases} u(x, y), & y > 0, \\ 0, & y = 0, \\ -u(x, -y), & y < 0. \end{cases} \]

则 \(\tilde u\) 在整个 \(\mathbb{C}\) 上调和。

为什么？下半平面的定义是为了让上下两块在实轴上"光滑接合"。先看连续性——上面的 \(u(x, 0) = 0\) 假设保证了过实轴时 \(\tilde u\) 取 0，连续。但这还不够，要它真的调和必须证明跨过实轴时 Laplace 方程仍然成立。

关键想法是用平均值性质来"判定"调和性：可以证明，一个连续函数 \(\tilde u\) 在某域上调和，当且仅当它在该域上对充分小的圆都满足平均值性质（这件事本身需要点工夫，本质上是 Weyl 引理的一个特殊情形，详见附录 A）。所以我们去验证 \(\tilde u\) 在实轴上的点也满足平均值性质——上下半平面上的点显然满足，因为 \(u\) 与 \(-u(\cdot, -y)\) 各自调和。

取实轴上一点 \(x_0\)，半径 \(r\) 的圆盘 \(D_r(x_0)\)。圆周积分是

\[ \frac{1}{2\pi} \int_0^{2\pi} \tilde u(x_0 + r\cos\theta, r\sin\theta)\, d\theta. \]

把它分成上下两半积分。上半 \(\theta \in (0, \pi)\) 上是 \(u(\ldots, r\sin\theta)\)（\(\sin\theta > 0\)）；下半 \(\theta \in (\pi, 2\pi)\) 上是 \(-u(\ldots, -r\sin\theta) = -u(x_0 + r\cos\theta, r |\sin\theta|)\)（因为 \(\sin\theta < 0\) 时 \(-\sin\theta > 0\)）。换元 \(\theta \to 2\pi - \theta\) 把下半的积分变量翻过来，发现它恰好等于上半积分的相反数——两者抵消，平均值是 0。而 \(\tilde u(x_0, 0) = 0\)。所以平均值性质成立。\(\square\)

这个论证的精髓是对称性：构造里的负号 \(-u(x, -y)\) 是为了让"下半的贡献"恰好抵消"上半的贡献"，从而平均值是边界上的 0 值。如果改成 \(+u(x, -y)\)（偶反射），那么平均值是两倍上半的平均，不等于边界 0 值——那种反射要求边界上的法向导数 \(\partial_y u = 0\)（Neumann 条件）才能成立。

反射原理（解析版）. 上面的调和反射可以"提升"为解析反射。设 \(f = u + iv\) 在 \(H\) 上解析，在 \(\overline{H}\) 上连续，且在实轴上 \(f\) 取实值（即 \(v(x, 0) = 0\)）。则定义

\[ \tilde f(z) = \begin{cases} f(z), & z \in H, \\ \overline{f(\bar z)}, & z \in \overline{H}^c. \end{cases} \]

给出 \(\mathbb{C}\) 上的解析延拓。

这条解析版的反射原理威力极大。比如，知道一个函数在上半圆 \(\{|z| \leq 1, \mathrm{Im}\, z > 0\}\) 上解析、在直径 \([-1, 1]\) 上取实值，就能立刻把它延拓到整个单位圆盘上。这种"由对称性获得延拓"的思想在椭圆函数、模形式、Riemann 映射等许多领域反复出现。

例 19.7.1. 设 \(u(x, y) = y\) 在上半平面调和，在 \(y = 0\) 上为 0。其反射延拓 \(\tilde u(x, y) = y\)（在下半平面 \(\tilde u = -u(x, -y) = -(-y) = y\)）。所以反射后还是 \(u(x, y) = y\)，本来就在全平面调和。这是个平凡例子，验证一下程序。

例 19.7.2. 设 \(u(x, y) = \mathrm{Im}\, z^2 = 2xy\) 在上半平面调和，在 \(y = 0\) 上为 0。反射给 \(\tilde u(x, y) = -u(x, -y) = -2x(-y) = 2xy\)。又得到原来的 \(2xy\)，全平面调和。两个例子都"自然延拓自己"——这是因为 \(\mathrm{Im}\, z^2\) 已经天然定义在全平面上、在实轴上为零；反射只是把那一部分的连续延拓写出来。真正非平凡的例子是当 \(u\) 在上半平面的奇异边界条件下定义时的反射。

19.8 应用例题：稳态二维热传导

把本章理论合到一起，做一个具体的物理问题。

问题. 一块半径为 1 的圆形金属薄片，一半边界（上半圆 \(0 < \theta < \pi\)）维持温度 \(100\ \mathrm{°C}\)，另一半（下半圆 \(\pi < \theta < 2\pi\)）维持 \(0\ \mathrm{°C}\)。问稳态时片内任一点的温度 \(T(x, y)\)？

翻译成数学. 稳态意味着 \(T\) 不再随时间变化，热传导方程 \(T_t = k \Delta T\) 退化为 \(\Delta T = 0\)。所以 \(T\) 在 \(D\) 内调和；边界条件正是上面 19.6 节例 19.6.2 那样的不连续边界值（差一个因子 100）。

求解. 由该例的结论，

\[ T(z) = 100 \cdot \frac{\omega}{2\pi}, \]

其中 \(\omega\) 是从内部点 \(z\) 看上半圆所张的角度（弧度）。这个角度可以用解析几何算出来。设 \(z = (x, y)\)，则它到边界点 \(1\) 与 \(-1\) 的方向分别是 \(\arg(1 - z)\) 与 \(\arg(-1 - z)\)。所张角是

\[ \omega = \arg(-1 - z) - \arg(1 - z) = \arg\frac{-1 - z}{1 - z}. \]

所以

\[\boxed{T(x, y) = \frac{100}{\pi} \arg\frac{-1 - z}{1 - z} = \frac{100}{\pi} \arctan\frac{2y}{x^2 + y^2 - 1} \text{(适当选择分支)}.}\]

第二个等号是把 \(\arg\) 用 \(\arctan\) 写出来；分支选择确保 \(0 \leq T \leq 100\)。

具体值. 在圆心 \(z = 0\) 处，\(\arg\dfrac{-1}{1} = \arg(-1) = \pi\)，所以 \(T(0, 0) = \dfrac{100}{\pi} \cdot \pi = 100/2 = 50\ \mathrm{°C}\)。这正是平均值性质告诉我们的——圆心的温度应当是边界温度的平均。

在 \(z = i/2\)（即 \((0, 0.5)\)，靠近热边界）处，

\[ \arg\frac{-1 - i/2}{1 - i/2} = \arg\frac{(-1 - i/2)(1 + i/2)}{|1 - i/2|^2}. \]

分子 \((-1)(1) + (-1)(i/2) + (-i/2)(1) + (-i/2)(i/2) = -1 - i/2 - i/2 + 1/4 = -3/4 - i\)。所以 \(\arg(-3/4 - i)\)，这是第三象限的角度，\(\arctan\dfrac{-1}{-3/4} = \arctan(4/3) \approx 0.93\)，加上 \(\pi\) 得到 \(\approx 4.07\) 弧度。所以 \(T(0, 0.5) \approx \dfrac{100}{\pi} \cdot 4.07 \approx 130\)。等等，这超过 100 了！

等等，让我重新查一下分支。我们要的 \(\omega\) 是从 \(z\) 看上半圆所张的角，应当在 \(0\) 到 \(2\pi\) 之间，而靠近上半圆时应当接近 \(2\pi\)（不是接近 0）；靠近下半圆时应当接近 0。重看公式：从 \(z\) 看 \(1\)（弧的起点）与 \(-1\)（弧的终点），逆时针扫过的角度。在圆心 \(z = 0\)，这个扫过角是从 \(\arg 1 = 0\) 走到 \(\arg(-1) = \pi\)，扫过 \(\pi\)。所以 \(\omega(0, 0) = \pi\)，对应 \(T = 50\)，对头。

但我把 \(\arg\) 比例系数写错了。正确的公式是 \(T = \dfrac{100}{\pi} \omega\)，其中 \(\omega\) 实际上在 \(0\) 到 \(\pi\) 之间（所以 \(T\) 在 0 到 100 之间）。而 \(\omega = \pi - \arg\dfrac{1 - z}{-1 - z}\)，或者更对称地

\[ \omega(z) = \arg(1 - z) - \arg(-1 - z) \pmod{2\pi}, \]

选择使其在 \([0, \pi]\) 内。具体到 \(z = i/2\)：\(\arg(1 - i/2) = \arctan(-1/2) \approx -0.464\)，\(\arg(-1 - i/2) = \pi + \arctan(1/2) \approx 3.605\)。差值 \(\approx -4.07\)，加上 \(2\pi\) 得 \(\approx 2.21\)。\(T \approx \dfrac{100}{\pi} \cdot 2.21 \approx 70.4\ \mathrm{°C}\)。靠近热边界，确实比中心 50 高，符合直觉。

这个例子展示了"理论 \(\to\) 公式 \(\to\) 数值"的完整链条。注意几件事：

  1. 边界条件不连续（在 \(\pm 1\) 处跳跃），但在内部 \(T\) 是解析光滑的——这是 Laplace 方程的"光滑化"特性的典型展现。
  2. 解的等温线是那些"从 \((1, 0)\) 与 \((-1, 0)\) 张成等角的曲线"，也就是过这两点的圆弧族。圆心 \((0, 0)\) 的等温线是经过 \((1, 0)\) 与 \((-1, 0)\) 的水平直径——\(50\ \mathrm{°C}\) 等温线。
  3. 对称性：\(T(x, y) + T(x, -y) = 100\)，因为问题本身在 \(y \to -y\) 下变成 \(0 \leftrightarrow 100\) 的互换。

更复杂的例题. 如果你把上半圆温度换成 \(g(\theta)\)（任意连续函数），同样的 Poisson 公式给出

\[ T(z) = \frac{1}{2\pi} \int_0^{2\pi} \frac{1 - |z|^2}{|e^{i\theta} - z|^2}\, g(e^{i\theta})\, d\theta. \]

哪怕 \(g\) 是奇形怪状的函数——只要连续——稳态温度就由这条积分公式完全决定。

下一章我们会看到，解析函数和调和函数还有更深的联系：傅里叶变换、Laplace 变换、量子力学波函数、信号处理——所有这些"现代分析"的基本工具，背后的统一线索都是复函数的性质。我们现在要把视野从"二维稳态"打开到"频域 vs 时域"的整个结构。
