第 2 章 复函数、极限与连续

第 1 章我们认识了复数本身——既作为代数对象（怎么算），又作为几何对象（在一张平面上长什么样）。本章把研究对象从单个复数升级到复函数 \(f : \mathbb{C} \to \mathbb{C}\)：把每个复数 \(z\) 送到另一个复数 \(w = f(z)\)。

这看起来好像只是把实变函数 \(f : \mathbb{R} \to \mathbb{R}\) 平移到复数版本，可是麻烦立刻冒出来：实变函数的图像是一条平面曲线，能直接画；复函数的图像是一个四维对象（输入两个实参 \(x, y\)、输出两个实参 \(u, v\)），完全画不出来。我们必须改变"看"函数的方式——不再画一张图，而是对照两张平面（\(z\)-平面和 \(w\)-平面），观察一些典型曲线被 \(f\) 送过去之后变成什么样。这就是 2.1 到 2.4 节的主题。

到 2.5 节我们正式引入复函数的极限。表面上 \(\lim_{z \to z_0} f(z) = L\) 的定义跟单变量极限几乎一字不差，可是 \(z\) 在复平面上是个二维点——它可以从任何方向逼近 \(z_0\)，方向有无穷多个！这种"全方向"性质和多元微积分里的二维极限非常相似，并且为下一章的柯西-黎曼方程埋下伏笔。2.7、2.8 节讲连续性，并教你用"双路径法"判别复极限不存在。本章不长，但每一节都为后续做铺垫。

2.1 复函数：定义域与值域

给定一个集合 \(\Omega \subset \mathbb{C}\)，一条复函数是一个映射

\[ f : \Omega \to \mathbb{C}, \qquad z \mapsto f(z). \]

其中 \(\Omega\) 叫做 \(f\) 的定义域（domain），是 \(f\) 接受的"输入复数"组成的集合；\(\Omega\) 在 \(f\) 下的像

\[ f(\Omega) = \{f(z) : z \in \Omega\} \subset \mathbb{C} \]

叫做 \(f\) 的值域（range，或像 image）。

这跟实变函数 \(f : A \to \mathbb{R}\) 的格式完全一致，只是把"实数"换成"复数"。来几个具体例子：

  • \(f(z) = z^2\)：定义域是整个 \(\mathbb{C}\)，值域是整个 \(\mathbb{C}\)（每个复数都有平方根，所以每个复数 \(w\) 都能写成某个 \(z^2\)）。
  • \(f(z) = 1/z\)：定义域是 \(\mathbb{C} \setminus \{0\}\)（\(z = 0\) 不能除），值域也是 \(\mathbb{C} \setminus \{0\}\)（结果不可能是 0）。
  • \(f(z) = \dfrac{z + 1}{z^2 + 1}\)：定义域是 \(\mathbb{C} \setminus \{i, -i\}\)（分母在 \(z = \pm i\) 处为 0）。
  • \(f(z) = e^z\)：定义域是 \(\mathbb{C}\)，值域是 \(\mathbb{C} \setminus \{0\}\)（指数函数永不取 0，理由 2.4 节会看到）。
  • \(f(z) = \mathrm{Re}\, z\)：定义域是 \(\mathbb{C}\)，但值域是 \(\mathbb{R}\)——它把每个复数映到它的实部，输出始终是实数。这种"出口在实轴"的复函数也很常见。

请你注意："定义域"在复分析里通常被默认为复平面的某个开集（往往还加上"连通"——这种又开又连通的集合在复分析里叫做区域，常用希腊字母 \(\Omega\) 或 \(D\) 表示）。这并不是吹毛求疵——为了让"复可微"有意义，我们必须能在每个内点的附近从各个方向逼近，而这正是开集所保证的。本章用 \(\Omega\) 这个记号作为暗示，但不强行限定。

2.2 \(z\)-平面与 \(w\)-平面：为什么不能"画图"

这一小节讲个关键认识——为什么我们说"复函数没法画图"。

实变函数 \(y = f(x)\)，它把实数 \(x\) 送到实数 \(y\)。我们画图的方法是：横轴放 \(x\)、纵轴放 \(y\)，每个 \(x\) 对应纵向高度 \(f(x)\)，画出一条曲线。所以图像是 \(\mathbb{R} \times \mathbb{R} = \mathbb{R}^2\) 里的一个一维子集——容易画，容易看。

复函数 \(w = f(z)\) 呢？输入 \(z = x + i y\) 是 \(\mathbb{R}^2\) 里一个点，需要两个实数描述；输出 \(w = u + i v\) 也是 \(\mathbb{R}^2\) 里一个点，又需要两个实数。"图像"就是 \(\mathbb{R}^2 \times \mathbb{R}^2 = \mathbb{R}^4\) 里的一个二维子集。四维空间里的二维曲面——人类的视觉无法直接处理。

那怎么办？我们换一种"看"法。把两张平面分开摆：

  • 左边一张叫做 \(z\)-平面，标 \((x, y)\) 坐标，代表输入复数 \(z\) 所在的平面；
  • 右边一张叫做 \(w\)-平面，标 \((u, v)\) 坐标，代表输出复数 \(w\) 所在的平面。

现在 \(f\) 是从左边平面到右边平面的"搬运"映射。要"看见" \(f\)，最有效的办法是：在 \(z\)-平面上画几条有特点的曲线（典型选择：竖直线 \(x = \text{const}\)、水平线 \(y = \text{const}\)，或圆 \(|z| = \text{const}\)、射线 \(\arg z = \text{const}\)），然后看 \(f\) 把这些曲线送到 \(w\)-平面的什么形状上去。这种"看图法"叫做映射图或image diagram。它没法把整个函数装进一张图里，但它能把这个函数的几何性格呈现出来。

用这种"网格映射"法看典型函数（\(z^2\)、\(1/z\)、\(e^z\)）的几何，是 2.4 节的主要工作。在那之前，我们先把每个复函数拆成"实部 + 虚部"两个实值函数——这是另一种把复函数"实化"的标准技巧。

2.3 实部与虚部：把 \(f(z)\) 写成 \(u(x, y) + i v(x, y)\)

每个复数 \(z\) 都能拆成 \(z = x + i y\)，每个复数 \(f(z)\) 也能拆成实部加虚部。把这两件事并起来，复函数 \(f\) 就总能写成

\[ \boxed{f(z) = u(x, y) + i v(x, y),} \]

其中 \(u, v\) 都是两个实变量 \((x, y)\) 的实值函数。\(u\) 叫做 \(f\) 的实部，\(v\) 叫做虚部。

这种拆分，看起来只是个记号游戏，可它非常有用——它把"复函数"变成了"两个二元实函数"，从而让你可以用熟悉的多元微积分（偏导、链式法则、二维极限）来分析它。第 3 章引入复可微和柯西-黎曼方程时，这种"实-虚拆分"将是核心工具。

来几个具体例子，体会一下拆分的具体样子：

例 2.1（拆 \(f(z) = z^2\)）　设 \(z = x + i y\)。计算：

\[ z^2 = (x + i y)^2 = x^2 + 2 i x y + i^2 y^2 = (x^2 - y^2) + i (2 x y). \]

所以 \(u(x, y) = x^2 - y^2\)、\(v(x, y) = 2 x y\)。这两个二元函数你应该都认得——\(x^2 - y^2\) 是双曲面（在等高线图里就是一族双曲线 \(x^2 - y^2 = c\)），\(2 x y\) 也是双曲面（等高线 \(2 x y = c\) 也是双曲线，但和前者旋转了 45°）。

例 2.2（拆 \(f(z) = 1/z\)）　设 \(z = x + i y \neq 0\)。要把 \(1/z\) 实化，使用 1.2 节那条"分母乘共轭"的招：

\[ \frac{1}{z} = \frac{\bar z}{z \bar z} = \frac{x - i y}{x^2 + y^2} = \frac{x}{x^2 + y^2} + i \cdot \frac{-y}{x^2 + y^2}. \]

所以 \(u(x, y) = \dfrac{x}{x^2 + y^2}\)，\(v(x, y) = \dfrac{-y}{x^2 + y^2}\)。注意分母 \(x^2 + y^2 = |z|^2\) 在 \(z \neq 0\) 时不为零，所以 \(u, v\) 都在 \((x, y) \neq (0, 0)\) 的区域上有定义。

例 2.3（拆 \(f(z) = e^z\)）　设 \(z = x + i y\)。我们坚持复指数 \(e^z\) 应当满足 \(e^{a + b} = e^a e^b\)，所以

\[ e^z = e^{x + i y} = e^x \cdot e^{i y} = e^x (\cos y + i \sin y), \]

用了 1.5 节的欧拉公式。所以 \(u(x, y) = e^x \cos y\)、\(v(x, y) = e^x \sin y\)。

从这条公式还可以读出复指数函数的几个重要事实：

  • \(|e^z| = e^x = e^{\mathrm{Re}\, z}\)：复指数的模只看实部，永远 \(> 0\)。所以 \(e^z\) 永远不为 0——这对应"值域不含 0"的事实。
  • \(\arg(e^z) = y = \mathrm{Im}\, z\)：复指数的幅角就是输入的虚部。
  • 复指数不是单射！\(e^{z + 2 i\pi} = e^z \cdot e^{2 i\pi} = e^z \cdot 1 = e^z\)，所以 \(e^z\) 关于 \(z\) 有周期 \(2 i\pi\)。这跟实指数函数 \(e^x\) 严格单调的好性质截然不同——这一点将在我们定义复对数 \(\log z\) 时酿成大麻烦（一个值有无穷多原像）。

所以从拆分中你看到，复指数虽然形式上长得跟实指数像，但它带来的几何/分析性格大不相同。下一节我们就系统地用映射图来观察 \(z^2\)、\(1/z\)、\(e^z\) 这三个最经典复函数的几何样子。

2.4 三个经典复函数的几何

本节把 \(f(z) = z^2\)、\(1/z\)、\(e^z\) 在 2.2 节那种"映射图"语言下逐个看清楚。每个例子都按下面的程序：先在 \(z\)-平面上选一组有特点的曲线，再用 2.3 的实部虚部公式（或极坐标）算它们在 \(w\)-平面上的像。

2.4.1 \(f(z) = z^2\)：把角度翻倍

用极坐标看 \(z^2\) 最直观。设 \(z = r e^{i\theta}\)，那么 \(w = z^2 = r^2 e^{i \cdot 2\theta}\)。一句话：模平方、角度翻倍。

用这个事实立刻可以看出：

  • 射线被映成射线：在 \(z\)-平面上以原点出发、角度 \(\theta_0\) 的射线，被映到 \(w\)-平面上以原点出发、角度 \(2\theta_0\) 的射线。比如正实轴（\(\theta = 0\)）映到自己；正虚轴（\(\theta = \pi/2\)）映到负实轴（\(2\theta = \pi\)）；第一象限的对角线（\(\theta = \pi/4\)）映到正虚轴。
  • 圆被映成圆：圆 \(|z| = r_0\) 被映到圆 \(|w| = r_0^2\)。比如单位圆 \(|z| = 1\) 映到自己；半径 2 的圆映到半径 4 的圆。
  • 角度不守恒，被翻倍：在 \(z\)-平面上从 \(\theta = 0\) 转到 \(\theta = \pi/2\)（90°）的弧段，在 \(w\)-平面上对应从 \(2\theta = 0\) 转到 \(2\theta = \pi\)（180°）的弧段——角度翻了一倍。所以整个上半平面 \(\{z : \mathrm{Im}\, z > 0\} = \{0 < \arg z < \pi\}\)被 \(z^2\) 映成整个被去除正实轴的复平面 \(\{0 < \arg w < 2\pi\}\)（也就是除去正实轴的所有 \(w\)）。"角度翻倍"让上半平面这块"角度宽 \(\pi\)" 的区域，覆盖到了"角度宽 \(2\pi\)" 的整个去切口平面。

更进一步，用直角坐标看：实部公式 \(u = x^2 - y^2\)、虚部 \(v = 2 x y\)。

  • 水平线 \(y = c\)（\(c > 0\)）的像：参数化 \(z = x + i c\)（\(x\) 跑遍 \(\mathbb{R}\)），\(u = x^2 - c^2\)、\(v = 2 c x\)。消去 \(x\)：从 \(v = 2 c x\) 得 \(x = v/(2 c)\)，代入 \(u\)：\(u = v^2/(4 c^2) - c^2\)。这是一条开口朝右、顶点在 \((-c^2, 0)\) 的抛物线。
  • 竖直线 \(x = a\)（\(a > 0\)）的像：\(z = a + i y\)，\(u = a^2 - y^2\)、\(v = 2 a y\)，类似消去 \(y\)：\(u = -v^2/(4 a^2) + a^2\)，是开口朝左、顶点在 \((a^2, 0)\) 的抛物线。

所以 \(z^2\) 把 \(z\)-平面上的"水平线 + 竖直线"网格，变成 \(w\)-平面上"两族正交的抛物线"。这两族抛物线在每一交点处都互相垂直——这是 \(z^2\) 的"共形性质"的一个具体体现，将在第 3 章后部、第 7 章再讨论。

2.4.2 \(f(z) = 1/z\)：反演

同样用极坐标。设 \(z = r e^{i\theta}\)（\(z \neq 0\)），那么

\[ \frac{1}{z} = \frac{1}{r e^{i\theta}} = \frac{1}{r} \cdot e^{-i\theta}. \]

一句话：模取倒数、幅角变号。"幅角变号"在几何上等于"关于实轴翻转"——也就是取共轭那个动作。所以 \(1/z\) 的几何可以分两步：

第一步，关于单位圆做反演：把每个点 \(z\) 拉到（或推到）射线 \(\arg z\) 上离原点 \(1/r\) 的位置。短的变长、长的变短，单位圆上的点不动（半径 1 不变）。

第二步，关于实轴翻转：把所得点关于实轴对称。

总结：

  • 圆 \(|z| = r_0\) 映到圆 \(|w| = 1/r_0\)。单位圆映到自己；半径 2 的圆映到半径 1/2 的圆。
  • 射线 \(\arg z = \theta_0\)（除原点外）映到射线 \(\arg w = -\theta_0\)。正实轴映到自己；正虚轴映到负虚轴。
  • 整个外部 \(|z| > 1\) 被翻到整个内部 \(|w| < 1\)（再带一个上下镜像）；圆盘 \(|z| < 1\) 翻到外部 \(|w| > 1\)。

更进一步：\(1/z\) 这个变换有一个非常美的性质——它把所有圆与直线映成圆与直线。具体说："不经过原点的圆"映成"不经过原点的圆"；"经过原点的圆"映成"不经过原点的直线"；"不经过原点的直线"映成"经过原点的圆"；"经过原点的直线"映成"经过原点的直线"。这种"圆-线 ⇄ 圆-线" 的整齐变换叫做分式线性变换，它的一般形式是 \(w = (a z + b)/(c z + d)\)，是第 7 章共形几何的主角之一。

2.4.3 \(f(z) = e^z\)：把竖条变扇形

用 2.3 的拆分 \(e^z = e^x (\cos y + i \sin y)\)，注意此时 \(|w| = e^x\)、\(\arg w = y\)。

所以 \(z = x + i y\) 在 \(z\)-平面上的几何位置（直角坐标 \((x, y)\)）翻译成 \(w\)-平面上的几何位置（极坐标 \((|w|, \arg w) = (e^x, y)\)）。这正好是从直角坐标到极坐标的翻译器。

具体例子：

  • 水平线 \(y = c\)：\(\arg w = c\)（常数），所以像是以原点为端点、辐角 \(c\) 的射线。\(x\) 从 \(-\infty\) 跑到 \(+\infty\) 时 \(|w| = e^x\) 从 0 跑到 \(\infty\)（永不到 0、永不到 \(\infty\)）——所以严格说不包括端点。
  • 竖直线 \(x = a\)：\(|w| = e^a\)（常数），所以像是以原点为心、半径 \(e^a\) 的圆。\(y\) 跑遍 \(\mathbb{R}\) 时 \(\arg w = y\) 跑遍 \(\mathbb{R}\)，把这个圆无穷多次绕过去——也就是这条像是无穷多次重叠的同一个圆周。
  • 水平条带 \(\{z : -\pi < \mathrm{Im}\, z < \pi\}\)（横宽 \(2\pi\) 的一条无限长水平条）：被映成整个去除负实轴的复平面 \(\{w : -\pi < \arg w < \pi, w \neq 0\}\)。这是个"切了一刀"的复平面。下一步：宽度 \(2 i\pi\) 的水平条恰好是 \(e^z\) 的一个"基本周期带"——它被一一映到这块切口平面，且这映射是双射。

这是个非常重要的图像。"竖条 ⇄ 切口平面"的对应将在第 4 章定义复对数 \(\log z\) 时再现：复对数想"反"复指数，但因为 \(e^z\) 的周期性，反不出来一个好函数——只能反出一个"主分支"，对应于把整张复平面切一刀（去掉负实轴）。这个"切一刀"，就是为了避开周期性带来的多值。

2.5 复极限：\(\lim_{z \to z_0} f(z) = L\)

开始正式讲分析。复极限的定义跟你在微积分里背过的实极限定义几乎一字不差——只是把"实数绝对值"换成"复数模"，其他什么都不变。

设 \(f\) 在 \(z_0\) 的某个去心邻域上有定义（也就是在 \(\{z : 0 < |z - z_0| < \delta_0\}\) 上有定义，对某个 \(\delta_0 > 0\)）。我们说 \(f\) 在 \(z_0\) 处的极限等于 \(L\)，记作 \(\lim_{z \to z_0} f(z) = L\)，如果对任意 \(\varepsilon > 0\)，存在 \(\delta > 0\) 使

\[ \boxed{0 < |z - z_0| < \delta \quad \Longrightarrow \quad |f(z) - L| < \varepsilon.} \]

用大白话说："只要 \(z\) 离 \(z_0\) 足够近（但不等于 \(z_0\)），\(f(z)\) 就离 \(L\) 任意近"。这跟 \(\mathbb{R}\) 上的极限定义一模一样——只是这里"离…近"用复数模度量，是平面上的欧氏距离，不是数轴上的距离。

来验证一个最简单的：\(\lim_{z \to z_0} z = z_0\)。直接用定义：取 \(\delta = \varepsilon\)，则 \(|z - z_0| < \delta\) 显然推出 \(|f(z) - z_0| = |z - z_0| < \varepsilon = \varepsilon\) ✓。

类似地容易验证 \(\lim_{z \to z_0}(c z + d) = c z_0 + d\)、\(\lim_{z \to z_0} z^2 = z_0^2\)（用 \(|z^2 - z_0^2| = |z - z_0| \cdot |z + z_0|\) 估）等等。所有"代数运算的极限算法"——和、积、商的极限——都跟实变情形形式上一样，证明也照搬。具体而言：

极限的算法定理：若 \(\lim_{z \to z_0} f(z) = A\)、\(\lim_{z \to z_0} g(z) = B\)，则

\[ \lim_{z \to z_0}(f + g)(z) = A + B, \qquad \lim_{z \to z_0}(f g)(z) = A B, \qquad \lim_{z \to z_0}\frac{f(z)}{g(z)} = \frac{A}{B} \quad (\text{若 } B \neq 0). \]

证明跟实变情形完全平行——把所有"绝对值"理解为"复数模"即可。我们不重复，留给你做练习。

2.6 复极限的"全方向"特性

定义虽然形式上一样，但实质有重要不同——这件事必须现在就强调，否则你下一章一碰柯西-黎曼方程会立刻跌跤。

实变极限 \(\lim_{x \to x_0} f(x) = L\) 的"\(x \to x_0\)"是一维逼近：\(x\) 只能从左边或右边沿数轴走过来。所以"左极限 = 右极限 = 极限"。两条路径，验证起来不烦。

复极限 \(\lim_{z \to z_0} f(z) = L\) 的"\(z \to z_0\)"是二维逼近：\(z\) 在复平面上一个去心邻域内，可以从任何方向逼近 \(z_0\)。是水平地从右边来？竖直地从上面来？沿一条 45° 的射线走？沿一条螺旋曲线？沿一条心电图似的复杂曲线？——每一条都是合法的逼近方式，都得给出同一个 \(L\)。无穷多种逼近方式，全部要协调一致。

这个"全方向"性质，是复极限的特质，也是复分析"刚强"性的根源。它带来两件事：

第一，判别"极限不存在"非常容易：只需找出两条不同路径，沿它们 \(f(z)\) 趋于不同的极限值——一旦找到这种"路径不一致"，极限就不能存在。这是 2.8 节"双路径法"的内容。

第二，判别"极限存在"非常严苛：你必须保证所有路径都给出同一个值。仅仅"沿水平、沿竖直方向都得到 \(L\)" 是不够的——还得沿曲线、沿对角线…一切都给出 \(L\)。所以"复极限存在"是个比"实极限存在"强得多的要求。

这个"二维全方向"性质和你在多元微积分里见过的二元极限 \(\lim_{(x, y) \to (x_0, y_0)} g(x, y)\) 完全相似——事实上完全等价。具体地：把 \(f(z) = u(x, y) + i v(x, y)\)，则

\[ \lim_{z \to z_0} f(z) = L \quad\Longleftrightarrow\quad \lim_{(x, y) \to (x_0, y_0)} u(x, y) = \mathrm{Re}\, L \;\text{且}\; \lim_{(x, y) \to (x_0, y_0)} v(x, y) = \mathrm{Im}\, L. \]

其中右边的两个极限，都是普通的二元实极限。所以"复极限存在/等于"完全可以化归为"两个二元实极限存在/等于"。这条等价是把复分析里的极限问题"翻译"成多元微积分问题的桥梁。

这个"全方向"特性还埋下了一个重要伏笔：将来定义复可微（也就是 \(f'(z_0) = \lim_{z \to z_0}\dfrac{f(z) - f(z_0)}{z - z_0}\)）时，"\(z \to z_0\)" 仍然是从所有方向、要求同一极限。这意味着复可微比 \(\mathbb{R}^2\) 上的可微强得多——它要求所有方向上的"差商"协调一致，从而逼出柯西-黎曼方程。第 3 章你会亲眼看见这个推导。

2.7 连续性

连续性的定义沿用实变情形。函数 \(f\) 在点 \(z_0\) 处连续，如果

\[ \lim_{z \to z_0} f(z) = f(z_0). \]

这有三层含义：(i) \(f\) 在 \(z_0\) 处有定义；(ii) 极限 \(\lim_{z \to z_0} f(z)\) 存在；(iii) 该极限恰好等于 \(f(z_0)\)。三件事缺一不可。

函数 \(f\) 在集合 \(\Omega\) 上连续，是说它在 \(\Omega\) 的每一点都连续。

用 \(\varepsilon\)-\(\delta\) 翻译："对任意 \(\varepsilon > 0\)，存在 \(\delta > 0\)，使 \(|z - z_0| < \delta\) 推出 \(|f(z) - f(z_0)| < \varepsilon\)"——这跟"\(\lim_{z \to z_0} f(z) = f(z_0)\)" 是一回事，只是没有"去心"的限制了（因为现在 \(z = z_0\) 时不等式 \(|f(z_0) - f(z_0)| = 0 < \varepsilon\) 平凡成立）。

把 2.5 节的极限算法定理直接套用，立刻得到连续函数的算法：连续函数的和、差、积、商（分母不为零处）都仍然连续；连续函数的复合也连续。具体地：

  • 多项式 \(P(z) = a_n z^n + a_{n-1} z^{n-1} + \cdots + a_0\)（系数 \(a_i \in \mathbb{C}\)）在整个 \(\mathbb{C}\) 上连续。
  • 有理函数 \(R(z) = P(z)/Q(z)\) 在 \(Q(z) \neq 0\) 的所有 \(z\) 处连续；分母为 0 的地方称为 \(R\) 的极点，将在第 5 章详细讨论。
  • 复指数 \(e^z\) 在整个 \(\mathbb{C}\) 上连续——理由：\(e^z = e^x \cos y + i e^x \sin y\)，实部、虚部都是 \((x, y)\) 的二元连续函数（连续函数复合 + 乘积），所以 \(e^z\) 连续。
  • 共轭函数 \(z \mapsto \bar z\) 在整个 \(\mathbb{C}\) 上连续：因为 \(|\bar z - \bar z_0| = |\overline{z - z_0}| = |z - z_0|\)，取 \(\delta = \varepsilon\) 立刻成立。
  • 模函数 \(z \mapsto |z|\) 在整个 \(\mathbb{C}\) 上连续（这是一个 \(\mathbb{C} \to \mathbb{R}_{\geq 0}\) 的实值函数）：用反向三角不等式 \(\big||z| - |z_0|\big| \leq |z - z_0|\)，取 \(\delta = \varepsilon\) 立成。
  • 实部、虚部 \(z \mapsto \mathrm{Re}\, z\)、\(z \mapsto \mathrm{Im}\, z\) 在整个 \(\mathbb{C}\) 上连续：因为 \(|\mathrm{Re}\, z - \mathrm{Re}\, z_0| \leq |z - z_0|\)，同样取 \(\delta = \varepsilon\)。

所以从这些"原子"出发，再用算法定理（和、积、商、复合）大量构造，几乎一切初等复函数都是连续的——只在分母为零、对数函数的支点等"奇异点"附近会出问题。

例 2.4（一个连续性的具体验证）　证明 \(f(z) = z \cdot \bar z = |z|^2\) 在整个 \(\mathbb{C}\) 上连续。

解：方法一（用算法）：\(z\)（恒等映射）连续；\(\bar z\) 连续（上面说过）；连续函数的乘积连续。所以 \(z \bar z = |z|^2\) 连续。

方法二（直接用 \(\varepsilon\)-\(\delta\)）：固定 \(z_0\)。要证 \(|z|^2 - |z_0|^2\) 在 \(|z - z_0|\) 小时小。用平方差分解：

\[ ||z|^2 - |z_0|^2| = ||z| - |z_0|| \cdot ||z| + |z_0|| \leq |z - z_0| \cdot (|z| + |z_0|). \]

第一步用了反向三角不等式。第二步控制 \(|z| \leq |z_0| + |z - z_0|\)，所以 \(|z| + |z_0| \leq 2 |z_0| + |z - z_0|\)。给定 \(\varepsilon > 0\)，先要求 \(|z - z_0| < \delta_1 := 1\)，于是 \(|z| + |z_0| \leq 2|z_0| + 1\)；再要求 \(|z - z_0| < \delta_2 := \dfrac{\varepsilon}{2|z_0| + 1}\)，那么

\[ ||z|^2 - |z_0|^2| \leq |z - z_0| \cdot (2|z_0| + 1) < \varepsilon. \]

取 \(\delta = \min(\delta_1, \delta_2)\) 即可。两种方法都给出连续性。注意第二种方法是练 \(\varepsilon\)-\(\delta\) 思维的好机会——估计的核心招数（用反向三角、用 \(|z| \leq |z_0| + |z - z_0|\) 控制）是复分析里的"标配技巧"，下一章证明微分时还要反复用。

2.8 双路径法：判别极限不存在

2.6 节提到，复极限的"全方向"特性让"判别极限不存在"变得相对容易——只要找出两条路径，沿它们 \(f(z)\) 趋于不同的极限，那么 \(\lim_{z \to z_0} f(z)\) 就不存在。这种技巧在多元微积分里你大概见过（叫"双路径法"），用在复分析里几乎一字不变。

原理表述：

若沿路径 \(\gamma_1\) 逼近 \(z_0\) 时 \(f(z) \to A\)，沿路径 \(\gamma_2\) 逼近时 \(f(z) \to B\)，且 \(A \neq B\)，则 \(\lim_{z \to z_0} f(z)\) 不存在。

这是因为如果极限存在并等于 \(L\)，那么沿任何路径都得趋于 \(L\)，特别地沿 \(\gamma_1\) 和 \(\gamma_2\) 都得是 \(L\)，矛盾。

来看几个具体例子。

例 2.5（\(\bar z / z\) 在 0 处）　考虑 \(\displaystyle\lim_{z \to 0}\frac{\bar z}{z}\)。问极限是否存在？

解：用极坐标 \(z = r e^{i\theta}\)，则 \(\bar z = r e^{-i\theta}\)，比值

\[ \frac{\bar z}{z} = \frac{r e^{-i\theta}}{r e^{i\theta}} = e^{-2 i \theta}. \]

关键：当 \(z \to 0\)（即 \(r \to 0\)），结果不依赖于 \(r\)——它只看 \(\theta\)。

  • 沿正实轴趋向 0（\(\theta = 0\)）：\(\bar z / z = e^0 = 1\)。
  • 沿正虚轴趋向 0（\(\theta = \pi/2\)）：\(\bar z / z = e^{-i\pi} = -1\)。
  • 沿对角线（\(\theta = \pi/4\)）趋向 0：\(\bar z / z = e^{-i\pi/2} = -i\)。

三条不同方向，三个不同极限：1，\(-1\)，\(-i\)。所以 \(\lim_{z \to 0} \bar z / z\) 不存在。

这件事很值得品味——\(\bar z / z\) 在 0 附近有界（始终模为 1，因为 \(|\bar z / z| = 1\) 对所有 \(z \neq 0\)），不发散，但极限就是不存在。它只是在每个方向上"卡"在不同的圆周点上，没有共同的极限点。这种现象在实变里也有，但在复分析里更普遍——因为方向多。

例 2.6（\(\mathrm{Re}(z) / z\) 在 0 处）　考虑 \(\displaystyle\lim_{z \to 0}\frac{\mathrm{Re}\, z}{z}\)。

解：写 \(z = r e^{i\theta}\)，则 \(\mathrm{Re}\, z = r \cos\theta\)，比值

\[ \frac{\mathrm{Re}\, z}{z} = \frac{r\cos\theta}{r e^{i\theta}} = \cos\theta \cdot e^{-i\theta}. \]

同样不依赖 \(r\)。沿不同方向：

  • 正实轴（\(\theta = 0\)）：\(\cos 0 \cdot e^0 = 1\)。
  • 正虚轴（\(\theta = \pi/2\)）：\(\cos(\pi/2) \cdot e^{-i\pi/2} = 0\)。

已经够了——两条路径给出 1 和 0，不一致。所以极限不存在。

例 2.7（\((\mathrm{Re}\, z)^2 / |z|\) 在 0 处）　考虑 \(\displaystyle\lim_{z \to 0}\frac{(\mathrm{Re}\, z)^2}{|z|}\)。

解：\(z = r e^{i\theta}\)，\(\mathrm{Re}\, z = r\cos\theta\)，\(|z| = r\)，比值

\[ \frac{(\mathrm{Re}\, z)^2}{|z|} = \frac{r^2\cos^2\theta}{r} = r \cos^2\theta. \]

当 \(r \to 0\) 时，无论 \(\theta\) 是多少，\(r\cos^2\theta \to 0\)（因为 \(|\cos^2\theta| \leq 1\) 有界）。所以这次所有路径都给出 0；极限存在，等于 0。

这一例和前两例的反差非常说明问题：单凭"几条路径试出某个一致值"不能断定极限存在；要证存在，要么所有方向算清楚（如这里 \(\cos^2\theta\) 有界），要么用 \(\varepsilon\)-\(\delta\) 严格估计。

用 \(\varepsilon\)-\(\delta\) 验证一下例 2.7：要使 \(|f(z) - 0| < \varepsilon\)，注意

\[ \left|\frac{(\mathrm{Re}\, z)^2}{|z|}\right| = \frac{(\mathrm{Re}\, z)^2}{|z|} \leq \frac{|z|^2}{|z|} = |z|, \]

因为 \(|\mathrm{Re}\, z| \leq |z|\)。所以取 \(\delta = \varepsilon\) 即可。

例 2.8（一个稍微复杂的例子）　考虑 \(\displaystyle\lim_{z \to 0}\frac{x y}{x^2 + y^2}\)，其中 \(z = x + i y\)。

解：（这是个实值函数，把 \(z\) 写成 \((x, y)\) 处理。）沿 \(x\) 轴（\(y = 0\)）：\(f = 0/(x^2) = 0\)，极限 0。沿 \(y\) 轴（\(x = 0\)）：\(f = 0\)，极限 0。沿对角线 \(y = x\)：\(f = x^2/(2 x^2) = 1/2\)，极限 \(1/2\)。三条路径给出两个不同的值 0 和 \(1/2\)，所以极限不存在。

这是多元微积分里的经典例子，照搬到复分析里仍然有效——因为复极限本质上就是 \(\mathbb{R}^2\) 上的二维极限。下一章引入复可微时，类似的"路径不一致"将变成强有力的工具——它会逼出柯西-黎曼方程，从而把"复可微"刻画成 \(u, v\) 偏导满足的两条特殊方程。

本章到此结束。我们已经把"复函数"看清楚了：它的定义、它怎么被画（其实是不画，而是看映射图），它的实部虚部拆分，它在三个经典例子（\(z^2\)、\(1/z\)、\(e^z\)）下的几何样子，以及它的极限和连续性。下一章是复分析真正"开光"的地方——我们要定义复可微。表面上跟实可微一样，可一旦把"全方向"的复极限要求带进去，事情就完全变了样：复可微比实可微强得多，强大到足以推出整个复分析的核心定理。具体来说，我们将看到柯西-黎曼方程从这一极限要求里"自然涌出"，看到复可微推出"调和性"，看到任何复可微函数都自动是解析的（无穷次可导，能展开成幂级数）——这些都是实变里完全没有的奇迹。准备好。
