第 6 章 多值函数与分支切割

上一章末尾我们看到一个反复出现的问题：复对数和复幂这两个对象天然"多值"——同一个 \(z\) 对应无穷多个 \(\log z\)，差 \(2\pi i\) 的整数倍。我们用"主值 \(\mathrm{Log}\, z\)"把多值压成单值，代价是被迫沿负实轴剪开一道"切割"，让函数在切割两侧不连续。

这种"剪开"操作不是 \(\log\) 独有的。任何"反复绕一个点必然多值"的函数——比如 \(\sqrt{z}\)、\(\sqrt{z^2 - 1}\)、\(\log(z^2 + 1)\)、\(\arctan z\)——都会面临同样的问题。本章要把这件事彻底讲清：什么样的点会引起多值（叫"分支点"）；如何用"分支切割"让函数局部单值；同一函数为什么允许有不同的切割方案；不同方案下函数的具体值如何不同；以及当我们绕过切割时函数值会怎样"跳跃"。最后我们花一节简单介绍黎曼面——它是把多值函数变成单值函数的最优雅方式。

本章和之前的章节稍有不同：它不是在"加一个新工具"，而是在"清理一片已经积累下来的混乱"。读完这章，再回头看 \(\log z\)、\(z^a\)、\(\sqrt{z^2 - 1}\)，你应该能闭眼说清"我现在站在哪个分支上、为什么允许我这样做、过线时会发生什么"。这些都将在第 7 章（留数）和第 8 章（围道积分）中频繁用到——尤其当被积函数带分支点时。

6.1 多值函数的本质

从 \(\log z\) 入手。我们已经知道 \(\log z = \ln |z| + i \arg z\)，而 \(\arg z\) 多值——任何两个值差 \(2\pi\) 的整数倍。所以 \(\log z\) 多值，差 \(2\pi i\) 的整数倍。

抽象地讲，"多值"的意思是：一个"函数"\(f\) 把 \(z\) 不是映成一个数，而是映成一个集合。这种"函数"严格来说不是高中意义上的函数——但因为它太自然出现在复分析里，我们沿用 \(\log z\) 的记号，同时心里清楚它代表的是无穷多个值的集合。

多值的根源是路径相关性。设想从 \(z_0 = 1\) 开始，连续地"走" \(z\) 沿一条路径绕到 \(z_0\) 附近的某个 \(z\)，并沿途连续地选择 \(\log z\) 的某个值（也就是连续选 \(\arg z\)）。如果路径不绕过原点，绕回起点时，连续延拓回到的 \(\log\) 值就是出发时那个值，没事。但如果路径绕过原点一圈（比如沿单位圆从 \(1\) 一路到 \(1\) 走一周），那么辐角会从 0 连续地增大到 \(2\pi\)——回到起点 \(z = 1\) 时，"连续延拓的"\(\log\) 值变成了 \(\ln 1 + i \cdot 2\pi = 2\pi i\)，不等于出发时的 \(0\)。函数值经过一圈"提升"了 \(2\pi i\)。

这件事——"沿闭合路径连续延拓后函数值不一定回到原值"——就是多值现象的本质。一个函数如果在每一条路径上都"回得来"，它就真的是单值的；只要存在某种闭合路径让它回不来，它就多值。

一个具体的视觉化。把 \(\log z\) 想象成一个"螺旋楼梯"：站在 \(z = 1\) 处的"地面"上，\(\log z = 0\)；沿单位圆逆时针走一圈，回到 \(z = 1\) 时你已经爬到了"二楼"，\(\log z = 2\pi i\)；再走一圈，三楼，\(\log z = 4\pi i\)；以此类推。顺时针走的话，下楼。每"一层"对应 \(\log z\) 的一个分支，\(z\) 平面上的同一个点 \(z = 1\) 对应螺旋楼梯上每一层都有的"同一个位置"。这架"楼梯"就是 \(\log z\) 的黎曼面，6.9 节会再讲。

在我们没法在两维复平面上把"楼层"画出来的情况下，我们退而求其次——剪开一刀，逼着自己不准绕原点，于是就回不到不同的层。这就是分支切割的几何动机。

6.2 分支点

稍微正式一点定义"什么样的点是麻烦的根源"。

定义（分支点）。设 \(f\) 是一个多值函数。点 \(z_0\) 叫 \(f\) 的分支点（branch point），如果对任意以 \(z_0\) 为内部点的足够小的闭合曲线 \(\gamma\)，沿 \(\gamma\) 连续延拓 \(f\) 一圈后，函数值不回到原值。

换言之，分支点是这样一个点：你不可能在它周围任何小邻域里把多值函数"压"成单值。无论怎么剪都剪不掉它的多值性，除非把切割从它出发延伸出去把"绕它"这件事禁止。

"\(\infty\) 是不是分支点"是一个常被忽略的问题。我们用一个简洁的定义统一处理：

无穷远点是 \(f\) 的分支点，当且仅当 \(f(1/w)\) 把 \(w = 0\) 当作分支点。

这是把 \(z = \infty\) 通过 \(z = 1/w\) "拉回到" \(w = 0\) 来检验的方法。

分支点不一定都"显然在某个特定位置"——某些函数有多个分支点（如 \(\sqrt{(z-1)(z+1)}\) 有 \(z = \pm 1\) 两个），某些有无穷多个（如 \(\log(\sin z)\) 在每个 \(z = k\pi\) 处都有）。本章主要讨论一两个分支点的情形；多个分支点是相同思想的累加。

6.3 例子：\(\log z\) 与 \(\sqrt{z}\) 的分支点

\(\log z\) 的分支点。我们已经见过：沿包围原点的闭合路径绕一圈，\(\log z\) 增加 \(2\pi i\)，不回原值。所以 \(z = 0\) 是分支点。

沿不包围原点的闭合路径绕一圈呢？\(\arg z\) 在路径上变化的净累计是 \(0\)（因为它没绕过 0），所以 \(\log z\) 也回到原值。所以这样的路径不会触发多值。这印证了"分支点是 0"的说法——只有"绕 0"才会出问题。

无穷远点呢？换 \(w = 1/z\)，那么 \(\log z = -\log w\)。\(\log w\) 在 \(w = 0\) 是分支点，所以 \(-\log w\) 在 \(w = 0\) 也是分支点，所以 \(\log z\) 在 \(z = \infty\) 也是分支点。

结论：\(\log z\) 的分支点是 \(\{0, \infty\}\)。这意味着任何分支切割都必须从 \(0\) 延伸到 \(\infty\)（沿某条曲线），把"在 \(0\) 和 \(\infty\) 间绕"这件事禁止。

\(\sqrt{z}\) 的分支点。\(\sqrt{z} = z^{1/2} = e^{\frac{1}{2} \log z} = |z|^{1/2} e^{i \arg z / 2}\)。绕原点一圈，\(\arg z\) 增加 \(2\pi\)，所以 \(\arg z / 2\) 增加 \(\pi\)，所以 \(e^{i \arg z / 2}\) 变成 \(e^{i\pi}\) 倍 = \(-1\) 倍。所以 \(\sqrt{z}\) 在绕原点一圈后变成原来的相反数。

再绕一圈呢？又乘 \((-1)\)，回到原值。所以 \(\sqrt{z}\) 是双值函数：每个非零 \(z\) 对应两个值（互为相反数），绕 0 一圈"切换分支"，绕 0 两圈才"回到原分支"。

\(z = 0\) 是分支点。同理 \(z = \infty\) 也是（同样换 \(w = 1/z\) 检验）。\(\sqrt{z}\) 的分支点也是 \(\{0, \infty\}\)。任何切割从 0 延到 \(\infty\)。

注意区别：\(\log z\) 是无穷值的多值（每绕一圈加 \(2\pi i\)，永远不重复），而 \(\sqrt{z}\) 是有限二值的多值（每绕一圈翻号，绕两圈回归）。

例 6.3.1（\(\sqrt[n]{z}\) 的分支点和值数）。\(\sqrt[n]{z} = z^{1/n} = |z|^{1/n} e^{i \arg z / n}\)。绕 0 一圈，\(\arg z / n\) 增加 \(2\pi/n\)，所以 \(\sqrt[n]{z}\) 乘上 \(e^{2\pi i / n}\)。绕 \(n\) 圈才完全回归。所以 \(\sqrt[n]{z}\) 是 \(n\) 值多值函数，分支点 \(\{0, \infty\}\)。

例 6.3.2（\(\log(z - a)\) 的分支点）。在 \(\log z\) 里把 \(z\) 平移成 \(z - a\)，分支点也跟着平移。所以 \(\log(z-a)\) 的分支点是 \(\{a, \infty\}\)。绕过 \(z = a\) 一圈才会让函数值跳 \(2\pi i\)。

6.4 分支切割

有了分支点的概念，"分支切割"现在就是一句具体的指令：

分支切割 \(C\)是连接分支点（包括 \(\infty\)）的一条曲线，目的是把它从复平面上"剪掉"，然后在剩下的开区域 \(\mathbb{C} \setminus C\) 上选一个 \(\arg z\)（或别的辐角变量）的连续选择，让多值函数变成单值。

关键要求："剪开"后，剩下的开区域必须是单连通的——也就是说，不可能在剩余区域里画出一条绕过分支点的闭合曲线。如果还能绕，多值就还在，没有真正解决问题。

对 \(\log z\)，分支点是 0 和 \(\infty\)。任何从 0 出发延伸到 \(\infty\) 的曲线都能当切割——可以是负实轴 \((-\infty, 0]\)，可以是正实轴 \([0, \infty)\)，可以是从 0 出发沿任意辐角的射线，甚至可以是一条弯弯曲曲的螺线（只要不自交、不闭合）。每种切割对应 \(\log z\) 的一个特定"分支"。

对 \(\sqrt{z}\)，同理。任意一条从 0 到 \(\infty\) 的简单曲线都可以做切割。

对带多个有限分支点的函数（如 \(\sqrt{(z-a)(z-b)}\)），切割方式更灵活——可以连接 \(a, b\) 与 \(\infty\) 的不同子集。这是 6.8 节的主题。

6.5 主分支 \(\mathrm{Log}\, z\)：切割沿负实轴

把 5.10 节学过的"主值对数"放回这个新框架里看一遍：\(\mathrm{Log}\, z = \ln |z| + i \mathrm{Arg}\, z\)，其中 \(\mathrm{Arg}\, z \in (-\pi, \pi]\)。

这相当于约定切割沿负实轴 \((-\infty, 0]\)。在 \(\mathbb{C} \setminus (-\infty, 0]\) 上，\(\mathrm{Arg}\, z\) 是连续单值的（取值范围 \((-\pi, \pi)\)），所以 \(\mathrm{Log}\, z\) 单值解析。

为什么主值约定切割是负实轴而不是正实轴？纯粹历史习惯——选负实轴让 \(\mathrm{Log}\) 在正实轴（最常用区域）上和实自然对数完全一致，连续地穿过正实轴不会出事。如果切割沿正实轴，那"\(\mathrm{Log}\, 1\) 是不是有定义"就成问题。所以历史上选了负实轴这个不太常用的方向。

这条切割上具体发生什么？取负实轴上一点 \(z_0 = -r\)（\(r > 0\)）。从上方靠近 \(z_0\)：\(z = -r + \epsilon i\)、\(\epsilon \to 0^+\)。\(\mathrm{Arg}\, z \to +\pi\)。所以 \(\mathrm{Log}\, z \to \ln r + i \pi\)。

从下方靠近 \(z_0\)：\(z = -r - \epsilon i\)、\(\epsilon \to 0^+\)。\(\mathrm{Arg}\, z \to -\pi\)。所以 \(\mathrm{Log}\, z \to \ln r - i \pi\)。

两个极限差 \(2\pi i\)。所以跨越切割时 \(\mathrm{Log}\) 值"跳" \(\pm 2\pi i\)。

这正是"切割"的代价——切割两侧的函数值不再连续，跳跃量等于函数绕过分支点一圈所累计的增量（这里是 \(2\pi i\)）。

6.6 替代分支：切割可以选别的曲线

重申一遍：对 \(\log z\)，负实轴只是一种选择。其他常见选择：

切割沿正实轴。约定 \(\arg z \in [0, 2\pi)\)。在 \(\mathbb{C} \setminus [0, \infty)\) 上 \(\log\) 单值。这种选择在做"绕原点的回路积分"时很方便——回路自然处于一个连通区域里，函数变化清晰。

切割沿正虚轴。约定 \(\arg z \in (-3\pi/2, \pi/2]\)（或别的等价范围）。在 \(\mathbb{C} \setminus \{i y : y \ge 0\}\) 上单值。

切割沿任意从 0 出发的射线 \(\arg z = \alpha\)。约定 \(\arg z \in (\alpha - 2\pi, \alpha]\)。在切除该射线的复平面上单值。

所有这些"分支"都是 \(\log z\) 的合法选择。它们彼此之间有什么关系？取任意两个分支 \(\log_1 z\)、\(\log_2 z\)，在它们公共的解析区域上，二者差 \(2\pi i\) 的整数倍——这个整数依据切割位置而定，可能在不同区域不同。

例 6.6.1（用沿正实轴切割的对数算 \(\log(-1)\)）。约定 \(\arg z \in [0, 2\pi)\)。\(\arg(-1) = \pi\)（这次唯一）。所以这一分支下 \(\log(-1) = 0 + i\pi = i\pi\)。和主值一致——巧合还是必然？

必然。因为 \(z = -1\) 不在切割上（切割是 \([0, \infty)\)，不含 \(-1\)），主值切割也是 \((-\infty, 0]\) 不含 \(-1\)（注意：主值切割不含 0，但严格来说包不包含 \(-1\) 取决于约定，按 \(\mathrm{Arg} \in (-\pi, \pi]\) 的约定，\(-1\) 在切割上而 \(\mathrm{Arg}(-1) = \pi\)）。对在两个分支共同解析区域里的点（比如 \(-1 + i\) 或 \(-1 - i\) 附近），两分支可能差 \(2\pi i\)。

例 6.6.2（在主值分支与"正实轴切割"分支下，\(z = -1 - i\) 的 \(\log\) 值差多少）。\(|-1-i| = \sqrt{2}\)。主值分支下 \(\mathrm{Arg}(-1-i) = -3\pi/4\)（在 \((-\pi, 0]\)），所以 \(\mathrm{Log}(-1-i) = \tfrac{1}{2}\ln 2 - i \cdot 3\pi/4\)。

"正实轴切割"分支下 \(\arg(-1-i) = 5\pi/4\)（在 \([0, 2\pi)\)），所以 \(\log(-1-i) = \tfrac{1}{2}\ln 2 + i \cdot 5\pi/4\)。

差：\(\tfrac{1}{2}\ln 2 + i \cdot 5\pi/4 - (\tfrac{1}{2}\ln 2 - i \cdot 3\pi/4) = i \cdot (5\pi/4 + 3\pi/4) = i \cdot 2\pi = 2\pi i\)。差 \(2\pi i\)，符合一般理论。

6.7 跨越切割时函数值的跳跃

选好切割后，函数在切割两侧的值有"跳跃"——这是切割的本质代价。我们把这件事一般化。

设 \(f\) 在 \(\mathbb{C} \setminus C\) 上是 \(F\) 的一个分支（\(F\) 是原始多值函数，\(C\) 是切割）。设 \(z_0\) 是切割 \(C\) 上的一个内部点。我们要计算的是

\[ \Delta f(z_0) := f(z_0^+) - f(z_0^-), \]

其中 \(z_0^+\) 是从切割"一侧"逼近 \(z_0\) 的极限，\(z_0^-\) 是从"另一侧"逼近的极限。这个差就叫做 \(f\) 在 \(z_0\) 的跳跃（jump）。

对 \(\mathrm{Log}\, z\) 沿负实轴切割（5.10 节）：跳跃 \(= 2\pi i\)（上方减下方）。

对 \(\sqrt{z}\) 沿负实轴切割（约定 \(\arg z \in (-\pi, \pi]\)）：

\[ \sqrt{z} = |z|^{1/2} e^{i \arg z / 2}. \]

取 \(z_0 = -r\)（\(r > 0\)）。从上方逼近 \(\arg z \to +\pi\)，所以 \(\sqrt{z} \to r^{1/2} e^{i\pi/2} = i \sqrt{r}\)。从下方逼近 \(\arg z \to -\pi\)，\(\sqrt{z} \to r^{1/2} e^{-i\pi/2} = -i \sqrt{r}\)。跳跃 \(= 2 i \sqrt{r}\)。

注意这次跳跃量依赖于 \(r\)（也就是依赖于切割上的具体位置），不像 \(\log\) 那样是常数 \(2\pi i\)。这是因为 \(\sqrt{z}\) 的"圈一圈"造成的不是"加常数"而是"乘常数 \(-1\)"——具体跳跃 = 函数值乘上 \((-1)\) 与原值之差 = \(-2 \sqrt{z}\)（在该侧）。

例 6.7.1（\(\log z\) 的另一种切割下跳跃）。把 \(\log z\) 切割沿正实轴（\(\arg z \in [0, 2\pi)\)）。在切割点 \(z_0 = r > 0\)：从上方（\(z = r + \epsilon i\)）逼近，\(\arg z \to 0^+\)；\(\log z \to \ln r + i \cdot 0 = \ln r\)。从下方（\(z = r - \epsilon i\)）逼近，\(\arg z \to 2\pi^-\)；\(\log z \to \ln r + i \cdot 2\pi\)。跳跃 \(= -2\pi i\)（上 - 下）。

所以跳跃量的大小都是 \(2\pi\)，但符号取决于切割方向。

这个"跳跃"在做围道积分时极重要。下一章我们会做"哑铃"型路径绕分支切割的积分（如 \(\int_0^1 x^a (1-x)^b dx\) 这类），那里"上下两侧的差"就是跳跃，是路径贡献的核心。

6.8 \(\sqrt{z^2 - 1}\)：两个分支点的情形

本节是本章的核心难点。函数 \(\sqrt{z^2 - 1} = \sqrt{(z-1)(z+1)}\) 在 \(z = +1\) 和 \(z = -1\) 各有一个分支点，外加可能的 \(z = \infty\)。我们要弄清楚：哪些点是分支点？切割可以怎么选？

分支点确认。绕 \(z = +1\) 一小圈（小圆 \(|z - 1| = \epsilon\)，\(\epsilon\) 很小，所以圆内不含 \(-1\)）：\((z - 1)\) 改变 \(2\pi\) 的辐角，\((z + 1)\) 改变 \(0\)（因为 \(-1\) 在圆外、不被绕过）。所以 \((z-1)(z+1)\) 总辐角增量 \(2\pi\)，\(\sqrt{(z-1)(z+1)}\) 增量 \(\pi\)，函数值乘 \((-1)\)。不回原值——所以 \(z = 1\) 是分支点。

同理 \(z = -1\) 也是分支点。

关键观察：绕同时包含 \(\pm 1\) 的大圈一周。\((z - 1)\) 改变 \(2\pi\)、\((z + 1)\) 也改变 \(2\pi\)，总改变 \(4\pi\)，\(\sqrt{}\) 改变 \(2\pi\)，函数值乘 \(e^{2\pi i} = 1\)——回到原值！所以"绕两个分支点同时一圈"是无害的。

这件事的几何含义重要：\(\infty\) 不一定是分支点。检验：换 \(w = 1/z\)，得 \(\sqrt{(1/w - 1)(1/w + 1)} = \sqrt{(1 - w)(1 + w)} / w = \sqrt{1 - w^2} / w\)。\(w = 0\) 处 \(\sqrt{1 - w^2}\) 在 \(w = 0\) 附近无问题（\(\sqrt{1 - 0} = 1\)，函数解析）。但有 \(1/w\) 因子，所以函数有极点，不是分支点。结论：\(\sqrt{z^2 - 1}\) 的分支点只有 \(\pm 1\)，不包括 \(\infty\)。

这就给了切割两种本质不同的方案：

方案 A：两段切割（"双切割"）。从 \(z = -1\) 沿负实轴到 \(-\infty\)；从 \(z = 1\) 沿正实轴到 \(+\infty\)。即切割 \(C_A = (-\infty, -1] \cup [1, +\infty)\)。

这里的逻辑：每段切割从一个有限分支点 \(\pm 1\) 延伸到 \(\infty\)。但由于 \(\infty\) 不是分支点，"两段都过 \(\infty\)" 在那里"消化掉"——绕一个大圈同时绕过 \(\pm 1\)（也就同时跨过两段切割）函数值不变，与"\(\infty\) 不是分支点"一致。

切割后的单值区域是 \(\mathbb{C} \setminus C_A\)，包含整个复平面去掉两条尾巴和两个端点。

方案 B：一段切割（"短切割"）。从 \(z = -1\) 直接到 \(z = +1\)，沿实轴线段 \([-1, +1]\)。这条切割只在两个有限分支点之间，不到 \(\infty\)。

这里的逻辑：切割连接两个分支点。绕这段切割的任何小圈都同时绕过 \(\pm 1\) 两个分支点——函数值无变化（如上所述）。绕只一个分支点（比如只绕 \(+1\)）的小圈则不可能不穿过切割。所以剩余区域里不存在"只绕一个分支点"的回路，多值被消除，函数单值。

切割后的单值区域是 \(\mathbb{C} \setminus [-1, 1]\)，是包含 \(\infty\) 的连通区域。

这两种方案都正确——都让函数在剩余区域上单值——但给的是不同的"分支"。它们之间的关系有点微妙。

例 6.8.1（在两种方案下，\(\sqrt{z^2 - 1}\) 在 \(z = 2\) 处的具体值）。\(z = 2\) 不在任何一种切割上（方案 A 的切割是 \((-\infty, -1] \cup [1, \infty)\)——含 \(2\)！所以方案 A 下 \(z = 2\) 在切割上，不能直接代）。我们换一个 \(z = 2 + 0.1 i\)（既不在 A 也不在 B 的切割上）。

方案 A：切割 \([1, \infty)\) 把 \(z = 2 + 0.1 i\) 隔在了切割上方。约定上方的辐角分支为：在 \(z\) 离 \(+1\) 的辐角 \(\arg(z - 1) \in (-\pi, \pi)\)（避开切割 \([1, \infty)\) 上的辐角 0）；在 \(z\) 离 \(-1\) 的辐角 \(\arg(z + 1) \in (-\pi, \pi)\)（避开切割 \((-\infty, -1]\) 上的辐角 \(\pi\)）……这件事方便起见交给具体计算。

让我们直接做方案 B 的计算。\(z = 2\)，\(z^2 - 1 = 3\)，所以 \(\sqrt{z^2 - 1} = \pm \sqrt{3}\)。具体取哪个分支，依赖于"主值"约定。一个常用约定：在 \(z\) 大于 \(1\) 的实数处取正值，即 \(\sqrt{z^2 - 1}|_{z = 2} = + \sqrt{3}\)。这就唯一定下了方案 B 下的分支。

例 6.8.2（方案 B 下绕切割的跳跃）。沿切割 \([-1, 1]\) 取一个内部点，比如 \(z_0 = 0\)。从上方逼近 \(z_0\)（\(z = i \epsilon\)、\(\epsilon \to 0^+\)）：\(z^2 - 1 = -\epsilon^2 - 1 \approx -1\)；\(\sqrt{-1}\) 在哪个分支？这取决于"\((z-1)(z+1)\) 的辐角的连续延拓"。

具体地，方案 B 下选择"在 \(z\) 大且实时取 \(+\sqrt{z^2-1}\)" 这一约定，从 \(z = 2\) 出发沿上半平面绕到 \(z = i \epsilon\)（不穿过切割）。沿途辐角的具体计算我们略——结果是 \(\sqrt{z^2-1}|_{z = 0^+ i \epsilon} = i\)（这个 \(i\) 而不是 \(-i\)）。

从下方逼近：同样办法（沿下半平面绕过去）给 \(\sqrt{z^2-1}|_{z = 0 - i \epsilon} = -i\)。

跳跃 \(= i - (-i) = 2i\)。

这件事对计算 \(\int_{-1}^{1} \sqrt{1 - x^2}\, dx\) 这类积分非常有用——经典的"哑铃围道"利用上下跳跃 \(2i\) 把实积分变成围道积分。

例 6.8.3（方案 A 下的切割含义不同）。方案 A 下，切割沿负实轴 \((-\infty, -1]\) 和正实轴 \([1, \infty)\)，区间 \((-1, 1)\) 不在切割上，所以函数在这里有定义。代入 \(z = 0\)：\(z^2 - 1 = -1\)，\(\sqrt{-1} = \pm i\)。哪一个？依"主值"约定，常常选"在 \(z = 0\) 处取 \(+i\)" 或类似规则。

方案 A 与方案 B 在重叠区域（比如上半平面 \(\mathrm{Im}\, z > 0\) 且 \(|z| < 1\)）上其实给同一个值或差一个负号——具体看分支选择。两套方案都"对"，但用途不同：方案 A 自然适合处理"\(z\) 沿实轴大值时的渐近行为"；方案 B 自然适合处理"\(\int_{-1}^{1}\) 这类有限实积分"。

6.9 黎曼面初探

分支切割的本质代价是不连续——切割两侧不连续。能不能想个办法让函数真的处处单值且连续？答案：黎曼面（Riemann surface）。

核心想法：把"复平面 + 分支切割"换成"多层堆叠的复平面"，每"绕一圈"就把读者从一层带到下一层。这样函数值就再也不"跳"——它跨越的不是跳跃，而是层次。

\(\log z\) 的黎曼面。无穷多层：每一层是去掉原点的复平面。第 \(k\) 层上 \(\log z\) 取值 \(\ln |z| + i (\arg z + 2k\pi)\)。各层在"原本切割"位置粘合：第 \(k\) 层切割上方接到第 \(k+1\) 层切割下方（无缝粘合）。

几何上像一架无限螺旋楼梯（在 6.1 节末尾我们已经画过这幅图）。\(\log z\) 在这架楼梯上是一个真正的单值连续函数，不需要任何切割。

\(\sqrt{z}\) 的黎曼面。两层：每层是去掉原点的复平面。第 1 层上 \(\sqrt{z} = + |z|^{1/2} e^{i \arg z / 2}\)（取主值），第 2 层是 \(-|z|^{1/2} e^{i \arg z / 2}\)。两层沿"切割"（如负实轴）粘合：第 1 层上方接第 2 层下方，第 1 层下方接第 2 层上方。这种粘合在三维空间里画不出来（必有自相交），但作为抽象的二维流形它存在。绕原点一圈从第 1 层到第 2 层，绕两圈回到第 1 层。

\(\sqrt{z^2 - 1}\) 的黎曼面。也是两层（\(\sqrt{}\) 自带"取负"对称性）。两层沿"切割"粘合，按方案 B 的切割（\([-1, 1]\)）想象：第 1 层和第 2 层沿这条线段相互交换。两层粘成一个"环面"般的紧致曲面（拓扑上是球面——简单情形）。

本节只是初探。完整的黎曼面理论需要更多准备——它属于复几何/代数几何的范畴，超出本书范围。但理念你应该把握：

  • 黎曼面是把"多值"问题彻底解决的正确几何工具。
  • 分支切割是黎曼面在二维复平面上的"投影"——把多层"压扁"到一层，代价是引入了切割上的不连续。
  • 实际计算时我们大都用"分支切割 + 主值"，因为容易写、容易算。但心里要清楚黎曼面是更深的真相。

6.10 在留数计算中处理分支函数

本节是为下一章（留数计算）做准备。当被积函数含有分支函数（\(\log z\)、\(\sqrt{z}\)、\(z^a\) 等）时，怎么处理切割？这里给两个常用模式。

模式 A：哑铃围道（处理 \(\int_a^b f(x) dx\) 中含分支因子）。比如 \(\int_0^1 x^a (1-x)^b dx\)（\(a, b > -1\)）。被积函数 \(z^a (1-z)^b\) 在复平面上有两个分支点 \(0\) 和 \(1\)，最自然的切割是 \([0, 1]\)（方案 B 风格）。围道：在切割上方贴近的"上半线段"+ 围两端的小圆 + 切割下方贴近的"下半线段"，组成一个"哑铃"或"狗骨头"形状。

沿这个围道，由 Cauchy 定理（被积函数在围道围出的区域里解析），积分 = 0；但围道由四段组成，把它们逐个写出，就把实积分表达成"上下两侧跳跃 \(\times\) 实积分"和"两端小圆贡献"的关系——后两者通常容易算。

关键的一步：上下两侧函数值的差 = 跳跃。对 \(z^a\) 沿正实轴切割 \([0, \infty)\)：上方 \(z = x\)、\(\arg z = 0\)、\(z^a = x^a\)；下方 \(z = x e^{2\pi i}\)、\(\arg z = 2\pi\)、\(z^a = x^a e^{2\pi i a}\)。差 \(x^a (1 - e^{2\pi i a})\)。这个差因子 \((1 - e^{2\pi i a})\) 让我们能把围道积分换成实积分乘一个因子。

模式 B：扇形围道（处理 \(\int_0^\infty f(x) dx\) 中含 \(\log\)）。比如 \(\int_0^\infty \dfrac{dx}{1 + x^3}\)。一个聪明的办法：考虑围道 = "正实轴 \([0, R]\) + \(R\) 的圆弧从 0 到某辐角 \(\alpha\) + 沿辐角 \(\alpha\) 的射线回到 0"。如果 \(\alpha = 2\pi/3\) 或别的让 \(z^3\) 变到原值的角度，那么沿射线的积分和沿正实轴的积分有一个显式关系，加上圆弧贡献（\(R \to \infty\) 时趋于 0）= 围道内极点的留数和。

这种"扇形"或"分数圈"围道避开了切割问题——只要围道完全在某个单值区域里，就不需要担心切割。

模式 C：含 \(\log\) 的对数因子技巧。比如 \(\int_0^\infty f(x) dx\) 当 \(f\) 是有理函数时可以考虑 \(\int f(z) \log z dz\)（用沿正实轴切割的 \(\log\)）。围道：正实轴上方大圆 + 大半圆 + 正实轴下方小圆 + 大半圆 → 在上下两侧 \(\log z\) 差 \(2\pi i\)，于是围道积分变成"\(2\pi i \int_0^\infty f(x) dx\)"加圆弧贡献。这把含 \(\log\) 的"辅助"积分用来取出原本不含 \(\log\) 的实积分。

例 6.10.1（一个具体计算的轮廓）。求 \(\int_0^\infty \dfrac{dx}{x^2 + 1}\)。这个积分大家都知道答案 \(\pi/2\)，但用模式 C（带 \(\log\)）来重做一遍，演示技巧。

考虑 \(\oint \dfrac{\log z}{z^2 + 1} dz\)，沿"键孔围道"：从 \(\epsilon\) 沿正实轴上方到 \(R\)，沿大圆到正实轴下方的 \(R\)（绕原点一圈），沿正实轴下方回到 \(\epsilon\)，沿小圆绕原点回到起点。\(\log z\) 沿正实轴切割（\(\arg z \in [0, 2\pi)\)）。

极点：\(z = \pm i\)。两极点的留数（用主值 \(\arg \pm i = \pi/2, 3\pi/2\)）：

\[ \mathrm{Res}_{z = i} \frac{\log z}{z^2 + 1} = \frac{\log i}{2 i} = \frac{i \pi/2}{2 i} = \frac{\pi}{4}. \] \[ \mathrm{Res}_{z = -i} \frac{\log z}{z^2 + 1} = \frac{\log(-i)}{-2i} = \frac{i \cdot 3\pi/2}{-2i} = -\frac{3\pi}{4}. \]

留数和 \(= \pi/4 - 3\pi/4 = -\pi/2\)。围道积分 \(= 2\pi i \cdot (-\pi/2) = -i \pi^2\)。

围道积分逐段：上方 \(\int_\epsilon^R \dfrac{\ln x}{x^2 + 1} dx\)；下方 \(-\int_\epsilon^R \dfrac{\ln x + 2\pi i}{x^2 + 1} dx\)（注意 \(\log\) 在下方多了 \(2\pi i\)，且方向反所以加负号）；大圆与小圆贡献 \(\to 0\)（\(R \to \infty\)、\(\epsilon \to 0\)）。

合并上下两段：\(\int_0^\infty \dfrac{\ln x - \ln x - 2\pi i}{x^2 + 1} dx = -2\pi i \int_0^\infty \dfrac{dx}{x^2 + 1}\)。设此积分 \(= I\)。等式：\(-2\pi i \cdot I = -i\pi^2\)，所以 \(I = \pi/2\)。✓

这套技巧的本质：用 \(\log\) 的"上下跳跃 \(2\pi i\)" 把不带 \(\log\) 的实积分 提取出来。这是处理一大类有理函数实积分的有效手法，不少在第 8 章会反复用到。

本章用一个比较抽象的话题（多值函数、分支切割、黎曼面）连接了上一章的具体计算（复对数、复幂）和下一章的实用技术（留数 + 围道）。读到这里，你应该具备的工具是：

  • 识别一个多值函数的分支点。
  • 选一个合理的切割让函数局部单值。
  • 计算函数跨越切割时的跳跃。
  • 知道在哪些围道积分里需要绕开/利用分支切割。
