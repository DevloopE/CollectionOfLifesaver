第 10 章 梯度与方向导数

偏导数告诉你"沿坐标轴方向" \(f\) 变化的速率。但"沿坐标轴"是任意人为的选择——为什么 \(x\)、\(y\) 比别的方向更特殊?它们没有,只不过我们用直角坐标系建立函数,所以坐标轴成了默认参考。现实里,你在山坡上常常想知道:"沿东偏北 30 度方向爬,坡有多陡?";化学反应里,你想知道把若干浓度同时小幅调整时反应速率的变化方向;机器学习里,你最关心的是参数往哪个方向走能让损失下降最快。这些情形要求的不是沿坐标轴的导数,而是沿任意方向的导数。这就是方向导数。

把所有偏导数打包成一个向量,就得到梯度。表面上,梯度只是把若干偏导数装进尖括号里,似乎只是个记号上的便利。但当你深入下去,会发现梯度有三条几何性质,让它成为多元微积分中最重要的对象之一,完全配得上一整章的篇幅:

  1. 沿任意单位方向 \(\mathbf{u}\) 的方向导数等于梯度与 \(\mathbf{u}\) 的点积。
  2. 梯度本身指向 \(f\) 在该点处增长最快的方向,而它的模 \(|\nabla f|\) 等于该方向上的(最大)变化率。
  3. 梯度与 \(f\) 的等高线(二维情形)或等值面(三维及以上情形)在每一点处垂直。

这第二条是多元微积分中最美的事实之一——它告诉你:函数最陡的方向是被它自己的偏导数完全决定的。我们会用整整两节的篇幅把它的几何含义、物理含义、和应用含义都细细说透。

10.1 方向导数

设 \(f(x, y)\) 是函数,\((a, b)\) 是定义域中一点,\(\mathbf{u} = \langle u_1, u_2\rangle\) 是单位向量(即 \(|\mathbf{u}| = 1\))。\(f\) 在 \((a, b)\) 沿 \(\mathbf{u}\) 的方向导数定义为

\[D_\mathbf{u} f(a, b) = \lim_{h \to 0}\frac{f(a + hu_1, b + hu_2) - f(a, b)}{h}.\]

把这条公式翻译成图像:你站在曲面上 \((a, b, f(a, b))\),朝水平面里指定的方向 \(\mathbf{u}\) 走一小段距离 \(h\)(\(h\) 是带符号的实数),你的脚下高度变化了多少?除以 \(h\),再让 \(h \to 0\),就得到"每单位距离的高度变化率"。所以方向导数是"沿 \(\mathbf{u}\) 方向看过去,曲面有多陡"的精确刻画。它的单位是"高度单位 / 长度单位",和坡度的单位完全一致。

顺带要明白,把 \(\mathbf{u}\) 限定为单位向量是有原因的,而且这个原因不是约定俗成。如果你把 \(\mathbf{u}\) 换成 \(2\mathbf{u}\)——同一个方向,但长度翻倍——那么沿这个新向量"走一步"实际上是沿原方向走两步,同样的 \(h\) 对应了两倍的位移,差商也跟着翻倍。但你没有换方向!这种"长度依赖"是不该出现的。要求单位向量,等于把"方向"和"长度"分开,确保方向导数只跟"方向"挂钩,不被"步长"污染。

偏导数其实是方向导数的特例。沿 \(\mathbf{u} = \langle 1, 0\rangle\) 的方向导数就是 \(f_x\);沿 \(\mathbf{u} = \langle 0, 1\rangle\) 就是 \(f_y\)。所以方向导数是偏导数的"无方向偏好"推广。

定义里那个极限不便于直接计算——每一题都让你算极限太累。好消息:如果 \(f\) 在 \((a, b)\) 可微(标准充分条件:偏导数连续),则方向导数有闭式公式:

\[\boxed{D_\mathbf{u} f(a, b) = f_x(a, b)\, u_1 + f_y(a, b)\, u_2.}\]

这是从第 8 章的线性近似立刻得来的:\(f(a + hu_1, b + hu_2) \approx f(a, b) + f_x(a, b)\cdot hu_1 + f_y(a, b)\cdot hu_2\)。代入定义,化简,取极限,就得上式。可微性保证了这个近似的误差是 \(o(h)\),所以除以 \(h\) 取极限时,误差归零,只剩下两项的线性组合。

看一道。\(f(x, y) = x^2 + xy + y^3\) 在 \((1, 2)\) 沿 \(\mathbf{u} = \langle 3/5, 4/5\rangle\) 的方向导数。

验证 \(|\mathbf{u}| = \sqrt{9/25 + 16/25} = \sqrt{25/25} = 1\),没问题。\(f_x = 2x + y\),\(f_x(1, 2) = 4\)。\(f_y = x + 3y^2\),\(f_y(1, 2) = 13\)。所以

\[D_\mathbf{u} f(1, 2) = 4 \cdot \tfrac{3}{5} + 13 \cdot \tfrac{4}{5} = \tfrac{12 + 52}{5} = \tfrac{64}{5} = 12.8.\]

这个数大约是 \(12.8\),意思是从 \((1, 2)\) 出发沿向量 \(\langle 3/5, 4/5\rangle\) 方向走每一个单位长度,函数值大约增加 \(12.8\)。

再看一道,带处理"非单位"方向的步骤。求 \(f(x, y) = e^{xy}\) 在 \((1, 0)\) 沿 \(\mathbf{v} = \langle 3, -4\rangle\) 的方向导数。

\(\mathbf{v}\) 不是单位向量(它的长度是 5),先把它单位化:\(\mathbf{u} = \mathbf{v}/|\mathbf{v}| = \langle 3/5, -4/5\rangle\)。\(f_x = ye^{xy}\),\(f_x(1, 0) = 0\);\(f_y = xe^{xy}\),\(f_y(1, 0) = 1\)。所以

\[D_\mathbf{u} f(1, 0) = 0 \cdot \tfrac{3}{5} + 1 \cdot (-\tfrac{4}{5}) = -\tfrac{4}{5}.\]

注意结果是负的——意思是沿 \(\mathbf{u}\) 这个方向走,函数值实际上在减少。这并不是错,只是说明 \(\mathbf{u}\) 指向的是"下坡"那一侧。

这里要把一个最容易栽跟头的细节讲明白:如果你忘了单位化,直接把 \(\langle 3, -4\rangle\) 代进公式,得到 \(0 \cdot 3 + 1 \cdot (-4) = -4\),整整大五倍——这是错的。题目要的是"每单位距离的变化率",而你算的是"沿 \(\mathbf{v}\) 走 \(\mathbf{v}\) 这么长的距离"的变化率。除以 \(|\mathbf{v}|\) 就修正回正确答案了。所以做这种题的第一步始终是检查向量是否单位化;不是,就先除以模长。

再看一道。\(f(x, y) = \ln(x^2 + y^2)\) 在 \((1, 1)\) 沿 \(\mathbf{u} = \langle 1/\sqrt{2}, 1/\sqrt{2}\rangle\) 的方向导数。\(f_x = \dfrac{2x}{x^2 + y^2}\) 在 \((1, 1)\) 是 \(1\);\(f_y = \dfrac{2y}{x^2 + y^2}\) 在 \((1, 1)\) 是 \(1\)。所以

\[D_\mathbf{u} f(1, 1) = 1\cdot\tfrac{1}{\sqrt{2}} + 1\cdot\tfrac{1}{\sqrt{2}} = \sqrt{2}.\]

这个 \(\sqrt{2}\) 不是巧合——\(\mathbf{u}\) 正好平分了 \(+x\) 和 \(+y\) 的方向,而 \(f_x = f_y = 1\) 又是对称的,所以两份贡献等量加起来。这暗示我们,在这一点 \(\mathbf{u}\) 指向的可能正是"上山最陡"的那个方向——下一节我们就会看到为什么。

10.2 梯度向量

把上一节的公式 \(D_\mathbf{u} f = f_x u_1 + f_y u_2\) 看作两个向量的点积:

\[D_\mathbf{u} f = \langle f_x, f_y\rangle \cdot \langle u_1, u_2\rangle.\]

所以方向导数等于一个固定的向量 \(\langle f_x, f_y\rangle\) 与方向 \(\mathbf{u}\) 的点积。这个固定的向量有它自己的名字:梯度。

\[\boxed{\nabla f = \left\langle \frac{\partial f}{\partial x},\ \frac{\partial f}{\partial y}\right\rangle.}\]

对三元函数:\(\nabla f = \langle f_x, f_y, f_z\rangle\)。一般地:\(n\) 元函数 \(f\) 的梯度是 \(\langle f_{x_1}, \ldots, f_{x_n}\rangle\),是 \(\mathbb{R}^n\) 中的向量。读作 "del \(f\)" 或 "grad \(f\)"。倒三角 \(\nabla\) 这个符号叫"nabla",来自一种希伯来竖琴的形状。

有了梯度,方向导数公式可以写得相当干净:

\[\boxed{D_\mathbf{u} f = \nabla f \cdot \mathbf{u} \quad (|\mathbf{u}| = 1).}\]

看一道。求 \(f(x, y, z) = x^2 yz + e^z\) 的梯度。\(f_x = 2xyz\),\(f_y = x^2 z\),\(f_z = x^2 y + e^z\)。所以 \(\nabla f = \langle 2xyz,\ x^2 z,\ x^2 y + e^z\rangle\)。这就是把三个偏导数装进尖括号里,没有额外动作。

再看一道。求 \(f(x, y) = x^3 - 3xy + y^3\) 在 \((2, 1)\) 处的梯度。\(f_x = 3x^2 - 3y\),\(f_x(2, 1) = 12 - 3 = 9\);\(f_y = -3x + 3y^2\),\(f_y(2, 1) = -6 + 3 = -3\)。所以 \(\nabla f(2, 1) = \langle 9, -3\rangle\)。它的模是 \(\sqrt{81 + 9} = \sqrt{90} = 3\sqrt{10}\)。这个数下一节会用到,因为它正好等于"在该点最大变化率"。

梯度的好处之一是它把信息打包,让公式简洁。后面我们还会看到链式法则、最优化的条件、向量场理论里都用 \(\nabla\) 把繁复的偏导数列表压缩成一个对象,既好写又好读。

10.3 为什么梯度指向"上坡方向"

这是梯度最重要、也最美的几何性质。它的理由其实只是一个点积公式,但它解释的几何现象之深远,绝对值得我们花时间细细品。

从方向导数 \(D_\mathbf{u} f = \nabla f \cdot \mathbf{u}\) 出发。点积有一种"角度"的写法:\(\nabla f \cdot \mathbf{u} = |\nabla f|\,|\mathbf{u}|\cos\theta\),其中 \(\theta\) 是 \(\nabla f\) 与 \(\mathbf{u}\) 的夹角。因为 \(|\mathbf{u}| = 1\),所以

\[D_\mathbf{u} f = |\nabla f|\cos\theta.\]

这条公式里 \(|\nabla f|\) 是一个固定的非负数(取决于在哪一点),\(\cos\theta\) 是一个在 \([-1, 1]\) 之间的数(取决于你选的方向)。所以变化率的大小完全由 \(\cos\theta\) 来调节,而 \(\cos\theta\) 的取值是几何性的、肉眼可见的。

把 \(\theta\) 当成滑杆来扫一遍:

  • 当 \(\theta = 0\)——也就是 \(\mathbf{u}\) 指向与 \(\nabla f\) 完全相同的方向——我们有 \(\cos\theta = 1\),所以 \(D_\mathbf{u} f = |\nabla f|\),这是最大的可能值。换句话说,沿梯度方向走每单位距离得到的高度增加是最多的。这是上坡最陡的方向。
  • 当 \(\theta = \pi/2\)——也就是 \(\mathbf{u}\) 与 \(\nabla f\) 垂直——\(\cos\theta = 0\),\(D_\mathbf{u} f = 0\)。沿这个方向走,瞬时高度不变。这是"沿等高线走"的方向。
  • 当 \(\theta = \pi\)——也就是 \(\mathbf{u}\) 指向与 \(\nabla f\) 完全相反的方向——\(\cos\theta = -1\),\(D_\mathbf{u} f = -|\nabla f|\),这是最小的可能值。沿这个方向走,高度下降得最快。这是下坡最陡的方向(也是机器学习里梯度下降算法选的那个方向)。

把这三种几何观察总结成三条事实,它们就是梯度的"三大性质":

  1. \(\nabla f\) 指向 \(f\) 在该点处增长最快的方向。
  2. \(|\nabla f|\) 是该方向上的(也是所有方向中最大的)变化率。
  3. 沿与 \(\nabla f\) 垂直的方向走,\(f\) 瞬时不变;换言之,等高线 / 等值面与梯度垂直。

第三条值得多看一眼。它说的是:在每一点 \((a, b)\),如果你沿与 \(\nabla f(a, b)\) 垂直的方向走一小步,\(f\) 的值不变(到一阶意义上)。所以你走的轨迹保持在同一条等高线 \(f(x, y) = f(a, b)\) 上(或这条等高线的切线上)。这意味着梯度处处垂直于等高线。把整个梯度场画在地图上,你会看到一根根小箭头总是与地图上的等高线正交——这正是地形图上"水流方向"的几何来源,水会沿最陡下降方向流,即与等高线正交,即与梯度反向。

让我们把这件事坐稳一点,因为它是整个梯度概念的灵魂。想象你站在山上某一点 \(P\),周围的山是函数 \(z = f(x, y)\) 的图象。你脚下的地形可以用穿过 \(P\) 的等高线来描绘——也就是与你海拔相同的所有点连成的那条曲线。如果你想"省力地兜山"——既不上坡也不下坡——你应该沿等高线走。如果你想"最快上山",你应当与等高线垂直地、朝海拔升高的那一侧走;这正是梯度方向。如果你想"最快下山",反过来,沿梯度的反方向走。

那为什么在数学上"最快"恰好对应"梯度方向"?直觉是:当你沿着等高线走,你正好把扰动的"全部资源"用在了"不变高度"这件事上,所以高度变化率为零。当你与等高线斜着走,扰动有一部分用在了"沿等高线"的方向(那部分对高度无贡献),只有垂直于等高线的那部分才真正改变高度。所以"全部扰动都用来改变高度"的方向必定是垂直于等高线的方向——这就是梯度。这个论证可以严格地用 \(\cos\theta\) 公式重写,但直觉版本就足够说服人。

这一现象在物理里随处可见。热力学里,温度梯度指向温度上升最快的方向,而热流(根据傅里叶定律)沿温度梯度的反方向流——热从热处流向冷处,正是"沿最陡下降方向"。流体力学里,压强梯度的反方向是流体加速的方向。电磁学里,电势梯度的反方向是电场,电荷沿电场加速。整个"梯度指向最快上升"的几何故事,在物理里穿着不同的衣服反复出现。

看一道。\(f(x, y) = x^2 + y^2\) 在 \((3, 4)\) 处增长最快的方向是什么?最大变化率是多少?

\(\nabla f = \langle 2x, 2y\rangle\),\(\nabla f(3, 4) = \langle 6, 8\rangle\),\(|\nabla f(3, 4)| = \sqrt{36 + 64} = 10\)。所以增长最快方向是 \(\langle 6, 8\rangle\) 的方向,单位化为 \(\langle 3/5, 4/5\rangle\)。最大变化率是 \(10\)。

把这个结果跟几何直觉对一对。\(f = x^2 + y^2\) 的等高线是一族同心圆 \(x^2 + y^2 = c\)。点 \((3, 4)\) 在以 \(5\) 为半径的等高线上。从该点最快上山的方向应该是"径向向外"——也就是从原点指向 \((3, 4)\) 的方向,等价于 \(\langle 3, 4\rangle\) 方向,单位化即 \(\langle 3/5, 4/5\rangle\)。这与梯度方向一致。

顺便验证一下"梯度垂直于等高线":在 \((3, 4)\) 处,等高线是过该点的圆 \(x^2 + y^2 = 25\) 的切线;切线的方向(可以用参数化算或者用直觉)是 \(\langle -4, 3\rangle\) 方向(切向与径向垂直,且要么是 \(\langle -4, 3\rangle\) 要么是 \(\langle 4, -3\rangle\))。\(\langle 6, 8\rangle\cdot\langle -4, 3\rangle = -24 + 24 = 0\),正交。

再看一道更现实的。设温度场 \(T(x, y) = 100 - x^2 - 2y^2\)(摄氏度,坐标以米为单位)。在点 \((1, 2)\),问:

  1. 蚂蚁朝哪个方向跑能让温度上升最快?
  2. 那个方向上,蚂蚁感受到温度的上升速率是多少?
  3. 蚂蚁朝什么方向跑能让温度不变?

\(\nabla T = \langle -2x, -4y\rangle\),\(\nabla T(1, 2) = \langle -2, -8\rangle\)。\(|\nabla T(1, 2)| = \sqrt{4 + 64} = \sqrt{68} = 2\sqrt{17}\)。

(1) 上升最快的方向就是梯度方向,即 \(\langle -2, -8\rangle\),单位化为 \(\langle -1/\sqrt{17}, -4/\sqrt{17}\rangle\)。也就是说,蚂蚁朝着原点偏左下的方向跑温度上升最快——这跟温度场的形状一致,因为 \(T\) 在原点 \((0, 0)\) 取得最大值 \(100\),离原点越远温度越低。

(2) 该方向上的最大上升速率是 \(|\nabla T(1, 2)| = 2\sqrt{17} \approx 8.25\) °C/m。

(3) 温度不变的方向,与梯度垂直。梯度是 \(\langle -2, -8\rangle\),垂直方向是 \(\langle -8, 2\rangle\)(把分量交换一个并取反号即可),或它的反向 \(\langle 8, -2\rangle\)。任何与 \(\langle -2, -8\rangle\) 垂直的非零向量都是答案——因为有两个相反方向都让温度瞬时不变。

再看一道带最优化味道的。一只小鸟从点 \((1, 1, 1)\) 起飞,周围气流让它下降的速率与 \(f(x, y, z) = x^2 + y^2 + z^2\) 的某方向导数对应。它必须沿一个单位向量飞行。问朝哪个方向飞,下降最快?

\(\nabla f = \langle 2x, 2y, 2z\rangle\),在 \((1, 1, 1)\) 处是 \(\langle 2, 2, 2\rangle\)。下降最快的方向是 \(-\nabla f\) 方向,即 \(\langle -2, -2, -2\rangle\),单位化为 \(\langle -1/\sqrt{3}, -1/\sqrt{3}, -1/\sqrt{3}\rangle\)。下降速率为 \(|\nabla f| = 2\sqrt{3} \approx 3.46\)(每单位距离)。

这就是"梯度下降"的雏形。所有现代机器学习里训练神经网络的算法,本质都是同一回事:在参数空间的某一点,求出损失函数的梯度,沿梯度的反方向走一小步,反复迭代,直到走到一个梯度近乎为零的点为止。这一切的合法性,完全依赖于本节的"梯度反方向是最陡下降方向"。

10.4 等值面的切平面

第三大性质——梯度垂直于等高线 / 等值面——能立刻给我们一个非常实用的几何工具:写出任何等值面在任意一点的切平面方程。

设 \(S\) 是三元方程 \(F(x, y, z) = c\) 的等值面,\(P_0 = (x_0, y_0, z_0)\) 是 \(S\) 上的点。则 \(S\) 在 \(P_0\) 处的切平面法向量就是 \(\nabla F(P_0)\),从而切平面方程是

\[\boxed{F_x(P_0)(x - x_0) + F_y(P_0)(y - y_0) + F_z(P_0)(z - z_0) = 0.}\]

为什么是这样?因为 \(\nabla F\) 与等值面垂直(由 10.3 节第三条性质),而垂直于曲面的向量正是它的法向量。一旦你有了过某点的法向量,过该点的切平面方程就是"法向量 \(\cdot\) (位置向量 \(-\) 切点)= 0"——这是第 2 章里建立平面方程的标准模板。

看一道。椭球面 \(x^2 + 2y^2 + 3z^2 = 6\) 在 \((1, 1, 1)\) 处的切平面。设 \(F = x^2 + 2y^2 + 3z^2 - 6\)(把 \(c = 6\) 移到左边作差;结果可以是 \(F = x^2 + 2y^2 + 3z^2\) 或减常数,梯度都一样)。\(\nabla F = \langle 2x, 4y, 6z\rangle\) 在 \((1, 1, 1)\) 处为 \(\langle 2, 4, 6\rangle\)。所以切平面是

\[2(x - 1) + 4(y - 1) + 6(z - 1) = 0,\]

即 \(2x + 4y + 6z = 12\),化简 \(x + 2y + 3z = 6\)。

再看一道。求曲面 \(z^2 = x^2 + y^2\)(一个上下双叶圆锥的合体)在点 \((3, 4, 5)\) 处的切平面。把方程改写为 \(F(x, y, z) = z^2 - x^2 - y^2 = 0\)。\(\nabla F = \langle -2x, -2y, 2z\rangle\) 在 \((3, 4, 5)\) 是 \(\langle -6, -8, 10\rangle\)。切平面:

\[-6(x - 3) - 8(y - 4) + 10(z - 5) = 0,\]

即 \(-6x - 8y + 10z = 0\),也就是 \(3x + 4y = 5z\)(两边除以 \(-2\) 然后整理)。

再看一道。求曲面 \(\sin(xy) + e^{z} = 1\) 在 \((0, 0, 0)\) 处的切平面。\(F = \sin(xy) + e^z - 1\)。\(F_x = y\cos(xy)\) 在原点是 \(0\);\(F_y = x\cos(xy)\) 在原点是 \(0\);\(F_z = e^z\) 在原点是 \(1\)。所以 \(\nabla F(0, 0, 0) = \langle 0, 0, 1\rangle\),切平面是 \(0(x) + 0(y) + 1(z - 0) = 0\),即 \(z = 0\)。也就是水平平面。这与几何直觉吻合:在原点附近,这个曲面的"陡峭"几乎全在 \(z\) 方向上,所以切平面是水平的。

把这一节的结论与第 8 章的切平面公式联系起来。8.1 节里我们对显式给定的曲面 \(z = f(x, y)\) 求过切平面,得到 \(z = f(a, b) + f_x(x - a) + f_y(y - b)\)。该曲面也可以写成等值面:令 \(F(x, y, z) = z - f(x, y)\),则 \(z = f(x, y)\) 等价于 \(F(x, y, z) = 0\)。这时 \(F_x = -f_x\)、\(F_y = -f_y\)、\(F_z = 1\),所以梯度法向量是 \(\langle -f_x, -f_y, 1\rangle\)。切平面方程

\[-f_x(x - a) - f_y(y - b) + 1\cdot(z - c) = 0,\]

整理得 \(z = c + f_x(x - a) + f_y(y - b)\),正是 8.1 节那条公式。两套方法相容,只是看待问题的角度不同——一个把曲面看成"高度场",另一个把它看成"等值面",最终给出同一张切平面。

另一个连贯性检查:对二元函数,如果你写出梯度法向量 \(\langle -f_x, -f_y, 1\rangle\),它的最后一个分量永远是 \(+1\)——意思是这种"显式曲面"的法向量永远有"向上"的分量,这对应几何上"曲面是高度场,有上下侧之分,法向量取向上一侧"的事实。如果你换看 \(F = f(x, y) - z\) 而不是 \(z - f(x, y)\),梯度就是 \(\langle f_x, f_y, -1\rangle\)——同一条法线,只是反向。法线是双向的,选哪一向取决于约定。

第 11 章里我们用梯度找临界点(梯度为零的点),并发展二阶导数判别法来分类极值——这是把梯度的几何洞察用到最优化里去的第一站。
