第 11 章 局部极值与最值

在一元微积分里，"求极值"是一套已经被打磨得近乎透明的程序：让 \(f'(x) = 0\)、解出临界点、再用 \(f''(x)\) 的正负来分辨极大或极小，最后在闭区间的端点处补上两个比较。整套方法之所以好用，是因为一维世界足够简单——函数图像是一条曲线，曲线在某点要么"上凸"要么"下凸"，没有第三种姿态。

到了多元函数，这套思路的骨架仍在：极值候选仍然出现在某种"导数为零"的位置；判别仍然要看二阶量。但维度一旦升高，世界忽然变得宽阔——一个点附近，函数可以同时朝某些方向上升、朝另一些方向下降。这种崭新的几何姿态有一个生动的名字：鞍点。它是一元情形里完全不存在的现象，也是多元极值理论里最值得仔细看清的画面。

本章我们沿着这条思路一步步铺开：先把"梯度等于零"作为多元版本的费马必要条件，看清它的几何意义；再借助二阶偏导构造一个判别式，把临界点干净地分成"极小、极大、鞍点、定理失效"四类；最后处理闭有界区域上的最值问题——这一步不只看内部的临界点，还得仔细搜遍边界。这三件事合起来构成一套完整的"无约束（或区域内）极值"工作流，也为下一章的拉格朗日乘数法打下背景。

11.1 临界点

多元函数的极值候选叫做临界点，定义是梯度等于零向量的点： \[ \nabla f(a, b) = \mathbf{0} \quad \Longleftrightarrow \quad f_x(a, b) = 0 \ \text{且}\ f_y(a, b) = 0. \] 这个定义为什么自然？回想 10.3 节的几何观察：梯度 \(\nabla f\) 指向 \(f\) 增长最快的方向，模长就是这个方向上的瞬时增长率。如果 \(\nabla f(a, b) \neq \mathbf{0}\)，那么从 \((a, b)\) 沿梯度方向稍微走一点，\(f\) 必然增加；沿反梯度方向走，\(f\) 必然减少。这就立刻说明 \((a, b)\) 不可能是局部极值点。所以——只要 \((a, b)\) 是局部极值点，必然有 \(\nabla f(a, b) = \mathbf{0}\)。

这正是一元世界里"\(f'(a) = 0\)"那条费马必要条件的完整推广。一元情形里只有一个偏导（就是普通的 \(f'\)），二元情形里需要两个偏导同时为零，三元情形里需要三个——总之，梯度 = 零向量 这一条简洁的式子捕捉到了所有维度。

从切平面的角度看也一样直观。在临界点处，曲面 \(z = f(x, y)\) 的切平面方程 \[ z - f(a, b) = f_x(a, b)(x - a) + f_y(a, b)(y - b) \] 退化成 \(z = f(a, b)\)——一个完全水平的平面。换言之，临界点正是曲面"切平面水平"的位置。这一画面也提示我们：水平切平面背后可以躲着三种不同的几何——山顶、谷底，以及鞍。光看"切平面水平"这一件事，无法分辨它们；这正是下一节要解决的问题。

先从两个具体例子出发，体会临界点的求解过程。考虑 \(f(x, y) = x^2 + y^2 - 4x + 6y + 13\)。这是一个二次多项式，几何上是一个朝上开口的抛物面。把两个偏导分别置零：\(f_x = 2x - 4 = 0\)，\(f_y = 2y + 6 = 0\)，立得 \(x = 2\)、\(y = -3\)。所以唯一的临界点是 \((2, -3)\)，此处 \(f(2, -3) = 4 + 9 - 8 - 18 + 13 = 0\)。如果把 \(f\) 配方成 \((x-2)^2 + (y+3)^2\)，可以一眼看出它是抛物面 \(z = u^2 + v^2\) 平移而来，\((2, -3)\) 显然是全局最低点。这个例子虽然初等，但揭示了一个值得记住的模式：纯二次的 \(f\) 总能配方，配方的最低点（或最高点）就是临界点。

再看一个三次多项式 \(f(x, y) = x^3 + y^3 - 3xy\)。这次偏导为 \[ f_x = 3x^2 - 3y, \qquad f_y = 3y^2 - 3x. \] 方程组化为 \(y = x^2\) 与 \(x = y^2\)。把第一式代入第二式得 \(x = (x^2)^2 = x^4\)，整理为 \(x^4 - x = x(x^3 - 1) = 0\)。于是 \(x = 0\) 或 \(x = 1\)，对应 \(y = 0\) 或 \(y = 1\)。两个临界点是 \((0, 0)\) 与 \((1, 1)\)。这两点的几何性质不同（一个是鞍点，一个是局部极小），但仅凭目前掌握的信息无从判断；需要等到下一节的二阶判别。

再举一个更"不规则"的例子：求 \(f(x, y) = \sin x + \sin y + \sin(x + y)\) 在矩形 \([0, \pi] \times [0, \pi]\) 内部的临界点。偏导为 \[ f_x = \cos x + \cos(x + y), \qquad f_y = \cos y + \cos(x + y). \] 两式相减得 \(\cos x = \cos y\)，因此在所给开区间里必有 \(x = y\)。代入 \(f_x = 0\)：\(\cos x + \cos 2x = 0\)。利用 \(\cos 2x = 2\cos^2 x - 1\)，方程变为 \(2\cos^2 x + \cos x - 1 = 0\)，即 \((2\cos x - 1)(\cos x + 1) = 0\)。在 \((0, \pi)\) 上 \(\cos x = -1\) 不出现，故 \(\cos x = 1/2\)、\(x = \pi/3\)。临界点为 \((\pi/3, \pi/3)\)，函数值 \(f = 3 \sin(\pi/3) = 3\sqrt{3}/2\)。

从这些例子里我们能总结出"找临界点"的工作节奏：写下两个偏导，让它们同时为零，把所得的二元方程组解干净。常用的代数手段无非三种——直接代入消元、两式相减寻找对称、提取公因子分类讨论。复杂的题目往往会出现"或 \(x = 0\)，或 …"这样的多种情形，要把所有分支都走到底。下一节我们将给临界点贴上几何标签。

11.2 二阶导数判别法

在临界点处，一阶信息已经被用尽——梯度全零，告诉我们此处可能是极值，但不告诉我们是哪一种。要分辨极大、极小还是鞍点，得看二阶。一元情形里 \(f''(a) > 0\) 表示曲线在 \(a\) 处"凹向上"，因此 \(a\) 是极小点；\(f''(a) < 0\) 则是极大；\(f''(a) = 0\) 时不能判定。多元情形里，二阶信息不再是单一的数，而是三个量：\(f_{xx}\)、\(f_{yy}\)、\(f_{xy}\)（按 11 章前的约定 \(f_{xy} = f_{yx}\)）。

它们如何决定临界点的几何，可以由二元泰勒展开窥见。设 \((a, b)\) 是 \(f\) 的临界点。把 \(f\) 在该点附近做二阶泰勒展开（一阶项为零，因为梯度为零）： \[ f(a + h, b + k) - f(a, b) \approx \tfrac{1}{2}\bigl[f_{xx}(a, b)\, h^2 + 2 f_{xy}(a, b)\, hk + f_{yy}(a, b)\, k^2\bigr]. \] 括号里那块二次型决定了 \(f\) 在 \((a, b)\) 附近"上升或下降"的姿态。它是关于 \((h, k)\) 的二次齐次式，几何上对应一张过原点的抛物面或鞍面。问题就化成：这个二次型对所有非零 \((h, k)\) 是恒为正、恒为负、还是有正有负？

恒为正叫"正定"——任何方向走 \(f\) 都增加，临界点是局部极小；恒为负叫"负定"——任何方向都减少，临界点是局部极大；正负都有叫"不定"——某些方向走上去、某些方向走下来，临界点是鞍点。

判别正定/负定/不定的代数工具是判别式 \[ D = f_{xx}(a, b)\, f_{yy}(a, b) - \bigl(f_{xy}(a, b)\bigr)^2. \] 这个量可以理解成那张二次型矩阵 \(\begin{pmatrix}f_{xx} & f_{xy} \\ f_{xy} & f_{yy}\end{pmatrix}\) 的行列式。线性代数的"\(2 \times 2\) 对称矩阵特征值符号"判别准则告诉我们：

  • 若 \(D > 0\) 且 \(f_{xx} > 0\)：两个特征值都是正的——二次型正定——\((a, b)\) 是局部极小。
  • 若 \(D > 0\) 且 \(f_{xx} < 0\)：两个特征值都是负的——二次型负定——\((a, b)\) 是局部极大。
  • 若 \(D < 0\)：两个特征值符号相反——二次型不定——\((a, b)\) 是鞍点。
  • 若 \(D = 0\)：判别失效，需要更高阶的信息或沿不同方向直接讨论。

这就是著名的二阶导数判别法，把它装进键值对： \[ \boxed{\ D = f_{xx} f_{yy} - f_{xy}^2,\quad \text{看 \(D\) 的符号；\(D > 0\) 时再看 \(f_{xx}\) 的符号。}\ } \] 有一个温柔的记忆方式：\(D > 0\) 表示"真极值"——非极大即极小，由 \(f_{xx}\) 的正负二选一；\(D < 0\) 表示"鞍点"，是新东西；\(D = 0\) 是边界情形，需要其他工具。注意 \(D > 0\) 时 \(f_{xx}\) 与 \(f_{yy}\) 必然同号（因为 \(f_{xx} f_{yy} > f_{xy}^2 \ge 0\)），所以拿哪一个看都行。

为什么要把二维这件事做得这么仔细？因为鞍点是高维世界独有的几何，值得花时间认真看一眼。最经典的例子是 \(f(x, y) = x^2 - y^2\)。在原点处 \(\nabla f = \mathbf{0}\)，函数沿 \(x\) 轴方向恢复成 \(y = x^2\)（一条朝上开口的抛物线，原点是极小），沿 \(y\) 轴方向却退化为 \(y = -y^2\)（朝下开口，原点是极大）。这就是马鞍面：南北方向是上凸的山脊，东西方向是下凹的谷底；原点既不是极大也不是极小。代入判别法：\(f_{xx} = 2\)、\(f_{yy} = -2\)、\(f_{xy} = 0\)，故 \(D = (2)(-2) - 0 = -4 < 0\)，确实是鞍点。

回到上一节留下的悬念：\(f(x, y) = x^3 + y^3 - 3xy\) 的临界点 \((0, 0)\) 与 \((1, 1)\) 究竟各是哪种？现在算 \(f_{xx} = 6x\)、\(f_{yy} = 6y\)、\(f_{xy} = -3\)。

在 \((0, 0)\)：\(D = (0)(0) - (-3)^2 = -9 < 0\)，所以是鞍点。直观上，从原点沿 \(y = x\) 方向走，\(f = 2x^3 - 3x^2\)，这条三次曲线从右边渐渐降到原点；沿 \(y = -x\) 方向 \(f = -3x^2\)，原点是局部极大。两个方向行为相反，对得上鞍点。

在 \((1, 1)\)：\(D = (6)(6) - 9 = 27 > 0\)，且 \(f_{xx}(1, 1) = 6 > 0\)，所以是局部极小，函数值 \(f(1, 1) = 1 + 1 - 3 = -1\)。

再做一个带着混合偏导的例子：\(f(x, y) = x^3 - 3x + y^3 - 3y^2\)。先求临界点：\(f_x = 3x^2 - 3 = 0\) 给 \(x = \pm 1\)；\(f_y = 3y^2 - 6y = 3y(y - 2) = 0\) 给 \(y = 0\) 或 \(y = 2\)。一共 \(2 \times 2 = 4\) 个临界点。二阶导：\(f_{xx} = 6x\)，\(f_{yy} = 6y - 6\)，\(f_{xy} = 0\)，故 \(D = 6x \cdot (6y - 6) = 36 x(y - 1)\)。

临界点\(D\)\(f_{xx}\)分类\(f\)
\((1, 0)\)\(-36\)\(6\)鞍点\(-2\)
\((1, 2)\)\(36\)\(6\)局部极小\(-6\)
\((-1, 0)\)\(36\)\(-6\)局部极大\(2\)
\((-1, 2)\)\(-36\)\(-6\)鞍点\(-2\)

这个例子干净地展示了判别法的全套表演——同一个函数里四种局面（极大、极小、鞍点两次）齐聚一堂。

下一例则展示判别失效的情形。考虑 \(f(x, y) = x^4 + y^4\)。显然 \((0, 0)\) 是临界点（梯度为 \(\langle 4x^3, 4y^3\rangle\)）。二阶导 \(f_{xx} = 12 x^2\)、\(f_{yy} = 12 y^2\)、\(f_{xy} = 0\) 全在原点取零，于是 \(D = 0\)。判别法束手无策。但直接观察：\(f(x, y) = x^4 + y^4 \ge 0\)，等号只在原点取到，所以原点其实是严格的全局极小。判别法失败不代表结论不存在；它只是提示我们去用别的工具——直接观察、沿不同方向比较、或更高阶的泰勒展开。再看 \(f(x, y) = x^4 - y^4\)：同样在原点 \(D = 0\)，但沿 \(x\) 轴是极小、沿 \(y\) 轴是极大，原点是鞍点。两个例子在判别式上长得一模一样（\(D = 0\)），结论却完全不同——这就是 \(D = 0\) 必须特别小心的原因。

最后做一个稍带挑战的几何题：求 \(f(x, y) = x^2 + y^2 + \tfrac{1}{x y}\)（限定 \(x, y > 0\)）的局部极小。偏导 \[ f_x = 2x - \frac{1}{x^2 y}, \qquad f_y = 2y - \frac{1}{x y^2}. \] 令 \(f_x = 0\)：\(2x^3 y = 1\)；令 \(f_y = 0\)：\(2 x y^3 = 1\)。两式相除得 \(x^2 / y^2 = 1\)，故在 \(x, y > 0\) 区域内 \(x = y\)。代回 \(2 x^4 = 1\)，\(x = (1/2)^{1/4}\)。此即唯一临界点。再算二阶判别可知它确为局部极小（\(f_{xx} = 2 + 2/(x^3 y) > 0\)，且 \(D > 0\) 通过直接代入验证）。这种"能看出唯一性 + 适合几何范围"的题，给出的是真正的最小值：\(f_{\min} = 2\sqrt{1/2} + \sqrt{2} = \sqrt{2} + \sqrt{2} = 2\sqrt{2}\)。这种"独子型"题目以后还会反复出现。

11.3 闭有界区域上的最值

一元微积分里，闭区间 \([a, b]\) 上的连续函数必能取到最大值和最小值——这是极值定理。求最值的标准方法只有两类候选：内部的临界点（导数为零的点），加上区间的两个端点。比较候选值，最大者为最大值，最小者为最小值。

多元情形里，相应的极值定理（A.2 节）说：闭有界区域 \(D\) 上的连续函数取得最大值和最小值。而求最值的候选则有两类：

  1. 区域 \(D\) 内部的临界点，即 \(\nabla f = \mathbf{0}\) 的点；
  2. 区域 \(D\) 边界上的"边界临界点"，即 \(f\) 限制到边界后取极值的位置。

这两条类比于一元情形里"内部临界点 + 端点"的两步。但二维区域的边界是一条曲线（如圆周、椭圆、矩形的四条边），三维区域的边界是一张曲面，所以"边界搜索"本身又是一道低一维的最值题。常用三种处理方式：

  • 把边界参数化，把 \(f\) 限制成单变量函数，再用一元方法找极值；
  • 把边界拆成几段（直线段、圆弧、抛物段……）逐段处理；
  • 用第 12 章的拉格朗日乘数法直接求"\(f\) 在 \(g = 0\) 上的极值"。

不论选哪种工具，最后都要把所有候选值汇总比较，决定最大与最小。如果区域有"角"（比如矩形的四个角），角点也要单独算上——它们是边界本身的"端点"。

下面是一个完整的圆盘上最值题。求 \(f(x, y) = x^2 + 2y^2 - x\) 在闭圆盘 \(D : x^2 + y^2 \le 4\) 上的最大值与最小值。

第一步：内部临界点。 由 \(f_x = 2x - 1\)、\(f_y = 4y\)，令两者为零，得 \(x = 1/2\)、\(y = 0\)。点 \((1/2, 0)\) 是否在 \(D\) 内？\(1/4 + 0 = 1/4 < 4\)，是。函数值 \(f(1/2, 0) = 1/4 - 1/2 = -1/4\)。这是一个候选。

第二步：边界 \(x^2 + y^2 = 4\) 上的搜索。 用参数化 \(x = 2\cos t\)、\(y = 2\sin t\)（\(t \in [0, 2\pi]\)）。代入 \(f\)： \[ g(t) = 4\cos^2 t + 8\sin^2 t - 2\cos t = 4\cos^2 t + 8(1 - \cos^2 t) - 2\cos t = 8 - 4\cos^2 t - 2\cos t. \] 为简洁，令 \(u = \cos t\)，得 \(g = 8 - 4u^2 - 2u\)。这是 \(u\) 的一元二次（其中 \(u \in [-1, 1]\)），\(dg/du = -8u - 2 = 0\) 给 \(u = -1/4\)。这是一个内部极值。\(g\) 在 \(u = -1/4\) 处的值为 \(8 - 4 \cdot 1/16 + 1/2 = 8 - 1/4 + 1/2 = 33/4\)。再看 \(u = 1\)（即 \((2, 0)\)）和 \(u = -1\)（即 \((-2, 0)\)）：分别给 \(g = 8 - 4 - 2 = 2\) 与 \(g = 8 - 4 + 2 = 6\)。

第三步：汇总比较。

  • 内部 \((1/2, 0)\)：\(f = -1/4\)。
  • 边界 \(\cos t = -1/4\)（对应两个对称的点）：\(f = 33/4 = 8.25\)。
  • 边界 \((2, 0)\)：\(f = 2\)。
  • 边界 \((-2, 0)\)：\(f = 6\)。
最大值是 \(33/4\)，在 \(\cos t = -1/4\) 的两点取得；最小值是 \(-1/4\)，在 \((1/2, 0)\) 取得。值得注意：最大值出现在边界，最小值出现在内部——这是常见的格局，但不绝对，万事还得算了再说。

第二个例子展示矩形区域上的工作方式——边界由四条线段拼成，每段独立处理。求 \(f(x, y) = x^2 + xy + y^2 - 6x\) 在矩形 \(R = [0, 5] \times [-3, 3]\) 上的最值。

内部：\(f_x = 2x + y - 6 = 0\)，\(f_y = x + 2y = 0\)。从第二式得 \(x = -2y\)，代入第一式：\(-4y + y - 6 = 0\)，\(y = -2\)，\(x = 4\)。点 \((4, -2)\) 在矩形内，函数值 \(f = 16 - 8 + 4 - 24 = -12\)。

边界分四段：

  • 下边 \(y = -3\)、\(x \in [0, 5]\)：\(f = x^2 - 3x + 9 - 6x = x^2 - 9x + 9\)。\(f' = 2x - 9 = 0\) 给 \(x = 9/2\)，函数值 \(f(9/2, -3) = 81/4 - 81/2 + 9 = -45/4\)。
  • 上边 \(y = 3\)、\(x \in [0, 5]\)：\(f = x^2 + 3x + 9 - 6x = x^2 - 3x + 9\)。\(f' = 2x - 3 = 0\) 给 \(x = 3/2\)，函数值 \(f(3/2, 3) = 9/4 - 9/2 + 9 = 27/4\)。
  • 左边 \(x = 0\)、\(y \in [-3, 3]\)：\(f = y^2\)，最小 \(0\)（\(y = 0\)）、最大 \(9\)（\(y = \pm 3\)）。
  • 右边 \(x = 5\)、\(y \in [-3, 3]\)：\(f = 25 + 5y + y^2 - 30 = y^2 + 5y - 5\)。\(f' = 2y + 5 = 0\) 给 \(y = -5/2\)，函数值 \(f(5, -5/2) = 25/4 - 25/2 - 5 = -45/4\)。
四个角的值：\(f(0, -3) = 9\)、\(f(0, 3) = 9\)、\(f(5, -3) = 25 - 15 + 9 - 30 = -11\)、\(f(5, 3) = 25 + 15 + 9 - 30 = 19\)。

把所有候选汇总：\(-12, -45/4, 27/4, 0, 9, -45/4, 9, 9, -11, 19\)。最大值是 \(19\)，在角点 \((5, 3)\) 取得；最小值是 \(-12\)，在内部临界点 \((4, -2)\) 取得。这个题清晰地说明：四条边、四个角、再加内部，缺一不可——边界搜索如果漏了一段，就可能漏掉真正的最大值。

再做一个三角形区域上的例子，体会"边界 = 直线段拼图"的处理方式。求 \(f(x, y) = x + 2y - x y\) 在三角形 \(T\)（顶点为 \((0, 0)\)、\((4, 0)\)、\((0, 3)\)）上的最值。

内部：\(f_x = 1 - y\)，\(f_y = 2 - x\)。临界点 \((2, 1)\) 在三角形内（直线 \(3x + 4y = 12\) 上 \(3(2) + 4(1) = 10 < 12\)），\(f(2, 1) = 2 + 2 - 2 = 2\)。

三条边：

  • 下边 \(y = 0\)、\(x \in [0, 4]\)：\(f = x\)，单调，端点 \(0\) 与 \(4\)；
  • 左边 \(x = 0\)、\(y \in [0, 3]\)：\(f = 2y\)，单调，端点 \(0\) 与 \(6\)；
  • 斜边 \(3x + 4y = 12\)，可参数化 \(y = 3 - 3x/4\)，\(x \in [0, 4]\)：\(f = x + 2(3 - 3x/4) - x(3 - 3x/4) = x + 6 - 3x/2 - 3x + 3x^2/4 = 3x^2/4 - 7x/2 + 6\)。\(f' = 3x/2 - 7/2 = 0\) 给 \(x = 7/3\)，对应 \(y = 3 - 7/4 = 5/4\)。函数值 \(f(7/3, 5/4) = 7/3 + 5/2 - 35/12 = 28/12 + 30/12 - 35/12 = 23/12\)。
顶点已经在边的端点中处理过：\((0, 0) \to f = 0\)，\((4, 0) \to f = 4\)，\((0, 3) \to f = 6\)。汇总比较，最大值是 \(6\)，在 \((0, 3)\) 取得；最小值是 \(0\)，在 \((0, 0)\) 取得。

从这几个例子中可以提炼出"闭有界区域上求最值"的标准节奏：先把内部所有临界点找出来（梯度为零），算下函数值；再把边界拆成可以参数化的部分（圆弧、直线段、椭圆弧……），逐段化为一元最值题；如果有"角点"（不光滑处），单独补上；最后汇总所有候选值，最大者即最大值、最小者即最小值。下一章里我们会用拉格朗日乘数法替代"参数化边界"的步骤——它的优势是不必显式参数化，缺点是要解一个非线性方程组；两套工具各有适合的场合。
