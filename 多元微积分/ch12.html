第 12 章 拉格朗日乘数法

有相当一类极值问题，本质并不是"在某区域里求 \(f\) 的极值"，而是"在某曲线（或曲面）上求 \(f\) 的极值"。比如：在椭圆 \(x^2 + 4y^2 = 4\) 上，函数 \(f(x, y) = xy\) 的最大值是多少？或者：在球面 \(x^2 + y^2 + z^2 = 1\) 上，函数 \(f = x + y + z\) 的最大值在哪里取得？这种"被一条等式约束捆住"的极值，就是带约束极值问题。

原则上有一条朴素出路：从约束 \(g(x, y) = 0\) 中解出某个变量（比如把 \(y\) 解成 \(x\) 的函数），代回到 \(f\) 里，化为一元最值题。但这条路在两点上很容易出毛病：约束未必能干净地解出某个变量（试试看从 \(x^2 + 4y^2 = 4\) 里解 \(y\)：得到 \(\pm\sqrt{1 - x^2/4}\)，要分两个分支讨论）；即便能解，结果通常不对称、丑陋而且容易计算出错。

拉格朗日乘数法提供了一条对称、几何意义明确、不必显式消元的方法，是多元微积分里最优雅的几个想法之一。它不在乎你能不能解出谁，只要你能写下梯度，就能写下方程组；剩下的全是代数。

本章我们先说一约束情形——这是绝大多数应用题的样貌——给出方法、跑几个经典例子（包括"椭圆内接矩形最大面积"和"和定积最大"等老题），然后讲背后的几何直觉（"等值线相切于约束曲线"），最后扩展到两个约束。

12.1 一个约束

设要极值化的函数是 \(f(x, y)\)，约束写成 \(g(x, y) = 0\) 的形式。拉格朗日条件说：极值候选点 \((x, y)\) 一定满足 \[ \boxed{\ \nabla f(x, y) = \lambda\, \nabla g(x, y) \quad \text{且}\quad g(x, y) = 0\ } \] 其中 \(\lambda\) 是一个待定的实数，叫做拉格朗日乘数。按分量展开就是一个三元方程组： \[ f_x = \lambda g_x, \qquad f_y = \lambda g_y, \qquad g(x, y) = 0. \] 解出 \((x, y, \lambda)\) 后，每个 \((x, y)\) 都是候选极值点。把 \(f\) 在所有候选点上的值算出来，最大者为最大值、最小者为最小值。

方法看起来神秘，但用起来非常机械：列梯度、写方程组、解、比较。下面我们用四个例子层层深入地体会它。

第一例就是开篇提到的椭圆题：求 \(f(x, y) = xy\) 在椭圆 \(g(x, y) = x^2 + 4y^2 - 4 = 0\) 上的极值。算两个梯度：\(\nabla f = \langle y, x\rangle\)，\(\nabla g = \langle 2x, 8y\rangle\)。方程组： \[ y = 2\lambda x,\qquad x = 8\lambda y, \qquad x^2 + 4y^2 = 4. \] 把第一式代入第二式：\(x = 8\lambda(2\lambda x) = 16 \lambda^2 x\)，整理 \(x(1 - 16\lambda^2) = 0\)。两种情形。

情形 A：\(x = 0\)。代入约束 \(4y^2 = 4\)，\(y = \pm 1\)。候选点 \((0, \pm 1)\)，函数值 \(f = 0\)。

情形 B：\(\lambda^2 = 1/16\)，即 \(\lambda = \pm 1/4\)。由 \(y = 2 \lambda x = \pm x/2\) 代入约束：\(x^2 + 4(x/2)^2 = 4\)，\(2x^2 = 4\)，\(x^2 = 2\)，\(x = \pm \sqrt{2}\)。
子情形 \(\lambda = 1/4\)：\(y = x/2\)，候选 \((\sqrt 2, \sqrt 2 / 2)\) 与 \((-\sqrt 2, -\sqrt 2/2)\)，函数值 \(f = \sqrt 2 \cdot \sqrt 2 / 2 = 1\)。
子情形 \(\lambda = -1/4\)：\(y = -x/2\)，候选 \((\sqrt 2, -\sqrt 2/2)\) 与 \((-\sqrt 2, \sqrt 2/2)\)，函数值 \(f = -1\)。

最大值 \(1\)，最小值 \(-1\)。\((0, \pm 1)\) 处的 \(f = 0\) 既不是最大也不是最小——但拉格朗日方法仍然忠实地把它们也找出来了，它给出的是所有候选，是不是真正的极值要靠最后比较。

第二例是一个真正的应用题——经典中的经典：在椭圆 \(\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = 1\) 内接一个边平行于坐标轴的矩形，问矩形面积最大是多少？

设矩形右上角的顶点是 \((x, y)\)（\(x, y > 0\)），由对称性，矩形的边长是 \(2x\) 和 \(2y\)，面积 \(A = 4xy\)。约束是 \(g(x, y) = x^2/a^2 + y^2/b^2 - 1 = 0\)。梯度 \[ \nabla A = \langle 4y, 4x\rangle, \qquad \nabla g = \Bigl\langle \frac{2x}{a^2}, \frac{2y}{b^2}\Bigr\rangle. \] 方程组：\(4y = \lambda \cdot 2x/a^2\)，\(4x = \lambda \cdot 2y/b^2\)。两式相乘消去 \(\lambda\)：\(16 xy = \lambda^2 \cdot 4xy/(a^2 b^2)\)。在 \(x, y > 0\) 的可行区域上 \(xy \neq 0\)，所以 \(\lambda^2 = 4 a^2 b^2\)，\(\lambda = 2 a b\)（取正根，便于后续）。代回 \(4y = 2ab \cdot 2x/a^2 = 4bx/a\)，得 \(y = bx/a\)。代入约束：\(x^2/a^2 + b^2 x^2/(a^2 b^2) = 1\)，即 \(2x^2/a^2 = 1\)，\(x = a/\sqrt 2\)，对应 \(y = b/\sqrt 2\)。最大面积为 \[ A_{\max} = 4 \cdot \frac{a}{\sqrt 2} \cdot \frac{b}{\sqrt 2} = 2ab. \] 注意几何上的优雅：最优矩形的顶点位置 \((a/\sqrt 2, b/\sqrt 2)\) 正好是椭圆参数 \(t = \pi/4\) 的位置——同时半轴方向各占一半。这对单位圆 \(a = b = 1\) 退化成"边长为 \(\sqrt 2\) 的正方形"，面积 \(2\)，与"圆内接最大矩形是正方形"的常识吻合。

第三例：求 \(f(x, y, z) = x y z\) 在 \(x + y + z = 3\) 的约束下，且 \(x, y, z > 0\) 时的最大值。这是"和定积最大"的经典几何问题，也是均值不等式 AM-GM 在三变量下的展开。 \[ \nabla f = \langle yz, xz, xy\rangle, \qquad \nabla g = \langle 1, 1, 1\rangle. \] 方程组：\(yz = \lambda\)，\(xz = \lambda\)，\(xy = \lambda\)。三式相等：\(yz = xz \Rightarrow z(y - x) = 0\)；在 \(z > 0\) 区域里 \(y = x\)。同理 \(z = x\)。所以 \(x = y = z\)，代入约束 \(3x = 3\)，\(x = 1\)。最大值 \(f(1, 1, 1) = 1\)。这告诉我们一个干净的事实：对所有满足 \(x + y + z = 3\) 的正数三元组，乘积不超过 \(1\)，且仅在 \(x = y = z = 1\) 时取等。换成对称形式：对正实数 \(x_1, x_2, x_3\)， \[ \frac{x_1 + x_2 + x_3}{3} \ge \sqrt[3]{x_1 x_2 x_3}, \] 等号当且仅当 \(x_1 = x_2 = x_3\) 时成立。这就是经典的算术-几何均值不等式。

第四例转向工程感更强的问题：一个不带顶盖的长方体储物盒，体积为 \(V_0\)，求表面积最小的尺寸（这是"最优包装"的一个简化版，实际中比如开口纸箱、上面敞开的水池）。设底面是 \(x \times y\) 的矩形，高为 \(z\)。体积 \(g(x, y, z) = xyz - V_0 = 0\)。表面积 \(S = xy + 2xz + 2yz\)（底面 + 四条侧壁，没有顶）。求 \(S\) 在 \(g = 0\) 上的最小。 \[ \nabla S = \langle y + 2z, x + 2z, 2x + 2y\rangle, \qquad \nabla g = \langle yz, xz, xy\rangle. \] 方程组（用 \(\lambda\) 是拉格朗日乘数）： \[ y + 2z = \lambda yz, \quad x + 2z = \lambda xz, \quad 2x + 2y = \lambda xy. \] 两式相减：\((y + 2z) - (x + 2z) = \lambda z(y - x)\)，化为 \(y - x = \lambda z (y - x)\)。若 \(y \neq x\)，则 \(\lambda z = 1\)。但物理上，由对称性也容易猜到 \(x = y\)，让我们直接采取这条路：设 \(x = y\)。则前两式合并为 \(x + 2z = \lambda xz\)；第三式变为 \(4x = \lambda x^2\)，即 \(\lambda x = 4\)（\(x > 0\)），所以 \(\lambda = 4/x\)。代入：\(x + 2z = (4/x) \cdot xz = 4z\)，即 \(x = 2z\)。再用约束 \(x \cdot x \cdot z = V_0\)：\(x^2 z = V_0\)，由 \(z = x/2\) 得 \(x^3/2 = V_0\)，\(x = (2 V_0)^{1/3}\)。最优尺寸：底面是 \(x = (2V_0)^{1/3}\) 的正方形，高 \(z = x/2 = (2V_0)^{1/3}/2\)。这告诉我们：开口盒的最优形状是底面正方形、高为底边一半。比起带顶盖的"立方体最优"，这次少一个顶面，几何就发生了相应的变化——"低胖"的盒子比"高瘦"的盒子表面更小。

第五例展示"不等式约束"如何转化成一系列等式约束。求 \(f(x, y) = x + y\) 在闭区域 \(\{(x, y) : x^2 + y^2 \le 4,\ y \ge 0\}\)（上半圆盘）上的最值。这是 11.3 节"闭有界区域最值"问题；用拉格朗日的视角处理边界。内部临界点：\(\nabla f = \langle 1, 1\rangle\)，永远不为零，所以内部没有候选——最值必在边界上。边界由两段组成：上半圆弧 \(x^2 + y^2 = 4\)（\(y \ge 0\)）和直径 \(y = 0\)（\(-2 \le x \le 2\)）。在圆弧段，约束 \(g = x^2 + y^2 - 4 = 0\)，方程组 \(1 = 2\lambda x\)，\(1 = 2\lambda y\)，立得 \(x = y\)，代入约束 \(2x^2 = 4\)，\(x = \sqrt 2\)（取 \(y \ge 0\)）。候选 \((\sqrt 2, \sqrt 2)\)，\(f = 2\sqrt 2\)。在直径段 \(y = 0\)，\(f = x\) 单调，端点 \(\pm 2\) 给值 \(\pm 2\)。汇总：最大 \(2\sqrt 2 \approx 2.83\)，在 \((\sqrt 2, \sqrt 2)\) 取得；最小 \(-2\)，在 \((-2, 0)\) 取得。

这五个例子合起来覆盖了一约束拉格朗日的常见情形：对称多项式、几何最优化、工程约束、"和定积最大"的均值不等式、与边界搜索的结合。每一题的工作流都是相同的：写下两个梯度，列方程组，解出候选点，比较函数值。整个过程的难点不在概念，而在代数——当方程组里出现"\(x = 0\) 或 …" 这样的分支时，必须把每条分支都走到底。

有一个隐含的前提需要稍稍提一句：拉格朗日方程 \(\nabla f = \lambda \nabla g\) 实际上要求 \(\nabla g \neq \mathbf{0}\)。如果在某点 \(\nabla g = \mathbf{0}\)（约束曲面在该点"奇异"），方程组可能漏掉它，必须单独检查。在标准教材题里这种情形几乎不出现，但应用问题里偶尔会撞上。

12.2 几何意义：等值线相切于约束曲线

拉格朗日条件 \(\nabla f = \lambda \nabla g\) 写出来是代数式，但它的灵魂是几何。把这件事想透一遍，方法本身就不再神秘——你甚至可以在没有公式的题目里靠肉眼直接看出答案。

设想约束 \(g(x, y) = 0\) 是平面上一条曲线 \(C\)。我们要找 \(f\) 限制在 \(C\) 上的极值点。同时把 \(f\) 的等值线 \(f(x, y) = c\) 画在同一张图上——\(c\) 取不同的值就得到一族水平曲线，可以想象成"地形图上的等高线"。问题就变成：在曲线 \(C\) 上走，等高线 \(c\) 的值能达到多大？

沿着 \(C\) 慢慢走，\(f\) 的值不停地变化——每经过一条等值线，\(f\) 就增大或减小一份。如果某时刻 \(C\) 与某条等值线横穿（不相切，斜着切过去），那么再走一小步，\(C\) 就会跨入另一条等值线——\(f\) 还在变。换言之，"横穿等值线"意味着 \(f\) 还在涨或跌，不可能是极值点。极值点必须出现在 \(C\) 与某条等值线相切的位置。在切点，再走一小步几乎没出发条等值线（要到二阶才能感知差异），\(f\) 的瞬时变化率是零——这正是极值的特征。

"相切"用梯度怎么写？两条曲线在某点相切，等价于它们在该点有相同的法方向。回想 10.3 节的核心事实：函数的梯度永远垂直于该函数的等值线。所以

  • \(\nabla g\) 垂直于约束曲线 \(C\)；
  • \(\nabla f\) 垂直于 \(f\) 的等值线。
两条曲线相切 \(\Longleftrightarrow\) 两个法向量平行 \(\Longleftrightarrow\) \(\nabla f\) 与 \(\nabla g\) 共线 \(\Longleftrightarrow\) 存在标量 \(\lambda\) 使得 \(\nabla f = \lambda \nabla g\)。这正是拉格朗日条件！

所以当你看到这条公式时，请脑子里立刻翻译为："在约束曲线上，目标函数的等值线必须与约束曲线相切。" 这种几何直觉在解题时极其有用——许多题目根本不需动笔列方程，只需在图上画出椭圆与一族双曲线，看它们什么时候"擦"在一起。

用这个直觉再看一下 12.1 的椭圆题：约束 \(C\) 是椭圆 \(x^2 + 4y^2 = 4\)，目标 \(f = xy\) 的等值线 \(xy = c\) 是一族双曲线。当 \(c\) 很小时，双曲线躲在远处，与椭圆相离；\(c\) 增大，双曲线靠近椭圆；到某个临界 \(c = c^\ast\)，双曲线擦到椭圆——这就是最大值的位置。再增大 \(c\)，双曲线超出椭圆，相交不再可能。所以 \(c^\ast\) 就是 \(f\) 在椭圆上的最大值——前一节算出来正是 \(c^\ast = 1\)。

从这个角度也可以解释一些"奇怪的特殊点"。在情形 A 里我们得到候选点 \((0, \pm 1)\)，\(f = 0\)。这两点几何上是椭圆的"南北两顶点"。在那里 \(\nabla f = \langle y, x\rangle = \langle \pm 1, 0\rangle\) 沿水平方向，而 \(\nabla g = \langle 2x, 8y\rangle = \langle 0, \pm 8\rangle\) 沿铅直方向——它们彼此垂直，不"平行"。但等等，$x = 0$ 时 \(\nabla f \cdot \langle 0, 1\rangle = x = 0\)；这意味着 \(\nabla f\) 在 \(C\) 切方向上的分量为零——也是"沿 \(C\) 走 \(f\) 不变"的状态。所以那里仍然是临界点，只不过对应"鞍点式"或"局部稳定但非极值"的情形。这是拉格朗日方法的特点：它给出所有候选，区分"极大、极小、其他"要靠最后的函数值比较。

另一种几何视角是地形比喻。把 \(z = f(x, y)\) 看作一片山地的高度图，\(C\) 是山上一条标好的小径。沿着小径走，海拔不断变化，最高处和最低处就是带约束的极值。在最高点，小径必须和某一条等高线相切——否则你再走一步就会离开等高线，要么登高要么下降，与"最高"矛盾。这个比喻里"等高线"对应 \(f\) 的等值线，"小径"对应 \(C\)，"切于等高线"就是拉格朗日条件。

把这层几何记牢之后，下一步去看两个约束的几何，就毫不费力。

12.3 两个约束

把场景升到三维。设有两个约束 \(g(x, y, z) = 0\) 与 \(h(x, y, z) = 0\)，每个约束分别定义一张曲面；它们的交集通常是一条空间曲线 \(C\)。问题是：\(f(x, y, z)\) 限制到 \(C\) 上的极值在哪儿？

几何直觉照旧："在极值点处，\(f\) 沿 \(C\) 的瞬时变化率为零；这等价于 \(\nabla f\) 与 \(C\) 垂直。" 但这次"\(C\) 的法方向"不再是单一的方向——三维空间里一条曲线在某点的法向是一整张平面（叫做法平面）。这张法平面由谁张成？由两个约束各自的梯度 \(\nabla g\) 与 \(\nabla h\) 张成（它们都垂直于自己的曲面，因而都垂直于 \(C\)，因而都在 \(C\) 的法平面里；通常它们线性独立，恰好张成整个法平面）。

所以"在极值点处 \(\nabla f\) 与 \(C\) 垂直"翻译成：\(\nabla f\) 必须落在 \(\nabla g\) 与 \(\nabla h\) 张成的法平面里。换言之，存在两个标量 \(\lambda\)、\(\mu\)，使得 \[ \boxed{\ \nabla f = \lambda \nabla g + \mu \nabla h, \quad g = 0, \quad h = 0.\ } \] 这就是两约束拉格朗日条件。两个新加的标量 \(\lambda, \mu\) 各对应一个约束。展开成分量是五个方程（三个梯度方程 + 两个约束），五个未知（\(x, y, z, \lambda, \mu\)）。

下面用一个例子完整跑一遍。求 \(f(x, y, z) = x + 2y + 3z\) 在两个约束下的极值：\(g = x + y + z - 1 = 0\)（一张平面）与 \(h = x^2 + y^2 + z^2 - 4 = 0\)（一张球面）。两曲面相交得到一个圆——这就是 \(C\)。

梯度：\(\nabla f = \langle 1, 2, 3\rangle\)，\(\nabla g = \langle 1, 1, 1\rangle\)，\(\nabla h = \langle 2x, 2y, 2z\rangle\)。方程： \[ 1 = \lambda + 2\mu x, \quad 2 = \lambda + 2\mu y, \quad 3 = \lambda + 2\mu z, \quad x + y + z = 1, \quad x^2 + y^2 + z^2 = 4. \] 第二式减第一式：\(1 = 2\mu(y - x)\)；第三式减第二式：\(1 = 2\mu(z - y)\)。两个差相等推出 \(y - x = z - y\)，即 \(2y = x + z\)。

由约束 \(x + y + z = 1\) 与 \(2y = x + z\) 联立：\(3y = 1\)，\(y = 1/3\)，\(x + z = 2/3\)。代入第二个约束：\(x^2 + 1/9 + z^2 = 4\)，\(x^2 + z^2 = 35/9\)。由恒等式 \((x + z)^2 = x^2 + 2xz + z^2\)，\(4/9 = 35/9 + 2xz\)，\(xz = -31/18\)。

现在 \(x, z\) 是和为 \(2/3\)、积为 \(-31/18\) 的一对数，即二次方程 \(t^2 - (2/3) t - 31/18 = 0\) 的根。判别式 \(4/9 + 4 \cdot 31/18 = 4/9 + 62/9 = 66/9\)，根为 \(t = (2 \pm \sqrt{66})/6\)。两个候选点： \[ \Bigl(\frac{2 + \sqrt{66}}{6},\ \frac{1}{3},\ \frac{2 - \sqrt{66}}{6}\Bigr)\quad \text{与}\quad \Bigl(\frac{2 - \sqrt{66}}{6},\ \frac{1}{3},\ \frac{2 + \sqrt{66}}{6}\Bigr). \] 代入 \(f = x + 2y + 3z\)。利用 \(x + z = 2/3\) 与 \(y = 1/3\)： \[ f = (x + z) + 2y + 2z = 2/3 + 2/3 + 2z = 4/3 + 2z. \] \(z\) 较大时 \(f\) 较大：\(z = (2 + \sqrt{66})/6\) 给 \(f = 4/3 + (2 + \sqrt{66})/3 = (6 + \sqrt{66})/3\)；\(z\) 较小时 \(f = (6 - \sqrt{66})/3\)。所以 \(f\) 在交线 \(C\) 上的最大、最小值分别为 \((6 \pm \sqrt{66})/3\)。

代数确实有一点重，但结构非常清晰：写出三条梯度方程加上两条约束，巧用对称（这里是"两两相减"），最后求解一元（或二元）方程组。两约束情形里"两两相减"是常用技巧——它直接给出"\(\nabla f\) 在 \(\nabla g\) 与 \(\nabla h\) 张成的平面里"这一几何条件的代数表达，常常能消去 \(\lambda\) 与 \(\mu\)。

再举一个简洁的例子。求 \(f(x, y, z) = x^2 + y^2 + z^2\)（即与原点距离的平方）在两条约束下的最小：\(x + y + z = 1\)、\(x + 2y + 3z = 6\)。几何上是要找两张平面交线（一条直线）上离原点最近的点。 \[ \nabla f = \langle 2x, 2y, 2z\rangle, \quad \nabla g = \langle 1, 1, 1\rangle, \quad \nabla h = \langle 1, 2, 3\rangle. \] 方程：\(2x = \lambda + \mu\)，\(2y = \lambda + 2\mu\)，\(2z = \lambda + 3\mu\)。可以看出 \(2y - 2x = \mu\)，\(2z - 2y = \mu\)，所以 \(y - x = z - y\)，即 \(2y = x + z\)。结合约束 \(x + y + z = 1\) 和 \(x + 2y + 3z = 6\)：第一式给 \(3y = 1\)（如果 \(2y = x + z\)），但代入 \(y = 1/3\) 得 \(x + z = 2/3\)。由 \(x + 2y + 3z = 6\)：\(x + 2/3 + 3z = 6\)，\(x + 3z = 16/3\)。结合 \(x + z = 2/3\)：相减得 \(2z = 14/3\)，\(z = 7/3\)，\(x = 2/3 - 7/3 = -5/3\)。最近点 \((-5/3, 1/3, 7/3)\)，最小距离平方 \(25/9 + 1/9 + 49/9 = 75/9 = 25/3\)。

在这种"求空间直线到原点的最近距离"问题里，拉格朗日方法的结构和线性代数里的"投影到子空间正交补"其实是同一件事——\(\nabla f = 2\langle x, y, z\rangle\) 与 \(\langle 1, 1, 1\rangle\)、\(\langle 1, 2, 3\rangle\) 共面，即 \((x, y, z)\) 在两条约束法向张成的平面里——这正是"原点到直线的垂足"的代数条件。两套语言、同一回事。

到此为止，我们完成了多元微分学的全部主线：偏导、链式法则、方向导数与梯度、切平面与近似、临界点与极值、拉格朗日乘数法。把它们串起来，多元微分学其实只在做一件事：用线性近似（一阶）和二次近似（二阶）来读懂多元函数在各种几何约束下的局部行为。从下一章（第 13 章）起，我们转入第二部分——多元积分学，先从矩形区域上的二重积分开始。
