第 14 章 一般区域上的二重积分

实际中要积的区域很少是矩形。两条抛物线之间的"叶状"片区，圆盘的一部分，由直线和曲线共同围出的三角形片，以及各种由不等式自然定义的奇形区域——这些才是我们日常遇到的形状。本章把第 13 章的方法推广到这种一般情况。

方法本身没变：仍然是富比尼把二重积分化为两次一元积分。变的是积分上下限的写法。在矩形上，外层和内层的上下限都是常数；在一般区域上，内层的上下限会变成"外层变量的函数"——因为区域的边界是曲线，沿不同的扫描方向走入走出区域的位置依赖于你站在哪。这一章的功夫，几乎全在画区域、判形状、写上下限这三件事上。请准备好你的草稿纸——在这一章里，不画图就开算是新手最常犯的错误。

14.1 I 型与 II 型区域

把区域 \(D\) 想成平面上一片被曲线围成的"地块"。在直角坐标里，一般有两种最自然的描述方式，分别叫 I 型区域和 II 型区域。

I 型区域：上下被两条曲线 \(y = g_1(x)\)、\(y = g_2(x)\) 夹住

当区域 \(D\) 长得像这样：

\[D = \{(x, y) : a \le x \le b,\ g_1(x) \le y \le g_2(x)\},\]

我们叫它 I 型。脑子里的画面是这样：在 \(x\) 轴上，\(D\) 投影为区间 \([a, b]\)；对每一个固定的 \(x \in [a, b]\)，画一条竖直的扫描线穿过 \(D\)，这条线进入 \(D\) 的位置是 \(y = g_1(x)\)，离开 \(D\) 的位置是 \(y = g_2(x)\)。换言之，\(D\) 的下边界曲线是 \(y = g_1(x)\)，上边界曲线是 \(y = g_2(x)\)，左右边界是两条竖直直线 \(x = a\) 和 \(x = b\)（这两条线在 \(g_1(x) = g_2(x)\) 处可能退化成两个角，让左右边界缩成一个点）。

I 型区域上的二重积分，沿"先竖直扫，再水平叠加"的方向写：

\[\boxed{\iint_D f(x, y)\,dA = \int_a^b \int_{g_1(x)}^{g_2(x)} f(x, y)\,dy\,dx.}\]

两个细节请记牢：内层的上下限 \(g_1(x), g_2(x)\) 是外层变量 \(x\) 的函数，而外层的上下限 \(a, b\) 是常数。如果你写出来的形式不是这样——比如内外都是常数，或者外层上下限里出现了 \(y\)——那就一定写错了。

II 型区域：左右被两条曲线 \(x = h_1(y)\)、\(x = h_2(y)\) 夹住

另一种描述把横竖交换。当区域长这样：

\[D = \{(x, y) : c \le y \le d,\ h_1(y) \le x \le h_2(y)\},\]

我们叫它 II 型。画面是这样：在 \(y\) 轴上，\(D\) 投影为区间 \([c, d]\)；对每一个固定的 \(y \in [c, d]\)，画一条水平的扫描线穿过 \(D\)，这条线进入 \(D\) 的位置是 \(x = h_1(y)\)，离开 \(D\) 的位置是 \(x = h_2(y)\)。所以 \(D\) 的左边界曲线是 \(x = h_1(y)\)，右边界曲线是 \(x = h_2(y)\)。

积分按"先水平扫，再竖直叠加"写：

\[\boxed{\iint_D f(x, y)\,dA = \int_c^d \int_{h_1(y)}^{h_2(y)} f(x, y)\,dx\,dy.}\]

同样地，内层上下限是外层变量 \(y\) 的函数，外层上下限是常数。

I 型 vs II 型：怎么选

许多区域同时是 I 型和 II 型——比如圆盘、矩形、三角形。这种时候你拥有自由：选哪种取决于哪种让被积函数更容易积。有些区域则只能写成其中一种（比如一个有"凹陷"或"中间断开"的形状只能拆成几块再分别处理，但本书的所有例题都不会让你遇到那种麻烦）。

判断"哪一种是 I 型、哪一种是 II 型"的最快方法，是看你画出区域后，沿哪条扫描方向能一刀穿到底而不在中途冒出再进入。如果竖直扫描线总是从同一条曲线进、同一条曲线出，那就是 I 型；如果水平扫描线满足这条性质，那就是 II 型。

14.2 设上下限：从画图到写积分

这是这一章最重要、也最容易翻车的一步。请把下面这套节奏练成本能：

  1. 把区域 \(D\) 画在草稿纸上。把所有给出的边界曲线画清楚，把它们的交点求出来。
  2. 决定先积 \(x\) 还是 \(y\)：也就是把 \(D\) 看成 I 型还是 II 型。挑哪种取决于哪种内积分更好做、哪种边界曲线更容易写。
  3. 对你最后积的那个变量（外层变量），找它在 \(D\) 上的全部范围 \([u_\min, u_\max]\)——这是常数上下限。
  4. 对你先积的那个变量（内层变量），画一条沿这个变量方向的扫描线穿过 \(D\)，求出它进入 \(D\) 与离开 \(D\) 时的位置。这两个位置都是外层变量的函数，就是内层上下限。

下面通过几道题把这套节奏走熟。

例 1. 求 \(\displaystyle\iint_D x\,dA\)，其中 \(D\) 是由 \(y = x^2\) 与 \(y = 2x\) 围成的区域。

第一步，画图。两条曲线在哪相交？解 \(x^2 = 2x\)，即 \(x(x - 2) = 0\)，给出 \(x = 0\) 与 \(x = 2\)，对应的 \(y\) 是 \(0\) 与 \(4\)。两交点 \((0, 0)\) 和 \((2, 4)\)。在区间 \(x \in (0, 2)\) 内验证一下哪条在上：取 \(x = 1\)，\(y = x^2 = 1\)，\(y = 2x = 2\)；直线 \(y = 2x\) 在抛物线 \(y = x^2\) 上方。所以区域夹在抛物线（下）与直线（上）之间，沿着 \(x\) 从 \(0\) 走到 \(2\)。

第二步，按 I 型设上下限。\(x \in [0, 2]\)；对每个 \(x\)，竖直扫描线从下边界 \(y = x^2\) 进入区域，从上边界 \(y = 2x\) 离开。所以

\[\iint_D x\,dA = \int_0^2 \int_{x^2}^{2x} x\,dy\,dx = \int_0^2 x \cdot (2x - x^2)\,dx = \int_0^2 (2x^2 - x^3)\,dx.\]

算一下：

\[\int_0^2 (2x^2 - x^3)\,dx = \frac{2x^3}{3} - \frac{x^4}{4}\Big|_0^2 = \frac{16}{3} - 4 = \frac{4}{3}.\]

第三步（验证），按 II 型再做一次。\(y\) 在区域上的范围是 \([0, 4]\)。对每个 \(y\)，画一条水平扫描线穿过区域，看它从哪条曲线进、从哪条曲线出。把 \(y = x^2\) 反解（取右支）得 \(x = \sqrt{y}\)，把 \(y = 2x\) 反解得 \(x = y/2\)。在 \(y \in (0, 4)\) 内取一个值看看，例如 \(y = 1\)：直线给 \(x = 1/2\)，抛物线（右支）给 \(x = 1\)。所以水平线在 \(x = y/2\)（左侧的直线）进入区域，在 \(x = \sqrt{y}\)（右侧的抛物线）离开。这就是 II 型的上下限：

\[\iint_D x\,dA = \int_0^4 \int_{y/2}^{\sqrt{y}} x\,dx\,dy = \int_0^4 \frac{x^2}{2}\Big|_{y/2}^{\sqrt{y}}\,dy = \int_0^4 \left(\frac{y}{2} - \frac{y^2}{8}\right)dy.\]

算一下：

\[\int_0^4 \left(\frac{y}{2} - \frac{y^2}{8}\right)dy = \frac{y^2}{4} - \frac{y^3}{24}\Big|_0^4 = 4 - \frac{64}{24} = 4 - \frac{8}{3} = \frac{4}{3}.\]

两种顺序得到同一个 \(\dfrac{4}{3}\)，对得上。这道题告诉你：不少区域两种顺序都能写，难度差不多。下一道题就不一样了。

例 2. 求 \(\displaystyle\iint_D xy\,dA\)，其中 \(D\) 是由 \(y = x\)、\(y = 0\)、\(x + y = 2\) 围成的三角形。

画图。三条边：\(y = x\)（过原点的 45 度线）、\(y = 0\)（\(x\) 轴）、\(x + y = 2\)（截距为 2 的斜线）。求两两交点：\(y = x\) 与 \(y = 0\) 在原点 \((0, 0)\) 相交；\(y = 0\) 与 \(x + y = 2\) 在 \((2, 0)\) 相交；\(y = x\) 与 \(x + y = 2\) 解得 \(x = 1\)、\(y = 1\)，即 \((1, 1)\)。所以这是顶点 \((0, 0)\)、\((2, 0)\)、\((1, 1)\) 的三角形。

这种"三个不同侧的边"的三角形，其实写成 II 型最简单，因为对每一个固定的 \(y\)，水平扫描线总是从直线 \(x = y\)（左边）进入，从直线 \(x = 2 - y\)（右边，由 \(x + y = 2\) 反解）离开。\(y\) 的范围是 \([0, 1]\)。所以

\[\iint_D xy\,dA = \int_0^1 \int_y^{2 - y} xy\,dx\,dy = \int_0^1 y \cdot \frac{x^2}{2}\Big|_y^{2 - y}\,dy = \int_0^1 \frac{y}{2}\left[(2 - y)^2 - y^2\right]dy.\]

展开方括号 \((2 - y)^2 - y^2 = (4 - 4y + y^2) - y^2 = 4 - 4y\)，于是被积式变成 \(\dfrac{y}{2}(4 - 4y) = 2y - 2y^2\)。

\[\int_0^1 (2y - 2y^2)\,dy = y^2 - \frac{2y^3}{3}\Big|_0^1 = 1 - \frac{2}{3} = \frac{1}{3}.\]

如果按 I 型来做，会发现需要把区域劈成两块处理：在 \(0 \le x \le 1\) 时上界是 \(y = x\)，在 \(1 \le x \le 2\) 时上界是 \(y = 2 - x\)（因为这时左侧那条 \(y = x\) 已经走出去了）。劈成两块当然能做，但工作量翻倍。这就是为什么看一眼区域，让选择服务于几何是这种题的灵魂。

例 3. 求由 \(y = \sqrt{x}\)、\(y = x^2\) 围成的区域的面积。

面积就是被积函数为 \(1\) 的二重积分：\(\text{Area}(D) = \iint_D 1\,dA\)。先求交点：\(\sqrt{x} = x^2\)，两边平方得 \(x = x^4\)，解出 \(x = 0\) 或 \(x = 1\)。在 \(x \in (0, 1)\) 内取一个值，例如 \(x = 1/4\)：\(\sqrt{x} = 1/2\)，\(x^2 = 1/16\)，所以 \(\sqrt{x}\) 在上方。

I 型：\(x \in [0, 1]\)，\(y \in [x^2, \sqrt{x}]\)。

\[\text{Area} = \int_0^1 \int_{x^2}^{\sqrt{x}} dy\,dx = \int_0^1 (\sqrt{x} - x^2)\,dx = \frac{2}{3}x^{3/2} - \frac{x^3}{3}\Big|_0^1 = \frac{2}{3} - \frac{1}{3} = \frac{1}{3}.\]

这种"夹在两条曲线之间"的面积题，很多教材在一元微积分里就讲过；你会发现它本质上正是二重积分 \(\iint_D 1\,dA\) 在 I 型上的展开。换一种说法：二重积分对应的"面积"概念，把一元里"曲线下面积"的故事统一了起来。

14.3 交换积分次序：用新顺序绕开"算不出"的内积分

有时候你写出一个累次积分，一动笔就发现内层积分根本写不出闭式——里头出现了 \(e^{x^2}\)、\(\sin(x^2)\)、\(\dfrac{\sin x}{x}\) 这一类没有初等原函数的家伙。这时候，换一种积分次序常常能让原本无解的题变成两步就完。

这种"换顺序"的技术，关键还是画区域。给定一个用 I 型上下限写出来的累次积分，你可以反推出对应的区域 \(D\)，然后把它当作 II 型重新写出来——内外变量互换，上下限相应改写。重写之后再看：内层有没有可能积出来。常常奇迹就发生在这一步。

把它做成一套配方：

  1. 从原积分的上下限读出区域 \(D\)，把 \(D\) 仔细画在草稿纸上。
  2. 把 \(D\) 用另一种类型重新参数化（外层和内层互换）。
  3. 写出新的累次积分，求解。

下面四道题把这套节奏练熟。

例 4. 计算 \(\displaystyle\int_0^1 \int_y^1 e^{x^2}\,dx\,dy\)。

第一念头：\(\int e^{x^2}\,dx\) 没有初等闭式。所以按现有顺序内积分根本写不出来。换顺序。

从给出的上下限读出区域：\(0 \le y \le 1\)，\(y \le x \le 1\)。也就是 \(D\) 是 \(\{(x, y) : 0 \le y \le x \le 1\}\)，画出来是顶点 \((0, 0)\)、\((1, 0)\)、\((1, 1)\) 的三角形（直角在 \((1, 0)\)）。

把它重新当作 I 型：\(x \in [0, 1]\)；对每个 \(x\)，\(y\) 从 \(0\) 走到 \(x\)。所以

\[\int_0^1 \int_y^1 e^{x^2}\,dx\,dy = \int_0^1 \int_0^x e^{x^2}\,dy\,dx = \int_0^1 x e^{x^2}\,dx.\]

(内积分 \(\int_0^x e^{x^2}\,dy = e^{x^2} \cdot x\)，因为被积式不含 \(y\)。) 现在用 \(u = x^2\)、\(du = 2x\,dx\)，

\[\int_0^1 x e^{x^2}\,dx = \frac{1}{2}\int_0^1 e^u\,du = \frac{1}{2}(e - 1).\]

原本无解，换序后两行解决——这就是这一节的全部魔法。

例 5. 计算 \(\displaystyle\int_0^1 \int_{\sqrt{x}}^1 \sin(y^3)\,dy\,dx\)。

同样，\(\int \sin(y^3)\,dy\) 没有初等闭式。换顺序。区域：\(0 \le x \le 1\)，\(\sqrt{x} \le y \le 1\)。也就是 \(D = \{(x, y) : 0 \le x \le 1,\ \sqrt{x} \le y \le 1\}\)。下边界 \(y = \sqrt{x}\) 等价于 \(x = y^2\)。

把 \(D\) 看成 II 型：对 \(y \in [0, 1]\)，水平扫描线从 \(x = 0\) 进入区域，从 \(x = y^2\) 离开（因为再往右就跨过了 \(y = \sqrt{x}\) 这条边界）。所以

\[\int_0^1 \int_{\sqrt{x}}^1 \sin(y^3)\,dy\,dx = \int_0^1 \int_0^{y^2} \sin(y^3)\,dx\,dy = \int_0^1 y^2 \sin(y^3)\,dy.\]

用 \(u = y^3\)、\(du = 3y^2\,dy\)：

\[\int_0^1 y^2 \sin(y^3)\,dy = \frac{1}{3}\int_0^1 \sin u\,du = \frac{1}{3}(1 - \cos 1).\]

又是一例：原本无解，换序立刻可解。注意每次"换序"前都要先把区域画出来——这是这一节里你应该养成的肌肉记忆。

例 6. 计算 \(\displaystyle\int_0^4 \int_{x/2}^2 e^{y^2}\,dy\,dx\)。

同样的理由：\(\int e^{y^2}\,dy\) 没有初等闭式。换顺序。区域：\(0 \le x \le 4\)，\(x/2 \le y \le 2\)。下边界 \(y = x/2\) 反过来说就是 \(x = 2y\)。所以 \(D\) 是顶点 \((0, 0)\)、\((4, 2)\)、\((0, 2)\) 的三角形。

把它当 II 型：\(y \in [0, 2]\)；对每个 \(y\)，水平扫描线从 \(x = 0\) 进入，从 \(x = 2y\) 离开。

\[\int_0^4 \int_{x/2}^2 e^{y^2}\,dy\,dx = \int_0^2 \int_0^{2y} e^{y^2}\,dx\,dy = \int_0^2 2y e^{y^2}\,dy.\]

用 \(u = y^2\)：

\[\int_0^2 2y e^{y^2}\,dy = \int_0^4 e^u\,du = e^4 - 1.\]

例 7. 计算 \(\displaystyle\int_0^1 \int_{3y}^3 e^{x^2}\,dx\,dy\)。

区域：\(0 \le y \le 1\)，\(3y \le x \le 3\)。也就是 \(\{(x, y) : 0 \le y \le 1,\ 3y \le x \le 3\}\)。下边界 \(x = 3y\)，即 \(y = x/3\)。所以 \(D\) 是顶点 \((0, 0)\)、\((3, 0)\)、\((3, 1)\) 的三角形。

当 I 型：\(x \in [0, 3]\)，\(y \in [0, x/3]\)。

\[\int_0^1 \int_{3y}^3 e^{x^2}\,dx\,dy = \int_0^3 \int_0^{x/3} e^{x^2}\,dy\,dx = \int_0^3 \frac{x}{3} e^{x^2}\,dx.\]

用 \(u = x^2\)：

\[\int_0^3 \frac{x}{3} e^{x^2}\,dx = \frac{1}{6}\int_0^9 e^u\,du = \frac{e^9 - 1}{6}.\]

四道换序题练下来，你应该已经看出共同的节奏：原积分按某个顺序写出来内积分无解 → 把区域画出来 → 重新按另一种类型描述 → 换序后内积分秒解。在考试与作业里见到 \(e^{x^2}\)、\(\sin(x^2)\)、\(\cos(x^2)\)、\(\dfrac{\sin x}{x}\)、\(e^{y^2}\)、\(\sqrt{1 + x^3}\) 之类没有初等原函数的家伙，第一反应就是：该不该换序？

14.4 综合例题

下面这两道题把本章的几件武器一起用起来。

例 8. 设 \(D\) 是由 \(y = x\)、\(y = 0\)、\(x = \pi\) 围成的三角形。求 \(\displaystyle\iint_D \frac{\sin x}{x}\,dA\)。

注意 \(\dfrac{\sin x}{x}\) 关于 \(x\) 没有初等原函数。所以一旦 \(x\) 出现在外层积分里（即先 \(y\) 后 \(x\)），最后那个外积分照样积不出来；但如果先对 \(y\) 积，因为 \(\dfrac{\sin x}{x}\) 关于 \(y\) 是常数，那一步几乎没东西要算，只乘上一个 \(y\) 的区间长度。再外层对 \(x\) 积，刚好把分母上的 \(x\) 抵消掉。

把 \(D\) 写成 I 型：\(x \in [0, \pi]\)，\(y \in [0, x]\)。

\[\iint_D \frac{\sin x}{x}\,dA = \int_0^\pi \int_0^x \frac{\sin x}{x}\,dy\,dx = \int_0^\pi \frac{\sin x}{x} \cdot x\,dx = \int_0^\pi \sin x\,dx = 2.\]

这道题就是教材上的"反序消去分母"经典——一道你应该会立刻识别出的题型。

例 9. 求 \(\displaystyle\iint_D y\,dA\)，其中 \(D\) 是由 \(y = x - 1\) 与 \(y^2 = 2x + 6\) 围成的区域。

画图。直线 \(y = x - 1\)，即 \(x = y + 1\)。抛物线 \(y^2 = 2x + 6\)，即 \(x = (y^2 - 6)/2\)，对称轴是 \(y = 0\)，顶点在 \((-3, 0)\)，开口朝右。求两条线的交点：把 \(x = y + 1\) 代进抛物线，\(y^2 = 2(y + 1) + 6 = 2y + 8\)，即 \(y^2 - 2y - 8 = 0\)，解出 \(y = 4\) 或 \(y = -2\)。对应 \(x = 5\) 或 \(x = -1\)。两交点 \((5, 4)\) 与 \((-1, -2)\)。

这个区域写 I 型很难（因为下边界沿着 \(x\) 走的时候要分段：\(x \in [-3, -1]\) 时上下边界都是抛物线的两支；\(x \in [-1, 5]\) 时上界是抛物线上支、下界是直线）；但写 II 型简洁多了：对 \(y \in [-2, 4]\)，水平扫描线从抛物线 \(x = (y^2 - 6)/2\) 进入区域，从直线 \(x = y + 1\) 离开（在 \(y \in (-2, 4)\) 内验证 \(y + 1 > (y^2 - 6)/2\)，例如 \(y = 0\) 时 \(1 > -3\)，对的）。

\[\iint_D y\,dA = \int_{-2}^4 \int_{(y^2 - 6)/2}^{y + 1} y\,dx\,dy = \int_{-2}^4 y\left[(y + 1) - \frac{y^2 - 6}{2}\right]dy.\]

方括号化简：\((y + 1) - \dfrac{y^2 - 6}{2} = \dfrac{2y + 2 - y^2 + 6}{2} = \dfrac{-y^2 + 2y + 8}{2}\)。整体被积式

\[y \cdot \frac{-y^2 + 2y + 8}{2} = \frac{-y^3 + 2y^2 + 8y}{2}.\]

\[\int_{-2}^4 \frac{-y^3 + 2y^2 + 8y}{2}\,dy = \frac{1}{2}\left[-\frac{y^4}{4} + \frac{2y^3}{3} + 4y^2\right]_{-2}^4.\]

在 \(y = 4\)：\(-\dfrac{256}{4} + \dfrac{128}{3} + 64 = -64 + \dfrac{128}{3} + 64 = \dfrac{128}{3}\)。

在 \(y = -2\)：\(-\dfrac{16}{4} + \dfrac{-16}{3} + 16 = -4 - \dfrac{16}{3} + 16 = 12 - \dfrac{16}{3} = \dfrac{20}{3}\)。

差是 \(\dfrac{128 - 20}{3} = \dfrac{108}{3} = 36\)。再乘前面的 \(\dfrac{1}{2}\) 得 \(18\)。所以 \(\iint_D y\,dA = 18\)。

这道题示范了"区域形状决定 I/II 型选择"的判断力——遇到开口向右的抛物线，几乎总该用 II 型；遇到开口向上的抛物线，几乎总该用 I 型。
