第 16 章 三重积分

到目前为止，我们已经把一元积分推广成了二重积分。一元积分把"一根线段上每一点的值"加起来；二重积分把"一片区域里每一点的值"加起来。下一步几乎是不可避免的：把"一块立体里每一点的值"加起来。这就是三重积分。

但说到这里，有一件事必须坦白。一元积分有一幅清楚的图像：曲线下方的面积。二重积分也有：曲面下方的体积。三重积分理论上可以说成"四维'超曲面'下方的超体积"——可是没有人见过四维的东西，所以这条几何直观此刻就走到了尽头。我们不应假装它还在那里。三重积分的几何图像必须从别处借来。

幸运的是，那个"别处"非常实在。三重积分有两个最重要的几何与物理用途，足以撑起整章：

  1. 计算三维区域 \(E\) 的体积：\(\displaystyle V = \iiint_E 1\,dV\)。
  2. 计算密度为 \(\rho(x, y, z)\) 的物体的质量、质心、转动惯量等物理量。

事实上，绝大多数三重积分的实际用途都落在这两条上。你不必为"四维体积"操心。当你看见 \(\iiint_E f\,dV\) 时，最稳的心理图像就是"把 \(E\) 切成无数小盒子，每个小盒子里的 \(f\) 乘上小盒子的体积，再把它们全部加起来"。这幅"切成小盒子再求和"的图，将贯穿本章。

本章的任务有三件：先把累次三重积分的机器搭起来，再讨论怎样为一个具体立体选择最自然的积分次序，最后把这套机器用到体积、质量与质心的计算上。第 17 章里，我们再换上柱面坐标和球面坐标这两件更趁手的工具。

16.1 累次三重积分

让我们先把记号摆清楚。设 \(E\) 是三维空间中的一个区域，\(f(x, y, z)\) 是定义在 \(E\) 上的（足够好的）函数。\(f\) 在 \(E\) 上的三重积分记为

\[ \iiint_E f(x, y, z)\,dV, \]

这里 \(dV\) 是体积元素。在直角坐标里，\(dV = dx\,dy\,dz\)。其余的坐标系下 \(dV\) 长得不一样——这是第 17 章的主角。

它的定义思路和二重积分一模一样，只是升了一维。把 \(E\) 用一张三维网格切成无数小长方体，第 \(i\) 个小盒子的体积记作 \(\Delta V_i\)，在这个小盒子里挑一个样本点 \((x_i^*, y_i^*, z_i^*)\)。把每个小盒子里的函数值乘以小盒子体积再求和，就得到一个黎曼和

\[ \sum_{i} f(x_i^*, y_i^*, z_i^*)\,\Delta V_i. \]

当网格越切越细、所有 \(\Delta V_i \to 0\) 时，如果这个和有极限，那个极限就被定义为三重积分。这一切都是为了把"对每个小盒子里的 \(f\) 乘小盒子的体积再加起来"这件事讲清楚。

富比尼定理在三维下依旧有效——而且更慷慨。它告诉我们：在合理的条件下，\(\iiint_E f\,dV\) 可以化成一层套一层的累次积分。三个变量按什么顺序积都行，所以一共有六种次序：

\[ dx\,dy\,dz,\quad dx\,dz\,dy,\quad dy\,dx\,dz,\quad dy\,dz\,dx,\quad dz\,dx\,dy,\quad dz\,dy\,dx. \]

理论上六种次序的结果完全一样；实际上，根据立体的形状，会有一两种次序明显比其它四种好算。挑次序的本事，跟二维情形里的"画图、看哪个变量好积"是一脉相承的。

层层递进的上下限

三重积分最常见的几何配置是这样：底下有一片二维区域 \(D\)（在 \(xy\) 平面里），上面被两张曲面 \(z = \phi_1(x, y)\) 与 \(z = \phi_2(x, y)\) 像三明治一样夹着，构成一块立体

\[ E = \{(x, y, z) : (x, y) \in D,\ \phi_1(x, y) \le z \le \phi_2(x, y)\}. \]

这种"先在 \(xy\) 平面上挖出一块影子，再让 \(z\) 从一张曲面跑到另一张曲面"的描述，是三重积分里最常出现的一种结构。它的累次形式是

\[ \iiint_E f\,dV = \iint_D \left[\int_{\phi_1(x, y)}^{\phi_2(x, y)} f(x, y, z)\,dz\right] dA. \]

里面那一层是先把 \(z\) 积掉——把立体上方一根竖直针上的 \(f\) 值沿 \(z\) 加起来；外面那一层就是把得到的二元函数在 \(xy\) 平面的影子 \(D\) 上做二重积分（这正是第 14 章的内容）。

设上下限的关键，是把"最里层、中层、最外层"分清楚。一个稳妥的步骤是：

  1. 先画图。在脑中或纸上看清 \(E\) 长什么样：哪两张曲面是上下盖？影子区域 \(D\) 是什么形状？这一步省不了。
  2. 选一个积分顺序。最常见的策略是先把"竖直方向"那个变量积掉——也就是当立体被两张 "\(z = \) 函数" 的曲面夹住时，把 \(z\) 放最里面。但这只是默认；如果立体是被两张 "\(x = \) 函数" 夹住，就先积 \(x\)。
  3. 层层往外写上下限：最内层的上下限可以同时依赖另两个变量；中层的上下限只依赖最外层那个变量；最外层的上下限必须是常数。如果哪一层的上下限错位了，几乎可以肯定是次序选错或者画图没画清。

来看一个最经典的练手立体——单位四面体。考虑积分 \(\iiint_E z\,dV\)，其中 \(E\) 是由 \(z = 0\)（底）、\(z = 1 - x - y\)（顶）、\(x \ge 0\)、\(y \ge 0\) 围成的四面体。这正是第一卦限里被斜平面 \(x + y + z = 1\) 切下的那一块。

第一步，画图。把它的影子先看清：上面那张斜面与底面 \(z = 0\) 在哪里相交？令 \(1 - x - y = 0\)，得到 \(x + y = 1\)。所以 \(D\)（影子）就是 \(xy\) 平面里 \(x \ge 0\)、\(y \ge 0\)、\(x + y \le 1\) 的直角三角形。在 \(D\) 内每一点 \((x, y)\) 上，\(z\) 从 \(0\) 一直走到 \(1 - x - y\)。

第二步，写出累次积分：

\[ \iiint_E z\,dV = \int_0^1 \int_0^{1 - x} \int_0^{1 - x - y} z\,dz\,dy\,dx. \]

第三步，从最里面算起。把 \(z\) 从 \(0\) 积到 \(1 - x - y\)：

\[ \int_0^{1 - x - y} z\,dz = \frac{(1 - x - y)^2}{2}. \]

中层是把 \((1 - x - y)^2/2\) 关于 \(y\) 积。这一层稍微费一点笔，但只是单变量积分。令 \(u = 1 - x - y\)，则 \(du = -dy\)，当 \(y = 0\) 时 \(u = 1 - x\)，\(y = 1 - x\) 时 \(u = 0\)：

\[ \int_0^{1 - x} \frac{(1 - x - y)^2}{2}\,dy = \frac{1}{2}\int_{1 - x}^{0} u^2 (-du) = \frac{1}{2}\int_0^{1 - x} u^2\,du = \frac{(1 - x)^3}{6}. \]

最外层关于 \(x\)：

\[ \int_0^1 \frac{(1 - x)^3}{6}\,dx = \frac{1}{6}\cdot\frac{-(1 - x)^4}{4}\bigg|_0^1 = \frac{1}{6}\cdot\frac{1}{4} = \frac{1}{24}. \]

所以 \(\iiint_E z\,dV = \dfrac{1}{24}\)。这个数本身有点物理意义：四面体的体积是 \(1/6\)，所以这个积分除以体积就是 \(z\) 坐标的平均值，等于 \(1/4\)。这与四面体质心高度恰好为顶点高度的四分之一这一事实吻合。

下面再做一个稍带"曲面"味道的例子。考虑\(\iiint_E (x + y)\,dV\)，其中 \(E\) 由 \(z = 0\)、\(z = 4 - x^2\) 与 \(0 \le y \le 3\) 围成。

这次的影子并不在 \(xy\) 平面里——更自然的"夹"是上下两张曲面 \(z = 0\) 和 \(z = 4 - x^2\)，但只有当 \(4 - x^2 \ge 0\)，即 \(-2 \le x \le 2\) 时，这两张曲面才能围出体积。\(y\) 的范围已直接给出。所以

\[ \iiint_E (x + y)\,dV = \int_{-2}^{2} \int_0^3 \int_0^{4 - x^2} (x + y)\,dz\,dy\,dx. \]

最内层：\(\int_0^{4 - x^2}(x + y)\,dz = (x + y)(4 - x^2)\)。中层关于 \(y\)：\(\int_0^3 (x + y)(4 - x^2)\,dy = (4 - x^2)\left[3x + \tfrac{9}{2}\right]\)。最外层关于 \(x\)：注意 \(3x(4 - x^2) = 12x - 3x^3\) 是奇函数，在对称区间 \([-2, 2]\) 上积出来等于 \(0\)；而 \(\tfrac{9}{2}(4 - x^2)\) 是偶函数，

\[ \int_{-2}^{2}\frac{9}{2}(4 - x^2)\,dx = \frac{9}{2}\cdot 2 \int_0^2 (4 - x^2)\,dx = 9 \left(8 - \frac{8}{3}\right) = 9 \cdot \frac{16}{3} = 48. \]

这一题里"奇函数 + 对称区间 \(\Rightarrow\) 等于零"的小技巧出现了一次——以后我们会反复看到，尤其是在球或对称立体上的积分里。它是把三重积分变短的最便宜的方式之一。

再来一个把"先 \(x\) 后 \(z\) 后 \(y\)" 这种次序逼出来的例子。考虑 \(\iiint_E 1\,dV\)（即体积），\(E\) 是由 \(x = y^2\)、\(x = 4\)、\(z = 0\)、\(z = 5\) 围成的立体。

这个立体在 \(yz\) 平面上的影子是矩形 \([-2, 2] \times [0, 5]\)（因为 \(y^2 \le 4\) 给出 \(-2 \le y \le 2\)，而 \(z\) 已直接限定）。在每个 \((y, z)\) 上，\(x\) 从 \(y^2\) 跑到 \(4\)。所以最自然的次序是 \(dx\,dy\,dz\)：

\[ V = \int_0^5 \int_{-2}^{2} \int_{y^2}^{4} dx\,dy\,dz = \int_0^5 dz \int_{-2}^{2} (4 - y^2)\,dy = 5 \cdot \frac{32}{3} = \frac{160}{3}. \]

注意我们这里的"层层递进"完全没沿用 \(z\) 在最里面的默认；当立体显然是被两张 "\(x = \) 函数" 夹住时，把 \(x\) 放最里面才是自然的。次序服务于立体，不是反过来。

16.2 选择积分次序

选择积分次序的关键，跟二维情形完全是同一个原则：先画图，再选最容易的次序。三维下的画图比二维难一些——往往需要在脑子里旋转几下，或者在草稿纸上分别画出立体在 \(xy\)、\(xz\)、\(yz\) 三个平面上的投影——但这一步省不得。

有几条几乎从不出错的判据可以帮你下决心：

  • 如果立体被两张曲面"\(z = \) 函数"夹住，先把 \(z\) 积掉。最内层的 \(z\) 上下限就是这两张曲面，外面的二重积分用立体在 \(xy\) 平面上的影子来描述。
  • 如果立体被两张曲面"\(x = \) 函数"夹住，先把 \(x\) 积掉。最内层的 \(x\) 上下限就是这两张曲面，外面在 \(yz\) 平面上的影子上做二重积分。
  • 如果立体被两张曲面"\(y = \) 函数"夹住，先把 \(y\) 积掉，类推。
  • 所有都满足时，看哪一种描述能让累次积分的上下限最干净——常数好过线性，线性好过更复杂的式子。

举一个会反复出现的例子。立体 \(E\) 由抛物面 \(z = x^2 + y^2\)（下方）与平面 \(z = 1\)（上方）围成，求它的体积。

先 \(z\) 后 \((x, y)\) 是首选。在影子 \(D\) 上每一点 \((x, y)\)，\(z\) 从 \(x^2 + y^2\) 跑到 \(1\)。影子是把上下两张曲面"对齐"得到的：当 \(x^2 + y^2 = 1\) 时两张曲面相交，所以 \(D\) 是单位圆盘 \(x^2 + y^2 \le 1\)。

\[ V = \iint_D (1 - x^2 - y^2)\,dA. \]

这二重积分最该换到极坐标——因为 \(D\) 是圆盘，被积函数里出现 \(x^2 + y^2\)。换过去：

\[ V = \int_0^{2\pi}\int_0^1 (1 - r^2)\,r\,dr\,d\theta = 2\pi \cdot \left[\frac{r^2}{2} - \frac{r^4}{4}\right]_0^1 = 2\pi \cdot \frac{1}{4} = \frac{\pi}{2}. \]

反过来若硬选 "先 \(x\) 后 \(z\) 后 \(y\)" 这样的次序，最内层立刻就要写 \(-\sqrt{z - y^2} \le x \le \sqrt{z - y^2}\)——能算，但累人。正确的次序就是省力的次序。

下一个例子展示的是"换次序也可以救一个看上去算不出的积分"。设

\[ I = \int_0^1 \int_x^1 \int_0^y e^{y^2}\,dz\,dy\,dx. \]

这是直接给出的累次积分，不是从立体反推上下限。最里层关于 \(z\)，被积函数与 \(z\) 无关，结果是 \(y\,e^{y^2}\)。第二层是 \(\int_x^1 y\,e^{y^2}\,dy\)——令 \(u = y^2\) 可以做出来，等于 \((e - e^{x^2})/2\)。可现在最外层 \(\int_0^1 (e - e^{x^2})/2\,dx\) 出现了 \(e^{x^2}\)，而 \(\int e^{x^2}\,dx\) 没初等表达式。死路。

救它的办法是换次序。把上下限读成立体描述：\(0 \le x \le 1\)，\(x \le y \le 1\)，\(0 \le z \le y\)。改先做 \(x\) 后 \(y\)：在 \(xy\) 平面里，区域是 \(0 \le x \le y \le 1\)，所以 \(0 \le y \le 1\)，\(0 \le x \le y\)。新次序写出来是

\[ I = \int_0^1 \int_0^y \int_0^y e^{y^2}\,dz\,dx\,dy = \int_0^1 e^{y^2}\cdot y \cdot y\,dy = \int_0^1 y^2 e^{y^2}\,dy. \]

嗯，这下也不轻松——但比先前那个 \(\int e^{x^2}\,dx\) 有救：用一次分部积分加上 \(\int y\,e^{y^2}\,dy = e^{y^2}/2\) 就能结清。重要的不是这道题的最终数字，而是那个教训：当一种次序把你逼进死角时，换一个次序常常豁然开朗。

16.3 应用：体积、质量与质心

三重积分的两条最常见用途——体积和物理量——都共用同一种"切成小盒子，把贡献加起来"的思维。

立体 \(E\) 的体积是被积函数取常数 \(1\) 时的特例：

\[ \boxed{V = \iiint_E 1\,dV.} \]

把这个公式的来源想清楚：每个小盒子的"贡献"就是它自己的体积 \(\Delta V_i\)，把所有 \(\Delta V_i\) 加起来当然就是 \(E\) 的总体积。

稍复杂一点的情形，是 \(E\) 中物体的密度依空间位置而变。设 \(\rho(x, y, z)\) 表示每单位体积的质量密度（注意：从这一节起 \(\rho\) 临时充当一个新角色，与球面坐标里的 \(\rho\) 同字母不同义，我会在第 17 章里再做提醒）。每个小盒子里的质量约为 \(\rho(x_i^*, y_i^*, z_i^*)\,\Delta V_i\)，把所有这些小贡献加起来就是物体的总质量：

\[ \boxed{m = \iiint_E \rho(x, y, z)\,dV.} \]

知道了质量，就能问"质量的平均位置"在哪里——这就是质心。质心的三个坐标 \((\bar x, \bar y, \bar z)\) 是质量加权下三个坐标的平均值：

\[ \boxed{\bar x = \frac{1}{m}\iiint_E x\,\rho\,dV,\quad \bar y = \frac{1}{m}\iiint_E y\,\rho\,dV,\quad \bar z = \frac{1}{m}\iiint_E z\,\rho\,dV.} \]

物理含义很直白：\(\bar x\) 是物体里所有点的 \(x\) 坐标按质量加权后的平均，\(\bar y, \bar z\) 同理。把物体悬挂在质心处，它在重力下能平衡——这是质心的力学定义，也是它最直接的物理意义。当密度均匀时（\(\rho\) 是常数），\(\rho\) 在分子分母里同时出现并约掉，质心就退化成几何上的形心，是纯粹由形状决定的。

顺手把另一组常用积分也写下来：关于坐标轴的转动惯量。绕 \(z\) 轴旋转时，每个小盒子离 \(z\) 轴的距离是 \(\sqrt{x^2 + y^2}\)，所以

\[ I_z = \iiint_E (x^2 + y^2)\,\rho\,dV, \]

类推 \(I_x = \iiint_E (y^2 + z^2)\rho\,dV\)、\(I_y = \iiint_E (x^2 + z^2)\rho\,dV\)。这些量在刚体力学里至关重要，但更重要的是它们都遵循同一个模板：把"每点要乘的小因子"乘上密度，再对整个立体做三重积分。三重积分这台机器，就是用来回答这一类问题的。

来把上一节那个抛物面与平面之间的立体重新拿来。已经知道 \(V = \pi/2\)。现在设密度 \(\rho \equiv 1\)，求质心的 \(z\) 坐标 \(\bar z\)（由对称性，\(\bar x = \bar y = 0\) 已经无须计算）。

此时 \(m = V = \pi/2\)，所以只需算 \(\iiint_E z\,dV\)。先把 \(z\) 积掉：

\[ \int_{x^2 + y^2}^{1} z\,dz = \frac{1 - (x^2 + y^2)^2}{2}. \]

外面的二重积分换到极坐标，区域是单位圆盘：

\[ \iiint_E z\,dV = \frac{1}{2}\int_0^{2\pi}\int_0^1 (1 - r^4)\,r\,dr\,d\theta = \frac{1}{2}\cdot 2\pi\cdot\left[\frac{r^2}{2} - \frac{r^6}{6}\right]_0^1 = \pi\cdot\frac{1}{3} = \frac{\pi}{3}. \]

所以

\[ \bar z = \frac{\pi/3}{\pi/2} = \frac{2}{3}. \]

这个数字值得停下来想一想。立体在 \(z \in [0, 1]\) 之间，几何中点是 \(z = 1/2\)。但质心 \(\bar z = 2/3\) 偏向上半部，原因很清楚：抛物面下方的立体——切成水平片来看——上方的圆片大、下方的圆片小，所以质量更"重"地集中在 \(z\) 大的一头。"质心高于几何中点"是这块立体形状的直接反映。

再来一个更具说服力的"质心"例子。考虑一个底面在 \(z = 0\) 处、半径为 \(R\)、高为 \(h\) 的实心圆锥，密度均匀。直觉告诉我们质心一定在轴线上 \(\bar x = \bar y = 0\)，且 \(\bar z\) 应该比 \(h/2\) 偏低，因为圆锥下宽上尖、质量集中在下面。但偏低多少？

把立体写出来：高度 \(z\) 处，水平截面是半径为 \(R(1 - z/h)\) 的圆。所以 \(D_z\) 是这个圆，面积为 \(\pi R^2 (1 - z/h)^2\)。体积

\[ V = \int_0^h \pi R^2 \left(1 - \frac{z}{h}\right)^2 dz = \pi R^2 \cdot \frac{h}{3} \]

（这正是熟悉的圆锥体积公式 \(\tfrac{1}{3}\pi R^2 h\)）。算 \(\iiint_E z\,dV\)，把"先 \(D_z\)，再 \(z\)"分开：

\[ \iiint_E z\,dV = \int_0^h z \cdot \pi R^2 \left(1 - \frac{z}{h}\right)^2 dz. \]

展开 \((1 - z/h)^2 = 1 - 2z/h + z^2/h^2\)，逐项积分：

\[ \int_0^h z\,dz - \frac{2}{h}\int_0^h z^2\,dz + \frac{1}{h^2}\int_0^h z^3\,dz = \frac{h^2}{2} - \frac{2h^2}{3} + \frac{h^2}{4} = h^2\left(\frac{6 - 8 + 3}{12}\right) = \frac{h^2}{12}. \]

所以 \(\iiint_E z\,dV = \pi R^2 h^2/12\)，于是

\[ \bar z = \frac{\pi R^2 h^2/12}{\pi R^2 h/3} = \frac{h}{4}. \]

圆锥的质心在距底面 \(h/4\) 处——这是固体力学里的经典结果，比几何中点 \(h/2\) 低得多。其几何意义就是：圆锥下大上小，水平截面的面积按 \((1 - z/h)^2\) 衰减，把质心狠狠地拉向底部。这一类"不均匀的形状必然把质心拉向较厚的一端"的几何直觉，是估算质心位置时最常用的"理智检验"。

到此为止，我们已经把直角坐标下的三重积分讲清楚了：定义、累次形式、上下限、次序、应用。但你大概也注意到了，凡是涉及圆形或球形的立体，都得在最后一步"换到极坐标"——这个动作太频繁，足以提示我们：直角坐标在面对圆柱与球时是笨拙的。第 17 章把柱面坐标和球面坐标加入工具箱，专门对付这一类立体；它们将让本来要写两页的积分缩成两行。
