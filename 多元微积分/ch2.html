第 2 章 直线、平面与曲面

一元微积分中最简单的对象是数轴上的点和线段;在二元情形里,是平面上的点、直线和曲线。到了三元,我们已经讨论过了点(第 1 章),现在该学怎样描述直线、平面以及一组标准的曲面(球面、抛物面、柱面、锥面等等)。这些是三维世界里最基本的几何对象,就像积木里的方块和圆柱:大多数工程结构、大多数物理模型,都可以用它们当骨架。

本章主要是"记账"。这里没有什么深刻想法——只是同一个几何对象有好几种写法,每种你都要会,因为你都会在不同场合遇到。好消息是:本章每一个公式都是垂直判定(点积 \(=\) 0,用于平面)或平行判定(叉积 \(=\) 0,用于直线)的直接推论。所以与其把每个公式当独立的咒语去背,不如反过来想:我们手里只有两件武器(点积、叉积),用它们能做出多少种几何描述?答案就是本章的内容——直线、平面、它们的交集、它们之间的距离,以及一组著名的二次曲面。读到本章末尾,你应该可以面对任何"求三维空间中某两个对象的关系"的问题,知道用哪个工具去切入。

2.1 三维直线

三维空间中的一条直线由两个数据确定:直线上的一个点和一个方向。方向是一个向量。点和方向向量一旦给定,直线就被唯一锁定——以那个点为锚,沿着那个向量正反两侧无限延展,得到的就是这条直线。

2.1.1 参数式

设 \(P_0 = (x_0, y_0, z_0)\) 是直线上一点,\(\mathbf{v} = \langle a, b, c \rangle\) 是方向向量。则任意点 \(P = (x, y, z)\) 在直线上当且仅当从 \(P_0\) 到 \(P\) 的位移是方向向量的某个倍数:

\[ \overrightarrow{P_0 P} = t \mathbf{v} \quad (t \in \mathbb{R}). \]

按分量写出:

\[ \boxed{x = x_0 + at, \qquad y = y_0 + bt, \qquad z = z_0 + ct.} \]

这就是直线的参数式。\(t\) 取遍所有实数时,\((x, y, z)\) 描出整条直线。\(t = 0\) 时你在 \(P_0\);\(t\) 增大时你沿 \(\mathbf{v}\) 方向走;\(t\) 减小时往反方向走。这种"直线就是匀速运动一个粒子的轨迹"的看法,我们会在第 3 章里大量运用——\(t\) 不仅是一个抽象参数,你完全可以把它当成时间。

例如,过 \((1, 2, -3)\) 沿方向 \(\langle 4, 0, -1 \rangle\) 的直线,参数式为

\[ x = 1 + 4t, \quad y = 2, \quad z = -3 - t. \]

注意 \(y\) 是常数:因为方向向量的 \(y\) 分量是零。每当方向向量某个分量为零,直线对应的那个坐标就保持常数——这条直线整个躺在 \(y = 2\) 这个水平面上。

再来一个例子。过两点 \(A = (1, 0, 2)\) 和 \(B = (3, -1, 4)\) 的直线怎么写参数式?方向向量取 \(\overrightarrow{AB} = \langle 2, -1, 2 \rangle\),起点用 \(A\):

\[ x = 1 + 2t, \quad y = -t, \quad z = 2 + 2t. \]

检验:\(t = 0\) 给出 \(A\) ✓,\(t = 1\) 给出 \((3, -1, 4) = B\) ✓。也可以选 \(B\) 当起点、用 \(\overrightarrow{BA}\) 当方向,得到的参数式表面看不一样,但描述的是同一条直线——参数式不是唯一的。如果换一个方向向量(取 \(\overrightarrow{AB}\) 的两倍),参数 \(t\) 的"速度"就翻倍,但点集还是同一条直线。这种"同一条直线的多种参数式"是本节常见的细节。

2.1.2 对称式

如果 \(a, b, c\) 都不为零,可以从每个参数方程中解出 \(t\):

\[ t = \frac{x - x_0}{a} = \frac{y - y_0}{b} = \frac{z - z_0}{c}. \]

不出现 \(t\) 的等式链就是直线的对称式:

\[ \boxed{\frac{x - x_0}{a} = \frac{y - y_0}{b} = \frac{z - z_0}{c}.} \]

读这个表达式的方法是:三个分式都等于同一个数(就是那个被消去的 \(t\)),所以它们之间彼此相等。这是两个等式而不是一个,合在一起把 \((x, y, z)\) 限制在一条一维的直线上(三维空间减去两个独立约束 \(=\) 一维),恰好对得上。

如果 \(a, b, c\) 中某个为零,相应的分母不能用,那个坐标就保持常数。例如 \(a = 0\) 时对称式变为

\[ x = x_0, \qquad \frac{y - y_0}{b} = \frac{z - z_0}{c}. \]

对称式适合在你不想引入参数的时候用——例如要找直线与平面的交点(可以直接联立),或者比较两条直线是否重合(对称式标准化后系数比较)。参数式适合"画出运动轨迹"的场合;对称式适合"做代数操作"的场合。两种写法各有用武之地,要会自由切换。

把上一小节的例子改写成对称式:过 \(A = (1, 0, 2)\) 沿方向 \(\langle 2, -1, 2 \rangle\) 的直线,对称式是

\[ \frac{x-1}{2} = \frac{y}{-1} = \frac{z-2}{2}. \]

2.1.3 两条直线:平行、相交还是异面?

二维空间中,两条不同的直线要么交于一点,要么平行。三维中还有第三种可能——异面:既不平行也不相交,只是在空间中错过。想象桌上随手扔下的两根筷子,它们大概率既不平行,也不接触——它们就异面了。这是三维世界相比二维世界多出来的一种几何关系。

设两条直线 \(L_1, L_2\),方向向量分别是 \(\mathbf{v}_1, \mathbf{v}_2\),线上各取一点 \(P_1, P_2\)。判别它们关系的做法是这样的:

  1. 先算 \(\mathbf{v}_1 \times \mathbf{v}_2\)。如果它是零向量,两条方向向量成比例,两条直线平行;此时还要进一步看一个点是否在另一条线上,以判断它们是否完全重合。
  2. 如果叉积非零,两线肯定不平行,接着算标量三重积 \(\overrightarrow{P_1 P_2} \cdot (\mathbf{v}_1 \times \mathbf{v}_2)\)。这个量为零,说明三个向量共面,两条直线在同一平面内,因此相交(此时把参数式联立解方程组就能找到交点)。如果三重积非零,三个向量不共面,两条直线异面。

简言之:第一步看方向是否成比例(平行?),第二步看四个点是否共面(共面?)。两个判定结合就把所有可能性穷尽了。我们在 2.2.2 节介绍完平面之后,会用一道完整例题把这套程序走一遍。

2.2 三维平面

一个平面由两个数据确定:平面上的一点,和一个法向量(与平面垂直的向量)。点与法向量一旦给定,平面被锁死。这跟直线的"点 + 方向向量"形成对照:直线被一条线方向定下来,平面被一条垂直方向定下来。这种"用垂直方向来描述对象"的思路,是三维几何里反复出现的招数。

2.2.1 法向量式

设 \(P_0 = (x_0, y_0, z_0)\) 在平面上,\(\mathbf{n} = \langle A, B, C \rangle\) 是一个法向量。任意点 \(P = (x, y, z)\) 在平面上当且仅当 \(\overrightarrow{P_0 P}\) 在平面内,也就是 \(\overrightarrow{P_0 P} \perp \mathbf{n}\):

\[ \mathbf{n} \cdot \overrightarrow{P_0 P} = 0. \]

这一行就是"垂直判定"在派上用场:任何在平面内的位移都垂直于法向量。按分量写出:

\[ \boxed{A(x - x_0) + B(y - y_0) + C(z - z_0) = 0.} \]

这就是平面的法向量式。展开整理可得一般式:

\[ Ax + By + Cz = D, \quad \text{其中 } D = Ax_0 + By_0 + Cz_0. \]

最重要的一句话:在方程 \(Ax + By + Cz = D\) 中,系数 \(A, B, C\) 就是某个法向量的分量。法向量可以从方程里直接读出来。这件事将反复出现,值得说两遍。给定 \(3x - y + 2z = 7\),一个法向量是 \(\langle 3, -1, 2 \rangle\),从系数原封不动地读出。\(D\) 的值不影响法向量的方向,只决定平面在空间里"平移"到哪里——\(D\) 改了,平面就平行地挪一挪。

来看一个具体例子:过点 \((2, 1, -1)\) 且法向量为 \(\langle 1, 4, -2 \rangle\) 的平面是什么?直接代:\(1(x - 2) + 4(y - 1) - 2(z + 1) = 0\),展开 \(x + 4y - 2z = 8\)。检验代入 \((2, 1, -1)\):\(2 + 4 + 2 = 8\) ✓。

反过来:平面 \(2x - 3y + 6z = 12\) 的法向量是 \(\langle 2, -3, 6 \rangle\)。注意这个法向量方向并不唯一——任何非零标量倍数(比如 \(\langle 4, -6, 12 \rangle\) 或 \(\langle 1, -3/2, 3\rangle\))都还是合法的法向量,只是对应的方程同时被乘以一个常数。习惯上我们会保留系数最简单的形式。

2.2.2 三点确定一个平面

如果给你三个不共线的点 \(P, Q, R\),怎么求过它们的平面?需要法向量。平面内的两个向量是 \(\overrightarrow{PQ}\) 与 \(\overrightarrow{PR}\)。与两者都垂直的向量是 \(\overrightarrow{PQ} \times \overrightarrow{PR}\)。这就是法向量。然后把 \(P\) 当大本营,代入法向量式即可。这套思路一旦熟悉,只需几行代数就能完成。

具体来一次。求过 \(P = (1, 0, 1)\)、\(Q = (2, 3, 0)\)、\(R = (-1, 1, 4)\) 的平面。\(\overrightarrow{PQ} = \langle 1, 3, -1 \rangle\),\(\overrightarrow{PR} = \langle -2, 1, 3 \rangle\)。法向量:

\[ \mathbf{n} = \overrightarrow{PQ} \times \overrightarrow{PR} = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \\ 1 & 3 & -1 \\ -2 & 1 & 3 \end{vmatrix} = \langle 9 + 1,\ 2 - 3,\ 1 + 6 \rangle = \langle 10, -1, 7 \rangle. \]

平面方程:\(10(x - 1) - 1(y - 0) + 7(z - 1) = 0\),即 \(10x - y + 7z = 17\)。代回检验:代 \(Q = (2, 3, 0)\) 得 \(10(2) - 3 + 0 = 17\) ✓;代 \(R = (-1, 1, 4)\) 得 \(-10 - 1 + 28 = 17\) ✓。两个都过——OK。

再来一道:求过 \(A = (0, 0, 1)\)、\(B = (1, 0, 0)\)、\(C = (0, 1, 0)\) 的平面。这三个点恰好是三个坐标轴上各取一个单位点。\(\overrightarrow{AB} = \langle 1, 0, -1 \rangle\),\(\overrightarrow{AC} = \langle 0, 1, -1 \rangle\)。叉积:

\[ \overrightarrow{AB} \times \overrightarrow{AC} = \mathbf{i}(0+1) - \mathbf{j}(-1-0) + \mathbf{k}(1-0) = \langle 1, 1, 1 \rangle. \]

所以平面是 \(1(x - 0) + 1(y - 0) + 1(z - 1) = 0\),化简 \(x + y + z = 1\)。这个平面方程出奇地干净,反映了三个截距(在三个坐标轴上的截距)都是 1 的对称性。事实上,任何"截距式"平面方程 \(\frac{x}{p} + \frac{y}{q} + \frac{z}{r} = 1\) 都过 \((p, 0, 0)\)、\((0, q, 0)\)、\((0, 0, r)\) 三点——这是一个很方便的特殊形式。

2.2.3 两平面的交线

两个不平行的平面交于一条直线。求这条直线的做法分两步:第一,直线方向与两个法向量都垂直,所以方向是 \(\mathbf{n}_1 \times \mathbf{n}_2\);第二,求线上一点,把 \(x, y, z\) 中某一个设为方便的值(通常 0),解剩下的两元一次方程组就行。这两步合起来就给出交线的参数式。

具体:求 \(x + y + z = 6\) 与 \(2x - y + z = 1\) 两平面的交线。法向量:\(\mathbf{n}_1 = \langle 1, 1, 1 \rangle\),\(\mathbf{n}_2 = \langle 2, -1, 1 \rangle\)。交线方向:

\[ \mathbf{v} = \mathbf{n}_1 \times \mathbf{n}_2 = \langle 1 + 1,\ 2 - 1,\ -1 - 2 \rangle = \langle 2, 1, -3 \rangle. \]

线上某点:试 \(z = 0\)。则 \(x + y = 6\)、\(2x - y = 1\)。两式相加:\(3x = 7\),\(x = 7/3\),\(y = 11/3\)。检验:满足两个方程。直线参数式:

\[ x = 7/3 + 2t, \quad y = 11/3 + t, \quad z = -3t. \]

检验代入 \(t = 0\) 时 \((x, y, z) = (7/3, 11/3, 0)\):\(x + y + z = 7/3 + 11/3 + 0 = 18/3 = 6\) ✓,\(2(7/3) - 11/3 + 0 = 14/3 - 11/3 = 3/3 = 1\) ✓。

如果要练判别两条直线关系的程序,这里来一道把它跟前面所学合在一起的题。设 \(L_1\) 是上面这条交线,\(L_2\) 是过 \((1, 0, 0)\) 沿方向 \(\langle 1, -1, 0 \rangle\) 的直线。它们是什么关系?第一步,叉积 \(\langle 2, 1, -3 \rangle \times \langle 1, -1, 0 \rangle = \langle 0 - 3, -3 - 0, -2 - 1 \rangle = \langle -3, -3, -3 \rangle\),非零,所以不平行。第二步,取 \(P_1 = (7/3, 11/3, 0)\) 在 \(L_1\) 上,\(P_2 = (1, 0, 0)\) 在 \(L_2\) 上,\(\overrightarrow{P_1 P_2} = \langle -4/3, -11/3, 0 \rangle\)。三重积 \(\langle -4/3, -11/3, 0 \rangle \cdot \langle -3, -3, -3 \rangle = 4 + 11 + 0 = 15\),非零,所以两线异面。

2.3 距离问题

"求 \(X\) 到 \(Y\) 的距离"问题有一小动物园:点到平面、点到直线、两条异面直线之间的距离。它们都有一行公式可写,而所有公式都来自投影(1.5.2 节)的同一个想法——把一段位移投影到合适的方向上,投影长度就是答案。一旦你看清这个统一的思路,就不必把每条公式都当独立的咒语去背。

2.3.1 点到平面的距离

给定平面 \(Ax + By + Cz = D\) 和点 \(P_1 = (x_1, y_1, z_1)\),\(P_1\) 到该平面的距离是

\[ \boxed{\text{距离} = \frac{|Ax_1 + By_1 + Cz_1 - D|}{\sqrt{A^2 + B^2 + C^2}}.} \]

这个公式怎么来的?在平面上任取一点 \(P_0\)。\(P_1\) 到平面的距离正是 \(\overrightarrow{P_0 P_1}\) 在法向量 \(\mathbf{n}\) 上投影的长度——因为最短距离总是沿垂直方向走。投影长度为 \(|\mathbf{n} \cdot \overrightarrow{P_0 P_1}| / |\mathbf{n}|\),代入并把 \(\mathbf{n} \cdot P_0 = D\) 简化,就得到上面那个公式。分子里的 \(Ax_1 + By_1 + Cz_1 - D\) 在几何上就是"把 \(P_1\) 代入平面方程左边减右边"得到的差——它度量了 \(P_1\) 偏离平面方程的程度,要除以 \(|\mathbf{n}| = \sqrt{A^2 + B^2 + C^2}\) 来得到真正的物理距离。

看一道:求 \((1, 2, 3)\) 到平面 \(2x - y + 2z = 9\) 的距离。代公式:

\[ \frac{|2(1) - 1(2) + 2(3) - 9|}{\sqrt{4 + 1 + 4}} = \frac{|2 - 2 + 6 - 9|}{3} = \frac{3}{3} = 1. \]

再来一道带平行平面的:平面 \(\Pi_1: x + 2y + 2z = 5\) 与 \(\Pi_2: x + 2y + 2z = 14\) 显然平行(同样的法向量)。它们之间的距离是多少?在 \(\Pi_1\) 上随便挑一个点,比如 \((5, 0, 0)\),代到 \(\Pi_2\) 的距离公式里:\(|5 - 14|/\sqrt{1 + 4 + 4} = 9/3 = 3\)。所以两个平面相距 3 个单位。这种"两个平行平面间的距离"题最常用的招就是把它转化为"取一个平面上的点,算到另一个平面的距离"。

2.3.2 点到直线的距离

设直线过 \(P_0\),方向 \(\mathbf{v}\),给定点 \(P_1\),则 \(P_1\) 到直线的距离是

\[ \boxed{\text{距离} = \frac{|\overrightarrow{P_0 P_1} \times \mathbf{v}|}{|\mathbf{v}|}.} \]

这个公式怎么来的?\(\overrightarrow{P_0 P_1}\) 与 \(\mathbf{v}\) 张成一个平行四边形,它的面积为 \(|\overrightarrow{P_0 P_1} \times \mathbf{v}|\)。这个面积也等于"底乘高":底就是 \(|\mathbf{v}|\)(\(\mathbf{v}\) 沿直线方向),高就是 \(P_1\) 到直线的垂直距离——也正是我们要找的距离。两边除以 \(|\mathbf{v}|\) 就解出。所以这条公式的几何精神是:"用平行四边形的面积反推它的高"。

例:求 \((1, 1, 1)\) 到过 \((0, 0, 0)\) 沿方向 \(\langle 1, 1, 0 \rangle\) 的直线的距离。\(\overrightarrow{P_0 P_1} = \langle 1, 1, 1 \rangle\)。叉积:\(\langle 1, 1, 1 \rangle \times \langle 1, 1, 0 \rangle = \langle 0 - 1,\ 1 - 0,\ 1 - 1\rangle = \langle -1, 1, 0 \rangle\),模 \(\sqrt{2}\)。\(|\mathbf{v}| = \sqrt{2}\)。距离 \(= \sqrt{2}/\sqrt{2} = 1\)。

再算一例。从 \(P_1 = (4, 1, 5)\) 到 \(L: x = t, y = 1 + 2t, z = 3 - t\)(过 \(P_0 = (0, 1, 3)\),方向 \(\mathbf{v} = \langle 1, 2, -1 \rangle\))的距离。\(\overrightarrow{P_0 P_1} = \langle 4, 0, 2 \rangle\)。叉积 \(\langle 4, 0, 2 \rangle \times \langle 1, 2, -1 \rangle = \langle 0 - 4,\ 2 - (-4),\ 8 - 0 \rangle = \langle -4, 6, 8 \rangle\),模 \(\sqrt{16 + 36 + 64} = \sqrt{116} = 2\sqrt{29}\)。\(|\mathbf{v}| = \sqrt{6}\)。距离 \(= 2\sqrt{29}/\sqrt{6} = 2\sqrt{29/6}\)。这种结果不是漂亮整数,但所有步骤都很机械。

2.3.3 异面两直线之间的距离

这是稍硬一点的一道题。设两条异面直线方向分别为 \(\mathbf{v}_1, \mathbf{v}_2\),线上各取一点 \(P_1, P_2\),则两直线间距离为

\[ \boxed{\text{距离} = \frac{|\overrightarrow{P_1 P_2} \cdot (\mathbf{v}_1 \times \mathbf{v}_2)|}{|\mathbf{v}_1 \times \mathbf{v}_2|}.} \]

这个公式怎么来的?向量 \(\mathbf{v}_1 \times \mathbf{v}_2\) 与两直线都垂直——它指向两直线之间的"公共法线"方向。把 \(\overrightarrow{P_1 P_2}\) 投影到这条公共法线上,投影长度就是两直线间最短距离。这跟点到平面距离的思路完全平行:都是"位移投影到法向量上"。如果你能看出这个统一性,本节三条公式就只是同一招的三种装扮。

例:求 \(L_1: x = 1 + t, y = 2, z = -t\) 与 \(L_2: x = -s, y = 1 + s, z = 1\) 之间的距离。\(\mathbf{v}_1 = \langle 1, 0, -1 \rangle\),\(\mathbf{v}_2 = \langle -1, 1, 0 \rangle\),\(P_1 = (1, 2, 0)\),\(P_2 = (0, 1, 1)\)。叉积:

\[ \mathbf{v}_1 \times \mathbf{v}_2 = \langle 0 \cdot 0 - (-1)(1),\ (-1)(-1) - (1)(0),\ (1)(1) - (0)(-1) \rangle = \langle 1, 1, 1 \rangle. \]

\(\overrightarrow{P_1 P_2} = \langle -1, -1, 1 \rangle\)。三重积:\(\langle -1, -1, 1 \rangle \cdot \langle 1, 1, 1 \rangle = -1 - 1 + 1 = -1\)。\(|\mathbf{v}_1 \times \mathbf{v}_2| = \sqrt{3}\)。所以距离 \(= |-1|/\sqrt{3} = 1/\sqrt{3} = \sqrt{3}/3\)。它们真的是异面吗?三重积 \(-1 \neq 0\),所以不共面,是异面 ✓。这种"距离公式顺便检验异面性"是一个甜蜜的副作用——如果三重积出来是零,你算出来的"距离"会是 0,正告诉你两线相交。

2.4 二次曲面——一场动物园游览

二次曲面是 \((x, y, z)\) 满足某个二次多项式方程的点集:

\[ Ax^2 + By^2 + Cz^2 + Dxy + Eyz + Fxz + Gx + Hy + Iz = J. \]

经过适当的旋转和平移,每个二次曲面都能化为大约半打"标准型"中的一种。你不需要知道旋转的理论(那是线性代数里关于"对称矩阵的对角化"的话题);你只需要按它们的标准方程认出每一种。把这个动物园背下来,日后画图时心里就有底。

下面我们把每种曲面分别介绍,配上方程、形状描述、它的截面长什么样,以及它在现实中的"原型物体"。

2.4.1 椭球面

方程:

\[ \frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1. \]

形状是被压扁的球,沿坐标轴的半轴长 \(a, b, c\)。如果 \(a = b = c\),退化为半径为 \(a\) 的球。如果你切它的水平截面(固定 \(z\)),得到的是椭圆;切竖直截面也得到椭圆。整个曲面是闭合的、有限的——它把空间分成内、外两部分。橄榄球、鸡蛋、行星(略带扁的)都是椭球的近似。

2.4.2 椭圆抛物面

方程:

\[ z = \frac{x^2}{a^2} + \frac{y^2}{b^2}. \]

开口向上的"碗"。固定 \(z = z_0 > 0\) 的截面是椭圆 \(x^2/a^2 + y^2/b^2 = z_0\);固定 \(x = 0\) 的截面是抛物线 \(z = y^2/b^2\),固定 \(y = 0\) 的截面也是抛物线 \(z = x^2/a^2\)。特例 \(a = b\):旋转抛物面,这正是抛物面型卫星天线的几何形状——每根从空中沿 \(+z\) 方向射来的电波,被表面反射后都汇聚到同一个焦点上。这种"汇聚"性质就是抛物面在工程里被频繁使用的原因。

2.4.3 双曲抛物面(鞍面)

方程:

\[ z = \frac{x^2}{a^2} - \frac{y^2}{b^2}. \]

马鞍形状,像一片"品客"薯片。固定 \(z\) 的截面是双曲线;在 \(y = 0\) 的平面里截面是抛物线 \(z = x^2/a^2\)(开口向上),在 \(x = 0\) 的平面里截面是抛物线 \(z = -y^2/b^2\)(开口向下)——所以这个曲面在一个方向上"上凸"、在垂直方向上"下凹",就像马鞍。它是二次曲面里最上镜的一个,也是最容易画错的一个。原点 \((0,0,0)\) 是它的一个鞍点:沿一个方向是局部最低点,沿另一个方向是局部最高点。第 13 章讨论极值时会回到这种点。

2.4.4 锥面(双锥)

方程:

\[ \frac{x^2}{a^2} + \frac{y^2}{b^2} = \frac{z^2}{c^2}. \]

两个圆锥(或椭圆锥)在原点处尖对尖。固定 \(z \neq 0\) 的截面是椭圆,且椭圆大小随 \(|z|\) 线性增大;\(z = 0\) 的"截面"只有原点。过原点的竖直平面截下来是一对相交直线。冰激凌甜筒的外表面就是锥面的一半。

2.4.5 单叶双曲面

方程:

\[ \frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1. \]

"束腰"形状:像在尖端被拉开成圆形腰部的锥。固定 \(z\) 的截面是椭圆,在 \(z = 0\) 处椭圆最小(刚好就是 \(x^2/a^2 + y^2/b^2 = 1\)),向上向下都越来越大。这个曲面有一个非凡的几何性质——它是"直纹面",意思是整个曲面可以由直线扫成。这就是为什么核电站的冷却塔做成这种形状:既能用直钢条搭建,又比纯圆柱更稳定、空气流动更顺畅。下次看到电视新闻里冷却塔冒白烟,你看到的就是单叶双曲面。

2.4.6 双叶双曲面

方程:

\[ \frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = -1. \]

两个分开的碗,沿 \(z\) 轴反向开口。中间没有曲面——如果你试着代 \(z = 0\) 进方程,得到 \(x^2/a^2 + y^2/b^2 = -1\),没有实数解,所以中间一段空空如也。截面只在 \(|z| \ge c\) 时存在。这种曲面在物理里出现于狭义相对论的"光锥"——时空中两个事件的关系曲面,其中类时间隔的事件构成双叶双曲面的两叶,一叶代表"过去",另一叶代表"未来"。

2.4.7 怎么把它们认出来

怎么区分这六种二次曲面?看符号和右端的常数。下面是一张速查表:

右端 \(=\) 1(或常数)右端 \(= z\)(一次)
三个平方项都正:椭球面。两个平方项同号:椭圆抛物面。
两正一负:单叶双曲面。两个平方项异号:鞍面(双曲抛物面)。
一正两负:双叶双曲面。
三个都正且右端 \(= 0\):锥面。

建议你按这张表把方程的所有可能形态默写一遍。一旦熟练,看到一个二次方程,只需扫一眼系数符号就能说出曲面是什么。

关于画图的实用建议:你会被要求画这些曲面。可靠的方法是画三组截面——通常令 \(x = 0\)、\(y = 0\)、\(z = 0\)(或一些方便的常数)——看这些截面是什么,然后把光滑曲面"拟合"到这些截面上。别直接根据公式徒手画——脑子凭空想象一个未知的三维曲面是非常难的,但凭三组截面拼合起来要靠谱得多。

试做一道:\(x^2 + 4y^2 + 9z^2 = 36\) 是什么曲面?两边除以 36:\(x^2/36 + y^2/9 + z^2/4 = 1\)。所以是椭球面,沿 \(x, y, z\) 轴的半轴长为 \(6, 3, 2\)。

再来一道:\(z = 4x^2 + y^2\)。三个平方项里有两个(\(x^2\) 和 \(y^2\))同号,右端是 \(z\) 一次,所以这是椭圆抛物面,开口朝 \(+z\) 方向。

再来一道:\(x^2 - y^2 + z^2 = 1\)。把符号排一下:正、负、正,合起来"两正一负",这是单叶双曲面,但它的"束腰"轴是 \(y\) 轴(因为 \(y\) 是带负号的那个)。在标准模板里我们习惯把腰带轴放在 \(z\),但只要换名字 \(y \leftrightarrow z\),实质完全一样。

再来一道:\(z = x^2 - y^2\)。两个平方项一正一负,右端是 \(z\) 一次,所以是鞍面。它在原点有鞍点。

2.5 柱面与其他棱柱型曲面

本书中"柱面"并不一定是你想的那个(饮料罐)。在多元微积分中,柱面的意思是:在某个平面里取一条曲线,沿着与该平面垂直的方向把它"挤出去"(英文叫 extrude),得到一张曲面。这样得到的曲面只依赖于两个坐标,第三个是"自由"的,可以取任意值。

例如,三维空间里的方程

\[ x^2 + y^2 = 1 \]

是一个半径为 1、轴沿 \(z\) 轴的圆柱面。方程没说 \(z\),所以 \(z\) 可以是任何数;对每个固定的 \(z\),得到的就是这个高度上的圆 \(x^2 + y^2 = 1\)。把所有圆竖直摞起来,就是圆柱面。这才是中学数学课本里那种圆柱——它只是"柱面"这个更广概念的一种特例。

判别柱面的方法很简单:如果一个三维方程里只出现三个变量中的两个,它代表一个柱面,轴沿那个缺席的变量方向。这条规则一旦记住,识别柱面就毫不费力。

来看几个例子:

  • \(y^2 + z^2 = 4\):半径 2 的圆柱面,轴沿 \(x\) 轴(因为 \(x\) 没出现)。
  • \(z = x^2\):抛物面槽,沿 \(y\) 轴方向"挤"出来——你可以想象一条抛物线沿 \(y\) 方向无限拉伸成一道滑梯。
  • \(z = \sin x\):正弦波纹瓦楞面,沿 \(y\) 方向延伸——像是工地围栏那种波浪铁皮的形状。
  • \(\frac{x^2}{4} + \frac{y^2}{9} = 1\):椭圆柱面,轴沿 \(z\) 轴。

后几个并不是饮料罐,但它们在多元微积分意义下都是柱面。"柱面"的定义关心的不是横截面是不是圆,而是"沿某方向平移不变"这种结构。

柱面在本书后面会反复出现,主要在两个地方:其一,作为重积分的积分区域边界(把面包形状的体积切片);其二,作为梯度场或向量场的等值面。在第 16 章我们会用柱坐标来高效处理这类对称性,所以现在就熟悉它们,后面会受益。

到此为止,你已经掌握了描述三维空间中的点、向量、直线、平面以及大多数标准曲面的语言。这些都是"静态"的几何对象——它们就在那里,不动。下一章我们走向微积分的第一步:让一个点动起来。一个向量值函数就是关于动点的"配方",而它的导数就是它的速度——和一元微积分里的想法一样,只是这次有三个坐标而不是一个。本章的所有公式——叉积、点积、距离、二次曲面的形状——都会以新的形式回来,只是这次它们活在运动中。
