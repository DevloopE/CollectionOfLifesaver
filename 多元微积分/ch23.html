第 23 章 参数曲面与曲面面积

在第 3 章里，我们学过怎样用一个参数 \(t\) 去描述空间中的一条曲线 \(\mathbf{r}(t)\)：参数 \(t\) 在某个区间里跑，曲线就在三维空间里被一笔一笔地"画"出来。一条曲线是一维的对象——它沿着自己延伸的方向只能让你前进或后退，所以一个参数就够了。

到本章我们要谈论的是曲面。一张曲面是二维的对象：站在曲面上的小蚂蚁，可以朝两个相互独立的方向走——比如沿着球面"东西"走，或者"南北"走。所以为了描述一张曲面，我们需要两个参数。把它们叫作 \(u\) 和 \(v\)，并把每一对参数 \((u, v)\) 送到三维空间中的一个点 \(\mathbf{r}(u, v)\)：

\[\mathbf{r}(u, v) = \langle x(u, v),\ y(u, v),\ z(u, v)\rangle,\qquad (u, v)\in D.\]

这里 \(D\) 是 \(uv\) 平面里的某个区域——参数的"户籍"——而 \(\mathbf{r}\) 把 \(D\) 整张铺到三维空间里的某张曲面 \(S\) 上。请你心里画一幅这样的画：左边是一张平整的二维"模板纸"\(D\)；右边是三维空间。\(\mathbf{r}\) 像一只手，把这张模板纸抓起来，在空间里揉成一个杯子的外壁、一个气球的表面、一段烟囱、一片屋顶。模板纸上的水平线变成曲面上的\(u\)-曲线，模板纸上的垂直线变成曲面上的\(v\)-曲线。这两族曲线在曲面上交叉成一张密密麻麻的"经纬网"——这就是参数化在曲面上留下的脚印。

本章的两件事我们要把它们彻底说透：第一，怎样把那些常见的曲面写成参数形式；第二，怎样用这种参数表达去算曲面的面积。第二件事的公式只有一个简短的式子 \(|\mathbf{r}_u \times \mathbf{r}_v|\,du\,dv\)，但它的几何含义，是这一章真正的核心，也是后两章曲面积分的基石。

23.1 曲面的参数化

学习一种新对象，最有效的办法是先把几样典型的例子各自看一眼，让眼睛习惯它们的"声音"。下面我把日常会反复遇到的曲面写成参数形式，你可以把这一节当作一份小目录。每一种参数化里我都画了一幅简短的"文字图"，告诉你两个参数分别在做什么。

显函数图 \(z = f(x, y)\)

当一张曲面恰好是某个函数 \(f(x, y)\) 的图时，参数化是最不需要动脑子的：把 \(x, y\) 自身用作参数，

\[\mathbf{r}(x, y) = \langle x,\ y,\ f(x, y)\rangle,\qquad (x, y)\in D.\]

这里 \(D\) 是函数 \(f\) 所定义的那一块平面区域。形象地说：你把 \(xy\) 平面里那一块定义域"拎起来"，每一点向上抬到高度 \(f(x, y)\)，得到的"地形"就是这张曲面。\(u\)-曲线（固定 \(y\) 让 \(x\) 变化）画的是地形上一道道东西走向的山脊或谷沟，\(v\)-曲线（固定 \(x\) 让 \(y\) 变）画的是南北走向的剖面。

球面（半径 \(R\)）

球面用球坐标里的 \(\phi\)（与 \(z\) 轴的夹角，俗称"纬度的余角"）和 \(\theta\)（在 \(xy\) 平面里的方位角，俗称"经度"）作为参数，

\[\mathbf{r}(\phi, \theta) = \langle R\sin\phi\cos\theta,\ R\sin\phi\sin\theta,\ R\cos\phi\rangle,\qquad \phi\in[0, \pi],\ \theta\in[0, 2\pi].\]

请把它当作"地球仪"来想。\(\phi\)-曲线（固定 \(\theta\)、让 \(\phi\) 走）是从北极到南极的一根根经线；\(\theta\)-曲线（固定 \(\phi\)、让 \(\theta\) 转）是绕着 \(z\) 轴的一圈圈纬线。在赤道处 \(\phi = \pi/2\)，纬线最大；越往南北两极，\(\sin\phi\) 越小，纬圈也越小，到极点处 \(\sin\phi = 0\)，纬圈缩成一点。这就是为什么"北极"和"南极"是参数化的两个奇点：在那里两个参数其实只剩一个真正能动。

圆柱面（半径 \(R\)，轴沿 \(z\)）

圆柱面更简单：让横截面沿着每一个高度 \(z\) 都画一个相同的圆，

\[\mathbf{r}(\theta, z) = \langle R\cos\theta,\ R\sin\theta,\ z\rangle,\qquad \theta\in[0, 2\pi],\ z\in[a, b].\]

\(\theta\)-曲线是水平的圆，\(z\)-曲线是竖直的母线。整张圆柱面就是这些母线沿着圆环排队"立起来"的样子。如果你要把它"剪开摊平"，沿任意一根母线竖着割一刀，圆柱就摊成一张矩形。这正是参数 \((\theta, z)\) 的"模板"形状。

圆锥面（半角 \(\alpha\)，顶点在原点）

把圆柱面的"母线"扭成倾斜的射线，就是圆锥。一种自然的参数化是用"沿母线走的距离"\(r\) 和方位角 \(\theta\)：

\[\mathbf{r}(r, \theta) = \langle r\sin\alpha\cos\theta,\ r\sin\alpha\sin\theta,\ r\cos\alpha\rangle,\qquad r\ge 0,\ \theta\in[0, 2\pi].\]

这里 \(\alpha\) 是母线与 \(z\) 轴所成的固定角；\(r=0\) 处所有母线都汇聚到顶点。换一种写法：用水平半径\(\rho = r\sin\alpha\) 和方位角作参数，得 \(\mathbf{r}(\rho, \theta) = \langle \rho\cos\theta,\ \rho\sin\theta,\ \rho\cot\alpha\rangle\)。两种写法描述的是同一张曲面，只是模板纸的格子不同。

旋转曲面（绕 \(z\) 轴旋转 \(xz\) 平面里的曲线 \(x = g(z)\)）

把曲线 \(x = g(z)\)（\(z\in[a, b]\)）绕 \(z\) 轴旋转一圈，得到的曲面可以这样参数化：

\[\mathbf{r}(z, \theta) = \langle g(z)\cos\theta,\ g(z)\sin\theta,\ z\rangle,\qquad z\in[a, b],\ \theta\in[0, 2\pi].\]

这是一个特别有用的家族：球面（\(g(z) = \sqrt{R^2 - z^2}\)）、圆柱面（\(g(z) = R\)）、圆锥面（\(g(z) = z\tan\alpha\)），甚至环面（甜甜圈）、酒杯外壁、花瓶——只要它"绕一根轴对称"，它就是一张旋转曲面。\(z\)-曲线是母线（旋转前那条曲线在不同方位的复制品），\(\theta\)-曲线是纬圈（每一个高度处的水平圆）。

平面（包含点 \(\mathbf{r}_0\)、由两向量 \(\mathbf{a}\)、\(\mathbf{b}\) 张成）

最简单也最常用：

\[\mathbf{r}(u, v) = \mathbf{r}_0 + u\mathbf{a} + v\mathbf{b},\qquad (u, v)\in D.\]

\(u\)-曲线是与 \(\mathbf{a}\) 平行的一条条直线，\(v\)-曲线是与 \(\mathbf{b}\) 平行的另一族直线。它们在平面里画出一张网格——这是所有参数化里最"诚实"的一张图，因为模板和曲面其实是同一种东西（仿射变换之差）。

同一张曲面、不同的参数化

请你记住一件事：参数化不是唯一的。一张球面，可以用 \((\phi, \theta)\)，可以用立体投影里的 \((u, v)\)，也可以用上半球的 \(z = \sqrt{R^2 - x^2 - y^2}\) 把 \((x, y)\) 当参数（仅覆盖一半）。这种"自由"是参数化最强大的地方：选一种适合手头题目的参数，能让计算变得轻得多。我们等到 24 章和 25 章时还要反复体会这一点。

23.2 曲面面积公式

给定参数曲面 \(\mathbf{r}(u, v)\)（\((u, v)\in D\)），它的面积有一条非常漂亮的公式：

\[\boxed{\,S = \iint_D |\mathbf{r}_u \times \mathbf{r}_v|\,du\,dv,\,}\]

其中 \(\mathbf{r}_u = \partial\mathbf{r}/\partial u\)、\(\mathbf{r}_v = \partial\mathbf{r}/\partial v\)。

这条公式短得像一句口诀，但它背后的几何意味却需要仔细品味。我们来一笔一笔把它推出来。

请把目光移回到那张"模板纸"\(D\) 上。在 \(D\) 里取一个微小的矩形：宽 \(du\)、高 \(dv\)，左下角在 \((u, v)\)。这一小块矩形被参数映射 \(\mathbf{r}\) 抬到三维空间，会变成曲面上一小块"近似平行四边形"。具体来讲，矩形的四个顶点 \((u, v)\)、\((u+du, v)\)、\((u, v+dv)\)、\((u+du, v+dv)\) 分别被送到

\[\mathbf{r}(u, v),\quad \mathbf{r}(u+du, v),\quad \mathbf{r}(u, v+dv),\quad \mathbf{r}(u+du, v+dv).\]

当 \(du, dv\) 很小时，按定义有线性近似

\[\mathbf{r}(u+du, v) \approx \mathbf{r}(u, v) + \mathbf{r}_u\,du,\qquad \mathbf{r}(u, v+dv) \approx \mathbf{r}(u, v) + \mathbf{r}_v\,dv.\]

所以这一小块在曲面上张成的平行四边形，由两条边

\[\mathbf{r}_u\,du\quad\text{和}\quad \mathbf{r}_v\,dv\]

所夹。一对向量所张成的平行四边形的面积，等于它们叉积的模——这是叉积最早出场时（第 1 章）就告诉过我们的事。所以这一小块面积近似

\[dS \approx |\mathbf{r}_u\,du \times \mathbf{r}_v\,dv| = |\mathbf{r}_u \times \mathbf{r}_v|\,du\,dv.\]

把所有这种小平行四边形加起来，再让 \(du, dv\to 0\)（即取黎曼和的极限），就得到

\[S = \iint_D |\mathbf{r}_u \times \mathbf{r}_v|\,du\,dv.\]

这就是曲面面积公式的几何含义：\(|\mathbf{r}_u \times \mathbf{r}_v|\) 度量的是"模板纸上单位面积"在曲面上被拉伸成多大面积"。它扮演的角色，与第 18 章里换元积分中的雅可比行列式 \(|\partial(x,y)/\partial(u,v)|\) 几乎一模一样——只是后者比较"二维到二维"的面积比，而前者比较"二维参数到三维曲面"的面积比。

有一种特殊情形特别值得专门写出来。当曲面恰好是函数图 \(z = f(x, y)\) 时，把 \(x, y\) 当参数：\(\mathbf{r}(x, y) = \langle x, y, f(x, y)\rangle\)，则

\[\mathbf{r}_x = \langle 1, 0, f_x\rangle,\qquad \mathbf{r}_y = \langle 0, 1, f_y\rangle.\]

叉积可以按行列式展开算出

\[\mathbf{r}_x \times \mathbf{r}_y = \langle -f_x,\ -f_y,\ 1\rangle,\qquad |\mathbf{r}_x \times \mathbf{r}_y| = \sqrt{1 + f_x^2 + f_y^2}.\]

于是显式曲面 \(z = f(x, y)\) 的面积公式就成了一条干净的二重积分

\[\boxed{\,S = \iint_D \sqrt{1 + f_x^2 + f_y^2}\,dA.\,}\]

很多教材用这条公式做曲面面积的"入门定义"。事实上它只是一般公式 \(\iint_D|\mathbf{r}_u\times\mathbf{r}_v|\,du\,dv\) 的一个具体的特例；你可以把它当作一个有用的速记。

把一元微积分里的弧长公式 \(\int\sqrt{1 + (f'(x))^2}\,dx\) 与之并排放着，你会感到这两者根本是同一种思想：弧长把"水平方向上微小一段 \(dx\)" 拉伸成长度 \(\sqrt{1 + (f')^2}\,dx\)；曲面面积把"\(xy\) 平面上微小一片 \(dA\)"拉伸成面积 \(\sqrt{1 + f_x^2 + f_y^2}\,dA\)。维数升了一阶，面孔却几乎没变。

23.3 例题

下面我们做四个题，把上面这条公式各种"打开方式"都过一遍。

例 1：球面的面积

求半径 \(R\) 球面的面积。

用前面给出的标准参数化 \(\mathbf{r}(\phi, \theta) = \langle R\sin\phi\cos\theta,\ R\sin\phi\sin\theta,\ R\cos\phi\rangle\)。先把两个偏导算出来：

\[\mathbf{r}_\phi = R\langle \cos\phi\cos\theta,\ \cos\phi\sin\theta,\ -\sin\phi\rangle,\]

\[\mathbf{r}_\theta = R\langle -\sin\phi\sin\theta,\ \sin\phi\cos\theta,\ 0\rangle.\]

叉积按行列式展开：

\[\mathbf{r}_\phi \times \mathbf{r}_\theta = R^2\langle \sin^2\phi\cos\theta,\ \sin^2\phi\sin\theta,\ \sin\phi\cos\phi\rangle.\]

它的模可以一步把 \(\sin\phi\) 提出来：

\[|\mathbf{r}_\phi \times \mathbf{r}_\theta| = R^2\sin\phi\sqrt{\sin^2\phi(\cos^2\theta + \sin^2\theta) + \cos^2\phi} = R^2\sin\phi.\]

剩下来就是把它积分：

\[S = \int_0^{2\pi}\int_0^\pi R^2\sin\phi\,d\phi\,d\theta = R^2 \cdot 2\pi \cdot 2 = 4\pi R^2.\]

这正是初等几何里那个著名的 \(4\pi R^2\)。注意"面元素"\(dS = R^2\sin\phi\,d\phi\,d\theta\) 与三重积分中球坐标的"体积元素"\(dV = \rho^2\sin\phi\,d\rho\,d\phi\,d\theta\) 之间，只差一个 \(d\rho\) 与一个 \(\rho\)——这是它们结构上的"亲属关系"。

例 2：抛物面的截顶

求曲面 \(z = x^2 + y^2\) 在 \(z\le 1\) 那部分（一个"碗"）的面积。

这是显函数情形。投影到 \(xy\) 平面是单位圆盘 \(D = \{x^2 + y^2 \le 1\}\)。算偏导：\(f_x = 2x\)、\(f_y = 2y\)，所以

\[\sqrt{1 + f_x^2 + f_y^2} = \sqrt{1 + 4x^2 + 4y^2}.\]

积分

\[S = \iint_D \sqrt{1 + 4(x^2 + y^2)}\,dA.\]

圆对称的被积式自然让人换极坐标。令 \(x = r\cos\theta\)、\(y = r\sin\theta\)，则 \(dA = r\,dr\,d\theta\)：

\[S = \int_0^{2\pi}\int_0^1 \sqrt{1 + 4r^2}\,r\,dr\,d\theta.\]

内层用代换 \(u = 1 + 4r^2\)，\(du = 8r\,dr\)：

\[\int_0^1 \sqrt{1 + 4r^2}\,r\,dr = \frac{1}{8}\int_1^5 \sqrt{u}\,du = \frac{1}{8}\cdot \frac{2}{3}\left(5^{3/2} - 1\right) = \frac{5\sqrt{5} - 1}{12}.\]

乘上外层的 \(2\pi\)：

\[S = \frac{\pi(5\sqrt{5} - 1)}{6}.\]

对一道想象不出"显然的几何答案"的题，唯一可信的检验方式，是把数代回去看量级是否合理。这里 \(5\sqrt{5} \approx 11.18\)，所以 \(S \approx \pi\cdot 10.18/6 \approx 5.33\)。底面圆盘面积是 \(\pi\approx 3.14\)，碗的"侧表面"应当比底大一些——5.33 落在合理区间。

例 3：圆柱侧面

求半径 \(R\)、高度从 \(z = 0\) 到 \(z = h\) 的圆柱侧面积。

用 \(\mathbf{r}(\theta, z) = \langle R\cos\theta,\ R\sin\theta,\ z\rangle\)。偏导：

\[\mathbf{r}_\theta = \langle -R\sin\theta,\ R\cos\theta,\ 0\rangle,\qquad \mathbf{r}_z = \langle 0, 0, 1\rangle.\]

叉积 \(\mathbf{r}_\theta \times \mathbf{r}_z = \langle R\cos\theta,\ R\sin\theta,\ 0\rangle\)，模 \(|\mathbf{r}_\theta \times \mathbf{r}_z| = R\)。所以

\[S = \int_0^h\int_0^{2\pi} R\,d\theta\,dz = R\cdot 2\pi\cdot h = 2\pi R h.\]

正是"剪开摊平"得到的矩形 \(2\pi R \times h\) 的面积。让人安心——参数面积公式与初等几何符合。

例 4：环面（甜甜圈）的面积

把 \(xz\) 平面上的圆 \((x - a)^2 + z^2 = b^2\)（\(0 < b < a\)）绕 \(z\) 轴旋转，得到一个环面。它的参数化可以写成

\[\mathbf{r}(u, v) = \langle (a + b\cos u)\cos v,\ (a + b\cos u)\sin v,\ b\sin u\rangle,\qquad u, v\in[0, 2\pi].\]

\(u\) 是绕"小圆"走的角度，\(v\) 是绕中心 \(z\) 轴走的角度。算偏导：

\[\mathbf{r}_u = \langle -b\sin u\cos v,\ -b\sin u\sin v,\ b\cos u\rangle,\]

\[\mathbf{r}_v = \langle -(a + b\cos u)\sin v,\ (a + b\cos u)\cos v,\ 0\rangle.\]

叉积按行列式展开（耐心一些）：

\[\mathbf{r}_u \times \mathbf{r}_v = -b(a + b\cos u)\langle \cos u\cos v,\ \cos u\sin v,\ \sin u\rangle.\]

模就把里面那个长度为 1 的向量带过去：

\[|\mathbf{r}_u \times \mathbf{r}_v| = b(a + b\cos u).\]

面积

\[S = \int_0^{2\pi}\int_0^{2\pi} b(a + b\cos u)\,du\,dv = 2\pi\cdot b\cdot[2\pi a + 0] = 4\pi^2 a b.\]

注意 \(\int_0^{2\pi}\cos u\,du = 0\)。这条结果有一种"对称的简洁"：它正是帕普斯定理给出的——绕轴旋转一条曲线生成的曲面面积等于"曲线长度乘以质心绕一周走过的距离"。曲线（小圆）周长 \(2\pi b\)，其质心绕 \(z\) 轴走一周距离 \(2\pi a\)，相乘正得 \(4\pi^2 ab\)。两条独立的路径给出同一答案，是很好的安心。

到此，我们已经能够把"一张曲面有多大"化为一道二重积分。下一步是把这条思路升级：不只是测量曲面本身的大小，而是把任何一个标量函数或向量场沿曲面积分。这就是第 24 章。
