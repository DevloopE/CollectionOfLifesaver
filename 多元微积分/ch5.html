第 5 章 多元函数

到目前为止，本书故事的主语始终是"输入一个数 \(t\)、输出一个向量 \(\mathbf{r}(t)\)"——曲线的世界。从这一章起我们把视角彻底翻转过来。输入将是一个向量——平面或空间中的一个点——而输出回到一个普通的实数。换句话说，我们要研究的对象是 \(f(x, y)\) 或 \(f(x, y, z)\) 这种形式的函数。

这种函数比你想象的要常见得多。一座山的海拔随经纬度变化，是 \(h(x, y)\)；一块金属板上某点的温度，是 \(T(x, y)\)；空气在房间各处的密度、压强，是 \(\rho(x, y, z)\) 与 \(p(x, y, z)\)。当你打开一份股票的盈亏热力图、一张天气云图、一张地形测绘图，你看到的就是某个二元函数的可视化。多元函数是描述世界的自然语言，因为世界很少只随一个变量在变。

本章不做微积分。微积分要从第 7 章才开始。这一章只做一件事：把这些函数看清楚。我们会画图，会画等高线，会做截面，会想象 \(\mathbb{R}^3\) 里的曲面。如果你直接跳过这一章去算偏导和梯度，你算出来的将是空洞的符号；但如果你先把图景印在脑子里，后面所有公式都会变成一种"你已经预感到了的东西"。多花一点时间在这章上，是值得的。

5.1 二元函数的图象

二元函数 \(f \colon D \to \mathbb{R}\) 是一条规则，其中 \(D \subseteq \mathbb{R}^2\)。给它喂入一对数 \((x, y) \in D\)，它吐出一个数 \(f(x, y)\)。我们用图象来想象它：

\[\text{graph}(f) = \{(x, y, z) \in \mathbb{R}^3 : z = f(x, y),\ (x, y) \in D\}.\]

请这样去想：在 \(\mathbb{R}^3\) 的"地板"——也就是 \(xy\) 平面——的每一个点 \((x, y)\) 上方，按高度 \(z = f(x, y)\) 钉上一颗钉子。所有这些钉头连成一张曲面，就是 \(f\) 的图象。地板上每个点正上方恰好一颗钉，所以图象总是"\(z\) 关于 \((x, y)\) 单值"的曲面——它从来不会自己折回去。

有些图象我们已经认识。线性函数 \(f(x, y) = ax + by + c\) 的图象是一张平面，倾斜方向由系数 \(a\)、\(b\) 决定。\(f(x, y) = x^2 + y^2\) 的图象是一只朝上开口的椭圆抛物面，像一个旋转对称的碗。\(f(x, y) = x^2 - y^2\) 的图象是双曲抛物面，俗称鞍面——形状像一副马鞍，沿 \(x\) 方向上凹、沿 \(y\) 方向下凹。\(f(x, y) = \sqrt{1 - x^2 - y^2}\) 的图象是单位球面的上半部分，定义域是闭圆盘 \(\{(x, y) : x^2 + y^2 \leq 1\}\)。\(f(x, y) = \sin(x)\cos(y)\) 的图象在两个方向上都周期重复，看起来像一摞鸡蛋盒。

关于定义域，我们要养成一个习惯：写出函数前先想清楚它在哪些 \((x, y)\) 上有意义。\(f(x, y) = \ln(x + y - 1)\) 的定义域是开半平面 \(\{(x, y) : x + y > 1\}\)，因为对数要求自变量为正。\(f(x, y) = \sqrt{4 - x^2 - y^2}\) 的定义域是闭圆盘 \(\{(x, y) : x^2 + y^2 \leq 4\}\)。\(f(x, y) = \dfrac{1}{xy}\) 的定义域是平面去掉两条坐标轴。每当定义域不是整个 \(\mathbb{R}^2\)，请把它写出来——这件事在第 6 章讨论极限与连续性时会真正派上用场，因为"\((x, y)\) 趋向于 \((a, b)\)"必须在定义域里发生。

5.1.1 截面法画图

考试或习题让你"画 \(f(x, y)\) 的图象"，请不要盯着公式凭空乱画。三维曲面太复杂，你大脑里没有合适的肌肉去直接想象它。正确做法是：用平面去切它，得到几条截面曲线，再把这些曲线在脑中拼起来。

常用的截面有三类。第一类，固定 \(x = 0\)（或别的常数）：得到的是 \(yz\) 平面里的曲线 \(z = f(0, y)\)。第二类，固定 \(y = 0\)：得到 \(xz\) 平面里的曲线 \(z = f(x, 0)\)。第三类，固定 \(z = c\)：在 \(xy\) 平面里得到的曲线方程是 \(f(x, y) = c\)，这条曲线就是等高线，下一节专门讨论。这三类截面合在一起，足以让你脑中浮现出曲面的整体形状。

来看一个具体例子：画 \(f(x, y) = x^2 - y^2\) 的图象。设 \(y = 0\)，得到 \(z = x^2\)，这是 \(xz\) 平面里向上开口的抛物线。设 \(x = 0\)，得到 \(z = -y^2\)，这是 \(yz\) 平面里向下开口的抛物线。设 \(z = 0\)，得到 \(x^2 = y^2\)，即两条直线 \(y = \pm x\)。设 \(z = 1\)，得到 \(x^2 - y^2 = 1\)，沿 \(x\) 轴方向开口的双曲线；设 \(z = -1\)，得到 \(y^2 - x^2 = 1\)，沿 \(y\) 轴方向开口的双曲线。把这五张截面拼起来，你看到的就是经典的鞍面：沿一根坐标轴上爬，沿另一根下滑，斜方向看是一组互补的双曲线。

再考虑一个，\(f(x, y) = e^{-(x^2 + y^2)}\)。\(y = 0\) 截面是钟形曲线 \(z = e^{-x^2}\)，最高点在原点；\(x = 0\) 截面也是同样的钟形 \(z = e^{-y^2}\)。\(z = c\)（\(0 < c \leq 1\)）截面满足 \(x^2 + y^2 = -\ln c\)，是同心圆。所以图象就是一个旋转对称的"高斯帽"，原点处有最高值 \(1\)，向四周指数式衰减。这种"先做几条截面再合成"的训练，做几次之后会变成本能。

第三个例子稍微挑战一点：\(f(x, y) = \dfrac{xy}{x^2 + y^2}\)，定义在原点之外。\(y = 0\) 截面是 \(z = 0\)，一条沿 \(x\) 轴铺开的水平线。\(x = 0\) 截面同样是 \(z = 0\)。乍看之下两个截面都是零，似乎曲面就在零附近。但是沿 \(y = x\) 截面：\(z = \dfrac{x^2}{2x^2} = \dfrac12\)，恒为 \(\dfrac12\)！沿 \(y = -x\) 截面：\(z = -\dfrac12\)。所以这张曲面的全貌相当怪异：四条不同射线给出四个不同的常数值。这个例子我们在第 6 章会再次相遇——它正是"多元极限不存在"的标志性反例。

5.2 等高线与等值面

有一种比"画三维曲面"更省力、信息更密集的方式：等高线图。\(f(x, y)\) 的等高线（也叫水平曲线、等值线）是指 \(f\) 取某个固定常数 \(c\) 的所有点构成的曲线：

\[\{(x, y) \in D : f(x, y) = c\}.\]

注意一个要害：等高线住在输入平面 \(xy\) 里，不住在三维曲面上。它是把高度 \(z = c\) 的水平切片正交投影到地板上而得到的曲线。换句话说，先用平面 \(z = c\) 切曲面，得到一圈三维曲线；再让这圈曲线"垂直落地"，留下的影子就是等高线。

这正是徒步地图的工作方式。任何一张地形图——你在国家公园门口看到的那种棕色细线密布的图——上面每一条线都是高程函数 \(h(x, y)\) 的一条等高线。沿着一条等高线行走，意味着海拔不变，你在山腰上水平绕行；穿过一条等高线就是上下海拔变化一段。线之间的间距告诉你坡度：等高线密集，坡陡；等高线稀疏，地势平缓。等高线相切或闭合在一个小圆里，那里通常是山顶或山谷的底部。等高线 X 形相交（很罕见），那里就是鞍点——一种"沿一个方向是顶、沿另一个方向是底"的地形。

这种地图—函数的对应关系不是一种简陋类比，而是数学家想象多元函数最贴心的工具。请把每个 \(f(x, y)\) 都想成一片山地的高程，把等高线图想成它的徒步地图。"梯度"以后会出现——它就是地图上指向最陡上坡方向的那一支箭头。"鞍点"、"局部极小"、"局部极大"——它们就是地形上的鞍、谷、峰。

来看几张具体的"地图"。函数 \(f(x, y) = x^2 + y^2\) 的等高线：方程 \(x^2 + y^2 = c\)。\(c > 0\) 时是半径为 \(\sqrt{c}\) 的圆；\(c = 0\) 时只剩原点一个点；\(c < 0\) 时无解，没有等高线。所以等高线图是一组以原点为中心的同心圆。等距取 \(c = 1, 2, 3, \dots\)，相邻圆的半径差是 \(\sqrt{c+1} - \sqrt{c}\)，这个差越往外越小，所以等高线越往外越密——意味着碗的内壁越往外越陡，这跟我们对抛物面的直觉一致。

函数 \(f(x, y) = y - x^2\) 的等高线：\(y - x^2 = c\)，即 \(y = x^2 + c\)。每条等高线都是同一条标准抛物线 \(y = x^2\) 的上下平移。\(c\) 越大，抛物线越靠上。整张图象就是把这条抛物线沿 \(z\) 方向"剃刀切开"得到的曲面——一个抛物柱面被斜搭在地板上。

函数 \(f(x, y) = xy\) 的等高线：方程 \(xy = c\)。\(c > 0\) 时是位于第一、三象限的双曲线 \(y = c/x\)；\(c < 0\) 时是位于第二、四象限的双曲线；\(c = 0\) 时退化成两条坐标轴 \(x = 0\) 与 \(y = 0\)。所以等高线图是一组围绕原点的双曲线，正负 \(c\) 各占两个象限，原点处是退化的"鞍点"型十字。这正好对应于 \(z = xy\) 的图象——它跟 \(z = x^2 - y^2\) 通过 \(45^\circ\) 旋转等价，所以图象也是一个鞍面。

把若干常数 \(c\)（通常是等差数列 \(c = -2, -1, 0, 1, 2, \dots\)）对应的等高线画在同一张纸上，并标注每条线的高度，就是等高线图。当题目让你"画 \(f\) 的图象"时，画一张精心标注的等高线图通常已经是非常好的答案——它包含了三维曲面几乎所有的形状信息，并且容易在二维纸面上呈现。

5.2.1 等值面（三元函数）

对三元函数 \(f(x, y, z)\)，类似的概念叫等值面（或水平面）：

\[\{(x, y, z) \in \mathbb{R}^3 : f(x, y, z) = c\}.\]

这是 \(\mathbb{R}^3\) 中的一张曲面。请注意：三元函数的"图象"将是 \(\mathbb{R}^4\) 中的对象，而我们没法直接画 \(\mathbb{R}^4\)。所以等值面是我们唯一实际可行的可视化手段。它就像三维世界里的等高线——只不过等高线是平面里的曲线，等值面是空间里的曲面。

看 \(f(x, y, z) = x^2 + y^2 + z^2\) 的等值面。\(c > 0\) 时方程 \(x^2 + y^2 + z^2 = c\) 给出半径为 \(\sqrt{c}\) 的球面；\(c = 0\) 时退化成一点；\(c < 0\) 时空集。所以这个 \(f\) 的等值面是一组以原点为中心的同心球面。如果把 \(f\) 想成"到原点的距离平方"，这非常合理——所有到原点距离相同的点构成一个球面。

看 \(f(x, y, z) = x^2 + y^2 - z^2\) 的等值面。这个稍丰富。\(c > 0\) 时方程 \(x^2 + y^2 - z^2 = c\) 是单叶双曲面，外形像核电站的冷却塔——绕 \(z\) 轴旋转的双曲线。\(c = 0\) 时是双圆锥 \(x^2 + y^2 = z^2\)，两个尖端在原点对接的无限锥。\(c < 0\) 时是双叶双曲面，分成上下两片各自远离 \(xy\) 平面。

同一个函数，三种性质截然不同的等值面，全靠常数 \(c\) 取值切换。锥面恰好是单叶与双叶之间的临界形状——这就是我们把锥面叫做退化二次曲面的原因，它正坐在两类双曲面互相变形过渡的中点上。等值面图集就像是给函数做了一次断层扫描，每个 \(c\) 一张片子，整套片子合起来才是函数的全貌。

5.3 三元及三元以上的函数

到此为止，所有我们做的事情对任意 \(n\) 个变量都自然推广。当变量太多时手写 \((x, y, z, w, u, v, \dots)\) 既笨拙又快用完字母，所以我们采用记号 \(\mathbf{x} = (x_1, x_2, \dots, x_n)\)。函数 \(f \colon D \to \mathbb{R}\) 且 \(D \subseteq \mathbb{R}^n\) 就是 \(n\) 元实值函数。

本书的例题主要集中在 \(n = 2\) 和 \(n = 3\) 上，原因很简单：在这两种情形下我们能画图、能借助几何直觉。但是请记住，所有要紧的代数公式——偏导数、梯度、链式法则、泰勒展开、拉格朗日乘子——都对任意 \(n\) 推广，公式的字面形式几乎不变，只是下标从两三个变成一串。每当你看到推导过程中有人写"由对称性，关于 \(z\) 同理"或者"以下论证对任意维数都成立"，那是作者在悄悄告诉你：这件事跟具体的 \(n\) 无关。

当 \(n \geq 4\) 时，几何可视化的确不再可行——人脑没有进化出第四个空间维度的肌肉。但代数是完全相同的，只是符号变长。在数据科学、机器学习、最优化里，你会面对几百乃至几百万个变量的函数（神经网络的损失函数 \(L(\mathbf{w})\) 通常有上亿个权重）。那时候本书所有公式都依然适用。所以在二、三维里把直觉打牢，是为高维计算储蓄的地基。
