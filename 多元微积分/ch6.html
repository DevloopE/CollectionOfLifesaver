第 6 章 极限与连续性

一元微积分里，写 \(\lim_{x \to a} f(x) = L\) 时，"\(x \to a\)"的方式只有两种：从左、从右。如果左右极限相等，全极限就存在；如果不等，全极限不存在。整个故事简单清爽。

把这套定义搬到二元，麻烦立刻翻倍。\(\lim_{(x, y) \to (a, b)} f(x, y) = L\) 的意思仍然是：\((x, y)\) 越靠近 \((a, b)\)，\(f(x, y)\) 就越靠近 \(L\)。但现在 \((x, y)\) 逼近 \((a, b)\) 的方式不再只有两种——平面上有无穷多条曲线可以走进 \((a, b)\)：沿 \(x\) 轴直线进来、沿 \(y\) 轴进来、沿任意方向的直线 \(y = mx + b - ma\)、沿抛物线、沿三次曲线、沿螺线、沿你能画出的任何怪曲线。为了让二元极限存在，无论沿哪条路径逼近 \((a, b)\)，\(f\) 都必须趋向同一个 \(L\)。

这一条要求把多元极限变成了一件远比一元棘手的事。坏消息：很多看起来"应该有极限"的表达式其实没有极限——某些路径偷偷给出不同的值。好消息：当极限确实存在时，计算它的技巧并不复杂；而当你怀疑它不存在时，证明它不存在也有一招屡试不爽的方法，叫做双路径法。这一章我们就把这两件事都学会，并把它们安放在"连续性"这个统一框架下。

6.1 多元极限的基本想法

非正式地说，\(\lim_{(x, y) \to (a, b)} f(x, y) = L\) 意味着：只要 \((x, y)\) 充分靠近 \((a, b)\) 但不等于 \((a, b)\)，\(f(x, y)\) 就可以任意靠近 \(L\)。这里的"靠近"是几何上的距离——欧几里得距离 \(\sqrt{(x - a)^2 + (y - b)^2}\)——它涵盖了所有方向，不是单单某一条线。

我们用 \(\varepsilon\)-\(\delta\) 严格化这个想法的方式跟一元几乎一样：把"区间"换成"圆盘"。形式定义见 A.1 节，眼下我们只需要这张图：以 \((a, b)\) 为圆心画一个半径 \(\delta\) 的小圆盘，要求圆盘里所有点（除 \((a, b)\) 自身）的 \(f\) 值都落在 \(L\) 的 \(\varepsilon\) 范围里。\(\varepsilon\) 任你给得多小，总能找到对应的 \(\delta\) 让这件事成立——这就是极限存在的几何含义。

关键是要内化这个画面：圆盘不分方向。它是一个二维的小区域，覆盖了从 \((a, b)\) 出发往任何方向走的足够近的所有点。所以"所有路径都得给出同一个 \(L\)"不是一种额外苛求，而是直接从这个几何定义里掉出来的——因为圆盘里的每一条曲线，无论形状如何，只要终点是 \((a, b)\)，最终的尾段都会被圆盘罩住，那段上的 \(f\) 值就被锁在 \([L - \varepsilon, L + \varepsilon]\) 里。

反过来用这个画面，就有一条极重要的逻辑：如果你能找到两条逼近 \((a, b)\) 的路径，使 \(f\) 沿它们趋向不同的极限值，那么二元极限必定不存在。因为如果它存在 \(= L\)，那么所有路径上的极限都得是 \(L\)；反向逆否一下，路径上极限不一致，全极限就不可能存在。这条原则是 6.3 节"双路径法"的全部根基，我们后面会反复使用它。

请暂停一下，把这件事想透。一元里"左极限等于右极限"是两个数字的一致性检验，简单。二元里"所有路径上的极限都相等"在原则上是无穷多个数字的一致性检验，绝不简单——靠一两条路径恰好给出同一个值，根本不能宣告全极限存在。所以在多元情形下，证明极限存在需要不等式控制（夹逼）；证明极限不存在只需要找到两条不一致的路径。这两个方向不对称，难度也不对称。

6.2 计算存在的极限

先说好消息：绝大多数"乖巧"的极限和一元一样平凡。如果 \(f(x, y)\) 是由多项式、指数、正弦、对数等连续函数通过加、减、乘、除、复合组合而成，并且 \((a, b)\) 落在它的自然定义域里——也就是说，代入时不会出现除零、负数对数、负数的非整数次幂等麻烦——那么直接代入就完事：

\[\lim_{(x, y) \to (a, b)} f(x, y) = f(a, b).\]

道理跟一元完全一样：连续函数在每一点的极限就是函数值。比如 \(\lim_{(x, y) \to (1, 2)} (x^2 + 3xy + y^3) = 1 + 6 + 8 = 15\)，直接代。再比如 \(\lim_{(x, y) \to (0, 0)} \cos(x + y) e^{xy} = \cos(0) \cdot e^0 = 1 \cdot 1 = 1\)，依然直接代。再比如 \(\lim_{(x, y) \to (\pi, 1)} \dfrac{\sin(xy)}{1 + x^2 y} = \dfrac{\sin(\pi)}{1 + \pi^2} = 0\)。这种题没有难度，只是热身。

真正有意思的极限是那些直接代入会得到不定式的——典型是 \(0/0\)，但也可能是 \(\infty - \infty\) 或其他形态。这时候你才需要思考。来做一个完整的例子。

计算 \(\lim_{(x, y) \to (0, 0)} \dfrac{x^2 y}{x^2 + y^2}\)。代入 \((0, 0)\)：分子分母都是零，是 \(0/0\) 不定式。所以得做点工作。一个合理的开局是先沿几条简单路径试试看是不是都给同一个值——如果路径之间已经打架，就直接用双路径法判定不存在；如果路径之间一致，再尝试夹逼证明。

沿 \(x\) 轴（\(y = 0\)）：\(f(x, 0) = 0/x^2 = 0\)，趋向 \(0\)。沿 \(y\) 轴（\(x = 0\)）：\(f(0, y) = 0/y^2 = 0\)，也是 \(0\)。沿对角线 \(y = x\)：\(f(x, x) = x^3 / (2x^2) = x/2 \to 0\)。再来沿任意过原点的直线 \(y = mx\)：

\[f(x, mx) = \frac{x^2 \cdot mx}{x^2 + m^2 x^2} = \frac{m x^3}{(1 + m^2) x^2} = \frac{mx}{1 + m^2} \to 0.\]

不论 \(m\) 是多少，沿这条直线都是 \(0\)。看起来非常鼓舞——但请记住，"试若干路径都得到一致值"不能证明极限存在。可能我还没想到的某条怪路径上 \(f\) 给出别的值。要稳妥，得用一个不依赖路径的方法：找一个上界把 \(|f(x, y)|\) 罩住，并让这个上界自动趋向于零。

这样做：注意 \(\dfrac{x^2}{x^2 + y^2} \leq 1\)（因为分子是分母的一部分），所以

\[\left|\frac{x^2 y}{x^2 + y^2}\right| = \frac{x^2}{x^2 + y^2} \cdot |y| \leq 1 \cdot |y| = |y|.\]

而 \((x, y) \to (0, 0)\) 时 \(|y| \to 0\)，由夹逼定理（A.2 节）就有 \(\lim_{(x, y) \to (0, 0)} \dfrac{x^2 y}{x^2 + y^2} = 0\)。证毕。

这就是一个完整的多元极限计算流程：先代入；如果是不定式，先尝试若干路径快速摸底——路径之间不一致就直接判不存在，路径之间一致就动手用不等式罩住绝对值并夹逼。这套程序你做几次会变成本能。控制并夹逼是其中最需要练习的一步，所以我们列出几条反复有用的不等式：

  • \(\dfrac{x^2}{x^2 + y^2} \leq 1\)、\(\dfrac{y^2}{x^2 + y^2} \leq 1\)；任何"一个二次项除以正的二次项之和"都不超过 \(1\)。
  • \(|x| \leq \sqrt{x^2 + y^2}\)、\(|y| \leq \sqrt{x^2 + y^2}\)；这个量我们记作 \(r\)，就是 \((x, y)\) 到原点的距离。
  • \(|\sin u| \leq |u|\)、\(|\sin u| \leq 1\)、\(|\cos u| \leq 1\)；标准的三角不等式。
  • \(|xy| \leq \tfrac12(x^2 + y^2)\)；这是均值不等式 \(2|xy| \leq x^2 + y^2\) 的直接改写。

再来一个应用上述工具的例子。计算 \(\lim_{(x, y) \to (0, 0)} \dfrac{\sin(x^2 + y^2)}{x^2 + y^2}\)。这就是一元极限 \(\lim_{u \to 0} \dfrac{\sin u}{u} = 1\) 的化身：令 \(u = x^2 + y^2\)，\((x, y) \to (0, 0)\) 蕴涵 \(u \to 0^+\)，所以原极限等于 \(1\)。这种"代换变量化为一元"的小技巧很值钱：只要你能识别出"被代入项在 \((a, b)\) 处趋向某个 \(u_0\)"并且外层是连续函数，你可以把多元问题压扁为一元。

再来一个。计算 \(\lim_{(x, y) \to (0, 0)} \dfrac{x^4 - y^4}{x^2 + y^2}\)。分子可以因式分解：\(x^4 - y^4 = (x^2 - y^2)(x^2 + y^2)\)。所以表达式化简为 \(x^2 - y^2\)，它在原点处的极限就是 \(0\)。这种"约分掉麻烦因子"的策略也是常用招式：先尝试代数化简，再看是否一目了然。

6.2.1 用极坐标算极限

当 \((a, b) = (0, 0)\) 时，换成极坐标常常让一切清晰。令 \(x = r\cos\theta\)、\(y = r\sin\theta\)。注意 \(r = \sqrt{x^2 + y^2}\) 就是到原点的距离，所以"\((x, y) \to (0, 0)\)"当且仅当"\(r \to 0\)"，与 \(\theta\) 无关——\(\theta\) 编码的是逼近的方向，\(r\) 编码的是逼近的远近。把这一点想透很关键，因为它告诉我们：在极坐标下，多元极限的考验就是"能否对所有 \(\theta\) 一致地让 \(r \to 0\) 时表达式趋向同一个值"。

来计算 \(\lim_{(x, y) \to (0, 0)} \dfrac{x^3 + y^3}{x^2 + y^2}\) 看看。代入：分子是 \(r^3 \cos^3\theta + r^3 \sin^3\theta = r^3(\cos^3\theta + \sin^3\theta)\)，分母是 \(r^2\)。整个表达式化为 \(r(\cos^3\theta + \sin^3\theta)\)。括号里的量是 \(\theta\) 的有界函数（绝对值不超过 \(2\)），而 \(r \to 0\) 把整个乘积压向零。所以无论 \(\theta\) 是什么，极限都是 \(0\)。

再来一个。\(\lim_{(x, y) \to (0, 0)} \dfrac{x^2 y^2}{x^2 + y^2}\)。代入：分子 \(r^4 \cos^2\theta \sin^2\theta\)，分母 \(r^2\)。化为 \(r^2 \cos^2\theta \sin^2\theta\)。\(r^2 \to 0\)，乘以有界量后仍趋向 \(0\)。所以极限 \(= 0\)。

极坐标的妙处和危险都浓缩在一个判据里。化简之后的表达式如果含有 \(r\) 的正幂（如 \(r \cdot g(\theta)\)、\(r^2 \cdot g(\theta)\)）且 \(g(\theta)\) 有界，那么 \(r \to 0\) 时它一致地趋向零，极限存在且为零。但如果化简后没有 \(r\) 的正幂因子，比如表达式是纯粹的 \(g(\theta)\)，那么不同方向给出不同值——\(\theta\) 转一圈，函数值绕了不一样的圈——这意味着极限不存在。

用这个判据看一个反例：\(\lim_{(x, y) \to (0, 0)} \dfrac{xy}{x^2 + y^2}\)。代入：分子 \(r^2 \cos\theta \sin\theta\)，分母 \(r^2\)。化为 \(\cos\theta \sin\theta = \tfrac12 \sin(2\theta)\)。没有 \(r\) 因子！这表示沿 \(\theta = 0\) 方向得 \(0\)，沿 \(\theta = \pi/4\) 方向得 \(\tfrac12\)。极限不存在，下一节我们会用双路径法重新证明它。

但极坐标也不是万能的。陷阱是：极坐标只刻画了沿射线的逼近行为，对一些非常细致的非线性路径（比如 \(y = x^k\) 这种 \(\theta\) 在 \(r \to 0\) 时也变化的路径），极坐标的"\(r \to 0\) 一致地"判据有时会被误用。所以严格地讲，要用极坐标证明极限存在，应该写：

\[|f(x, y) - L| \leq h(r),\quad \text{且 } h(r) \to 0 \text{ 当 } r \to 0,\]

其中 \(h\) 是不依赖于 \(\theta\) 的函数。如果你只是分了分情况说"对每个 \(\theta\) 有 \(\lim_{r \to 0} f = 0\)"，那并不严格——它只覆盖了直线路径。下面 6.3 节里我们会看到一个反例，每条直线都给出 \(0\)，但极限并不存在。

6.3 证明极限不存在（双路径法）

当你怀疑某个极限不存在时，最干净的证明就是双路径法：找出两条逼近 \((a, b)\) 的路径，让 \(f\) 沿它们趋向不同的值。仅此一笔，极限不存在的判定就完成了。

先做一个快速例子。证明 \(\lim_{(x, y) \to (0, 0)} \dfrac{xy}{x^2 + y^2}\) 不存在。沿 \(y = 0\)：\(f(x, 0) = 0\)，极限为 \(0\)。沿 \(y = x\)：\(f(x, x) = \dfrac{x^2}{2x^2} = \dfrac12\)，极限为 \(\dfrac12\)。\(0 \neq \dfrac12\)，二元极限不存在。三行话就完事——这就是双路径法的优雅之处。

再做一个稍稍狡猾的。证明 \(\lim_{(x, y) \to (0, 0)} \dfrac{x^2 - y^2}{x^2 + y^2}\) 不存在。沿 \(x\) 轴（\(y = 0\)）：\(f(x, 0) = x^2/x^2 = 1\)。沿 \(y\) 轴（\(x = 0\)）：\(f(0, y) = -y^2/y^2 = -1\)。\(1 \neq -1\)，极限不存在。

但是有些反例需要你跳出直线来找。看这个：证明 \(\lim_{(x, y) \to (0, 0)} \dfrac{x^2 y}{x^4 + y^2}\) 不存在。先尝试沿过原点的任意直线 \(y = mx\)：

\[f(x, mx) = \frac{x^2 \cdot mx}{x^4 + m^2 x^2} = \frac{m x^3}{x^2(x^2 + m^2)} = \frac{mx}{x^2 + m^2} \to 0\ (x \to 0).\]

对任何 \(m\) 都得 \(0\)。沿 \(x\) 轴（\(y = 0\)）也得 \(0\)。所有过原点的直线一齐给出 \(0\)——这种均一让人非常想宣告"极限存在 \(= 0\)"，但请耐住性子，先想想有没有非线性路径会捣乱。

试 \(y = x^2\)：

\[f(x, x^2) = \frac{x^2 \cdot x^2}{x^4 + (x^2)^2} = \frac{x^4}{2x^4} = \frac12.\]

沿这条抛物线恒为 \(\tfrac12\)！沿直线一律得 \(0\)，沿抛物线 \(y = x^2\) 一律得 \(\tfrac12\)。极限不存在。

这个例子的教训值得加粗：只试直线是不够的。当你试遍直线得到一致值，先问自己——这个表达式有没有"幂次平衡"的非线性路径在等着我？接下来这一段就是回答"该试什么样的非线性路径"的启发式。

看分子分母的次数。把可能路径写成 \(y = x^k\) 的形式（\(k > 0\) 是某个实数）。代入分子和分母，考察各项关于 \(x\) 的次数。如果存在某个 \(k\)，使得分子的最低次和分母里某些项的最低次正好抵消——也就是说，沿这条路径分子分母都是同阶的——那么沿这条路径的极限就是一个非零常数，常常和直线方向上的零不一致。

用这条启发式重做 \(\dfrac{x^2 y}{x^4 + y^2}\)：沿 \(y = x^k\)，分子 \(x^2 \cdot x^k = x^{2+k}\)；分母里两项 \(x^4\) 与 \(x^{2k}\)。要让分母里 \(y^2\) 项 \(x^{2k}\) 与 \(x^4\) 项同阶，取 \(k = 2\)。这时分母变成 \(x^4 + x^4 = 2x^4\)，与分子 \(x^{2+2} = x^4\) 同阶，整个商是常数 \(\tfrac12\)。这就回收了我们刚才用 \(y = x^2\) 找到的反例——但现在你看到了"为什么是 \(y = x^2\) 而不是 \(y = x^3\) 或 \(y = x^{1/2}\)"。

再用这个启发式找一个反例。考虑 \(\lim_{(x, y) \to (0, 0)} \dfrac{x^3 y}{x^6 + y^2}\)。沿 \(y = x^k\)：分子 \(x^{3+k}\)，分母里 \(x^6\) 与 \(x^{2k}\)。让 \(2k = 6\)，得 \(k = 3\)。试 \(y = x^3\)：

\[f(x, x^3) = \frac{x^3 \cdot x^3}{x^6 + x^6} = \frac12.\]

而沿 \(y = 0\)：\(f(x, 0) = 0\)。极限不存在。

这个"次数平衡"的把戏在分式型极限里几乎所向披靡。你会发现：一旦分母里出现某变量的高次项，对应的"杀手路径"就是把另一个变量取成相应的高次曲线。多做几道，这种眼力就长出来了。

6.4 连续性

跟一元情形完全一致：函数 \(f\) 在 \((a, b)\) 连续，当且仅当 \(f(a, b)\) 有定义、\(\lim_{(x, y) \to (a, b)} f(x, y)\) 存在，且二者相等：

\[\lim_{(x, y) \to (a, b)} f(x, y) = f(a, b).\]

跟一元情形也一致地，所有你能写得出的初等函数——多项式、有理函数（在分母不为零处）、指数、对数（在自变量为正处）、三角、反三角，以及它们经过加、减、乘、除、复合后的组合——在它们有定义的所有点都连续。这件事并不神秘，本质就是把"一元连续函数的代数和复合还是连续函数"逐一翻译到二元，所有论证一字不改。

所以"有趣的"连续性问题永远关于分段定义的函数，特别是在分段拼接的边界点上。看两个例子。

设 \(f(x, y) = \dfrac{xy}{x^2 + y^2}\) 在 \((x, y) \neq (0, 0)\) 时按公式取值，而在原点处人为约定 \(f(0, 0) = 0\)。问 \(f\) 在原点连续吗？\(f(0, 0) = 0\)。\(\lim_{(x, y) \to (0, 0)} f(x, y) = 0\)？我们在 6.3 节已经证明这个极限不存在。极限不存在，自然也就谈不上等于函数值。所以 \(f\) 在原点不连续。它在原点之外处处连续，但唯独被"补"上的那一点是间断点。

把分子稍变一下：设 \(f(x, y) = \dfrac{x^2 y}{x^2 + y^2}\) 在 \((x, y) \neq (0, 0)\) 时按公式取值，而 \(f(0, 0) = 0\)。问连续吗？\(f(0, 0) = 0\)。极限呢？我们在 6.2 节用 \(\left|\tfrac{x^2 y}{x^2 + y^2}\right| \leq |y|\) 加夹逼定理证明了极限是 \(0\)。所以极限和函数值都是 \(0\)，连续。

这两个例子并排放着特别有教益。表达式只差一个 \(x\) 因子，连续性的判定结果完全相反。原因是分子上多一个 \(x\) 让整个量"小一阶"——这正好把 \(\dfrac{x^2 y}{x^2 + y^2}\) 的大小压到 \(|y|\) 之下，让夹逼定理可以下手；而原来的 \(\dfrac{xy}{x^2 + y^2}\) 在 \(y = x\) 路径上不衰减为零，所以极限存在不了。分子的次数和分母的次数差距决定了一切。

一个收尾提示：连续性的定义没什么新意，难点全在极限的计算上。第 7 章我们将定义偏导数，它本质上只需要沿着一根坐标轴的极限，比本章里这种"全方向"的极限简单许多。但请记住一个微妙之处，到了第 8 章我们会撞上：偏导数存在不蕴涵函数连续。一个函数可以在一点上同时拥有 \(f_x\) 和 \(f_y\)，却在那一点不连续——届时我们要用本章的工具来看清这个反直觉的事实。
