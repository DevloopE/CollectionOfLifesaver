第 7 章 偏导数

在一元微积分里，导数 \(f'(x)\) 度量 \(f\) 随 \(x\) 变化的速率，几何上是切线斜率。但当输入变成一对数 \((x, y)\) 时，"\(f\) 变化的速率"变得含糊——你向哪个方向运动，速率就不一样。沿 \(x\) 方向走一步的高度变化，可能完全不同于沿 \(y\) 方向走一步、或沿对角线走一步的变化。所以在多元世界，"导数"分裂成了许多片段，每个方向有它自己的"速率"。

最简单也最重要的两块片段，叫做偏导数。它们度量当你沿一个坐标方向运动而保持其他坐标固定时，\(f\) 的瞬时变化率。这是把"二维变化"简化成"两个一维变化"的最直接方式，所有更高级的概念——梯度、方向导数、链式法则、切平面、泰勒展开——都建在偏导数之上。

幸运的是，偏导数的计算容易到几乎不需要新知识：把除一个变量之外的所有变量当作常数，然后按一元求导法则做。难的是概念——它意味着什么、它不意味着什么、它对函数图象的几何含义、二阶混合偏导数顺序无关的"克莱罗定理"为什么深刻。这一章我们把这两条线一起讲透。

7.1 偏导数：定义与计算

二元函数 \(f(x, y)\) 在点 \((a, b)\) 处关于 \(x\) 的偏导数定义为

\[f_x(a, b) = \lim_{h \to 0} \frac{f(a + h, b) - f(a, b)}{h}.\]

仔细看这个差商：分子比较 \(f\) 在 \((a + h, b)\) 与 \((a, b)\) 的值——也就是说，\(y\) 坐标始终被钉在 \(b\)，只让 \(x\) 在 \(a\) 附近变动。所以这个极限正是"把 \(y = b\) 固定后所得一元函数 \(g(x) = f(x, b)\)"在 \(x = a\) 处的普通导数。一切关于一元导数的工具——乘积法则、商法则、链式法则、极限计算——都原样适用。

对应地，

\[f_y(a, b) = \lim_{h \to 0} \frac{f(a, b + h) - f(a, b)}{h}\]

是把 \(x = a\) 固定后关于 \(y\) 求的导数。

偏导数有几种常见记号你都会遇到：

\[f_x = \frac{\partial f}{\partial x} = \partial_x f = D_x f.\]

那个圆滚滚的符号 \(\partial\)（读作"偏"或"del"）不是普通的 \(d\)。它是偏导数专用，它的视觉作用是提醒你"其他变量被固定"。在一些动力学、热力学、量子力学的文献里，偏导数还会写成 \(\left(\dfrac{\partial f}{\partial x}\right)_y\)，下标的 \(y\) 明确告诉你"在 \(y\) 不变的前提下求"。这种细致记号在变量众多、约束复杂的物理学问题里特别有用，但本书里我们大多数时候直接用 \(\partial f / \partial x\)，能省力就省力。

7.1.1 计算：把别的变量当常数

实际算偏导数从不去用极限定义。你做的事就是这句话：要算 \(\partial f/\partial x\)，把所有其他变量都当成常数，然后按一元求导法则关于 \(x\) 求导。\(\partial f/\partial y\) 同理，把 \(x\) 等当常数。剩下的纯粹是一元微积分。

来一组例子练手。

设 \(f(x, y) = x^3 + 4xy^2 + y^4\)。求 \(f_x\) 时，把 \(y\) 当常数：\(x^3\) 求导得 \(3x^2\)；\(4xy^2\) 是 \(x\) 的一次项，\(y^2\) 是常数系数，求导得 \(4y^2\)；\(y^4\) 不含 \(x\)，是常数，求导得 \(0\)。所以 \(f_x = 3x^2 + 4y^2\)。求 \(f_y\) 时，把 \(x\) 当常数：\(x^3\) 求导得 \(0\)；\(4xy^2\) 求导得 \(8xy\)；\(y^4\) 求导得 \(4y^3\)。所以 \(f_y = 8xy + 4y^3\)。

设 \(f(x, y) = e^{xy}\)。\(f_x\)：\(y\) 是常数，\(e^{xy}\) 关于 \(x\) 是 \(e^{(\text{常数})x}\)，求导带出系数 \(y\)，得 \(f_x = y e^{xy}\)。\(f_y\) 对称，得 \(f_y = x e^{xy}\)。

设 \(f(x, y) = \sin(x^2 + y)\)。\(f_x\)：链式法则不能漏，外层 \(\sin\) 求导是 \(\cos\)，内层 \(x^2 + y\) 关于 \(x\) 求导是 \(2x\)，所以 \(f_x = 2x\cos(x^2 + y)\)。\(f_y\)：内层关于 \(y\) 求导是 \(1\)，所以 \(f_y = \cos(x^2 + y)\)。

设 \(f(x, y) = \ln(x^2 + y^2)\)。\(f_x\)：外层 \(\ln u\) 求导得 \(1/u\)，内层 \(x^2 + y^2\) 关于 \(x\) 求导得 \(2x\)，所以 \(f_x = \dfrac{2x}{x^2 + y^2}\)。同理 \(f_y = \dfrac{2y}{x^2 + y^2}\)。

三变量的情形毫无新意。设 \(f(x, y, z) = xyz + x^2 y^3 z^4\)。\(f_x\)：把 \(y\)、\(z\) 当常数，得 \(yz + 2xy^3 z^4\)。\(f_y\)：把 \(x\)、\(z\) 当常数，得 \(xz + 3x^2 y^2 z^4\)。\(f_z\)：把 \(x\)、\(y\) 当常数，得 \(xy + 4x^2 y^3 z^3\)。

最常见的失误发生在链式法则上。当被求导的"外层函数"包着一个还含 \(x\) 的内层时，不要忘记乘上内层的导数。在第三个例子里，\(f_x = \cos(x^2 + y) \cdot \dfrac{\partial}{\partial x}(x^2 + y) = 2x\cos(x^2 + y)\)，那个 \(2x\) 是必需的。漏掉它是新手最容易犯的错——如果一道偏导题让你算错了，第一时间检查你有没有漏掉链式因子。

7.1.2 在某点求值

求"\(f\) 在某具体点处的偏导值"时，先把偏导数算成关于全部变量的函数，再代入。来一个示例：设 \(f(x, y) = xe^{xy}\)，求 \(f_y(2, 0)\)。

先算 \(f_y\)：用乘积法则——\(x\) 关于 \(y\) 是常数，所以 \(f_y = x \cdot \dfrac{\partial}{\partial y}(e^{xy}) = x \cdot xe^{xy} = x^2 e^{xy}\)。再代入 \((2, 0)\)：\(f_y(2, 0) = 4 \cdot e^0 = 4\)。

反过来千万别先代入再求导——那样你会先把 \(y\) 设成 \(0\)，得到 \(f(x, 0) = x\)，再"对 \(y\) 求导"——可是 \(y\) 已经被消掉了，你得到的 \(0\) 完全是错的。这种"先求导后代入"的次序是铁律。

7.2 几何意义与物理意义

几何上，\(f_x(a, b)\) 有一个干净的画面：它是当你站在曲面 \(z = f(x, y)\) 上的点 \((a, b, f(a, b))\)，沿 \(+x\) 方向走（\(y\) 保持等于 \(b\)）时，所走轨迹在该点的切线斜率。

把这个画面具象化：用平面 \(y = b\) 切曲面，得到一条交线。这条交线住在平面 \(y = b\) 里，可以用 \((x, z)\) 坐标描述为 \(z = f(x, b)\)，是一条普通的一元曲线。\(f_x(a, b)\) 就是这条交线在 \(x = a\) 处切线的斜率——一个完全一元的概念，只不过它是从二元函数"切片"得到的。

同样地，\(f_y(a, b)\) 是用平面 \(x = a\) 切曲面、得到曲线 \(z = f(a, y)\) 后，该曲线在 \(y = b\) 处切线的斜率，对应的方向是沿 \(+y\) 走、\(x\) 保持等于 \(a\)。所以 \(f_x\) 和 \(f_y\) 是同一张曲面在同一点上沿两条互相垂直的"测量方向"上的两条切线斜率。

对 \(f(x, y) = x^2 + y^2\) 在点 \((1, 2)\) 上做一次实验：\(f_x = 2x\)，所以 \(f_x(1, 2) = 2\)——意思是站在抛物面上、面朝 \(+x\) 方向走，曲面以速率 \(2\) 抬升（每水平走一步 \(\Delta x\)，高度大致升 \(2\Delta x\)）。\(f_y = 2y\)，\(f_y(1, 2) = 4\)——沿 \(+y\) 方向走时抬升速率是 \(4\)，更陡。这跟你对 \(z = x^2 + y^2\) 这只碗的几何直觉吻合：在 \((1, 2)\) 上，\(y\) 方向距离原点更远，碗壁更陡。

把上面这个画面再和 5.2 节的"地形图"接起来：在等高线图上的某个点 \((a, b)\)，\(f_x\) 是"向东走时海拔变化率"，\(f_y\) 是"向北走时海拔变化率"。等高线密集的地方，沿穿过等高线的方向偏导数大；等高线稀疏的地方，对应方向偏导数小。某条等高线在你脚下沿正东走，意味着东方向上 \(f_x = 0\)（海拔不变），而南北方向 \(f_y\) 才是真正衡量海拔变化的数。这种"等高线图 + 偏导数"的双重视角是多元微积分的基础语言，请把它内化。

物理上，偏导数的解释更朴素：它是"保持其他变量不变时，\(f\) 关于一个变量的瞬时变化率"。设 \(T(x, y, z, t)\) 是空间点 \((x, y, z)\) 在时刻 \(t\) 的温度，则 \(\partial T / \partial x\) 是"沿 \(x\) 方向走时温度的变化率"，比如向东走一米空气会变得多热；\(\partial T / \partial t\) 是"在固定地点上随时间的升温率"，比如此时此处每秒钟升温几度；\(\partial T / \partial z\) 可能描述高空与地表的温度差……每一个偏导数都对应一个具体的物理测量，每个都把空气和时间的某一种"变化"分离出来供你单独考量。这种分离是热力学、流体力学、电磁学里所有偏微分方程的共同骨架。

7.3 高阶偏导数；克莱罗定理

偏导数本身又是函数，所以你可以再对它求偏导。从 \(f(x, y)\) 出发，可能的二阶偏导数有四个：

  • \(f_{xx} = \dfrac{\partial^2 f}{\partial x^2}\)：对 \(x\) 求两次偏导。
  • \(f_{yy} = \dfrac{\partial^2 f}{\partial y^2}\)：对 \(y\) 求两次偏导。
  • \(f_{xy} = \dfrac{\partial^2 f}{\partial y\, \partial x}\)：先对 \(x\)，再对 \(y\)。
  • \(f_{yx} = \dfrac{\partial^2 f}{\partial x\, \partial y}\)：先对 \(y\)，再对 \(x\)。

关于记号有个值得忍耐的怪事。下标记号 \(f_{xy}\) 中下标的书写顺序就是求导顺序："先 \(x\) 再 \(y\)"。但 \(\partial\) 记号 \(\dfrac{\partial^2 f}{\partial y\, \partial x}\) 的求导顺序是从右往左读："先 \(x\)（最右那个 \(\partial\)），再 \(y\)"。两种约定恰好相反。这一点真的会把人搅糊涂，但你必须学会读两种记号都不出错。一个记忆口诀：下标顺其自然，分母从右到左。

所幸还有一个好消息能让这种顺序焦虑大半消失：对几乎所有你会遇到的函数，二阶混合偏导数 \(f_{xy}\) 与 \(f_{yx}\) 相等。

这个结果叫做克莱罗定理（也称施瓦茨定理）：若 \(f\)、\(f_x\)、\(f_y\)、\(f_{xy}\)、\(f_{yx}\) 在点 \((a, b)\) 的某个开圆盘内都连续，则

\[f_{xy}(a, b) = f_{yx}(a, b).\]

用一句口语：在合理的连续性假设下，求偏导的顺序无关紧要。这意味着你算 \(f_{xy}\) 时可以挑两条路径里更顺手的那一条来走，不必担心选错了顺序结果不同。

来验证一下。设 \(f(x, y) = x^3 y^2 + e^{xy}\)，分别用两种顺序算 \(f_{xy}\)。

先 \(x\) 再 \(y\)：\(f_x = 3x^2 y^2 + ye^{xy}\)。再对 \(y\) 求：\(f_{xy} = 6x^2 y + (e^{xy} + y \cdot xe^{xy}) = 6x^2 y + e^{xy}(1 + xy)\)。

先 \(y\) 再 \(x\)：\(f_y = 2x^3 y + xe^{xy}\)。再对 \(x\) 求：\(f_{yx} = 6x^2 y + (e^{xy} + x \cdot ye^{xy}) = 6x^2 y + e^{xy}(1 + xy)\)。

两次答案完全相同——这正是克莱罗定理在替我们打保票。

克莱罗定理的"为什么"是什么？它的深意能用一个二阶差分论证看清。考虑下面这个看似毫无来头的二维差分：

\[\Delta(h, k) = f(a + h, b + k) - f(a + h, b) - f(a, b + k) + f(a, b).\]

这是"沿矩形顶点取值的交替和"。你可以从两条不同的路径来组织它。先算 \(y\) 方向的差分，再算 \(x\) 方向：

\[\Delta(h, k) = \big[f(a + h, b + k) - f(a + h, b)\big] - \big[f(a, b + k) - f(a, b)\big].\]

方括号里的两项都是"在固定 \(x\) 时 \(f\) 关于 \(y\) 的变化"，所以方括号约等于 \(k \cdot f_y\)。两个方括号之差又是"\(f_y\) 关于 \(x\) 的变化"，约等于 \(h \cdot (f_y)_x = h \cdot f_{yx}\)。整个起来 \(\Delta \approx hk \cdot f_{yx}\)。

同样这个 \(\Delta\) 也可以反过来分组，先算 \(x\) 方向再算 \(y\) 方向：

\[\Delta(h, k) = \big[f(a + h, b + k) - f(a, b + k)\big] - \big[f(a + h, b) - f(a, b)\big],\]

类似地得到 \(\Delta \approx hk \cdot f_{xy}\)。

同一个 \(\Delta\)，同样除以 \(hk\) 后取极限，给出两个表达式 \(f_{xy}\) 与 \(f_{yx}\)。如果偏导都连续，这两个极限必须相等——这就是克莱罗定理本质上的证明思路。它告诉你混合偏导数的相等不是巧合，而是 \(\Delta\) 这种"矩形差分"用两种顺序展开都得到同一个数的代数事实。形式化的版本（带中值定理）见 A.4 节。

更高阶的偏导数同样推广。对大多数良好（足够光滑）的函数，你按任何顺序对各变量求任意次数的偏导，结果只依赖于"对每个变量求导的次数"，而不依赖具体顺序。所以在物理学和工程文献里你会看见 \(\dfrac{\partial^3 f}{\partial x^2 \partial y}\) 这种写法，并不指明 \(y\) 的求导是第几步进行的——因为根本无所谓。

最后值得一提：克莱罗定理的连续性假设并非空摆设。存在一个著名的人工反例（涉及 \(f(x, y) = \dfrac{xy(x^2 - y^2)}{x^2 + y^2}\)，原点处定义为 \(0\)），它在原点处偏导数都存在，但 \(f_{xy}(0, 0) = -1 \neq 1 = f_{yx}(0, 0)\)。问题恰好出在原点处 \(f_{xy}\) 与 \(f_{yx}\) 不连续——克莱罗定理的前提失效。除了作为警示，这个反例你大概一辈子也遇不到——你将处理的所有"自然"函数都满足克莱罗定理。详见 A.4 节。

第 8 章我们将利用偏导数构造曲面 \(z = f(x, y)\) 的切平面——一元切线的高维类比；以及函数的线性化，即一元线性近似 \(f(a + h) \approx f(a) + f'(a)h\) 在多元世界里的对应物。可以预告一下：那个公式将长成 \(f(a + h, b + k) \approx f(a, b) + f_x(a, b)\, h + f_y(a, b)\, k\)——你看，两个偏导数自然地各占一项，分别贡献 \(x\) 方向和 \(y\) 方向上的一阶变化。\(f_x\) 与 \(f_y\) 是构成多元微积分一切线性近似的两块基石。
