第 8 章 切平面与线性化

一元微积分里有一个被反复使用的小奇迹：导数 \(f'(a)\) 给出 \(y = f(x)\) 在 \(x = a\) 处切线的斜率，而那条切线是函数图象在该点附近的最佳直线近似。整个一元微积分的应用——线性近似、微分、误差估计、牛顿法——几乎都从这一根切线引出来。曲线很复杂,直线很简单,所以我们用直线骗过去一段,只要不走太远就骗得过去。

多元的故事完全平行,只不过"切线"被换成了切平面。它仍是图象的最佳线性近似——在二维输入下,所谓"线性"已经不是直线,而是平面。如果你愿意把这个比喻贯彻到底:把一张曲面想成揉皱的纸,切平面就是把它在某一点附近"压平"以后得到的那张水平的硬纸板;只要你不离那一点太远,硬纸板和曲面差不到哪儿去。等到你真的需要"看见"切平面是什么东西的时候,这一章给的图景会很顶用。

我们要做三件事:先建立切平面公式,把它从两条偏导数斜率拼起来;然后把切平面看成一个近似函数,叫它线性化,用它代替原函数估算东西;最后用全微分把"小变化引起的小变化"写得干净利落,处理误差传播这一类极其常见的工程问题。

8.1 图象的切平面

设 \(z = f(x, y)\) 是一张曲面,\((a, b)\) 是定义域中的一点。如果 \(f\) 在 \((a, b)\) 附近"足够光滑",意思是偏导数 \(f_x, f_y\) 在该点连续(这是可微的标准充分条件,详见 A.3 节),那么曲面在点 \((a, b, f(a,b))\) 处有切平面,它的方程是

\[\boxed{z = f(a, b) + f_x(a, b)(x - a) + f_y(a, b)(y - b).}\]

这个公式不是凭空蹦出来的,它有非常清楚的几何来源,而且只要把两条独立的"切线"在心里拼一拼,就能自己推出来。我们慢慢说。

想象你站在曲面上的点 \(P_0 = (a, b, f(a,b))\),要描述曲面在你脚下那一小片是什么样子。先别贪心,只朝两个方向看。

第一个方向,固定 \(y = b\) 不动,只让 \(x\) 变化。这就是把整张曲面与平面 \(y = b\) 求交,得到 \(x\)–\(z\) 平面里的一条曲线 \(z = f(x, b)\)。这条曲线在 \(x = a\) 处的斜率,根据偏导数的定义,正是 \(f_x(a, b)\)。所以如果你沿着 \(+x\) 方向小步走 \(h\) 个单位,高度大约抬升 \(f_x(a, b)\,h\)。这是一根二维世界里熟悉的切线,完全没有新东西。

第二个方向,固定 \(x = a\) 不动,只让 \(y\) 变化。同理,你得到曲线 \(z = f(a, y)\) 与一根斜率 \(f_y(a, b)\) 的切线。沿 \(+y\) 走 \(k\) 个单位,高度大约抬升 \(f_y(a, b)\,k\)。

现在的关键观察是:这两根切线是相交于 \(P_0\) 的两条不平行的直线。在三维空间里,过一点的两条不平行直线恰好确定唯一一张平面。这张平面就是切平面。如果你从 \(P_0\) 出发,移动了 \((h, k)\),意味着 \(x\) 加了 \(h\) 同时 \(y\) 加了 \(k\),那么按照线性叠加的精神,高度的总改变量大约是两份贡献相加:\(f_x(a, b)\,h + f_y(a, b)\,k\)。把 \(h = x - a\)、\(k = y - b\) 代回去,把基础高度 \(f(a, b)\) 加上,就是上面的方程。

于是切平面的方程其实是"切线公式套两遍"。一元情形里 \(y = f(a) + f'(a)(x - a)\);多元情形里,你给每一个独立变量加一项 \(f_{(\text{该变量})}(a, b) \cdot (\text{该变量} - \text{基点})\)。这种"每变量配一项"的结构,我们后面还会反复看到——它的灵魂就是线性。

看一道具体题来把公式坐稳。求 \(z = x^2 + y^2\) 在 \((1, 2)\) 处的切平面。先算基点高度:\(f(1, 2) = 1 + 4 = 5\)。然后两条偏导数:\(f_x = 2x\) 在 \((1,2)\) 是 \(2\);\(f_y = 2y\) 在 \((1, 2)\) 是 \(4\)。代入公式:

\[z = 5 + 2(x - 1) + 4(y - 2) = 2x + 4y - 5.\]

检查一下:把 \((x, y) = (1, 2)\) 代回平面方程,得 \(z = 2 + 8 - 5 = 5\),正好等于 \(f(1, 2)\)。这是个最起码的合理性测验:切平面应当通过切点。

这个曲面 \(z = x^2 + y^2\) 是一个朝上开口的旋转抛物面。在点 \((1, 2, 5)\) 处,它正好处在距离原点的圆 \(x^2 + y^2 = 5\) 上方。如果你站在那一点,曲面在你脚下既向 \(+x\) 也向 \(+y\) 方向上扬,正切平面的两个斜率 \(2\) 和 \(4\) 反映了这种上扬,而且 \(y\) 方向更陡——这跟你直觉上感到"\(y = 2\) 比 \(x = 1\) 更远离原点,所以那个方向坡更陡"是一致的。

再看一道。求 \(z = e^{x}\cos(y)\) 在 \((0, 0)\) 处的切平面。基点高度 \(f(0, 0) = 1\)。\(f_x = e^x \cos y\) 在原点是 \(1\);\(f_y = -e^x \sin y\) 在原点是 \(0\)。所以切平面是

\[z = 1 + 1\cdot(x - 0) + 0\cdot(y - 0) = 1 + x.\]

这个结果几何上可以这样读:在原点这一小片,曲面看起来像一条直线 \(z = 1 + x\)(画在 \(y = 0\) 的那张平面里)被沿 \(y\) 轴方向平移出去形成的"水平条带"。原因是在 \(y = 0\) 附近,\(\cos y\) 几乎等于 \(1\),而且它对 \(y\) 的一阶变化为零(\(\sin 0 = 0\))。也就是说,沿 \(y\) 方向曲面瞬间是平的;只有沿 \(x\) 方向才有非零斜率。

再多一道。求 \(z = \ln(x^2 + y^2)\) 在 \((1, 0)\) 处的切平面。\(f(1, 0) = \ln 1 = 0\)。\(f_x = \dfrac{2x}{x^2 + y^2}\) 在 \((1, 0)\) 是 \(2\);\(f_y = \dfrac{2y}{x^2 + y^2}\) 在 \((1, 0)\) 是 \(0\)。所以切平面 \(z = 0 + 2(x - 1) + 0\cdot(y - 0) = 2x - 2\)。这一题里 \(f_y\) 又变成零了,和上一题一样,但理由不同:这次是因为分子里那个 \(y\) 让偏导数对 \(y = 0\) 这条横轴上每一点都是零。

最后一道偏理论的。求 \(z = xy\) 在 \((2, -3)\) 处的切平面。\(f(2, -3) = -6\)。\(f_x = y\) 在 \((2, -3)\) 是 \(-3\);\(f_y = x\) 在 \((2, -3)\) 是 \(2\)。所以切平面是 \(z = -6 - 3(x - 2) + 2(y + 3) = -3x + 2y\)。整张曲面 \(z = xy\) 是一个鞍面;切平面在不同的点上斜率会很不同——比如在原点,两个偏导数都是零,切平面就是水平的 \(z = 0\)。鞍点这件事我们留到第 11 章再细究。

8.2 线性化与微分

把切平面当作 \(f\) 的近似函数来用,有专门的名字:线性化。我们记它为 \(L(x, y)\):

\[\boxed{L(x, y) = f(a, b) + f_x(a, b)(x - a) + f_y(a, b)(y - b).}\]

只要 \((x, y)\) 在 \((a, b)\) 附近,就有 \(L(x, y) \approx f(x, y)\)。这个近似的精度随你离 \((a, b)\) 的远近而变好/变差:贴着基点的时候,精度是惊人地好;走远了,曲面的"曲"性就显现出来,平面与曲面分道扬镳。

为什么我们在意这种近似?最直接的理由就是:像 \(f(x, y) = \sqrt{x^2 + y^2}\) 这种函数,直接代入算非整数点是麻烦的,但偏导数的形式很简单,只需要做几次乘法和加法就能给出一个相当不错的估计值。计算器、计算机都还没出现的年代,这是工程师的命根子。今天虽然不再有人靠这个开根号,但把复杂函数在某点附近换成线性函数这件事本身——我们叫它"线性化的思维方式"——遍布所有应用数学:误差分析、控制系统、机器学习里的梯度下降,都离不开它。

把变化量的记号引进来,故事会更整洁。记 \(\Delta x = x - a\)、\(\Delta y = y - b\) 是输入的变化量,\(\Delta f = f(x, y) - f(a, b)\) 是输出的真实变化量。线性化的关系翻译过来就是

\[\Delta f \approx f_x(a, b)\,\Delta x + f_y(a, b)\,\Delta y.\]

右边这一对加项有自己的名字,叫 \(f\) 的全微分,记作

\[df = f_x(a, b)\, dx + f_y(a, b)\, dy.\]

这里我们故意写 \(dx, dy\) 而不写 \(\Delta x, \Delta y\),想强调它是"微小量"——但本质上这只是一种写法上的约定,运算时把 \(dx, dy\) 当作就是 \(\Delta x, \Delta y\) 处理。整个等式说的事是同一件:

\[\Delta f \approx df = f_x \, dx + f_y \, dy.\]

这里有一段值得静下来想清楚的哲学。\(\Delta f\) 是真实的变化:你输入真的从 \((a, b)\) 移到了 \((a + dx, b + dy)\),输出真的发生了某个改变。而 \(df\) 是切平面预测的变化:它假装你脚下还是一张平面,然后按那张平面的斜率乘以位移给出一个估计。所以 \(\Delta f\) 是事实,\(df\) 是模型。这个模型的"误差" \(\Delta f - df\) 是高阶小量,即在 \((dx, dy) \to (0, 0)\) 时它趋向 0 的速度比 \(\sqrt{dx^2 + dy^2}\) 还要快——也就是说,在足够小的尺度上,事实和模型几乎一致。

记号上有一处必须留心,免得后面被绕进去。一元里我们写 \(df = f'(x)\,dx\),把 \(f'\) 写成 \(\dfrac{df}{dx}\),那两个 \(d\) 看起来是一样的。多元里就不一样了:偏导数用的是 \(\partial\)(圆头),全微分用的是 \(d\)(直头)。\(df\) 由所有偏导数组合而成,不是单独某一个偏导数。等到你写 \(\dfrac{\partial f}{\partial x}\) 的时候,你处理的是"只让 \(x\) 变其他不动";等到你写 \(df\) 的时候,你已经在让所有变量同时变。

这一节的最后再举一个抽象一点的例子,把全微分的样子坐稳。设 \(z = x^2 \ln(y)\)。则 \(\dfrac{\partial z}{\partial x} = 2x\ln(y)\),\(\dfrac{\partial z}{\partial y} = \dfrac{x^2}{y}\)。所以全微分 \(dz = 2x\ln(y)\,dx + \dfrac{x^2}{y}\,dy\)。当 \((x, y) = (1, e)\) 时,这变成 \(dz = 2\,dx + \dfrac{1}{e}\,dy\)。如果你给 \(x\) 一个小扰动 \(dx = 0.01\)、给 \(y\) 一个小扰动 \(dy = -0.02\),那么 \(z\) 大约变化 \(2(0.01) + \dfrac{1}{e}(-0.02) = 0.02 - 0.02/e \approx 0.02 - 0.0074 \approx 0.0126\)。

8.3 估计小变化与误差传播

线性化最常见的两种用途几乎是同一回事的两种打扮:用偏导数和已知点的数据,把一个不便直接算的量"用近似的方式"算出来;或者反过来,从输入的不确定性估计输出的不确定性,这件事工程上叫误差传播。两种情形下,你做的都是用 \(df\) 代替 \(\Delta f\)。

比如,用线性化估计 \(\sqrt{(2.05)^2 + (2.95)^2}\)。一开始我会想:这个数我没办法心算。但我注意到 \(2.05\) 离 \(2\) 很近,\(2.95\) 离 \(3\) 很近,而 \(\sqrt{2^2 + 3^2} = \sqrt{13}\) 是我能直接处理的"基点"。所以把 \(f(x, y) = \sqrt{x^2 + y^2}\),要算 \(f(2.05, 2.95)\),基点选 \((a, b) = (2, 3)\),则 \(\Delta x = 0.05\)、\(\Delta y = -0.05\)。

偏导数:\(f_x = \dfrac{x}{\sqrt{x^2 + y^2}}\),\(f_x(2, 3) = \dfrac{2}{\sqrt{13}}\);\(f_y = \dfrac{y}{\sqrt{x^2 + y^2}}\),\(f_y(2, 3) = \dfrac{3}{\sqrt{13}}\)。把它们装进线性化公式:

\[f(2.05, 2.95) \approx \sqrt{13} + \frac{2}{\sqrt{13}}(0.05) + \frac{3}{\sqrt{13}}(-0.05) = \sqrt{13} + \frac{0.1 - 0.15}{\sqrt{13}} = \sqrt{13} - \frac{0.05}{\sqrt{13}}.\]

由 \(\sqrt{13} \approx 3.6056\) 与 \(0.05/\sqrt{13} \approx 0.01387\),估计值约为 \(3.5917\)。如果你拿计算器去验证:\(\sqrt{4.2025 + 8.7025} = \sqrt{12.905} \approx 3.5924\),误差约 \(0.0007\)。在这种距离下,线性近似的精度相当让人满意。

顺便说一句,这个例子里 \(\Delta x\) 和 \(\Delta y\) 的符号对答案至关重要——它们一正一负,所以两个偏导数的贡献部分互相抵消,留下的是一个小负数。如果你疏忽地把两者都当成正数,会得到错误的方向。

再看一个味道完全不同的应用:误差传播。一个圆柱体的半径 \(r = 5\,\mathrm{cm}\)、高度 \(h = 10\,\mathrm{cm}\),但每个量在测量时都可能有不超过 \(0.1\,\mathrm{cm}\) 的误差。问体积可能有多大误差?

体积 \(V = \pi r^2 h\) 是 \((r, h)\) 的函数。它的全微分是

\[dV = \frac{\partial V}{\partial r}\,dr + \frac{\partial V}{\partial h}\,dh = 2\pi r h\, dr + \pi r^2\, dh.\]

在 \((r, h) = (5, 10)\) 处:\(dV = 2\pi(5)(10)\, dr + \pi(25)\, dh = 100\pi\, dr + 25\pi\, dh\)。

最坏情况是 \(|dr|, |dh| \le 0.1\) 而且同号(所以两份贡献加起来,而不是相互抵消):

\[|dV| \le 100\pi(0.1) + 25\pi(0.1) = 12.5\pi \approx 39.3\,\mathrm{cm}^3.\]

体积本身约为 \(250\pi \approx 785\,\mathrm{cm}^3\),所以相对误差约 \(39.3/785 \approx 5\%\)。这个数字其实告诉了我们一件实际的事:在估算一个圆柱体积时,半径的小误差比高的小误差影响大得多——\(100\pi\) 的系数比 \(25\pi\) 大四倍。如果想把体积测准,优先把半径测准。

顺带把容易栽跟头的地方挑明:误差传播题里的 \(|dr|, |dh|\) 都按正值代入并相加,因为各分量误差最坏的情况是同号——一同把体积往上推或一同把体积往下推。如果你直接把它们减掉,得到的是"误差恰好相互抵消"的最好情形,通常不是题目要的。

再看一个工程感很强的例子:测量直角三角形的两直角边,得 \(a = 3.00\,\mathrm{m}\)、\(b = 4.00\,\mathrm{m}\),每个测量误差最大 \(0.01\,\mathrm{m}\)。估计斜边 \(c = \sqrt{a^2 + b^2}\) 的最大可能误差。

\(c\) 的全微分:\(\dfrac{\partial c}{\partial a} = \dfrac{a}{\sqrt{a^2 + b^2}} = \dfrac{a}{c}\),\(\dfrac{\partial c}{\partial b} = \dfrac{b}{c}\)。在 \((a, b) = (3, 4)\) 处,\(c = 5\),所以两个偏导数分别是 \(0.6\) 和 \(0.8\)。最坏情况下,

\[|dc| \le 0.6(0.01) + 0.8(0.01) = 0.014\,\mathrm{m},\]

即斜边大约 \(5\,\mathrm{m}\) 上下浮动 \(1.4\,\mathrm{cm}\)。同样地,如果你想把斜边测得更准,先把那条较长的直角边测准——它的偏导数 \(0.8\) 比另一条的 \(0.6\) 大,影响更敏感。

再来一个百分比误差版本——这种问法在物理化学里非常常见。一只单摆的周期由 \(T = 2\pi\sqrt{L/g}\) 给出。如果摆长 \(L\) 测量有 \(0.5\%\) 的相对误差,重力加速度 \(g\) 估值有 \(0.2\%\) 的相对误差,问周期 \(T\) 的相对误差大约是多少?

这种题最干净的做法是先取对数:\(\ln T = \ln(2\pi) + \tfrac{1}{2}\ln L - \tfrac{1}{2}\ln g\)。两边取微分:

\[\frac{dT}{T} = \tfrac{1}{2}\,\frac{dL}{L} - \tfrac{1}{2}\,\frac{dg}{g}.\]

最坏情况:\(\left|\dfrac{dT}{T}\right| \le \tfrac{1}{2}(0.005) + \tfrac{1}{2}(0.002) = 0.0035\),即周期的相对误差不超过 \(0.35\%\)。这个对数把乘除法变成加减法、把幂变成系数的窍门,在工程估计里值得牢记——它本质上还是全微分,只是穿了件好看的衣服。

第 9 章我们建立多元链式法则,把"小变化引起小变化"的一阶贡献"传递"过函数复合,这样我们处理嵌套依赖关系时就有一套清楚的规则可循。
