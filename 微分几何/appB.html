B.1 Lie 群:连续对称性

对称性在数学和物理里无处不在。离散 对称(置换、晶格、群)通常用群论 处理(本系列抽象代数读本)。连续 对称——平面绕原点旋转、空间平移、时空 Lorentz 变换——需要新工具: Lie 群。

定义: 一个Lie 群 \(G\) 是一个流形(ch21) 同时也是一个群, 使乘法 \(\mu : G \times G \to G\) 与逆 \(\iota : G \to G\) 都光滑。

"光滑流形" + "群结构" 这两套结构在 \(G\) 上必须兼容(乘法 + 逆是光滑映射)。Sophus Lie 1873 年引入这个概念时, 是为了对称性能微分——能问"无穷小 对称性是什么"。

最朴素例子: \(\mathbb{R}^n\) 在加法下。流形 = \(\mathbb{R}^n\), 群结构 = 向量加法。光滑性显然。这是阿贝尔(交换) Lie 群——结构最简单。

更有趣: \((\mathbb{R}^*, \cdot)\) 是非零实数在乘法下——一个 1 维 Lie 群。\((\mathbb{C}^*, \cdot)\) 是 2 维。

下一节给真正"非平凡" 的——矩阵 Lie 群。

B.2 矩阵 Lie 群:三个家族

大部分实战 Lie 群是矩阵群: 是 \(GL_n(\mathbb{R})\)(\(n \times n\) 实可逆矩阵)的子群, 同时也是子流形。它们由线性方程或保某种结构 来定义。

例 1: \(GL_n(\mathbb{R}) = \{A \in \mathbb{R}^{n \times n} : \det A \neq 0\}\)。开子集 \(\subset \mathbb{R}^{n^2}\), 维度 \(n^2\)。乘法 \(AB\) 显然光滑(分量是多项式), 逆 \(A^{-1}\) 由 Cramer 公式给, 在 \(\det \neq 0\) 处光滑。所以 \(GL_n\) 是 \(n^2\) 维 Lie 群。不 紧(\(\det\) 可任意大), 不 连通(两个分量: \(\det > 0\) 和 \(\det < 0\))。

例 2: 特殊正交群 \(SO(n) = \{A : A^T A = I, \det A = 1\}\)。即 \(\mathbb{R}^n\) 中保内积 + 保定向的线性变换。维度 \(n(n-1)/2\)。\(n = 2\): \(SO(2) =\) 圆周(2D 旋转), 1 维。\(n = 3\): \(SO(3)\) 是 3D 旋转群, 3 维, 同胚于实射影空间 \(\mathbb{RP}^3\) (一个非平凡的紧致 3 维流形)。\(n = 4\): \(SO(4)\) 是 6 维, 在物理学里和 \(SU(2) \times SU(2)\)双重 覆盖关系。

为什么 \(SO(n)\) 是流形? 用附录 A.5 隐函数定理。\(F : \mathbb{R}^{n^2} \to \mathbb{R}^{n(n+1)/2}\), \(F(A) = A^T A - I\)(对称矩阵, 有 \(n(n+1)/2\) 独立分量)。\(F^{-1}(0) = O(n)\) 加 \(\det = 1\) 切到 \(SO(n)\)。隐函数定理给 \(F^{-1}(0)\) 是 \(n^2 - n(n+1)/2 = n(n-1)/2\) 维子流形。

例 3: 特殊酉群 \(SU(n) = \{A \in \mathbb{C}^{n \times n} : A^* A = I, \det A = 1\}\)。即复 \(\mathbb{C}^n\) 中保 Hermite 内积 + 行列式 1 的变换。维度 \(n^2 - 1\)。\(n = 2\): \(SU(2)\) 是 3 维, 同胚于3 球面 \(S^3\)——这件事极不显然但极重要(物理学里"电子自旋" 用 \(SU(2)\) 描述, 它和 \(SO(3)\)双重覆盖 给出"自旋 1/2" 现象)。\(n = 3\): \(SU(3)\) 是 8 维, 强相互作用的对称群(色 SU(3))。

B.3 Lie 代数:Lie 群在单位元处的"线性化"

Lie 群是个流形, 它在单位元 \(e \in G\) 处有切空间 \(T_e G\)。这个切空间叫Lie 代数, 通常记 \(\mathfrak{g}\)。

关键事实: \(\mathfrak{g}\) 上有一个自然的"括号" 运算 \([\cdot, \cdot] : \mathfrak{g} \times \mathfrak{g} \to \mathfrak{g}\), 满足:

  • 双线性。
  • 反对称 \([X, Y] = -[Y, X]\)。
  • Jacobi 恒等式 \([X, [Y, Z]] + [Y, [Z, X]] + [Z, [X, Y]] = 0\)。

这种带括号的向量空间叫Lie 代数。

对矩阵 Lie 群, Lie 代数有具体 公式:

  • \(\mathfrak{gl}_n = \) 所有 \(n \times n\) 矩阵, \([X, Y] = XY - YX\)(矩阵换位)。
  • \(\mathfrak{so}(n) = \{X : X^T = -X\}\)(反对称 矩阵), 维度 \(n(n-1)/2\), \([X, Y] = XY - YX\)。
  • \(\mathfrak{su}(n) = \{X : X^* = -X, \mathrm{tr}\, X = 0\}\)(反 Hermite+迹零矩阵), 维度 \(n^2 - 1\)。

为什么 \(\mathfrak{so}(n)\) 是反对称? 取 \(SO(n)\) 中过 \(I\) 的曲线 \(A(t)\), \(A(0) = I\)。条件 \(A^T A = I\) 求导给 \(A'(0)^T + A'(0) = 0\)。所以 \(X = A'(0) \in T_I SO(n)\) 必反对称。这是"群的限制 → 代数的限制" 的最直接例子。

Lie 群和 Lie 代数的关系靠指数映射:

\[ \boxed{\ \exp : \mathfrak{g} \to G, \quad \exp(X) = I + X + \frac{X^2}{2!} + \frac{X^3}{3!} + \cdots \ } \]

(矩阵幂级数, 对所有 \(X\) 收敛)。这个指数映射和ch14 黎曼几何里的指数映射同名——而且它们是同一概念在不同语境下的化身: 都是"把切空间映回流形"。在 Lie 群里, \(\exp\) 把 Lie 代数(切于单位元的)沿"左不变向量场" 流到 Lie 群上。

Lie 第三定理: 任何抽象 Lie 代数都是某个 Lie 群的代数。所以"Lie 群的局部结构 = Lie 代数"——这是一个线性化原理, 类似流形的"切空间是流形的局部线性化"。

B.4 不变度量与 Killing 形式

给一个 Lie 群 \(G\), 我们能在它上面放一个左不变(或双侧不变) 的黎曼度量。具体: 在单位元的切空间 \(T_e G = \mathfrak{g}\) 上选一个内积 \(\langle \cdot, \cdot \rangle_e\), 然后用左乘 \(L_g\) 把它运到每个 \(g \in G\):

\[ \langle X, Y \rangle_g := \langle (L_{g^{-1}})_* X, (L_{g^{-1}})_* Y \rangle_e. \]

这给整个 \(G\) 一个黎曼度量, 满足左乘是等距。这种度量自动让 \(G\) 是齐性空间(homogeneous)——任何两点几何上等价。

对紧 Lie 群(\(SO(n), SU(n)\) 等), 存在双侧不变 度量(同时也右不变)。这种度量更对称, 它的测地线就是单参数子群 \(\{\exp(tX) : t \in \mathbb{R}\}\)——指数映射的像。

对半单 Lie 代数(simple 或semi-simple), 有一种典范 的内积叫Killing 形式:

\[ \boxed{\ B(X, Y) := \mathrm{tr}(\mathrm{ad}_X \circ \mathrm{ad}_Y),\ } \]

其中 \(\mathrm{ad}_X : \mathfrak{g} \to \mathfrak{g}\) 是 \(\mathrm{ad}_X(Y) = [X, Y]\)。这是个对称双线性形式。

关键事实(Cartan 准则): \(\mathfrak{g}\) 半单 ⟺ \(B\) 非退化。

对紧 Lie 代数, \(B\) 是负定 的, 所以 \(-B\) 是个内积——这给了紧 Lie 群一个典范的双侧不变度量。这就是物理学里"规范场论的几何"——杨-Mills 场强用的就是这种 Killing-诱导度量。

例。\(\mathfrak{so}(3)\) 中, 选基 \(\{E_1, E_2, E_3\}\)(三个标准生成元), \([E_i, E_j] = \epsilon_{ijk} E_k\)。算 \(B(E_i, E_j) = -2 \delta_{ij}\)(Levi-Civita 给的)。所以 Killing 形式让 \(\{E_i\}\) 是个负定正交基, \(SO(3)\) 上的标准度量(\(\sim\) 圆球度量)就是 \(-B/2\)。

B.5 联络与曲率张量(更细致版)

本书ch22 给了 Christoffel 符号 \(\Gamma^k_{ij}\) 与 Riemann 张量 \(R^l_{ijk}\) 的定义。这一节把它们提升到更现代的"联络" 语言, 这正是黎曼几何 / 规范场论用的。

抽象定义: 黎曼流形 \((M, g)\) 上的仿射联络 \(\nabla\) 是一种"沿向量场求导" 的算子: 给两个向量场 \(X, Y\), 给出新向量场 \(\nabla_X Y\), 满足:

  • 对 \(X\) 是 \(C^\infty(M)\)-线性: \(\nabla_{fX + Y} Z = f \nabla_X Z + \nabla_Y Z\)。
  • 对 \(Y\) 是 \(\mathbb{R}\)-线性 + Leibniz: \(\nabla_X (fY) = (Xf) Y + f \nabla_X Y\)。

Levi-Civita 联络: 在黎曼流形上, 存在唯一 的联络满足两个额外条件:

  • 无挠: \(\nabla_X Y - \nabla_Y X = [X, Y]\)。
  • 度量相容: \(X \langle Y, Z \rangle = \langle \nabla_X Y, Z \rangle + \langle Y, \nabla_X Z \rangle\)。

这条唯一性叫"黎曼几何基本定理"。在坐标里, Levi-Civita 联络的分量就是 Christoffel 符号:

\[ \nabla_{\partial_i} \partial_j = \Gamma^k_{ij} \partial_k. \]

具体公式 ch22 给过:

\[ \Gamma^k_{ij} = \frac{1}{2} g^{kl}(\partial_i g_{lj} + \partial_j g_{il} - \partial_l g_{ij}). \]

曲率张量 用联络换位定义:

\[ R(X, Y) Z := \nabla_X \nabla_Y Z - \nabla_Y \nabla_X Z - \nabla_{[X, Y]} Z. \]

这是三参数 \(\to\) 一参数 的张量——告诉你"沿不同顺序求导 差多少"。这就是 ch22 的 Riemann 张量, 但写成无坐标形式。

从 \(R\) 缩并(contract)得 Ricci 张量 \(R_{ij}\), 再缩并得标量曲率 \(R\)——和ch23 里 GR 用的完全同款。

这就是为什么 Cartan / Lee / do Carmo 等黎曼几何教材一开始就用"联络 + 张量" 的语言——它独立于 坐标, 让证明结构清楚。

B.6 Cartan 结构方程预告

把上面联络 + 曲率重新用微分形式(ch12)写, 得到 Cartan 1899 年的"结构方程"。这是黎曼几何里最优雅的公式之一。

设流形 \(M\) 上选一个正交标架场 \(\{e_1, \ldots, e_n\}\)。对偶 1-形式 \(\{\theta^1, \ldots, \theta^n\}\) 满足 \(\theta^i(e_j) = \delta^i_j\)。联络 1-形式 \(\omega^i_j\) 由

\[ \nabla e_j = \omega^i_j \otimes e_i \]

给出。曲率 2-形式 \(\Omega^i_j\) 由

\[ R(\cdot, \cdot) e_j = \Omega^i_j \otimes e_i \]

给出。

Cartan 第一 结构方程(无挠):

\[ \boxed{\ d\theta^i + \omega^i_j \wedge \theta^j = 0.\ } \]

Cartan 第二 结构方程(曲率):

\[ \boxed{\ \Omega^i_j = d\omega^i_j + \omega^i_k \wedge \omega^k_j.\ } \]

这两条公式让计算曲率变得大幅 简化——不用算 Christoffel 符号一个个偏导, 直接做 \(d\theta\) 和 \(\omega \wedge \omega\)。它在物理 / 工程实际计算 GR 时用得最多——一旦你有度量的好选择(标架不一定要是坐标基), 算曲率就是几行 Cartan 演算。

更深一层: 这两条方程把"挠" 与"曲率" 都识别为"外微分 + 楔积" 的具体表达——这建立了"规范场论" 的几何框架。在规范场论里:

  • \(\omega\) = "规范势"(电磁场的 \(A_\mu\), Yang-Mills 的非阿贝尔推广)。
  • \(\Omega\) = "场强"(电磁场张量 \(F_{\mu\nu}\), Yang-Mills 场强)。
  • \(\Omega = d\omega + \omega \wedge \omega\) = 标准物理公式 \(F = dA + A \wedge A\)。

所以本书ch12 入门的微分形式 + 这一节的 Cartan 方程, 是物理学家描述基本相互作用的同一种 数学。爱因斯坦的引力 + Maxwell 的电磁 + 杨-Mills 的强弱核力, 都是这个公式的某种推广。

B.7 通向 Riemann 几何

"黎曼几何" 这门学科的入口在哪?

主要话题:

  • 测地线方程的整体理论: Hopf-Rinow 定理(ch15)的完整证明; Jacobi 场(测地线"邻居" 的演化, 由 Riemann 张量驱动); 共轭点 / 截断点。
  • 比较几何: 给曲率上下界, 推出体积、直径、谱的整体结论。Bonnet-Myers 定理(Ricci 曲率 \(\geq c > 0\) ⟹ 直径 \(\leq \pi/\sqrt c\))、Toponogov 比较定理。
  • 整体定理: 球定理(\(1/4 < K \leq 1\) ⟹ 流形同胚 \(S^n\))、Cheeger 有限性定理、几何分析里的 Ricci 流(Hamilton 1982, Perelman 2003 用它证明 Poincaré 猜想)。
  • 对称空间: 黎曼几何里"最对称" 的流形, 由 Lie 群作用分类(\(\mathbb{R}^n, S^n, \mathbb{H}^n\), 复 / 四元数射影空间, 等等)。
  • 谱几何: Laplace-Beltrami 算子的特征值序列编码流形的几何信息(Mark Kac "Can one hear the shape of a drum?"——一般不能, 但能听到很多)。

推荐书目:

  • do Carmo, Riemannian Geometry: 经典入门, 直接从本书 ch22 风格往下走。
  • Lee, Introduction to Riemannian Manifolds: 现代风格, 张量与联络优先。
  • Petersen, Riemannian Geometry: 偏比较几何与几何分析。
  • Jost, Riemannian Geometry and Geometric Analysis: 进阶, 偏分析视角。

对Lie 群 / 表示论:

  • Hall, Lie Groups, Lie Algebras, and Representations: 矩阵 Lie 群入门, 不需要太多流形背景。
  • Fulton-Harris, Representation Theory: 标准研究生教材, 严整。
  • Knapp, Lie Groups Beyond an Introduction: 半单 Lie 群完整理论。

对规范场论 / 现代几何物理:

  • Baez-Muniain, Gauge Fields, Knots, and Gravity: 物理优先, 快速通向 Yang-Mills + 量子拓扑。
  • Nakahara, Geometry, Topology and Physics: 百科风格, 物理学家用。
  • Frankel, The Geometry of Physics: 数学偏向, 微分形式 + 规范理论 + GR。

B.8 总结:本书工具如何延伸

把本书 24 章的工具映射到下一步:

  • ch6 正则曲面 → \(SO(n), SU(n)\) 等矩阵 Lie 群是子流形(B.2)。
  • ch7 第一基本形式 → 黎曼度量 → 不变度量 / Killing 形式(B.4)。
  • ch13 测地线 → Lie 群的单参数子群(指数映射的像)。
  • ch14 指数映射 → Lie 群的指数映射 \(\exp : \mathfrak{g} \to G\)(B.3)。
  • ch12 微分形式 → Cartan 结构方程 + 规范场论(B.6)。
  • ch22 Christoffel + Riemann 张量 → Levi-Civita 联络 + 曲率张量的现代表述(B.5)。
  • ch23 GR → 弦理论 / 规范场论 / 几何分析(B.7)。

请你看见: 本书是古典 微分几何, 但古典里每个工具都已经 是现代的种子。从 do Carmo 风格的"曲面在 \(\mathbb{R}^3\) 中" 到 Lee 风格的"抽象流形 + 张量场", 距离不远——只需把"嵌入到 \(\mathbb{R}^3\)" 这件事去掉, 用抽象的 流形 + 张量代替, 就到现代黎曼几何。

这就是为什么我们在ch21 引入抽象流形, 在ch22 引入黎曼度量——它们是从"古典曲面" 通往"抽象黎曼几何" 的桥。本书结束在桥的一端; 你下一步要去的就是桥的另一端, 那里有整个 现代几何与物理在等你。

祝你的下一段旅程顺利。
