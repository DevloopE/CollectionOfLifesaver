第 1 章 引论：从弯曲到曲率

请你做一道你看得见 的事。拿一张 A4 纸。把它卷 成圆筒——不要折，只是卷。再放回平面。再卷。这件事你能做无数次, 不撕不裂。

现在再拿一张纸, 试着把它包裹 到一个橘子上——不撕不揉, 让纸贴橘子皮。做不到。无论怎么努力, 纸要么撕开, 要么起皱褶, 要么留下一块没贴上。

请你停下来想一秒。第一件事（卷成圆筒）和第二件事（贴到球面）从平面到曲面 的几何变换看起来都"很温和"——没拉伸纸, 没改变纸上任何两点的距离, 只是让纸在 3D 空间里换个姿态。但一件能做, 一件做不到。区别在哪？

区别叫做Gauss 曲率。1827 年 Gauss 证明了一条让他自己用拉丁文激动写下"Theorema Egregium"（"卓越定理"）的事实——一个 2D 生物只靠在曲面上量距离, 就能区分"能从纸卷出来" 的曲面和"不能从纸卷出来" 的曲面。这道区分的工具就是 Gauss 曲率。它内蕴——不依赖曲面如何嵌入 3D 空间, 只依赖曲面自身 的几何。

这一章把"曲率" 这个概念从最朴素的"圆弯不弯" 一步步建立到 Gauss Theorema Egregium 的入口。整本书 24 章基本就是这个故事的展开。

1.1 一道你画过却没问过的事：圆怎么"弯"

画一个半径为 1 的圆。再画一个半径为 5 的圆。两个都"弯"——但你看得见 第一个比第二个更弯。

这件直觉的事怎么数学化？答案是曲率 \(\kappa\)。对一个圆, 定义

\[ \kappa = \frac{1}{r}. \]

半径 \(r\) 越大, 曲率越小; 直线 \(r = \infty\), 曲率 \(\kappa = 0\)。这条定义符合直觉——大圆"更直", 小圆"更弯"。

但这只对圆。一般曲线（比如抛物线、椭圆、心形线）每个点的"弯" 都不同——曲线的某些段更直, 某些段更弯。怎么逐点 定义曲率？

核心想法：在曲线 \(\gamma\) 上的每一点 \(P\), 找一个最佳贴合的圆——那个圆叫"密切圆"（osculating circle）。它的半径 \(r\) 倒数 \(1/r\) 就是曲线在 \(P\) 处的曲率。

具体怎么找密切圆？用Taylor 展开二阶项。把 \(\gamma\) 在 \(P\) 附近展到二阶, 二阶项决定弯曲。第 3 章给具体公式 \(\kappa(t) = |\gamma' \times \gamma''| / |\gamma'|^3\); 弧长参数化下简化为 \(\kappa(s) = |\gamma''(s)|\)。

所以"曲线弯多少" 完全数学化——给曲线 \(\gamma\) + 点 \(P\), 给一个数 \(\kappa(P)\)。这个数捕获了我们直观说的"弯"。

1.2 曲面也"弯", 但弯有方向

把这套搬到曲面上有个新问题：曲线只有一种切方向, 弯曲就是一种意思。曲面在每点有一整个切平面——不同方向上"弯多少" 不一样。

具体看几个例子：

球面（半径 \(r\)）。每个点切平面是 2 维。任选切方向 \(v\), 沿 \(v\) 走的曲线"弯" 多少？同一个圆——大圆——曲率 \(1/r\)。所有方向都一样。所以球面"各向同性弯"。

圆柱面（半径 \(r\)）。沿圆周方向走, 是个圆, 曲率 \(1/r\)。沿轴向走, 是直线, 曲率 \(0\)。两个垂直方向曲率相差 很大。圆柱"各向异性弯"——一个方向圆弯, 一个方向直。

鞍面（如 \(z = x^2 - y^2\)）。沿一个方向（\(x\)-轴）向上 弯（凸），沿垂直方向（\(y\)-轴）向下 弯（凹）。两个曲率异号。

所以曲面在每点的"弯"由所有切方向上的曲率 一起描述。把它们组织起来——主曲率 \(\kappa_1, \kappa_2\) 是最大 与最小 的方向曲率（互相垂直的两个方向）。其它方向的曲率夹在 \(\kappa_2\) 与 \(\kappa_1\) 之间。

三个例子的主曲率：

  • 球面（\(r\)）：\(\kappa_1 = \kappa_2 = 1/r\)。
  • 圆柱（\(r\)）：\(\kappa_1 = 1/r, \kappa_2 = 0\)。
  • 鞍面：\(\kappa_1 > 0, \kappa_2 < 0\)（两数异号）。

但记住两个 数比记一个 数烦。能不能搓成一个？

1.3 两个搓成一个：Gauss 曲率

定义

\[ \boxed{\ K := \kappa_1 \kappa_2.\ } \]

叫Gauss 曲率。三个例子：

  • 球面（\(r\)）：\(K = (1/r)(1/r) = 1/r^2 > 0\)。
  • 圆柱（\(r\)）：\(K = (1/r) \cdot 0 = 0\)。
  • 鞍面：\(K = \kappa_1 \kappa_2 < 0\)（异号相乘）。

所以 \(K\) 把曲面分成三类:

  • \(K > 0\): "椭圆型"（球面、椭球面）——两方向都弯到同侧。
  • \(K = 0\): "抛物型"或"平的"（平面、圆柱、圆锥）——至少一方向直。
  • \(K < 0\): "双曲型"（鞍面、双曲面）——两方向异向弯, 像马鞍。

这种三分对每个点都成立——一个曲面可以在不同点是不同类型（比如环面有 \(K > 0\) 的外侧 + \(K < 0\) 的内侧 + \(K = 0\) 的顶 / 底圈）。

还有一个有用的辅助量——平均曲率

\[ H := \frac{\kappa_1 + \kappa_2}{2}. \]

它度量曲面"平均" 弯多少。\(H = 0\) 的曲面叫极小曲面——肥皂膜的形状（ch19）。\(H\) 与 \(K\) 把"所有切方向的曲率信息" 浓缩为两个数。

1.4 一道让你不安的实验：纸卷得了圆柱, 卷不了球面

回到开头那两件事。把一张平面纸卷成圆柱——没问题。为什么？

关键观察：纸是平的, \(K_{\text{纸}} = 0\)。圆柱也满足 \(K_{\text{圆柱}} = 0\)（一方向 \(1/r\), 另一方向 \(0\), 乘积 \(0\)）。两个曲面 \(K\) 相同。所以从纸到圆柱, "Gauss 曲率没变"。

试着把纸贴到球面上——做不到。为什么？

因为球面 \(K = 1/r^2 > 0 \neq 0 = K_{\text{纸}}\)。"Gauss 曲率不一样"。如果你能把纸"不拉伸" 贴到球面上, 那就是说存在一个保距离的双射纸 → 球面。这种"等距" 双射会保持所有内蕴几何量——其中就有 \(K\)。但是 \(K\) 在两边不一样。所以等距双射不存在。所以纸不能 不拉伸地贴到球面。

请你停下来感受这件事的深度。我们直觉上觉得"纸贴不到球面" 是一种物理 限制（"纸太硬了"）。但其实它是数学 限制——纸 \(K = 0\) 与球面 \(K \neq 0\) 不可能 等距, 无论你的纸多软。一个 2D 生物在纸上量距离, 量出来的所有 几何关系（包括 \(K\)）都是 \(K = 0\) 风格的; 球面上的 2D 生物量出来全是 \(K = 1/r^2\) 风格的。两边的几何本质 不同。

这就是为什么每张世界地图都失真。地球（近似球面 \(K \approx 1/R^2\)）不能等距摊到平面（\(K = 0\)）。Mercator 投影保形（保角度）但纬度高的国家被严重放大（格陵兰看起来比非洲大）。Gall-Peters 投影保面积但形状变形。没有 一种世界地图能同时保面积、保角度、保距离——这是 Gauss 不让的。

1.5 Gauss 1827：\(K\) 是内蕴 的

上一节的论证暗藏 一件深得不可思议的事。我们说"纸不能贴球面" 因为等距 保 \(K\)。但等距的定义只用到"距离" 这个内蕴信息——曲面上两点间的最短曲面路径长度。所以"等距保 \(K\)" 等价于说"\(K\) 由距离决定"——即 \(K\) 只是内蕴几何的函数, 不依赖曲面在 \(\mathbb{R}^3\) 中怎么嵌入。

这件事极非显然。回看 \(K\) 的定义——\(\kappa_1, \kappa_2\) 是主曲率, 用切平面 + 法向量 定义, 法向量需要 3D 嵌入！\(K = \kappa_1 \kappa_2\) 看起来彻底是外蕴 量——它说"曲面在 3D 空间里弯成什么样"。可 Gauss 证明它居然 等于一个只用曲面距离表达的量。

形式陈述（Gauss 1827, "Theorema Egregium"）：

\[ \boxed{\ K = \kappa_1 \kappa_2 \text{ 是内蕴量, 只依赖第一基本形式}.\ } \]

"第一基本形式" 是ch7 的概念——它是一种把切向量长度 写成关于参数化的对称矩阵的工具, 不 涉及法向量。Theorema Egregium 说 \(K\) 这个看上去外蕴的东西, 居然能用第一基本形式（纯内蕴）写出来。ch11 给完整证明——历史上 Gauss 证明的具体公式（用 Christoffel 符号）。

这件事的哲学 意义：内蕴几何独立 于嵌入。一个 2D 生物在曲面上爬行, 不需要"外面 3D 空间" 的概念, 就能完全 重建曲面的本质几何。这种"抽象掉嵌入" 的视角直接通向 Riemann 1854 年的抽象流形 概念——研究"带度量的空间" 而不依赖任何外部嵌入。本书ch21-22 给抽象流形的入口, 那里 Gauss-Riemann 的视角真正成熟。

1.6 这本书的路线

24 章五段式：

  • 曲线 (ch2-5)：参数化 → 弧长 → 平面曲线 + 曲率 → 空间曲线 + Frenet-Serret → 整体曲线（4-顶点 + 等周）。这一段是热身——曲率概念的最简单化身。
  • 曲面基础 (ch6-12)：正则曲面 → 第一基本形式（内蕴度量）→ Gauss 映射 → 第二基本形式 + 主曲率 → Gauss / 平均曲率 → Theorema Egregium → 微分形式入门。这是本书的核心。
  • 测地线 (ch13-15)：曲面上的"直线" + 测地坐标 + 完备性。是 ch16-17 Gauss-Bonnet 的准备。
  • Gauss-Bonnet (ch16-18)：本书第一座大山——\(\int_M K \, dA = 2\pi \chi(M)\)（紧致曲面）。几何决定拓扑。
  • 选题 + 桥梁 (ch19-24)：极小曲面 → 曲面分类 → 抽象流形 → 黎曼度量 → 广义相对论 → 计算几何。这一段把本书的工具用出去, 通向更高级的几何与物理。

整本书的"主弧线"是：从"圆有曲率 \(1/r\)" 这种小学几何, 走到 Gauss-Bonnet 这种"积分 + 拓扑"，再到 Riemann 抽象流形的入口, 最后摸到爱因斯坦相对论。整条弧线让你看见同一个 "曲率" 概念在不同抽象层次的化身。

1.7 临别话

古典微分几何特别适合 Banner 风格——几乎所有概念都能画出来, 几乎所有定理都来自具体的可视 实验。请你在读后面 23 章时, 每一章 都准备一张草稿纸——画曲线、画曲面、画切向量、画测地线。微分几何是最不能"纯符号" 学的数学之一; 不画图你就看不见在做什么。

具体一些：第 4 章 Frenet-Serret 你画一个螺旋线, 把 \(T, N, B\) 三个标架画在曲线上几个点; ch9 第二基本形式你画一个鞍面, 看法向量在不同方向变化; ch16 Gauss-Bonnet 你画一个三角形在球面上, 自己算"三内角和 - π = 球面三角形面积 / R²"。这种动手的章节直接对应 Banner 教学法的核心——具体先于抽象。

下一章我们从最朴素的开始：曲线的参数化。一个曲线（比如圆）有无穷多种 参数化（按弧长、按角度、按 \(x\) 投影、按其他什么）。哪种"最自然"？\(\varepsilon\) 是弧长参数化——按真实弧长走。这种参数化让曲率公式变得最干净, 是后续所有几何分析的标配。
