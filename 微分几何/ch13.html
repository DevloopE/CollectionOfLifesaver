第 13 章 测地线：曲面上的"直线"

请你做一道你能画 的事。在桌上放一张 A4 纸, 画两个点, 用尺把它们连起来。这条线是直线——你不假思索就知道, 因为"直线是两点间最短路径" 这件事, 是平面几何最基本的事实。

现在拿一个橘子放在桌上, 在橘皮上点两个点。"它们之间的直线" 在哪？

请你停一秒。这个问句的表面 答案是"橘子上没有直线"——因为橘皮是弯的, 任何画在橘皮上的曲线都不直。但这是个不该满足的答案。一个 2D 生物住在橘皮上, 它需要 知道两点间最短路径。它需要"直线" 这个概念在橘皮上的对应物。它叫测地线。

对球面（橘子近似球面）, 测地线是大圆——经过球心的平面截球面得到的圆。第 1 章 提到的飞行航线就沿大圆走——纽约到东京的最短路径不是地图上的直线（Mercator 投影把它弄歪了）, 而是绕过北极附近的一段大圆。这件事每个长程国际航班乘客都看过——飞机座位前的小屏幕上那条弯弯的航线, 就是球面测地线。

这一章的目标是把"测地线" 这个概念定义清楚、方程写下来、例子算出来。我们会发现一件让人惊讶的事：测地线有两条看上去完全无关 的等价定义, 一个用变分（最短路径）, 一个用加速度（直线"不偏"）。两条等价是测地线理论的核心, 也是下下章 Gauss-Bonnet 的基础。

13.1 球面"大圆" 是飞行最短路径

先讲清楚"大圆" 是什么。球面 \(S^2 \subset \mathbb{R}^3\) 上的大圆 是过球心的平面 与 \(S^2\) 相交得到的圆——比如赤道（过球心的水平面截球）、子午圈（过球心的垂直面截球）、或任何一个"过球心" 的平面截球得到的圆。

"过球心" 这条很关键——如果截球的平面不 过球心（比如纬度圈）, 截出来是个小圆, 它的半径比球小, 它不是 测地线。

具体看：纽约（北纬 40 度, 西经 74 度）到东京（北纬 35 度, 东经 140 度）。这两个点都在北纬 35-40 度附近。一个朴素的想法是"沿纬度走"——保持纬度不变, 横穿太平洋。这条路在地图上看起来"水平直线"。但它不是最短的。

真正最短的路径是大圆航线: 从纽约出发, 先北上（向阿拉斯加方向）, 然后越过白令海峡附近, 再南下 到东京。这条曲线在 Mercator 地图上看起来像个"北弯的弧"——但它真的 比沿纬度走要短约 2000 公里（占总航程 14%）。

请你停下来想：直觉里"沿纬度走" 看起来直, "北弯的弧" 看起来弯。可真实 的最短路径是后者。这是因为我们的"看起来直" 直觉来自地图（Mercator 投影）, 而第 1 章 说过, 地图本质失真——球面到平面的投影不可能 同时保面积、保角度、保距离。"看起来直" 在地图上, 不等于"真的最短"。

这件事的几何原因是：球面没有"普通直线" 这种东西。球面上能画的曲线全部都弯——区别只是弯多少、怎么弯。其中"弯得最少" 的（即只为遵守球面 而弯, 不为别的而弯）的曲线就是大圆。它们在球面这个内蕴几何里, 扮演直线的角色。

这种思维转换是关键: 把"直线"从"绝对的"概念变为"相对于背景几何的"概念。在平面几何里直线是绝对的; 在球面几何里"直线的对应物" 是大圆; 在双曲几何里是双曲圆弧; 在一般曲面里——是测地线。

13.2 测地线定义一: 弧长泛函的临界点

把"两点间最短路径" 这件事数学化。设 \(S\) 是个曲面, \(p, q \in S\) 是两点。考虑所有曲线 \(\gamma : [0, 1] \to S\) 满足 \(\gamma(0) = p, \gamma(1) = q\)。它们的弧长

\[ L[\gamma] := \int_0^1 |\gamma'(t)| \, dt = \int_0^1 \sqrt{I(\gamma'(t), \gamma'(t))} \, dt \]

是一个泛函——一个把"曲线" 输入、输出"实数" 的映射。这里 \(I\) 是第 7 章 的第一基本形式。它告诉你在曲面 \(S\) 上, 切向量 \(v\) 的"真实长度" 是 \(\sqrt{I(v, v)}\)。

"最短路径" 就是使 \(L[\gamma]\) 最小 的曲线。这是个变分 问题——在所有连接 \(p, q\) 的曲线中找出 \(L\) 最小的那条。

但有件事要小心。"最小化" 这个要求强——可能不存在 最小化曲线（比如下下章 会看到不完备曲面的反例）。变分法的标准做法是先弱化要求: 找临界点——\(L\) 的"导数" 为零的曲线。临界点不一定是最小, 但所有 最小都是临界点。

定义（测地线, 弧长版本）。曲线 \(\gamma : [0, 1] \to S\) 是测地线, 若它是弧长泛函 \(L\) 的临界点——即对任何"变分"（保持端点不动的微小扰动）, \(L\) 的一阶变化为零。

具体: 取曲线族 \(\gamma_s(t)\), \(s \in (-\varepsilon, \varepsilon)\), \(\gamma_0 = \gamma\), \(\gamma_s(0) = p, \gamma_s(1) = q\)（端点固定）。\(\gamma\) 是测地线 ⟺

\[ \left. \frac{d}{ds} L[\gamma_s] \right|_{s = 0} = 0 \quad \text{对一切这样的变分}. \]

这条定义抓住了"最短路径" 的本质。它纯内蕴——只用 \(I\) 不用 \(\mathbb{R}^3\) 嵌入。它把"测地线" 化为一个变分问题, 我们可以用变分法标准工具（Euler-Lagrange）求解, 13.4 节做。

但它有缺点: 抽象, 不直接给方程; 而且"临界点" 不一定是最短——它可能是极大、鞍点、或局部（不是全局）最短。我们需要一条更几何 的定义来补足。这就是 13.3 节。

13.3 测地线定义二: 没法向加速度

换一个思路。"直线" 在平面上的另一个等价刻画是没加速度——一个粒子沿直线匀速 运动, 加速度恒为零（牛顿第一定律的几何版）。

把这条搬到曲面上有个问题。曲面上沿曲线 \(\gamma(t)\) 走的粒子, 它的加速度 \(\gamma''(t)\) 是 \(\mathbb{R}^3\) 里的向量——可能不切于 \(S\)。比如赤道上跑的粒子（球面上的大圆）, 它的加速度指向球心——不切于球面, 而是沿球面法向。

这种"法向加速度" 是粒子被迫留在曲面上 必需的——是球面"把粒子拉向自己" 的力。它不 是粒子"主动转弯" 的力, 是外蕴 限制。

所以"曲面上的直线" 应该是主动加速度为零 的曲线——即切向 加速度为零、只剩法向 加速度（被曲面强迫的）。这就是几何意义上的"不偏"——粒子没转弯, 它沿"当前方向" 走得最远（在曲面允许的范围内）。

定义（测地线, 加速度版本）。曲线 \(\gamma : I \to S\)（按弧长参数化, \(|\gamma'| = 1\)）是测地线, 若它的加速度 \(\gamma''(t)\) 处处沿曲面法向——即

\[ \boxed{\ \gamma''(t) \perp T_{\gamma(t)} S \quad \text{对所有 } t.\ } \]

等价地, 用协变导数（ch11 概念）的语言: 测地线满足

\[ \frac{D \gamma'}{dt} = 0, \]

其中 \(D / dt\) 是切平面方向的导数（即 \(\gamma''\) 投影到 \(T S\) 的部分）。

用这条定义验证球面大圆: 大圆上跑的粒子加速度指向球心——而球心方向就是球面法向。所以加速度纯法向, 切向部分为 0。大圆是测地线, 验证。

用这条定义看"纬度圈"（小圆）: 沿纬度跑的粒子, 加速度也指向纬度圈的圆心——但纬度圈的圆心不在球心, 而是在球的对称轴 上。所以加速度方向不 是球面法向, 它有切向分量（指向赤道 的方向）。所以纬度圈不是 测地线。

这条定义的好处: 直接、几何。给一条曲线, 算 \(\gamma''\), 看它有没有切向部分——立即知道是不是测地线。

缺点: 它依赖 \(\mathbb{R}^3\) 嵌入（用 \(\gamma''\) 的向量含义）。我们说"测地线是内蕴的" 但这条定义看起来外蕴。修法: 用协变导数 \(D/dt\) 写, 它纯内蕴（只用 \(I\) 与 Christoffel 符号定义）。13.5 节给方程。

13.4 两条定义等价

关键定理: 13.2 与 13.3 的两条定义等价。

定理。曲线 \(\gamma\) 是 \(L[\gamma]\) 的临界点（13.2）⟺ 它满足 \(D \gamma' / dt = 0\)（13.3, 在弧长参数化下）。

证明思路（标准变分法）。假设 \(\gamma\) 已弧长参数化, 即 \(|\gamma'| = 1\)。取变分 \(\gamma_s(t)\), 写

\[ \frac{\partial \gamma_s}{\partial s}\bigg|_{s = 0} =: V(t), \]

这是变分向量场——告诉你怎么 扰动曲线。端点固定意味着 \(V(0) = V(1) = 0\)。

用第一基本形式写 \(L[\gamma_s] = \int_0^1 \sqrt{I(\partial_t \gamma_s, \partial_t \gamma_s)} \, dt\)。求导（标准链式 + 分部积分）:

\[ \frac{d}{ds} L[\gamma_s] \bigg|_{s = 0} = \int_0^1 \frac{I(\partial_t \gamma_0, \partial_s \partial_t \gamma_0)}{|\partial_t \gamma_0|} dt = \int_0^1 I(\gamma', V') dt \]

（用 \(|\gamma'| = 1\)）。分部积分（边界项 \(I(\gamma', V) \big|_0^1 = 0\) 因 \(V\) 在端点为零）:

\[ = -\int_0^1 I\left(\frac{D \gamma'}{dt}, V\right) dt. \]

这条对一切允许 \(V\) 等于零, 当且仅当 \(D \gamma' / dt \equiv 0\)（变分法基本引理: 若 \(\int f \cdot V = 0\) 对一切 \(V\), 则 \(f = 0\)）。

证毕。

注意一件事: 上面证明里我们做了"分部积分"——这一步需要 \(\gamma\) 二阶可微。我们整章默认 \(\gamma \in C^2\)。要严格化需更细的正则性讨论, 标准微分几何课程都会处理。

这条等价定理告诉我们: "局部最短" 与 "没法向加速度" 是同一件事。这件事的物理直觉很自然——一个粒子如果沿最短路径走, 它没"动机" 转弯（任何转弯都让路径变长）, 所以加速度为零。反过来, 加速度为零的粒子沿当前方向 走, 这种"不浪费" 的运动自然给最短路径（局部）。

"局部" 这个词重要——测地线是局部 最短, 不一定全局。例: 球面上从北极到赤道某点, 有两条 大圆弧——短的（顺着走）和长的（绕过南极反向走）。两条都是测地线, 但只有短的那条是全局最短。

13.5 测地线方程

把 \(D \gamma' / dt = 0\) 写成具体的 ODE。设曲面有局部参数化 \(\boldsymbol{x}(u^1, u^2)\), 第一基本形式系数 \(g_{ij} = I(\partial_i, \partial_j)\)（\(\partial_i = \partial \boldsymbol{x} / \partial u^i\)）。ch11 定义了 Christoffel 符号

\[ \Gamma^k_{ij} = \frac{1}{2} g^{kl} \left( \frac{\partial g_{il}}{\partial u^j} + \frac{\partial g_{jl}}{\partial u^i} - \frac{\partial g_{ij}}{\partial u^l} \right), \]

其中 \(g^{kl}\) 是 \(g_{ij}\) 的逆矩阵（隐含对 \(l\) 求和, Einstein 约定）。

\(\Gamma^k_{ij}\) 是纯内蕴——只依赖 \(g_{ij}\) 与其偏导。它告诉你"切向量沿坐标方向移动时, 它的分量变化" 的修正项。

测地线方程: 设 \(\gamma(t) = \boldsymbol{x}(u^1(t), u^2(t))\), 则 \(D \gamma' / dt = 0\) 化为

\[ \boxed{\ \ddot{u}^k + \Gamma^k_{ij}(\boldsymbol{u}) \, \dot{u}^i \dot{u}^j = 0, \quad k = 1, 2.\ } \]

这是测地线方程——一组二阶 非线性 ODE, \(\Gamma^k_{ij}\) 作为系数（依赖位置 \(u\)）, 二次项 \(\dot{u}^i \dot{u}^j\) 反映"测地线的弯曲" 由"当前速度方向" 决定。

具体推导（草图）。\(\gamma(t) = \boldsymbol{x}(\boldsymbol{u}(t))\) 求一阶: \(\gamma'(t) = \partial_i \boldsymbol{x} \cdot \dot{u}^i\)。求二阶:

\[ \gamma'' = \partial_i \boldsymbol{x} \cdot \ddot{u}^i + \partial_i \partial_j \boldsymbol{x} \cdot \dot{u}^i \dot{u}^j. \]

关键步: 把 \(\partial_i \partial_j \boldsymbol{x}\) 分解为切平面分量与法向分量。Gauss 公式给

\[ \partial_i \partial_j \boldsymbol{x} = \Gamma^k_{ij} \partial_k \boldsymbol{x} + L_{ij} \boldsymbol{n}, \]

其中 \(L_{ij}\) 是第二基本形式系数, \(\boldsymbol{n}\) 是单位法向。所以

\[ \gamma'' = \underbrace{(\ddot{u}^k + \Gamma^k_{ij} \dot{u}^i \dot{u}^j) \partial_k \boldsymbol{x}}_{\text{切向}} + \underbrace{L_{ij} \dot{u}^i \dot{u}^j \boldsymbol{n}}_{\text{法向}}. \]

测地线条件 \(\gamma''\) 切向分量 = 0 立即给出 \(\ddot{u}^k + \Gamma^k_{ij} \dot{u}^i \dot{u}^j = 0\)。

这条方程的形式 重要——它出现在物理（爱因斯坦广义相对论ch23 里"自由下落粒子" 沿时空测地线运动）、机器人（机械臂"自然" 运动是其位形空间的测地线）、计算机图形（曲面上"笔直" 路径绘制）等。

13.6 几个例子

具体算几个曲面的测地线, 把抽象方程变成可见曲线。

例 1: 平面 \(\mathbb{R}^2\)。用直角坐标 \((u^1, u^2) = (x, y)\), \(I = dx^2 + dy^2\), 即 \(g_{ij} = \delta_{ij}\) 是常数矩阵。Christoffel 符号涉及 \(g_{ij}\) 偏导数, 全部为零。测地线方程

\[ \ddot{x} = 0, \quad \ddot{y} = 0 \]

解 \(x(t) = x_0 + v_1 t, y(t) = y_0 + v_2 t\)——直线。预期。

例 2: 球面 \(S^2\)（半径 1）。用经纬度 \((\theta, \varphi)\)（\(\theta\) 是余纬, \(\varphi\) 是经度）, \(\boldsymbol{x}(\theta, \varphi) = (\sin\theta \cos\varphi, \sin\theta \sin\varphi, \cos\theta)\)。算第一基本形式

\[ I = d\theta^2 + \sin^2\theta \, d\varphi^2, \]

即 \(g_{11} = 1, g_{22} = \sin^2 \theta, g_{12} = 0\)。Christoffel 符号:

\[ \Gamma^1_{22} = -\sin\theta \cos\theta, \quad \Gamma^2_{12} = \Gamma^2_{21} = \cot\theta, \]

其余为零（细节省略, 是直接求 \(g_{ij}\) 偏导）。测地线方程

\[ \ddot{\theta} - \sin\theta\cos\theta \, \dot{\varphi}^2 = 0, \quad \ddot{\varphi} + 2\cot\theta \, \dot{\theta}\dot{\varphi} = 0. \]

这组 ODE 不容易直接积分——但用对称性 可以猜出解。比如 \(\varphi = \text{const}\)（沿子午圈）、\(\theta = \dot{\theta}_0 t + \theta_0\)——这是子午圈, 是大圆。验证: \(\ddot{\theta} = 0\), \(\dot{\varphi} = 0\), 第一方程 \(0 - \sin\theta\cos\theta \cdot 0 = 0\) ✓; 第二方程 \(0 + 0 = 0\) ✓。

更一般的——任何大圆, 用旋转对称性可证它满足方程。这给出: 球面测地线 = 大圆, 验证。

例 3: 圆柱面 \(C\)（半径 1）。参数化 \(\boldsymbol{x}(u, v) = (\cos u, \sin u, v)\), \(I = du^2 + dv^2\)（!）即平面度量! 圆柱面是可展曲面（ch1 提到, \(K = 0\), 能从平面卷出）, 在 \((u, v)\) 坐标下度量与平面相同。

所以 \(\Gamma^k_{ij} = 0\), 测地线方程 \(\ddot{u} = \ddot{v} = 0\), 解 \(u(t) = u_0 + a t, v(t) = v_0 + b t\)——在 \((u, v)\) 平面里的直线。但 \(u\) 是角度, 周期 \(2\pi\)——所以圆柱面测地线在 3D 中是:

  • \(b = 0\)（纯角向）: 圆周（沿圆柱周向走的圆）。
  • \(a = 0\)（纯轴向）: 直线（沿圆柱轴线走）。
  • 一般情况: 螺旋线（"螺纹" 形状, 一边转一边升）。

这件事可以更直观看: 把圆柱沿一条母线剪开, 摊成平面长方形。圆柱测地线对应剪开后平面里的直线——再把长方形卷回, 直线变螺旋线 / 圆周 / 直线。这种"展开" 视角对所有可展曲面（圆柱、圆锥、切平面构成的曲面）都成立。

例 4: 圆环面 \(T^2\)。圆环面参数化 \(\boldsymbol{x}(u, v) = ((R + r\cos v) \cos u, (R + r\cos v) \sin u, r \sin v)\), \(R\) 大半径, \(r\) 小半径。第一基本形式

\[ I = (R + r \cos v)^2 du^2 + r^2 dv^2. \]

测地线种类多样:

  • 外赤道（\(v = 0\), 大圆周）: 测地线。这是圆环最外圈, 半径 \(R + r\)。
  • 内赤道（\(v = \pi\), 小圆周）: 测地线。这是圆环最内圈, 半径 \(R - r\)。
  • 子午圈（\(u = \text{const}\), 沿"切片" 的小圆）: 测地线。半径 \(r\)。
  • 螺旋型测地线: 一般测地线绕圆环面"缠绕"（既绕主圆又绕切片）。这些可能永不闭合（在 \(T^2\) 上稠密）。

圆环测地线的丰富性反映了它不是 常曲率曲面（外侧 \(K > 0\), 内侧 \(K < 0\), 顶 / 底圈 \(K = 0\)）。混合的曲率让测地线行为多样, 是研究"测地线流"（曲面上的"动力系统"）经典模型。

13.7 测地线纯由 \(I\) 决定

请你看测地线方程

\[ \ddot{u}^k + \Gamma^k_{ij} \dot{u}^i \dot{u}^j = 0 \]

里出现的东西。\(u^k\) 是位置坐标; \(\Gamma^k_{ij}\) 是 Christoffel 符号——只用 \(g_{ij}\)（第一基本形式系数）+ 它们的偏导数。没 出现 \(L_{ij}\)（第二基本形式）, 没 出现法向量 \(\boldsymbol{n}\), 没 出现 \(\mathbb{R}^3\) 嵌入。

这意味着: 测地线纯内蕴——两个等距的曲面（即 \(I\) 一样的曲面）有完全相同的测地线（在等距对应下）。

这件事链回第 11 章 Theorema Egregium 主题——内蕴几何独立于嵌入。Gauss 曲率内蕴, 测地线也内蕴。这两件事从同一个"第一基本形式决定一切" 母题里长出。

具体一个例子让你看见这件事的力量: 圆柱面 \(C\) 与平面 \(\mathbb{R}^2\) 的一片矩形等距（\(C\) 是 \(\mathbb{R}^2\) "卷起来")。所以 \(\mathbb{R}^2\) 里直线对应 \(C\) 上的测地线——这正是 13.6 例 3 看到的"剪开圆柱看见直线"。这个对应不是"巧合"——是等距保测地线 的具体化身。

反过来, 球面 \(S^2\) 与 \(\mathbb{R}^2\) 不 等距（Gauss 曲率不同）。所以 \(\mathbb{R}^2\) 里直线没法 通过等距对应到球面测地线——这就是为什么"世界地图" 里"大圆航线" 看起来不是直线: 球面与平面的"近似嵌入"（Mercator 投影）不 是等距, 它扭曲了"直线" 这个概念。

13.8 局部存在唯一性

测地线方程是二阶 ODE。ODE 标准定理（Picard-Lindelöf）给我们:

定理（测地线局部存在唯一性）。对任意 \(p \in S, v \in T_p S\), 存在唯一的测地线 \(\gamma : (-\varepsilon, \varepsilon) \to S\) 满足

\[ \gamma(0) = p, \quad \gamma'(0) = v. \]

\(\varepsilon\) 是某正数, 依赖 \(p, v\)。

证明思路。测地线方程 \(\ddot{u}^k = -\Gamma^k_{ij} \dot{u}^i \dot{u}^j\) 是显式 二阶 ODE。化为一阶系统（引入辅助变量 \(w^k = \dot{u}^k\)）:

\[ \dot{u}^k = w^k, \quad \dot{w}^k = -\Gamma^k_{ij}(\boldsymbol{u}) w^i w^j. \]

右边是 \((u, w)\) 的光滑 函数（\(\Gamma^k_{ij}\) 光滑, \(w^i w^j\) 多项式）。Picard 定理给出: 给初值 \((u_0, w_0)\), 存在唯一光滑解, 至少在某小时间区间。\(\square\)

"初值 \(p, v\)" 这两条信息——位置 + 速度——唯一确定测地线。这件事的几何直觉: 在曲面某点出发, 选定方向 和速度, 然后"沿这个方向匀速走, 不转弯"——这给你唯一的测地线。

"速度" 这件事可换为"方向": 如果你只关心"沿哪个方向走", 不关心"走多快", 用单位向量 \(v / |v|\) 代替 \(v\), 同时调整时间参数 \(t \to |v| t\)。改速度只是重参数化 测地线, 不改它的像（轨迹）。所以对每个方向（\(T_p S\) 中单位向量）, 有唯一一条 测地线（作为像）。

这条"位置 + 方向 → 唯一测地线" 是下一章 指数映射 的基础。指数映射 \(\exp_p : T_p S \to S\) 把切向量 \(v\) 送到"沿 \(v\) 方向走单位时间" 终点——它把切平面（线性结构）和曲面（弯曲结构）连接起来, 是连接抽象的微分几何与具体曲面的关键工具。

"局部" 这个词在定理里很关键——\(\varepsilon\) 可能很小, 测地线也许不能延伸到所有时间。一个测地线能 延伸多久, 是完备性 问题, ch15 处理。我们会看到"测地完备" 是个深刻的几何条件, 等价于一些其它看上去 完全无关的条件（Hopf-Rinow 定理）。

下一章我们把"沿方向走测地线" 这件事系统化, 引入指数映射 与测地坐标——这是把切平面 上的线性结构"卷" 到曲面上, 让我们能用线性代数工具研究曲面附近的几何。Gauss 引理和"曲率公式 \(K = -\partial_r^2 \sqrt G / \sqrt G\)" 都从那里出来。
