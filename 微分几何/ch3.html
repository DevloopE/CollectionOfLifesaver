第 3 章 平面曲线：曲率与切向量

第 1 章给了曲率的直觉: 大圆"更直", 小圆"更弯"; \(\kappa = 1/r\) 对圆。本章把这种直觉形式化——给任何（光滑、正则）平面曲线每点一个曲率值。

关键工具来自上一章末尾埋下的那条事实: 在弧长参数化下 \(|\gamma'(s)| = 1\), 把这条等式两边求导得 \(\gamma''(s) \perp \gamma'(s)\)。所以在弧长参数化下, \(\gamma''(s)\) 是垂直于切方向 的向量——它度量切方向转弯 的速率。这就是曲率。

本章结构: (1) 推 \(\gamma'' \perp \gamma'\), (2) 定义平面曲率 \(\kappa = |\gamma''|\) 验证圆 \(\kappa = 1/r\), (3) 一般参数化下的可计算公式, (4) 带符号曲率（左转 / 右转）, (5) 密切圆 + 抛物线 / 椭圆 / 心形线的曲率算, (6) 复原定理（给 \(\kappa(s)\) 复原 \(\gamma\)）, (7) 几何分类（\(\kappa = 0\) 直线, \(\kappa\) 常数 = 圆）。

3.1 弧长参数化下的关键正交关系

设 \(\gamma(s): I \to \mathbb{R}^2\) 是弧长参数化 的平面曲线, 即 \(|\gamma'(s)| = 1\) 处处。这等价于

\[ \gamma'(s) \cdot \gamma'(s) = 1. \]

两边对 \(s\) 求导（用乘积法则）:

\[ \gamma''(s) \cdot \gamma'(s) + \gamma'(s) \cdot \gamma''(s) = 0, \]

即 \(2 \gamma''(s) \cdot \gamma'(s) = 0\)。所以

\[ \boxed{\ \gamma''(s) \perp \gamma'(s).\ } \]

这条正交关系 在弧长参数化下恒成立。它的几何含义: 切向量 \(\gamma'(s)\) 是单位向量, 沿曲线移动时只能转方向, 不能变长度; 它的导数 \(\gamma''(s)\) 度量转方向的速率 + 方向, 必然垂直于 \(\gamma'(s)\)。

用直观语言: \(\gamma'(s)\) 是船头朝向, \(\gamma''(s)\) 是船头转动的角速度向量——它的方向 告诉你船在向哪边转, 它的长度 告诉你转得多快。后者就是曲率。

3.2 平面曲线曲率的定义

定义（弧长参数化下的平面曲率）. 设 \(\gamma(s): I \to \mathbb{R}^2\) 是 \(C^2\) 弧长参数化曲线。在 \(s \in I\) 处的曲率 是

\[ \boxed{\ \kappa(s) := |\gamma''(s)|.\ } \]

它是非负实数, 度量曲线在 \(s\) 处转弯有多急。\(\kappa = 0\) 表示曲线在该点不转（局部直）; \(\kappa\) 大表示转得急。

验证圆 \(\kappa = 1/r\). 半径 \(r\) 圆的弧长参数化是 \(\gamma(s) = (r \cos(s/r), r \sin(s/r))\)。求导:

\[ \gamma'(s) = (-\sin(s/r), \cos(s/r)), \quad |\gamma'(s)| = 1. \ \checkmark \]

再求导:

\[ \gamma''(s) = \big(-\frac{1}{r}\cos(s/r), -\frac{1}{r}\sin(s/r)\big). \]

长度:

\[ |\gamma''(s)| = \sqrt{\frac{1}{r^2}\cos^2(s/r) + \frac{1}{r^2}\sin^2(s/r)} = \frac{1}{r}. \]

所以 \(\kappa = 1/r\) 处处。✓

这一步是对 ch1 那条"圆的曲率 \(1/r\)" 朴素定义的正式验证: 我们的形式定义 \(\kappa = |\gamma''(s)|\) 在圆上吐回了那个朴素值。证明从 ch2 的"弧长参数化让 \(|\gamma'| = 1\)" + ch3 的正交关系自然长出。

顺手还有两条:

  • 直线 \(\gamma(s) = p + s v\)（\(|v| = 1\) 弧长参数化）。\(\gamma'(s) = v\), \(\gamma''(s) = 0\), \(\kappa = 0\) 处处。直线"不弯" 数学上对应 \(\kappa \equiv 0\)。
  • 圆: \(\kappa\) 是常数(\(1/r\))。

所以 \(\kappa\) 的两个最简单值都对应熟悉的几何对象: \(\kappa \equiv 0\) 是直线, \(\kappa\) 是非零常数是圆。后面 §3.7 会形式陈述这种"曲率决定形状" 的命题。

3.3 一般参数化下的曲率公式

问题: ch2 强调大多数曲线没有可写出来的弧长参数化（抛物线、椭圆、心形线都没有）。那 \(\kappa = |\gamma''(s)|\) 这条定义对它们怎么算？

答: 把曲率转写 成一般参数化下的表达式。它恒等于 \(|\gamma''(s)|\)（因为 \(\kappa\) 是几何不变量）, 但用一般参数化 \(\gamma(t)\) 与它的导数表达。

命题（平面曲率的一般公式）. 设 \(\gamma(t) = (x(t), y(t))\) 是 \(C^2\) 正则平面曲线。则它在 \(t\) 处的曲率

\[ \boxed{\ \kappa(t) = \frac{|x'(t) y''(t) - y'(t) x''(t)|}{(x'(t)^2 + y'(t)^2)^{3/2}} = \frac{|\gamma'(t) \times \gamma''(t)|}{|\gamma'(t)|^3}.\ } \]

这里 \(\gamma' \times \gamma''\) 在 2D 中视作一个标量（叉乘的 \(z\)-分量）: \(\gamma' \times \gamma'' = x' y'' - y' x''\)。

推导. 设 \(s(t)\) 是弧长函数, \(s'(t) = v(t) := |\gamma'(t)|\)。弧长参数化 \(\tilde \gamma(s) := \gamma(t(s))\)。链式法则:

\[ \tilde \gamma'(s) = \gamma'(t) \cdot \frac{1}{v(t)} = \frac{\gamma'(t)}{v(t)}. \]

这是单位向量。再求导（对 \(s\)）:

\[ \tilde \gamma''(s) = \frac{d}{ds}\left(\frac{\gamma'(t)}{v(t)}\right) = \frac{1}{v(t)} \cdot \frac{d}{dt}\left(\frac{\gamma'(t)}{v(t)}\right) = \frac{1}{v} \cdot \frac{\gamma''(t) v(t) - \gamma'(t) v'(t)}{v(t)^2} = \frac{v \gamma'' - v' \gamma'}{v^3}. \]

这个向量的模平方:

\[ |\tilde \gamma''(s)|^2 = \frac{|v \gamma'' - v' \gamma'|^2}{v^6}. \]

展开分子:

\[ |v \gamma'' - v' \gamma'|^2 = v^2 |\gamma''|^2 - 2 v v' (\gamma' \cdot \gamma'') + (v')^2 |\gamma'|^2. \]

注意 \(v^2 = |\gamma'|^2 = \gamma' \cdot \gamma'\), 求导得 \(2 v v' = 2 \gamma' \cdot \gamma''\), 即 \(v v' = \gamma' \cdot \gamma''\)。代入:

\[ = v^2 |\gamma''|^2 - 2 (\gamma' \cdot \gamma'')^2 / v^2 \cdot v^2 \cdot \ldots \]

等等, 简化得更直接: 用恒等式 \(|a|^2 |b|^2 - (a \cdot b)^2 = |a \times b|^2\)（在 2D 中 \(a \times b\) 是标量, 在 3D 是向量, 模平方都对）:

\[ v^2 |\gamma''|^2 - (\gamma' \cdot \gamma'')^2 = |\gamma'|^2 |\gamma''|^2 - (\gamma' \cdot \gamma'')^2 = |\gamma' \times \gamma''|^2. \]

所以分子 \(= v^2 |\gamma' \times \gamma''|^2 / v^2 \cdot \ldots\) 直接重做这步: 把上面 \(|v \gamma'' - v' \gamma'|^2\) 展开

\[ = v^2 (\gamma'' \cdot \gamma'') - 2 v v' (\gamma' \cdot \gamma'') + (v')^2 (\gamma' \cdot \gamma'). \]

用 \(v v' = \gamma' \cdot \gamma''\):

\[ = v^2 |\gamma''|^2 - 2 (v v')^2 + (v')^2 v^2 = v^2 |\gamma''|^2 - (v v')^2 = v^2 [|\gamma''|^2 v^2 - (v')^2 v^2]/v^2. \]

更干净地: \(|v \gamma'' - v' \gamma'|^2 = v^2 |\gamma''|^2 + (v')^2 |\gamma'|^2 - 2 v v' (\gamma' \cdot \gamma'') = v^2 |\gamma''|^2 + (v v')^2 - 2 (vv')^2 = v^2 |\gamma''|^2 - (vv')^2 = v^2 |\gamma''|^2 - (\gamma' \cdot \gamma'')^2\)。最后用恒等式

\[ v^2 |\gamma''|^2 - (\gamma' \cdot \gamma'')^2 = |\gamma'|^2 |\gamma''|^2 - (\gamma' \cdot \gamma'')^2 = |\gamma' \times \gamma''|^2. \]

所以

\[ |\tilde \gamma''(s)|^2 = \frac{|\gamma' \times \gamma''|^2}{v^6} = \frac{|\gamma' \times \gamma''|^2}{|\gamma'|^6}. \]

开根号:

\[ \kappa(t) = |\tilde \gamma''(s)| = \frac{|\gamma' \times \gamma''|}{|\gamma'|^3}. \]

得证。

注意. 这条公式不 需要先求弧长参数化（这点很重要——大多数曲线弧长参数化算不出）。你只要知道 \(\gamma'(t), \gamma''(t)\), 直接代公式。

3.4 带符号曲率: 左转还是右转

定义 \(\kappa = |\gamma''(s)|\) 给的是非负数。但平面曲线还有转向 信息: 沿曲线走时向左转 还是向右转。这个信息丢了。

带符号版本: 在弧长参数化下令 \(T(s) := \gamma'(s)\)（单位切向量）, 设

\[ N(s) := T(s) \text{ 逆时针旋转 } 90°. \]

具体: 若 \(T = (a, b)\), 则 \(N = (-b, a)\)。这是左侧法向量。\(\{T, N\}\) 是平面上每点一组正交基, 随 \(s\) 移动。

由 \(\gamma''(s) \perp \gamma'(s) = T(s)\), \(\gamma''(s)\) 必沿 \(\pm N(s)\) 方向。定义

\[ \boxed{\ \gamma''(s) = \kappa_s(s) N(s).\ } \]

这里 \(\kappa_s\) 是带符号曲率: 它是 \(\gamma''\) 在 \(N\) 方向上的有符号分量。\(|\kappa_s| = |\gamma''| = \kappa\) 即先前定义的非负曲率。

几何含义. \(\kappa_s > 0\) 表示曲线向左 弯（朝 \(N\) 方向）; \(\kappa_s < 0\) 表示向右 弯。

一般参数化下:

\[ \boxed{\ \kappa_s(t) = \frac{x'(t) y''(t) - y'(t) x''(t)}{(x'(t)^2 + y'(t)^2)^{3/2}}.\ } \]

这是 §3.3 公式去掉绝对值。

例子. 单位圆 \(\gamma(t) = (\cos t, \sin t)\) 逆时针描出。\(x' = -\sin t, y' = \cos t, x'' = -\cos t, y'' = -\sin t\)。\(x' y'' - y' x'' = (-\sin t)(-\sin t) - (\cos t)(-\cos t) = \sin^2 t + \cos^2 t = 1\)。\((x'^2 + y'^2)^{3/2} = 1\)。所以 \(\kappa_s = 1\) 处处, 正号——逆时针圆向左 弯, 符号为正。

反之 \(\gamma(t) = (\cos t, -\sin t)\)（顺时针单位圆）: \(\kappa_s = -1\)——向右弯, 负号。

请你停下来注意: 带符号曲率 \(\kappa_s\) 依赖于定向（哪边算"左"）+ 方向（沿哪边描出曲线）。换方向描 \(\kappa_s \to -\kappa_s\)。但 \(|\kappa_s| = \kappa\) 与定向方向无关——它是几何对象本身的属性。本书后面 \(\kappa\) 默认指非负曲率; 需要符号信息时单独说"带符号曲率"。

3.5 密切圆 + 例子计算

密切圆（osculating circle）的概念: 曲线 \(\gamma\) 在点 \(P = \gamma(t_0)\) 的密切圆是与 \(\gamma\) 在 \(P\) 处到二阶 重合的圆——它和 \(\gamma\) 共享同一个切向量与同一个曲率值。

具体构造: 在 \(P\) 处取单位切 \(T\)、单位法 \(N\)（左法）。圆心在 \(P + (1/\kappa) N\)（沿曲线弯向那一侧距离 \(1/\kappa\)）, 半径 \(R = 1/\kappa\)。这个圆的曲率恰好 \(\kappa\), 和 \(\gamma\) 在 \(P\) 处一致, 且共享切向量。

所以密切圆是曲线在该点的最佳圆形近似。\(R = 1/\kappa\) 叫曲率半径。

这是 ch1 §1.1 那个直观"大圆更直, 小圆更弯" 的最终形式: 曲线在每点贴一个圆, 圆的半径越小说明曲线在该点弯得越急; \(\kappa = 1/R\) 把这件事变成数。

下面跑三个具体计算, 让公式落地。

例 1（抛物线 \(y = x^2\)）. \(\gamma(t) = (t, t^2)\)。\(\gamma'(t) = (1, 2t), \gamma''(t) = (0, 2)\)。叉乘标量 \(x' y'' - y' x'' = 1 \cdot 2 - 2t \cdot 0 = 2\)。\(|\gamma'|^2 = 1 + 4t^2\)。所以

\[ \kappa(t) = \frac{2}{(1 + 4t^2)^{3/2}}. \]

讨论: \(\kappa(0) = 2\)（最大）; \(\kappa(\pm \infty) \to 0\)（远端越来越像直线）。这与图像吻合——抛物线在原点附近最弯, 远端张开几乎像两条直线。

\(t = 0\) 处 \(R = 1/2\), 密切圆半径 \(1/2\)。圆心？切向量 \(\gamma'(0) = (1, 0)\), 单位切 \(T = (1, 0)\), 左法 \(N = (0, 1)\)。圆心 \(P + R N = (0, 0) + (1/2)(0, 1) = (0, 1/2)\)。所以原点处密切圆是中心 \((0, 1/2)\) 半径 \(1/2\) 的圆: \(x^2 + (y - 1/2)^2 = 1/4\)。展开 \(x^2 + y^2 - y + 1/4 = 1/4\), 即 \(y = x^2 + y^2\)。把 \(y = x^2\) 抛物线上的点 \((x, x^2)\) 代入右边得 \(x^2 + x^4 = x^2 (1 + x^2)\), 比左边 \(x^2\) 多了 \(x^4\) 项——四阶差。所以密切圆到三阶 与抛物线吻合（四阶才出现差距）。这就是"密切"（osculating, 拉丁"亲吻" 之意）的具体含义。

例 2（椭圆 \(\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1\)）. \(\gamma(t) = (a \cos t, b \sin t)\)。\(\gamma' = (-a \sin t, b \cos t)\), \(\gamma'' = (-a \cos t, -b \sin t)\)。叉乘标量 \(x' y'' - y' x'' = (-a \sin t)(-b \sin t) - (b \cos t)(-a \cos t) = ab \sin^2 t + ab \cos^2 t = ab\)。模 \(|\gamma'|^2 = a^2 \sin^2 t + b^2 \cos^2 t\)。所以

\[ \kappa(t) = \frac{ab}{(a^2 \sin^2 t + b^2 \cos^2 t)^{3/2}}. \]

讨论: 在 \(t = 0\)（点 \((a, 0)\), 椭圆长轴端点） \(\kappa = ab / b^3 = a/b^2\); 在 \(t = \pi/2\)（点 \((0, b)\), 短轴端点） \(\kappa = ab / a^3 = b/a^2\)。设 \(a > b\), 则 \(a/b^2 > b/a^2\)（前者大后者小）。长轴端点曲率最大, 短轴端点曲率最小——对应几何: 椭圆在长轴端点最尖, 在短轴端点最平。

这正是椭圆的"四个顶点"——曲率局部极值点。下一章 ch5 的 4-顶点定理会说这是下界: 任何简单闭凸平面曲线至少有 4 个顶点, 椭圆恰好达到这个下界。

例 3（心形线 \(r = 1 + \cos \theta\) 极坐标）. 转笛卡尔: \(\gamma(\theta) = ((1 + \cos\theta)\cos\theta, (1 + \cos\theta)\sin\theta)\)。这是常见的"心形" 闭曲线。曲率公式（极坐标 \(r(\theta)\) 形式）

\[ \kappa(\theta) = \frac{r^2 + 2(r')^2 - r r''}{(r^2 + (r')^2)^{3/2}} \]

在这里 \(r = 1 + \cos\theta, r' = -\sin\theta, r'' = -\cos\theta\):

\[ r^2 = 1 + 2\cos\theta + \cos^2\theta, \quad (r')^2 = \sin^2\theta, \quad r r'' = -(1+\cos\theta)\cos\theta = -\cos\theta - \cos^2\theta. \]

分子: \(r^2 + 2(r')^2 - r r'' = 1 + 2\cos\theta + \cos^2\theta + 2\sin^2\theta + \cos\theta + \cos^2\theta = 3 + 3\cos\theta + 2 + 0 = \ldots\) 让我重算: \(\cos^2 + 2\sin^2 = \cos^2 + 2(1-\cos^2) = 2 - \cos^2\)。所以分子 \(= 1 + 2\cos\theta + 2 - \cos^2\theta + \cos\theta + \cos^2\theta = 3 + 3\cos\theta = 3(1 + \cos\theta)\)。分母:

\[ r^2 + (r')^2 = (1+\cos\theta)^2 + \sin^2\theta = 1 + 2\cos\theta + \cos^2\theta + \sin^2\theta = 2 + 2\cos\theta = 2(1+\cos\theta). \]

所以

\[ \kappa(\theta) = \frac{3(1+\cos\theta)}{[2(1+\cos\theta)]^{3/2}} = \frac{3}{2 \sqrt{2(1+\cos\theta)}} = \frac{3}{2 \sqrt{2} \sqrt{1+\cos\theta}}. \]

用 \(1 + \cos\theta = 2\cos^2(\theta/2)\):

\[ \kappa(\theta) = \frac{3}{2\sqrt{2} \cdot \sqrt{2} |\cos(\theta/2)|} = \frac{3}{4 |\cos(\theta/2)|}. \]

讨论: \(\theta = 0\) 处（心形线"圆头" 右端） \(\kappa = 3/4\); \(\theta = \pi\) 处（心形线尖点） \(\kappa \to \infty\)。这条公式精确捕捉了心形线尖点处的"曲率发散"——尖点处曲线无限弯, 弧长参数化在尖点不光滑（这是 \(\gamma'\) 在该点为零、奇异点的征兆——心形线尖点处 \(r = 0\) 同时 \(\gamma' = 0\)）。所以 §2.7 的"正则曲线" 假设在尖点处不满足, 这里的曲率定义本来要小心; 这个例子正好展示弯曲极端的几何后果。

3.6 曲率决定曲线（复原定理）

这一节给古典微分几何的第一座小山: 给曲率函数 \(\kappa(s)\), 复原曲线。

复原定理（平面曲线）. 设 \(\kappa: I \to \mathbb{R}\) 是给定的连续函数（任意符号）。则存在唯一（至刚体变换 + 反射）的弧长参数化平面曲线 \(\gamma: I \to \mathbb{R}^2\) 使其带符号曲率 \(\kappa_s(s) = \kappa(s)\)。

"至刚体变换" 意思是: 你能平移、旋转曲线得到不同的位置 / 朝向, 但形状唯一。

证明（构造性）. 设 \(\theta(s) := \int_0^s \kappa(\sigma) d\sigma\)（积一下曲率得"切向角度"）。定义

\[ T(s) := (\cos \theta(s), \sin \theta(s)), \quad N(s) := (-\sin\theta(s), \cos\theta(s)), \]

\(T\) 是设想的单位切, \(N\) 是它的左法。注意 \(T'(s) = (-\sin\theta \cdot \kappa, \cos\theta \cdot \kappa) = \kappa(s) N(s)\)——满足曲率定义。

设

\[ \gamma(s) := \gamma(0) + \int_0^s T(\sigma) d\sigma = \gamma(0) + \left(\int_0^s \cos\theta(\sigma) d\sigma, \int_0^s \sin\theta(\sigma) d\sigma\right). \]

验证: \(\gamma'(s) = T(s) = (\cos\theta, \sin\theta)\), 模 1, 弧长参数化。\(\gamma''(s) = T'(s) = \kappa(s) N(s)\), 所以带符号曲率 \(\kappa_s(s) = \kappa(s)\)。✓

唯一性（至刚体）: 起点 \(\gamma(0)\) 是平移自由度（2 个自由度）, 起始切方向 \(\theta(0)\) 是旋转自由度（1 个自由度）, 共 3 个自由度——平面刚体变换恰好 3 个参数（2 平移 + 1 旋转）。给定刚体规约（如固定 \(\gamma(0) = 0, \theta(0) = 0\)）, \(\gamma\) 由 \(\kappa\) 完全 确定。

这条定理是后面 ch4 空间曲线复原定理的原型。它是说: 曲率函数是平面曲线（除去刚体变换）的完整不变量。两条平面曲线形状相同当且仅当它们在弧长参数化下的曲率函数相同。

例子.

\(\kappa(s) \equiv 0\): \(\theta(s) = 0\), \(T = (1, 0)\), \(\gamma(s) = (s, 0)\)——直线。✓

\(\kappa(s) \equiv c > 0\) 常数: \(\theta(s) = cs\), \(T = (\cos cs, \sin cs)\), \(\gamma(s) = (\sin(cs)/c, -(\cos(cs) - 1)/c)\)——半径 \(1/c\) 圆。✓

\(\kappa(s) = s\)（线性增长）: \(\theta(s) = s^2/2\), \(\gamma(s) = (\int \cos(\sigma^2/2) d\sigma, \int \sin(\sigma^2/2) d\sigma)\)——这就是Cornu 螺线（Euler 螺线）, 工程上铁路轨道过渡段的形状。它的曲率随弧长线性增长, 所以列车进入弯道时曲率渐变, 离心力不会突然出现。

3.7 几何分类: \(\kappa = 0\) 直线; \(\kappa\) 常数 = 圆

把上面例子凝结:

命题. 设 \(\gamma\) 是 \(C^2\) 正则平面曲线。则:

  1. \(\gamma\) 是直线 \(\Longleftrightarrow\) \(\kappa(s) = 0\) 处处。
  2. \(\gamma\) 是圆（或圆弧） \(\Longleftrightarrow\) \(\kappa(s)\) 是非零常数。

证明（两方向都已包含在 §3.2 + §3.6 里）: \(\Leftarrow\) 用复原定理; \(\Rightarrow\) 用直线 / 圆的弧长参数化算 \(\kappa\)。

这个命题是\(\kappa\) 这一概念的验收: 它把"不弯" 与"弯得均匀" 这两种几何最简单的形状精确 对应到曲率上, 分别是 \(\kappa \equiv 0\) 与 \(\kappa\) = 非零常数。其余情形对应"弯得不均匀"——抛物线、椭圆、心形线、Cornu 螺线等等。

\(\kappa(s)\) 是非常数的曲线形状无穷丰富: 上一节告诉我们, 任意指定 \(\kappa(s)\) 都对应一个具体的（局部）曲线。所以平面曲线形状 与曲率函数 之间是一一对应（至刚体）。曲率是平面曲线的完整 DNA。

3.8 临别提示: 通向 ch4

本章建立了平面曲线的曲率。3D 空间曲线还需要一个新东西——扭率（torsion）。直觉: 平面曲线只能在同一个平面 内弯; 空间曲线可以"偏离 当前的弯曲平面"——这种偏离速率就是扭率 \(\tau\)。

具体例子: 螺旋线 \(\gamma(t) = (\cos t, \sin t, t)\) 既绕 \(z\) 轴弯（曲率）又沿 \(z\) 上升（扭率）。它的曲率与扭率都是非零常数。本书 ch4 会看到一个空间曲线的形状由 \((\kappa, \tau)\) 一对函数完全决定（Frenet-Serret 复原定理）。

请你动手 试一道: 半径 \(R\) 圆的弧长参数化是 \(\gamma(s) = (R\cos(s/R), R\sin(s/R))\)。算出 \(\gamma'\) 与 \(\gamma''\), 验证 \(|\gamma'| = 1, \kappa = |\gamma''| = 1/R\)。再用一般公式 \(\kappa = |x' y'' - y' x''| / (x'^2 + y'^2)^{3/2}\) 用角度 参数化 \(\gamma(t) = (R\cos t, R\sin t)\) 重算一遍, 答案应该一样（独立于参数化）。这个 60 秒手算把"曲率是几何不变量" 这一抽象命题落到具体计算。
