第 5 章 整体曲线：4-顶点定理 + 等周不等式

前面三章是局部: 在曲线的每个点说话——切向量、曲率、扭率、密切圆。本章转换视角——看整条闭合 曲线作为整体 时, 有什么不可避免 的几何事实。

这是 Banner 风格里反复出现的"从点到整体" 的跳跃。局部信息（每点曲率）累积起来, 不是简单线性求和——它们之间有整体约束。两个最经典的例子:

  1. 4-顶点定理（Mukhopadhyaya 1909, Kneser 1912）: 任何简单闭凸平面曲线至少 有 4 个顶点（曲率函数的局部极值点）。不能少于 4——这是个整体 约束, 曲线必须 在曲率上"波动够多次"。
  2. 等周不等式（Steiner / Hurwitz 1902）: 周长 \(L\) 固定的简单闭曲线围出面积 \(A \le L^2/(4\pi)\), 等号 \(\Leftrightarrow\) 圆。圆是周长固定下围最大面积的形状——一个整体最优 的命题。

本章证这两条。技巧上, 4-顶点用极坐标 + 反证 + 对称性; 等周用 Fourier 展开 + Wirtinger 不等式（Hurwitz 1902 的优雅做法）。两个证明都展示了"整体几何" 怎么用分析工具 攻——这是后面 ch16-17 Gauss-Bonnet 的预备热身。

5.1 简单闭曲线 + 凸曲线

先要的一些基本概念。

定义. 一条平面曲线 \(\gamma: [a, b] \to \mathbb{R}^2\) 称为

  • 闭合（closed）: \(\gamma(a) = \gamma(b)\) 且各阶导数也在端点匹配（光滑闭合）。
  • 简单（simple）: 在 \([a, b)\) 上是单射, 即不自交。
  • 简单闭曲线: 同时是简单与闭合的, 即"不自交的闭环"。

简单闭曲线的标准等价描述: 它是 \(\mathbb{R}^2\) 中一个嵌入圆周（拓扑圆周）的像。由Jordan 曲线定理, 它把平面分成有界内部 + 无界外部。"围出" 的面积是有界内部的面积。

凸性. 简单闭曲线 \(\gamma\) 称为凸 的, 如果它围出的内部区域是凸集（即区域中任两点连线段全在区域内）。等价地, 曲线在每点的切线 把平面分成两半, 整条曲线在同一 半平面内（除切点本身外）。

用带符号曲率 的语言: 设 \(\gamma\) 是逆时针 描出的简单闭凸曲线, 那么 \(\kappa_s(s) \ge 0\) 处处。反之: 若 \(\gamma\) 简单闭合且 \(\kappa_s \ge 0\) 处处, 它必凸。所以"凸" \(\Leftrightarrow\) "带符号曲率不变号"（在适当方向描点下）。本书后面以"凸" 一词时默认这个等价。

简单闭凸曲线的例子: 圆、椭圆、心形线（去掉尖点的光滑版本）、卵形线 (\(x^4 + y^4 = 1\)) 等。

5.2 顶点 (vertex) 的定义

定义. 简单闭曲线 \(\gamma\) 的顶点（vertex）是曲率函数 \(\kappa(s)\) 的局部极值点——即 \(\kappa'(s) = 0\) 处 (并不只是临界点, 而是真正的极大或极小)。

"顶点" 这个词来自椭圆——椭圆有四个顶点 \((\pm a, 0), (0, \pm b)\), 恰好对应曲率 \(\kappa(t) = ab/(a^2\sin^2 t + b^2\cos^2 t)^{3/2}\) 的四个局部极值: 在 \(t = 0, \pi\) 处（长轴端点）\(\kappa = a/b^2\) 是极大; 在 \(t = \pi/2, 3\pi/2\) 处（短轴端点）\(\kappa = b/a^2\) 是极小。所以椭圆恰好 4 个顶点。

顶点在几何上对应"密切圆半径变化的零点"——即曲率最弯或最直的位置。沿椭圆走时, 你能感觉到 在长轴端点曲率最大（密切圆最小）, 然后渐缓到短轴端点（密切圆最大）, 又渐弯回来——这种"缓 / 急 / 缓 / 急" 的循环让椭圆有 4 个顶点。

圆呢？圆的曲率是常数, 处处相等。这种情况一般约定所有点都是顶点（或说"退化情形"）。4-顶点定理对这种"处处常数曲率" 的情形需要单独说明: 圆是个退化 的例子, 4-顶点结论显然成立（甚至无穷多顶点）。

5.3 4-顶点定理

定理（Mukhopadhyaya 1909, Kneser 1912 的扩展）. 任何简单闭凸平面 \(C^3\) 曲线至少 有 4 个顶点。

这条定理在 1909 由 Mukhopadhyaya 证, 假设凸性。后来 Kneser (1912)、Bose (1932)、Osserman (1985) 等推广到非凸情形, 证明"任何 简单闭 \(C^2\) 平面曲线至少 4 个顶点"——更广。本书只证凸的版本, 它是最干净的。

朴素观察. \(\kappa(s)\) 是 \([0, L]\) 上的连续函数（\(L\) 是曲线周长）, 在闭区间 上必有最大值 与最小值——所以至少 2 个顶点（一极大 + 一极小）。所以"至少 2 个" 是平凡的。"至少 4 个" 才是本章的内容。

难点: 直接用极值点数论证只能给 2; 需要更深的整体 论证才能从 2 提升到 4。

证明（凸的情形）. 反证。假设 \(\kappa(s)\) 只有2 个 极值点（一个极大 \(M\), 一个极小 \(m\), 且 \(M > m\)）。

设 \(s_1\) 是极小点, \(s_2\) 是极大点（\(s_1 < s_2\), 把曲线 \([0, L]\) 分成两段 \([s_1, s_2]\) 与 \([s_2, s_1 + L]\)）。在 \([s_1, s_2]\) 上 \(\kappa\) 单调递增（从 \(m\) 到 \(M\)）; 在 \([s_2, s_1+L]\) 上单调递减（从 \(M\) 到 \(m\)）。所以 \(\kappa'(s)\) 在 \([s_1, s_2]\) 上 \(\ge 0\), 在 \([s_2, s_1+L]\) 上 \(\le 0\)。也就是说, \(\kappa'(s)\) 在曲线上恰好换一次号。

取过 \(\gamma(s_1)\) 和 \(\gamma(s_2)\) 的直线 \(\ell\)。由凸性, \(\ell\) 把曲线分成两段: 一段在 \(\ell\) 上方, 一段在下方。具体哪段在上方与 \(\kappa'\) 的符号有几何关联（下面会用）。

选 \(\ell\) 的法向量 \(n\)（垂直 \(\ell\), 长度 1, 方向取定）。考虑积分

\[ J := \oint_\gamma \kappa'(s) \, \langle \gamma(s), n \rangle \, ds. \]

这里 \(\langle \gamma(s), n \rangle\) 是点 \(\gamma(s)\) 在 \(n\) 方向上的投影; 这个投影从 \(\ell\) 起算（即取 \(\ell\) 上的点投影为 0）。

分两段算:

段 \([s_1, s_2]\)（\(\kappa' \ge 0\)）, 假设这段在 \(\ell\) 的正 侧（\(\langle \gamma, n \rangle \ge 0\)）。则被积函数 \(\kappa' \cdot \langle \gamma, n \rangle \ge 0\)。

段 \([s_2, s_1+L]\)（\(\kappa' \le 0\)）, 这段在负 侧（\(\langle \gamma, n \rangle \le 0\)）。则被积函数 \((\le 0) \cdot (\le 0) \ge 0\)。

所以 \(J \ge 0\), 等号仅 在 \(\kappa\) 退化（处处常数）时取, 但凸非圆的曲线 \(\kappa\) 不常数, 所以 \(J > 0\)。

另一方面, 用分部积分 算 \(J\):

\[ J = \oint \kappa'(s) \langle \gamma(s), n \rangle ds = \oint \frac{d \kappa}{ds} \langle \gamma, n \rangle ds. \]

分部积分（注意 \(\kappa\) 与 \(\gamma\) 都周期性, 边界项消失）:

\[ J = - \oint \kappa(s) \frac{d}{ds}\langle \gamma(s), n \rangle ds = -\oint \kappa(s) \langle \gamma'(s), n \rangle ds. \]

\(\gamma'(s) = T(s)\) 单位切。\(\langle T(s), n \rangle\) 是切向量在 \(n\) 上的分量。注意 \(T(s) = (\cos\theta(s), \sin\theta(s))\), 设 \(n = (\cos\alpha, \sin\alpha)\), 则 \(\langle T, n\rangle = \cos(\theta - \alpha)\)。

用 ch3 §3.6 的 \(\theta'(s) = \kappa_s(s) = \kappa(s)\)（凸正向描出, 所以正号）。所以 \(\kappa(s) = \theta'(s)\), 代入:

\[ J = -\oint \theta'(s) \cos(\theta - \alpha) ds = -\oint d[\sin(\theta - \alpha)] = -[\sin(\theta - \alpha)]_0^L = 0 \]

因为 \(\theta(L) - \theta(0) = 2\pi\)（凸闭曲线绕一圈, 切方向角度增 \(2\pi\)）, \(\sin(\theta(L) - \alpha) = \sin(\theta(0) + 2\pi - \alpha) = \sin(\theta(0) - \alpha)\)。

所以 \(J = 0\)。

矛盾: 一边 \(J > 0\), 一边 \(J = 0\)。所以"\(\kappa\) 只有 2 个极值点" 不成立。

由 \(\kappa\) 在 \([0, L]\) 上连续 + 周期, 极值点数必为偶数（极大与极小交替出现）。所以至少 2; 我们证了至少 4。

得证。

请你停下来感受这个证明的几何核心: 我们没有去"数" 顶点, 而是构造了一个积分, 用两种不同方式 算它——一种说它正, 一种说它零。这种"同一物两种算法给矛盾" 的反证, 是整体几何里反复出现的技巧。

5.4 4-顶点定理的下界紧致: 椭圆

椭圆 \(\gamma(t) = (a\cos t, b\sin t), a > b > 0\) 的曲率（ch3 §3.5）

\[ \kappa(t) = \frac{ab}{(a^2\sin^2 t + b^2 \cos^2 t)^{3/2}}. \]

令 \(f(t) := a^2 \sin^2 t + b^2 \cos^2 t = b^2 + (a^2-b^2)\sin^2 t\)。\(\kappa\) 极值 \(\Leftrightarrow f\) 极值。\(f'(t) = (a^2-b^2) \cdot 2 \sin t \cos t = (a^2-b^2)\sin 2t\)。零点: \(t = 0, \pi/2, \pi, 3\pi/2\)——恰好 4 个。前两段（\(t = 0, \pi\)）\(f\) 取最小 \(b^2\), 对应 \(\kappa\) 最大 \(a/b^2\)（长轴端点）; 后两段 \(t = \pi/2, 3\pi/2\) \(f\) 最大 \(a^2\), \(\kappa\) 最小 \(b/a^2\)（短轴端点）。

所以椭圆恰好 4 个顶点——4-顶点定理的下界紧致。

这是 4-顶点定理的"验证物": 它告诉我们"至少 4 个" 不能改进到"至少 5 个"——椭圆已经达到下界。任何不 是椭圆的非圆光滑凸闭曲线一般有更多 顶点（如 \(x^4 + y^4 = 1\) 有 8 个, \(x^6 + y^6 = 1\) 有 12 个等）, 但永远不少于 4。

请你动手 想想这件事的对称含义: 椭圆有 \(\mathbb{Z}_2 \times \mathbb{Z}_2\) 对称（关于两个轴对称）, 这种对称恰好 强制 4 个顶点。更高对称（如圆 \(O(2)\) 对称）给无穷顶点; 更低对称（如一般凸曲线无对称）给至少 4 个但通常更多。

5.5 等周不等式: 陈述

这是几何里最古老 的优化命题之一, 古希腊人就在问: 在固定周长下, 哪种平面图形围出最大面积？

古希腊朴素答案: 圆。证明却等了 2000 多年——最早严格证明（用变分法）出自 19 世纪。Hurwitz 1902 给出今天最优雅的Fourier 证明, 这是我们要看的。

定理（等周不等式）. 设 \(\gamma\) 是平面上 \(C^1\) 简单闭曲线, 周长 \(L\), 围出面积 \(A\)。则

\[ \boxed{\ A \le \frac{L^2}{4\pi}.\ } \]

等号成立 \(\Longleftrightarrow\) \(\gamma\) 是圆。

(对圆: 周长 \(L = 2\pi r\), 面积 \(A = \pi r^2 = (2\pi r)^2 / (4\pi) = L^2/(4\pi)\)——等号成立。)

等价形式. 把不等式改写: \(L^2 \ge 4\pi A\), 即"周长平方至少 4π 倍面积"。

无量纲形式: 定义等周比 \(I := L^2 / A\), 不等式说 \(I \ge 4\pi\)。圆 \(I = 4\pi\); 正方形 \(L = 4 a, A = a^2, I = 16\); 正六边形 \(L = 6a, A = (3\sqrt{3}/2) a^2, I = 36/(3\sqrt{3}/2) = 24/\sqrt{3} \approx 13.86\); 圆 \(I = 4\pi \approx 12.57\)——比正六边形还小, 是最小等周比。

5.6 等周不等式的 Hurwitz 证明（Fourier 法）

设 \(\gamma\) 弧长参数化, 周长 \(L\)。坐标 \(\gamma(s) = (x(s), y(s))\), 周期 \(L\):

\[ x(s + L) = x(s), \quad y(s + L) = y(s). \]

归一化: 把 \(s\) 放缩到 \([0, 2\pi]\), 即设 \(s = L t / (2\pi)\), \(t \in [0, 2\pi]\)。这样 \(x(t), y(t)\) 是 \(2\pi\)-周期函数。新的弧长元 \(ds = (L/2\pi) dt\), 弧长参数化条件 \((dx/ds)^2 + (dy/ds)^2 = 1\) 变为

\[ (\dot x)^2 + (\dot y)^2 = (L/2\pi)^2 \]

(其中 \(\dot{} = d/dt\))。积一下:

\[ \int_0^{2\pi} (\dot x^2 + \dot y^2) dt = 2\pi \cdot (L/2\pi)^2 = L^2 / (2\pi). \]

面积公式（Green）: \(A = \frac{1}{2}\oint (x \dot y - y \dot x) dt = \frac{1}{2}\int_0^{2\pi}(x \dot y - y \dot x) dt\)。

把 \(x(t), y(t)\) 用 Fourier 级数展开:

\[ x(t) = \sum_{n \in \mathbb{Z}} a_n e^{int}, \quad y(t) = \sum_{n \in \mathbb{Z}} b_n e^{int}. \]

由 \(x, y\) 实值, \(\overline{a_n} = a_{-n}, \overline{b_n} = b_{-n}\)。

导数: \(\dot x = \sum in \, a_n e^{int}\), \(\dot y = \sum in \, b_n e^{int}\)。Parseval:

\[ \int_0^{2\pi} \dot x^2 dt = 2\pi \sum n^2 |a_n|^2, \quad \int_0^{2\pi} \dot y^2 dt = 2\pi \sum n^2 |b_n|^2. \]

所以

\[ \int (\dot x^2 + \dot y^2) dt = 2\pi \sum n^2 (|a_n|^2 + |b_n|^2) = L^2/(2\pi). \]

即

\[ \sum_n n^2 (|a_n|^2 + |b_n|^2) = \frac{L^2}{(2\pi)^2}. \qquad (\star) \]

面积:

\[ \int (x \dot y - y \dot x) dt = \int \big(\sum_m a_m e^{imt}\big)\big(\sum_n in \, b_n e^{int}\big) dt - \int \big(\sum_m b_m e^{imt}\big)\big(\sum_n in \, a_n e^{int}\big) dt. \]

用正交性 \(\int e^{imt} e^{int} dt = 2\pi \delta_{m, -n}\) （即 \(m + n = 0\) 时 \(2\pi\), 否则 0）:

\[ \int x \dot y dt = 2\pi \sum_n in \, a_{-n} b_n = 2\pi \sum_n in \, \overline{a_n} b_n. \]

\[ \int y \dot x dt = 2\pi \sum_n in \, \overline{b_n} a_n. \]

所以

\[ \int (x \dot y - y \dot x) dt = 2\pi \sum_n in (\overline{a_n} b_n - \overline{b_n} a_n) = 2\pi \sum_n in \cdot 2i \, \text{Im}(\overline{a_n} b_n) = -4\pi \sum n \, \text{Im}(\overline{a_n} b_n). \]

面积

\[ A = \frac{1}{2} \cdot (-4\pi) \sum n \, \text{Im}(\overline{a_n} b_n) = -2\pi \sum_n n \, \text{Im}(\overline{a_n} b_n). \]

(由对称性 \(n\) 与 \(-n\) 项相消组合, 最后 \(A = 2\pi \sum_{n>0} n (\text{虚分量项})\), 这里我们直接处理整体。注意符号约定的方向选择, 下面改用绝对值估计避方向问题.)

取绝对值估计:

\[ |A| \le 2\pi \sum_n |n| \, |\text{Im}(\overline{a_n} b_n)| \le 2\pi \sum_n |n| \cdot |a_n| |b_n|. \]

用 AM-GM \(|a_n| |b_n| \le (|a_n|^2 + |b_n|^2)/2\):

\[ |A| \le \pi \sum_n |n| (|a_n|^2 + |b_n|^2). \]

再用 \(|n| \le n^2\)（对 \(|n| \ge 1\); \(n = 0\) 项贡献为 0 因为有 \(|n|\) 系数）:

\[ |A| \le \pi \sum_n n^2 (|a_n|^2 + |b_n|^2) = \pi \cdot \frac{L^2}{(2\pi)^2} = \frac{L^2}{4\pi}. \]

得证 \(A \le L^2/(4\pi)\)。

等号情形. 上面不等式 \(|n| \le n^2\) 等号当且仅当 \(|n| \in \{0, 1\}\), 即所有 \(|n| \ge 2\) 的 Fourier 系数为零。AM-GM 等号当 \(|a_n| = |b_n|\) 且 \(\overline{a_n} b_n\) 是纯虚数（即 \(b_n = \pm i a_n\)）。

所以等号 \(\Leftrightarrow\) \(x, y\) 只有 \(n = 0, \pm 1\) 项, 且 \(b_1 = -i a_1\)（取右手定向时）。即

\[ x(t) = a_0 + a_1 e^{it} + \overline{a_1} e^{-it}, \quad y(t) = b_0 + b_1 e^{it} + \overline{b_1} e^{-it} = b_0 - i a_1 e^{it} + i \overline{a_1} e^{-it}. \]

设 \(a_1 = (R/2) e^{i\phi}\):

\[ x(t) = a_0 + R \cos(t + \phi), \quad y(t) = b_0 + R \sin(t + \phi). \]

这正是圆: 中心 \((a_0, b_0)\), 半径 \(R\)。等号成立 \(\Leftrightarrow\) \(\gamma\) 是圆。✓

得证。

请你停下来欣赏这个证明: 它把"找最大面积" 的变分 问题翻译成"Fourier 系数估计" 的分析 问题; 然后用Parseval + Cauchy-Schwarz 给一个纯代数 不等式。优美在所有几何信息全压缩到 \(n^2\) vs \(|n|\) 这个单纯不等式上。等号紧致地告诉你"最优形状是圆"——因为只有圆的 Fourier 谱只有 \(\pm 1\) 阶项。

5.7 应用: 物理 + 生物

等周不等式 + 4-顶点定理在真实世界 有大量应用——任何"表面张力 + 体积约束" 的系统都遵循它们。

(1) 肥皂泡. 表面张力极小化 表面积。给定包含的气体体积, 肥皂泡形状是最小表面积 形状——3D 中的等周不等式说: 球。所以肥皂泡是球——等周问题在 3D 的解。本节是平面（2D）的版本; 3D 版本陈述: 体积 \(V\) 的封闭曲面表面积 \(S \ge \sqrt[3]{36\pi V^2}\), 等号 \(\Leftrightarrow\) 球。

(2) 细胞表面. 红细胞、上皮细胞等的形状受表面张力 + 细胞膜弹性能影响。"最小面积下囊住固定体积" 的偏好把细胞推向球形（小细胞或某些干细胞确实很圆）。但红细胞演化出双凹盘形是为优化气体扩散表面 / 体积比——一个反方向 的等周问题（最大化表面 / 体积）。生物形态学常常在这两类等周问题之间博弈。

(3) 水滴. 失重环境下水滴是球; 在重力下被压扁但仍接近球冠。这都是表面张力极小化等周 问题的体现。

(4) 4-顶点 + 翻滚物体. 4-顶点定理保证一个平面凸闭曲线至少 有 4 个曲率极值——4 个"最尖" 与"最钝" 处。一个均匀重 2D 物体在水平面上无摩擦翻滚时, 稳定平衡 位置对应曲率极小（最钝）处, 不稳定平衡 对应曲率极大处。所以 4-顶点定理在某种意义上保证每个均匀凸 2D 物体 至少有 2 个稳定 + 2 个不稳定平衡——共 4 个平衡。3D 类似: 著名的 Gömböc 物体（Várkonyi-Domokos 2007）证明3D 中存在 只有 1 个稳定 + 1 个不稳定平衡的均匀凸物体——这种"2-顶点 3D 类比" 的存在极其反直觉, 因为它告诉我们 4-顶点定理的3D 类比不成立。

(5) 等周谬误（实际世界中的"最优形状" 直觉）. 古希腊故事: 迦太基女王 Dido 被允许"用一张牛皮围出一块土地"。她把牛皮剪成细长条围成圆形——等周不等式的实际应用。如果换成正方形或多边形, 围出的面积都更小。

5.8 临别提示: 通向 ch6

这一章结束了曲线段（ch2-5）。我们建立了:

  • 参数化与弧长（ch2）
  • 平面曲线曲率（ch3）
  • 空间曲线 Frenet-Serret + \(\kappa, \tau\) 复原定理（ch4）
  • 整体闭曲线: 4-顶点 + 等周（ch5）

下一章 ch6 起转向曲面。曲线是 1D, 曲面是 2D——多了一维带来本质上更丰富的几何。曲面在每点不只有一个切方向, 而是一整个切平面; "弯" 不再是一个数, 而是切平面上的对称双线性形式（第二基本形式）。本书核心工作（ch6-12）就是把这套 2D 曲率理论建起来, 直至 ch11 的 Theorema Egregium——本书的第一个高潮。

请你回顾 ch1 那段开篇的"纸卷得了圆筒, 卷不了球面" 的实验。现在你已经有了曲线的曲率工具。曲面的曲率工具（主曲率、Gauss 曲率、平均曲率）是这套思想升一维 的产物。Theorema Egregium 说: Gauss 曲率虽然定义看上去依赖 3D 嵌入, 实际上只依赖曲面内在的距离结构。这条"内蕴 vs 外蕴" 的区分在曲线情形不存在（1D 曲线没有"嵌入信息之外的几何"——Frenet-Serret 完全描述了它的形状）。曲面里这条区分是核心的, 也是本书后面所有内容的根。

请你动手 算一道收尾: 椭圆 \(x^2/4 + y^2 = 1\) (即 \(a = 2, b = 1\)) 的周长 \(L\) 与面积 \(A\)。面积容易: \(A = \pi a b = 2\pi\)。周长是椭圆积分, 数值上 \(L \approx 9.6884\)。验证 \(L^2/(4\pi) \approx 7.469 \cdot 4 / (4\pi) \approx 7.469 \approx \pi a^2 = 4\pi\)? 不对——重算: \(L^2/(4\pi) = 9.6884^2/(4\pi) \approx 93.86/12.57 \approx 7.47\); \(A = 2\pi \approx 6.28\)。所以 \(A \approx 6.28 < 7.47\)——等周不等式严格成立, 椭圆"不如圆有效率"。这种数值验证是把抽象不等式落到具体数。把它当成本章工具的一次小的验收。
