第 8 章 Gauss 映射：法向量场

上一章讲内蕴 几何——第一基本形式只看曲面"自身" 的距离 / 角度 / 面积, 不管曲面在 \(\mathbb{R}^3\) 里怎么放。

这一章和下一章换视角, 看外蕴 几何——曲面在 \(\mathbb{R}^3\) 中"怎么弯"。最关键的工具是法向量: 每个 \(p \in S\) 上有一个垂直于 \(T_p S\) 的单位向量 \(N(p) \in \mathbb{R}^3\)（其实 \(N(p) \in S^2\), 因为 \(|N(p)| = 1\)）. 把这些法向量当成一个映射 \(N: S \to S^2\), 这就是Gauss 映射。

Gauss 1827 的天才洞察: 法向量怎么变（即 \(dN_p\)）就是曲面"怎么弯" 的形式表达。\(dN_p\) 是 \(T_p S \to T_p S\) 的线性算子（下面解释为什么是这个映射, 而不是 \(T_p S \to T_{N(p)} S^2\)）, 它的特征值 \(-\kappa_1, -\kappa_2\)（取负号惯例）就是主曲率, 特征方向就是主方向。这是后面 ch9 第二基本形式 + ch10 Gauss 曲率公式 + ch11 Theorema Egregium 的工具底座。

8.1 单位法向量

从最具体的开始。\(p \in S\), 切空间 \(T_p S \subset \mathbb{R}^3\) 是 2 维子空间, 它的正交补（在 \(\mathbb{R}^3\) 中）是 1 维。所以正交补里有恰好两个 单位向量, 它们互为相反数。任选一个叫 \(N(p)\), 另一个就是 \(-N(p)\).

具体怎么算：用一张参数化 \(X(u, v)\), 切空间基 \((X_u, X_v)\), 法向量是它们的叉积:

\[ N(p) = \frac{X_u(q) \times X_v(q)}{|X_u(q) \times X_v(q)|}, \quad p = X(q). \]

叉积自动垂直于两个因子, 除以模长得单位向量。

例（球面）. \(X(\theta, \varphi) = (r \sin \theta \cos \varphi, r \sin \theta \sin \varphi, r \cos \theta)\). \(X_\theta \times X_\varphi\) 直接算（用 ch7.4 的偏导）, 答案是

\[ X_\theta \times X_\varphi = r^2 \sin \theta \cdot (\sin \theta \cos \varphi, \sin \theta \sin \varphi, \cos \theta) = r \sin \theta \cdot X(\theta, \varphi). \]

归一化, \(N = X / r = (\sin \theta \cos \varphi, \sin \theta \sin \varphi, \cos \theta)\). 几何上, 球面在每点的法向量就是从球心指向那点的径向单位向量。这是常识, 现在有公式。

例（圆柱）. \(X(u, v) = (r \cos u, r \sin u, v)\). \(X_u = (-r \sin u, r \cos u, 0), X_v = (0, 0, 1)\). \(X_u \times X_v = (r \cos u, r \sin u, 0)\), 模长 \(r\). \(N = (\cos u, \sin u, 0)\). 几何上, 圆柱面法向量从轴心指向曲面点的径向方向（投影到 \(xy\) 平面）, \(z\) 分量为零（因为圆柱"不向上向下弯"）.

注意：法向量的方向是约定. 我们选 \(X_u \times X_v\) 而不是 \(X_v \times X_u\)（差一个符号）, 这是参数化下的"正向"。换一张参数化, 法向量可能反向。

这引出一个微妙的整体问题: 一个曲面能不能选出处处一致 的 \(N\), 即一个连续 的 \(N: S \to S^2\)？

答案是: 不一定。这就是可定向性。一个曲面叫可定向 的, 如果存在连续的单位法向量场 \(N: S \to S^2\). 否则叫不可定向。

大部分熟悉的曲面（球、椭球、圆柱、环面、平面）都可定向。最著名的不可定向例子: Möbius 带——一张纸条扭半圈黏起来, 沿曲面走一圈, 法向量从"朝上" 变成"朝下", 没办法处处一致。这种曲面没全局法向量。

本章后面默认曲面可定向, 选定一个 \(N\)。

8.2 Gauss 映射 \(N: S \to S^2\)

把单位法向量看成映射:

定义 8.1. 设 \(S \subset \mathbb{R}^3\) 是可定向正则曲面, 选定一个连续单位法向量场 \(N: S \to S^2\). 这个映射 \(N\) 叫做曲面 \(S\) 的Gauss 映射。

"映射" 这个词是关键的视角转换。以前我们把 \(N(p)\) 看成"p 处那个法向量"——一个 \(\mathbb{R}^3\) 中的向量。现在我们把 \(N(p)\) 看成"单位球面 \(S^2\) 中的一点"——\(N(p) \in S^2\). 同一个对象, 两种解读。第二种解读让我们能问 \(N\) 这个映射的微分 是什么——这就是后面所有曲率论的钥匙。

8.3 三个例子的 Gauss 映射图

把 \(N\) 当映射 \(N: S \to S^2\), 可以问"它的像是什么"。三个例子让你看清楚。

球面 \(S^2_r\). 上面算了 \(N(p) = p / r\). 把球面每点除以 \(r\), 得到单位球面 \(S^2\). 所以 \(N: S^2_r \to S^2\), \(p \mapsto p/r\), 是一个同胚（是个缩放）。

请你注意: 球面的 Gauss 映射本质就是恒等（差一个 \(1/r\) 缩放）. \(N\) 是 1-1 满的, 像是整个 \(S^2\). 直觉上球面"每个方向都被某点的法向量指到", 因为球面"各方向都凸"。

圆柱面. \(N(p) = (\cos u, \sin u, 0)\), 即 \(N\) 的像总落在 \(xy\) 平面 \(z = 0\) 上, 即赤道圆 \(S^1 \subset S^2\). 把整个圆柱面（无穷长一个二维曲面）映到 \(S^2\) 上的一个圆（一维）！这是个剧烈降维。

原因: 圆柱沿轴方向"不弯", 所以法向量在轴方向不变化。圆柱上 \(z\) 不同但 \((x, y)\) 同的两点（同一条母线上）有同一个 法向量。所以整条母线被映到同一点 \((\cos u, \sin u, 0) \in S^2\). 这个"整条母线压成一点" 反映出圆柱"沿轴方向曲率为零"。

平面. 平面 \(z = 0\) 上每点法向量都是 \((0, 0, 1)\). 所以 \(N: \text{平面} \to S^2\) 是常映射, 像是一个 点 \((0, 0, 1) \in S^2\)。整个平面（二维）被压到一点（零维）！

把三例并排:

  • 球面: 二维 → 二维（同胚到 \(S^2\)）.
  • 圆柱: 二维 → 一维（降一维）.
  • 平面: 二维 → 零维（降两维）.

"Gauss 映射的像维数" 反映"曲面在多少独立方向上弯". 这给我们一个朴素直觉的形式版本: 球面"各向都弯"（满秩）, 圆柱"一向弯一向直"（秩 1）, 平面"哪都不弯"（秩 0）。下一节我们形式化这个"弯多少" 的概念——\(dN_p\) 的秩就是它。

8.4 \(dN_p\) 是 \(T_p S \to T_p S\) 的自伴算子

这一节是本章核心。\(N: S \to S^2\) 是光滑映射（光滑性: \(N\) 在参数化下表达为 \((X_u \times X_v)/|X_u \times X_v|\), 这是光滑函数）, 所以有微分

\[ dN_p: T_p S \to T_{N(p)} S^2. \]

具体来说, \(dN_p(v)\) 这样定义: 取 \(\gamma: (-\varepsilon, \varepsilon) \to S\), \(\gamma(0) = p, \gamma'(0) = v\). 复合 \(N \circ \gamma: (-\varepsilon, \varepsilon) \to S^2\) 是 \(S^2\) 上一条曲线, 它在 \(t = 0\) 的速度

\[ dN_p(v) := \frac{d}{dt}\bigg|_{t = 0} N(\gamma(t)) \in T_{N(p)} S^2. \]

所以 \(dN_p(v)\) 是"沿 \(v\) 方向走时, 法向量变化的速度". 这就是"曲面在 \(v\) 方向弯多少" 的形式版本。

关键观察. \(T_{N(p)} S^2\) 是 \(\mathbb{R}^3\) 中的什么？\(S^2\) 在 \(N(p)\) 处的切平面 = 与 \(N(p)\) 垂直的 2 维子空间. 而 \(T_p S\) = 与 \(N(p)\) 垂直的 2 维子空间（因为 \(N(p)\) 是 \(T_p S\) 的法向量）. 它俩是同一个子空间！

所以 \(dN_p\) 自动有定义域 \(T_p S\)、像在 \(T_{N(p)} S^2 = T_p S\) 中, 我们可以把它看成 \(T_p S\) 上的线性算子:

\[ dN_p: T_p S \to T_p S. \]

这是一个微妙但重要的"记号合并"。把目标和源看成同一个空间, 我们就可以问 \(dN_p\) 的特征值、特征向量、自伴性等线性代数问题。

定理 8.1（\(dN_p\) 自伴）. \(dN_p: T_p S \to T_p S\) 是自伴算子（关于 \(I_p\) 内积）, 即对所有 \(v, w \in T_p S\),

\[ \langle dN_p(v), w \rangle = \langle v, dN_p(w) \rangle. \]

证明. 取一张参数化 \(X(u, v)\), \(p = X(q_0)\). 自伴性等价于在基 \((X_u, X_v)\) 下证两件事:

\[ \langle dN(X_u), X_v \rangle = \langle X_u, dN(X_v) \rangle. \tag{*} \]

（自伴性只需在基向量对 上验证, 双线性自动延拓.）

注意 \(dN(X_u) = N_u\)（\(N\) 沿 \(u\) 方向的偏导, 因为 \(X_u\) 是 \(u\) 方向的"基向量化" 速度）. 同理 \(dN(X_v) = N_v\).

所以 (*) 等价于 \(\langle N_u, X_v \rangle = \langle X_u, N_v \rangle\).

关键事实: \(N\) 处处垂直于 \(X_u, X_v\), 即 \(\langle N, X_u \rangle = 0, \langle N, X_v \rangle = 0\). 对 \(\langle N, X_v \rangle = 0\) 关于 \(u\) 求偏导:

\[ \langle N_u, X_v \rangle + \langle N, X_{vu} \rangle = 0. \]

所以 \(\langle N_u, X_v \rangle = -\langle N, X_{vu} \rangle\).

同样, 对 \(\langle N, X_u \rangle = 0\) 关于 \(v\) 求偏导得

\[ \langle N_v, X_u \rangle = -\langle N, X_{uv} \rangle. \]

用混合二阶导对称: \(X_{uv} = X_{vu}\)（\(X\) 是 \(C^\infty\)）. 所以 \(\langle N, X_{uv} \rangle = \langle N, X_{vu} \rangle\). 因此

\[ \langle N_u, X_v \rangle = -\langle N, X_{vu} \rangle = -\langle N, X_{uv} \rangle = \langle N_v, X_u \rangle. \]

这正是 (*). \(\square\)

请你停下来感受这个证明。最关键的一步是混合偏导对称——一个关于 \(C^\infty\) 函数的极朴素事实, 在这里给我们送出了 \(dN_p\) 的自伴性。这种"用平凡的微积分事实推出深刻的几何事实" 是微分几何的常态。

"自伴" 这件事在线性代数里有立刻可用的推论: 自伴算子可正交对角化——存在一组正交基使矩阵对角。这是下一节的事。

8.5 \(dN_p\) 自伴 ⟹ 正交对角化

定理 8.2. 设 \(A: V \to V\) 是有限维内积空间 \(V\) 上的自伴算子. 则存在一组正交 基 \((e_1, \ldots, e_n)\) 由 \(A\) 的特征向量组成, 对应特征值 \(\lambda_1, \ldots, \lambda_n\) 都是实数, 在这组基下 \(A\) 矩阵对角:

\[ [A] = \mathrm{diag}(\lambda_1, \ldots, \lambda_n). \]

这是线性代数的谱定理——不是这本书的内容, 我们假定你熟。具体到我们的情况, \(V = T_p S\) 是 2 维, \(A = -dN_p\)（取负号是惯例, 下一节解释）, 特征值是 \(\kappa_1, \kappa_2\), 特征向量 \(e_1, e_2\) 是正交 的（在 \(I_p\) 下）单位向量.

注意 \(dN_p\) 自伴在 \(T_p S\) 的每个点 都成立, 所以每个 \(p\) 都有自己的 \(\kappa_1(p), \kappa_2(p)\) 和正交主方向。这些数和方向是逐点 的曲面几何信息。

8.6 主曲率（取负号惯例）

定义 8.2. 设 \(p \in S\), \(dN_p: T_p S \to T_p S\) 是 Gauss 映射的微分. 主曲率

\[ \kappa_1(p), \kappa_2(p) := -dN_p \text{ 的特征值}. \]

"取负号" 这件事来自直觉。我们希望"球面 Gauss 曲率 \(\kappa = 1/r\)" 是正 的, 不是负的。算一下球面: 上面 8.1 算 \(N(p) = p/r\), 所以 \(N(\gamma(t)) = \gamma(t) / r\), \(dN(v) = v/r\). \(dN_p\) 是同位映射的 \(1/r\) 倍. 它的特征值都是 \(1/r\)（任何向量都是特征向量）. 取负号, \(-dN_p\) 的特征值都是 \(-1/r\)。这给 \(\kappa = -1/r < 0\)？

不对——我们要 \(\kappa > 0\). 问题在 \(N\) 的方向。我们选 \(N(p) = p/r\), 法向量从球心向外指（球面外 法）. 这选择下, 当向量沿曲面运动, 法向量"跟着外侧" 转, \(dN\) 与 \(I\) 同向, 所以"取负号" 让特征值还是 \(1/r > 0\)？

更仔细: \(dN(v) = (1/r) v\), 所以 \(-dN(v) = -(1/r) v\), 特征值是 \(-1/r\). 这不 给我们想要的 \(+1/r\)。

问题在法向量约定。如果选内 法 \(N = -p/r\)（向球心指）, 则 \(dN(v) = -(1/r) v\), \(-dN(v) = (1/r) v\), 特征值 \(1/r\)。这给我们想要的 \(\kappa_1 = \kappa_2 = 1/r > 0\)。

所以"取负号" 加上"选合适的法向量方向" 一起让 \(\kappa\) 在球面上 \(> 0\)。

正式说: 主曲率的符号 依赖法向量约定。换 \(N \to -N\), \(dN \to -dN\), \(\kappa \to -\kappa\). 但主曲率的乘积 \(K = \kappa_1 \kappa_2\)（Gauss 曲率, ch10）不 依赖法向量方向（两个 \(\kappa\) 都换符号, 乘积不变）. 这是为什么 Gauss 曲率"客观" 而主曲率"有约定成分"。

本书统一约定: 选外 法（"指向凸侧"——球面指外、凸的圆柱外侧指外, 等等）。在这个约定下大部分熟悉曲面的主曲率为正（球面 \(1/r\)、圆柱凸方向 \(1/r\) 等）。但这只是约定; 后面具体公式不区分符号约定时会按上下文重选。

正式定义重述, 取约定 \(N\) 的方向使球面主曲率 \(> 0\):

定义 8.2'. 主曲率 \(\kappa_1, \kappa_2\) 是 \(-dN_p\) 的两个实特征值（按惯例, \(\kappa_1 \geq \kappa_2\)）。

8.7 主方向

主曲率 \(\kappa_1, \kappa_2\) 各自对应一个特征空间。它们分别是 \(T_p S\) 中的主方向 \(e_1, e_2\). 由谱定理它们正交。

主方向的几何意义: 沿 \(e_i\) 方向走, 法向量变化最"极端"——\(\kappa_i\) 是法向量变化率的极值之一（最大或最小）. 后面 ch9 给法向曲率的精确版本: 沿任何方向 \(v\) 的"法向曲率" \(\kappa_n(v) = II(v, v) / I(v, v)\), 它的最大 = \(\kappa_1\), 最小 = \(\kappa_2\), 极值在 \(e_1, e_2\) 方向取到。

主方向的非唯一性 出现在 \(\kappa_1 = \kappa_2\) 的点（叫脐点）. 此时 \(-dN_p\) 是常数倍恒等, 所有方向都是特征向量. 球面的每个点都是脐点（因为 \(\kappa_1 = \kappa_2 = 1/r\)）. 平面也每点脐（\(\kappa_1 = \kappa_2 = 0\)）. 椭球面（非球）只有 4 个脐点（一对长轴端 + 一对短轴端... 实际上椭球的脐点定理是一个有趣的整体结果）.

8.8 几个例子的主曲率

例 1（球面 \(S^2_r\)）. 算过 \(dN(v) = (1/r) v\)（约定外法）, 取负号得 \(-dN(v) = -(1/r) v\). 特征值 \(-1/r\). 但我们要"球面 \(\kappa > 0\)", 所以正确约定是内法 \(N = -p/r\). 此时

\[ \kappa_1 = \kappa_2 = \frac{1}{r}. \]

所有方向都是主方向（脐点）。Gauss 曲率 \(K = \kappa_1 \kappa_2 = 1/r^2\). 平均曲率 \(H = (\kappa_1 + \kappa_2)/2 = 1/r\). ✓ 与 ch1 一致。

例 2（圆柱面 \(x^2 + y^2 = r^2\)）. 法向量 \(N(u, v) = (\cos u, \sin u, 0)\). 算 \(dN\):

  • 沿 \(u\) 方向: \(N_u = (-\sin u, \cos u, 0)\). 同时 \(X_u = (-r \sin u, r \cos u, 0)\). 所以 \(N_u = X_u / r\), 即 \(dN(X_u) = N_u = (1/r) X_u\). 主方向 \(X_u\)（圆周方向）, 主值 \(1/r\), 取负号得 \(-1/r\). 但我们想 \(\kappa > 0\), 这又是法向量约定问题——选合适方向给 \(\kappa = 1/r > 0\).
  • 沿 \(v\) 方向: \(N_v = (0, 0, 0) = 0\). 所以 \(dN(X_v) = 0\). 主方向 \(X_v\)（轴方向）, 主值 \(0\).

所以圆柱面在每点的主曲率是

\[ \kappa_1 = \frac{1}{r}, \quad \kappa_2 = 0. \]

主方向: \(\kappa_1\) 方向是圆周（\(X_u\) 方向）, \(\kappa_2 = 0\) 方向是母线（\(X_v\) 方向）. 几何上完全直觉: 沿圆周走"看见弯" 程度 \(1/r\), 沿轴走"看见弯" 程度 \(0\)（直线）.

Gauss 曲率 \(K = (1/r) \cdot 0 = 0\). 圆柱"有平面的内蕴几何"——这就是 ch1 + ch7 反复说的"纸卷成圆柱" 的形式背景。

平均曲率 \(H = (1/r + 0) / 2 = 1/(2r)\). 不为零. 圆柱不是极小曲面（极小曲面 \(H \equiv 0\), 见 ch19）.

例 3（鞍面 \(z = x^2 - y^2\)）. 这个例子完整算需要 ch9 的第二基本形式, 这里给出结果, 算具体细节留到 ch9.5.

原点 \((0, 0, 0)\) 处:

\[ \kappa_1 = 2 \text{（沿 \(x\) 方向）}, \quad \kappa_2 = -2 \text{（沿 \(y\) 方向）}. \]

沿 \(x\) 方向 \(z = x^2\), 凸（\(\kappa > 0\)）; 沿 \(y\) 方向 \(z = -y^2\), 凹（\(\kappa < 0\)）. 两数异号, Gauss 曲率 \(K = -4 < 0\). 这是 ch1 说的"双曲型" 局部形状的鼻祖。

请你停下来比较三例的"主曲率组合":

  • 球面: \((1/r, 1/r)\), 同号同值. 两方向都凸, "各向同性正弯"。
  • 圆柱: \((1/r, 0)\), 一正一零. 一向弯一向直, "各向异性"。
  • 鞍面: \((+, -)\), 异号. 两方向异向弯, "双曲型"。

这三例正好对应 \(K > 0, K = 0, K < 0\) 三种局部形状（ch1.3 介绍, ch9.8 形式化）。

8.9 临别话

这一章建立了三件事:

  1. Gauss 映射 \(N: S \to S^2\)——把每个曲面点映到它的单位法向量.
  2. \(dN_p\) 是 \(T_p S\) 上的自伴算子（关键证明用混合二阶导对称）.
  3. 主曲率 \(\kappa_1, \kappa_2\) 是 \(-dN_p\) 的特征值, 主方向是它们的特征方向, 在 \(I_p\) 下正交.

球面 / 圆柱 / 鞍面三例的主曲率分别 \((1/r, 1/r), (1/r, 0), (+, -)\), 完美对应 ch1 的"三类"。

但 \(dN_p\) 是个抽象算子, 用它直接算曲率有点别扭。下一章我们把 \(dN_p\) 通过双线性形式表达——这就是第二基本形式 \(II_p(v, w) = -\langle dN_p(v), w \rangle\). 用它和第一基本形式 \(I_p\) 一起, 主曲率会变成广义特征值问题 \(\det(II - \kappa I) = 0\), 计算上特别可操作。这一对 \((I, II)\) 之后会成为曲面几何的"一对眼睛"——\(I\) 看内蕴, \(II\) 看外蕴。
