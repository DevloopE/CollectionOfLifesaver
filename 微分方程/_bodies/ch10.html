第 10 章 用拉普拉斯解 ODE

第 9 章我们费了不少劲学拉普拉斯变换。本章开始用它。先把要点重述一遍:拉普拉斯变换把"时间域里的微分方程"翻译成"\(s\)-域里的代数方程",代数方程容易解,解完之后再翻译回时间域,就得到了原方程的解。整个流程像一个加工厂:输入 \(t\)-域,输出 \(t\)-域,中间经过 \(s\)-域。本章把这条流水线一步步走通。

我们要回答的具体问题是:

  • 10.1 给定一个带初值的常微分方程,完整流程是什么?
  • 10.2 反向操作怎么做?——给定 \(Y(s)\),怎么"还原"出 \(y(t)\)?这一节是关键技巧:部分分式展开。
  • 10.3 完整推演几个例题,涵盖单根、重根、复根三种情形。
  • 10.4 与第 5-8 章的方法相比,什么时候拉普拉斯更省事?(剧透:几乎所有带初值的问题。)

10.1 解决 IVP 的"流程"

拉普拉斯方法的整套流程可以概括为四步:

\[ \boxed{\; \text{(1) 两边取 } \mathcal{L} \;\to\; \text{(2) 用微分定理把 } y' , y'' \text{ 替换成 } sY, s^2 Y \text{ 等并代入初值} \;\to\; \text{(3) 解出 } Y(s) \;\to\; \text{(4) 反变换 } y(t) = \mathcal{L}^{-1}\{Y(s)\}. \;} \]

看起来抽象,直接用一个例子走一遍最清楚。考虑

\[ y' + 2y = e^{-t}, \quad y(0) = 1. \]

这是一阶线性 ODE,我们在第 3 章学过用积分因子解。现在用拉普拉斯重做。

第一步,两边取 \(\mathcal{L}\)。 把 \(\mathcal{L}\) 当作一个算子,作用在等号左右两边:

\[ \mathcal{L}\{y' + 2y\} = \mathcal{L}\{e^{-t}\}. \]

左边用线性性拆开:

\[ \mathcal{L}\{y'\} + 2 \mathcal{L}\{y\} = \mathcal{L}\{e^{-t}\}. \]

右边查表:\(\mathcal{L}\{e^{-t}\} = 1/(s + 1)\)。

第二步,用微分定理。 设 \(Y(s) = \mathcal{L}\{y(t)\}\),则 \(\mathcal{L}\{y'\} = s Y(s) - y(0) = s Y(s) - 1\)(代入了初值 \(y(0) = 1\))。代入:

\[ (s Y(s) - 1) + 2 Y(s) = \frac{1}{s + 1}. \]

注意:这是一个代数方程,变量是 \(Y(s)\)。所有"微分"都已经消失了——这就是变换的威力。还有,初值 \(y(0) = 1\) 已经"嵌"进方程里了——你不需要先求通解再代初值,初值是免费送的。

第三步,解出 \(Y(s)\)。 把含 \(Y\) 的项移到一边:

\[ (s + 2) Y(s) = \frac{1}{s + 1} + 1 = \frac{1 + (s + 1)}{s + 1} = \frac{s + 2}{s + 1}. \] \[ Y(s) = \frac{s + 2}{(s + 1)(s + 2)} = \frac{1}{s + 1}. \]

(分子分母约去 \(s + 2\)。)所以 \(Y(s) = 1/(s + 1)\)。

第四步,反变换。 查表,\(1/(s + 1) = \mathcal{L}\{e^{-t}\}\),所以

\[ y(t) = e^{-t}. \]

检验:\(y(0) = e^0 = 1\) ✓;\(y' = -e^{-t}\),\(y' + 2y = -e^{-t} + 2 e^{-t} = e^{-t}\) ✓。完美。

整个过程没有出现任何积分(除了开头变换 \(\mathcal{L}\{e^{-t}}\) 那一步,但那是查表)。比第 3 章的积分因子法,代价低不少。

这套四步流程是死的。难度不在流程本身,而在第四步——反变换。"知道 \(Y(s)\),怎么找到 \(y(t)\)?"上面这道题里 \(Y(s) = 1/(s + 1)\) 一眼就在变换表里。但实际问题里你解出来的 \(Y(s)\) 通常是个分式,而且分母不会刚好长成表里的样子。这时候需要一个机械化的拆解技术,也就是下一节的部分分式展开。

10.2 反拉普拉斯变换 \(\mathcal{L}^{-1}\)

反变换 \(\mathcal{L}^{-1}\) 满足:如果 \(F(s) = \mathcal{L}\{f(t)\}\),则 \(f(t) = \mathcal{L}^{-1}\{F(s)\}\)。它是一个线性算子(由变换的线性性自动继承)。

原则上反变换有一个复积分公式(Bromwich 积分),但本课不需要——我们用查表 + 部分分式就能搞定教科书里 99% 的题。

10.2.1 查表反查

第 9 章那张表如果横着看,就是反变换表。比如:

\(F(s)\)\(f(t) = \mathcal{L}^{-1}\{F(s)\}\)
\(\dfrac{1}{s}\)\(1\)
\(\dfrac{1}{s^{n+1}}\)\(\dfrac{t^n}{n!}\)
\(\dfrac{1}{s - a}\)\(e^{at}\)
\(\dfrac{s}{s^2 + \omega^2}\)\(\cos(\omega t)\)
\(\dfrac{\omega}{s^2 + \omega^2}\)\(\sin(\omega t)\)
\(\dfrac{s - a}{(s - a)^2 + \omega^2}\)\(e^{at}\cos(\omega t)\)
\(\dfrac{\omega}{(s - a)^2 + \omega^2}\)\(e^{at}\sin(\omega t)\)
\(\dfrac{1}{(s - a)^{n+1}}\)\(\dfrac{t^n}{n!} e^{at}\)

实际操作时,你拿到一个 \(F(s)\),先看分母长什么样——这决定了答案的"骨架":

  • 分母是 \((s - a)^k\)? 答案含 \(t^j e^{at}\) 形式的项。
  • 分母是 \(s^2 + \omega^2\)? 答案含 \(\cos(\omega t), \sin(\omega t)\)。
  • 分母是 \((s - a)^2 + \omega^2\)? 答案含 \(e^{at}\cos(\omega t), e^{at}\sin(\omega t)\)。
  • 分母是 \(s^2 - a^2\)? 答案含 \(\cosh(at), \sinh(at)\)(或者等价地,\(e^{at}, e^{-at}\))。

但这是模式识别,具体的数还得拆开。这就是下一段。

10.2.2 部分分式展开

这是反变换的万能武器。一句话:把一个有理分式 \(P(s)/Q(s)\)(分子次数低于分母)拆成几个"简单分式"之和,每个简单分式都在变换表里。

部分分式的"简单分式"种类有限,与 \(Q(s)\) 的因式分解有关:

  1. \(Q(s)\) 含因子 \((s - a)\)(单根):对应一个 \(\dfrac{A}{s - a}\) 形式的项。
  2. \(Q(s)\) 含因子 \((s - a)^k\)(\(k\) 重根):对应 \(\dfrac{A_1}{s - a} + \dfrac{A_2}{(s - a)^2} + \cdots + \dfrac{A_k}{(s - a)^k}\) 这一串项。
  3. \(Q(s)\) 含因子 \(s^2 + bs + c\)(不可约二次,即判别式 \(b^2 - 4c < 0\)):对应一个 \(\dfrac{Bs + C}{s^2 + bs + c}\) 形式的项。
  4. 这种二次因子的 \(k\) 重:类似第 2 条,每一阶都给出一项。

列好框架以后,通分,比较系数,解出未知数 \(A, A_1, \ldots, B, C, \ldots\),拆完。这个过程纯粹是代数操练,不涉及微积分。

例 10.2.1(单根,实数)。求 \(\mathcal{L}^{-1}\{(2s + 3)/((s - 1)(s + 2))\}\)。

分母是 \((s - 1)(s + 2)\),两个不同的实单根。设

\[ \frac{2s + 3}{(s - 1)(s + 2)} = \frac{A}{s - 1} + \frac{B}{s + 2}. \]

通分:

\[ \frac{A(s + 2) + B(s - 1)}{(s - 1)(s + 2)} = \frac{2s + 3}{(s - 1)(s + 2)}. \]

所以 \(A(s + 2) + B(s - 1) = 2s + 3\)。展开 \((A + B)s + (2A - B) = 2s + 3\),所以 \(A + B = 2\),\(2A - B = 3\)。相加:\(3A = 5\),\(A = 5/3\),\(B = 2 - 5/3 = 1/3\)。所以

\[ \frac{2s + 3}{(s - 1)(s + 2)} = \frac{5/3}{s - 1} + \frac{1/3}{s + 2}. \]

反变换查表:

\[ \mathcal{L}^{-1}\left\{\frac{2s + 3}{(s - 1)(s + 2)}\right\} = \frac{5}{3} e^{t} + \frac{1}{3} e^{-2t}. \]

有个小技巧——叫遮盖法(cover-up method)——可以更快算 \(A, B\) 这种单根的系数。要求 \(A\)(对应 \(s = 1\) 的根),把式子两边同乘 \((s - 1)\):

\[ \frac{2s + 3}{s + 2} = A + \frac{B(s - 1)}{s + 2}. \]

令 \(s = 1\),右边的 \(B\) 项被 \(s - 1\) 杀死,得 \(A = (2 \cdot 1 + 3)/(1 + 2) = 5/3\)。同样,要求 \(B\),乘 \((s + 2)\) 然后令 \(s = -2\):\(B = (2 \cdot (-2) + 3)/((-2) - 1) = -1/(-3) = 1/3\)。一致。这种"代根"法只对单根有效,但常常一秒钟出答案。

例 10.2.2(重根)。求 \(\mathcal{L}^{-1}\{(s + 1)/(s - 2)^3\}\)。

分母是 \((s - 2)\) 的三重根。框架是

\[ \frac{s + 1}{(s - 2)^3} = \frac{A}{s - 2} + \frac{B}{(s - 2)^2} + \frac{C}{(s - 2)^3}. \]

通分:\(A(s - 2)^2 + B(s - 2) + C = s + 1\)。这里如果展开比较系数也行,但更快的是 变量代换 \(u = s - 2\),把分母里的根挪到 \(0\)。

\[ \frac{s + 1}{(s - 2)^3} = \frac{(u + 2) + 1}{u^3} = \frac{u + 3}{u^3} = \frac{1}{u^2} + \frac{3}{u^3} = \frac{1}{(s - 2)^2} + \frac{3}{(s - 2)^3}. \]

所以 \(A = 0, B = 1, C = 3\)。反变换:

  • \(\mathcal{L}^{-1}\{1/(s - 2)^2\} = t e^{2t}\)(用 \(\mathcal{L}\{t e^{at}\} = 1/(s - a)^2\))。
  • \(\mathcal{L}^{-1}\{3/(s - 2)^3\} = 3 \cdot t^2 e^{2t}/2 = (3/2) t^2 e^{2t}\)(用 \(\mathcal{L}\{t^2 e^{at}/2\} = 1/(s - a)^3\))。

合并:

\[ \mathcal{L}^{-1}\left\{\frac{s + 1}{(s - 2)^3}\right\} = t e^{2t} + \frac{3}{2} t^2 e^{2t} = e^{2t}\left(t + \frac{3 t^2}{2}\right). \]

"重根 \(\to\) 多项式因子"——这跟第 6 章里齐次解里"重根加 \(t\) 因子"的现象是同一回事的两个侧面。

例 10.2.3(复根/不可约二次)。求 \(\mathcal{L}^{-1}\{2s/(s^2 + 4s + 13)\}\)。

分母 \(s^2 + 4s + 13\) 的判别式是 \(16 - 52 = -36 < 0\),所以是不可约二次,对应复根。不要急着用复根分解——更省事的方法是配方:

\[ s^2 + 4s + 13 = (s^2 + 4s + 4) + 9 = (s + 2)^2 + 9. \]

这个形式提醒我们答案应当含 \(e^{-2t} \cdot \{\cos(3t) \text{ 或 } \sin(3t)\}\)。

分子怎么处理?目标是把 \(2s\) 凑成 \((s + 2)\) 和常数的组合,因为变换表上是这两个量。\(2s = 2(s + 2) - 4\),所以

\[ \frac{2s}{(s + 2)^2 + 9} = \frac{2(s + 2)}{(s + 2)^2 + 9} - \frac{4}{(s + 2)^2 + 9}. \]

第一项是 \(\mathcal{L}\{2 e^{-2t} \cos(3t)\}\)。第二项缺一个 \(\omega = 3\) 因子,凑成 \(-\frac{4}{3} \cdot \frac{3}{(s + 2)^2 + 9}\),也就是 \(-\frac{4}{3} \mathcal{L}\{e^{-2t} \sin(3t)\}\)。所以

\[ \mathcal{L}^{-1}\left\{\frac{2s}{s^2 + 4s + 13}\right\} = 2 e^{-2t}\cos(3t) - \frac{4}{3} e^{-2t}\sin(3t). \]

这道题的关键技巧:遇到不可约二次,先配方把分母化成 \((s - a)^2 + \omega^2\) 的形式,然后凑分子让它能拆成"\(s - a\)"项 + 常数项,分别对应 \(\cos\) 和 \(\sin\)。这是几乎每道含复根的题都要做的标准动作。

例 10.2.4(混合)。求 \(\mathcal{L}^{-1}\{(s^2 + 1)/(s(s - 1)(s^2 + 4))\}\)。

分母有三种因子:\(s\)(单根)、\(s - 1\)(单根)、\(s^2 + 4\)(不可约二次)。框架:

\[ \frac{s^2 + 1}{s(s - 1)(s^2 + 4)} = \frac{A}{s} + \frac{B}{s - 1} + \frac{Cs + D}{s^2 + 4}. \]

用遮盖法求 \(A, B\):

  • \(A\):令 \(s = 0\),\(A = (0 + 1)/((0 - 1)(0 + 4)) = 1/(-4) = -1/4\)。
  • \(B\):令 \(s = 1\),\(B = (1 + 1)/(1 \cdot (1 + 4)) = 2/5\)。

剩下 \(C, D\) 比较系数。通分等式:

\[ A(s - 1)(s^2 + 4) + B s (s^2 + 4) + (Cs + D) s (s - 1) = s^2 + 1. \]

把 \(A = -1/4\),\(B = 2/5\) 代进去,展开两边比较 \(s^3, s^2, s^1, s^0\) 系数。但更巧的是挑两个特殊 \(s\) 值:取 \(s = i\)(\(s^2 + 4\) 在 \(s = 2i\) 取零;\(s = i\) 也行,但下面取 \(s = 2i\) 让最复杂的项消失更直接)。设 \(s = 2i\),\(s^2 + 4 = 0\):

\[ (2iC + D)(2i)(2i - 1) = (2i)^2 + 1 = -3. \] \[ (2iC + D)(2i)(2i - 1) = (2iC + D)(4i^2 - 2i) = (2iC + D)(-4 - 2i). \]

展开:\((-4 - 2i)(2iC + D) = -8iC - 4D - 4i^2 C - 2iD = -8iC - 4D + 4C - 2iD = (4C - 4D) + (-8C - 2D)i\)。

这等于 \(-3\)(实数),所以实部 \(4C - 4D = -3\),虚部 \(-8C - 2D = 0\)。后者给 \(D = -4C\),代入前者 \(4C - 4(-4C) = -3\),\(20C = -3\),\(C = -3/20\),\(D = 12/20 = 3/5\)。

所以

\[ \frac{s^2 + 1}{s(s - 1)(s^2 + 4)} = \frac{-1/4}{s} + \frac{2/5}{s - 1} + \frac{-3s/20 + 3/5}{s^2 + 4}. \]

把最后一项拆开:\(-\frac{3}{20} \cdot \frac{s}{s^2 + 4} + \frac{3}{5} \cdot \frac{1}{s^2 + 4} = -\frac{3}{20}\cos(2t) + \frac{3}{10} \cdot \frac{2}{s^2 + 4} = -\frac{3}{20}\cos(2t) + \frac{3}{10}\sin(2t)\)。

反变换合在一起:

\[ \mathcal{L}^{-1}\{\cdots\} = -\frac{1}{4} + \frac{2}{5} e^t - \frac{3}{20}\cos(2t) + \frac{3}{10}\sin(2t). \]

这道题虽然啰嗦,但每一步都机械。重点不是答案的具体数,而是流程:先列框架,再求系数,最后查表反变换。把这套套路在不同例子上跑五遍,就内化了。

10.3 完整推演

把 10.1 的"四步流程"和 10.2 的"反变换技巧"组合起来,我们就可以解任何常系数线性 ODE。下面三道完整推演,涵盖单根、重根、复根三种情形。

例 10.3.1(单根,二阶 IVP)。解

\[ y'' - 5y' + 6y = 0, \quad y(0) = 1, \quad y'(0) = 4. \]

这道题用第 6 章的特征方程方法当然容易,但用拉普拉斯走一遍能把流程演熟。

第一步,变换。 两边取 \(\mathcal{L}\):\(\mathcal{L}\{y''\} - 5 \mathcal{L}\{y'\} + 6 \mathcal{L}\{y\} = 0\)。

第二步,代入微分定理与初值。

\[ \mathcal{L}\{y''\} = s^2 Y - s y(0) - y'(0) = s^2 Y - s - 4. \] \[ \mathcal{L}\{y'\} = s Y - y(0) = s Y - 1. \]

代入方程:

\[ (s^2 Y - s - 4) - 5(s Y - 1) + 6 Y = 0. \] \[ (s^2 - 5s + 6) Y - s - 4 + 5 = 0 \implies (s^2 - 5s + 6) Y = s - 1. \]

第三步,解 \(Y\)。

\[ Y(s) = \frac{s - 1}{s^2 - 5s + 6} = \frac{s - 1}{(s - 2)(s - 3)}. \]

第四步,反变换。 部分分式:

\[ \frac{s - 1}{(s - 2)(s - 3)} = \frac{A}{s - 2} + \frac{B}{s - 3}. \]

遮盖法:\(A = (2 - 1)/(2 - 3) = -1\),\(B = (3 - 1)/(3 - 2) = 2\)。所以

\[ Y(s) = -\frac{1}{s - 2} + \frac{2}{s - 3}, \quad y(t) = -e^{2t} + 2 e^{3t}. \]

检验:\(y(0) = -1 + 2 = 1\) ✓;\(y'(t) = -2 e^{2t} + 6 e^{3t}\),\(y'(0) = -2 + 6 = 4\) ✓。

看到没?整个过程没有"先求通解再代初值"这种两步走——初值在第二步就已经塞进了代数方程,出来的 \(Y(s)\) 直接对应特解。这就是拉普拉斯的简洁。

例 10.3.2(重根,带强迫)。解

\[ y'' - 4y' + 4y = e^{2t}, \quad y(0) = 0, \quad y'(0) = 0. \]

注意特征多项式 \(s^2 - 4s + 4 = (s - 2)^2\) 有重根 2,而右端 \(e^{2t}\) 也对应根 2——这是经典的共振(本节末尾再讨论)。我们走流程,看拉普拉斯怎么自动处理。

变换:\(\mathcal{L}\{y''\} = s^2 Y - 0 - 0 = s^2 Y\),\(\mathcal{L}\{y'\} = s Y\),\(\mathcal{L}\{e^{2t}\} = 1/(s - 2)\)。

\[ s^2 Y - 4 s Y + 4 Y = \frac{1}{s - 2} \implies (s - 2)^2 Y = \frac{1}{s - 2}. \] \[ Y(s) = \frac{1}{(s - 2)^3}. \]

反变换:从 \(\mathcal{L}\{t^n e^{at}/n!\} = 1/(s - a)^{n+1}\),取 \(n = 2, a = 2\):\(\mathcal{L}\{t^2 e^{2t}/2\} = 1/(s - 2)^3\)。所以

\[ y(t) = \frac{t^2 e^{2t}}{2}. \]

检验:\(y(0) = 0\) ✓;\(y'(t) = t e^{2t} + t^2 e^{2t}\),\(y'(0) = 0\) ✓;再算 \(y''\) 代入方程能验证(留给读者)。

这道题里特别值得注意:在用拉普拉斯做的过程中,你根本不需要识别"共振"——分母自动变成了 \((s - 2)^3\),反变换自动得到 \(t^2\) 因子。换言之,共振在 \(s\)-域里只是"分母多了一阶",自动产生了 \(t\) 的幂次,你完全不必做特殊处理。这是拉普拉斯的另一个优点:它把"共振 vs 非共振"这种条件判断从你身上拿走了。

例 10.3.3(复根,带正弦强迫)。解

\[ y'' + y = \sin(2t), \quad y(0) = 1, \quad y'(0) = 0. \]

变换:\(s^2 Y - s + Y = \mathcal{L}\{\sin(2t)\} = 2/(s^2 + 4)\)。整理:

\[ (s^2 + 1) Y = s + \frac{2}{s^2 + 4} \implies Y = \frac{s}{s^2 + 1} + \frac{2}{(s^2 + 1)(s^2 + 4)}. \]

第一项 \(s/(s^2 + 1)\) 直接反变换:\(\cos t\)。第二项要部分分式。设

\[ \frac{2}{(s^2 + 1)(s^2 + 4)} = \frac{As + B}{s^2 + 1} + \frac{Cs + D}{s^2 + 4}. \]

通分:\((As + B)(s^2 + 4) + (Cs + D)(s^2 + 1) = 2\)。展开:\((A + C) s^3 + (B + D) s^2 + (4A + C) s + (4B + D) = 2\)。所以

\[ A + C = 0, \quad B + D = 0, \quad 4A + C = 0, \quad 4B + D = 2. \]

从 \(A + C = 0\) 与 \(4A + C = 0\):相减 \(3A = 0\),\(A = 0\),\(C = 0\)。从 \(B + D = 0\) 与 \(4B + D = 2\):相减 \(3B = 2\),\(B = 2/3\),\(D = -2/3\)。

\[ \frac{2}{(s^2 + 1)(s^2 + 4)} = \frac{2/3}{s^2 + 1} - \frac{2/3}{s^2 + 4}. \]

反变换:第一项 \((2/3) \sin t\);第二项 \(-(2/3) \cdot (1/2) \sin(2t) = -(1/3) \sin(2t)\)(因为 \(2/(s^2 + 4) = \mathcal{L}\{\sin(2t)\}\),所以 \(1/(s^2 + 4) = (1/2) \mathcal{L}\{\sin(2t)\}\))。合并:

\[ y(t) = \cos t + \frac{2}{3} \sin t - \frac{1}{3} \sin(2t). \]

检验初值:\(y(0) = 1 + 0 - 0 = 1\) ✓;\(y'(t) = -\sin t + (2/3)\cos t - (2/3)\cos(2t)\),\(y'(0) = 0 + 2/3 - 2/3 = 0\) ✓。

注意结构:\(\cos t\) 是齐次解的"自然振动"部分,\((2/3)\sin t\) 是齐次解的另一支(因为初值约束的需要它出现),\(-(1/3)\sin(2t)\) 是被强迫频率 \(\omega = 2\) 的"特解"。这跟第 7 章未定系数法的结果完全一致——只不过这里我们没有"猜形式"那一步,所有的事都是代数自动出来的。

例 10.3.4(共振)。解

\[ y'' + 4y = \cos(2t), \quad y(0) = 0, \quad y'(0) = 0. \]

注意自然频率 \(\omega_0 = 2\),驱动频率 \(\omega = 2\),完全相等——这是共振情形。

变换:\(s^2 Y + 4 Y = \mathcal{L}\{\cos(2t)\} = s/(s^2 + 4)\)。

\[ (s^2 + 4) Y = \frac{s}{s^2 + 4} \implies Y = \frac{s}{(s^2 + 4)^2}. \]

这个 \(s/(s^2 + \omega^2)^2\) 对应什么?第 9 章 9.9 节我们算过 \(\mathcal{L}\{t \sin(\omega t)\} = 2 \omega s/(s^2 + \omega^2)^2\)。这里 \(\omega = 2\):\(\mathcal{L}\{t \sin(2t)\} = 4s/(s^2 + 4)^2\)。所以

\[ \frac{s}{(s^2 + 4)^2} = \frac{1}{4} \mathcal{L}\{t \sin(2t)\} \implies y(t) = \frac{t}{4} \sin(2t). \]

检验:\(y(0) = 0\) ✓;\(y'(t) = (1/4)\sin(2t) + (t/2)\cos(2t)\),\(y'(0) = 0\) ✓;\(y'' = (1/2)\cos(2t) + (1/2)\cos(2t) - t\sin(2t) = \cos(2t) - t\sin(2t)\),\(y'' + 4y = \cos(2t) - t\sin(2t) + 4 \cdot (t/4)\sin(2t) = \cos(2t)\) ✓。

这就是共振响应:振幅线性增长,因为有那个 \(t\) 因子。物理上,如果你在天然频率上推荡秋千,每一次推都加力,振幅就会越来越大。代数上,这个 \(t\) 因子来自 \(s/(s^2 + 4)^2\)——分母的"重二次因子"。这又一次展示了拉普拉斯的"自动":你不必手工"识别共振再加 \(t\)",系数表自己就把这件事处理完了。

例 10.3.5(齐次三阶)。解

\[ y''' - y' = 0, \quad y(0) = 0, \quad y'(0) = 1, \quad y''(0) = 0. \]

三阶,所以需要三个初值。变换:

\[ \mathcal{L}\{y'''\} = s^3 Y - s^2 y(0) - s y'(0) - y''(0) = s^3 Y - 0 - s - 0 = s^3 Y - s. \] \[ \mathcal{L}\{y'\} = s Y - 0 = s Y. \] \[ (s^3 Y - s) - s Y = 0 \implies (s^3 - s) Y = s \implies Y = \frac{s}{s^3 - s} = \frac{s}{s(s - 1)(s + 1)} = \frac{1}{(s - 1)(s + 1)}. \]

(分子分母约去 \(s\)。)部分分式:

\[ \frac{1}{(s - 1)(s + 1)} = \frac{A}{s - 1} + \frac{B}{s + 1}. \]

遮盖法:\(A = 1/(1 + 1) = 1/2\),\(B = 1/(-1 - 1) = -1/2\)。

\[ y(t) = \frac{1}{2} e^t - \frac{1}{2} e^{-t} = \sinh t. \]

(用了 \(\sinh t = (e^t - e^{-t})/2\) 这个定义。)

检验:\(y(0) = 0\) ✓;\(y' = \cosh t\),\(y'(0) = 1\) ✓;\(y'' = \sinh t\),\(y''(0) = 0\) ✓;\(y''' = \cosh t\),\(y''' - y' = \cosh t - \cosh t = 0\) ✓。

10.3.6 含右端是"指数 × 三角"的题

例 10.3.6(用 \(s\)-平移定理)。解

\[ y'' + 2y' + 2y = e^{-t}\cos t, \quad y(0) = 0, \quad y'(0) = 0. \]

右端项是 \(e^{-t}\cos t\),用 \(s\)-平移:\(\mathcal{L}\{\cos t\} = s/(s^2 + 1)\),所以 \(\mathcal{L}\{e^{-t}\cos t\} = (s + 1)/((s + 1)^2 + 1)\)。

左端的特征多项式 \(s^2 + 2s + 2 = (s + 1)^2 + 1\)——和右端项的分母一样!这又是一个"共振"的变种(根的位置匹配)。

变换方程:\((s^2 + 2s + 2) Y = (s + 1)/((s + 1)^2 + 1)\),即

\[ ((s + 1)^2 + 1) Y = \frac{s + 1}{(s + 1)^2 + 1} \implies Y = \frac{s + 1}{((s + 1)^2 + 1)^2}. \]

形式是 \((s + 1)/((s + 1)^2 + 1)^2\)。先做平移:设 \(F(s) = s/(s^2 + 1)^2\),则 \(Y(s) = F(s + 1)\),由 \(s\)-平移逆向操作 \(y(t) = e^{-t} f(t)\),其中 \(f(t) = \mathcal{L}^{-1}\{F(s)\}\)。

而 \(s/(s^2 + 1)^2\) 我们在 9.9 节算过:\(\mathcal{L}\{t \sin t\} = 2 \cdot 1 \cdot s/(s^2 + 1)^2\),所以 \(s/(s^2 + 1)^2 = (1/2) \mathcal{L}\{t \sin t\}\),也就是 \(f(t) = (1/2) t \sin t\)。

合起来:

\[ y(t) = \frac{1}{2} e^{-t} t \sin t. \]

检验留给读者(步骤稍繁但能算通)。这种"指数 × 三角"的强迫项,在第 7-8 章的未定系数法里很容易让人写错猜测形式;在拉普拉斯里,只是一次 \(s\)-平移加一次"\(t\) 乘"模式,两步就完了。

10.4 与第 5-8 章的方法对比

到这里你已经看到拉普拉斯方法在 IVP 问题上很省事。但什么时候更省事?什么时候反而该用经典方法?这一节把账算清楚。

10.4.1 拉普拉斯什么时候赢

核心场景:带初值的常系数线性 ODE。具体地,以下情况下,拉普拉斯几乎一定更省事:

  1. 初值给在 \(t = 0\) 处。 微分定理直接吸收 \(y(0), y'(0), \ldots\),不需要"先求通解,再代初值"两步走。如果初值给在 \(t = a \ne 0\) 处,可以做平移 \(\tilde t = t - a\) 化到 \(t = 0\),仍然能用,但稍麻烦。
  2. 右端是分段函数、单位阶跃或冲击。 第 11 章的 \(t\)-平移和 Dirac \(\delta\) 让这类题在 \(s\)-域里只是几个 \(e^{-as}\) 因子,经典方法处理起来非常笨。
  3. 右端含 \(e^{at} \cdot p(t) \cdot \cos(\omega t)\) 这种"指数 × 多项式 × 三角"的混合项。 未定系数法要写超长的猜测形式,共振情形还要乘 \(t\) 修正;拉普拉斯里只是几条标准定理的组合。
  4. 系统(联立 ODE)。 几个未知函数耦合的方程组,在 \(s\)-域里变成几个未知 \(Y_i(s)\) 的线性代数方程组,克莱姆法则或消元就能解。这比"求向量特征值再写矩阵指数"路径要直观得多。
  5. 右端含卷积。 第 11 章会看到,卷积在 \(s\)-域里就是"相乘",代数化操作非常顺。

10.4.2 经典方法什么时候赢

反过来,以下情况下经典方法可能更直接:

  1. 只要通解,不要特解。 比如"写出二阶方程的通解",经典方法的"特征方程 → 求根 → 写齐次解"流程就能直接给答案,而拉普拉斯需要先有初值。
  2. 变系数方程。 比如 \(t y'' + y' + t y = 0\)(贝塞尔方程),拉普拉斯一般做不了——\(\mathcal{L}\{t y''\}\) 涉及 \(F(s)\) 对 \(s\) 求导,会得到关于 \(Y(s)\) 的微分方程(往往跟原方程一样难)。
  3. 非线性 ODE。 拉普拉斯本质上是线性算子(\(\mathcal{L}\{f g\} \ne \mathcal{L}\{f\} \mathcal{L}\{g\}\)),非线性方程没有什么好办法。
  4. 右端项太离奇。 比如右端是 \(e^{t^2}\),它没有拉普拉斯变换(超指数增长)。这种情况下你只能数值求解或换其他变换。

10.4.3 一个对照表

问题特征推荐方法
常系数 + IVP + 简单右端(指数/多项式/三角)拉普拉斯,通常更短
常系数 + IVP + 分段或冲击右端拉普拉斯几乎是唯一选择(第 11 章)
常系数 + 求通解经典(特征方程 + 未定系数)
常系数线性方程组(IVP)拉普拉斯
变系数方程经典(级数法、特殊技巧),拉普拉斯一般不行
非线性方程都难,需要数值或定性分析
带卷积的积分方程或积分-微分方程拉普拉斯

所以一句话总结:"常系数 + IVP + 任何特别难处理的右端 → 拉普拉斯"。这是它的拿手菜,而且非常拿手。

10.4.4 经典方法做不了的一个例子

来感受一下"非拉普拉斯不可"的题。考虑

\[ y'' + y = f(t), \quad y(0) = 0, \quad y'(0) = 0, \]

其中

\[ f(t) = \begin{cases} 1, & 0 \le t < \pi \\ 0, & t \ge \pi \end{cases}. \]

用经典方法:必须分两段解,在 \([0, \pi]\) 上解 \(y'' + y = 1\),在 \([\pi, \infty)\) 上解 \(y'' + y = 0\),然后在 \(t = \pi\) 处把 \(y\) 和 \(y'\) "对接"起来。能算,但麻烦。

下一章的拉普拉斯方法只需要把 \(f(t)\) 写成 \(1 - u(t - \pi)\)(其中 \(u\) 是单位阶跃),然后 \(\mathcal{L}\{f\} = 1/s - e^{-\pi s}/s\),代进方程一步到位,最后用 \(t\)-平移定理反变换。整个过程一气呵成,不需要"分段对接"。这道题第 11 章会再做一遍,作为阶跃函数登场的招牌动作。

本章把拉普拉斯方法用在了"标准的"常系数线性 ODE 上。下一章引入两个新角色——单位阶跃函数 和 Dirac \(\delta\) 函数——以及一个连接两个函数的强大工具:卷积。这些扩展让拉普拉斯方法的覆盖面再上一个台阶,可以处理"开关"、"敲打"、"任意外力"这类工程上极常见的右端项。
