第 16 章 幂级数解

到这里为止,我们解线性常微分方程的"武器库"基本上只有一件主力武器:常系数线性方程的特征方程法。一旦系数变成 \(x\) 的函数,我们就只能依靠少数几招——比如可以化为常系数的 Euler 方程,或者侥幸瞧出一个解然后用降阶法把第二个解凑出来。这显然不够。物理和工程里冒出来的方程绝大多数都是变系数的:量子力学里的氢原子径向方程,声学里描述圆鼓振动的方程,行星运动的微扰方程,弹性力学的薄壳方程……它们都是线性二阶变系数,而且它们的解几乎从来不是初等函数。

那怎么办?把方程扔了不管?当然不行。我们需要一种新的解的概念——一种不要求把答案写成 \(\sin\)、\(\cos\)、\(e^x\)、多项式这种东西的有限组合,而是允许"无穷多项加起来"的那种解。这就是幂级数解。本章和下一章的主线就是:用幂级数当作"通用候选答案",代进方程里,逼着方程告诉我们答案的每一项系数应该是什么。这套方法不需要灵感,不需要瞧出"凑巧的解",只需要耐心和组合代数。它的代价是答案不再是封闭表达式,而是一个无穷级数;它的回报是,你将能解出几乎所有出现在物理学里的二阶线性方程。

本章会:

  • 说明为什么变系数方程"逼"我们走向级数;
  • 把幂级数最关键的几条事实复习一遍(收敛半径、可微可积、唯一性);
  • 把"代入级数,等系数"这套形式做法讲清楚;
  • 用三个详细例子打熟手感:\(y'' + y = 0\)(应得 \(\sin x, \cos x\))、Airy 方程 \(y'' - x y = 0\)、以及一道带普通项的非齐次例题;
  • 给出"普通点"的概念与存在性定理;
  • 给出收敛半径的"距离最近奇异点"估计;
  • 多刷几道变系数例题,把套路彻底打入肌肉记忆。

奇异点的处理放到第 17 章——那里会出现 Frobenius 方法,以及一连串带名字的特殊函数 (Bessel、Legendre、Hermite、Laguerre)。本章先把"普通点"这个最干净的情形吃透。

16.1 为什么需要幂级数

来看看下面这个无害的方程:

\[ y'' - x y = 0. \]

它叫做Airy 方程,是数学物理里的明星之一。它出现在光学(光在焦点附近的衍射图样)、量子力学(线性势能下的薛定谔方程)、平面波在缓变折射率介质里的传播,等等。它看起来朴素得不能再朴素——\(y'' = xy\),没有任何技巧的余地。可是请你试着用我们已经学过的所有方法去解它:

  • 它是线性的,但系数不是常数(\(-x\) 不是常数),所以特征方程法不能用。
  • 它不是 Euler 方程(Euler 方程是 \(x^2 y'' + a x y' + b y = 0\),系数和 \(x\) 的幂次匹配上了导数阶数,这里不是)。
  • 没有显然能"瞧"出来的解。\(y = \sin x\)?\(y'' = -\sin x\),\(xy = x\sin x\),不相等。\(y = e^x\)?\(y'' = e^x\),\(xy = x e^x\),不相等。\(y = e^{x^2/2}\)?\(y' = x e^{x^2/2}\),\(y'' = e^{x^2/2} + x^2 e^{x^2/2}\),\(xy = x e^{x^2/2}\),也不相等。

事实上,可以证明,Airy 方程的两个独立解都不是初等函数。它们叫做 \(\mathrm{Ai}(x)\) 和 \(\mathrm{Bi}(x)\),今天的特殊函数表上各占一行,但你无法用 \(\sin\)、\(\cos\)、\(\exp\)、对数、多项式、根式有限多次组合得到它们。这就好像一开始 \(\pi\) 不是有理数:它是真实存在的、有用的、可以无限精确逼近的,但你必须接受"它落在初等表达的世界之外"。

那这种方程怎么办?这里有一个非常关键的转念:"解"不一定要是封闭表达式。如果我们能写出

\[ y(x) = a_0 + a_1 x + a_2 x^2 + a_3 x^3 + \cdots = \sum_{n=0}^{\infty} a_n x^n \]

而且能把每一个系数 \(a_n\) 都算出来,那这就是一个完美的解——你想知道 \(y(0.3)\) 是多少?把级数前 20 项一加,精度比工程师全部要求的还高;你想画出 \(y\) 的图像?把级数当作多项式去算几百个点,曲线就出来了;你想研究它的渐近行为?有专门的方法把级数变形再做估计。

关键事实是:一大类变系数线性方程,在某些区间上,确实存在收敛的幂级数解。而且这种级数解的系数可以由方程通过递推关系逐项算出来。这就是幂级数法的核心承诺。它解决的是一个原本没希望的问题:把"我没有任何初等技巧能搞它"的方程,化成"我只需要会算下标和会做组合代数"的练习。

一句话总结本章的精神:当你不能写出答案是什么时,你至少可以写出答案的每一位。这就是幂级数法。

16.2 幂级数复习

在动手把级数代进微分方程之前,先把幂级数本身的几条关键事实复习一遍。如果你高级微积分学得很熟,这一节可以扫一眼跳过;如果不熟,请慢慢看,后面 every 一道例题都要用到这里的工具。

16.2.1 收敛半径

所谓幂级数(以 \(0\) 为中心的)就是形如

\[ \sum_{n=0}^{\infty} a_n x^n = a_0 + a_1 x + a_2 x^2 + \cdots \]

的级数。每一项是一个固定数 \(a_n\) 乘以 \(x^n\)。注意:幂级数本身只是一串"形式上写下来"的东西,它在某个具体的 \(x\) 上未必有值——只有当对应的数项级数收敛时,这个 \(x\) 才是它的"合法输入"。

收敛与否取决于 \(x\) 离中心的距离。基本结果(由 Abel 给出)是:存在一个 \(R \in [0, +\infty]\),叫做收敛半径,使得

  • 当 \(|x| < R\) 时级数绝对收敛;
  • 当 \(|x| > R\) 时级数发散;
  • 当 \(|x| = R\) 时,可能收敛也可能发散——这是边界,要逐个检查。

计算 \(R\) 最常用的两个公式:

\[ \frac{1}{R} = \lim_{n \to \infty} \left| \frac{a_{n+1}}{a_n} \right| \quad \text{(比值法)}, \qquad \frac{1}{R} = \limsup_{n \to \infty} |a_n|^{1/n} \quad \text{(根值法)}. \]

不要担心这两个公式的细则——本章我们用的几乎全是比值法。

来看几个标准例子,它们都是后面要反复出现的"老朋友":

\[ e^x = \sum_{n=0}^{\infty} \frac{x^n}{n!}, \qquad \sin x = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n+1}}{(2n+1)!}, \qquad \cos x = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n}}{(2n)!}. \]

这三个的收敛半径都是 \(R = +\infty\),也就是对所有实数 \(x\) 都收敛。验证一下 \(e^x\):比值法给出 \(|a_{n+1}/a_n| = (n!)/((n+1)!) = 1/(n+1) \to 0\),所以 \(1/R = 0\),\(R = \infty\)。

对比一个反例,几何级数 \(\sum x^n = 1/(1-x)\),其中 \(a_n = 1\),比值是 \(1\),所以 \(R = 1\)——这个级数只在 \(|x| < 1\) 内收敛,在 \(x = 1\) 时给出了 \(\sum 1\) 显然发散。

展开点也可以不是 \(0\)。一般地,以 \(x_0\) 为中心的幂级数是

\[ \sum_{n=0}^{\infty} a_n (x - x_0)^n, \]

它的收敛区间形如 \((x_0 - R, x_0 + R)\)。这种"换中心"是把方程展开在不同点附近的解时常用的;本章绝大多数例子都把中心取在 \(0\),只有少数题目我们会改用 \(x_0 = 1\) 之类。

16.2.2 可微、可积、唯一性

幂级数最舒服的一件事是:在收敛区间内部,你可以逐项求导和积分,而且求导以后的级数有同样的收敛半径。具体地,设 \(f(x) = \sum a_n x^n\) 在 \(|x| < R\) 上收敛,则在 \(|x| < R\) 上:

\[ f'(x) = \sum_{n=1}^{\infty} n a_n x^{n-1}, \qquad \int_0^x f(t)\, dt = \sum_{n=0}^{\infty} \frac{a_n}{n+1} x^{n+1}. \]

请反复看一下求导那一项:求和从 \(n = 1\) 开始,因为 \(n = 0\) 那一项 \(a_0\) 是常数,导数为零。这个"下标的位移"是后面所有计算里都要小心的细节,你练几次就习惯了。

第二个关键事实是唯一性:如果两个幂级数 \(\sum a_n x^n\) 和 \(\sum b_n x^n\) 在 \(0\) 周围的某个开区间上恒等(也就是,两个级数在那一段上代表同一个函数),那么逐项相等:\(a_n = b_n\) 对所有 \(n\) 成立。

这是把"两个级数相等"翻译成"两串系数相等"的合法许可证。整个本章的"代入级数,然后比对系数"这套做法,合法性就来自这条唯一性定理。如果没有它,我们看到 \(\sum a_n x^n = 0\) 时还不能下结论"所以 \(a_n = 0\) 对所有 \(n\)";有了它,我们立刻就能说"是的,所有系数都是零"。

这一条尤其值得被打入肌肉记忆——后面每次"等系数"一步,你都在悄悄用这条定理。

16.2.3 求和指标的位移

在做"代入级数"时,经常需要把不同求和指标的级数对齐。例如

\[ \sum_{n=2}^{\infty} n(n-1) a_n x^{n-2} \]

(这是 \(y(x) = \sum a_n x^n\) 的二阶导)和

\[ \sum_{n=0}^{\infty} a_n x^{n+2} \]

(这是 \(x^2 y\)),它们的求和指标完全不一样,无法直接合并。要合并就得做指标平移。

方法很简单。第一个里令 \(m = n - 2\),则 \(n = m + 2\),\(n = 2\) 对应 \(m = 0\),于是

\[ \sum_{n=2}^{\infty} n(n-1) a_n x^{n-2} = \sum_{m=0}^{\infty} (m+2)(m+1) a_{m+2} x^{m}. \]

(然后因为 \(m\) 只是 dummy 变量,我们重新把它叫做 \(n\),变成 \(\sum_{n=0}^{\infty}(n+2)(n+1)a_{n+2} x^n\)。)指标平移这个动作以后会做几十次,记住一句口诀就够:"x 的指数提多少,求和起点就提多少,系数下标就降多少"。具体一点:把每个级数都改写成 \(\sum_{n=k}^{\infty}(\text{某式}) x^n\) 的形式,然后取所有 \(k\) 中最大的当公共起点,前面零零散散漏出来的几项再单独写出来对系数。这一段听起来繁,做几遍后就完全自动化了。

16.3 形式做法

下面是把幂级数代进方程的"标准菜谱"。你把它当作"一份食谱"来记,本章和下一章里几乎所有例题都按这个流程走。

设要解

\[ P(x) y'' + Q(x) y' + R(x) y = 0, \]

而且 \(P\)、\(Q\)、\(R\) 在 \(x = x_0\) 附近都是"漂亮"的(详细的"漂亮"在 16.6 节定义)。

  1. 设解为
\[ y(x) = \sum_{n=0}^{\infty} a_n (x - x_0)^n. \]
  1. 逐项求导得到
\[ y'(x) = \sum_{n=1}^{\infty} n a_n (x - x_0)^{n-1}, \qquad y''(x) = \sum_{n=2}^{\infty} n(n-1) a_n (x - x_0)^{n-2}. \]
  1. 代入方程,把 \(P, Q, R\) 也展开为 \((x - x_0)\) 的幂级数(如果它们已经是多项式就更舒服了)。
  2. 合并同次幂(必要时做指标平移)。
  3. 设系数等于零(因为整个级数恒为零,所以每一项的系数都为零)。这就给出关于 \(a_n\) 的递推关系。
  4. 逐项算出 \(a_n\)。\(a_0, a_1\) 通常是任意的两个常数(对应于二阶方程的两个任意常数),后面的系数被它们决定。
  5. 整理出两个独立解 \(y_1, y_2\)(把 \(a_0 = 1, a_1 = 0\) 与 \(a_0 = 0, a_1 = 1\) 分开做)。
  6. (选)估计收敛半径。

就八步。看起来繁,做完一道例题你就觉得它是机械的。下一节就开始详细的第一道。

16.4 详细推演:用幂级数解 \(y'' + y = 0\)

当然,我们知道 \(y'' + y = 0\) 的解是 \(\cos x\) 和 \(\sin x\)——它是二阶常系数齐次方程,特征方程 \(r^2 + 1 = 0\),特征根 \(r = \pm i\)。我们已经会解。所以为什么要用幂级数法做一遍?

原因有两个。第一,这是一个我们能验证结果对不对的例题。如果级数法做对了,我们应该得到 \(\cos\) 和 \(\sin\) 的级数。第二,也是更重要的,这道题让你看到一个深刻的事实:那些我们日常使用的"特殊函数",比如 \(\sin\) 和 \(\cos\),其实就是某个微分方程的幂级数解——级数法不是在"造"新东西,它是在系统化地呈现出我们已有的函数。当我们后面遇到 Bessel、Legendre 时,会发现"那些奇怪的特殊函数"和 \(\sin\) 没有本质差别,只不过它们对应的方程稍微复杂一点。

开始。设

\[ y(x) = \sum_{n=0}^{\infty} a_n x^n. \]

则

\[ y''(x) = \sum_{n=2}^{\infty} n(n-1) a_n x^{n-2}. \]

代入 \(y'' + y = 0\):

\[ \sum_{n=2}^{\infty} n(n-1) a_n x^{n-2} + \sum_{n=0}^{\infty} a_n x^n = 0. \]

第一项里 \(x\) 的幂是 \(n - 2\),第二项里是 \(n\)。要合并,把第一项的指标平移:令 \(m = n - 2\),

\[ \sum_{n=2}^{\infty} n(n-1) a_n x^{n-2} = \sum_{m=0}^{\infty} (m+2)(m+1) a_{m+2} x^m. \]

(改 \(m\) 回 \(n\)。)于是方程变成

\[ \sum_{n=0}^{\infty} \big[ (n+2)(n+1) a_{n+2} + a_n \big] x^n = 0. \]

由唯一性,每一个 \(x^n\) 的系数都为零:

\[ (n+2)(n+1) a_{n+2} + a_n = 0 \quad \Longrightarrow \quad a_{n+2} = -\frac{a_n}{(n+2)(n+1)}. \]

这就是递推关系。漂亮!它把每个偶数下标的系数与上一个偶数下标联系起来,奇数下标也一样。

把 \(a_0\) 和 \(a_1\) 留作任意常数,推一推:

偶数下标:

\[ a_2 = -\frac{a_0}{2 \cdot 1} = -\frac{a_0}{2!}, \quad a_4 = -\frac{a_2}{4 \cdot 3} = \frac{a_0}{4!}, \quad a_6 = -\frac{a_4}{6 \cdot 5} = -\frac{a_0}{6!}, \ldots \]

规律:\(a_{2k} = (-1)^k a_0 / (2k)!\)。

奇数下标:

\[ a_3 = -\frac{a_1}{3 \cdot 2} = -\frac{a_1}{3!}, \quad a_5 = -\frac{a_3}{5 \cdot 4} = \frac{a_1}{5!}, \quad a_7 = -\frac{a_5}{7 \cdot 6} = -\frac{a_1}{7!}, \ldots \]

规律:\(a_{2k+1} = (-1)^k a_1 / (2k+1)!\)。

所以

\[ y(x) = a_0 \sum_{k=0}^{\infty} \frac{(-1)^k x^{2k}}{(2k)!} + a_1 \sum_{k=0}^{\infty} \frac{(-1)^k x^{2k+1}}{(2k+1)!}. \]

现在请你抬头看一眼 16.2 节里写过的 \(\cos x\) 和 \(\sin x\) 的级数——它们就是上面这两个级数!所以

\[ y(x) = a_0 \cos x + a_1 \sin x. \]

结果完全正确。两个独立解就是 \(y_1(x) = \cos x\) 和 \(y_2(x) = \sin x\),正如所料。

这件事值得停下来想一想。我们没有用任何"\(\sin, \cos\) 是什么"的预先知识——我们只是把幂级数代进方程,等系数,然后整出来的两个级数恰好就是 \(\sin x\) 和 \(\cos x\)。换个角度看:你完全可以反过来定义 \(\sin\) 和 \(\cos\) 为方程 \(y'' + y = 0\) 在 \(0\) 点附近满足初值条件 \((y(0), y'(0)) = (1, 0)\) 与 \((0, 1)\) 的两个解。事实上,Cauchy 在 19 世纪就提出过这种"用微分方程来定义初等函数"的看法——这种看法把所有"特殊函数"放在一个统一的视角下:它们都是某个二阶 ODE 的幂级数解。

收敛半径方面,这个级数和指数级数类似,比值 \(|a_{n+2}/a_n| = 1/[(n+2)(n+1)] \to 0\),所以 \(R = +\infty\)——\(\sin x, \cos x\) 在整条实轴上都有定义,这与我们已知的事实吻合。

16.5 Airy 方程的级数解

现在用同一套机械流程去捕获一个我们事先不认识其解的方程:Airy 方程

\[ y'' - x y = 0. \]

系数 \(P(x) = 1, Q(x) = 0, R(x) = -x\),它们都是多项式,任意点都"漂亮"。在 \(x_0 = 0\) 附近设

\[ y(x) = \sum_{n=0}^{\infty} a_n x^n. \]

则 \(y'' = \sum_{n=2}^{\infty} n(n-1) a_n x^{n-2}\),而 \(xy = \sum_{n=0}^{\infty} a_n x^{n+1}\)。代入方程:

\[ \sum_{n=2}^{\infty} n(n-1) a_n x^{n-2} - \sum_{n=0}^{\infty} a_n x^{n+1} = 0. \]

把两项的求和指标都平移到"\(x^n\)"的统一形式。第一项:令 \(m = n - 2\),\(\sum_{m=0}^{\infty}(m+2)(m+1)a_{m+2} x^m\)。第二项:令 \(m = n + 1\),\(\sum_{m=1}^{\infty} a_{m-1} x^m\)。注意第二项的求和是从 \(m = 1\) 开始的,不是 \(m = 0\)——因为原来 \(n \geq 0\) 对应 \(m \geq 1\)。

(改 \(m\) 为 \(n\)。)合并时要把 \(m = 0\) 那项单独提出来:

\[ \underbrace{2 \cdot 1 \cdot a_2}_{\text{第一项 } n=0} \cdot x^0 + \sum_{n=1}^{\infty} \big[ (n+2)(n+1) a_{n+2} - a_{n-1} \big] x^n = 0. \]

由唯一性,每个系数为零:

\[ 2 a_2 = 0 \implies a_2 = 0, \] \[ (n+2)(n+1) a_{n+2} - a_{n-1} = 0, \quad n \geq 1 \implies a_{n+2} = \frac{a_{n-1}}{(n+2)(n+1)}. \]

这是 Airy 方程的递推关系——它把下标差 3 的系数联系起来。所以 \(a_0, a_3, a_6, \ldots\) 是一族,\(a_1, a_4, a_7, \ldots\) 是一族,\(a_2, a_5, a_8, \ldots\) 是又一族。而我们已经知道 \(a_2 = 0\),所以第三族整族都为零(\(a_5 = a_2/(5 \cdot 4) = 0\),\(a_8 = a_5/(8 \cdot 7) = 0\),……)。我们其实只需要追两族。

第一族(从 \(a_0\) 出发,\(n + 2\) 是 \(3, 6, 9, \ldots\) 这种模 3 余 0 的下标——把 \(n = 1\) 代进递推得 \(a_3 = a_0/(3 \cdot 2)\)):

\[ a_3 = \frac{a_0}{3 \cdot 2}, \quad a_6 = \frac{a_3}{6 \cdot 5} = \frac{a_0}{6 \cdot 5 \cdot 3 \cdot 2}, \quad a_9 = \frac{a_6}{9 \cdot 8} = \frac{a_0}{9 \cdot 8 \cdot 6 \cdot 5 \cdot 3 \cdot 2}, \ldots \]

分母是几个连乘的"成对"因子,你可以把它写成

\[ a_{3k} = \frac{a_0}{(3k)(3k-1) \cdot (3k-3)(3k-4) \cdots 6 \cdot 5 \cdot 3 \cdot 2}. \]

这个公式看着乱,可它就是这样。如果你会写 \(\Gamma\) 函数,它有更紧凑的形式;不会也无所谓——逐项算就行。

第二族(从 \(a_1\) 出发,\(n = 2\) 给 \(a_4 = a_1/(4 \cdot 3)\)):

\[ a_4 = \frac{a_1}{4 \cdot 3}, \quad a_7 = \frac{a_4}{7 \cdot 6} = \frac{a_1}{7 \cdot 6 \cdot 4 \cdot 3}, \quad a_{10} = \frac{a_7}{10 \cdot 9} = \frac{a_1}{10 \cdot 9 \cdot 7 \cdot 6 \cdot 4 \cdot 3}, \ldots \]

所以两个独立解是

\[ y_1(x) = 1 + \frac{x^3}{6} + \frac{x^6}{180} + \frac{x^9}{12960} + \cdots, \] \[ y_2(x) = x + \frac{x^4}{12} + \frac{x^7}{504} + \frac{x^{10}}{45360} + \cdots. \]

(\(y_1\) 取 \(a_0 = 1, a_1 = 0\);\(y_2\) 取 \(a_0 = 0, a_1 = 1\)。)

通解是 \(y(x) = a_0 y_1(x) + a_1 y_2(x)\)。著名的 Airy 函数 \(\mathrm{Ai}\) 与 \(\mathrm{Bi}\) 是这两个解的线性组合,它们的具体常数系数被选成让 \(\mathrm{Ai}, \mathrm{Bi}\) 在 \(x \to \infty\) 时有干净的渐近形式(分别是衰减和增长),但这超出本节范围。

关于收敛半径:Airy 方程的系数 \(P(x) = 1\) 永远不为零,\(Q, R\) 在所有点都漂亮,所以"最近的奇异点"在无穷远——根据 16.7 节即将给出的定理,级数解的收敛半径是 \(R = \infty\)。也就是说,\(y_1, y_2\) 对所有实数 \(x\) 都收敛,它们是名副其实的"整函数"。

注意一个有趣的对照:\(y'' + y = 0\) 的两个独立解 \(\cos, \sin\) 是有界的、振荡的;Airy 方程 \(y'' = xy\) 的解则不有界(\(x > 0\) 时它们指数式地增长,\(x < 0\) 时它们振荡)。两者只差一个常数 vs. 线性的系数前面的负号,行为却天差地别。这正是变系数线性方程的"个性"。

16.6 普通点与存在性定理

到这里你已经做过两道幂级数题——一道熟悉的(\(\sin, \cos\)),一道不熟的(Airy)。两道都成功了。但凭什么保证这套方法对一般方程也行?会不会有些方程,你代入级数做完递推,但最后那个级数的收敛半径是 0,毫无用处?这一节回答这个问题。

把方程统一写成标准形式:

\[ y'' + p(x) y' + q(x) y = 0, \]

其中 \(p(x) = Q(x)/P(x)\),\(q(x) = R(x)/P(x)\)。这一步的关键意义是:很多方程是写成 \(P y'' + Q y' + R y = 0\) 的,要把它除以 \(P(x)\)之后才能讲下面的故事。

定义:点 \(x_0\) 是方程的普通点 (ordinary point),如果 \(p(x)\) 与 \(q(x)\) 在 \(x_0\) 处都解析(也就是,在 \(x_0\) 附近都能展开为收敛的幂级数);否则 \(x_0\) 是方程的奇异点。

对于"漂亮的"方程——\(p, q\) 都是有理函数(多项式 / 多项式)——这个定义有一个非常好用的等价说法:\(x_0\) 是普通点当且仅当 \(P(x_0) \neq 0\)(也就是,以 \(P(x_0) = 0\) 为条件的那些点 是奇异点的候选)。这是检查一个具体方程的"机械办法"。

例子。看方程 \((x^2 - 1) y'' + xy' + y = 0\)。这里 \(P(x) = x^2 - 1\),所以 \(P(\pm 1) = 0\),奇异点是 \(\pm 1\),其他所有点都是普通点。例如 \(x_0 = 0, 2, -3\) 都是普通点。再看方程 \(y'' + (\sin x) y' + e^x y = 0\)。这里 \(p(x) = \sin x, q(x) = e^x\),两个都解析(它们是整函数),所以所有点都是普通点。再看 Bessel 方程 \(x^2 y'' + xy' + (x^2 - \nu^2) y = 0\)。把它写成 \(y'' + (1/x) y' + (1 - \nu^2/x^2) y = 0\),这里 \(p = 1/x, q = 1 - \nu^2/x^2\) 在 \(x = 0\) 处不解析(\(1/x\) 在那里发散),所以 \(x = 0\) 是 Bessel 方程的奇异点。Bessel 方程会成为下一章的主角。

现在是核心定理。

普通点的存在性定理。如果 \(x_0\) 是方程 \(y'' + p(x) y' + q(x) y = 0\) 的普通点,则方程在 \(x_0\) 附近存在两个线性独立的幂级数解

\[ y_1(x) = \sum_{n=0}^{\infty} a_n (x - x_0)^n, \qquad y_2(x) = \sum_{n=0}^{\infty} b_n (x - x_0)^n, \]

且它们的收敛半径至少等于 \(p\) 和 \(q\) 在 \(x_0\) 处的幂级数展开的收敛半径中较小的那一个。

这个定理给我们三件好东西:

  1. 存在性。我们用幂级数法去做不会"扑空";级数解一定存在,而且是两个独立的。
  2. 系数被两参数族地决定。\(a_0, a_1\)(也就是 \(y(x_0), y'(x_0)\) 这两个初值)是任意的;一旦给定它们,后面所有 \(a_n\) 都被递推关系确定。
  3. 收敛半径有下界。下一节具体怎么估计这个下界。

定理的证明要用到一些复变知识(把 \(p, q\) 看成复变函数,在复平面上的奇异点),所以我们略去——但定理本身对实变量幂级数完全有效,你可以放心使用。

16.7 收敛半径估计

用了存在性定理我们得到了"\(y\) 的级数收敛半径 \(\geq\) \(p, q\) 的级数收敛半径中较小者"。现在的问题是:\(p, q\) 的级数收敛半径怎么估?

对于有理函数 \(p, q\) (这是绝大多数实际方程的情形),答案极其简洁:

有理函数在 \(x_0\) 处的幂级数收敛半径 \(=\) \(x_0\) 到该函数的最近极点的距离(在复平面上量距离!)。

所以最终的实用判据是:

\[ \boxed{\text{级数解 } y(x) \text{ 在 } x_0 \text{ 附近的收敛半径} \geq \text{从 } x_0 \text{ 到方程最近奇异点(在复平面)的距离}.} \]

"在复平面上量距离"这一点千万不要忽略。一些方程在实数轴上看起来没有奇异点,但其实在虚轴上有,这会限制级数的收敛半径。下面的例子会展示这一点。

例 16.7.1. 估计方程 \((1 + x^2) y'' + 2 x y' - 2 y = 0\) 在 \(x_0 = 0\) 附近级数解的收敛半径。

这里 \(P(x) = 1 + x^2 = 0\) 给出 \(x = \pm i\),也就是奇异点在虚轴上,距离原点 \(1\)。所以级数解在 \(|x| < 1\) 上至少收敛。这是一个绝佳的"复平面奇异点限制实变量级数"的例子:实数 \(x\) 跑到 \(x = 1\) 时方程本身一点也没有发癫,但它附近的复奇异点 \(\pm i\) 还是限制了级数的收敛半径。

例 16.7.2. 估计 Legendre 方程 \((1 - x^2) y'' - 2x y' + n(n+1) y = 0\) 在 \(x_0 = 0\) 附近级数解的收敛半径。

奇异点:\(1 - x^2 = 0\) 给 \(x = \pm 1\)。从 \(0\) 到 \(\pm 1\) 的距离都是 \(1\)。所以级数解收敛半径至少是 \(1\)。事实上当 \(n\) 不是非负整数时这个 \(R\) 恰好就是 \(1\)(也就是说,级数在 \(|x| = 1\) 处发散);当 \(n\) 是非负整数时奇迹发生:级数截断成多项式——也就是 Legendre 多项式 \(P_n(x)\)——这种"截断"方式给出的解就是整个实轴上的多项式。这件事下一章 17.6 节会详细讲。

例 16.7.3. 估计 Airy 方程 \(y'' - xy = 0\) 在 \(x_0 = 0\) 附近的收敛半径。

这里 \(P(x) = 1\) 永远不为零,所以方程没有奇异点。最近奇异点在无穷远。所以级数解的收敛半径是 \(\infty\)——这与我们在 16.5 节直接观察到的"对所有 \(x\) 都收敛"完全一致。

这套估计办法的好处:你不需要先把级数算出来就能预测它的收敛范围。这在做工程估计时极其方便——比如你想用前 50 项级数算 \(y(0.7)\),但不知道能不能算,先看奇异点在哪,如果最近奇异点离 \(0\) 有 \(1\) 那么远,\(0.7\) 在收敛区间里,放心算。

16.8 例题汇总

这一节集中刷四道变系数例题,从简到难,把 16.3 节的"形式做法"打入肌肉记忆。

16.8.1 例:\(y' = y\) 的级数解(健身热身)

它有显然的解 \(y = e^x\),所以这是给级数法做的"对答案"练习。

设 \(y = \sum a_n x^n\),则 \(y' = \sum_{n \geq 1} n a_n x^{n-1} = \sum_{n \geq 0}(n+1) a_{n+1} x^n\)。代入 \(y' - y = 0\):

\[ \sum_{n=0}^{\infty} \big[ (n+1) a_{n+1} - a_n \big] x^n = 0. \]

等系数:\(a_{n+1} = a_n / (n+1)\)。所以 \(a_1 = a_0, a_2 = a_0/2, a_3 = a_0/(3 \cdot 2) = a_0/3!, \ldots, a_n = a_0/n!\)。于是

\[ y(x) = a_0 \sum_{n=0}^{\infty} \frac{x^n}{n!} = a_0 e^x. \]

对了。这道题告诉你:即便是一阶方程,级数法也可以一步把整个解推出来,而且收敛半径是 \(\infty\)。

16.8.2 例:Hermite 方程的预演

看方程 \(y'' - 2x y' + 2 \lambda y = 0\),其中 \(\lambda\) 是常数。这就是Hermite 方程(在量子力学的简谐振子里出现,我们 17.7 节再细讲),以 \(0\) 为普通点。

设 \(y = \sum a_n x^n\),则

\[ y' = \sum_{n=1}^{\infty} n a_n x^{n-1}, \qquad y'' = \sum_{n=2}^{\infty} n(n-1) a_n x^{n-2}. \]

所以 \(2x y' = \sum_{n=1}^{\infty} 2n a_n x^n\),平移 \(y''\) 到 \(\sum_{n=0}^{\infty}(n+2)(n+1) a_{n+2} x^n\)。代入方程:

\[ \sum_{n=0}^{\infty} (n+2)(n+1) a_{n+2} x^n - \sum_{n=1}^{\infty} 2n a_n x^n + \sum_{n=0}^{\infty} 2 \lambda a_n x^n = 0. \]

把 \(n = 0\) 项单独提出来再合并:

\[ \big[ 2 a_2 + 2\lambda a_0 \big] + \sum_{n=1}^{\infty} \big[ (n+2)(n+1) a_{n+2} - 2 n a_n + 2 \lambda a_n \big] x^n = 0. \]

等系数:

\[ a_2 = -\lambda a_0, \qquad a_{n+2} = \frac{2(n - \lambda)}{(n+2)(n+1)} a_n \text{ for } n \geq 1. \]

(注意 \(n = 0\) 那项也满足这个递推:\(a_2 = 2(0 - \lambda)/(2 \cdot 1) \cdot a_0 = -\lambda a_0\),一致。所以可以统一写成 \(a_{n+2} = 2(n-\lambda) a_n / [(n+2)(n+1)]\) 对所有 \(n \geq 0\)。)

这个递推有一个非常迷人的性质:如果 \(\lambda\) 恰好是非负整数 \(N\),那么 \(a_{N+2} = 2(N - N) a_N / [\cdots] = 0\),从而 \(a_{N+4} = 0, a_{N+6} = 0, \ldots\) 整列后续都为零;也就是说,如果 \(N\) 是偶数,从 \(a_0\) 出发的那个级数截断成 \(N\) 次多项式;如果 \(N\) 是奇数,从 \(a_1\) 出发的级数截断成 \(N\) 次多项式。这些多项式经过适当归一化就是Hermite 多项式 \(H_n(x)\)。例如:

  • \(\lambda = 0\):\(a_2 = 0\),从 \(a_0\) 出发只有一项 \(y_1 = 1\)。这就是 \(H_0\)。
  • \(\lambda = 1\):\(a_3 = 0\),从 \(a_1\) 出发是 \(y_2 = x\)。这就是 \(H_1\)(归一化后是 \(2x\))。
  • \(\lambda = 2\):\(a_4 = 0\),从 \(a_0\) 出发是 \(y_1 = 1 - 2x^2\)。这就是 \(H_2\)(归一化后是 \(4x^2 - 2\))。

"参数取整数时级数截断成多项式"是所有经典正交多项式族(Hermite, Legendre, Laguerre, Chebyshev……)共享的关键现象。本章只让你瞥一眼,下一章再细讲。

16.8.3 例:\((1 - x) y' = y\) 的级数解(系数稍微复杂一点)

这是一阶方程,可以用分离变量直接解出来:\(\int dy/y = \int dx/(1 - x)\),得到 \(\ln |y| = -\ln|1-x| + C\),也就是 \(y = K/(1 - x)\)。这是一个对照——我们的级数法应当给出 \(1/(1-x) = \sum x^n\) 的结构。

设 \(y = \sum a_n x^n\)。\(y' = \sum_{n \geq 1} n a_n x^{n-1} = \sum_{n \geq 0}(n+1) a_{n+1} x^n\)。\((1-x) y' = y' - x y' = \sum_{n \geq 0}(n+1) a_{n+1} x^n - \sum_{n \geq 1} n a_n x^n\)。代入方程 \((1-x)y' - y = 0\):

\[ \sum_{n=0}^{\infty} (n+1) a_{n+1} x^n - \sum_{n=1}^{\infty} n a_n x^n - \sum_{n=0}^{\infty} a_n x^n = 0. \]

提出 \(n = 0\) 项,再合并:\([a_1 - a_0] + \sum_{n=1}^{\infty} [(n+1) a_{n+1} - n a_n - a_n] x^n = 0\),也就是

\[ a_1 = a_0, \qquad (n+1) a_{n+1} = (n+1) a_n \text{ for } n \geq 1 \implies a_{n+1} = a_n. \]

(再次注意 \(n = 0\) 也满足:\(a_1 = a_0\)。)所有系数相等:\(a_n = a_0\) 对所有 \(n\)。所以

\[ y(x) = a_0 \sum_{n=0}^{\infty} x^n = \frac{a_0}{1 - x}, \qquad |x| < 1. \]

对了。注意收敛半径是 \(1\),正是 \(p(x) = -1/(1 - x)\) 的奇异点 \(x = 1\) 到原点的距离。

16.8.4 例:\(y'' + x y' + y = 0\) 在 \(x_0 = 0\) 附近的级数解

这道方程在概率论里叫做"Mehler-Hermite 方程"的近亲——它和正态分布的密度有关。现在用形式做法做一遍。

设 \(y = \sum a_n x^n\)。则 \(xy' = \sum_{n=1}^{\infty} n a_n x^n\),\(y'' = \sum_{n=0}^{\infty}(n+2)(n+1) a_{n+2} x^n\)。代入:

\[ \sum_{n=0}^{\infty}(n+2)(n+1) a_{n+2} x^n + \sum_{n=1}^{\infty} n a_n x^n + \sum_{n=0}^{\infty} a_n x^n = 0. \]

对 \(n = 0\):\(2 a_2 + a_0 = 0 \implies a_2 = -a_0/2\)。

对 \(n \geq 1\):\((n+2)(n+1) a_{n+2} + (n + 1) a_n = 0 \implies a_{n+2} = -a_n/(n+2)\)。

(再次,\(n = 0\) 一致。所以 \(a_{n+2} = -a_n/(n+2)\) 对所有 \(n \geq 0\)。)

偶数列:\(a_2 = -a_0/2, a_4 = -a_2/4 = a_0/(2 \cdot 4), a_6 = -a_4/6 = -a_0/(2 \cdot 4 \cdot 6), \ldots\),也就是 \(a_{2k} = (-1)^k a_0 / (2 \cdot 4 \cdots 2k) = (-1)^k a_0 / (2^k k!)\)。

奇数列:\(a_3 = -a_1/3, a_5 = -a_3/5 = a_1/(3 \cdot 5), a_7 = -a_5/7 = -a_1/(3 \cdot 5 \cdot 7), \ldots\),即 \(a_{2k+1} = (-1)^k a_1 /[3 \cdot 5 \cdots (2k+1)] = (-1)^k a_1 \cdot 2^k k! / (2k+1)!\)。

所以两个独立解是

\[ y_1(x) = \sum_{k=0}^{\infty} \frac{(-1)^k}{2^k k!} x^{2k} = \sum_{k=0}^{\infty} \frac{(-x^2/2)^k}{k!} = e^{-x^2/2}, \]

第一个解被认了出来——\(e^{-x^2/2}\)。第二个解

\[ y_2(x) = x - \frac{x^3}{3} + \frac{x^5}{15} - \frac{x^7}{105} + \cdots \]

不是初等函数,但和误差函数 \(\mathrm{erf}\) 相关——具体地,\(y_2(x) = e^{-x^2/2} \int_0^x e^{t^2/2}\, dt\)(用降阶法可以验证;过程留给你做练习)。这道题漂亮地展示了:同一个变系数方程,一个解可以是初等函数,另一个解必须靠级数表达。这种"半初等"现象在变系数方程里其实非常普遍。

方程的系数 \(p(x) = x, q(x) = 1\) 都是多项式(没有奇异点),所以两个级数都对所有 \(x\) 收敛。验证一下 \(y_1\):它就是 \(e^{-x^2/2}\) 的级数,确实在所有 \(x\) 上收敛,完美一致。

16.8.5 例:Chebyshev 方程的初步

最后看一个会出现奇异点的例子——它是变系数例题里"奇异点强加收敛半径限制"的最直观示范。Chebyshev 方程是

\[ (1 - x^2) y'' - x y' + n^2 y = 0, \qquad n \text{ 是非负整数}. \]

奇异点在 \(x = \pm 1\),距离原点 \(1\)。所以以 \(x_0 = 0\) 为中心的级数解收敛半径至少是 \(1\)。

设 \(y = \sum a_n x^n\)。算各项:

\[ (1 - x^2) y'' = \sum_{n=0}^{\infty} (n+2)(n+1) a_{n+2} x^n - \sum_{n=2}^{\infty} n(n-1) a_n x^n, \] \[ -x y' = -\sum_{n=1}^{\infty} n a_n x^n, \qquad n^2 y = \sum_{n=0}^{\infty} n^2 a_n x^n. \]

(我们用 \(n\) 既作下标又作方程里的整参数,确实容易混。让我把那个方程里的整参数叫做 \(\nu\) 来避免混淆,把方程改写成

\[ (1 - x^2) y'' - x y' + \nu^2 y = 0.) \]

合并,等系数,得到

\[ a_{m+2} = \frac{m^2 - \nu^2}{(m+2)(m+1)} a_m, \quad m \geq 0. \]

(自己检验。)与 Hermite 方程一样,如果 \(\nu\) 是非负整数,某一时刻 \(m = \nu\) 让分子变零,从而递推中断,级数截断成多项式。这些多项式归一化以后是Chebyshev 多项式 \(T_\nu(x)\):\(T_0 = 1, T_1 = x, T_2 = 2x^2 - 1, T_3 = 4x^3 - 3x, \ldots\),它们在数值分析中(最佳逼近、伪谱方法)无处不在。

如果 \(\nu\) 不是整数,级数永远不截断,收敛半径就严格等于 \(1\)——级数在 \(x = \pm 1\) 处发散。这就是奇异点对收敛半径的"硬上限"作用的最直白展示。

本章到此结束。我们建立了三件事:(1) 对普通点附近的方程,幂级数解保证存在,而且至少在到最近奇异点的距离内收敛;(2) 找系数完全是机械的,只需指标平移和等系数;(3) 这套方法把许多"我们认识"的特殊函数(\(\sin, \cos, e^x\),Hermite/Chebyshev 多项式,Airy 函数 \(\mathrm{Ai}/\mathrm{Bi}\))整齐地放到同一个框架里。

但还有一个我们绕开了的问题:如果展开点是奇异点呢?Bessel 方程在 \(0\) 处奇异、Legendre 方程在 \(\pm 1\) 处奇异——而物理问题恰恰会问"在那些点附近的解长什么样"。这就需要一个能处理奇异点的级数方法,叫做 Frobenius 方法。它是下一章的主题,也是把"特殊函数"完整入袋的最后一关。
