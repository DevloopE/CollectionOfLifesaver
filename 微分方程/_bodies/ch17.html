第 17 章 Frobenius 方法与特殊函数

第 16 章里我们建立了一个很满意的事实:在普通点附近,二阶线性变系数方程总有两个独立的幂级数解,机械流程就能算出来。可是仔细一看,物理学最重要的几个方程——氢原子的径向方程、圆鼓的振动方程、行星轨道的扰动方程——它们的"自然展开点"恰好是奇异点。比如 Bessel 方程

\[ x^2 y'' + x y' + (x^2 - \nu^2) y = 0 \]

需要在 \(x = 0\) 附近研究,因为 \(x\) 是径向距离,\(0\) 是最重要的物理位置("极点"那里);Legendre 方程的关键展开点 \(\pm 1\) 也是它的奇异点。如果硬把幂级数 \(y = \sum a_n x^n\) 代进 Bessel 方程,马上会出问题——你将看到从 \(a_0\) 到 \(a_2\) 之间多了一个跨不过去的鸿沟,递推关系给不出任何信息(下面 17.1 节展示)。所以我们需要一个升级版的级数方法,允许解里有"非整数次的 \(x\)"和"\(\ln x\) 项"。这就是 Frobenius 方法。

本章会:

  • 展示为什么普通幂级数法在奇异点会失败;
  • 定义"正则奇异点",给出"哪些奇异点能被 Frobenius 救回来";
  • 引入 Frobenius 试解 \(y = x^r \sum a_n x^n\),推导指标方程;
  • 讨论指标方程两根 \(r_1, r_2\) 的三种情形,以及第二解何时含 \(\ln x\);
  • 详细推演 Bessel 方程,得到 \(J_\nu, Y_\nu\) 的级数表达,并简述物理应用(圆鼓振动、圆柱波导);
  • 详细推演 Legendre 方程,得到 Legendre 多项式 \(P_n(x)\) 与它在球面问题中的角色;
  • 速通 Hermite 与 Laguerre 方程(量子力学的常客);
  • 用一段哲学性的小节回答"这些特殊函数为什么特殊"。

17.1 普通幂级数在奇异点为何失败

我们用 Bessel 方程 \(\nu = 0\) 的最简单情形来看故障是怎么发生的:

\[ x^2 y'' + x y' + x^2 y = 0. \]

把它写成标准形式:除以 \(x^2\),\(y'' + (1/x) y' + y = 0\)。\(p(x) = 1/x\) 在 \(0\) 处发散,所以 \(0\) 是奇异点。先按 16 章的套路设 \(y = \sum_{n=0}^{\infty} a_n x^n\)、试一试。

\(y' = \sum_{n=1}^{\infty} n a_n x^{n-1}, y'' = \sum_{n=2}^{\infty} n(n-1) a_n x^{n-2}\)。代入原方程 \(x^2 y'' + xy' + x^2 y = 0\):

\[ \sum_{n=2}^{\infty} n(n-1) a_n x^n + \sum_{n=1}^{\infty} n a_n x^n + \sum_{n=0}^{\infty} a_n x^{n+2} = 0. \]

整理一下,把第三项指标 \(m = n + 2\) 平移成 \(\sum_{m=2}^{\infty} a_{m-2} x^m\),然后合并:

\[ \big[ a_1 \cdot x \big] + \sum_{n=2}^{\infty} \big[ n(n-1) a_n + n a_n + a_{n-2} \big] x^n = 0. \]

(\(n = 0\) 项消失了,因为前两个求和都不贡献,而第三项也是从 \(n = 2\) 起的。)等系数:

\[ a_1 = 0, \quad n^2 a_n + a_{n-2} = 0 \implies a_n = -\frac{a_{n-2}}{n^2}, \quad n \geq 2. \]

到这里听起来像 16 章一样顺利:递推关系出来了,从 \(a_0\) 出发就能逐项算 \(a_2, a_4, a_6, \ldots\),从 \(a_1 = 0\) 出发奇数下标全是零……可是这只给我们一个解,不是两个!整个方程是二阶的,我们需要两个独立解,但普通幂级数只给了我们一个(从 \(a_0\) 出发的那个),另一个(从 \(a_1\) 出发的)被强行设成了零。换句话说,我们丢失了一个自由度。

这是奇异点处普通幂级数法的典型失败模式:它只给出一个解,而那个解里的另一个独立函数往往含 \(\ln x\) 或非整数次的 \(x\),根本不是普通幂级数能表示的。我们需要扩大候选答案的范围。

17.2 正则奇异点

不是所有奇异点都能被救回来——有些奇异点太"野"了,任何级数方法都不行。Frobenius 方法只对一类温和的奇异点起作用,叫做正则奇异点 (regular singular point)。

定义。把方程写成 \(y'' + p(x) y' + q(x) y = 0\)。点 \(x_0\) 是正则奇异点,如果它是奇异点(\(p\) 或 \(q\) 在 \(x_0\) 处不解析),但

\[ (x - x_0) p(x) \quad \text{和} \quad (x - x_0)^2 q(x) \]

都在 \(x_0\) 处解析。也就是,\(p\) 在 \(x_0\) 处至多有一阶极点,\(q\) 至多有二阶极点。

"正则"这个词的含义是:奇异性虽然存在,但被以"恰当的代数比例"包住——\(p\) 配 \(1/(x - x_0)\)、\(q\) 配 \(1/(x - x_0)^2\),正好是 Frobenius 试解 \(y = x^r \sum a_n x^n\) 能消化的那种比例。

不是这种比例的奇异点叫做不规则奇异点 (irregular singular point),Frobenius 方法对它们无效;那种情况要靠"渐近级数"或其他更深的工具,本书不涉及。

例 17.2.1. Bessel 方程 \(x^2 y'' + xy' + (x^2 - \nu^2) y = 0\)。标准形式:\(y'' + (1/x) y' + (1 - \nu^2/x^2) y = 0\)。\(p = 1/x\) 在 \(0\) 处一阶极点,\(q = 1 - \nu^2/x^2\) 在 \(0\) 处二阶极点。\(xp = 1\) 解析,\(x^2 q = x^2 - \nu^2\) 解析。所以 \(x = 0\) 是 Bessel 方程的正则奇异点。

例 17.2.2. Legendre 方程 \((1-x^2)y'' - 2xy' + n(n+1)y = 0\)。标准形式:\(y'' - \frac{2x}{1-x^2} y' + \frac{n(n+1)}{1-x^2} y = 0\)。在 \(x = 1\) 处:\(p = -2x/(1-x^2) = -2x/[(1-x)(1+x)]\) 一阶极点;\(q\) 一阶极点(没到二阶)。\((x-1)p\) 与 \((x-1)^2 q\) 都解析。所以 \(\pm 1\) 都是正则奇异点。

例 17.2.3. 方程 \(x^3 y'' + y = 0\)。标准形式 \(y'' + (1/x^3) y = 0\),\(q = 1/x^3\) 在 \(0\) 处三阶极点。\(x^2 q = 1/x\) 仍不解析。所以 \(x = 0\) 是不规则奇异点。Frobenius 方法对此无效。

本章后面所有的奇异点都将是正则的。

17.3 Frobenius 方法与指标方程

核心想法。如果 \(x_0\) 是正则奇异点,试解为

\[ y(x) = (x - x_0)^r \sum_{n=0}^{\infty} a_n (x - x_0)^n = \sum_{n=0}^{\infty} a_n (x - x_0)^{n+r}, \]

其中 \(r\) 是待定的实数(或复数,但本书全用实情形),不必是整数。

方便起见,以下设 \(x_0 = 0\)(总可以平移),所以试解就是

\[ y(x) = x^r \sum_{n=0}^{\infty} a_n x^n = \sum_{n=0}^{\infty} a_n x^{n+r}, \quad a_0 \neq 0. \]

注意"\(a_0 \neq 0\)"这个约定:它确保 \(x^r\) 是 \(y\) 的"领头幂",而不会被一个 \(a_0 = 0\) 偷偷地变成 \(x^{r+1}\) 的领头幂(否则 \(r\) 就含糊了)。

把它代进方程会发生什么?在 \(x = 0\) 附近,把方程乘以 \(x^2\) 改写成

\[ x^2 y'' + x \cdot [x p(x)] y' + [x^2 q(x)] y = 0, \]

这里 \(P_0(x) := xp(x), Q_0(x) := x^2 q(x)\) 都解析(因为 \(0\) 是正则奇异点)。把它们展成幂级数 \(P_0(x) = p_0 + p_1 x + p_2 x^2 + \cdots\),\(Q_0(x) = q_0 + q_1 x + q_2 x^2 + \cdots\)。

试解 \(y = x^r \sum a_n x^n\) 给出

\[ y' = \sum_{n=0}^{\infty}(n+r) a_n x^{n+r-1}, \quad y'' = \sum_{n=0}^{\infty}(n+r)(n+r-1) a_n x^{n+r-2}, \]

所以

\[ x^2 y'' = \sum_{n=0}^{\infty}(n+r)(n+r-1) a_n x^{n+r}, \quad x y' = \sum_{n=0}^{\infty}(n+r) a_n x^{n+r}. \]

代入:

\[ \sum_{n=0}^{\infty}(n+r)(n+r-1) a_n x^{n+r} + P_0(x) \sum_{n=0}^{\infty}(n+r) a_n x^{n+r} + Q_0(x) \sum_{n=0}^{\infty} a_n x^{n+r} = 0. \]

整体除以 \(x^r\)。看 \(x^0\) 项的系数(也就是最低次那一项的系数)——它来自 \(n = 0\) 和 \(P_0, Q_0\) 的常数项 \(p_0, q_0\):

\[ \big[ r(r - 1) + p_0 r + q_0 \big] a_0 = 0. \]

因为 \(a_0 \neq 0\),括号必须为零:

\[ \boxed{r(r - 1) + p_0 r + q_0 = 0.} \]

这就是指标方程 (indicial equation)。它是关于 \(r\) 的二次方程,通常给出两个根 \(r_1, r_2\)(实数情形约定 \(r_1 \geq r_2\))。它的根告诉你"\(y\) 在 \(0\) 附近以哪个幂次起步"。

关键事实:

  • 对每个根 \(r_i\),把它代回到 \(x^{n+r}\)(\(n \geq 1\))那些系数为零的条件,会给出一个递推关系,从而决定 \(a_1, a_2, \ldots\)。这样我们就构造出一个"以 \(r_i\) 起步的"Frobenius 解 \(y_i = x^{r_i} \sum a_n^{(i)} x^n\)。
  • 能否得到两个独立解,以及第二解是否含 \(\ln x\) 项,完全取决于 \(r_1, r_2\) 之间的关系。这就是下一节的三种情形。

17.4 三种情形

设指标方程的两根为 \(r_1 \geq r_2\)(实数;复根的情形本书略去)。下面是 Frobenius 方法的"三大教条":

情形 I:\(r_1 - r_2\) 不是整数(包括 \(r_1 - r_2\) 是非整数有理数或无理数)。两个独立 Frobenius 解

\[ y_1(x) = x^{r_1} \sum_{n=0}^{\infty} a_n x^n, \qquad y_2(x) = x^{r_2} \sum_{n=0}^{\infty} b_n x^n, \quad a_0, b_0 \neq 0, \]

都存在,系数分别由对 \(r = r_1, r_2\) 的递推算出。这是最舒服的情形——两个 \(r\) 根毫无干扰地各搞各的。

情形 II:\(r_1 = r_2 = r\)(重根)。第一个解

\[ y_1(x) = x^r \sum_{n=0}^{\infty} a_n x^n \]

用 Frobenius 方法照做。第二个解必须含 \(\ln x\):

\[ y_2(x) = y_1(x) \ln x + x^r \sum_{n=0}^{\infty} c_n x^n. \]

(系数 \(c_n\) 也由代入方程逐项决定;通常也设 \(c_0 = 0\) 以避免被 \(y_1\) 吞掉。)

情形 III:\(r_1 - r_2\) 是正整数(且 \(r_1 \neq r_2\))。第一个解 \(y_1(x) = x^{r_1} \sum a_n x^n\) 总能用 Frobenius 方法做出。第二个解可能是

\[ y_2(x) = x^{r_2} \sum_{n=0}^{\infty} b_n x^n \]

(普通的 Frobenius 解,不带 \(\ln\)),也可能必须带 \(\ln\) 项

\[ y_2(x) = K \cdot y_1(x) \ln x + x^{r_2} \sum_{n=0}^{\infty} c_n x^n, \]

其中 \(K\) 可能等于 \(0\),也可能等于某个非零常数。具体哪个,要看代入过程中"卡壳那一项"是否真的卡死。下面 Bessel 方程的整数 \(\nu\) 情形会展示这种情况。

这三种情形的判别非常机械。计算流程:

  1. 把方程化标准形式;
  2. 验明 \(0\) 是正则奇异点;
  3. 读出 \(p_0 = \lim_{x \to 0} xp(x), q_0 = \lim_{x \to 0} x^2 q(x)\),写指标方程;
  4. 解 \(r_1 \geq r_2\),判断属于哪一情形;
  5. 对 \(r_1\) 跑递推,得 \(y_1\);
  6. 按情形 I/II/III 处理 \(y_2\)。

下面把这一切套到 Bessel 方程上去——它会同时展示情形 I 和情形 III。

17.5 Bessel 方程

Bessel 方程是数学物理里出现频率仅次于"\(y'' + y = 0\)"的方程,大概排第二。它是

\[ x^2 y'' + x y' + (x^2 - \nu^2) y = 0, \]

其中 \(\nu \geq 0\) 是参数。这个方程会在你做柱坐标下的拉普拉斯方程或波动方程时自动冒出来——不是哪个伟人灵光一闪发明的,而是几何强加的。

17.5.1 指标方程

标准形式:\(y'' + (1/x) y' + (1 - \nu^2/x^2) y = 0\)。\(p_0 = \lim x \cdot (1/x) = 1, q_0 = \lim x^2 \cdot (1 - \nu^2/x^2) = -\nu^2\)。指标方程:

\[ r(r - 1) + r - \nu^2 = 0 \implies r^2 = \nu^2 \implies r = \pm \nu. \]

所以 \(r_1 = \nu, r_2 = -\nu\)。差是 \(2\nu\)。

判别情形:

  • 如果 \(2\nu\) 不是整数(即 \(\nu\) 既非整数也非半整数 \(1/2, 3/2, \ldots\)),情形 I,两个独立 Frobenius 解。
  • 如果 \(\nu = 0\),情形 II,重根 \(r = 0\),第二解必含 \(\ln\)。
  • 如果 \(\nu = 1, 2, 3, \ldots\) 是正整数,情形 III,第二解通常需要 \(\ln\) 项。
  • 如果 \(\nu = 1/2, 3/2, \ldots\) 是半整数,看似情形 III(差是整数),但不需要 \(\ln\)——这是情形 III 中"幸运的子情形"(\(K = 0\))。

17.5.2 第一解 \(J_\nu(x)\)

用 \(r = \nu\) 设 \(y = x^\nu \sum_{n=0}^{\infty} a_n x^n = \sum_{n=0}^{\infty} a_n x^{n+\nu}\)。代入方程,经过一段熟练的"算各项 \rightarrow 合并 \rightarrow 等系数"的工作(细节相当于 17.1 节的复杂版),会得到 \(a_1 = 0\) 以及递推

\[ a_n = -\frac{a_{n-2}}{n(n + 2\nu)}, \quad n \geq 2. \]

奇数下标全为零(因为 \(a_1 = 0\) 且每往后跳两步)。偶数下标:\(a_{2k} = -a_{2k-2}/[(2k)(2k+2\nu)] = -a_{2k-2}/[2^2 k(k + \nu)]\)。逐次递推:

\[ a_{2k} = \frac{(-1)^k a_0}{2^{2k} k! (1 + \nu)(2 + \nu) \cdots (k + \nu)}. \]

用 \(\Gamma\) 函数把分母写紧凑:\((1+\nu)(2+\nu)\cdots(k+\nu) = \Gamma(k + \nu + 1)/\Gamma(\nu + 1)\)。约定 \(a_0 = 1/[2^\nu \Gamma(\nu + 1)]\)(这是历史选定的归一化),得到

\[ \boxed{J_\nu(x) = \sum_{k=0}^{\infty} \frac{(-1)^k}{k!\, \Gamma(k + \nu + 1)} \left( \frac{x}{2} \right)^{2k + \nu}.} \]

这就是第一类 Bessel 函数。它是 Bessel 方程在 \(0\) 附近的一个特定解,对所有 \(x \geq 0\) 收敛。如果你在科学计算软件里调用 besselj(nu, x),出来的就是它。

对 \(\nu = 0\),前几项:

\[ J_0(x) = 1 - \frac{x^2}{4} + \frac{x^4}{64} - \frac{x^6}{2304} + \cdots \]

对 \(\nu = 1\):

\[ J_1(x) = \frac{x}{2} - \frac{x^3}{16} + \frac{x^5}{384} - \cdots \]

从图像上看,\(J_\nu(x)\) 像一个"被慢慢压低的振荡函数"——有点像 \(\sin x / \sqrt{x}\) 那种感觉。事实上,\(J_\nu\) 在大 \(x\) 时的渐近形式正是

\[ J_\nu(x) \sim \sqrt{\frac{2}{\pi x}} \cos\left( x - \frac{\nu \pi}{2} - \frac{\pi}{4} \right). \]

它告诉你为什么 Bessel 函数在物理上扮演的角色像"圆柱坐标里的余弦"——大半径处它就是个被衰减的余弦波。

17.5.3 第二解 \(Y_\nu(x)\)

对非整数 \(\nu\),把上面整个过程重做一遍但用 \(r = -\nu\),会得到

\[ J_{-\nu}(x) = \sum_{k=0}^{\infty} \frac{(-1)^k}{k!\, \Gamma(k - \nu + 1)} \left( \frac{x}{2} \right)^{2k - \nu}. \]

这是 Bessel 方程的另一个独立解。所以 \(\nu\) 非整数时,通解是 \(c_1 J_\nu + c_2 J_{-\nu}\)。

但是 \(\nu = n\) 是非负整数时事情变糟:可以证明 \(J_{-n}(x) = (-1)^n J_n(x)\),也就是说 \(J_{-n}\) 和 \(J_n\) 不再独立!这正是情形 III 在我们眼前展开:第二个 Frobenius 解失效,我们必须含 \(\ln\) 项地构造第二解。

统一处理两种情况(整数和非整数)的标准定义是第二类 Bessel 函数:

\[ Y_\nu(x) = \frac{J_\nu(x) \cos(\nu \pi) - J_{-\nu}(x)}{\sin(\nu \pi)}. \]

对非整数 \(\nu\),这只是 \(J_\nu\) 与 \(J_{-\nu}\) 的某个特定线性组合,所以与 \(J_\nu\) 独立。对整数 \(\nu = n\),分子分母都是零,需要取极限——结果含 \(\ln\) 项,正是情形 III 所要求的形式。例如 \(Y_0\) 在 \(0\) 附近

\[ Y_0(x) = \frac{2}{\pi}\left[ \ln(x/2) + \gamma \right] J_0(x) + \frac{2}{\pi} \sum_{k=1}^{\infty} \frac{(-1)^{k+1} H_k}{(k!)^2} \left(\frac{x}{2}\right)^{2k}, \]

其中 \(\gamma\) 是欧拉-马歇罗尼常数,\(H_k = 1 + 1/2 + \cdots + 1/k\)。注意领头的 \(\ln\) 项——\(Y_0\) 在 \(x \to 0^+\) 时像 \(\ln x\) 那样发散。这是为什么物理上"在原点处必须有限"的边界条件会强行迫使 \(Y_n\) 的系数为零,只剩 \(J_n\)。

所以 Bessel 方程的通解(对所有 \(\nu \geq 0\))是

\[ y(x) = c_1 J_\nu(x) + c_2 Y_\nu(x). \]

17.5.4 物理应用:圆鼓振动

为什么这一切值得关心?最经典的例子:圆鼓的振动。设一张半径 \(R\) 的圆膜,固定边缘,初始位置和速度给定。膜的位移 \(u(r, \theta, t)\) 满足二维波动方程

\[ \frac{\partial^2 u}{\partial t^2} = c^2 \nabla^2 u = c^2 \left( \frac{1}{r} \frac{\partial}{\partial r}\Big( r \frac{\partial u}{\partial r} \Big) + \frac{1}{r^2} \frac{\partial^2 u}{\partial \theta^2} \right). \]

用分离变量 \(u = T(t) F(r) G(\theta)\),会得到 \(G\) 满足简单的 \(G'' + m^2 G = 0\)(给出 \(\sin m\theta, \cos m\theta\)),\(T\) 满足 \(T'' + \omega^2 T = 0\)(给出时间振荡),而 \(F\) 满足

\[ r^2 F'' + r F' + (k^2 r^2 - m^2) F = 0, \]

其中 \(k = \omega/c\)。换变量 \(x = kr\),方程变成整数阶 Bessel 方程

\[ x^2 F'' + x F' + (x^2 - m^2) F = 0. \]

所以 \(F(r) = J_m(kr)\) 或 \(Y_m(kr)\)。物理要求 \(u\) 在 \(r = 0\)(圆心)处有限,所以 \(Y_m\) 那部分必须扔掉(它在 \(0\) 处发散像 \(\ln r\) 或 \(r^{-m}\)),只剩 \(F(r) = J_m(kr)\)。

边界 \(r = R\) 处膜固定,\(u = 0\),所以 \(J_m(kR) = 0\)。这把 \(k\)(从而 \(\omega\))量子化:它们对应 \(J_m\) 的零点。\(J_0\) 的第一个正零点 \(\approx 2.405\),这就是直径为 \(R\) 的鼓的最低振动频率(有名的"基音")。整张鼓的振动模式就是各种 \(J_m(kr) \cos(m\theta) \cos(\omega t)\) 的叠加,所谓"鼓的合声"——这一切的代数核心,就是 Bessel 方程的两个 Frobenius 解和"原点有限"的物理选取。

17.5.5 物理应用:圆柱波导

类似地,在圆形截面的电磁波导(微波传输线)里,电磁场的横向分布满足

\[ \nabla_\perp^2 \psi + k_c^2 \psi = 0, \]

在柱坐标下分离变量,径向部分又是 Bessel 方程。波导内允许的"模式"对应 \(J_m\) 或 \(J_m'\) 的零点(取决于电模或磁模的边界条件),从而决定截止频率。WiFi 路由器、雷达、毫米波 5G 设备——这一切的工程参数都来自 Bessel 函数零点表。

所以 Bessel 方程绝不是抽象数学的玩具:它写在你客厅墙后面那台无线路由器的设计文档里。

17.6 Legendre 方程

第二大客户。Legendre 方程是

\[ (1 - x^2) y'' - 2 x y' + n(n + 1) y = 0, \]

其中 \(n\) 是参数(不必整数,但物理感兴趣的几乎都是非负整数)。它的关键奇异点是 \(\pm 1\)(从 \(1 - x^2\) 来),而 \(0\) 是普通点。我们先在 \(0\) 附近做幂级数,然后讨论 \(n\) 整数时的"截断奇迹"。

17.6.1 在 \(0\) 附近的级数解

(技术上 \(0\) 是普通点,所以这一段是 16 章的方法,不需要 Frobenius。但既然 Legendre 方程是本章主角,我们把它放这里。)

设 \(y = \sum_{k=0}^{\infty} a_k x^k\)。代入,做指标平移和合并,得到递推

\[ a_{k+2} = \frac{k(k+1) - n(n+1)}{(k+2)(k+1)} a_k = \frac{(k - n)(k + n + 1)}{(k+2)(k+1)} a_k. \]

(自己检验。)

所以两个独立解

\[ y_1(x) = 1 + \sum_{\text{偶 } k \geq 2}(\cdots)x^k, \qquad y_2(x) = x + \sum_{\text{奇 } k \geq 3}(\cdots)x^k. \]

关键观察:如果 \(n\) 是非负整数,那么递推分子里的 \((k - n)\) 在 \(k = n\) 时变零,从而 \(a_{n+2} = 0\),整列从 \(a_n\) 之后截断。结果:

  • \(n\) 偶数:\(y_1\) 是 \(n\) 次多项式(\(y_2\) 是无穷级数);
  • \(n\) 奇数:\(y_2\) 是 \(n\) 次多项式(\(y_1\) 是无穷级数)。

这些多项式解,经过规范化让 \(P_n(1) = 1\),叫做Legendre 多项式。前几个:

\[ P_0(x) = 1, \quad P_1(x) = x, \quad P_2(x) = \frac{1}{2}(3x^2 - 1), \quad P_3(x) = \frac{1}{2}(5x^3 - 3x), \quad P_4(x) = \frac{1}{8}(35 x^4 - 30 x^2 + 3). \]

它们对所有实 \(x\) 收敛(就是多项式呀!),包括奇异点 \(x = \pm 1\)。这是"截断"的另一个回报:不仅得到了简单的有限和形式,还把奇异点带来的发散完全规避了。

对应的"另一个解"(那个无穷级数,不截断的那个)叫做第二类 Legendre 函数 \(Q_n(x)\),它在 \(x = \pm 1\) 处发散像 \(\ln\)。在涉及球面的物理问题中,要求"在两极有限"会立刻把 \(Q_n\) 扔掉,只留下 \(P_n\)。

17.6.2 物理应用:球面问题

为什么是这一族多项式?把一个静电势 \(\phi\) 的拉普拉斯方程 \(\nabla^2 \phi = 0\) 写在球坐标 \((r, \theta, \varphi)\) 下,做分离变量 \(\phi = R(r) \Theta(\theta) \Phi(\varphi)\),角度部分会得到一个方程,如果设 \(x = \cos\theta\),它就恰好变成 Legendre 方程:

\[ \frac{d}{dx}\Big[ (1 - x^2) \frac{d\Theta}{dx} \Big] + n(n+1) \Theta = 0. \]

(这就是你为什么在球面上做势能展开会自动看到 \(P_n(\cos\theta)\) 出现:它是球面上的"自然函数"。)

因此,绕一个带电球壳之外的电势可以写成

\[ \phi(r, \theta) = \sum_{n=0}^{\infty} \left( A_n r^n + B_n r^{-(n+1)} \right) P_n(\cos\theta). \]

这就是著名的多极展开:\(n = 0\) 项是单极(\(\sim 1/r\)),\(n = 1\) 项是偶极,\(n = 2\) 项是四极,等等。\(P_n(\cos\theta)\) 不是凑出来的——它是球对称问题里不能不出现的角度依赖。

类似地,在量子力学里,氢原子的角动量本征函数 \(Y_l^m(\theta, \varphi)\)(球谐函数)是 \(P_l\) 的"亲戚"——更确切说是 associated Legendre 函数 \(P_l^m\) 乘 \(e^{im\varphi}\)。所以中学化学里"\(s, p, d, f\) 轨道的形状"——那些花瓣状的 \(2p_x, 3d_{xy}\)——在数学上就是 \(P_l^m(\cos\theta) e^{im\varphi}\) 的图。

17.7 Hermite 与 Laguerre 方程

这两个方程在量子力学里频繁登场。我们已经在 16.8.2 节预演过 Hermite 方程的级数解了,这里把两者一起做一个对照速通。

17.7.1 Hermite 方程

方程:\(y'' - 2 x y' + 2 \lambda y = 0\)。它的所有点都是普通点(系数都是多项式)。在 16.8.2 节我们已经看到:递推 \(a_{n+2} = 2(n - \lambda) a_n / [(n+2)(n+1)]\) 在 \(\lambda = N\) 整数时让某一列截断,得到 \(N\) 次多项式——归一化后是Hermite 多项式 \(H_n(x)\)。前几个:

\[ H_0 = 1, \quad H_1 = 2x, \quad H_2 = 4x^2 - 2, \quad H_3 = 8x^3 - 12 x, \quad H_4 = 16 x^4 - 48 x^2 + 12. \]

物理出处:量子力学的一维谐振子。哈密顿量 \(\hat{H} = \frac{\hat{p}^2}{2m} + \frac{1}{2} m \omega^2 \hat{x}^2\),薛定谔方程 \(\hat{H} \psi = E \psi\) 经过去单位化 \(x \to \xi\),变成

\[ -\psi'' + \xi^2 \psi = \varepsilon \psi. \]

设 \(\psi = e^{-\xi^2/2} u(\xi)\),代入,\(u\) 满足 Hermite 方程 \(u'' - 2\xi u' + (\varepsilon - 1) u = 0\),即 \(\lambda = (\varepsilon - 1)/2\)。物理要求 \(\psi \to 0\) 在 \(\xi \to \pm \infty\),这等价于要求 \(u\) 的级数截断(否则 \(u\) 会指数增长压不住 \(e^{-\xi^2/2}\))。截断的条件:\(\lambda\) 必须是非负整数 \(n\),即 \(\varepsilon = 2n + 1\),也就是

\[ E_n = \hbar \omega \left( n + \tfrac{1}{2} \right). \]

这就是著名的"等距能级"。每一个能级 \(n\) 的本征函数是 \(\psi_n(\xi) \propto H_n(\xi) e^{-\xi^2/2}\)——也就是说,Hermite 多项式是谐振子的本征函数。Hermite 多项式的"截断条件"和谐振子的"能级量子化"是同一件事。这是数学和物理之间最深刻的"翻译"之一。

17.7.2 Laguerre 方程

方程:\(x y'' + (1 - x) y' + \lambda y = 0\)(这是简单形式;广义 Laguerre 方程多带一个参数 \(\alpha\))。\(0\) 是正则奇异点,所以是 Frobenius 题。指标方程给出 \(r = 0, 0\)(重根,情形 II)。第一解通过 \(r = 0\) 的递推得到。

设 \(y = \sum_{n \geq 0} a_n x^n\),代入,得到

\[ a_{n+1} = \frac{n - \lambda}{(n+1)^2} a_n. \]

(和 Hermite 一样,这里也会出现"截断奇迹":\(\lambda = N\) 非负整数时,\(a_{N+1} = 0\),级数截断成 \(N\) 次多项式。)

归一化后这些多项式是Laguerre 多项式 \(L_n(x)\)。前几个:

\[ L_0 = 1, \quad L_1 = 1 - x, \quad L_2 = \frac{1}{2}(x^2 - 4x + 2), \quad L_3 = \frac{1}{6}(-x^3 + 9x^2 - 18 x + 6). \]

物理出处:氢原子的径向方程。把氢的薛定谔方程在球坐标下分离变量,角度部分给球谐函数(用到 Legendre 那边),径向方程经过几步代换变成 Laguerre 方程(确切说是广义 Laguerre 方程)。截断条件给出主量子数 \(n\),从而能级

\[ E_n = -\frac{13.6\, \mathrm{eV}}{n^2}. \]

所以氢原子能级公式 \(E_n = -E_1/n^2\) 这一个高中物理课就背的结果——背后藏的是 Laguerre 方程级数解的截断。

"特殊函数家族"的一个核心模式开始浮现:把一个二阶线性方程(可能含奇异点)做级数解,然后要求"在某个无穷点不爆炸",这个要求迫使参数取整数值,从而级数截断成多项式。这一族多项式就是该物理系统的本征函数,而那些整数就是量子数。

Hermite \(\to\) 谐振子能级 \(n + 1/2\);Laguerre \(\to\) 氢能级 \(n\);Legendre \(\to\) 角动量量子数 \(\ell\);(广义)Laguerre \(\times\) (associated)Legendre \(\to\) 整个氢原子的本征态。这种"一族特殊函数 \(\Leftrightarrow\) 一类本征值问题"的对应是数学物理的核心机制。

17.8 这些"特殊函数"为什么特殊?

本章的最后一节是哲学性的——但它是这一切的"为什么"。

读到这里你可能有两个疑问:

  1. 为什么是这些方程?Bessel、Legendre、Hermite、Laguerre,为什么宇宙偏好它们?
  2. 为什么物理学的几乎每一个角落都在用同样的几个特殊函数?明明数学上有无穷多个二阶线性 ODE,为什么实际只见这五六种?

答案是同一个:对称性。

更具体地:把一个偏微分方程(波动方程、薛定谔方程、拉普拉斯方程……)写在带某种对称性的坐标系下,用分离变量法剥离时间和角度依赖,你不可避免地会得到几个"经典的二阶线性 ODE"。

  • 把拉普拉斯方程 \(\nabla^2 u = 0\) 放到柱坐标(\(r, \theta, z\))做分离变量,径向方程是 Bessel 方程。
  • 把 \(\nabla^2 u = 0\) 放到球坐标 \((r, \theta, \varphi)\) 做分离变量,角度方程(关于 \(\theta\))是 Legendre 方程。
  • 把谐振子的薛定谔方程化简,得到 Hermite 方程。
  • 把氢原子的径向薛定谔方程化简,得到 Laguerre 方程。
  • 把高维球面(\(S^n\) 上)的拉普拉斯方程做调和分析,得到 Gegenbauer 方程(Legendre 的高维亲戚)。

每一类"特殊函数"对应于某种对称性(平移、旋转、轴对称、球对称、SO(3) 群、SO(n) 群、SU(2) 群、……)留下的"自然函数"。物理学之所以反复出现这几个,不是因为造物主偏爱它们,而是因为时空本身就有这些对称性。给定一个有圆对称的物理问题(圆鼓、圆柱波导、原子的轨道角动量),你不可能避开 Bessel 或 Legendre。

这给了"特殊函数"一个统一的看法:它们是对称群表示理论在初等水平上的化身。具体到数学家的语言:

  • Bessel 函数 \(J_\nu\) 是欧氏群 \(E(2)\) 的不可约表示矩阵元。
  • Legendre 多项式 \(P_n(\cos\theta)\)、球谐 \(Y_l^m\) 是 SO(3) 的不可约表示矩阵元。
  • Hermite 多项式 \(H_n\) 是 Heisenberg 群表示的本征函数。
  • Jacobi 多项式涵盖几乎所有"经典正交多项式",对应 SU(2) 的表示理论。

这是另一本书(表示论、调和分析)的故事,但你应当知道:你在本章学到的"幂级数解 + 截断成多项式 + 命名"这套机器,实际上是更深一层的代数结构(对称群)在 ODE 层面留下的脚印。

从最实用的角度说,本章给了你两件东西。一是方法:遇到二阶线性变系数方程时,先验明展开点是普通点还是正则奇异点,然后或者用幂级数,或者用 Frobenius;遇到奇异点处含 \(\ln\) 的情形,知道那是情形 II 或 III 的标志。二是家谱:Bessel \(\to\) 圆柱、Legendre \(\to\) 球面、Hermite \(\to\) 谐振子、Laguerre \(\to\) 氢原子,这套对应让你在以后的物理课上看到 \(J_0, P_n, H_n, L_n\) 时不会慌。它们不是天上掉下来的,它们就是几个微分方程的级数解,而那些方程是几何对称强加的。

下一章我们将彻底告别"找解的形式"这个主题,转向定性的看待 ODE 的方法:相平面、平衡点、稳定性。给定一个非线性方程 \(x'' = -\sin x\)(单摆),我们不会再问"它的解是什么级数",而是问"它的轨道在 \((x, x')\) 平面上长什么样"。这是非线性动力系统的入口。
