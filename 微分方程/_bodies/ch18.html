第 18 章 偏微分方程总览

到这一章为止,本书的整个故事都发生在常微分方程(ODE)的领域里——未知量是一个只依赖于一个自变量的函数,通常那个自变量是时间 \(t\)。一个粒子的位置 \(x(t)\)、一个电路里的电流 \(I(t)\)、一个细菌种群的大小 \(N(t)\)——所有这些都是单变量函数。导数都是普通的导数,方程都长着 \(\dot{x} = f(x, t)\) 或者 \(\ddot{x} + \omega^2 x = 0\) 这种形态。我们用了二十多页讲解了一阶线性、可分离、恰当方程、二阶线性、拉普拉斯变换、级数解、相平面、稳定性。整个工具箱都对准了"时间演化"这一件事。

但现实里很少有东西只随时间变化。一根金属棒被加热的时候,它每一处的温度都不同,且每一处的温度都同时随时间变化——所以温度是 \(u(x, t)\),依赖于位置 \(x\) 和时间 \(t\) 两个自变量。一根弹奏中的吉他弦,它在位置 \(x\) 处的位移 \(u(x, t)\) 也是位置和时间的函数。一片金属板上的稳态温度分布 \(u(x, y)\) 依赖于两个空间坐标。一团扩散的烟、一只飞行的飞机周围的气流、一颗恒星内部的辐射场——这些物理对象天然要求多变量函数。它们的"运动方程"于是就不再是 ODE,而是偏微分方程(partial differential equation, 简称 PDE)。

本章的任务是把这道门推开。我们不打算在这一章里求解任何 PDE——求解要等到第 19、20 章。这一章只做三件事:第一,告诉你 PDE 长什么样、和 ODE 有什么本质区别;第二,介绍三大经典 PDE(热方程、波动方程、拉普拉斯方程),它们贯穿了整个数学物理;第三,给你一张解题地图——告诉你后面两章我们要走哪条路、为什么走那条路、其他路是什么样子。读完这一章,你应该能在看到一个 PDE 的时候立刻说出"它长成这样,所以应该用这种方法",哪怕你还不会真正动手算。

关于"为什么 PDE 比 ODE 难"再多说一句。ODE 的解空间通常是一个有限维的向量空间(二阶线性齐次方程的解是两个基础解的线性组合,所以解空间是二维的)。PDE 的解空间一般是无限维的——这就是为什么我们后面要用傅里叶级数(无穷多项的和)来组装解。这种维数上的飞跃,才是 PDE 比 ODE 难的真正原因。但好消息是:通过一种叫"分离变量"的技巧,我们可以把 PDE 化成几个 ODE,然后再用本书前面学的工具去解每一个 ODE,最后把它们重新组装成 PDE 的解。这条路线就是本章 18.7 节要详细讲的"总轮廓"。

18.1 从一个变量到多个变量

先把 ODE 和 PDE 的书面差别讲清楚。ODE 里的未知函数只有一个自变量,所以它的导数都是普通的 \(d/dt\) 或者 \(d/dx\):

\[ \frac{dy}{dt} = -ky, \qquad \frac{d^2x}{dt^2} + \omega^2 x = 0. \]

这些是 ODE。PDE 里的未知函数有两个或更多自变量,所以它的导数都是偏导数 \(\partial/\partial t\)、\(\partial/\partial x\):

\[ \frac{\partial u}{\partial t} = \alpha \frac{\partial^2 u}{\partial x^2}, \qquad \frac{\partial^2 u}{\partial t^2} = c^2 \frac{\partial^2 u}{\partial x^2}, \qquad \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0. \]

这些是 PDE。\(\partial\) 这个符号读作"偏",它告诉你"求导的时候,把其他变量当成常数"。\(\partial u/\partial t\) 的意思是:固定 \(x\),只让 \(t\) 变,看 \(u\) 怎么变。\(\partial u/\partial x\) 的意思是:固定 \(t\),只让 \(x\) 变,看 \(u\) 怎么变。两者完全不是同一回事,所以它们必须用不同的符号写出来——这正是 \(\partial\) 取代了 \(d\) 的原因。

为了书写方便,我们经常用下标记号来表示偏导数:

\[ u_t = \frac{\partial u}{\partial t}, \quad u_x = \frac{\partial u}{\partial x}, \quad u_{tt} = \frac{\partial^2 u}{\partial t^2}, \quad u_{xx} = \frac{\partial^2 u}{\partial x^2}, \quad u_{xt} = \frac{\partial^2 u}{\partial x \partial t}. \]

这种写法把字数砍成原来的三分之一,所以下面整本 PDE 的内容里我们都会优先用下标。请你立刻习惯它:看到 \(u_{xx}\) 不要再去想 \(\partial^2 u/\partial x^2\),直接读成"对 \(x\) 求两次偏导"就行。

那么 PDE 的"解"长什么样?ODE 的解是一个函数(一条曲线)。PDE 的解仍然是一个函数,但它有两个(或更多)输入,所以图像是一个曲面。例如,热方程的解 \(u(x, t)\) 描述了一根杆上每一点 \(x\) 在每一时刻 \(t\) 的温度——这个图像在 \(xt\) 平面上方画出一张曲面,沿 \(x\) 方向看是温度随空间的分布,沿 \(t\) 方向看是温度随时间的演化。这种"既是空间故事又是时间故事"的双重身份,是 PDE 解最直观的一面。

看一道很小的题来感受一下。验证 \(u(x, t) = e^{-t} \sin x\) 满足 PDE \(u_t = u_{xx}\)。这是一个非常温和的练习,但它告诉你"验证 PDE 的解"和"验证 ODE 的解"流程一模一样:算每一项,代回原方程,看看相不相等。

算 \(u_t\):\(t\) 是变量,\(x\) 当常数,所以 \(u_t = -e^{-t} \sin x\)。算 \(u_x\):\(x\) 是变量,\(t\) 当常数,\(u_x = e^{-t} \cos x\)。再对 \(x\) 求一次:\(u_{xx} = -e^{-t} \sin x\)。比较 \(u_t\) 与 \(u_{xx}\):两者都等于 \(-e^{-t} \sin x\),完全相等 ✓。所以 \(u(x, t) = e^{-t} \sin x\) 确实是这个 PDE 的解。

注意 \(u(x, t) = e^{-t} \sin x\) 这个解的形状:它是 \(x\) 的函数 \(\sin x\) 与 \(t\) 的函数 \(e^{-t}\) 的乘积。这种"\(X(x) \cdot T(t)\) 形式"的解,正是 18.7 节我们要重点利用的——分离变量法的全部出发点就是先去找这种乘积形式的解,再把多个这样的解叠加起来。

顺便提一个重要现象:PDE 的"通解"通常含有任意函数,而不是 ODE 那样的"任意常数"。例如最简单的 PDE \(u_x = 0\)(对 \(x\) 求偏导为零),它的解是\(u(x, y) = f(y)\),其中 \(f\) 是任意一元函数。这跟 ODE \(dy/dx = 0\) 的解 \(y = C\)(\(C\) 是任意常数)对照起来很有教育意义:ODE 的不定性是有限维(一个常数),PDE 的不定性是无限维(一个任意函数)。这又一次说明,PDE 的解空间天然是无限维的。要把任意函数 \(f\) 钉死,我们需要的"边界数据"也是无限多的——这就是 18.4 节边界条件的故事。

18.2 三大经典 PDE

整个数学物理的 PDE 浩瀚如海,但有三个方程是绝对的中心。它们刻画了三种最基本的物理过程,所有更复杂的 PDE 几乎都是它们的某种推广或耦合。把这三个方程吃透,你就掌握了 PDE 世界的"骨架"。

18.2.1 热方程

第一个是热方程(heat equation),也叫扩散方程:

\[ \boxed{u_t = \alpha\, u_{xx}.} \]

这里 \(u(x, t)\) 通常表示"某一点的温度",\(\alpha > 0\) 是一个叫做热扩散系数的常数,刻画了介质把热量"传开"的快慢。这个方程的物理意义来自一个很自然的实验观察:一个点的温度上升或下降的速率,正比于它周围温度的"凸度"。如果这一点比周围冷(\(u_{xx} > 0\),凹向上),热量就从两边流过来,温度升高(\(u_t > 0\));如果这一点比周围热(\(u_{xx} < 0\),凹向下),热量流出去,温度下降(\(u_t < 0\))。\(u_t\) 与 \(u_{xx}\) 的符号一致,系数是 \(\alpha\)——这就是热方程。

热方程的应用范围远远超出"传热"。它统治着任何"扩散"过程:墨水在水里扩散、香味在房间里扩散、股票价格在市场上的随机游走(布朗运动)、神经突触上的离子流——只要有"梯度驱动的流",数学上往往就归结为热方程。\(\alpha\) 的物理含义会变(热扩散系数、分子扩散系数、波动率平方),但方程的形状不变。这种跨学科的统一性,正是数学物理最迷人的地方:同一组符号,在物理、生物、金融三个领域里描述看起来风马牛不相及的现象。

18.2.2 波动方程

第二个是波动方程(wave equation):

\[ \boxed{u_{tt} = c^2\, u_{xx}.} \]

这里 \(u(x, t)\) 表示一根弦在位置 \(x\)、时刻 \(t\) 处偏离平衡位置的位移,\(c > 0\) 是一个叫做波速的常数。注意它和热方程的关键区别:左边是 \(u_{tt}\),不是 \(u_t\)——也就是说,这是一个关于时间的二阶方程。这个微小的差别带来了完全不同的物理:热方程描述"扩散"(信息消散),波动方程描述"传播"(信息以速度 \(c\) 传出去而不损失)。

从牛顿第二定律 \(F = ma\) 推导波动方程很直观:把一根弦想成一长串小质量,每一小段所受的张力的合力(由张力和邻段的弯曲产生)就是 \(c^2 u_{xx}\) 那一项,而 \(u_{tt}\) 就是加速度。所以波动方程其实就是把 \(F = ma\) 写在一根连续的弦上。

波动方程的应用同样跨越多个领域:吉他弦、空气中的声波、水面上的涟漪、电磁场(光、电波、X 射线)、地震波。任何"恢复力正比于位移的二阶导"的连续介质,运动方程都长这个样子。我们将在第 20 章详细求解它,会发现它的解天然分裂成"右行波"与"左行波"两部分——这就是 D'Alembert 解的核心。

18.2.3 拉普拉斯方程

第三个是拉普拉斯方程(Laplace's equation):

\[ \boxed{u_{xx} + u_{yy} = 0.} \]

注意这个方程没有时间。两个自变量 \(x\) 和 \(y\) 都是空间坐标,所以它描述的是"稳态"——也就是不随时间变化的分布。事实上,如果你拿热方程,把杆换成一片金属板,然后等它冷热平衡(\(u_t = 0\)),剩下的就是 \(u_{xx} + u_{yy} = 0\),正是拉普拉斯方程。所以拉普拉斯方程是热方程在稳态极限下的样子。

同样的方程也出现在静电学里:真空中无电荷的区域,电势 \(\phi\) 满足 \(\phi_{xx} + \phi_{yy} = 0\)(在二维)。还有理想流体的速度势、引力势(无质量分布的区域)、肥皂膜的形状、复变函数的实部和虚部——它们都满足拉普拉斯方程。满足拉普拉斯方程的函数被赋予了一个专门的名字:调和函数(harmonic functions)。这一族函数有许多优美的性质:它们没有"局部极大或极小"(任何极值都必然在边界上取到),它们等于以自身为中心、任何小圆盘上的平均值——这就是著名的"平均值性质"。这些都是数学分析里专门的话题。

拉普拉斯算子 \(\Delta = \partial^2/\partial x^2 + \partial^2/\partial y^2\)(在三维里 \(\Delta = \partial_x^2 + \partial_y^2 + \partial_z^2\))出现得如此频繁,以至于上面三个方程经常被简写成:

\[ u_t = \alpha \Delta u, \qquad u_{tt} = c^2 \Delta u, \qquad \Delta u = 0. \]

当你以后看到这种符号,就知道是热方程、波动方程、拉普拉斯方程了。这种简洁性也提示你:这三个方程其实是同一个"扩散/传播/平衡"故事的三种时间结构——没有时间(拉普拉斯)、一阶时间(热)、二阶时间(波动)。把这三种时间结构理解透,就理解了线性 PDE 世界的"主轴"。

18.3 椭圆 / 抛物 / 双曲

线性二阶 PDE 在数学上有一个标准分类。考虑两个自变量的线性二阶 PDE 的最一般形式:

\[ A\, u_{xx} + 2B\, u_{xy} + C\, u_{yy} + (\text{低阶项}) = 0, \]

其中 \(A, B, C\) 是常数(或可以是 \(x, y\) 的函数,但暂时当成常数)。这个方程的"性格"由判别式 \(B^2 - AC\) 决定,完全类比圆锥曲线的分类:

  • 椭圆型(elliptic):\(B^2 - AC < 0\)。代表方程:拉普拉斯方程 \(u_{xx} + u_{yy} = 0\),\(A = C = 1\),\(B = 0\),\(B^2 - AC = -1 < 0\)。
  • 抛物型(parabolic):\(B^2 - AC = 0\)。代表方程:热方程 \(u_t - \alpha u_{xx} = 0\),把 \(t\) 当 \(y\),则 \(A = -\alpha\),\(B = 0\),\(C = 0\),\(B^2 - AC = 0\)。
  • 双曲型(hyperbolic):\(B^2 - AC > 0\)。代表方程:波动方程 \(u_{tt} - c^2 u_{xx} = 0\),\(A = -c^2\),\(B = 0\),\(C = 1\),\(B^2 - AC = c^2 > 0\)。

三大经典 PDE 不是任意挑出来的——它们正好是三种类型各自的"原型"。所有椭圆型 PDE 在性格上都像拉普拉斯方程(描述稳态、平衡、调和),所有抛物型 PDE 都像热方程(描述扩散、不可逆的"磨平"过程),所有双曲型 PDE 都像波动方程(描述以有限速度传播的波)。这种"分类带来对应直觉"的逻辑,是 PDE 入门的关键之一:看到一个新方程,先算判别式确定类型,你立刻就知道它的解长什么样、怎么解、解的物理意义大致是什么。

这里值得说一句:这三种类型对应的"问题恰当性"也不一样,这是 18.5 节要谈的话题。先剧透一句:椭圆型问题靠边界条件来定解;抛物型和双曲型问题需要初始条件 + 边界条件;给错了条件可能根本无解或者有无穷多解。所以分类并不是数学家的洁癖——它直接影响"应该提怎样的问题"。

看一道分类的小题。判断 PDE \(3 u_{xx} + 4 u_{xy} + u_{yy} = 0\) 的类型。读出 \(A = 3\)、\(2B = 4\) 即 \(B = 2\)、\(C = 1\)。判别式 \(B^2 - AC = 4 - 3 = 1 > 0\)。所以是双曲型。再看 \(u_{xx} + 4 u_{xy} + 4 u_{yy} = 0\):\(A = 1\),\(B = 2\),\(C = 4\),\(B^2 - AC = 4 - 4 = 0\),所以是抛物型。最后看 \(u_{xx} + 2 u_{yy} = 0\):\(A = 1\),\(B = 0\),\(C = 2\),\(B^2 - AC = -2 < 0\),所以是椭圆型。掌握了这种快速分类,看到一个不认识的方程就能下意识地把它归到对应"家族"。

18.4 边界条件与初始条件

ODE 也需要"额外信息"才能挑出唯一解——一阶 ODE 需要一个初值,二阶 ODE 需要两个。PDE 的情形更丰富:既要"时间"上的初值(初始条件,initial condition,简称 IC),又要"空间"上的边值(边界条件,boundary condition,简称 BC)。这两个名字一定要分清楚。

看热方程 \(u_t = \alpha u_{xx}\) 的具体例子。物理设定:一根长度为 \(L\) 的金属棒,放在 \(0 \le x \le L\) 上。要确定 \(u(x, t)\),你必须告诉我:

  • 初始条件:在 \(t = 0\) 时,棒上每一点的温度是什么?即 \(u(x, 0) = f(x)\),其中 \(f\) 是一个已知函数。
  • 边界条件:对每一时刻 \(t > 0\),棒的两端 \(x = 0\) 和 \(x = L\) 处发生什么?最常见的是两端温度固定为零:\(u(0, t) = 0\),\(u(L, t) = 0\)。这模拟"两端浸在冰水里"的物理情形。

有了 IC 和 BC,棒的温度演化就唯一确定。注意 IC 给的是整条空间的初始数据(一个函数),BC 给的是整段时间的端点数据(每一时刻一个值)——它们的"形状"是互补的。

边界条件有几种标准类型,值得记住名字:

  • 狄利克雷边界条件(Dirichlet BC):规定 \(u\) 在边界上的值。例如 \(u(0, t) = 0\)。物理意义:两端温度被强制保持为 0。
  • 诺伊曼边界条件(Neumann BC):规定 \(u\) 在边界上的导数。例如 \(u_x(0, t) = 0\)。物理意义:边界不导热(绝缘),没有热流穿过。
  • 罗宾边界条件(Robin BC):规定 \(u\) 与 \(u_x\) 的某个线性组合。例如 \(u_x + h u = 0\)。物理意义:边界与外界按"温差"换热(牛顿冷却律)。

这三种边界条件覆盖了大部分实际物理情形。它们对应的数学问题求解方法略有不同——不同的 BC 会让我们在 19.3 节解出来的"本征值问题"长不同的样子。本书第 19、20 章主要用齐次狄利克雷条件 \(u(0, t) = u(L, t) = 0\),因为它最简单也最对称;真实工程问题里则需要根据物理情境选合适的 BC。

波动方程 \(u_{tt} = c^2 u_{xx}\) 的情形再多一层。因为时间是二阶的,所以初始条件需要给两个:不仅要给初始位移 \(u(x, 0) = f(x)\),还要给初始速度 \(u_t(x, 0) = g(x)\)。这正好类比二阶 ODE 需要两个初值。物理上很自然:吉他弦的运动既取决于你最初把它拉到哪里(\(f\)),也取决于你松手时它的速度是多少(\(g\))。BC 同样需要 \(u(0, t) = u(L, t) = 0\),代表两端被钉死。

拉普拉斯方程 \(u_{xx} + u_{yy} = 0\) 没有时间,所以只有 BC,没有 IC。例如在矩形 \(0 \le x \le a\)、\(0 \le y \le b\) 内求 \(u\),你需要给定矩形四条边上 \(u\) 的值——这一组数据完全确定 \(u\) 在矩形内部。这种"边界数据 → 内部解"的格局,正是椭圆型问题的标志。

18.5 完备问题:存在、唯一、连续

给一个 PDE 配上 IC 和 BC,这样形成的问题有意义吗?也就是说,它有解吗?如果有,解唯一吗?如果数据稍微改一点,解会不会剧烈变化?这三个问题分别叫做存在性、唯一性、连续依赖性(对数据的连续依赖)。一个同时满足这三条的问题,被法国数学家哈达马(Hadamard)称为完备问题(well-posed problem)。

为什么三条都重要?

  • 存在性:如果方程根本没有解,那物理建模就出错了——大自然显然给出了某种结果,如果你的方程没解,说明你写错了方程或者条件。
  • 唯一性:如果有多个解,你不知道哪个对应真实物理。这等于"数学没法预言物理"。
  • 连续依赖性:这一条最微妙。在实验里,你测到的初始数据总有一点误差;如果解对数据极度敏感,那即使方程数学上完美,你也算不出可靠预测——因为微小的输入误差会被放大成巨大的输出误差。

三条全满足,这个问题就是"恰到好处"的——既可解,又可靠。本书第 19、20 章的物理设定都是完备问题,所以你不需要担心。

不完备的问题确实存在,而且在物理里有时正是关键。一个经典例子是"反向"热方程 \(u_t = -\alpha u_{xx}\)(把扩散沿时间倒着走),它对数据不连续依赖——给定今天的温度分布,要反推一千年前的初始温度,即使理论上有唯一解,数据中任何微小误差都会被指数级放大,实际不可解。这是为什么"热扩散是不可逆的"在数学上的体现。把这种现象记在心里,会让你以后看到"反问题"(inverse problem)时,知道它为什么麻烦。

18.6 求解 PDE 的整体策略

"怎么解 PDE?"——回答这个问题之前,先认清现实:绝大多数 PDE 没有解析解。哪怕看起来温和的方程,在复杂的几何形状或非线性的情形下,通常只能数值求解。但是,对于本书会涉及的"经典几何 + 线性方程",有四类标准技术,每一类都对应特定的方程结构:

  1. 分离变量法(separation of variables)。适用于:在简单几何(矩形、圆盘、球)上的线性齐次 PDE,带齐次 BC。这是本书第 19、20 章的主线方法。它把 PDE 化成几个 ODE,然后用傅里叶级数把解组装起来。
  2. 积分变换法(integral transforms)。把 PDE 的某个变量变换掉:对时间用拉普拉斯变换(可以用我们第 12 章学到的工具),对无界空间用傅里叶变换。变换后 PDE 变成 ODE 或代数方程,解出来再反变换回去。适合无穷长杆、无穷平面等"无边界"问题。
  3. 特征线法(method of characteristics)。专门对付一阶 PDE 与某些双曲型方程。把 PDE 沿"特征曲线"化成 ODE。波动方程的 D'Alembert 解就是特征线法最简洁的应用,我们将在 20.7 节展示。
  4. 数值方法(numerical methods)。有限差分、有限元、谱方法。当几何复杂或方程非线性时,这是唯一现实的路线。本书不深入,但你应该知道它们的存在,因为科学计算的大半事业就建立在这些技术上。

本书第 19、20 章的所有手算技巧都属于第 1 类(分离变量)和第 3 类(特征线)的入门。第 2 类(积分变换)在第 19 章末尾会"轻描淡写"地提一下傅里叶变换,但不深入。第 4 类完全留给数值分析课。

怎么挑方法?粗略地说:

  • 有限区域 + 齐次 BC → 分离变量。
  • 无限区域 → 变换法。
  • 一阶或纯双曲 + 没有边界 → 特征线。
  • 其他,或几何复杂 → 数值。

这个"地图"会引导你后面看到任何一个 PDE 时知道大致该往哪个方向走。本书着重训练分离变量,因为它最具教育意义——它把"傅里叶级数"和"PDE"两件事一次性教会你,而且推广到其他几何只是细节问题。

18.7 分离变量法的总轮廓

这一节是后两章的"剧本提纲"。我们用最简单的例子——一根两端温度为零、初始温度分布为 \(f(x)\) 的有限长杆上的热方程——把分离变量法的整个流程讲一遍。第 19 章会重新走一遍同样的流程,但带着所有计算细节;第 20 章则把它套用到波动方程上。

问题:

\[ u_t = \alpha\, u_{xx}, \quad 0 \le x \le L,\ t > 0, \]

带 BC \(u(0, t) = u(L, t) = 0\) 与 IC \(u(x, 0) = f(x)\)。

分离变量法的赌注是:假设解可以写成"\(x\) 的函数 × \(t\) 的函数"的形式:

\[ u(x, t) = X(x)\, T(t). \]

这是一个非常强的假设——大多数 \(u(x, t)\) 都不能这样分解。但奇迹的地方在于:虽然单个分离形式不能表示一般的 \(f\),但很多个分离形式的线性叠加可以表示几乎任何 \(f\)。这正是傅里叶级数的力量,也是为什么分离变量法在线性 PDE 里会成功。

把 \(u = X(x) T(t)\) 代入 \(u_t = \alpha u_{xx}\)。\(u_t = X(x) T'(t)\),\(u_{xx} = X''(x) T(t)\)。代入得:

\[ X(x) T'(t) = \alpha X''(x) T(t). \]

把 \(X\) 与 \(T\) 各自挪到一边(同时除以 \(\alpha X(x) T(t)\)):

\[ \frac{T'(t)}{\alpha T(t)} = \frac{X''(x)}{X(x)}. \]

这个等式的两边长得很奇怪——左边只依赖 \(t\),右边只依赖 \(x\)。但等式对所有 \(x\) 和所有 \(t\) 都成立。如果我固定 \(x\) 改变 \(t\),左边变,右边不变;但等式必须保持。结论只有一个:两边都必须等于一个常数(这是分离变量法的关键时刻)。把这个常数记作 \(-\lambda\)(选这个符号是为了让后面的物理解释方便):

\[ \frac{T'(t)}{\alpha T(t)} = \frac{X''(x)}{X(x)} = -\lambda. \]

这一个等式立刻分裂成两个 ODE:

\[ T'(t) + \alpha \lambda\, T(t) = 0, \qquad X''(x) + \lambda\, X(x) = 0. \]

左边是关于 \(t\) 的一阶线性 ODE,右边是关于 \(x\) 的二阶线性 ODE。本书第 3 章和第 6 章都会解。所以"PDE → 两个 ODE"这一步,正是分离变量法的核心招数:把困难翻倍降维。

边界条件呢?\(u(0, t) = X(0) T(t) = 0\) 对所有 \(t\) 成立——这意味着 \(X(0) = 0\)(否则 \(T \equiv 0\),解就是平凡的 0)。同理 \(X(L) = 0\)。所以空间方程带齐次边界条件:

\[ X'' + \lambda X = 0, \quad X(0) = X(L) = 0. \]

这就是一个本征值问题:对什么样的 \(\lambda\),这个 ODE+BC 有非平凡解?这种 \(\lambda\) 叫本征值,对应的非零解叫本征函数。我们将在 19.3 节详细解出本征值是 \(\lambda_n = (n\pi/L)^2\),本征函数是 \(X_n(x) = \sin(n\pi x/L)\),其中 \(n = 1, 2, 3, \ldots\)。

每个本征值对应一个时间方程的解:\(T_n(t) = e^{-\alpha \lambda_n t}\)。乘起来得到一族分离形式的解:

\[ u_n(x, t) = e^{-\alpha (n\pi/L)^2 t} \sin\!\left(\frac{n\pi x}{L}\right), \quad n = 1, 2, 3, \ldots \]

每一个都满足 PDE 与 BC,但都不一定满足 IC \(u(x, 0) = f(x)\)。怎么办?叠加它们:

\[ u(x, t) = \sum_{n=1}^\infty b_n\, e^{-\alpha (n\pi/L)^2 t} \sin\!\left(\frac{n\pi x}{L}\right). \]

这一步利用了 PDE 的线性:任意线性组合的解还是解,而 BC 是齐次的,所以叠加后也满足 BC。剩下的任务是选系数 \(b_n\) 让 \(t = 0\) 时这个级数等于给定的 \(f(x)\):

\[ f(x) = \sum_{n=1}^\infty b_n \sin\!\left(\frac{n\pi x}{L}\right). \]

这正是傅里叶正弦级数!\(b_n\) 由"傅里叶系数公式"算出。一旦 \(b_n\) 算出,代回上面的级数就给出了完整的解 \(u(x, t)\)。整个流程:分离 → 解 ODE → 叠加 → 用 IC 定系数,这就是分离变量法的"四步舞"。

第 19 章会把每一步的细节都展开,包括为什么 \(\lambda < 0\) 与 \(\lambda = 0\) 的情形被排除,为什么本征函数恰好是 \(\sin\),系数公式怎么从正交性导出,以及一道带具体 \(f(x) = x(L-x)\) 的完整算例。第 20 章把整个过程在波动方程上重做一遍——你会看到,空间部分完全一样(\(\sin(n\pi x/L)\)),只有时间部分从 \(e^{-\alpha \lambda_n t}\) 换成了 \(\cos\) 与 \(\sin\) 的振荡组合。这种"空间几何决定本征函数,时间结构决定时间因子"的分工,是 PDE 思维里反复出现的主题。

18.8 Sturm-Liouville 问题

上一节空间方程 \(X'' + \lambda X = 0\) 配上齐次边界条件,是一个本征值问题。这种问题在 PDE 里如此常见,数学家给它起了个专门的名字:Sturm-Liouville 问题(简称 SL 问题)。SL 问题的一般形式是

\[ -\big(p(x) y'\big)' + q(x) y = \lambda\, w(x) y, \]

带某种齐次 BC。这里 \(p, q, w\) 是给定函数,\(\lambda\) 是待求的本征值,\(y\) 是待求的本征函数。我们的 \(X'' + \lambda X = 0\) 是 SL 问题里 \(p \equiv 1\)、\(q \equiv 0\)、\(w \equiv 1\) 的特例。

SL 问题之所以重要,是因为它的本征函数族有一些非常优美的性质,几乎是"为分离变量法量身定制":

  • 本征值是实数且可以排成升序列 \(\lambda_1 < \lambda_2 < \cdots\),趋向无穷。
  • 不同本征值对应的本征函数正交(在带权 \(w\) 的内积意义下)。
  • 这些本征函数构成一个完备函数系——任何"合理"的函数都可以展开成它们的级数。

这三条就是傅里叶级数为什么能工作的"幕后老板"。19.6 节我们用正交性导出傅里叶系数公式,本质上就是在用 SL 理论的第二条性质。第三条(完备性)的严格证明需要泛函分析,但你只需要知道它给了你"任何合理的 \(f\) 都能展开"的承诺。

本书不深入 SL 理论,但建议你记住这个名字——以后如果学到圆盘上的拉普拉斯方程,本征函数就是贝塞尔函数;球面上,本征函数是球谐函数。这些"特殊函数"都是各自几何对应的 SL 问题的本征函数。背后的统一框架是相同的,只不过每种几何长出不同的"花"。

到这里,我们已经把 PDE 的"地图"摆了出来:三大方程 + 三种类型 + 一种主流求解方法 + 一个数学骨架(SL)。第 19 章我们扎进热方程的细节,把分离变量从头到尾算一遍,顺带把傅里叶级数完整地讲清楚——它本身就是 PDE 的副产品中最辉煌的一项数学发明。第 20 章再把同样的方法搬到波动方程上,并加上 D'Alembert 的特征线视角作为对照。
