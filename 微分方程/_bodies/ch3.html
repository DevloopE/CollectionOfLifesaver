第 3 章 一阶线性方程

第 2 章你学会了一招几乎无脑的解法：碰到 \(\dfrac{dy}{dx} = f(x) g(y)\) 这种"\(x\) 部分和 \(y\) 部分能分开"的方程，把变量分离、两边积分，搞定。这一招有多好用，你已经亲眼见过——人口、衰变、冷却、自由落体、化学反应几乎全是它的功劳。但它也有一个明显的局限：右边必须能写成"\(x\) 函数乘 \(y\) 函数"。一旦右边出现"\(x\) 与 \(y\) 的和"或更复杂的纠缠，分离变量法就死掉了。

来看一个无辜的例子：

\[ \frac{dy}{dx} + 2 x y = x. \]

这显然是一阶方程，含 \(y'\) 和 \(y\)，结构很简单。但要分离它，你得把 \(y' = x - 2 x y = x (1 - 2 y)\) ——咦？竟然是可分离的，因为右边能写成 \(x \cdot (1 - 2y)\)。OK 这个例子运气好。再换一个：

\[ \frac{dy}{dx} + 2 x y = x^2. \]

右边变成 \(y' = x^2 - 2 x y\)。能否分离？\(x^2 - 2 x y\)——这是 \(x\) 与 \(y\) 的线性组合，无论怎么提公因子，你都把 \(x\) 部分和 \(y\) 部分掰不开。它不是可分离的。但它仍然结构非常工整——\(y\) 与 \(y'\) 都只以"一次"出现（这正是"线性"的含义），系数随 \(x\) 变化。这就是本章要解决的对象：一阶线性方程。

这一章讲一种漂亮的方法叫做积分因子法。它的核心想法值得你提前知道：我们将给方程两边乘上一个精心选择的"魔法函数" \(\mu(x)\)，使得乘完之后左边突然能合并成一个完整的导数 \((\mu(x) y)'\)——之后两边对 \(x\) 积一次分，方程立刻被解开。整个 trick 的精髓就在于"找出那个 \(\mu(x)\)"。本章先教你这件事。

本章结构：3.1 节给出一阶线性方程的标准形式与术语；3.2 节用一个非常具体的例子动机化"积分因子"这个想法；3.3 节推导积分因子的明确公式 \(\mu(x) = e^{\int p(x) dx}\)；3.4 节给出整套通解公式；3.5 节讲初值问题与唯一性；3.6 节大量例题；3.7 节回到本章开头的话题——线性 vs 可分离，何时用哪种。

读这一章的时候，我特别建议你在草稿纸上跟着每一步演算。"积分因子"这个想法在文字上看起来朴素得一目了然——给方程乘个东西、合并成乘积导数、再积分——但真正自己亲手算过几遍之后才能形成"反射"，看到一阶线性方程时立刻知道下一步是什么。这种反射在做大题、考试、解工程问题时是无价的。

3.1 标准形式

一个一阶线性方程指的是能写成下面这种形式的方程：

\[ \boxed{\frac{dy}{dx} + p(x)\, y = q(x).} \]

这叫做标准形。要点：

  • 方程是一阶的——只出现 \(y\) 与 \(y'\)，不涉及 \(y''\) 等更高阶导数。
  • \(y'\) 系数是 1（如果原方程里 \(y'\) 的系数不是 1，先把它除过去；见下面）。
  • \(y\) 出现一次，被 \(x\) 的某个函数 \(p(x)\) 乘着。
  • 右边 \(q(x)\) 完全只与 \(x\) 有关——不依赖 \(y\)。

"线性"这个词的来由：等式左边只含 \(y\) 和 \(y'\) 的"一次"组合，\(y\) 不会以 \(y^2\)、\(\sin y\)、\(y \cdot y'\) 等非线性方式出现。这正是 1.3.2 节里我们对线性方程下定义时说的那种结构。

很多方程看起来不是这种形式，但容易化成这种形式。比如：

  • \((1 + x^2) y' + 2 x y = e^x\)。\(y'\) 前有系数 \((1 + x^2)\)；两边除以这个系数得 \(y' + \dfrac{2x}{1 + x^2} y = \dfrac{e^x}{1 + x^2}\)。这就是标准形，\(p(x) = \dfrac{2x}{1+x^2}\)、\(q(x) = \dfrac{e^x}{1+x^2}\)。
  • \(y' = x y + e^x\)。把 \(xy\) 移到左边：\(y' - x y = e^x\)，即 \(p(x) = -x\)、\(q(x) = e^x\)。
  • \(x y' = y + x^2\)（\(x \neq 0\)）。整理：\(y' - \dfrac{1}{x} y = x\)，所以 \(p(x) = -\dfrac{1}{x}\)、\(q(x) = x\)。

遇到一个含 \(y'\) 与 \(y\) 的方程，第一件事就是把它写成标准形。除非已经是标准形，否则不要急着套公式。把 \(y'\) 系数化为 1 这一步，决定了 \(p(x)\) 和 \(q(x)\) 的具体表达式。

如果右边 \(q(x) \equiv 0\)，方程叫做齐次的：\(y' + p(x) y = 0\)。齐次的一阶线性方程其实是可分离的（把 \(p(x) y\) 移到右边 \(y' = -p(x) y\)，分离 \(\dfrac{dy}{y} = -p(x) dx\)，积分），所以它是个特例。第 3 章的核心是处理 \(q(x) \neq 0\) 的"非齐次"情形——这才需要积分因子。

3.2 灵感：寻找一个魔法因子

从一个具体的方程开始：

\[ y' + 2 x y = x^2. \]

它的左边长得让人发痒——\(y'\) 加 \(y\) 乘某个系数。这是不是某个乘积的导数？回忆乘积法则：\((u v)' = u' v + u v'\)。如果我把 \(\mu(x) y\) 看成乘积 \(u v\)（\(u = \mu\)、\(v = y\)），那么

\[ \big(\mu(x) y\big)' = \mu'(x) y + \mu(x) y'. \]

嗯，\(\mu(x) y' + \mu'(x) y\)——和我们方程左边 \(y' + 2x y\) 形状很像，只要左边被 \(\mu(x)\) 乘一下。事实上，把原方程左右两边都乘以某个待定函数 \(\mu(x)\)：

\[ \mu(x) y' + 2 x \mu(x) y = x^2 \mu(x). \]

对照乘积法则的展开 \((\mu y)' = \mu y' + \mu' y\)，我们希望左边恰好就是 \((\mu y)'\)。比较系数：要让 \(\mu y' + 2 x \mu y\) 等于 \(\mu y' + \mu' y\)，必须有

\[ \mu'(x) = 2 x\, \mu(x). \]

瞧——这又是一个微分方程！但它是可分离的，而且只关乎 \(\mu\) 一个函数：

\[ \frac{d\mu}{\mu} = 2 x \, dx \;\Longrightarrow\; \ln |\mu| = x^2 + C \;\Longrightarrow\; \mu(x) = e^{x^2} \]

（取 \(C = 0\) 即可——任何常数倍的 \(\mu\) 都行，挑最简单的）。

这个 \(\mu(x) = e^{x^2}\) 就是我们的"魔法因子"。让我们看看它怎么魔法。把原方程乘以 \(e^{x^2}\)：

\[ e^{x^2} y' + 2 x e^{x^2} y = x^2 e^{x^2}. \]

左边正是 \((e^{x^2} y)'\)（你可以求导验证：\((e^{x^2})' \cdot y + e^{x^2} \cdot y' = 2x e^{x^2} y + e^{x^2} y'\) ✓）。所以方程变成

\[ \frac{d}{dx}\Big(e^{x^2} y\Big) = x^2 e^{x^2}. \]

对两边对 \(x\) 积分：

\[ e^{x^2} y = \int x^2 e^{x^2} \, dx + C. \]

这一步把方程从微分方程降级成了"求一个不定积分"——也就是普通的微积分问题。剩下来要解决 \(\int x^2 e^{x^2} dx\)，但这是另一回事，跟微分方程的"骨头"已经无关了（这个积分实际上没有初等闭式，会带 \(\mathrm{erf}\) 函数；本例的目的只是讲思路而非算到底）。换一个右边好积的例子，下面的演示就更圆满。

这就是积分因子法。三件事：

  1. 找出魔法因子 \(\mu(x)\)，使得方程左边乘上 \(\mu\) 之后正好是 \((\mu y)'\)。
  2. 把方程写成 \((\mu y)' = \mu(x) q(x)\)。
  3. 对两边积分，得到 \(\mu(x) y = \int \mu(x) q(x) dx + C\)，再把 \(\mu\) 除回去。

真正巧妙的只有第 1 步——找 \(\mu\)。下一节我们对一般情形把它做出来。

这一招的精神值得停下来玩味一下。我们没有直接攻击原方程，而是通过引入一个新对象 \(\mu\)，再用一个更简单的方程（关于 \(\mu\) 的可分离方程）把它求出来，最后用 \(\mu\) 来"打通"原方程。这种"间接路线"——绕一个圈把硬问题变软——在数学的更高层经常出现。它的物理类比是：你想把一辆熄火的车推过一座小山，直接推不动；你先到山顶给它一根缆绳挂在更高处的滑轮上，再借助这套机械装置，问题就解开了。\(\mu\) 就是那根缆绳。

3.3 推导积分因子的公式

在标准形 \(y' + p(x) y = q(x)\) 上重复上面的逻辑。希望 \(\mu(x)\) 满足"乘上后左边变成 \((\mu y)'\)"，也就是

\[ \mu(x)\, y' + \mu(x)\, p(x)\, y = \big(\mu(x) y\big)' = \mu'(x) y + \mu(x) y'. \]

对照系数（\(y'\) 的系数已经一致），\(y\) 的系数要相等：

\[ \mu(x) p(x) = \mu'(x). \]

这是一个关于 \(\mu\) 的可分离方程：

\[ \frac{d\mu}{\mu} = p(x) \, dx \;\Longrightarrow\; \ln |\mu| = \int p(x) \, dx + C_1 \;\Longrightarrow\; \mu(x) = C_2\, e^{\int p(x) dx}. \]

积分常数 \(C_2\) 取什么都行（任何非零倍数的 \(\mu\) 都能起到同样作用——它会出现在方程两边并互相约掉）。最方便的是取 \(C_2 = 1\)：

\[ \boxed{\mu(x) = e^{\int p(x)\, dx}.} \]

这就是积分因子的明确公式。两件事：

  • \(\int p(x) dx\) 这里取一个"任意"原函数即可——你不必加积分常数（任何加常数都只是把 \(\mu\) 乘以一个总体常数，约掉）。
  • 这个公式自动保证 \(\mu(x) > 0\)（指数函数永远为正），所以你大可放心地用它去乘原方程，不必担心改变方程或丢失绝对值号。

具体跑一遍：对方程 \(y' + 2 x y = x^2\)，\(p(x) = 2x\)，\(\int p \, dx = x^2\)，\(\mu = e^{x^2}\)。和我们 3.2 节算出来的一样 ✓。

再来：对 \(y' - \dfrac{1}{x} y = x\)（标准形见 3.1 节最后一例），\(p(x) = -1/x\)，\(\int p \, dx = -\ln|x|\)，\(\mu = e^{-\ln|x|} = 1/|x|\)。在 \(x > 0\) 区域上简化为 \(\mu = 1/x\)。所以用 \(1/x\) 乘方程：\(\dfrac{1}{x} y' - \dfrac{1}{x^2} y = 1\)，左边正是 \((\dfrac{y}{x})'\)（验证：\((\dfrac{y}{x})' = \dfrac{y'}{x} - \dfrac{y}{x^2}\) ✓）。所以方程化为 \(\dfrac{d}{dx}\left(\dfrac{y}{x}\right) = 1\)，积一下得 \(\dfrac{y}{x} = x + C\)，即 \(y = x^2 + C x\)。验证：\(y' = 2x + C\)；\(y' - y/x = 2x + C - (x^2 + Cx)/x = 2x + C - x - C = x\) ✓。

3.4 通解公式

把上面三步连起来写成一个公式。对一阶线性方程

\[ y' + p(x) y = q(x), \]

取 \(\mu(x) = e^{\int p(x) dx}\)，方程变形为 \((\mu y)' = \mu q\)。两边积分得 \(\mu y = \int \mu q \, dx + C\)。再除以 \(\mu\)：

\[ \boxed{\quad y(x) = \frac{1}{\mu(x)} \left( \int \mu(x)\, q(x) \, dx + C \right). \quad} \]

这就是一阶线性方程的通解公式。它含一个待定常数 \(C\)（来自不定积分），符合一阶方程"通解含一个常数"的预期。

使用这个公式的标准流程：

  1. 把方程写成标准形 \(y' + p(x) y = q(x)\)。
  2. 计算 \(\mu(x) = e^{\int p(x) dx}\)。
  3. 计算 \(\int \mu(x) q(x) dx\)。
  4. 把上一步结果加常数 \(C\)，再除以 \(\mu\)。

实际操作中，记下面这件事会让你少出错：你可以选择"直接套公式"，也可以选择"乘上 \(\mu\) 后把左边写成 \((\mu y)'\) 再积分"。两者完全等价；前者公式化、后者过程化。我个人推荐过程化，因为它更"看得见"——你能直观地看到 \((\mu y)'\) 这一步合并的发生，万一算错也容易在上下文里找到。但当你对方法熟练后，公式化无疑更省事。

顺便提一笔通解的结构。把通解公式拆开：

\[ y(x) = \underbrace{\frac{C}{\mu(x)}}_{\text{齐次部分}} + \underbrace{\frac{1}{\mu(x)}\int \mu(x) q(x) dx}_{\text{非齐次特解}}. \]

第一项 \(C / \mu(x) = C e^{-\int p \, dx}\) 是齐次方程 \(y' + p y = 0\) 的通解；第二项是非齐次方程 \(y' + p y = q\) 的某个特解。这种"齐次通解 + 非齐次特解"的结构是所有线性微分方程通解的共同骨架，你将在第 4 章二阶线性方程那里再见到一次更隆重的版本。

这个结构有一个重要的实际后果：叠加原理。如果你能把右端 \(q(x) = q_1(x) + q_2(x)\) 拆成两部分，并且分别求出 \(y' + py = q_1\) 的特解 \(y_1\) 与 \(y' + py = q_2\) 的特解 \(y_2\)，那么 \(y_1 + y_2\) 就是 \(y' + py = q\) 的一个特解。线性这个性质让你可以"分而治之"——把一个复杂的右端拆成几个简单的右端分别处理，再把答案加起来。这件事在右端是若干指数、若干正弦的组合时尤其方便，工程师们日常使用。

3.5 初值问题与唯一性

给一阶线性方程加上一个初始条件 \(y(x_0) = y_0\)，就构成一个 IVP：

\[ y' + p(x) y = q(x), \qquad y(x_0) = y_0. \]

解法不变——先求通解 \(y(x) = \mu^{-1}(\int \mu q \, dx + C)\)，再用初始条件确定 \(C\) 的值，得到唯一的特解。

关于唯一性的好消息：只要 \(p(x)\) 与 \(q(x)\) 在某个包含 \(x_0\) 的开区间上都连续，这个 IVP 在该区间上必定有唯一解。这件事不需要援引一般的皮卡尔定理——因为我们有一个明确公式，从公式里能直接读出答案存在且唯一。

具体来说，把通解公式里的不定积分换成定积分（从 \(x_0\) 积起），就能直接消掉常数 \(C\)：

\[ \mu(x) y(x) - \mu(x_0) y_0 = \int_{x_0}^{x} \mu(s) q(s) \, ds, \]

整理得

\[ y(x) = \frac{1}{\mu(x)}\left( \mu(x_0) y_0 + \int_{x_0}^{x} \mu(s) q(s) \, ds \right). \]

注意每一项都是明确算出来的——只要 \(p, q\) 连续，所有积分都存在，公式就给出唯一一个 \(y(x)\)。这是线性方程的特权：解的存在唯一性与"找解的公式"是合一的。非线性方程则没有这种好事——能保证存在唯一性的只有定理，找解还需要别的本事。

线性 IVP 的另一个特点：解的"存在区间"不会突然出现奇怪的爆破。只要 \(p, q\) 在 \((a, b)\) 上连续，IVP 的解就在整个 \((a, b)\) 上都存在。对比 2.2.4 节里 \(y' = 1 + y^2\) 那种"在 \(\pi/2\) 处突然飞掉"的有限时间爆破——非线性方程才会出这种事，线性方程不会。这是线性方程之所以好用的核心原因之一。

把这件事记成对比表：

一阶线性一阶非线性
是否有通用解法是（积分因子）无（看具体情形）
解的存在唯一性\(p, q\) 连续即可保证需要 \(\partial f/\partial y\) 也连续（皮卡尔条件）
解的存在区间跟 \(p, q\) 的连续区间一致可能"提前爆破"，存在区间小
叠加原理成立不成立
典型例子RC 电路、混合容器、变利率储蓄逻辑斯蒂、\(y' = 1 + y^2\)

这表的内容你日后会反复用到，特别是当题目突然给你一个"看起来比较吓人"的方程，你的第一反应就该是先看看它是不是线性的——是的话整本书都给你工具了；不是的话才需要花心思想怎么换元、怎么近似。

3.6 详细例题

3.6.1 例 1：纯计算（找 \(\mu\) 的代表性练手）

解 \(y' + 3 y = e^{2 x}\)、\(y(0) = 1\)。

已是标准形，\(p(x) = 3\)（常数）、\(q(x) = e^{2x}\)。

积分因子：\(\int p \, dx = 3 x\)，所以 \(\mu(x) = e^{3 x}\)。

乘以方程：\(e^{3x} y' + 3 e^{3x} y = e^{2x} \cdot e^{3x} = e^{5x}\)。左边是 \((e^{3x} y)'\)。

积分：\(e^{3x} y = \int e^{5x} dx + C = \dfrac{1}{5} e^{5x} + C\)。

除回 \(\mu\)：\(y(x) = \dfrac{1}{5} e^{2x} + C e^{-3x}\)。

用 \(y(0) = 1\)：\(1 = 1/5 + C\)，所以 \(C = 4/5\)。

最终特解：\(y(x) = \dfrac{1}{5} e^{2 x} + \dfrac{4}{5} e^{-3 x}\)。

验证：\(y'(x) = \dfrac{2}{5} e^{2x} - \dfrac{12}{5} e^{-3x}\)；\(3 y = \dfrac{3}{5} e^{2x} + \dfrac{12}{5} e^{-3x}\)；相加：\(y' + 3y = e^{2x}\) ✓。\(y(0) = 1/5 + 4/5 = 1\) ✓。

3.6.2 例 2：变系数

解 \(x y' + y = \cos x\)（\(x > 0\)）。

先化标准形。两边除以 \(x\)：\(y' + \dfrac{1}{x} y = \dfrac{\cos x}{x}\)。所以 \(p(x) = 1/x\)、\(q(x) = (\cos x)/x\)。

积分因子：\(\int (1/x) dx = \ln x\)（\(x > 0\)，去掉绝对值），\(\mu = e^{\ln x} = x\)。

乘上 \(\mu = x\)：\(x y' + y = \cos x\)——咦，正好就是原方程！这不是巧合：原方程左边 \(x y' + y\) 已经是 \((x y)'\) 的展开。这种现象叫"方程已经是恰当的"，省去一步代数。

所以 \((x y)' = \cos x\)，积分得 \(x y = \sin x + C\)，

\[ y(x) = \frac{\sin x + C}{x}. \]

这是通解。注意它在 \(x = 0\) 处可能有奇异——原方程 \(x y' + y = \cos x\) 在 \(x = 0\) 处的"\(y'\) 系数"为零，使得方程退化，所以解在 \(x = 0\) 附近的行为需要特别讨论。这种"系数在某点变为零"的方程叫奇点方程，本书后面（第 8 章左右）会专门处理；这里你只需要意识到：把 \(p(x) = 1/x\) 当作处处连续的函数是不对的，它在 \(x = 0\) 处发散，所以唯一性定理只在 \(x > 0\) 或 \(x < 0\) 各自的区间上成立。

验证：\(y' = \dfrac{x \cos x - (\sin x + C)}{x^2}\)；代入原方程左边：

\[ x y' + y = \frac{x \cos x - (\sin x + C)}{x} + \frac{\sin x + C}{x} = \frac{x \cos x}{x} = \cos x \;\checkmark. \]

3.6.3 例 3：RC 电路放电与充电

电工学里最经典的一阶线性方程来自 RC 电路（一个电阻 \(R\) 与一个电容 \(C\) 串联，外加一个电压源 \(V(t)\)）。设电容上的电压为 \(v(t)\)。基尔霍夫电压定律给出

\[ R C\, \frac{dv}{dt} + v = V(t). \]

整理为标准形：\(\dfrac{dv}{dt} + \dfrac{1}{RC} v = \dfrac{V(t)}{RC}\)。所以 \(p(t) = 1/(RC)\)（常数）、\(q(t) = V(t)/(RC)\)。把时间常数 \(\tau = RC\) 引进来更省事：\(v' + \dfrac{1}{\tau} v = \dfrac{V(t)}{\tau}\)。

积分因子：\(\mu(t) = e^{t/\tau}\)。乘上去得 \((e^{t/\tau} v)' = e^{t/\tau} V(t)/\tau\)，积分

\[ v(t) = e^{-t/\tau} \left( v(0) + \int_0^t e^{s/\tau} \frac{V(s)}{\tau} ds \right). \]

这是普适公式，对任何电压源 \(V(t)\) 都对。来看几个具体情形。

情形 A：电源为常数 \(V_0\)（恒压充电）。积分计算：\(\int_0^t e^{s/\tau} \dfrac{V_0}{\tau} ds = V_0 (e^{t/\tau} - 1)\)。所以

\[ v(t) = e^{-t/\tau}\big(v(0) + V_0 (e^{t/\tau} - 1)\big) = V_0 + (v(0) - V_0) e^{-t/\tau}. \]

看见了吗？这个公式跟牛顿冷却公式一模一样！\(V_0\) 是"环境温度"（最终电压），\(v(0)\) 是"初始温度"，时间常数 \(\tau\) 控制趋向最终值的速度。如果初始电容空（\(v(0) = 0\)），那么 \(v(t) = V_0 (1 - e^{-t/\tau})\)——典型的"S 型上爬"，时间 \(\tau\) 后达到 \(V_0\) 的 63%，时间 \(5\tau\) 后达到约 99%。

情形 B：电源关掉（放电）。取 \(V_0 = 0\)、\(v(0) = V_0^*\)（之前充满到 \(V_0^*\)）。代入：\(v(t) = V_0^* e^{-t/\tau}\)。指数衰减，时间常数 \(\tau\)。这就是相机闪光灯断电后电容电压下降的过程，也是示波器上 RC 测量最经典的曲线。

情形 C：电源为正弦波 \(V(t) = V_0 \cos(\omega t)\)。积分计算需要积 \(\int e^{s/\tau} \cos(\omega s) ds\)，这是积分学里的标准积分（用分部积分两次或用复数化方法），结果是 \(\dfrac{e^{s/\tau}(\cos(\omega s)/\tau + \omega \sin(\omega s))}{1/\tau^2 + \omega^2}\)。代入再化简，最终能得到 \(v(t) = A \cos(\omega t - \phi) + B e^{-t/\tau}\)，其中 \(A, \phi\) 是确定的常数，\(B\) 由初始条件决定。物理含义：第一项是稳态响应（与输入同频但相位滞后 \(\phi\)、振幅缩小 \(A\)），第二项是暂态响应（指数衰减，几个 \(\tau\) 后消失）。这是所有交流电路分析的入门曲。

3.6.4 例 4：变利率储蓄

设你有一个储蓄账户，初始本金 \(P_0\) 元。利息按瞬时复利方式增长，年利率为 \(r(t)\)（可能随时间变动）。你每年还以连续方式存入 \(d(t)\) 元。账户余额 \(P(t)\) 满足

\[ \frac{dP}{dt} = r(t) P + d(t). \]

整理为标准形：\(P' - r(t) P = d(t)\)，所以 \(p(t) = -r(t)\)、\(q(t) = d(t)\)。积分因子 \(\mu(t) = e^{-\int r(s) ds}\)。

具体例子：\(r(t) \equiv 0.05\)（年利率 5%）、\(d(t) = 1000\)（每年存 1000 元）、\(P(0) = 0\)（白手起家）。

\(\mu = e^{-0.05 t}\)。方程：\((e^{-0.05 t} P)' = 1000 e^{-0.05 t}\)。积分：\(e^{-0.05 t} P = 1000 \cdot \dfrac{e^{-0.05 t}}{-0.05} + C = -20000 e^{-0.05 t} + C\)。

用 \(P(0) = 0\)：\(0 = -20000 + C\)，所以 \(C = 20000\)。

最终：\(P(t) = -20000 + 20000 e^{0.05 t} = 20000 (e^{0.05 t} - 1)\)。

检查具体值：30 年后 \(P(30) = 20000 (e^{1.5} - 1) \approx 20000 \cdot 3.482 - 20000 \approx 49640\) 元。也就是说每年存 1000 元、5% 复利，30 年存到约 5 万元。这就是连续复利的力量——比 1000 × 30 = 30000 多了两万出头。

3.6.5 例 5：混合容器

一个 100 升的水箱里初始装有 50 升盐水，含 10 公斤盐（浓度 0.2 kg/L）。从顶部以 4 升/分钟的速率注入清水（无盐），底部以同样 4 升/分钟的速率排出充分混合后的溶液。求任意时刻水箱里的盐量 \(s(t)\)。

建模。由于注入与排出速率相等，水箱里的水总量保持 50 升不变。所以"瞬时浓度"是 \(s(t)/50\) kg/L。每分钟排出的盐量是 \((s/50) \cdot 4 = (4/50) s = 0.08 s\) kg。注入的盐量是 0（清水）。所以

\[ \frac{ds}{dt} = (\text{进}) - (\text{出}) = 0 - 0.08 s = -0.08 s. \]

这其实是齐次的、可分离的、退化的线性方程。\(p(t) = 0.08\)、\(q(t) = 0\)。\(\mu = e^{0.08 t}\)。\((e^{0.08 t} s)' = 0\)，所以 \(e^{0.08 t} s = C\)，\(s(t) = C e^{-0.08 t}\)。\(s(0) = 10\) 给 \(C = 10\)。最终 \(s(t) = 10 e^{-0.08 t}\)。指数衰减。

把题目稍微变难。同样设置，但注入的不是清水，而是浓度 0.5 kg/L 的盐水（仍以 4 L/min 注入并排出）。则每分钟注入盐量 \(0.5 \cdot 4 = 2\) kg，排出盐量仍是 \(0.08 s\) kg：

\[ \frac{ds}{dt} = 2 - 0.08 s. \]

这次 \(q \neq 0\)，必须用积分因子。标准形 \(s' + 0.08 s = 2\)。\(\mu = e^{0.08 t}\)。乘上去：\((e^{0.08 t} s)' = 2 e^{0.08 t}\)，积分得 \(e^{0.08 t} s = 25 e^{0.08 t} + C\)，于是 \(s(t) = 25 + C e^{-0.08 t}\)。\(s(0) = 10\) 给 \(C = -15\)。最终 \(s(t) = 25 - 15 e^{-0.08 t}\)。

检查极限：\(t \to \infty\) 时 \(s \to 25\) kg。意思是水箱里盐量趋向于 50 升 × 0.5 kg/L = 25 kg——也就是浓度趋向于注入水的浓度。直觉对得上。

这种"混合容器"题目是工程教材里一阶线性方程的最常见情境。一旦你看到"流入流出"的字眼，几乎可以下意识写出 \(\dfrac{d(\text{量})}{dt} = (\text{进入率}) - (\text{排出率})\)，然后整理成标准形。

3.6.6 例 6：含 \(\sin\) 的右端

解 \(y' + y = \sin x\)、\(y(0) = 0\)。

已是标准形。\(p = 1\)、\(q = \sin x\)。\(\mu = e^x\)。乘方程：\((e^x y)' = e^x \sin x\)。

积分：\(\int e^x \sin x \, dx\) 是个经典积分，要么用分部积分两次，要么记一个公式：\(\int e^{ax} \sin(bx) dx = \dfrac{e^{ax}(a \sin bx - b \cos bx)}{a^2 + b^2} + C\)。这里 \(a = b = 1\)，得 \(\int e^x \sin x \, dx = \dfrac{e^x (\sin x - \cos x)}{2} + C\)。

所以 \(e^x y = \dfrac{e^x (\sin x - \cos x)}{2} + C\)，化简：

\[ y(x) = \frac{\sin x - \cos x}{2} + C\, e^{-x}. \]

用 \(y(0) = 0\)：\(0 = \dfrac{0 - 1}{2} + C\)，所以 \(C = 1/2\)。

最终特解：\(y(x) = \dfrac{\sin x - \cos x}{2} + \dfrac{1}{2} e^{-x}\)。

解释这个解的物理。第一项是受迫响应——一个与输入正弦同频但相位不同的稳态振荡；第二项是暂态响应——一个指数衰减项。\(t\) 大时第二项消失，系统进入稳态，只剩前一项。这正是 3.6.3 节 RC 电路情形 C 的同一个故事：线性常系数方程加上正弦输入 = 稳态正弦输出 + 衰减暂态。这条规律在工程里被反复使用：信号处理、控制理论、电路分析的"频率响应"分析全都依赖于此。

3.6.7 例 7：带阻力的下落（线性版）

把第 2 章的 \(m v' = m g - b v\) 重做一遍，但用线性方程的方法而不是分离变量。整理：\(v' + (b/m) v = g\)。这是常系数线性方程，\(p = b/m\)、\(q = g\)。\(\mu = e^{(b/m) t}\)。\((e^{(b/m) t} v)' = g\, e^{(b/m) t}\)，积分得 \(e^{(b/m) t} v = \dfrac{m g}{b} e^{(b/m) t} + C\)，所以

\[ v(t) = \frac{m g}{b} + C e^{-(b/m) t}. \]

初始条件 \(v(0) = v_0\) 给 \(C = v_0 - mg/b\)，最终：

\[ v(t) = \frac{m g}{b} + \left(v_0 - \frac{m g}{b}\right) e^{-(b/m) t}. \]

这跟第 2 章的结果一模一样。线性方法和分离变量方法在这里给出同一个答案——因为 \(v' = g - (b/m) v\) 既是线性的也是可分离的（自治方程）。所以你可以挑你喜欢的方法。

3.7 何时用可分离，何时用线性

到这里你手里有了两件武器。决定用哪一件，是面对一阶方程时第一件要判断的事情。

常见的方程类型可以这样划分：

方程类型判别方法
\(y' = f(x)\)（只与 \(x\) 有关）右边无 \(y\)直接积分
\(y' = g(y)\)（只与 \(y\) 有关，自治）右边无 \(x\)分离变量
\(y' = f(x) g(y)\)（可分离）右边可拆为 \(x\)、\(y\) 两因子的积分离变量
\(y' + p(x) y = q(x)\)（线性）左边是 \(y\) 与 \(y'\) 的"一次"线性组合积分因子
同时是可分离与线性形如 \(y' + p(x) y = 0\) 或 \(y' = a y + b\)哪种顺手用哪种
都不是右边含 \(y\) 的非线性函数（\(y^2\)、\(\sin y\)、…），又不能分离需要更高级方法（齐次方程换元、伯努利、精确方程，等等）

判断的几个练习。下面每一行你心里默念一下："属于哪一类？该用什么方法？"答案附后。

  1. \(y' = x^2 \cos y\)。
  2. \(y' + 3 y = \sin x\)。
  3. \(y' = x + y\)。
  4. \(y' = e^{x - y}\)。
  5. \(y' + x y = x y^2\)。
  6. \(x y' + 2 y = x e^x\)。

答案：
1. 可分离（\(f(x) = x^2\)、\(g(y) = \cos y\)）。分离变量。
2. 线性（\(p = 3\)、\(q = \sin x\)）。积分因子 \(\mu = e^{3x}\)。
3. 线性（写成 \(y' - y = x\)，\(p = -1\)、\(q = x\)）。不是可分离！\(x + y\) 不能拆成 \(x\) 因子乘 \(y\) 因子。
4. 可分离（\(e^{x-y} = e^x \cdot e^{-y}\)）。分离变量。不是线性，因为 \(e^{-y}\) 是 \(y\) 的非线性函数。
5. 既不线性也不可分离。这是一个伯努利方程 \(y' + x y = x y^2\)，可以用换元 \(u = 1/y\) 化成线性。本书后面会教。
6. 线性（化标准形 \(y' + (2/x) y = e^x\)）。积分因子 \(\mu = e^{\int 2/x \, dx} = e^{2 \ln x} = x^2\)。注意 \(x \neq 0\) 的限制。

如果你能在 5 秒内做出这些判断，你就建立起了"看到方程先分类"的反射弧。这是后面所有章节的基础工作流程：先分类、再选方法。

临走前再总结一句关于"积分因子"这件事的精神。我们的策略并不是去硬解原方程，而是让 \(\mu y\) 这个新的复合对象成为方程的主角。一旦把方程改写成 \((\mu y)' = \mu q\)，左边就退化为一个"完整的导数"——剩下的求解就只是积分。这种"通过引入一个辅助函数把方程的左边变成完整的导数"的思路，在更高级的微分方程理论中被推广为恰当方程 (exact equation) 的概念，再往上推到流体力学和热力学就是积分形式的守恒律。本章只是这条思路的最朴素入门。

到这里第一阶段三章告一段落。我们已经从"什么是微分方程"出发，覆盖了一阶方程的两大主力解法——可分离与线性。靠这两件工具，第 1 章列出的所有经典模型都能完整求解。下一章我们升级到二阶方程，那里将出现更丰富的现象：振动、共振、阻尼、特征方程、叠加原理。本章里反复强调的"线性方程的解结构 = 齐次通解 + 非齐次特解"那一句话，到了二阶将变成核心定理。先把这一章的内容内化掉再前进——尤其是积分因子的推导步骤、3.7 节的分类反射、以及"代回去验证"的习惯。这三件事内化了，剩下的就是熟练度问题了。
