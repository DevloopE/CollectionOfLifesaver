第 4 章 一阶方程的应用与建模

到这里为止,你已经认识了三种解一阶方程的工具:可分离变量(2.2 节)、一阶线性方程的积分因子(3.2 节)、以及恰当方程及其积分因子(3.5 节)。这是一套相当齐全的弹药。但如果一直只是面对"求解 \(y' = xy + e^x\)"这种已经写好的方程,微分方程就和填字游戏没什么本质区别——拿到题、识别类型、套公式、写答案。这不是这门课真正想教你的事。

真正的事情是这样的:你面前是一杯刚倒出来的咖啡,大约 \(85^\circ\)C,你想知道它要多久才会降到能入口的 \(60^\circ\)C;你面前是一个池塘,有水流不断进来又不断流出,水里溶有某种化学物质,你想知道两小时后池塘里这种物质的浓度是多少;你面前是一个充电中的电容,你想知道经过电阻和电容串联,电压随时间是怎么爬升的。这些问题没有"\(y'\)"出现在题面上。它们是关于世界的问题。把它们翻译成微分方程,是这一章要教的核心技能。这套技能叫做建模。

建模这件事的难处,不在数学,而在翻译:把"咖啡降温"这句中文翻译成 \(\frac{dT}{dt} = -k(T - T_a)\) 这条数学语句。一旦翻译完成,后面就是把第 2、3 章学过的方法用一遍。所以你应该把这一章看作一座桥:左边是物理、化学、生物、工程的实际问题,右边是你已经会的求解技术,中间这条桥就是建模。本章我们将做以下几件事:

  • 给出一套通用的建模步骤,把"翻译"这件玄学的事拆成可重复的流程;
  • 详细推演六个经典模型:牛顿冷却、人口增长(指数与 Logistic)、混合容器、RC 电路、含空气阻力的自由落体、连续复利;
  • 引入一类特别重要的方程——自治方程 \(y' = f(y)\)——它们的右端不显含 \(t\),很多物理模型自然地落入这一类;
  • 对自治方程发展一套不解方程也能看出长期行为的工具:相直线;
  • 把这套工具用到 Logistic 方程上,看一个完整的"定性分析"是什么样子。

这一章里你会反复看到一句话:"先把方程写出来。" 不论你最终能否解出来,把方程写出来本身就已经是把物理问题数学化的关键一步——它让模糊的因果直觉变成了可以用代数操作的对象。许多研究问题至今都只能写到方程为止,还没有解析解,但仅仅写出方程就已经使我们能用计算机模拟它、能问"在什么参数范围内行为会改变",就已经能换来巨大的洞察。所以建模本身就是回报,即使解方程是后话。

4.1 建模的一般步骤

建模听起来玄,但其实有套路。把它拆开来看,大致是五步:

  1. 识别因变量与自变量。 先问自己:我关心的"会随什么变化的量"是什么?它依赖于谁?咖啡问题里,因变量是温度 \(T\),自变量是时间 \(t\)。池塘问题里,因变量是某种物质的总量 \(Q\),自变量是 \(t\)。这一步看起来浅,实际上经常是初学者最容易翻车的地方——你必须想清楚自己究竟在求什么。
  2. 写出物理(或化学、生物)规律。 这一步用到学科背景。牛顿冷却定律说"温差越大,降温越快";质量守恒说"出去的减进来的等于积累";基尔霍夫定律说"环路上的电压代数和为零"。这些规律告诉你"\(\frac{dT}{dt}\)"或者"\(\frac{dQ}{dt}\)"在每一刻应该等于什么。
  3. 把规律化为微分方程。 上一步说出口的那句"等于什么",直接写成等式,就是一个微分方程。注意把单位、符号都带上,常常你会看到"输入率"和"输出率"这种语言,把它们各自表达成 \(t\)(或者别的变量)的函数,然后做减法。
  4. 解方程。 这步是前几章的内容。识别这是什么类型——可分离?线性?恰当?——然后用对应的方法。如果方程超出本章学过的类型,有时也得求助数值方法或定性分析。
  5. 回到原始问题验证并解读。 解出 \(T(t)\) 之后,问"\(t \to \infty\) 时 \(T\) 趋于什么?这合理吗?如果 \(t = 0\) 时温度是 \(85^\circ\),公式给出的是 \(85^\circ\) 吗?"如果初始条件代回去对不上,你某处算错了;如果长期行为不合理(比如咖啡变得比室温还冷),你模型本身可能写错了。这一步是对自己的检查。

第 5 步常被忽略,但它是把"解题机器"和"建模的人"区分开来的关键。一个会解方程但从不回头检查的人,只是个被动的运算员;一个把答案放回物理直觉里反复检验的人,才真的在建模。后面每一道题我都会做完这一步,你也应该养成这个习惯。

建模还有两个附加注意点。第一,近似从一开始就在那里。"咖啡的温度在每一时刻是均匀的"是一个近似(实际上靠近杯壁会冷一些);"流入的盐水瞬间和池水混合均匀"也是一个近似(实际上需要时间扩散)。模型不是真实,模型是真实的可计算简化。这种简化决定了模型的适用范围。第二,参数要从数据来。牛顿冷却定律里有个常数 \(k\),它取决于杯子材质、表面积等,不能凭空而来。通常我们要么从实验测一个数据点把 \(k\) 标定出来,要么从已知的物理参数算出来。本章的例题里我都会明示这一步。

4.2 经典模型详解

把上面五步用到六个经典模型上。每一个都演完整一遍——识别变量、写规律、解方程、解读。

4.2.1 牛顿冷却:咖啡降温

设你刚倒出一杯咖啡,温度 \(T_0 = 90^\circ\)C,把它放在 \(T_a = 20^\circ\)C 的房间里。十分钟后你测得它已经是 \(70^\circ\)C。问:再过多久它会降到 \(40^\circ\)C(差不多是入口温度)?

第一步:因变量是咖啡温度 \(T\),自变量是时间 \(t\)(分钟)。

第二步:物理规律是牛顿冷却定律——一个物体的温度变化率,与它和环境温度之差成正比。用一句话写:温差越大,降得越快;温差为零,就不再降。这条直觉对应的方程是

\[ \frac{dT}{dt} = -k(T - T_a), \]

其中 \(k > 0\) 是一个比例常数,由材料、表面积等决定。注意符号:\(T > T_a\) 时(咖啡比房间热),右端为负,温度下降;\(T < T_a\) 时(假设你在烤箱里放了块冰),右端为正,温度上升。这正符合直觉。

第三步:方程已经写出来了。这是一个一阶线性方程(右端展开是 \(-kT + kT_a\)),也是可分离的。我们用分离变量法解:

\[ \frac{dT}{T - T_a} = -k\, dt. \]

两端积分,\(\ln|T - T_a| = -kt + C_1\),指数化得到

\[ T - T_a = A e^{-kt}, \quad A = \pm e^{C_1}. \]

用初始条件 \(T(0) = T_0\) 定 \(A\):\(T_0 - T_a = A\)。所以解是

\[ \boxed{\,T(t) = T_a + (T_0 - T_a)\, e^{-kt}.\,} \]

这个公式值得记住。它说:咖啡的"超出室温的部分"按指数衰减。当 \(t \to \infty\),\(e^{-kt} \to 0\),\(T \to T_a\)——咖啡最终降到室温,完全合理。\(t = 0\) 时,\(T = T_a + (T_0 - T_a) = T_0\),初始条件也对得上。第 5 步的两个检查都通过了。

第四步还差一步:具体常数 \(k\) 没定。我们只能从实验数据来标定。题目说 \(t = 10\) 时 \(T = 70\)。代入:

\[ 70 = 20 + 70 e^{-10k}, \quad e^{-10k} = \frac{50}{70} = \frac{5}{7}. \]

所以 \(-10k = \ln(5/7)\),\(k = \frac{1}{10}\ln(7/5) \approx 0.0336\) /分钟。

现在回答原问题。要 \(T = 40\):

\[ 40 = 20 + 70 e^{-kt}, \quad e^{-kt} = \frac{20}{70} = \frac{2}{7}, \quad t = \frac{1}{k}\ln(7/2). \]

代入 \(k\):\(t = \frac{\ln(7/2)}{(1/10)\ln(7/5)} = \frac{10 \ln(7/2)}{\ln(7/5)} \approx \frac{10 \times 1.2528}{0.3365} \approx 37.2\) 分钟。所以从倒出来到能喝,大约需要 37 分钟。把它分成 10 分钟+ 27 分钟,前 10 分钟降了 \(20^\circ\),后 27 分钟才再降 \(30^\circ\)——降温速率随温差变小而变慢,这正是指数衰减的特征。

常见错误。 一个新手很容易写成 \(\frac{dT}{dt} = -kT\),漏掉了室温 \(T_a\)。这样的方程意味着"咖啡最终降到 \(0^\circ\)",显然不对。物理规律说的是"温差驱动",所以方程里必须出现温差 \(T - T_a\),而不是 \(T\) 本身。每次写完这种方程,都先在脑子里测试"\(t \to \infty\) 给出什么",看是否合理。

4.2.2 人口模型:指数增长与 Logistic 方程

这是建模史上最经典的两个模型,也是Logistic 方程的入门处。我们一步步搭起来。

第一个模型(马尔萨斯,1798):指数增长。 设 \(P(t)\) 是某种人口在时刻 \(t\) 的数量。最朴素的假设是:每个个体在单位时间内贡献相同的"净增长率"——即出生减死亡。如果这个净增长率是 \(r\),那么单位时间内的总增量就是 \(rP\):

\[ \frac{dP}{dt} = rP. \]

这是可分离的,解是 \(P(t) = P_0 e^{rt}\),其中 \(P_0 = P(0)\)。指数增长。

这个模型在短时间、人口远低于环境承载量时确实成立。1900 年到 1950 年的世界人口大致就符合 \(r \approx 0.01\) /年的指数增长。但显然它不可能永远成立——按 \(r = 0.01\) /年外推,世界人口在大约 7 万年后会比可观测宇宙的所有原子还多。这种荒谬让我们必须修正模型。

第二个模型(Verhulst,1838):Logistic 方程。 物理上的修正是:当人口接近某个环境承载量 \(K\) 时(食物有限、空间有限、传染病等等),净增长率应该下降;到达 \(K\) 时,净增长率为零;超过 \(K\) 时,净增长率应为负(死亡多于出生)。最简单能实现这件事的修改,是把 \(r\) 换成一个随 \(P\) 线性下降的函数 \(r(1 - P/K)\):

\[ \boxed{\,\frac{dP}{dt} = rP\left(1 - \frac{P}{K}\right).\,} \]

这就是Logistic 方程。检查它的物理意义:\(P\) 很小时 \(P/K \approx 0\),括号 \(\approx 1\),方程退化为 \(dP/dt \approx rP\),即指数增长——和马尔萨斯模型在小人口时一致。\(P = K\) 时,括号 \(= 0\),\(dP/dt = 0\),人口稳定。\(P > K\) 时,括号 \(< 0\),\(dP/dt < 0\),人口下降。三种情形都符合直觉。

现在解它。这是一个可分离方程:

\[ \frac{dP}{P(1 - P/K)} = r\, dt. \]

左端的部分分式分解。让 \(\frac{1}{P(1 - P/K)} = \frac{A}{P} + \frac{B}{1 - P/K}\)。两端乘 \(P(1 - P/K)\):\(1 = A(1 - P/K) + BP\)。代 \(P = 0\):\(A = 1\)。代 \(P = K\):\(BK = 1\),即 \(B = 1/K\)。所以

\[ \frac{1}{P(1 - P/K)} = \frac{1}{P} + \frac{1/K}{1 - P/K}. \]

把它积分:

\[ \int \left(\frac{1}{P} + \frac{1/K}{1 - P/K}\right) dP = \ln|P| - \ln|1 - P/K| = \ln\left|\frac{P}{1 - P/K}\right|. \]

(中间用了链式法则:\(\int \frac{1/K}{1 - P/K}\,dP = -\ln|1 - P/K|\)。)所以

\[ \ln\left|\frac{P}{1 - P/K}\right| = rt + C. \]

指数化、合并常数,设 \(0 < P < K\)(初始人口未超过承载量),

\[ \frac{P}{1 - P/K} = A e^{rt}. \]

用 \(P(0) = P_0\) 定 \(A\):\(A = P_0/(1 - P_0/K)\)。把这个 \(A\) 代回去,然后从 \(\frac{P}{1 - P/K} = A e^{rt}\) 解出 \(P\):

\[ P = (1 - P/K) A e^{rt}, \quad P\left(1 + \frac{A e^{rt}}{K}\right) = A e^{rt}, \quad P = \frac{A e^{rt}}{1 + A e^{rt}/K}. \]

把 \(A\) 代入并整理(分子分母同时除以某个量),得到一个比较干净的形式:

\[ \boxed{\,P(t) = \frac{K\, P_0}{P_0 + (K - P_0)\, e^{-rt}}.\,} \]

这个公式值得我们认真读。\(t = 0\) 时,分母 \(= P_0 + (K - P_0) = K\),分子 \(= K P_0\),所以 \(P(0) = P_0\),初始条件对。\(t \to \infty\) 时,\(e^{-rt} \to 0\),分母 \(\to P_0\),所以 \(P \to K\)——人口趋于承载量。曲线形状像一个拉长的"S",起初指数增长,中段减速,最后水平渐近 \(K\)。这就是Logistic 曲线(也叫 S 形曲线、增长饱和曲线),它在生态学、流行病学、技术普及率、产品销售等领域反复出现。

一个具体数值。 设某岛屿能容纳 \(K = 1000\) 只兔子,初始放入 \(P_0 = 100\) 只,内禀增长率 \(r = 0.5\) /年。问 5 年后多少?代:

\[ P(5) = \frac{1000 \times 100}{100 + 900 e^{-2.5}} = \frac{100{,}000}{100 + 900 \times 0.0821} = \frac{100{,}000}{100 + 73.9} \approx 575. \]

5 年后约 575 只,大约是承载量的 60%。再过几年它会更接近 1000。把这个数代回方程检验:右端 \(rP(1-P/K) = 0.5 \times 575 \times 0.425 \approx 122\) 只/年——这意味着它现在每年还以 122 只的速度往上长,几年内就能逼近 1000。一切自洽。

4.5 节我们会用相直线来不解方程地直接看出 Logistic 方程的所有定性行为——你会看到为什么 \(P = K\) 是稳定的、为什么 \(P = 0\) 是不稳定的。

4.2.3 混合容器:盐水稀释

这是化学工程里的原型问题。设一个池子最初装有 100 升纯水。从某时刻起,以 \(2\) 升/分钟的速率往池子里注入浓度为 \(3\) 克/升的盐水,同时以 \(2\) 升/分钟的速率把池中水(已经混合均匀)抽出。问:\(t\) 分钟后,池子里盐的总量 \(Q(t)\) 是多少?最终趋于多少?

第一步:因变量 \(Q(t)\) 是池中盐的总质量(克),自变量是 \(t\)(分钟)。注意我故意用总量而不是浓度;后者只是 \(Q/V\),其中 \(V\) 是水的体积。

第二步:质量守恒。每一时刻,

\[ \frac{dQ}{dt} = (\text{流入率}) - (\text{流出率}). \]

"流入率"是单位时间内盐进入池子的速率:

\[ \text{流入率} = (\text{流入液体速率}) \times (\text{流入液体浓度}) = 2 \text{ 升/分钟} \times 3 \text{ 克/升} = 6 \text{ 克/分钟}. \]

"流出率"是单位时间内盐离开池子的速率。这一项稍微tricky,因为流出液体的浓度不是常数——它就等于此刻池子的浓度 \(Q/V\)。这里 \(V = 100\)(因为流入流出速率相等,体积不变)。所以

\[ \text{流出率} = 2 \text{ 升/分钟} \times \frac{Q}{100} \text{ 克/升} = \frac{Q}{50} \text{ 克/分钟}. \]

第三步:方程是

\[ \frac{dQ}{dt} = 6 - \frac{Q}{50}. \]

这是一个一阶线性方程,右端可写成 \(\frac{1}{50}(300 - Q)\)。注意这和牛顿冷却的形式一模一样!"流入率减去与 \(Q\) 成正比的流出率"和"室温减去当前温度乘以常数",在数学上是同一种结构。我们也可以直接用上一节的解形式套,但这次我们走积分因子的标准流程,作为复习。改写为

\[ \frac{dQ}{dt} + \frac{1}{50}Q = 6. \]

积分因子 \(\mu = e^{t/50}\)。两端乘 \(\mu\):

\[ \frac{d}{dt}\left(e^{t/50} Q\right) = 6 e^{t/50}. \]

积分:\(e^{t/50} Q = 6 \times 50 e^{t/50} + C = 300 e^{t/50} + C\)。除掉积分因子:

\[ Q(t) = 300 + C e^{-t/50}. \]

用初值 \(Q(0) = 0\)(初始是纯水,无盐):\(0 = 300 + C\),\(C = -300\)。所以

\[ \boxed{\,Q(t) = 300\left(1 - e^{-t/50}\right) \text{ 克}.\,} \]

第五步:检验。\(t = 0\):\(Q = 0\),对。\(t \to \infty\):\(Q \to 300\) 克,意味着浓度 \(\to 3\) 克/升,正好是流入液体的浓度——这非常合理:长时间后,池水必然被稀释到与流入液体同样的浓度。\(t = 50\) 时,\(Q = 300(1 - e^{-1}) \approx 190\) 克,约 63% 的进度;\(t = 100\) 时约 86%;\(t = 200\) 时约 98%。

变化版本:流入流出速率不等。 如果流入是 3 升/分钟而流出仍是 2 升/分钟,那么池子体积就以 \(1\) 升/分钟的速率上涨:\(V(t) = 100 + t\)。这时流出率应为 \(2 \times Q/V = 2Q/(100+t)\),方程变成

\[ \frac{dQ}{dt} + \frac{2}{100 + t} Q = 9 \quad (\text{流入率 } = 3 \times 3 = 9). \]

这仍是线性方程,但系数不再是常数了。积分因子 \(\mu = \exp\!\left(\int \frac{2}{100+t}\, dt\right) = (100+t)^2\),然后照搬流程。这种"体积变化"的版本你会在习题里碰到。要点是:每次都先想清楚"流入率和流出率分别等于什么",再写方程,不要急。

4.2.4 RC 电路:充放电

把一个电源电压 \(V\)、电阻 \(R\)、电容 \(C\) 串联起来,合上开关,就是最简单的电路。这种电路就叫RC 电路。我们要研究电容上的电压 \(v(t)\) 怎样随时间变化。

电学规律有三条要用:

  • 欧姆定律:电阻上的电压 \(= R \times i\),其中 \(i\) 是电流;
  • 电容方程:电容上的电压等于电荷除以电容值,即 \(v = q/C\);电流是电荷的时间导数 \(i = dq/dt\);
  • 基尔霍夫电压定律:沿环路一圈,电压代数和为零。

把这些拼起来。环路上,电源电压等于电阻电压加电容电压:

\[ V = Ri + v = R\frac{dq}{dt} + \frac{q}{C}. \]

用 \(v\) 来作为因变量更方便。\(q = Cv\),\(dq/dt = C\, dv/dt\)。代入:

\[ V = RC \frac{dv}{dt} + v. \]

整理为标准形式:

\[ \boxed{\,RC\frac{dv}{dt} + v = V.\,} \]

这又是一个一阶线性常系数方程。注意它的形式:\(\tau \frac{dv}{dt} + v = V\),其中 \(\tau = RC\) 称为时间常数。除以 \(RC\),得 \(dv/dt = (V - v)/(RC)\)——再一次,"驱动力"\(\propto\)"目标减当前"。和牛顿冷却、和混合容器,完全是同一种数学结构。这种"线性松弛"是物理学里最普适的几种行为之一,值得你把这个形式深深记住:

\[ \frac{dy}{dt} = \frac{1}{\tau}\left(y_\infty - y\right) \quad \Longrightarrow \quad y(t) = y_\infty + (y_0 - y_\infty)e^{-t/\tau}. \]

对比看,RC 电路的解(初始 \(v(0) = 0\),即电容初始未充电)是

\[ v(t) = V\left(1 - e^{-t/(RC)}\right). \]

这是充电过程。\(t \to \infty\),\(v \to V\),电容充满,电流停止。\(t = RC\) 时,\(v \approx 0.632 V\),约六成。\(t = 5RC\) 时,\(v \approx 0.993 V\),已接近 1。这就是为什么工程上常用 \(5\tau\) 作为"充满"的经验时间。

对应的放电过程:把已充电至 \(V_0\) 的电容(没有电源)接到电阻上。这时方程是 \(RC\, dv/dt + v = 0\)(没有外电源驱动),解是 \(v(t) = V_0 e^{-t/(RC)}\):指数衰减。这描述了相机闪光灯放电、电子手表的电压保持等等。

具体例子。 \(R = 1000\) Ω,\(C = 10^{-3}\) F,\(V = 5\) V。时间常数 \(\tau = 1\) 秒。从零充到 \(4\) V 需要多久?

\[ 4 = 5(1 - e^{-t}), \quad e^{-t} = 1/5, \quad t = \ln 5 \approx 1.61 \text{ 秒}. \]

4.2.5 含空气阻力的自由落体

把一个质量 \(m\) 的物体从静止释放,只考虑重力,加速度恒为 \(g\),速度无限增长。这显然和经验不符——跳伞员在打开伞前会加速到一个稳定的"终端速度"然后就不再加速了。原因是空气阻力。

空气阻力的常用建模有两种:在低速下,阻力 \(\propto\) 速度,即 \(F_\text{阻} = bv\);在高速下,阻力 \(\propto\) 速度的平方,\(F_\text{阻} = bv^2\)。我们先做线性版本(更容易解析求解),然后简单提一下平方版本。

设向下为正方向。质量 \(m\) 的物体,受重力 \(mg\)(向下,正)和阻力 \(-bv\)(向上,负,因为阻力总是反方向于运动)。牛顿第二定律 \(m\, dv/dt = F_\text{合}\):

\[ m\frac{dv}{dt} = mg - bv. \]

这是一个一阶线性方程,且和我们看过的"线性松弛"形式完全一样!整理为

\[ \frac{dv}{dt} = g - \frac{b}{m}v = \frac{b}{m}\left(\frac{mg}{b} - v\right). \]

"时间常数"是 \(m/b\),"目标速度"是 \(v_\infty = mg/b\)。这就是终端速度。在这个速度下,重力恰好被阻力抵消,物体不再加速。直接套通解:

\[ v(t) = \frac{mg}{b} + \left(v_0 - \frac{mg}{b}\right) e^{-bt/m}. \]

若静止释放(\(v_0 = 0\)):

\[ \boxed{\,v(t) = \frac{mg}{b}\left(1 - e^{-bt/m}\right).\,} \]

检查:\(t = 0\),\(v = 0\),对。\(t \to \infty\),\(v \to mg/b\),终端速度。早期 \(t \ll m/b\) 时,\(e^{-bt/m} \approx 1 - bt/m\),代入得 \(v \approx gt\)——符合"无空气阻力时自由下落"的直觉,因为短时间内速度还小,阻力还可以忽略。

具体数字。 一个 \(m = 70\) kg 的人从飞机跳出,设 \(b = 14\) kg/s(典型值,带未打开的小阻力)。终端速度 \(v_\infty = 70 \times 9.8 / 14 = 49\) m/s,约 176 km/h——和真实跳伞员未开伞时的速度量级一致。时间常数 \(m/b = 5\) 秒,所以大约 \(5 \times 5 = 25\) 秒后接近终端速度。

位置。 如果你还想知道下落距离 \(y(t)\)(向下为正,初始位置 \(y_0 = 0\)),只需积分一次:

\[ y(t) = \int_0^t v(s)\, ds = \frac{mg}{b}\left[t - \frac{m}{b}(1 - e^{-bt/m})\right]. \]

这一步只用到普通微积分,没有新的方程。许多"两阶段"问题(比如先做了 30 秒自由落体再开伞,问开伞时下落了多远)就用这个公式。

关于平方阻力。 高速下 \(F_\text{阻} = bv^2\),方程是 \(m\,dv/dt = mg - bv^2\)。这仍是可分离的(\(dv/(mg - bv^2)\) 可以分式分解为 \(\arctan\) 或 \(\tanh\) 形式),解出来是 \(v(t) = v_\infty \tanh(gt/v_\infty)\),其中 \(v_\infty = \sqrt{mg/b}\)。形式不同,但定性行为一样:终端速度存在,接近终端速度时呈指数性接近。物理直觉对、代数表达式各异。

4.2.6 利息复利:连续与离散

设你在银行存了 \(P_0\) 元,年利率 \(r\)。如果一年只结算一次("年复利"),一年后本利和是 \(P_0(1 + r)\),\(t\) 年后是 \(P_0(1 + r)^t\)。如果每月结算一次("月复利"),一年后是 \(P_0(1 + r/12)^{12}\)。一般地,每年结算 \(n\) 次:

\[ P(t) = P_0 \left(1 + \frac{r}{n}\right)^{nt}. \]

取 \(n \to \infty\) 的极限,我们得到连续复利:

\[ P(t) = P_0 \lim_{n \to \infty}\left(1 + \frac{r}{n}\right)^{nt} = P_0 e^{rt}. \]

这正是 \(e\) 在金融里出现的原因。值得注意:这个连续复利公式不只是离散公式的极限,它也可以从微分方程直接得到。如果资金增长率每一刻都和余额成比例,

\[ \frac{dP}{dt} = rP, \]

解就是 \(P(t) = P_0 e^{rt}\)。两条路殊途同归。这告诉我们一件事:"连续复利"和"瞬时增长率正比于本金"是同一回事,只是表述不同。

对比一下数字。 \(P_0 = 1000\) 元,\(r = 5\%\),10 年。年复利 \(P = 1000 \times 1.05^{10} \approx 1628.89\) 元;月复利 \(\approx 1000 \times (1 + 0.05/12)^{120} \approx 1647.01\) 元;连续复利 \(1000 e^{0.5} \approx 1648.72\) 元。可以看到:复利越频繁,最终金额越大,但收敛得很快——从月复利到连续复利只差不到 2 元。这是 \(\lim_n (1 + r/n)^n = e^r\) 的实际意义。

带注资。 假设你不止存了一笔,还每年以连续速率 \(D\) 元/年继续注资(比如自动定投)。方程变成

\[ \frac{dP}{dt} = rP + D. \]

这是一阶线性,可以用积分因子 \(\mu = e^{-rt}\):

\[ \frac{d}{dt}(e^{-rt} P) = D e^{-rt}, \quad e^{-rt} P = -\frac{D}{r}e^{-rt} + C. \]

所以 \(P(t) = -D/r + C e^{rt}\),用 \(P(0) = P_0\) 定 \(C = P_0 + D/r\):

\[ P(t) = \left(P_0 + \frac{D}{r}\right) e^{rt} - \frac{D}{r}. \]

检查:\(t = 0\) 给出 \(P_0\),对;\(D = 0\) 退化为纯复利,对。这个公式在退休规划里很常用——给定目标 \(P(T) = M\)、年限 \(T\)、利率 \(r\),你可以反解出每年需要存多少。

4.3 自治方程 \(y' = f(y)\) 与平衡解

回顾一下上一节的六个模型,它们的方程是:

\[ \frac{dT}{dt} = -k(T - T_a), \quad \frac{dP}{dt} = rP\left(1 - \frac{P}{K}\right), \quad \frac{dQ}{dt} = 6 - \frac{Q}{50}, \] \[ \frac{dv}{dt} = g - \frac{b}{m}v, \quad \frac{dP}{dt} = rP, \quad \frac{dv}{dt} = g - \frac{b}{m}v. \]

注意一件事:每一个方程的右端都只依赖于因变量,不显含自变量 \(t\)。这是一类相当特殊也相当广的方程,值得给它一个名字:

自治方程(autonomous equation)是指形如 \(y' = f(y)\) 的方程——右端只依赖 \(y\),不显含自变量 \(t\)。换言之,"规律"本身不会随时间变化,只是当前状态决定下一步的变化率。

大多数物理学的基本规律都是自治的——重力的方向不会因为今天是星期二而变。所以自治方程在建模里出现得特别频繁。非自治的例子也不少,比如带季节性变化的人口模型 \(P' = (r + a\sin(2\pi t))P\),或者带交流电源的 RC 电路 \(RC\, v' + v = V_0 \sin(\omega t)\)——这些右端显含 \(t\)。

自治方程有一个特别简单的"平衡"概念。称 \(y = y^*\) 是方程 \(y' = f(y)\) 的平衡解(也叫稳态解、定常解、临界点),如果 \(f(y^*) = 0\)。这意味着:如果系统初始就在 \(y^*\),它将永远停留在 \(y^*\)——因为变化率为零,什么都不会动。

找平衡解超级容易:解代数方程 \(f(y) = 0\) 即可,不用碰任何积分。来几个例子。

例 1:牛顿冷却 \(T' = -k(T - T_a)\)。\(f(T) = -k(T - T_a) = 0\) 当且仅当 \(T = T_a\)。唯一平衡解是 \(T = T_a\)——咖啡的温度等于室温时,它就不变了。这正是你的物理直觉。

例 2:Logistic 方程 \(P' = rP(1 - P/K)\)。\(rP(1 - P/K) = 0\) 当且仅当 \(P = 0\) 或 \(P = K\)。两个平衡解:\(P = 0\)(灭绝)和 \(P = K\)(承载量)。这两个状态都是"如果初始在那里,就一直在那里"的。

例 3:含线性阻力的下落 \(v' = g - bv/m\)。\(g - bv/m = 0\) 给出 \(v = mg/b\)——终端速度。这正是我们之前算出的结果,但现在我们看到它从一个更高的视角自然出现:它就是这个自治方程的唯一平衡解。

例 4:稍微不同的方程 \(y' = y^2 - 4\)。\(y^2 - 4 = 0\) 给出 \(y = 2\) 或 \(y = -2\)。两个平衡解。

例 5:\(y' = \sin y\)。\(\sin y = 0\) 给出 \(y = n\pi\),\(n \in \mathbb{Z}\)。无穷多平衡解。

找出平衡解只回答了一个问题:"哪些状态是不动的?"它没回答更重要的问题:"如果初始不在平衡解,那会怎么样?"咖啡如果初始 \(85^\circ\),会向 \(20^\circ\) 的平衡解靠近,这是稳定的——附近的状态会被吸引过来。但是 Logistic 方程的 \(P = 0\) 平衡呢?如果你引入一对兔子(\(P\) 离 \(0\) 很近但不为零),它们会迅速繁殖、远离 \(0\),所以 \(P = 0\) 是不稳定的——附近的状态会被排斥出去。这种"吸引/排斥"的对比,要用下一节的相直线工具来系统化地分析。

4.4 自治方程的相直线

给一个自治方程 \(y' = f(y)\),不解它就能看出它的所有解的定性行为——这个工具叫相直线(phase line)。它是 1 维"相图"。

构造方法分三步:

  1. 在一条 \(y\) 轴(画成竖直的或水平的都行,本书我们画成竖直的——上面 \(y\) 大,下面 \(y\) 小)上,标出所有平衡解 \(y^*\)。
  2. 在每一段(平衡解之间的区间)里,看 \(f(y)\) 的符号。\(f(y) > 0\) 那段,\(y\) 增大,在轴上画一个向上的箭头;\(f(y) < 0\) 那段,\(y\) 减小,画向下的箭头。
  3. 根据箭头方向判断每个平衡解的稳定性:
  • 如果两边的箭头都指向该平衡解(下方箭头向上,上方箭头向下),它是稳定(stable)的。附近的解会朝它收敛。
  • 如果两边的箭头都背离该平衡解(下方箭头向下,上方箭头向上),它是不稳定(unstable)的。附近的解会远离它。
  • 如果一边吸引另一边排斥,它是半稳定(semi-stable)的——一侧的解会收敛,另一侧的解会远离。

这套工具的优点是:你只用判断 \(f(y)\) 在几个区间里的符号,就能定性地知道任何初始条件下的解长期会走向哪里。不用解方程。

例 1:\(y' = -2y\)。平衡解 \(y = 0\)。\(y > 0\) 时 \(f = -2y < 0\),箭头向下;\(y < 0\) 时 \(f > 0\),箭头向上。两边箭头都指向 \(y = 0\),所以 \(y = 0\) 是稳定的。任何初始条件最终都收敛到 \(0\)。把这一结论和精确解 \(y = y_0 e^{-2t}\) 对照——确实如此。

例 2:\(y' = 2y\)。平衡解 \(y = 0\)。\(y > 0\) 时 \(f > 0\),箭头向上;\(y < 0\) 时 \(f < 0\),箭头向下。两边都背离 \(0\),所以 \(y = 0\) 不稳定。任何非零初值都会跑向 \(\pm \infty\)。精确解 \(y = y_0 e^{2t}\) 验证了这点。

例 3:\(y' = y^2 - 4 = (y-2)(y+2)\)。平衡解 \(y = 2\) 和 \(y = -2\)。在每个区间看符号:

  • \(y > 2\):\((+)(+) > 0\),箭头向上;
  • \(-2 < y < 2\):\((-)(+) < 0\),箭头向下;
  • \(y < -2\):\((-)(-) > 0\),箭头向上。

看 \(y = 2\):下方(\(-2 < y < 2\))箭头向下、上方(\(y > 2\))箭头向上,两边都背离——不稳定。看 \(y = -2\):下方(\(y < -2\))向上、上方(\(-2 < y < 2\))向下,两边都指向——稳定。

结论:任何 \(y_0 < -2\) 的初值都会向 \(-2\) 收敛;\(-2 < y_0 < 2\) 的也都收敛到 \(-2\);\(y_0 > 2\) 的会往 \(+\infty\) 跑(实际上是有限时间发散,但这不影响定性结论);恰在 \(y_0 = -2\) 或 \(y_0 = 2\) 的就停在那。这一切都不需要解方程。事实上 \(y' = y^2 - 4\) 的精确解是 \(y(t) = 2\frac{1 + Ce^{4t}}{1 - Ce^{4t}}\)(分离变量、部分分式),长得很丑;但相直线告诉我们的"哪些初值收敛到哪、哪些发散",一目了然。

例 4:\(y' = y(1-y)^2\)。先找平衡:\(y = 0\) 或 \(y = 1\)。看符号:

  • \(y > 1\):\((+)(+)^2 > 0\),向上;
  • \(0 < y < 1\):\((+)(+)^2 > 0\),向上;
  • \(y < 0\):\((-)(+)^2 < 0\),向下。

看 \(y = 0\):下方向下、上方向上,两边背离,不稳定。看 \(y = 1\):下方向上(吸引)、上方也向上(排斥)——半稳定。这种半稳定情况经常被新手漏掉:一侧吸引、另一侧排斥。它对应平衡点恰好是 \(f(y)\) 的二重零点(\((1-y)^2\) 让 \(y = 1\) 是二重根)。

稳定性的导数判据。 有一个更快的方法判断稳定性:计算 \(f'(y^*)\)。若 \(f'(y^*) < 0\),稳定;若 \(f'(y^*) > 0\),不稳定;若 \(f'(y^*) = 0\),不能下结论(要看高阶或看相直线)。

为什么?把 \(f(y)\) 在 \(y^*\) 附近做一阶 Taylor 展开:\(f(y) \approx f(y^*) + f'(y^*)(y - y^*) = f'(y^*)(y - y^*)\)。设 \(z = y - y^*\),\(z' \approx f'(y^*) z\)。如果 \(f'(y^*) < 0\),这是衰减(\(z \to 0\),即 \(y \to y^*\));\(f'(y^*) > 0\) 是增长(\(y\) 远离 \(y^*\))。这就是线性化稳定性分析的最基本版本,在二阶系统里我们会看到它的二维版本。

用导数判据复查例 3 的两个平衡点。\(f(y) = y^2 - 4\),\(f'(y) = 2y\)。\(f'(2) = 4 > 0\),不稳定;\(f'(-2) = -4 < 0\),稳定。和相直线给的结论一致。

4.5 例:用相直线分析 Logistic 方程

这一节我们把刚搭好的工具,应用到本章的明星方程——Logistic 方程

\[ \frac{dP}{dt} = rP\left(1 - \frac{P}{K}\right) = f(P), \quad r > 0,\; K > 0, \]

从而不解方程就回答这个方程下人口的所有可能行为。

第一步:平衡解。 \(f(P) = 0\) 当且仅当 \(P = 0\) 或 \(P = K\)。两个。

第二步:符号分析。 在三段区间里看 \(f(P)\):

  • \(P > K\):\(P > 0\) 但 \(1 - P/K < 0\),所以 \(f < 0\),箭头向下。
  • \(0 < P < K\):\(P > 0\) 且 \(1 - P/K > 0\),所以 \(f > 0\),箭头向上。
  • \(P < 0\):\(P < 0\) 且 \(1 - P/K > 0\),所以 \(f < 0\),箭头向下。

(虽然 \(P < 0\) 在物理上没意义——人口怎么可能为负——但数学上方程允许,我们仍可以分析其相直线。)

第三步:稳定性。

  • \(P = 0\):下方箭头向下(背离),上方箭头向上(背离)。两边都背离——不稳定。
  • \(P = K\):下方箭头向上(指向),上方箭头向下(指向)。两边都指向——稳定。

用导数判据复核:\(f'(P) = r(1 - 2P/K)\)。\(f'(0) = r > 0\),不稳定;\(f'(K) = r(1 - 2) = -r < 0\),稳定。一致。

第四步:解读。 结论是:

  • 任何正的初始人口 \(P_0 > 0\)(无论是 1 只兔子还是 999 只),都会随时间趋于承载量 \(K\)。这正是 Logistic 模型的根本结论:种群在长期内会饱和到环境承载量。
  • \(P_0 = 0\) 是平衡解但不稳定。理论上"零兔子"会一直为零,但任何小扰动(放进一对兔子)都会使种群爆发式离开,迈向 \(K\)。这刻画了一个不稳定平衡的本性:你"理论上"可以维持它,但现实里不可能维持。
  • \(P_0 = K\) 也是平衡解,而且是稳定的——这是种群的"自然栖息状态"。

这就是不解方程能告诉我们的全部定性图像。把这些和我们 4.2.2 节算出的精确解

\[ P(t) = \frac{KP_0}{P_0 + (K - P_0)e^{-rt}} \]

对照,是完全一致的:\(t \to \infty\),\(e^{-rt} \to 0\),\(P \to K\)。这就是相直线工具的力量——它给出的定性结论是对的,而且只用到关于 \(f(P)\) 符号的几次心算,不需要任何积分。

解曲线的形状。 相直线进一步告诉我们解的图形长什么样。在 \(0 < P_0 < K\) 区间里,\(P\) 一路单调上升到 \(K\),不可能"过冲"或"振荡"——一阶自治方程的解永远是单调的(在两个相邻平衡解之间,符号不变,所以 \(dP/dt\) 不会改号)。

什么时候上升最快?求 \(f(P) = rP(1 - P/K)\) 的最大值。一阶条件 \(f'(P) = r(1 - 2P/K) = 0\) 给出 \(P = K/2\)。所以解曲线在 \(P = K/2\) 时上升最快——这是 Logistic 曲线的拐点(inflection point)。在拐点之前,曲线"加速"上升;拐点之后,曲线"减速"上升,渐近 \(K\)。这种"先加速、后减速"的 S 形,是 Logistic 模型的视觉签名。

带捕捞的 Logistic 模型。 作为最后一个细节,看一个稍复杂的版本。如果某种鱼按照 Logistic 增长,但同时被以恒定速率 \(h\) 捕捞:

\[ \frac{dP}{dt} = rP\left(1 - \frac{P}{K}\right) - h. \]

这是 \(P\) 的一个二次函数(开口向下),以 \(h\) 的大小不同,根的情况会变。

  • 若 \(h\) 小:\(f(P) = -\frac{r}{K}P^2 + rP - h\) 有两个正根 \(P_-\) 和 \(P_+\)(\(P_- < P_+\))。看符号易知 \(P_-\) 不稳定、\(P_+\) 稳定。所以只要初始人口 \(P_0 > P_-\),系统会稳到一个低于 \(K\) 的新平衡 \(P_+\)。这就是可持续渔业的状态。
  • 临界:\(h\) 增大到某个 \(h_c\),两根合并成一个二重根 \(P_c = K/2\)(因为最大可能增长率出现在 \(P = K/2\),\(h_c = rK/4\))。这是分岔点。
  • 若 \(h > h_c\):\(f(P) < 0\) 恒成立,无平衡解,所有初值都向 \(-\infty\) 走——种群灭绝。

这是分岔分析(bifurcation analysis)的入门例子:模型的某个参数缓慢变化,导致平衡解的数目和稳定性质发生质变。在生态学、医学、气候学里,这种"参数过临界即崩溃"的现象有重大现实意义——所谓"过度捕捞"和"种群崩溃",数学背后就是这种分岔。

这一章我们把一阶方程的工具用到现实问题里,认识到:许多看似不同的物理过程(冷却、稀释、充电、阻力下落)在数学上是同一种"线性松弛";Logistic 方程及其变体捕捉了"有限承载"的本质;而相直线是分析自治方程长期行为的强力工具——不解方程也能看出整个图像。下一章我们升级到二阶方程,你将看到一个全新的世界:振荡。线性松弛只能趋近,二阶系统可以振荡,而振荡正是物理学半壁江山(波、电磁、量子)的基本语言。
