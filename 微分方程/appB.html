附录 B 常用积分与变换表

这份附录是一份翻查手册。我们建议你做题之前先尝试自己回忆和推导；只有当你卡住或者要节省时间的时候，再翻这里。把这些表硬背下来不是目标——理解它们怎么来、知道哪一条对应哪种结构，才是。

B.1 基本积分速查

以下都是不定积分。常数 \(C\) 全部省略。\(a, n\) 是常数，\(a > 0\) 时取主分支。

被积函数原函数条件
\(x^n\)\(\dfrac{x^{n+1}}{n + 1}\)\(n \ne -1\)
\(\dfrac{1}{x}\)\(\ln|x|\)\(x \ne 0\)
\(e^{ax}\)\(\dfrac{1}{a} e^{ax}\)\(a \ne 0\)
\(a^x\)\(\dfrac{a^x}{\ln a}\)\(a > 0, a \ne 1\)
\(\sin(ax)\)\(-\dfrac{1}{a}\cos(ax)\)\(a \ne 0\)
\(\cos(ax)\)\(\dfrac{1}{a}\sin(ax)\)\(a \ne 0\)
\(\tan x\)\(-\ln|\cos x| = \ln|\sec x|\)
\(\cot x\)\(\ln|\sin x|\)
\(\sec x\)\(\ln|\sec x + \tan x|\)
\(\csc x\)\(-\ln|\csc x + \cot x|\)
\(\sec^2 x\)\(\tan x\)
\(\csc^2 x\)\(-\cot x\)
\(\sec x \tan x\)\(\sec x\)
\(\sinh(ax)\)\(\dfrac{1}{a} \cosh(ax)\)
\(\cosh(ax)\)\(\dfrac{1}{a} \sinh(ax)\)
\(\dfrac{1}{a^2 + x^2}\)\(\dfrac{1}{a} \arctan\dfrac{x}{a}\)\(a > 0\)

\(\dfrac{1}{\sqrt{x^2 + a^2}}\)\(\ln(x + \sqrt{x^2 + a^2}) = \mathrm{arsinh}\dfrac{x}{a}\)
\(\dfrac{1}{\sqrt{x^2 - a^2}}\)\(\ln|x + \sqrt{x^2 - a^2}|\)\(|x| > a\)
\(\dfrac{1}{x \sqrt{x^2 - a^2}}\)\(\dfrac{1}{a}\mathrm{arcsec}\dfrac{|x|}{a}\)\(|x| > a\)
\(\dfrac{1}{a^2 - x^2}\)\(\dfrac{1}{2a} \ln\left|\dfrac{a + x}{a - x}\right|\)\(|x| \ne a\)

常用三角积分（用倍角降幂）：

\[ \int \sin^2 x \, dx = \frac{x}{2} - \frac{\sin 2x}{4}, \qquad \int \cos^2 x \, dx = \frac{x}{2} + \frac{\sin 2x}{4}. \]

分部积分（IBP）口诀：

\[ \int u \, dv = u v - \int v \, du. \]

典型用法：

\[ \int x \, e^{ax}\, dx = \frac{e^{ax}}{a^2}(ax - 1), \qquad \int x \sin(ax)\, dx = \frac{\sin(ax)}{a^2} - \frac{x \cos(ax)}{a}, \] \[ \int e^{ax} \cos(bx)\, dx = \frac{e^{ax}(a\cos bx + b \sin bx)}{a^2 + b^2}, \qquad \int e^{ax} \sin(bx)\, dx = \frac{e^{ax}(a\sin bx - b \cos bx)}{a^2 + b^2}. \]

B.2 拉普拉斯变换表

定义：

\[ F(s) = \mathcal{L}\{f(t)\} = \int_0^\infty e^{-st} f(t)\, dt. \]

下表按字母（中文拼音）顺序，但每条都同时给出 \(f(t)\) 和 \(F(s)\)。\(a, n, \omega\) 是常数。

\(f(t)\)\(F(s) = \mathcal{L}\{f\}\)条件
\(1\)\(\dfrac{1}{s}\)\(s > 0\)
\(t\)\(\dfrac{1}{s^2}\)\(s > 0\)
\(t^n\)（\(n\) 整数）\(\dfrac{n!}{s^{n+1}}\)\(s > 0\)
\(t^a\)（\(a > -1\)）\(\dfrac{\Gamma(a+1)}{s^{a+1}}\)\(s > 0\)
\(e^{at}\)\(\dfrac{1}{s - a}\)\(s > a\)
\(t e^{at}\)\(\dfrac{1}{(s - a)^2}\)\(s > a\)
\(t^n e^{at}\)\(\dfrac{n!}{(s - a)^{n+1}}\)\(s > a\)
\(\sin(\omega t)\)\(\dfrac{\omega}{s^2 + \omega^2}\)\(s > 0\)
\(\cos(\omega t)\)\(\dfrac{s}{s^2 + \omega^2}\)\(s > 0\)
\(\sinh(at)\)\(\dfrac{a}{s^2 - a^2}\)\(s > |a|\)
\(\cosh(at)\)\(\dfrac{s}{s^2 - a^2}\)\(s > |a|\)
\(e^{at}\sin(\omega t)\)\(\dfrac{\omega}{(s - a)^2 + \omega^2}\)\(s > a\)
\(e^{at}\cos(\omega t)\)\(\dfrac{s - a}{(s - a)^2 + \omega^2}\)\(s > a\)
\(t \sin(\omega t)\)\(\dfrac{2 \omega s}{(s^2 + \omega^2)^2}\)\(s > 0\)
\(t \cos(\omega t)\)\(\dfrac{s^2 - \omega^2}{(s^2 + \omega^2)^2}\)\(s > 0\)
单位阶跃 \(u(t - a)\)\(\dfrac{e^{-as}}{s}\)\(a \ge 0\)
\(\delta(t - a)\)（Dirac）\(e^{-as}\)\(a \ge 0\)
\(u(t - a) f(t - a)\)\(e^{-as} F(s)\)（位移定理）
\(e^{at} f(t)\)\(F(s - a)\)（频域位移）
\(t \cdot f(t)\)\(-F'(s)\)
\(\dfrac{f(t)}{t}\)（若极限存在）\(\displaystyle\int_s^\infty F(\sigma) \, d\sigma\)
\(\displaystyle\int_0^t f(\tau)\, d\tau\)\(\dfrac{F(s)}{s}\)
\(f'(t)\)\(s F(s) - f(0)\)
\(f''(t)\)\(s^2 F(s) - s f(0) - f'(0)\)
\((f * g)(t) = \displaystyle\int_0^t f(\tau) g(t - \tau)\,d\tau\)\(F(s) G(s)\)（卷积定理）

常用反变换技巧速查：

  • 分式分解：把 \(F(s)\) 拆成简单部分（\(\frac{A}{s - a}\)、\(\frac{Bs + C}{s^2 + \omega^2}\) 等），逐项查表。
  • 配方：\(s^2 + bs + c = (s + b/2)^2 + (c - b^2/4)\)，把 \(F\) 化成"位移过的"标准形式。
  • 位移定理：\(F(s - a) \leftrightarrow e^{at} f(t)\)，\(e^{-as} F(s) \leftrightarrow u(t - a) f(t - a)\)——遇到 \(e^{-as}\) 想阶跃，遇到 \(F(s - a)\) 想 \(e^{at}\) 包裹。
  • 卷积：当 \(F = G \cdot H\) 而 \(G, H\) 都易反时，\(f = g * h\)。

B.3 傅里叶变换简表

定义（对称约定）：

\[ \hat{f}(\omega) = \mathcal{F}\{f\}(\omega) = \int_{-\infty}^{\infty} f(t)\, e^{-i\omega t}\, dt, \qquad f(t) = \frac{1}{2\pi}\int_{-\infty}^{\infty} \hat{f}(\omega)\, e^{i\omega t}\, d\omega. \]

不同教材的归一化约定略有不同；对应做替换即可。

\(f(t)\)\(\hat{f}(\omega)\)
\(\delta(t)\)\(1\)
\(1\)\(2\pi \delta(\omega)\)
\(e^{i\omega_0 t}\)\(2\pi \delta(\omega - \omega_0)\)
\(\cos(\omega_0 t)\)\(\pi[\delta(\omega - \omega_0) + \delta(\omega + \omega_0)]\)
\(\sin(\omega_0 t)\)\(-i\pi[\delta(\omega - \omega_0) - \delta(\omega + \omega_0)]\)
矩形脉冲 \(\mathrm{rect}(t/T)\)（宽 \(T\)）\(T\, \mathrm{sinc}(\omega T / 2) = T \dfrac{\sin(\omega T/2)}{\omega T/2}\)
\(e^{-a |t|}\)（\(a > 0\)）\(\dfrac{2a}{a^2 + \omega^2}\)
\(e^{-a t^2}\)（\(a > 0\)）\(\sqrt{\pi/a}\, e^{-\omega^2/(4a)}\)（高斯自变换）
\(e^{-at} u(t)\)（\(a > 0\)）\(\dfrac{1}{a + i\omega}\)
\(\dfrac{1}{a^2 + t^2}\)（\(a > 0\)）\(\dfrac{\pi}{a} e^{-a |\omega|}\)
\(f'(t)\)\(i \omega \hat{f}(\omega)\)
\(t f(t)\)\(i \dfrac{d\hat{f}}{d\omega}\)
\(f(t - t_0)\)\(e^{-i\omega t_0} \hat{f}(\omega)\)
\(e^{i\omega_0 t} f(t)\)\(\hat{f}(\omega - \omega_0)\)
\((f * g)(t)\)\(\hat{f}(\omega) \hat{g}(\omega)\)
\(f(t) g(t)\)\(\dfrac{1}{2\pi} (\hat{f} * \hat{g})(\omega)\)

Parseval 等式（能量守恒）：

\[ \int_{-\infty}^{\infty} |f(t)|^2\, dt = \frac{1}{2\pi}\int_{-\infty}^{\infty} |\hat{f}(\omega)|^2\, d\omega. \]

B.4 三角恒等式速查

毕氏：

\[ \sin^2 x + \cos^2 x = 1, \quad 1 + \tan^2 x = \sec^2 x, \quad 1 + \cot^2 x = \csc^2 x. \]

和差化积/角加法：

\[ \sin(\alpha \pm \beta) = \sin\alpha\cos\beta \pm \cos\alpha\sin\beta, \] \[ \cos(\alpha \pm \beta) = \cos\alpha\cos\beta \mp \sin\alpha\sin\beta, \] \[ \tan(\alpha \pm \beta) = \dfrac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha\tan\beta}. \]

倍角：

\[ \sin 2x = 2 \sin x \cos x, \qquad \cos 2x = \cos^2 x - \sin^2 x = 2\cos^2 x - 1 = 1 - 2\sin^2 x, \] \[ \tan 2x = \frac{2\tan x}{1 - \tan^2 x}. \]

半角（降幂）：

\[ \sin^2 x = \frac{1 - \cos 2x}{2}, \qquad \cos^2 x = \frac{1 + \cos 2x}{2}. \]

积化和差：

\[ \sin\alpha \sin\beta = \tfrac{1}{2}[\cos(\alpha - \beta) - \cos(\alpha + \beta)], \] \[ \cos\alpha \cos\beta = \tfrac{1}{2}[\cos(\alpha - \beta) + \cos(\alpha + \beta)], \] \[ \sin\alpha \cos\beta = \tfrac{1}{2}[\sin(\alpha + \beta) + \sin(\alpha - \beta)]. \]

和差化积：

\[ \sin\alpha + \sin\beta = 2 \sin\tfrac{\alpha+\beta}{2}\cos\tfrac{\alpha-\beta}{2}, \] \[ \cos\alpha + \cos\beta = 2 \cos\tfrac{\alpha+\beta}{2}\cos\tfrac{\alpha-\beta}{2}, \] \[ \cos\alpha - \cos\beta = -2 \sin\tfrac{\alpha+\beta}{2}\sin\tfrac{\alpha-\beta}{2}. \]

双曲恒等式（注意符号差异）：

\[ \cosh^2 x - \sinh^2 x = 1, \qquad \cosh 2x = \cosh^2 x + \sinh^2 x, \qquad \sinh 2x = 2 \sinh x \cosh x. \]

Euler 公式（一切三角的根）：

\[ e^{ix} = \cos x + i \sin x, \qquad \cos x = \frac{e^{ix} + e^{-ix}}{2}, \qquad \sin x = \frac{e^{ix} - e^{-ix}}{2i}. \]

B.5 微分公式速查

\(f(x)\)\(f'(x)\)
\(x^n\)\(n x^{n-1}\)
\(e^{ax}\)\(a e^{ax}\)
\(a^x\)\(a^x \ln a\)
\(\ln x\)\(\dfrac{1}{x}\)
\(\log_a x\)\(\dfrac{1}{x \ln a}\)
\(\sin x\)\(\cos x\)
\(\cos x\)\(-\sin x\)
\(\tan x\)\(\sec^2 x\)
\(\cot x\)\(-\csc^2 x\)
\(\sec x\)\(\sec x \tan x\)
\(\csc x\)\(-\csc x \cot x\)
\(\arcsin x\)\(\dfrac{1}{\sqrt{1 - x^2}}\)
\(\arccos x\)\(-\dfrac{1}{\sqrt{1 - x^2}}\)
\(\arctan x\)\(\dfrac{1}{1 + x^2}\)
\(\sinh x\)\(\cosh x\)
\(\cosh x\)\(\sinh x\)
\(\tanh x\)\(\mathrm{sech}^2 x\)

三大复合规则：

\[ (uv)' = u'v + uv', \qquad \left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}, \qquad (f \circ g)'(x) = f'(g(x)) g'(x). \]

隐函数微分：对 \(F(x, y) = 0\) 求导得 \(F_x + F_y \cdot y' = 0\)，所以 \(y' = -F_x / F_y\)。

常用一阶 ODE 模板的封闭形式（备查）：

  • \(y' + p(t) y = q(t)\)：积分因子 \(\mu = \exp(\int p\, dt)\)，解 \(y = \dfrac{1}{\mu}\left[\int \mu q\, dt + C\right]\)。
  • \(y' = f(t) g(y)\)：分离 \(\dfrac{dy}{g(y)} = f(t)\, dt\)，两边积分。
  • \(y' + p y = q y^n\)（Bernoulli）：代换 \(v = y^{1-n}\) 化为线性。
  • 恰当方程 \(M\, dx + N\, dy = 0\)，若 \(M_y = N_x\)，存在 \(F\) 使 \(F_x = M, F_y = N\)，解为 \(F(x, y) = C\)。

常用二阶常系数 ODE 模板：

  • \(a y'' + b y' + c y = 0\)：特征 \(a r^2 + br + c = 0\)。两实根 → \(y = c_1 e^{r_1 t} + c_2 e^{r_2 t}\)；重根 → \((c_1 + c_2 t) e^{r t}\)；复根 \(\alpha \pm i\beta\) → \(e^{\alpha t}(c_1 \cos\beta t + c_2 \sin\beta t)\)。
  • 非齐次：用待定系数（外驱是多项式 / 指数 / 三角）或参数变易（外驱是其他）。
