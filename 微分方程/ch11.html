第 11 章 阶跃、冲击与卷积

第 9-10 章已经把拉普拉斯方法的"基本盘"打通。本章把这套方法的覆盖范围再扩大一截,用来处理三种新对象:

  • 单位阶跃函数 \(u(t - a)\)——它在 \(t = a\) 之前是 \(0\),之后是 \(1\)。物理上对应"在 \(t = a\) 时刻打开开关"或"加上一个一直作用下去的常力"。带它你就能优雅地处理分段右端项,把"分段对接"那种笨重做法扔掉。
  • Dirac \(\delta\) 函数 \(\delta(t - a)\)——一个在 \(t = a\) 处"无穷尖、无穷瘦,但总积分等于 \(1\)" 的"广义函数"。它是"瞬时单位冲量"的数学化:锤子敲一下、撞球瞬间撞击、电路里的瞬时火花,都能用 \(\delta\) 来建模。
  • 卷积 \((f * g)(t) = \int_0^t f(\tau) g(t - \tau) d\tau\)——两个函数的一种特殊"乘积",在 \(s\)-域里恰好对应普通的乘积:\(\mathcal{L}\{f * g\} = F(s) G(s)\)。这个事实把"两个 \(s\)-域分式相乘的反变换"问题转化成时间域的积分。

章节安排:

  • 11.1-11.4 阶跃函数:定义、变换、\(t\)-平移定理、应用到分段右端的 ODE。
  • 11.5-11.7 \(\delta\) 函数:定义、物理直觉、变换、冲击响应。
  • 11.8-11.10 卷积:定义、卷积定理、应用例题。

11.1 单位阶跃函数

定义:

\[ \boxed{\,u(t - a) = \begin{cases} 0, & t < a \\ 1, & t \ge a \end{cases}.\,} \]

它也叫 Heaviside 函数(以英国电气工程师 Oliver Heaviside 命名),记号有时也写作 \(H(t - a)\) 或 \(u_a(t)\)。本书一律用 \(u(t - a)\)。

函数图像就是 \(t < a\) 时贴在 \(t\)-轴上,到 \(t = a\) 瞬间"跳"到 \(1\),然后保持 \(1\) 一直延伸到右边。一个忽然打开就再不关掉的开关。

它最重要的用处是把分段函数写成单一表达式。比如

\[ g(t) = \begin{cases} 3, & 0 \le t < 2 \\ 5, & 2 \le t < 7 \\ 0, & t \ge 7 \end{cases} \]

看上去像一段"楼梯"。怎么写成单个表达式?思路:每一次"台阶"的变化都用一个阶跃来制造。

从 \(g \equiv 3\) 开始(在 \(t = 0\) 之后)。在 \(t = 2\) 处要把 \(g\) 从 \(3\) 加到 \(5\),也就是加一个 \(2\),用 \(2 u(t - 2)\)。在 \(t = 7\) 处要把 \(g\) 从 \(5\) 减到 \(0\),也就是减 \(5\),用 \(-5 u(t - 7)\)。所以

\[ g(t) = 3 + 2 u(t - 2) - 5 u(t - 7). \]

用一个值检验:\(t = 1\) 时 \(g = 3 + 0 - 0 = 3\) ✓;\(t = 4\) 时 \(g = 3 + 2 - 0 = 5\) ✓;\(t = 10\) 时 \(g = 3 + 2 - 5 = 0\) ✓。

这种"用阶跃把楼梯堆起来"的技术,是拉普拉斯能优雅处理分段函数的全部秘密。

另一种常见模式:在某段时间内"打开"一个函数。比如想让 \(\sin t\) 只在 \([2, 5]\) 这段时间内出现,其他时间为零。这是

\[ \sin t \cdot [u(t - 2) - u(t - 5)]. \]

方括号里那个差就是一个"窗"——在 \([2, 5]\) 上是 \(1\),其它时间是 \(0\)。乘以 \(\sin t\) 就把 \(\sin t\) "切"出来这一段。这个"窗函数 = 两个阶跃之差"的写法以后会大量见到。

11.2 阶跃函数的拉普拉斯

用定义直接积分:

\[ \mathcal{L}\{u(t - a)\} = \int_0^\infty e^{-st} u(t - a) dt = \int_a^\infty e^{-st} dt \]

(因为 \(u(t - a) = 0\) 在 \([0, a)\) 上,所以那一段贡献为零)

\[ = \left[-\frac{1}{s} e^{-st}\right]_a^\infty = \frac{e^{-as}}{s} \quad (s > 0). \]

所以

\[ \boxed{\,\mathcal{L}\{u(t - a)\} = \frac{e^{-as}}{s} \quad (a \ge 0).\,} \]

注意 \(a = 0\) 时 \(u(t) \equiv 1\)(在 \([0, \infty)\) 上),公式给 \(1/s\),跟 \(\mathcal{L}\{1\}\) 一致。

这条公式让分段函数的变换变得机械化。继续上面的例子,\(g(t) = 3 + 2 u(t - 2) - 5 u(t - 7)\):

\[ \mathcal{L}\{g(t)\} = \frac{3}{s} + \frac{2 e^{-2s}}{s} - \frac{5 e^{-7s}}{s} = \frac{3 + 2 e^{-2s} - 5 e^{-7s}}{s}. \]

这跟第 9 章那道"分段函数硬积分"的例子比起来——一行就完了。

11.3 \(t\)-平移定理(第二平移定理)

这是阶跃函数的"主菜"。它处理的是函数 \(f\) 被 \(u(t - a)\) "切"出来后再变换的情况。

\[ \boxed{\,\mathcal{L}\{u(t - a) f(t - a)\} = e^{-as} F(s).\,} \]

注意函数必须是 \(f(t - a)\) 而不是 \(f(t)\)——也就是说,\(f\) 也要做相同的"右移"。这是初学者最容易栽的地方,后面专门讲。

证明:用定义。

\[ \mathcal{L}\{u(t - a) f(t - a)\} = \int_0^\infty e^{-st} u(t - a) f(t - a) dt = \int_a^\infty e^{-st} f(t - a) dt. \]

换元 \(\tau = t - a\),\(d\tau = dt\),当 \(t = a\) 时 \(\tau = 0\),\(t \to \infty\) 时 \(\tau \to \infty\):

\[ = \int_0^\infty e^{-s(\tau + a)} f(\tau) d\tau = e^{-as} \int_0^\infty e^{-s\tau} f(\tau) d\tau = e^{-as} F(s). \]

完。这个定理跟第 9 章的 \(s\)-平移定理 \(\mathcal{L}\{e^{at} f(t)\} = F(s - a)\) 形成对偶:

  • \(s\)-平移:在时间域乘 \(e^{at}\),在 \(s\)-域右移 \(a\)。
  • \(t\)-平移:在时间域右移 \(a\)(并用 \(u\) 切掉前面),在 \(s\)-域乘 \(e^{-as}\)。

这两个对称性是拉普拉斯几何的核心。

用法举例 11.3.1。求 \(\mathcal{L}\{u(t - 3) (t - 3)^2\}\)。这正是 \(t\)-平移定理的标准形式:\(f(t) = t^2\),\(a = 3\)。

\[ F(s) = \mathcal{L}\{t^2\} = \frac{2}{s^3}, \quad \mathcal{L}\{u(t - 3)(t - 3)^2\} = e^{-3s} \cdot \frac{2}{s^3} = \frac{2 e^{-3s}}{s^3}. \]

用法举例 11.3.2(常见陷阱)。求 \(\mathcal{L}\{u(t - 2) t^2\}\)。注意:这里函数是 \(t^2\),不是 \((t - 2)^2\)。要套定理,必须先把 \(t^2\) 重写成"\((t - 2) + 2\) 的某个东西":

\[ t^2 = ((t - 2) + 2)^2 = (t - 2)^2 + 4(t - 2) + 4. \]

所以

\[ u(t - 2) t^2 = u(t - 2)(t - 2)^2 + 4 u(t - 2)(t - 2) + 4 u(t - 2), \]

对每一项分别用定理(\(F(s)\) 分别是 \(2/s^3, 1/s^2, 1/s\)):

\[ \mathcal{L}\{u(t - 2) t^2\} = e^{-2s}\left(\frac{2}{s^3} + \frac{4}{s^2} + \frac{4}{s}\right). \]

这个"先把函数重新表达成 \(t - a\) 的多项式"的小动作非常关键。如果你硬算 \(\mathcal{L}\{u(t - 2) t^2\}\) 而不重写,直接套定理就错了——你得到的是 \(\mathcal{L}\{u(t - 2)(t - 2)^2\}\),那是另一个函数。

反方向:已知 \(F(s)\) 含 \(e^{-as}\),逆变换。比如 \(\mathcal{L}^{-1}\{e^{-3s}/s^2\}\)。识别 \(e^{-3s}\)(对应 \(a = 3\))乘以 \(1/s^2\)(对应 \(t\) 的变换),所以 \(F(s) = 1/s^2 \to f(t) = t\),逆变换是 \(u(t - 3)(t - 3)\)——也就是从 \(t = 3\) 开始增加的"斜坡"。

规律:看到 \(e^{-as}\) 出现,先把它"摘下来",对剩下的部分做反变换得到 \(f(t)\),最后把答案写成 \(u(t - a) f(t - a)\)(把 \(t\) 都换成 \(t - a\),前面套个 \(u\))。

11.4 求解分段右端的 ODE

把 11.1-11.3 的全部技巧用到一道实际题上。考虑一个被"踢一下"的弹簧:

例 11.4.1。求解

\[ y'' + y = \begin{cases} 1, & 0 \le t < \pi \\ 0, & t \ge \pi \end{cases}, \quad y(0) = 0, \quad y'(0) = 0. \]

(这就是第 10.4.4 节预告过的题。)

第一步,把右端写成阶跃函数表达式。这相当于一个"\([0, \pi]\) 内为 \(1\)"的窗:\(f(t) = 1 - u(t - \pi)\)。变换:\(F(s) = 1/s - e^{-\pi s}/s\)。

第二步,变换方程(注意零初值):\(s^2 Y + Y = F(s)\),所以

\[ (s^2 + 1) Y = \frac{1}{s} - \frac{e^{-\pi s}}{s} \implies Y = \frac{1}{s(s^2 + 1)} - \frac{e^{-\pi s}}{s(s^2 + 1)}. \]

第三步,反变换。先做 \(1/(s(s^2 + 1))\) 的部分分式:

\[ \frac{1}{s(s^2 + 1)} = \frac{A}{s} + \frac{Bs + C}{s^2 + 1}. \]

通分:\(A(s^2 + 1) + (Bs + C) s = 1\),也就是 \((A + B) s^2 + Cs + A = 1\)。所以 \(A = 1, C = 0, B = -1\)。

\[ \frac{1}{s(s^2 + 1)} = \frac{1}{s} - \frac{s}{s^2 + 1}. \]

反变换:\(\mathcal{L}^{-1}\{1/(s(s^2 + 1))\} = 1 - \cos t\)。

第二项 \(e^{-\pi s}/(s(s^2 + 1))\) 用 \(t\)-平移:\(\mathcal{L}^{-1}\{e^{-\pi s} \cdot G(s)\} = u(t - \pi) g(t - \pi)\),其中 \(g(t) = 1 - \cos t\)。所以这一项的反变换是 \(u(t - \pi)(1 - \cos(t - \pi))\),又因 \(\cos(t - \pi) = -\cos t\),化简为 \(u(t - \pi)(1 + \cos t)\)。

合并:

\[ y(t) = (1 - \cos t) - u(t - \pi)(1 + \cos t). \]

分段写出来更清楚:

\[ y(t) = \begin{cases} 1 - \cos t, & 0 \le t < \pi \\ (1 - \cos t) - (1 + \cos t) = -2\cos t, & t \ge \pi \end{cases}. \]

第一段是从静止开始,弹簧被一个常力推动,慢慢被推大;到 \(t = \pi\) 力撤掉,弹簧成为自由振动 \(-2\cos t\),振幅是 \(2\)。这是"先施压一段时间,再松开,看自由振动"的标准物理图像。

检验:\(t = \pi^-\) 时 \(y = 1 - \cos\pi = 2\),\(t = \pi^+\) 时 \(y = -2\cos\pi = 2\),连续 ✓;\(y'(\pi^-) = \sin\pi = 0\),\(y'(\pi^+) = 2\sin\pi = 0\),也连续 ✓。"撤掉外力的瞬间"的位置和速度都被自动接好了——这是拉普拉斯的好处:不需要手工对接,代数自动处理。

例 11.4.2(矩形脉冲下的电路)。一个 RC 电路 \(RC \frac{dq}{dt} + q = V(t)\),\(R = 1, C = 1, q(0) = 0\),其中

\[ V(t) = \begin{cases} V_0, & 0 \le t < T \\ 0, & t \ge T \end{cases}. \]

方程化为 \(q' + q = V_0 (1 - u(t - T))\)。变换:

\[ sQ - 0 + Q = V_0\left(\frac{1}{s} - \frac{e^{-Ts}}{s}\right) \implies (s + 1) Q = \frac{V_0(1 - e^{-Ts})}{s}. \] \[ Q(s) = \frac{V_0}{s(s + 1)} - \frac{V_0 e^{-Ts}}{s(s + 1)}. \]

部分分式 \(1/(s(s + 1)) = 1/s - 1/(s + 1)\),反变换 \(1 - e^{-t}\)。所以 \(\mathcal{L}^{-1}\{V_0/(s(s + 1))\} = V_0(1 - e^{-t})\)。第二项含 \(e^{-Ts}\),反变换是 \(V_0 u(t - T)(1 - e^{-(t - T)})\)。合并:

\[ q(t) = V_0(1 - e^{-t}) - V_0 u(t - T)(1 - e^{-(t - T)}). \]

分段:

\[ q(t) = \begin{cases} V_0(1 - e^{-t}), & 0 \le t < T \\ V_0(1 - e^{-t}) - V_0(1 - e^{-(t - T)}) = V_0(e^{-(t - T)} - e^{-t}) = V_0 e^{-t}(e^T - 1), & t \ge T \end{cases}. \]

第一段:电容慢慢充电(指数趋向 \(V_0\))。第二段:电压撤掉后,电容上的电荷指数衰减。这正是 RC 充放电的标准曲线。整个过程一个公式包揽,非常优雅。

11.5 Dirac \(\delta\) 函数

从一个物理问题入手。一根弹簧上的物体,在 \(t = 0\) 时被锤子瞬间敲了一下。"瞬间"是什么意思?如果敲击持续 \(\tau\) 秒,产生总冲量 \(I\)(动量改变量),那么作用力大约是 \(I/\tau\)。当 \(\tau \to 0\)(敲得"越来越快"),力会无限大,但积分(也就是冲量)始终是 \(I\)。我们想要描写这种"无限大、无限短、积分有限"的力。

这就是 \(\delta\) 函数想干的事。Dirac \(\delta\) 函数(以物理学家 Paul Dirac 命名)的"形式定义"是:

\[ \delta(t - a) = \begin{cases} 0, & t \ne a \\ \infty, & t = a \end{cases}, \quad \int_{-\infty}^\infty \delta(t - a) dt = 1. \]

"在 \(t = a\) 处无穷尖,其它地方为零,但积分等于 \(1\)。" 这显然不是一个普通函数——任何一个有限值的函数,改在某一点取无穷,积分还是不变;\(\delta\) 必须是一种"极限对象"。严格来讲,\(\delta\) 是一个分布(distribution),数学上有完整的处理(L. Schwartz 在 1940 年代开创),不过对工程使用我们只需要它的几个性质,完全不必动用分布理论。

怎么把它当成"极限对象"理解?设

\[ \delta_\epsilon(t) = \begin{cases} 1/\epsilon, & 0 \le t \le \epsilon \\ 0, & \text{其它} \end{cases}. \]

这是一个高 \(1/\epsilon\)、宽 \(\epsilon\) 的矩形脉冲,面积始终是 \(1\)。当 \(\epsilon \to 0\),它越来越尖、越来越瘦,极限就是 \(\delta(t)\) 的"行为"。

\(\delta\) 函数最重要的性质是"挑选性"(sifting property):对任何在 \(t = a\) 处连续的函数 \(f\),

\[ \boxed{\,\int_{-\infty}^\infty f(t) \delta(t - a) dt = f(a).\,} \]

意思是:\(\delta(t - a)\) 把 \(f\) 在 \(t = a\) 处的值"挑出来"。证明思路:\(\delta\) 在 \(t \ne a\) 处为零,所以积分只在 \(t = a\) 一点有贡献,而那一点 \(f \approx f(a)\),把常数 \(f(a)\) 提出来,剩下 \(\int \delta = 1\),得 \(f(a)\)。

11.6 物理解释:\(\delta\) 是"瞬时单位冲量"

回到锤子敲弹簧。设方程是 \(m y'' + ky = F(t)\),其中 \(F(t)\) 是受力。如果在 \(t = a\) 处给出一个总冲量 \(I\) 的瞬时打击,我们写

\[ F(t) = I \delta(t - a). \]

"\(I\) 倍的 \(\delta\) 函数集中在 \(t = a\)"——这就是冲量为 \(I\) 的瞬时冲击的标准建模。注意:\(F(t)\) 在物理上是力,\(I\) 是冲量(力对时间的积分),量纲对得上。

另一种常见情形:打开一个开关瞬间产生火花、撞击瞬间产生力——这些都用 \(\delta\) 建模。基本上凡是"持续时间相对系统响应时间小到可以忽略"的输入,都该写成 \(\delta\)。

11.7 \(\delta\) 的拉普拉斯与冲击响应

用挑选性算 \(\mathcal{L}\{\delta(t - a)\}\):

\[ \mathcal{L}\{\delta(t - a)\} = \int_0^\infty e^{-st} \delta(t - a) dt = e^{-as} \quad (a \ge 0). \]

所以

\[ \boxed{\,\mathcal{L}\{\delta(t - a)\} = e^{-as}.\,} \]

取 \(a = 0\) 得 \(\mathcal{L}\{\delta(t)\} = 1\)。注意这是一个不寻常的"变换":\(F(s) = 1\) 当 \(s \to \infty\) 时不趋于零!这违反了第 9.3 节我们提到的"普通函数的拉普拉斯变换在无穷远处必趋零"。理由是 \(\delta\) 不是普通函数——它是分布。这种"反常"恰恰是 \(\delta\) 的本质标志。

下面来看 \(\delta\) 在 ODE 里的应用:冲击响应。考虑一个被瞬时冲量 \(1\) 击打的振子(假设系统从静止):

\[ y'' + 2 \zeta \omega_0 y' + \omega_0^2 y = \delta(t), \quad y(0) = 0, \quad y'(0) = 0. \]

解 \(y(t)\) 称为冲击响应函数(impulse response),记作 \(h(t)\)。它的物理意义:在系统静止状态下,给一个瞬时的单位冲量,它后续的运动模样。

例 11.7.1。求 \(y'' + y = \delta(t)\),\(y(0) = 0, y'(0) = 0\) 的解。

变换:\(s^2 Y + Y = 1\),\((s^2 + 1) Y = 1\),\(Y = 1/(s^2 + 1)\),反变换 \(y(t) = \sin t\)。

所以单位冲量产生 \(\sin t\) 的振动,振幅恰好是 \(1\)。物理解释:冲量 \(I = 1\) 给质量为 \(m = 1\) 的振子带来速度 \(\Delta v = I/m = 1\)。所以系统初始条件其实变成了 \(y(0) = 0, y'(0) = 1\)(\(\delta\) 等价于"瞬时把速度提到 1"),自由振动方程的解就是 \(\sin t\)。

这是一个非常重要的洞见:冲击响应等价于"重置初始条件,然后做自由振动"。具体来说,在 \(y'' + py' + qy = \delta(t)\)、零初值下,\(\delta\) 的效果是把 \(y'(0)\) 从 \(0\) 推到 \(1\)(因为冲量给 \(y'\) 一个"跳跃")。这跟"齐次方程 + 初值 \(y(0) = 0, y'(0) = 1\)" 等价。

例 11.7.2。求 \(y'' + 4y = 3\delta(t - \pi)\),\(y(0) = 1, y'(0) = 0\) 的解。

变换:\(s^2 Y - s + 4 Y = 3 e^{-\pi s}\),\((s^2 + 4) Y = s + 3 e^{-\pi s}\),

\[ Y = \frac{s}{s^2 + 4} + \frac{3 e^{-\pi s}}{s^2 + 4}. \]

反变换:第一项 \(\cos(2t)\)。第二项用 \(t\)-平移,\(\mathcal{L}^{-1}\{1/(s^2 + 4)\} = (1/2)\sin(2t)\),所以 \(\mathcal{L}^{-1}\{3 e^{-\pi s}/(s^2 + 4)\} = (3/2) u(t - \pi) \sin(2(t - \pi)) = (3/2) u(t - \pi) \sin(2t)\)(用了 \(\sin(2t - 2\pi) = \sin(2t)\))。合并:

\[ y(t) = \cos(2t) + \frac{3}{2} u(t - \pi) \sin(2t). \]

分段:

  • 在 \([0, \pi)\) 上,\(y = \cos(2t)\)——自由振动,初值 \(y(0) = 1, y'(0) = 0\)。
  • 在 \([\pi, \infty)\) 上,\(y = \cos(2t) + (3/2)\sin(2t)\)——\(\pi\) 时刻被踹了一脚,从此振幅变成 \(\sqrt{1 + 9/4} = \sqrt{13}/2\)。

物理上:振子安静地振动着,\(t = \pi\) 时一个冲量打过去,瞬间改变了它的速度,从此振动得更猛了。这种"突变"用经典方法处理需要分段对接,用 \(\delta\) 一行式子。

11.8 卷积

本章的第三个新工具。卷积(convolution)是定义在两个函数上的二元运算:

\[ \boxed{\,(f * g)(t) = \int_0^t f(\tau) g(t - \tau) d\tau.\,} \]

记号 \(f * g\) 读作 "\(f\) 卷积 \(g\)"。它是一个关于 \(t\) 的函数,定义在 \([0, \infty)\) 上(我们在拉普拉斯的语境里只考虑右半轴)。

这个定义看起来挺特别——为什么要这样积?直觉上,\((f * g)(t)\) 是把 \(g\) 的"过去历史"用 \(f\) 加权叠加起来。\(g(t - \tau)\) 表示"\(\tau\) 时间以前的 \(g\) 值",\(f(\tau)\) 是权重,从 \(\tau = 0\) 到 \(\tau = t\) 积分就是把这些"历史 \(g\)"以 \(f\) 为加权叠起来。这种"用过去信号预测现在响应"的模式在工程里到处都是——线性时不变系统的输出永远是输入与冲击响应的卷积。

卷积有几条重要性质:

  • 交换律:\(f * g = g * f\)。证:在卷积积分里换元 \(\sigma = t - \tau\),\(d\sigma = -d\tau\),得 \(\int_0^t g(\sigma) f(t - \sigma) d\sigma = (g * f)(t)\)。
  • 分配律:\(f * (g + h) = f * g + f * h\)。来自积分的线性性。
  • 结合律:\((f * g) * h = f * (g * h)\)。证明稍长,但成立。
  • 常数提取:\((cf) * g = c(f * g)\)。

注意:卷积不满足"\(f * 1 = f\)"——这是初学者容易误会的。\(f * 1 = \int_0^t f(\tau) d\tau\),是 \(f\) 的积分而非 \(f\) 本身。卷积也没有简单的"乘法单位元"——除非你接受 \(f * \delta = f\)(我们 11.10 节再说)。

来算一个具体的卷积。设 \(f(t) = e^{at}\),\(g(t) = e^{bt}\)(\(a \ne b\)),求 \(f * g\)。

\[ (f * g)(t) = \int_0^t e^{a\tau} e^{b(t - \tau)} d\tau = e^{bt} \int_0^t e^{(a - b)\tau} d\tau = e^{bt} \cdot \frac{e^{(a - b) t} - 1}{a - b} = \frac{e^{at} - e^{bt}}{a - b}. \]

这是一个有用的小公式:两个不同指数的卷积是它们的差除以指数差。注意当 \(a = b\) 时这个公式不能用(分母为零),但能取极限:用洛必达对 \(a\) 求导得 \(t e^{at}\),也就是 \(e^{at} * e^{at} = t e^{at}\)。

11.9 卷积定理

本章的明珠。卷积定理说:

\[ \boxed{\,\mathcal{L}\{f * g\} = F(s) G(s).\,} \]

用文字说:卷积在时间域里是一个积分,在 \(s\)-域里只是普通的乘法。这是为什么 \(s\)-域如此让人感动——它把困难的时间域积分变成了代数。

证明用变换变量的双重积分:

\[ F(s) G(s) = \left(\int_0^\infty e^{-s\tau} f(\tau) d\tau\right) \left(\int_0^\infty e^{-su} g(u) du\right). \]

合并成一个双重积分(在 \(\tau, u\) 都从 \(0\) 到 \(\infty\) 的"第一象限"上):

\[ F(s) G(s) = \iint_{\tau \ge 0, u \ge 0} e^{-s(\tau + u)} f(\tau) g(u) d\tau du. \]

换元 \(t = \tau + u\)(\(t\) 是新积分变量),保留 \(\tau\) 不动,\(u = t - \tau\)。雅可比 \(\partial(\tau, u)/\partial(\tau, t) = 1\),所以 \(d\tau du = d\tau dt\)。新区域:\(\tau \ge 0, t \ge \tau\),也就是 \(0 \le \tau \le t < \infty\)。

\[ F(s) G(s) = \int_0^\infty e^{-st} \left(\int_0^t f(\tau) g(t - \tau) d\tau\right) dt = \int_0^\infty e^{-st} (f * g)(t) dt = \mathcal{L}\{f * g\}. \]

就这么干净。整个证明的关键是"换变量从 \((\tau, u)\) 到 \((\tau, t)\),交换积分顺序"——这是双重积分的标准技巧,你在多元微积分课里见过。

用刚才算的 \(e^{at} * e^{bt} = (e^{at} - e^{bt})/(a - b)\) 验证一下定理:

\[ \mathcal{L}\{e^{at} * e^{bt}\} = \mathcal{L}\left\{\frac{e^{at} - e^{bt}}{a - b}\right\} = \frac{1}{a - b}\left(\frac{1}{s - a} - \frac{1}{s - b}\right) = \frac{1}{a - b} \cdot \frac{(s - b) - (s - a)}{(s - a)(s - b)} = \frac{1}{(s - a)(s - b)}. \]

另一边,\(F(s) G(s) = (1/(s - a))(1/(s - b)) = 1/((s - a)(s - b))\)。一致 ✓。

11.10 卷积应用:难反变换的 \(F(s) G(s)\)

卷积定理的最重要用法是这样:你解出来 \(Y(s)\) 是某个 \(F(s) G(s)\) 的乘积,而 \(F\) 和 \(G\) 各自的反变换都好做,但合在一起的 \(FG\) 反变换比较麻烦(比如部分分式很丑)。这时候

\[ y(t) = \mathcal{L}^{-1}\{F(s) G(s)\} = (f * g)(t), \]

把问题转化为时间域的卷积积分。

例 11.10.1。求 \(\mathcal{L}^{-1}\{1/((s^2 + 1)^2)\}\)。

把分母拆成两个因子:\(1/((s^2 + 1)^2) = (1/(s^2 + 1)) \cdot (1/(s^2 + 1))\)。每个因子的反变换是 \(\sin t\)。所以由卷积定理

\[ \mathcal{L}^{-1}\{1/((s^2 + 1)^2)\} = \sin t * \sin t = \int_0^t \sin\tau \sin(t - \tau) d\tau. \]

用积化和差 \(\sin A \sin B = (1/2)(\cos(A - B) - \cos(A + B))\),取 \(A = \tau, B = t - \tau\):

\[ \sin\tau \sin(t - \tau) = \tfrac{1}{2}(\cos(2\tau - t) - \cos t). \] \[ \int_0^t \sin\tau \sin(t - \tau) d\tau = \tfrac{1}{2} \int_0^t \cos(2\tau - t) d\tau - \tfrac{1}{2} \int_0^t \cos t \, d\tau \] \[ = \tfrac{1}{2} \cdot \frac{\sin(2\tau - t)}{2} \Big|_0^t - \tfrac{1}{2} t \cos t = \tfrac{1}{4}(\sin t - \sin(-t)) - \tfrac{t \cos t}{2} = \tfrac{\sin t}{2} - \tfrac{t \cos t}{2}. \]

所以 \(\mathcal{L}^{-1}\{1/((s^2 + 1)^2)\} = (\sin t - t \cos t)/2\)。

顺便说,这个结果跟用第 9.9 节的 \(t\)-乘公式得到的一样:从 \(\mathcal{L}\{t \cos t\} = (s^2 - 1)/(s^2 + 1)^2\),\(\mathcal{L}\{\sin t\} = 1/(s^2 + 1)\),你可以验证 \((1/2)(\sin t - t \cos t)\) 的变换确实是 \(1/((s^2 + 1)^2)\)。多种方法都能算,卷积是其中一种。

例 11.10.2(冲击响应 = 卷积核)。考虑一个一般的二阶 IVP:

\[ y'' + py' + qy = f(t), \quad y(0) = 0, \quad y'(0) = 0, \]

变换:\((s^2 + ps + q) Y = F(s)\),\(Y = F(s)/(s^2 + ps + q)\)。设 \(H(s) = 1/(s^2 + ps + q)\),则 \(Y = F \cdot H\)。由卷积定理

\[ y(t) = (f * h)(t) = \int_0^t f(\tau) h(t - \tau) d\tau, \]

其中 \(h(t) = \mathcal{L}^{-1}\{H(s)\}\) 正是冲击响应(11.7 节那个)。

这给了一个深刻的物理图像:系统对任意外力 \(f(t)\) 的响应,等于外力与冲击响应的卷积。换句话说,只要你测出系统对一次"敲打"的响应 \(h\),那么它对任何外力的响应都可以写成 \(h * f\)——一种"线性叠加":每个时刻 \(\tau\) 的小冲量 \(f(\tau) d\tau\) 在之后产生 \(h(t - \tau) f(\tau) d\tau\) 的响应,把这些响应全部叠加(也就是积分)就是总响应。这就是工程里"系统的传递函数"和"冲击响应"被如此重视的原因。

例 11.10.3(具体的 \(h * f\))。求 \(y'' + y = e^{-t}\),\(y(0) = 0, y'(0) = 0\) 的解,用卷积。

这里 \(p = 0, q = 1\),\(H(s) = 1/(s^2 + 1)\),\(h(t) = \sin t\)。所以

\[ y(t) = (e^{-t} * \sin t) = \int_0^t e^{-\tau} \sin(t - \tau) d\tau. \]

展开 \(\sin(t - \tau) = \sin t \cos\tau - \cos t \sin\tau\):

\[ y(t) = \sin t \int_0^t e^{-\tau} \cos\tau \, d\tau - \cos t \int_0^t e^{-\tau} \sin\tau \, d\tau. \]

计算两个积分(分部积分,或更省事的用复指数 \(e^{-\tau}(\cos\tau \pm i\sin\tau) = e^{(-1 \pm i)\tau}\) 一次性算两个)。结果:

\[ \int_0^t e^{-\tau} \cos\tau d\tau = \frac{1}{2}(1 - e^{-t}\cos t + e^{-t}\sin t), \] \[ \int_0^t e^{-\tau} \sin\tau d\tau = \frac{1}{2}(1 - e^{-t}\cos t - e^{-t}\sin t). \]

(留给读者验证;用 \(\int e^{-\tau} \cos\tau d\tau = -e^{-\tau}(\cos\tau - \sin\tau)/2\) 等。)代入合并(对 \(\sin t, \cos t, e^{-t}\sin t \cos t\) 几项做一些化简):

\[ y(t) = \frac{1}{2}(\sin t - \cos t + e^{-t}). \]

检验:\(y(0) = (0 - 1 + 1)/2 = 0\) ✓;\(y'(t) = (\cos t + \sin t - e^{-t})/2\),\(y'(0) = (1 + 0 - 1)/2 = 0\) ✓;\(y''(t) = (-\sin t + \cos t + e^{-t})/2\),\(y'' + y = (-\sin t + \cos t + e^{-t})/2 + (\sin t - \cos t + e^{-t})/2 = e^{-t}\) ✓。

当然,这道题用部分分式直接做也行——\(Y = 1/((s + 1)(s^2 + 1))\),拆成 \(A/(s + 1) + (Bs + C)/(s^2 + 1)\),求出 \(A = 1/2, B = -1/2, C = 1/2\),得到一样的答案。卷积只是另一条路径,在某些题里更直观。

例 11.10.4(\(\delta\) 是卷积单位元)。卷积定理告诉我们 \(\mathcal{L}\{f * \delta\} = F(s) \cdot 1 = F(s)\),所以 \(f * \delta = f\)。\(\delta\) 是卷积下的"单位元"。这正符合直觉:把任意函数和"瞬时冲量"卷在一起,等于函数本身——因为 \(\delta\) 把所有东西都"取在 \(t = a\) 一点"。一般地

\[ f * \delta(\cdot - a) = f(t - a) \]

(读者用挑选性自己验证)。这个事实在信号处理里经常用——\(\delta\) 是"原始信号",任何信号都可以看作 \(\delta\) 的"加权积分"(也就是 \(\delta\) 的卷积)。

例 11.10.5(积分方程)。考虑积分方程

\[ y(t) = t + \int_0^t y(\tau) \sin(t - \tau) d\tau. \]

右端的积分正好是 \(y * \sin t\),所以 \(y = t + y * \sin t\)。两边取拉普拉斯:

\[ Y(s) = \frac{1}{s^2} + Y(s) \cdot \frac{1}{s^2 + 1}. \] \[ Y(s) \left(1 - \frac{1}{s^2 + 1}\right) = \frac{1}{s^2} \implies Y(s) \cdot \frac{s^2}{s^2 + 1} = \frac{1}{s^2} \implies Y(s) = \frac{s^2 + 1}{s^4} = \frac{1}{s^2} + \frac{1}{s^4}. \]

反变换:

\[ y(t) = t + \frac{t^3}{6}. \]

检验:\(y'(t) = 1 + t^2/2\),\(y''(t) = t\),\(y(t) - t = t^3/6\),要验证 \(\int_0^t (\tau + \tau^3/6) \sin(t - \tau) d\tau = t^3/6\)——可以做但比较繁,信任拉普拉斯就好。这道题的关键是看出"原方程含有卷积",然后用卷积定理把卷积变成乘法,代数解出 \(Y\)。这种积分方程(unknown function 出现在积分号里)经典方法很难下手,拉普拉斯+卷积定理几乎一招搞定。

例 11.10.6(综合应用——一个被锤子敲的衰减振子)。考虑

\[ y'' + 2y' + 5y = 4 \delta(t - 2), \quad y(0) = 0, \quad y'(0) = 0. \]

变换:\(s^2 Y + 2s Y + 5Y = 4 e^{-2s}\),\((s^2 + 2s + 5) Y = 4 e^{-2s}\)。配方:\(s^2 + 2s + 5 = (s + 1)^2 + 4\)。

\[ Y = \frac{4 e^{-2s}}{(s + 1)^2 + 4} = e^{-2s} \cdot \frac{4}{(s + 1)^2 + 4} = e^{-2s} \cdot 2 \cdot \frac{2}{(s + 1)^2 + 4}. \]

识别 \(2/((s + 1)^2 + 4) = \mathcal{L}\{e^{-t}\sin(2t)\}\)(\(s\)-平移: \(\mathcal{L}\{\sin(2t)\} = 2/(s^2 + 4)\),平移得 \(2/((s + 1)^2 + 4)\))。所以括号里的反变换是 \(2 e^{-t}\sin(2t)\)。再用 \(t\)-平移:

\[ y(t) = u(t - 2) \cdot 2 e^{-(t - 2)} \sin(2(t - 2)). \]

分段:

  • \(t < 2\):\(y = 0\)。系统静止,等待。
  • \(t \ge 2\):\(y = 2 e^{-(t - 2)} \sin(2(t - 2))\)。\(t = 2\) 时被冲量 \(4\) 击打,从此做衰减振动,振幅按 \(e^{-(t - 2)}\) 衰减。

这道题串起了本章所有工具:\(\delta\)(冲击)、\(t\)-平移(因为冲击在 \(t = 2\) 而非 \(t = 0\))、\(s\)-平移(因为有衰减项 \(e^{-t}\))、加上配方处理复根。把它当成本章总练习,值得自己完整再做一遍。

到这里,拉普拉斯方法的"完整工具箱"已经齐了。前 11 章我们走过了一阶 ODE、二阶常系数 ODE、变系数和级数解、系统、再加上拉普拉斯变换的全套——常微分方程在工程和物理里 90% 以上的应用都已经在你的射程之内。

如果你回头看一眼 \(s\)-域和时间域之间的"对偶字典":\(s \leftrightarrow d/dt\),\(1/s \leftrightarrow \int_0^t\),\(e^{at} \cdot \leftrightarrow s \to s - a\),\(\cdot \to t - a\) (with \(u\)) \(\leftrightarrow e^{-as}\),\(\delta \leftrightarrow 1\),\(*\leftrightarrow \times\)——这些对应不仅仅是技术,它们暗示着两个域之间的某种深刻同构。后面如果你学傅里叶变换,会看到这种"换域"的思路更普遍——拉普拉斯只是其中的一个特例。但这是另一本书的故事了。
