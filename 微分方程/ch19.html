第 19 章 热方程与傅里叶级数

第 18 章我们把 PDE 的总图摆了出来,并描绘了"分离变量法"的四步流程:把 \(u(x, t) = X(x) T(t)\) 代入,分成两个 ODE,解 ODE,用初始条件叠加。这一章我们把那张提纲完整地走一遍,在最经典的物理设定上——一根长 \(L\) 的金属棒、两端浸在冰水里、初始温度分布给定——动手把每一步算到底。

本章的另一半是傅里叶级数本身。傅里叶级数是把任意一个周期函数(或者半区间上的任意一个函数)展开成 \(\sin\) 和 \(\cos\) 的无穷级数。它是 19 世纪由傅里叶(Joseph Fourier)在研究热方程时发明的——也就是说,本章这两件事(热方程 + 傅里叶级数)在历史上根本就是同一项工作。傅里叶发现,要解热方程,必须把初始温度分布展开成 \(\sin\) 级数;为了证明展开真的"等于"原函数,他不得不发明一整门函数级数收敛的理论。这门理论后来变成了现代分析学的基石——甚至 \(\delta\) 函数、希尔伯特空间、量子力学的态矢量,根都扎在这一刻。所以你正在学的不是一门"小技巧",而是一段奠基的数学。

本章的结构:先建立物理问题(19.1),走完分离变量四步(19.2 到 19.6),然后把傅里叶级数完整讲一遍(19.7),再处理半区间展开和正余弦级数的关系(19.8),做一道完整的 \(f(x) = x(L - x)\) 的算例(19.9),反思物理直觉(19.10),最后简要介绍傅里叶变换(19.11)。读完这一章,你应该能独立解任何"两端固定 + 给定初始分布"的热方程,并能熟练求任何"合理"函数的傅里叶系数。

19.1 经典问题

把物理设定写得明明白白。一根长度为 \(L\) 的均匀金属棒,沿 \(x\) 轴放在 \(0 \le x \le L\) 上。棒的两端 \(x = 0\) 与 \(x = L\) 浸在冰水里,所以那里的温度始终为 0。在 \(t = 0\) 时刻,棒上温度分布为某个已知函数 \(f(x)\)(可能是把棒中间烤热,可能是某种线性分布,等等)。问:在每个 \(t > 0\),棒上每一点 \(x\) 的温度 \(u(x, t)\) 是多少?

用方程写下来,这个问题是:

\[ \boxed{\begin{aligned} & u_t = \alpha\, u_{xx}, & 0 < x < L,\ t > 0, \\ & u(0, t) = 0,\ u(L, t) = 0, & t > 0, \\ & u(x, 0) = f(x), & 0 \le x \le L. \end{aligned}} \]

第一行是 PDE(热方程,\(\alpha > 0\) 是材料的热扩散系数,例如铜约为 \(10^{-4}\,\mathrm{m^2/s}\),石头约为 \(10^{-6}\))。第二行是边界条件(BC):两端温度恒为 0。第三行是初始条件(IC):\(t = 0\) 时整根棒的温度分布。我们要在这三组条件全部满足的前提下,求出 \(u(x, t)\) 这张曲面。

这个问题之所以是"经典",是因为它把所有 PDE 的标准要素都包含了:线性方程 + 齐次 BC + 非齐次 IC。我们将会看到,齐次 BC 让分离变量法畅通无阻,而非齐次 IC 是真正的"信息源"——它告诉我们解里每一项的"权重"。如果 BC 也非齐次(例如两端温度是别的常数),需要先把问题变形成齐次 BC 的形式;这是 19 章末尾的题外话,我们暂不深入。

在动手之前再做一个小提醒:为什么"两端温度为零"是 \(u(0, t) = 0\) 而不是 \(u_x(0, t) = 0\)?因为题目说的是"两端的温度是 0",不是"两端没有热流"。\(u_x = 0\) 才是绝缘(无热流)的条件——那是另一个问题,叫"绝热端 BC"。这两个 BC 长得只差一个 \(_x\),物理意义却南辕北辙:浸冰水 vs 包保温层。务必读懂题目,然后选对 BC。

19.2 分离变量

按照 18.7 节的剧本,假设 \(u(x, t) = X(x) T(t)\),代入热方程:

\[ X(x) T'(t) = \alpha X''(x) T(t). \]

两边同时除以 \(\alpha X(x) T(t)\),把变量分开:

\[ \frac{T'(t)}{\alpha\, T(t)} = \frac{X''(x)}{X(x)}. \]

现在做关键观察。等式左边只依赖 \(t\),右边只依赖 \(x\)。如果你固定 \(x\)让 \(t\) 变,左边变,但右边一动不动——可它们必须相等。结论:左右两边都必然等于一个不依赖 \(x\) 也不依赖 \(t\) 的常数。

把这个常数记作 \(-\lambda\)(添加负号纯粹为了让 19.4 节的指数衰减解写起来漂亮——别太纠结正负号,这是惯例):

\[ \frac{T'(t)}{\alpha\, T(t)} = \frac{X''(x)}{X(x)} = -\lambda. \]

这一等式立刻变成两个 ODE:

\[ T'(t) = -\alpha \lambda\, T(t), \qquad X''(x) = -\lambda\, X(x), \]

或者整理成标准形式:

\[ T'(t) + \alpha \lambda\, T(t) = 0, \qquad X''(x) + \lambda\, X(x) = 0. \]

第一个是关于 \(t\) 的一阶线性齐次 ODE,通解 \(T(t) = C e^{-\alpha \lambda t}\)。第二个是关于 \(x\) 的二阶线性齐次 ODE。它的通解形式取决于 \(\lambda\) 的符号——这正是接下来要分析的事。

BC 怎么搬过来?\(u(0, t) = X(0) T(t) = 0\) 对所有 \(t\) 都成立。如果 \(T(t) \equiv 0\),那 \(u \equiv 0\),平凡解,不要;所以必有 \(X(0) = 0\)。同理 \(X(L) = 0\)。所以空间方程带齐次 BC:

\[ X'' + \lambda X = 0, \quad X(0) = 0, \quad X(L) = 0. \]

这正是一个本征值问题。下一节我们解出来。

19.3 空间问题:本征值与本征函数

问题:对什么样的 \(\lambda\),

\[ X'' + \lambda X = 0, \quad X(0) = X(L) = 0 \]

有非平凡(即不恒为 0)的解?能让它有非平凡解的 \(\lambda\) 叫本征值,对应的非零解叫本征函数。我们要把所有可能的本征值找出来。

分三种情形(按 \(\lambda\) 的符号):

情形 1:\(\lambda < 0\)。记 \(\lambda = -\mu^2\),\(\mu > 0\)。方程变为 \(X'' - \mu^2 X = 0\),特征根是 \(\pm\mu\),通解

\[ X(x) = A e^{\mu x} + B e^{-\mu x}. \]

带入 \(X(0) = 0\):\(A + B = 0\),即 \(B = -A\)。所以 \(X(x) = A(e^{\mu x} - e^{-\mu x}) = 2A \sinh(\mu x)\)。再带 \(X(L) = 0\):\(2A \sinh(\mu L) = 0\)。但 \(\mu L > 0\) 时 \(\sinh(\mu L) \neq 0\),所以必须 \(A = 0\),从而 \(X \equiv 0\)。没有非平凡解。结论:\(\lambda < 0\) 不是本征值。

情形 2:\(\lambda = 0\)。方程变为 \(X'' = 0\),通解 \(X = Ax + B\)。带 \(X(0) = 0\):\(B = 0\)。带 \(X(L) = 0\):\(AL = 0\),即 \(A = 0\)。还是 \(X \equiv 0\)。所以 \(\lambda = 0\) 也不是本征值。

情形 3:\(\lambda > 0\)。记 \(\lambda = \mu^2\),\(\mu > 0\)。方程变为 \(X'' + \mu^2 X = 0\),通解

\[ X(x) = A \cos(\mu x) + B \sin(\mu x). \]

带 \(X(0) = 0\):\(A = 0\)。所以 \(X(x) = B \sin(\mu x)\)。再带 \(X(L) = 0\):\(B \sin(\mu L) = 0\)。要非平凡 \(B \neq 0\),所以必须 \(\sin(\mu L) = 0\),即

\[ \mu L = n \pi, \quad n = 1, 2, 3, \ldots \]

(\(n = 0\) 给 \(\mu = 0\) 已被情形 2 排除;\(n\) 取负值得到的 \(\sin\) 只是符号翻转,本征函数是同一个,所以只取正整数)。所以本征值是

\[ \boxed{\lambda_n = \left(\frac{n \pi}{L}\right)^2, \quad n = 1, 2, 3, \ldots} \]

对应的本征函数(取 \(B = 1\),反正常数会在叠加时被吸收掉)是

\[ \boxed{X_n(x) = \sin\!\left(\frac{n \pi x}{L}\right), \quad n = 1, 2, 3, \ldots} \]

停一下,看看这些本征函数长什么样。\(X_1(x) = \sin(\pi x/L)\) 在 \([0, L]\) 上是一个完整的"驼峰",两端为零,中间最高;\(X_2(x) = \sin(2\pi x/L)\) 在 \([0, L]\) 上有两个驼峰,中间多了一个零点;\(X_n\) 有 \(n\) 个驼峰,\(n - 1\) 个内部零点。\(n\) 越大,函数振荡得越快——这是"频率"的一种几何表现。

顺便一个观察:这些 \(\sin(n\pi x/L)\) 在 \([0, L]\) 上两两正交:

\[ \int_0^L \sin\!\left(\frac{n\pi x}{L}\right) \sin\!\left(\frac{m\pi x}{L}\right) dx = \begin{cases} 0, & m \neq n, \\ L/2, & m = n. \end{cases} \]

这条性质在 19.6 节求傅里叶系数时会非常重要。它的证明用积化和差公式,我们留到 19.7.2 节再做。

本征值问题为什么能这样三段论?因为 \(\sin\) 是唯一一个"在 \([0, L]\) 两端都过零、且能构造无穷多种振荡频率"的函数。\(\cos\) 在 \(x = 0\) 不过零(\(\cos 0 = 1\)),所以被 \(X(0) = 0\) 这个 BC 排除。如果换 BC——比如 \(X'(0) = X'(L) = 0\)(绝热端)——本征函数会换成 \(\cos\) 而不是 \(\sin\),这是 19.8 节"半区间余弦展开"对应的物理图景。BC 的形状直接决定了本征函数的形状,这是 PDE 思维里非常深的一层。

19.4 时间部分

每个本征值 \(\lambda_n = (n\pi/L)^2\) 对应一个时间方程

\[ T_n'(t) = -\alpha \lambda_n T_n(t), \]

这是一阶线性 ODE,通解

\[ T_n(t) = C_n e^{-\alpha \lambda_n t} = C_n \exp\!\left[-\alpha \left(\frac{n \pi}{L}\right)^2 t\right]. \]

这是一个指数衰减。注意衰减速率是 \(\alpha (n\pi/L)^2 = \alpha n^2 \pi^2 / L^2\)——它正比于 \(n^2\)。所以 \(n\) 越大(本征函数振荡越快),衰减得越快;\(n\) 翻倍,衰减常数变 4 倍。这一条物理直觉非常重要,我们 19.10 节会专门讨论它的意义。

把 \(X_n\) 与 \(T_n\) 乘起来,得到分离形式的解

\[ u_n(x, t) = X_n(x) T_n(t) = C_n e^{-\alpha (n\pi/L)^2 t} \sin\!\left(\frac{n\pi x}{L}\right). \]

每个 \(u_n\) 都满足 PDE \(u_t = \alpha u_{xx}\) 和 BC \(u(0, t) = u(L, t) = 0\)——你可以代回去检验。但每个 \(u_n\) 的初值是 \(C_n \sin(n\pi x/L)\),不是给定的 \(f(x)\)。所以单个 \(u_n\) 不够——除非 \(f\) 恰好就是某个 \(\sin\) 函数,否则我们必须叠加多个 \(u_n\)。

19.5 叠加

热方程是线性齐次的:如果 \(u_1\) 与 \(u_2\) 都是解,那它们的任意线性组合 \(c_1 u_1 + c_2 u_2\) 也是解;如果都满足齐次 BC,叠加也满足齐次 BC。所以最一般的"分离形式之和"是

\[ \boxed{u(x, t) = \sum_{n=1}^\infty b_n\, e^{-\alpha (n\pi/L)^2 t} \sin\!\left(\frac{n\pi x}{L}\right).} \]

这一行公式就是热方程边值问题形式上的通解。它满足 PDE,满足 BC,只剩一件事:在 \(t = 0\) 时它得等于给定的 \(f(x)\)。

令 \(t = 0\):

\[ f(x) = \sum_{n=1}^\infty b_n \sin\!\left(\frac{n\pi x}{L}\right). \]

这就是把 \(f\) 展开成 \(\sin\) 级数,即傅里叶正弦级数。19.6 节告诉你怎么算 \(b_n\)。一旦 \(b_n\) 算出,代回上面 \(u(x, t)\) 的表达式,问题就完全解决了。

到这里整个分离变量法的主轴已经走完。剩下的真正难点全在傅里叶级数:为什么任意"合理"的 \(f\) 真的能展成 \(\sin\) 级数?系数 \(b_n\) 的公式是什么?为什么这个公式正确?接下来我们一一回答。

19.6 用初始条件确定系数

系数 \(b_n\) 怎么求?利用 19.3 节末尾的正交性。回忆:

\[ \int_0^L \sin\!\left(\frac{n\pi x}{L}\right) \sin\!\left(\frac{m\pi x}{L}\right) dx = \begin{cases} 0, & m \neq n, \\ L/2, & m = n. \end{cases} \]

把 \(f(x) = \sum_{k=1}^\infty b_k \sin(k\pi x/L)\) 两边都乘 \(\sin(m\pi x/L)\),在 \([0, L]\) 上积分:

\[ \int_0^L f(x) \sin\!\left(\frac{m\pi x}{L}\right) dx = \sum_{k=1}^\infty b_k \int_0^L \sin\!\left(\frac{k\pi x}{L}\right) \sin\!\left(\frac{m\pi x}{L}\right) dx. \]

右边的求和里,正交性把除了 \(k = m\) 之外的所有项都打成零。剩下唯一的非零项是 \(k = m\),它给出 \(b_m \cdot L/2\)。所以

\[ \int_0^L f(x) \sin\!\left(\frac{m\pi x}{L}\right) dx = b_m \cdot \frac{L}{2}, \]

解出

\[ \boxed{b_n = \frac{2}{L} \int_0^L f(x) \sin\!\left(\frac{n\pi x}{L}\right) dx.} \]

(我把哑变量从 \(m\) 改回 \(n\) 以匹配上面的指标。)这就是傅里叶正弦系数公式。它告诉你如何从 \(f\) 算出每一个 \(b_n\),只要你能算这个积分。

这个推导有一个非常重要的隐含步骤:"两边乘 \(\sin\) 然后积分"这种"挑出一项"的技巧,跟向量空间里"两边点乘 \(\mathbf{e}_k\) 挑出第 \(k\) 个分量"完全是一回事。把 \(\sin(n\pi x/L)\) 看成一组"基向量",把内积定义为 \(\langle f, g \rangle = \int_0^L f(x) g(x) dx\),那么正交性就是 \(\langle X_n, X_m \rangle = 0\)(\(m \neq n\)),系数公式就是 \(b_n = \langle f, X_n \rangle / \langle X_n, X_n \rangle\)——这是无穷维的"投影"。这一抽象观点是泛函分析的开端,我们在这里只是借用它来组织傅里叶级数的逻辑。

来一道直接的小算例。\(L = 1\),\(f(x) = 1\)(常数函数,棒在 \(t = 0\) 时温度处处为 1)。求 \(b_n\):

\[ b_n = \frac{2}{1} \int_0^1 1 \cdot \sin(n\pi x) dx = 2 \cdot \left[-\frac{\cos(n\pi x)}{n\pi}\right]_0^1 = \frac{2}{n\pi}\big(1 - \cos(n\pi)\big). \]

\(\cos(n\pi) = (-1)^n\),所以 \(1 - \cos(n\pi) = 1 - (-1)^n\),它在 \(n\) 偶数时是 0,\(n\) 奇数时是 2。所以

\[ b_n = \begin{cases} 4/(n\pi), & n \text{ 奇}, \\ 0, & n \text{ 偶}. \end{cases} \]

解是

\[ u(x, t) = \sum_{n \text{ 奇}} \frac{4}{n\pi} e^{-\alpha (n\pi)^2 t} \sin(n\pi x). \]

这就是"棒初始温度处处为 1、两端浸冰水"的温度演化。\(t = 0\) 时,所有项加起来给出 \(f(x) = 1\)(在 \(0 < x < 1\) 内;在端点 \(x = 0, 1\) 处级数收敛到 0,因为 \(\sin(n\pi \cdot 0) = \sin(n\pi \cdot 1) = 0\),与 BC 一致——这暗示着 \(f\) 在端点的值与 BC 不一致时,级数在端点会"妥协"到 BC,而不是 \(f\) 的值;19.7.3 节的 Dirichlet 收敛定理会把这件事讲得更清楚)。

到这里,热方程问题在原则上已经解决:给一个 \(f(x)\),按公式算 \(b_n\),代回级数,就得到 \(u(x, t)\)。我们 19.9 节会做一道更复杂的 \(f(x) = x(L-x)\) 的算例。但现在让我们转头,把傅里叶级数本身完整地讲清楚——不仅是用 \(\sin\),而是用最一般的形式。

19.7 傅里叶级数

这一节是本章的"另一个一半"。它脱离开热方程,把傅里叶级数作为一个独立的数学对象介绍——它适用于任何周期函数,而不仅仅是那些出现在 PDE 里的边值问题。

19.7.1 周期函数与三角级数

函数 \(f\) 叫做周期为 \(2L\)(或者说周期 \(2L\)),如果 \(f(x + 2L) = f(x)\) 对所有 \(x\) 成立。\(\sin x\) 与 \(\cos x\) 是周期为 \(2\pi\) 的;\(\sin(\pi x/L)\) 与 \(\cos(\pi x/L)\) 是周期为 \(2L\) 的;\(\sin(n\pi x/L)\) 与 \(\cos(n\pi x/L)\) 是周期为 \(2L/n\) 的——但因为 \(2L/n\) 整除 \(2L\),所以它们也都是周期 \(2L\) 的(只是周期更短而已)。所以"以 \(2L\) 为周期"的全部 \(\sin\) 与 \(\cos\) 是

\[ 1, \cos\!\left(\frac{\pi x}{L}\right), \cos\!\left(\frac{2\pi x}{L}\right), \ldots, \sin\!\left(\frac{\pi x}{L}\right), \sin\!\left(\frac{2\pi x}{L}\right), \ldots \]

(常数 1 也算"周期 \(2L\)",因为常数对任何平移都不变。它对应 \(n = 0\) 的 \(\cos\) 项。)

傅里叶的大胆猜想是:任何"合理"的周期 \(2L\) 函数 \(f\) 都可以写成它们的无穷级数

\[ \boxed{f(x) = \frac{a_0}{2} + \sum_{n=1}^\infty \left[a_n \cos\!\left(\frac{n\pi x}{L}\right) + b_n \sin\!\left(\frac{n\pi x}{L}\right)\right].} \]

这个级数叫 \(f\) 的傅里叶级数。\(a_0/2\) 那个 1/2 不是排印错误,而是为了让 \(a_0\) 的系数公式与其他 \(a_n\) 公式形式一致(否则 \(a_0\) 会比别的多一个 1/2,记起来很烦)。"合理"的精确意义在 19.7.3 节会说;现在你只需要知道几乎所有物理里出现的函数都"合理"。

这个想法在 1807 年被提出来的时候,几乎没有人相信。当时的数学界(包括拉格朗日和拉普拉斯本人)认为"任意函数"——比如有跳跃的方波——绝不可能用光滑的 \(\sin, \cos\) 的和拼出来。傅里叶坚持自己是对的,但他没有给出严格证明。后来狄利克雷(Dirichlet)在 1829 年给出了第一个收敛性的严格证明,确立了傅里叶级数的合法地位。这段历史告诉你,数学有时候要等想象力先于严格性走在前面。

19.7.2 傅里叶系数公式

系数 \(a_n\) 与 \(b_n\) 由"乘 + 积分"挑出。先记下基本正交关系(在整个周期 \([-L, L]\) 上,而不是 \([0, L]\) 上):

\[ \int_{-L}^L \cos\!\left(\frac{n\pi x}{L}\right) \cos\!\left(\frac{m\pi x}{L}\right) dx = \begin{cases} 0, & m \neq n, \\ L, & m = n \neq 0, \\ 2L, & m = n = 0. \end{cases} \] \[ \int_{-L}^L \sin\!\left(\frac{n\pi x}{L}\right) \sin\!\left(\frac{m\pi x}{L}\right) dx = \begin{cases} 0, & m \neq n, \\ L, & m = n. \end{cases} \] \[ \int_{-L}^L \cos\!\left(\frac{n\pi x}{L}\right) \sin\!\left(\frac{m\pi x}{L}\right) dx = 0 \quad \text{(任何 } n, m\text{)}. \]

这些都可以用积化和差公式直接验证。例如 \(\cos A \cos B = \tfrac{1}{2}[\cos(A - B) + \cos(A + B)]\),代进去积分 \(\cos(\text{非零整数倍 } \pi x/L)\) 在 \([-L, L]\) 上等于零,所以剩下的就是上面的格局。

有了正交性,把傅里叶级数 \(f(x) = a_0/2 + \sum (a_n \cos + b_n \sin)\) 两边乘 \(\cos(m\pi x/L)\) 在 \([-L, L]\) 上积分,只有 \(a_m\cdot L\)(当 \(m \neq 0\))或 \(a_0/2 \cdot 2L = a_0 L\)(当 \(m = 0\))这一项幸存。所以

\[ \boxed{a_n = \frac{1}{L} \int_{-L}^L f(x) \cos\!\left(\frac{n\pi x}{L}\right) dx, \quad n = 0, 1, 2, \ldots} \]

类似地,乘 \(\sin\) 然后积分得到

\[ \boxed{b_n = \frac{1}{L} \int_{-L}^L f(x) \sin\!\left(\frac{n\pi x}{L}\right) dx, \quad n = 1, 2, 3, \ldots} \]

注意这里 \(b_n\) 的公式与 19.6 节的"半区间正弦系数公式"\(b_n = (2/L)\int_0^L f \sin\)不同——前者积分在 \([-L, L]\) 上(全周期),后者在 \([0, L]\) 上(半周期)。19.8 节会解释它们怎么互相关联。

看一道经典算例:\(f(x)\) 在 \([-\pi, \pi]\) 上等于 \(x\),周期 \(2\pi\)(即 \(L = \pi\))。这就是著名的"锯齿波"。算傅里叶系数:

\(a_n\):因为 \(x\) 是奇函数,\(\cos\) 是偶函数,\(x \cos(nx)\) 是奇函数,\(\int_{-\pi}^{\pi}\) 奇函数 \(= 0\)。所以 \(a_n = 0\) 对所有 \(n \ge 0\)。傅里叶级数纯由 \(\sin\) 项组成。

\(b_n\):

\[ b_n = \frac{1}{\pi} \int_{-\pi}^{\pi} x \sin(nx) dx = \frac{2}{\pi} \int_0^\pi x \sin(nx) dx \]

(被积函数 \(x \sin(nx)\) 是偶函数,所以 \([-\pi, \pi]\) 上的积分等于 \(2 \times [0, \pi]\) 上的积分)。

用分部积分,取 \(u = x\),\(dv = \sin(nx) dx\),则 \(du = dx\),\(v = -\cos(nx)/n\):

\[ \int_0^\pi x \sin(nx) dx = \left[-\frac{x \cos(nx)}{n}\right]_0^\pi + \frac{1}{n}\int_0^\pi \cos(nx) dx \] \[ = -\frac{\pi \cos(n\pi)}{n} + \frac{1}{n^2}[\sin(nx)]_0^\pi = -\frac{\pi (-1)^n}{n} + 0 = \frac{(-1)^{n+1} \pi}{n}. \]

所以

\[ b_n = \frac{2}{\pi} \cdot \frac{(-1)^{n+1} \pi}{n} = \frac{2(-1)^{n+1}}{n}. \]

傅里叶级数为

\[ x \sim 2 \sum_{n=1}^\infty \frac{(-1)^{n+1}}{n} \sin(nx) = 2\left[\sin x - \frac{\sin 2x}{2} + \frac{\sin 3x}{3} - \cdots\right]. \]

("\(\sim\)" 而不是 "\(=\)" 是因为这级数虽然在 \(-\pi < x < \pi\) 内部确实等于 \(x\),但在端点 \(x = \pm \pi\) 收敛到 0(\(\sin(n\pi) = 0\)),不等于 \(\pm \pi\) 本身——再次反映周期延拓与原函数在跳跃点的妥协。)

19.7.3 收敛:Dirichlet 条件

傅里叶级数什么时候真的收敛到 \(f\)?狄利克雷在 1829 年给出了第一组充分条件,叫Dirichlet 条件:如果 \(f\) 在一个周期内

  • 分段连续(可以有有限个跳跃点),
  • 分段单调(可以有有限个极值),

那么 \(f\) 的傅里叶级数在每一点 \(x_0\) 都收敛,而且

  • 如果 \(f\) 在 \(x_0\) 连续,级数收敛到 \(f(x_0)\);
  • 如果 \(f\) 在 \(x_0\) 跳跃,级数收敛到 \(\tfrac{1}{2}\big[f(x_0^-) + f(x_0^+)\big]\)(左右极限的平均)。

这在物理上几乎覆盖一切——任何"实际可测"的温度分布、电压波形、声压等都满足 Dirichlet 条件。所以从应用角度,你可以放心使用傅里叶级数,不必担心收敛问题。

跳跃点的"取平均"行为很有意思。回到上面的锯齿波 \(f(x) = x\)(\(-\pi < x < \pi\),周期延拓),在 \(x = \pi\) 处 \(f\) 跳跃:左极限 \(f(\pi^-) = \pi\),右极限 \(f(\pi^+) = -\pi\)(因为周期延拓到下一周期开始);二者平均是 0。所以傅里叶级数在 \(x = \pi\) 收敛到 0——和我们上面看到 \(\sin(n\pi) = 0\) 完全一致。这是 Dirichlet 定理的具体体现。

另一个值得提的现象是吉布斯现象(Gibbs phenomenon):当 \(f\) 有跳跃时,部分和 \(S_N(x) = \sum_{n=1}^N\) 在跳跃点附近永远会"超调"约 9% 的跳跃幅度,无论 \(N\) 多大。这是傅里叶级数在跳跃点处不一致收敛的反映。它在工程上很重要——任何用有限项 \(\sin/\cos\) 重构方波的电路都会出现这种"耳朵"。但在 \(L^2\) 范数意义下,\(S_N \to f\) 完美无瑕。这种"逐点行为"与"积分行为"分裂的现象,是傅里叶级数特有的微妙之处。

19.7.4 复指数形式

傅里叶级数还有一个更紧凑的写法,用复指数。回忆 Euler 公式 \(e^{i\theta} = \cos\theta + i \sin\theta\),所以

\[ \cos\theta = \frac{e^{i\theta} + e^{-i\theta}}{2}, \quad \sin\theta = \frac{e^{i\theta} - e^{-i\theta}}{2i}. \]

把这些代进 \(a_n \cos(n\omega x) + b_n \sin(n\omega x)\)(其中 \(\omega = \pi/L\)),整理一下,你会发现整个傅里叶级数可以写成

\[ \boxed{f(x) = \sum_{n=-\infty}^\infty c_n e^{i n \omega x}, \quad \omega = \frac{\pi}{L}} \]

(注意求和从 \(-\infty\) 到 \(+\infty\),包括负 \(n\)),其中

\[ \boxed{c_n = \frac{1}{2L} \int_{-L}^L f(x) e^{-i n \omega x} dx.} \]

系数 \(c_n\) 与 \(a_n, b_n\) 之间的关系是

\[ c_0 = \frac{a_0}{2}, \quad c_n = \frac{a_n - i b_n}{2}, \quad c_{-n} = \frac{a_n + i b_n}{2} = \overline{c_n} \quad (n \ge 1). \]

(最后那个等式只有当 \(f\) 是实函数时成立——\(c_{-n}\) 与 \(c_n\) 互为复共轭。这是实函数的一种对称约束。)

为什么用复指数写更好?第一,公式更对称:只有一个 \(c_n\),不需要分 \(a_n\) 和 \(b_n\)。第二,\(\sin\) 与 \(\cos\) 的微分会换形(\(\sin' = \cos\),\(\cos' = -\sin\)),而 \(e^{i n \omega x}\) 求导只是乘 \(in\omega\),不换形——这在做傅里叶变换或者解 PDE 时非常方便。第三,跟傅里叶变换(19.11 节)的过渡更自然。在物理学和工程中(尤其是信号处理与量子力学),复指数形式是事实上的标准。

19.8 半区间展开

到目前为止傅里叶级数都谈"周期 \(2L\)"的函数,定义在 \([-L, L]\) 上。但热方程的初始数据 \(f(x)\) 只定义在 \([0, L]\) 上,根本没有周期或负 \(x\) 的概念。怎么把它套进傅里叶级数的框架?

方法是周期延拓:把 \(f\) 在 \([0, L]\) 上的定义"延伸"到 \([-L, L]\) 上,然后再周期 \(2L\) 地铺满整个数轴。延拓有两种自然方式:

  • 奇延拓:在 \([-L, 0]\) 上定义 \(f(-x) := -f(x)\),也就是把图像绕原点点对称翻转。然后再以 \(2L\) 为周期重复。这样得到的是一个奇函数,所有 \(a_n = 0\),只剩 \(b_n \sin\) 项。
  • 偶延拓:在 \([-L, 0]\) 上定义 \(f(-x) := f(x)\),也就是把图像绕 \(y\) 轴镜面翻转。然后再周期 \(2L\) 重复。这样得到偶函数,所有 \(b_n = 0\),只剩 \(a_n \cos\) 项。

奇延拓得到的傅里叶级数叫傅里叶正弦级数:

\[ f(x) = \sum_{n=1}^\infty b_n \sin\!\left(\frac{n\pi x}{L}\right), \quad b_n = \frac{2}{L}\int_0^L f(x) \sin\!\left(\frac{n\pi x}{L}\right) dx. \]

(\(b_n\) 的 \(2/L\) 系数比 19.7 节里的 \(1/L\) 大一倍——因为奇延拓让 \([-L, 0]\) 上的积分等于 \([0, L]\) 上的积分,把它们加起来正好翻倍。)这正是19.6 节里求热方程系数用到的公式。

偶延拓得到的傅里叶级数叫傅里叶余弦级数:

\[ f(x) = \frac{a_0}{2} + \sum_{n=1}^\infty a_n \cos\!\left(\frac{n\pi x}{L}\right), \quad a_n = \frac{2}{L}\int_0^L f(x) \cos\!\left(\frac{n\pi x}{L}\right) dx. \]

选哪种延拓?看 BC。如果 BC 是 \(u(0, t) = u(L, t) = 0\)(温度端值为 0),需要本征函数在端点过零,选 \(\sin\)——奇延拓。如果 BC 是 \(u_x(0, t) = u_x(L, t) = 0\)(绝热端,无热流),需要本征函数的导数在端点为零(\(\cos' = -\sin\),在 \(x = 0\) 与 \(x = L\) 即 \(n\pi\) 处 \(\sin\) 为 0),选 \(\cos\)——偶延拓。所以"BC 选延拓,延拓选展开"——这是一条非常清晰的关联链。

19.9 完整算例:\(f(x) = x(L - x)\)

把 19.1 到 19.8 整套机器组装起来,做一道有血有肉的算例。问题:长度 \(L\) 的杆,两端浸冰水(\(u(0, t) = u(L, t) = 0\)),初始温度 \(u(x, 0) = f(x) = x(L - x)\)(中间最热,呈倒抛物线分布,在 \(x = L/2\) 处达到最大值 \(L^2/4\),两端为 0——和 BC 自洽,这是好兆头)。求 \(u(x, t)\)。

第一步:写出形式解

\[ u(x, t) = \sum_{n=1}^\infty b_n e^{-\alpha (n\pi/L)^2 t} \sin\!\left(\frac{n\pi x}{L}\right). \]

第二步:求系数 \(b_n\)。

\[ b_n = \frac{2}{L}\int_0^L x(L - x) \sin\!\left(\frac{n\pi x}{L}\right) dx. \]

展开 \(x(L - x) = Lx - x^2\),分两个积分。我直接用分部积分硬算 \(\int_0^L (Lx - x^2) \sin(n\pi x/L) dx\)。设 \(\beta = n\pi/L\),被积函数是 \((Lx - x^2) \sin(\beta x)\)。

第一项 \(\int_0^L Lx \sin(\beta x) dx = L \int_0^L x \sin(\beta x) dx\)。分部积分,\(u = x\),\(dv = \sin(\beta x) dx\),\(v = -\cos(\beta x)/\beta\):

\[ \int_0^L x \sin(\beta x) dx = \left[-\frac{x \cos(\beta x)}{\beta}\right]_0^L + \frac{1}{\beta}\int_0^L \cos(\beta x) dx \] \[ = -\frac{L \cos(\beta L)}{\beta} + \frac{1}{\beta^2}[\sin(\beta x)]_0^L = -\frac{L \cos(n\pi)}{\beta} + 0 \]

(因为 \(\beta L = n\pi\),\(\sin(n\pi) = 0\)),所以 \(= -L(-1)^n / \beta\),乘以 \(L\) 给出 \(-L^2 (-1)^n / \beta\)。

第二项 \(\int_0^L x^2 \sin(\beta x) dx\)。分部积分,\(u = x^2\),\(dv = \sin(\beta x) dx\),\(v = -\cos(\beta x)/\beta\):

\[ \int_0^L x^2 \sin(\beta x) dx = \left[-\frac{x^2 \cos(\beta x)}{\beta}\right]_0^L + \frac{2}{\beta}\int_0^L x \cos(\beta x) dx \] \[ = -\frac{L^2 (-1)^n}{\beta} + \frac{2}{\beta} I_2, \]

其中 \(I_2 = \int_0^L x \cos(\beta x) dx\)。再分部:\(u = x\),\(dv = \cos(\beta x) dx\),\(v = \sin(\beta x)/\beta\):

\[ I_2 = \left[\frac{x \sin(\beta x)}{\beta}\right]_0^L - \frac{1}{\beta}\int_0^L \sin(\beta x) dx = 0 + \frac{1}{\beta^2}[\cos(\beta x)]_0^L = \frac{(-1)^n - 1}{\beta^2}. \]

(用 \(\sin(n\pi) = 0\)。)代回:

\[ \int_0^L x^2 \sin(\beta x) dx = -\frac{L^2 (-1)^n}{\beta} + \frac{2}{\beta} \cdot \frac{(-1)^n - 1}{\beta^2} = -\frac{L^2 (-1)^n}{\beta} + \frac{2[(-1)^n - 1]}{\beta^3}. \]

把两项合起来:

\[ \int_0^L (Lx - x^2) \sin(\beta x) dx = -\frac{L^2 (-1)^n}{\beta} - \left[-\frac{L^2 (-1)^n}{\beta} + \frac{2[(-1)^n - 1]}{\beta^3}\right] = -\frac{2[(-1)^n - 1]}{\beta^3}. \]

所以

\[ b_n = \frac{2}{L} \cdot \left(-\frac{2[(-1)^n - 1]}{\beta^3}\right) = -\frac{4}{L} \cdot \frac{(-1)^n - 1}{\beta^3}. \]

代入 \(\beta = n\pi/L\),\(\beta^3 = n^3 \pi^3 / L^3\):

\[ b_n = -\frac{4}{L} \cdot \frac{[(-1)^n - 1] L^3}{n^3 \pi^3} = -\frac{4 L^2 [(-1)^n - 1]}{n^3 \pi^3} = \frac{4 L^2 [1 - (-1)^n]}{n^3 \pi^3}. \]

\(1 - (-1)^n\) 在 \(n\) 偶数时为 0,奇数时为 2。所以

\[ b_n = \begin{cases} \dfrac{8 L^2}{n^3 \pi^3}, & n \text{ 奇数}, \\ 0, & n \text{ 偶数}. \end{cases} \]

最终:

\[ \boxed{u(x, t) = \sum_{n \text{ 奇}} \frac{8 L^2}{n^3 \pi^3} e^{-\alpha (n\pi/L)^2 t} \sin\!\left(\frac{n\pi x}{L}\right).} \]

第三步:核对 \(t = 0\)。在 \(t = 0\) 时

\[ u(x, 0) = \sum_{n \text{ 奇}} \frac{8 L^2}{n^3 \pi^3} \sin\!\left(\frac{n\pi x}{L}\right). \]

这应当等于 \(x(L - x)\)。验证主要靠数值代入。例如取 \(x = L/2\):\(\sin(n\pi/2)\) 在 \(n = 1, 3, 5, \ldots\) 时为 \(1, -1, 1, -1, \ldots\),所以前几项给出

\[ u(L/2, 0) \approx \frac{8 L^2}{\pi^3}\left[1 - \frac{1}{27} + \frac{1}{125} - \cdots\right]. \]

方括号里的级数收敛到 \(\pi^3/32\)(可以查表或者信我),所以乘起来正好得 \(L^2/4\)——与 \(f(L/2) = (L/2)(L - L/2) = L^2/4\) 一致 ✓。如果你对这个核对没把握,可以编程数值算出前 100 项的和,在几个 \(x\) 点比较一下 \(\sum b_n \sin(\cdot)\) 与 \(x(L - x)\)——会非常接近。

这道算例值得反复琢磨。注意 \(b_n \propto 1/n^3\) 衰减得相当快——这说明级数收敛迅速,只需取前 5、6 项就能把 \(t = 0\) 的初始分布逼近得不错。这种"系数衰减快"反映 \(f(x) = x(L - x)\) 是个非常光滑的函数(\(C^\infty\) 的多项式);如果 \(f\) 有跳跃,系数衰减只是 \(O(1/n)\),收敛要慢得多。这条"光滑度 ↔ 系数衰减率"的关系是傅里叶分析里非常深的对偶。

19.10 物理直觉:热方程"磨平细节"

退一步看 19.5 节那个解的形状:

\[ u(x, t) = \sum_{n=1}^\infty b_n e^{-\alpha (n\pi/L)^2 t} \sin\!\left(\frac{n\pi x}{L}\right). \]

第 \(n\) 项的衰减速率正比于 \(n^2\)。所以 \(n = 1\) 项(基波,只有一个驼峰)衰减最慢,\(n = 100\) 项(高频,100 个驼峰)衰减得是基波的 10000 倍快。等到 \(t\) 足够大时,所有高频项早已熄灭,只剩一两个低频项,温度分布趋于一个平滑的"基波形状"\(\sin(\pi x/L)\),最终衰减到全 0(因为两端浸冰水,系统能量被持续抽走)。

这就是热方程的磨平性:任何尖锐的、突变的、振荡剧烈的初始分布,在热扩散下都会被指数级地"磨平"——细节(高频成分)衰减快,粗轮廓(低频成分)衰减慢。一根棒,中间烫了一个很尖的小热点,几秒之内热点就摊开成一个温和的小驼峰,几分钟内整根棒接近于 0。从信息论的角度看,热方程抹去信息:看到 \(t\) 时刻的温度分布,你完全没法精确反推 \(t = 0\) 时的细节(因为高频信息已经按指数被抹掉,任何反向重构都会被噪声放大成一团乱麻)——这就是 18.5 节提到的"反向热方程对数据不连续依赖"的本质。

这条直觉在工程里有许多回响:图像处理里的"高斯模糊"本质上是把图像作为初值演化热方程一段时间,所以它必然抹掉细小纹理而保留大轮廓;扩散滤波被用来去除噪声,代价是同时模糊掉边缘。边缘保护扩散(Perona–Malik)等改进版本就是为了对抗热方程"无差别磨平"的天性。

下面这个对比记着:热方程是不可逆的、磨平的、走向死寂的。这与下一章波动方程截然相反——波动方程保留细节、不抹信息、可逆传播。你弹了一下吉他弦,几秒之后弦还在嗡嗡振动,所有"高频泛音"都没消失(理想情况下),正是因为波动方程的解是振荡而不是衰减。这种对比是 PDE 性格分类(抛物 vs 双曲)的物理体现。

19.11 短谈傅里叶变换

傅里叶级数适用于有限区间(或周期函数)。如果你的物理问题发生在无穷长的区域——比如一根无限长的杆,或者一个无界平面上的扩散——那本征值就不再是离散的 \(\lambda_n = (n\pi/L)^2\),而是连续的实数。求和变成积分,这就是傅里叶变换(Fourier transform)。

傅里叶变换把函数 \(f(x)\)(\(-\infty < x < \infty\))变成另一个函数 \(\hat{f}(k)\),由

\[ \hat{f}(k) = \int_{-\infty}^\infty f(x) e^{-i k x} dx \]

定义。逆变换是

\[ f(x) = \frac{1}{2\pi}\int_{-\infty}^\infty \hat{f}(k) e^{i k x} dk. \]

(系数前 \(2\pi\) 的位置依据约定不同有几种摆法,看书。)\(\hat{f}(k)\) 把 \(f\) 分解成不同频率 \(k\) 的纯正弦波的叠加——把 19.7.4 节复指数级数中"求和"的指标 \(n\)(离散)换成了"积分"的连续指标 \(k\)。物理上 \(k\) 叫"波数",对应空间频率;在时间方面则叫"角频率 \(\omega\)"。

傅里叶变换在解无界问题上特别强大。无穷长杆上的热方程 \(u_t = \alpha u_{xx}\)(无 BC,只有 \(u(x, 0) = f(x)\) 这一 IC)的解可以直接写出:

\[ u(x, t) = \frac{1}{\sqrt{4\pi \alpha t}}\int_{-\infty}^\infty f(y) e^{-(x - y)^2 / (4\alpha t)} dy. \]

这就是著名的热核(heat kernel)解。它本质上是把 \(f\) 与一个高斯函数做卷积——而高斯就是热方程在初值为 \(\delta\) 函数时的基本解。这条公式既可以用傅里叶变换从头推出来,也可以用其他方法(如格林函数)推出来。这里我们只是让你知道它存在;细节属于更深入的 PDE 课程。

更广的应用:傅里叶变换是信号处理(从音频压缩到图像 JPEG)、量子力学(波函数的位置-动量表示)、晶体学(衍射图样)的核心工具。本书不深入它,但本章的傅里叶级数是它的"离散兄弟"——理解了级数,变换就是把"离散求和"换成"连续积分"的形式上的飞跃,直觉相似。

到这里第 19 章结束。我们解决了"两端固定 + 给定初值"的热方程,把傅里叶级数从动机到收敛全部交代,做了完整算例,并瞥了一眼无穷区域的傅里叶变换。下一章把同一套机器搬到波动方程上,你会看到几乎一字不差的"分离 → 解 ODE → 用本征函数叠加 → 用 IC 定系数"流程,只不过时间方程换成了二阶振荡——所以解会"振铃"而不是"衰减"。请保持你刚学到的傅里叶级数的肌肉记忆,因为 20 章一开始就会用到。
