第 2 章 一阶可分离方程

第 1 章我们认识了"微分方程"这个对象，但只看不动手。从这一章开始，我们要真正动手解方程了。最先登场的是可分离方程——一类形式特别简单的一阶方程，它的解法基本只靠一个想法："把 \(x\) 和 \(y\) 各挪一边，然后两边各积一次分。"听起来朴素得几乎像是耍赖，但只要这个想法用得上，整个求解过程就基本结束了。

这一类方程为什么值得专门花一章？三个理由。第一，它在实践中出奇地常见——第 1 章里我们列举的那些经典模型（指数增长、放射衰变、牛顿冷却、带阻力的自由落体、化学反应、电容放电），全都是可分离的。会解可分离方程，等于一次性接管了这些应用领域的核心数学。第二，可分离方程是后面所有方法的"参照系"——以后我们碰到的每一种新方法，几乎都可以问一句"它和可分离方程的关系是什么"。第三，可分离方程的解法把"两边对什么变量积分"这件事讲得最清楚，是把"莱布尼兹记号" \(\dfrac{dy}{dx}\) 的妙处第一次真正榨出来用——那种"\(dy\) 和 \(dx\) 像普通量一样可以乘除"的代数操作，将在这里得到合法的解释。

本章先把"可分离"这个形式说清楚（2.1 节），然后教你那一招分离变量法（2.2 节），再讨论解的写法（2.3 节）和一个最容易被忘记的小陷阱（2.4 节）。然后是大量的例题——人口、自由落体、化学反应、混合溶液（2.5 节）。最后教你在算完之后怎么"代回去检验"自己有没有出错（2.6 节）。

2.1 什么样的方程是可分离的

一个一阶方程 \(\dfrac{dy}{dx} = F(x, y)\) 叫做可分离方程，如果右边 \(F(x, y)\) 能写成两个一元函数的乘积——一个只与 \(x\) 有关，一个只与 \(y\) 有关：

\[ \boxed{\frac{dy}{dx} = f(x)\, g(y).} \]

"可分离"这个名字直接来自这个形式：右边的 \(x\) 和 \(y\) 可以"分离"开来，分别住在自己的因子里，没有缠绕。

看几个例子立刻有感觉：

  • \(y' = x y\)。可分离。\(f(x) = x\)、\(g(y) = y\)。
  • \(y' = (x^2 + 1)(y^3 - y)\)。可分离。\(f(x) = x^2 + 1\)、\(g(y) = y^3 - y\)。
  • \(y' = e^{x + y}\)。可分离。因为 \(e^{x+y} = e^x \cdot e^y\)，所以 \(f(x) = e^x\)、\(g(y) = e^y\)。这是初看时不太明显但很常考的一种"伪装"。
  • \(y' = \dfrac{x}{y^2}\)。可分离。\(f(x) = x\)、\(g(y) = 1/y^2\)（暂时假设 \(y \neq 0\)）。
  • \(y' = y\)。可分离，且更特殊——它甚至只依赖于 \(y\)，不依赖于 \(x\)（这种叫自治方程）。
  • \(y' = f(x)\)（右边只与 \(x\) 有关）。可分离的退化情形。求解就是直接对两边 \(x\) 积分。

下面这些就不是可分离的（至少不显然是）：

  • \(y' = x + y\)。和。无法写成 \(f(x) g(y)\) 的形式——一个和不是一个积。这个方程要等到第 3 章用积分因子法解。
  • \(y' = \dfrac{x + y}{x - y}\)。同样，含和与差。这是个"齐次方程"，可以用换元 \(v = y/x\) 转成可分离的，但它本身并不直接是可分离的。
  • \(y' = x y + x^2\)。两项的和；分不开。

判别一个方程是否可分离，最稳妥的办法是把右边写出来，看你能不能把所有 \(x\) 拢进一个因子、所有 \(y\) 拢进另一个因子。如果做不到，就不是可分离。如果做得到——哪怕代价是先做点代数化简，比如把 \(e^{x+y}\) 拆成 \(e^x e^y\)、把 \(\sin(x) \cos(y) - \cos(x) \sin(y)\) 化成 \(\sin(x - y)\)（不可分离！）——也算可分离。

有一种常见的伪可分离形式值得警惕：\(y' = h(x + y)\) 或 \(y' = h(ax + by)\)。它看起来把 \(x\) 和 \(y\) 通过一个表达式 \(x + y\) 卷在了一起，单一函数 \(h\) 作用在它上面。这种方程一般不是可分离的，但通过换元 \(u = x + y\)（或 \(u = ax + by\)）能转成可分离的——这是另一种叫做"约化为可分离"的小技巧。本章我们暂不展开。

另一类常见的"长得像可分离实则需要加工"的方程是齐次方程：\(y' = F(y/x)\)，右边是 \(y/x\) 的某个函数。最经典的例子是 \(y' = (x + y)/(x - y)\)——分子分母都是 \(x\) 与 \(y\) 的一次齐次函数。直接分离做不到，但换元 \(v = y/x\) 后能把它变成关于 \(v\) 的可分离方程。这一招在某些教材里叫"齐次方程换元法"。本章不深入，但你可以心里有这么一个念头：当一个方程里 \(x\) 和 \(y\) 总是以同次幂的组合出现时，"\(v = y/x\)" 几乎一定能起作用。

顺便提醒：本节标题里的"齐次方程"——指的是 \(F(y/x)\) 这种"把 \(x, y\) 同时乘以同一个常数后方程不变"的几何意义上的齐次。这跟第 3 章里我们要遇到的"齐次线性方程"（指 \(y' + p y = 0\) 这种"右边为零"的齐次）不是同一个意思。同一个词，两种用法，要根据上下文区分。

2.2 分离变量法

现在到了关键的一招。给定可分离方程

\[ \frac{dy}{dx} = f(x) g(y), \]

我们要求 \(y(x)\)。先把 \(g(y)\) 挪到左边、\(dx\) 挪到右边：

\[ \frac{dy}{g(y)} = f(x) \, dx. \]

然后两边各做一个不定积分：

\[ \int \frac{dy}{g(y)} = \int f(x) \, dx. \]

左边算出来是 \(y\) 的某个函数（带个积分常数），右边算出来是 \(x\) 的某个函数（也带个积分常数）。把两边的积分常数合并成一个 \(C\)（写在右边或左边都行），就得到一条 \(x\) 与 \(y\) 的关系式——这就是方程的通解，多半呈隐式。如果幸运，能再把 \(y\) 显式解出来。

事就是这么简单。先看看方法在简单例子上的运行，再回头讲讲"\(\dfrac{dy}{dx}\) 的两边为什么可以这样像乘除"。

2.2.1 第一例：\(y' = y\)

这是 1.4 节里我们靠"猜"得到 \(y = C e^x\) 的那个方程。用分离变量法严格走一遍：

\[ \frac{dy}{dx} = y \;\Longrightarrow\; \frac{dy}{y} = dx. \]

两边积：

\[ \int \frac{dy}{y} = \int dx \;\Longrightarrow\; \ln|y| = x + C_1. \]

把指数函数施加到两边，把绝对值与常数一起处理：\(|y| = e^{x + C_1} = e^{C_1} \cdot e^x\)，所以 \(y = \pm e^{C_1} \cdot e^x\)。把 \(\pm e^{C_1}\) 视为一个新的任意常数 \(C\)（它可以取任意非零实数），最终得

\[ y(x) = C e^x. \]

这正是我们之前猜出来的通解。再补一笔：当 \(C = 0\) 时，\(y \equiv 0\) 也是原方程的解（左右都是 0），但这个解在分离变量的中途——具体来说，在我们用 \(g(y) = y\) 去除两边时——被悄悄丢掉了，因为那一步要求 \(y \neq 0\)。我们 2.4 节会专门讨论这个"零解"陷阱。把零解补回来后，全部解就是 \(y = C e^x\)，\(C \in \mathbb{R}\)（含 \(C = 0\)）。

2.2.2 关于 \(\dfrac{dy}{dx}\) 的代数游戏

很多读者第一次看到"\(\dfrac{dy}{dx} = y\) 推出 \(\dfrac{dy}{y} = dx\)"时会皱眉头：\(\dfrac{dy}{dx}\) 不是一个分数，\(dy\) 和 \(dx\) 并不是真正的"小量"——是莱布尼兹的记号，看起来像个分数，但严格意义上不是。我们怎么可以"两边乘 \(dx\)"再"两边除 \(g(y)\)"？

答案是：这一步在形式上是合法的，因为它能等价于一个严格的链式法则操作。我们把上面的步骤"严格化"一遍。原方程：

\[ \frac{dy}{dx} = f(x) g(y). \]

假设 \(g(y) \neq 0\)，两边除以 \(g(y)\)：

\[ \frac{1}{g(y)} \cdot \frac{dy}{dx} = f(x). \]

左边可以这样看：如果记 \(G(y) = \int \dfrac{dy}{g(y)}\)，则按链式法则 \(\dfrac{d}{dx} G(y(x)) = G'(y) \cdot \dfrac{dy}{dx} = \dfrac{1}{g(y)} \cdot \dfrac{dy}{dx}\)。所以方程变成

\[ \frac{d}{dx} G(y(x)) = f(x), \]

两边对 \(x\) 积分得 \(G(y(x)) = \int f(x) \, dx + C\)，正是上面"分离变量"那一招得到的同一个结果。

所以，分离变量法只是链式法则的逆向使用，加上一个标准的不定积分。把 \(dy\)、\(dx\) 当成可乘除的"小量"是合法的助记法，背后的严谨性由链式法则保证。从此以后，你大可以放心地把 \(dy\) 和 \(dx\) 像普通分数一样移来移去——本书后面会反复这样写。

2.2.3 第二例：\(y' = x y\)

分离变量：

\[ \frac{dy}{y} = x \, dx \;\Longrightarrow\; \int \frac{dy}{y} = \int x \, dx \;\Longrightarrow\; \ln|y| = \tfrac{1}{2} x^2 + C_1. \]

取指数：\(|y| = e^{C_1} e^{x^2/2}\)，把符号与常数吸进新常数 \(C\)，得

\[ y(x) = C e^{x^2 / 2}. \]

验证：\(y'(x) = C \cdot x \cdot e^{x^2/2} = x y\) ✓。再次注意 \(y \equiv 0\) 也是解（对应 \(C = 0\)），它在分离时被丢过一次。

2.2.4 第三例：\(y' = (1 + y^2)\)

分离：

\[ \frac{dy}{1 + y^2} = dx \;\Longrightarrow\; \arctan y = x + C \;\Longrightarrow\; y(x) = \tan(x + C). \]

这次解能显式写出来。注意它有一个特性：\(\tan(x + C)\) 在 \(x + C = \pi/2 + n\pi\) 处发散——也就是说，解只在某个有限的开区间上存在，不能延拓到全体实数。这种"有限存在区间"的现象在非线性方程里很常见，叫做有限时间爆破（finite-time blow-up）。如果你解的是从 \(y(0) = 0\) 出发的初值问题，那 \(C = 0\)，解是 \(\tan x\)，存在区间是 \((-\pi/2, \pi/2)\)。物理上你可以把它想象成一个"反馈越来越强"的过程——\(y\) 越大，\(y' = 1 + y^2\) 就越大，\(y\) 增长得越快，反过来又让 \(y'\) 更大，一路加速直到 \(\pi/2\) 处冲到无穷大。

2.2.5 第四例：\(y' = e^{x + y}\)

这是 2.1 节提到的"伪装"例子。先把右边拆开：\(e^{x+y} = e^x \cdot e^y\)。所以是可分离的：

\[ \frac{dy}{e^y} = e^x \, dx, \quad \text{即} \quad e^{-y} \, dy = e^x \, dx. \]

积分：\(-e^{-y} = e^x + C_1\)，整理 \(e^{-y} = -e^x - C_1\)。把右边写成 \(C - e^x\)（吸收符号），取负对数：

\[ y(x) = -\ln(C - e^x). \]

定义域要求 \(C - e^x > 0\)（因为对数里要求正），即 \(x < \ln C\)。又是一个"解只在某个区间上存在"的非线性方程。当 \(x \to \ln C^-\) 时，对数里 \(C - e^x \to 0^+\)，\(\ln \to -\infty\)，所以 \(y \to +\infty\)——又一个有限时间爆破。

用具体初始条件确定常数。比如 \(y(0) = 0\)：\(0 = -\ln(C - 1)\)，所以 \(\ln(C-1) = 0\)，\(C = 2\)。最终特解 \(y(x) = -\ln(2 - e^x)\)，定义在 \(x < \ln 2 \approx 0.693\) 上。验证：\(y' = -\dfrac{-e^x}{2 - e^x} = \dfrac{e^x}{2 - e^x}\)；又 \(e^{x + y} = e^x \cdot e^y = e^x \cdot e^{-\ln(2 - e^x)} = \dfrac{e^x}{2 - e^x}\)，与 \(y'\) 一致 ✓。

2.3 隐式解 vs 显式解

解微分方程的最终目标当然是 \(y\) 写成 \(x\) 的明确函数——这叫显式解。但很多时候你做完积分，得到的是一条 \(F(x, y) = C\) 的关系，不能（或不容易）把 \(y\) 单独解出来。这时它就叫做隐式解。隐式解仍然是合法、合用的解；很多时候它甚至比硬要解出来的显式解更紧凑、更有几何含义。

看一个例子：

\[ \frac{dy}{dx} = -\frac{x}{y}. \]

分离：\(y \, dy = -x \, dx\)，两边积：\(\dfrac{1}{2} y^2 = -\dfrac{1}{2} x^2 + C_1\)，化简：

\[ \boxed{x^2 + y^2 = C,} \]

其中 \(C = 2 C_1 > 0\)（必须正，因为左边是平方和）。这是一族同心圆，圆心在原点，半径 \(\sqrt{C}\)。隐式解的形式比显式更对称、更几何。如果非要写显式，得分上下半圆：

\[ y = \sqrt{C - x^2} \quad (\text{上半圆}), \qquad y = -\sqrt{C - x^2} \quad (\text{下半圆}), \]

各自定义在 \(|x| < \sqrt{C}\) 上。一个解就被切成了两段——这正是为什么隐式形式更得人心。

什么时候坚持要显式呢？如果你接下来需要对 \(y(x)\) 做某个具体的代入或评估（比如 IVP 里要用初始条件定常数、再代入算 \(y(2)\) 的具体值），显式形式就方便。否则隐式形式留着不动也无伤大雅——只要写出"任意 \(x\) 与 \(y\) 满足 \(F(x, y) = C\) 即为该方程的一组解"，就算交差。

把上面的例子做成 IVP：求解 \(y' = -x/y\)、\(y(0) = 3\)。代入隐式通解 \(x^2 + y^2 = C\)：用初始条件 \(0 + 9 = C\)，所以 \(C = 9\)。隐式特解：\(x^2 + y^2 = 9\)。要显式：因为 \(y(0) = 3 > 0\)，挑上半圆 \(y(x) = \sqrt{9 - x^2}\)，定义在 \(|x| < 3\) 上。验证：\(y'(x) = \dfrac{-x}{\sqrt{9 - x^2}} = -\dfrac{x}{y}\) ✓。

2.4 被丢掉的零解

分离变量法有一个非常容易被忽略的小坑，所有人在初学时都会踩一次：在我们把 \(g(y)\) 从分母上"赶到左边"那一步，需要的隐含条件是 \(g(y) \neq 0\)。也就是说，我们把方程两边除以了 \(g(y)\)，而除法不允许除以零。但 \(g(y) = 0\) 这种情况，恰恰对应着方程的某些常数解——这些解叫做平衡解，它们经常在分离的那一刻被悄无声息地丢掉。

讲清楚这件事，靠下面这个观察。如果存在某个 \(y_0\) 使 \(g(y_0) = 0\)，那么常数函数 \(y(x) \equiv y_0\) 就是原方程 \(y' = f(x) g(y)\) 的解——因为左边 \(y' = 0\)（常数函数的导数为零），右边 \(f(x) g(y_0) = f(x) \cdot 0 = 0\)。所以 \(g\) 的每个零点都对应一个平衡解。

这些解通常不会自动出现在用分离变量得到的"通解"\(F(x, y) = C\) 的某个具体 \(C\) 取值里。要么它们对应"\(C \to \infty\)"这种边界极限（在计算时被忽略），要么它们根本就脱离了那一族曲线。所以你必须额外把它们补到答案里。

看一下 \(y' = y\) 这个例子。\(g(y) = y\)，零点是 \(y = 0\)。所以 \(y(x) \equiv 0\) 是一个解。我们 2.2.1 节得到的通解 \(y = C e^x\) 在 \(C = 0\) 时刚好就是这个零解。所以这个例子里，零解"自动"被通解吸收，没有额外问题。

再看 \(y' = y^{2/3}\) 这个之前提到过的"病态"例子。\(g(y) = y^{2/3}\)，零点是 \(y = 0\)。所以 \(y \equiv 0\) 是一个解。分离变量：\(y^{-2/3} dy = dx\)，积分 \(3 y^{1/3} = x + C\)，于是

\[ y(x) = \left(\frac{x + C}{3}\right)^3. \]

这一族解里，没有任何一个 \(C\) 值能让 \(y\) 恒为 0——它们全是三次曲线，只在 \(x = -C\) 这一点穿过 0，其他时候不为 0。所以零解 \(y \equiv 0\) 必须额外补上：完整解集是 \(\{ y = ((x + C)/3)^3 : C \in \mathbb{R} \} \cup \{ y \equiv 0 \}\)。

更进一步，因为 \(y \equiv 0\) 可以"在任意 \(x = a\) 处"突然变成 \(((x - a)/3)^3\) 那一支三次曲线，整个解的非唯一性变得相当严重——这正是 1.6 节预告过的、皮卡尔定理失效的地方。我们到第 4 章再细讲这件事。

实践中的动作：每次用分离变量法之前，先看一眼 \(g(y) = 0\) 的根。把这些 \(y_0\) 写下来，告诉自己 "\(y \equiv y_0\) 也是解"。算完通解之后，检查一下你的通解是不是已经包含了这些零解；如果没有，把它们补上。这个习惯会让你的答案更完整，也能在大题打分时挽回容易丢的"完整性"分数。

反过来想这件事也很值得。"丢解"的根源是除以零——而 \(g(y) = 0\) 的那些 \(y_0\) 在原方程中具有特殊地位：它们让 \(y' = 0\)，即解曲线在那里的斜率为零。如果一条解曲线"碰到"了 \(y = y_0\) 这条水平线，它会立刻变成水平的，接下来要么沿着这条水平线一直走（保持是平衡解），要么离开这条线（如果从这条线"出发"也算解）。这就是 1.6 节预告的非唯一性的几何根源。皮卡尔条件（\(\partial f/\partial y\) 在初值点连续）的真正物理含义是：禁止 \(g\) 在 \(y_0\) 处出现"导数无穷"或"斜率无穷"的奇异，从而保证解曲线一旦与平衡线接触，就必须被吸进去并且不能再离开。\(y' = 3 y^{2/3}\) 那个例子里 \(g(y) = 3 y^{2/3}\) 在 0 处的导数是 \(2 y^{-1/3} \to \infty\)，恰好违背皮卡尔条件，于是解能从 0 这条线随时"分叉"出去。

把这件事记成一句口号：除以零的时候，丢的不是无关紧要的特例，而是平衡解，而平衡解恰好是物理意义最丰富的那些解。所以从今往后，分离变量第一步——把 \(g(y)\) 移到分母——一旦完成，你就立刻在边上写一行 "\(y \equiv y_0\)（\(g(y_0) = 0\) 的根）也是解"，永远不要忘。

2.5 详细例题

规则懂了，现在用它解决第 1 章列出的那些建模问题。每个例题都按同一套流程走：建模 → 写出方程 → 判别可分离 → 分离变量 → 两边积分 → 用初始条件定常数 → 代回检验 → 解释物理。这一遍走完，你应该对"分离变量"的肌肉记忆基本到位。

2.5.1 例 1：人口指数增长

解 \(\dfrac{dP}{dt} = k P\)，\(P(0) = P_0\)，其中 \(k > 0\) 是繁殖率。

分离：\(\dfrac{dP}{P} = k \, dt\)。

积分：\(\ln |P| = k t + C_1\)，所以 \(P = C e^{k t}\)（吸收 \(\pm e^{C_1}\) 进新常数 \(C\)）。

用初始条件 \(P(0) = P_0\)：\(P_0 = C e^0 = C\)，所以 \(C = P_0\)。

最终特解：

\[ \boxed{P(t) = P_0 \, e^{k t}.} \]

这就是经典的指数增长公式。注意零解 \(P \equiv 0\) 自动被包进去了（\(P_0 = 0\) 的情形）——一个种群初始为 0 就永远是 0，再合理不过。

来一个具体数字：某细菌培养液在 \(t = 0\) 时有 1000 个细菌，1 小时后有 4000 个。问 3 小时后有多少？先用 1 小时的数据求 \(k\)：

\[ 4000 = 1000 \cdot e^{k \cdot 1} \;\Longrightarrow\; e^k = 4 \;\Longrightarrow\; k = \ln 4. \]

然后 \(P(3) = 1000 \cdot e^{3 \ln 4} = 1000 \cdot 4^3 = 64000\)。3 小时后有 6.4 万个。和你直觉一致——每小时翻 4 倍，3 小时翻 \(4^3\) 倍。

2.5.2 例 2：放射性衰变与碳测年

解 \(\dfrac{dN}{dt} = -\lambda N\)，\(N(0) = N_0\)。

步骤跟人口几乎一样，结果是 \(N(t) = N_0 e^{-\lambda t}\)（指数为负，因为 \(\lambda > 0\) 且方程右边带负号）。

"半衰期" \(T_{1/2}\) 的定义：经过这么久后剩下一半。即 \(N(T_{1/2}) = N_0 / 2\)，代入得 \(\dfrac{1}{2} = e^{-\lambda T_{1/2}}\)，取对数：\(\lambda = \dfrac{\ln 2}{T_{1/2}}\)。这就是 1.2.2 提到的关系式。

具体：碳-14 半衰期 5730 年。一个考古样本里测得 \(^{14}\mathrm{C}\) 含量是新鲜样本的 25%。它有多老？

\[ 0.25 \, N_0 = N_0 \, e^{-\lambda t} \;\Longrightarrow\; t = \frac{\ln 4}{\lambda} = \frac{\ln 4}{\ln 2 / 5730} = \frac{2 \ln 2}{\ln 2} \cdot 5730 = 2 \cdot 5730 = 11460 \text{ 年}. \]

两个半衰期，吻合（每半衰期剩 \(1/2\)，两次后剩 \(1/4 = 25\%\)）。这种"指数衰变"模型让你能从化学测量直接推出年代——是考古学的一个核心工具。

2.5.3 例 3：牛顿冷却

解 \(\dfrac{dT}{dt} = -k(T - T_a)\)，\(T(0) = T_0\)。

这次右边不是 \(\pm k T\)，多了一个常数。但仍然可分离：

\[ \frac{dT}{T - T_a} = -k \, dt \;\Longrightarrow\; \ln |T - T_a| = -k t + C_1 \;\Longrightarrow\; T - T_a = C e^{-k t}. \]

初始条件：\(T(0) - T_a = T_0 - T_a = C\)，所以

\[ \boxed{T(t) = T_a + (T_0 - T_a)\, e^{-k t}.} \]

这个公式的"几何"意思一目了然。\(T_a\) 是渐近线（环境温度），\(T_0 - T_a\) 是"初始温差"，\(e^{-k t}\) 是温差以指数速率衰减的因子。当 \(t \to \infty\)，\(T \to T_a\)——咖啡终归要凉到与房间同温。当 \(t = 0\)，\(T = T_0\)——初始时刻确实是初始温度。两头都对得上。

具体：90 ℃ 的咖啡放在 20 ℃ 的房间，10 分钟后温度降到 70 ℃。问再过 10 分钟（共 20 分钟）时，咖啡是几度？

先用 10 分钟数据求 \(k\)：\(70 = 20 + (90 - 20) e^{-k \cdot 10} = 20 + 70 e^{-10k}\)，所以 \(50 = 70 e^{-10k}\)，\(e^{-10k} = 5/7\)，\(k = \dfrac{1}{10} \ln(7/5) \approx 0.0336\) /分钟。

然后 \(T(20) = 20 + 70 e^{-20 k} = 20 + 70 (e^{-10k})^2 = 20 + 70 \cdot (5/7)^2 = 20 + 70 \cdot 25/49 = 20 + 250/7 \approx 55.7\) ℃。

所以再过 10 分钟咖啡降到约 55.7 ℃。注意一个有意思的事实：第一个 10 分钟降了 20 ℃（90 → 70），第二个 10 分钟却只降了约 14 ℃（70 → 55.7）——温差越小，降温越慢，这是牛顿冷却的几何特征。指数衰减的"半衰式"行为：每隔同样时间，温差按相同比例缩小。

2.5.4 例 4：自由落体加阻力

解 \(m \dfrac{dv}{dt} = m g - b v\)，\(v(0) = 0\)（从静止开始下落）。

把方程整理成可分离形式。两边除以 \(m\)：\(\dfrac{dv}{dt} = g - \dfrac{b}{m} v\)。记 \(\alpha = b/m\)、\(v_\infty = g/\alpha = m g / b\)（终端速度）。则方程变成 \(\dfrac{dv}{dt} = \alpha (v_\infty - v)\)——结构上和牛顿冷却一模一样，只是字母换了。

分离与积分（步骤略，跟例 3 同型）：

\[ v(t) = v_\infty + (v_0 - v_\infty) e^{-\alpha t} = v_\infty \big(1 - e^{-\alpha t}\big), \]

其中第二个等号用了 \(v_0 = 0\)。

解释。\(v(0) = 0\) ✓。\(t \to \infty\) 时 \(v \to v_\infty\)，速度趋向终端速度。"达到 99% 终端速度"需要 \(e^{-\alpha t} = 0.01\)，即 \(t = \dfrac{\ln 100}{\alpha} \approx \dfrac{4.6 m}{b}\)。对一个体重 70 kg、阻力系数 \(b \approx 13\) kg/s 的没开伞的跳伞员来讲，\(\alpha \approx 0.186\) /s，达 99% 终端速度需要约 25 秒——之后基本就匀速下落了。

2.5.5 例 5：化学反应（一级反应）

化学动力学里最简单的反应模型是一级反应：反应物 A 单独分解成产物 B，反应速率与 A 的浓度成正比。设 \([A](t)\) 是 A 在时刻 \(t\) 的浓度。"反应速率"在化学里就是 \(-d[A]/dt\)（A 在减少，所以加负号让速率为正）。一级反应的方程：

\[ -\frac{d[A]}{dt} = k\, [A], \quad \text{即} \quad \frac{d[A]}{dt} = -k\, [A]. \]

这又是"指数衰减"方程！解：\([A](t) = [A]_0 e^{-k t}\)。在化学里你也常常看到"半衰期"这个量——过了 \(T_{1/2}\) 后反应物剩一半，关系仍然是 \(T_{1/2} = (\ln 2)/k\)。

一个具体题。\(N_2 O_5\) 在 45 ℃ 下分解，是一级反应，\(k = 6.2 \times 10^{-4}\) /s。从 0.50 mol/L 开始，多久后浓度降到 0.10 mol/L？

\[ 0.10 = 0.50 \cdot e^{-k t} \;\Longrightarrow\; e^{-k t} = 1/5 \;\Longrightarrow\; t = \frac{\ln 5}{k} = \frac{1.609}{6.2 \times 10^{-4}} \approx 2596 \text{ 秒} \approx 43 \text{ 分钟}. \]

下一步可以做更复杂的二级反应 \(d[A]/dt = -k [A]^2\)——它仍然可分离。我留作课后练习：分离 \([A]^{-2} d[A] = -k \, dt\)，积分得 \(-1/[A] = -k t - 1/[A]_0\)，于是 \(1/[A] = 1/[A]_0 + k t\)，即 \([A](t) = \dfrac{[A]_0}{1 + k [A]_0 t}\)。注意这次的衰减是幂次型而不是指数型——浓度按 \(1/t\) 量级慢慢趋零，远比一级反应慢。这种"反应级数"的差异在化学动力学里有具体的实验诊断意义：把数据画成 \(\ln [A]\) vs \(t\) 是直线，则反应是一级；画 \(1/[A]\) vs \(t\) 是直线，则是二级。同样的方程语法，背后是不同的反应机理。

2.5.6 例 6：双曲型动力学 \(y' = (1 - y)(1 + y)\)

解 \(\dfrac{dy}{dx} = 1 - y^2 = (1 - y)(1 + y)\)。\(g(y) = (1-y)(1+y)\) 的零点是 \(y = \pm 1\)，所以 \(y \equiv 1\) 与 \(y \equiv -1\) 都是平衡解，先记下。

分离：\(\dfrac{dy}{(1-y)(1+y)} = dx\)。左边用部分分式：\(\dfrac{1}{(1-y)(1+y)} = \dfrac{1/2}{1-y} + \dfrac{1/2}{1+y}\)。积分：

\[ \frac{1}{2}\Big(-\ln|1-y| + \ln|1+y|\Big) = x + C_1, \quad \text{即} \quad \frac{1}{2}\ln\left|\frac{1+y}{1-y}\right| = x + C_1. \]

两边乘 2 取指数：\(\left|\dfrac{1+y}{1-y}\right| = e^{2 C_1} e^{2x}\)，把符号与常数吸成 \(A\)：\(\dfrac{1+y}{1-y} = A e^{2x}\)。解 \(y\)：

\[ y(x) = \frac{A e^{2x} - 1}{A e^{2x} + 1}. \]

这是一个相当漂亮的解。让 \(t \to \infty\)，\(A e^{2x}\) 占绝对优势，\(y \to 1\)；让 \(x \to -\infty\)，\(y \to -1\)。所以所有"内部"的解（\(|y| < 1\)）都是从 \(-1\) 平稳过渡到 \(+1\) 的 S 型曲线。回头看两个平衡解：\(y \equiv 1\) 是稳定平衡（吸引），\(y \equiv -1\) 是不稳定平衡（排斥）——把方向场画出来一目了然。这种解叫做双曲正切波形（事实上当 \(A = 1\) 时，\(y(x) = \tanh x\) 恰好就是 \(\tanh\) 函数）。

2.5.7 例 7：逻辑斯蒂方程

把 1.2.1 提到的"现实化"人口模型解出来：

\[ \frac{dP}{dt} = k P \left(1 - \frac{P}{M}\right), \qquad P(0) = P_0, \]

其中 \(M > 0\) 是承载量，\(k > 0\) 是低密度时的繁殖率。

注意先看 \(g(P) = P(1 - P/M) = 0\) 的根：\(P = 0\) 与 \(P = M\)。这两个分别是平衡解 \(P \equiv 0\)（彻底灭绝）和 \(P \equiv M\)（饱和）。把它们记下来，免得最后忘掉。

分离：\(\dfrac{dP}{P(1 - P/M)} = k \, dt\)。左边的积分需要部分分式分解。注意 \(P(1 - P/M) = P (M - P)/M\)，所以

\[ \frac{dP}{P(1 - P/M)} = \frac{M \, dP}{P (M - P)} = \left(\frac{1}{P} + \frac{1}{M - P}\right) dP, \]

用了部分分式 \(\dfrac{M}{P(M-P)} = \dfrac{1}{P} + \dfrac{1}{M-P}\)（验证：通分后分子是 \((M-P) + P = M\) ✓）。

所以左边积分等于 \(\ln|P| - \ln|M - P| = \ln \left| \dfrac{P}{M - P} \right|\)。两边对 \(t\) 积分得

\[ \ln \left| \frac{P}{M - P} \right| = k t + C_1 \;\Longrightarrow\; \frac{P}{M - P} = A e^{k t}, \]

其中 \(A\) 是新的任意常数（吸收了符号与 \(e^{C_1}\)）。用 \(P(0) = P_0\) 定 \(A\)：\(A = P_0 / (M - P_0)\)。然后把 \(P\) 解出来：

\[ \boxed{P(t) = \frac{M P_0}{P_0 + (M - P_0) e^{-k t}}.} \]

验证关键点。\(t = 0\)：分母为 \(P_0 + (M - P_0) = M\)，于是 \(P(0) = M P_0 / M = P_0\) ✓。\(t \to \infty\)：\(e^{-kt} \to 0\)，分母 \(\to P_0\)，于是 \(P \to M\) ✓——种群最终趋于承载量。当 \(P_0\) 远小于 \(M\) 时，前期 \(P\) 的增长接近指数（因为 \(1 - P/M \approx 1\)，方程退化为 \(P' \approx kP\)），后期被压平到 \(M\)。这就是著名的"S 型曲线"，在生物、市场普及、传染病早期等领域反复出现。

顺便：两个平衡解 \(P \equiv 0\) 和 \(P \equiv M\) 都被通解吸收了吗？\(P \equiv M\) 是 \(t \to \infty\) 极限，可以视为某种"\(P_0 = M\)"的退化情形（代入公式即得 \(P(t) \equiv M\)）；\(P \equiv 0\) 对应 \(P_0 = 0\)（分子分母都变成 0，需要小心处理，但极限上确实是 0）。所以实际上两个平衡解都已自然地被这个公式涵盖。

2.5.8 例 8：药物代谢——给完一剂之后

把"混合容器"的思路搬到药代动力学里。设一位病人血液里某药物浓度为 \(C(t)\)（mg/L）。药物以一级反应被肝脏代谢清除（速率常数 \(k\)），且没有外源补给。则

\[ \frac{dC}{dt} = -k C, \quad C(0) = C_0, \]

解：\(C(t) = C_0 e^{-k t}\)。这跟放射衰变一模一样，"半衰期"概念在药代里也叫 \(t_{1/2}\)，意义相同。

具体：阿司匹林口服后血液里峰值浓度 \(C_0\) = 50 mg/L，\(k = 0.139\) /h（对应半衰期约 5 小时）。问 12 小时后血液里还剩多少？

\[ C(12) = 50 \cdot e^{-0.139 \cdot 12} = 50 \cdot e^{-1.668} \approx 50 \cdot 0.189 = 9.4 \text{ mg/L}. \]

大约还剩 19%。这就是为什么药品说明书上会写"每 4-6 小时一次"——不是为了简便，而是要在药效完全衰退前补上下一剂，让血液浓度大致稳定在治疗窗内。

这道题表面上看跟放射性、跟人口、跟 RC 放电毫无关系，但数学上是同一条方程。再次印证了我们在第 1 章里说过的话：少数几个微分方程，描述了非常多的现象。

2.6 验证：把答案代回去

解完一个微分方程后，永远把答案代回原方程检验一下。这件事不花什么时间，却能在你算错时第一时间警示你。微分方程的可贵之处就在这里：哪怕你不会解，你至少永远会验证。

验证的标准动作：

  1. 写下你的解 \(y(x)\)。
  2. 对它求导，得到 \(y'(x)\)（如果是高阶方程，求到所需的最高阶）。
  3. 把 \(y\)、\(y'\)（以及 \(y''\) 等等）代入原方程的左右两边。
  4. 化简两边；它们应该对所有 \(x\) 恒等。
  5. 如果还有初始条件，把 \(x = x_0\) 代入 \(y(x_0)\)（以及 \(y'(x_0)\) 等等），检查是否符合给定值。

我们把上面 2.5 节的几个解逐一抽检一下。

解 \(P(t) = P_0 e^{k t}\) 验证 \(P' = k P\)。求导：\(P' = P_0 \cdot k \cdot e^{k t} = k \cdot P_0 e^{k t} = k P\) ✓。初始：\(P(0) = P_0 e^0 = P_0\) ✓。

解 \(T(t) = T_a + (T_0 - T_a) e^{-k t}\) 验证 \(T' = -k(T - T_a)\)。求导：\(T'(t) = (T_0 - T_a) \cdot (-k) e^{-k t} = -k (T_0 - T_a) e^{-k t}\)。再算 \(-k(T - T_a)\)：注意 \(T - T_a = (T_0 - T_a) e^{-k t}\)，于是 \(-k(T - T_a) = -k (T_0 - T_a) e^{-k t}\)，与 \(T'\) 一致 ✓。初始：\(T(0) = T_a + (T_0 - T_a) = T_0\) ✓。

解 \(P(t) = \dfrac{M P_0}{P_0 + (M - P_0) e^{-k t}}\) 验证逻辑斯蒂方程。这个验证略繁，但是好的练习。求导用商法则：

\[ P'(t) = \frac{- M P_0 \cdot (M - P_0)(-k) e^{-k t}}{[P_0 + (M - P_0) e^{-k t}]^2} = \frac{k M P_0 (M - P_0) e^{-k t}}{[\,\cdot\,]^2}. \]

另一边 \(k P (1 - P/M)\)：

\[ k P \cdot \frac{M - P}{M} = \frac{k}{M} \cdot \frac{M P_0}{[\,\cdot\,]} \cdot \frac{M [P_0 + (M - P_0) e^{-k t}] - M P_0}{[\,\cdot\,]} = \frac{k}{M} \cdot \frac{M P_0 \cdot M (M - P_0) e^{-k t}}{[\,\cdot\,]^2} \]

化简后等于 \(\dfrac{k M P_0 (M - P_0) e^{-k t}}{[\,\cdot\,]^2}\)，与 \(P'\) 一致 ✓。这种"代回看相不相同"的过程从来不优雅，但它是你唯一不需要任何额外洞察就能确认答案正确的手段。建议每次解完一个 IVP，都至少在草稿上做一遍。

本章到此结束。可分离方程的全部内容你已经掌握：判别一个方程是否可分离 → 分离 → 两边积分 → 隐式或显式写解 → 不要忘记零解 → 用初始条件定常数 → 代回验证。这条流程将在你日后处理无数实际建模问题时反复使用。

临走前回顾一下我们这一章里反复见到的几张方程"脸"：

  • \(y' = ky\) 与 \(y' = -ky\)。指数增长 / 指数衰减。出现在人口、放射衰变、化学一级反应、药代、电容放电、连续复利等等。解：\(y = y_0 e^{\pm k t}\)。
  • \(y' = -k(y - y_*)\)（仿射形式）。带平衡点的指数趋近。出现在牛顿冷却、终端速度下落、RC 充电（恒压）。解：\(y = y_* + (y_0 - y_*) e^{-k t}\)。
  • \(y' = ky(1 - y/M)\)。逻辑斯蒂方程（自我限制增长）。出现在生物种群、市场普及、新技术扩散。解：S 型曲线。
  • \(y' = 1 + y^2\)。带反馈加速的方程。出现在某些非线性物理。解：\(\tan(x + C)\)，有限时间爆破。

这四种"脸"在工程与科学课程里出现的频率远高于其他类型。把它们的解曲线"长相"印在脑子里，将来碰到具体应用题，往往不用解就能猜出答案的趋势——只需要把数代进去把常数定下来。

下一章我们处理另一类同样基础、同样实用的一阶方程：线性方程。它们形如 \(y' + p(x) y = q(x)\)，跟可分离的不同之处在于：右边有一个让人讨厌的 \(p(x) y\) 项，把 \(x\) 与 \(y\) 缠到了一起。我们将引入"积分因子"这一漂亮技巧，硬生生把那一团缠绕给解开。看完第 3 章你就拥有了求解一阶方程的两大主力工具——其中绝大多数实际问题都被它们俩接管了。
