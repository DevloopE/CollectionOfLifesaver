第 20 章 波动方程

第 19 章的整套机器——分离变量、本征函数、傅里叶级数——是为热方程建立的,但它的力量远不止于此。同样的步骤,几乎一字不变,可以套到波动方程上。区别在哪里?在时间方程:热方程的时间因子是衰减的指数 \(e^{-\alpha \lambda_n t}\),而波动方程的时间因子是振荡的 \(\cos(c\sqrt{\lambda_n} t)\) 与 \(\sin(c\sqrt{\lambda_n} t)\)。这一点细微的代换,把整个方程从"信息消散"翻转成"信息振荡保留",物理上从"扩散"翻转成"传播"。本章就是把这个翻转一步步演下来。

除了分离变量法,本章还要介绍一个完全不同的视角——D'Alembert 解。它把波动方程的解写成"右行波 + 左行波"的形式,在无界域上特别透彻地解释了为什么波动方程要"传播"东西。这两种视角(本征函数级数 vs 行波)是同一套解的两种"表象",像量子力学里位置表象与动量表象一样,各有各的擅长场景:本征函数视角对有界域和共振分析最自然(乐器的振动模),行波视角对无界域和因果性分析最自然(光以速度 \(c\) 走出去)。学完本章你应当两种都会、都能切换。

本章结构:20.1-20.6 是"对应版"的分离变量,与第 19 章对照着读;20.7-20.8 是 D'Alembert 解;20.9 是热方程与波动方程的物理对比;20.10 给一道经典完整算例(拨弦)。

20.1 弦振动方程

物理设定:一根长 \(L\) 的弦(吉他弦、小提琴弦),沿 \(x\) 轴拉在 \(0 \le x \le L\) 上,两端 \(x = 0\) 与 \(x = L\) 被钉死(\(u = 0\) 那里)。弦上每一点 \(x\) 在时刻 \(t\) 偏离平衡位置的位移记作 \(u(x, t)\)。在 \(t = 0\) 时,你给弦两个初始信息:它的初始形状 \(u(x, 0) = f(x)\),以及它的初始速度 \(u_t(x, 0) = g(x)\)。问:之后每个 \(t\),弦的形状是什么?

\[ \boxed{\begin{aligned} & u_{tt} = c^2\, u_{xx}, & 0 < x < L,\ t > 0, \\ & u(0, t) = 0,\ u(L, t) = 0, & t > 0, \\ & u(x, 0) = f(x),\ u_t(x, 0) = g(x), & 0 \le x \le L. \end{aligned}} \]

波速 \(c\)(单位 m/s)由弦的张力 \(T\) 与线密度 \(\rho\) 决定:\(c = \sqrt{T/\rho}\)。这是从牛顿第二定律应用到一小段弦上的物理推导得出的——本书不展开,但你应该相信"张力越大、密度越小,波就传得越快",这一点和直觉一致(吉他弦拉得越紧调子越高)。

初始条件为什么需要两个?因为时间方程 \(u_{tt} = c^2 u_{xx}\) 关于时间是二阶的——和二阶 ODE 类比,需要两个初值才能定解。物理上很直白:一根弦的运动既取决于你最初把它拉到哪里(\(f\)),也取决于你松手时它是否带速度(\(g\))。如果你拉成形状 \(f\) 然后松手,初速度是零,\(g \equiv 0\);如果你"打"弦让它从平衡位置开始有速度,但形状仍是平直的,则 \(f \equiv 0\)、\(g\) 给定。两种激发方式产生不同的演化。

顺便插一句历史:波动方程是 18 世纪由达朗贝尔(d'Alembert)、欧拉(Euler)、丹·伯努利(Daniel Bernoulli)三位大师围绕"振动弦的形状"问题反复争论得出的。Bernoulli 主张"任何初始形状都可以分解成正弦模态的叠加"——这正是分离变量法的核心思想,但当时被欧拉等人强烈质疑(他们不相信"任意函数能写成 \(\sin\) 级数")。半个世纪后傅里叶在热方程中重新提出同样的想法,才最终把这场争论拍板了。所以你下面看到的 \(\sin\) 模态分解,是 200 年前一群顶级数学家吵了几十年才接受的洞见。

20.2 分离变量

同样的剧本。假设 \(u(x, t) = X(x) T(t)\),代入波动方程:

\[ X(x) T''(t) = c^2 X''(x) T(t). \]

除以 \(c^2 X(x) T(t)\):

\[ \frac{T''(t)}{c^2 T(t)} = \frac{X''(x)}{X(x)}. \]

左只依赖 \(t\),右只依赖 \(x\)——同样的论证告诉你两边等于一个常数 \(-\lambda\):

\[ \frac{T''(t)}{c^2 T(t)} = \frac{X''(x)}{X(x)} = -\lambda, \]

分裂成两个 ODE:

\[ T''(t) + c^2 \lambda\, T(t) = 0, \qquad X''(x) + \lambda\, X(x) = 0. \]

BC 同样把 \(X(0) = X(L) = 0\) 搬过来。空间问题与第 19 章一模一样:

\[ X'' + \lambda X = 0, \quad X(0) = X(L) = 0. \]

所以本征值与本征函数也一模一样:

\[ \lambda_n = \left(\frac{n\pi}{L}\right)^2, \quad X_n(x) = \sin\!\left(\frac{n\pi x}{L}\right), \quad n = 1, 2, 3, \ldots \]

(请重读 19.3 节确认这一推导。)只有时间方程不一样——下一节解。

这种"空间方程一致、时间方程不同"的现象很值得在大脑里留下一个印章:空间几何决定本征函数,时间结构决定时间因子。换一个 PDE,换一个时间方程;换一个几何,换一组本征函数。这是分离变量法的"模块化设计":你可以把空间部分(\(\sin(n\pi x/L)\))像积木一样反复使用。

20.3 时间部分:振荡而非衰减

每个本征值 \(\lambda_n\) 对应时间方程

\[ T_n''(t) + c^2 \lambda_n T_n(t) = 0, \]

这是一个二阶线性齐次 ODE。把 \(c^2 \lambda_n = c^2 (n\pi/L)^2 = \omega_n^2\),其中

\[ \boxed{\omega_n = \frac{c n \pi}{L}} \]

叫第 \(n\) 个本征频率(或本征角频率)。时间方程化为

\[ T_n''(t) + \omega_n^2 T_n(t) = 0. \]

这是经典的简谐振子方程!通解是

\[ T_n(t) = a_n \cos(\omega_n t) + b_n \sin(\omega_n t). \]

对照热方程的时间解 \(T_n(t) = C_n e^{-\alpha \lambda_n t}\)(指数衰减):波动方程的时间解是振荡。它不衰减,也不增长——能量被永远振荡地保存在第 \(n\) 个模态里(理想情况下,无阻尼)。这就是"波动方程不抹细节"的根源:每一个频率成分一直保留下去,只是在两端反弹来回。

"角频率"\(\omega_n\) 与日常概念中的"频率"\(f_n\)(每秒周期数)的关系是 \(\omega_n = 2\pi f_n\),所以 \(f_n = c n / (2L)\)。乐器学把 \(f_1 = c/(2L)\) 称为基频(fundamental frequency),把 \(f_2, f_3, \ldots\) 称为泛音(overtones)。一根 0.65m 的吉他高音 E 弦,如果调到张力让 \(c \approx 430\) m/s,那基频是 \(430/(2 \cdot 0.65) \approx 330\) Hz——正是 E4 音的频率(标准吉他高音 E)。改变张力或长度都会改 \(c\) 或 \(L\),从而改频率,这正是调音和按弦的物理原理。

20.4 完整通解

每个本征值 \(\lambda_n\) 给出分离形式的解

\[ u_n(x, t) = \big[a_n \cos(\omega_n t) + b_n \sin(\omega_n t)\big] \sin\!\left(\frac{n\pi x}{L}\right), \]

由线性叠加得

\[ \boxed{u(x, t) = \sum_{n=1}^\infty \big[a_n \cos(\omega_n t) + b_n \sin(\omega_n t)\big] \sin\!\left(\frac{n\pi x}{L}\right),} \]

其中 \(\omega_n = c n \pi / L\)。这是带固定端 BC 的波动方程的形式通解。它满足 PDE 与 BC,只剩 IC——下一节用 IC 把 \(a_n\) 与 \(b_n\) 钉死。

注意每个 \(n\) 项是一个驻波:空间部分 \(\sin(n\pi x/L)\) 决定波形(在哪里振、在哪里不振),时间部分 \(\cos + \sin\) 决定振幅随时间的振荡。"驻波"的意思是它没有移动——波节(零点)固定在 \(x = kL/n\),\(k = 0, 1, \ldots, n\) 处永不动,中间的弦上下振动。每一根吉他弦或者钢琴弦发出的声音,本质就是这些驻波的叠加。

20.5 用 IC 求系数

初始位移条件 \(u(x, 0) = f(x)\):令 \(t = 0\),所有 \(\sin(\omega_n \cdot 0) = 0\) 项消失,所有 \(\cos(\omega_n \cdot 0) = 1\) 项保留,得

\[ f(x) = \sum_{n=1}^\infty a_n \sin\!\left(\frac{n\pi x}{L}\right). \]

这正是 \(f\) 的傅里叶正弦展开,所以

\[ \boxed{a_n = \frac{2}{L}\int_0^L f(x) \sin\!\left(\frac{n\pi x}{L}\right) dx.} \]

初始速度条件 \(u_t(x, 0) = g(x)\):先对 \(t\) 求导,

\[ u_t(x, t) = \sum_{n=1}^\infty \big[-a_n \omega_n \sin(\omega_n t) + b_n \omega_n \cos(\omega_n t)\big] \sin\!\left(\frac{n\pi x}{L}\right), \]

令 \(t = 0\) 给

\[ g(x) = \sum_{n=1}^\infty b_n \omega_n \sin\!\left(\frac{n\pi x}{L}\right). \]

这是 \(g\) 的傅里叶正弦展开,但系数是 \(b_n \omega_n\),所以

\[ b_n \omega_n = \frac{2}{L}\int_0^L g(x) \sin\!\left(\frac{n\pi x}{L}\right) dx, \]

即

\[ \boxed{b_n = \frac{2}{L \omega_n}\int_0^L g(x) \sin\!\left(\frac{n\pi x}{L}\right) dx = \frac{2}{c n \pi}\int_0^L g(x) \sin\!\left(\frac{n\pi x}{L}\right) dx.} \]

有了 \(a_n\) 与 \(b_n\),代回 20.4 节的级数,弦的运动 \(u(x, t)\) 就完全确定。流程和热方程一模一样,只是因为时间是二阶,需要两组系数(\(a_n\) 给位移,\(b_n\) 给速度)。

看一道极简练习。\(L = \pi\)(把数字凑整以省力),\(c = 1\),初始位移 \(f(x) = 3\sin x + \sin(2x)\),初始速度 \(g(x) = 0\)。求 \(u(x, t)\)。

初始位移已经"恰好"是基频与第二谐波的叠加,所以系数不必积分就能直接读出:\(a_1 = 3\),\(a_2 = 1\),其他 \(a_n = 0\)。初始速度为 0,所以所有 \(b_n = 0\)。\(\omega_n = c n \pi / L = n \pi / \pi = n\)。所以

\[ u(x, t) = 3 \cos(t) \sin x + \cos(2t) \sin(2x). \]

就这。两个驻波,各自以 \(\omega_1 = 1\) 和 \(\omega_2 = 2\) 振荡,叠加而成弦的运动。\(t = 0\) 时给出 \(3\sin x + \sin(2x) = f\) ✓;速度 \(u_t = -3\sin t \cdot \sin x - 2 \sin(2t)\sin(2x)\) 在 \(t = 0\) 给出 0 ✓。两个 IC 都满足,完工。

这道题为什么这么简单?因为 \(f\) 已经写成本征函数的有限和。一般情况下 \(f\) 是任意的,你必须做积分得到无穷多个 \(a_n\)。但只要 \(f\) 与 \(g\) 已经写成 \(\sin\) 级数,系数就直接读出——这是傅里叶级数最干脆的用法。

20.6 振动模态、基频与泛音

每一个 \(n\) 项 \([a_n \cos(\omega_n t) + b_n \sin(\omega_n t)] \sin(n\pi x/L)\) 叫弦的第 \(n\) 个振动模态(\(n\)-th mode of vibration)。第 1 模态(基频)是最低的纯音;第 2 模态(第一泛音)频率是基频的两倍,听起来高一个八度;第 3、第 4 模态依次升高。一根弦总是同时以这无穷多个模态振动,每个模态有自己的振幅和相位——具体振幅取决于你怎么"激发"它。

这就是为什么不同乐器的同一个音(比如都是中央 C)听起来音色不同。中央 C 的基频是 261.63 Hz——所有调成中央 C 的乐器基频都一样。但泛音的相对强度差别巨大:小提琴的高频泛音很强,所以听起来"明亮";单簧管的偶数泛音较弱(因为吹口相当于一个"封闭端"边界条件,改变了本征函数的形状),所以听起来"温柔"。这种"基频相同但泛音构成不同导致音色不同"的现象叫做音色(timbre),它的物理本质就是傅里叶系数 \(a_n, b_n\) 的不同分布。

对乐器演奏家而言,这个直觉非常有用:

  • 弦长决定基频:\(f_1 = c/(2L)\),按短一半的弦,基频翻倍,音高一个八度。这就是吉他指板上 12 品在弦的中点位置的原因。
  • 张力决定波速:\(c = \sqrt{T/\rho}\),拉紧弦提升 \(T\),\(c\) 增大,基频升高。这就是调音的物理。
  • 触弦点决定泛音:在弦的中点用力弹,会激发奇数模态(因为奇数模态在中点不为零,容易吸收能量),压制偶数模态(它们在中点是节点);在弦的 1/3 处弹,会压制第 3 模态,等等。这就是吉他"自然泛音"的原理。
  • 触弦方式决定振幅分布:用指尖弹(初始位移大、初始速度近零)主要激发 \(a_n\) 模态;用拨片打(初始速度大、初始位移近零)主要激发 \(b_n\) 模态;用弓拉(连续供能)是更复杂的受迫问题。

这些都不是音乐家玄学,而是 20.4-20.5 节公式的直接推论。一段优美的小提琴音乐,本质上是一个连续的"傅里叶系数控制问题"。

20.7 D'Alembert 解

分离变量给出的解是本征函数级数,用一组无穷个驻波拼出运动。但波动方程还有一个完全不同的、闭式的解,在无界(没有 BC,只有 IC)的情形下特别透彻。这就是 1747 年达朗贝尔发现的解。

考虑无界波动方程

\[ u_{tt} = c^2 u_{xx}, \quad x \in \mathbb{R},\ t > 0, \]

带 IC \(u(x, 0) = F(x)\),\(u_t(x, 0) = G(x)\)(用大写 \(F, G\) 强调这是无界版本,与 20.1 节有界版本的 \(f, g\) 区别)。D'Alembert 公式给出

\[ \boxed{u(x, t) = \frac{1}{2}\big[F(x + ct) + F(x - ct)\big] + \frac{1}{2c}\int_{x - ct}^{x + ct} G(s)\, ds.} \]

这个公式直接、闭式、不需要积分变换、不需要级数。它怎么来的?通过一个非常巧妙的变量替换:引入特征坐标

\[ \xi = x - ct, \quad \eta = x + ct. \]

用链式法则把 \(u_{tt}\) 与 \(u_{xx}\) 都换成对 \(\xi, \eta\) 的偏导,你会发现 \(u_{tt} - c^2 u_{xx} = -4 c^2 u_{\xi \eta}\)。所以波动方程在新坐标下变成

\[ u_{\xi \eta} = 0. \]

这是一个非常简单的 PDE!对 \(\eta\) 积分一次得 \(u_\xi = h(\xi)\)(任意函数);再对 \(\xi\) 积分得 \(u = \int h(\xi) d\xi + \psi(\eta) = \phi(\xi) + \psi(\eta)\),其中 \(\phi, \psi\) 是任意一元函数。换回原坐标:

\[ \boxed{u(x, t) = \phi(x - ct) + \psi(x + ct).} \]

这就是波动方程的通解(在无界域上),由两个任意函数 \(\phi, \psi\) 组成。\(\phi(x - ct)\) 描述一个以速度 \(c\) 向右移动的波(把 \(x\) 增大 \(ct\),\(\phi\) 的值不变,所以图像保持不变地向右滑动);\(\psi(x + ct)\) 描述一个以速度 \(c\) 向左移动的波。所以波动方程的每一个解都是右行波 + 左行波的组合——这就是波动方程"传播"性的根本表现。

用 IC 把 \(\phi, \psi\) 钉死。\(u(x, 0) = \phi(x) + \psi(x) = F(x)\)。\(u_t(x, 0) = -c\phi'(x) + c\psi'(x) = G(x)\),即 \(\psi'(x) - \phi'(x) = G(x)/c\)。积分这个等式,\(\psi(x) - \phi(x) = (1/c)\int_0^x G(s) ds + C\)(常数 \(C\) 后面会消掉)。结合 \(\phi + \psi = F\) 解出

\[ \phi(x) = \frac{1}{2}F(x) - \frac{1}{2c}\int_0^x G(s) ds - \frac{C}{2}, \quad \psi(x) = \frac{1}{2}F(x) + \frac{1}{2c}\int_0^x G(s) ds + \frac{C}{2}. \]

代回 \(u = \phi(x - ct) + \psi(x + ct)\):

\[ u(x, t) = \frac{1}{2}\big[F(x - ct) + F(x + ct)\big] + \frac{1}{2c}\left[\int_0^{x + ct} G(s) ds - \int_0^{x - ct} G(s) ds\right] \] \[ = \frac{1}{2}\big[F(x - ct) + F(x + ct)\big] + \frac{1}{2c}\int_{x - ct}^{x + ct} G(s) ds, \]

这正是 D'Alembert 公式。常数 \(C\) 在两项相减时消去了,所以解唯一。

看一道用 D'Alembert 公式的题。\(c = 1\),\(F(x) = e^{-x^2}\)(一个高斯包),\(G(x) = 0\)(从静止开始)。求 \(u(x, t)\)。

\[ u(x, t) = \frac{1}{2}\big[e^{-(x - t)^2} + e^{-(x + t)^2}\big]. \]

解读:初始的高斯包"分裂"成两个高度减半的高斯包,一个以速度 1 向右走 (\(e^{-(x - t)^2}\)),一个以速度 1 向左走(\(e^{-(x + t)^2}\))。它们形状不变,只是位置移动。这就是波分裂(splitting):一个静止释放的初始扰动,会变成大小相同方向相反的两个行波。这跟你在水面上"啪"一下扔颗石头看到圆环涟漪以速度 \(c\) 向外扩展的物理图景完全一致(只是水面是二维,以圆环而不是直线传播)。

再来一道。\(c = 2\),\(F(x) = 0\)(初始位置平直),\(G(x) = 1\) 在 \(|x| < 1\) 内,\(G \equiv 0\) 其他地方(初速给一个方波"敲击")。求 \(u(x, t)\)。

\[ u(x, t) = \frac{1}{2 \cdot 2}\int_{x - 2t}^{x + 2t} G(s) ds = \frac{1}{4}\cdot\big(\text{区间 } [x - 2t, x + 2t] \text{ 与 } [-1, 1] \text{ 的相交长度}\big). \]

当 \(t\) 足够大让积分区间完全覆盖 \([-1, 1]\)(即 \(2t > x + 1\) 且 \(2t > 1 - x\)),交集长度为 2,\(u(x, t) = 1/2\)。当积分区间完全错开 \([-1, 1]\)(即 \(x - 2t > 1\) 或 \(x + 2t < -1\)),交集为 0,\(u = 0\)。中间过渡区间得分情况算。这就是"敲击"在弦上以速度 \(c\) 传出的扰动:每一点最终被传到一个稳态位移 1/2,稳态从距离敲击点的传播时间开始。

20.8 D'Alembert 解的物理意义

D'Alembert 公式的第一项 \(\tfrac{1}{2}[F(x - ct) + F(x + ct)]\) 把初始位移 \(F\) 分成两半,各自以速度 \(c\) 向左右行走。第二项 \(\tfrac{1}{2c}\int_{x - ct}^{x + ct} G(s) ds\) 把初始速度 \(G\) 在区间 \([x - ct, x + ct]\) 上累计——这个区间被称为点 \((x, t)\) 的依赖域(domain of dependence):只有初始数据在这个区间内才能影响 \(u\) 在 \((x, t)\) 处的值。

反过来,从某一点 \(x_0\) 在 \(t = 0\) 时刻开始,初始数据能影响的区域是影响域:在时刻 \(t\),\(x_0\) 的初始数据会影响 \(|x - x_0| \le ct\) 这个区间内的 \(u\)。这一区域是一个张角为 \(\pm c\) 的"光锥"——任何信号都不能传得比 \(c\) 快,这是波动方程内置的"光速极限"。

这种有限传播速度的特性,正是波动方程跟热方程的根本区别。热方程是无限传播速度的:在 \(t > 0\) 的任意时刻,杆上某一点的初始热量都会立刻"感觉到"——因为热核 \((4\pi\alpha t)^{-1/2} e^{-x^2/(4\alpha t)}\) 在所有 \(x\) 处都是非零的,只是衰减得快。这违反相对论,但热方程是非相对论的扩散近似;真实的物理(比如电磁波)用波动方程描述,是相对论性的。

所以波动方程和热方程在"信息传递"的本性上对立:

  • 波动方程:有限传播速度 \(c\),无能量损失,信息振荡保留,可逆。
  • 热方程:无限传播速度,但高频成分指数衰减,信息消散,不可逆。

这条对比是抛物型 vs 双曲型 PDE 性格差异在物理直觉上的最浓缩表达。

20.9 与热方程的对比:保留细节 vs 磨平细节

把第 19.10 节的"磨平"和上一节的"保留"放在一起,做一组干净的对比表:

热方程 \(u_t = \alpha u_{xx}\)波动方程 \(u_{tt} = c^2 u_{xx}\)
类型抛物型双曲型
时间阶一阶二阶
初始条件数1 个(位移)2 个(位移 + 速度)
时间因子\(e^{-\alpha\lambda_n t}\)(衰减)\(\cos(\omega_n t), \sin(\omega_n t)\)(振荡)
对细节磨平(高频快速衰减)保留(高频永远振荡)
可逆性不可逆(熵增)可逆(时间反演对称)
传播速度无限(任何点立即感受)有限(\(c\) 严格上限)
稳态行为趋于 0(被 BC 抽空)永远振荡
典型场景扩散、传热、布朗运动声波、电磁波、机械振动

这两栏几乎条条相反,可方程的形状只差一个 \(_t\) 与 \(_{tt}\)。这是 PDE 教育里一个深刻的教训:方程左边的微分阶数,几乎完全决定了解的物理性格。一阶时间 → 不可逆耗散;二阶时间 → 可逆振荡。这种"形式 → 行为"的强对应,让你看到一个新方程时往往不必算就能猜出大致行为。例如 \(u_t = -i u_{xx}\)(注意虚数 \(-i\))是薛定谔方程(自由粒子),时间也是一阶,但乘了 \(i\),让"衰减"变成"振荡的相位旋转"——所以薛定谔方程兼具热方程的"无限传播"与波动方程的"信息保留"的杂合性格。这是量子力学的核心怪异之一。本书不深入,但你应该看出"PDE 的形式微调如何彻底重塑物理"。

20.10 完整算例:拨弦

把整个第 20 章的工具用到最经典的吉他物理:拨弦的演化。

问题:长度 \(L\) 的弦,两端固定,在距左端 \(L/2\) 处(中点)被拉到位移 \(h\) 后无初速松开。求 \(u(x, t)\)。

初始数据:

\[ f(x) = \begin{cases} 2hx/L, & 0 \le x \le L/2, \\ 2h(L - x)/L, & L/2 \le x \le L \end{cases}, \qquad g(x) = 0. \]

(这是一个三角形:在中点拔到 \(h\),向两端线性下降到 0。)

因为 \(g = 0\),所有 \(b_n = 0\)。算 \(a_n\):

\[ a_n = \frac{2}{L}\int_0^L f(x) \sin\!\left(\frac{n\pi x}{L}\right) dx = \frac{2}{L}\left[\int_0^{L/2} \frac{2hx}{L}\sin\!\left(\frac{n\pi x}{L}\right) dx + \int_{L/2}^L \frac{2h(L - x)}{L}\sin\!\left(\frac{n\pi x}{L}\right) dx\right]. \]

把常数提出:\(a_n = \frac{4h}{L^2}\big[I_1 + I_2\big]\),其中 \(I_1 = \int_0^{L/2} x \sin(n\pi x/L) dx\),\(I_2 = \int_{L/2}^L (L - x)\sin(n\pi x/L) dx\)。

\(I_1\):分部积分,\(u = x\),\(dv = \sin(n\pi x/L) dx\),\(v = -L \cos(n\pi x/L)/(n\pi)\):

\[ I_1 = \left[-\frac{xL \cos(n\pi x/L)}{n\pi}\right]_0^{L/2} + \frac{L}{n\pi}\int_0^{L/2}\cos(n\pi x/L) dx \] \[ = -\frac{L^2 \cos(n\pi/2)}{2 n\pi} + \frac{L^2}{n^2\pi^2}\sin(n\pi/2). \]

\(I_2\):换元 \(y = L - x\),\(dy = -dx\),当 \(x = L/2\) 时 \(y = L/2\),当 \(x = L\) 时 \(y = 0\)。同时 \(\sin(n\pi x/L) = \sin(n\pi(L - y)/L) = \sin(n\pi - n\pi y/L) = \sin(n\pi)\cos(n\pi y/L) - \cos(n\pi)\sin(n\pi y/L)\)。\(\sin(n\pi) = 0\),\(\cos(n\pi) = (-1)^n\),所以 \(\sin(n\pi x/L) = -(-1)^n \sin(n\pi y/L) = (-1)^{n+1}\sin(n\pi y/L)\)。代入:

\[ I_2 = \int_{L/2}^0 y \cdot (-1)^{n+1}\sin(n\pi y/L) (-dy) = (-1)^{n+1}\int_0^{L/2} y \sin(n\pi y/L) dy = (-1)^{n+1} I_1. \]

所以 \(I_1 + I_2 = I_1[1 + (-1)^{n+1}]\)。这个量在 \(n\) 偶数时是 \(I_1 \cdot 0 = 0\),在 \(n\) 奇数时是 \(I_1 \cdot 2 = 2 I_1\)。所以偶数模态完全不被激发!这是中点拨弦的特殊性:中点 \(x = L/2\) 是所有偶数本征函数 \(\sin(2k\pi x/L)\) 的节点,而拉动弦的"凸点"恰好在节点上,激发不了它们。这是上文 20.6 节"触弦点决定泛音"那条规则的具体例子。

对 \(n\) 奇数,\(\cos(n\pi/2) = 0\)(因为 \(n\pi/2\) 是 \(\pi/2\) 的奇数倍,\(\cos\) 在那里为 0),\(\sin(n\pi/2) = (-1)^{(n-1)/2}\)(交替为 \(+1, -1, +1, \ldots\),\(n = 1, 3, 5, 7, \ldots\))。所以 \(I_1 = L^2 (-1)^{(n-1)/2} / (n^2 \pi^2)\),\(I_1 + I_2 = 2 I_1 = 2 L^2 (-1)^{(n-1)/2} / (n^2 \pi^2)\)。

代回 \(a_n = (4h/L^2)(I_1 + I_2)\):

\[ a_n = \frac{4h}{L^2} \cdot \frac{2 L^2 (-1)^{(n-1)/2}}{n^2 \pi^2} = \frac{8 h (-1)^{(n-1)/2}}{n^2 \pi^2} \quad (n \text{ 奇}), \]

\(a_n = 0\) 当 \(n\) 偶。最终

\[ \boxed{u(x, t) = \sum_{n=1, 3, 5, \ldots} \frac{8 h (-1)^{(n-1)/2}}{n^2 \pi^2} \cos\!\left(\frac{c n\pi t}{L}\right) \sin\!\left(\frac{n\pi x}{L}\right).} \]

系数前几项是 \(8h/\pi^2 \cdot \{1, 0, -1/9, 0, 1/25, 0, -1/49, \ldots\}\),所以基频项振幅最大,后续奇数模态以 \(1/n^2\) 衰减,符号交替——这套模态分布就是"中点拨弦"的物理指纹。乐器学家会告诉你,中点拨弦听起来比较"哑"(偶数泛音缺失,八度泛音被禁掉),所以吉他演奏家一般不在 12 品上方拨弦,而在更接近琴桥的地方拨——那里能激发出更丰富的偶数泛音,声音更"亮"。这就是物理 + 数学 + 音乐的一次完整闭环。

核对 \(t = 0\):级数

\[ \sum_{n=1, 3, 5, \ldots} \frac{8 h (-1)^{(n-1)/2}}{n^2 \pi^2} \sin(n\pi x/L) \]

应当等于三角形 \(f(x)\)。在 \(x = L/2\) 时 \(\sin(n\pi/2) = (-1)^{(n-1)/2}\)(\(n\) 奇数),所以每一项里的"两个 \((-1)^{(n-1)/2}\)"相乘等于 1,级数变成 \(\sum_{n \text{ 奇}} (8h)/(n^2 \pi^2) = (8h/\pi^2)(1 + 1/9 + 1/25 + \ldots)\)。括号内级数收敛到 \(\pi^2/8\),所以总和是 \(8h/\pi^2 \cdot \pi^2/8 = h\) ✓——正是中点的初始位移。整个解自洽。

最后留一个观察:解的级数是有限传播速度 \(c\) 的——尽管它写成无穷级数。怎么从 D'Alembert 视角理解这一点?把无界的 D'Alembert 公式

\[ u(x, t) = \frac{1}{2}[F(x - ct) + F(x + ct)] \]

套用到"周期延拓后的三角波 \(F\)"上,你会得到与级数完全等价的答案。两端固定的边界,在 D'Alembert 视角里就是把初始数据奇延拓到整个数轴并按 \(2L\) 周期化——边界反射就是周期延拓中跨越边界时位移翻号的结果。这两种视角(傅里叶级数 vs D'Alembert 周期延拓)给出同样的解,但解释机制不同:级数视角强调"模态分解",D'Alembert 视角强调"行波反射"。把同一物理用两套数学语言看,都是必要的。

到这里第 20 章结束。我们用分离变量解了固定端波动方程,展示了振动模态、基频、泛音的物理意义,引入了 D'Alembert 解描述行波传播,并在拨弦例子里把全部工具汇总演练一次。读到这里你应该能对任何"线性二阶 PDE + 简单几何 + 齐次 BC"自信地动手——这正是 PDE 入门的胜利标志。本书的 PDE 之旅在此告一段落;往后真实研究的方向有变系数 PDE、非线性 PDE、高维问题、数值方法、广义函数解,每一条都是独立的学科。但所有这些更高级的发展,根都扎在这三章里。
