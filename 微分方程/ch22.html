第 22 章 综合应用：从模型到解

到这一章，你手里的工具已经很齐整：

  • 一阶方程的所有解析技巧（可分离、线性、积分因子、Bernoulli 等）；
  • 二阶常系数齐次/非齐次方程，包括待定系数法和参数变易法；
  • Laplace 变换，用来对付有外驱、有间断、有 \(\delta\) 的方程；
  • 线性方程组的特征值/特征向量解法；
  • 非线性系统的相图、平衡点、线性化、Lyapunov 稳定性；
  • Fourier 级数和经典 PDE 三件套（热方程、波动方程、Laplace 方程）；
  • 第 21 章的数值方法。

这是一大堆工具。每一个工具我们学的时候都用一些"干净"的小例子。但真实问题不会贴标签告诉你"这是个 Bernoulli 方程"或者"这是个边值问题"——你必须自己识别。本章把所有这些工具放进一个工具箱，然后用 5 个真正的应用问题，从头到尾走一遍：建模、识别类型、选择方法、解出来、解释物理。

本章涉及：

  • 双弹簧-质量系统：一个机械模型如何变成线性方程组，又怎么用特征值法解出"模态"（22.1）。
  • 连锁化学反应 \(A \to B \to C\)：常系数线性方程组的优雅例子，以及它的长期行为（22.2）。
  • SIR 流行病模型：一个非线性系统，怎么找平衡点、做线性化、做数值模拟（22.3）。
  • 带外驱的 RLC 电路：Laplace 变换大显身手（22.4）。
  • 受热并震动的弦：两个 PDE 的耦合，简单的简化策略（22.5）。
  • "我该用哪种方法"：决策树式的总结（22.6）。

这一章不引入任何新工具，只是组合。读完之后，整本书就成了一个连贯的整体——这就是综合的价值。

22.1 双弹簧-质量系统

设两个相同的弹簧（劲度系数 \(k\)），把两个相同的质量块 \(m\) 串在一面墙和一根固定杆之间，如下：

墙— 弹簧 —[质量 \(m_1\)]— 弹簧 —[质量 \(m_2\)]— 弹簧 —墙

设 \(x_1(t), x_2(t)\) 分别是两块质量相对于平衡位置的位移。我们要找运动方程。

22.1.1 物理 → 方程组

对 \(m_1\)：左弹簧伸长 \(x_1\)，对它施加恢复力 \(-k x_1\)；中间弹簧的伸长是 \(x_2 - x_1\)（如果 \(m_2\) 比 \(m_1\) 走得更远，中间弹簧被拉长），所以中间弹簧对 \(m_1\) 施加力 \(+k(x_2 - x_1)\)（向右把 \(m_1\) 拉过来）。Newton 第二定律：

\[ m \ddot{x}_1 = -k x_1 + k(x_2 - x_1) = -2k x_1 + k x_2. \]

对 \(m_2\)：右弹簧压缩 \(x_2\)，恢复力 \(-k x_2\)；中间弹簧对 \(m_2\) 的力是 \(-k(x_2 - x_1) = k(x_1 - x_2)\)。所以

\[ m \ddot{x}_2 = k(x_1 - x_2) - k x_2 = k x_1 - 2 k x_2. \]

把它们整理成矩阵形式。设 \(\omega_0^2 = k/m\)：

\[ \ddot{\mathbf{x}} = -\omega_0^2 \, A \, \mathbf{x}, \qquad \mathbf{x} = \begin{pmatrix} x_1 \\ x_2 \end{pmatrix}, \qquad A = \begin{pmatrix} 2 & -1 \\ -1 & 2 \end{pmatrix}. \]

这是个二阶方程组。我们在第 13 章学过怎么处理：寻找形式 \(\mathbf{x}(t) = \mathbf{v} \cos(\omega t + \phi)\) 的解，代入后得到

\[ -\omega^2 \mathbf{v} = -\omega_0^2 A \mathbf{v} \quad\Longleftrightarrow\quad A \mathbf{v} = \frac{\omega^2}{\omega_0^2} \mathbf{v}. \]

所以 \(\omega^2 / \omega_0^2\) 必须是 \(A\) 的特征值，\(\mathbf{v}\) 必须是对应的特征向量。这正是"模态分析"。

22.1.2 求矩阵 \(A\) 的特征值

特征方程：\(\det(A - \lambda I) = (2 - \lambda)^2 - 1 = 0\)，即 \(\lambda^2 - 4\lambda + 3 = 0\)，所以 \(\lambda_1 = 1, \lambda_2 = 3\)。

对应频率：\(\omega_1 = \omega_0 \cdot 1 = \sqrt{k/m}\)，\(\omega_2 = \omega_0 \sqrt{3} = \sqrt{3 k/m}\)。

这就是两个简正频率——系统能做"纯振动"的两个频率。

22.1.3 求模态（特征向量）

对 \(\lambda_1 = 1\)：\((A - I)\mathbf{v} = \begin{pmatrix} 1 & -1 \\ -1 & 1 \end{pmatrix} \mathbf{v} = 0\)。任何 \(\mathbf{v}_1 \propto (1, 1)^T\)。

对 \(\lambda_2 = 3\)：\((A - 3I)\mathbf{v} = \begin{pmatrix} -1 & -1 \\ -1 & -1 \end{pmatrix} \mathbf{v} = 0\)。任何 \(\mathbf{v}_2 \propto (1, -1)^T\)。

22.1.4 物理解释模态

第一模态 \(\mathbf{v}_1 = (1, 1)^T\)：\(x_1\) 与 \(x_2\) 总是同向同步移动。两块质量像被绑在一起，中间弹簧从不变长——它的长度恒等于平衡长度，因此对动力学完全没有贡献。这两块质量只感受到外侧两弹簧，每块感受到劲度 \(k\)，所以频率正是 \(\sqrt{k/m} = \omega_0\)。这是"对称模态"。

第二模态 \(\mathbf{v}_2 = (1, -1)^T\)：\(x_1\) 与 \(x_2\) 总是反向移动。中间弹簧的伸缩变得最剧烈：当 \(x_1 = +a\) 时 \(x_2 = -a\)，中间弹簧的伸长是 \(2a\)。这个模态对应的"有效劲度"是外弹簧加上中间弹簧的两倍贡献，频率是 \(\sqrt{3} \omega_0\)，比第一模态更高。这是"反对称模态"。

通解：

\[ \mathbf{x}(t) = c_1 \begin{pmatrix} 1 \\ 1 \end{pmatrix} \cos(\omega_0 t + \phi_1) + c_2 \begin{pmatrix} 1 \\ -1 \end{pmatrix} \cos(\sqrt{3}\omega_0 t + \phi_2). \]

四个待定常数 \(c_1, c_2, \phi_1, \phi_2\) 由 4 个初始条件 \(x_1(0), x_2(0), \dot x_1(0), \dot x_2(0)\) 决定。

具体例子：\(x_1(0) = 1, x_2(0) = 0, \dot x_1(0) = \dot x_2(0) = 0\)（拉开 \(m_1\) 然后松手）。代入：\(c_1 \cos\phi_1 + c_2 \cos\phi_2 = 1\)，\(c_1 \cos\phi_1 - c_2 \cos\phi_2 = 0\)。所以 \(c_1 \cos\phi_1 = c_2 \cos\phi_2 = 1/2\)。速度初值 \(= 0\) 推出 \(\sin\phi_1 = \sin\phi_2 = 0\)，所以 \(\phi_1 = \phi_2 = 0\)，\(c_1 = c_2 = 1/2\)。

\[ x_1(t) = \tfrac{1}{2}\bigl[\cos(\omega_0 t) + \cos(\sqrt{3}\omega_0 t)\bigr], \qquad x_2(t) = \tfrac{1}{2}\bigl[\cos(\omega_0 t) - \cos(\sqrt{3}\omega_0 t)\bigr]. \]

这两个表达式有美丽的物理含义：因为两个频率比不是有理数，运动是准周期的，永不严格重复。如果你不动手画就把答案放出来，它看起来像神秘的"拍频"——能量在 \(m_1\) 和 \(m_2\) 之间慢慢来回摆动。这就是耦合振子的典型行为，模态分析把这种看似复杂的图景"对角化"成了两个独立的简谐振动。

22.2 连锁化学反应 A → B → C

设 \(A\) 以速率常数 \(k_1\) 转化成 \(B\)，\(B\) 又以速率常数 \(k_2\) 转化成 \(C\)：

\[ A \xrightarrow{k_1} B \xrightarrow{k_2} C. \]

设 \(a(t), b(t), c(t)\) 分别是三种物质的浓度，初值 \(a(0) = a_0, b(0) = c(0) = 0\)。

22.2.1 设方程组

反应速率方程：

\[ \dot a = -k_1 a, \qquad \dot b = k_1 a - k_2 b, \qquad \dot c = k_2 b. \]

这是一阶线性方程组。注意它是三角结构——\(a\) 不依赖 \(b, c\)；\(b\) 依赖 \(a\) 但不依赖 \(c\)；\(c\) 依赖 \(b\) 但不再反过来影响。

22.2.2 解析解（链式做）

第一个方程是单变量的，立即解出：\(a(t) = a_0 e^{-k_1 t}\)。

把它代入第二个方程：

\[ \dot b + k_2 b = k_1 a_0 e^{-k_1 t}. \]

这是一阶线性，积分因子 \(e^{k_2 t}\)：

\[ \frac{d}{dt}\bigl(b e^{k_2 t}\bigr) = k_1 a_0 e^{(k_2 - k_1) t}. \]

积分（假设 \(k_1 \ne k_2\)）：

\[ b e^{k_2 t} = \frac{k_1 a_0}{k_2 - k_1} e^{(k_2 - k_1) t} + C. \]

由 \(b(0) = 0\)，\(C = -k_1 a_0 / (k_2 - k_1)\)。

\[ b(t) = \frac{k_1 a_0}{k_2 - k_1}\bigl(e^{-k_1 t} - e^{-k_2 t}\bigr). \]

最后由质量守恒（\(a + b + c = a_0\)）：

\[ c(t) = a_0 - a(t) - b(t) = a_0\left(1 - e^{-k_1 t} - \frac{k_1}{k_2 - k_1}(e^{-k_1 t} - e^{-k_2 t})\right). \]

整理一下：

\[ c(t) = a_0 \left( 1 + \frac{k_1 e^{-k_2 t} - k_2 e^{-k_1 t}}{k_2 - k_1} \right). \]

22.2.3 用特征值法重做

把方程组写成 \(\dot{\mathbf{u}} = M \mathbf{u}\)，其中 \(\mathbf{u} = (a, b, c)^T\)，

\[ M = \begin{pmatrix} -k_1 & 0 & 0 \\ k_1 & -k_2 & 0 \\ 0 & k_2 & 0 \end{pmatrix}. \]

这是下三角矩阵，特征值就在对角线上：\(\lambda_1 = -k_1, \lambda_2 = -k_2, \lambda_3 = 0\)。零特征值对应的特征向量是 \((0, 0, 1)^T\)——也就是 \(c\) 守恒方向（与质量守恒一致：当所有 \(A, B\) 都变成 \(C\)，系统进入定态）。

这两种解法殊途同归，但特征值法把"长期行为"看得更清楚：长期 \(t \to \infty\)，所有非零特征值贡献的项都被指数衰减压死，只剩下零特征值的方向——即 \(\mathbf{u}(\infty) = (0, 0, a_0)^T\)。所有 \(A\) 都变成 \(C\)，符合常识。

22.2.4 长期行为分析

更细一点：

  • \(a(t)\) 单调下降到 0，时间尺度 \(1/k_1\)。
  • \(b(t)\) 先升后降，在 \(t^* = \ln(k_2/k_1)/(k_2 - k_1)\) 处达到峰值（对 \(b(t)\) 求导让它等于 0 算出）。这个峰值是有限的——\(B\) 是一个瞬态中间产物，被产生得比消耗得快的阶段才积累。
  • \(c(t)\) 单调上升到 \(a_0\)，时间尺度由较小的 \(\min(k_1, k_2)\) 决定（最慢一步是限速步骤）。

"限速步骤"是化学动力学的口头禅——长链反应的总体速率被最慢的那一步控制。这正是我们的特征值分析告诉我们的：\(c(t)\) 趋于 \(a_0\) 的速度是由 \(\max(e^{-k_1 t}, e^{-k_2 t})\) 决定的，也就是由较小的速率常数决定。

这一节展示的是线性方程组的整套机器：用解析方法（链式或特征值）写出公式，然后从特征值的符号和大小读出物理图景。这对所有线性常系数系统都通用——化学、电路、人口、放射性衰变链，全是同一个故事。

22.3 SIR 流行病模型

这是一个真正非线性的应用，把第 15 章的相图工具和第 21 章的数值方法都用上。

22.3.1 模型方程

在一个总人口为 \(N\) 的封闭群体里，每个人在任意时刻处于三种状态之一：易感（\(S\)）、感染（\(I\)）、康复或移除（\(R\)）。基本假设：

  • 易感者与感染者按比例 \(\beta\) 接触并被感染：每单位时间有 \(\beta S I / N\) 个新感染（双线性，"质量作用律"）。简化常用 \(\beta SI\)（吸收 \(N\) 进 \(\beta\)）。
  • 感染者按速率 \(\gamma\) 康复：每单位时间有 \(\gamma I\) 个康复。
\[ \dot S = -\beta S I, \qquad \dot I = \beta S I - \gamma I, \qquad \dot R = \gamma I. \]

注意 \(\dot S + \dot I + \dot R = 0\)，所以 \(S + I + R = N\) 常数（守恒）。我们可以只关心 \((S, I)\) 平面。

22.3.2 找平衡点

设 \(\dot S = \dot I = 0\)。\(\dot S = 0\) 给 \(I = 0\)；\(\dot I = I(\beta S - \gamma) = 0\) 也给 \(I = 0\)（或 \(S = \gamma/\beta\) 但只在 \(I = 0\) 时同时成立）。

所以平衡点是整条 \(S\) 轴，\(I = 0\)。这并不奇怪——只要没有当前的感染者，疾病就停留在那个状态。这种"非孤立的平衡集"是 SIR 模型的特点，也告诉你：传统的"局部线性化"在这里要小心。

22.3.3 关键无量纲量：基本再生数

把第二个方程改写为

\[ \dot I = I (\beta S - \gamma). \]

所以 \(I\) 在 \(S > \gamma/\beta\) 时增加，在 \(S < \gamma/\beta\) 时减少。临界量是 \(S^* = \gamma/\beta\)。

从 \(S = N\)（开始时所有人都易感）出发，\(I\) 会增长当且仅当 \(N > \gamma/\beta\)，等价于

\[ \boxed{R_0 := \frac{\beta N}{\gamma} > 1.} \]

这就是著名的基本再生数。它的物理意义：一个感染者在他整个感染期 \(1/\gamma\) 内，会接触 \(\beta N\) 单位时间的易感人群，因此平均产生 \(R_0 = \beta N / \gamma\) 个新感染者。\(R_0 > 1\) 意味着每个感染者制造出超过一个新感染者，疫情扩散；\(R_0 < 1\) 意味着每个感染者制造出不到一个，疫情自行熄灭。这就是为什么所有公共卫生措施都瞄准"把 \(R_0\) 压到 1 以下"——通过疫苗减少 \(N\) 中可被感染的人数，通过隔离/口罩减少 \(\beta\)，通过治疗加快 \(\gamma\)。

22.3.4 一条守恒律：相轨道

把 \(\dot I / \dot S = (\beta S - \gamma) I / (-\beta S I) = -1 + \gamma / (\beta S)\)，所以

\[ \frac{dI}{dS} = -1 + \frac{\gamma}{\beta S}. \]

积分：

\[ I(S) = -S + \frac{\gamma}{\beta} \ln S + C. \]

用初始条件 \((S_0, I_0)\) 定 \(C = I_0 + S_0 - (\gamma/\beta) \ln S_0\)。这是相平面里的轨道方程——告诉你 \(I\) 随 \(S\) 怎么变（消去了时间）。

22.3.5 长期行为

从这条轨道方程可以读出几件事：

  • 峰值发生在 \(\dot I = 0\) 也就是 \(S = \gamma/\beta\) 处：\(I_{\max} = N - \gamma/\beta - (\gamma/\beta)\ln(N \beta / \gamma) + (\gamma/\beta) \ln(N) - 1\)（如果 \(I_0 \approx 0\)）。这是"医院床位"决定的关键量。
  • \(t \to \infty\) 时 \(I \to 0\)（因为 \(I\) 总在减少或经过峰值后单调减少），但 \(S\) 不会到 0——总有一部分人从未被感染过！这部分人通过 \(I = -S + (\gamma/\beta) \ln S + C\) 在 \(I = 0\) 时给出超越方程
\[ S_\infty - \frac{\gamma}{\beta} \ln S_\infty = S_0 - \frac{\gamma}{\beta} \ln S_0 + I_0, \]

它没有闭式解，但可以用数值方法（牛顿迭代）解出 \(S_\infty\)。这个 \(S_\infty > 0\) 意味着群体免疫：即使没有疫苗，疫情也会自然终结，留下一些从未感染的人。

22.3.6 数值模拟

取具体参数：\(N = 1000\)，\(\beta = 0.0003\)，\(\gamma = 0.1\)（即 \(R_0 = 3\)），\(I_0 = 1\)，\(S_0 = 999\)。用 RK4 解，\(h = 0.1\)。Python 风格：

from scipy.integrate import solve_ivp

def sir(t, y, beta=0.0003, gamma=0.1):
    S, I, R = y
    return [-beta*S*I, beta*S*I - gamma*I, gamma*I]

sol = solve_ivp(sir, [0, 200], [999, 1, 0], dense_output=True)

跑出来：

  • \(t \approx 30\) 时感染数 \(I\) 达到峰值约 300（也就是有 30% 的人同时感染）；
  • \(t \approx 80\) 后 \(I\) 降到几乎为零；
  • \(S_\infty \approx 60\)，意味着大约 6% 的人从未感染。

这就是没有任何干预的"自然进程"。如果通过口罩等措施把 \(\beta\) 降低到原来的 1/3（\(R_0 = 1\)），峰值大幅压低，疫情持续时间拉长但可控；如果再低，疫情迅速消失。这就是"压平曲线"策略的数学基础。

这一节连接的工具：相图分析（求平衡点）、线性化（在 \(I = 0\) 平衡线附近研究 \(\dot I = (\beta S - \gamma) I\)，告诉我们 \(R_0\)）、相平面守恒律（消去时间得到 \(I(S)\) 公式）、超越方程数值解（求 \(S_\infty\)）、数值积分（看动力学过程）。这五样合在一起，整个 SIR 故事就讲清楚了。

22.4 RLC 电路含周期外驱

串联 RLC 电路：电阻 \(R\)、电感 \(L\)、电容 \(C\)，外加一个电压源 \(V(t)\)。设回路电流是 \(i(t)\)，电容上的电荷是 \(q(t)\)，则 \(i = \dot q\)。基尔霍夫电压定律给出：

\[ L \ddot q + R \dot q + \frac{1}{C} q = V(t). \]

这是常系数二阶非齐次方程。当 \(V(t)\) 是分段或脉冲信号时，Laplace 变换是首选工具。

22.4.1 例题：方波驱动

取 \(L = 1, R = 2, C = 1\)。\(V(t) = 1\) 当 \(0 \le t < 1\)，\(V(t) = 0\) 当 \(t \ge 1\)。初始 \(q(0) = 0, \dot q(0) = 0\)。

用单位阶跃函数：\(V(t) = u(t) - u(t - 1)\)。Laplace 变换 \(\mathcal{L}\{V\} = \frac{1}{s} - \frac{e^{-s}}{s} = \frac{1 - e^{-s}}{s}\)。

对方程做 Laplace（用零初值）：

\[ (s^2 + 2s + 1) Q(s) = \frac{1 - e^{-s}}{s}, \]

所以

\[ Q(s) = \frac{1 - e^{-s}}{s(s + 1)^2}. \]

分两块：\(Q(s) = F(s) - e^{-s} F(s)\) 其中 \(F(s) = \frac{1}{s(s+1)^2}\)。

分式分解 \(F\)：

\[ \frac{1}{s(s + 1)^2} = \frac{A}{s} + \frac{B}{s + 1} + \frac{D}{(s + 1)^2}. \]

通分凑：\(1 = A(s + 1)^2 + B s (s + 1) + D s\)。代 \(s = 0\)：\(A = 1\)。代 \(s = -1\)：\(D \cdot (-1) = 1\)，所以 \(D = -1\)。比较 \(s^2\) 系数：\(A + B = 0\)，\(B = -1\)。

所以 \(F(s) = \frac{1}{s} - \frac{1}{s+1} - \frac{1}{(s+1)^2}\)，反变换 \(f(t) = 1 - e^{-t} - t e^{-t}\)。

由位移定理 \(\mathcal{L}^{-1}\{e^{-s} F(s)\} = u(t-1) f(t-1)\)，最终

\[ q(t) = f(t) - u(t-1) f(t-1) = \bigl(1 - e^{-t} - t e^{-t}\bigr) - u(t-1)\bigl(1 - e^{-(t-1)} - (t-1) e^{-(t-1)}\bigr). \]

读法：在 \(0 \le t < 1\) 阶段，电压是 1，电荷按 \(1 - e^{-t} - t e^{-t}\) 增长，趋向于 1（充电）；在 \(t \ge 1\) 阶段，电压撤去，剩下的"被减号那块"开始抵消，电荷开始放电指数衰减。

这种含开关、脉冲的电路问题，几乎只能用 Laplace 优雅地处理。如果用经典 Frobenius 或参数变易，需要在 \(t = 1\) 处分段拼接、匹配连续性，工作量大得多，还容易出错。

22.4.2 共振

另一个经典：\(V(t) = V_0 \cos(\omega t)\)。这种"纯正弦驱动"的稳态解在 12.5 节做过。结论：稳态电流的振幅是 \(V_0 / |Z(\omega)|\)，其中

\[ |Z(\omega)|^2 = R^2 + \left(\omega L - \frac{1}{\omega C}\right)^2. \]

当 \(\omega L = 1/(\omega C)\) 即 \(\omega = 1/\sqrt{LC}\) 时阻抗最小，电流最大——共振频率。这也是收音机选台的物理原理：调电容 \(C\) 让共振频率对准你想收的电台。

22.5 一根受热并震动的弦

设一根固定在两端、长度为 \(L\) 的弦，既会沿横向振动，又会沿纵向因受热而变形。在最简化的耦合模型里，温度 \(T(x, t)\) 满足热方程，横向位移 \(u(x, t)\) 满足波动方程，但它们通过弦的张力或杨氏模量耦合：温度高的地方，弹性减小，因此波速 \(c\) 依赖于 \(T\)。

\[ \partial_t T = \alpha \partial_x^2 T, \] \[ \partial_t^2 u = c^2(T) \, \partial_x^2 u, \qquad c^2(T) = c_0^2 (1 - \kappa T). \]

这是耦合 PDE 系统。一般来说没有闭式解。但通过分离时间尺度可以做巨大化简：热扩散通常比横振动慢得多，所以把 \(T\) 当成准静态背景——先解热方程到达准稳态 \(T^*(x)\)，再把 \(c^2(T^*(x))\) 代入波动方程做局部模态分析。

这是工程问题里非常常用的策略——多尺度分析。两个 PDE 时间尺度差大时，就先把慢的那个解到准稳态，把它当成快的那个的"参数"。否则你就要面对真正的耦合 PDE，需要数值方法（差分、有限元）。

本节我们不展开数值，只指出：第 17-19 章的 Fourier 工具仍然能用——只要在分离变量后承认 \(c^2\) 随 \(x\) 变，Sturm-Liouville 理论保证仍有正交本征模，只是本征函数不再是简单的 \(\sin(n\pi x / L)\)，而要解一个变系数本征值问题。这又是一道"组合工具"的题：分离变量 + Sturm-Liouville + 数值（如有限差分求本征函数）。
