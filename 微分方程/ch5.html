第 5 章 二阶线性齐次方程(常系数)

到这里你对一阶方程已经相当熟练。无论它是可分离的、线性的、还是恰当的,你都有一套方法去解。也学会了不解方程时怎样定性分析。但一阶方程能描写的物理过程是有限的——它们大体上是"松弛型"的:温度趋近室温、电容趋近电源电压、人口趋近承载量,都是单调地走向某个平衡。

但物理世界的另一半是振荡。一根弹簧上挂的小球,被拉一下放手,它会反复来回摆动,不会单调趋近原点。一根钟摆,在小角度下也是这样的振动。一个 LC 电路里的电流会周期性地来回流动。一个分子里的两个原子在它们的平衡距离附近振颤。这些行为都不能用一阶方程描写——一阶自治方程的解必单调,无法振荡。要描写振荡,必须升级。

升级的方向是二阶方程,即出现 \(y''\) 的方程。原因是物理上,加速度(\(y''\))由力决定,而很多物理力(比如弹簧的弹性恢复力)依赖于位移,这就让 \(y''\) 自然出现。本章我们专攻最简单也最重要的一类二阶方程:

\[ ay'' + by' + cy = 0, \]

其中 \(a, b, c\) 是常数,\(a \neq 0\)。"齐次"指右端为零;"常系数"指 \(a, b, c\) 不依赖于 \(x\)。这两个限制让这一类方程拥有一套近乎完美的解法——你将看到,解只用一个二次方程的根就能完全表达。这一章我们将做以下几件事:

  • 把"猜指数解"这件事说清楚——为什么试 \(y = e^{rx}\),为什么这把所有可能性都罩住了;
  • 把得到的特征方程 \(ar^2 + br + c = 0\) 的三种根的情形(两个不同实根、重根、共轭复根)分别处理,得到对应的通解形式;
  • 解释一个非常重要的概念:解空间是二维的——任何解都可以唯一写成两个"基础解"的线性组合;
  • 引入朗斯基行列式(Wronskian)作为判别两个解线性无关的工具;
  • 说清楚"二阶方程的初值问题需要两个初值"——\(y(x_0)\) 和 \(y'(x_0)\),它们一起唯一决定一个解;
  • 每种情形配两道详细例题,把整个流程跑熟。

这一章是后面一切高阶线性方程的母体——非齐次方程(第 7 章)、变系数方程(第 8 章)、振动应用(第 9 章)、Laplace 变换(第 11 章),都建立在你这里掌握的"特征方程一二三种根"的基础之上。学好这一章,后面就轻松一半。

5.1 形式 \(ay'' + by' + cy = 0\)

先把术语固定下来。形如

\[ \boxed{\,a y'' + b y' + c y = 0,\quad a \neq 0\,} \]

的方程叫做二阶常系数线性齐次方程。把每个词解读一下:

  • 二阶:出现的最高导数是二阶 \(y''\)。
  • 常系数:\(a, b, c\) 是常数,不依赖于 \(x\)。这是本章的限制;变系数(\(a(x), b(x), c(x)\))将在第 8 章处理。
  • 线性:方程对 \(y, y', y''\) 都是线性的——没有 \(y^2\)、\(yy'\)、\(\sin y\) 这种非线性项。
  • 齐次:右端为零。如果右端是某个 \(g(x) \neq 0\),就叫"非齐次"——那是第 7 章的事。

常见的物理例子。无阻尼弹簧振子:\(my'' + ky = 0\)(质量 \(m\),弹性系数 \(k\),无摩擦)。有阻尼弹簧振子:\(my'' + cy' + ky = 0\)(\(c\) 是阻尼系数)。LC 电路:\(L q'' + q/C = 0\)(电感 \(L\)、电容 \(C\)、电荷 \(q\))。RLC 电路:\(L q'' + R q' + q/C = 0\)。这些方程都可以写成 \(ay'' + by' + cy = 0\) 的形式;它们的解就是各种振荡(无阻尼)或衰减振荡(有阻尼)。

线性齐次方程有一条核心性质,叫叠加原理:如果 \(y_1\) 和 \(y_2\) 都是方程的解,那么对任何常数 \(c_1, c_2\),\(c_1 y_1 + c_2 y_2\) 也是方程的解。证明很短:把它代回去,

\[ a(c_1 y_1 + c_2 y_2)'' + b(c_1 y_1 + c_2 y_2)' + c(c_1 y_1 + c_2 y_2) \] \[ = c_1(a y_1'' + b y_1' + c y_1) + c_2(a y_2'' + b y_2' + c y_2) = c_1 \cdot 0 + c_2 \cdot 0 = 0. \quad\checkmark \]

叠加原理是线性方程的根本特征——它意味着解的集合是一个向量空间(如果你学过线性代数,这一刻应该响起警报)。后面 5.4 节会专门展开这个对应。

现在的问题变成:怎么找到这些解?

5.2 寻找指数解 → 特征方程

一阶常系数方程 \(y' + ky = 0\) 的解是 \(y = e^{-kx}\)——指数函数。这给了我们一个启发:对二阶方程,试一试 \(y = e^{rx}\) 看看?

这一招的高明之处是:指数函数求一次导还是指数函数(乘上一个常数),求两次导还是指数函数(乘上一个常数的平方)。这意味着把它代入任何常系数线性方程,都会得到同样形状的指数函数,只剩下一个关于 \(r\) 的代数方程。我们把这件事做出来。

设 \(y = e^{rx}\)。则 \(y' = re^{rx}\),\(y'' = r^2 e^{rx}\)。代入 \(ay'' + by' + cy = 0\):

\[ a r^2 e^{rx} + b r e^{rx} + c e^{rx} = 0, \quad e^{rx}(ar^2 + br + c) = 0. \]

因为 \(e^{rx}\) 永远不等于零(对任何实 \(r\) 和 \(x\)),括号里必须为零:

\[ \boxed{\,ar^2 + br + c = 0.\,} \]

这就是特征方程(characteristic equation,也叫辅助方程 auxiliary equation)。它把一个微分方程问题化成了一个二次方程问题——而二次方程是我们高中就熟悉的。

怎么从微分方程"读出"特征方程?有一个机械化的对应:

\[ y'' \leftrightarrow r^2, \quad y' \leftrightarrow r, \quad y \leftrightarrow 1. \]

把方程里的每个 \(y, y', y''\) 都替换成对应的 \(r\) 的幂,就得到特征方程。例如:

  • \(y'' - 5y' + 6y = 0\) 的特征方程是 \(r^2 - 5r + 6 = 0\);
  • \(y'' + 4y = 0\) 的特征方程是 \(r^2 + 4 = 0\);
  • \(2y'' + 3y' - y = 0\) 的特征方程是 \(2r^2 + 3r - 1 = 0\)。

这种"读出"完全是机械的,你做过几次就会反射性地写。

下一步是解这个二次方程。用万能的二次公式:

\[ r = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}. \]

判别式 \(\Delta = b^2 - 4ac\) 决定了根的三种情形:

  1. \(\Delta > 0\):两个不同实根 \(r_1 \neq r_2\)。
  2. \(\Delta = 0\):一个二重实根 \(r_1 = r_2 = -b/(2a)\)。
  3. \(\Delta < 0\):一对共轭复根 \(r = \alpha \pm \beta i\),其中 \(\alpha = -b/(2a)\),\(\beta = \sqrt{-\Delta}/(2a)\)。

这三种情形对应三种不同的通解形式。下面分别处理。

5.3 三种情形

5.3.1 情形一:两个不同实根

设特征方程的两根是 \(r_1 \neq r_2\),都是实数。那么我们手上有两个解:\(y_1 = e^{r_1 x}\) 和 \(y_2 = e^{r_2 x}\)。叠加原理告诉我们,

\[ \boxed{\,y(x) = c_1 e^{r_1 x} + c_2 e^{r_2 x}\,} \]

对任何常数 \(c_1, c_2\) 都是解。这就是这种情形下的通解(general solution)。"通解"两个字意思是说:任何解都可以写成这种形式;反过来,这个公式取遍所有 \(c_1, c_2\) 时也只产生方程的解,不多也不少。我们 5.4 节会论证这一点。

例 1. 解 \(y'' - 5y' + 6y = 0\)。

特征方程 \(r^2 - 5r + 6 = 0\),即 \((r-2)(r-3) = 0\),根 \(r_1 = 2, r_2 = 3\)。所以通解是

\[ y(x) = c_1 e^{2x} + c_2 e^{3x}. \]

验证:\(y' = 2c_1 e^{2x} + 3c_2 e^{3x}\),\(y'' = 4c_1 e^{2x} + 9c_2 e^{3x}\)。代入左端:\((4 - 10 + 6)c_1 e^{2x} + (9 - 15 + 6)c_2 e^{3x} = 0 + 0 = 0\)。\(\checkmark\)

例 2. 解 \(y'' + 3y' - 4y = 0\),初值 \(y(0) = 2\),\(y'(0) = 1\)。

特征方程 \(r^2 + 3r - 4 = (r+4)(r-1) = 0\),根 \(r_1 = -4, r_2 = 1\)。通解 \(y = c_1 e^{-4x} + c_2 e^x\)。

用初值定 \(c_1, c_2\)。\(y(0) = c_1 + c_2 = 2\)。\(y'(0) = -4c_1 + c_2 = 1\)。两式相减:\(-5c_1 = -1\),\(c_1 = 1/5\),\(c_2 = 9/5\)。所以

\[ y(x) = \tfrac{1}{5} e^{-4x} + \tfrac{9}{5} e^x. \]

验证 \(y(0) = 1/5 + 9/5 = 2\) \(\checkmark\),\(y'(0) = -4/5 + 9/5 = 1\) \(\checkmark\)。

5.3.2 情形二:重根 \(r_1 = r_2 = r\)

设特征方程 \(ar^2 + br + c = 0\) 有一个二重根 \(r\)。那么 \(y_1 = e^{rx}\) 是一个解,但只有一个。我们却需要两个线性无关的解才能给出二阶方程的通解(理由 5.4 节展开)。怎么办?

能不能再用一个 \(y_2 = e^{rx}\)?不行——这就是 \(y_1\) 本身,不是新的。\(c_1 e^{rx} + c_2 e^{rx} = (c_1 + c_2) e^{rx}\) 只有一个独立常数,不能满足两个独立的初值条件。我们需要另一种形状的解。

答案是:第二个解是 \(y_2 = x e^{rx}\)。整体通解是

\[ \boxed{\,y(x) = (c_1 + c_2 x) e^{rx}.\,} \]

为什么是 \(xe^{rx}\)?有几种解释路径,我们走两条最直观的。

解释一:取极限。设特征方程的两根是 \(r\) 和 \(r + \varepsilon\),\(\varepsilon\) 很小。那么 \(e^{rx}\) 和 \(e^{(r+\varepsilon)x}\) 都是解,\(\frac{e^{(r+\varepsilon)x} - e^{rx}}{\varepsilon}\) 也是解(它是两个解的线性组合,系数 \(1/\varepsilon\) 和 \(-1/\varepsilon\)),只要 \(\varepsilon \neq 0\)。让 \(\varepsilon \to 0\):

\[ \frac{e^{(r+\varepsilon)x} - e^{rx}}{\varepsilon} = e^{rx} \cdot \frac{e^{\varepsilon x} - 1}{\varepsilon} \to e^{rx} \cdot x = xe^{rx}. \]

所以 \(xe^{rx}\) 自然地从两个不同实根的极限产生——它是"重根"作为"两根并合"时不丢失的"第二维度"。

解释二:直接代入验证。 我们可以直接验证 \(y_2 = xe^{rx}\) 是解,前提是 \(r\) 是重根(等价于 \(2ar + b = 0\),即 \(r = -b/(2a)\))。算导数:

\[ y_2 = xe^{rx}, \quad y_2' = e^{rx} + rxe^{rx} = (1 + rx)e^{rx}, \] \[ y_2'' = re^{rx} + r(1 + rx)e^{rx} = (2r + r^2 x)e^{rx}. \]

代入左端 \(ay'' + by' + cy\):

\[ e^{rx}\left[a(2r + r^2 x) + b(1 + rx) + cx\right] = e^{rx}\left[(2ar + b) + (ar^2 + br + c)x\right]. \]

\(ar^2 + br + c = 0\) 因为 \(r\) 是特征方程的根;\(2ar + b = 0\) 因为 \(r\) 是重根(这等价于 \(r\) 是 \(\Delta = b^2 - 4ac\) 的零的特征,即 \(r = -b/(2a)\)),所以两个括号都为零,结果是 \(0\)。\(\checkmark\)

例 3. 解 \(y'' - 6y' + 9y = 0\)。

特征方程 \(r^2 - 6r + 9 = (r-3)^2 = 0\),重根 \(r = 3\)。通解

\[ y(x) = (c_1 + c_2 x) e^{3x}. \]

例 4. 解 \(4y'' + 4y' + y = 0\),初值 \(y(0) = 1, y'(0) = 0\)。

特征方程 \(4r^2 + 4r + 1 = (2r+1)^2 = 0\),重根 \(r = -1/2\)。通解

\[ y(x) = (c_1 + c_2 x) e^{-x/2}. \]

初值。\(y(0) = c_1 = 1\)。算 \(y'\):

\[ y' = c_2 e^{-x/2} + (c_1 + c_2 x)\cdot (-1/2) e^{-x/2} = \left[c_2 - \tfrac{1}{2}(c_1 + c_2 x)\right] e^{-x/2}. \]

\(y'(0) = c_2 - c_1/2 = c_2 - 1/2 = 0\),所以 \(c_2 = 1/2\)。最终

\[ y(x) = \left(1 + \tfrac{1}{2} x\right) e^{-x/2}. \]

5.3.3 情形三:共轭复根 \(\alpha \pm \beta i\)

设判别式 \(\Delta = b^2 - 4ac < 0\)。这时特征方程的根是一对共轭复数:

\[ r = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} = \frac{-b}{2a} \pm \frac{\sqrt{4ac - b^2}}{2a} i = \alpha \pm \beta i, \]

其中 \(\alpha = -b/(2a)\)(实部),\(\beta = \sqrt{4ac - b^2}/(2a) > 0\)(虚部)。

形式上,情形一的公式照样能写:

\[ \boxed{\,y(x) = c_1 e^{(\alpha + \beta i)x} + c_2 e^{(\alpha - \beta i)x}.\,} \]

这形式上是解。但它有一个问题:它涉及复指数,系数 \(c_1, c_2\) 也要选成复数才能让 \(y\) 取实值。这对实际计算和物理解释都不方便。我们更想要一个完全实数的通解形式。

怎么把复指数翻译成正弦余弦?用 Euler 公式:\(e^{i\theta} = \cos\theta + i\sin\theta\)。所以 \(e^{(\alpha + \beta i)x} = e^{\alpha x}(\cos\beta x + i\sin\beta x)\),\(e^{(\alpha - \beta i)x} = e^{\alpha x}(\cos\beta x - i\sin\beta x)\)。两个复解都是 \(e^{\alpha x}\) 乘以一个三角函数线性组合。这两个解的实数化版本是 \(e^{\alpha x}\cos\beta x\) 和 \(e^{\alpha x}\sin\beta x\),从而通解写成

\[ y(x) = e^{\alpha x}\left(c_1 \cos\beta x + c_2 \sin\beta x\right). \]

下一章我们将正式做这个"实数化"的工作——把复指数解严格地转换成实数三角函数,并解释为什么这么做合法。这一章为了不抢戏,我们仍把通解先写成复指数形式 \(c_1 e^{(\alpha + \beta i)x} + c_2 e^{(\alpha - \beta i)x}\)(或者你愿意的话,直接写成 \(e^{\alpha x}(A\cos\beta x + B\sin\beta x)\),两种形式我们都允许使用,在 6.1 节会建立两者的等价性)。

例 5. 写出 \(y'' + 4y = 0\) 的特征方程的根并形式上写下通解。

特征方程 \(r^2 + 4 = 0\),根 \(r = \pm 2i\)。所以 \(\alpha = 0, \beta = 2\),形式上的通解是

\[ y(x) = c_1 e^{2ix} + c_2 e^{-2ix}, \quad \text{或等价地} \quad y(x) = A\cos 2x + B\sin 2x. \]

第二种写法是上面的实数化结果。物理上,这个方程描述无阻尼振子(把 \(y\) 想成位移),解是周期 \(\pi\) 的纯振荡。

例 6. 写出 \(y'' - 2y' + 5y = 0\) 的特征方程的根并形式上写下通解。

特征方程 \(r^2 - 2r + 5 = 0\),用公式 \(r = (2 \pm \sqrt{4 - 20})/2 = 1 \pm 2i\)。所以 \(\alpha = 1, \beta = 2\),通解形式上是

\[ y(x) = c_1 e^{(1 + 2i)x} + c_2 e^{(1 - 2i)x}, \quad \text{或} \quad y(x) = e^x(A\cos 2x + B\sin 2x). \]

这个方程对应一个有阻尼但欠阻尼的振子(因为 \(\alpha = 1 > 0\),实际上是负阻尼,即增幅),解是包络指数增长的振荡。

5.4 解空间是二维的

这一节解释:为什么在每一种情形里我们都给出"通解 = \(c_1 \times\)(一个解)\(+ c_2 \times\)(另一个解)"的形式,涉及恰好两个独立常数? 为什么不是一个、不是三个、不是五个?

答案的核心是:二阶常系数线性齐次方程的解集是一个二维向量空间。我们一步步解释这句话。

首先,叠加原理(5.1 节)告诉我们,解的集合 \(V = \{y : ay'' + by' + cy = 0\}\) 在加法和数乘下封闭——\(y_1, y_2 \in V \Rightarrow c_1 y_1 + c_2 y_2 \in V\)。这正是向量空间的定义。如果你学过线性代数,这一刻应该回想起第 8 章里"基"和"维数"的概念。

其次,这个向量空间的维数是 2。证明的关键是存在唯一性定理(我们这里只陈述,严格证明在第 8 章):对于二阶线性方程 \(y'' + p(x)y' + q(x)y = 0\),给定任何初值 \(y(x_0) = a, y'(x_0) = b\),存在唯一的解。这个事实告诉我们:解被两个数 \((y(x_0), y'(x_0))\) 唯一决定,所以解空间和 \(\mathbb{R}^2\) 是一一对应的。\(\mathbb{R}^2\) 是二维的,所以解空间也是二维的。

"维数为 2"的意思是:可以找到两个解 \(y_1, y_2\),使得每一个解都唯一地写成 \(c_1 y_1 + c_2 y_2\) 的形式。这两个解叫做解空间的基(basis)——本章我们也叫它们"基础解组"或"基本解组"(fundamental set of solutions)。

但不是任意两个解都构成基。比如 \(y_1 = e^{2x}\) 和 \(y_2 = 3e^{2x}\) 都是 \(y'' - 4y' + 4y = (y'' - 4y' + 4y)\)... 等等,\(y'' - 4y = 0\) 的解,但你用它们的线性组合 \(c_1 y_1 + c_2 y_2 = (c_1 + 3c_2) e^{2x}\) 只能得到一个独立常数,不能覆盖所有的解。问题出在 \(y_2 = 3 y_1\)——它们是线性相关的。

正式地:两个函数 \(y_1, y_2\) 在区间 \(I\) 上称为线性相关,如果存在不全为零的常数 \(k_1, k_2\) 使得 \(k_1 y_1(x) + k_2 y_2(x) = 0\) 对所有 \(x \in I\) 成立——等价地,一个是另一个的常数倍(假设两者都不恒为零)。如果这样的 \(k_1, k_2\) 不存在(只有 \(k_1 = k_2 = 0\) 才能使等式恒为零),它们就线性无关。

所以"两个解构成基"等价于"它们线性无关"。这把"找通解"这件事归结为"找两个线性无关的解"。下面验证我们 5.3 节得到的三种情形里的两个解确实线性无关。

  • 情形一:\(y_1 = e^{r_1 x}, y_2 = e^{r_2 x}\),\(r_1 \neq r_2\)。比值 \(y_2/y_1 = e^{(r_2 - r_1)x}\) 不是常数(因 \(r_2 - r_1 \neq 0\))。所以线性无关。
  • 情形二:\(y_1 = e^{rx}, y_2 = xe^{rx}\)。比值 \(y_2/y_1 = x\),不是常数。所以线性无关。
  • 情形三:复数情形里 \(y_1 = e^{(\alpha + \beta i)x}, y_2 = e^{(\alpha - \beta i)x}\)。比值 \(y_2/y_1 = e^{-2\beta i x}\),不是常数(因 \(\beta \neq 0\))。所以线性无关。

三种情形下,我们给出的两个解都构成解空间的基。所以"\(c_1 y_1 + c_2 y_2\)"确实是所有解的形式——这就是为什么我们叫它"通解"。

这个"维数 = 2"的事实有一个非常实用的推论:二阶方程的初值问题需要恰好两个初值(\(y(x_0), y'(x_0)\))来唯一确定解。一阶方程只需一个初值,因为解空间是一维的;二阶需要两个,因为解空间是二维的。你应该能预测到:\(n\) 阶常系数线性齐次方程的解空间是 \(n\) 维的,需要 \(n\) 个初值。这个规律将在第 8 章正式扩展。

5.5 朗斯基行列式

"两个函数是否线性无关"是一个理论问题——上一节我们靠"看比值是否常数"来判断,这对简单函数还行,但对一般情况怎么办?有一个机械化的判别工具:朗斯基行列式(Wronskian)。

给定两个可微函数 \(y_1(x), y_2(x)\),它们的朗斯基行列式是

\[ \boxed{\,W(y_1, y_2)(x) = \begin{vmatrix} y_1(x) & y_2(x) \\ y_1'(x) & y_2'(x) \end{vmatrix} = y_1(x) y_2'(x) - y_2(x) y_1'(x).\,} \]

它是一个关于 \(x\) 的函数。判别准则是这样的:

定理(线性无关判别)。设 \(y_1, y_2\) 是某个二阶线性齐次方程在区间 \(I\) 上的解。则:

  • 如果 \(W(y_1, y_2)(x_0) \neq 0\) 在某点 \(x_0 \in I\),则 \(y_1, y_2\) 在 \(I\) 上线性无关(从而构成解空间的基)。
  • 如果 \(W(y_1, y_2)(x) = 0\) 对所有 \(x \in I\),则 \(y_1, y_2\) 线性相关。

(注意"两个函数都是某个方程的解"这个前提很重要;对一般的两个函数,Wronskian 可以恒为零而函数本身却线性无关——这个反例和 Abel 定理有关,我们这里不展开。)

简言之:"\(W \neq 0\)"是线性无关的充分条件,而对方程的解组,它也是必要条件。所以你只需算 Wronskian 在某一个方便的点(通常是 \(x = 0\))上的值,看是否非零,就能确认。

来用 Wronskian 复查我们三种情形里的解组。

情形一:\(y_1 = e^{r_1 x}, y_2 = e^{r_2 x}\)。

\[ W = e^{r_1 x} \cdot r_2 e^{r_2 x} - e^{r_2 x} \cdot r_1 e^{r_1 x} = (r_2 - r_1) e^{(r_1 + r_2)x}. \]

因 \(r_1 \neq r_2\),\(W \neq 0\) 处处。线性无关。\(\checkmark\)

情形二:\(y_1 = e^{rx}, y_2 = xe^{rx}\)。\(y_2' = (1 + rx)e^{rx}\)。

\[ W = e^{rx} (1 + rx) e^{rx} - xe^{rx} \cdot re^{rx} = e^{2rx}(1 + rx - rx) = e^{2rx}. \]

处处不为零。线性无关。\(\checkmark\)

情形三(复指数版本):\(y_1 = e^{(\alpha + \beta i)x}, y_2 = e^{(\alpha - \beta i)x}\)。

\[ W = e^{(\alpha + \beta i)x}(\alpha - \beta i)e^{(\alpha - \beta i)x} - e^{(\alpha - \beta i)x}(\alpha + \beta i)e^{(\alpha + \beta i)x} \] \[ = e^{2\alpha x}\left[(\alpha - \beta i) - (\alpha + \beta i)\right] = -2\beta i \cdot e^{2\alpha x}. \]

因 \(\beta \neq 0\),\(W \neq 0\)。线性无关(在复数意义下)。\(\checkmark\)

对实数化的版本 \(y_1 = e^{\alpha x}\cos\beta x, y_2 = e^{\alpha x}\sin\beta x\),你可以自己算一遍,Wronskian 是 \(\beta e^{2\alpha x}\),也不为零。

例 7. 验证 \(y_1 = e^{2x}\) 和 \(y_2 = e^{-3x}\) 在 \(\mathbb{R}\) 上线性无关。

\[ W = e^{2x}(-3 e^{-3x}) - e^{-3x}(2 e^{2x}) = -3e^{-x} - 2e^{-x} = -5 e^{-x} \neq 0. \checkmark \]

例 8. \(y_1 = \sin x\) 和 \(y_2 = \cos x\) 是 \(y'' + y = 0\) 的两个解。它们线性无关吗?

\[ W = \sin x \cdot (-\sin x) - \cos x \cdot \cos x = -\sin^2 x - \cos^2 x = -1 \neq 0. \checkmark \]

所以 \(y'' + y = 0\) 的通解可以写成 \(y = c_1 \sin x + c_2 \cos x\)(等同于 5.3.3 节的实数化形式,这里 \(\alpha = 0, \beta = 1\))。

5.6 初值问题

5.4 节告诉我们二阶方程的解空间是二维的。所以要从通解 \(c_1 y_1 + c_2 y_2\) 中"挑出"一个具体的解,需要两个条件来定 \(c_1, c_2\)。最常见的两个条件就是初值条件:

\[ y(x_0) = a, \quad y'(x_0) = b. \]

"初值"两字提示这通常是从某一时刻 \(x_0\) 开始的演化问题——\(y(x_0)\) 是初始位置,\(y'(x_0)\) 是初始速度。物理上这两件事独立给定:你可以把弹簧拉到某个位置(初始位移)然后给它任意一个初始速度,再放手,这两件事完全可以分开设定。

把通解代入两个初值条件,得到关于 \(c_1, c_2\) 的一个 \(2 \times 2\) 线性方程组:

\[ \begin{cases} c_1 y_1(x_0) + c_2 y_2(x_0) = a, \\ c_1 y_1'(x_0) + c_2 y_2'(x_0) = b. \end{cases} \]

这个方程组的系数行列式正是 \(W(y_1, y_2)(x_0)\)——朗斯基行列式!如果 Wronskian 非零,方程组有唯一解,初值问题有唯一解。如果 Wronskian 为零(基础解组没选好),方程组无解或有无穷解,这告诉我们 \(y_1, y_2\) 没构成基。所以 Wronskian 不仅是线性无关的判别工具,它的非零也是"用通解满足任意初值"这件事能办成的保证。

我们 5.3 节其实已经做过这种初值题(例 2 和例 4),只是那时还没用 Wronskian 的语言。下面把流程系统化:

解二阶常系数齐次初值问题的流程

  1. 写出特征方程 \(ar^2 + br + c = 0\),求根。
  2. 判别式分情况,写下通解 \(y = c_1 y_1 + c_2 y_2\)(三种情形对应三种形式)。
  3. 计算 \(y'\)(对 \(c_1, c_2\) 仍然是常数)。
  4. 代入两个初值,解出 \(c_1, c_2\)。
  5. 把 \(c_1, c_2\) 代回通解,得到具体解。
  6. (可选但推荐)代回检验初值。

这套流程对所有三种情形都适用——情形一和情形二我们已经做过例子,下面再补一道情形三的。

例 9. 解 \(y'' + 9y = 0\),\(y(0) = 3, y'(0) = -6\)。

第一步。特征方程 \(r^2 + 9 = 0\),根 \(r = \pm 3i\)。\(\alpha = 0, \beta = 3\)。

第二步。通解(实数形式)是 \(y = c_1 \cos 3x + c_2 \sin 3x\)。

第三步。\(y' = -3c_1 \sin 3x + 3c_2 \cos 3x\)。

第四步。代入初值:\(y(0) = c_1 = 3\),\(y'(0) = 3c_2 = -6\),所以 \(c_2 = -2\)。

第五步。具体解:\(y(x) = 3\cos 3x - 2\sin 3x\)。

第六步。检验:\(y(0) = 3 \cdot 1 - 2 \cdot 0 = 3\) \(\checkmark\);\(y'(0) = -9\sin 0 - 6\cos 0 = 0 - 6 = -6\) \(\checkmark\)。

5.7 详细例题汇总

每种情形再各做两道,把流程跑熟。

情形一:两个不同实根

例题 A. 解 \(y'' - y' - 6y = 0\)。

特征方程 \(r^2 - r - 6 = 0\)。判别式 \(\Delta = 1 + 24 = 25 > 0\)。根 \(r = (1 \pm 5)/2\),即 \(r_1 = 3, r_2 = -2\)。通解

\[ y(x) = c_1 e^{3x} + c_2 e^{-2x}. \]

验证一项:\(y_1 = e^{3x}\),\(y_1' = 3e^{3x}\),\(y_1'' = 9e^{3x}\),代入 \(9 - 3 - 6 = 0\) \(\checkmark\)。另一项 \(y_2 = e^{-2x}\),\(y_2' = -2e^{-2x}\),\(y_2'' = 4e^{-2x}\),代入 \(4 + 2 - 6 = 0\) \(\checkmark\)。

例题 B. 解 \(2y'' - 7y' + 3y = 0\),\(y(0) = 1, y'(0) = 0\)。

特征方程 \(2r^2 - 7r + 3 = 0\)。判别式 \(49 - 24 = 25 > 0\)。根 \(r = (7 \pm 5)/4\),即 \(r_1 = 3, r_2 = 1/2\)。通解

\[ y(x) = c_1 e^{3x} + c_2 e^{x/2}. \]

\(y' = 3c_1 e^{3x} + (1/2) c_2 e^{x/2}\)。初值:\(y(0) = c_1 + c_2 = 1\),\(y'(0) = 3c_1 + c_2/2 = 0\)。从第二式:\(c_2 = -6 c_1\)。代入第一式:\(c_1 - 6c_1 = -5c_1 = 1\),所以 \(c_1 = -1/5\),\(c_2 = 6/5\)。最终

\[ y(x) = -\tfrac{1}{5} e^{3x} + \tfrac{6}{5} e^{x/2}. \]

检验:\(y(0) = -1/5 + 6/5 = 1\) \(\checkmark\);\(y'(0) = -3/5 + 3/5 = 0\) \(\checkmark\)。物理直觉:\(t = 0\) 时位置 1、速度 0,但解里有一项 \(e^{3x}\),\(c_1 = -1/5 < 0\)——这意味着随着 \(x\) 增大,\(y\) 会先在 \(c_2 e^{x/2}\) 项的牵引下上升,然后被 \(c_1 e^{3x}\) 这一负的、增长更快的项拉下去,最后 \(y \to -\infty\)。可以验证转折点在 \(y' = 0\) 时:\(3c_1 e^{3x} + (c_2/2)e^{x/2} = 0\) 给出 \(e^{5x/2} = -c_2/(6c_1) = 1\),即 \(x = 0\)。所以转折点恰在初始处,之后立即开始下降。

情形二:重根

例题 C. 解 \(y'' + 4y' + 4y = 0\)。

特征方程 \(r^2 + 4r + 4 = (r + 2)^2 = 0\)。重根 \(r = -2\)。通解

\[ y(x) = (c_1 + c_2 x) e^{-2x}. \]

验证:\(y_2 = xe^{-2x}\),\(y_2' = (1 - 2x)e^{-2x}\),\(y_2'' = (-2 - 2(1 - 2x))e^{-2x} = (-4 + 4x)e^{-2x}\)。代入:

\[ y_2'' + 4 y_2' + 4 y_2 = e^{-2x}\left[(-4 + 4x) + 4(1 - 2x) + 4x\right] = e^{-2x}(0) = 0. \checkmark \]

例题 D. 解 \(y'' - 10y' + 25y = 0\),\(y(0) = 0, y'(0) = 1\)。

特征方程 \(r^2 - 10r + 25 = (r - 5)^2 = 0\)。重根 \(r = 5\)。通解

\[ y(x) = (c_1 + c_2 x) e^{5x}. \]

导数:\(y' = c_2 e^{5x} + 5(c_1 + c_2 x) e^{5x} = (5c_1 + c_2 + 5c_2 x) e^{5x}\)。初值:\(y(0) = c_1 = 0\)。\(y'(0) = 5c_1 + c_2 = c_2 = 1\)。最终

\[ y(x) = x e^{5x}. \]

检验:\(y(0) = 0\) \(\checkmark\);\(y' = e^{5x} + 5x e^{5x}\),\(y'(0) = 1\) \(\checkmark\)。

情形三:共轭复根

例题 E. 解 \(y'' + 6y' + 13y = 0\)。

特征方程 \(r^2 + 6r + 13 = 0\)。判别式 \(\Delta = 36 - 52 = -16 < 0\),情形三。\(r = (-6 \pm \sqrt{-16})/2 = -3 \pm 2i\)。\(\alpha = -3, \beta = 2\)。通解

\[ y(x) = c_1 e^{(-3 + 2i)x} + c_2 e^{(-3 - 2i)x}, \quad \text{或} \quad y(x) = e^{-3x}(A\cos 2x + B\sin 2x). \]

(实数化在第 6 章详讲;这里两种写法都可视为正确。)物理上,\(\alpha = -3 < 0\) 说明这是过阻尼振荡的反面——欠阻尼:振荡的振幅按 \(e^{-3x}\) 衰减,这是阻尼振子的典型行为。

例题 F. 解 \(y'' - 2y' + 10y = 0\),\(y(0) = 1, y'(0) = 0\)。

特征方程 \(r^2 - 2r + 10 = 0\)。\(\Delta = 4 - 40 = -36\)。根 \(r = (2 \pm 6i)/2 = 1 \pm 3i\)。\(\alpha = 1, \beta = 3\)。通解

\[ y(x) = e^x(A\cos 3x + B\sin 3x). \]

导数:\(y' = e^x(A\cos 3x + B\sin 3x) + e^x(-3A\sin 3x + 3B\cos 3x) = e^x[(A + 3B)\cos 3x + (B - 3A)\sin 3x]\)。初值:\(y(0) = A = 1\);\(y'(0) = A + 3B = 1 + 3B = 0\),所以 \(B = -1/3\)。最终

\[ y(x) = e^x\left(\cos 3x - \tfrac{1}{3}\sin 3x\right). \]

检验:\(y(0) = 1 \cdot 1 = 1\) \(\checkmark\);\(y'(0) = 1 \cdot (1 + 3 \cdot (-1/3)) = 1 \cdot 0 = 0\) \(\checkmark\)。注意 \(\alpha = 1 > 0\),所以解的振幅按 \(e^x\) 增长——这对应"负阻尼"系统(比如反馈放大器),在物理上少见但数学上完全合理。

小结:解二阶常系数齐次方程的总流程

把整章压成一张速查表:

判别式特征根通解形式典型物理
\(\Delta > 0\)两不同实根 \(r_1, r_2\)\(y = c_1 e^{r_1 x} + c_2 e^{r_2 x}\)过阻尼振子(无振荡,纯衰减)
\(\Delta = 0\)重根 \(r\)\(y = (c_1 + c_2 x)e^{rx}\)临界阻尼振子(刚刚不振荡)
\(\Delta < 0\)共轭复根 \(\alpha \pm \beta i\)\(y = e^{\alpha x}(A\cos\beta x + B\sin\beta x)\)欠阻尼振子(衰减振荡)/ 无阻尼振子(\(\alpha = 0\))

关于"为什么 \(\Delta\) 的符号决定了振荡与否",直觉上是这样的:特征方程 \(ar^2 + br + c = 0\) 来自 \(ay'' + by' + cy = 0\),如果把它解读为弹簧振子(\(a = m, b = c_\text{阻尼}, c = k\)),\(\Delta = b^2 - 4ac\) 比较的是"阻尼的平方"与"惯性 \(\times\) 弹性"。阻尼大,\(\Delta > 0\),衰减占主导,无振荡;阻尼适中(临界),\(\Delta = 0\),刚刚抑制住振荡;阻尼小,\(\Delta < 0\),振荡得以发生。第 9 章会专门用物理语言讲透这件事。

这一章我们做的事是:把"二阶常系数线性齐次方程"这一类方程完全解决了——你会写出特征方程,会按三种情形分别给出通解,会用初值定常数。这套技术看起来初等,但实际上它支撑了物理学和工程学里巨大的篇章。下一章我们做两件事:把情形三的"复指数"严格地实数化(看到 Euler 公式 \(e^{i\theta} = \cos\theta + i\sin\theta\) 在这里如何被使用),并把这些工具用到一个具体的振动应用——简谐运动和阻尼振子,你会真正看到方程的解和你日常感知的振动现象如何对应。
