第 6 章 复根与振荡

第 5 章里我们处理了二阶常系数线性齐次方程 \(ay'' + by' + cy = 0\) 的特征方程 \(ar^2 + br + c = 0\),并得到了两种实根情形:两个不同实根给出 \(y = c_1 e^{r_1 x} + c_2 e^{r_2 x}\),一个重实根给出 \(y = (c_1 + c_2 x)e^{rx}\)。我们故意把判别式 \(b^2 - 4ac < 0\) 的情形留到了下一章——也就是这一章。本章要解决的就是它。

当判别式为负时,特征方程没有实根,而是给出一对共轭复根 \(r = \alpha \pm \beta i\)。这听起来麻烦——一个物理问题里的解不可能真的取复数值。但只要我们按部就班地把复指数展开,实数解会自动浮出水面,而且它带来的图像比前两种情形都更生动:它就是振荡——弹簧来回摆动、电路里的电流交变、单摆周期摇摆。换句话说,本章的复根并不是一种"额外的怪情形",反而是真实世界里最常见的一种情形。

本章将依次处理:

  • 6.1 复指数 \(e^{(\alpha + \beta i)x}\) 的速通,以及欧拉公式 \(e^{i\theta} = \cos\theta + i\sin\theta\) 在这里到底起什么作用;
  • 6.2 把复指数实数化,得到形如 \(y = e^{\alpha x}(c_1 \cos\beta x + c_2 \sin\beta x)\) 的解;
  • 6.3 振幅-相位形式 \(y = R e^{\alpha x}\cos(\beta x - \phi)\),以及怎么从 \(c_1, c_2\) 算出 \(R\) 和 \(\phi\);
  • 6.4 \(\alpha\) 与 \(\beta\) 的物理含义——衰减率与角频率;
  • 6.5 同一个二阶方程在三种判别式下的三种行为:欠阻尼 / 临界阻尼 / 过阻尼;
  • 6.6 详细例题:弹簧-质量、RLC 电路初探,把第 8 章物理应用提前打个底。

本章不需要太多新数学;欧拉公式如果你在上卷附录 B 里见过,这里可以直接用。它需要的是把"复数"和"实数"两套语言来回切换的熟练度。一旦你能在 \(e^{(2 + 3i)x}\) 与 \(e^{2x}(\cos 3x + i\sin 3x)\) 之间无缝转换,本章就算过关了。

6.1 复指数与欧拉公式速通

先把符号铺开。复数 \(z = \alpha + \beta i\) 的指数 \(e^z\) 是什么意思?如果 \(z\) 是实数,这个问题有现成的答案——\(e^x\) 就是那条熟悉的指数曲线。但当 \(z\) 是 \(2 + 3i\) 这种东西时,我们没有"\(2.71828^{2+3i}\)" 的直接计算法。我们必须定义 \(e^z\),并让定义在两件事上同时合理:对实 \(z\) 还原为普通指数;对所有 \(z\) 满足指数律 \(e^{z_1 + z_2} = e^{z_1} e^{z_2}\)。

这里的关键事实——欧拉公式——告诉我们怎么定义。对任意实数 \(\theta\),

\[ e^{i\theta} = \cos\theta + i\sin\theta. \]

这是十八世纪欧拉的杰作。怎么相信这个公式?最快的途径是把三者的泰勒级数都写出来:

\[ e^{i\theta} = \sum_{n=0}^{\infty}\frac{(i\theta)^n}{n!} = 1 + i\theta - \frac{\theta^2}{2!} - i\frac{\theta^3}{3!} + \frac{\theta^4}{4!} + \cdots \]

把实部和虚部分开,实部是 \(1 - \theta^2/2! + \theta^4/4! - \cdots\),正是 \(\cos\theta\) 的泰勒级数;虚部是 \(\theta - \theta^3/3! + \theta^5/5! - \cdots\),正是 \(\sin\theta\) 的泰勒级数。两者相加正好就是 \(\cos\theta + i\sin\theta\)。完整推导上卷附录 B 已经给过,这里我们直接采用。

欧拉公式的几何含义同样优雅。\(e^{i\theta}\) 是复平面上一个模为 1 的点,它到原点的距离恒等于 \(\sqrt{\cos^2\theta + \sin^2\theta} = 1\)。当 \(\theta\) 从 0 增加,这个点沿单位圆逆时针转动;\(\theta = \pi/2\) 时它在 \(i\),\(\theta = \pi\) 时在 \(-1\),\(\theta = 2\pi\) 时回到 \(1\)。所以"复指数 \(e^{i\theta}\)" 就是"单位圆上转角 \(\theta\) 的那个点"——一个把"指数"与"旋转"等同起来的奇迹。

有了 \(e^{i\theta}\),复指数 \(e^{\alpha + \beta i}\) 就直接由指数律给出:

\[ e^{\alpha + \beta i} = e^{\alpha} \cdot e^{\beta i} = e^{\alpha}(\cos\beta + i\sin\beta). \]

它的模是 \(e^{\alpha}\),辐角是 \(\beta\)。换句话说:实部 \(\alpha\) 控制"放大或缩小",虚部 \(\beta\) 控制"旋转角度"。这是一个值得装裱起来的图像。

把它推广到 \(x\) 的函数。对每一个实数 \(x\),

\[ e^{(\alpha + \beta i)x} = e^{\alpha x + \beta i x} = e^{\alpha x}\bigl(\cos\beta x + i\sin\beta x\bigr). \]

这是本章最常用的一行。把它读熟:左边是抽象的复指数,右边是一个实数函数(\(e^{\alpha x}\cos\beta x\)) 加上 \(i\) 倍另一个实数函数(\(e^{\alpha x}\sin\beta x\))。

顺便练一下导数。我们希望 \(\frac{d}{dx}e^{rx} = r e^{rx}\) 对复 \(r\) 也成立。验证一下 \(r = \alpha + \beta i\) 的情形:

\[ \frac{d}{dx}\bigl[e^{\alpha x}(\cos\beta x + i\sin\beta x)\bigr] = e^{\alpha x}\bigl[\alpha(\cos\beta x + i\sin\beta x) + (-\beta\sin\beta x + i\beta\cos\beta x)\bigr] \]

把第二个括号里的虚部 \(i\) 提出来,得到 \(\alpha + i\beta\) 这个公因子,再乘回 \(e^{\alpha x}(\cos\beta x + i\sin\beta x)\),正是 \((\alpha + \beta i) e^{(\alpha + \beta i)x}\)。所以求导规则不用动,复指数照样满足 \((e^{rx})' = r e^{rx}\)。这意味着第 5 章里"代入特征方程"的全部代数,都对复 \(r\) 直接成立——我们什么都不必重做,只需要把答案翻译回实数。

最后再把欧拉公式的两条推论列出来,后面要用:

\[ \cos\theta = \frac{e^{i\theta} + e^{-i\theta}}{2}, \qquad \sin\theta = \frac{e^{i\theta} - e^{-i\theta}}{2i}. \]

这两个等式从 \(e^{i\theta} = \cos\theta + i\sin\theta\) 与 \(e^{-i\theta} = \cos\theta - i\sin\theta\) 相加相减立得。它们说"三角函数其实就是复指数的特殊组合",而我们后面在做实数化时,正好要走相反的方向——把复指数组合回三角函数。

6.2 把复指数实数化

来正经地解一个方程。考虑

\[ y'' + 4y = 0. \]

特征方程 \(r^2 + 4 = 0\) 给出 \(r = \pm 2i\)。判别式 \(b^2 - 4ac = 0 - 16 = -16 < 0\),没有实根。按第 5 章的"两个不同根"模式,通解的形式上应该是

\[ y = C_1 e^{2ix} + C_2 e^{-2ix}, \]

其中 \(C_1, C_2\) 是复常数。这是数学上正确的,但不是物理上想要的——一个弹簧的位置不可能等于"\(C_1\) 倍某个复函数加上 \(C_2\) 倍另一个复函数"。我们想要的是实数解。

办法是把欧拉公式代回去。\(e^{2ix} = \cos 2x + i\sin 2x\),\(e^{-2ix} = \cos 2x - i\sin 2x\)。所以

\[ y = C_1(\cos 2x + i\sin 2x) + C_2(\cos 2x - i\sin 2x) \] \[ = (C_1 + C_2)\cos 2x + i(C_1 - C_2)\sin 2x. \]

把 \(c_1 = C_1 + C_2\)、\(c_2 = i(C_1 - C_2)\) 当作两个新的常数,通解就写成

\[ y = c_1\cos 2x + c_2\sin 2x. \]

这是实数函数,只要我们选 \(c_1, c_2\) 为实数,\(y\) 就是实数。具体来说,要让 \(c_1, c_2\) 为实数,只需 \(C_1\) 与 \(C_2\) 互为共轭复数(\(C_2 = \bar{C_1}\))——这样 \(C_1 + C_2 = 2\Re C_1\) 是实数,\(i(C_1 - C_2) = i \cdot 2i\Im C_1 = -2\Im C_1\) 也是实数。

不过这些复数代数我们以后基本不会再写。整章的规则可以一行总结:

\[ \boxed{\text{若特征方程的根为 } r = \alpha \pm \beta i,\quad \text{则 } y = e^{\alpha x}\bigl(c_1\cos\beta x + c_2\sin\beta x\bigr).} \]

其中 \(c_1, c_2\) 是任意两个实常数,由初始条件确定。这就是本章的"主公式"。它从此取代第 5 章那个"两个不同实根 \(\Rightarrow c_1 e^{r_1 x} + c_2 e^{r_2 x}\)"的模式,适用于复根情形。

为什么它一定是解?因为 \(e^{(\alpha + \beta i)x}\) 与 \(e^{(\alpha - \beta i)x}\) 都是齐次方程的复值解(直接代入就能验证),所以它们的任意线性组合也是解,而且这个组合的实部和虚部分别是

\[ \Re\bigl(e^{(\alpha + \beta i)x}\bigr) = e^{\alpha x}\cos\beta x, \qquad \Im\bigl(e^{(\alpha + \beta i)x}\bigr) = e^{\alpha x}\sin\beta x. \]

因为方程是线性的且系数是实数,实部和虚部各自就都是实值解。这两个实值解之间相互线性无关(它们的 Wronskian 不为零,这一点会留给习题——你做一下就能看到 \(e^{2\alpha x} \beta\),只要 \(\beta \ne 0\) 就非零)。所以它们构成基,通解是它们的线性组合,这就是上面的主公式。

回到例子。\(y'' + 4y = 0\) 中 \(\alpha = 0\),\(\beta = 2\),所以 \(y = c_1\cos 2x + c_2\sin 2x\)。如果再给一组初始条件 \(y(0) = 3\)、\(y'(0) = 4\),代入:\(y(0) = c_1 = 3\),\(y'(x) = -2c_1\sin 2x + 2c_2\cos 2x\),\(y'(0) = 2c_2 = 4\),所以 \(c_2 = 2\)。最终 \(y = 3\cos 2x + 2\sin 2x\)。

来一个虚部和实部都不平凡的例子。解 \(y'' - 2y' + 5y = 0\)。特征方程 \(r^2 - 2r + 5 = 0\),用求根公式 \(r = (2 \pm \sqrt{4 - 20})/2 = (2 \pm \sqrt{-16})/2 = 1 \pm 2i\)。所以 \(\alpha = 1\)、\(\beta = 2\),通解

\[ y = e^{x}(c_1\cos 2x + c_2\sin 2x). \]

注意:这个解既有指数因子 \(e^x\)(在放大),又有三角振荡。如果再加初始条件 \(y(0) = 0\)、\(y'(0) = 1\),那么 \(y(0) = c_1 = 0\)。求导:\(y' = e^x(c_1\cos 2x + c_2\sin 2x) + e^x(-2c_1\sin 2x + 2c_2\cos 2x)\),代入 \(c_1 = 0\) 后 \(y' = e^x(c_2\sin 2x + 2c_2\cos 2x)\),所以 \(y'(0) = 2c_2 = 1\),\(c_2 = 1/2\)。最终 \(y = \tfrac{1}{2}e^x \sin 2x\)。

来一个衰减振荡的例子。\(y'' + 2y' + 5y = 0\)。特征方程 \(r^2 + 2r + 5 = 0\),根 \(r = (-2 \pm \sqrt{4 - 20})/2 = -1 \pm 2i\)。所以 \(\alpha = -1\)(注意符号,与上面那道题相反),\(\beta = 2\),通解

\[ y = e^{-x}(c_1\cos 2x + c_2\sin 2x). \]

这一次指数因子 \(e^{-x}\) 在衰减,所以 \(y\) 的振幅会随时间越变越小,最终趋于零。后面 6.4 节会回过头来给 \(\alpha\) 这个符号一个物理解释——它就是物理学里说的"衰减率"。

这里特别值得提醒一句:你可能会想"为什么不写成 \(c_1 e^{ix} + c_2 e^{-ix}\) 之类",答案是没人禁止你这么写——它确实也是合法的复值通解——但实际计算的时候它非常麻烦,而且你交作业时老师想要的几乎肯定是实数形式。养成把复指数立刻翻译成 \(e^{\alpha x}(c_1\cos\beta x + c_2\sin\beta x)\) 的习惯,以后省下大量返工。

6.3 振幅-相位形式

上一节的主公式 \(y = e^{\alpha x}(c_1\cos\beta x + c_2\sin\beta x)\) 是数学上最方便的形式,因为它直接由初始条件 \(y(0) = c_1\)、\(y'(0)\) 与 \(c_2\) 的简单组合确定。但物理上有一种更直观的形式,我们经常需要把它改写过来。

来看一个具体的例子。设 \(c_1 = 3\)、\(c_2 = 4\)、\(\alpha = 0\)、\(\beta = 2\),也就是 \(y = 3\cos 2x + 4\sin 2x\)。这条曲线在做什么?如果你用计算器画一下,你会发现它仍然是一条单一频率的余弦波,只不过振幅不是 3 也不是 4,而是某个其他值,而且它的峰值不在 \(x = 0\)。能不能直接把它写成一条 \(R\cos(2x - \phi)\) 的形式?可以——这是一条三角恒等式技巧。

展开 \(R\cos(\beta x - \phi) = R(\cos\beta x \cos\phi + \sin\beta x \sin\phi) = (R\cos\phi)\cos\beta x + (R\sin\phi)\sin\beta x\)。把它和 \(c_1\cos\beta x + c_2\sin\beta x\) 比较,只要

\[ R\cos\phi = c_1, \qquad R\sin\phi = c_2, \]

两种写法就完全等价。把这两式平方相加,得 \(R^2\cos^2\phi + R^2\sin^2\phi = c_1^2 + c_2^2\),即 \(R^2 = c_1^2 + c_2^2\)。把它们相除,得 \(\tan\phi = c_2/c_1\)。所以

\[ \boxed{R = \sqrt{c_1^2 + c_2^2},\qquad \tan\phi = \frac{c_2}{c_1}.} \]

这给出了实数解的振幅-相位形式:

\[ y = R e^{\alpha x}\cos(\beta x - \phi). \]

把它读出来:\(R\) 是振幅(峰值的高度);\(e^{\alpha x}\) 是包络(振幅怎么随时间放大或缩小);\(\beta\) 是角频率(每单位时间转过的弧度);\(\phi\) 是相位(波形的水平偏移,峰值出现在 \(\beta x = \phi\) 处)。

回到我们刚才那个例子,\(c_1 = 3\)、\(c_2 = 4\)。\(R = \sqrt{9 + 16} = 5\),\(\tan\phi = 4/3\),所以 \(\phi = \arctan(4/3) \approx 0.927\) 弧度,约 \(53.1°\)。于是

\[ 3\cos 2x + 4\sin 2x = 5\cos(2x - 0.927). \]

这一行很有用——它把"两个三角项的和"压缩成"一个三角项",对画图、找峰值、求最大值都极其方便。\(y\) 的最大值显然就是 \(R = 5\),发生在 \(2x - 0.927 = 0\),也就是 \(x \approx 0.463\) 处。

关于 \(\phi\) 的象限。\(\arctan\) 函数只返回 \((-\pi/2, \pi/2)\) 之间的值,所以单凭 \(\tan\phi = c_2/c_1\) 不足以唯一确定 \(\phi\) 的位置。要正确选择,你必须同时看 \(c_1\) 和 \(c_2\) 的符号:

  • 若 \(c_1 > 0\),则 \(\phi\) 在第一或第四象限,\(\arctan(c_2/c_1)\) 直接给出正确值。
  • 若 \(c_1 < 0\),则 \(\phi\) 在第二或第三象限,\(\arctan(c_2/c_1)\) 算出来的值要加上 \(\pi\)。
  • 若 \(c_1 = 0\),则 \(\tan\phi\) 没定义;此时若 \(c_2 > 0\) 则 \(\phi = \pi/2\),若 \(c_2 < 0\) 则 \(\phi = -\pi/2\)(或者你愿意写 \(3\pi/2\) 也行——相位差 \(2\pi\) 整数倍是无关紧要的)。

这正是计算器上 atan2(c_2, c_1) 函数所做的事——它考虑两个分量的符号,直接返回正确的象限。如果你手算,养成"画一个 \((c_1, c_2)\) 在平面上的点,看在哪个象限"的小习惯,基本不会错。

来一个练手:把 \(y = -2\cos 3x + 2\sqrt{3}\sin 3x\) 化成振幅-相位形式。\(R = \sqrt{4 + 12} = 4\)。\(\tan\phi = 2\sqrt{3}/(-2) = -\sqrt{3}\)。如果不看象限,\(\arctan(-\sqrt{3}) = -\pi/3\),但 \(c_1 = -2 < 0\)、\(c_2 = 2\sqrt{3} > 0\),所以 \((c_1, c_2)\) 在第二象限,正确的 \(\phi = -\pi/3 + \pi = 2\pi/3\)。最终

\[ -2\cos 3x + 2\sqrt{3}\sin 3x = 4\cos(3x - 2\pi/3). \]

验证一下:\(4\cos\phi = 4\cos(2\pi/3) = -2\) ✓;\(4\sin\phi = 4\sin(2\pi/3) = 2\sqrt{3}\) ✓。对了。

另一种约定。有些书用 \(R\sin(\beta x + \phi)\) 而不是 \(R\cos(\beta x - \phi)\),或者写 \(R\cos(\beta x + \phi)\)(\(\phi\) 前是加号)。它们之间只差一个相位的平移和符号——本质上都是同一族函数。本书采用 \(R\cos(\beta x - \phi)\),这样 \(\phi > 0\) 表示波形向右平移(峰值出现在更晚的时间),与"延迟相位"这个物理直觉吻合。如果你看到别的书用别的约定,问自己一句"它的 \(\phi\) 是哪种符号约定下的",就能转换。

6.4 \(\alpha\) 与 \(\beta\) 的物理含义

现在我们已经能熟练写出 \(y = R e^{\alpha x}\cos(\beta x - \phi)\) 这种通解。是时候停下来想一想,\(\alpha\) 和 \(\beta\) 在物理上分别代表什么。

先说 \(\beta\)。它出现在 \(\cos(\beta x - \phi)\) 里。三角函数的周期是 \(2\pi\),所以这个余弦的周期是

\[ T = \frac{2\pi}{\beta}. \]

反过来看,每单位 \(x\) 经过 \(\beta\) 弧度,所以 \(\beta\) 叫做角频率(angular frequency,通常单位是"弧度每秒",如果 \(x\) 是时间)。它和我们日常说的频率(每秒振动几次)的关系是 \(f = \beta/(2\pi)\)——除以 \(2\pi\) 是因为一个完整振动是 \(2\pi\) 弧度。

举例:\(y'' + 4y = 0\) 给出 \(\beta = 2\),所以周期 \(T = \pi\)。\(y'' + 100y = 0\) 给出 \(\beta = 10\),周期 \(T = \pi/5 \approx 0.628\)——振动比前者快 5 倍。一般地,\(y'' + \omega_0^2 y = 0\) 这种"无阻尼振子"方程的角频率就是 \(\omega_0\),这正是它符号的来源:用 \(\omega_0\) 而不是 \(\beta\) 来强调它就是这个系统的固有角频率。

再说 \(\alpha\)。它出现在 \(e^{\alpha x}\) 这个包络因子里。看 \(R e^{\alpha x}\cos(\beta x - \phi)\):无论余弦本身怎么振荡,它的振幅在以指数 \(R e^{\alpha x}\) 在变化。三种情况:

  • \(\alpha < 0\):\(e^{\alpha x}\) 随 \(x\) 增加而衰减,振幅越来越小,最终归零。这就是衰减振荡。\(\alpha\) 的绝对值越大,衰减越快。物理上,\(\alpha\) 越负代表系统的"阻尼越大",能量耗散越快。
  • \(\alpha = 0\):\(e^{\alpha x} = 1\),包络是常数 \(R\),系统做简谐振动(无阻尼)。这种理想化的情形对应没有摩擦、没有空气阻力的弹簧,数学上很干净,物理上是个有用的极限。
  • \(\alpha > 0\):\(e^{\alpha x}\) 随 \(x\) 增加而放大,振幅越来越大。这种情形在控制系统里是不稳定的标志——一架飞机出现增长振荡通常意味着马上要散架。

因此,\(\alpha\) 和 \(\beta\) 各管一摊:\(\beta\) 决定振多快,\(\alpha\) 决定振多久(或者会不会越振越大)。给定一个二阶常系数齐次方程,只要把它的特征方程根算出来,你立刻就能写出系统的"振多快"和"振多久"——不必再求一遍解。这是本章的核心实用价值。

来一个具体的例子。考虑 \(y'' + 0.4 y' + 4.04 y = 0\),特征方程根

\[ r = \frac{-0.4 \pm \sqrt{0.16 - 16.16}}{2} = \frac{-0.4 \pm \sqrt{-16}}{2} = -0.2 \pm 2i. \]

所以 \(\alpha = -0.2\)、\(\beta = 2\)。系统的振荡周期是 \(2\pi/2 = \pi\),振幅按 \(e^{-0.2 x}\) 衰减。一个有用的指标是半衰期——振幅衰减到一半所需的时间:\(e^{-0.2 t_{1/2}} = 1/2\),解得 \(t_{1/2} = \ln 2/0.2 \approx 3.466\)。也就是说,大约每经过 3.5 秒(假设 \(x\) 是秒),振幅会减半。

另一种衡量方式是 \(\alpha\) 的倒数,它有个名字叫时间常数 \(\tau = 1/|\alpha|\)。在此例中 \(\tau = 5\)。物理上,\(\tau\) 是包络衰减到原来的 \(1/e \approx 36.8\%\) 所需的时间。\(\tau\) 与半衰期相差一个因子 \(\ln 2 \approx 0.693\),换算关系 \(t_{1/2} = \tau \ln 2\)。两种说法各有用处:工程师爱说 \(\tau\),实验室里测振动的人爱说 \(t_{1/2}\)。

6.5 三种情形对比:欠阻尼 / 临界阻尼 / 过阻尼

到这里,我们已经把二阶常系数齐次方程的全部三种情形走完了一遍。把它们摆到一起对比,会看出一个非常优雅的"统一图景"——同一个方程,在调节判别式的过程中,可以连续地从一种行为过渡到另一种。

考虑物理学里最常见的二阶方程

\[ m y'' + c y' + k y = 0, \]

其中 \(m, c, k > 0\)(分别代表质量、阻尼系数、弹性系数)。特征方程 \(m r^2 + c r + k = 0\) 的判别式是 \(\Delta = c^2 - 4 m k\)。把根写出来:

\[ r = \frac{-c \pm \sqrt{c^2 - 4mk}}{2m}. \]

三种情形:

过阻尼 (overdamped):\(c^2 > 4mk\),即 \(\Delta > 0\)。两个不同实根,而且都是负的(因为 \(-c < 0\) 且 \(\sqrt{c^2 - 4mk} < c\))。通解 \(y = c_1 e^{r_1 x} + c_2 e^{r_2 x}\),没有振荡,只是两个衰减指数的叠加。物理图像:阻尼太大,系统挤过平衡位置(或者根本到不了),然后慢慢蹭回去。开门时门弓特别紧的那种感觉就是过阻尼——门关不上,缓缓挪到关上为止。

临界阻尼 (critically damped):\(c^2 = 4mk\),即 \(\Delta = 0\)。一个重实根 \(r = -c/(2m) < 0\)。通解 \(y = (c_1 + c_2 x) e^{r x}\),仍然没有振荡,但比过阻尼"刚好"再快一点——它是所有不振荡情形里衰减最快的那一个。物理图像:这是阻尼调到"刚好不振"的临界点;门弹簧就该这么调——既不让门"砰"地撞回去,又能尽快关好。

欠阻尼 (underdamped):\(c^2 < 4mk\),即 \(\Delta < 0\)。一对共轭复根 \(r = \alpha \pm \beta i\),其中 \(\alpha = -c/(2m) < 0\)、\(\beta = \sqrt{4mk - c^2}/(2m) > 0\)。通解 \(y = e^{\alpha x}(c_1\cos\beta x + c_2\sin\beta x)\),既有振荡又有衰减。物理图像:这是真正的"振荡"情形——弹簧来回摆动几次后慢慢停下,门弹回去几下才平衡。日常的弹簧、单摆、汽车悬挂(在调好的范围内)都属于欠阻尼。

三种情形可以排成一张表:

判别式根的类型通解形式振荡?物理名称
\(\Delta > 0\)两实根 \(r_1 \ne r_2\)(均负)\(c_1 e^{r_1 x} + c_2 e^{r_2 x}\)否过阻尼
\(\Delta = 0\)重实根 \(r\)(负)\((c_1 + c_2 x) e^{r x}\)否临界阻尼
\(\Delta < 0\)复根 \(\alpha \pm \beta i\)(\(\alpha < 0\))\(e^{\alpha x}(c_1\cos\beta x + c_2\sin\beta x)\)是欠阻尼

临界阻尼是过阻尼与欠阻尼之间的分水岭。从工程角度看,这是绝大多数设计上希望的状态——比临界更阻尼一点,反应迟缓;比临界更欠一点,会有不必要的振荡。汽车减震器的目标就是把系统调到尽可能接近临界,既能吸收颠簸,又不让车反复弹跳。

下面来一道把三种情形跑一遍的题。考虑同一个 \(m = 1\)、\(k = 4\) 的弹簧,只改 \(c\) 的值,看看会发生什么。\(\Delta = c^2 - 16\),所以临界值是 \(c = 4\)。

\(\boldsymbol{c = 1}\)(欠阻尼):\(r = (-1 \pm \sqrt{1 - 16})/2 = (-1 \pm \sqrt{-15})/2 = -0.5 \pm i\sqrt{15}/2\)。\(\alpha = -0.5\)、\(\beta = \sqrt{15}/2 \approx 1.936\)。系统会振荡,周期 \(2\pi/\beta \approx 3.245\),振幅按 \(e^{-0.5 x}\) 衰减(每经过 \(\ln 2/0.5 \approx 1.386\) 秒振幅减半)。

\(\boldsymbol{c = 4}\)(临界阻尼):\(r = -2\),重根。\(y = (c_1 + c_2 x)e^{-2x}\)。无振荡,衰减时间常数 \(1/2 = 0.5\) 秒。

\(\boldsymbol{c = 5}\)(过阻尼):\(r = (-5 \pm \sqrt{25 - 16})/2 = (-5 \pm 3)/2\),即 \(r_1 = -1\)、\(r_2 = -4\)。\(y = c_1 e^{-x} + c_2 e^{-4x}\)。无振荡,主要由衰减较慢的 \(e^{-x}\) 项决定——所以反而比临界阻尼回到平衡要慢!这一点常常违反初学者的直觉:阻尼加大并不一定让系统回得更快;过阻尼反而比临界阻尼更迟钝。

这条违反直觉的事实,是工程上"调到临界"的核心理由。如果你设计一台仪器希望它"晃动一次后尽快归位",答案是临界阻尼;如果你设计一个测量系统希望"完全不要过冲",最近临界但偏过阻尼一点点是上选;如果你设计一个钟摆希望"尽量长时间走下去",欠阻尼且 \(\alpha\) 尽量接近 0(用尽可能小的摩擦)。

6.6 详细例题

6.6.1 例题:基本复根

解 \(y'' + 9y = 0\),初值 \(y(0) = 2\)、\(y'(0) = -3\)。

特征方程 \(r^2 + 9 = 0\),根 \(r = \pm 3i\)。\(\alpha = 0\)、\(\beta = 3\)。通解

\[ y = c_1\cos 3x + c_2\sin 3x. \]

初值:\(y(0) = c_1 = 2\)。求导:\(y' = -3c_1\sin 3x + 3c_2\cos 3x\),\(y'(0) = 3c_2 = -3\),所以 \(c_2 = -1\)。最终

\[ y = 2\cos 3x - \sin 3x. \]

把它化成振幅-相位形式:\(R = \sqrt{4 + 1} = \sqrt{5}\)。\(\tan\phi = -1/2\),且 \(c_1 = 2 > 0\),所以 \(\phi = \arctan(-1/2) \approx -0.4636\) 弧度(即 \(-26.57°\))。于是

\[ y = \sqrt{5}\cos(3x + 0.4636). \]

(注意:这里 \(\phi\) 取负值,所以 \(\cos(3x - \phi)\) 实际上是 \(\cos(3x + 0.4636)\),我把它写成更直观的形式。)振幅 \(\sqrt{5} \approx 2.236\),最大值在 \(3x + 0.4636 = 0\),即 \(x \approx -0.155\)——下一个峰值在 \(x \approx -0.155 + 2\pi/3 \approx 1.939\) 处。

6.6.2 例题:衰减振荡

解 \(y'' + 4y' + 13y = 0\),初值 \(y(0) = 1\)、\(y'(0) = 0\)。

特征方程 \(r^2 + 4r + 13 = 0\),根 \(r = (-4 \pm \sqrt{16 - 52})/2 = (-4 \pm \sqrt{-36})/2 = -2 \pm 3i\)。\(\alpha = -2\)、\(\beta = 3\)。通解

\[ y = e^{-2x}(c_1\cos 3x + c_2\sin 3x). \]

初值 \(y(0) = c_1 = 1\)。求导:

\[ y' = -2 e^{-2x}(c_1\cos 3x + c_2\sin 3x) + e^{-2x}(-3c_1\sin 3x + 3c_2\cos 3x) \] \[ = e^{-2x}\bigl[(-2 c_1 + 3c_2)\cos 3x + (-2c_2 - 3c_1)\sin 3x\bigr]. \]

\(y'(0) = -2c_1 + 3c_2 = 0\),代入 \(c_1 = 1\) 得 \(c_2 = 2/3\)。最终

\[ y = e^{-2x}\bigl(\cos 3x + \tfrac{2}{3}\sin 3x\bigr). \]

振幅-相位形式:\(R = \sqrt{1 + 4/9} = \sqrt{13/9} = \sqrt{13}/3\)。\(\tan\phi = (2/3)/1 = 2/3\),\(c_1 > 0\) 所以 \(\phi = \arctan(2/3) \approx 0.5880\) 弧度。所以

\[ y = \frac{\sqrt{13}}{3} e^{-2x}\cos(3x - 0.588). \]

振幅以 \(e^{-2x}\) 衰减,半衰期 \(\ln 2/2 \approx 0.347\) 秒——很快。每过 0.35 秒,峰值就减半;3.5 秒后,振幅降到原来的 \(2^{-10} \approx 0.001\),实际上已经"看不见"了。

6.6.3 例题:弹簧-质量(初探)

一个 1 千克的物体挂在弹簧上,弹簧的弹性系数 \(k = 25\) N/m,空气阻尼系数 \(c = 6\) Ns/m。把物体从平衡位置往下拉 0.1 米后松手(初速度为零)。物体什么时候第一次回到平衡位置?

设 \(y(t)\) 为物体偏离平衡位置的位移(向下为正),则牛顿第二定律给出

\[ y'' + 6y' + 25y = 0, \]

初值 \(y(0) = 0.1\)、\(y'(0) = 0\)。特征方程 \(r^2 + 6r + 25 = 0\),根 \(r = (-6 \pm \sqrt{36 - 100})/2 = -3 \pm 4i\)。所以欠阻尼,\(\alpha = -3\)、\(\beta = 4\)。通解

\[ y = e^{-3t}(c_1\cos 4t + c_2\sin 4t). \]

初值 \(y(0) = c_1 = 0.1\)。求导 \(y'(0) = -3c_1 + 4c_2 = 0\),解出 \(c_2 = 3 \cdot 0.1/4 = 0.075\)。所以

\[ y(t) = e^{-3t}(0.1\cos 4t + 0.075\sin 4t). \]

振幅-相位形式:\(R = \sqrt{0.01 + 0.005625} = \sqrt{0.015625} = 0.125\)。\(\tan\phi = 0.075/0.1 = 0.75\),\(\phi = \arctan(0.75) \approx 0.6435\)。所以

\[ y(t) = 0.125\, e^{-3t}\cos(4t - 0.6435). \]

"第一次回到平衡位置" 意味着 \(y(t) = 0\) 的第一个 \(t > 0\)。\(e^{-3t}\) 永远不为零,所以问题归结为 \(\cos(4t - 0.6435) = 0\),即 \(4t - 0.6435 = \pi/2 + k\pi\)(\(k = 0, 1, 2, \ldots\))。最小的 \(t > 0\) 来自 \(k = 0\):

\[ 4t = \pi/2 + 0.6435 = 2.214,\qquad t = 0.554\text{ 秒}. \]

所以物体大约 0.55 秒后第一次穿过平衡位置。物理上你会感到弹簧来回摆动几次(理论上无穷多次),但每次振幅都减小到上次的 \(e^{-3 \cdot 2\pi/4} = e^{-3\pi/2} \approx 0.0090\) 倍——也就是说,只一个周期之后振幅就基本消失了。这就是欠阻尼但接近临界(因为这里 \(c^2 = 36\) 与 \(4mk = 100\) 比起来已经不算小)的典型行为。

6.6.4 例题:RLC 电路初探

一个 RLC 串联电路:电感 \(L = 1\) H,电阻 \(R = 2\) Ω,电容 \(C = 1/26\) F。在 \(t = 0\) 时电容上有初始电荷 \(Q_0 = 5\) 库仑,电流为零。求电荷 \(Q(t)\)。

从基尔霍夫电压定律,\(L Q'' + R Q' + Q/C = 0\),代入数值:

\[ Q'' + 2 Q' + 26 Q = 0, \]

初值 \(Q(0) = 5\)、\(Q'(0) = 0\)(电流 \(I = Q'\) 初始为零)。特征方程 \(r^2 + 2r + 26 = 0\),根 \(r = (-2 \pm \sqrt{4 - 104})/2 = -1 \pm 5i\)。欠阻尼,\(\alpha = -1\)、\(\beta = 5\)。通解

\[ Q(t) = e^{-t}(c_1\cos 5t + c_2\sin 5t). \]

初值 \(Q(0) = c_1 = 5\)。求导:\(Q'(t) = e^{-t}\bigl[(-c_1 + 5c_2)\cos 5t + (-c_2 - 5c_1)\sin 5t\bigr]\),\(Q'(0) = -c_1 + 5c_2 = 0\),解出 \(c_2 = 1\)。所以

\[ Q(t) = e^{-t}(5\cos 5t + \sin 5t). \]

振幅-相位形式:\(R_0 = \sqrt{25 + 1} = \sqrt{26}\)(我把振幅记作 \(R_0\) 是为了避免与电阻 \(R\) 混淆)。\(\tan\phi = 1/5 = 0.2\),\(\phi \approx 0.1974\)。所以

\[ Q(t) = \sqrt{26}\, e^{-t}\cos(5t - 0.1974). \]

电流 \(I(t) = Q'(t)\) 也随之振荡衰减,频率与 \(Q\) 同步。电路里能量在电感的磁场和电容的电场之间来回转换,而电阻不断把这部分能量转化为热——这就是为什么振幅在指数衰减。注意 \(L Q'' + R Q' + Q/C = 0\) 的形式与 \(m y'' + c y' + k y = 0\) 完全一样:类比关系是 \(L \leftrightarrow m\)、\(R \leftrightarrow c\)、\(1/C \leftrightarrow k\)。这个类比在 8.5 节会被认真展开;这里先点一下,以后回来时你会觉得"啊这就是同一个故事"。

6.6.5 例题:把三种情形放在同一题里

对方程 \(y'' + b y' + 4y = 0\)(\(b > 0\)),分别取 \(b = 3\)、\(b = 4\)、\(b = 5\),求通解形式并说明属于哪一类。

\(b = 3\):判别式 \(9 - 16 = -7 < 0\),欠阻尼。根 \(r = (-3 \pm i\sqrt{7})/2\)。通解 \(y = e^{-3x/2}\bigl(c_1\cos(\sqrt{7}x/2) + c_2\sin(\sqrt{7}x/2)\bigr)\)。

\(b = 4\):判别式 \(16 - 16 = 0\),临界阻尼。根 \(r = -2\)(重)。通解 \(y = (c_1 + c_2 x)e^{-2x}\)。

\(b = 5\):判别式 \(25 - 16 = 9 > 0\),过阻尼。根 \(r = (-5 \pm 3)/2\),即 \(r_1 = -1\)、\(r_2 = -4\)。通解 \(y = c_1 e^{-x} + c_2 e^{-4x}\)。

三种通解形式完全不同,但它们描述的都是"小阻尼弹簧"型的物理:输入相同的初始扰动,看它怎么回到平衡。从 \(b = 3 \to 4 \to 5\) 你能感受到一个连续的过渡:振荡频率慢慢减小到零(欠阻尼),根从复变实(临界),然后两实根分开(过阻尼)。判别式 \(\Delta\) 像一个"调谐旋钮",不同位置对应不同的物理行为。

6.6.6 一个微妙之处:振幅是否真的"在 \(R e^{\alpha x}\) 这条曲线内"?

说"\(y = R e^{\alpha x}\cos(\beta x - \phi)\) 的振幅是 \(R e^{\alpha x}\)" 听起来天经地义,但严格来讲这只是包络——它是 \(y\) 各次峰值连成的曲线,而不是瞬时振幅。\(y\) 的实际峰值不是发生在余弦取 1 的地方,而是发生在 \(y' = 0\) 的地方,而 \(y'\) 包含了 \(e^{\alpha x}\) 求导带来的 \(\alpha\) 项,稍微把峰值时刻往前推了一点点。

具体计算:\(y = R e^{\alpha x}\cos(\beta x - \phi)\),求导用乘积法则,

\[ y' = R e^{\alpha x}\bigl[\alpha\cos(\beta x - \phi) - \beta\sin(\beta x - \phi)\bigr]. \]

令 \(y' = 0\),得 \(\tan(\beta x - \phi) = \alpha/\beta\)。如果 \(\alpha = 0\)(无衰减),这就是 \(\sin(\beta x - \phi) = 0\),即 \(\beta x - \phi = k\pi\),峰值在 \(\cos(\beta x - \phi) = \pm 1\) 处,也就是包络曲线上。但若 \(\alpha < 0\),\(\tan(\beta x - \phi) = \alpha/\beta\) 是个负数,\(\beta x - \phi\) 比"刚到 \(\pi\) 的整数倍"略微提前,所以实际峰值比 \(e^{\alpha x}\) 包络给出的略低。

不过这种"略低"的差异在实际中通常很小,而且只在 \(|\alpha|\) 与 \(\beta\) 量级相当时才明显。一般工程估算中,把 \(R e^{\alpha x}\) 当作"振幅曲线"是完全可以接受的近似,而且它在画图上是 \(y(x)\) 那条曲线的"包裹线"——你画一下立刻就能看明白。这就是"包络"这个名字的来源。

至此本章结束。下一章我们处理非齐次方程 \(ay'' + by' + cy = g(x)\):右端不再是零,而是被某个外力或外部信号驱动。这正是真实物理系统的常态——弹簧上挂着重物时弹簧已经被"挂"在那里了;电路里有电源时电源就是 \(g(t)\);摆受到周期性推动时,推力就是 \(g(t)\)。本章学的齐次解 \(y_h\) 在那里仍然是基础,但要再加上一个对应外力的特解 \(y_p\)。下一章就告诉你怎么找 \(y_p\)。
