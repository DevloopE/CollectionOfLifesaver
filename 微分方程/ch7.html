第 7 章 非齐次方程的特解

第 5、6 两章我们解决了二阶常系数线性齐次方程 \(ay'' + by' + cy = 0\),分别覆盖了不同实根、重根、复根三种情形。但齐次方程意味着"右端为零",物理上这表示没有外力:弹簧被拉动后自由释放,没有手在按它;电路里没有电源,只是把电容的初始电荷放掉;摆开始摇动后没有人再去推它。现实中绝大多数有趣的问题都有外力——一个驱动信号 \(g(x)\) 出现在右端:

\[ ay'' + by' + cy = g(x). \]

本章就处理这个加了右端的非齐次方程。结论会很漂亮——通解就是"齐次解 + 任一特解"——但找特解的过程涉及两个相对独立的方法,各有各的脾气。本章铺开的是:

  • 7.1 通解结构定理:为什么 \(y = y_h + y_p\) 给出全部解?
  • 7.2 待定系数法:猜一个形式,代回方程定系数,适用于 \(g(x)\) 是多项式、指数、三角函数及其乘积的情形;
  • 7.3 一张完整的"\(g\) 的形态 ↔ 试解的形式"对照表,加 5–6 道完整例题,以及"共振情形"的乘 \(x\) 修正;
  • 7.4 参数变易法:更一般也更通用,适用于 \(g\) 是任意连续函数的情形(包括 \(\tan x\)、\(\ln x\) 这种待定系数法解决不了的);
  • 7.5 怎么选——两种方法各擅长什么,什么时候用哪种。

本章的两种方法在思路上完全不同。待定系数法是"靠形式猜"——基于对 \(g(x)\) 形状的观察,猜一个含未知系数的试解,代入定出系数,简单直接但只对一类特殊的 \(g\) 有效。参数变易法是"靠齐次解的基算"——把齐次解的两个基函数当作"骨架",在它们前面挂上待定的函数而不是常数,然后用 Wronskian 行列式给出明确的积分公式;它通用,但需要算积分,有时积分本身比方程难。学完这两种,你能解决任何"二阶常系数线性方程"的非齐次问题——这一类方程在物理学和工程学里出现得太频繁,以至于本章会成为全书最常被回顾的一章之一。

7.1 通解结构定理

先把基本事实摆清。考虑非齐次方程

\[ L[y] := ay'' + by' + cy = g(x), \]

其中 \(L\) 是把函数 \(y\) 映到函数 \(ay'' + by' + cy\) 的线性算子("线性"的意思是 \(L[c_1 y_1 + c_2 y_2] = c_1 L[y_1] + c_2 L[y_2]\))。对应的齐次方程是 \(L[y] = 0\)。

定理(通解结构)。设 \(y_p\) 是非齐次方程 \(L[y] = g\) 的任一特解,\(y_h = c_1 y_1 + c_2 y_2\) 是齐次方程 \(L[y] = 0\) 的通解(\(y_1, y_2\) 是基)。则非齐次方程的通解是

\[ \boxed{y = y_h + y_p = c_1 y_1 + c_2 y_2 + y_p.} \]

证明分两步,简单到值得自己写一遍。

第一步:任何 \(y_h + y_p\) 形式的函数都是 \(L[y] = g\) 的解。这是线性的直接结果:

\[ L[y_h + y_p] = L[y_h] + L[y_p] = 0 + g = g. \checkmark \]

第二步:\(L[y] = g\) 的每一个解都能写成 \(y_h + y_p\) 的形式。设 \(Y\) 是任意一个解,即 \(L[Y] = g\)。那么 \(L[Y - y_p] = L[Y] - L[y_p] = g - g = 0\),所以 \(Y - y_p\) 是齐次方程的解,即 \(Y - y_p = c_1 y_1 + c_2 y_2 = y_h\),换句话说 \(Y = y_h + y_p\)。 \(\square\)

这个证明看起来朴素,但它的含义很深刻:寻找非齐次方程的全部解,被分解成了两件事——找齐次方程的全部解(基),再找非齐次方程的任意一个解。前者我们在第 5、6 章已经全部解决,后者就是本章余下要做的事。

有一条来自这个结构的实用推论是叠加原理:如果 \(g(x) = g_1(x) + g_2(x)\),且 \(y_{p,1}\) 是 \(L[y] = g_1\) 的特解,\(y_{p,2}\) 是 \(L[y] = g_2\) 的特解,则 \(y_p = y_{p,1} + y_{p,2}\) 是 \(L[y] = g\) 的特解。证明同样靠线性:\(L[y_{p,1} + y_{p,2}] = g_1 + g_2 = g\)。物理意义:多种不同外力的作用可以"分别求出特解再相加"。这条原理后面会反复用到——遇到 \(g(x) = 3 e^x + 5\sin 2x\) 这种东西,你不必硬猜一个怪形式,只要分别处理 \(3 e^x\) 部分和 \(5\sin 2x\) 部分,再相加就行。

7.2 待定系数法

第一种找特解的方法叫待定系数法(undetermined coefficients)——也叫"形式猜测法"。它的精神是:如果右端 \(g(x)\) 是某种"在求导下保持形式"的函数(比如多项式、指数、三角函数),那么 \(y_p\) 也极可能是同一形态的函数,只是系数未定。我们写下一个含未定系数的试解,代回方程,比较两边系数,把未定系数解出来。

7.2.1 简单的指数情形

来一个最简单的例子。求 \(y'' - 3y' + 2y = 5 e^{4x}\) 的特解。

第一步,先看右端 \(g(x) = 5 e^{4x}\)。它是一个指数函数,而求导只会让指数不变(\(e^{4x}\) 求导得 \(4 e^{4x}\)),所以代入 \(y'' - 3y' + 2y\) 后,左端必然是某个数乘 \(e^{4x}\)。要让它等于 \(5 e^{4x}\),自然该把 \(y_p\) 设为

\[ y_p = A e^{4x}, \]

其中 \(A\) 是待定系数。

第二步,代入。\(y_p' = 4A e^{4x}\),\(y_p'' = 16 A e^{4x}\)。

\[ y_p'' - 3y_p' + 2y_p = 16 A e^{4x} - 12 A e^{4x} + 2 A e^{4x} = 6 A e^{4x}. \]

要求等于 \(5 e^{4x}\),即 \(6A = 5\),所以 \(A = 5/6\)。最终

\[ y_p = \tfrac{5}{6} e^{4x}. \]

整体通解是 \(y = y_h + y_p\)。先解齐次:\(r^2 - 3r + 2 = 0\),根 \(r = 1, 2\),所以 \(y_h = c_1 e^x + c_2 e^{2x}\)。最终

\[ y = c_1 e^x + c_2 e^{2x} + \tfrac{5}{6} e^{4x}. \]

这就是非齐次方程的全部解。两个常数 \(c_1, c_2\) 由初始条件确定;特解部分 \(\frac{5}{6} e^{4x}\) 不含任何待定常数——它是一个具体的函数。

7.2.2 多项式情形

右端是多项式时,试解也是同次的多项式(含全部低次项)。来一个例子:求 \(y'' + y = x^2 + 1\) 的特解。

右端是二次多项式,所以猜 \(y_p = A x^2 + B x + C\)。求导:\(y_p' = 2Ax + B\)、\(y_p'' = 2A\)。代入:

\[ y_p'' + y_p = 2A + Ax^2 + Bx + C = Ax^2 + Bx + (2A + C). \]

等于 \(x^2 + 0 \cdot x + 1\),所以比较系数:\(A = 1\),\(B = 0\),\(2A + C = 1\) 解出 \(C = -1\)。最终

\[ y_p = x^2 - 1. \]

关键提醒:即使 \(g(x) = x^2 + 1\) 的一次项系数为零,试解里仍然必须带 \(B x\) 项。原因是导数会"降次":\(y_p' = 2Ax + B\),所以 \(B\) 项虽然在 \(g(x)\) 里"看不见",代入方程后还是会出现在某些位置。如果你漏掉它,系数比较时会出现矛盾。一般规则:多项式试解的次数和 \(g\) 一样,但要写全所有低次项。

7.2.3 三角函数情形

右端是 \(\sin\) 或 \(\cos\) 时,试解必须同时包含 \(\sin\) 和 \(\cos\)——因为求导会让 \(\sin\) 变 \(\cos\),反之亦然。例子:求 \(y'' + y' + y = 2\sin 3x\) 的特解。

试解

\[ y_p = A\sin 3x + B\cos 3x. \]

哪怕 \(g\) 里只有 \(\sin\) 没有 \(\cos\),试解里也要有 \(\cos\) 项。

计算:\(y_p' = 3A\cos 3x - 3B\sin 3x\)、\(y_p'' = -9A\sin 3x - 9B\cos 3x\)。代入:

\[ y_p'' + y_p' + y_p = (-9A - 3B + A)\sin 3x + (-9B + 3A + B)\cos 3x \] \[ = (-8A - 3B)\sin 3x + (3A - 8B)\cos 3x. \]

比较系数:\(-8A - 3B = 2\) 且 \(3A - 8B = 0\)。从第二式 \(A = 8B/3\),代入第一式 \(-8 \cdot 8B/3 - 3B = 2\),即 \(-(64/3 + 9/3)B = 2\),\(-(73/3) B = 2\),\(B = -6/73\)。\(A = 8(-6/73)/3 = -16/73\)。最终

\[ y_p = -\tfrac{16}{73}\sin 3x - \tfrac{6}{73}\cos 3x. \]

不算漂亮,但完全正确。这种"两个未知数的二乘二线性方程组"是三角型试解的标准副产物;你解得多了,自然就快。

7.2.4 共振情形:试解与齐次解相同的应对

下面这件事非常重要:有时按上述规则猜的试解会失效,因为它恰好就是齐次方程的解,代入后两边都为零,根本定不出系数。这种情形就是共振(resonance)。例子:求 \(y'' - 3y' + 2y = e^{2x}\) 的特解。

按 7.2.1 的规则,猜 \(y_p = A e^{2x}\)。代入:\(y_p' = 2Ae^{2x}\)、\(y_p'' = 4Ae^{2x}\),所以 \(y_p'' - 3y_p' + 2y_p = (4A - 6A + 2A)e^{2x} = 0\)。等于 \(e^{2x}\) 显然解不出 \(A\)——左边永远是零!

问题出在哪里?齐次方程 \(y'' - 3y' + 2y = 0\) 的特征根是 \(r = 1, 2\),所以 \(e^{2x}\) 本身就是齐次解。把齐次解代入算子 \(L\),结果当然是零;把它常数倍 \(A e^{2x}\) 代入,结果还是零。我们必须改试解的形式。

共振修正规则:如果按通常规则猜出的试解恰好是齐次解(对应单根),把试解乘以 \(x\)。如果对应的是齐次方程的重根,则乘以 \(x^2\)。

所以这里改猜 \(y_p = A x e^{2x}\)。计算:

\[ y_p' = A e^{2x} + 2 A x e^{2x} = A(1 + 2x)e^{2x}, \] \[ y_p'' = 2A e^{2x} + A(1 + 2x)(2)e^{2x} = A(2 + 2 + 4x)e^{2x} = A(4 + 4x)e^{2x}. \]

代入:

\[ y_p'' - 3y_p' + 2y_p = A(4 + 4x)e^{2x} - 3A(1 + 2x)e^{2x} + 2A x e^{2x} \] \[ = A e^{2x}\bigl[(4 + 4x) - 3(1 + 2x) + 2x\bigr] = A e^{2x}(4 + 4x - 3 - 6x + 2x) = A e^{2x}(1 + 0) = A e^{2x}. \]

等于 \(e^{2x}\),所以 \(A = 1\),特解 \(y_p = x e^{2x}\)。

整体通解:

\[ y = c_1 e^x + c_2 e^{2x} + x e^{2x}. \]

"乘以 \(x\)"这一招的几何含义:由于 \(e^{2x}\) 已经在齐次解的零空间里,我们必须从一个线性无关于齐次解的方向猜起。\(x e^{2x}\) 与 \(e^{2x}\) 线性无关,所以它能把方程"撑开"。这与第 5 章重根情形的"乘以 \(x\)"在思路上是同构的——都是"被堵在零空间外面"时的同一种突破手段。

看一个重根的共振例子。求 \(y'' - 4y' + 4y = e^{2x}\) 的特解。齐次特征 \(r^2 - 4r + 4 = (r-2)^2 = 0\),重根 \(r = 2\),所以 \(y_h = (c_1 + c_2 x)e^{2x}\)。这意味着 \(e^{2x}\) 和 \(x e^{2x}\) 都是齐次解。试解必须再升一次,猜 \(y_p = A x^2 e^{2x}\)。

\[ y_p' = 2A x e^{2x} + 2A x^2 e^{2x} = 2A(x + x^2)e^{2x}, \] \[ y_p'' = 2A(1 + 2x)e^{2x} + 2A(x + x^2)(2)e^{2x} = 2A(1 + 2x + 2x + 2x^2)e^{2x} = 2A(1 + 4x + 2x^2)e^{2x}. \]

代入 \(y_p'' - 4y_p' + 4y_p\):

\[ 2A(1 + 4x + 2x^2)e^{2x} - 8A(x + x^2)e^{2x} + 4A x^2 e^{2x} \] \[ = 2A e^{2x}\bigl[(1 + 4x + 2x^2) - 4(x + x^2) + 2x^2\bigr] \] \[ = 2A e^{2x}\bigl[1 + 4x + 2x^2 - 4x - 4x^2 + 2x^2\bigr] = 2A e^{2x} \cdot 1 = 2A e^{2x}. \]

等于 \(e^{2x}\),所以 \(A = 1/2\),\(y_p = \tfrac{1}{2} x^2 e^{2x}\)。

"乘以 \(x^2\)" 把试解推到与齐次空间(\(e^{2x}\) 与 \(x e^{2x}\))同时线性无关的位置。这种"层层升幂"的逻辑可以无限推广——如果某个 \(e^{rx}\) 是 \(k\) 重根,对应的试解就是 \(A x^k e^{rx}\)。

7.3 待定系数法的对照表与例题

把上一节散落的规则整理成一张对照表。设我们要解 \(L[y] = g(x)\)。下表中假定试解还不是齐次解;若是,把试解乘以 \(x\)(单根)或 \(x^2\)(重根)。

\(g(x)\) 的形态试解 \(y_p\)
常数 \(K\)\(A\)
多项式 \(P_n(x)\)(次数 \(n\))\(A_n x^n + A_{n-1} x^{n-1} + \cdots + A_0\)(全次幂都写)
\(K e^{\alpha x}\)\(A e^{\alpha x}\)
\(K\sin\beta x\) 或 \(K\cos\beta x\)\(A\sin\beta x + B\cos\beta x\)
\(P_n(x) e^{\alpha x}\)\(\bigl(A_n x^n + \cdots + A_0\bigr) e^{\alpha x}\)
\(P_n(x)\sin\beta x\) 或 \(P_n(x)\cos\beta x\)\(\bigl(A_n x^n + \cdots + A_0\bigr)\sin\beta x + \bigl(B_n x^n + \cdots + B_0\bigr)\cos\beta x\)
\(K e^{\alpha x}\sin\beta x\) 或 \(K e^{\alpha x}\cos\beta x\)\(e^{\alpha x}(A\sin\beta x + B\cos\beta x)\)
\(P_n(x) e^{\alpha x}\sin\beta x\) 或 \(\cos\beta x\)\(e^{\alpha x}\bigl[(A_n x^n + \cdots + A_0)\sin\beta x + (B_n x^n + \cdots + B_0)\cos\beta x\bigr]\)

使用步骤,反复念叨直到自动化:

  1. 把齐次方程的特征方程解出来,记下根。
  2. 看 \(g(x)\) 的形态,从上表查到对应的试解。
  3. 检查共振:试解中的指数/三角因子是否对应于齐次方程的特征根?若对应于单根,把试解乘以 \(x\);若对应于重根,乘以 \(x^2\)。具体地,若 \(g\) 含 \(e^{\alpha x}\) 而 \(\alpha\) 是单根,乘 \(x\);若 \(g\) 含 \(e^{\alpha x}\sin\beta x\) 或 \(\cos\beta x\) 而 \(\alpha \pm \beta i\) 是单复根,乘 \(x\)。
  4. 把试解代入方程,展开,比较系数,解出未知常数。
  5. 写下 \(y = y_h + y_p\)。

下面跑几道完整例题,把表里的不同行各踩一遍。

7.3.1 例:多项式 × 指数

解 \(y'' - y = x e^{2x}\),求通解。

齐次:\(r^2 - 1 = 0\),根 \(r = \pm 1\),所以 \(y_h = c_1 e^x + c_2 e^{-x}\)。

右端 \(g = x e^{2x}\)。\(\alpha = 2\) 不是齐次根(只有 \(\pm 1\) 是),无共振。从表中查到试解

\[ y_p = (Ax + B)e^{2x}. \]

计算:\(y_p = (Ax + B)e^{2x}\),\(y_p' = A e^{2x} + 2(Ax + B)e^{2x} = [A + 2(Ax + B)]e^{2x} = (2Ax + A + 2B)e^{2x}\),\(y_p'' = 2A e^{2x} + 2(2Ax + A + 2B)e^{2x} = (2A + 4Ax + 2A + 4B)e^{2x} = (4Ax + 4A + 4B)e^{2x}\)。

\[ y_p'' - y_p = (4Ax + 4A + 4B)e^{2x} - (Ax + B)e^{2x} = \bigl[(4A - A)x + (4A + 4B - B)\bigr]e^{2x} \] \[ = (3Ax + 4A + 3B)e^{2x}. \]

等于 \(x e^{2x}\):比较系数 \(3A = 1\) 得 \(A = 1/3\);\(4A + 3B = 0\) 得 \(B = -4A/3 = -4/9\)。所以

\[ y_p = \bigl(\tfrac{x}{3} - \tfrac{4}{9}\bigr)e^{2x}. \]

通解 \(y = c_1 e^x + c_2 e^{-x} + \bigl(x/3 - 4/9\bigr)e^{2x}\)。

7.3.2 例:三角共振

解 \(y'' + 4y = \cos 2x\),求通解。

齐次:\(r^2 + 4 = 0\),根 \(r = \pm 2i\),\(y_h = c_1\cos 2x + c_2\sin 2x\)。

右端 \(g = \cos 2x\)。按表本应猜 \(y_p = A\sin 2x + B\cos 2x\)。但是注意——这正是齐次解!\(\alpha + \beta i = 0 + 2i\) 是齐次单复根,所以发生共振。修正:

\[ y_p = x(A\sin 2x + B\cos 2x). \]

计算:\(y_p = Ax\sin 2x + Bx\cos 2x\)。求导(乘积法则):

\[ y_p' = A\sin 2x + 2Ax\cos 2x + B\cos 2x - 2Bx\sin 2x. \] \[ y_p'' = 2A\cos 2x + 2A\cos 2x - 4Ax\sin 2x - 2B\sin 2x - 2B\sin 2x - 4Bx\cos 2x \] \[ = 4A\cos 2x - 4Ax\sin 2x - 4B\sin 2x - 4Bx\cos 2x. \]

所以

\[ y_p'' + 4y_p = 4A\cos 2x - 4Ax\sin 2x - 4B\sin 2x - 4Bx\cos 2x + 4Ax\sin 2x + 4Bx\cos 2x \] \[ = 4A\cos 2x - 4B\sin 2x. \]

等于 \(\cos 2x\):比较系数 \(4A = 1\) 得 \(A = 1/4\);\(-4B = 0\) 得 \(B = 0\)。所以

\[ y_p = \tfrac{1}{4} x\sin 2x. \]

通解 \(y = c_1\cos 2x + c_2\sin 2x + \tfrac{1}{4} x\sin 2x\)。

注意特解里有 \(x\sin 2x\) 这种"振幅随 \(x\) 线性增长"的项。这正是物理学家所说的共振现象——驱动频率与系统固有频率相等时,响应不再是稳定振荡,而是振幅无限增长!8.4 节会回到这件事并作更细致的讨论。

7.3.3 例:叠加原理

解 \(y'' + y = 4 e^x + 3\sin 2x\),求通解。

齐次:\(r^2 + 1 = 0\),\(r = \pm i\),\(y_h = c_1\cos x + c_2\sin x\)。

右端是两项,用叠加原理拆开。

第一部分,\(g_1 = 4 e^x\):\(\alpha = 1\) 不是齐次根,试解 \(y_{p,1} = A e^x\)。代入 \(y'' + y\):\(A e^x + A e^x = 2A e^x = 4 e^x\),\(A = 2\)。所以 \(y_{p,1} = 2 e^x\)。

第二部分,\(g_2 = 3\sin 2x\):\(\alpha + \beta i = 0 + 2i\) 不是齐次根(齐次是 \(\pm i\)),无共振。试解 \(y_{p,2} = C\sin 2x + D\cos 2x\)。

\[ y_{p,2}'' + y_{p,2} = -4C\sin 2x - 4D\cos 2x + C\sin 2x + D\cos 2x = -3C\sin 2x - 3D\cos 2x. \]

等于 \(3\sin 2x\):\(-3C = 3\)、\(-3D = 0\),所以 \(C = -1\)、\(D = 0\)。\(y_{p,2} = -\sin 2x\)。

合并:\(y_p = y_{p,1} + y_{p,2} = 2 e^x - \sin 2x\)。通解

\[ y = c_1\cos x + c_2\sin x + 2 e^x - \sin 2x. \]

叠加原理让我们把"看起来要一次猜五个未定系数"的题分成两道独立小题,每道只猜一两个常数。这种"分而治之"的策略,在 \(g\) 是几项之和时极其有用。

7.3.4 例:多项式 × 三角

解 \(y'' - 2y' + y = (x - 1)\cos 3x\),求特解。

齐次:\(r^2 - 2r + 1 = (r - 1)^2 = 0\),重根 \(r = 1\),\(y_h = (c_1 + c_2 x) e^x\)。

右端 \(g = (x - 1)\cos 3x\)。这是"一次多项式 × 余弦"。\(\alpha + \beta i = 0 + 3i\) 不是齐次根(齐次是实数 1 重根),无共振。从表中查试解

\[ y_p = (Ax + B)\sin 3x + (Cx + D)\cos 3x. \]

代入会得到一个比较冗长的代数推导,但思路完全直接:逐项求导,合并 \(\sin 3x\) 与 \(\cos 3x\) 系数,设它们分别等于右端对应项,得到 4 个方程解 4 个未知数。

具体来算。\(y_p = (Ax + B)\sin 3x + (Cx + D)\cos 3x\)。

\[ y_p' = A\sin 3x + 3(Ax + B)\cos 3x + C\cos 3x - 3(Cx + D)\sin 3x \] \[ = [A - 3(Cx + D)]\sin 3x + [3(Ax + B) + C]\cos 3x \] \[ = (-3Cx + A - 3D)\sin 3x + (3Ax + 3B + C)\cos 3x. \] \[ y_p'' = -3C\sin 3x + 3(-3Cx + A - 3D)\cos 3x + 3A\cos 3x - 3(3Ax + 3B + C)\sin 3x \] \[ = [-3C - 9Ax - 9B - 3C]\sin 3x + [3A - 9Cx + 3A - 9D + 3A \text{wait... let me redo}] \]

把这部分小心地重做。\(y_p' = (A - 3D - 3Cx)\sin 3x + (3B + C + 3Ax)\cos 3x\)。再求一次:

\[ y_p'' = (-3C)\sin 3x + 3(A - 3D - 3Cx)\cos 3x + (3A)\cos 3x - 3(3B + C + 3Ax)\sin 3x. \]

合并:

\[ y_p'' = \bigl[-3C - 3(3B + C + 3Ax)\bigr]\sin 3x + \bigl[3(A - 3D - 3Cx) + 3A\bigr]\cos 3x \] \[ = (-9Ax - 9B - 6C)\sin 3x + (-9Cx + 6A - 9D)\cos 3x. \]

现在把 \(y_p'' - 2 y_p' + y_p\) 拼起来。

\(\sin 3x\) 系数:\((-9Ax - 9B - 6C) - 2(-3Cx + A - 3D) + (Ax + B)\) \(= -9Ax - 9B - 6C + 6Cx - 2A + 6D + Ax + B\) \(= (-9A + 6C + A)x + (-9B - 6C - 2A + 6D + B)\) \(= (-8A + 6C)x + (-8B - 6C - 2A + 6D)\)。

\(\cos 3x\) 系数:\((-9Cx + 6A - 9D) - 2(3Ax + 3B + C) + (Cx + D)\) \(= -9Cx + 6A - 9D - 6Ax - 6B - 2C + Cx + D\) \(= (-9C - 6A + C)x + (6A - 9D - 6B - 2C + D)\) \(= (-6A - 8C)x + (6A - 6B - 2C - 8D)\)。

右端 \(g = (x - 1)\cos 3x = x\cos 3x - \cos 3x\),所以 \(\sin 3x\) 系数全为零,\(\cos 3x\) 中 \(x\) 的系数为 1,常数为 \(-1\)。比较:

  • \(\sin 3x\) 中 \(x\) 项:\(-8A + 6C = 0\),即 \(C = 4A/3\)。
  • \(\sin 3x\) 中常数:\(-8B - 6C - 2A + 6D = 0\)。
  • \(\cos 3x\) 中 \(x\) 项:\(-6A - 8C = 1\)。
  • \(\cos 3x\) 中常数:\(6A - 6B - 2C - 8D = -1\)。

把 \(C = 4A/3\) 代入第三式:\(-6A - 32A/3 = 1\),即 \(-18A/3 - 32A/3 = -50A/3 = 1\),所以 \(A = -3/50\)。\(C = 4(-3/50)/3 = -4/50 = -2/25\)。

从第二式:\(-8B - 6(-2/25) - 2(-3/50) + 6D = 0\),即 \(-8B + 12/25 + 3/25 + 6D = 0\),即 \(-8B + 6D + 15/25 = 0\),即 \(-8B + 6D = -3/5\)。

从第四式:\(6(-3/50) - 6B - 2(-2/25) - 8D = -1\),即 \(-18/50 - 6B + 4/25 - 8D = -1\),即 \(-9/25 + 8/50 - 6B - 8D = -1\),化为 \(-9/25 + 4/25 - 6B - 8D = -1\),即 \(-5/25 - 6B - 8D = -1\),即 \(-1/5 - 6B - 8D = -1\),所以 \(-6B - 8D = -4/5\),即 \(6B + 8D = 4/5\)。

从两式 \(\{-8B + 6D = -3/5,\ 6B + 8D = 4/5\}\) 解 \(B, D\)。乘 3 与乘 4 后相加:\(-24B + 18D = -9/5\),\(24B + 32D = 16/5\),相加得 \(50D = 7/5\),\(D = 7/250\)。代回 \(6B + 8(7/250) = 4/5\),\(6B + 56/250 = 200/250\),\(6B = 144/250\),\(B = 24/250 = 12/125\)。

所以 \(A = -3/50\)、\(B = 12/125\)、\(C = -2/25\)、\(D = 7/250\),

\[ y_p = \bigl(-\tfrac{3}{50}x + \tfrac{12}{125}\bigr)\sin 3x + \bigl(-\tfrac{2}{25}x + \tfrac{7}{250}\bigr)\cos 3x. \]

这道题展示了待定系数法在"\(g\) 比较复杂"时的全部代数痛苦——需要解一个 4 元线性方程组。但思路始终一样:猜形式,代入,比较,解线性方程组。 算式不优雅,逻辑不复杂。

7.3.5 例:常数右端

解 \(y'' + 3y' + 2y = 6\),求通解。

齐次:\(r^2 + 3r + 2 = (r+1)(r+2) = 0\),\(y_h = c_1 e^{-x} + c_2 e^{-2x}\)。

右端是常数 6,猜 \(y_p = A\)。代入:\(0 + 0 + 2A = 6\),\(A = 3\)。所以 \(y_p = 3\),通解

\[ y = c_1 e^{-x} + c_2 e^{-2x} + 3. \]

"常数右端"是最简单的非齐次情形,工程上叫做稳态——当瞬态部分 \(c_1 e^{-x} + c_2 e^{-2x}\) 衰减到零后,系统稳定在 \(y = 3\) 这个"新平衡"上。这正是为什么 \(y_p\) 里的 3 是有物理意义的:它就是右端 6 除以"刚度系数" 2,描述系统对常值输入的最终响应。

7.3.6 例:重根共振 + 三角

解 \(y'' + 9y = 4\sin 3x\),求通解。

齐次:\(r^2 + 9 = 0\),\(r = \pm 3i\),\(y_h = c_1\cos 3x + c_2\sin 3x\)。

右端 \(g = 4\sin 3x\)。\(\alpha + \beta i = 0 + 3i\) 是齐次单复根(注意:对实系数方程,共轭复根 \(\pm 3i\) 算一对单根,不是重根)。共振,试解乘 \(x\):

\[ y_p = x(A\sin 3x + B\cos 3x). \]

(参考 7.3.2 节,模式完全一样,只换成了 \(\sin\)。)计算与前面类似,得

\[ y_p'' + 9y_p = -6 B\sin 3x + 6A\cos 3x. \]

(读者可以自己核实——把 7.3.2 的代数从头跑一遍。)等于 \(4\sin 3x\):\(-6B = 4\),\(B = -2/3\);\(6A = 0\),\(A = 0\)。所以 \(y_p = -\tfrac{2}{3} x\cos 3x\),通解

\[ y = c_1\cos 3x + c_2\sin 3x - \tfrac{2}{3} x\cos 3x. \]

同样,共振导致特解里出现 \(x\cos 3x\) 这种振幅线性增长的项。物理意义在 8.4 节展开。

7.4 参数变易法

待定系数法漂亮但有局限:它只对"求导后形式不变"的 \(g\)(即多项式、指数、三角及其乘积)有效。如果 \(g(x) = \tan x\)、\(\ln x\)、\(\sec x\) 这种东西,猜什么形式都不行。参数变易法(variation of parameters)是一种普适的方法,只要齐次方程的两个基已经求出,无论 \(g\) 多么古怪,都能写出 \(y_p\) 的显式积分公式。代价是有时候要算的积分本身很难,但至少思路是不变的。

7.4.1 推导

设齐次方程 \(L[y] = 0\) 的两个线性无关解为 \(y_1, y_2\),其通解 \(y_h = c_1 y_1 + c_2 y_2\)。参数变易的思路是:把常数 \(c_1, c_2\) 替换成函数 \(u_1(x), u_2(x)\),设特解形式

\[ y_p = u_1(x) y_1(x) + u_2(x) y_2(x). \]

这里有两个未知函数 \(u_1, u_2\),所以我们需要两个方程来定它们。一个方程当然来自"\(y_p\) 满足非齐次方程",但我们还有一个选择的自由度——可以再加一个方便的条件,简化代数。最常用的选择是

\[ u_1' y_1 + u_2' y_2 = 0. \tag{$*$} \]

这条选择并不强迫,只是为了让接下来的求导漂亮。具体说,先求 \(y_p'\):

\[ y_p' = u_1' y_1 + u_1 y_1' + u_2' y_2 + u_2 y_2'. \]

如果加上条件 \((*)\),\(y_p' = u_1 y_1' + u_2 y_2'\),很干净。再求一次:

\[ y_p'' = u_1' y_1' + u_1 y_1'' + u_2' y_2' + u_2 y_2''. \]

把 \(y_p, y_p', y_p''\) 代入 \(L[y_p] = a y_p'' + b y_p' + c y_p\):

\[ L[y_p] = a(u_1' y_1' + u_1 y_1'' + u_2' y_2' + u_2 y_2'') + b(u_1 y_1' + u_2 y_2') + c(u_1 y_1 + u_2 y_2). \]

把含 \(u_1\) 与含 \(u_2\) 的项分别归并:

\[ = u_1\bigl(a y_1'' + b y_1' + c y_1\bigr) + u_2\bigl(a y_2'' + b y_2' + c y_2\bigr) + a(u_1' y_1' + u_2' y_2'). \]

但 \(y_1, y_2\) 是齐次解,所以前两个括号都是零!剩下

\[ L[y_p] = a(u_1' y_1' + u_2' y_2'). \]

等于 \(g(x)\),所以

\[ u_1' y_1' + u_2' y_2' = \frac{g(x)}{a}. \tag{$**$} \]

把方程 \((*)\) 与 \((**)\) 联立:

\[ \begin{cases} y_1 u_1' + y_2 u_2' = 0,\\ y_1' u_1' + y_2' u_2' = g/a. \end{cases} \]

这是一个 2×2 线性方程组,把 \(u_1', u_2'\) 当未知。系数矩阵的行列式是Wronskian

\[ W = \begin{vmatrix} y_1 & y_2 \\ y_1' & y_2' \end{vmatrix} = y_1 y_2' - y_1' y_2. \]

因为 \(y_1, y_2\) 线性无关,\(W \ne 0\)。用 Cramer 法则解出

\[ \boxed{u_1' = -\frac{y_2 \, g(x)/a}{W} = -\frac{y_2\, g}{a W},\qquad u_2' = \frac{y_1 \, g(x)/a}{W} = \frac{y_1\, g}{a W}.} \]

对它们积分得 \(u_1, u_2\),最后写出 \(y_p = u_1 y_1 + u_2 y_2\)。

很多书把方程写成"标准形式" \(y'' + p(x) y' + q(x) y = f(x)\)(即 \(a = 1\)),那时分母里的 \(a\) 消失,公式简化为 \(u_1' = -y_2 f/W\)、\(u_2' = y_1 f/W\)。本书在做例题时也通常会先把方程除以 \(a\),让最高阶系数为 1,再用这个干净的公式。

7.4.2 第一道例题:与待定系数对照

用参数变易法解 \(y'' + y = \sin x\),并与待定系数法对照。

齐次:\(r^2 + 1 = 0\),\(y_1 = \cos x\)、\(y_2 = \sin x\)、\(y_h = c_1\cos x + c_2\sin x\)。

注意 \(g = \sin x\) 恰好是齐次解,所以这是共振情形。在待定系数法里我们会乘 \(x\) 处理;而参数变易法对共振完全无感——它走的是一条不同的逻辑通道,共振对它没有特殊影响。

计算 Wronskian:\(W = \cos x \cdot \cos x - (-\sin x)\sin x = \cos^2 x + \sin^2 x = 1\)。方程已经是标准形式 \(a = 1\)。所以

\[ u_1' = -\frac{y_2 \cdot g}{W} = -\sin x \cdot \sin x = -\sin^2 x, \] \[ u_2' = \frac{y_1 \cdot g}{W} = \cos x \cdot \sin x. \]

积分:

\[ u_1 = -\int \sin^2 x \,dx = -\int \frac{1 - \cos 2x}{2}\, dx = -\frac{x}{2} + \frac{\sin 2x}{4}. \] \[ u_2 = \int \sin x \cos x\, dx = \frac{1}{2}\int \sin 2x \, dx = -\frac{\cos 2x}{4}. \]

(积分常数可以略去,因为它们对应的项 \(C_1 \cos x + C_2 \sin x\) 已经包含在齐次解里了。)所以

\[ y_p = u_1\cos x + u_2\sin x = \Bigl(-\frac{x}{2} + \frac{\sin 2x}{4}\Bigr)\cos x + \Bigl(-\frac{\cos 2x}{4}\Bigr)\sin x \] \[ = -\frac{x}{2}\cos x + \frac{1}{4}(\sin 2x \cos x - \cos 2x \sin x) = -\frac{x}{2}\cos x + \frac{1}{4}\sin(2x - x) = -\frac{x}{2}\cos x + \frac{1}{4}\sin x. \]

右边那个 \(\frac{1}{4}\sin x\) 仍是齐次解,可以并入 \(y_h\) 里(即把 \(c_2\) 重新定义),所以本质上的 \(y_p\) 是

\[ y_p = -\tfrac{1}{2} x\cos x. \]

用待定系数法的话,猜 \(y_p = x(A\sin x + B\cos x)\) 算出 \(A = 0, B = -1/2\) 也是 \(y_p = -\tfrac{1}{2} x\cos x\)。两种方法殊途同归。

这道题展示了参数变易法的一个事实:它直接给出特解,但有时会"附赠"一些齐次解项;这些项可以直接吸收进 \(y_h\),不影响最终通解。所以做完参数变易后,看到答案里有齐次解片段,不必惊慌——直接合并即可。

7.4.3 第二道:\(g = \tan x\),待定系数法做不了

解 \(y'' + y = \tan x\)。

齐次同上:\(y_1 = \cos x\)、\(y_2 = \sin x\)、\(W = 1\)。

\(g = \tan x = \sin x/\cos x\),它不是任何"求导后形式不变"的函数,待定系数法在这里无法启动。但参数变易法照常运转:

\[ u_1' = -\sin x \tan x = -\frac{\sin^2 x}{\cos x} = -\frac{1 - \cos^2 x}{\cos x} = -\sec x + \cos x. \] \[ u_2' = \cos x \tan x = \sin x. \]

积分:

\[ u_1 = \int (\cos x - \sec x)\, dx = \sin x - \ln|\sec x + \tan x|. \] \[ u_2 = \int \sin x\, dx = -\cos x. \]

所以

\[ y_p = (\sin x - \ln|\sec x + \tan x|)\cos x + (-\cos x)\sin x \] \[ = \sin x \cos x - \cos x \ln|\sec x + \tan x| - \sin x \cos x = -\cos x \ln|\sec x + \tan x|. \]

通解:

\[ y = c_1\cos x + c_2\sin x - \cos x \ln|\sec x + \tan x|. \]

这是一个完整的解,而 \(g = \tan x\) 完全在待定系数法的能力之外。注意 \(\int \sec x\, dx = \ln|\sec x + \tan x|\) 是一个标准结果(微积分课里的"积分技巧"),如果你忘了它就该回去复习——参数变易法的难点常常不在方法本身,而在中途要算的那个积分。

7.4.4 第三道:\(g = \ln x\)

解 \(y'' - y = \ln x\)(\(x > 0\))。

齐次:\(r^2 - 1 = 0\),\(y_1 = e^x\)、\(y_2 = e^{-x}\)。Wronskian \(W = e^x(-e^{-x}) - e^x \cdot e^{-x} = -1 - 1 = -2\)。

\(g = \ln x\),又是待定系数法搞不定的。

\[ u_1' = -\frac{y_2 g}{W} = -\frac{e^{-x}\ln x}{-2} = \frac{e^{-x}\ln x}{2}. \] \[ u_2' = \frac{y_1 g}{W} = \frac{e^x \ln x}{-2} = -\frac{e^x\ln x}{2}. \]

积分 \(\int e^{-x}\ln x\,dx\) 与 \(\int e^x\ln x\,dx\) 都不是初等函数(它们涉及"指数积分函数" Ei 这种特殊函数)。所以最后我们只能写

\[ y_p = \frac{e^x}{2}\int e^{-x}\ln x\,dx - \frac{e^{-x}}{2}\int e^x\ln x\,dx \]

(注意外面那个 \(y_1 = e^x\) 与 \(y_2 = e^{-x}\) 并到积分外面了)。这看起来不像"答案",但它确实是 \(y_p\) 的显式表达——就好比 \(\int_0^x e^{-t^2}\,dt\) 是误差函数 erf 的"显式表达",虽然不能再化简但已是写得出来的最终形式。在数值计算或者在物理问题中,这种"含未算积分的表达式"完全有用——你可以代入数值积分,可以分析它的渐近行为,也可以拿它去推导别的东西。

这一题给出本章的一个重要教训:参数变易法的"通用性"不等于"算得出闭形式"。它把"找特解"还原为"算两个积分",但这两个积分本身可能没有初等表达。这是任何"完全通用"的方法都得付出的代价——通用越大,具体技巧越少。

7.4.5 第四道:\(g = \sec x\)

解 \(y'' + y = \sec x\)。

齐次:\(y_1 = \cos x\)、\(y_2 = \sin x\)、\(W = 1\)。

\[ u_1' = -\sin x \sec x = -\tan x. \] \[ u_2' = \cos x \sec x = 1. \]

积分:

\[ u_1 = -\int \tan x\,dx = \ln|\cos x|,\qquad u_2 = x. \]

所以

\[ y_p = \cos x \ln|\cos x| + x\sin x. \]

通解 \(y = c_1\cos x + c_2\sin x + \cos x \ln|\cos x| + x\sin x\)。

这是一个"看上去很神奇"的特解,因为它把 \(\ln|\cos x|\) 和 \(x\sin x\) 这两种表面上无关的函数组合在一起,而且确实满足 \(y'' + y = \sec x\)(读者可以通过求二阶导验证)。这种"形式上的复杂性"——你绝不可能凭直觉猜出来——正是参数变易法的价值所在。

7.4.6 第五道:\(g = e^x/x\)

解 \(y'' - 2y' + y = e^x/x\)(\(x > 0\))。

齐次:\(r^2 - 2r + 1 = (r-1)^2 = 0\),重根,\(y_1 = e^x\)、\(y_2 = x e^x\)、\(y_h = (c_1 + c_2 x)e^x\)。Wronskian \(W = e^x(e^x + x e^x) - e^x \cdot x e^x = e^{2x}\)。

\(g = e^x/x\)。这个右端如果用待定系数法,就算包括"指数 × 多项式"也覆盖不了它(\(1/x\) 不是多项式)。

\[ u_1' = -\frac{x e^x \cdot e^x/x}{e^{2x}} = -\frac{e^{2x}}{e^{2x}} = -1. \] \[ u_2' = \frac{e^x \cdot e^x/x}{e^{2x}} = \frac{1}{x}. \]

积分:

\[ u_1 = -x,\qquad u_2 = \ln|x| = \ln x. \]

所以

\[ y_p = (-x)e^x + (\ln x)(x e^x) = -x e^x + x e^x \ln x = x e^x(\ln x - 1). \]

通解 \(y = (c_1 + c_2 x)e^x + x e^x(\ln x - 1)\)。注意 \(-x e^x\) 那项可以并入 \(c_2 x e^x\) 里,所以也可写成 \(y = (c_1 + c_2 x)e^x + x e^x \ln x\)(常数 \(c_2\) 已经吸收了 \(-1\))。两种写法等价。

7.5 两种方法何时用哪种

把两种方法放到一起对比:

待定系数法参数变易法
适用范围\(g\) 是多项式 / 指数 / 三角及其乘积任意连续 \(g\)
步骤猜形式 → 代入 → 解线性方程组求 Wronskian → 算两个积分
难点共振时要乘 \(x\) 修正;系数代数有时复杂积分本身可能很难甚至非初等
对常系数齐次方程方便同样适用,但通常更费力
对变系数齐次方程不直接适用(需对方程做特殊变换)仍可用,只要先求出 \(y_1, y_2\)

实战中的选择策略:

  1. 先看 \(g\)。如果 \(g\) 落在表 7.3 的范围内(多项式、指数、正/余弦、它们的乘积或叠加),优先用待定系数法——它的代数虽然有时麻烦,但思路简单,不涉及不熟悉的积分。
  2. 注意共振。猜形式后必须检查它是不是齐次解;若是,乘 \(x\) 或 \(x^2\)。这一步是新手最容易漏的,养成"猜完试解后画一秒钟齐次根"的习惯。
  3. 否则用参数变易法。\(g\) 含 \(\tan, \sec, \ln, 1/x, 1/(1+x^2)\) 等"非标准"形式时,直接走参数变易公式。
  4. 线性叠加可拆。若 \(g = g_1 + g_2\) 且 \(g_1\) 是"标准型"而 \(g_2\) 不是,把它拆开:对 \(g_1\) 用待定系数,对 \(g_2\) 用参数变易,最后相加。
  5. 变系数方程(\(a, b, c\) 是 \(x\) 的函数)只能用参数变易法——前提是齐次方程的两个基已经求出。怎么求齐次基是另一回事(常用幂级数解法,见第 11、12 章)。

来一道分类题作收尾。判断下列 \(g(x)\) 在解 \(y'' - y = g(x)\) 时各自适合哪种方法。

  • \(g = e^x\):待定系数,共振(齐次根 \(r = 1\)),试解 \(A x e^x\)。
  • \(g = e^{2x} + 3\):待定系数,无共振,试解 \(A e^{2x} + B\)。
  • \(g = x \cos x\):待定系数,无共振,试解 \((Ax + B)\sin x + (Cx + D)\cos x\)。
  • \(g = e^x \sin x\):待定系数,无共振(\(\alpha + \beta i = 1 + i \ne \pm 1\)),试解 \(e^x(A\sin x + B\cos x)\)。
  • \(g = 1/x\):待定系数无效,用参数变易法。
  • \(g = \arctan x\):待定系数无效,参数变易法。
  • \(g = e^x/(1 + x^2)\):待定系数无效,参数变易法,且积分会涉及不可初等表达的形式。

分类的要诀就一条:右端属不属于"求导封闭"的那个小族(多项式、指数、三角及它们的乘积)?属——待定系数;不属——参数变易。学完本章,你已经掌握了二阶常系数线性方程的全部解法工具。第 8 章会立刻把它们投入到物理实战中——弹簧、电路、摆——你将看到这两种方法在真实场景里的用法,以及"共振"在物理学里的真实面孔。
