第 8 章 二阶方程的物理应用

前几章我们做了大量代数:特征方程、判别式、复根、待定系数、参数变易。所有这些都是工具——它们的存在意义只在于解决问题。本章把工具拿出来用。我们将看到第 5–7 章的那些抽象结论,在弹簧、电路、摆这些日常物体上,以多么直接、多么具体的方式呈现出来。

更重要的是,本章会让你看到一个非凡的事实:同一个方程 \(m y'' + c y' + k y = F(t)\) 同时描述弹簧的振动、电路里的电荷,以及小角度的摆动。物理对象完全不同——一个是机械系统、一个是电磁现象、一个是引力系统——但它们服从同一条方程。这种"数学结构的统一"在自然科学里反复出现,而二阶常系数线性方程是其中最经典的例子之一。学完本章,你就理解了为什么微分方程被认为是所有理工科的通用语言。

本章覆盖:

  • 8.1 弹簧-质量系统(无阻尼,自由振动);
  • 8.2 阻尼弹簧——三种阻尼对应第 6 章的三种情形;
  • 8.3 强迫振动——把驱动力放进右端;
  • 8.4 共振:振幅函数 \(R(\omega)\) 在 \(\omega \to \omega_0\) 时的剧烈放大;
  • 8.5 RLC 电路与弹簧的完整类比;
  • 8.6 单摆——从非线性原方程到小角度的线性近似;
  • 8.7 拍现象——两个接近频率的叠加产生的"调幅"。

8.1 弹簧-质量系统(无阻尼自由振动)

一根弹簧的一端固定在天花板上,另一端挂一个质量为 \(m\) 的物体。设 \(y(t)\) 为物体在时间 \(t\) 时偏离平衡位置的位移(向下为正)。当物体在平衡位置时弹簧的弹力恰好抵消重力,所以这是一个"零位置";偏离平衡时,弹簧产生一个恢复力把物体往平衡位置拉。

实验事实(胡克定律):弹簧的恢复力 \(F_{\text{spring}}\) 与位移成正比,方向相反:

\[ F_{\text{spring}} = -k y, \]

其中 \(k > 0\) 是弹性系数(spring constant),单位 N/m。负号体现"恢复"——\(y > 0\)(物体在平衡下方)时弹簧往上拉(\(F < 0\));\(y < 0\) 时往下推(\(F > 0\))。

(为什么把重力消掉了?如果你测的是物体距弹簧自然长度的位移而非平衡位置,方程会带一个常数项 \(-mg\) 在右端,但常数项的特解就是把零点移动一下——也就是平衡位置——之后又回到 \(F = -ky\) 的纯净形式。所以本章默认我们已经在平衡位置上建立坐标系。)

牛顿第二定律 \(F = ma = m y''\) 给出

\[ m y'' = -k y \quad\Longleftrightarrow\quad y'' + \frac{k}{m} y = 0. \]

定义 \(\omega_0 = \sqrt{k/m}\),方程变成

\[ \boxed{y'' + \omega_0^2 y = 0.} \]

这是简谐振子方程。从第 6 章我们知道它的根 \(r = \pm i\omega_0\),通解

\[ y(t) = c_1\cos\omega_0 t + c_2\sin\omega_0 t = R\cos(\omega_0 t - \phi), \]

其中 \(R = \sqrt{c_1^2 + c_2^2}\)、\(\tan\phi = c_2/c_1\)。这就是简谐运动:周期 \(T = 2\pi/\omega_0 = 2\pi\sqrt{m/k}\),振幅 \(R\) 由初值决定。

物理解读:周期 \(T\) 不依赖于振幅。无论你把物体拉离平衡位置 1 厘米还是 10 厘米,振动一次的时间都是同一个值——这是简谐振动最迷人的性质,也是钟摆能用于计时的基础(在小角度近似下)。

看一道题:一个 0.5 kg 的物体挂在 \(k = 8\) N/m 的弹簧上,从平衡位置往下拉 0.2 m 后以 0.4 m/s 的初速度向上推开。求位置 \(y(t)\) 及振幅。

\(\omega_0 = \sqrt{8/0.5} = 4\) rad/s。方程 \(y'' + 16 y = 0\),通解 \(y = c_1\cos 4t + c_2\sin 4t\)。初值:\(y(0) = 0.2\),所以 \(c_1 = 0.2\)。\(y'(0) = -0.4\)(向上为负),所以 \(4 c_2 = -0.4\),\(c_2 = -0.1\)。所以

\[ y(t) = 0.2\cos 4t - 0.1\sin 4t. \]

振幅 \(R = \sqrt{0.04 + 0.01} = \sqrt{0.05} = \sqrt{5}/10 \approx 0.224\) m。\(\tan\phi = -0.1/0.2 = -0.5\),\(c_1 > 0\) 故 \(\phi = \arctan(-0.5) \approx -0.4636\) 弧度。所以

\[ y(t) = 0.224\cos(4t + 0.464). \]

振动周期 \(T = \pi/2 \approx 1.571\) 秒。注意振幅 \(0.224 > 0.2\) 是合理的——初始位移 0.2 m 加上向上的初速度让物体在另一侧能去到比 0.2 m 更远的位置。

8.2 阻尼弹簧

真实的弹簧有摩擦——空气阻力、弹簧本身的内耗、悬挂处的摩擦。这些综合起来一般可以近似为一个正比于速度的阻力(对低速、缓慢运动这是合理近似):

\[ F_{\text{damp}} = -c y', \]

其中 \(c > 0\) 是阻尼系数(damping coefficient),单位 Ns/m。负号表示"阻力总是反对运动方向"。

把它加进牛顿方程:

\[ m y'' = -k y - c y' \quad\Longleftrightarrow\quad m y'' + c y' + k y = 0. \]

这正是 6.5 节那个"标准方程"。三种情形:

  • \(c^2 < 4 m k\) 欠阻尼,通解 \(y = R e^{-(c/2m)t}\cos(\beta t - \phi)\),其中 \(\beta = \sqrt{4mk - c^2}/(2m)\)。振荡但振幅指数衰减。
  • \(c^2 = 4 m k\) 临界阻尼,\(y = (c_1 + c_2 t)e^{-(c/2m)t}\)。不振荡,以"恰到好处"的速度回到平衡。
  • \(c^2 > 4 m k\) 过阻尼,\(y = c_1 e^{r_1 t} + c_2 e^{r_2 t}\)(\(r_1, r_2\) 都是负实数)。不振荡且回归比临界还慢。

我们已经在 6.5 节走过细节,此处只补一个工程上的视角。阻尼比(damping ratio)定义为

\[ \zeta = \frac{c}{2\sqrt{mk}}. \]

它把三种情形归一化:\(\zeta < 1\) 欠阻尼,\(\zeta = 1\) 临界,\(\zeta > 1\) 过阻尼。在此约定下,系统的特征根可以漂亮地写成 \(r = -\zeta\omega_0 \pm \omega_0\sqrt{\zeta^2 - 1}\),其中 \(\omega_0 = \sqrt{k/m}\) 是无阻尼固有角频率。在欠阻尼情形,\(\sqrt{\zeta^2 - 1}\) 是虚数,根写成 \(-\zeta\omega_0 \pm i \omega_0\sqrt{1 - \zeta^2}\),所以

  • 衰减率 \(\alpha = -\zeta\omega_0\),即 \(\zeta\) 越大衰减越快;
  • 振荡频率 \(\beta = \omega_0\sqrt{1 - \zeta^2}\),叫阻尼角频率;它比无阻尼角频率 \(\omega_0\) 略小,因为阻尼"拖慢"了振动。

当 \(\zeta\) 很小(轻阻尼)时 \(\beta \approx \omega_0\);当 \(\zeta \to 1\)(临界)时 \(\beta \to 0\)——振动周期发散到无穷,所以在临界阻尼下系统真的"不振荡"。这给了"临界"一个生动的解读:它正是振荡周期开始无限增长、终于消失的那个临界阻尼水平。

例题:\(m = 1\)、\(k = 100\)、\(c = 12\)。则 \(\omega_0 = 10\),\(\zeta = 12/(2 \cdot 10) = 0.6\),欠阻尼。\(\beta = 10\sqrt{1 - 0.36} = 10\sqrt{0.64} = 8\)。所以阻尼角频率是 8 rad/s,周期 \(2\pi/8 = \pi/4 \approx 0.785\) 秒。衰减率 \(\alpha = -6\),时间常数 \(1/6 \approx 0.167\) 秒——大约六分之一秒后振幅降到 \(1/e \approx 36.8\%\)。这个系统将在大约一秒内振荡几下后基本停止。

8.3 强迫振动

把一个驱动外力 \(F(t)\) 接到方程的右端:

\[ m y'' + c y' + k y = F(t). \]

最有意思也最具实际意义的情形是周期驱动,即 \(F(t) = F_0 \cos\omega t\)(也可以写成 \(\sin\),只差相位)。\(\omega\) 是驱动角频率(externally imposed),\(F_0\) 是驱动幅度。这种情形对应的物理图像是:有人以稳定的频率 \(\omega\) 摇晃弹簧的悬挂端,问弹簧会怎么响应。

方程

\[ m y'' + c y' + k y = F_0 \cos\omega t. \]

解的结构 \(y = y_h + y_p\)。\(y_h\) 是齐次解,我们已经知道:在欠阻尼情形下它指数衰减,所以经过一段时间后可以忽略。这部分叫瞬态(transient)——它依赖于初始条件,但会消失。\(y_p\) 是特解,它由 \(F_0\cos\omega t\) 决定,长期持续。这部分叫稳态(steady state)——长期来看,系统的位移就是 \(y_p\),与初始条件无关,只与驱动力有关。

下面用待定系数法求 \(y_p\)。设 \(\omega \ne \omega_0\)(暂时排除共振)。试解 \(y_p = A\cos\omega t + B\sin\omega t\)。求导:\(y_p' = -A\omega\sin\omega t + B\omega\cos\omega t\),\(y_p'' = -A\omega^2\cos\omega t - B\omega^2\sin\omega t\)。代入:

\[ m(-A\omega^2\cos - B\omega^2\sin) + c(-A\omega\sin + B\omega\cos) + k(A\cos + B\sin) \]

(\(\cos\omega t\) 和 \(\sin\omega t\) 略去自变量 \(\omega t\) 节省空间)。整理 \(\cos\omega t\) 项:

\[ (-m\omega^2 + k)A + c\omega B = (k - m\omega^2)A + c\omega B. \]

\(\sin\omega t\) 项:

\[ (-m\omega^2 + k)B - c\omega A = (k - m\omega^2)B - c\omega A. \]

等于 \(F_0\cos\omega t + 0\cdot\sin\omega t\),所以

\[ \begin{cases} (k - m\omega^2)A + c\omega B = F_0,\\ -c\omega A + (k - m\omega^2)B = 0. \end{cases} \]

用 Cramer 法则。系数行列式 \(D = (k - m\omega^2)^2 + (c\omega)^2 > 0\)(只要不同时 \(c = 0\) 与 \(\omega = \omega_0\),否则就是无阻尼共振,后面 8.4 单独处理)。所以

\[ A = \frac{(k - m\omega^2) F_0}{D},\qquad B = \frac{c\omega F_0}{D}. \]

把 \(y_p\) 改写为振幅-相位形式 \(y_p = R\cos(\omega t - \phi)\)。振幅

\[ \boxed{R(\omega) = \sqrt{A^2 + B^2} = \frac{F_0}{\sqrt{(k - m\omega^2)^2 + (c\omega)^2}}.} \]

相位 \(\tan\phi = B/A = c\omega/(k - m\omega^2)\)。

这就是稳态响应的振幅函数。它把"输入信号 \(F_0\cos\omega t\)"映成"输出响应 \(R(\omega)\cos(\omega t - \phi)\)"——同样的频率,但振幅按 \(R(\omega)/F_0\) 这个增益变化,且相位滞后 \(\phi\)。整个 \(R(\omega)\) 与 \(\phi(\omega)\) 的曲线刻画了系统对各种频率的"频率响应",是工程学里所有信号分析的起点。

把振幅化简一下,引入 \(\omega_0 = \sqrt{k/m}\) 与 \(\zeta = c/(2\sqrt{mk})\):

\[ R(\omega) = \frac{F_0/k}{\sqrt{(1 - \omega^2/\omega_0^2)^2 + (2\zeta\omega/\omega_0)^2}}. \]

这就是工程书上常画的"幅频特性"曲线。\(F_0/k\) 是静态变形(假设力常驻不动,弹簧会被压多远),后面那个根号是动态放大因子。

例题:\(m = 1\)、\(c = 0.4\)、\(k = 4\),驱动 \(F(t) = 5\cos 2t\)。求稳态响应。

\(\omega = 2\),\(\omega_0 = 2\),\(\zeta = 0.4/(2\cdot 2) = 0.1\)。注意 \(\omega = \omega_0\),即接近共振——但因为 \(c \ne 0\),仍可以代公式。

\[ R = \frac{5}{\sqrt{(4 - 4)^2 + (0.4 \cdot 2)^2}} = \frac{5}{0.8} = 6.25. \]

所以稳态振幅是 6.25——比静态变形 \(F_0/k = 5/4 = 1.25\) 大了 5 倍!这就是共振附近的振幅放大。下一节将正式讨论。

相位 \(\tan\phi = 0.4 \cdot 2/(4 - 4) = \infty\),所以 \(\phi = \pi/2\)。也就是输出信号的峰值出现在驱动信号峰值之后 \(T/4\)(四分之一周期)的位置——这是共振点的标志性"\(90°\) 滞后"现象。

8.4 共振

从振幅公式

\[ R(\omega) = \frac{F_0}{\sqrt{(k - m\omega^2)^2 + (c\omega)^2}} \]

看 \(\omega\) 怎么变,\(R(\omega)\) 怎么变。先算极端:

  • \(\omega \to 0\)(慢驱动):\(R \to F_0/k\),即静态变形。物理意义:慢慢推弹簧,响应就是稳态压缩,不振荡。
  • \(\omega \to \infty\)(快驱动):\(R \to F_0/(m\omega^2) \to 0\)。物理意义:推得太快,质量惯性来不及响应,振幅趋于零。
  • \(\omega = \omega_0 = \sqrt{k/m}\)(驱动频率等于固有频率):分母里的 \((k - m\omega^2)\) 项为零,只剩下 \(c\omega\),所以 \(R = F_0/(c\omega_0)\)——振幅反比于阻尼。\(c\) 越小,\(R\) 越大。

让我们看 \(R(\omega)\) 在哪里取最大。求 \(R'(\omega) = 0\)(等价于求被开方式 \((k - m\omega^2)^2 + (c\omega)^2\) 取最小)。对被开方式求导:

\[ \frac{d}{d\omega}\bigl[(k - m\omega^2)^2 + (c\omega)^2\bigr] = 2(k - m\omega^2)(-2m\omega) + 2 c^2 \omega \] \[ = -4m\omega(k - m\omega^2) + 2c^2 \omega = 2\omega\bigl[-2m(k - m\omega^2) + c^2\bigr]. \]

令为零:\(\omega = 0\)(不算)或 \(-2m(k - m\omega^2) + c^2 = 0\),即

\[ \omega_{\text{max}}^2 = \frac{2mk - c^2}{2m^2} = \omega_0^2 - \frac{c^2}{2m^2} = \omega_0^2(1 - 2\zeta^2). \]

这是共振频率(确切地说,是稳态振幅取最大值的那个驱动频率)。它比无阻尼固有频率 \(\omega_0\) 略小,差距由阻尼决定。当 \(\zeta\) 很小时,\(\omega_{\text{max}} \approx \omega_0\);当 \(\zeta = 1/\sqrt{2} \approx 0.707\) 时 \(\omega_{\text{max}} = 0\),也就是阻尼太大了根本没有"共振峰"——\(R(\omega)\) 是一个单调下降的函数。

当 \(c \to 0\)(无阻尼极限),\(\omega_{\text{max}} \to \omega_0\) 而且 \(R(\omega_0) \to \infty\)。无阻尼共振时振幅无界增长——这正是 7.3.2 节那道 \(y'' + 4y = \cos 2x\) 给出的 \(\frac{x}{4}\sin 2x\) 解的物理含义:特解里出现 \(t \cdot$振荡$\) 的项,意味着真正的振幅以线性方式无限增长。

具体推导一下无阻尼共振。考虑 \(m y'' + k y = F_0\cos\omega_0 t\),其中 \(\omega_0 = \sqrt{k/m}\)。齐次解 \(y_h = c_1\cos\omega_0 t + c_2\sin\omega_0 t\) 包含 \(\cos\omega_0 t\),所以共振。乘 \(t\) 修正:试解 \(y_p = t(A\cos\omega_0 t + B\sin\omega_0 t)\)。求导后代入(略,结果是)

\[ y_p = \frac{F_0}{2 m\omega_0} t\sin\omega_0 t. \]

振幅 \(\frac{F_0}{2 m\omega_0} t\) 随时间无限增长。物理上,这就是经典的"共振崩塌":桥梁被频率匹配的人群行走震塌、玻璃杯被歌唱家的高音震碎、塔科马海峡大桥被周期风震毁——后者是一个被反复研究的真实例子,虽然不是纯线性共振(那里的机制叫"气动弹性颤振"),但思路相同。

有阻尼时,虽然振幅有上限,但仍可能很大。把 \(\omega_{\text{max}}^2 = \omega_0^2(1 - 2\zeta^2)\) 代入 \(R(\omega)\),化简后

\[ R_{\text{max}} = \frac{F_0}{c\omega_0\sqrt{1 - \zeta^2}} \approx \frac{F_0}{c\omega_0} \quad (\text{small } \zeta). \]

这里有几个 \(\omega_0\) 与 \(c\)、\(m\) 的关系,推导起来要小心,但结论是清晰的:轻阻尼时共振峰高且窄,重阻尼时共振峰平且宽。

例题:对 \(y'' + 0.2 y' + y = \cos\omega t\)(即 \(m = 1, c = 0.2, k = 1\)),画出 \(R(\omega)\) 的几个关键值。\(\omega_0 = 1\),\(\zeta = 0.1\)。

  • \(\omega = 0\):\(R = 1/k = 1\)。
  • \(\omega = 0.5\):\(R = 1/\sqrt{(1 - 0.25)^2 + (0.1)^2} = 1/\sqrt{0.5625 + 0.01} = 1/\sqrt{0.5725} \approx 1.32\)。
  • \(\omega = 1\)(共振点附近):\(R = 1/\sqrt{0 + 0.04} = 1/0.2 = 5\)。
  • \(\omega = 2\):\(R = 1/\sqrt{(1-4)^2 + (0.4)^2} = 1/\sqrt{9.16} \approx 0.330\)。
  • \(\omega = 5\):\(R = 1/\sqrt{(1-25)^2 + 1} = 1/\sqrt{577} \approx 0.0416\)。

清晰地看到:在 \(\omega = 1\) 附近振幅放大 5 倍,远离 \(\omega = 1\) 后振幅骤降。这就是工程师设计抗振结构时极力避开共振频率的根本原因——离共振哪怕远一点,振动就基本消失。

8.5 RLC 电路:与弹簧的完全类比

电路里有三种基本元件:电感 \(L\)(像质量,储存动能/磁场能)、电阻 \(R\)(像阻尼,耗散能量为热)、电容 \(C\)(像弹簧,储存势能/电场能)。把它们串联,接上时变电压源 \(E(t)\),用基尔霍夫电压定律:

\[ L\frac{dI}{dt} + RI + \frac{Q}{C} = E(t), \]

其中 \(Q\) 是电容上的电荷,\(I = dQ/dt\) 是回路中的电流。把 \(I = Q'\) 代入:

\[ \boxed{L Q'' + R Q' + \frac{1}{C} Q = E(t).} \]

把它和弹簧方程 \(m y'' + c y' + k y = F(t)\) 对照:

机械量电学量意义
\(m\)(质量)\(L\)(电感)惯性 / 储存动能或磁场能
\(c\)(阻尼)\(R\)(电阻)耗散
\(k\)(弹性)\(1/C\)(电容倒数)恢复 / 储存势能或电场能
\(y\)(位移)\(Q\)(电荷)"广义坐标"
\(y'\)(速度)\(I\)(电流)"广义速度"
\(F\)(外力)\(E\)(电压源)驱动

对应关系是逐项的——同样的二阶常系数线性方程在两个完全不同的物理学场合下分别用不同的符号写出。这意味着第 6、7 章对弹簧讲过的话原封不动地适用于电路:

  • 欠阻尼 RLC 电路(\(R^2 < 4L/C\)):电荷振荡衰减。这就是收音机里调谐回路的工作原理。
  • 临界阻尼 (\(R^2 = 4L/C\)):电荷无振荡地回到零,以最快速度。
  • 过阻尼 (\(R^2 > 4L/C\)):电荷无振荡地缓慢回零。
  • 无阻尼共振 (\(R = 0\), \(\omega = 1/\sqrt{LC}\)):电流幅度无限增长——这是 LC 振荡器在理论上的工作模式。
  • 有阻尼共振 (\(R \ne 0\)):驱动频率 \(\omega \approx 1/\sqrt{LC}\) 时电流响应最大,这就是收音机选台的物理原理——你转动调谐旋钮就是在改 \(C\) 的值,从而调整 \(\omega_0 = 1/\sqrt{LC}\) 让它匹配你想接收的电台频率。

例题:RLC 电路 \(L = 1\) H、\(R = 4\) Ω、\(C = 0.04\) F,接 \(E(t) = 100\sin 5t\) V。求稳态电流。

方程 \(Q'' + 4 Q' + 25 Q = 100\sin 5t\)。\(\omega = 5\),\(\omega_0 = \sqrt{1/LC} = \sqrt{25} = 5\),共振!但有阻尼,所以振幅有限。试解 \(Q_p = A\cos 5t + B\sin 5t\)。代入:

\(Q_p'' = -25 A\cos 5t - 25 B\sin 5t\),\(Q_p' = -5A\sin 5t + 5B\cos 5t\)。

\[ Q_p'' + 4 Q_p' + 25 Q_p = (-25A + 20B + 25A)\cos 5t + (-25B - 20A + 25B)\sin 5t = 20B\cos 5t - 20A\sin 5t. \]

等于 \(100\sin 5t\):\(20 B = 0\) 得 \(B = 0\);\(-20 A = 100\) 得 \(A = -5\)。所以 \(Q_p = -5\cos 5t\),稳态电流

\[ I_p = Q_p' = 25\sin 5t. \]

幅度 25 A,频率 5 rad/s。注意一个有趣的现象:\(I_p\) 与 \(E(t) = 100\sin 5t\) 同相(都是 \(\sin 5t\))——共振点恰好是电流相位与电压相位"对齐"的地方。这在电路里叫纯电阻行为:整个 LC 部分的"阻抗"互相抵消,只剩下电阻 \(R\) 起作用,所以电流 \(I = E/R = 100/4 = 25\) A,与上面算出的振幅一致。这并不是巧合——是共振条件下 LC 阻抗相消的精确表现。

8.6 单摆

挂在长度 \(L\) 绳子下的质量为 \(m\) 的物体,绳子的另一端固定。设 \(\theta(t)\) 为绳子与竖直方向的夹角(向某一侧偏离为正)。在无摩擦假设下,物体只受重力。重力沿绳子方向的分量被绳子的拉力抵消,有效力是垂直于绳子的那个分量,大小 \(mg\sin\theta\),方向把物体往平衡位置(\(\theta = 0\))拉。物体的位移弧长是 \(s = L\theta\),所以加速度沿弧 \(\ddot s = L\ddot\theta\)。

牛顿第二定律

\[ m L \ddot\theta = -mg\sin\theta \quad\Longleftrightarrow\quad \ddot\theta + \frac{g}{L}\sin\theta = 0. \]

这是非线性方程!\(\sin\theta\) 出现在里面。一般来说,无法用本书前面学的方法直接解(它的解涉及一种叫"椭圆函数"的特殊函数)。

但是,对小角度,我们有 \(\sin\theta \approx \theta\)(泰勒展开 \(\sin\theta = \theta - \theta^3/6 + \cdots\),保留首项)。代入得

\[ \boxed{\ddot\theta + \frac{g}{L}\theta = 0,} \]

这是简谐振子方程!固有角频率 \(\omega_0 = \sqrt{g/L}\),周期

\[ T = 2\pi\sqrt{\frac{L}{g}}. \]

这就是中学物理里讲的"摆的周期公式"。注意它的两个非凡之处:

  1. 周期不依赖于振幅——只要振幅小,无论摆的多还是少,一次摆动的时间都一样。这是伽利略发现的(据说他在比萨大教堂里观察吊灯摆动时,用自己的脉搏计时,发现幅度不同的吊灯摆动一次居然时间一样)。
  2. 周期不依赖于摆的质量。重的摆和轻的摆同步摆动。这是因为重力同时影响"驱动力"和"惯性",两者刚好相消。

例题:\(L = 1\) m,\(g = 9.8\) m/s²。周期 \(T = 2\pi\sqrt{1/9.8} \approx 2.007\) 秒。所以一根 1 米长的绳挂的摆,大约 2 秒摆动一次。事实上,设计 1 秒摆动一次的"秒摆"长度是 \(L = g/\pi^2 \cdot 1 \approx 0.248\) m——大约 25 厘米。这是十七世纪计时学的核心数字。

但小角度近似什么时候失效?如果取下一阶 \(\sin\theta = \theta - \theta^3/6\),用微扰计算可证明真实周期为

\[ T(\theta_{\max}) = 2\pi\sqrt{L/g}\left(1 + \frac{1}{16}\theta_{\max}^2 + \cdots\right). \]

当 \(\theta_{\max} = 0.1\) 弧度(约 \(5.7°\))时修正项是 \(0.0625\%\),可以忽略;当 \(\theta_{\max} = 0.5\) 弧度(约 \(28.6°\))时修正项是 \(1.56\%\),仍小但可测;当 \(\theta_{\max} = 1.5\) 弧度(约 \(85.9°\))时修正约 14%。所以"小角度近似"对真实钟摆(摆动通常不超过 ±5°)非常准确,对游乐场荡秋千就开始失真了。

给摆加阻尼:

\[ \ddot\theta + \gamma\dot\theta + (g/L)\sin\theta = 0, \]

小角度近似变成 \(\ddot\theta + \gamma\dot\theta + (g/L)\theta = 0\)——和阻尼弹簧形式完全一样。三种阻尼情形完全适用,你可以把它们的图像直接搬过来。

8.7 拍现象

把两个频率非常接近但不相等的余弦相加,会得到一种特殊的波形——它看起来像一个"被慢调制的"余弦,振幅在 0 与最大之间周期性地涨落。这种现象叫拍(beats),在物理学和音乐里都很有名。

来推导。考虑 \(y(t) = \cos\omega_1 t + \cos\omega_2 t\),其中 \(\omega_1, \omega_2\) 接近但不等。用和差化积公式

\[ \cos\alpha + \cos\beta = 2\cos\frac{\alpha + \beta}{2}\cos\frac{\alpha - \beta}{2}, \]

得

\[ y(t) = 2\cos\Bigl(\frac{\omega_1 + \omega_2}{2} t\Bigr)\cos\Bigl(\frac{\omega_1 - \omega_2}{2} t\Bigr). \]

设 \(\bar\omega = (\omega_1 + \omega_2)/2\)(平均频率)、\(\Delta\omega = (\omega_1 - \omega_2)/2\)(差频率的一半)。则

\[ y(t) = 2\cos\bar\omega t \cdot \cos\Delta\omega t. \]

因为 \(\omega_1 \approx \omega_2\),\(\Delta\omega \ll \bar\omega\),所以可以把 \(2\cos\Delta\omega t\) 视为慢变包络,\(\cos\bar\omega t\) 是快振荡载波。听起来效果是:你听到一个频率 \(\bar\omega/(2\pi)\) 的"音调",但它的响度在 \(0\) 和最大之间周期性涨落,涨落周期是 \(2\pi/(2\Delta\omega) = \pi/\Delta\omega\)(对应于 \(\cos\Delta\omega t\) 从 \(+1\) 到 \(0\) 又回到 \(+1\) 的半周期,因为我们听的是响度即 \(|\cos\Delta\omega t|\),它的周期是 \(\cos\) 周期的一半)。

所以听到的"拍频"是

\[ f_{\text{beat}} = \frac{\Delta\omega}{\pi}\cdot\frac{1}{2\pi}\cdot 2 = \frac{|\omega_1 - \omega_2|}{2\pi} = |f_1 - f_2|. \]

简单地说:拍频 = 两个频率之差。所以钢琴调音师同时按下钢琴某个键和音叉,如果两者频率相差 1 Hz,就会听到每秒 1 次的"忽强忽弱"——拍。当两者频率严格相等时,拍消失,这正是调音师追求的状态。

例题:\(\omega_1 = 100\)、\(\omega_2 = 102\)(单位 rad/s)。\(\bar\omega = 101\),\(\Delta\omega = 1\)。听到的"音调"约 \(101/(2\pi) \approx 16.1\) Hz(对人耳来说接近最低可闻范围),拍频 \(2/(2\pi) \approx 0.318\) Hz——大约每 3.14 秒响一次拍。波形是 \(2\cos 101 t \cdot \cos t\)——一个慢慢振荡(周期 \(2\pi\) 秒)的包络,内部装着快振荡。

拍现象在微分方程的语境里有一个干净的来源:把强迫振动 \(y'' + \omega_0^2 y = F_0\cos\omega t\) 在 \(\omega \to \omega_0\)(无阻尼,接近共振但不共振)的极限里看。把通解 \(y_h + y_p\) 写出来,经过一点儿代数变形可以表达成

\[ y(t) = \frac{F_0}{m(\omega_0^2 - \omega^2)}\bigl(\cos\omega t - \cos\omega_0 t\bigr) + (\text{齐次项}), \]

右端括号里 \(\cos\omega t - \cos\omega_0 t\) 用和差公式:

\[ \cos\omega t - \cos\omega_0 t = -2\sin\frac{\omega + \omega_0}{2}t \cdot \sin\frac{\omega - \omega_0}{2}t. \]

所以位移呈现"快振荡 \(\sin\bar\omega t\) 被慢振荡 \(\sin(\Delta\omega t)\) 包络"的图像——这正是拍。当 \(\omega \to \omega_0\)(\(\Delta\omega \to 0\)),包络的周期 \(\to \infty\),意思是它"还来不及完成一次涨落",振幅就一直在增长——这正是无阻尼共振的极限行为。

所以拍与共振之间存在一个连续过渡:

  • \(\omega\) 远离 \(\omega_0\):响应是稳态的小振荡。
  • \(\omega\) 接近 \(\omega_0\) 但不等:出现拍——振幅周期性涨落,涨落周期 \(\sim 1/|\omega - \omega_0|\),越接近共振涨落越慢。
  • \(\omega = \omega_0\)(无阻尼):涨落周期 \(\to \infty\),振幅持续增长(\(\sim t \cdot\)振荡)。这就是共振的"无界增长"行为的来源。

这种在物理参数连续变化时,系统行为也连续变化的现象,是研究微分方程的一个反复出现的主题:解的定性结构(振荡 / 不振荡、有界 / 无界、稳定 / 不稳定)随参数 smoothly 变化或在特定值处发生定性转折。我们已经在 6.5 节看过(三种阻尼情形)、在 8.4 节看过(共振峰的位置随阻尼变化)、现在又看到(拍向共振的过渡)。这种"参数依赖"的视角,在更高级的微分方程理论(动力系统、分歧理论)里被发展为完整的语言。
