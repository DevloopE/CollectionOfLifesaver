附录 A 集合、关系、等价类速查

本书正文里"陪集"" 商群"" 商环"" 商空间" 这些概念都依赖一个底层的集合论操作——用一个等价关系把集合分成等价类。这条操作如果没在底子里跑顺，到第 3 章陪集出现的那一刻容易卡壳。这个附录把这套基础集中处理一遍——也顺带把"函数"" 偏序"" 选择公理" 这些边角概念整理在一处，正文用到时（如 Zorn 引理在第 10 章自由群存在性、第 20 章代数闭包存在性里）你能回来查。

  • 集合的基本运算 + 笛卡尔积。
  • 函数：单 / 满 / 双射、像 / 原像、复合。
  • 关系：自反 / 对称 / 反对称 / 传递。
  • 等价关系 + 等价类 + 划分（三者的等价）。
  • 偏序与全序、上下界。
  • Zorn 引理 + 良序原理 + 选择公理（陈述 + 三者等价）。

A.1 集合的基本操作

设 \(A, B\) 是集合。

  • 并集：\(A \cup B = \{x : x \in A \text{ 或 } x \in B\}\)。
  • 交集：\(A \cap B = \{x : x \in A \text{ 且 } x \in B\}\)。
  • 差集：\(A \setminus B = \{x : x \in A \text{ 且 } x \notin B\}\)。
  • 对称差：\(A \triangle B = (A \setminus B) \cup (B \setminus A)\)。
  • 幂集：\(\mathcal P(A) = \{S : S \subseteq A\}\)（\(A\) 的所有子集组成的集合）。\(|\mathcal P(A)| = 2^{|A|}\) 当 \(A\) 有限时。
  • 笛卡尔积：\(A \times B = \{(a, b) : a \in A, b \in B\}\)。

De Morgan 律（在某全集 \(U\) 内）：\((A \cup B)^c = A^c \cap B^c\)，\((A \cap B)^c = A^c \cup B^c\)。一种"对偶性"——并和交在补运算下交换。

分配律：\(A \cap (B \cup C) = (A \cap B) \cup (A \cap C)\) 和 \(A \cup (B \cap C) = (A \cup B) \cap (A \cup C)\)。这两条不像数的分配律——这里两边对称，两个 分配律都成立。

A.2 函数：单 / 满 / 双射

函数 \(f : A \to B\) 是从 \(A\) 到 \(B\) 的映射——给每个 \(a \in A\) 指定一个唯一 \(f(a) \in B\)。\(A\) 叫定义域（domain），\(B\) 叫陪域（codomain），\(\{f(a) : a \in A\}\) 叫像（image，记 \(\mathrm{Im}\, f\) 或 \(f(A)\)）。

注意陪域 和像 不一定相等——\(f : \mathbb{R} \to \mathbb{R}\)，\(f(x) = x^2\)，陪域 \(\mathbb{R}\)，像 \([0, \infty)\)。

三种性质：

  • 单射（injective / 1-1）：\(f(a_1) = f(a_2) \Rightarrow a_1 = a_2\)。"不重复"。
  • 满射（surjective / onto）：对每个 \(b \in B\) 存在 \(a \in A\) 使 \(f(a) = b\)。即像 = 陪域。
  • 双射（bijective）：单 + 满。这种 \(f\) 有逆函数 \(f^{-1} : B \to A\)。

例。\(f : \mathbb{Z} \to \mathbb{Z}\)，\(f(n) = 2n\)。单（\(2 n_1 = 2 n_2 \Rightarrow n_1 = n_2\)）但不满（奇数不在像里）。所以非双射，没有 \(\mathbb{Z} \to \mathbb{Z}\) 的逆。

例。\(f : \mathbb{R} \to \mathbb{R}_{\ge 0}\)，\(f(x) = x^2\)。满（每个 \(y \ge 0\) 都是 \(\sqrt y\) 的像）但不单（\(f(1) = f(-1) = 1\)）。

例。\(f : \mathbb{R} \to \mathbb{R}\)，\(f(x) = x^3\)。双射，\(f^{-1}(y) = \sqrt[3] y\)。

像 / 原像。设 \(f : A \to B\)。对 \(S \subseteq A\)，\(f(S) = \{f(a) : a \in S\}\)（"正向像"）。对 \(T \subseteq B\)，\(f^{-1}(T) = \{a \in A : f(a) \in T\}\)（"原像" 或"逆像"）。注意 \(f^{-1}(T)\) 总有意义（即使 \(f\) 没有逆函数）——它只是"哪些 \(a\) 的像在 \(T\) 里"。

同态的核（第 4 章）= 单位元的原像 \(\ker \rho = \rho^{-1}(\{e_H\})\)——这就用了原像概念。

复合。\(f : A \to B\) 和 \(g : B \to C\) 复合 \(g \circ f : A \to C\)，\((g \circ f)(a) = g(f(a))\)。复合结合 \((h \circ g) \circ f = h \circ (g \circ f)\) 但一般不交换。这就是第 1 章 1.2 节看到的"对称的组合不交换" 的根。

A.3 关系

关系 \(R\) on \(A\) 是 \(A \times A\) 的一个子集——\(R \subseteq A \times A\)。当 \((a, b) \in R\) 时写 \(a R b\) 或 \(a \sim b\)。

关系的四条标准性质：

  • 自反（reflexive）：对所有 \(a \in A\)，\(a \sim a\)。
  • 对称（symmetric）：\(a \sim b \Rightarrow b \sim a\)。
  • 反对称（antisymmetric）：\(a \sim b\) 且 \(b \sim a\) \(\Rightarrow\) \(a = b\)。
  • 传递（transitive）：\(a \sim b\) 且 \(b \sim c\) \(\Rightarrow\) \(a \sim c\)。

不同性质组合给不同类型：

  • 自反 + 对称 + 传递 = 等价关系。
  • 自反 + 反对称 + 传递 = 偏序。

本书最常用是等价关系（陪集、商）和偏序（理想包含、Zorn 引理）。

例。\(\le\) on \(\mathbb{R}\)：自反、反对称、传递。偏序（也是全序，A.5 节）。

例。"整除" \(|\) on \(\mathbb{Z}_{>0}\)：自反（\(n | n\)）、反对称（\(a | b\) 且 \(b | a\) \(\Rightarrow\) \(a = b\) for positive）、传递（\(a | b\) 且 \(b | c\) \(\Rightarrow\) \(a | c\)）。偏序 但不 全序（2 和 3 不可比）。

例。"同余 mod \(n\)" \(a \equiv b \pmod n\)：自反、对称、传递。等价关系。商集 \(\mathbb{Z}/n\mathbb{Z}\) 是它的等价类集，第 2 章群论的标准例子。

A.4 等价关系 ↔ 划分

等价关系是抽象代数最重要的工具。每个子群（第 3 章）、每个正规子群（第 4 章）、每个理想（第 12 章）都通过"等价关系" 给出商集。下面把"等价关系" 和"划分" 这两个表面不同的概念严格联系起来。

设 \(\sim\) 是 \(A\) 上的等价关系。对 \(a \in A\)，

\[ [a] := \{x \in A : x \sim a\} \]

叫 \(a\) 的等价类。\(A / \sim := \{[a] : a \in A\}\) 叫商集（quotient set）。

定义。 \(A\) 的一个划分（partition）是一族子集 \(\{P_i\}_{i \in I}\) 满足：

  • 每个 \(P_i \neq \emptyset\)；
  • \(i \neq j \Rightarrow P_i \cap P_j = \emptyset\)（两两不相交）；
  • \(\bigcup_i P_i = A\)（合起来覆盖整个 \(A\)）。

说人话：划分把 \(A\) 切成不重不漏的块。

定理（等价 ↔ 划分）。 \(A\) 上的等价关系一一对应于 \(A\) 的划分：

  • 给等价 \(\sim\)，则 \(\{[a] : a \in A\}\) 是 \(A\) 的划分；
  • 给划分 \(\{P_i\}\)，定义 \(a \sim b \Leftrightarrow a, b\) 在同一个 \(P_i\) 里——这是等价关系；
  • 两个对应互为逆。

证明（等价 → 划分方向）。要证 \(\{[a]\}\) 是划分。每个 \([a]\) 非空（\(a \in [a]\)，自反）。两个等价类要么相等要么不相交：若 \([a] \cap [b] \neq \emptyset\)，取 \(c \in [a] \cap [b]\)，则 \(c \sim a\) 和 \(c \sim b\)，对称给 \(a \sim c\)，传递给 \(a \sim b\)。然后 \(x \in [a] \Rightarrow x \sim a \sim b \Rightarrow x \sim b \Rightarrow x \in [b]\)，反向同；所以 \([a] = [b]\)。最后 \(\bigcup [a] = A\)（每个 \(a\) 在自己的 \([a]\) 里）。\(\square\)

这条对应是所有 "商" 操作的根。陪集 \(gH\) 是同余 \(g_1 \sim g_2 \Leftrightarrow g_1^{-1} g_2 \in H\) 的等价类（第 3 章）。商环 \(R/I\) 是同余 \(r_1 \sim r_2 \Leftrightarrow r_1 - r_2 \in I\) 的等价类（第 12 章）。模的商类似。每次"商"，下面就是这条等价关系-划分对应在干活。

"良定义" 问题。商集上想定义运算时，常见手段是"用代表元定义"——比如 \(\mathbb{Z}/n\mathbb{Z}\) 上加法定义为 \([a] + [b] = [a + b]\)。但这要求结果不依赖代表元的选择：如果 \(a' \sim a\) 和 \(b' \sim b\)，要 \([a' + b'] = [a + b]\)。这条"良定义"（well-defined）检查是商代数里反复出现的步骤——第 4 章"正规子群" 的核心是说：商群乘法良定义 ⟺ 子群正规。

A.5 偏序与全序

设 \(\le\) 是 \(A\) 上的关系，自反 + 反对称 + 传递。则 \((A, \le)\) 是偏序集（partially ordered set，poset）。

"偏" 因为不要求每对元素可比。如果还有：

  • 三歧（trichotomy）：对所有 \(a, b\)，\(a \le b\) 或 \(b \le a\)。

则称全序（total order，线性序）。

例。\((\mathbb{Z}, \le)\) 全序。\((\mathcal P(A), \subseteq)\) 偏序但非 全序当 \(|A| \ge 2\)（取 \(A = \{1, 2\}\)，\(\{1\}\) 和 \(\{2\}\) 不可比）。

上界 / 下界 / 极大 / 极小。设 \((A, \le)\) 偏序，\(S \subseteq A\)。

  • \(u \in A\) 是 \(S\) 的上界：对每个 \(s \in S\)，\(s \le u\)。
  • \(\ell \in A\) 是 \(S\) 的下界：对每个 \(s \in S\)，\(\ell \le s\)。
  • \(m \in S\) 是 \(S\) 的极大元：不存在 \(s \in S\) 满足 \(m < s\)（即 \(m \le s\) 且 \(m \neq s\)）。
  • \(\sup S\)（上确界）：\(S\) 的最小 上界（如果存在）。

"极大" 不等于"最大"。在偏序里可能有几个互不可比的极大元——例如 \((\{a, b\}, \le_{\text{trivial}})\) 上 \(a, b\) 都是极大但都不是最大。

A.6 选择公理 / Zorn 引理 / 良序原理

本节是这个附录里最微妙的——三条等价 的集合论命题，每条独立看起来"显然" 或"不显然"，但合起来是抽象数学经常默用的"非构造性" 工具。

选择公理（Axiom of Choice，AC）。给一族非空集合 \(\{A_i\}_{i \in I}\)，存在一个函数 \(f : I \to \bigcup A_i\) 使每个 \(f(i) \in A_i\)。

说人话：从每个非空集里同时各挑一个元素——挑得出来。\(I\) 有限时这是显然的（连续挑）。\(I\) 无限时这条不 平凡——它说"无限多次挑选"也能合在一起。

1908 年 Zermelo 把它列为公理；20 世纪整体被绝大部分数学家接受。少数构造主义者拒绝它，但本书完全采用 AC。

良序原理（Well-Ordering Principle）。每个集合都能被赋一个全序使其良序——即每个非空子集有最小元。

"良序"（well-ordered）= 全序 + 每个非空子集有最小元。\(\mathbb{N}\) 在标准 \(\le\) 下良序——这是数学归纳法的根。\((\mathbb{Q}, \le)\) 不 良序（开区间 \((0, 1) \cap \mathbb{Q}\) 没最小元）。良序原理说：你能重新 给 \(\mathbb{Q}\) 排个序使它良序。这条听起来奇怪——但它是 AC 的一种形式。

Zorn 引理（Zorn's Lemma）。设 \((A, \le)\) 偏序集。如果每条链（chain，全序子集）有上界，则 \(A\) 有极大元。

"链有上界 \(\Rightarrow\) 有极大元" ——这是抽象数学里证"极大 对象存在" 的标准模板。具体例：

  • 每个向量空间有基：取"线性无关子集" 偏序集，链的并仍线性无关给上界，Zorn 给极大线性无关 = 基。
  • 每个非平凡环有极大理想：取"真理想" 偏序集，链的并仍是真理想给上界，Zorn 给极大理想（第 12 章用过）。
  • 每个域有代数闭包：在某种偏序集上 Zorn（第 20 章）。
  • 自由群 / 自由模存在：相关构造里 Zorn 反复出现。

定理（三者等价）。 AC ⟺ 良序原理 ⟺ Zorn 引理。

三个看起来很不同的命题逻辑等价。证明（不展开，参考集合论教材）由"循环蕴含" 完成：AC \(\Rightarrow\) Zorn（用超限归纳）、Zorn \(\Rightarrow\) 良序（取"定义在 \(A\) 子集上的良序" 的偏序集）、良序 \(\Rightarrow\) AC（每个 \(A_i\) 的最小元就是选择函数的值）。

本书使用Zorn 引理 比 AC 直接形式多——它的"极大对象存在" 句式正好是抽象代数最常需要的。但记住本质上是同一件事——每次你看到 Zorn 引理的应用，下面就是"选择公理"。

"非构造" 警告。Zorn 给"极大对象存在"，但不给 你怎么构造 它。"每个向量空间有基"——但当向量空间是 \(\mathbb{R}\) over \(\mathbb{Q}\)（一个无可数维的向量空间）时，没人能写出一组基。Zorn 保证基存在，但这个基无法显式给。这种"知道存在但写不出来" 是 AC 风格证明的特征。本书用 Zorn 时通常会指出这一点——构造性 vs 存在性是两件事。

A.7 备查公式表

正文用得最多的几条速查：

公式说明
\(|A \times B| = |A| \cdot |B|\)笛卡尔积大小
\(|A \cup B| = |A| + |B| - |A \cap B|\)容斥原理
\(|\mathcal P(A)| = 2^{|A|}\)幂集大小（有限时）
\(|A/\sim| \le |A|\)商集大小不超过原集
所有等价类大小之和 = \(|A|\)划分性质
\(f^{-1}(T_1 \cap T_2) = f^{-1}(T_1) \cap f^{-1}(T_2)\)原像保交（也保并）
\(f(S_1 \cap S_2) \subseteq f(S_1) \cap f(S_2)\)正向像不 保交（一般）

最后一条值得注意——正向像和原像在集合操作下的行为不 对称。原像表现得更好。这是为什么第 4 章定义同态的核 用的是原像（\(\rho^{-1}(\{e\})\)）而不是正向像。

A.8 实战提醒

  • 遇到"商"（商群、商环、商空间）——下面一定是某个等价关系；找它，看划分是什么，确认运算良定义。
  • 遇到"极大 对象存在"（极大理想、代数闭包、向量空间基）——下面通常是 Zorn 引理。
  • "同构" 的本质是双射 + 保结构。两步：先验双射，再验运算/序/拓扑/...被保。
  • "良定义" 检查是商代数的标准步骤——切勿跳过。"用代表元定义" 必须验证不依赖代表元。
  • 偏序里"极大" 不等于"最大"——可能有多个互不可比的极大元；想证唯一性要单独论证。
