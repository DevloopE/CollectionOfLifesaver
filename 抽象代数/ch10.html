第 10 章 自由群、生成元与关系

\(D_6\) 你已经见过——第 1 章从正六边形对称里数出 12 个元素：

\[ \{e, r, r^2, r^3, r^4, r^5, s, rs, r^2 s, r^3 s, r^4 s, r^5 s\}. \]

请你重新看这条清单。12 个元素全部由两个 字母 \(r, s\) + 它们的乘积构成。这两个字母是生成元。

但只有"生成元" 还不够。\(\{r, s\}\) 在没有任何约束 下能生成无穷多串：\(r, r^2, r^3, \ldots, rs, srs, sr^2srs, \ldots\) 没尽头。\(D_6\) 之所以是有限 的，是因为它满足三条关系：

\[ r^6 = e, \quad s^2 = e, \quad srs = r^{-1}. \]

第一条说"转 6 次回到原位"，第二条说"反射两次回原位"，第三条说"先反射再转 60° 再反射 = 转 -60°"（你可以画一下六边形验证）。这三条把无穷多串压缩 到 12 个元素的 \(D_6\)。

这章正面对待这个想法："群可以用生成元 + 关系来描述"。最极端的情况——什么关系都不加，得到的群叫自由群。所有有限或无限群都是某个自由群的商。这给我们一种统一 的"群的写法"，叫 group presentation。

10.1 自由群 \(F_S\)：reduced 字在拼接下的群

取一个集合 \(S\)（叫"字母表"）。我们要造一个由 \(S\) 中字母自由组合而成、没有任何额外约束 的群。

第一步，把 \(S\) 的每个 \(a\) 配上一个"形式逆元" \(a^{-1}\)（这只是个新符号，不 是某个具体运算）。把 \(S \cup S^{-1}\) 叫扩展字母表。

第二步，定义"字"（word）——扩展字母表里有限长度的串：\(a b c^{-1} a^{-1} d\)、\(\varnothing\)（空字）、\(a b a^{-1} b^{-1}\) 这些都是字。

第三步，定义"reduce"。如果一个字里出现 \(a a^{-1}\) 或 \(a^{-1} a\)（两个互逆字母相邻），把它去掉。比如 \(a b b^{-1} c \to a c\)。反复施这条规则直到字里不再出现互逆相邻——这种字叫reduced。

关键事实：reduce 操作的结果 不依赖你按什么次序做（即"reduce" 是 confluent 的；可严格证明，本节略）。所以每个字有唯一 的 reduced form。

第四步，定义群运算。两个 reduced 字 \(u, v\) 的积 定义为"拼接 \(uv\) 之后再 reduce"。比如 \(u = ab, v = b^{-1} c\)，拼接 \(uv = abb^{-1}c\)，reduce \(\to ac\)。

定义。\(S\) 上的自由群 \(F_S\) 是所有 \(S \cup S^{-1}\) 上 reduced 字的集合，运算是"拼接 + reduce"。恒等元是空字 \(\varnothing\)。\(a_1 a_2 \cdots a_k\) 的逆元是 \(a_k^{-1} \cdots a_2^{-1} a_1^{-1}\)。

验证群公理。结合律：拼接结合，reduce 不破坏结合（因 reduce 唯一）。恒等：空字与任何字拼接 = 那个字。逆元：\(u u^{-1}\) 拼接后所有字母对消完，等于 \(\varnothing\)。 \(\Box\)

例子。\(S = \{a\}\) 的自由群 \(F_1 = \{\ldots, a^{-2}, a^{-1}, \varnothing, a, a^2, \ldots\} \cong \mathbb{Z}\)（同构于整数加法群——\(a^k\) 对应整数 \(k\)）。

例子。\(S = \{a, b\}\) 的自由群 \(F_2\) 无限，非阿贝尔。元素如 \(a, ab, aba^{-1}, abab^{-1}a^{-1}, \ldots\) 没尽头。\(ab \neq ba\) ——拼接顺序 改变，且没有任何关系告诉我们它们相等，所以它们就是 \(F_2\) 中不同的两个元素。

请你停下来想一件事：\(F_2\) 是所有 两生成元群的"母亲"。任何两生成元群（比如 \(D_6\) 由 \(r, s\) 生成）都是 \(F_2\) 的商——把 \(F_2\) 里某些元素强行设为 \(e\)（即模掉一个正规子群）就得到那个群。这条"商" 视角是 group presentation 的核心。

10.2 普遍性质：\(F_S\) 是"自由" 的真正含义

"自由" 这个词在数学里有形式 的含义，由普遍性质（universal property）刻画。不光是"没约束"，还有更精确的：

定理（自由群的普遍性质）。设 \(S\) 是集合、\(F_S\) 是 \(S\) 上自由群、\(\iota : S \hookrightarrow F_S\) 是把字母 \(a\) 当作长度 1 的字 \(a\) 的嵌入。对任意 群 \(G\) 和任意 函数 \(\phi : S \to G\)，存在唯一 的群同态 \(\tilde \phi : F_S \to G\) 使 \(\tilde \phi \circ \iota = \phi\)。

用图说：

S  ──ι──▶ F_S
 \         |
  \φ      ∃! ̃φ
   ▼      ▼
        G

这个性质完全 决定 \(F_S\)（即任何满足这条性质的群同构于 \(F_S\)）。

用人话："在 \(F_S\) 上定义同态" 等价于 "选定每个生成元的去处"。你只需指定 \(\phi(a), \phi(b), \ldots\)（即每个字母被送到 \(G\) 的哪个元素），\(\tilde \phi\) 就被自动唯一决定—— \(\tilde \phi(a b a^{-1}) = \phi(a) \phi(b) \phi(a)^{-1}\)。

这条性质的证明：唯一性来自 \(\tilde \phi\) 必须满足 \(\tilde \phi(a_1 \cdots a_k) = \tilde \phi(a_1) \cdots \tilde \phi(a_k) = \phi(a_1) \cdots \phi(a_k)\)（同态性强制）；存在性是验证这条公式良定（即不依赖字的具体写法、reduce 后还是同一个）。

这是 \(F_S\) 的"朴素到极致" 的证据：它没有任何额外结构，所以从它出去的同态只受"同态性" 这条最弱的约束。任何额外 的群（比如 \(D_6\)）会因满足某条关系（比如 \(r^6 = e\)）而限制 哪些 \(\phi\) 能延拓到该群——但 \(F_S\) 上没有 任何这种限制。

10.3 Group presentation \(\langle S | R \rangle\)

把"加一些关系把自由群变小" 形式化。

取 \(F_S\)。再取一个子集 \(R \subseteq F_S\) ——这些是我们想"强行设为 \(e\)" 的元素。比如对 \(D_6\)，\(S = \{r, s\}\)、\(R = \{r^6, s^2, srsr\}\)（注：\(srs = r^{-1} \Leftrightarrow srsr = e\)）。

要让 \(R\) 中元素都 = \(e\)，等于商掉包含 \(R\) 的某个正规子群。最小的这种正规子群叫"由 \(R\) 正规生成的子群"，记 \(N(R)\) ——它是 \(\{x r x^{-1} : r \in R, x \in F_S\}\) 生成的子群（"正规化" 是因为正规子群对共轭封闭）。

定义。由生成元 \(S\) 和关系 \(R\) 给出的presentation 是商群

\[ \boxed{\ \langle S \mid R \rangle := F_S / N(R).\ } \]

记号读法：\(\langle r, s \mid r^6, s^2, srsr \rangle\) 表示"由 \(r, s\) 生成、满足 \(r^6 = s^2 = srsr = e\) 的群"。等价的更读得顺的写法：\(\langle r, s \mid r^6 = e, s^2 = e, srs = r^{-1} \rangle\)。两种等价。

从普遍性质推出：\(\langle S | R \rangle\) 也有自己的普遍性质。从 \(\langle S | R \rangle\) 出去的同态 = 从 \(F_S\) 出去且杀掉 \(R\) 的同态。具体：

定理。群 \(G\) 上"满足关系 \(R\) 的 \(S\) 的赋值"（即 \(\phi : S \to G\) 使诱导 \(\tilde \phi\) 在 \(R\) 上为 \(e\)）一一对应于 "群同态 \(\langle S | R \rangle \to G\)"。

这条性质是 group presentation 在实践中的用法 ——定义一个群 = 给一个 presentation。下面给四个例子。

10.4 四个 presentation 范例

例 1：循环群 \(\mathbb{Z}_n\)。

\[ \mathbb{Z}_n = \langle a \mid a^n = e \rangle. \]

一个生成元 \(a\)，关系是 \(a^n = e\)。元素是 \(\{e, a, a^2, \ldots, a^{n-1}\}\)。验证：\(F_{\{a\}} = \mathbb{Z}\)，模掉 \(N(\{a^n\}) = n \mathbb{Z}\)（因为 \(a\) 上自由群是阿贝尔的，正规子群 = 子群），得 \(\mathbb{Z} / n \mathbb{Z} = \mathbb{Z}_n\)。

例 2：整数 \(\mathbb{Z}\)。

\[ \mathbb{Z} = \langle a \mid \rangle. \]

一个生成元、无关系。"无关系" 用空集表达。这就是 \(F_{\{a\}}\)，已经知道 \(\cong \mathbb{Z}\)。

例 3：\(\mathbb{Z} \times \mathbb{Z}\)。

\[ \mathbb{Z} \times \mathbb{Z} = \langle a, b \mid ab = ba \rangle. \]

两个生成元，一个关系——它们交换。这条关系把 \(F_2\)（无限非阿贝尔）压成阿贝尔群 \(\mathbb{Z}^2\)。元素 = \(a^m b^n\)（\(m, n \in \mathbb{Z}\)），同构 \((m, n) \in \mathbb{Z} \times \mathbb{Z}\)。

例 4：二面体群 \(D_n\)。

\[ D_n = \langle r, s \mid r^n = e, s^2 = e, srs = r^{-1} \rangle. \]

三个关系，正好是 \(n\) 边形对称的三条几何事实。元素 \(\{r^k, r^k s : 0 \le k < n\}\) 共 \(2n\) 个。这条 presentation 是 \(D_n\) 最经济的描述——比"\(2n\) 个元素的列表 + 乘法表"（\(4 n^2\) 项）紧凑得多。

请你停下来欣赏：四个看起来非常不同的群——\(\mathbb{Z}_n\) 有限、\(\mathbb{Z}\) 无限阿贝尔、\(\mathbb{Z}^2\) 二维无限阿贝尔、\(D_n\) 有限非阿贝尔——用同一种记号 \(\langle S | R \rangle\) 写出。这是 group presentation 的统一 力。任何群（无论有限无限、阿贝尔或非）都有 presentation——证明：取 \(S = G\) 自己（每个元素都是生成元），\(R\) = 所有满足的乘积关系，则 \(\langle G | R \rangle = G\)。这"平凡 presentation" 不实用但存在性给出。

实用的 presentation 总希望 \(S, R\) 都尽可能小。"有限 presentation"（finitely presented）= 存在 \(S, R\) 都有限的 presentation。\(D_n, \mathbb{Z}_n, \mathbb{Z}, \mathbb{Z}^2\) 都有限 presentation。

10.5 Cayley 图：把 presentation 画出来

有了 \(\langle S | R \rangle\)，可以把群"画" 出来。Cayley 图（Cayley graph）：

  • 节点 = 群中每个元素。
  • 对每个生成元 \(a \in S\)，从 \(g\) 到 \(ga\) 画一条有向 边（涂上 \(a\) 的颜色）。

例 \(\mathbb{Z}_6 = \langle a | a^6 = e \rangle\) 的 Cayley 图：6 个节点排成一圈 \(e, a, a^2, \ldots, a^5\)，每个节点向下一个画箭头。是个六边形。

例 \(D_3 = \langle r, s | r^3 = s^2 = e, srs = r^{-1} \rangle\) 的 Cayley 图：6 个节点排成两个同心三角形（外圈 \(e, r, r^2\)，内圈 \(s, sr, sr^2\)），\(r\) 边沿圈走、\(s\) 边连接里外。是个棱柱图。

例 \(\mathbb{Z}^2 = \langle a, b | ab = ba \rangle\) 的 Cayley 图：无限大方格，每个格点 \((m, n)\) 用 \(a\) 边到 \((m+1, n)\)、用 \(b\) 边到 \((m, n+1)\)。这是 \(\mathbb{Z}^2\) 的晶格（lattice）图。

例 \(F_2 = \langle a, b | \rangle\)（无关系）的 Cayley 图：从 \(e\) 出发，每个节点伸出 4 条边（\(a, a^{-1}, b, b^{-1}\)）；走出去就回不到原点（除了沿原路返回）。这是一棵无穷 4-正则树。

Cayley 图把抽象群结构变成可视的几何对象。一些群论问题在 Cayley 图上变成几何 / 拓扑问题（这是几何群论，20 世纪后半叶兴起的领域，Gromov 是关键人物之一）。比如"群增长"（一个球内有多少元素，半径 \(n\) 的球大小怎么随 \(n\) 增长）就是 Cayley 图的几何不变量——不同群（多项式增长、指数增长）有非常不同的 Cayley 图几何。

10.6 word problem 不可判定

给一个 presentation \(\langle S | R \rangle\)，自然的算法问题：

Word Problem：给一个 \(F_S\) 中的字 \(w\)，判定 \(w\) 在 \(\langle S | R \rangle\) 中是否 = \(e\)。

这条问题看起来"应当能算"——你只要不断用关系 \(R\) 化简 \(w\)，看能不能化到空字嘛。可惜：

定理（Novikov 1955, Boone 1958）。存在有限 presentation 的群，其 word problem 算法不可判定。即不存在一个总能在有限步内回答"\(w = e\) 吗" 的图灵机。

这是数学里第一批"具体的 算法不可判定" 结果之一。它背后是把图灵机模拟成 group presentation 的关系——构造一个 group presentation 使 word problem 等价于"这台图灵机会停吗"，即停机问题，已知不可判定。

这条结果哲学上很冲击：群论已经知道所有 群都能写成 presentation，但判断两个 presentation 是不是同一个群 都不可算。它告诉我们 group presentation 的描述能力非常 强——强到包含图灵机这种通用计算装置。

实际效果：你不应该期待一个"万能算法" 能从 \(\langle S | R \rangle\) 算出群的所有性质（阶、共轭类、子群结构……）。具体 群（\(D_n, S_n, \mathbb{Z}^k\) 等）有具体 的算法，但统一 的算法不存在。

不过有些子类 的 group presentation 是可判定的。比如有限 群（你可以列出所有元素、做穷举乘法表）、阿贝尔 群（用第 8 章结构定理 + Smith 标准形可解 word problem）、双曲 群（Gromov 引入的几何条件下，word problem 在 polynomial 时间可解）。把"哪一类 presentation 可解" 弄清楚是组合 / 几何群论的活问题。

这一章给出了"群" 的构造性 看法：每个群是某个自由群的商。这种看法在第 12 章"子群、陪集、商群" 的更深讨论里反复出现，并在第 17 章模理论里得到一种平行版本——"自由模" 起类似的角色。下一章我们换跑道：把"群"（一种运算）扩展到"环"（两种运算），开始这本书的第二大块。
