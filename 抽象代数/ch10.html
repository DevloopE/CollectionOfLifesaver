第 10 章 自由群、生成元与关系

到这里我们已经看了大量"具体已存在"的群——\(\mathbb{Z}_n, S_n, A_n, D_n, \) 各种 Sylow 子群、有限交换群的分类。一个自然的反向问题：能不能反过来"造"一个群？给我几个生成元，再加上几条关系，然后一切其它结构由这些生成元和关系自动决定。这就是群表示（group presentation）的思想。

极端情况下，"零关系"——只给生成元，不施加任何约束——得到的群叫自由群。它是这一组生成元能产生的"最大可能"群，所有其它带这一组生成元的群都是它的商。这个"自由"的意义和线性代数里"自由模 = 没有任何关系"的意义完全一致。

本章我们把自由群、群表示、Cayley 图三件事讲清，最后简短提一下"判定问题"在群表示上的不可解性——这是 20 世纪计算群论与可计算性理论的交汇。

  • 10.1 自由群 \(F_S\) 的字（reduced word）模型。
  • 10.2 \(F_2\) 的几何：无限正则二叉树。
  • 10.3 自由群的普遍性质——一切群论同态由生成元处的取值唯一决定。
  • 10.4 群表示 \(\langle S \mid R \rangle = F_S / N(R)\)。
  • 10.5 例：\(D_n\) 的标准 presentation。
  • 10.6 例：\(\mathbb{Z}, \mathbb{Z}_n, \mathbb{Z}^2\) 的 presentation。
  • 10.7 Cayley 图——把群表示画成几何图。
  • 10.8 算法不可解性：判定两个 presentation 同构在一般情形不可判定。

10.1 自由群 \(F_S\)

设 \(S\) 是一个集合（可有限可无限）。我们要构造一个群 \(F_S\)，里面只含 \(S\) 的元素及其逆，乘法尽量"无约束"地拼接。

定义。对每个 \(s \in S\) 引入一个符号 \(s^{-1}\)。一个字是 \(\{s, s^{-1} : s \in S\}\) 中元素的有限序列 \(w = a_1 a_2 \cdots a_n\)（允许空字 \(\varepsilon\)）。

字 \(w\) 称为约化的（reduced），如果没有相邻的"\(s s^{-1}\)"或"\(s^{-1} s\)"出现。

定义。\(F_S\) 是所有约化字组成的集合，乘法定义为拼接然后约化——把两字 \(w_1 w_2\) 拼接，反复消去出现的 \(s s^{-1}\) 或 \(s^{-1} s\) 直到约化为止。\(F_S\) 中单位元为空字 \(\varepsilon\)，逆元 \((a_1 \cdots a_n)^{-1} = a_n^{-1} \cdots a_1^{-1}\)。

这是\(S\) 上的自由群。\(|S| = n\) 时记 \(F_n\)。

需要论证：约化字模型给出群结构（结合律、单位、逆），且约化结果不依赖约化次序。这条约化唯一性是关键技术细节，标准证法是引入"形式幺半群 + 约化运算"或者用 van der Waerden 的"基础引理"。本章不证（书末习题）。

例 \(S = \{a\}\) 单生成。\(F_1\) 中所有约化字 \(\{\ldots, a^{-2}, a^{-1}, \varepsilon, a, a^2, \ldots\} = \{a^k : k \in \mathbb{Z}\}\)。乘法 \(a^k \cdot a^l = a^{k+l}\)。所以 \(F_1 \cong \mathbb{Z}\)。

例 \(S = \{a, b\}\) 两生成。\(F_2\) 中字像 \(a, b, a b, b a, a b a^{-1}, a^2 b^3 a^{-1} b^{-2}, \ldots\)，全是约化字。\(F_2\) 不可交换：\(a b \ne b a\)（它们都是约化字，不相等）。事实上 \(F_2\) 含无限多不同元素，比 \(\mathbb{Z}\) 大得多。

10.2 \(F_2\) 的几何：无限正则四叉树

\(F_2\) 的元素结构很难凭抽象文字感受。它的Cayley 图给出强烈的几何直观。

\(F_2\) 的 Cayley 图：每个约化字是一个顶点，从顶点 \(w\) 引出 4 条带标号的边——分别向 \(wa, wa^{-1}, wb, wb^{-1}\)。

结果是一棵无限的正则 4-叉树（每个顶点恰有 4 个邻居，无环路——为什么没有环？因为 \(F_2\) 中"约化字"本身没有非平凡关系，不存在 \(w_1 w_2 \cdots w_k = e\) 这种"回到原点"的非平凡循环）。

这棵树有几个让人惊讶的几何性质：

  • 双曲性：它是 Gromov 意义下的"\(\delta\)-双曲空间"，曲率"无穷负"——这种几何让 \(F_2\) 在几何群论里是"最双曲"的群之一。
  • 遵循 Banach–Tarski 悖论：\(F_2\) 是 \(\mathrm{SO}(3)\) 的子群（事实），可以让"球面分解成有限块再重组成两个球"成为可能（详细在实分析读本关于不可测集的章节）。

"自由群有非常大的子群空间"是 Nielsen–Schreier 定理：自由群的任意子群都是自由群。\(F_2\) 含无穷秩的自由子群——这点与有限维线性代数的"子空间维数小于等于全空间"形成强烈反差。

10.3 自由群的普遍性质

"自由"这两个字的精确数学含义是：

定理（自由群的普遍性质）。设 \(S\) 是集合，\(F_S\) 是 \(S\) 上的自由群（带嵌入 \(\iota : S \hookrightarrow F_S\)）。对任意群 \(G\) 与任意映射 \(f : S \to G\)，存在唯一群同态 \(\tilde f : F_S \to G\) 使得 \(\tilde f \circ \iota = f\)。

\[ \boxed{\ \forall f : S \to G,\ \exists ! \tilde f : F_S \to G \ \text{群同态}, \ \tilde f|_S = f.\ } \]

证明。\(\tilde f\) 在生成元上必定 \(\tilde f(s) = f(s)\)。对一般约化字 \(w = s_1^{\epsilon_1} \cdots s_n^{\epsilon_n}\)，必定 \(\tilde f(w) = f(s_1)^{\epsilon_1} \cdots f(s_n)^{\epsilon_n}\)。所以唯一性立即得到。存在性：把上述公式当作定义；要验它是良定的群同态——这恰好需要 \(F_S\) 中"约化"操作的良定性，前节已说明。

这个定理换种说法：\(F_S\) 是"集合 \(\to\) 群"这个遗忘函子的左伴随的值。任意群 \(G\) 的"\(S\) 索引子集"和"\(F_S \to G\) 的同态"一一对应。

实用推论：每个群 \(G\) 都是某个自由群的商。证明：取 \(S = G\)，\(f : S \to G\) 是恒等。\(\tilde f : F_G \to G\) 由普遍性给。\(\tilde f\) 显然满射（每个 \(g \in G\) 是 \(\tilde f(g)\)），所以 \(G \cong F_G / \ker \tilde f\)。这把"任何群" \(=\) "自由群除以某个正规子群"这件事一举证明。

10.4 群表示 \(\langle S \mid R \rangle\)

有了上一节"任何群是自由群的商"作为后盾，群表示就有清晰意义了。

定义。设 \(S\) 是集合，\(R \subseteq F_S\) 是若干字（"关系子"）。\(R\) 在 \(F_S\) 中的正规闭包 \(N(R)\) 定义为包含 \(R\) 的最小正规子群，即 \(R\) 中所有元素及其在 \(F_S\) 中所有共轭生成的子群。

群

\[ \boxed{\ \langle S \mid R \rangle := F_S / N(R)\ } \]

叫由 \(S\) 生成且关系为 \(R\) 的群，或称这是它的一个 group presentation。

语言上，\(\langle S \mid R \rangle\) 描述的是"由 \(S\) 自由生成，并强制 \(R\) 中所有字等于 \(e\)"的群。在表达式里写 "\(r = e\)"（\(r \in R\)）通常省略 \(= e\)，直接写 \(r\)；或者写成方程 \(u = v\)（即 \(u v^{-1} \in R\)）。

这个构造满足一个对应的带关系的普遍性质：

定理。对任意群 \(G\) 与映射 \(f : S \to G\) 满足"\(f\) 在每个 \(r \in R\) 上消失"（即 \(\tilde f(r) = e\) 对每个 \(r\)），存在唯一同态 \(\langle S \mid R \rangle \to G\) 把每个 \(s\) 送到 \(f(s)\)。

意思是：\(\langle S \mid R \rangle\) 是满足这些关系的最大群。任何"满足这些关系"的群都从它产生。

10.5 例：\(D_n\) 的 presentation

\(D_n\) 是正 \(n\) 边形的对称群，阶 \(2n\)。它由两个元素生成：旋转 \(r\)（旋转角 \(2\pi/n\)）与反射 \(s\)。这两个元素满足：

  • \(r^n = e\)（旋转 \(n\) 次回到原点）；
  • \(s^2 = e\)（反射两次还原）；
  • \(s r s = r^{-1}\)（先旋转再反射 \(=\) 先反射再反向旋转）。

所以 \(D_n\) 的 presentation：

\[ \boxed{\ D_n = \langle r, s \mid r^n = e,\ s^2 = e,\ s r s = r^{-1} \rangle.\ } \]

这个 presentation 让你不用借助"正多边形几何"也能写下 \(D_n\)——纯代数。

验证 \(|D_n| = 2n\)：用关系 \(s r s = r^{-1}\) 反复挪 \(s\)，每个字可以化为 \(r^i\) 或 \(r^i s\)（\(0 \le i < n\)）。所以最多 \(2n\) 个不同元素。每个字两两不等（自由部分剩 \(\mathbb{Z}_n \times \{e, s\}\)，关系不进一步合并），所以恰为 \(2n\)。

类似的标准 presentation：

  • 四元数群 \(Q_8 = \langle a, b \mid a^4 = e,\ b^2 = a^2,\ b a b^{-1} = a^{-1} \rangle\)；
  • 对称群 \(S_n = \langle s_1, \ldots, s_{n-1} \mid s_i^2 = e,\ (s_i s_{i+1})^3 = e,\ s_i s_j = s_j s_i\ (|i - j| \ge 2) \rangle\)（Coxeter presentation）。

Coxeter presentation 是\(S_n\) 在线性代数"反射群"和后续第 24 章 表示论里的标准入口。

10.6 例：\(\mathbb{Z}, \mathbb{Z}_n, \mathbb{Z}^2\)

\(\mathbb{Z}\) 的 presentation。它是"一个生成元，无关系"的群：

\[ \mathbb{Z} = \langle x \mid \rangle. \]

这就是 \(F_1\)，单生成自由群。

\(\mathbb{Z}_n\) 的 presentation。给一个生成元加一条关系 \(x^n = e\)：

\[ \mathbb{Z}_n = \langle x \mid x^n \rangle. \]

\(\mathbb{Z} \times \mathbb{Z}\) 的 presentation。两个生成元，加一条交换关系 \(x y = y x\)，等价地 \(x y x^{-1} y^{-1} = e\)（这个表达式叫换位子）：

\[ \mathbb{Z} \times \mathbb{Z} = \langle x, y \mid x y x^{-1} y^{-1} \rangle. \]

这条 presentation 透露一个深刻事实——一旦在 \(F_2\) 中加上"\(x, y\) 交换"这一条关系，整个群从无穷维"自由"跌落到二维交换 \(\mathbb{Z}^2\)。这就是 group presentation 的强大：极少几条关系可以剧烈改变群的大小。

更一般，\(\mathbb{Z}^n\)（\(n\) 元交换自由群）：

\[ \mathbb{Z}^n = \langle x_1, \ldots, x_n \mid x_i x_j = x_j x_i\ \forall i, j \rangle. \]

把所有的换位子 \([x_i, x_j] = x_i x_j x_i^{-1} x_j^{-1}\) 都设为 \(e\) 就是"全部交换"。

"自由群 + 全部换位子关系"得到的群叫 \(F_n\) 的交换化 \(F_n^{\mathrm{ab}}\)，等于 \(\mathbb{Z}^n\)。这是从"非交换最大群"到"交换最大群"的标准约化。

10.7 Cayley 图与几何直观

group presentation 的另一种可视化方式是Cayley 图。

定义。设 \(G = \langle S \mid R \rangle\) 一个 presentation。\(G\) 的 Cayley 图是一个有向带标号图：

  • 顶点 \(=\) \(G\) 的元素；
  • 对每个 \(g \in G\) 与 \(s \in S\)，从 \(g\) 到 \(g s\) 引一条标号 \(s\) 的有向边。

这个图把"群"变成"几何对象"，对很多群行为给出极强的视觉理解。

例。\(\mathbb{Z}_6 = \langle x \mid x^6 \rangle\) 的 Cayley 图是一个 6-顶点圆环（每个顶点一条出边到下一个）。\(\mathbb{Z}^2 = \langle x, y \mid xy = yx \rangle\) 的 Cayley 图是平面无限方格。\(F_2 = \langle a, b \mid \rangle\) 的 Cayley 图是无限 4-正则树（10.2 节）。\(D_4 = \langle r, s \mid r^4, s^2, srsr \rangle\) 的 Cayley 图是 8-顶点的"双层环"（外环 \(r\)-边的 4-环，内环 \(r\)-边的 4-环，外内两层之间用 \(s\)-边连）。

Cayley 图把"群乘积"变成"沿边走"——你想算 \(g h\)，就在 Cayley 图中从 \(g\) 出发，按 \(h\) 的字母逐字沿对应标号的边走，最后到达的顶点就是 \(gh\)。

几何群论的整门学问基本上就是"研究 Cayley 图作为度量空间的几何"——曲率、增长率、双曲性、可设等距嵌入等。Gromov 1980 年代发展的几何群论用 Cayley 图把"群论"和"非欧几何"连成一线。这条线在实分析读本提到的 amenability 概念里也再出现。

10.8 不可解性：Boone–Novikov 定理

group presentation 给"什么群存在"一个非常一般的答案。但它带来一个尴尬的问题——给两个 presentation \(\langle S_1 \mid R_1 \rangle\) 与 \(\langle S_2 \mid R_2 \rangle\)，判定它们同构是否有算法？

1955 年 Novikov 与 1958 年 Boone 各自独立给出震惊的答案：

定理（Boone–Novikov）。"判定两个有限呈现群是否同构"在一般情形是算法不可解的。事实上更基本的字问题——"给一个有限 presentation \(\langle S \mid R \rangle\) 与一个字 \(w\)，判定 \(w\) 是否等于 \(e\)"——也已经不可解。

"不可解"在这里是 Turing 意义下：不存在任何 Turing 机能在有限时间内对任意输入正确判定字问题。

这条定理的证明走的是计算性理论的路：把 Turing 机的"停机问题"嵌入到一个具体的群表示中。Markov 与 Adyan 接着证明群同构问题、可解性、有限性、可数性等几乎一切群论性质在 presentation 上都是不可判定的。

"代数对象上的判定问题不可解"是 20 世纪可计算性理论与代数交汇的主线。它告诉你：尽管 group presentation 是描述群的"通用语言"，这门语言本身就有不可逾越的算法极限。具体某个群（如 \(\mathbb{Z}_n, S_n, F_n\)）的字问题可以高效解决；但"任意有限呈现群"作为一类对象，你没有统一的算法来全部处理它们。

\[ \boxed{\ \text{字问题（word problem）在一般有限呈现群上算法不可解.}\ } \]

群论这一"上半部分"——纯群论——到这里告一段落。前 10 章我们从"对称的几何直觉"出发（第 1 章），建立了群、子群、商群、同构定理（2-5 章），引入了群作用与 Sylow（6-7 章），分类了有限交换群（8 章），分析了对称群与置换（9 章），最后给"群的普遍构造"一个完整框架（10 章）。

下一章我们换一个完全不同的代数结构——环（ring）。一个环里同时有两种运算——加法（构成交换群）与乘法（结合，对加法分配）。整数 \(\mathbb{Z}\)、多项式 \(\mathbb{Z}[x]\)、矩阵环、连续函数环，都是环。环论比群论维度更高、结构更丰富，最终通向域论与Galois 理论，让我们能解决那些 19 世纪悬而未决的代数终极问题——尺规作图与多项式可解性。
