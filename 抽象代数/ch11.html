第 11 章 环：定义与基本例子

请你停下来回想一下：从第 1 章到第 10 章，所有的"群" 例子——\(D_6, \mathbb{Z}, GL_n, S_n, F_2\) ——只关心一种 运算。要么是加法（\(\mathbb{Z}\) 上的 \(+\)），要么是合成（\(D_6\) 上的对称组合），要么是乘法（\(GL_n\) 上的矩阵乘法）。一种，再多就不算群了。

但你以前接触过的很多对象——\(\mathbb{Z}\) 自己、矩阵、多项式——并不只有一种 运算。整数同时 有加法和乘法。矩阵同时 有加法和乘法。多项式同时 有加法和乘法。而且这两种运算不 是无关的：它们靠分配律

\[ a (b + c) = ab + ac, \qquad (a + b) c = ac + bc \]

连起来。

把"双运算 + 分配律" 抽象出来，得到的就是环。这一章给环的公理 + 一组标准例子。从这里开始这本书进入它的第二段——环、域、模。整个段落的目标是：把"群论的统一" 推到"同时有加法和乘法" 的世界里。

11.1 环的公理

定义。一个环 \((R, +, \cdot)\) 是一个集合 \(R\) 配两个二元运算 \(+\)（叫加法）和 \(\cdot\)（叫乘法），满足：

  • (R1) \((R, +)\) 是阿贝尔群。即加法结合、有零元 \(0\)、每个 \(a\) 有加法逆 \(-a\)、加法交换。
  • (R2) \((R, \cdot)\) 结合：\((a \cdot b) \cdot c = a \cdot (b \cdot c)\)。
  • (R3) 分配律：\(a (b + c) = ab + ac\)，\((a + b) c = ac + bc\)（左右两条都要——乘法不假设交换）。
  • (R4) 存在乘法单位 \(1 \in R\) 使 \(1 \cdot a = a \cdot 1 = a\)。

注释。乘法不假设交换——\(ab \neq ba\) 一般成立。如果额外 要求 \(ab = ba\)，叫交换环（commutative ring）。乘法不假设有逆——绝大部分环里不是每个非零元都有乘法逆（\(\mathbb{Z}\) 里 2 没有乘法逆，因为 \(1/2 \notin \mathbb{Z}\)）。如果额外 要求每个非零元有逆，叫除环（division ring）；交换的除环叫域（field，第 15 章）。

有些教材的"环" 定义不要求 \(1\)（叫"rng"——少了 i 的环，意指无 identity）。本书统一要求 \(1\) ——所有"环" 默认有乘法单位。这是 Atiyah-Macdonald, Eisenbud, Lang 等现代代数教材的约定。

从公理出发的两条立即推论：

  • \(0 \cdot a = 0\)。证明：\(0 \cdot a = (0 + 0) \cdot a = 0 \cdot a + 0 \cdot a\)（分配律）。两边减 \(0 \cdot a\) 得 \(0 = 0 \cdot a\)。
  • \((-1) \cdot a = -a\)。证明：\(0 = 0 \cdot a = (1 + (-1)) \cdot a = a + (-1) a\)，所以 \((-1) a\) 是 \(a\) 的加法逆。

这些"显然" 的事实不 是公理，是从公理推出 的。把它们写一遍是养成好的代数习惯——以后看到环你会反射去问"这两条还成立吗"。

11.2 数环 \(\mathbb{Z}, \mathbb{Q}, \mathbb{R}, \mathbb{C}\)

第一组例子是你最熟的：

整数环 \(\mathbb{Z}\)。加法 + 乘法都是熟知的。\((\mathbb{Z}, +)\) 是阿贝尔群，乘法结合 + 交换 + 有 1，分配律是初中算术的事。不是域——2 没乘法逆。

有理数 \(\mathbb{Q}\)、实数 \(\mathbb{R}\)、复数 \(\mathbb{C}\)。三者都是域——非零元都有逆。\(\mathbb{Q} \subset \mathbb{R} \subset \mathbb{C}\) 是子环（且子域）链。

这些是所谓数环，它们的公理验证是分钟级的——你已经 在小学到初中里默认接受了所有公理，只是没人指出它们叫"环公理"。

请你停下来想：把"满足环公理" 这件事抽象出来之后，所有 关于环的定理同时关于 \(\mathbb{Z}, \mathbb{Q}, \mathbb{R}, \mathbb{C}\)。这正是第 1 章 1.4 节说的"群的统一" 在环上的版本——同一组公理同时管四 个数系。

11.3 模环 \(\mathbb{Z}_n\)

把 \(\mathbb{Z}\) "模 \(n\)" ——就是把所有差 是 \(n\) 倍数的整数视为相等。结果是 \(\mathbb{Z}_n = \{0, 1, 2, \ldots, n - 1\}\)，加法乘法都"模 \(n\) 之后取代表元"。

例 \(\mathbb{Z}_6\)：\(4 + 5 = 9 \equiv 3 \pmod 6\)，所以 \(4 + 5 = 3\) 在 \(\mathbb{Z}_6\) 中。\(4 \cdot 5 = 20 \equiv 2 \pmod 6\)，所以 \(4 \cdot 5 = 2\)。

验证 \(\mathbb{Z}_n\) 是环。\((\mathbb{Z}_n, +)\) 是阶 \(n\) 的循环阿贝尔群（第 8 章反复用过）。乘法继承自 \(\mathbb{Z}\) 的结构所以结合 + 交换 + 有 1 + 分配律。所以 \(\mathbb{Z}_n\) 是交换环。

它什么时候是域？要每个非零元有乘法逆。

命题。\(\mathbb{Z}_n\) 是域 \(\Leftrightarrow\) \(n\) 是素数。

证明草。\(\Leftarrow\)：若 \(n = p\) 素，对任意 \(0 < a < p\)，\(\gcd(a, p) = 1\)。Bézout 给整数 \(x, y\) 使 \(a x + p y = 1\)，模 \(p\) 得 \(a x \equiv 1\)，所以 \(x \mod p\) 是 \(a\) 的乘法逆。\(\Rightarrow\)：若 \(n\) 合数 \(n = a b\)（\(1 < a, b < n\)），则 \(a, b \in \mathbb{Z}_n\) 都非零但 \(a b = 0\) ——所以 \(a\) 不能有乘法逆（否则两边乘 \(a^{-1}\) 得 \(b = 0\)）。\(\Box\)

所以 \(\mathbb{Z}_2, \mathbb{Z}_3, \mathbb{Z}_5, \mathbb{Z}_7, \ldots\) 是域。这些"素数模" 域有专门记号 \(\mathbb{F}_p\)，是第 15 章有限域和第 23 章编码理论的基本对象。

反过来，\(\mathbb{Z}_4, \mathbb{Z}_6, \mathbb{Z}_8, \ldots\) 是环但不是 域。\(\mathbb{Z}_6\) 中 \(2 \cdot 3 = 0\) ——两个非零元素的积是 0。这种现象叫零因子，11.8 节专门讨论。

11.4 矩阵环 \(M_n(R)\)

取一个环 \(R\)（比如 \(\mathbb{R}\)）。\(R\) 上 \(n \times n\) 矩阵的集合 \(M_n(R)\)，配上常规的矩阵加法和矩阵乘法，是一个环。

验证：

  • 加法：分量逐位加，显然阿贝尔群。零元是零矩阵 \(0\)，\(A\) 的加法逆是 \(-A\)。
  • 乘法：标准矩阵乘法，结合（线性代数知识）。乘法单位是单位矩阵 \(I\)。
  • 分配律：从 \(R\) 的分配律推出来。

但关键不同：矩阵乘法不交换 一般。例如 \(2 \times 2\) 矩阵

\[ A = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}, \quad B = \begin{pmatrix} 0 & 0 \\ 1 & 0 \end{pmatrix}, \quad AB = \begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix} \neq \begin{pmatrix} 0 & 0 \\ 0 & 1 \end{pmatrix} = BA. \]

所以 \(M_n(R)\)（\(n \ge 2\)）是非交换 环。这是这一章第一个 自然出现的非交换环——你以前学的所有数环都是交换的，矩阵环把你拽出"乘法交换" 的舒适区。

另外注意：\(M_n(R)\) 中有零因子。比如 \(A B = 0\) 但 \(A, B\) 都非零的情况大量存在（\(A = B = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}\) 给 \(A^2 = 0\)）。所以 \(M_n(R)\) 从不 是域（\(n \ge 2\)）。这件事很重要——线性代数在矩阵代数上无所不能，但"求逆" 这件事很受限制。\(M_n(R)\) 中可逆元素构成一个群，叫第 1 章说过的 \(GL_n(R)\) ——一般线性群。可逆矩阵不 构成环（不对加法封闭：\(I + (-I) = 0\) 不可逆），但构成群。

11.5 多项式环 \(R[x]\) 与 \(R[x_1, \ldots, x_n]\)

取一个交换环 \(R\)（比如 \(\mathbb{R}\) 或 \(\mathbb{Z}\)）。

\(R[x]\) ——\(R\) 系数的一元多项式环——元素是形如

\[ p(x) = a_0 + a_1 x + a_2 x^2 + \cdots + a_n x^n \]

的形式表达式（\(a_i \in R\)，\(n \ge 0\) 任意，但只有有限个 \(a_i\) 非零）。

"形式 表达式" 这个词关键。\(x\) 不是 任何具体数——它是个"不定元"。\(R[x]\) 不是某个函数集合，是形式 多项式（即系数序列 \((a_0, a_1, a_2, \ldots)\) 几乎全为 0）的代数。两个多项式相等 \(\Leftrightarrow\) 系数对应相等。

加法：系数逐位加。

乘法：标准多项式乘法 \((p \cdot q)(x) = \sum_k \big(\sum_{i + j = k} a_i b_j\big) x^k\)。

验证环公理是机械的（系数运算继承 \(R\) 的环结构）。\(R[x]\) 是交换环（如果 \(R\) 交换）。乘法单位是常数多项式 \(1\)（即 \(a_0 = 1\)，其它 \(a_i = 0\)）。

例：\(\mathbb{Z}[x]\)、\(\mathbb{Q}[x]\)、\(\mathbb{R}[x]\)、\(\mathbb{C}[x]\) ——所有这些都是交换环。\(\mathbb{C}[x]\) 因 \(\mathbb{C}\) 代数闭（复分析），有特别简单的因式分解（每个非零多项式分解成线性因子之积）；其它三个不是。这条差异在第 16 章 UFD和第 18 章域扩张反复出现。

多变量。\(R[x_1, x_2, \ldots, x_n]\) ——\(R\) 系数的多元 多项式环。元素是 \(x_i\) 的形式幂积之和，比如 \(3 x_1^2 x_2 - 5 x_1 x_3^4 + x_2 x_3 \in \mathbb{Z}[x_1, x_2, x_3]\)。同样是交换环。

多项式环是抽象代数和代数几何 的核心对象。代数几何研究的"仿射簇" = 多项式方程组的解集，整套理论靠 \(\mathbb{C}[x_1, \ldots, x_n]\) 这种环构建（Hilbert 零点定理、Gröbner 基、范畴-簇对应都在这条路线上）。

11.6 高斯整数 \(\mathbb{Z}[i]\)

取 \(\mathbb{Z}\) 加上一个新元素 \(i\)（满足 \(i^2 = -1\)），让它们生成的最小子环

\[ \mathbb{Z}[i] = \{a + b i : a, b \in \mathbb{Z}\}. \]

这叫高斯整数（Gaussian integers），由 Gauss 引入研究 19 世纪初的数论问题。它是 \(\mathbb{C}\) 的子环。

加法：\((a + bi) + (c + di) = (a + c) + (b + d) i\)。

乘法：\((a + bi)(c + di) = (ac - bd) + (ad + bc) i\)（用 \(i^2 = -1\) 展开）。

验证子环（"子环" 这个概念会在第 12 章正式定义）：包含 \(1\)，对加 + 乘 + 取负封闭。是。

\(\mathbb{Z}[i]\) 不是域（\(2\) 没乘法逆——\(1/2\) 不在 \(\mathbb{Z}[i]\)）。但它有比 \(\mathbb{Z}\) 更丰富的结构：

  • 它是欧几里得整环——存在带余除法（用复模 \(N(a + bi) = a^2 + b^2\) 作"大小"测度）。
  • 它是主理想整环（PID）——每个理想由单一元素生成。
  • 它是唯一分解整环（UFD）——每个非零非单位元唯一（除单位倍数 + 重排）写成不可约元素之积。

这些性质把它放在第 16 章 UFD的一个重要位置——它给出"\(\mathbb{Z}\) 之外的真正有趣的 UFD" 的标准例子。\(\mathbb{Z}[i]\) 中"素元"（不可约元）的分类——哪些有理素数 \(p\) 在 \(\mathbb{Z}[i]\) 里仍是素元、哪些分解——直接给出 Fermat 著名的"素数能写成两平方和当且仅当 \(p \equiv 1 \pmod 4\) 或 \(p = 2\)" 定理。这是代数数论 的入门典范。

类似的对象还有 \(\mathbb{Z}[\sqrt{-5}] = \{a + b \sqrt{-5} : a, b \in \mathbb{Z}\}\) ——但它不是 UFD：\(6 = 2 \cdot 3 = (1 + \sqrt{-5})(1 - \sqrt{-5})\) 是两种本质不同的分解。这条"分解失唯" 的发现催生了理想论 ——Dedekind 19 世纪后期把"数不能唯一分解" 修复成"理想能唯一分解"，是现代代数数论的根。

11.7 函数环 \(C(X)\)

取一个拓扑空间 \(X\)（最简单：\(X = [0, 1]\) 闭区间）。考虑所有连续 实值函数 \(f : X \to \mathbb{R}\) 的集合，记 \(C(X)\)。

加法：\((f + g)(x) = f(x) + g(x)\)（点态加）。

乘法：\((f \cdot g)(x) = f(x) g(x)\)（点态乘）。

验证环公理：每条都从 \(\mathbb{R}\) 的环公理"逐点" 推出。零元是常 0 函数，乘法单位是常 1 函数。是交换环。

这条环和前面所有例子结构上 不同：

  • 它无穷 大（无穷多函数）。
  • 它有大量 零因子——\(f \cdot g = 0\) 不要求 \(f = 0\) 或 \(g = 0\)，只要 \(f, g\) 的支撑（非零集）不重叠就行。例 \(f(x) = \max(x - 1/2, 0)\)、\(g(x) = \max(1/2 - x, 0)\) 都是 \([0, 1]\) 上非零连续函数，但 \(f g \equiv 0\)。
  • 它远不是 PID 或 UFD——在 \(C([0, 1])\) 上做"分解" 没有好的概念。

这条环的"代数" 看起来比 \(\mathbb{Z}\) 弱很多，但它的分析 特别丰富——\(C(X)\) 是泛函分析 ch4 的核心对象之一（\(C([0, 1])\) 上 sup 范数 \(\|f\| = \sup_x |f(x)|\) 把它变成 Banach 空间，再加上共轭（\(f \to \bar f\)）变成 \(C^*\)-代数）。

整本书中"代数对象同时是分析对象" 的最重要例子就是函数环。它告诉你抽象代数不止 服务于离散数学——\(C^*\)-代数、von Neumann 代数、调和分析里到处都是把"环结构 + 拓扑结构" 同时考虑的对象。20 世纪后半叶 Connes 的非交换几何 把这条线推到极致——把"非交换 \(C^*\)-代数" 当成"非交换空间" 研究。

11.8 单位、零因子、幂等元

环里的元素按"它和乘法的关系" 可分几种特殊类型。这一节把它们一一命名。

单位（unit）。\(u \in R\) 是单位 \(\Leftrightarrow\) 存在 \(v \in R\) 使 \(u v = v u = 1\)。即 \(u\) 有乘法逆。

所有单位构成集合 \(R^\times\)，它在乘法下是群 ——叫 \(R\) 的单位群。

例：

  • \(\mathbb{Z}^\times = \{1, -1\}\)：只有 \(\pm 1\) 有乘法逆。
  • \(\mathbb{Q}^\times = \mathbb{Q} \setminus \{0\}\)：所有非零有理数。
  • \(\mathbb{Z}_n^\times = \{a : \gcd(a, n) = 1\}\)：与 \(n\) 互素的剩余类。\(|\mathbb{Z}_n^\times| = \varphi(n)\)（欧拉函数）。
  • \(M_n(\mathbb{R})^\times = GL_n(\mathbb{R})\)：可逆矩阵。
  • \(\mathbb{Z}[i]^\times = \{1, -1, i, -i\}\)：4 个元素，是 \(\mathbb{Z}_4\) 这个循环群。

"每个非零元都是单位" = "是域"。所以"单位" 在域之外是个真 信息——单位群比 \(R \setminus \{0\}\) 严格小。

零因子（zero divisor）。\(a \neq 0\) 是零因子 \(\Leftrightarrow\) 存在 \(b \neq 0\) 使 \(a b = 0\) 或 \(b a = 0\)。

例：

  • \(\mathbb{Z}\)、\(\mathbb{Q}\)、\(\mathbb{R}\)、\(\mathbb{C}\)：没有 零因子。这些环叫整环（integral domain）——交换环 + 有 1 + 没零因子。
  • \(\mathbb{Z}_6\)：\(2 \cdot 3 = 0\)，所以 2、3 都是零因子。一般 \(\mathbb{Z}_n\) 当 \(n\) 合数时有零因子，\(n\) 素时没零因子（11.3 节）。
  • \(M_n(\mathbb{R})\)：大量零因子（任何奇异 矩阵都是零因子）。
  • \(C([0, 1])\)：大量零因子（11.7 节例子）。

"没零因子" 是个很强的条件——它让"消去律" 成立：\(ab = ac, a \neq 0 \Rightarrow b = c\)（在整环中）。这条性质是 \(\mathbb{Z}\) 和 \(\mathbb{Z}_6\) 数论行为天差地别的根源。第 14 章专门分类整环的层级（整环 → 唯一分解整环 → 主理想整环 → 欧几里得整环 → 域），"没零因子" 是最低门槛。

幂等元（idempotent）。\(e \in R\) 是幂等元 \(\Leftrightarrow\) \(e^2 = e\)。

例：

  • 任何环里 \(0\) 和 \(1\) 都是幂等元。这两个叫"平凡" 幂等。
  • \(\mathbb{Z}_6\)：\(3^2 = 9 \equiv 3 \pmod 6\)、\(4^2 = 16 \equiv 4 \pmod 6\)。所以 \(3, 4\) 都是非平凡幂等。
  • \(M_n(\mathbb{R})\)：任何投影矩阵 \(P\)（\(P^2 = P\)）是幂等元。例如把 \(\mathbb{R}^3\) 投到 \(xy\) 面的 \(P = \text{diag}(1, 1, 0)\)。
  • \(C([0, 1])\)：只有 \(0\) 和 \(1\) 是幂等元（\(f(x)^2 = f(x)\) 强制 \(f(x) \in \{0, 1\}\)，但 \(f\) 连续 + \([0, 1]\) 连通 \(\Rightarrow f\) 是常 0 或常 1）。

幂等元的存在≠ 平凡"在交换环上等价于环可分解为两个非平凡子环之积"（具体：\(R \cong eR \times (1 - e) R\)）。所以非平凡幂等元揭示环的乘积结构。\(\mathbb{Z}_6\) 的非平凡幂等正反映了 CRT 分解 \(\mathbb{Z}_6 \cong \mathbb{Z}_2 \times \mathbb{Z}_3\)。

这三类元素——单位 / 零因子 / 幂等——是研究任何 新环时的反射性提问：它的单位群多大？有没有零因子？有没有非平凡幂等？这三个答案立即告诉你环的"形状" ——它接近域、接近整环、还是接近乘积分解？

下一章我们正式定义"子环" 和环里最重要的特殊子集——理想。理想之于环就像正规子群之于群——它是构造商环的工具，整个环 / 域 / 模理论的代数操作都建在理想之上。从此环论开始展开它真正的力量。
