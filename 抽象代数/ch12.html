第 12 章 理想与商环

在群论里你已经熟一条流程：正规子群 给商群。把群 \(G\) 按某个正规子群 \(N\) 的陪集分组，陪集之间能定义乘法，得到的 \(G/N\) 自己是个群。"正规" 这条额外要求是为了保证陪集乘法良定义。

环论里有完全平行的故事——但要平行得彻底，"正规子群" 必须升级成理想。这升级不是装饰：环有两种运算（加法 + 乘法），陪集需要同时支持加 和乘。"加法良定义" 只要求子群正规——而环的加法子群自动正规（环的加法部分是阿贝尔群，所有子群都正规）；这条免费给。乘法 良定义却要一条新条件，比"正规" 强得多。这条新条件就是"理想"的灵魂。

这一章的核心问题是：陪集乘法 \((a + I)(b + I) = ab + I\) 要良定义，\(I\) 必须满足什么？ 答案会自然把"理想"这个名字从问题里逼出来。

12.1 一个你已经会的例子：\(\mathbb{Z}/n\mathbb{Z}\)

从你最熟悉的环 \(\mathbb{Z}\) 入手。固定 \(n = 5\)。看 \(\mathbb{Z}\) 中所有5 的倍数：

\[ (5) := 5\mathbb{Z} = \{\ldots, -10, -5, 0, 5, 10, 15, \ldots\}. \]

记号 \((5)\) 表示"5 生成的"——所有 5 的整数倍。它是 \(\mathbb{Z}\) 的子集，而且 在加法下是子群（两个 5 的倍数相加还是 5 的倍数）。

把 \(\mathbb{Z}\) 按这个子群分陪集。陪集 \(a + (5)\) 就是"和 \(a\) 模 5 同余的所有整数":

\[ 0 + (5), \quad 1 + (5), \quad 2 + (5), \quad 3 + (5), \quad 4 + (5). \]

这5 个 陪集是 \(\mathbb{Z}\) 模 5 的全部"剩余类"。集合记号 \(\mathbb{Z}/5\mathbb{Z}\)。

这 5 个陪集自己有加法：\((1 + (5)) + (3 + (5)) = 4 + (5)\)；\((3 + (5)) + (4 + (5)) = 7 + (5) = 2 + (5)\)。这就是初等数论里学过的"模 5 加法"。

它们也有乘法：\((2 + (5)) \cdot (3 + (5)) = 6 + (5) = 1 + (5)\)。"模 5 乘法"。你早就用过——只是当时没意识到这是陪集乘法。

所以 \(\mathbb{Z}/5\mathbb{Z}\) 既是加法群，又有兼容的乘法——它是个环，5 元素的环。它从 \(\mathbb{Z}\) 通过模 5 制造出来。

请你停下来想：我们刚才做了什么？把一个无穷 大的环 \(\mathbb{Z}\)，按某个特殊子集 \((5)\) 折叠，得到一个5 元素 的新环 \(\mathbb{Z}/5\mathbb{Z}\)。这种"折叠" 是环论核心的构造手段。这一章要做的，就是把"\((5) \subset \mathbb{Z}\) 起到的作用"抽象出来——任意环 \(R\) 里能起这种作用的子集叫理想，折叠出的新环叫商环。

为什么 \((5)\) 能这么用？看清两条性质：

  • \((5)\) 在加法下封闭——5 的倍数加 5 的倍数还是 5 的倍数。它是加法子群。
  • \((5)\) "吸收" 乘法——5 的倍数乘任何 整数还是 5 的倍数。\(5k \cdot n = 5(kn)\)。

第二条是关键。它比"乘法子群" 强——我们没要求 \((5)\) 在乘法下自我封闭（其实是的，但那不是重点）；我们要求它能吸收外面的乘法。把整个 \(\mathbb{Z}\) 乘进 \((5)\)，结果还在 \((5)\) 里。

这条吸收性质是为陪集乘法良定义 准备的。下一节我们看到这一点。

12.2 理想：定义

把上面对 \((5) \subset \mathbb{Z}\) 看到的两条性质抽象到任意环。

定义。 设 \(R\) 是环。子集 \(I \subseteq R\) 叫 \(R\) 的左理想，如果：

  • (I1) \(I\) 在加法下是子群（\(0 \in I\)；\(a, b \in I \Rightarrow a - b \in I\)）；
  • (I2L) \(I\) 吸收左 乘——对任意 \(r \in R\) 与 \(a \in I\)，有 \(ra \in I\)。

类似地把 (I2L) 换成 \(ar \in I\) 得右理想。双侧理想（简称理想）就是同时左、右都是的。

本章里大部分时候我们在交换环 里工作（乘法 \(ab = ba\)），左 = 右 = 双侧，三个名字塌成一个。除非另说，"理想" 默认双侧 / 交换环里的版本。

请注意 (I2) 的强度。它不只说"\(I\) 在自身的乘法下封闭"——\(I \cdot I \subseteq I\) 是自动的（因为 \(I\) 元素是 \(R\) 元素，乘积落进 \(I\) 因为 (I2) 把"任何 \(r \in R\)" 乘进去）。它说整个 \(R\) 乘进去都不会跑出来。这条比"乘法子环" 严格得多。

对比子环： 子环只需要"加 / 减 / 乘 在自身内封闭"——\(I \cdot I \subseteq I\)。理想要"外面 乘进来都不跑"——\(R \cdot I \subseteq I\)。理想是 子环（强一条），反过来一般不真。

例子：\(\mathbb{Z} \subset \mathbb{Q}\) 是子环但不 是理想——\(\mathbb{Q}\) 里的 \(1/2\) 乘 \(\mathbb{Z}\) 里的 \(1\) 等于 \(1/2 \notin \mathbb{Z}\)，吸收失败。

另例：固定 \(a \in R\)，集合

\[ (a) := \{ra : r \in R\} \]

是 \(R\) 的理想。验证：加法封闭（\(ra + sa = (r+s)a\)）；吸收（\(s(ra) = (sr)a\)）。这叫主理想——单一元素 \(a\) 生成的理想。\((5) \subset \mathbb{Z}\) 就是 \(a = 5\) 的主理想。

更一般，对一组元素 \(a_1, \ldots, a_k \in R\)，

\[ (a_1, \ldots, a_k) := \{r_1 a_1 + \cdots + r_k a_k : r_i \in R\} \]

是由 \(a_1, \ldots, a_k\) 生成的 理想。它是包含 \(\{a_1, \ldots, a_k\}\) 的最小理想。

例：在 \(\mathbb{Z}\) 中 \((6, 9) = ?\) 答案是 \((3)\)——因为 \(6\) 与 \(9\) 的最大公约数是 3，由 Bézout \(3 = 2 \cdot 6 - 9\) 在 \((6, 9)\) 里；反过来 \(6, 9\) 都在 \((3)\) 里。一般在 \(\mathbb{Z}\) 中 \((a, b) = (\gcd(a, b))\)——所有有限生成理想都主。这一观察是下一章 PID 的伏笔。

"平凡" 理想： \(\{0\}\) 永远是理想（叫零理想，记 \((0)\)）；\(R\) 自身永远是理想（含 \(1\) 时）。如果 \(I\) 包含任何一个单位（可逆元 \(u\)），则 \(1 = u^{-1} u \in I\)，再由吸收 \(r = r \cdot 1 \in I\) 对所有 \(r\)——\(I = R\)。

所以含单位的理想 = 整个环。这条在域里有戏剧性后果——下一章我们会看，域只有两个理想：\((0)\) 和域自身。理由：域中除 0 外全是单位。

12.3 商环 \(R/I\)：构造与陪集乘法

现在做"折叠"。给定环 \(R\) 与理想 \(I\)。考虑加法陪集

\[ a + I := \{a + x : x \in I\}, \quad a \in R. \]

不同 \(a\) 可以给出同一陪集：\(a + I = b + I \iff a - b \in I\)。把所有不同陪集的集合记 \(R/I\)。

\(R/I\) 上有加法——这是因为 \(I\) 是加法子群，标准群论给：

\[ (a + I) + (b + I) := (a + b) + I. \]

验证良定义（不同代表给同一答案）的步骤你在群论学过：如果 \(a + I = a' + I\)、\(b + I = b' + I\)，则 \(a - a', b - b' \in I\)，所以 \((a + b) - (a' + b') = (a - a') + (b - b') \in I\)，即 \((a + b) + I = (a' + b') + I\)。

关键问题——乘法：

\[ (a + I)(b + I) := ab + I. \]

这条定义 写下来容易，但我们必须验证它良定义。也就是说：换 \(a \to a' = a + x\)（\(x \in I\)），换 \(b \to b' = b + y\)（\(y \in I\)），应该得到同一个陪集。

算 \(a' b'\)：

\[ a' b' = (a + x)(b + y) = ab + ay + xb + xy. \]

所以 \(a' b' - ab = ay + xb + xy\)。要让这等于 \(I\) 中元素（即 \(a'b' + I = ab + I\)），我们需要

\[ ay + xb + xy \in I. \]

逐项看：

  • \(xy\)：\(x, y \in I\)。\(xy \in I\) 因为 \(I\) 是子环（\(x \in I\)、\(y \in R\)，吸收给 \(xy \in I\)）。
  • \(xb\)：\(x \in I\)、\(b \in R\)——吸收给 \(xb \in I\)。需要右 吸收。
  • \(ay\)：\(a \in R\)、\(y \in I\)——吸收给 \(ay \in I\)。需要左 吸收。

所以三项都进 \(I\)，和也进 \(I\)。良定义成立。

但这一推导用 了 \(I\) 的两侧吸收——左乘任何环元素进 \(I\) 还在 \(I\)，右乘也是。如果 \(I\) 只是子环（\(I \cdot I \subseteq I\)）而不 是理想，"\(ay\)" 这一项一般跑出 \(I\) 之外，乘法就不良定义。

请你停下来欣赏一下："理想" 这个名字现在不再像随便起的——它恰好 是为了让陪集乘法良定义所需要的最小条件。把"乘法良定义" 拆开看，它逼出来"两侧吸收"——这就是理想。

商环。 \(R/I\) 配上述加法 + 乘法，是个环——结合律 / 分配律从 \(R\) 继承下来（用代表元验证），加法零元 \(0 + I = I\)，乘法单位 \(1 + I\)（如果 \(R\) 有 \(1\)）。它叫 \(R\) 模 \(I\) 的商环。

典范投影。 映射

\[ \pi : R \to R/I, \quad a \mapsto a + I \]

是环同态——保加保乘。它的核 \(\ker \pi = \{a \in R : a + I = I\} = I\)。

反过来：任何环同态的核都是理想。验证：若 \(\varphi : R \to S\) 是环同态，则 \(\ker \varphi = \{a \in R : \varphi(a) = 0\}\)。加法子群没问题（核是加法群同态的核）；吸收：若 \(a \in \ker \varphi\)、\(r \in R\)，则 \(\varphi(ra) = \varphi(r) \varphi(a) = \varphi(r) \cdot 0 = 0\)，所以 \(ra \in \ker \varphi\)。同理 \(ar \in \ker \varphi\)。核是双侧理想。

这给了"理想 ↔ 同态核" 的对应——和群论里"正规子群 ↔ 同态核" 完全平行。

12.4 第一同构定理（环版）

群论里你证过：群同态 \(\varphi : G \to H\) 总能因子化为 \(G \to G/\ker \varphi \to \mathrm{im}\, \varphi\)，其中第二箭头是同构。环同态有完全一样的版本。

定理（第一同构定理，环）。 设 \(\varphi : R \to S\) 是环同态。则 \(\ker \varphi\) 是 \(R\) 的理想，\(\mathrm{im}\, \varphi\) 是 \(S\) 的子环，并且

\[ R / \ker \varphi \;\cong\; \mathrm{im}\, \varphi \]

（同构由 \(a + \ker \varphi \mapsto \varphi(a)\) 给出）。

证明思路完全平行群版——验证映射良定义、保加保乘、双射。良定义因为 \(\ker \varphi\) 中元素被 \(\varphi\) 送到 0。保加保乘因为 \(\varphi\) 是同态。单射因为只有 \(\ker \varphi\) 被送到 0。满到 \(\mathrm{im}\, \varphi\) 是定义。

怎么用： 要证两个环 \(R/I\) 与某个具体环 \(T\) 同构，找一个满同态 \(\varphi : R \to T\) 使 \(\ker \varphi = I\)；然后调用第一同构定理。下面三个例子让你看清这一招。

例 1。 \(\mathbb{Z}/n\mathbb{Z}\)。同态 \(\varphi : \mathbb{Z} \to \mathbb{Z}/n\mathbb{Z}\) 取 \(a \mapsto a \bmod n\)。满（每个剩余类都有原像）。核 \(= n\mathbb{Z} = (n)\)。所以 \(\mathbb{Z}/(n) \cong \mathbb{Z}/n\mathbb{Z}\)——这是定义，不算新结果。但它显示构造的统一性。

例 2。 \(\mathbb{R}[x]/(x^2 + 1) \cong \mathbb{C}\)。考虑同态

\[ \varphi : \mathbb{R}[x] \to \mathbb{C}, \quad p(x) \mapsto p(i), \]

"在 \(x = i\) 处求值"。验证：保加 \((p+q)(i) = p(i) + q(i)\)；保乘 \((pq)(i) = p(i) q(i)\)。满：任意 \(a + bi \in \mathbb{C}\) 是多项式 \(a + bx\) 在 \(i\) 处的值。

核：\(\ker \varphi = \{p(x) \in \mathbb{R}[x] : p(i) = 0\}\)。哪些实系数多项式以 \(i\) 为根？由于 \(i\) 与 \(-i\) 共轭、实系数多项式根成共轭对，\(p(i) = 0 \Rightarrow p(-i) = 0\)。所以 \(p\) 至少有 \(x - i\) 和 \(x + i\) 两根，即 \((x - i)(x + i) = x^2 + 1\) 整除 \(p\)。即 \(p \in (x^2 + 1)\)。反之 \((x^2+1) \subset \ker \varphi\) 显然。所以 \(\ker \varphi = (x^2 + 1)\)。

第一同构定理给

\[ \mathbb{R}[x]/(x^2 + 1) \;\cong\; \mathbb{C}. \]

这是一个相当深的结论——它告诉你复数 \(\mathbb{C}\) 不是上帝给的，是 \(\mathbb{R}\) 通过添加 \(x^2 + 1\) 的根代数地构造出来的。"\(i\)" 这个符号在 \(\mathbb{R}[x]/(x^2 + 1)\) 里就是陪集 \(x + (x^2 + 1)\)——一个完全实数的对象。

这种"通过 \(R[x]/(p(x))\) 添加根" 的构造是第 18 章整个域扩张理论的种子。Galois 理论的全部技术都建立在这条之上。

例 3。 \(\mathbb{Z}[x]/(x) \cong \mathbb{Z}\)。同态 \(\varphi : \mathbb{Z}[x] \to \mathbb{Z}\) 取 \(p(x) \mapsto p(0)\)（求常数项）。满（任意 \(n\) 是常数多项式 \(n\) 的值）。核：\(p(0) = 0\) 当且仅当 \(p\) 没常数项，即 \(p\) 被 \(x\) 整除，\(p \in (x)\)。所以 \(\mathbb{Z}[x]/(x) \cong \mathbb{Z}\)。

类似 \(\mathbb{Z}[x]/(x - 5) \cong \mathbb{Z}\)（用 \(p \mapsto p(5)\)）。一般 \(R[x]/(x - a) \cong R\) 对任意 \(a \in R\)——"模掉 \(x - a\)" 等价于"把 \(x\) 设为 \(a\)"。

第二、第三同构定理在环里也有平行版本，证明完全照搬群版，本章不再展开。重点看下面：商环结构告诉我们原环 的什么。

12.5 极大理想 \(\iff\) 商环是域

给一个具体问题：什么样的理想 \(I\) 让商 \(R/I\) 是域？

"域" 是每个非零元都可逆 的环（详见下一章）。这等价于"除 \((0)\) 外没有真理想"——若 \(J \subsetneq F\) 是非零理想，取 \(0 \neq a \in J\)，由 \(F\) 是域 \(a\) 可逆，\(1 = a^{-1} a \in J\)，所以 \(J = F\)，矛盾。所以域只有 \((0)\) 和自身两个理想。

所以 \(R/I\) 是域 \(\iff R/I\) 没有真理想 \(\iff R/I\) 中只有 \((0)\) 和 \(R/I\) 自身。

把这条翻译 回 \(R\)：\(R/I\) 的理想恰好 对应包含 \(I\) 的 \(R\) 的理想（这是商环的"对应定理"——和群版一样，下一节我们会用到）。所以"\(R/I\) 没有真理想" \(\iff\) "\(I\) 与 \(R\) 之间没有真理想"。

这正是极大理想 的定义。

定义。 理想 \(I \subsetneq R\) 叫极大，如果不存在理想 \(J\) 满足 \(I \subsetneq J \subsetneq R\)——\(I\) 已经"挤到了顶"，再大就只能是 \(R\) 自己。

定理。 在交换环 \(R\)（含 1）中，

\[ \boxed{\ I \text{ 极大} \iff R/I \text{ 是域}.\ } \]

证明上面已经写出。

例 1。 \((p) \subset \mathbb{Z}\)（\(p\) 素数）极大——商 \(\mathbb{Z}/(p) = \mathbb{Z}/p\mathbb{Z}\) 是域 \(\mathbb{F}_p\)。极大性也可直接验证：包含 \((p)\) 的理想形如 \((d)\)（\(d | p\)，\(d \in \{1, p\}\)）；要么 \((d) = (p)\)，要么 \((d) = (1) = \mathbb{Z}\)，没有真中间。

例 2。 \((4) \subset \mathbb{Z}\) 不 极大：\((4) \subsetneq (2) \subsetneq \mathbb{Z}\)。商 \(\mathbb{Z}/(4) = \mathbb{Z}/4\mathbb{Z}\) 不 是域：\(2 \cdot 2 = 4 \equiv 0\)，\(2\) 不可逆（\(2 \cdot ? = 1\) 在 \(\mathbb{Z}/4\) 里无解）。

例 3。 \((x^2 + 1) \subset \mathbb{R}[x]\) 极大——因为商 \(\cong \mathbb{C}\) 是域。极大性也可直接看：\(x^2 + 1\) 在 \(\mathbb{R}[x]\) 中不可约（无实根），所以包含 \((x^2 + 1)\) 的真理想就只能是 \((1) = \mathbb{R}[x]\)。

这条经验在 \(k[x]\)（\(k\) 域）中一般化为：\((p(x))\) 极大 \(\iff p(x)\) 不可约。这是我们在 ch15、ch18 反复要用的事实。

12.6 素理想 \(\iff\) 商环是整环

"域" 是个强条件。退一步问：什么样的 \(I\) 让 \(R/I\) 没零因子（即整环，下一章详细定义）？

"\(R/I\) 没零因子" 的意思是：\((a + I)(b + I) = 0 + I\) 蕴含 \(a + I = 0 + I\) 或 \(b + I = 0 + I\)。翻译：

\[ ab \in I \implies a \in I \text{ 或 } b \in I. \]

这就是素理想 的定义。

定义。 理想 \(\mathfrak{p} \subsetneq R\) 叫素（prime），如果对所有 \(a, b \in R\)，

\[ ab \in \mathfrak{p} \implies a \in \mathfrak{p} \text{ 或 } b \in \mathfrak{p}. \]

定理。 在交换环 \(R\)（含 1）中，

\[ \boxed{\ \mathfrak{p} \text{ 素} \iff R/\mathfrak{p} \text{ 是整环}.\ } \]

证明就是上面那段翻译。

名字"素" 的由来：在 \(\mathbb{Z}\) 中 \((p)\) 素 \(\iff p\) 是素数（\(p | ab \Rightarrow p | a\) 或 \(p | b\) 是素数的核心性质）。所以"素理想" 是把"素数" 的乘法行为搬到任意环。

极大 vs 素：

  • 极大 \(\Rightarrow R/I\) 是域 \(\Rightarrow R/I\) 是整环（域是整环，下一章）\(\Rightarrow\) 素。所以极大蕴含素。
  • 反过来一般不真。例：\((0) \subset \mathbb{Z}\) 是素（\(\mathbb{Z}/(0) = \mathbb{Z}\) 是整环），但不 极大（\((0) \subsetneq (2) \subsetneq \mathbb{Z}\)）。

但有一类环里两者重合：PID（主理想环）——\((0)\) 之外的素理想恰好是极大理想。这条等价（"PID 中非零素理想 = 极大理想"）会在下一章 PID 理论里再回来。

12.7 \(\mathbb{Z}[i]\) 中一个让人意外的例子

用上面两条判别准则，看一个具体环。高斯整数 \(\mathbb{Z}[i] = \{a + bi : a, b \in \mathbb{Z}\}\)——把 \(i\) 加进 \(\mathbb{Z}\) 得到的环（也可以看作 \(\mathbb{Z}[x]/(x^2 + 1)\)）。它在数论里出现频繁，尤其是在"哪些素数能写成两个平方和" 这类问题里。

看理想 \((2) \subset \mathbb{Z}[i]\)。问：

  1. \((2) \subset \mathbb{Z}\) 是素的（\(2\) 是素数）。\((2) \subset \mathbb{Z}[i]\) 还素吗？
  2. \((2) \subset \mathbb{Z}[i]\) 极大吗？

看 (1)：在 \(\mathbb{Z}[i]\) 中，

\[ (1 + i)(1 - i) = 1 - i^2 = 1 + 1 = 2. \]

所以 \(2 \in (2)\) 等于 \((1+i)(1-i)\)。问 \(1 + i \in (2)\) 吗？\((2)\) 是 \(\{2(a + bi) : a, b \in \mathbb{Z}\} = \{2a + 2bi : a, b \in \mathbb{Z}\}\)——所有"两个分量都偶" 的高斯整数。\(1 + i\) 两个分量都奇，不 在 \((2)\) 里。同样 \(1 - i\) 也不在 \((2)\) 里。但它们的乘积 \(2\) 在。不素。

所以 \((2) \subset \mathbb{Z}[i]\) 不 是素理想。它的商 \(\mathbb{Z}[i]/(2)\) 有零因子。事实上 \(\mathbb{Z}[i]/(2)\) 有 4 个元素 \(\{0, 1, i, 1 + i\}\)（mod 2 的实部 + 虚部），其中 \((1 + i)^2 = 2i \equiv 0\)——\(1 + i\) 是非零零因子。

更进一步分解：\(2 = (1 + i)(1 - i)\)，但 \(1 - i = -i (1 + i)\)（验：\(-i(1 + i) = -i + 1 = 1 - i\)），且 \(-i\) 是 \(\mathbb{Z}[i]\) 的单位（\((-i)(i) = 1\)）。所以 \(1 - i\) 是 \(1 + i\) 乘单位——它们"等同"（associates）。

翻译：在 \(\mathbb{Z}[i]\) 中

\[ 2 = -i (1 + i)^2. \]

把单位忽略不计，\(2 = (1+i)^2\)——2 在 \(\mathbb{Z}[i]\) 里是某个数的平方。这个"某个数" 是 \(1 + i\)，它在 \(\mathbb{Z}[i]\) 中素（理想 \((1 + i)\) 素）。

看 (2)：\((2) \subsetneq (1 + i) \subsetneq \mathbb{Z}[i]\)。包含关系：\(2 = (1+i)(1-i) \in (1+i)\) 给左边；\(1 \notin (1+i)\)（否则 \((1+i)\) 含单位，等于全环——但 \((1+i)\) 严真，比如 \(1 \notin (1+i)\) 因为 \(1 = (1+i)(a + bi)\) 解出 \(a + bi = 1/(1+i) = (1-i)/2 \notin \mathbb{Z}[i]\)）。所以 \((1 + i)\) 是 \((2)\) 与全环间的中间真理想——\((2)\) 不极大。

但 \((1 + i)\) 极大！商 \(\mathbb{Z}[i]/(1+i)\) 有 2 个元素 \(\{0, 1\}\)，它就是 \(\mathbb{F}_2\)——是域。所以 \((1 + i)\) 极大。

这个小练习教什么： 同一个数 2 在不同环里的素分解 不同。在 \(\mathbb{Z}\) 里 2 是素；在 \(\mathbb{Z}[i]\) 里 2 是 \((1+i)^2\) 乘个单位——它分歧（ramified）。这种"素数在环扩展里的行为" 是代数数论的核心，它的几何 / 数论后果延伸出整本代数数论的内容。

对比一下其它素数：

  • \(p = 5\) 在 \(\mathbb{Z}[i]\) 里 \(5 = (2 + i)(2 - i)\)——分裂成两个不等的素理想。
  • \(p = 3\) 在 \(\mathbb{Z}[i]\) 里仍然素——\((3) \subset \mathbb{Z}[i]\) 还是素 / 极大理想。
  • \(p = 2\) 像上面，\(2 = -i(1+i)^2\)——分歧。

这三种情况（分裂 / 不动 / 分歧）由 \(p \bmod 4\) 决定：\(p \equiv 1\) 分裂、\(p \equiv 3\) 不动、\(p = 2\) 分歧。这是 Fermat 关于"哪些素数 = 两平方和" 那条经典定理的代数版。

12.8 这一章给你的反射动作

读完这一章你应该有以下几个反射：

  • 看到"环 \(R\) 的理想 \(I\)"，立即想"商环 \(R/I\)"。两者是孪生概念——一来必带着另一。
  • 看到"\(R/I\) 是域"，立即想"\(I\) 极大"。看到"\(R/I\) 是整环"，立即想"\(I\) 素"。
  • 要研究环同态 \(\varphi : R \to S\)，第一同构定理几乎总是切入点：核 = 理想 \(I\)，像 \(\cong R/I\)，问题转成研究 \(R/I\) 与 \(S\) 的关系。
  • "添加一个新元素满足某个方程"（"添加 \(\alpha\) 使 \(p(\alpha) = 0\)"）的标准实现是 \(R[x]/(p(x))\)。\(\mathbb{R}\) 添加 \(i\) 得 \(\mathbb{R}[x]/(x^2 + 1) = \mathbb{C}\) 是这条路径的开端。

第三条让构造性代数 进入视野。原本"\(\mathbb{C}\) 是包含 \(\sqrt{-1}\) 的什么神秘扩张" 的问题，被替换成"在 \(\mathbb{R}[x]\) 上模掉 \(x^2 + 1\)" 的纯代数操作。这种"用商环来构造扩张" 的眼光，是第 18 章 之后整个 域扩张与 Galois 理论的方法论基础。

下一章我们正面研究"整环" 这个概念：没零因子 的环。\(\mathbb{Z}, k[x], \mathbb{Z}[i]\) 都是整环——它们之所以"行为像数"，根本原因是没零因子。我们会看在整环里你能像 \(\mathbb{Z}\) 构造 \(\mathbb{Q}\) 那样构造一个分式域，把整环嵌入一个域。这一构造让"整环 ⇄ 域" 这条桥真正立起来。
