第 13 章 整环、域、商域

请你做一道初中也会反对的算术。在 \(\mathbb{Z}_6\)（即 \(\mathbb{Z}/6\mathbb{Z}\)，模 6 的整数）里：

\[ 2 \cdot 3 = 6 \equiv 0 \pmod 6. \]

两个非零元素的乘积是 0。在你写过一辈子算术的 \(\mathbb{Z}\) 里这从没发生过——\(\mathbb{Z}\) 里 \(ab = 0\) 必有 \(a = 0\) 或 \(b = 0\)。可这条性质在 \(\mathbb{Z}_6\) 里崩了。

这条崩坏不只是"长得怪"。它有一连串严重后果：

  • "约去律" 失效：\(2 \cdot 3 = 2 \cdot 0\)（在 \(\mathbb{Z}_6\) 里）但 \(3 \neq 0\)。所以你不能"两边除以 2"。
  • "分式" 失效：\(2/3\) 在 \(\mathbb{Z}_6\) 里没法定义为"那个 \(x\) 使 \(3x = 2\)"——因为 \(3\) 没有乘法逆。
  • "整除关系" 复杂化：\(\mathbb{Z}_6\) 中 \(2 | 0\)、\(3 | 0\)，但 \(0 \neq 0 \cdot 2 \cdot 3\) 这种唯一分解不再成立。

"\(ab = 0 \Rightarrow a = 0\) 或 \(b = 0\)" 是一条非常基本 的性质。环论里有专门名字给"满足这条性质" 的环——它们是整环。这一章把整环的范围理清楚（哪些环是、哪些不是），把更强的"域" 摆在它旁边对比，最后做一件漂亮的事：把任何整环嵌入 一个域，叫商域。

最后这件事意义重大。它说：整环和域之间只差"取分式"。\(\mathbb{Z} \subset \mathbb{Q}\) 这条最熟悉的嵌入只是一般机制 的一个特例。这种机制是代数构造 的标准范式之一。

13.1 \(\mathbb{Z}_6\) 里的故事

把 \(\mathbb{Z}_6\) 的乘法表写出来一部分：

\(\cdot\)012345
0000000
1012345
2024024
3030303
4042042
5054321

看 \(2\) 那一行：\(2 \cdot 3 = 0\)，\(2\) 那一列也是。看 \(3\) 那一行：\(3 \cdot 0 = 3 \cdot 2 = 3 \cdot 4 = 0\)，三个非零元 \(0, 2, 4\) 都被 \(3\) 灭成 0。看 \(4\) 也是——\(4 \cdot 3 = 0\)。

非零元素 \(2, 3, 4\) 都"会和某个非零的东西乘出 0"。它们叫 \(\mathbb{Z}_6\) 的零因子。

定义。 环 \(R\) 中元素 \(a \neq 0\) 叫零因子（zero divisor），如果存在 \(b \neq 0\) 使 \(ab = 0\) 或 \(ba = 0\)。

看 \(\mathbb{Z}_6\) 中 \(1\) 和 \(5\)：它们的整行都没有 0（除了 \(\cdot 0\) 的位置）。它们不 是零因子。事实上 \(1, 5\) 都可逆：\(1 \cdot 1 = 1\)、\(5 \cdot 5 = 25 = 24 + 1 \equiv 1\)。

这两件事不是巧合——下面会看到 有限 环里 "非零因子" 等价于 "可逆"。

从 \(\mathbb{Z}_6 = 2 \cdot 3\) 你怀疑：模 \(n\) 出现零因子当且仅当 \(n\) 不素。验证：若 \(n = ab\)（\(1 < a, b < n\)），则在 \(\mathbb{Z}_n\) 中 \(a \cdot b = n \equiv 0\)，且 \(a, b\) 非零。反过来若 \(n\) 素、\(ab \equiv 0 \pmod n\)，则 \(n | ab\)，由 \(n\) 素 \(n | a\) 或 \(n | b\)，即 \(a \equiv 0\) 或 \(b \equiv 0\)。

所以 \(\mathbb{Z}_n\) 没零因子 \(\iff n\) 是素数。\(\mathbb{Z}_2, \mathbb{Z}_3, \mathbb{Z}_5, \mathbb{Z}_7, \ldots\) 没零因子；\(\mathbb{Z}_4, \mathbb{Z}_6, \mathbb{Z}_8, \mathbb{Z}_9, \ldots\) 有。

"没零因子" 这条性质有名字。

13.2 整环

定义。 整环（integral domain，又译"整域"）是一个交换环 \(R\)（含 \(1 \neq 0\)），没零因子——即 \(a, b \in R\) 满足 \(ab = 0\) 必有 \(a = 0\) 或 \(b = 0\)。

等价地：非零元素的乘积非零。

"含 \(1 \neq 0\)" 排除了平凡环 \(\{0\}\)（其中 \(0 = 1\)）——平凡环按定义不算整环。

例子分类。

  • \(\mathbb{Z}\)：整环。两个非零整数相乘非零是初等结果。
  • \(\mathbb{Q}, \mathbb{R}, \mathbb{C}\)：整环。它们其实是域（更强）。
  • \(\mathbb{Z}[i]\)：整环。验证：\((a + bi)(c + di) = 0 \Rightarrow |a + bi|^2 |c + di|^2 = 0 \Rightarrow (a^2+b^2)(c^2+d^2) = 0\) 在 \(\mathbb{Z}\) 中，所以 \(a^2 + b^2 = 0\) 或 \(c^2 + d^2 = 0\)，即 \(a + bi = 0\) 或 \(c + di = 0\)。
  • \(\mathbb{Z}[x]\) 与一般 \(R[x]\)（\(R\) 整环）：整环。证明用多项式次数——若 \(f\) 次数 \(m\)、\(g\) 次数 \(n\)，则 \(fg\) 的最高项是首项系数之积乘 \(x^{m+n}\)，因 \(R\) 整环非零，不抵消。
  • \(\mathbb{Z}_n\)（\(n\) 合数）：不 整环。例如 \(\mathbb{Z}_6, \mathbb{Z}_4, \mathbb{Z}_{15}\)。
  • \(M_n(\mathbb{R})\)（\(n \times n\) 实矩阵环、\(n \ge 2\)）：不 整环。例如 \(\begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix} \cdot \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix} = 0\)，但因子非零。而且 不交换——双重不满足。
  • \(\mathbb{Z} \times \mathbb{Z}\)（直积，按分量加乘）：不 整环。\((1, 0) \cdot (0, 1) = (0, 0)\)，因子非零。
  • \(C[0, 1]\)（\([0, 1]\) 上连续函数环）：不 整环。取 \(f(x) = \max(0, x - 1/2)\)、\(g(x) = \max(0, 1/2 - x)\)，两个都非零（一个支集 \([1/2, 1]\)、一个 \([0, 1/2]\)），但它们的乘积处处 0。

反思一下："整环" 的范围比"环" 窄很多——大部分有趣的环（矩阵 / 函数空间 / 直积 / 模 \(n\)）都不 是整环。整环像是一类"行为最像数" 的环——其中"约去律" 成立、"次数" 行为正常、"因子分解" 有意义。整环是数论 / 经典代数 的天然舞台，矩阵 / 函数空间是分析 / 几何 的天然舞台。

整环里的约去律。 在整环 \(R\) 里，若 \(ac = bc\) 且 \(c \neq 0\)，则 \(a = b\)。证明：\(ac - bc = 0 \Rightarrow (a - b)c = 0\)，由 \(c \neq 0\) 与无零因子，\(a - b = 0\)，即 \(a = b\)。

这条约去律 在整环里恢复了"数" 的算术行为。在 \(\mathbb{Z}_6\) 里它失败：\(2 \cdot 3 = 2 \cdot 0\) 但 \(3 \neq 0\)。有零因子 与约去律失败 是同一件事的两面。

13.3 域

整环之上还有一个更强的概念。

定义。 域（field）是一个交换环 \(F\)（含 \(1 \neq 0\)），其中每个非零元素都可逆——对每个 \(a \in F \setminus \{0\}\) 存在 \(a^{-1} \in F\) 使 \(a a^{-1} = 1\)。

等价的说法：\(F \setminus \{0\}\) 在乘法下是阿贝尔群（叫"乘法群"，记 \(F^\times\) 或 \(F^*\)）。

定理。 域是整环。

证明：设 \(F\) 是域，\(ab = 0\)。若 \(a \neq 0\)，乘 \(a^{-1}\)：\(a^{-1}(ab) = a^{-1} \cdot 0 = 0\)，左边 \(= (a^{-1}a)b = b\)，所以 \(b = 0\)。

反过来不真——\(\mathbb{Z}\) 是整环但不是域（\(2 \in \mathbb{Z}\) 没乘法逆）。整环到域差一步：让所有非零元素可逆。

标准例子。

  • \(\mathbb{Q}, \mathbb{R}, \mathbb{C}\)：经典域。
  • \(\mathbb{F}_p = \mathbb{Z}/p\mathbb{Z}\)（\(p\) 素数）：有限域。每个非零元都可逆——这个等价于"\(p\) 素数"，下面证。
  • \(\mathbb{Q}(i) = \{a + bi : a, b \in \mathbb{Q}\}\)：域。\((a + bi)^{-1} = (a - bi)/(a^2 + b^2)\)，分母在 \(\mathbb{Q}\) 里非零（除非 \(a = b = 0\)）。
  • \(\mathbb{Q}(\sqrt 2) = \{a + b\sqrt 2 : a, b \in \mathbb{Q}\}\)：域。\((a + b\sqrt 2)^{-1} = (a - b\sqrt 2)/(a^2 - 2b^2)\)。分母非零因 \(\sqrt 2 \notin \mathbb{Q}\)。
  • \(\mathbb{F}_p(x)\) = \(\mathbb{F}_p\) 上有理函数域：所有形式 \(p(x)/q(x)\)（\(q \neq 0\)）的有理表达式。

反例：

  • \(\mathbb{Z}\)：整环非域（无逆）。
  • \(\mathbb{Z}_n\)（\(n\) 合数）：连整环都不是。
  • \(\mathbb{Z}[i]\)：整环非域。例如 \(2 \in \mathbb{Z}[i]\) 没逆——\(1/2 \notin \mathbb{Z}[i]\)。

定理。 \(\mathbb{Z}_n\) 是域 \(\iff n\) 是素数。

证明：若 \(n\) 合数，已经知道它有零因子，连整环都不是。若 \(n = p\) 素数：要验证每个 \(a \in \{1, \ldots, p-1\}\) 在 \(\mathbb{Z}_p\) 中可逆。由于 \(\gcd(a, p) = 1\)（\(p\) 素），Bézout 给整数 \(s, t\) 使 \(sa + tp = 1\)，模 \(p\)：\(sa \equiv 1 \pmod p\)。所以 \(s \bmod p\) 是 \(a\) 的逆。

所以从 \(\mathbb{Z}_p\)（\(p\) 素），数论给我们一族有限域。这族域在编码理论、密码学、数论中是基石。第 23 章详细研究有限域。

定理。 有限整环 是域。

证明：设 \(R\) 有限整环、\(a \in R\) 非零。映射 \(\phi : R \to R\)，\(\phi(x) = ax\)。由约去律单射；有限集合上单射等于满射，所以存在 \(b\) 使 \(ab = 1\)，即 \(b = a^{-1}\)。

这条定理把"整环 + 有限" 自动升级成"域"。\(\mathbb{Z}_p\) 是它的实例（既整环又有限）。但无限 整环不一定升级——\(\mathbb{Z}\) 整环但非域。

13.4 整环的特征

整环 \(R\) 中考虑"\(1 + 1 + \cdots + 1\)（\(n\) 个）" 这个序列。在 \(\mathbb{Z}\) 中它给 \(n\)、永远不到 0。在 \(\mathbb{Z}_p\) 中它给 \(0, 1, 2, \ldots, p-1, 0, 1, \ldots\)，第 \(p\) 步到 0。

定义

\[ \mathrm{char}(R) := \min \{n \ge 1 : \underbrace{1 + 1 + \cdots + 1}_{n \text{ 个}} = 0\}, \]

若没有这样的 \(n\)，定义 \(\mathrm{char}(R) = 0\)。这叫 \(R\) 的特征。

定理。 整环的特征要么 0，要么素数。

证明：设 \(\mathrm{char}(R) = n \ge 1\)。若 \(n = ab\) 是合分解（\(1 < a, b < n\)），考虑 \(R\) 中 \(\underbrace{1 + \cdots + 1}_a \cdot \underbrace{1 + \cdots + 1}_b = \underbrace{1 + \cdots + 1}_{ab} = \underbrace{1 + \cdots + 1}_n = 0\)。即两个非零元（按 \(n\) 最小性，\(a, b < n\) 的"\(1\) 的和" 都非零）的乘积为 0——\(R\) 有零因子，矛盾。

所以 \(n\) 必素。完。

  • \(\mathrm{char}(\mathbb{Z}) = \mathrm{char}(\mathbb{Q}) = \mathrm{char}(\mathbb{R}) = \mathrm{char}(\mathbb{C}) = 0\)。
  • \(\mathrm{char}(\mathbb{F}_p) = p\)。
  • \(\mathrm{char}(\mathbb{F}_p(x)) = p\)（特征跟基础域走）。

特征 \(p\) 域里的"新生儿梦想"。 在特征 \(p\) 域 \(F\) 里，

\[ (a + b)^p = a^p + b^p. \]

证明：二项展开 \((a+b)^p = \sum_{k=0}^p \binom{p}{k} a^k b^{p-k}\)；中间项 \(\binom{p}{k}\)（\(1 \le k \le p-1\)）都被 \(p\) 整除（用 \(\binom{p}{k} = p! / (k!(p-k)!)\)，分子有 \(p\)、分母没——除完仍含 \(p\)），所以在 \(F\) 中归零；只剩首尾。

这条"加法的 \(p\) 次方等于各分量 的 \(p\) 次方"在普通分析里像是"梦想"（学生常以为 \((a+b)^2 = a^2 + b^2\)，错的；但特征 2 域里它真的 对）。所以叫"freshman's dream"。

特征 \(p\) 上由此可以定义 Frobenius 同态

\[ \mathrm{Frob}_p : F \to F, \quad x \mapsto x^p. \]

它是环同态（保加由"freshman's dream"，保乘从指数律），单射（\(F\) 是域），是有限域上 Galois 理论的核心生成元。这是第 23 章的事。

13.5 商域：把整环嵌进域

整环不是域——但你能 把它嵌入一个域。\(\mathbb{Z}\) 嵌入 \(\mathbb{Q}\) 是最熟悉的例子；这一节把这种嵌入一般化。

问题：给整环 \(D\)。能不能找到一个域 \(F\)、和单射环同态 \(\iota : D \hookrightarrow F\)，使得 \(F\) "尽可能小"？

答案：能。这个 \(F\) 叫 \(D\) 的商域（field of fractions），记 \(\mathrm{Frac}(D)\)，构造如下。

构造。 模仿 \(\mathbb{Q} = \mathbb{Z}\) 的分式构造。

第 1 步。考虑对集 \(D \times (D \setminus \{0\})\)。元素 \((a, b)\) 我们直觉上想把它当 \(a/b\)。

第 2 步。在这个集合上定义关系：

\[ (a, b) \sim (c, d) \iff ad = bc. \]

这就是分数等价 \(a/b = c/d \iff ad = bc\) 的形式。验证它是等价关系：

  • 反身：\(ab = ba\)（交换）。
  • 对称：显然。
  • 传递：若 \(ad = bc\) 且 \(cf = de\)，乘前式由 \(f\)：\(adf = bcf\)；用 \(cf = de\)：\(adf = bde\)。所以 \(d(af - be) = 0\)。\(d \neq 0\) 且 \(D\) 整环 \(\Rightarrow af = be\)。即 \((a, b) \sim (e, f)\)。

第 3 步。把等价类记 \([a, b]\) 或 \(a/b\)。设

\[ \mathrm{Frac}(D) := \{[a, b] : a \in D, b \in D \setminus \{0\}\}. \]

第 4 步。定义加法和乘法：

\[ [a, b] + [c, d] := [ad + bc, bd], \quad [a, b] \cdot [c, d] := [ac, bd]. \]

这就是分数加 / 乘的标准公式。\(bd \neq 0\) 因 \(D\) 整环（\(b, d \neq 0\) \(\Rightarrow bd \neq 0\)）。

验证良定义（不同代表给同结果）：略——和 \(\mathbb{Q}\) 构造一字不差。

第 5 步。验证 \(\mathrm{Frac}(D)\) 是域：

  • 加法零元：\([0, 1]\)。
  • 乘法单位：\([1, 1]\)。
  • 加法逆：\([a, b]\) 的逆是 \([-a, b]\)。
  • 乘法逆：若 \([a, b] \neq [0, 1]\)（即 \(a \neq 0\)），那 \([a, b] \cdot [b, a] = [ab, ba] = [1, 1]\)（用 \(ab \cdot 1 = 1 \cdot ba\) 验等价）。所以 \([a, b]^{-1} = [b, a]\)。
  • 结合 / 分配 / 交换：从 \(D\) 继承。

所以 \(\mathrm{Frac}(D)\) 确实是域。

第 6 步。嵌入 \(\iota : D \hookrightarrow \mathrm{Frac}(D)\) 取 \(a \mapsto [a, 1]\)。验证它是单射环同态：保加 \([a, 1] + [b, 1] = [a + b, 1]\)；保乘 \([a, 1][b, 1] = [ab, 1]\)；单射因 \([a, 1] = [b, 1] \iff a = b\)。

所以我们把 \(D\) 嵌进一个域 \(\mathrm{Frac}(D)\)。完成。

"最小性" 的精确意义是：任何把 \(D\) 嵌入域 \(F\) 的单射环同态 \(\varphi : D \hookrightarrow F\) 都唯一 延拓为单射环同态 \(\widetilde{\varphi} : \mathrm{Frac}(D) \hookrightarrow F\)（取 \([a, b] \mapsto \varphi(a) \varphi(b)^{-1}\)）。这是商域的泛性质——它是包含 \(D\) 的"最便宜" 的域。

请你停下来欣赏：构造的每一步都抄 了你小学学分数的过程——分式 \(a/b\)、等价 \(ad = bc\)、加法通分 \(ad + bc \over bd\)、乘法 \(ac \over bd\)。我们做的只是把"整数" 换成"任意整环元素"。整个构造从你 8 岁开始就在做。这一节是把那段记忆形式化。

13.6 商域的例子

例 1：\(\mathrm{Frac}(\mathbb{Z}) = \mathbb{Q}\)。 这是定义。

例 2：\(\mathrm{Frac}(k[x]) = k(x)\)。 设 \(k\) 是域，\(k[x]\) 是 \(k\) 上多项式环（整环）。它的商域记 \(k(x)\)，是\(k\) 上有理函数域——所有形式 \(p(x)/q(x)\)（\(p, q \in k[x]\)、\(q \neq 0\)，按 \(p/q = p'/q' \iff pq' = p'q\) 等价）的"分式"。

这个域非常重要。它把"多项式" 升级到"有理函数"，让"\(1/(x - 1)\)" 这种表达式成为合法的代数对象。当 \(k = \mathbb{R}\) 或 \(\mathbb{C}\) 时这接到分析里的有理函数；当 \(k = \mathbb{F}_p\) 时这是函数域 类的基本对象，是数论里"函数域 vs 数域" 类比的载体。

例 3：\(\mathrm{Frac}(\mathbb{Z}[i]) = \mathbb{Q}(i)\)。 高斯整数的商域是高斯有理数 \(\mathbb{Q}(i) = \{a + bi : a, b \in \mathbb{Q}\}\)。

验证它真是 \(\mathrm{Frac}(\mathbb{Z}[i])\)：任何 \([a + bi, c + di]\)（\(c + di \neq 0\)）是

\[ \frac{a + bi}{c + di} = \frac{(a + bi)(c - di)}{c^2 + d^2} = \frac{ac + bd}{c^2 + d^2} + \frac{bc - ad}{c^2 + d^2} i, \]

分母 \(c^2 + d^2 \in \mathbb{Z}\) 非零（\(c + di \neq 0\) 给）。所以这个商在 \(\mathbb{Q}(i)\) 里。反之 \(\mathbb{Q}(i)\) 中的 \((p/q) + (r/s)i\) 等于 \([(ps + qri), qs] \in \mathrm{Frac}(\mathbb{Z}[i])\)。所以 \(\mathrm{Frac}(\mathbb{Z}[i]) = \mathbb{Q}(i)\)。

同样地 \(\mathrm{Frac}(\mathbb{Z}[\sqrt 2]) = \mathbb{Q}(\sqrt 2)\)、\(\mathrm{Frac}(\mathbb{Z}[\omega]) = \mathbb{Q}(\omega)\)（\(\omega = e^{2\pi i / 3}\)）等等。这一族构造在数论里给"数域"——\(\mathbb{Q}\) 的有限扩张，是代数数论的基本对象。

例 4：什么时候 \(\mathrm{Frac}(D) = D\)？ 当且仅当 \(D\) 已经是域。这时取分式不增加任何新元素。所以"把整环嵌进域" 的过程只对真正非 域的整环有效。

例 5：非整环不能这么做。 在 \(\mathbb{Z}_6\) 里你想搞分式吗？想定义 \(1/2\)？\(2\) 是零因子（\(2 \cdot 3 = 0\)）；如果给 \(2\) 一个逆 \(c\)，则 \(1 = 2c\)，乘 \(3\)：\(3 = 6c = 0\)——但 \(3 \neq 0\)，矛盾。有零因子的环不能整体反转成域。

更一般地，"局部化" 这种推广构造（把某一些 元素反转，不一定全部）是第 16 章之后的高级内容。商域是局部化最干净的特例——把所有 非零元素同时反转，要求环本身没零因子。

13.7 这一章给你的反射动作

读完这一章你应该有：

  • 看到环要先问："是不是整环？是不是域？特征是几？"——这三个问题决定它能用哪些工具。
  • 看到 \(\mathbb{Z}_n\)，立即想"\(n\) 素 \(\Leftrightarrow\) 域"。这条等价是 \(\mathbb{Z}_n\) 的全部秘密。
  • 看到整环，立即想"它有商域"。这给你一条标准升级路径——把"无逆元"问题搬到商域里去解决。
  • 看到特征 \(p\) 的域，立即想 Frobenius \(x \mapsto x^p\)——它是个环同态，让"\(p\) 次方" 变成代数操作而不是分析操作。

整环和域的层级是这本书后面整套结构的支柱：

\[ \text{交换环} \supset \text{整环} \supset \text{UFD} \supset \text{PID} \supset \text{Euclid 整环} \supset \text{域}. \]

每一层都比前一层多一条性质，每一层都有典型例。\(\mathbb{Z}, k[x], \mathbb{Z}[i]\) 这些"经典" 整环里的层级关系——以及它们各自站 在哪一层——是下一章的主题。我们会看到 \(\mathbb{Z}\) 的"唯一素分解" 这件高中事实其实是因为 \(\mathbb{Z}\) 站在PID 这一层；任何 PID 都自动有唯一分解，叫 UFD。但不是 每个 UFD 都是 PID——\(\mathbb{Z}[x]\) 是 UFD 但不 PID。这条层级会让你看清"为什么有的环有唯一分解、有的没有"——以及这种差别从哪 来。
