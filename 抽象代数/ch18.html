第 18 章 域的扩张：把"扩多大"变成不变量

请你回想最熟悉的一对域：\(\mathbb{R} \subset \mathbb{C}\)。\(\mathbb{C}\) 是 \(\mathbb{R}\) 的"扩张"——它包含 \(\mathbb{R}\)，加上一个新元素 \(i\)（满足 \(i^2 = -1\)）。任何复数都能唯一写成 \(a + bi\)，\(a, b \in \mathbb{R}\)。这种"唯一" 让我们说一句很物理的话：

"\(\mathbb{C}\) 是 \(\mathbb{R}\) 上 2 维的"——基是 \(\{1, i\}\)。

"维数" 这个词突然出现在这里很自然——但请你停下来想一想：这个维数到底是什么？\(\mathbb{C}\) 不是向量空间——它是个域！可它同时是个 \(\mathbb{R}\)-向量空间——把 \(\mathbb{R}\) 当标量，\(\mathbb{C}\) 当向量。\(\{1, i\}\) 是它的基，所以它在这种眼光下 2 维。

这种"把扩域看作底域上向量空间" 的视角是这一章打开的全部门户。它把"扩张" 这种定性的概念变成"扩张次数 \([K:F]\)" 这种定量的不变量。而不变量一旦有数字，就能比、能算、能证。

18.1 域扩张的定义和"自动" 向量空间结构

定义。一个域扩张 \(K/F\) 是两个域 \(F \subseteq K\)，且 \(F\) 是 \(K\) 的子域（也就是 \(F\) 用 \(K\) 的运算自身是域）。我们写 \(K/F\) 表示"\(K\) over \(F\)"，不是商，仅仅是表"扩张" 的记号。

例：\(\mathbb{Q} \subset \mathbb{R}\), \(\mathbb{R} \subset \mathbb{C}\), \(\mathbb{Q} \subset \mathbb{Q}(\sqrt 2) := \{a + b\sqrt 2 : a, b \in \mathbb{Q}\}\)。每一对都是域扩张。

关键观察：

命题。若 \(K/F\) 是域扩张，则 \(K\) 在 \(F\) 数乘下自动是个 \(F\)-向量空间。

证明：把 \(K\) 当阿贝尔群（用它自身加法），把 \(F \subseteq K\) 当标量，标量乘法是 \(K\) 自身乘法限制到 \(F \times K\)。验证向量空间四条公理——它们都是 \(K\) 的域公理的特例。完。

这条命题不显眼但极重要。它说：每个域扩张都双重身份——既是域，也是 \(F\)-向量空间。我们把这两个身份同时盯着，就能用线性代数的工具研究域。

18.2 扩张次数 \([K:F]\)

既然 \(K\) 是 \(F\)-向量空间，它就有维数。

定义。\(K/F\) 的扩张次数 \([K:F]\) 是 \(K\) 作为 \(F\)-向量空间的维数：

\[ [K:F] := \dim_F K. \]

可能是有限正整数，可能是 \(\infty\)。

当 \([K:F]\) 有限时叫有限扩张，否则叫无限扩张。本章主要兴趣在有限扩张——它们和"代数元" 紧密相关，是下一章的主题。

请你停下来体会名字。"次数" 这个词不是随便挑——它会和后面"最小多项式的次数"（19.3）拉上关系，那不是巧合而是同一个数。这种同名是 19.5 节的命题。

18.3 例 1：\([\mathbb{C}:\mathbb{R}] = 2\)

主张：\(\{1, i\}\) 是 \(\mathbb{C}\) 作为 \(\mathbb{R}\)-向量空间的基。

张成：任何 \(z \in \mathbb{C}\) 可唯一写成 \(z = a + bi\)，\(a, b \in \mathbb{R}\)。这是复数的标准表达——存在来自 \(\mathbb{C}\) 的构造，唯一来自 \(1, i\) 在 \(\mathbb{C}\) 中的线性无关。

线性无关：设 \(a \cdot 1 + b \cdot i = 0\) 对 \(a, b \in \mathbb{R}\)。即 \(a + bi = 0\)。复数等于零意味着实部和虚部都为零，即 \(a = b = 0\)。所以 \(\{1, i\}\) 在 \(\mathbb{R}\) 上线性无关。

所以 \([\mathbb{C}:\mathbb{R}] = 2\)。

18.4 例 2：\([\mathbb{Q}(\sqrt 2):\mathbb{Q}] = 2\)

定义 \(\mathbb{Q}(\sqrt 2) := \{a + b\sqrt 2 : a, b \in \mathbb{Q}\}\)。

先验证它是域。加减明显封闭。乘法：\((a + b\sqrt 2)(c + d\sqrt 2) = (ac + 2bd) + (ad + bc)\sqrt 2 \in \mathbb{Q}(\sqrt 2)\)。逆元（对非零 \(a + b\sqrt 2\)）：

\[ \frac{1}{a + b\sqrt 2} = \frac{a - b\sqrt 2}{a^2 - 2b^2}. \]

分母 \(a^2 - 2b^2\) 不是 \(0\)：如果 \(a^2 = 2b^2\) 而 \(a, b \in \mathbb{Q}\) 不全为零，则 \((a/b)^2 = 2\)——但 \(\sqrt 2\) 无理（高中证），矛盾。所以分母非零，逆元在 \(\mathbb{Q}(\sqrt 2)\) 里。

这就是个域。\(\{1, \sqrt 2\}\) 是它在 \(\mathbb{Q}\) 上的基：

张成：定义里的元素直接是 \(a \cdot 1 + b \cdot \sqrt 2\)。

线性无关：设 \(a + b\sqrt 2 = 0\), \(a, b \in \mathbb{Q}\)。如果 \(b \neq 0\)，\(\sqrt 2 = -a/b \in \mathbb{Q}\)——和 \(\sqrt 2\) 无理矛盾。所以 \(b = 0\)，从而 \(a = 0\)。

所以 \([\mathbb{Q}(\sqrt 2):\mathbb{Q}] = 2\)。和 \([\mathbb{C}:\mathbb{R}] = 2\) 同次数。这说明"次数 2" 不是 \(\mathbb{C}\) 独有——它是"加了一个二次代数元" 的标志。这种相似性在19 章会被一般化为"\([F(\alpha):F] = \deg m_\alpha\)"。

18.5 例 3：\([\mathbb{Q}(\sqrt 2, \sqrt 3):\mathbb{Q}] = 4\)

这是个比前两个深一点的例子。\(\mathbb{Q}(\sqrt 2, \sqrt 3)\) 定义为同时包含 \(\sqrt 2\) 和 \(\sqrt 3\) 的 \(\mathbb{Q}\) 上最小子域——具体长什么样？

朴素尝试：先加 \(\sqrt 2\) 得 \(\mathbb{Q}(\sqrt 2)\)，再加 \(\sqrt 3\)。后者意味着把 \(\sqrt 3\) 加到 \(\mathbb{Q}(\sqrt 2)\) 里，得到 \(\mathbb{Q}(\sqrt 2)(\sqrt 3) = \{u + v\sqrt 3 : u, v \in \mathbb{Q}(\sqrt 2)\}\)。代入 \(u = a + b\sqrt 2\), \(v = c + d\sqrt 2\)（\(a, b, c, d \in \mathbb{Q}\)）：

\[ u + v\sqrt 3 = a + b\sqrt 2 + c\sqrt 3 + d\sqrt 2 \sqrt 3 = a + b\sqrt 2 + c\sqrt 3 + d\sqrt 6. \]

所以猜测 \(\{1, \sqrt 2, \sqrt 3, \sqrt 6\}\) 是基，扩张次数 4。

张成已经显出（任何元素是上面那种四项形式）。剩下证线性无关。

设 \(a + b\sqrt 2 + c\sqrt 3 + d\sqrt 6 = 0\), \(a, b, c, d \in \mathbb{Q}\)。要证全为零。

把式子写成 \((a + c\sqrt 3) + (b + d\sqrt 3)\sqrt 2 = 0\)。这是 \(\mathbb{Q}(\sqrt 3)\) 上 \(\{1, \sqrt 2\}\) 的线性组合——如果 \(\sqrt 2 \notin \mathbb{Q}(\sqrt 3)\)，则 \(\{1, \sqrt 2\}\) 在 \(\mathbb{Q}(\sqrt 3)\) 上线性无关，所以 \(a + c\sqrt 3 = 0\) 和 \(b + d\sqrt 3 = 0\)。再用 \(\{1, \sqrt 3\}\) 在 \(\mathbb{Q}\) 上线性无关，得 \(a = b = c = d = 0\)。

所以剩下的事就是证 \(\sqrt 2 \notin \mathbb{Q}(\sqrt 3)\)。反证：若 \(\sqrt 2 = u + v\sqrt 3\), \(u, v \in \mathbb{Q}\)。两边平方：\(2 = u^2 + 3v^2 + 2uv\sqrt 3\)。\(\sqrt 3\) 无理 ⟹ \(2uv = 0\)。要么 \(u = 0\)（则 \(2 = 3v^2\) ⟹ \(v^2 = 2/3\) ⟹ \(v\) 不有理），要么 \(v = 0\)（则 \(\sqrt 2 = u\) ⟹ 有理，假）。两边都失败，所以 \(\sqrt 2 \notin \mathbb{Q}(\sqrt 3)\)。

所以 \([\mathbb{Q}(\sqrt 2, \sqrt 3):\mathbb{Q}] = 4\)，基 \(\{1, \sqrt 2, \sqrt 3, \sqrt 6\}\)。

请你停下来注意这次怎么得到 4：把扩张分两步做。第一步 \(\mathbb{Q} \to \mathbb{Q}(\sqrt 2)\) 扩 2。第二步 \(\mathbb{Q}(\sqrt 2) \to \mathbb{Q}(\sqrt 2, \sqrt 3)\) 也扩 2（因为 \(\sqrt 3 \notin \mathbb{Q}(\sqrt 2)\)，论证类似）。两步合起来 \(2 \times 2 = 4\)。这种"次数相乘" 不是巧合，是下一节的塔公式。

18.6 塔公式：扩张次数会乘起来

定理（塔公式）。设 \(F \subseteq M \subseteq K\) 是三层域扩张。则

\[ [K:F] \;=\; [K:M] \cdot [M:F]. \]

当左右某一边是 \(\infty\)，另一边相应地也是 \(\infty\)。

证明。假设两边都有限——记 \([K:M] = m\), \([M:F] = n\)。要证 \([K:F] = mn\)。

选 \(M\) 的 \(F\)-基 \(\{u_1, \ldots, u_n\}\)，\(K\) 的 \(M\)-基 \(\{v_1, \ldots, v_m\}\)。我宣布：\(K\) 在 \(F\) 上的基是

\[ B := \{u_i v_j : 1 \leq i \leq n, 1 \leq j \leq m\}. \]

这是个 \(mn\)-元素集合。证它是 \(K\) 的 \(F\)-基。

张成：取任意 \(k \in K\)。因为 \(\{v_j\}\) 是 \(K\) 的 \(M\)-基，可写

\[ k = \sum_{j=1}^m b_j v_j, \qquad b_j \in M. \]

每个 \(b_j \in M\) 又可按 \(\{u_i\}\) 展开：

\[ b_j = \sum_{i=1}^n a_{ij} u_i, \qquad a_{ij} \in F. \]

合起来：

\[ k = \sum_{j=1}^m \Big( \sum_{i=1}^n a_{ij} u_i \Big) v_j = \sum_{i,j} a_{ij} (u_i v_j). \]

这是 \(B\) 在 \(F\) 上的线性组合。所以 \(B\) 张成 \(K\) over \(F\)。

线性无关：设 \(\sum_{i,j} a_{ij} u_i v_j = 0\), \(a_{ij} \in F\)。重新排列：

\[ \sum_{j=1}^m \Big( \sum_{i=1}^n a_{ij} u_i \Big) v_j = 0. \]

内层括号 \(\sum_i a_{ij} u_i \in M\)（因为 \(a_{ij} \in F \subseteq M\), \(u_i \in M\)）。所以这是 \(\{v_j\}\) 在 \(M\) 上的线性组合等于零。\(\{v_j\}\) 是 \(K/M\) 的基 ⟹ 线性无关 over \(M\) ⟹ 每个 \(\sum_i a_{ij} u_i = 0\)（对每个 \(j\)）。

每个 \(\sum_i a_{ij} u_i = 0\) 是 \(\{u_i\}\) 在 \(F\) 上的线性组合等于零。\(\{u_i\}\) 是 \(M/F\) 的基 ⟹ 每个 \(a_{ij} = 0\)。

所以 \(B\) 在 \(F\) 上线性无关，且张成。\(B\) 是基，\([K:F] = |B| = mn\)。

无限情况：如果某中间维数无限，构造同样地给出 \(K\) 上 \(F\) 不可数（或至少无限）多个线性无关元素，所以 \([K:F]\) 也无限。完。

请你停下来欣赏这条公式的简洁。它说"两步扩张的次数等于两步次数之积"——比"两步加法的距离等于两段距离之和" 还要漂亮，因为乘比加更结构。它在22 章证 Galois 主定理时是每页都用的工具。

立即推论：

推论。若 \([K:F]\) 有限，则任何中间域 \(F \subseteq M \subseteq K\) 也使 \([M:F]\) 有限，且 \([M:F] | [K:F]\)。

证明：塔公式 \([K:F] = [K:M][M:F]\)，左边有限 ⟹ 两个因子都有限；且 \([M:F]\) 整除 \([K:F]\)。

这条推论非常实用。例如想找 \(\mathbb{Q}(\sqrt 2, \sqrt 3)\)（次数 4）的中间域，候选只能有次数 1 或 2——次数 1 是 \(\mathbb{Q}\) 自身，次数 2 是 \(\mathbb{Q}(\sqrt 2), \mathbb{Q}(\sqrt 3), \mathbb{Q}(\sqrt 6)\) 这三个（每个都对应一个 \(\mathbb{Q}(\sqrt 2, \sqrt 3)\) 中的二次元素）。"次数整除" 限定了中间域可能的结构。Galois 主定理把这种"中间域 / 子群"对应做到极致。

18.7 单扩张 \(F(\alpha)\) 与 \([F(\alpha):F]\)

本章最后一个概念：把"加一个元素" 这件事正式命名。

定义。设 \(K/F\) 是域扩张，\(\alpha \in K\)。由 \(\alpha\) 生成的单扩张 \(F(\alpha)\) 是 \(K\) 中包含 \(F\) 和 \(\alpha\) 的最小子域。

"最小" 在集合包含意义下，等价于 \(F(\alpha) = \bigcap \{L : F \cup \{\alpha\} \subset L \subset K, L \text{ 是子域}\}\)。这种"最小子域" 叫"由 \(\alpha\) 生成"。

例：

  • \(\mathbb{R}(i) = \mathbb{C}\)，因为 \(\mathbb{R}\) 加 \(i\) 之后已经能造所有 \(a + bi\)。
  • \(\mathbb{Q}(\sqrt 2) = \{a + b\sqrt 2 : a, b \in \mathbb{Q}\}\)（前面验证过）。
  • \(\mathbb{Q}(\pi) = \{f(\pi)/g(\pi) : f, g \in \mathbb{Q}[x], g(\pi) \neq 0\}\)。这个看起来比 \(\mathbb{Q}(\sqrt 2)\) 大很多——下一章会看到这种"大" 不是错觉，它来自 \(\pi\) 是超越。

\(F(\alpha)\) 的具体结构（含多大）取决于 \(\alpha\) 的性质：

  • 若 \(\alpha\) 满足某个 \(F\) 上多项式（"代数"），\(F(\alpha)\) 是有限扩张，\([F(\alpha):F]\) 等于 \(\alpha\) 的"最小多项式" 次数（19.5）。
  • 若 \(\alpha\) 不满足任何 \(F\) 上多项式（"超越"），\(F(\alpha)\) 是无限扩张，同构于有理函数域 \(F(x)\)。

这两种情况的对比是下一章的主线：代数 vs 超越，以及怎么用最小多项式 完全把握代数情况。

18.8 这一章你带走的

四件事：

  • "域扩张 \(K/F\)" 的关键观察：\(K\) 自动是 \(F\)-向量空间。这把"扩张" 从定性概念转成定量的扩张次数 \([K:F] = \dim_F K\)。
  • 三个标准例子的次数：\([\mathbb{C}:\mathbb{R}] = 2\), \([\mathbb{Q}(\sqrt 2):\mathbb{Q}] = 2\), \([\mathbb{Q}(\sqrt 2, \sqrt 3):\mathbb{Q}] = 4\)。每次都靠"造基 + 证线性无关"。
  • 塔公式 \([K:F] = [K:M][M:F]\)：扩张次数是乘的。它告诉我们任何中间域的次数整除整体次数——这条整除是 Galois 理论的核心算术。
  • 单扩张 \(F(\alpha)\)：把"加一个元素" 形式化。\(F(\alpha)\) 的具体大小取决于 \(\alpha\) 是代数还是超越——下一章的主题。

这一章只准备了语言。所有"扩张次数 = 多少 = 为什么" 的具体计算到下一章最小多项式 上场之后才有清爽答案。下一章你会看到：每个代数元 \(\alpha\) 都被它的最小多项式 \(m_\alpha(x)\) 完全决定——\(F(\alpha) \cong F[x]/(m_\alpha)\)，\([F(\alpha):F] = \deg m_\alpha\)。这是非常具体可算的事，把 18 章的抽象立刻落地。
