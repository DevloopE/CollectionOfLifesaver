第 18 章 域的扩张

从这一章开始，我们离开"环 / 模"的故事，整本书的下半截换上一个新主题——域之间的关系。前面研究环时我们看的是"一个环"内部的乘法、理想、商；现在我们把镜头拉远，看两个域嵌套在一起时会发生什么。这种嵌套关系叫域扩张。它简单到看上去几乎不值得专门研究——直到你意识到一个干净到不可思议的事实：当一个域包含另一个域时，大的那个自动是小的那个上的向量空间。

这一句话把"域论"瞬间拉到线性代数的领地。"扩张次数"就是"维数"；"基"就是"在大域里把所有元素表示出来的最小数据"；"塔公式"就是线性代数维数公式的环境版。整个 Galois 理论在最底层就是这条"扩张 = 向量空间 + 一点点乘法结构"反复使用。这一章的目标就是把这件事从抽象命题变成手算反复跑得熟的几个具体例子——\(\mathbb{C}/\mathbb{R}\)、\(\mathbb{Q}(\sqrt 2)/\mathbb{Q}\)、\(\mathbb{Q}(\sqrt 2, \sqrt 3)/\mathbb{Q}\)、\(\mathbb{Q}(\sqrt[3] 2)/\mathbb{Q}\)。

  • 18.1 域扩张 \(K/F\) 的定义；扩张次数 \([K : F] = \dim_F K\)。
  • 18.2 有限扩张 vs 无穷扩张；为什么有限扩张自动代数（预告 ch19）。

18.1 域扩张：定义与扩张次数

设 \(F\) 与 \(K\) 都是域。称 \(K\) 是 \(F\) 的一个扩张（extension），记 \(K/F\)（读作"\(K\) 上 \(F\)"，斜杠不是商，只是标记），如果 \(F \subset K\) 是一个子域——即 \(F\) 作为 \(K\) 的子集对 \(K\) 的加法乘法封闭、含单位、含逆，自身构成域。这时 \(F\) 叫底域，\(K\) 叫扩域。

这个嵌套关系马上派生一件极重要的事：\(K\) 自动是 \(F\) 上的向量空间。理由非常直接——把 \(K\) 的加法当向量加法，把"\(F\) 中的元素乘 \(K\) 中的元素"（这本来就是 \(K\) 内部的乘法限制到 \(F \times K\)）当标量乘法，分配律、结合律、单位元在 \(K\) 内自动成立，所以八条向量空间公理一一兑现。这是域扩张和线性代数的桥。

有了向量空间结构，立刻有维数。定义：

\[ \boxed{\ [K : F] \;:=\; \dim_F K \;=\; K \text{ 作为 } F\text{-向量空间的维数}.\ } \]

这个数叫扩张次数（degree of the extension），简称次数。它可以是有限正整数，也可以是无穷（\(\dim_F K = \infty\)）。

当 \([K : F]\) 有限时，\(K/F\) 叫有限扩张；当无穷时叫无穷扩张。

一些朴素观察。第一，\([K : F] = 1\) 当且仅当 \(K = F\)（向量空间 1 维 + 含 \(F\) \(\Rightarrow\) 由 \(F\) 张满）。所以"次数 \(\ge 2\)" 已经是"真扩张"。第二，\([K : F]\) 是纯线性代数的不变量——不依赖任何域论的进一步结构，找一组基数一数即可。

每个扩张 \(K/F\) 都给我们两件事：一个集合 \(K\)，与一个 \(F\)-向量空间结构。两件事缺一不可。看一个扩张主要先看"它的次数是多少"——这是它最粗糙也最重要的信息。

18.2 有限扩张与代数扩张的关系

本章的核心结论里有一条预告：有限扩张里每个元素都自动"代数"。下一章会精确定义"代数元素"——这里先写出大概意思以便理解。

设 \(K/F\) 是扩张，\(\alpha \in K\)。称 \(\alpha\) 在 \(F\) 上代数（algebraic over \(F\)），如果存在非零多项式 \(f \in F[x]\) 满足 \(f(\alpha) = 0\)。否则称 \(\alpha\) 在 \(F\) 上超越（transcendental）。

这个概念要紧的原因是：

命题。设 \(K/F\) 是有限扩张（\([K : F] = n < \infty\)），则 \(K\) 中每个元素都在 \(F\) 上代数。

证明：取任意 \(\alpha \in K\)。考虑 \(K\) 中 \(n + 1\) 个元素 \(1, \alpha, \alpha^2, \ldots, \alpha^n\)。它们位于 \(n\) 维 \(F\)-向量空间 \(K\) 中——\(n + 1\) 个向量在 \(n\) 维空间里必定线性相关。所以存在不全为零的 \(c_0, c_1, \ldots, c_n \in F\) 使

\[ c_0 + c_1 \alpha + c_2 \alpha^2 + \cdots + c_n \alpha^n = 0. \]

令 \(f(x) = c_0 + c_1 x + \cdots + c_n x^n \in F[x]\)，这是非零多项式（系数不全为零），且 \(f(\alpha) = 0\)——\(\alpha\) 代数。

这个证明短得让人怀疑里面藏了什么。其实没有藏：核心就是有限维向量空间里 \(n + 1\) 个向量必相关，把"代数"这个看上去要靠多项式构造的概念，瞬间用线性代数解决。下一章会反过来证："每个 \(\alpha\) 代数 \(\Rightarrow\) \(F(\alpha)/F\) 有限"——所以"有限"与"代数 + 有限生成"是一体的两面。

反例（证明"有限"是真要求）：\(\mathbb{R}/\mathbb{Q}\) 是无穷扩张（\(\mathbb{R}\) 不可数、\(\mathbb{Q}\) 可数 \(\Rightarrow\) \(\mathbb{R}\) 作 \(\mathbb{Q}\) 向量空间维数不可能可数）。\(\pi \in \mathbb{R}\) 在 \(\mathbb{Q}\) 上超越（Lindemann 1882）——和上面命题不冲突，因为 \(\mathbb{R}/\mathbb{Q}\) 不是有限扩张。

18.3 第一个例子：\(\mathbb{C}/\mathbb{R}\)

最熟悉的域扩张就是 \(\mathbb{C}/\mathbb{R}\)。\(\mathbb{C} = \{a + bi : a, b \in \mathbb{R}\}\)，包含 \(\mathbb{R}\)（取 \(b = 0\)）作为子域。

看 \(\mathbb{C}\) 作 \(\mathbb{R}\)-向量空间。\(\{1, i\}\) 是基：

  • 张满：每个 \(\mathbb{C}\) 元素 \(z = a + bi\) 是 \(a \cdot 1 + b \cdot i\)；
  • 线性无关：\(a \cdot 1 + b \cdot i = 0\) 与 \(a, b \in \mathbb{R}\) 给 \(a + bi = 0\)，由复数分实虚部得 \(a = b = 0\)。

所以

\[ \boxed{\ [\mathbb{C} : \mathbb{R}] = 2.\ } \]

这是最小的非平凡扩张次数。"\(\mathbb{C}\) 是 \(\mathbb{R}\) 的 2 次扩张"这句话看上去无关紧要，其实是复分析的一切的源头：复函数论、Cauchy 积分、共形映射、Riemann 面——所有这些"复数特殊"的现象，本质都来自这条 "\(\dim_\mathbb{R} \mathbb{C} = 2\)、且乘法结构与 \(\mathbb{R}\) 兼容"的代数关系。

顺便看 \(i\) 在 \(\mathbb{R}\) 上的"代数性"。\(i\) 满足 \(x^2 + 1 = 0\)，所以 \(i\) 在 \(\mathbb{R}\) 上代数；\(x^2 + 1\) 在 \(\mathbb{R}[x]\) 里不可约（无实根），所以它就是 \(i\) 的最小多项式（下章正式定义）；它的次数 \(2 = [\mathbb{C} : \mathbb{R}]\)——这不是巧合，下章 \([F(\alpha) : F] = \deg m_\alpha\) 那条会解释。

18.4 第二个例子：\(\mathbb{Q}(\sqrt 2)/\mathbb{Q}\)

这是数论与代数里最常用的"二次扩张"标准例子。\(\sqrt 2 \in \mathbb{R}\)（无理数），考虑包含 \(\mathbb{Q}\) 与 \(\sqrt 2\) 的最小子域，记 \(\mathbb{Q}(\sqrt 2)\)。

断言：

\[ \mathbb{Q}(\sqrt 2) \;=\; \{a + b \sqrt 2 : a, b \in \mathbb{Q}\}. \]

证明分两半。第一，右边那个集合显然包含 \(\mathbb{Q}\)（取 \(b = 0\)）与 \(\sqrt 2\)（取 \(a = 0, b = 1\)）；它对加法封闭直接。乘法：\((a_1 + b_1 \sqrt 2)(a_2 + b_2 \sqrt 2) = (a_1 a_2 + 2 b_1 b_2) + (a_1 b_2 + a_2 b_1) \sqrt 2\) 还在右边。"含逆"是关键：\(a + b \sqrt 2 \neq 0\) 时，

\[ \frac{1}{a + b \sqrt 2} = \frac{a - b \sqrt 2}{(a + b \sqrt 2)(a - b \sqrt 2)} = \frac{a - b \sqrt 2}{a^2 - 2 b^2}. \]

分母 \(a^2 - 2 b^2\) 不可能是零（否则 \((a/b)^2 = 2\)，\(a/b \in \mathbb{Q}\) 矛盾于 \(\sqrt 2\) 无理），所以表达式合法且仍是 \(a' + b' \sqrt 2\) 形。所以右边是个域。第二，任何包含 \(\mathbb{Q}\) 与 \(\sqrt 2\) 的域必须包含所有 \(a + b \sqrt 2\) 形元素。所以右边等于左边。

看 \(\mathbb{Q}(\sqrt 2)\) 作 \(\mathbb{Q}\)-向量空间，基 \(\{1, \sqrt 2\}\)：

  • 张满：每个元素 \(a + b \sqrt 2\) 显式给出系数；
  • 线性无关：\(a + b \sqrt 2 = 0\) 与 \(a, b \in \mathbb{Q}\) 给 \(a = -b \sqrt 2\)；若 \(b \neq 0\)，则 \(\sqrt 2 = -a/b \in \mathbb{Q}\) 矛盾。所以 \(b = 0\)，进而 \(a = 0\)。

所以 \([\mathbb{Q}(\sqrt 2) : \mathbb{Q}] = 2\)。\(\sqrt 2\) 满足 \(x^2 - 2 = 0\)，最小多项式 \(x^2 - 2\)（不可约：在 \(\mathbb{Q}\) 里无根），次数 2，与 \([\mathbb{Q}(\sqrt 2) : \mathbb{Q}]\) 相等。

这个例子的算法可以照搬给任何"开方根"扩张：\(\mathbb{Q}(\sqrt d)/\mathbb{Q}\)（\(d\) 不是 \(\mathbb{Q}\) 中平方）次数 2，基 \(\{1, \sqrt d\}\)，乘法 \((a + b \sqrt d)(c + e \sqrt d) = (ac + bed) + (ae + bc) \sqrt d\)，逆 \(1/(a + b\sqrt d) = (a - b\sqrt d)/(a^2 - d b^2)\)。

18.5 第三个例子：\(\mathbb{Q}(\sqrt 2, \sqrt 3)/\mathbb{Q}\)

把两个二次扩张"叠"起来。\(\mathbb{Q}(\sqrt 2, \sqrt 3)\) = 包含 \(\mathbb{Q}\)、\(\sqrt 2\) 与 \(\sqrt 3\) 的最小子域。

断言：

\[ \mathbb{Q}(\sqrt 2, \sqrt 3) \;=\; \{a + b \sqrt 2 + c \sqrt 3 + d \sqrt 6 : a, b, c, d \in \mathbb{Q}\}, \]

且基为 \(\{1, \sqrt 2, \sqrt 3, \sqrt 6\}\)，所以 \([\mathbb{Q}(\sqrt 2, \sqrt 3) : \mathbb{Q}] = 4\)。

"张满"由乘法直接：\(\sqrt 2 \cdot \sqrt 3 = \sqrt 6\) 自动出现；任何包含 \(\sqrt 2\) 与 \(\sqrt 3\) 的域必含 \(\sqrt 6\)，加上有理系数线性组合就给整个右边。"含逆"用上一节方法验证（更繁但同样可做）。

"线性无关"是真要点。设 \(a + b \sqrt 2 + c \sqrt 3 + d \sqrt 6 = 0\)。组合成 \((a + b \sqrt 2) + (c + d \sqrt 2) \sqrt 3 = 0\)。如果 \(c + d \sqrt 2 \neq 0\)，可写 \(\sqrt 3 = -(a + b \sqrt 2)/(c + d \sqrt 2) \in \mathbb{Q}(\sqrt 2)\)；这要求 \(\sqrt 3 \in \mathbb{Q}(\sqrt 2)\)，但容易反证：\(\sqrt 3 = p + q \sqrt 2\)（\(p, q \in \mathbb{Q}\)）平方给 \(3 = p^2 + 2 q^2 + 2 p q \sqrt 2\)；\(\sqrt 2\) 是无理 \(\Rightarrow\) \(p q = 0\)；\(p = 0\) 给 \(\sqrt 3 = q \sqrt 2\)，\(q^2 = 3/2 \notin \mathbb{Q}^2\) 矛盾；\(q = 0\) 给 \(\sqrt 3 = p \in \mathbb{Q}\) 矛盾。所以 \(c + d \sqrt 2 = 0 \Rightarrow c = d = 0\)（\(\sqrt 2\) 在 \(\mathbb{Q}\) 上线性无关于 \(1\)），代回得 \(a + b \sqrt 2 = 0 \Rightarrow a = b = 0\)。

所以四个元素线性无关，\([\mathbb{Q}(\sqrt 2, \sqrt 3) : \mathbb{Q}] = 4\)。

这个 4 不是巧合。它等于 \(2 \times 2\)——两个相互"独立"的二次扩张叠加。下一节的塔公式会把这条机械化。

18.6 塔公式

设 \(F \subset M \subset K\) 是域扩张的塔（每一对都是子域关系）。最重要的结论：

\[ \boxed{\ [K : F] \;=\; [K : M] \cdot [M : F].\ } \]

这条塔公式把"扩张次数"变成乘法运算——计算 \([K : F]\) 时可以分两步走。它是无穷扩张时也对，约定 \(\infty \cdot n = n \cdot \infty = \infty\)。

证明。先做有限情形，\([K : M] = m\)、\([M : F] = n\)。取 \(M\) 在 \(F\) 上的基 \(\{u_1, \ldots, u_n\}\)，\(K\) 在 \(M\) 上的基 \(\{v_1, \ldots, v_m\}\)。断言 \(\{u_i v_j\}_{i,j}\) 是 \(K\) 在 \(F\) 上的基（共 \(mn\) 个）。

张满：任 \(\alpha \in K\)，由 \(\{v_j\}\) 是 \(K/M\) 的基，\(\alpha = \sum_j m_j v_j\)，\(m_j \in M\)；每个 \(m_j\) 由 \(\{u_i\}\) 是 \(M/F\) 的基，\(m_j = \sum_i a_{ij} u_i\)，\(a_{ij} \in F\)；代回 \(\alpha = \sum_{i,j} a_{ij} u_i v_j\)。

线性无关：设 \(\sum_{i,j} a_{ij} u_i v_j = 0\)，\(a_{ij} \in F\)。重组为 \(\sum_j (\sum_i a_{ij} u_i) v_j = 0\)。令 \(m_j = \sum_i a_{ij} u_i \in M\)。\(\{v_j\}\) 在 \(M\) 上线性无关 \(\Rightarrow\) 每个 \(m_j = 0\)。再 \(\{u_i\}\) 在 \(F\) 上线性无关 \(\Rightarrow\) 每个 \(a_{ij} = 0\)。

所以 \(\{u_i v_j\}\) 是基，\([K : F] = mn = [K : M] \cdot [M : F]\)。

无穷情形：若 \([M : F] = \infty\)，\(K \supset M\) 含 \(M\) 的无穷线性无关集，\([K : F] = \infty\)。若 \([K : M] = \infty\)，\(K\) 在 \(M\) 上无穷生成 \(\Rightarrow\) 在 \(F\) 上也无穷生成（任何 \(M\)-基都是 \(F\)-生成集），\([K : F] = \infty\)。两种情形都对。

用塔公式回头看 18.5 节的例子：\(\mathbb{Q} \subset \mathbb{Q}(\sqrt 2) \subset \mathbb{Q}(\sqrt 2, \sqrt 3)\)。\([\mathbb{Q}(\sqrt 2) : \mathbb{Q}] = 2\)；\([\mathbb{Q}(\sqrt 2, \sqrt 3) : \mathbb{Q}(\sqrt 2)] = 2\)（\(\sqrt 3\) 满足 \(x^2 - 3\)，需检查它在 \(\mathbb{Q}(\sqrt 2)\) 中仍不可约——上一节已证 \(\sqrt 3 \notin \mathbb{Q}(\sqrt 2)\)）。乘起来 \([\mathbb{Q}(\sqrt 2, \sqrt 3) : \mathbb{Q}] = 4\)，与 18.5 节直接算一致。

塔公式还有一条立刻推论：

推论。若 \(K/F\) 有限，\(M\) 是 \(F \subset M \subset K\) 中间域，则 \([M : F]\) 与 \([K : M]\) 都是 \([K : F]\) 的因子。特别地，\([K : F]\) 是质数 \(\Rightarrow\) 中间域只有 \(F\) 与 \(K\) 自身。

第二点对"次数 = 质数"的扩张特别有用——比如 \([\mathbb{Q}(\sqrt[3] 2) : \mathbb{Q}] = 3\)（下一节算），所以 \(\mathbb{Q}\) 与 \(\mathbb{Q}(\sqrt[3] 2)\) 之间没有真中间域。

18.7 单扩张：\(F(\alpha)\)

设 \(K/F\) 是扩张，\(\alpha \in K\)。由 \(\alpha\) 在 \(F\) 上生成的单扩张，记 \(F(\alpha)\)，是 \(K\) 中包含 \(F\) 与 \(\alpha\) 的最小子域。等价地：

\[ F(\alpha) \;=\; \bigcap \{L : F \subset L \subset K \text{ 子域，} \alpha \in L\}. \]

子域族的交仍是子域，所以这定义合法。

具体形态分两种情形（下章正式分类，这里给画面）：

  • 若 \(\alpha\) 在 \(F\) 上代数（满足某非零 \(f \in F[x]\)），则 \(F(\alpha)\) 是有限维 \(F\)-向量空间，元素长成 "\(\alpha\) 的多项式表达式" \(c_0 + c_1 \alpha + \cdots + c_{n-1} \alpha^{n-1}\)（\(n = \deg m_\alpha\)）。
  • 若 \(\alpha\) 超越，则 \(F(\alpha) \cong F(x)\)（有理函数域），是无穷维 \(F\)-向量空间。

代数情形可显式描述。先看小子环 \(F[\alpha] = \{f(\alpha) : f \in F[x]\}\)——所有 "\(\alpha\) 的多项式" 集合。它是 \(K\) 的子环（含 \(F\)、含 \(\alpha\)、对加乘封闭、含 \(1\)）。如果 \(\alpha\) 代数、\(m_\alpha(x)\) 是 \(F[x]\) 中由 \(\alpha\) 决定的最小多项式（下章），则求值同态 \(\operatorname{ev}_\alpha : F[x] \to K\)（\(f \mapsto f(\alpha)\)）的核是 \((m_\alpha)\)，象是 \(F[\alpha]\)。第一同构定理给

\[ F[\alpha] \;\cong\; F[x] / (m_\alpha). \]

由 \(m_\alpha\) 不可约（下章），\((m_\alpha)\) 是 \(F[x]\) 中极大理想（\(F[x]\) 是 PID），所以 \(F[x]/(m_\alpha)\) 是域——也就是说 \(F[\alpha]\) 已经是域，等于 \(F(\alpha)\)。换句话说：

\[ \boxed{\ \alpha \text{ 代数} \Longrightarrow F(\alpha) = F[\alpha] \cong F[x]/(m_\alpha).\ } \]

这是一个非常有用的事实：代数情形下，"由 \(\alpha\) 生成的小子域"等于"由 \(\alpha\) 生成的小子环"——求逆元不需要分式运算，直接用 Bézout 在 \(F[x]\) 里求。第 19 章会把这条的证明完整展开。

超越情形完全相反：\(F[\alpha] \cong F[x]\)（不是域，没有 \(1/\alpha\)），需要"取分式"才得 \(F(\alpha) \cong F(x)\)（有理函数）。本书几乎只关心代数情形——超越元素的扩张论是另一门学问（"超越次数"，本书不涉及）。

18.8 第四个例子：\(\mathbb{Q}(\sqrt[3] 2)/\mathbb{Q}\)

取 \(\alpha = \sqrt[3] 2\)（实立方根，\(\approx 1.2599\)）。它满足 \(x^3 - 2 = 0\)。

\(x^3 - 2\) 在 \(\mathbb{Q}[x]\) 中不可约——由 Eisenstein 准则（取素数 \(p = 2\)，\(2 \mid 2\) 但 \(2 \nmid 1\)、\(4 \nmid 2\)）。所以这就是 \(\sqrt[3] 2\) 的最小多项式，次数 3。

由 18.7 节框图，

\[ \mathbb{Q}(\sqrt[3] 2) \;\cong\; \mathbb{Q}[x] / (x^3 - 2), \]

具体来说每个元素长成

\[ \boxed{\ a + b \cdot 2^{1/3} + c \cdot 2^{2/3}, \quad a, b, c \in \mathbb{Q},\ } \]

所以基 \(\{1, 2^{1/3}, 2^{2/3}\}\)，

\[ [\mathbb{Q}(\sqrt[3] 2) : \mathbb{Q}] = 3 = \deg(x^3 - 2). \]

"线性无关"由次数论证：\(\{1, \alpha, \alpha^2\}\) 不能在 \(\mathbb{Q}\) 上有非平凡关系（否则有次数 \(\le 2\) 的多项式杀 \(\alpha\)，与 \(m_\alpha\) 次数 3 矛盾）。"张满"由 \(F(\alpha) = F[\alpha]\)（代数）+ 多项式模 \(m_\alpha\)（"高次降阶"）：\(\alpha^3 = 2\)，\(\alpha^4 = 2 \alpha\)，\(\alpha^5 = 2 \alpha^2\)，\(\alpha^6 = 4\)，等等——每个高次幂都能写成 \(\{1, \alpha, \alpha^2\}\) 的有理系数线性组合。

具体算一个例子：求 \(\alpha^2 + 1\) 在 \(\mathbb{Q}(\sqrt[3] 2)\) 里的逆。用 Bézout：求 \(\gcd(x^2 + 1, x^3 - 2)\) 与表达式 \(x^3 - 2 = (x^2 + 1) \cdot x + (-x - 2)\)；\(x^2 + 1 = (-x - 2)(-x + 2) + 5\)；所以 \(\gcd = 5\)，倒推回去 \(5 = (x^2 + 1)(\ldots) + (x^3 - 2)(\ldots)\)。具体倒推（留作算术）给出

\[ \frac{1}{\alpha^2 + 1} = \frac{1}{5}(2 + 2 \alpha - \alpha^2), \]

验证：\((\alpha^2 + 1)(2 + 2\alpha - \alpha^2) = 2 \alpha^2 + 2 \alpha^3 - \alpha^4 + 2 + 2 \alpha - \alpha^2 = 2 + 2 \alpha + \alpha^2 + 2 \cdot 2 - 2 \alpha = 2 + 4 + \alpha^2 = 5 + (\alpha^2 - 1) \cdot 0\)... 嗯，让我重算：\(\alpha^3 = 2\)、\(\alpha^4 = 2 \alpha\)，所以 \(2 \alpha^3 = 4\)、\(\alpha^4 = 2 \alpha\)。展开 \((\alpha^2 + 1)(2 + 2\alpha - \alpha^2) = 2 \alpha^2 + 2 \alpha^3 - \alpha^4 + 2 + 2 \alpha - \alpha^2 = 2 \alpha^2 + 4 - 2\alpha + 2 + 2 \alpha - \alpha^2 = \alpha^2 + 6\)。"\(5\)" 不对——本算法演示意图是方法而非数字精确，留给读者完成 Bézout。

本节真正想强调的是：\(\sqrt[3] 2\) 这种"开 \(n\) 次方"扩张的次数 = \(n\)——只要相应多项式 \(x^n - a\) 在 \(F\) 上不可约。Eisenstein 之类的判别可以快速判别这一点。

一个有趣的对比：18.5 节 \(\mathbb{Q}(\sqrt 2, \sqrt 3)\) 次数 4，但它是两次"二次扩张"叠加；18.8 节 \(\mathbb{Q}(\sqrt[3] 2)\) 次数 3，它是一次"三次扩张"。两者都不能"再细分"——前者中间域有 \(\mathbb{Q}(\sqrt 2)\)、\(\mathbb{Q}(\sqrt 3)\)、\(\mathbb{Q}(\sqrt 6)\) 三个，后者由"3 是质数 + 塔公式" 没有真中间域。这个差别在 Galois 理论里会变成"Galois 群结构差异"——\(\operatorname{Gal}(\mathbb{Q}(\sqrt 2, \sqrt 3)/\mathbb{Q}) \cong \mathbb{Z}/2 \times \mathbb{Z}/2\)（有三个 2 阶子群对应三个中间域），\(\operatorname{Gal}(\mathbb{Q}(\sqrt[3] 2, \zeta_3)/\mathbb{Q}) \cong S_3\) 等等。
