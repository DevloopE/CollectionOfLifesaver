第 21 章 Galois 主定理

请你拿铅笔，准备做一道数数题。下面会出现两份清单：一份关于子域，一份关于子群。我让你把两份都数一遍，得到的两个数会相等。然后你会问"为什么"——答案就是 Galois 主定理。

21.1 一个让你数到 5-5 的小实验

看域扩张

\[ K := \mathbb{Q}(\sqrt 2, \sqrt 3) = \{a + b \sqrt 2 + c \sqrt 3 + d \sqrt 6 : a, b, c, d \in \mathbb{Q}\}. \]

这是 \(\mathbb{Q}\) 的 4 维扩张：\(\{1, \sqrt 2, \sqrt 3, \sqrt 6\}\) 是它在 \(\mathbb{Q}\) 上的一组基。\(\sqrt 6 = \sqrt 2 \cdot \sqrt 3\)，所以 4 维不奇怪——你也可以验证 \([\mathbb{Q}(\sqrt 2, \sqrt 3) : \mathbb{Q}] = [\mathbb{Q}(\sqrt 2, \sqrt 3) : \mathbb{Q}(\sqrt 2)] \cdot [\mathbb{Q}(\sqrt 2) : \mathbb{Q}] = 2 \cdot 2 = 4\)（因为 \(\sqrt 3\) 在 \(\mathbb{Q}(\sqrt 2)\) 上仍 2 次：\(\sqrt 3 \notin \mathbb{Q}(\sqrt 2)\) 因为 \(\mathbb{Q}(\sqrt 2) \cap \mathbb{R}^+\) 上正元素都是 \((a + b \sqrt 2)^2 = a^2 + 2 b^2 + 2 ab \sqrt 2\) 形式，等于 3 强迫 \(ab = 0\)，然后 \(a = 0\) 给 \(2 b^2 = 3\) 无理；\(b = 0\) 给 \(a^2 = 3\) 无理）。

清单 1：\(K\) 与 \(\mathbb{Q}\) 之间的所有中间域。"中间域" 指域 \(L\) 满足 \(\mathbb{Q} \subseteq L \subseteq K\)。我数一遍：

  1. \(\mathbb{Q}\) 自己（最小的中间域，1 维）；
  2. \(\mathbb{Q}(\sqrt 2)\)（2 维，含 \(\sqrt 2\) 不含 \(\sqrt 3\)）；
  3. \(\mathbb{Q}(\sqrt 3)\)（2 维，含 \(\sqrt 3\) 不含 \(\sqrt 2\)）；
  4. \(\mathbb{Q}(\sqrt 6)\)（2 维，含 \(\sqrt 6\) 但不 含 \(\sqrt 2\) 或 \(\sqrt 3\)——因为 \((a + b \sqrt 6)^2 = a^2 + 6 b^2 + 2 ab \sqrt 6\) 等于 2 强迫 \(ab = 0\) 然后无解）；
  5. \(K\) 自己（最大的中间域，4 维）。

共 5 个。这是全部——你可能会试 \(\mathbb{Q}(\sqrt 2 + \sqrt 3)\)，但那个数的最小多项式 \(x^4 - 10 x^2 + 1\) 是 4 次（你可以算），所以 \([\mathbb{Q}(\sqrt 2 + \sqrt 3) : \mathbb{Q}] = 4\)，等于 \(K\) 全维——\(\mathbb{Q}(\sqrt 2 + \sqrt 3) = K\)。其它你能想的"新" 子域比如 \(\mathbb{Q}(\sqrt 2 + 2)\) 都和上面 5 个之一相同（这一个等于 \(\mathbb{Q}(\sqrt 2)\)）。

所以中间域恰好 5 个。

清单 2：\(K\) 在 \(\mathbb{Q}\) 上的全部"自同构"。"自同构" 指域同构 \(\sigma : K \to K\) 满足 \(\sigma(q) = q\) 对所有 \(q \in \mathbb{Q}\)。这种 \(\sigma\) 由它在生成元 \(\sqrt 2, \sqrt 3\) 上的值完全决定。\(\sigma\) 必须保持 \(\sqrt 2\) 的最小多项式 \(x^2 - 2\) 的根集合 \(\{\pm \sqrt 2\}\)（因为 \(\sigma(\sqrt 2)^2 = \sigma(\sqrt 2 \cdot \sqrt 2) = \sigma(2) = 2\)），所以 \(\sigma(\sqrt 2) \in \{+\sqrt 2, -\sqrt 2\}\)；同理 \(\sigma(\sqrt 3) \in \{+\sqrt 3, -\sqrt 3\}\)。任意组合都可以——给出 \(2 \times 2 = 4\) 个候选自同构（且它们都确实是自同构，因为 \(\sqrt 2, \sqrt 3\) 的选择互不冲突）：

  • \(\text{id}\)：\(\sqrt 2 \mapsto \sqrt 2\)，\(\sqrt 3 \mapsto \sqrt 3\)；
  • \(\sigma_2\)：\(\sqrt 2 \mapsto -\sqrt 2\)，\(\sqrt 3 \mapsto \sqrt 3\)；
  • \(\sigma_3\)：\(\sqrt 2 \mapsto \sqrt 2\)，\(\sqrt 3 \mapsto -\sqrt 3\)；
  • \(\sigma_2 \sigma_3 = \sigma_{2,3}\)：\(\sqrt 2 \mapsto -\sqrt 2\)，\(\sqrt 3 \mapsto -\sqrt 3\)。

4 个自同构，每个 2 阶（\(\sigma^2 = \text{id}\)），互相可换——这是群 \(\mathbb{Z}_2 \times \mathbb{Z}_2\)。称它为 \(G := \text{Gal}(K/\mathbb{Q})\)。

现在数 \(G\) 的子群。共多少个？

  1. \(\{e\}\)（平凡子群，1 阶）；
  2. \(\langle \sigma_2 \rangle = \{e, \sigma_2\}\)（2 阶）；
  3. \(\langle \sigma_3 \rangle = \{e, \sigma_3\}\)（2 阶）；
  4. \(\langle \sigma_{2,3} \rangle = \{e, \sigma_{2,3}\}\)（2 阶）；
  5. \(G\) 自己（4 阶）。

共 5 个子群。

请你停下来比一下：

  • 5 个中间域 \(\mathbb{Q} \subseteq L \subseteq K\)；
  • 5 个子群 \(\{e\} \subseteq H \subseteq G\)。

5 = 5——这不是巧合。它是同一对集合之间存在双射 的迹象。本节稍后我们会精确给出这双射；现在先让你直觉地匹配：

中间域 \(L\)维 \([L:\mathbb{Q}]\)"对应的" 子群 \(H\)阶 \(|H|\)
\(K\)4\(\{e\}\)1
\(\mathbb{Q}(\sqrt 2)\)2\(\langle \sigma_3 \rangle\)（不动 \(\sqrt 2\)）2
\(\mathbb{Q}(\sqrt 3)\)2\(\langle \sigma_2 \rangle\)（不动 \(\sqrt 3\)）2
\(\mathbb{Q}(\sqrt 6)\)2\(\langle \sigma_{2,3} \rangle\)（不动 \(\sqrt 6\)）2
\(\mathbb{Q}\)1\(G\)4

注意：

  • 对应是反序 的——大子域 \(\leftrightarrow\) 小子群，小子域 \(\leftrightarrow\) 大子群。
  • 维数和阶满足 \([L:\mathbb{Q}] \cdot |H| = [K:\mathbb{Q}] = 4\)，即 \([L:\mathbb{Q}] = [G:H]\)（指数）。
  • "对应" 的具体规则似乎是：子群 \(H\) ↔ "被 \(H\) 全部固定的元素组成的子域"。比如 \(\langle \sigma_3 \rangle = \{e, \sigma_3\}\)：\(\sigma_3\) 把 \(\sqrt 3\) 翻号但保 \(\sqrt 2\)，所以被 \(\sigma_3\) 固定的元素是不含 \(\sqrt 3\) 项的，即形如 \(a + b \sqrt 2\)——正是 \(\mathbb{Q}(\sqrt 2)\)。

这三个观察就是Galois 主定理 的内容。下面把整个故事一般化、严格化、证清楚。

21.2 Galois 扩张：把"双射存在" 的扩张挑出来

不是所有 有限扩张都有 21.1 那样的整齐对应。比如 \(\mathbb{Q}(\sqrt[3]{2})/\mathbb{Q}\) 不正规（ch20.8），它的中间域和"自同构群" 之间对应是残缺 的——自同构只有 \(\{e\}\)（\(\sqrt[3]{2}\) 必须送到自己，因为 \(K\) 是实子域，另两个根复，没地方去），但中间域有 \(\mathbb{Q}\) 与 \(K\) 两个。1 个子群 vs 2 个中间域——双射不存在。

所以"主定理对哪类扩张成立" 是个关键问题。答案：正规 + 可分 + 有限。

定义。有限域扩张 \(K/F\) 称为Galois 扩张，如果它正规 且可分。

在特征 0 域上"可分" 自动（ch20.9），所以"\(K/\mathbb{Q}\) Galois" = "\(K/\mathbb{Q}\) 有限正规"。本课主要在 \(\mathbb{Q}\) 上工作，所以 Galois 扩张实际上 = 某个 \(f \in \mathbb{Q}[x]\) 的分裂域。

Galois 扩张的关键定量性质：

定理。\(K/F\) 是 Galois 扩张 ⟺ \(|\text{Aut}(K/F)| = [K:F]\)。这里 \(\text{Aut}(K/F)\) 是 \(K\) 在 \(F\) 上的自同构群（\(\sigma : K \to K\) 域同构且 \(\sigma|_F = \text{id}\)）。

"自同构数 = 扩张次数" 是 Galois 扩张的定量标志。比如 21.1 中 \(K = \mathbb{Q}(\sqrt 2, \sqrt 3)\)：\([K:\mathbb{Q}] = 4\)，自同构数 \(|G| = 4\)，确实 Galois。反之 \(\mathbb{Q}(\sqrt[3]{2})/\mathbb{Q}\)：\([K:\mathbb{Q}] = 3\)，自同构数 1，不 Galois。

这个定理的证明用 ch20.5 同构延拓引理：把 \(F\)-自同态 \(\sigma : K \to K\) 看成"延拓 \(\text{id}_F : F \to F\) 到 \(K\)" 的方式。每用一次"延拓到下一根" 的一步，可选根数 \(\le\) 该最小多项式根数 \(\le\) 该最小多项式次数；把所有步乘起来 \(\le [K:F]\)。等号成立 ⟺ 每步可选数 = 该次数 ⟺ 每个最小多项式分裂且无重根 ⟺ 正规 + 可分。详细证明展开太长，留作 21.7 节主定理证明的副产物。

可惜 21.1 节我们一开始就玩了"先数自同构再宣布 Galois"——其实 \(\mathbb{Q}(\sqrt 2, \sqrt 3)\) 是 \((x^2 - 2)(x^2 - 3) \in \mathbb{Q}[x]\) 的分裂域，所以正规；特征 0 自然可分；故 Galois。

21.3 Galois 群与基本性质

定义。Galois 扩张 \(K/F\) 的Galois 群 \(\text{Gal}(K/F)\) 是 \(\text{Aut}(K/F)\) 配合函数合成 作运算。

它是个有限群，阶 \([K:F]\)。具体的 \(\sigma \in \text{Gal}(K/F)\) 长这样：把 \(K\) 看作有限维 \(F\)-向量空间，\(\sigma\) 是 \(F\)-线性变换；同时它保乘法（\(\sigma(ab) = \sigma(a)\sigma(b)\)），保 1。这两条加在一起非常严格——不是任意 \(F\)-线性变换都满足。

Galois 群的关键作用方式：它在多项式根集 上作用。设 \(p \in F[x]\) 是 \(F\)-不可约，\(\alpha_1, \ldots, \alpha_n \in K\) 是 \(p\) 的根。任何 \(\sigma \in \text{Gal}(K/F)\) 把 \(\alpha_i\) 送到某个 \(\alpha_j\)——因为 \(\sigma(\alpha_i)\) 满足同一个 \(p\)（系数被 \(\sigma\) 不动），且 \(\sigma\) 是双射所以"送到根" 是根集到根集的双射。所以 Galois 群嵌入根集的置换群。这个观察是 ch22 五次方程不可解的核心工具。

具体例 21.1：\(\text{Gal}(K/\mathbb{Q})\) 作用在 \(\{\sqrt 2, -\sqrt 2\}\) 上由 \(\sigma_2\) 翻号，作用在 \(\{\sqrt 3, -\sqrt 3\}\) 上由 \(\sigma_3\) 翻号——两个独立的 \(\mathbb{Z}_2\) 作用，群是 \(\mathbb{Z}_2 \times \mathbb{Z}_2\)。

21.4 两个标准例子

例 1：\(\text{Gal}(\mathbb{C}/\mathbb{R}) = \mathbb{Z}_2\)。\(\mathbb{C}\) 是 \(\mathbb{R}\) 的 2 维扩张，是 \(x^2 + 1\) 的分裂域，所以正规可分（特征 0），是 Galois 扩张。Galois 群 2 阶。两个元素是 \(\text{id}\) 与复共轭 \(\sigma : a + bi \mapsto a - bi\)。这就是 ch1 节末提到的"\(\mathbb{C}\) 在 \(\mathbb{R}\) 上的 Galois 群是 2 元素群"——历史上 Galois 理论的最简单非平凡例子。

例 2：\(\text{Gal}(\mathbb{Q}(\zeta_p)/\mathbb{Q}) = (\mathbb{Z}/p)^*\)。设 \(p\) 素数，\(\zeta_p := e^{2\pi i / p}\) 是本原 \(p\) 次单位根。\(\zeta_p\) 的最小多项式是分圆多项式

\[ \Phi_p(x) = x^{p-1} + x^{p-2} + \cdots + x + 1 = \frac{x^p - 1}{x - 1}, \]

\(p - 1\) 次不可约（用 Eisenstein 在 \(\Phi_p(x+1)\) 上验证；这是经典练习）。\(\Phi_p\) 的根全部 是 \(\zeta_p^k\)，\(k = 1, \ldots, p-1\)（也就是所有非平凡 \(p\) 次单位根）。所以 \(\mathbb{Q}(\zeta_p)\) 由 \(\zeta_p\) 一个生成且 含 \(\Phi_p\) 全部根（其它 \(\zeta_p^k\) 是 \(\zeta_p\) 的幂），所以 \(\mathbb{Q}(\zeta_p)\) 是 \(\Phi_p\) 的分裂域，正规——Galois。

\(\text{Gal}(\mathbb{Q}(\zeta_p)/\mathbb{Q})\) 的元素 \(\sigma\) 必把 \(\zeta_p\) 送到 \(\Phi_p\) 的某根 \(\zeta_p^k\)，\(k \in \{1, \ldots, p-1\}\)。每个这样的 \(k\) 都给出一个合法 \(\sigma_k : \zeta_p \mapsto \zeta_p^k\)（要验证 \(\sigma_k\) 真是域同构——这是 ch19 单根扩张同构的应用）。所以 Galois 群同构于 \((\mathbb{Z}/p)^* = \{1, 2, \ldots, p-1\}\) 在乘法下——一个 \(p - 1\) 阶循环 群（这是费马小定理的著名推论）。

具体 \(p = 7\)：\((\mathbb{Z}/7)^* = \{1, 2, 3, 4, 5, 6\}\) 6 阶循环（生成元 3 或 5）。\(\mathbb{Q}(\zeta_7)/\mathbb{Q}\) 6 维，6 个自同构，对应 \(\zeta_7^k\) 的 6 种"重新选生成元" 方式。这个例子在 ch22 里讨论正 17 边形作图时会再出现——\(p = 17\) 时 \((\mathbb{Z}/17)^*\) 16 阶循环 = \(2^4\)，是 2 的幂，导致正 17 边形可作图。

21.5 不动域 \(\text{Fix}(H)\)

21.1 节我们注意到：每个子群 \(H\) 对应"被 \(H\) 全部不动的子域"。把这件事一般化命名。

定义。设 \(K\) 是域，\(H\) 是 \(K\) 的某些自同构组成的子群。\(H\) 的不动域 是

\[ \text{Fix}(H) := \{x \in K : \sigma(x) = x \text{ 对所有 } \sigma \in H\}. \]

容易验证 \(\text{Fix}(H)\) 是 \(K\) 的子域：

  • \(0, 1 \in \text{Fix}(H)\) 因为 \(\sigma(0) = 0, \sigma(1) = 1\) 对一切自同构；
  • \(x, y \in \text{Fix}(H) \Rightarrow x \pm y, xy \in \text{Fix}(H)\) 因为 \(\sigma(xy) = \sigma(x)\sigma(y) = xy\) 等；
  • \(x \in \text{Fix}(H), x \neq 0 \Rightarrow x^{-1} \in \text{Fix}(H)\) 因为 \(\sigma(x^{-1}) = \sigma(x)^{-1} = x^{-1}\)。

而且 \(F \subseteq \text{Fix}(H)\) 对任何 \(H \subseteq \text{Gal}(K/F)\)，因为 \(F\) 的元素被所有 \(F\)-自同构不动。所以 \(\text{Fix}(H)\) 是个中间域：\(F \subseteq \text{Fix}(H) \subseteq K\)。

具体例 21.1：

  • \(\text{Fix}(\{e\}) = K\)（恒等不动一切）；
  • \(\text{Fix}(\langle \sigma_2 \rangle)\)：\(\sigma_2\) 不动的元素 = "\(\sqrt 2\) 翻号后还相等" 的元素 = "\(\sqrt 2\) 系数为 0" 的元素 = \(\{a + c \sqrt 3 + d \sqrt 6 \cdot 0 : \ldots\}\)——再细看，\(\sigma_2(a + b\sqrt 2 + c \sqrt 3 + d \sqrt 6) = a - b \sqrt 2 + c \sqrt 3 - d \sqrt 6\)（注意 \(\sqrt 6 = \sqrt 2 \cdot \sqrt 3\)，\(\sigma_2\) 把它送到 \(-\sqrt 2 \cdot \sqrt 3 = -\sqrt 6\)）。等于自身要求 \(b = d = 0\)。所以 \(\text{Fix}(\langle \sigma_2 \rangle) = \{a + c \sqrt 3 : a, c \in \mathbb{Q}\} = \mathbb{Q}(\sqrt 3)\)；
  • 同理 \(\text{Fix}(\langle \sigma_3 \rangle) = \mathbb{Q}(\sqrt 2)\)；
  • \(\text{Fix}(\langle \sigma_{2,3} \rangle)\)：\(\sigma_{2,3}\) 把 \(\sqrt 2 \mapsto -\sqrt 2, \sqrt 3 \mapsto -\sqrt 3, \sqrt 6 \mapsto \sqrt 6\)（两个负号抵消）。等于自身要求 \(b = c = 0\)。所以 \(\text{Fix} = \{a + d \sqrt 6\} = \mathbb{Q}(\sqrt 6)\)；
  • \(\text{Fix}(G) = \mathbb{Q}\)（被全部 4 个自同构同时不动 = 同时 \(b = c = d = 0\)）。

把 \(H\) 与 \(\text{Fix}(H)\) 排在一起，正是 21.1 节末表格里"对应" 的反方向——子群大 的不动域小。

另一方向有个对称的映射：每个中间域 \(L\) 对应它的固定者群

\[ \text{Gal}(K/L) := \{\sigma \in \text{Gal}(K/F) : \sigma|_L = \text{id}\}. \]

这是 \(\text{Gal}(K/F)\) 的子群，且 \(\text{Gal}(K/L) \subseteq \text{Gal}(K/F)\)（条件更紧——固定 \(L\) 比固定 \(F\) 强，因为 \(L \supseteq F\)）。

所以我们手头有两个互逆候选 的映射：

\[ \Phi : \{H \le G\} \to \{\text{中间域}\}, \quad H \mapsto \text{Fix}(H); \] \[ \Psi : \{\text{中间域}\} \to \{H \le G\}, \quad L \mapsto \text{Gal}(K/L). \]

主定理说：在 Galois 扩张的情形下，\(\Phi\) 与 \(\Psi\) 互逆——两个集合大小相等且结构对应整齐。

21.6 Galois 主定理：完整陈述

定理（Galois 主定理）。设 \(K/F\) 是有限 Galois 扩张，\(G := \text{Gal}(K/F)\)。则：

(A) 双射对应。映射

\[ \Phi : H \mapsto \text{Fix}(H), \qquad \Psi : L \mapsto \text{Gal}(K/L) \]

是互逆 的双射，反序 地对应 \(G\) 的子群与 \(F \subseteq L \subseteq K\) 中间域。即：

  • \(\Psi(\Phi(H)) = H\) 对一切 \(H \le G\)；
  • \(\Phi(\Psi(L)) = L\) 对一切中间域 \(L\)；
  • \(H_1 \subseteq H_2 \Leftrightarrow \text{Fix}(H_1) \supseteq \text{Fix}(H_2)\)；
  • \(L_1 \subseteq L_2 \Leftrightarrow \text{Gal}(K/L_1) \supseteq \text{Gal}(K/L_2)\)。

(B) 维数关系。对应的子群与中间域满足

\[ [L : F] = [G : H], \qquad [K : L] = |H|, \]

其中 \(H = \text{Gal}(K/L)\)。

(C) 子扩张的 Galois 性。对任何中间域 \(L\)，扩张 \(K/L\) 也是 Galois 扩张，且 \(\text{Gal}(K/L) = \Psi(L) = H\)。

(D) 正规对应。中间扩张 \(L/F\) 是正规（即 Galois，因可分自动）⟺ \(H = \Psi(L)\) 是 \(G\) 的正规子群。这种情形下

\[ \text{Gal}(L/F) \cong G / H. \]

请你停下来欣赏这条定理。它把"域论问题" 翻译成"群论问题"——这是 Galois 思想的真正威力。比如：

  • "中间域有几个？" → 等价于"\(G\) 子群有几个？"——后者是有限群组合问题，可数可算。
  • "哪个中间域 \(L\) 是 \(F\) 的正规扩张？" → 等价于"哪个子群 \(H\) 是 \(G\) 的正规子群？"——后者用 ch3 的中心、换位子等工具就能查。
  • "扩张 \(K/L\) 怎么样？" → \(\text{Gal}(K/L) = H\) 直接读出。

本质上，有限 Galois 扩张的全部 子结构由它的 Galois 群决定。这是 19 世纪以来代数最深刻的发现之一。

21.7 主定理证明思路

完整证明分几步。我们把每步的主张写清，关键引理证明草图给到，技术细节略。

步 1（Artin 引理）。设 \(K\) 是域，\(H\) 是 \(K\) 的有限自同构子群（任意——不假设 \(K\) 是某个 Galois 扩张顶端）。设 \(F := \text{Fix}(H)\)。则 \(K/F\) 是有限 Galois 扩张，且 \(\text{Gal}(K/F) = H\)，\([K:F] = |H|\)。

这条引理是主定理证明的核心 工具。它把"先选群 \(H\)，再造扩张" 的方向走通了。证明草图：

(i) \(K/F\) 代数：对任 \(\alpha \in K\)，构造多项式 \(p_\alpha(x) := \prod_{\sigma \in H} (x - \sigma(\alpha)) \in K[x]\)。它的系数是 \(H\)-作用的对称函数（基本对称多项式 \(e_k(\sigma_1\alpha, \ldots, \sigma_n\alpha)\)），所以被 \(H\) 不动 \(\Rightarrow\) 在 \(F = \text{Fix}(H)\) 中。所以 \(p_\alpha \in F[x]\)，\(\alpha\) 满足它，\(\alpha\) 在 \(F\) 上代数。次数 \(\le |H|\)。

(ii) \(K/F\) 有限：取 \(\alpha\) 使 \([F(\alpha):F]\) 最大；对任 \(\beta \in K\)，\(F(\alpha, \beta) = F(\gamma)\) 对某 \(\gamma\)（用本原元定理，特征 0 时由 \(\gamma = \alpha + c \beta\) 选合适 \(c\) 给出）。但 \([F(\gamma):F] \ge [F(\alpha):F]\)（左边含右边）；最大性给等号 \(\Rightarrow F(\gamma) = F(\alpha) \Rightarrow \beta \in F(\alpha)\)。所以 \(K = F(\alpha)\)，\([K:F] = \deg(\text{min poly of } \alpha) \le |H|\)。

(iii) \(|\text{Gal}(K/F)| \ge |H|\)：\(H \subseteq \text{Gal}(K/F)\) 因为每个 \(\sigma \in H\) 是 \(K\) 的自同构且不动 \(F = \text{Fix}(H)\)。所以 \(|\text{Gal}(K/F)| \ge |H|\)。

(iv) \([K:F] \ge |\text{Gal}(K/F)|\)：这是Dedekind 字符独立性（distinct field automorphisms are linearly independent over the field）的推论。\(\sigma_1, \ldots, \sigma_m \in \text{Gal}(K/F)\) 不同，看作 \(K \to K\) 的函数，则它们 \(K\)-线性独立；它们也是 \(F\)-线性独立；但它们都把 \(F\) 不动 \(\Rightarrow\) 它们决定于在 \(K\) 的某 \(F\)-基上的值；这把 \(m \le \dim_F K = [K:F]\) 给出。

把 (i)-(iv) 串：\(|H| \ge [K:F]\)（从 (ii)）\(\ge |\text{Gal}(K/F)|\)（从 (iv)）\(\ge |H|\)（从 (iii)）。三个 \(\ge\) 形成环，全是等号。所以 \([K:F] = |\text{Gal}(K/F)| = |H|\)。再加上 \(H \subseteq \text{Gal}(K/F)\) 与等阶 \(\Rightarrow H = \text{Gal}(K/F)\)。最后 \([K:F] = |\text{Gal}(K/F)| \Rightarrow K/F\) 是 Galois（21.2 定理）。\(\square\)（草图）

步 2（主定理 (A)(B)(C)）。给定 Galois \(K/F\)，\(G = \text{Gal}(K/F)\)。

\(\Psi \circ \Phi = \text{id}_{\{H\}}\)：取 \(H \le G\)。由 Artin 引理（\(K\) 上自同构群 \(H\) 给的不动域 \(\text{Fix}(H)\)），\(K/\text{Fix}(H)\) 是 Galois 且 \(\text{Gal}(K/\text{Fix}(H)) = H\)。但 \(\Psi(\Phi(H)) = \Psi(\text{Fix}(H)) = \text{Gal}(K/\text{Fix}(H)) = H\)。完成。

\(\Phi \circ \Psi = \text{id}_{\{L\}}\)：取中间域 \(L\)。要证 \(\text{Fix}(\text{Gal}(K/L)) = L\)。一边 \(L \subseteq \text{Fix}(\text{Gal}(K/L))\) 显然（\(\text{Gal}(K/L)\) 元素不动 \(L\)）。另一边：在 \(L\) 上 \(K/L\) 是 Galois 扩张（这一点是 (C) 的内容；它的证明是：\(K/F\) 正规 \(\Rightarrow\) \(K\) 含 \(F\)-不可约多项式 \(f\) 全部根；同 \(f\) 看作 \(L\)-不可约因子的乘积，每个因子的根仍在 \(K\) 内；所以 \(K/L\) 也正规。可分性同样从 \(F\) 继承到 \(L\)）。由 Artin 应用于 \(K\) 与 \(\text{Gal}(K/L)\)：\(\text{Fix}(\text{Gal}(K/L)) = \text{ground field}\) 是某个域；这个域既包含 \(L\) 又被 \(\text{Gal}(K/L)\) 不动（按定义）；最关键是它的维数——\([K : \text{Fix}(\text{Gal}(K/L))] = |\text{Gal}(K/L)| = [K:L]\)（最后等号是 \(K/L\) Galois）。所以 \(\text{Fix}(\text{Gal}(K/L)) = L\) 因为它在 \(K\) 中含 \(L\) 且与 \(L\) 同维。完成。

反序对应、维数关系都从这两个互逆 + Artin 引理直接读出。

步 3（主定理 (D)：正规对应）。设 \(L\) 中间域，\(H = \Psi(L) = \text{Gal}(K/L)\)。

(\(\Rightarrow\)) 设 \(L/F\) 正规。对任意 \(\sigma \in G\) 与任意 \(\tau \in H\)，要证 \(\sigma \tau \sigma^{-1} \in H\)，即 \(\sigma \tau \sigma^{-1}|_L = \text{id}\)。取 \(\alpha \in L\)，要算 \(\sigma \tau \sigma^{-1}(\alpha)\)。设 \(\beta := \sigma^{-1}(\alpha)\)。问 \(\beta\) 在 \(L\) 里吗？\(\beta\) 满足 \(\alpha\) 的最小多项式 \(m_\alpha \in F[x]\)（因为 \(\sigma\) 把 \(\beta\) 送到 \(\alpha\)，而 \(m_\alpha \in F[x] \subseteq L[x]\) 在 \(\sigma\) 下不动），所以 \(\beta\) 是 \(m_\alpha\) 的某根。\(L/F\) 正规 \(\Rightarrow\) \(L\) 含 \(m_\alpha\) 全部根 \(\Rightarrow\) \(\beta \in L\)。因此 \(\tau(\beta) = \beta\)（\(\tau \in H\) 不动 \(L\)），所以 \(\sigma \tau \sigma^{-1}(\alpha) = \sigma \tau (\beta) = \sigma(\beta) = \alpha\)。所以 \(\sigma \tau \sigma^{-1}\) 不动 \(\alpha\)；对一切 \(\alpha \in L\) 有效 \(\Rightarrow \sigma \tau \sigma^{-1} \in H\)。所以 \(H \trianglelefteq G\)。

(\(\Leftarrow\)) 设 \(H \trianglelefteq G\)。要证 \(L/F\) 正规：取 \(\alpha \in L\) 与它的 \(F\)-最小多项式 \(m_\alpha \in F[x]\)；要证 \(m_\alpha\) 全部根在 \(L\) 中。\(K/F\) 正规 \(\Rightarrow\) \(m_\alpha\) 全部根在 \(K\) 中（设为 \(\alpha = \alpha_1, \ldots, \alpha_r\)）。每个 \(\alpha_i\) 是 \(\sigma_i(\alpha)\) 对某 \(\sigma_i \in G\)（这是 ch20.5 同构延拓引理的应用——任何 \(F\)-同构 \(F(\alpha) \to F(\alpha_i)\) 延拓到 \(K\)）。要证 \(\alpha_i \in L\)，等价于 \(\sigma_i(\alpha)\) 被 \(H\) 全部不动（因 \(L = \text{Fix}(H)\)）。对任 \(\tau \in H\)：\(\tau(\sigma_i(\alpha)) = \sigma_i(\sigma_i^{-1} \tau \sigma_i)(\alpha)\)。括号里的 \(\sigma_i^{-1} \tau \sigma_i \in H\)（\(H\) 正规）。所以它不动 \(L \ni \alpha\)。所以 \(\sigma_i^{-1} \tau \sigma_i (\alpha) = \alpha\)，所以 \(\tau(\sigma_i(\alpha)) = \sigma_i(\alpha) = \alpha_i\)。\(\tau\) 任意 \(\Rightarrow \alpha_i \in \text{Fix}(H) = L\)。\(L/F\) 正规。

商群同构 \(\text{Gal}(L/F) \cong G/H\) 是限制映射 \(G \to \text{Gal}(L/F)\)（\(\sigma \mapsto \sigma|_L\)）的诱导：核 = "限制到 \(L\) 是 id 的 \(\sigma\)" = \(H\)；像（满）由维数算 \(|G|/|H| = [L:F] = |\text{Gal}(L/F)|\)；第一同构定理给商。\(\square\)

这条证明是抽象代数的标杆性证明之一——它融合 了向量空间维数（步 1 的 (iv)）、群作用（步 1 的 (i)）、商环（ch19 单根扩张、ch20.5 同构延拓引理）、同构定理（步 3 的最后）。学完它你算是真懂 了 ch1-20 所有工具的合奏。

21.8 把 21.1 例子完整跑通

回到 \(K = \mathbb{Q}(\sqrt 2, \sqrt 3)\) 与 \(G = \mathbb{Z}_2 \times \mathbb{Z}_2 = \{e, \sigma_2, \sigma_3, \sigma_{2,3}\}\)。把主定理的对应一字一句对完。

验证：

  • 反序：\(\{e\} \subset \langle \sigma_2 \rangle, \langle \sigma_3 \rangle, \langle \sigma_{2,3} \rangle \subset G\) ↔ \(K \supset \mathbb{Q}(\sqrt 2), \mathbb{Q}(\sqrt 3), \mathbb{Q}(\sqrt 6) \supset \mathbb{Q}\)。\(\checkmark\)
  • 维数：每行 \([L:\mathbb{Q}] \cdot [K:L] = [K:\mathbb{Q}] = 4\)。\(\checkmark\)
  • 所有中间扩张都正规：因为 \(G\) 阿贝尔 \(\Rightarrow\) 一切子群正规 \(\Rightarrow\) 一切中间扩张是 Galois 的。\(\checkmark\)

三个 2 阶子群对应的子扩张 \(L/\mathbb{Q}\) 都是 Galois，\(\text{Gal}(L/\mathbb{Q}) \cong G / H\)：

取 \(L = \mathbb{Q}(\sqrt 2)\)，\(H = \langle \sigma_3 \rangle = \{e, \sigma_3\}\)。\(G/H = \{H, \sigma_2 H\} \cong \mathbb{Z}_2\)。直接看：\(\text{Gal}(\mathbb{Q}(\sqrt 2)/\mathbb{Q}) = \{\text{id}, \sqrt 2 \mapsto -\sqrt 2\} = \mathbb{Z}_2\)。\(\checkmark\)

非平凡限制例：\(\mathbb{Q}(\sqrt 2 + \sqrt 3)\) 是哪个中间域？

设 \(\alpha := \sqrt 2 + \sqrt 3\)。\(\alpha \in K\)。\(\mathbb{Q}(\alpha)\) 是某个中间域；要确定哪个。用主定理：\(\mathbb{Q}(\alpha)\) 对应于 \(\text{Gal}(K / \mathbb{Q}(\alpha)) = \{\sigma \in G : \sigma(\alpha) = \alpha\}\)。算每个 \(\sigma\) 作用：

  • \(e(\alpha) = \sqrt 2 + \sqrt 3 = \alpha\)。\(\checkmark\)
  • \(\sigma_2(\alpha) = -\sqrt 2 + \sqrt 3 \neq \alpha\)。
  • \(\sigma_3(\alpha) = \sqrt 2 - \sqrt 3 \neq \alpha\)。
  • \(\sigma_{2,3}(\alpha) = -\sqrt 2 - \sqrt 3 = -\alpha \neq \alpha\)。

只有 \(e\) 不动 \(\alpha\)，所以 \(\text{Gal}(K/\mathbb{Q}(\alpha)) = \{e\}\)，对应中间域 \(\mathbb{Q}(\alpha) = K\)。所以 \(\mathbb{Q}(\sqrt 2 + \sqrt 3) = K = \mathbb{Q}(\sqrt 2, \sqrt 3)\)——这是 21.1 节末提到的"本原元" 现象。一个看似只用一根的 \(\mathbb{Q}(\alpha)\) 等于一个用两根的 \(\mathbb{Q}(\sqrt 2, \sqrt 3)\)。

主定理告诉你这件事是系统的——对任何有限 Galois \(K/F\) 在特征 0 上，本原元定理（Steinitz）保证 \(K = F(\alpha)\) 对某 \(\alpha\)（21.7 步 1 (ii) 顺手证了这个）。所以一切有限 Galois 扩张都是单 扩张——一个根就够了。

21.9 这一章我们做了什么

从一个数数题出发——\(\mathbb{Q}(\sqrt 2, \sqrt 3)\) 有 5 个中间域，它的自同构群 \(\mathbb{Z}_2 \times \mathbb{Z}_2\) 有 5 个子群，整齐相等。把这件事一般化：

  • 挑出"这种好对应成立" 的扩张——Galois 扩张（正规 + 可分 + 有限），用"自同构数 = 扩张次数" 作定量标志。
  • 定义两个候选互逆映射：\(\Phi(H) = \text{Fix}(H)\) 与 \(\Psi(L) = \text{Gal}(K/L)\)。
  • Galois 主定理：在 Galois 扩张上，\(\Phi\) 与 \(\Psi\) 互逆——子群与中间域反序双射，正规子群 ↔ 正规中间扩张，商群 = 中间扩张的 Galois 群。
  • 证明的核心是Artin 引理——给定有限自同构群 \(H\)，它的不动域自动给一个 Galois 扩张。这条引理把"先群后域" 的方向走通。
  • 把 \(\mathbb{Q}(\sqrt 2, \sqrt 3)\) 例子完整对完，包括"\(\sqrt 2 + \sqrt 3\) 生成整个 \(K\)" 的本原元现象。

这一章是本书的高峰之一。你掌握了"用群论控制域扩张" 的语言。下一章把这个语言用在两个古典数学问题 上——古希腊就有的尺规作图三大难题，与 19 世纪初困扰所有数学家的"五次方程根式解" 问题。两件事都靠 Galois 主定理给出最终判决。
