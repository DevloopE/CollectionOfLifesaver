第 23 章 应用：编码理论与有限域

本书前面 22 章把抽象代数的骨架立起来了——群、环、域、模、Galois 理论。这一章和下一章把这副骨架放回现实世界，看它在两个具体应用领域里怎么变活——这一章是密码学和编码理论，下一章是晶体学和表示论。两章的目的不是把这两个领域当独立学科教完——那要各自一本书——而是让你看到：抽象代数不是数学家自娱自乐的语言游戏，它是 21 世纪通信、芯片、互联网、核磁共振、X 射线晶体学的底层语法。

这一章我们要解的具体问题是：怎么造出一个 8 元素的域？听起来像是个无关紧要的技术问题。但它把"什么是域"" 有限域分类"" 编码理论怎么用域"" 密码学怎么用环" 这一长串话题串起来。读完这一章你会看到：你手机里的 SD 卡纠错、CD 光盘抗刮、卫星通信抗噪、QR 二维码识别、HTTPS 网络加密——背后同一套有限域代数。

23.1 一个让你警觉的钩子：\(\mathbb{Z}_8\) 不是域

从最朴素的尝试开始。我们要 8 个元素的域。第一 反应：取 \(\mathbb{Z}_8 = \mathbb{Z}/8\mathbb{Z}\) 不就完了吗？8 个元素 \(\{0, 1, 2, 3, 4, 5, 6, 7\}\)，加法和乘法都是模 8 的标准运算。

问题是：这不是域。看：

\[ 2 \cdot 4 = 8 \equiv 0 \pmod 8. \]

两个非零元素相乘等于 0。这就是第 13 章讲的零因子。域里不允许有零因子（因为域 ⟹ 整环 ⟹ 无零因子）。所以 \(\mathbb{Z}_8\) 顶多是个环，不是域。

更具体的，\(2\) 在 \(\mathbb{Z}_8\) 里没有乘法逆——你解 \(2 x \equiv 1 \pmod 8\)，把 \(x = 0, 1, \ldots, 7\) 代进去：\(2x = 0, 2, 4, 6, 0, 2, 4, 6\)，从来不到 1。\(2\) 在 \(\mathbb{Z}_8\) 里是不可逆 的。

同样的毛病不只 8 有。\(\mathbb{Z}_n\) 是域当且仅当 \(n\) 是素数（这条事实第 13 章 13.4 节正式证过：\(\mathbb{Z}/n\mathbb{Z}\) 是域 ⟺ \(n\) 素 ⟺ \(\mathbb{Z}/n\mathbb{Z}\) 整环）。\(n\) 不素时总有 \(n = ab\) with \(1 < a, b < n\)，于是 \(a \cdot b \equiv 0\) 是零因子。

所以"把 \(\mathbb{Z}\) 模一个数" 这条最朴素的造域路只通向素数大小的域：\(\mathbb{F}_2, \mathbb{F}_3, \mathbb{F}_5, \mathbb{F}_7, \mathbb{F}_{11}, \ldots\) 这些可以；\(\mathbb{F}_4, \mathbb{F}_8, \mathbb{F}_9\) 这些朴素 路走不通。

请你停下来想：是不是大小为合数的域根本不存在？这是个很自然的猜测。如果你心里这样想，下一节会让你失望——8 元素的域真 存在，只是不长成 \(\mathbb{Z}_8\) 的样子。这种"朴素的造法走不通但对象本身存在" 的局面，正是这一章的核心。

23.2 撞墙后的 reframe：\(\mathbb{F}_8 \neq \mathbb{Z}/8\mathbb{Z}\)

朴素方法走不通。我们换个思路。回想第 15 章 15.5 节：在多项式环 \(k[x]\) 里，把 \(k[x]\) 模一个不可约多项式 \(p(x)\)\) 得到的商环 \(k[x] / (p(x))\) 是域。这条事实第 15 章已经证了——不可约多项式生成极大理想，第 13 章命题"商环模极大理想是域"立刻给结论。

把这条用到 \(k = \mathbb{F}_2\)。考虑多项式 \(g(x) = x^3 + x + 1 \in \mathbb{F}_2[x]\)。这是3 次 多项式，在 \(\mathbb{F}_2\) 上不可约——验证：\(g(0) = 1 \neq 0\)，\(g(1) = 1 + 1 + 1 = 1 \neq 0\)（在 \(\mathbb{F}_2\) 里 \(1 + 1 = 0\)，所以 \(1 + 1 + 1 = 1\)），所以没有 \(\mathbb{F}_2\) 里的根。3 次没根的多项式不可约（如果可约，必有 1 次因子，等价于有根）。

所以 \(\mathbb{F}_2[x] / (x^3 + x + 1)\) 是一个域。它有多少元素？

商环里的元素是 \(\mathbb{F}_2[x]\) 里多项式的等价类——两个多项式同类当且仅当它们的差被 \(g(x)\) 整除。每个等价类有唯一 的次数 \(< 3\) 的代表：用 \(g(x)\) 长除法，每个 \(f(x)\) 写成 \(f(x) = q(x) g(x) + r(x)\)，\(\deg r < 3\)，\(r\) 就是代表。

所以等价类一一对应于次数 \(< 3\) 的多项式 \(a_0 + a_1 x + a_2 x^2\)（\(a_i \in \mathbb{F}_2\)）。这种多项式有 \(2^3 = 8\) 个：

\[ 0, \quad 1, \quad x, \quad x + 1, \quad x^2, \quad x^2 + 1, \quad x^2 + x, \quad x^2 + x + 1. \]

正好 8 个元素。这就是 \(\mathbb{F}_8\)：

\[ \boxed{\ \mathbb{F}_8 := \mathbb{F}_2[x] / (x^3 + x + 1).\ } \]

请你停下来对比两个东西。\(\mathbb{Z}_8\) 的元素是整数（\(\{0, 1, \ldots, 7\}\)），运算是整数模 8。\(\mathbb{F}_8\) 的元素是多项式（次数 \(< 3\)，系数在 \(\{0, 1\}\)），运算是多项式加法和乘法 + 把超过 2 次的项用 \(x^3 = x + 1\) 化简。它们都是 8 元素的代数对象，但结构完全不同——一个有零因子，一个没有；一个不是域，一个是。

这种"真的造出 8 元素域，但要走多项式商环的路" 是有限域全部秘密的入口。下一节给一般的事实。

23.3 命名：有限域基本事实

把上一节的现象推广。设 \(\mathbb{F}\) 是任何 有限域。问：\(|\mathbb{F}|\) 可以是哪些数？

定理（有限域大小定理）。 任何有限域 \(\mathbb{F}\) 的元素个数必为某素数幂 \(p^n\)（\(p\) 素，\(n \ge 1\)）。反之，对每个素数幂 \(q = p^n\)，恰好存在一个（在同构意义下）大小为 \(q\) 的域，记作 \(\mathbb{F}_q\) 或 \(\mathrm{GF}(q)\)（"Galois Field"）。

这条定理的两个方向都不是平凡的，但每一条的思路都能看见，让我把它们各讲一遍。

方向 1：\(|\mathbb{F}|\) 必为素数幂。

设 \(\mathbb{F}\) 是有限域。看 \(\mathbb{F}\) 的特征——即满足 \(p \cdot 1 = 0\) 的最小正整数 \(p\)（\(\mathbb{F}\) 有限所以这种 \(p\) 必存在）。\(p\) 必是素数：若 \(p = ab\) with \(a, b > 1\)，则 \(0 = p \cdot 1 = (a \cdot 1)(b \cdot 1)\)，但域无零因子，所以 \(a \cdot 1 = 0\) 或 \(b \cdot 1 = 0\)，与 \(p\) 是最小 的矛盾。

所以 \(\mathbb{F}\) 包含一个同构于 \(\mathbb{F}_p\) 的子域（由 \(\{0, 1, 2 \cdot 1, \ldots, (p-1) \cdot 1\}\) 组成）。把 \(\mathbb{F}\) 看成 \(\mathbb{F}_p\) 上的向量空间——加法是 \(\mathbb{F}\) 的加法，标量乘是 \(\mathbb{F}_p\) 元素乘 \(\mathbb{F}\) 元素。\(\mathbb{F}\) 有限所以这个向量空间是有限维，设维数为 \(n\)。则 \(\mathbb{F}\) 作为 \(\mathbb{F}_p\)-向量空间同构于 \(\mathbb{F}_p^n\)，元素个数 \(p^n\)。

这条证明的关键一步是把域看成它的素子域上的向量空间——这是一个第 18 章反复用到的视角（"扩张次数 = 维数"），这里再现一次。

方向 2：每个 \(p^n\) 对应唯一的 \(\mathbb{F}_q\)。

"存在" 部分：用上一节的方法。在 \(\mathbb{F}_p[x]\) 里挑一个 \(n\) 次不可约 多项式 \(g(x)\)（这种总存在，事实是 \(\mathbb{F}_p[x]\) 里每个次数都有不可约多项式——下一节讨论怎么找）；做商 \(\mathbb{F}_p[x] / (g(x))\)；它是 \(p^n\) 元素的域。

"唯一" 部分（同构意义下）：这条更深，要用第 20 章 20.3 节的"分裂域唯一" 定理。具体的，\(\mathbb{F}_q\) 同构于多项式 \(x^q - x \in \mathbb{F}_p[x]\) 的分裂域——这条的关键观察是：\(\mathbb{F}_q\) 里每个元素 \(a\) 都满足 \(a^q = a\)（这是有限群 \(\mathbb{F}_q^*\) 的 Lagrange——\(\mathbb{F}_q^*\) 阶 \(q - 1\)，所以 \(a^{q-1} = 1\) 对每个 \(a \neq 0\)，再乘 \(a\) 得 \(a^q = a\)；\(a = 0\) 也满足）。所以 \(\mathbb{F}_q\) 的所有元素都是 \(x^q - x\) 的根；这个 \(q\) 次多项式恰好有 \(q\) 个根；这就是 \(x^q - x\) 在 \(\mathbb{F}_q\) 里完全分裂。两个 \(p^n\) 元素的域都是 \(x^{p^n} - x\) 的分裂域，由第 20 章分裂域唯一性，两者同构。

请你欣赏这个结果有多干净：每个素数幂大小恰好一种域，没有大小不是素数幂的有限域。这是有限域世界的全图——\(\mathbb{F}_2, \mathbb{F}_3, \mathbb{F}_4, \mathbb{F}_5, \mathbb{F}_7, \mathbb{F}_8, \mathbb{F}_9, \mathbb{F}_{11}, \mathbb{F}_{13}, \mathbb{F}_{16}, \mathbb{F}_{17}, \mathbb{F}_{19}, \mathbb{F}_{23}, \mathbb{F}_{25}, \mathbb{F}_{27}, \mathbb{F}_{29}, \mathbb{F}_{31}, \mathbb{F}_{32}, \mathbb{F}_{37}, \ldots\)。中间没有 \(\mathbb{F}_6, \mathbb{F}_{10}, \mathbb{F}_{12}, \mathbb{F}_{14}, \mathbb{F}_{15}, \ldots\)——这些"非素数幂大小的域" 根本不存在。

23.4 跑一遍：\(\mathbb{F}_4, \mathbb{F}_8, \mathbb{F}_9\) 的乘法表

抽象的话讲完，我们做点具体的——把三个最小的"非素 数大小有限域" 的乘法表写出来，让你看见 它们怎么运作。

\(\mathbb{F}_4 = \mathbb{F}_2[x] / (x^2 + x + 1)\)。 验证 \(x^2 + x + 1\) 在 \(\mathbb{F}_2\) 上不可约：\(g(0) = 1, g(1) = 1\)，没根。商有 4 个元素：\(0, 1, x, x + 1\)。把 \(x\) 称为 \(\alpha\)（满足 \(\alpha^2 + \alpha + 1 = 0\)，即 \(\alpha^2 = \alpha + 1\)）。

乘法表（\(\mathbb{F}_4\) 的所有非零元 \(\{1, \alpha, \alpha + 1\}\) 之间的乘法）：

×1\(\alpha\)\(\alpha + 1\)
11\(\alpha\)\(\alpha + 1\)
\(\alpha\)\(\alpha\)\(\alpha + 1\)1
\(\alpha + 1\)\(\alpha + 1\)1\(\alpha\)

验证一行：\(\alpha \cdot \alpha = \alpha^2 = \alpha + 1\)。\(\alpha \cdot (\alpha + 1) = \alpha^2 + \alpha = (\alpha + 1) + \alpha = 1\)（在特征 2 里 \(2\alpha = 0\)）。\((\alpha + 1)^2 = \alpha^2 + 1 = (\alpha + 1) + 1 = \alpha\)。漂亮——表对称、每行每列正好是 \(\{1, \alpha, \alpha + 1\}\) 的一个置换（这就是群 的乘法表的特征——下一节我们看到 \(\mathbb{F}_q^*\) 是循环群）。

\(\mathbb{F}_8 = \mathbb{F}_2[x] / (x^3 + x + 1)\)。 设 \(\alpha = x \mod g\)。则 \(\alpha^3 = \alpha + 1\)。8 个元素：\(0, 1, \alpha, \alpha^2, \alpha + 1, \alpha^2 + 1, \alpha^2 + \alpha, \alpha^2 + \alpha + 1\)。

不全列乘法表——8×8 表 64 项。我列非零元的幂：

\(k\)\(\alpha^k\)（用 \(\alpha^3 = \alpha + 1\) 化简）
0\(1\)
1\(\alpha\)
2\(\alpha^2\)
3\(\alpha + 1\)
4\(\alpha^2 + \alpha\)
5\(\alpha^2 + \alpha + 1\)
6\(\alpha^2 + 1\)
7\(1\)（回到起点）

验证 \(\alpha^4 = \alpha \cdot \alpha^3 = \alpha(\alpha + 1) = \alpha^2 + \alpha\)。\(\alpha^5 = \alpha \cdot \alpha^4 = \alpha(\alpha^2 + \alpha) = \alpha^3 + \alpha^2 = (\alpha + 1) + \alpha^2 = \alpha^2 + \alpha + 1\)。\(\alpha^6 = \alpha \cdot \alpha^5 = \alpha^3 + \alpha^2 + \alpha = (\alpha + 1) + \alpha^2 + \alpha = \alpha^2 + 1\)。\(\alpha^7 = \alpha \cdot \alpha^6 = \alpha^3 + \alpha = (\alpha + 1) + \alpha = 1\)。正好 7 步回到 1——这意味着 \(\mathbb{F}_8^* = \{1, \alpha, \alpha^2, \ldots, \alpha^6\}\) 是个 7 元素循环群，由 \(\alpha\) 生成。

有了这张幂表，乘法变得极容易：把每个非零元写成 \(\alpha^k\)，相乘加指数，模 7 化简。比如 \((\alpha^2 + \alpha) \cdot (\alpha^2 + 1) = \alpha^4 \cdot \alpha^6 = \alpha^{10} = \alpha^3 = \alpha + 1\)。把"乘法表" 替换成"指数表 + 加法 mod 7"——这是有限域计算的标准技巧（"对数表" 法），密码学里也叫"Zech logarithm"。

\(\mathbb{F}_9 = \mathbb{F}_3[x] / (x^2 + 1)\)。 验证 \(x^2 + 1\) 在 \(\mathbb{F}_3\) 上不可约：\(g(0) = 1, g(1) = 2, g(2) = 4 + 1 = 5 = 2\)，没根。9 个元素：\(\{a + b i : a, b \in \mathbb{F}_3\}\)（用 \(i\) 标记 \(x \mod g\)，满足 \(i^2 = -1 = 2\)）：

\[ 0, 1, 2, i, 1 + i, 2 + i, 2i, 1 + 2i, 2 + 2i. \]

这看起来就像复数模 3——这是 \(\mathbb{F}_9\) 的常用画法。乘法： \((1 + i)(2 + i) = 2 + i + 2i + i^2 = 2 + 3i + 2 = 4 + 0 = 1\)（mod 3）。所以 \(2 + i\) 是 \(1 + i\) 的乘法逆。

请你停下来想一件事：\(\mathbb{F}_9\) 是用 \(x^2 + 1\) 造的，但 \(\mathbb{F}_9\) 本身和你用什么不可约多项式造无关——上一节定理说"\(p^n\) 元素的域唯一同构"。如果你换成 \(\mathbb{F}_3[x] / (x^2 + x + 2)\)（也是不可约的 2 次），得到的是同一 个域，只是元素的"名字" 不同。具体的，两者之间有一个明确的同构。这种"构造方式不唯一但结果唯一" 是有限域世界最舒服的事——你可以选最方便的不可约多项式，结果不变。

23.5 一般化（一）：Frobenius 自同构

有限域有一个特殊的自同构，是有限域 Galois 理论的核心——它叫 Frobenius 自同构。

设 \(\mathbb{F}_q\)（\(q = p^n\)）是个有限域。定义

\[ \boxed{\ \sigma : \mathbb{F}_q \to \mathbb{F}_q, \quad \sigma(a) = a^p.\ } \]

断言：\(\sigma\) 是 \(\mathbb{F}_q\) 的自同构（双射 + 保持加法乘法）。

"保持乘法" 显然：\(\sigma(ab) = (ab)^p = a^p b^p = \sigma(a) \sigma(b)\)。

"保持加法" 不显然——一般情况下 \((a + b)^p \neq a^p + b^p\)（你以前学过的二项式展开！）。但在特征 \(p\) 的环里：

\[ (a + b)^p = \sum_{k=0}^p \binom{p}{k} a^k b^{p-k}. \]

关键在于中间项 \(\binom{p}{k}\)（\(0 < k < p\)）都被 \(p\) 整除——因为 \(\binom{p}{k} = p! / (k!(p-k)!)\)，分子有一个 \(p\) 因子，分母（\(k! (p-k)!\)，\(k, p-k < p\)）没有 \(p\)（\(p\) 素），所以分数 \(= p \cdot (\text{某整数})\)。在特征 \(p\) 域里 \(p \cdot 1 = 0\)，所以这些中间项消失，剩下

\[ (a + b)^p = a^p + b^p. \]

这条"新生神奇式"（freshman's dream）在 \(\mathbb{R}\) 或 \(\mathbb{Q}\) 里是错的——但在特征 \(p\) 的世界里对的。这是有限域 / 特征 \(p\) 代数的最反直觉也最重要的事实。

"双射" 部分：\(\sigma\) 是有限集到自身的同态，且核 \(\{a : a^p = 0\} = \{0\}\)（因为域无零因子，\(a^p = 0 \Rightarrow a = 0\)），所以单射；有限集自身的单射自动满射；所以双射。\(\square\)

所以 \(\sigma\) 是 \(\mathbb{F}_q\) 的自同构。它的不动点 是哪些？\(\sigma(a) = a \Leftrightarrow a^p = a \Leftrightarrow a^p - a = 0\)。这是一个 \(p\) 次方程，最多 \(p\) 个根。但 \(\mathbb{F}_p \subseteq \mathbb{F}_q\) 里所有元素都满足 \(a^p = a\)（Fermat 小定理 + 0），所以 \(\mathbb{F}_p\) 里这 \(p\) 个元素就是全部根。

结论：\(\sigma\) 的不动点子域 \(= \mathbb{F}_p\)。

再做一件事：\(\sigma\) 的阶 是多少？即多少次 \(\sigma\) 复合等于 \(\text{id}\)？\(\sigma^k(a) = a^{p^k}\)，所以 \(\sigma^k = \text{id} \Leftrightarrow a^{p^k} = a\) 对所有 \(a \in \mathbb{F}_q\) 成立 \(\Leftrightarrow p^k \ge q\)（精确说 \(p^k \equiv 1 \mod (q - 1)\)，但因为 \(\mathbb{F}_q\) 整体满足 \(a^q = a\)，最小的 \(k\) 是 \(n\)）\(\Leftrightarrow k\) 是 \(n\) 的倍数。所以 \(\sigma\) 的阶是 \(n\)。

由 \(\sigma\) 生成的群 \(\langle \sigma \rangle = \{\text{id}, \sigma, \sigma^2, \ldots, \sigma^{n-1}\}\) 是 \(n\) 阶循环群。它就是 \(\mathbb{F}_q / \mathbb{F}_p\) 的 Galois 群。

结论这条极漂亮：

\[ \boxed{\ \mathrm{Gal}(\mathbb{F}_{p^n} / \mathbb{F}_p) \cong \mathbb{Z}/n\mathbb{Z}, \quad \text{由 Frobenius } \sigma(a) = a^p \text{ 生成}.\ } \]

对比一般域扩张的 Galois 群可以非常复杂（不一定循环、不一定阿贝尔），有限域的 Galois 群永远是循环的——这是有限域世界出人意料的简单。Galois 对应"中间域 ↔ Galois 子群"在这里变成"中间域 ↔ 循环群的子群 ↔ \(n\) 的因子"——\(\mathbb{F}_{p^n}\) 和 \(\mathbb{F}_p\) 之间的中间域恰好对应 \(n\) 的因子，每个因子 \(d\) 给一个 \(\mathbb{F}_{p^d}\) 的中间域。

23.6 一般化（二）：\(\mathbb{F}_q^*\) 是循环群

23.4 节我们看到 \(\mathbb{F}_8^* = \{1, \alpha, \alpha^2, \ldots, \alpha^6\}\) 是 7 阶循环群，由 \(\alpha\) 生成。这不 是 \(\mathbb{F}_8\) 特殊——是有限域的乘法群永远是循环群。

定理。 \(\mathbb{F}_q^* = \mathbb{F}_q \setminus \{0\}\) 是 \(q - 1\) 阶循环群。

"循环" 意味着存在一个生成元 \(g \in \mathbb{F}_q^*\) 使每个非零元都能写成 \(g^k\)（\(k \in \{0, 1, \ldots, q-2\}\)）。这种 \(g\) 叫 \(\mathbb{F}_q\) 的原根（primitive root），或者叫 \(\mathbb{F}_q^*\) 的生成元。原根不 唯一——共有 \(\varphi(q - 1)\) 个（Euler 函数）。

证明思路。\(\mathbb{F}_q^*\) 是有限阿贝尔群（\(\mathbb{F}_q\) 的乘法本来就交换），由第 8 章 8.4 节"有限阿贝尔群结构定理"可写成 \(\mathbb{Z}/n_1 \oplus \cdots \oplus \mathbb{Z}/n_r\)。要证它是循环 等价于证它没有"两个独立的 \(\mathbb{Z}/p\) 子群"，即对每个素 \(p\)，\(\mathbb{F}_q^*\) 的 \(p\)-Sylow 子群是循环的。

用一招：方程 \(x^d = 1\) 在域 里最多 \(d\) 个根（多项式 \(x^d - 1\) 最多 \(d\) 个根，第 15 章事实）。如果 \(\mathbb{F}_q^*\) 有两个独立 \(\mathbb{Z}/p\) 子群，则 \(\{x : x^p = 1\}\) 至少 \(p^2\) 个元素，矛盾。所以每个 Sylow 都循环，所以整体循环。\(\square\)

这条事实让有限域的"对数" 概念有意义——固定一个原根 \(g\)，每个非零元 \(a\) 唯一对应 \(k \in \{0, \ldots, q-2\}\) 使 \(a = g^k\)。这个 \(k\) 叫 \(a\) 的离散对数（discrete logarithm），记 \(\log_g a = k\)。

"给 \(g, a\) 求 \(k\)" 这件事——叫离散对数问题——在 \(\mathbb{F}_q^*\) 里没有快速算法。这条"难度" 是密码学 的基础：Diffie-Hellman 密钥交换、ElGamal 加密、椭圆曲线密码学，都建立在离散对数难解 这个假设上。

有限域的"乘法是简单的（循环），但反向取对数是难的" 这种不对称是密码学整个领域的根。后面 23.8 节我们看用不同 不对称（整数因子分解难）造的 RSA。

23.7 应用一：Reed-Solomon 编码

把有限域 这个工具放到一个具体应用里：编码理论。

问题。Alice 想给 Bob 发一段数据。中间通道有噪声——比如卫星到地面的射频信号、CD 表面的划痕、SD 卡的 bit flip——会偶尔把 0 变成 1 或反过来。怎么让 Bob 在收到错码时不仅能检测，还能纠正？

朴素办法：每个 bit 重复 3 次。Alice 发 0 就发 000，发 1 就发 111。Bob 收到的可能是 010（中间被 flip 一位）——投票多数 → 译为 0。这能纠正 1 个错，但代价是三倍带宽。极不经济。

1960 年 Reed 和 Solomon 用有限域多项式 设计了一种远更经济的编码。基本想法：

编码。 Alice 想发的数据是 \(k\) 个 \(\mathbb{F}_q\) 元素：\((m_0, m_1, \ldots, m_{k-1})\)。她构造一个 \(< k\) 次多项式

\[ m(x) = m_0 + m_1 x + \cdots + m_{k-1} x^{k-1} \in \mathbb{F}_q[x]. \]

挑 \(n \ge k\) 个不同的 \(\mathbb{F}_q\) 元素 \(\alpha_1, \ldots, \alpha_n\)（要求 \(n \le q\)）。编码 是把 \(m(x)\) 在这 \(n\) 个点求值：

\[ c = (m(\alpha_1), m(\alpha_2), \ldots, m(\alpha_n)). \]

Alice 把这 \(n\) 个 \(\mathbb{F}_q\) 元素发出去。注意 \(n > k\) 所以发出去的多于 原数据，多出的 \(n - k\) 个值是冗余。

解码。 Bob 收到 \(\tilde c = (\tilde c_1, \ldots, \tilde c_n)\)，其中可能有些位被噪声改了。他怎么找回 \(m(x)\)？

如果 \(\tilde c\) 没出错，任意 \(k\) 个值就够——\(k\) 个点决定一个 \(< k\) 次多项式（第 15 章 Lagrange 插值）。如果有 \(t\) 个错，Bob 需要在 \(\binom{n}{k}\) 种"挑 \(k\) 个" 里找一组都对 的——只要错位数 \(t\) 不太大，正确多项式 仍能从大多数 \(k\)-元子集里被恢复。

具体的：Reed-Solomon 码能纠正 \(\lfloor (n - k)/2 \rfloor\) 个错。所以为了纠正 \(t\) 个错，你只需要 \(n = k + 2t\) 个发送点——比"重复 \(2t + 1\) 次" 经济得多（重复要 \(k(2t+1)\) 个 bit；RS 只要 \(k + 2t\) 个有限域元素）。

具体例：CD 上用 RS(255, 223)（在 \(\mathbb{F}_{256} = \mathbb{F}_{2^8}\) 上），每个码字 \(n = 255\) 字节里 \(k = 223\) 是数据，\(2t = 32\) 是冗余，可纠正 \(t = 16\) 个字节错——抵抗 CD 表面的连续划痕。SD 卡、QR 二维码、卫星通信、深空探测（Voyager 的图像下传）都用 RS 或它的变体。

请你停下来欣赏：纠错的全部秘密在"多项式由它的几个值决定"——这是 \(\mathbb{F}_q[x]\) 的代数事实。Lagrange 插值在第 15 章我们当成"插值技术"，这里它是整个数字时代抗噪通信的数学根。RS 编码不能不 在 \(\mathbb{F}_q\) 上做——\(\mathbb{R}\) 上虽然也能做插值，但有限域提供的"有限多个发送 + 接收符号 + 每个符号都能用 bit 串表示" 才让这套技术可实现。

23.8 应用二：RSA 加密简介

另一个具体应用，用不同 的代数结构（整数模合数）造。

RSA（Rivest–Shamir–Adleman，1978）是公开密钥加密的最早实用算法。它的代数基础不 是有限域而是合数模整数环 \(\mathbb{Z}/n\mathbb{Z}\)（\(n = pq\)，\(p, q\) 大素数），用 Euler 函数 \(\varphi(n)\) 和 Euler 定理。

预备：Euler 定理。 对 \(\gcd(a, n) = 1\)，

\[ a^{\varphi(n)} \equiv 1 \pmod n, \]

其中 \(\varphi(n) = |(\mathbb{Z}/n\mathbb{Z})^*|\) 是 \(\mathbb{Z}/n\) 里和 \(n\) 互素的元素个数。\(p\) 素时 \(\varphi(p) = p - 1\)；\(n = pq\)（两个不同素）时 \(\varphi(pq) = (p-1)(q-1)\)。

这条本质上是群 \((\mathbb{Z}/n\mathbb{Z})^*\) 上 Lagrange 定理——\(a\) 在这个 \(\varphi(n)\) 阶群里的阶整除 \(\varphi(n)\)。

RSA 算法。

密钥生成（Bob 做）。

  1. 挑两个大素数 \(p, q\)（实际中各 \(\ge 1024\) bit）。算 \(n = pq\)，\(\varphi(n) = (p-1)(q-1)\)。
  2. 挑 \(e\)（"加密指数"）满足 \(\gcd(e, \varphi(n)) = 1\)。常用 \(e = 65537\)（一个 Fermat 素 \(2^{16} + 1\)）。
  3. 用扩展 Euclid 算法找 \(d\) 满足 \(e d \equiv 1 \pmod{\varphi(n)}\)。这个 \(d\) 是"解密指数"。
  4. 公开：\((n, e)\)。保密：\((p, q, d, \varphi(n))\)。

加密（Alice 给 Bob 发消息 \(m\)）。 设 \(m \in \mathbb{Z}/n\)（消息编成 \(n\) 以下的整数）。Alice 算密文

\[ c \equiv m^e \pmod n \]

用公开 的 \((n, e)\) 即可——任何人都能加密。

解密（Bob 收到 \(c\)）。 Bob 算

\[ m' \equiv c^d \pmod n. \]

断言 \(m' = m\)。验证：\(m' = c^d = (m^e)^d = m^{ed}\)。\(ed \equiv 1 \pmod{\varphi(n)}\)，所以 \(ed = 1 + k \varphi(n)\) 某 \(k\)。所以 \(m^{ed} = m \cdot (m^{\varphi(n)})^k\)。当 \(\gcd(m, n) = 1\) 时 Euler 定理给 \(m^{\varphi(n)} \equiv 1\)，所以 \(m^{ed} \equiv m\)。（\(\gcd(m, n) > 1\) 的边角情况单独处理，结论仍对——用中国剩余定理在 \(\mathbb{Z}/p\) 和 \(\mathbb{Z}/q\) 上分别验证。）

安全性：攻击者拿到 \((n, e, c)\) 想找 \(m\)。他要算 \(d\)，就得算 \(\varphi(n) = (p-1)(q-1)\)，就得知道 \(p, q\)——也就是分解 \(n\)。整数分解没有已知多项式时间算法（最好 \(\sim e^{(\log n)^{1/3}}\)），\(n = 2048\) bit 时实际不可行。所以"分解大整数难" 这个假设支撑 RSA 的安全性。

反过来如果你能分解 \(n\)，立刻能算 \(\varphi(n)\)、再算 \(d\)、整个体系崩溃。这就是量子计算 让 RSA 紧张的原因——Shor 算法（1994）能在量子计算机上多项式时间分解整数。一旦大规模量子计算机普及，RSA 就完了。后量子密码学（如基于格的加密）正在为这一天做准备。

请你欣赏：RSA 全部依赖 \((\mathbb{Z}/n\mathbb{Z})^*\) 是阿贝尔群和 Euler 定理这两条 19 世纪以前就有的事实——但 它直到 1978 年才被发明，因为它"反向需要分解" 这个不对称要等到现代密码学的视角才被认识到。古典 数学结果 + 现代 计算/通信问题 = 新 应用——这是抽象代数现实意义最浓的一面。

总结。 这一章把"朴素地造大小 \(n\) 的域" 这个具体问题撞墙后，reframe 成"用多项式商环造"，立刻打开有限域的全图——大小限于 \(p^n\)、每种唯一同构、Frobenius 自同构是 Galois 群的生成元、乘法群循环、原根、离散对数。然后两个具体应用：Reed-Solomon 码（用有限域上的多项式插值做纠错）和 RSA（用 \(\mathbb{Z}/n\) 上 Euler 定理做加密）。这两个例子都体现一个 21 世纪通讯/安全的核心模式：古典代数事实 + "反向问题难" 的不对称 = 实用算法。
