查找：按"我现在想做什么"

  1. 识别一个对象是不是群：对照第 2 章三条公理（结合 + 恒等 + 逆）；阿贝尔加一条交换。
  2. 给一个具体小阶群做分类（如 \(|G| = 12\)）：用 Sylow（第 7 章）算出每个素 \(p\) 的 Sylow \(p\)-子群个数 \(n_p\)；约束 \(n_p \equiv 1 \pmod p\) 且 \(n_p | |G|/p^k\)；分情况讨论。\(|G| = 12\) 给 \(\mathbb{Z}_{12}, \mathbb{Z}_2 \oplus \mathbb{Z}_6, A_4, D_6, \langle a, b : a^4 = b^3 = e, ba = a b^{-1}\rangle\) 等。
  3. 判一个多项式是否不可约：低次（2、3 次）看有没有根；4 次以上用第 14 章 Eisenstein（如果可应用）；\(\mathbb{F}_p[x]\) 上小次数可枚举。
  4. 构造 \(p^n\) 元素的有限域：找 \(\mathbb{F}_p[x]\) 里 \(n\) 次不可约多项式 \(g(x)\)，做 \(\mathbb{F}_p[x]/(g)\) 23.3。
  5. 证明某 \(\alpha\) 在某域 \(K\) 上代数：找 \(K[x]\) 里的最小多项式 \(\alpha\) 满足。\([K(\alpha) : K] = \deg(\text{最小多项式})\) 第 19 章。
  6. 用 Galois 主定理算中间域：先算 \(\mathrm{Gal}(L/K)\)；列它所有子群；每个子群对应一个中间域，子群越大 ↔ 中间域越小 第 21 章。
  7. 判某个尺规作图问题可不可做：把目标长度的最小多项式次数 算出来；可作图 ⟺ 该次数是 2 的方幂 第 22 章。
  8. 判某次方程是否有求根公式：它的 Galois 群可解 ⟺ 有求根公式 第 22 章。\(n \le 4\) 总有；\(n \ge 5\) 一般不有（\(S_n\) 不可解）。
  9. 算 RSA 加密 / 解密：选两素 \(p, q\) → \(n = pq, \varphi(n) = (p-1)(q-1)\) → 选 \(e\) 互素 \(\varphi(n)\) → 算 \(d = e^{-1} \mod \varphi(n)\) → 加密 \(c = m^e \mod n\)，解密 \(m = c^d \mod n\) 23.8。
  10. 判物理系统能级简并度：找哈密顿对称群 \(G\)；找 \(G\) 的不可约表示分类；每个能级简并度 = 该能级特征空间所对应不可约的维数 24.9。
  11. "我搞 ML / 计算机":群作用 + 等变神经网络 第 6 章；模结构定理 + 数据降维理论根 第 17 章；密码学 / 哈希 第 23 章；范畴论 + 函数式编程 附录 B。
  12. 诊断"我证不出来"：步骤——（1）写出所有给定的"结构"；（2）写出要证的命题，问"它属于群论 / 环论 / 模论 / 域论 / Galois"；（3）翻到对应章节，看类似命题怎么证；（4）如果是商 / 同构，多半是某个同构定理或泛性质；（5）如果是"极大对象" 存在，多半是 Zorn。
