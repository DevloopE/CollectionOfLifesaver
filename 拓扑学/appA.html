附录 A 集合论与等价类速查

本书反复用到的集合语言——并 / 交 / 笛卡尔积 / 函数 / 等价关系 / 商集 / 偏序——这个附录集中处理。它们是实分析、代数、拓扑 共同的最基础工具, 但每本书的口径稍有不同。这里只摆本书实际用到 的版本——尤其是商拓扑 与任意积 用到的那几条。

  • 集合运算 + De Morgan。
  • 函数（单 / 满 / 双 / 像 / 逆像）。
  • 等价关系 + 划分 + 商集——商拓扑的代数前置。
  • 偏序 + 全序——Zorn 引理（附录 B）的语言。
  • Cantor 对角法 + \(\mathbb{R}\) 不可数。
  • 任意 index 集的笛卡尔积——选择公理引入。

A.1 集合运算

给集合 \(A, B\)（同一全集 \(X\) 子集）：

  • 并 \(A \cup B = \{x : x \in A \text{ 或 } x \in B\}\)。
  • 交 \(A \cap B = \{x : x \in A \text{ 且 } x \in B\}\)。
  • 差 \(A \setminus B = \{x : x \in A \text{ 且 } x \notin B\}\)。
  • 对称差 \(A \triangle B = (A \setminus B) \cup (B \setminus A)\)。
  • 余集（在全集 \(X\) 内）\(A^c = X \setminus A\)。

任意 index 族 \(\{A_\alpha\}_{\alpha \in I}\)（\(I\) 是 index 集, 可有限可无穷, 可可数可不可数）：

\[\bigcup_{\alpha \in I} A_\alpha = \{x : \exists \alpha, x \in A_\alpha\}, \qquad \bigcap_{\alpha \in I} A_\alpha = \{x : \forall \alpha, x \in A_\alpha\}.\]

De Morgan 律（任意 index）：

\[\boxed{\ \Big(\bigcup_\alpha A_\alpha\Big)^c = \bigcap_\alpha A_\alpha^c, \qquad \Big(\bigcap_\alpha A_\alpha\Big)^c = \bigcup_\alpha A_\alpha^c.\ }\]

这是拓扑学反复用的工具——"开集" 和"闭集" 通过 De Morgan 互译："任意开集并" \(\leftrightarrow\) "任意闭集交"，"有限开集交" \(\leftrightarrow\) "有限闭集并"。第 5 章闭集公理就是开集公理通过 De Morgan 翻过去的。

笛卡尔积（两个集合）：\(A \times B = \{(a, b) : a \in A, b \in B\}\), 元素是有序对。

多个集合：\(A_1 \times A_2 \times \cdots \times A_n = \{(a_1, \ldots, a_n) : a_i \in A_i\}\), 元素是有序 \(n\)-元组。

任意 index 笛卡尔积——A.7 节专门讲（这是选择公理的所在）。

A.2 函数：单射 / 满射 / 双射, 像 / 逆像

函数 \(f : A \to B\) 给每个 \(a \in A\) 配唯一 \(f(a) \in B\)。

  • 单射（injective）：\(f(a_1) = f(a_2) \Rightarrow a_1 = a_2\)。
  • 满射（surjective）：每个 \(b \in B\) 都有原像 \(a\) 使 \(f(a) = b\)。
  • 双射（bijective）：单 + 满。等价于有逆: 存在 \(g : B \to A\) 使 \(g \circ f = \mathrm{id}_A\)、\(f \circ g = \mathrm{id}_B\)。

给 \(S \subseteq A\), 像 \(f(S) = \{f(a) : a \in S\} \subseteq B\)。给 \(T \subseteq B\), 逆像（preimage）\(f^{-1}(T) = \{a \in A : f(a) \in T\} \subseteq A\)。注意：逆像即使 \(f\) 不可逆也良定——它不需要 \(f\) 有逆函数。

逆像和集合运算交换：

\[f^{-1}\Big(\bigcup_\alpha T_\alpha\Big) = \bigcup_\alpha f^{-1}(T_\alpha), \qquad f^{-1}\Big(\bigcap_\alpha T_\alpha\Big) = \bigcap_\alpha f^{-1}(T_\alpha), \qquad f^{-1}(B \setminus T) = A \setminus f^{-1}(T).\]

这一族"逆像保运算" 是第 6 章"连续 = 开集逆像开" 这条等价定义能成立的代数原因。逆像让"开集闭集系统" 在映射下表现良好。

注意"正像" \(f(\cdot)\) 不这么干净——\(f(A \cap B) \subseteq f(A) \cap f(B)\) 但未必相等（反例：\(f(x) = x^2\), \(A = \{1\}, B = \{-1\}\), \(f(A \cap B) = f(\emptyset) = \emptyset\) 但 \(f(A) \cap f(B) = \{1\}\)）。正像保并不保交不保余。

A.3 等价关系 + 划分 + 商集

给集合 \(X\), 一个等价关系 \(\sim\) 是 \(X \times X\) 的子集（即一个二元关系）满足：

  • (R) 反身：\(x \sim x\)。
  • (S) 对称：\(x \sim y \Rightarrow y \sim x\)。
  • (T) 传递：\(x \sim y\) 且 \(y \sim z \Rightarrow x \sim z\)。

给 \(x \in X\), 它的等价类 \([x] = \{y \in X : y \sim x\}\)。

定理（等价类划分）：等价类 \(\{[x] : x \in X\}\) 给 \(X\) 的划分——即两两不交且并 = \(X\)。等价的, 给一个划分 \(\{P_\alpha\}\) 也定义一个等价关系（\(x \sim y \iff\) 同属一块）。

"等价关系 \(\leftrightarrow\) 划分" 这一对应是基础。证明（\(\Rightarrow\) 方向）：(R) 给 \(x \in [x]\), 所以并 = \(X\)。两个等价类要么相同要么不交：若 \(z \in [x] \cap [y]\), 则 \(z \sim x\) 且 \(z \sim y\), 由 (S) (T) 得 \(x \sim y\), 所以 \([x] = [y]\)。\(\square\)

商集 \(X / \sim\) = 等价类全体 = \(\{[x] : x \in X\}\)。配商映射 \(q : X \to X/\sim\), \(q(x) = [x]\) ——把每个元素送到其等价类。

例 1：\(X = \mathbb{Z}\), \(x \sim y \iff x - y\) 偶。等价类两个：\([0] = \) 偶数, \([1] = \) 奇数。\(\mathbb{Z}/\sim = \{[0], [1]\} \cong \mathbb{Z}/2\mathbb{Z}\)。

例 2：\(X = \mathbb{R}\), \(x \sim y \iff x - y \in \mathbb{Z}\)。等价类是 \(\mathbb{R}\) 模 \(\mathbb{Z}\), 每类挑代表元在 \([0, 1)\) 中。\(\mathbb{R}/\mathbb{Z} \leftrightarrow [0, 1)\) 作为集合, 配上"0 与 1 等同" 给一个圆 \(S^1\)（这正是第 10 章商拓扑的核心例子）。

例 3：\(X = S^n\)（\(n\) 维球面）, \(x \sim y \iff y = \pm x\)（对径点等同）。商 \(S^n / \pm = \mathbb{R}P^n\)（实射影空间, ch24）。

商集是商拓扑（ch10）的代数底层——商拓扑就是给商集配上一个由商映射诱导的最大 拓扑。

A.4 偏序 + 全序

给集合 \(X\), 一个偏序（partial order）\(\le\) 是二元关系满足：

  • (R) 反身：\(x \le x\)。
  • (A) 反对称：\(x \le y\) 且 \(y \le x \Rightarrow x = y\)。
  • (T) 传递：\(x \le y\) 且 \(y \le z \Rightarrow x \le z\)。

"偏" 因为不 要求任意两元素可比——可能存在 \(x, y\) 使 \(x \not\le y\) 且 \(y \not\le x\), 这种 \(x, y\) 叫"不可比"。

若加一条 \((TR)\) 三歧：\(\forall x, y, x \le y \text{ 或 } y \le x\), 那 \(\le\) 叫全序（total order, 或 linear order）。

例：

  • \(\mathbb{R}\) 上自然 \(\le\) 是全序。
  • 给 \(X\) 的子集族 \(\mathcal{P}(X)\), 关系"子集" \(\subseteq\) 是偏序——\(A, B \subseteq X\) 不一定有 \(A \subseteq B\) 或 \(B \subseteq A\)。
  • \(\mathbb{Z}_{>0}\) 上"整除" \(a \mid b\) 是偏序——\(2 \nmid 3\) 且 \(3 \nmid 2\), 不可比。

偏序集（poset）\((X, \le)\) 的链（chain）：一个全序子集 \(C \subseteq X\)（即 \(C\) 中任两元素可比）。

上界：\(u \in X\) 是 \(S \subseteq X\) 的上界, 当 \(\forall s \in S, s \le u\)。

极大元：\(m \in X\) 极大, 当 \(\forall x \in X, m \le x \Rightarrow x = m\)（即"没有比 \(m\) 严格大的"）。注意"极大" 不等于"最大"——偏序里可能没有最大元（要求 \(\forall x, x \le \text{最大}\)）但有极大元（不可比的元素一起当极大）。

这套语言为Zorn 引理（附录 B）服务：

Zorn 引理：偏序集 \(X\) 中如果每条链都有上界, 则 \(X\) 有极大元。

它是选择公理的等价表述, 是线性代数证明任何向量空间有基、泛函分析 Hahn-Banach、代数证明每个非零环有极大理想、本书 Tychonoff 定理（ch16） 等的工具。

A.5 Cantor 对角法 + \(\mathbb{R}\) 不可数

两个集合等势（equinumerous）当存在双射。"有相同元素个数" 的精确版本（即使两边都无穷）。

\(\mathbb{N}\) 等势的集合叫可数无穷（countably infinite）：\(\mathbb{Z}, \mathbb{Q}\) 都可数无穷。

可数无穷比有限大、比连续统小。\(\mathbb{R}\) 不 可数——比 \(\mathbb{N}\) 严格大。这是 Cantor 1874 的著名结果, 给了"不同大小的无穷" 概念。

定理（Cantor）：\(\mathbb{R}\) 不可数。等价地：\([0, 1]\) 不可数。

证明（对角法）。反证。设 \([0, 1]\) 可数, 列成序列 \(x_1, x_2, x_3, \ldots\)。每个 \(x_n\) 写成 10 进小数：

\[x_n = 0.x_{n,1} x_{n,2} x_{n,3} \cdots\]

构造一个新数 \(y = 0.y_1 y_2 y_3 \ldots\)，规则：

\[y_n = \begin{cases} 5 & x_{n,n} \neq 5 \\ 6 & x_{n,n} = 5 \end{cases}.\]

那 \(y \in [0, 1]\), 但 \(y \neq x_n\) 对所有 \(n\)（在第 \(n\) 位与 \(x_n\) 不一样）。可序列遍历了所有 \([0, 1]\) 元素。矛盾。\(\square\)

"对角化" 是非常通用的反证模式。Russell 悖论（"所有不属于自身的集合的集合"）、Gödel 不完全性、停机问题不可判定都用对角化。本书ch11 Tychonoff 拓扑里偶尔间接用到。

\(\mathbb{R}\) 的势记 \(\mathfrak{c}\)（"continuum"）, \(\mathbb{N}\) 的势记 \(\aleph_0\)。Cantor 证明 \(\mathfrak{c} = 2^{\aleph_0}\)。"是否存在介于二者之间的势" 是连续统假设（CH）, Gödel + Cohen 证明它独立于 ZFC（既不能证又不能反证）。这是数学基础的名场面。

A.6 任意笛卡尔积

两个集合的笛卡尔积 \(A \times B\) 你熟。\(n\) 个的也熟。无穷多 集合的呢？

设 \(\{X_\alpha\}_{\alpha \in I}\) 是 index 族（\(I\) 任意 index 集, 可不可数）。任意笛卡尔积 形式定义：

\[ \boxed{\ \prod_{\alpha \in I} X_\alpha = \Big\{f : I \to \bigcup_\alpha X_\alpha \ \Big| \ f(\alpha) \in X_\alpha \text{ 对所有 } \alpha \in I\Big\}.\ } \]

"任意 index 笛卡尔积 = 选择函数族"。即一个元素是一个映射 \(f\), 给每个 index \(\alpha\) 选一个 \(f(\alpha) \in X_\alpha\)。

当 \(I = \{1, 2, \ldots, n\}\), 这等价于 \(n\)-元组 \((f(1), f(2), \ldots, f(n)) = (a_1, \ldots, a_n)\)。两个定义一致。

当 \(I = \mathbb{N}\), 元素是序列 \((a_1, a_2, \ldots)\)。

当 \(I = \mathbb{R}\) 或更大, 这是个巨大 的集合。"函数视角" 的好处是不依赖某种"列表" 结构, 适应任意 index。

关键问题：任意笛卡尔积非空吗？即如果每个 \(X_\alpha\) 非空, 是否一定存在选择函数 \(f\)？

对有限 \(I\) 答案是显然是——挨个选, 一定成。

对可数 \(I\)（如 \(\mathbb{N}\)）答案仍然是（用归纳）。

对不可数 \(I\) 答案不能从 ZF 公理推出——这正是选择公理（Axiom of Choice, AC）所断言：

选择公理（AC）：对任意非空集族 \(\{X_\alpha\}_{\alpha \in I}\), 存在选择函数 \(f \in \prod_{\alpha} X_\alpha\)。

等价地：任意非空集族的笛卡尔积非空。

这是数学最有争议（也最有用）的公理之一——下一个附录 B 专门讲它。

A.7 选择公理引入

选择公理在本书具体出现的几个地方：

  • ch11 任意积拓扑（Tychonoff topology）的定义 用任意笛卡尔积。
  • ch16 Tychonoff 定理"任意紧空间的积仍紧"——证明本质需要 AC（事实上 Tychonoff 定理与 AC 等价！）。
  • ch20 Urysohn 引理某种证法。
  • ch21 Urysohn 度量化定理的某些步骤。
  • ch11 "任意度量空间积有可数基" 等技术结果。

对大多数具体 拓扑学问题（\(\mathbb{R}^n\) 上的、可数空间上的）AC 不需要——可数族的选择由"可数选择公理"（AC\(_\omega\), 是 ZF 定理）解决。AC 真正介入的是不可数 index 的情形——Tychonoff 定理是最经典例。

请你把 A.6 节的"任意笛卡尔积非空 \(\iff\) 选择公理" 这条对应记住——它是看穿"哪些证明本质上需要 AC" 的最简单标志。任何时候你看到"对每个 \(\alpha \in I\)（\(I\) 不可数）选一个 \(x_\alpha \in X_\alpha\)" 的步骤, 你就在用 AC。
