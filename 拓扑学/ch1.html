第 1 章 引论：从距离到开集

请你回答一道你以为 自己知道答案的题：

函数 \(f(x) = x^2\) 在 \(\mathbb{R}\) 上连续吗？

当然连续。怎么证明 它连续？

教科书答案——\(\varepsilon\)-\(\delta\)：给任意 \(\varepsilon > 0\) 与任意 \(x_0 \in \mathbb{R}\)，找 \(\delta > 0\) 使 \(|x - x_0| < \delta \Rightarrow |x^2 - x_0^2| < \varepsilon\)。具体计算 \(|x^2 - x_0^2| = |x - x_0| \cdot |x + x_0|\)，取 \(\delta = \min(1, \varepsilon / (1 + 2|x_0|))\)，验证完。\(\checkmark\)

但是停下来想一下——你真 是这样验证 \(x^2\) 连续的吗？很可能不是。你"看见" 抛物线是条没有断点的曲线，"显然" 连续。\(\varepsilon\)-\(\delta\) 是用来万一被问起 的形式答案，不是你真正 用来理解连续的工具。

这一章要做的就是把这种"看见连续" 的直觉形式化——并且形式化得不依赖距离。形式化完了你会发现：连续真正 需要的是"开集" 这个概念，不需要"距离"。一旦你接受这件事，整个拓扑学就在你眼前打开。

1.1 连续的另一种说法："开集的逆像是开集"

给一个映射 \(f : X \to Y\) 在度量空间之间。实分析读本第 5 章给过这条等价：

\[ \boxed{\ f \text{ 连续} \iff \forall V \text{ 开} \subseteq Y, \ f^{-1}(V) = \{x : f(x) \in V\} \text{ 开} \subseteq X.\ } \]

用人话：每个开集的逆像还是开集。这条等价的证明就一段——用 \(\varepsilon\)-\(\delta\) 翻译"开" 为"对每个点都有一个含它的开球" 即可。

验证 \(f(x) = x^2\)。取 \(V \subseteq \mathbb{R}\) 开，比如 \(V = (1, 4)\)。那 \(f^{-1}(V) = \{x : 1 < x^2 < 4\} = (-2, -1) \cup (1, 2)\)。两个开区间的并——开集。\(\checkmark\)

取 \(V = (-1, 1)\)：\(f^{-1}(V) = \{x : x^2 < 1\} = (-1, 1)\)，开。再取 \(V = (4, \infty)\)：\(f^{-1}(V) = (-\infty, -2) \cup (2, \infty)\)，开。每个开集的逆像都开——\(f\) 连续。

请你停下来盯住这条等价仔细看一下。它说"连续" 这件事，可以完全 用"开集" 描述——没 出现 \(\varepsilon, \delta\)、没 出现"距离"、没 出现 \(\mathbb{R}\) 的算术结构。只 出现集合 + 函数 + "开集" 这个原始概念。

这就是关键。如果一个空间上有"开集" 概念（不一定来自距离），就能 谈连续。问题是——什么样的空间有"开集"？

1.2 "开集" 满足三条公理

在度量空间 \((X, d)\) 里，"开集" 由距离定义：\(U\) 开 \(\iff\) 每个 \(x \in U\) 都有半径 \(r > 0\) 使开球 \(B(x, r) \subseteq U\)。这是一种构造。

但"哪些集合是开的" 这件事满足三条抽象 性质——你完全可以不通过距离 直接列出来：

  • (O1) \(\emptyset\) 开，\(X\) 开。
  • (O2) 任意多开集的并仍开。\(\{U_\alpha\} \text{ 开} \Rightarrow \bigcup_\alpha U_\alpha\) 开。
  • (O3) 有限多开集的交仍开。\(U_1, \ldots, U_n \text{ 开} \Rightarrow \bigcap_{i=1}^n U_i\) 开。

验证（度量空间情形）：(O1) 显然。(O2) 任意并：每个 \(x \in \bigcup U_\alpha\) 在某个 \(U_{\alpha_0}\) 里，那个开球工作。(O3) 有限交：每个 \(x\) 在每个 \(U_i\) 里都有球 \(B(x, r_i) \subseteq U_i\)，取 \(r = \min r_i > 0\)，球 \(B(x, r)\) 同时落在所有 \(U_i\) 里。有限性必要——\(\inf\) 无穷多个正数可能 \(= 0\)。事实上 \(\bigcap_n (-1/n, 1/n) = \{0\}\)不 开。

反过来——给一个任意 集合 \(X\)，再给一族 \(X\) 的子集 \(\tau\)，如果 \(\tau\) 满足上面三条公理，我们就说 \(\tau\) 是 \(X\) 上的拓扑，\((X, \tau)\) 是拓扑空间。不需要距离——只需要满足公理的"开集" 集族。

这就是拓扑空间 的诞生。第 3 章正式定义。这一章只让你看到这条思路怎么从连续这件具体事 长出来。

1.3 几个具体的"没距离" 拓扑

看几个例子让"开集独立于距离" 这件事变成具体：

例 1：Sierpiński 空间。取 \(X = \{0, 1\}\)，定义

\[ \tau = \{\emptyset, \{1\}, \{0, 1\}\}. \]

验证三条公理：(O1) \(\emptyset, X \in \tau\)，\(\checkmark\)。(O2) 并：\(\emptyset \cup \{1\} = \{1\}\)，\(\{1\} \cup X = X\)，全在 \(\tau\) 里。(O3) 交：\(\emptyset \cap \{1\} = \emptyset\)，\(\{1\} \cap X = \{1\}\)，全在 \(\tau\) 里。所以 \(\tau\) 是 \(X\) 上的拓扑。

这个拓扑不 来自任何距离。原因——任何度量给出的拓扑都满足"\(\{x_0\}\) 开 \(\iff x_0\) 是孤立点"。在 Sierpiński 里 \(\{1\}\) 开但 \(\{0\}\) 不开——非对称。但任何度量自动给出对称："\(\{0\}\) 与 \(\{1\}\) 都同等程度 是孤立点或不是"，得到的拓扑里 \(\{0\}\) 与 \(\{1\}\) 要么都开要么都不开。所以 Sierpiński 拓扑不可 来自度量。

这个 2 元素小空间在计算机科学(domain theory) 里反复出现——它是"未终止 / 终止" 二值的拓扑模型。

例 2：余有限拓扑。取 \(X = \mathbb{R}\)（或任意无穷集），定义

\[ \tau = \{\emptyset\} \cup \{U \subseteq X : X \setminus U \text{ 有限}\}. \]

"开集 = 空集 + 余集有限的集合"。验证三条公理用 De Morgan + "有限并仍有限 / 任意交里有限" 即可。

这个拓扑也不 来自距离——它太粗（开集太少）。具体：\(\mathbb{R}\) 上任何度量给出的拓扑里，\((-1, 1)\) 是开集；但在余有限拓扑里 \((-1, 1)\) 不 开（其余集 \((-\infty, -1] \cup [1, \infty)\) 不有限）。

余有限拓扑虽然"怪"，但在代数几何 里至关重要——\(\mathbb{A}^1_k\) (仿射直线代数簇) 上的 Zariski 拓扑就是它。代数几何里"开集 = 多项式不为零的集合"，余集是多项式零点（有限多个），正好对应余有限。

例 3：离散拓扑 vs 平凡拓扑。给任意集合 \(X\)：

  • 离散 \(\tau_d = \mathcal{P}(X)\) (所有子集都开) — "什么都开"
  • 平凡 \(\tau_0 = \{\emptyset, X\}\) — "只有空集和全集开"

两个极端，都满足公理，都是合法拓扑。任何 \(X\) 上的"中间" 拓扑都被夹在这两者之间。它们是探索拓扑性质边界的好工具。

1.4 同胚：拓扑等价

有了"拓扑空间" 概念，"什么算同一个空间" 也变了。在度量空间里"同" 是等距同构——保距离的双射。在拓扑空间里"同" 是同胚——保开集结构 的双射。

形式上：双射 \(f : X \to Y\) 是同胚，如果 \(f\) 与 \(f^{-1}\) 都连续（都把开集映成开集）。

同胚比等距同构宽松得多。看几个例子：

圆 \(S^1 \subset \mathbb{R}^2\) 与椭圆。它们度量上完全不同（圆周长 \(2\pi r\)，椭圆周长是椭圆积分），但拓扑上同胚——一个连续可逆的"压扁" 映射就行。所以"圆" 与"椭圆" 在拓扑学眼里是同一个 对象。

开区间 \((0, 1)\) 与 \(\mathbb{R}\)。同胚！同胚映射 \(f(x) = \tan(\pi x - \pi/2)\) 把 \((0, 1)\) 双向连续地拉成整个 \(\mathbb{R}\)。所以"有界开区间" 与"整条实数线" 拓扑同。这件事让"有界" 不是拓扑性质——它依赖度量。

咖啡杯与甜甜圈。这是流传最广的例子。两者都是"带一个洞的实心 3D 物体"，可以连续变形相互转化。拓扑学家"分不清咖啡杯和甜甜圈" 就是这个意思——它们在拓扑下同。事实上"洞数"（亏格 \(g\)）是拓扑不变量：球面 \(g = 0\)、甜甜圈 \(g = 1\)、双甜甜圈 \(g = 2\)。这是第 23 章基本群与代数拓扑 的最基本观察。

圆 \(S^1\) 与 8 字 \(\infty\)。这两个不 同胚！"8 字" 中间有个交点 \(p\)——拿掉它剩下两个连通片；圆的任意点拿掉后剩下还是连通的（一段开弧）。"连通片个数" 是拓扑不变量（第 18 章）——拿掉一点后连通片数不变。所以圆与 8 字拓扑不同。

请你停下来欣赏这件事的分量：通过"拿一个点拓扑变化怎样" 这种纯组合操作，我们能区分 不同的拓扑空间。这种"拓扑不变量" 是整本书的主线之一。

1.5 拓扑学要研究什么

把上面四节合一起，拓扑学的主题清晰了：

对象：拓扑空间——集合 + 满足三公理的开集族。

等价：同胚——保开集结构的双射。

核心问题：

  • 什么是拓扑性质（同胚下不变）？给一个清单：紧致、连通、Hausdorff、可数、度量化、紧致化等。
  • 什么样的空间能配距离（"度量化"）？什么样不能？这是 Urysohn 度量化定理（ch21）回答的。
  • 怎么"组合" 拓扑空间（积、商、子空间）？这套构造 是 ch8-11 内容。
  • 怎么区分 拓扑空间？需要不变量——基本群（ch23）、同调群（代数拓扑, 超出本书）等。

把这些问题一层一层回答，就是这本书 24 章的全部。

1.6 这本书的路线

24 章五段式：

  • 地基（ch2-7）：度量空间复习 → 拓扑空间公理 → 基与子基 → 闭集等几何词汇 → 连续 + 同胚。读完这 7 章你能用拓扑语言。
  • 构造（ch8-11）：子空间 + 积 + 商 + 任意积（Tychonoff 拓扑）。这 4 章把"从已知拓扑造新拓扑" 的所有标准方法学完。
  • 性质（ch12-19）：分离公理（T0-T4 + Hausdorff）+ 可数性 + 紧致性 + 连通性。每个性质都是"同胚下不变" 的不变量。
  • 高级（ch20-22）：Urysohn 引理 + Tietze 延拓 + 度量化定理 + Baire 范畴。这 3 章是"正规空间能干什么" 的高峰。
  • 应用 + 桥梁（ch23-24）：基本群 → 通向代数拓扑；流形 → 通向微分几何。这两章给本书的下一步 入口。

1.7 临别话

拓扑学的"奇怪" 在于它故意 把数学里最直观的东西（距离）抽掉。一开始这种抽离让人不舒服——咖啡杯怎么能和甜甜圈"相同"？\(\mathbb{R}\) 怎么能和 \((0, 1)\) "相同"？

但是抽离一段时间后你会发现：这种抽离让"形状" 第一次成为可数学化的概念。"有几个洞""能不能撕开""能不能压扁到一点"——这些直观的几何问题，在拓扑学里有精确答案。基本群、同调群、Euler 示性数等不变量，把"形状" 翻译成代数对象，让我们能计算 它。

20 世纪几何 / 物理 / 生物 / 数据科学的几大革命都直接或间接靠拓扑：广义相对论的时空流形、规范场论的纤维丛、DNA 拓扑学、TDA（拓扑数据分析）、Künsken-Ribet-Wiles 路径上 Galois 表示、Lurie 的高阶范畴论。每一个背后都有"把距离抽掉, 留下结构" 的拓扑学。

下一章我们先回度量空间收尾——把实分析里学过的紧致 / 连通 / 完备等概念用面向拓扑 的视角重新组织一遍。然后ch3 起正式跳到抽象拓扑空间。
