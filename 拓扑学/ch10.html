第 10 章 商拓扑

请你拿一张纸做一个小实验。

裁一个长方形纸条 \([0, 1] \times [0, 1]\)。把左边 那条 \(\{0\} \times [0, 1]\) 与右边 那条 \(\{1\} \times [0, 1]\) 粘起来——按"同样的 \(y\) 粘到一起" 的规则——\((0, y)\) 与 \((1, y)\) 现在是同一 个点。你得到一个圆柱面 \(S^1 \times [0, 1]\)。再把上下 两条 \([0, 1] \times \{0\}\) 与 \([0, 1] \times \{1\}\) 也按"同样的 \(x\)" 粘起来——你得到一个环面（甜甜圈表面）\(T^2\)。

另外的玩法：左右粘的时候翻转一下——\((0, y)\) 与 \((1, 1 - y)\) 粘起来——你得到一条 Möbius 带（莫比乌斯环）。所有四边都按某种方向粘起来：左右照常 \((0, y) \sim (1, y)\)，上下翻转 \((x, 0) \sim (1 - x, 1)\)——你得到 Klein 瓶。把对边按对径 粘起来 \((0, y) \sim (1, 1 - y)\) 与 \((x, 0) \sim (1 - x, 1)\)——得到实射影平面 \(\mathbb{R}P^2\)。

这些"把若干点视为同一点" 的几何粘合，在拓扑学里有正式名字——商拓扑。这一章把"粘合" 形式化，再具体跑几个例子让你看到：从 \(\mathbb{R}^n\) 的子空间出发，纯靠粘合，可以造出环面、Möbius 带、Klein 瓶、射影空间这些新的 拓扑空间——其中很多不能 嵌进 \(\mathbb{R}^3\)（如 Klein 瓶, \(\mathbb{R}P^2\)）。

10.1 用正方形粘成圆柱 / 环面 / Klein 瓶 / 实射影面

形式化之前先把图画清楚。基础对象：单位正方形 \(I^2 := [0, 1] \times [0, 1]\)。下面四个粘合规则（\(\sim\) 表示"视为同一点"）：

(1) 圆柱 \(S^1 \times [0, 1]\)：仅左右边按 \(y\) 一致：

\[(0, y) \sim (1, y), \quad y \in [0, 1].\]

左右两条边的 \((0, y)\) 与 \((1, y)\) 视为同一点。其他点不动。结果——把纸条卷起来形成的开口圆柱面。

(2) 环面 \(T^2 = S^1 \times S^1\)：左右与 上下都"正粘"：

\[(0, y) \sim (1, y), \quad (x, 0) \sim (x, 1).\]

先卷成圆柱（左右粘），再把圆柱两端按上下圈对应粘——把圆柱"弯过来" 接成甜甜圈表面。这是 \(\mathbb{R}^3\) 中可见的标准甜甜圈。

(3) Möbius 带：左右"翻转粘"：

\[(0, y) \sim (1, 1 - y).\]

左边 \((0, y)\) 与右边的颠倒 位置 \((1, 1 - y)\) 粘——粘之前把右边那条上下倒一下。结果是一条不可定向 的带——你顺着它走一圈会回到起点的"背面"。这是 1858 年 Möbius 与 Listing 独立发现的著名空间。

(4) Klein 瓶：左右正粘 + 上下翻转粘：

\[(0, y) \sim (1, y), \quad (x, 0) \sim (1 - x, 1).\]

左右先卷成圆柱，再把上下两端按"翻转" 粘——这一步在 \(\mathbb{R}^3\) 里做不到（它"必须" 自交）。所以 Klein 瓶不能嵌入 \(\mathbb{R}^3\) 不自交，只能嵌入 \(\mathbb{R}^4\) 里。它是闭曲面、不可定向、无边界——独立完整的空间。

(5) 实射影平面 \(\mathbb{R}P^2\)：所有对边都"对径" 粘：

\[(0, y) \sim (1, 1 - y), \quad (x, 0) \sim (1 - x, 1).\]

更紧凑的描述：取单位圆盘 \(D^2\)，把边界圆 \(S^1\) 上对径点 \(p\) 与 \(-p\) 粘——也得到 \(\mathbb{R}P^2\)。这是最简单的非平凡射影空间，在代数几何 / 同伦论 / 物理（如自旋 \(SO(3) \cong \mathbb{R}P^3\) 的模型）里反复出现。\(\mathbb{R}P^2\) 也不能 嵌入 \(\mathbb{R}^3\) 不自交。

请停下来盯着这五个例子。它们都从同一 起点（单位正方形）出发，粘合规则 不同就给出截然不同的新空间——其中有的不可定向、有的不能嵌入 \(\mathbb{R}^3\)、有的与 \(\mathbb{R}^n\) 中的几何模型完全不同。这种"粘合" 是拓扑学独有的造物方法。

问题来了——"粘合后的新空间" 上的开集 应该是什么？这就是商拓扑要回答的。

10.2 商映射 \(p: X \to X / \sim\)

给拓扑空间 \(X\) 与 \(X\) 上的等价关系 \(\sim\)（满足自反、对称、传递）。等价关系把 \(X\) 划分成等价类——每个等价类是 \(X\) 的一个非空子集 \([x] := \{x' \in X : x' \sim x\}\)。商集合：

\[ X / \sim := \{[x] : x \in X\}. \]

注意：\(X / \sim\) 的"点" 是 \(X\) 的子集（等价类），而不是 \(X\) 的点。不过我们把每个等价类抽象当成"新空间的点" 来看。

商映射：

\[ p: X \to X / \sim, \quad p(x) := [x]. \]

"送一个点到它所在的等价类"。这个 \(p\) 显然是满射（每个等价类至少含一个点）。它一般不是 单射——同一等价类里的不同点都映到同一处。

例（圆柱）：\(X = I^2 = [0, 1]^2\)。等价关系 \((x, y) \sim (x', y')\) \(\iff\) "(x, y) = (x', y')" 或 "{x, x'} = {0, 1} 且 \(y = y'\)"。每个等价类大多是单点 \(\{(x, y)\}\)，但若 \(x \in \{0, 1\}\) 类含两点 \(\{(0, y), (1, y)\}\)。商集合 \(I^2 / \sim\) 形象上就是把左右边粘起来后的圆柱面上的点的集合。

10.3 商拓扑：开集由 \(p\) 反推

现在到关键设计——\(X / \sim\) 上"开集" 是什么？

策略：让商映射 \(p: X \to X / \sim\) 连续——同时 让它"尽可能传递" \(X\) 的开集结构。

定义（商拓扑）：\(X / \sim\) 上的子集 \(V\) 开 \(\iff p^{-1}(V) \subseteq X\) 开。

形式：

\[ \boxed{\ \tau_{X/\sim} := \{V \subseteq X/\sim : p^{-1}(V) \in \tau_X\}.\ } \]

验证三公理：

  • (O1) \(p^{-1}(\emptyset) = \emptyset \in \tau_X\)，\(p^{-1}(X/\sim) = X \in \tau_X\)。\(\checkmark\)
  • (O2) 任意并：\(p^{-1}\left(\bigcup V_\alpha\right) = \bigcup p^{-1}(V_\alpha)\)（逆像与并交换）。每个 \(p^{-1}(V_\alpha)\) 开 \(\Rightarrow\) 并开。\(\checkmark\)
  • (O3) 有限交：\(p^{-1}\left(\bigcap_i V_i\right) = \bigcap_i p^{-1}(V_i)\)。有限多个开集的交开。\(\checkmark\)

所以 \(\tau_{X/\sim}\) 合法。

\(p\) 在这个拓扑下连续——证明就是定义：\(V\) 开 \(\iff p^{-1}(V)\) 开。

关键观察（最细性）：商拓扑是 \(X/\sim\) 上让 \(p\) 连续的最细（开集最多的）拓扑。

证明：设 \(\tau'\) 是 \(X/\sim\) 的另一个拓扑使 \(p\) 连续。则任意 \(V \in \tau'\)，\(p^{-1}(V) \in \tau_X\)，按定义 \(V \in \tau_{X/\sim}\)。所以 \(\tau' \subseteq \tau_{X/\sim}\)——\(\tau_{X/\sim}\) 包含所有"让 \(p\) 连续" 的拓扑，是它们的上界，最细。\(\square\)

请停下来对比 ch8、ch9：

  • 子空间拓扑：让包含 \(i\) 连续的最粗 拓扑（开集最少）；
  • 积拓扑：让投影 \(\pi_i\) 都连续的最粗 拓扑；
  • 商拓扑：让商映射 \(p\) 连续的最细 拓扑（开集最多）。

方向不同，原因在映射的方向：子空间和积里"箭头朝外"（从子空间到大空间, 从积到分量），所以要让箭头连续就要"给出发地少开集" → 最粗。商映射"箭头朝内"（从大空间到商空间），所以要让箭头连续就要"给目的地少开集 = 给出发地多开集" → 商空间侧最细。这是一种对偶。

10.4 商映射的通用性质

积拓扑的通用性质（9.4）告诉我们"从外部到积空间" 的映射连续 \(\iff\) 分量连续。商拓扑的通用性质方向相反——"从商空间到外部" 的映射连续怎么验证？

定理（商空间的通用性质）：设 \(p: X \to X/\sim\) 是商映射，\(f: X/\sim \to Y\) 是任一映射。则

\[ \boxed{\ f \text{ 连续} \iff f \circ p: X \to Y \text{ 连续}.\ } \]

证明。(\(\Rightarrow\)) \(p\) 连续, \(f\) 连续 \(\Rightarrow f \circ p\) 连续——连续函数的复合连续。

(\(\Leftarrow\)) 设 \(f \circ p\) 连续。要证 \(f\) 连续——取 \(Y\) 中开集 \(V\)，证 \(f^{-1}(V)\) 在 \(X/\sim\) 中开。

按商拓扑定义：\(f^{-1}(V) \in \tau_{X/\sim} \iff p^{-1}(f^{-1}(V)) \in \tau_X\)。但 \(p^{-1}(f^{-1}(V)) = (f \circ p)^{-1}(V)\)（这是逆像复合的代数）。\(f \circ p\) 连续 \(\Rightarrow (f \circ p)^{-1}(V) \in \tau_X\)。\(\checkmark\square\)

这条性质极其 实用——它让"定义商空间上的映射并验证连续" 这件抽象事归约成"定义 \(X\) 上的映射, 验证它在等价类上一致, 验证它连续" 这件具体事。

一个细节："定义 \(X\) 上的映射, 验证它在等价类上一致"——具体指：要给出 \(\bar{f}: X/\sim \to Y\)，最直接的办法是先有 \(g: X \to Y\)，验证 当 \(x \sim x'\) 时 \(g(x) = g(x')\)（这叫"well-defined on classes"），然后把 \(\bar{f}([x]) := g(x)\)。这条 \(\bar{f}\) 显然满足 \(\bar{f} \circ p = g\)。所以"\(\bar{f}\) 连续 \(\iff g\) 连续" 由通用性质给出。

下面跑例子。

10.5 跑环面例子：\(I^2 / \sim \cong S^1 \times S^1\)

取 \(X = I^2 = [0, 1] \times [0, 1]\)（标准子空间拓扑），等价关系 \(\sim\) 为环面规则（10.1 第 (2) 例）：

\[ (x, 0) \sim (x, 1), \quad (0, y) \sim (1, y). \]

具体一点。把所有"四个角" 的点 \((0, 0), (0, 1), (1, 0), (1, 1)\) 都视为同一点（按规则它们两两相通）。所有"上下边对应点" 视为同点。所有"左右边对应点" 视为同点。其他点孤立。

商空间 \(I^2 / \sim\) 形式上是个抽象集合，但我们要证它同胚 于具体的环面 \(S^1 \times S^1\)，其中

\[ S^1 = \{(\cos \theta, \sin \theta) : \theta \in \mathbb{R}\} \subseteq \mathbb{R}^2. \]

构造同胚：定义 \(g: I^2 \to S^1 \times S^1\) 由

\[ g(x, y) := \big((\cos 2\pi x, \sin 2\pi x), (\cos 2\pi y, \sin 2\pi y)\big). \]

\(g\) 连续：每分量是 \(\cos 2\pi x\) 等连续函数（实分析）。

\(g\) 在等价类上一致：若 \((x, 0) \sim (x, 1)\)，算 \(g(x, 0) = (\ldots, (1, 0))\) 与 \(g(x, 1) = (\ldots, (\cos 2\pi, \sin 2\pi)) = (\ldots, (1, 0))\)——相等！同理 \(g(0, y) = ((1, 0), \ldots) = g(1, y)\)。一致。\(\checkmark\)

由通用性质 \(g\) 诱导出连续映射 \(\bar{g}: I^2 / \sim \to S^1 \times S^1\)。

\(\bar{g}\) 是双射：满 \(g\) 给出（\(g\) 满, 因为 \((\cos 2\pi x, \sin 2\pi x)\) 横扫 \(S^1\) 当 \(x \in [0, 1)\)）；单——若 \(g(x, y) = g(x', y')\)，按 \(\cos / \sin\) 周期 \(x \equiv x' \pmod{1}\)、\(y \equiv y' \pmod{1}\)，在 \([0, 1]\) 内这恰好 给出 \((x, y) \sim (x', y')\)。\(\checkmark\)

\(\bar{g}\) 是同胚——还要证逆连续。这一步用到"紧致空间到 Hausdorff 空间的连续双射是同胚"（ch15 标准结论）：

  • \(I^2\) 紧致（\([0, 1]\) 紧 + 积传, ch9）；
  • 商映射是连续满射, 像还是 \(I^2 / \sim\) 紧致（ch15: 紧致空间的连续像紧致）；
  • \(S^1 \times S^1 \subseteq \mathbb{R}^2 \times \mathbb{R}^2\) Hausdorff（子空间继承）；
  • 所以 \(\bar{g}: I^2 / \sim \to S^1 \times S^1\) 是紧 \(\to\) Hausdorff 的连续双射 \(\Rightarrow\) 同胚。

\(\boxed{\ I^2 / \sim \cong S^1 \times S^1.\ }\) 抽象的"把矩形对边粘起来" 与具体的"两个圆相乘" 是同一 拓扑空间——这就是环面。

请停一下欣赏：粘合（商拓扑）与积（乘积拓扑）两条独立路径 给同一个空间。这种"构造路径殊途同归" 是拓扑学很美的一面。

10.6 跑 Möbius 带、Klein 瓶、\(\mathbb{R}P^2\)

(a) Möbius 带 \(M\)。取 \(I^2\) 与 \((0, y) \sim (1, 1-y)\)。

商集合 \(M := I^2 / \sim\)。它有边界——上下边 \([0, 1] \times \{0, 1\}\) 上的点不被粘合，构成一条 闭曲线（不是两条）。这是 Möbius 带的著名性质："只有一条边"。

验证："边界" 由 \([0, 1] \times \{0\}\) 与 \([0, 1] \times \{1\}\) 组成。在原矩形里它们是两段。但因为 \((0, 0) \sim (1, 1)\) 与 \((1, 0) \sim (0, 1)\)（按 \(\sim\) 规则），两段在端点 上接起来——\((0, 0)\) 接 \((1, 1)\)，\((0, 1)\) 接 \((1, 0)\)——形成一条连通的闭曲线。所以 \(M\) 的边只有一条。

同样 \(M\) 是不可定向 的：你顺着中线走一圈，回到起点你的"左右" 反了。这条性质留 ch24 流形章正式证明。

\(M\) 可以嵌入 \(\mathbb{R}^3\)（你拿张纸条扭一下粘起来就是它），所以这个商空间是有具体几何模型的。

(b) Klein 瓶 \(K\)。取 \(I^2\) 与 \((0, y) \sim (1, y)\) 加 \((x, 0) \sim (1 - x, 1)\)。

左右正粘（先卷成圆柱），上下翻转。后一步在 \(\mathbb{R}^3\) 里没法不自交完成，所以 \(K\) 在 \(\mathbb{R}^3\) 里只有"自交模型"（你看到的 Klein 瓶图通常都有一处自交穿过自身），但作为抽象 拓扑空间它没有 自交。

事实：\(K\) 嵌入 \(\mathbb{R}^4\) 不自交。证明思路：在 \(\mathbb{R}^4\) 里上下粘合时可以"多用一维躲开" 而不必交穿。

\(K\) 是无边界、闭、不可定向曲面——与球面 \(S^2\)（无边、闭、可定向）、环面 \(T^2\)（无边、闭、可定向）、\(\mathbb{R}P^2\)（无边、闭、不可定向）一起构成"闭曲面分类定理" 的基本砖（ch24 综述）。

(c) \(\mathbb{R}P^2\) 的两种描述。

第一种（正方形）：取 \(I^2\) 与 \((0, y) \sim (1, 1-y)\)、\((x, 0) \sim (1-x, 1)\)。

第二种（圆盘对径）：取 \(D^2 := \{(x, y) : x^2 + y^2 \leq 1\}\)，等价关系：边界点 \(p \in S^1\) 与对径点 \(-p\) 视为同一。

第三种（球面对径）：取 \(S^2 := \{(x, y, z) : x^2 + y^2 + z^2 = 1\}\)，等价关系：每对对径点 \(p \sim -p\) 视为同一。这是射影空间 的通用定义：\(\mathbb{R}P^n := S^n / \{p \sim -p\}\)。

三种描述同胚（练习；用通用性质 + 紧到 Hausdorff 的方法）。

\(\mathbb{R}P^2\) 用处广："所有过原点的直线" 这件事在线性代数里是 \(\mathbb{R}^3 \setminus \{0\}\) 模 \(\sim\)（\(v \sim \lambda v, \lambda \neq 0\)），结果就是 \(\mathbb{R}P^2\)。这就是为什么它叫"射影"——它参数化了 \(\mathbb{R}^3\) 的方向（不分前后）。

更一般 \(\mathbb{R}P^n\)：\(\mathbb{R}^{n+1} \setminus \{0\} / (v \sim \lambda v) \cong S^n / (p \sim -p)\)——\(n\) 维实射影空间。\(\mathbb{R}P^3\) 同胚于 \(SO(3)\) 旋转群（这是物理"角动量代数" 的几何根源）。\(\mathbb{R}P^n\) 在代数拓扑、代数几何、规范场论里是核心对象。

10.7 商空间陷阱：商一般不 Hausdorff

子空间继承 Hausdorff（ch8.5）、积传 Hausdorff（ch9.7）。可惜商不传——即使 \(X\) Hausdorff, \(X / \sim\) 一般不 Hausdorff。看反例：

反例 1（行带双原点）：取 \(X = \mathbb{R} \sqcup \mathbb{R}\)（两份实数线的不交并）。每份带标准拓扑, \(X\) 是两条平行线，Hausdorff。

定义等价关系：第一条线的 \(x\) 与第二条线的 \(x\) 粘起来——除了 \(x = 0\) 外。也就是 \(x_1 \sim x_2\) iff \(x_1 = x_2 \neq 0\)。其他单点孤立。

商空间 \(X / \sim\) 是什么？除 0 外两条线粘成一条 \(\mathbb{R}\)；而 \(0\) 处有两个点（"第一条的 0" 与"第二条的 0" 没被粘）。这就是"带双原点的实数线"——一条 \(\mathbb{R}\) 上有两个 点占据"原点" 位置。

验证不 Hausdorff：取这两个"原点" \(0_1, 0_2\)。任何含 \(0_1\) 的开集 \(V_1\) 必含 \(0_1\) 的某个开邻域 \((-\varepsilon, \varepsilon)_1\)（带下标 1 表示"第一条线"）。同样 \(V_2\) 必含 \((-\varepsilon, \varepsilon)_2\)。但 \((-\varepsilon, \varepsilon)_1 \setminus \{0_1\}\) 与 \((-\varepsilon, \varepsilon)_2 \setminus \{0_2\}\) 在商空间里是同一个 集合（都被粘合到 \((-\varepsilon, \varepsilon) \setminus \{0\}\)）。所以 \(V_1 \cap V_2 \supseteq (-\varepsilon, \varepsilon) \setminus \{0\} \neq \emptyset\)——分不开。

这个空间叫"带双点的直线"（line with two origins），是教科书最经典的"商打破 Hausdorff" 例子。

反例 2（粗化等价）：取 \(X = \mathbb{R}\)，等价关系定义：所有 \(x \neq 0\) 互相等价（粘成一类），\(0\) 自成一类。

商空间 \(X / \sim = \{[0], [\text{else}]\}\)——两个点。商拓扑下：\(p^{-1}(\{[\text{else}]\}) = \mathbb{R} \setminus \{0\}\) 开 \(\Rightarrow \{[\text{else}]\}\) 开。\(p^{-1}(\{[0]\}) = \{0\}\) 不 开 \(\Rightarrow \{[0]\}\) 不 开。

所以商空间拓扑 \(\tau = \{\emptyset, \{[\text{else}]\}, \{[0], [\text{else}]\}\}\)——这正是 ch1.3 的 Sierpiński 空间！它不 Hausdorff（一个点在另一点的"每个邻域" 里）。

所以"商空间是不是 Hausdorff" 真的不能从 \(X\) 导出——要单独算。下一节给一条充分条件。

10.8 良态商：等价关系是闭子集时商 Hausdorff

要让商空间 Hausdorff，等价关系必须"足够规则"。一条最常用的充分条件：

定理：设 \(X\) 是 Hausdorff 空间，商映射 \(p: X \to X/\sim\) 是开 映射（即 \(U\) 开 \(\Rightarrow p(U)\) 开）。若等价关系图

\[ R := \{(x, x') \in X \times X : x \sim x'\} \]

是 \(X \times X\) 的闭 子集（积拓扑），则 \(X/\sim\) Hausdorff。

证明思路。取 \([x_1] \neq [x_2] \in X/\sim\)。则 \(x_1 \not\sim x_2\)，即 \((x_1, x_2) \notin R\)。\(R\) 闭 \(\Rightarrow X \times X \setminus R\) 开，含 \((x_1, x_2)\)。按积拓扑基存在开矩形 \(U_1 \times U_2 \subseteq X \times X \setminus R\) 使 \((x_1, x_2) \in U_1 \times U_2\)。也就是 \(x_i \in U_i\)，且没有 \((u_1, u_2) \in U_1 \times U_2\) 满足 \(u_1 \sim u_2\)（否则 \((u_1, u_2) \in R\)）。

\(p\) 开 \(\Rightarrow p(U_1), p(U_2)\) 是 \(X/\sim\) 中开集，分别含 \([x_1], [x_2]\)。验它们不相交：若 \([y] \in p(U_1) \cap p(U_2)\)，存在 \(u_1 \in U_1, u_2 \in U_2\) 使 \(p(u_1) = [y] = p(u_2)\)，即 \(u_1 \sim u_2\)——但我们排除了。\(\checkmark\square\)

这条定理给一个验证商 Hausdorff 的具体路径：(1) 验商映射开（这步对很多自然商成立, 如群作用商）, (2) 验关系图 \(R\) 在 \(X \times X\) 中闭。

例（环面）：\(I^2 / \sim \to S^1 \times S^1\)。正向 已经知道环面 Hausdorff（嵌入 \(\mathbb{R}^4\)）。逆推：商映射 \(p: I^2 \to T^2\) 开（点在内部时显然, 边界点处用粘合规则验）；关系图 \(R\) 由有限多条粘合规则给出, 闭。两条都满足, 商 Hausdorff——核对一致。

例（射影空间 \(\mathbb{R}P^n\)）：\(S^n / (p \sim -p)\)。商映射开（\(p\) 与 \(-p\) 周围的小球各自开, 粘起来仍开）。关系图 \(R = \{(p, p) : p \in S^n\} \cup \{(p, -p) : p \in S^n\}\)，由两条对角线（\(\Delta = \{(p, p)\}\) 与 \(\Delta' = \{(p, -p)\}\)）的并组成。\(\Delta\) 在 \(S^n \times S^n\) 中闭（Hausdorff 空间的对角线总闭, 这是 Hausdorff 的等价刻画——见 ch12）；\(\Delta'\) 也闭（用 \((p, q) \mapsto p + q\) 连续, 它是该映射的零点集）。两个闭集的并闭 \(\Rightarrow R\) 闭。所以 \(\mathbb{R}P^n\) Hausdorff。\(\checkmark\)

反例 1 套用：带双原点的直线。它的关系图 \(R\) 在 \((\mathbb{R} \sqcup \mathbb{R})^2\) 里不 闭——按定义 \(R\) 含所有 \((x_1, x_2)\) 使 \(x_1 = x_2 \neq 0\)，加对角线点 \((x_1, x_1), (x_2, x_2)\)。但不 含 \((0_1, 0_2)\)（这两个 \(0\) 没被粘）。然而 \((0_1, 0_2)\) 是一族 \((x_1, x_2)\) 当 \(x \to 0\) 的极限点，也不在 \(R\) 里——所以 \(R\) 不闭，定理失效，商不 Hausdorff。诊断准确。

所以你看：商空间 Hausdorff 与否，可以由"关系图闭不闭" 这条几何条件直接读出。这是一条非常清亮的判据，配 ch12 分离公理章会反复用。
