第 12 章 分离公理：T0-T4 与 Hausdorff

请你先回到第 1 章那个 2 元素小空间——Sierpiński 空间

\[ X = \{0, 1\}, \qquad \tau = \{\emptyset, \{1\}, \{0, 1\}\}. \]

三条公理我们已经验证过了。它是 拓扑空间，没有任何犯规之处。但请你停下来仔细问一句：在这个空间里，"0 和 1 能不能分开"？

"分开" 在拓扑里有一个最朴素的意思——找两个不交 的开集 \(U, V\)，一个含 0、一个含 1。在度量空间里这是免费的：\(0\) 和 \(1\) 距离 1，取半径 \(1/2\) 的两个开球就行。但在 Sierpiński 拓扑里我们试一下。

含 0 的开集只有一个——\(\{0, 1\}\) 自己（因为 \(\{0\}\) 不是开集，\(\{1\}\) 也不含 0）。它包含 1。所以"能含 0 又不含 1" 的开集不存在。0 与 1 在这个拓扑里分不开。

这件事不是技术细节——它有真 后果。考虑常值序列 \(0, 0, 0, \ldots\) 在 Sierpiński 上。它收敛到 0 吗？当然——任何含 0 的开邻域都包含所有项 0。但它同时 收敛到 1：唯一含 1 的开邻域是 \(\{1\}\) 与 \(\{0, 1\}\)，前者不含 0 不需考虑（不是 1 的邻域？我们要看含 1 的邻域），等一下——\(\{1\}\) 是开集且含 1，是 1 的邻域；它不含 0，所以序列 \(0, 0, 0, \ldots\) 的项最终不 全在 \(\{1\}\) 里。所以 \(0, 0, \ldots\) 不 收敛到 1。

等一下重新算：含 1 的开邻域有 \(\{1\}\) 和 \(\{0, 1\}\)。要"序列收敛到 1" 意味着对每个 含 1 的开邻域，序列从某项起全在里面。\(\{1\}\) 不含 0——\(0, 0, \ldots\) 没有项落进去——所以不 收敛到 1。

反方向：常值 \(1, 1, 1, \ldots\) 收敛到 1（显然），它收敛到 0 吗？含 0 的开邻域只有 \(\{0, 1\}\)，所有项 1 都在里面——满足条件——所以 \(1, 1, \ldots\) 同时 收敛到 0 与 1！

这就是分不开 的真正后果——序列极限不唯一。\(1, 1, 1, \ldots\) 同时收敛到两个不同点。这种空间里"极限" 这个词失去意义。

这一章把"能分开多少" 系统化——T0、T1、T2 (Hausdorff)、T3、T4 五条分离公理。每条都说"某种东西能用不交开集分开"。它们组成一条阶梯，越往上要求越强、可用工具越多。大部分实际研究的拓扑空间至少满足 Hausdorff (T2)——所以你今后看到"设 \(X\) 是 Hausdorff 空间" 这种开头，要想起背后这件事：它在保证极限唯一、连续映射可由值决定。

12.1 Sierpiński 引子：分不开导致什么

把上面的观察整理成定理形式。

观察 12.1. 在 Sierpiński 空间 \((\{0, 1\}, \tau)\) 上，常值序列 \(1, 1, 1, \ldots\) 收敛到 \(0\) 也 收敛到 \(1\)。所以"序列极限" 不唯一。

证明已经在引子里给出。把它看作一面警告——任何不满足"两点能分开" 的拓扑空间都会 出现这种现象。

更一般地：

观察 12.2. 设 \(X\) 是拓扑空间，\(x \neq y \in X\)。如果每个 含 \(x\) 的开邻域都含 \(y\)，那么任何收敛到 \(y\) 的序列也收敛到 \(x\)。

证明一行：序列 \(x_n \to y\) 意味着对 \(y\) 的每个 开邻域 \(V\)，\(x_n\) 最终落入 \(V\)。但 \(x\) 的每个 开邻域 \(U\) 也是 \(y\) 的开邻域（因为 \(y \in U\) 由假设），所以 \(x_n\) 也最终落入 \(U\)，即 \(x_n \to x\)。\(\blacksquare\)

这条观察告诉我们：分离公理的最低 起点必须是"有某个开集区分 \(x\) 与 \(y\)"。否则\(x\) 与 \(y\) 在拓扑里看起来一样——任何序列、任何收敛、任何连续函数都分不清它们。

12.2 T0：用一个开集区分两点

把"最低 区分要求" 写下来。

定义 12.3 (T0 / Kolmogorov)。 拓扑空间 \(X\) 称为 \(T_0\)，如果对任意两个不同的点 \(x \neq y\)，存在某个 开集 \(U\) 使

\[ (x \in U \text{ 且 } y \notin U) \quad \text{或} \quad (y \in U \text{ 且 } x \notin U). \]

用人话：至少有一个开集区分得开 \(x\) 和 \(y\)——含其中一个不含另一个。注意这里的"或"——只要一边能区分就够了，不 要求两边都能。

验证：

  • Sierpiński \(\{0, 1\}\) 配 \(\tau = \{\emptyset, \{1\}, \{0, 1\}\}\)：开集 \(\{1\}\) 含 1 不含 0——区分 0 与 1 这两点的一边。所以 Sierpiński 是 \(T_0\)。
  • 平凡拓扑 \(\tau_0 = \{\emptyset, X\}\) 在 \(|X| \geq 2\) 时不 \(T_0\)——唯一的非空开集是 \(X\)，无法区分任何两点。
  • 度量空间永远 \(T_0\)：\(x \neq y \Rightarrow d(x, y) > 0\)，开球 \(B(x, d(x, y) / 2)\) 含 \(x\) 不含 \(y\)。\(\checkmark\)

所以 \(T_0\) 是一条非常弱的条件——平凡拓扑都不满足，但只要"开集多到能看出点的不同"，就 \(T_0\)。

等价说法：\(T_0 \iff\) 不同点有不同的开邻域族。即映射 \(x \mapsto \mathcal{N}(x) := \{U \text{ 开} : x \in U\}\) 从 \(X\) 到 \(\mathcal{P}(\tau)\) 是单射。这条等价让"\(T_0\) 表达点能被开集区分" 变得直观——\(T_0\) 拓扑里"点" 与"它的邻域族" 一一对应。

12.3 T1：每个单点是闭集

T0 还允许"一边能区分"——一边不行没关系。下一级要求两边都行。

定义 12.4 (T1 / Fréchet)。 拓扑空间 \(X\) 称为 \(T_1\)，如果对任意 \(x \neq y\)，存在开集 \(U \ni x, y \notin U\) 并且 开集 \(V \ni y, x \notin V\)。

注意 \(U\) 与 \(V\) 不 要求不交——只要各自能"一边盯紧、另一边排除"。

Sierpiński 不 \(T_1\)：含 0 不含 1 的开集没有。

余有限拓扑 \(\tau_{\text{cofin}}\) 在无限集 \(X\) 上是 \(T_1\)：取 \(U = X \setminus \{y\}\)，余集 \(\{y\}\) 有限——\(U\) 是开集，含 \(x\) 不含 \(y\)。同理 \(V = X \setminus \{x\}\) 含 \(y\) 不含 \(x\)。\(\checkmark\)

定理 12.5 (T1 等价单点闭)。 \(X\) 是 \(T_1 \iff\) 对每个 \(x \in X\)，\(\{x\}\) 是闭集。

证明。

(\(\Rightarrow\))。假设 \(T_1\)。固定 \(x\)。要证 \(\{x\}\) 闭，等价于 \(X \setminus \{x\}\) 开。对每个 \(y \in X \setminus \{x\}\)（即 \(y \neq x\)），由 \(T_1\) 取开集 \(V_y \ni y, x \notin V_y\)。即 \(V_y \subseteq X \setminus \{x\}\)。所以 \(X \setminus \{x\} = \bigcup_{y \neq x} V_y\)——开集的并，开。

(\(\Leftarrow\))。假设每个单点闭。对 \(x \neq y\)，\(\{y\}\) 闭，故 \(U := X \setminus \{y\}\) 开且含 \(x\)（因 \(x \neq y\)），不含 \(y\)。同理 \(V := X \setminus \{x\}\) 开含 \(y\) 不含 \(x\)。\(T_1\)。\(\blacksquare\)

这条等价是 \(T_1\) 的主刻画。今后看到"\(\{x\}\) 闭" 就要想起 \(T_1\)。在大多数 你研究的空间里这件事自动成立——但代数几何里 Spec 的拓扑（"素谱"）就不 \(T_1\)，因为非极大素理想的"点" 的闭包包含其它"点"。

12.4 T2 (Hausdorff)：两点有不交开邻域

到了拓扑学里出现频率最高的一条分离公理。

定义 12.6 (T2 / Hausdorff)。 拓扑空间 \(X\) 称为 \(T_2\) 或 Hausdorff，如果对任意 \(x \neq y\)，存在不交 开集 \(U \ni x, V \ni y\)。即 \(U \cap V = \emptyset\)。

这就是引子里"能分开" 的精确说法——两个点不仅有"区分开集"（\(T_1\)），更有互斥 的开邻域。

典型例子：

  • 所有度量空间 Hausdorff——\(B(x, d(x,y)/2)\) 与 \(B(y, d(x,y)/2)\) 不交。\(\checkmark\)
  • \(\mathbb{R}^n\) 标准拓扑 Hausdorff（度量情形）。
  • 离散拓扑 Hausdorff（取 \(U = \{x\}, V = \{y\}\)）。

非例：

  • Sierpiński——已说，不 Hausdorff。
  • 余有限拓扑在无限集上——不 Hausdorff。任何两个非空开集的余集都有限，所以两个非空开集的余集之并 有限，交 的余集 \(=\) 余集之并有限——意味交是"余有限集"，非空（因为 \(X\) 无限）。所以任何两个非空开集相交。无法分开。但它是 \(T_1\)。

这第二个例子很重要——它告诉我们 \(T_1 \not\Rightarrow T_2\)。两条公理真 不一样。

"Hausdorff" 在大多数现代拓扑文献里是默认假设。一打开任何分析 / 几何 / 代数拓扑教材，你看到"设 \(X\) 是拓扑空间" 默认就是 Hausdorff——因为不 Hausdorff 时极限唯一性、紧致集闭等基本结论会失败，做不下去。

12.5 T3 (regular)：点 + 闭集分开

到这里"分" 的对象升级——从"两个点" 升级到"一个点 + 一个不含它的闭集"。

定义 12.7 (T3 / regular)。 拓扑空间 \(X\) 称为 regular 如果它是 \(T_1\)并且 对任意闭集 \(F\) 与点 \(x \notin F\)，存在不交 开集 \(U \ni x, V \supseteq F\)。称为 \(T_3\) 通常等同 regular。

请你停下来想这条要求在说什么。它比 Hausdorff 强：取 \(F = \{y\}\) 单点（在 \(T_1\) 下闭），就回到"点 \(x\) 与点 \(y\) 用不交开集分开"——即 Hausdorff。但 regular 要求不止 单点闭集——要求所有 闭集（可能非常大）都能与外部点分开。

等价刻画（很常用）：

定理 12.8. \(X\) 是 \(T_1\)。则 \(X\) regular \(\iff\) 对每个点 \(x\) 与含 \(x\) 的每个开集 \(W\)，存在开集 \(U\) 使 \(x \in U \subseteq \overline{U} \subseteq W\)。

用人话：每个开邻域里"能塞进一个更小的开邻域且其闭包仍在里面"。这条性质叫"开邻域的闭包细化"，在很多构造（比如 Urysohn 引理）里反复用。

证明。

(\(\Rightarrow\))。给 \(x \in W\)，\(W\) 开。则 \(F := X \setminus W\) 闭且 \(x \notin F\)。由 regular 取不交开集 \(U \ni x, V \supseteq F\)。则 \(U \subseteq X \setminus V\) 闭集——所以 \(\overline{U} \subseteq X \setminus V \subseteq X \setminus F = W\)。即 \(x \in U \subseteq \overline{U} \subseteq W\)。\(\checkmark\)

(\(\Leftarrow\))。给 \(F\) 闭、\(x \notin F\)。则 \(W := X \setminus F\) 开含 \(x\)。由假设取 \(U\) 使 \(x \in U \subseteq \overline{U} \subseteq W\)。则 \(V := X \setminus \overline{U}\) 开，含 \(F\)（因 \(\overline{U} \subseteq W = X \setminus F \Rightarrow F \subseteq X \setminus \overline{U}\)），且 \(U \cap V = U \cap (X \setminus \overline{U}) \subseteq U \cap (X \setminus U) = \emptyset\)。\(\blacksquare\)

典型例子：所有度量空间 regular（事实上 normal，下一节）。\(\mathbb{R}^n\)、所有 \(\mathbb{R}^n\) 子空间、所有 Banach 空间——都 regular。

历史小注：早期拓扑文献里 \(T_3 = \) "regular + \(T_1\)"，但有些作者把 regular 不要求 \(T_1\)。本书采取"regular = \(T_3\) = (R0) + 点闭集分开 + \(T_1\)" 的强约定，避免歧义。

12.6 T4 (normal)：两个闭集分开

把"分" 的对象再升级——两个不交闭集。

定义 12.9 (T4 / normal)。 拓扑空间 \(X\) 称为 normal 如果它是 \(T_1\)并且 对任意两个不交闭集 \(F, G\)，存在不交 开集 \(U \supseteq F, V \supseteq G\)。\(T_4\) 通常等同 normal。

同样的"用闭包细化邻域" 等价：

定理 12.10. \(X\) 是 \(T_1\)。\(X\) normal \(\iff\) 对每个闭集 \(F\) 与含 \(F\) 的每个开集 \(W\)，存在开集 \(U\) 使 \(F \subseteq U \subseteq \overline{U} \subseteq W\)。

证明完全类比 12.8——用 \(F\) 替换"点"。

normal 是分离阶梯里最强 的一档。它强到允许构造非平凡的连续函数——这就是Urysohn 引理（第 20 章）：

\[ \boxed{\ X \text{ normal} + F, G \text{ 不交闭} \Rightarrow \exists \text{ 连续 } f : X \to [0, 1], \ f|_F = 0, f|_G = 1.\ } \]

这条引理把"能用开集分" 升级成"能用连续函数分"——是后续度量化定理（Urysohn 度量化）的关键工具。它需要 normal——更弱的条件不够。

典型例子：

  • 所有度量空间 normal（详见 12.9 节）。
  • 所有紧致 Hausdorff 空间 normal（第 15 章详证）。

非例：

  • Sorgenfrey 直线的积 \(\mathbb{R}_\ell \times \mathbb{R}_\ell\)（"Sorgenfrey 平面"）是 regular Hausdorff 但不 normal。这是经典反例 (Munkres §31)。说明 normal 不 被任意积保持——比 regular 难得多。

12.7 阶梯：T4 ⟹ T3 ⟹ T2 ⟹ T1 ⟹ T0

把五条整理成阶梯。

定理 12.11. \(T_4 \Rightarrow T_3 \Rightarrow T_2 \Rightarrow T_1 \Rightarrow T_0\)。

证明。

\(T_4 \Rightarrow T_3\)：normal 给"两闭集分开"。取 \(F = \{x\}\)（在 \(T_1\) 下闭），\(G\) 任意闭集不含 \(x\)，得到不交开集分开 \(\{x\}\) 与 \(G\)——即 regular。

\(T_3 \Rightarrow T_2\)：取 \(F = \{y\}\)（在 \(T_1\) 下闭），\(x \neq y\)，得到不交开集分开 \(x\) 与 \(\{y\}\)——即 Hausdorff。

\(T_2 \Rightarrow T_1\)：Hausdorff 给两个不交开邻域 \(U \ni x, V \ni y\)——分别能区分两点。\(T_1\) 满足。

\(T_1 \Rightarrow T_0\)：\(T_1\) 的"双向区分" 显然蕴含 \(T_0\) 的"单向区分"。\(\blacksquare\)

反向不 成立。我们已看过两个反例：

  • Sierpiński 是 \(T_0\) 但不 \(T_1\)（不是单点闭：\(\{1\}\) 开但 \(\{0\}\) 不闭）。
  • 余有限拓扑（无限集上）是 \(T_1\) 但不 \(T_2\)。

另两个标准反例：

  • "含两点不能用开集分" 的某些 Hausdorff 空间不 regular（构造稍长）。Munkres §31 给具体例子。
  • Sorgenfrey 平面 regular Hausdorff 但不 normal。

这些反例的存在证明阶梯真 是阶梯——每一档都是真正多一份要求。

12.8 Hausdorff 的关键后果

Hausdorff 是阶梯里"最常用" 的一档。它给出三件本书反复用的事。

(1) 紧致集闭。 在 Hausdorff 空间 \(X\) 里，每个紧致子集 \(K \subseteq X\) 是闭集。

证明（梗概，详见第 14 章）：要证 \(X \setminus K\) 开。给 \(y \notin K\)，对每个 \(x \in K\) 由 Hausdorff 取不交开集 \(U_x \ni x, V_x \ni y\)。族 \(\{U_x\}_{x \in K}\) 覆盖 \(K\)，由紧致取有限子覆盖 \(U_{x_1}, \ldots, U_{x_n}\)。则 \(V := V_{x_1} \cap \cdots \cap V_{x_n}\) 是 \(y\) 的开邻域且与 \(\bigcup U_{x_i} \supseteq K\) 不交。\(V \subseteq X \setminus K\)。每个 \(y\) 都有这样的 \(V\)——\(X \setminus K\) 开。\(\checkmark\)

这是 Hausdorff 的第一 个关键作用——它让"紧致" 这个性质在组合（取闭包、补集）下良好。在非 Hausdorff 空间里紧致集可能不闭——比如 Sierpiński 上 \(\{0\}\) 紧致（有限集自动紧致）但不闭（\(\{1\}\) 才是开集，\(\{0\}\) 的补 \(\{1\}\) 开 \(\Rightarrow \{0\}\) 闭——等等再想想）。Sierpiński 上单点 \(\{0\}\)：\(X \setminus \{0\} = \{1\}\) 是开集，所以 \(\{0\}\) 闭。\(\{1\}\) 的补 \(\{0\}\) 闭——\(\{1\}\) 开。所以 Sierpiński 上单点都有闭/开属性，但不 Hausdorff。这说明"紧致集闭" 在非 Hausdorff 下可能仍成立——但不能保证。一般非 Hausdorff 反例可见 \(\mathbb{R}\) 配余有限拓扑加一些技巧。

(2) 序列极限唯一。 在 Hausdorff 空间 \(X\) 里，如果序列 \((x_n) \to x\) 且 \((x_n) \to y\)，则 \(x = y\)。

证明。反证：假设 \(x \neq y\)。由 Hausdorff 取不交开集 \(U \ni x, V \ni y\)。\(x_n \to x \Rightarrow x_n\) 最终 \(\in U\)；\(x_n \to y \Rightarrow x_n\) 最终 \(\in V\)。最终两边都满足意味着 \(x_n\) 最终 \(\in U \cap V = \emptyset\)，矛盾。\(\blacksquare\)

这就是引子里 Sierpiński 现象在 Hausdorff 下被消除——分开两点保证序列不能"同时 收敛到两边"。

(3) 连续映射"等价于" 决定。 设 \(Y\) Hausdorff，\(f, g : X \to Y\) 连续，\(D \subseteq X\) 稠密。如果 \(f|_D = g|_D\)，则 \(f = g\)。

证明。集合 \(E := \{x \in X : f(x) = g(x)\}\)。要证 \(E = X\)。已知 \(D \subseteq E\)，\(D\) 稠密——\(\overline{D} = X\)。要证 \(E\) 闭。

等价证明 \(X \setminus E = \{x : f(x) \neq g(x)\}\) 开。给 \(x \in X \setminus E\)，\(f(x) \neq g(x)\)。由 Hausdorff（在 \(Y\) 里）取不交开集 \(U \ni f(x), V \ni g(x)\)。则 \(W := f^{-1}(U) \cap g^{-1}(V)\) 是 \(x\) 的开邻域（\(f, g\) 连续保证 \(f^{-1}(U), g^{-1}(V)\) 开）。对 \(z \in W\)：\(f(z) \in U, g(z) \in V\)，因 \(U \cap V = \emptyset\) 必 \(f(z) \neq g(z)\)，即 \(z \in X \setminus E\)。所以 \(W \subseteq X \setminus E\)。\(X \setminus E\) 开 \(\Rightarrow E\) 闭 \(\Rightarrow E \supseteq \overline{D} = X\) \(\Rightarrow E = X\)。\(\blacksquare\)

这条结论叫"连续映射在稠密子集上的值决定整体"。在泛函分析 里这是"有界算子由稠密定义域延拓唯一" 的基础。需要 目标空间 Hausdorff——目标非 Hausdorff 时两个连续映射可在稠密集相同但全局不同。

12.9 度量空间自动 T4

最后看一件事：度量空间永远 normal。这条让我们不必担心阶梯——任何熟悉的实分析空间（\(\mathbb{R}^n\)、Banach、Hilbert）都自动满足所有分离公理。

定理 12.12. 度量空间 \((X, d)\) normal。

证明。

显然 \(T_1\)：单点 \(\{x\}\) 闭。事实上 \(\{x\} = \bigcap_n \overline{B(x, 1/n)}\)（闭集的交闭），或更简单：\(X \setminus \{x\} = \bigcup_{y \neq x} B(y, d(x, y) / 2)\)，开集的并开。

给两个不交闭集 \(F, G\)。要构造不交开集 \(U \supseteq F, V \supseteq G\)。

关键工具——距离到集合。定义 \(d(x, A) := \inf_{a \in A} d(x, a)\) 对任意集合 \(A \neq \emptyset\)。这函数 \(d(\cdot, A) : X \to \mathbb{R}\) 连续（\(|d(x, A) - d(y, A)| \leq d(x, y)\)），且 \(d(x, A) = 0 \iff x \in \overline{A}\)。所以对闭集 \(A\)，\(d(x, A) = 0 \iff x \in A\)。

对 \(x \in F\)：\(x \notin G\)（不交），\(G\) 闭，故 \(d(x, G) > 0\)。设 \(r_x := d(x, G) / 2 > 0\)。

对 \(y \in G\)：\(y \notin F\)，故 \(d(y, F) > 0\)。设 \(s_y := d(y, F) / 2 > 0\)。

定义 \(U := \bigcup_{x \in F} B(x, r_x)\)，\(V := \bigcup_{y \in G} B(y, s_y)\)。开（开球的并）。\(U \supseteq F\)，\(V \supseteq G\)（每个 \(x\) 在 \(B(x, r_x) \subseteq U\)）。

剩下证 \(U \cap V = \emptyset\)。反证：取 \(z \in U \cap V\)。则有 \(x \in F, y \in G\) 使

\[ d(z, x) < r_x = d(x, G) / 2, \qquad d(z, y) < s_y = d(y, F) / 2. \]

由三角不等式 \(d(x, y) \leq d(x, z) + d(z, y) < d(x, G) / 2 + d(y, F) / 2\)。

但 \(d(x, G) \leq d(x, y)\)（因 \(y \in G\)）与 \(d(y, F) \leq d(x, y)\)（因 \(x \in F\)）。代入：\(d(x, y) < d(x, y) / 2 + d(x, y) / 2 = d(x, y)\)。矛盾。所以 \(U \cap V = \emptyset\)。\(\blacksquare\)

这条定理一句话总结：所有度量空间满足整条阶梯 T0 到 T4。这就是为什么实分析读本从来不区分这些公理——它们在 \(\mathbb{R}^n\) 与 Banach 空间里同时为真。抽象拓扑 才需要分开它们：因为存在不来自度量的拓扑（Sierpiński、余有限、Sorgenfrey、Spec），这些场合阶梯之间真有差异。

反过来——不 是每个 normal Hausdorff 空间都来自度量。能不能配度量是度量化问题。Urysohn 度量化定理（第 21 章）回答："第二可数 + normal + Hausdorff" 的空间可度量化。这把分离公理与可数性公理（下一章）结合，给出可度量化的充分 条件。

12.10 临别话

分离公理把"能分多开" 这条直观感觉变成五档可计较的条件。在实用 上：

  • Hausdorff 是几乎默认假设。除非你明确做非 Hausdorff 拓扑（代数几何 Spec、domain theory、descriptive set theory 的某些角落），都假设 Hausdorff。
  • regular 在度量空间外有时缺失，但在大多数 流形 / 紧致 / 可数的标准空间里成立。
  • normal 是能造连续函数 的门槛。Urysohn 引理 + Tietze 延拓需要它。但 normal 在积 下不保——所以构造上要小心。

下一章我们换一类公理——可数性公理。它问的不是"能分多开"，而是"开集族能不能被可数地控制"。这条性质决定"序列" 这种可数对象在拓扑里够不够用——在第一可数空间里序列充分，否则得用网 或滤子。
