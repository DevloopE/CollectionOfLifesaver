第 2 章 度量空间复习

上一章承诺过：连续这件事，真正需要的是"开集"，不需要"距离"。下一章 ch3 起就把"开集" 抽象出来当原始概念。

但在跳到抽象之前，我们要在这一章做最后一站准备：把你在实分析里学过的度量空间 按"面向拓扑" 的视角重新整理一遍。同样的概念——开球、收敛、连续、紧致、连通——你过去都用距离 描述。这一章每个概念都给两版：

  • 用距离的版本：你熟悉的 \(\varepsilon\)-\(\delta\) 风格定义。
  • 用开集的等价版本：抽掉距离, 只剩开集。

这种"双重写" 让 ch3 跳到无距离时不撞墙——你会发现每个抽象拓扑定义都是 你熟悉的度量定义的等价改写。换言之，抽象拓扑空间不是新对象, 是把度量空间里多余的部分（距离的具体数值）忘掉之后剩下的部分。

2.1 度量公理 + 4 个例子

请你先复习度量的形式定义。度量空间 是一对 \((X, d)\)，其中 \(X\) 是集合，\(d : X \times X \to [0, \infty)\) 是度量，满足：

  • (M1) \(d(x, y) = 0 \iff x = y\)（正定性）。
  • (M2) \(d(x, y) = d(y, x)\)（对称性）。
  • (M3) \(d(x, z) \le d(x, y) + d(y, z)\)（三角不等式）。

看四个具体例子让"度量" 变成具体经验。

例 1：欧氏度量。\(X = \mathbb{R}^n\)，\(d_2(x, y) = \sqrt{\sum_i (x_i - y_i)^2}\)。这是几何课的"距离"。三角不等式 = Cauchy-Schwarz 的几何形式。验证略。

例 2：sup 度量。同样在 \(\mathbb{R}^n\) 上，定义 \(d_\infty(x, y) = \max_i |x_i - y_i|\)。验证三公理（M1-M3）：(M1) \(\max = 0\) iff 每项 \(= 0\) iff \(x = y\)。(M2) 对称显然。(M3) 三角：每项 \(|x_i - z_i| \le |x_i - y_i| + |y_i - z_i| \le d_\infty(x, y) + d_\infty(y, z)\)，对所有 \(i\) 取 max。\(\checkmark\)

这个 sup 度量的"球" 是正方体（每条坐标轴方向 \(\le r\)），不是球。所以"球" 这个词的几何形状取决于度量。

例 3：离散度量。任意 \(X\)，定义

\[ d_{\text{disc}}(x, y) = \begin{cases} 0, & x = y, \\ 1, & x \neq y. \end{cases} \]

验证：(M1)(M2) 显然。(M3) 三角：左边 \(\le 1\)，右边 \(\ge 0\)；如果左边 \(= 1\) 那 \(x \neq z\)，所以 \(y\) 不能同时等于 \(x\) 和 \(z\)，至少一个 \(d(x, y), d(y, z)\) 等于 1，右边 \(\ge 1\)。\(\checkmark\)

这个度量"蠢"——它不区分远近, 只区分相同 / 不同。但它是个合法度量, 而且产生极端 的拓扑（每个单点集都开——见 2.7）。它在反例库里很有用。

例 4：\(L^p\) 度量（函数空间）。\(X = C([0, 1], \mathbb{R})\)（[0,1] 上连续函数全体），\(p \in [1, \infty)\)，

\[ d_p(f, g) = \left( \int_0^1 |f(x) - g(x)|^p \, dx \right)^{1/p}.

三角不等式 = Minkowski 不等式（详见实分析 ch11）。当 \(p = \infty\), 改为 \(d_\infty(f, g) = \sup_{x \in [0,1]} |f(x) - g(x)|\)。

这个例子关键——它告诉你"度量空间" 这个概念不限于几何里的 \(\mathbb{R}^n\)。函数也可以"有距离"，距离可以是积分平均（\(L^1\)）、均方（\(L^2\)）、最大偏差（\(L^\infty\)）。整个泛函分析就建在这种"函数也有距离" 的视角上。

2.2 开球 → 开集 → 度量拓扑

给度量空间 \((X, d)\)，开球 是

\[ B(x_0, r) = \{x \in X : d(x_0, x) < r\}, \quad r > 0. \]

注意是严格 小于。"球面上" 的点不算。

开集定义。\(U \subseteq X\) 开 \(\iff\) 每个 \(x \in U\) 都有半径 \(r > 0\) 使 \(B(x, r) \subseteq U\)。

也就是说"开" = "每个点都被自己附近的一整个球包住"——没有"边缘" 上的点。

例子。\(\mathbb{R}\) 配欧氏度量：开区间 \((a, b)\) 开。验证：取 \(x \in (a, b)\)，\(r = \min(x - a, b - x) > 0\)，\(B(x, r) = (x - r, x + r) \subseteq (a, b)\)。\(\checkmark\)

非例子。\([0, 1]\) 不 开。验证：取 \(x = 0\)，任何 \(r > 0\) 给的 \(B(0, r) = (-r, r)\) 含负数，不在 \([0, 1]\) 里。0 是"边" 点。

关键事实。度量空间里"开集" 满足上一章 ch1.2 的三公理 (O1)(O2)(O3)。

验证：(O1) \(\emptyset\) 没有点要检查（vacuously 真），\(X\) 显然每点配球。(O2) 任意并：每个 \(x \in \bigcup U_\alpha\) 在某 \(U_{\alpha_0}\) 里，\(U_{\alpha_0}\) 给的球 \(B(x, r) \subseteq U_{\alpha_0} \subseteq \bigcup U_\alpha\)。(O3) 有限交：每个 \(x \in \bigcap_{i=1}^n U_i\) 在每个 \(U_i\) 里都有球 \(B(x, r_i)\)，取 \(r = \min r_i > 0\)（这里要有限性！），球 \(B(x, r)\) 同时落在所有 \(U_i\) 里。

所以度量给的"开集" 集族——记作 \(\tau_d\)——是 \(X\) 上的拓扑（满足三公理的开集族）。我们叫它度量诱导的拓扑，简称度量拓扑。

请你停下来感受这个翻译：度量空间 = 集合 + 距离，自动产生 = 集合 + 开集族（满足三公理）。距离 这一信息编码 进了"哪些集合是开集"。下一章 ch3 起我们干脆把距离扔掉, 只保留开集族——发现剩下的东西已经足以谈连续。

但注意：从距离到开集这一步会丢掉一些 信息。两个不同的度量可以给同一个拓扑——见 2.7 节。

2.3 收敛：度量版 vs 开集版

序列 \(\{x_n\} \subseteq X\) 收敛 到 \(x \in X\)，记 \(x_n \to x\)，度量版定义：

\[ \forall \varepsilon > 0, \exists N, \forall n \ge N : d(x_n, x) < \varepsilon. \]

等价地，实数序列 \(d(x_n, x) \to 0\)。

请你把这条等价改写为"用开集"。"\(d(x_n, x) < \varepsilon\)" 翻成"\(x_n \in B(x, \varepsilon)\)"，所以条件 = "\(\forall \varepsilon > 0\), \(\exists N\), \(\forall n \ge N\)，\(x_n \in B(x, \varepsilon)\)"。

但这里 \(B(x, \varepsilon)\) 是个"含 \(x\) 的开集"。事实上每个含 \(x\) 的开集 \(U\)，按开集定义，包含某个 \(B(x, \varepsilon) \subseteq U\)。所以"球" 可以替换为"含 \(x\) 的任意开集"，等价：

\[ \boxed{\ x_n \to x \iff \forall U \text{ 开}, x \in U, \exists N, \forall n \ge N : x_n \in U.\ } \]

用人话：每个含 \(x\) 的开集, 最终包含全部 \(x_n\)。

请你停下来盯住这条等价。它没有 提到 \(\varepsilon, \delta\), 没有 提到距离——只用了"开集"。所以"收敛" 和"连续" 一样, 真正 需要的是开集结构, 不是距离。这条等价整个 搬到 ch3 起的抽象拓扑空间——它是"序列收敛" 在抽象空间的定义。

例。\(\mathbb{R}\) 上 \(x_n = 1/n\)。每个含 0 的开集 \(U\) 包含某 \((-\varepsilon, \varepsilon)\)；只要 \(n \ge 1/\varepsilon\) 就有 \(1/n < \varepsilon\)，\(x_n \in U\)。所以 \(x_n \to 0\)。

警告：在一般 拓扑空间里, "收敛" 不再控制连续——序列收敛可能不够 抓住拓扑结构。这是ch15 可数性会处理的事——"第一可数" 空间里序列还够, 否则要用更广的"网" 或"滤子"。但在度量空间里序列足够。

2.4 连续：度量版 vs 开集版

这就是第 1 章 1.1 节那条等价的细节版。给度量空间之间的映射 \(f : X \to Y\)。

度量版定义。\(f\) 在 \(x_0\) 连续 \(\iff\)

\[ \forall \varepsilon > 0, \exists \delta > 0, \forall x : d_X(x, x_0) < \delta \implies d_Y(f(x), f(x_0)) < \varepsilon. \]

\(f\) 连续 \(\iff\) 在每点都连续。

开集版等价。\(f\) 连续 \(\iff\) 每个开集 \(V \subseteq Y\) 的逆像 \(f^{-1}(V) \subseteq X\) 开。

这条等价的证明就一段, 我把上一章的承诺兑现。

"度量版 \(\Rightarrow\) 开集版"：取 \(V \subseteq Y\) 开, 要证 \(f^{-1}(V)\) 开。取 \(x_0 \in f^{-1}(V)\), 即 \(f(x_0) \in V\)。\(V\) 开意味着 \(\exists \varepsilon > 0\) 使 \(B(f(x_0), \varepsilon) \subseteq V\)。\(f\) 在 \(x_0\) 连续意味着 \(\exists \delta > 0\) 使 \(d_X(x, x_0) < \delta \Rightarrow d_Y(f(x), f(x_0)) < \varepsilon\)，即 \(f(x) \in B(f(x_0), \varepsilon) \subseteq V\)，即 \(x \in f^{-1}(V)\)。所以 \(B(x_0, \delta) \subseteq f^{-1}(V)\)，\(f^{-1}(V)\) 开。

"开集版 \(\Rightarrow\) 度量版"：固定 \(x_0 \in X\)，给任意 \(\varepsilon > 0\)，\(B(f(x_0), \varepsilon) \subseteq Y\) 开, 所以 \(f^{-1}(B(f(x_0), \varepsilon))\) 开, 含 \(x_0\)。所以 \(\exists \delta > 0\) 使 \(B(x_0, \delta) \subseteq f^{-1}(B(f(x_0), \varepsilon))\), 即 \(d_X(x, x_0) < \delta \Rightarrow d_Y(f(x), f(x_0)) < \varepsilon\)。\(f\) 在 \(x_0\) 连续。\(\square\)

请你停下来欣赏这条等价的无距离 性质：右边"每个开集的逆像开" 完全是关于开集 的语句, 没有出现度量。所以"连续" 这个概念可以移植 到任何只有开集结构的空间——这就是 ch3 起的拓扑空间。

序列刻画。第三种等价（度量空间里）：\(f\) 在 \(x_0\) 连续 \(\iff\) 对任何 \(x_n \to x_0\) 有 \(f(x_n) \to f(x_0)\)。这条在度量空间里成立, 但在一般 拓扑空间里只是必要、不一定充分（除非空间是第一可数的）。所以"开集版" 是连续的真正 定义, "序列版" 只在好的空间里也对。

2.5 紧致：Heine-Borel 与序列紧

紧致 是度量空间最重要的性质之一。它有三种 等价定义（在度量空间里, 不一定在一般拓扑里）。

定义 A（开覆盖紧致）。\(X\) 紧致 \(\iff\) 每个 \(X\) 的开覆盖 \(\{U_\alpha\}\) 都有有限 子覆盖。

"开覆盖" = 一族开集 \(\{U_\alpha\}\) 满足 \(X = \bigcup_\alpha U_\alpha\)。"有限子覆盖" = 从中选有限多个 \(U_{\alpha_1}, \ldots, U_{\alpha_n}\) 仍覆盖 \(X\)。

这条纯开集 定义 \(\Rightarrow\) 它直接搬到一般拓扑空间, 是 ch16 紧致性的原始 定义。

定义 B（序列紧）。\(X\) 序列紧 \(\iff\) 每个序列 \(\{x_n\} \subseteq X\) 有收敛子列。

这是更几何的版本。Bolzano-Weierstrass 定理（实分析 ch3）说 \([a, b]\) 序列紧。

定义 C（Heine-Borel）。当 \(X = \mathbb{R}^n\), \(K \subseteq \mathbb{R}^n\) 紧致 \(\iff\) \(K\) 有界 且闭。

这是最常用的"判紧致" 工具——但只 对 \(\mathbb{R}^n\)（更一般的有限维赋范空间）成立。一般度量空间里"有界 + 闭" 不一定紧致（反例：\(\ell^2\) 中闭单位球不紧致, 因为它含一族正交点列 \(e_1, e_2, \ldots\) 没有收敛子列）。

等价性（度量空间里）。定义 A \(\iff\) 定义 B。证明用 Lebesgue 数引理 + 全有界性。\(\mathbb{R}^n\) 里再加 \(\iff\) 定义 C（Heine-Borel）。

请你记住：开覆盖紧致 是真正 的紧致, 因为它不依赖 度量, 在抽象拓扑空间里仍有定义。序列紧 在度量空间里等价, 在一般拓扑空间里不 等价（一般空间里两个性质独立）。

紧致性的意义：

  • 有限性条件——把"无穷个" 减为"有限多个"。这条让很多分析论证（找最大值、用一致连续）变得可行。
  • 紧致空间上连续函数 → 有界、达到极值——极值定理的拓扑根源。
  • 紧致 + Hausdorff = "近乎" 度量化（ch20 Urysohn）——紧致性强大到几乎逼出度量。

这一切都在 ch16 详谈, 这里只是预告。

2.6 完备性

Cauchy 列：\(\{x_n\}\) 是 Cauchy \(\iff\) \(\forall \varepsilon > 0, \exists N, \forall m, n \ge N : d(x_m, x_n) < \varepsilon\)。

完备：\((X, d)\) 完备 \(\iff\) 每个 Cauchy 列收敛。

例。\(\mathbb{R}\) 完备（实分析 ch2 关键定理）。\(\mathbb{Q}\) 不 完备（\(x_n = (1 + 1/n)^n \to e \notin \mathbb{Q}\)）。完备性是 \(\mathbb{R}\) 区别于 \(\mathbb{Q}\) 的关键 区分点。

完备性的意义：

  • 不动点定理（Banach）——完备空间上压缩映射有唯一不动点。这是实分析 ch9证常微分方程解存在性的工具。
  • Baire 范畴（ch22）——完备度量空间不是可数多"瘦" 集的并。这条让"大多数函数不可微" 这种反直觉事实成立。
  • 函数空间分析——\(L^p\)、\(C[a, b]\)（sup 度量）都完备, 是泛函分析的根基。

关键：完备性不 是拓扑性质——它依赖度量。同一个拓扑下, 一个度量完备、另一个不完备的事完全可能。

反例：\(\mathbb{R}\) 与开区间 \((-1, 1)\) 是同胚（如 ch1.4 节例：\(f(x) = \tan(\pi x / 2)\) 把 \((-1,1)\) 双向连续打到 \(\mathbb{R}\)）。所以它们拓扑同。但 \(\mathbb{R}\) 配标准度量完备, \((-1, 1)\) 配限制度量不 完备（\(x_n = 1 - 1/n\) 是 Cauchy, 极限 \(1 \notin (-1, 1)\)）。

请你停下来想：完备性依赖 度量的具体数值, 不只是开集族。所以它不 是同胚不变量, 不是拓扑性质。在抽象拓扑空间里"完备" 这个概念没法直接定义——你必须先有度量。

这件事预告了"度量化问题"（ch21 Urysohn 度量化定理）：哪些拓扑空间可以 配上完备度量？这是个深入的问题。

2.7 度量等价：何时两个度量给同一个拓扑

同一个集合 \(X\) 上可以有多 个度量。比如 \(\mathbb{R}^n\) 上至少有 \(d_1, d_2, d_\infty\)：

  • \(d_1(x, y) = \sum_i |x_i - y_i|\)（曼哈顿 / 出租车）
  • \(d_2(x, y) = \sqrt{\sum_i (x_i - y_i)^2}\)（欧氏）
  • \(d_\infty(x, y) = \max_i |x_i - y_i|\)（sup）

三个度量的球 形状不同：\(d_1\) 球是菱形、\(d_2\) 球是圆、\(d_\infty\) 球是方。距离也数值 不同：\(d_1((0, 0), (1, 1)) = 2\)、\(d_2 = \sqrt 2\)、\(d_\infty = 1\)。

但它们给同一个拓扑——同一个开集族。验证用不等式夹：

\[ d_\infty(x, y) \le d_2(x, y) \le d_1(x, y) \le n \cdot d_\infty(x, y). \]

具体, 每个 \(d_2\)-球套住某个 \(d_\infty\)-球, 反之亦然——所以 \(d_2\) 与 \(d_\infty\) 定义同样的"开集"。

定义。\(X\) 上两个度量 \(d_1, d_2\) 拓扑等价 \(\iff\) 它们给同一个开集族 \(\tau_{d_1} = \tau_{d_2}\)。

判别法。\(d_1 \sim d_2 \iff\) 每个 \(d_1\)-球内部含某个 \(d_2\)-球, 反之亦然。形式上: 对每个 \(x, r > 0\)，\(\exists r' > 0\) 使 \(B_{d_2}(x, r') \subseteq B_{d_1}(x, r)\), 反向同。

更强 的等价：强等价（\(\iff\) 存在常数 \(c, C > 0\) 使 \(c \cdot d_1 \le d_2 \le C \cdot d_1\)）。强等价 \(\Rightarrow\) 拓扑等价 \(\Rightarrow\) 但反之不一定。例如 \(d\) 与 \(\min(d, 1)\)（截断度量）拓扑等价但不 强等价（\(d\) 可以无界, 后者 \(\le 1\)）。

请你停下来想这件事的意义：度量空间里有很多冗余信息（具体距离数值）, 真正决定"拓扑结构" 的只是开集族。同一个拓扑可以由多个度量给。所以从度量到拓扑这一步, 我们故意 把"具体距离" 这条多余信息丢掉。

这种"丢信息" 听起来像是损失, 但其实是抽象的力量——只看本质, 不看细节。物理类比：温度计的读数（°C 还是 °F）依赖具体度量, 但"水开" 这件事不依赖度量——它是拓扑（开集结构）的事。

同样的"等价度量" 故事在泛函分析里以"等价范数" 形式出现：有限维上所有范数等价, 无限维上不等价范数给完全不同的拓扑。这件事是泛函分析比有限维线性代数复杂的根源。

2.8 出发——把距离扔掉

这一章把度量空间的所有主要概念按"面向拓扑" 视角整理了。回顾这张地图：

概念度量版开集等价版是拓扑性质？
开集每点配球开集族（公理给）是 / 定义
收敛 \(x_n \to x\)\(d(x_n, x) \to 0\)每个含 \(x\) 开集终含 \(x_n\)是
连续\(\varepsilon\)-\(\delta\)开集逆像开是
紧致序列紧 / Heine-Borel开覆盖有限子覆盖是
完备Cauchy 列收敛(无对应)不是

请你盯住"是拓扑性质？" 那一栏。除"完备" 外, 所有概念都有 纯开集等价版——意味着它们不依赖度量, 可以移植到抽象拓扑空间。"完备" 是唯一例外, 它实质 依赖具体距离, 不能移植。

这就是为什么下一章 ch3 起的拓扑空间 不是无中生有——它是把度量空间公共结构（开集 + 三公理）抽出来, 把度量丢掉 之后剩下的东西。每个你以前在度量空间里熟悉的概念（除完备外）在抽象空间里都有对应。

下一章我们正式给"拓扑空间" 定义, 看许多不来自距离 的拓扑例子, 让"开集独立于距离" 这件事变成具体经验。然后 ch4 给"基" 概念——告诉你不 用列出全部开集, 只列一些"生成元" 就行, 类比向量空间的基。
