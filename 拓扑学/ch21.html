第 21 章 Urysohn 度量化定理

请你回到本书第 1 章那条核心观察——"开集"才是连续的本质，"距离"只是一种构造 开集的实现。这条观察打开了拓扑学，但也立刻引出一个反方向的问题：

哪些拓扑空间能反过来配上一个距离，使距离诱导的拓扑恰好等于原来的拓扑？

这就是度量化问题（metrization problem）。它问的是"抽象拓扑" 与"度量拓扑" 之间的边界——什么样的拓扑空间被允许"反向回到度量世界"。

Urysohn 在 1925 年给了一个充分 条件：第二可数（有可数基）+ T4（正规 + Hausdorff）\(\Rightarrow\) 可度量化。这条定理的证明非常美——它用 Urysohn 引理造出一族可数多连续函数，把空间嵌入 Hilbert 立方体 \([0, 1]^\mathbb{N}\)，再从那里拉回度量。完整的度量化刻画（必要 + 充分）由 Bing、Nagata、Smirnov 在 1950 年代独立完成，复杂得多，本章末仅陈述。

21.1 度量化问题

形式化地——拓扑空间 \((X, \tau)\) 称为可度量化（metrizable），如果存在 \(X\) 上一个度量 \(d\)，使 \(d\) 诱导的拓扑 \(\tau_d\) 恰好等于 \(\tau\)。

这是一条存在性问题——是否 存在度量；不是构造性的——找哪个 度量不重要，能找到一个就行。

典型已知可度量化空间：

  • \(\mathbb{R}^n\)：标准欧氏度量，
  • 任何 Banach 空间：范数度量，
  • 所有有限离散空间：离散度量 \(d(x, y) = 1\) for \(x \neq y\)，
  • 有理数 \(\mathbb{Q}\)、Cantor 集 \(C\)：从 \(\mathbb{R}\) 子空间继承度量。

典型已知不可度量化空间（本章 21.6, 21.7 讲）：

  • "长直线"\(L\)（虽 T4 + 局部欧氏，但不第二可数），
  • Sorgenfrey 直线 \(\mathbb{R}_\ell\)（虽 T5 + 第一可数 + 可分，但乘积 \(\mathbb{R}_\ell^2\) 病理），
  • 余有限拓扑（不 Hausdorff，ch1）。

21.2 必要条件——度量空间长什么样

要找充分条件，先盘点必要 条件——度量空间必然 满足什么拓扑性质？这帮我们对"能度量化" 有最低标准。

命题（度量空间的拓扑性质）。设 \((X, d)\) 是度量空间。则

  1. \(X\) 是 Hausdorff（T2），事实上更强——T4（正规）+ T1。
  2. \(X\) 是第一可数——每点 \(x\) 有可数邻域基（取 \(\{B(x, 1/n)\}_{n \geq 1}\)）。
  3. \(X\) 是完全正规（T5）。

证明（草图）。(1) Hausdorff——\(x \neq y\)，取 \(r = d(x, y)/2\)，开球 \(B(x, r)\) 与 \(B(y, r)\) 不交。T4——给两个不交闭集 \(A, B\)，用 \(d(\cdot, A)\) 与 \(d(\cdot, B)\)（ch20.1）定义

\[ U := \{x : d(x, A) < d(x, B)\}, \qquad V := \{x : d(x, B) < d(x, A)\}, \]

\(U, V\) 不交开集（连续函数严格不等的解集开），\(A \subseteq U\)（\(A\) 上 \(d(x, A) = 0 < d(x, B)\)，因 \(A \cap B = \emptyset\)），\(B \subseteq V\) 同理。\(\checkmark\)

(2) 第一可数——\(\{B(x, 1/n)\}_n\) 是可数邻域基。\(\checkmark\)

(3) T5（完全正规）即每对分离子集能被不交开集分开（"分离" 即互不属于 对方闭包）。度量空间满足，证明类似 T4 但用集距离。

所以一个空间要可度量化，至少得 T4 + 第一可数。但这两条不够——下面 21.7 会看 Sorgenfrey 直线满足这两条但不可度量化。

21.3 Urysohn 度量化定理：陈述

Urysohn 度量化定理。设拓扑空间 \(X\) 是第二可数（有可数拓扑基）+ T4 + Hausdorff。则 \(X\) 可度量化。

请注意条件——第二可数（不是第一可数！）+ T4 + Hausdorff。第二可数是比第一可数强 得多的条件——它要求整个 拓扑只用可数多开集生成，而不只是每点局部可数。

第二可数的常用例子：\(\mathbb{R}^n\)（用开有理球——可数多）；任何可分 度量空间（可分 + 度量 \(\Rightarrow\) 第二可数）；任何离散可数 集。

反例：\(\ell^\infty\)（有界数列空间，标准范数）不 第二可数（虽然度量），所以"第二可数" 比"度量" 严格强一些。Urysohn 定理给出的是"第二可数" 这一子类的度量化。

更精确——"第二可数 + T4 + T1" 等价于"可分 + 可度量化"。这是 Urysohn 定理的精确等价表述。

21.4 证明思路：嵌入 Hilbert 立方体

整个证明的策略——嵌入。如果能把 \(X\) 嵌入一个已知度量空间 \(Y\) 作为子空间，那 \(X\) 就继承 了 \(Y\) 的度量。

选哪个 \(Y\)？候选——Hilbert 立方体

\[ Q := [0, 1]^\mathbb{N} = \{(x_1, x_2, \ldots) : x_i \in [0, 1]\} \]

带积拓扑（ch11 Tychonoff）。这是一个紧致 Hausdorff 空间，并且可度量化——例如

\[ d_Q\big((x_i), (y_i)\big) := \sum_{i=1}^\infty \frac{|x_i - y_i|}{2^i} \]

诱导的拓扑就是积拓扑（标准事实，用 \(\sum 2^{-i} = 1\) 收敛 + 逐项收敛 \(\iff\) 度量收敛验证）。

嵌入方法——构造一族可数 多连续函数 \(\{f_n : X \to [0, 1]\}_{n \geq 1}\)，让

\[ F : X \to Q, \qquad F(x) := (f_1(x), f_2(x), \ldots) \]

是嵌入（同胚到像）。这要求 \(\{f_n\}\) 分离点（\(F\) 单）+ 分离点与闭集（\(F^{-1}\) 在像上连续）。

这一族 \(\{f_n\}\) 怎么造？用 Urysohn 引理——T4 + 第二可数合在一起恰好提供"可数多对不交闭集 + 用 Urysohn 各造一个连续函数"的能力。

21.5 完整证明

设 \(X\) 是第二可数 + T4 + T1。可数基记 \(\mathcal{B} = \{B_1, B_2, \ldots\}\)。我们要造嵌入 \(F : X \to Q\)。

第一步——选可数对 \((B_n, B_m)\)。考虑可数族

\[ \mathcal{P} := \{(B_n, B_m) : n, m \in \mathbb{N}, \overline{B_n} \subseteq B_m\}. \]

\(\mathcal{P}\) 可数（\(\mathbb{N} \times \mathbb{N}\) 的子集）。把 \(\mathcal{P}\) 重新编号成 \(\{(B_{n_k}, B_{m_k})\}_{k = 1}^\infty\)。

第二步——用 Urysohn 引理造 \(\{f_k\}\)。对每对 \((B_{n_k}, B_{m_k}) \in \mathcal{P}\)，\(\overline{B_{n_k}}\) 闭，\(B_{m_k}^c\) 闭，二者不交（因 \(\overline{B_{n_k}} \subseteq B_{m_k}\)）。Urysohn 给连续 \(f_k : X \to [0, 1]\) 使

\[ f_k|_{\overline{B_{n_k}}} = 0, \qquad f_k|_{B_{m_k}^c} = 1. \]

等价——在 \(B_{m_k}\) 之外，\(f_k = 1\)；在 \(\overline{B_{n_k}} \subseteq B_{m_k}\) 上 \(f_k = 0\)。

第三步——验证 \(\{f_k\}\) 分离点与闭集。断言：对每个 \(x \in X\) 与每个含 \(x\) 的开集 \(U\)，存在某个 \(f_k\) 使 \(f_k(x) = 0\) 且 \(f_k|_{U^c} = 1\)。

证明。\(\{x\}\) 闭（T1），\(U^c\) 闭，二者不交。T4 给开集 \(V\) 使 \(\{x\} \subseteq V \subseteq \overline{V} \subseteq U\)（用 ch20.3 等价）。

\(\mathcal{B}\) 是基，存在 \(B_m \in \mathcal{B}\) 使 \(x \in B_m \subseteq U\)。再由 T4 + \(\{x\}\) 闭 + \(B_m\) 开取 \(V'\) 开使 \(x \in V' \subseteq \overline{V'} \subseteq B_m\)。再用 \(\mathcal{B}\) 是基取 \(B_n \in \mathcal{B}\) 使 \(x \in B_n \subseteq V'\)，所以 \(\overline{B_n} \subseteq \overline{V'} \subseteq B_m\)。

所以 \((B_n, B_m) \in \mathcal{P}\)。对应某个 \(k\) 给 \(f_k\) 使 \(f_k|_{\overline{B_n}} = 0\)（包含 \(x\)，所以 \(f_k(x) = 0\)）且 \(f_k|_{B_m^c} = 1\)。\(B_m \subseteq U\) 即 \(U^c \subseteq B_m^c\)，所以 \(f_k|_{U^c} = 1\)。\(\checkmark\)

第四步——定义嵌入 \(F\)。令

\[ F : X \to Q, \qquad F(x) := (f_1(x), f_2(x), \ldots). \]

连续——每个分量 \(f_k\) 连续，所以到积拓扑的映射连续（积拓扑的定义就是"每个分量连续即整体连续"）。\(\checkmark\)

单——给 \(x \neq y\)，取 \(U := X \setminus \{y\}\) 开（T1，\(\{y\}\) 闭）含 \(x\)。第三步给 \(f_k\) 使 \(f_k(x) = 0\) 且 \(f_k(y) = 1\)（\(y \in U^c\)）。所以 \(F(x)_k = 0 \neq 1 = F(y)_k\)，\(F(x) \neq F(y)\)。\(\checkmark\)

\(F^{-1}\) 在像上连续——这是嵌入的最难一步。等价说"\(F\) 是开映射到像"，即 \(U\) 在 \(X\) 上开 \(\Rightarrow F(U)\) 在 \(F(X) \subseteq Q\) 上（相对）开。

取 \(U \subseteq X\) 开，\(x \in U\)。要找 \(F(x)\) 在 \(F(X)\) 中的（相对）开邻域 \(W \cap F(X) \subseteq F(U)\)。

第三步给 \(f_k\) 使 \(f_k(x) = 0\) 且 \(f_k|_{U^c} = 1\)。考虑 \(Q\) 上的开集

\[ W := \{(z_1, z_2, \ldots) \in Q : z_k < 1/2\}. \]

\(W\) 是 \(Q\) 上开（\([0, 1/2)\) 是 \([0, 1]\) 开，乘以其余 \([0, 1]\)，积拓扑）。\(F(x) \in W\)（\(f_k(x) = 0 < 1/2\)）。

断言 \(W \cap F(X) \subseteq F(U)\)。设 \(F(y) \in W\)，即 \(f_k(y) < 1/2\)。若 \(y \in U^c\) 则 \(f_k(y) = 1 \geq 1/2\)，矛盾。所以 \(y \in U\)，\(F(y) \in F(U)\)。\(\checkmark\)

所以 \(F\) 是嵌入。\(F : X \to F(X) \subseteq Q\) 是同胚到像。

第五步——拉回度量。\(Q\) 上的度量 \(d_Q\) 限制到 \(F(X)\) 上是度量。\(F : X \to F(X)\) 是同胚。定义

\[ \boxed{\ d(x, y) := d_Q(F(x), F(y)) = \sum_{k=1}^\infty \frac{|f_k(x) - f_k(y)|}{2^k}.\ } \]

这是 \(X\) 上度量（直接验证三公理 + 同胚保 \(d(x, y) = 0 \iff x = y\)），并且诱导拓扑等于原拓扑（因 \(F\) 是同胚）。所以 \(X\) 可度量化。\(\blacksquare\)

请你停下来欣赏证明的整体形状：

  1. 第二可数——可数基 \(\mathcal{B}\) 给可数族 \(\mathcal{P}\) 给可数族 \(\{f_k\}\)。第二可数是"把"任意多函数" 减少到"可数多函数"" 的关键。
  2. T4 + Hausdorff——Urysohn 引理给"每对不交闭集对应一个连续函数"。T4 是"制造连续函数" 的引擎。
  3. 合在一起——可数多连续函数分离点 + 分离点和闭集，给嵌入到 \(Q\)。\(Q\) 是已知度量空间，所以 \(X\) 继承度量。

这是一种"把抽象空间嵌入一个具体的"模型"空间" 的策略，在拓扑学、表示论里反复出现——例如可数 Hausdorff 群嵌入 \(U(\ell^2)\)、紧致 Hausdorff 空间嵌入 Tychonoff 立方体 \([0, 1]^I\) 等。

21.6 反例一：长直线不可度量化

"长直线" \(L\) 是把不可数 多个 \([0, 1)\) 拼起来——具体取良序集 \(\omega_1\)（最小不可数序数），令

\[ L := \omega_1 \times [0, 1) \quad \text{字典序拓扑}. \]

"\((\alpha, t) < (\beta, s) \iff \alpha < \beta\) 或 (\(\alpha = \beta\) 且 \(t < s\))"。这空间像是不可数多条 \([0, 1)\) 首尾相接。

它的奇怪性质：

  • 局部欧氏——每点有邻域同胚于 \(\mathbb{R}\) 的开区间。
  • Hausdorff、T4。
  • 第一可数。
  • 道路连通。
  • 不第二可数（任何拓扑基至少 \(\aleph_1\) 多元素）。
  • 不可分（任意可数子集都"太短"——它在某个 \([0, \alpha) \times [0, 1)\) 内，\(\alpha < \omega_1\)）。

断言：长直线不可度量化。

证明思路。可度量化 \(\Rightarrow\) 任何连续函数 \(L \to [0, 1]\) 在某点取严格中间值（中间值定理需道路连通——这步不是问题）。但更细地——

关键事实：长直线上任何连续函数 \(g : L \to \mathbb{R}\) 都最终常值——存在 \(\alpha_0 < \omega_1\) 使 \(g\) 在 \([\alpha_0, \omega_1) \times [0, 1)\) 上常值。

原因（草图）：用 \(\omega_1\) 是正则 序数 + \(g\) 连续——若 \(g\) 在序数序列 \(\alpha_n \to \omega_1\) 上变化，但 \(\sup\{\alpha_n\}\) 在 \(\omega_1\) 内（任何 \(\omega_1\) 内的可数 \(\sup\) 仍在 \(\omega_1\) 内），导致 \(g\) 在那里有极限，限制了变化。

所以 \(L\) 上连续函数不分离 \(L\) 的远端两点（它们都映到同一极限值）。如果 \(L\) 可度量化，距离函数 \(d(\cdot, p)\) 对每个 \(p\) 是连续函数 \(L \to \mathbb{R}\)，所以也最终常值——但距离 函数最终常值意味着远处所有点到 \(p\) 距离相同，与"度量分离" 矛盾。所以 \(L\) 不可度量化。\(\blacksquare\)

请欣赏这例子的本质：长直线是 T4 + Hausdorff，但不 第二可数——所以 Urysohn 度量化定理不适用，事实上结论不成立。这告诉我们 Urysohn 定理里的"第二可数" 不能去掉。

21.7 反例二：Sorgenfrey 直线不可度量化

Sorgenfrey 直线 \(\mathbb{R}_\ell\) 是 \(\mathbb{R}\) 配下界半开区间拓扑——基为 \(\{[a, b) : a < b\}\)。

性质：

  • Hausdorff（更强 T5/完全正规），
  • 第一可数（\(\{[x, x + 1/n)\}_n\) 是 \(x\) 的可数邻域基），
  • 可分（\(\mathbb{Q}\) 稠密），
  • 不 第二可数（任何基至少 \(\mathfrak{c} = 2^{\aleph_0}\) 多——每个 \([x, x + 1)\) 必须在基中含某个含 \(x\) 的元素，而这迫使不同的 \(x\) 用不同的基元），
  • Lindelöf（每开覆盖有可数子覆盖——这条非平凡）。

注意 \(\mathbb{R}_\ell\) 是第一可数 + 可分 但不第二可数。在度量 空间里，可分 \(\Leftrightarrow\) 第二可数 \(\Leftrightarrow\) Lindelöf——这三条等价。所以一旦看到"可分但不第二可数"，可立刻断言"不可度量化"。\(\mathbb{R}_\ell\) 正是。

但这个反例真正的力量在乘积——Sorgenfrey 平面 \(\mathbb{R}_\ell^2\)：

  • \(\mathbb{R}_\ell\) 是正规，
  • 但 \(\mathbb{R}_\ell^2\) 是正规失败！

具体——\(\mathbb{R}_\ell^2\) 上"反对角线" \(\Delta := \{(x, -x) : x \in \mathbb{R}\}\) 是闭子集（用 \(\mathbb{R}_\ell^2\) 的拓扑验证），且作为子空间是离散且不可数 的。这件事用 cardinality argument 推出 \(\mathbb{R}_\ell^2\) 不 正规——大致："不可数闭离散子集 + Lindelöf 失败 + 集合论计数 \(\Rightarrow\) 找到不交闭集分不开"。

这告诉我们：正规性不对乘积封闭。这正是 Bing-Nagata-Smirnov 度量化里强化"正规" 为"完全正规" 的动机。

21.8 完整刻画：Bing–Nagata–Smirnov 度量化定理

把 Urysohn 度量化定理推到必要 + 充分，需要把"第二可数" 替换成更精细的"\(\sigma\)-局部有限基"。结果：

Bing–Nagata–Smirnov 度量化定理（1950 年代）。拓扑空间 \(X\) 可度量化 \(\iff\) \(X\) 是 T3（正则 Hausdorff）且有\(\sigma\)-局部有限基。

"\(\sigma\)-局部有限基"指——拓扑基 \(\mathcal{B} = \bigcup_{n=1}^\infty \mathcal{B}_n\)，其中每个 \(\mathcal{B}_n\) 是局部有限族（每点有邻域只交 \(\mathcal{B}_n\) 的有限多元素）。

这条件比第二可数弱得多——任何度量空间都满足（不必可分，例如不可数离散度量空间），所以是必要。它充分 也用类似 Urysohn 的嵌入策略，但嵌入到的是不 可分的"分层" 度量乘积。

本书不证明 Bing–Nagata–Smirnov，留给后续课。Urysohn 版（第二可数 + T4 \(\Rightarrow\) 度量化）已经覆盖了大量实际遇到的情形——所有可分的 Banach 空间、所有第二可数流形、所有可数 CW 复形等。

21.9 临别话

度量化定理是抽象拓扑 通向分析 的桥梁——它告诉你哪些抽象拓扑空间能退化 回度量空间，从而能用所有度量分析的工具（Cauchy 序列、完备性、Banach 不动点、Lebesgue 积分）。

Urysohn 版的两条假设——第二可数 + T4——分别对应"大小可控"和"分离公理足够强"。前者保证嵌入到的是可数 维度的乘积空间（\([0, 1]^\mathbb{N}\)），后者保证有"足够多" 连续函数实施嵌入。两条合在一起恰好够。

反例（长直线、Sorgenfrey）告诉你这两条都不能去。这种"正反两面" 的研究——给出充分条件 + 给出违反任一条件的反例——是拓扑学典型的定理 + 反例 风格。每个看似无聊的反例背后都对应一种"这里出了什么意外" 的可视化场景。

下一章ch22 我们换一个角度——既然度量空间 这么重要，"完备度量空间" 这一子类 还有更深的拓扑性质：Baire 范畴定理。这条定理把"完备性"翻译成"开稠集的可数交仍稠"，给"大多数" 这种存在性论证提供统一工具，并通向泛函分析三大原理。
