第 3 章 拓扑空间

ch1 已经预告了主角：拓扑空间 = 集合 + 满足三公理的开集族。ch2 在度量空间里验证了这条公理化是对 的——度量诱导的开集自动满足三公理, 而所有"面向连续 / 收敛" 的概念都能 用开集表达。

这一章正式定义拓扑空间 并立即看大量例子。重点不在定义本身——它就一行——重点在例子。具体看到许多不 来自距离的拓扑、同一集合上多个不同拓扑, 才能让"拓扑独立于距离" 从口号变成肌肉记忆。

这一章写完你应该能反射性地回答：

  • 给一个奇怪空间, 它是不是拓扑空间？（验证三公理）
  • 同一集合上有多种拓扑选择吗？（一定有, 至少离散和平凡）
  • 哪些拓扑不 来自度量？（Sierpiński、余有限、Sorgenfrey 都不）
  • 拓扑之间能比较 吗？（按"开集多少" 排——粗 / 细）

3.1 拓扑空间的形式定义

给一个集合 \(X\)。\(X\) 上的拓扑 \(\tau\) 是 \(X\) 的子集族（\(\tau \subseteq \mathcal{P}(X)\)）, 满足：

  • (O1) \(\emptyset \in \tau\), \(X \in \tau\)。
  • (O2) 任意并：\(\{U_\alpha\}_{\alpha \in I} \subseteq \tau \Rightarrow \bigcup_{\alpha \in I} U_\alpha \in \tau\)。（指标集 \(I\) 可以是任意集合, 包括不可数。）
  • (O3) 有限 交：\(U_1, \ldots, U_n \in \tau \Rightarrow \bigcap_{i=1}^n U_i \in \tau\)。

对 \(\tau\) 中的元素 \(U\)，我们称 \(U\) 是 开集。这对 \((X, \tau)\) 称为拓扑空间。

请你盯住这个定义看一会儿——它极简。没有距离, 没有 \(\varepsilon\), 没有数字, 只有集合 + 子集族 + 三条组合公理。这是数学定义最简的形式之一。

"X 上的拓扑" = X 上的"开集" 的选择——而"开集" 这个词在抽象层级上就是一族满足三公理的子集, 不带任何先验意义。

(O3) 为什么只有限 交？这是上一章的关键观察的形式化。任意交不行——\(\bigcap_n (-1/n, 1/n) = \{0\}\) 在 \(\mathbb{R}\) 上不开。如果允许任意交, 那从开集出发能"切" 出任意单点, 拓扑会塌缩为离散——失去意义。所以三公理里 (O2) 用任意, (O3) 限有限——这是非对称 的, 故意的。

注意：我们没说拓扑是唯一 的——同一个 \(X\) 可以有多个不同 \(\tau\)。下面 3.3 节起就看这种"同一集合多种拓扑" 的例子。

3.2 第一个例子：度量诱导拓扑

给度量空间 \((X, d)\)，度量诱导拓扑：

\[ \tau_d = \{U \subseteq X : \forall x \in U, \exists r > 0, B(x, r) \subseteq U\}. \]

上一章 2.2 节已验证这是 \(X\) 上的拓扑（满足三公理）。

所以每个 度量空间自动是拓扑空间。\(\mathbb{R}, \mathbb{R}^n, [0, 1]\)、欧氏球面、流形、函数空间, 全是拓扑空间。"拓扑空间" 这个范畴包含 度量空间作为子集。

但反过来不对——有些拓扑空间不 来自任何度量。这是下面 3.3-3.7 节几个例子要展示的事。"不来自度量的拓扑" 是拓扑学这个学科存在的理由。如果所有拓扑都来自度量, 我们就只需要度量空间, 不需要"拓扑空间" 这个更一般的概念。

"可度量化" 拓扑：一个拓扑空间 \((X, \tau)\) 可度量化 \(\iff\) \(\exists\) 度量 \(d\) 使 \(\tau = \tau_d\)。"哪些拓扑可度量化" 是拓扑学的核心问题之一, ch21 Urysohn 度量化定理给一个最重要的回答。

3.3 极端例子：离散与平凡拓扑

给任意 集合 \(X\)。两个"边界" 拓扑总是合法的：

离散拓扑 \(\tau_d = \mathcal{P}(X)\) — 所有 子集都开。

验证：(O1) \(\emptyset, X \in \mathcal{P}(X)\)。(O2)(O3) 任意子集族的并 / 交还是子集——都在 \(\mathcal{P}(X)\) 里。\(\checkmark\)

"离散" 这个名字的来由：每个单点集 \(\{x\}\) 都开。这意味着每个点"孤立"——它周围有一个只含自己的开邻域。直观地, 空间是"分裂" 的——点之间没有"连接"。

离散拓扑来自 度量——离散度量（ch2.1 例 3）正给离散拓扑：每个 \(\{x\} = B(x, 1/2)\)。

平凡拓扑（也叫 不可分 拓扑）\(\tau_0 = \{\emptyset, X\}\) — 只有 \(\emptyset\) 和 \(X\) 开。

验证：(O1) 显然。(O2)(O3) 任意并 / 交两个 \(\{\emptyset, X\}\) 的元素仍在 \(\{\emptyset, X\}\) 里。\(\checkmark\)

"平凡" 的几何含义：空间"融合" 在一起, 你分 不开任何子部分。每两个点都是"同一邻域里的同伴"。极端的"糊在一起" 状态。

平凡拓扑（当 \(|X| \ge 2\) 时）不 来自度量。因为度量空间每个不同的两点 \(x \neq y\) 距离 \(d(x, y) > 0\), 球 \(B(x, d(x,y)/3)\) 与 \(B(y, d(x,y)/3)\) 不相交, 它们俩是不同开集（Hausdorff 性质）。但平凡拓扑里只有 \(\emptyset\) 和 \(X\) 两个开集——找不到分隔 \(x, y\) 的不相交开集。所以平凡拓扑非度量化。

极端比较。同一个 \(X\) 上离散和平凡是两个极端：

  • 离散有 \(2^{|X|}\) 个开集（最多）。
  • 平凡只有 2 个开集（最少）。

任何 \(X\) 上的"合法" 拓扑都被夹在两者之间——\(\tau_0 \subseteq \tau \subseteq \tau_d\)（包含关系按集合）。这是 3.8 节"拓扑比较" 的预告。

这两个极端是反例机器。每当你怀疑某个性质 P 是不是拓扑不变量、是不是必然成立, 第一反应——测试 \(\tau_d\) 和 \(\tau_0\)。它们极端到能滤掉 你考虑的大部分性质。

3.4 余有限拓扑 + 余可数拓扑

给无穷集 \(X\)（典型是 \(\mathbb{R}\) 或 \(\mathbb{N}\)）。余有限拓扑：

\[ \tau_{\text{cof}} = \{\emptyset\} \cup \{U \subseteq X : X \setminus U \text{ 有限}\}. \]

"开集 = 空集 + 余集有限"。验证三公理：

(O1) \(\emptyset\) 在定义里, \(X \setminus X = \emptyset\) 有限, \(X\) 也开。\(\checkmark\)

(O2) 任意并：\(X \setminus \bigcup_\alpha U_\alpha = \bigcap_\alpha (X \setminus U_\alpha)\)。每个 \(X \setminus U_\alpha\) 有限, 所以它们的任意 交也有限（任意交一族有限集还是有限集——它是其中任何一个的子集）。所以 \(\bigcup U_\alpha\) 余集有限, 开。\(\checkmark\)

(O3) 有限 交：\(X \setminus \bigcap_{i=1}^n U_i = \bigcup_{i=1}^n (X \setminus U_i)\)。有限多 个有限集的并仍有限。\(\checkmark\)

所以 \(\tau_{\text{cof}}\) 是 \(X\) 上的拓扑。

这个拓扑非常粗。\(\mathbb{R}\) 上的开区间 \((0, 1)\)不 是余有限开集（其余集 \((-\infty, 0] \cup [1, \infty)\) 不有限）。所以欧氏开集不是 余有限开集——这两个拓扑不同。

事实上, 余有限拓扑里开集很少——只有"除去有限多个点的 \(X\)"。这种"稀少" 的开集结构产生了一些奇怪现象：

  • 余有限拓扑下任意两个非空开集必相交（它们的余集是有限集, 它们的并集 = 余有限集的余集 = 有限集; 如果两者不交则 \(X\) 是有限并集 = 有限——矛盾, 假设 \(X\) 无穷）。
  • 所以余有限拓扑不 是 Hausdorff（不能用不相交开集分离两点）。
  • 所以它不 来自度量（度量必 Hausdorff）——又一个非度量化拓扑。

余有限拓扑虽然"怪", 在代数几何 里至关重要——\(\mathbb{A}^1_k\) 上的 Zariski 拓扑就是它。代数几何的"开集" 是多项式不为零的集合, 余集是多项式零点（有限多个）。整个代数几何在拓扑上活在余有限（更高维：Zariski）拓扑里。

余可数拓扑: 同样思路, 把"有限" 替换成"可数"：

\[ \tau_{\text{cocount}} = \{\emptyset\} \cup \{U \subseteq X : X \setminus U \text{ 可数}\}. \]

验证类似（可数并仍可数, 有限交可数仍可数）。在不可数集（如 \(\mathbb{R}\)）上, 这是另一个 不来自度量的拓扑——它的"奇怪" 性质（任意两个非空开集相交）和余有限类似。它在分析里偶尔用作反例。

3.5 Sierpiński 空间 + 计算机科学

取 \(X = \{0, 1\}\)（最小的有意思的两点空间）。考虑拓扑

\[ \tau_S = \{\emptyset, \{1\}, \{0, 1\}\}. \]

验证：(O1) \(\emptyset, X \in \tau_S\)。(O2)(O3) 三个集合的所有可能并 / 交全在 \(\tau_S\) 里——容易枚举。\(\checkmark\)

这就是Sierpiński 空间。两个点 0 与 1 在拓扑上不平等：\(\{1\}\) 开但 \(\{0\}\) 不开。

这个不对称排除了度量化。原因（ch1.3 已说过, 这里展开）：度量空间里"\(\{x\}\) 开 \(\iff x\) 是孤立点 \(\iff\) 周围有一个邻域只含自己"——这是个对称概念, 任何度量自动让所有 满足条件的点都满足。如果在 Sierpiński 里 \(\{1\}\) 开, 度量结构会要求"\(\{0\}\) 同等程度也开"或"都不开", 不允许"一个开一个不开" 的不对称。所以 Sierpiński 拓扑不可 来自度量。

计算机科学含义。Sierpiński 在域论（domain theory，编程语言语义的数学理论）里反复出现。把 \(\{0, 1\}\) 看成"是否终止" 的状态：1 = "终止"，0 = "未知 / 还在算 / 永不终止"。

从外部看一个程序——你只能观察到 它已经终止（开集 \(\{1\}\)）, 但不能在有限时间确认 它不会终止（\(\{0\}\) 不开——你需要等无穷长时间才能确认）。这就是停机问题不可判定 在拓扑层面的影像——\(\{0\}\) 的"不开" 编码了"不可观测确认" 的算法事实。

请你停下来欣赏这件事：一个 2 元素小空间, 从纯数学公理出发是合法拓扑, 拒绝度量化, 在计算理论 里编码了停机问题。这种"奇怪空间, 实际意义" 的对应关系是拓扑学的魔法时刻 之一。

更一般的：所有"半决定问题"（如递归可枚举集）都有这种"开集 = 可证明" 的拓扑结构, 这条思路把计算复杂性 翻译成拓扑学。Scott 域、可计算实数、概率程序的语义——背后都是 Sierpiński 风格的不对称拓扑。

3.6 序拓扑（在偏序集上）

给偏序集 \((X, \le)\)（自反 + 反对称 + 传递, 但不一定全序）。可以从序结构 自然定义一个拓扑。

当 \(X\) 是全序 集（任两元素可比）, 标准做法用开区间：

\[ (a, b) := \{x \in X : a < x < b\}, \quad (-\infty, b) := \{x : x < b\}, \quad (a, \infty) := \{x : x > a\}. \]

把这些"开区间"（含三种类型）作为子基（ch4 概念, 这里先用）, 它们的有限交构成基, 有限交的任意并构成开集——这就给出序拓扑。

例 1。\(X = \mathbb{R}\) 配标准序 \(\le\)。序拓扑 = 标准欧氏拓扑（开区间作为基）。序拓扑恢复欧氏拓扑——这是个一致性验证。

例 2。\(X = \mathbb{N}\) 配标准序。序拓扑：每个单点 \(\{n\}\) 开（取 \((n-1, n+1) = \{n\}\)）——所以这是离散拓扑。同样, 任何全序的有限 集或离散 排列的全序集（如 \(\mathbb{Z}\)）的序拓扑是离散的。

例 3。\(X = [0, 1]\) 配标准序。序拓扑：开区间 \((a, b)\), 加上"含端点" 的半开 邻域 \([0, b)\) 和 \((a, 1]\)（这两个分别是 \((-\infty, b)\) 与 \((a, \infty)\) 限制到 \([0, 1]\)）。验证它们也是欧氏开集（\([0, b) = (- 1, b) \cap [0, 1]\)）。又是欧氏拓扑。

例 4。序数 \(\omega_1\)（最小不可数序数）配标准序——给出"长直线" 类型的奇怪空间, 在反例库里有用。我们暂不展开。

序拓扑的意义：让纯 序信息（没有距离、没有几何）自动产生 拓扑结构。这件事在序理论、集合论拓扑、泛代数 里至关重要。

3.7 Sorgenfrey 直线：可分但非度量化

这是个非常重要 的反例空间, 拓扑学家叫它下极限拓扑直线 或 Sorgenfrey 直线。

取 \(X = \mathbb{R}\)。考虑用半开区间 \([a, b)\) 作为基：

\[ \mathcal{B} = \{[a, b) : a < b\}. \]

"开集" = "这种半开区间的任意并"。这给一个拓扑 \(\tau_S\), Sorgenfrey 拓扑（也叫下极限拓扑, lower limit topology）。验证 \(\mathcal{B}\) 是基的两条充要条件 (ch4 会展开)：

  • \(\mathbb{R} = \bigcup_{n \in \mathbb{Z}} [n, n+1)\), 覆盖。
  • 两个 \([a, b) \cap [c, d) = [\max(a, c), \min(b, d))\), 仍是 \([\cdot, \cdot)\) 形式。

所以 \(\mathcal{B}\) 合法生成拓扑。\(\checkmark\)

Sorgenfrey 拓扑比标准欧氏拓扑严格更细。

原因 1：每个标准开区间 \((a, b) = \bigcup_{n \ge 1} [a + 1/n, b)\), 是 Sorgenfrey 开集的并, 所以 \((a, b) \in \tau_S\)。意味着 \(\tau_{\text{euc}} \subseteq \tau_S\)（Sorgenfrey 比欧氏多）。

原因 2：\([0, 1)\) 是 Sorgenfrey 开（按定义）但不是 欧氏开（左端点 0 没有欧氏邻域全在内部）。所以 \(\tau_{\text{euc}} \subsetneq \tau_S\)（严格更细）。

奇怪性质 1：可分。\(\mathbb{Q}\) 是可数稠密子集——每个 Sorgenfrey 开集 \([a, b)\) 包含某有理数。所以 Sorgenfrey 直线可分。

奇怪性质 2：非可分度量化（具体: 非第二可数）。欧氏直线"可分" + "第二可数"（有可数基, ch15）——所以是可分度量化 的典型。Sorgenfrey 直线可分 但不 第二可数（每个含 \(a\) 的开集必须含某 \([a, a + \varepsilon)\), 不同 \(a\) 给本质不同基元素, 不可数多个 \(a\) → 基不可数）。

但有定理（Urysohn）：第二可数 + 正则 \(\Rightarrow\) 可度量化。Sorgenfrey 是正则但不 第二可数, 所以这条定理不直接给度量化。事实上 Sorgenfrey 直线不 度量化。证明用"Sorgenfrey 平方" \(\mathbb{R}_\ell \times \mathbb{R}_\ell\) 不正规——但度量空间必正规, 所以两端不能调和。详细论证留 ch20 后做。

Sorgenfrey 直线的意义：

  • 它是"可分但非度量化" 的标志性反例——告诉你"可分" 不蕴含"度量化"。
  • 它"几乎" 像欧氏直线但又关键性质不同——是测试拓扑性质的"分辨率" 工具。
  • 它积 起来产生更奇怪的反例（Sorgenfrey 平方非正规——反 Tychonoff 直觉）。

请你停下来欣赏这件事：把 \(\mathbb{R}\) 上的"开集" 定义微微改一下（从 \((a, b)\) 到 \([a, b)\)）, 你就得到一个本质不同、不来自度量的拓扑。这种"同一集合可以承载本质不同的拓扑" 是拓扑学的核心现象——同一个 \(\mathbb{R}\), 装载不同的开集结构, 给出不同的"几何"。

3.8 比较拓扑：粗 vs 细

同一集合 \(X\) 上的两个拓扑 \(\tau_1, \tau_2\) 怎么比较？答案：按开集多少。

定义。\(\tau_1\) 比 \(\tau_2\) 粗（coarser，也叫 weaker / smaller）\(\iff\) \(\tau_1 \subseteq \tau_2\)（\(\tau_1\) 的开集都是 \(\tau_2\) 的开集）。等价: \(\tau_2\) 比 \(\tau_1\) 细（finer / stronger / larger）。

所以"粗" = 开集少, "细" = 开集多。这条命名容易混淆——记住：

\[ \boxed{\ \tau_1 \subseteq \tau_2 \iff \tau_1 \text{ 比 } \tau_2 \text{ 粗 } \iff \tau_2 \text{ 比 } \tau_1 \text{ 细}.\ } \]

记忆诀窍："开集多" 像"筛子细", 能"筛" 出更精细的结构——所以"开集多" = "细"。

例 1。任何 \(\tau\) 满足 \(\tau_0 \subseteq \tau \subseteq \tau_d\)。平凡是最粗, 离散是最细。

例 2。\(\mathbb{R}\) 上：\(\tau_{\text{cof}}\)（余有限）比 \(\tau_{\text{euc}}\)（欧氏）粗。验证：每个余有限开集 \(U\)（\(U \neq \emptyset\)）的余集 \(\mathbb{R} \setminus U\) 有限, 所以 \(U\) 是 \(\mathbb{R}\) 减去有限多个点——是欧氏开集（开集差去闭集 = 开集）。所以 \(\tau_{\text{cof}} \subseteq \tau_{\text{euc}}\)。

例 3。\(\mathbb{R}\) 上：\(\tau_{\text{euc}}\)（欧氏）比 \(\tau_S\)（Sorgenfrey）粗——已在 3.7 验证。

所以 \(\mathbb{R}\) 上至少有这条从粗到细 的链：

\[ \tau_0 \subsetneq \tau_{\text{cof}} \subsetneq \tau_{\text{euc}} \subsetneq \tau_S \subsetneq \tau_d. \]

这就是同一个 \(\mathbb{R}\) 上的拓扑格 的一条线索——实际拓扑空间多得多, 这只是其中很小一部分。

拓扑格：\(X\) 上所有拓扑构成一个偏序集（按 \(\subseteq\)）。任意一族拓扑 \(\{\tau_\alpha\}\) 的交 \(\bigcap \tau_\alpha\) 仍是拓扑（验证三公理对每个 \(\tau_\alpha\) 都成立, 所以对它们的交也成立）。所以这个偏序集有任意 inf（交）, 它就是个完备格——任意拓扑族也有 sup（"包含每个 \(\tau_\alpha\) 的最粗拓扑", 由 \(\bigcup \tau_\alpha\) 生成的拓扑）。

这个格结构的意义：当你说"X 上的拓扑" 时, 不是指一个 拓扑——而是从一个大量选择 中挑一个。每个选择给一个不同的"拓扑空间", 不同的连续函数集, 不同的紧致集, 不同的同伦类型。

"选哪个拓扑" 取决于你想研究什么。研究分析 → 度量拓扑; 研究代数几何 → Zariski（粗如余有限）; 研究域论 → Sierpiński 风格（粗且不对称）; 研究反例 → 离散 / 平凡 / Sorgenfrey 等。拓扑是工具, 不是给定。

3.9 一个反射动作

读完这一章你应该养成这个反射：看到"集合 X 上的拓扑" 这种字眼, 立即问"哪个拓扑？"

"\(\mathbb{R}\)" 不自动 携带一个唯一拓扑——它是个集合, 你选择 在它上面装载什么拓扑。99% 的场合默认欧氏, 但其余 1% 场合（Zariski、Sorgenfrey、p-adic、Stone）拓扑选择决定了一切。"R" 这个符号本身只指集合, 拓扑是额外结构。

这条反射在数论、代数几何、泛函分析 里都有用。比如同一个函数空间 \(C[0, 1]\) 配 \(L^p\) 范数和 sup 范数给本质不同的拓扑——一个完备一个不完备, 一个紧子集多一个少。"哪个范数" = "哪个拓扑"。

下一章我们处理"怎么具体描述一个拓扑" 的工程问题。\(\mathbb{R}\) 上欧氏拓扑有不可数无穷 多个开集——你不可能"列出全部"。但好消息是你不用——只需列一些"生成元"（开区间）, 通过任意并 + 有限交"生成" 全部开集。这种"生成元" 叫基, 类比线性代数里向量空间的基。下一章定义并展示 \(\mathbb{R}^n\)、积空间、子空间等的标准基。
