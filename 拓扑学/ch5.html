第 5 章 闭集、内部、闭包、边界

把"开集" 当成拓扑学的官方词汇是 ch3 的事。可你哪怕只读过几篇分析的书都能注意到，日常几何讨论里"闭" "边界" "内部" 出现的次数不比 "开" 少。"\(\bar A\) 表示 \(A\) 的闭包"、"\(\partial A\) 是 \(A\) 的边界"、"\(\mathrm{int}(A)\) 是 \(A\) 的内部"——这些写法在测度论、几何拓扑、PDE 里都是背景常识，不写出来都默认。

问题是：如果"开集" 已经被公理选成基本对象，闭集 / 内部 / 闭包 / 边界 是不是多余 的？还是说它们其实就是"开集" 的不同重组方式？答案——后者。这一章把这些日常词汇从"开集" 出发严格定义，并且给一些朴素直觉里容易错 的事打几针预防针：

  • "闭包 = 集合 \(\cup\) 边界" 在每个空间都对吗？
  • "边界" 会不会包含原集合的内部？
  • \(\bar{\mathbb{Q}}\) 在 \(\mathbb{R}\) 里是什么？\(\mathrm{int}(\mathbb{Q})\) 又是什么？
  • "稠密" 这件事在不同拓扑下能多么离奇？

每一个回答都会暴露你的几何直觉里某条偷偷依赖度量 的假设。读完这一章你的脑子里"闭包" 不再是"加上边界点"——而是"含 A 的最小闭集"，是从公理里长出来的代数操作。

5.1 闭集 = 开集补

设 \((X, \tau)\) 是拓扑空间。闭集 的定义朴素到没什么可说：

\[ \boxed{\ F \subseteq X \text{ 闭} \iff X \setminus F \in \tau \text{（即补集开）}.\ } \]

就这一行。"闭" 是"开" 在补集运算下的影子；它不是 一个新公理，而是开集的派生 概念。

这条定义直接翻译 开集公理 (O1)-(O3)。把 \(U_\alpha\) 换成 \(F_\alpha = X \setminus U_\alpha\)，并 \(\leftrightarrow\) 交（De Morgan：\(X \setminus \bigcup U_\alpha = \bigcap (X \setminus U_\alpha)\)）：

  • (C1) \(\emptyset\) 闭，\(X\) 闭。（\(X \setminus \emptyset = X\) 开，\(X \setminus X = \emptyset\) 开。）
  • (C2) 任意多闭集的交 仍闭。（\(\bigcap F_\alpha = X \setminus \bigcup (X \setminus F_\alpha)\)，并里面每个开 \(\Rightarrow\) 并开 \(\Rightarrow\) 补闭。）
  • (C3) 有限多闭集的并 仍闭。（与 (O3) 对偶。）

请注意 (C2) 和 (C3) 的不对称 与 (O2)-(O3) 完全镜像：闭集对任意交 闭、对有限并 闭。这种"任意 vs 有限" 的不对称是拓扑结构的根性质，本章后面所有反例都和它脱不了干系。

对偶定义法。有些教科书反过来：先写 (C1)-(C3) 公理，把"闭集族" 当原始概念，然后定义"开 = 闭的补"。两种方式数学上完全等价。我们选"开" 优先，因为连续映射 的定义（\(f^{-1}\) 把开集拉回开集）天然偏向开。

具体例子。欧氏 \(\mathbb{R}\) 上：\([a, b]\) 闭（补集 \((-\infty, a) \cup (b, \infty)\) 开）；\((a, b)\) 不 闭；\([a, b)\) 既不开也不闭（补集 \((-\infty, a) \cup [b, \infty)\) 含 \([b, \infty)\) 不开）。请你停下盯一会儿——"非开即闭" 这种二元思维是初学最容易犯的错。集合可以既不开也不闭（\([a, b)\) ）、可以既开又闭（\(\emptyset, X\) 在每个空间都既开又闭；离散拓扑里每个 子集都既开又闭）。开 / 闭是两条独立 的属性，可同时成立、可同时失败、可只有一条成立。

"既开又闭"叫 clopen ——这词在英文里是 closed-open 的合体。一个空间里 clopen 集合多 \(\Leftrightarrow\) 它"断成多块"，这正是 ch18 连通性 的几何刻画。在 \(\mathbb{R}\) 上 clopen 集只有 \(\emptyset\) 和 \(\mathbb{R}\)（连通），在 \(\mathbb{Q}\) 上 clopen 集多得是（不连通），在 Cantor 集上 clopen 是基（完全不连通）。

5.2 闭包 \(\bar A\)：含 A 的最小闭集

给一个集合 \(A \subseteq X\)，闭包 想抓住的几何画面是"把 A 加上它的所有极限点"。但这个朴素描述在抽象拓扑里要小心——"极限点" 本身得用拓扑定义。我们用一种不依赖任何辅助概念 的方式把闭包定义下来：

\[ \boxed{\ \bar A = \bigcap \{F : A \subseteq F, \ F \text{ 闭}\}.\ } \]

"所有含 A 的闭集的交"。这个交非空——\(X\) 本身就是一个含 \(A\) 的闭集（(C1)）。这个交是闭集——任意多闭集的交是闭集（(C2)）。所以 \(\bar A\) 是含 A 的最小闭集：

  • (a) \(A \subseteq \bar A\)；
  • (b) \(\bar A\) 闭；
  • (c) 若 \(F\) 闭且 \(A \subseteq F\)，则 \(\bar A \subseteq F\)。

这三条恰是"极小" 的代数刻画。把 \(\bar A\) 当作一个算子 \(\bar{\,\cdot\,} : \mathcal{P}(X) \to \mathcal{P}(X)\)，它满足 Kuratowski 闭包公理：

  • (K1) \(\overline{\emptyset} = \emptyset\)；
  • (K2) \(A \subseteq \bar A\)；
  • (K3) \(\overline{\bar A} = \bar A\)（幂等）；
  • (K4) \(\overline{A \cup B} = \bar A \cup \bar B\)。

(K1)：\(\emptyset\) 自身闭，所以含 \(\emptyset\) 的最小闭集是 \(\emptyset\)。(K2) 由定义。(K3)：\(\bar A\) 已经闭，含 \(\bar A\) 的最小闭集就是它自己。(K4) 一个方向 \(\supseteq\) 由 \(A \subseteq A \cup B \Rightarrow \bar A \subseteq \overline{A \cup B}\) 同理 \(\bar B \subseteq \overline{A \cup B}\)；另一方向 \(\bar A \cup \bar B\) 是有限个闭集的并，闭，含 \(A \cup B\)，故是含 \(A \cup B\) 的某闭集，最小那个 \(\overline{A \cup B}\) 含进它。

事实上 (K1)-(K4) 是"闭包" 概念的独立公理化：给一个集合 \(X\) 和一个算子 \(c : \mathcal{P}(X) \to \mathcal{P}(X)\) 满足 (K1)-(K4)，定义"闭集 = 算子的不动点"，再定义"开 = 闭的补"，得到的开集族恰好 满足 (O1)-(O3)。所以"用闭包公理定义拓扑" 与"用开集公理定义拓扑" 严格等价——这是 Kuratowski 1922 年的一个漂亮观察。

三种刻画。除了"极小闭集"，闭包还有两种更可计算 的刻画。先列出来：

(I) \(x \in \bar A \iff \) 每个 含 \(x\) 的开集 \(U\) 都满足 \(U \cap A \neq \emptyset\)；

(II) （度量空间情形）\(x \in \bar A \iff\) 存在 \(A\) 中点列 \(x_n \to x\)；

(III) （定义）\(x \in \bar A \iff x\) 在所有含 \(A\) 的闭集里。

(III) 是定义本身。证 (I) \(\iff\) (III)。

(III) \(\Rightarrow\) (I)：取 \(x \in \bar A\)，给开 \(U \ni x\)。假设 \(U \cap A = \emptyset\)，那 \(A \subseteq X \setminus U\) 且 \(X \setminus U\) 闭——\(\bar A \subseteq X \setminus U\)，与 \(x \in \bar A \cap U\) 矛盾。

(I) \(\Rightarrow\) (III)：设每个开 \(U \ni x\) 都交 \(A\) 非空。要证 \(x \in \bar A\)。反证，若 \(x \notin \bar A\)，那 \(x \in X \setminus \bar A\)，这是开集（\(\bar A\) 闭）。但 \(\bar A \cap A \supseteq A\)（由 (K2)），所以 \(X \setminus \bar A\) 不交 \(A\)——与 (I) 矛盾。\(\square\)

(I) 是闭包最实用 的刻画——从单点 \(x\) 检查"它的每个开邻域是否都触到 A" 即可，无需算闭集族的交。整本书里我们更常用 (I) 而不是 (III)。

(II)仅 在度量空间（或第一可数空间，ch15）里成立。证明：\((\Leftarrow)\)给 \(x_n \to x\) 与开 \(U \ni x\)，存在 \(N\) 使 \(n \geq N\) 时 \(x_n \in U\)，且 \(x_n \in A\)，故 \(U \cap A \neq \emptyset\)，由 (I) \(x \in \bar A\)。\((\Rightarrow)\)给 \(x \in \bar A\)（度量空间情形），开球 \(B(x, 1/n)\) 是 \(x\) 的开邻域，由 (I) 有 \(x_n \in B(x, 1/n) \cap A\)。这族 \(x_n\) 是 \(A\) 中点列，\(d(x_n, x) < 1/n \to 0\)。

请注意 (II) 在非 第一可数空间会塌——序列收敛抓不到拓扑闭包 这件事我们在 ch15 谈"序列 vs 网" 时还要专门讲。简单说：你在 \(\mathbb{R}\) 上习惯的"闭 = 序列闭" 是度量幸运，不是拓扑普遍真理。

5.3 内部 \(\mathrm{int}(A)\) 与边界 \(\partial A\)

"内部" 与"闭包" 完全对偶——把所有"闭" 换成"开"、把"含 A 的最小闭集" 换成"在 A 里的最大开集"：

\[ \boxed{\ \mathrm{int}(A) = \bigcup \{U : U \subseteq A, \ U \text{ 开}\}.\ } \]

"所有含于 A 的开集的并"。它的对偶刻画：

  • (a') \(\mathrm{int}(A) \subseteq A\)；
  • (b') \(\mathrm{int}(A)\) 开；
  • (c') 若 \(U\) 开且 \(U \subseteq A\)，则 \(U \subseteq \mathrm{int}(A)\)。

"含于 A 的最大开集"。\(\mathrm{int}\) 也是个算子，满足"开 的 Kuratowski 公理"——\(\mathrm{int}(X) = X\)、\(\mathrm{int}(A) \subseteq A\)、\(\mathrm{int}(\mathrm{int}(A)) = \mathrm{int}(A)\)、\(\mathrm{int}(A \cap B) = \mathrm{int}(A) \cap \mathrm{int}(B)\)。这些和闭包公理完全对称（把 \(\bar{\cdot}\) 换 \(\mathrm{int}\)、\(\subseteq\) 换 \(\supseteq\)、\(\cup\) 换 \(\cap\)、\(\emptyset\) 换 \(X\)）。

对偶关系：

\[ \boxed{\ \mathrm{int}(A) = X \setminus \overline{X \setminus A}, \qquad \bar A = X \setminus \mathrm{int}(X \setminus A).\ } \]

验证：\(X \setminus \overline{X \setminus A}\) 是所有 含 \(X \setminus A\) 的闭集 \(F\) 的补——即所有含于 \(A\) 的开集 \(X \setminus F\) 的并，正是 \(\mathrm{int}(A)\)。

有了内部 + 闭包，边界 自然定义：

\[ \boxed{\ \partial A = \bar A \setminus \mathrm{int}(A).\ } \]

"闭包减内部"。几何画面：内部是"稳稳在 A 里" 的部分，闭包是"稳稳贴着 A" 的部分（A 加它的极限），两者之差正是"夹在 A 边界上、可能在 A 里也可能不在" 的一层皮。

用 (I) 刻画：

\[ x \in \partial A \iff \text{每个含 } x \text{ 的开集 } U \text{ 同时交 } A \text{ 与 } X \setminus A \text{ 非空}. \]

"每个邻域里既有 A 的点又有 A 外的点"。这条最朴素，画面也最清楚。证：\(\partial A = \bar A \cap \overline{X \setminus A}\)（因 \(\mathrm{int}(A) = X \setminus \overline{X \setminus A}\)，所以 \(X \setminus \mathrm{int}(A) = \overline{X \setminus A}\)，再交 \(\bar A\)）。\(x \in \bar A \cap \overline{X \setminus A}\)由 (I) 等价于"每个开邻域 \(U \ni x\) 同时与 A 和 \(X \setminus A\) 相交"。

顺手得：\(\partial A = \bar A \cap \overline{X \setminus A}\) 是两个闭集的交，故 \(\partial A\) 总是闭集。请你记住这条——它是后面很多分析的起点。

分块画面。把 \(X\) 沿 \(A\) 切成三块：

\[ X = \mathrm{int}(A) \ \sqcup \ \partial A \ \sqcup \ \mathrm{int}(X \setminus A), \]

其中 \(\sqcup\) 是不交并。这三块两两不交（按定义）且并起来是 \(X\)。\(\mathrm{int}(A)\)是"稳在 A 内"，\(\mathrm{int}(X \setminus A)\)是"稳在 A 外"，\(\partial A\)是"骑在边界上"。这是闭包 / 内部 / 边界三者关系最有用的画面，记住它。

5.4 \(\bar A = A \cup \partial A\)（恒成立）

本章钩子里抛了一句"闭包 = 集合 \(\cup\) 边界 在每个空间都对吗"——朴素直觉立刻喊"对"，但请你停下来真正验证 一下。

用 (I) 证。\((\subseteq)\) 给 \(x \in \bar A\)。两情况：(i) \(x \in A\)，那 \(x \in A \cup \partial A\)。(ii) \(x \notin A\)，那 \(x \in X \setminus A\)，每个含 \(x\) 的开集 \(U\) 显然交 \(X \setminus A \ni x\) 非空；又 \(x \in \bar A\) 给出 \(U \cap A \neq \emptyset\)。两条合起来 \(x \in \partial A\)。

\((\supseteq)\) \(A \subseteq \bar A\) 由 (K2)；\(\partial A = \bar A \cap \overline{X \setminus A} \subseteq \bar A\)。\(\square\)

所以"闭包 = 集合 \(\cup\) 边界" 真是恒成立的——但原因 不是钩子里写的"集合加边界"。\(A\) 与 \(\partial A\) 可能有重叠：\(A\) 中的边界点 既在 \(A\) 里也在 \(\partial A\) 里。所以这条等式应该读作"闭包是 A 和它边界的并"，而不是"闭包是 A 与边界这两块拼在一起"——后者暗示不交并，但 \(A \cup \partial A\) 一般不是不交并。

钩子里第二句问"边界会不会包含原集合的内部"。答案——不会，按定义 \(\partial A = \bar A \setminus \mathrm{int}(A)\)，已经把内部减掉了。

但请你警惕另一件相关的事：边界可能与原集合相交。具体——\(A \cap \partial A\) 不一定空。例 \(A = [0, 1] \subseteq \mathbb{R}\)：\(\bar A = [0, 1]\)，\(\mathrm{int}(A) = (0, 1)\)，\(\partial A = \{0, 1\}\)。\(A \cap \partial A = \{0, 1\} \neq \emptyset\)——\(A\) 含它自己的边界点。这是因为 \(A\) 本身闭了。如果取 \(A = (0, 1)\)：\(\bar A = [0, 1]\)，\(\mathrm{int}(A) = (0, 1)\)，\(\partial A = \{0, 1\}\)，这次 \(A \cap \partial A = \emptyset\)——\(A\) 不含边界点。

从这两例你能看出："A 含不含自己的边界" 等价于"A 闭不闭"。具体：

\[ A \text{ 闭} \iff \bar A = A \iff \partial A \subseteq A. \]

对偶：

\[ A \text{ 开} \iff \mathrm{int}(A) = A \iff \partial A \cap A = \emptyset. \]

"开 = 不含自己边界"，"闭 = 含自己全部边界"。这是最简洁的几何刻画，请你装进脑子。

5.5 反例库：在四个非平凡拓扑里算具体集合

抽象算子吓人，具体计算才让概念落地。我们在四个标准空间里挑同一个集合 \(A\)，看 \(\bar A\)、\(\mathrm{int}(A)\)、\(\partial A\) 在每处长什么样。这一节不 学新概念——只反复练。

5.5.1 离散拓扑

\((X, \tau_d)\)，\(\tau_d = \mathcal{P}(X)\)。每个子集都开 \(\Rightarrow\) 每个子集都闭（补集也开）。所以对任意 \(A \subseteq X\)：

\[ \bar A = A, \quad \mathrm{int}(A) = A, \quad \partial A = \emptyset. \]

"边界永远空"——离散空间里没有边界点。几何上：每个点都是孤立的，没有"夹在两侧" 的点。这是离散拓扑"没有几何" 的最干净表述。

5.5.2 余有限拓扑

\(X = \mathbb{R}\)，\(\tau = \{\emptyset\} \cup \{U : X \setminus U \text{ 有限}\}\)。闭集 \(=\) 空集 \(\cup\) 有限集 \(\cup\) 全集（按 De Morgan）。

取 \(A = \mathbb{Z}\)。

  • \(\bar A\)：含 \(\mathbb{Z}\) 的最小闭集。\(\mathbb{Z}\) 无限，所以含 \(\mathbb{Z}\) 的"有限闭集" 不存在。"空集" 不含 \(\mathbb{Z}\)。剩下唯一选项 \(\mathbb{R}\)——\(\bar{\mathbb{Z}} = \mathbb{R}\)！
  • \(\mathrm{int}(A)\)：含于 \(\mathbb{Z}\) 的最大开集。开集 \(\neq \emptyset\) 必余集有限——但 \(\mathbb{Z}\) 的余集 \(\mathbb{R} \setminus \mathbb{Z}\)不有限，故含于 \(\mathbb{Z}\) 的非空 开集不存在。\(\mathrm{int}(\mathbb{Z}) = \emptyset\)。
  • \(\partial A = \bar A \setminus \mathrm{int}(A) = \mathbb{R} \setminus \emptyset = \mathbb{R}\)。

边界 \(=\) 整条实线——全空间！这与欧氏 \(\mathbb{R}\) 上的 \(\partial \mathbb{Z} = \mathbb{Z}\) 完全不同。原因：余有限里"开" 极少，每个点的"每个开邻域" 都几乎是整个 \(\mathbb{R}\)，必交 \(\mathbb{Z}\) 与 \(\mathbb{R} \setminus \mathbb{Z}\) 都非空。

把这件事多想一遍——在余有限拓扑下，\(\mathbb{Z}\) 在 \(\mathbb{R}\) 里"稠密到塞满全空间"。"稠密" 这件事强烈依赖拓扑（5.7 节专谈）。

5.5.3 Sierpiński 空间

\(X = \{0, 1\}\)，\(\tau = \{\emptyset, \{1\}, X\}\)。闭集 \(= \{\emptyset, \{0\}, X\}\)（取补）。注意 \(\{0\}\) 闭、\(\{1\}\) 不 闭，且 \(\{1\}\) 开、\(\{0\}\) 不 开——非对称的小空间。

四个子集挨个算。

\(A = \emptyset\)：\(\bar A = \emptyset\)、\(\mathrm{int}(A) = \emptyset\)、\(\partial A = \emptyset\)。无聊。

\(A = \{0\}\)：\(\{0\}\) 已闭，\(\bar A = \{0\}\)。\(\mathrm{int}(A) = \) 含于 \(\{0\}\) 的最大开集 \(= \emptyset\)（因为 \(\{1\}\)、\(X\) 都不含于 \(\{0\}\)）。\(\partial A = \{0\}\)。

\(A = \{1\}\)：\(\bar A = \) 含 \(\{1\}\) 的最小闭集 \(= X\)（\(\{0\}\) 不含 \(\{1\}\)，所以最小闭集只能是 \(X\)）！\(\mathrm{int}(A) = \{1\}\)（\(\{1\}\) 自身开）。\(\partial A = X \setminus \{1\} = \{0\}\)。

\(A = X\)：闭包内部边界与 \(A = \emptyset\) 完全对偶——\(\bar X = X\)、\(\mathrm{int}(X) = X\)、\(\partial X = \emptyset\)。

请你停下盯一下 \(A = \{1\}\) 这一栏：\(\bar{\{1\}} = X = \{0, 1\}\)！点 0 在 \(\{1\}\) 的闭包里——尽管 \(\{1\}\) 看起来一点也不"靠近" 0。原因：0 的唯一非空 开邻域是 \(X\)，它确实交 \(\{1\}\) 非空（按 (I)）。Sierpiński 拓扑里"0 看得到 1，但 1 看不到 0"——非对称的几何。

5.5.4 Sorgenfrey 直线

\(\mathbb{R}_\ell\)：以 \(\{[a, b) : a < b\}\) 为基的拓扑（这族基的合法性 ch4 验证）。每个 \([a, b)\) 是基本开集，其它开集是它们的并。

注意 \([a, b)\) 同时是闭集——其补集 \((-\infty, a) \cup [b, \infty) = \bigcup_n [a - n, a) \cup \bigcup_n [b, b+n)\) 是基本开集的并，故开。所以 \([a, b)\) 在 Sorgenfrey 里既开又闭（clopen）。这与欧氏 \(\mathbb{R}\) 上 \([a, b)\) "既不开也不闭" 形成强烈对照——同一个集合，不同拓扑下命运完全不同。

取 \(A = (0, 1] = \{x : 0 < x \leq 1\}\)。

  • \(\mathrm{int}(A)\)：找 \(A\) 内的开集。\([a, b) \subseteq (0, 1]\)需要 \(a > 0\) 且 \(b \leq 1\)。但 \(b\) 可以取 1 吗？\([a, 1) \subseteq (0, 1]\)对，\([a, 1)\)是 \(A\) 的子集且开。能否包含点 \(1\)？要含 1 的开集得是 \([1, c)\) 或它的并；\([1, c) \subseteq (0, 1]\)需 \(c \leq 1\)，但 \(c > 1\)（基本开集要求 \(a < b\)），矛盾。所以不存在 开集既含 1 又含于 \((0, 1]\)。结论：\(\mathrm{int}(A) = (0, 1)\)（\(=\) 所有 \([a, 1)\) 之并，\(0 < a < 1\)）。
  • \(\bar A\)：找 \(A\) 的闭包。点 0 在 \(\bar A\) 吗？0 的基本开邻域形如 \([0, c)\)，与 \((0, 1]\) 交得 \((0, c) \cap (0, 1]\)非空（取 \(c \leq 1\)），所以 0 \(\in \bar A\)？等等，按 (I)，0 \(\in \bar A \iff\) 0 的每个 开邻域都交 \(A\) 非空。\([0, c) \cap (0, 1] = (0, \min(c, 1)] \neq \emptyset\)（设 \(c > 0\)）——确实每个非空。但等等 0 \(\notin (0, 1]\)，会不会有反例邻域？没有——基本开邻域只能形如 \([0, c)\)（必须含 0 且形如 \([a, b)\) 给 \(a = 0\)）。所以 0 \(\in \bar A\)。
  • 但 等一下。请你把这件事跟欧氏 \(\mathbb{R}\) 上比一下——欧氏里 \(\overline{(0, 1]} = [0, 1]\)，把 0 加进来。Sorgenfrey 里 \(\overline{(0, 1]} = ?\) 我们刚证 0 \(\in \bar A\)。点 1 自然在 \(A\) 里（在 \(\bar A\) 里）。\((0, 1) \subseteq A \subseteq \bar A\)。点超过 1 的呢？取 \(x > 1\)，邻域 \([x, x + \epsilon)\) 与 \((0, 1]\) 交\(= \emptyset\)（因 \(x > 1\)），所以 \(x \notin \bar A\)。点小于 0 的，类似不在。结论：\(\bar A = [0, 1]\)。和欧氏一样的答案。
  • 但过程 不一样。请你停下注意：Sorgenfrey 里点 0 的开邻域都向右（\([0, c)\)），不能向左。这就是为什么 \([0, c) \cap (0, 1]\) 非空——交集向右是 \((0, c)\)。如果换成 \(A = [0, 1)\)，问 \(-1 \in \bar A\) 吗？\(-1\) 的邻域是 \([-1, c)\)，与 \([0, 1)\) 交 \(= [0, \min(c, 1))\)，需 \(c > 0\) 才非空，所以 \([-1, 0) \cap [0, 1) = \emptyset\) 对应取 \(c = 0\) ——不行，\(c > -1\)。取 \(c = 0\)：\([-1, 0) \cap [0, 1) = \emptyset\)！所以存在邻域 \([-1, 0)\)不交 \(A\)，故 \(-1 \notin \bar{[0, 1)}\)。但欧氏里 \(-1\) 也不在 \(\overline{[0, 1)} = [0, 1]\) 里，所以这条没区别。
  • \(\partial A = \bar A \setminus \mathrm{int}(A) = [0, 1] \setminus (0, 1) = \{0, 1\}\)。

这个例子里 Sorgenfrey 与欧氏给出同样 的 \(\bar A, \partial A\)，但 \(\mathrm{int}(A)\) 不同——欧氏给 \((0, 1)\)，Sorgenfrey 也给 \((0, 1)\)，等等，这次内部相同！让我换个反例更清楚。

取 \(A = [0, 1)\)。Sorgenfrey 里：\(A\) 自己就是基本开集，是 开集，所以 \(\mathrm{int}(A) = A = [0, 1)\)。\(\bar A\)：1 \(\in \bar A\) 吗？1 的邻域 \([1, c)\) 与 \([0, 1)\) 交 \(= \emptyset\)（因 \(c > 1\)，所以 \([1, c) \cap [0, 1) = \emptyset\)）。存在 邻域不交 \(A\)，故 1 \(\notin \bar A\)。\(\bar A = [0, 1)\)（不加点 1）。\(\partial A = \bar A \setminus \mathrm{int}(A) = \emptyset\)！

对照欧氏 \(\mathbb{R}\) 上同一个 \(A = [0, 1)\)：\(\mathrm{int}(A) = (0, 1)\)、\(\bar A = [0, 1]\)、\(\partial A = \{0, 1\}\)。

同一个集合 \([0, 1)\)，Sorgenfrey 上"整个 A 都是内部" "无边界"，欧氏上"0 与 1 都是边界点"。边界这件事强烈依赖拓扑——这是这一章最该带走的反直觉。

5.5.5 经典：\(\mathbb{Q}\) 在 \(\mathbb{R}\) 里

欧氏 \(\mathbb{R}\) 上取 \(A = \mathbb{Q}\)。算 \(\bar A\)、\(\mathrm{int}(A)\)、\(\partial A\)。

  • \(\bar{\mathbb{Q}}\)：每个 \(x \in \mathbb{R}\) 的开邻域 \(B(x, \epsilon) = (x - \epsilon, x + \epsilon)\) 都含有理数（有理数稠密）。所以每个 \(x\) 都满足 (I) \(\Rightarrow x \in \bar{\mathbb{Q}}\)。结论 \(\bar{\mathbb{Q}} = \mathbb{R}\)。
  • \(\mathrm{int}(\mathbb{Q})\)：含于 \(\mathbb{Q}\) 的开集。任何非空开集 \(U \subseteq \mathbb{R}\) 含某个 \((a, b)\)，里有无理数（无理数也稠密）——\(U \not\subseteq \mathbb{Q}\)。故含于 \(\mathbb{Q}\) 的开集只能是 \(\emptyset\)。\(\mathrm{int}(\mathbb{Q}) = \emptyset\)。
  • \(\partial \mathbb{Q} = \mathbb{R} \setminus \emptyset = \mathbb{R}\)。

"有理数的边界是整条实数线"——经典的拓扑反直觉。每个实数 \(x\)（无论有理无理）都是 \(\mathbb{Q}\) 的边界点：每个邻域里既有有理数也有无理数。

这也是为什么"测度" 与"拓扑" 是两套独立 的几何工具——\(\mathbb{Q}\) 拓扑上"满"（\(\bar{\mathbb{Q}} = \mathbb{R}\)、\(\partial \mathbb{Q} = \mathbb{R}\)），但测度 上"零测"（\(\lambda(\mathbb{Q}) = 0\)）。同一个集合的两种几何描述完全相反。这种反差在实分析里反复出现。

5.6 极限点、孤立点、完美集

"极限点" 的定义直接从闭包刻画 (I) 长出来——把"\(U \cap A\) 非空" 强化成"\(U \cap A\) 非空且不只有 x"：

\[ \boxed{\ x \text{ 是 } A \text{ 的极限点} \iff \forall \text{ 开 } U \ni x, \ (U \setminus \{x\}) \cap A \neq \emptyset.\ } \]

"x 的每个挖掉自身的邻域都还能碰到 A 的点"。极限点集合记作 \(A'\)。

请注意定义里没要求 \(x \in A\) ——\(x\) 可以在 \(A\) 里也可以不在。比较：

  • \(x \in \bar A\)：每个开邻域交 \(A\) 非空（可能只有 \(x\) 自己）；
  • \(x \in A'\)：每个开邻域挖掉 x 后还交 \(A\) 非空。

差别——\(A'\) 比 \(\bar A\) 严格"更强"。事实：

\[ \bar A = A \cup A'. \]

证：\(\supseteq\) 由 \(A \subseteq \bar A\) 与 \(A' \subseteq \bar A\)（极限点显然在闭包里）。\(\subseteq\) 给 \(x \in \bar A\)。若 \(x \in A\) 完。否则 \(x \notin A\)，则 \((U \setminus \{x\}) \cap A = U \cap A \neq \emptyset\)（因 \(x \notin A\)），故 \(x \in A'\)。\(\square\)

对应的，孤立点：

\[ \boxed{\ x \in A \text{ 是 } A \text{ 的孤立点} \iff \exists \text{ 开 } U \ni x, \ U \cap A = \{x\}.\ } \]

"x 在 A 里，但有一个开邻域只在 A 里碰到 x 自己"。孤立点不是 极限点（按定义）：孤立点的某个邻域不 包含 \(A \setminus \{x\}\) 的点。

所以 \(A\) 的点恰分两类——孤立点 \(\sqcup\) 极限点（在 \(A\) 里的）。

例。\(A = \{0\} \cup \{1/n : n \geq 1\} \subseteq \mathbb{R}\)（欧氏）。

  • 0 是极限点：每个 \((-\epsilon, \epsilon)\) 含无穷多 \(1/n\)，故挖掉 0 后非空。
  • 每个 \(1/n\) 是孤立点：取小邻域 \((1/n - \delta, 1/n + \delta)\) 比 \(1/n\) 与 \(1/(n+1)\) 之间距离 \(\frac{1}{n(n+1)}\) 还小，邻域里只有 \(1/n\) 一点属于 A。
  • \(A' = \{0\}\)，\(\bar A = A \cup \{0\} = A\) ——\(A\) 闭。

完美集（perfect set）：

\[ \boxed{\ A \text{ 完美} \iff A \text{ 闭且每个 } x \in A \text{ 都是极限点（即 } A \subseteq A'\text{）}.\ } \]

"闭、且无孤立点"。等价：\(A = A'\)（结合 \(A\) 闭得 \(\bar A = A = A \cup A'\) 即 \(A' \subseteq A\)；反向 \(A \subseteq A'\) 即定义）。

例：\([0, 1] \subseteq \mathbb{R}\) 完美——闭，且每点 \(x\) 的邻域都含无穷多点。\(\mathbb{R}\) 完美。

反例：\(A = \{1/n : n \geq 1\}\)不 完美——每个 \(1/n\) 是孤立点。

非平凡 例：Cantor 集 \(C \subseteq [0, 1]\)（取 \([0, 1]\) 反复挖中间三分之一）。\(C\) 完美——每个 Cantor 点都是极限点（任何邻域含无穷多 Cantor 点），\(C\) 闭。Cantor 集还不可数，测度零，完全不连通——一个点集论里的怪兽。

Cantor-Bendixson 定理（实分析 ch6）：每个非空完美集（在 Polish 空间里）都不可数，势 \(\geq 2^{\aleph_0}\)。这是连续统假设附近的核心事实。

5.7 稠密集：\(\bar A = X\)

稠密的定义朴素：

\[ \boxed{\ A \text{ 在 } X \text{ 中稠密} \iff \bar A = X.\ } \]

等价（按 (I)）：每个非空开集 \(U \subseteq X\) 都满足 \(U \cap A \neq \emptyset\)——"A 触到每个开集"。

经典例：\(\mathbb{Q}\) 在欧氏 \(\mathbb{R}\) 中稠密。我们刚算 \(\bar{\mathbb{Q}} = \mathbb{R}\)。同样 \(\mathbb{R} \setminus \mathbb{Q}\)（无理数）稠密、\(\mathbb{R} \setminus \mathbb{Q}\)（事实上比 \(\mathbb{Q}\) 还"稠密"——它不可数）。

稠密 + 可数的集合特别有用——叫可分（separable）：

\[ X \text{ 可分} \iff \exists \text{ 可数稠密子集 } A \subseteq X. \]

\(\mathbb{R}\) 可分（\(\mathbb{Q}\) 可数稠密）。\(\mathbb{R}^n\) 可分（\(\mathbb{Q}^n\) 可数稠密）。\(C([0,1])\) 可分（多项式可数稠密——由 Stone-Weierstrass，实分析 ch11）。\(L^p\) 可分（\(p < \infty\)）。不可分 例：\(\ell^\infty\)（实数有界序列空间，sup 范数）、\(L^\infty\)。

可分这件事在分析里至关重要——许多构造（Riesz 表示、Hahn-Banach 应用）需要可分背景。我们在 ch15 谈"第二可数 \(\Rightarrow\) 可分" 时再展开。

稠密 vs 余有限。余有限拓扑下的 \(\mathbb{R}\) 里，每个无穷子集都稠密——上面已算 \(\mathbb{Z}\) 稠密。事实上任意 \(A\) 无穷 \(\Rightarrow \bar A = \mathbb{R}\)（含 \(A\) 的闭集不能有限，只能是 \(\mathbb{R}\)）。"稠密" 在余有限下变得几乎琐碎——每个无穷集都稠密，可分性也变得免费。这又一次说明拓扑越粗，几何性质越廉价。

无处稠密。对偶概念：\(A\) 在 \(X\) 中无处稠密（nowhere dense）\(\iff \mathrm{int}(\bar A) = \emptyset\) ——闭包的内部空。例：\(\mathbb{Z} \subseteq \mathbb{R}\)（欧氏）：\(\bar{\mathbb{Z}} = \mathbb{Z}\)，\(\mathrm{int}(\mathbb{Z}) = \emptyset\)，无处稠密。Cantor 集 \(C \subseteq [0, 1]\)：\(\bar C = C\)，\(\mathrm{int}(C) = \emptyset\)（C 在 \([0,1]\) 里"含不了任何区间"——每个区间被挖掉中间），无处稠密。

"稠密" 与"无处稠密" 是两个极端。中间的"第一范畴 / 第二范畴" 集合，是 ch22 Baire 范畴定理 的主题。

5.8 临别话

这一章把"闭 / 内 / 边 / 闭包" 这四个日常词全部从开集出发严格化了。重点不是定义本身——你大概早会写——而是几个反直觉：

  • "开" 与"闭" 不是二分；可以既开又闭（clopen）或既不开也不闭。
  • "边界" 强烈依赖拓扑——同一个集合 \(A = [0, 1)\) 在欧氏 \(\mathbb{R}\) 里有边界 \(\{0, 1\}\)、在 Sorgenfrey 里无边界。
  • "稠密" 也强烈依赖拓扑——\(\mathbb{Z}\) 在欧氏 \(\mathbb{R}\) 里"处处不稠密"，在余有限 \(\mathbb{R}\) 里"稠密填满"。
  • "极限点" 与"序列极限" 的关系——后者只在度量（或第一可数）情形管用，一般拓扑空间需要"网" 这种推广（ch15 谈）。
  • 同一个 \(\mathbb{Q}\) 在 \(\mathbb{R}\) 里测度零、拓扑稠——拓扑与测度是独立 的几何观点。

这五条加起来给你一个肌肉记忆：看到任何关于"边界 / 闭包 / 稠密" 的论断，先问"在哪个拓扑下"。这条问句是"用拓扑思考" 的入口。
