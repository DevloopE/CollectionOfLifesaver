第 7 章 同胚与拓扑性质

ch1 的"圆与椭圆同胚、圆与 8 字不同胚" 这一对例子，是整本书最早 一次接触拓扑学的"区分能力"——拓扑能告诉你两个空间是不是同一个，而且不依赖距离这种细节。这种区分的正式工具 就是这一章：

  • "同一个" 的精确版本——同胚（homeomorphism）；
  • "什么算拓扑学家眼里的属性"——拓扑性质（topological invariants）；
  • "怎么证两个空间不同胚"——找一条拓扑性质让一边有、一边没。

读完这一章你能真正 区分 \(\mathbb{R}\) 与 \(\mathbb{R}^2\)、圆 \(S^1\) 与 8 字 \(\infty\)、\([0, 1]\) 与 \([0, 1)\)。这些区分以后看像 trivial，但它们是拓扑学最早的正面成果——证明"这两个空间是不同的" 比想象难得多，因为你要排除所有可能 的双连续双射。

7.1 同胚定义

设 \((X, \tau_X)\)、\((Y, \tau_Y)\) 是拓扑空间。一个映射 \(h : X \to Y\) 是同胚，如果：

  • (i) \(h\) 是双射（既单又满）；
  • (ii) \(h\) 连续；
  • (iii) \(h^{-1}\) 连续。

简记：双连续双射。当存在一个 \(X\) 到 \(Y\) 的同胚时，记 \(X \cong Y\)，读作"X 与 Y 同胚"。

这个定义把"同一个" 翻译成"有一个能恢复整个拓扑结构的双射"。具体——\(h\) 把 \(X\) 的开集与 \(Y\) 的开集一一对应：\(U \in \tau_X \iff h(U) \in \tau_Y\)（用 (ii) 与 (iii) 一起验证）。所以两个同胚的拓扑空间在开集层面 完全没区别——它们就是同一个抽象的"开集格"，只是给元素贴了不同标签。

等价表述。双射 \(h : X \to Y\) 是同胚 \(\iff\) \(h\) 是开映射（open map：\(U\) 开 \(\Rightarrow h(U)\) 开）且 \(h\) 连续。理由：对双射 \(h\)，\(h\) 是开映射 \(\iff h^{-1}\) 把开集拉回开集 \(\iff h^{-1}\) 连续。

同样：双连续双射 \(\iff\) 双射 + 连续 + 闭映射（\(F\) 闭 \(\Rightarrow h(F)\) 闭），用 6.2 节 A \(\iff\) B 那条对偶。

这些等价以后会反复用——比如证某双射是同胚时，选最方便的那个版本 检查（开映射、闭映射、\(h^{-1}\) 连续，三选一）。

7.2 注意陷阱：双射 + 连续 \(\nRightarrow\) 同胚

(iii) 这条"\(h^{-1}\) 连续" 在初学常被忽略——直觉上"连续双射" 听起来已经够了。错。

反例（再一次靠恒等映射）。

\(X = \mathbb{R}_d\)（离散拓扑）、\(Y = \mathbb{R}\)（欧氏）。考虑恒等映射 \(\mathrm{id} : \mathbb{R}_d \to \mathbb{R}\)。

  • 双射？是。
  • 连续？取欧氏开集 \(V \subseteq \mathbb{R}\)，\(\mathrm{id}^{-1}(V) = V \subseteq \mathbb{R}_d\)。\(\mathbb{R}_d\) 离散，每个 子集都开。所以 \(V\) 在 \(\mathbb{R}_d\) 中开。是 连续。
  • \(\mathrm{id}^{-1} : \mathbb{R} \to \mathbb{R}_d\) 连续？取 \(\mathbb{R}_d\) 开集 \(\{0\}\)（单点集，离散里开）。\(\mathrm{id}(\{0\}) = \{0\} \subseteq \mathbb{R}\)。问 \(\{0\}\) 在欧氏 \(\mathbb{R}\) 里开吗？不开。所以 \(\mathrm{id}^{-1}\) 不连续。

结论：恒等映射 \(\mathbb{R}_d \to \mathbb{R}\) 双射 + 连续，但不 是同胚。事实上 \(\mathbb{R}_d\) 与 \(\mathbb{R}\) 不同胚——下面 7.6 节给一条拓扑性质区分它们。

这个反例的几何含义：从"更细" 的拓扑（\(\mathbb{R}_d\)）到"更粗" 的拓扑（\(\mathbb{R}\)），恒等映射连续——因为粗拓扑的开集变多了，逆像（自身）总是细拓扑里的开集（细拓扑里什么都开）。反方向 不连续——从粗到细要把"少的开集" 拉成"多的开集"，做不到。

同胚要求双向 都连续——意味着两端拓扑同等丰富。"等同胚" 是比"等连续映射存在" 严格强的等价关系。

请你停下盯一下：双射 + 连续 + 满射 也不 够。具体：\(\mathrm{id} : \mathbb{R}_d \to \mathbb{R}\) 是双射、连续、满（双射本来就满）——但不同胚。所以"满" 这个修饰也不能补救——必须是 \(h^{-1}\) 连续。

两个特例。下面两条充分条件让"双射 + 连续 \(\Rightarrow\) 同胚" 成立，但都需要额外的几何条件：

  • 定理（ch16）：\(X\) 紧致、\(Y\) Hausdorff、\(h : X \to Y\) 连续双射 \(\Rightarrow h\) 同胚。证明思路：闭子集 \(F \subseteq X\) → \(F\) 紧致（紧致空间的闭子集紧致）→ \(h(F)\) 紧致（连续映射保紧致）→ \(h(F)\) 闭（Hausdorff 里紧致 = 闭）→ \(h\) 闭映射 → 同胚。
  • 定理（不变域定理 \(\mathbb{R}^n\) 内部不变性）：\(h : U \to V\) 连续双射，\(U \subseteq \mathbb{R}^n\) 开 \(\Rightarrow V \subseteq \mathbb{R}^n\) 开 + \(h\) 同胚（Brouwer 定理，深度结果，超出本书）。

但这两条都需要 紧致或欧氏背景。一般情形下"双连续双射" 必须显式验证两侧。

7.3 同胚是等价关系

命题。 "\(\cong\)" 是拓扑空间集合上的等价关系。

证。三条要查。

反身。\(\mathrm{id} : X \to X\) 双射、连续（6.4.2）、\(\mathrm{id}^{-1} = \mathrm{id}\) 也连续。\(X \cong X\)。

对称。若 \(h : X \to Y\) 同胚，则 \(h^{-1} : Y \to X\) 也是双射 + 连续（按定义）+ \((h^{-1})^{-1} = h\) 也连续（按定义）。\(Y \cong X\)。

传递。若 \(h : X \to Y\)、\(k : Y \to Z\) 同胚，则 \(k \circ h : X \to Z\) 是双射（双射的复合是双射）、连续（连续的复合连续，6.5）、\((k \circ h)^{-1} = h^{-1} \circ k^{-1}\) 是 \(k^{-1}\) 与 \(h^{-1}\) 的复合，仍连续。\(X \cong Z\)。\(\square\)

这条让"同胚等价类" 成为合法 的概念——拓扑空间的全体在 \(\cong\) 下分成等价类，每类叫一个"拓扑型"（topology type 或 homeomorphism class）。"圆型" "球型" "甜甜圈型" "双甜甜圈型" 等等都是具体的等价类。

拓扑学的全部 任务可以浓缩成一句话：列出所有拓扑型，并给出区分它们的方法。这件事在低维（曲面，2 维）已经完整做到（ch24 流形讨论："闭曲面分类定理"），在3 维 是 Perelman 2003 年完成的（Geometrization 与 Poincaré 猜想），在4 维 至今仍未完全解决（4 维是个怪洞）。

7.4 拓扑性质

"拓扑性质"（topological property，或 topological invariant）的定义：

\[ \boxed{\ P \text{ 是拓扑性质} \iff X \text{ 满足 } P \text{ 且 } X \cong Y \Rightarrow Y \text{ 也满足 } P.\ } \]

"同胚下保持的性质"。这是一个朴素到几乎没说什么的定义——但它的意义 全在反过来用：若 X 满足 P 而 Y 不满足 P，则 X 与 Y 不同胚。这是拓扑学区分空间 的标准武器。

哪些性质是拓扑性质？粗略说——用拓扑结构（开集 / 闭集 / 闭包 / 内部 / 边界 / 极限点 / 邻域 / 连续映射 / ...) 表述、不用度量 / 代数 / 测度等额外结构的性质 都是拓扑性质。这是非正式判据，但实际上很有用。

不 是拓扑性质的例子：

  • "有界"——\((0, 1)\) 有界、\(\mathbb{R}\) 不有界，但两者同胚（\(f(x) = \tan(\pi x - \pi/2)\)）。"有界" 依赖度量。
  • "完备"（Cauchy 列收敛）——\((0, 1)\) 不完备（\(1/n\) Cauchy 不收敛在 \((0, 1)\) 内）、\(\mathbb{R}\) 完备，两者同胚。"完备" 依赖度量。
  • "测度有限"——同上，\((0, 1)\) 测度 1、\(\mathbb{R}\) 测度无穷，同胚。"测度" 依赖测度结构。
  • "线性"（向量空间结构）——\(\mathbb{R}\) 与圆 \(S^1\) 都是 1 维流形，但 \(S^1\) 没向量空间结构（事实上 \(\mathbb{R}\) 与 \(S^1\) 不同胚——但仅仅因为 \(S^1\) "是闭的"，不是因为线性结构差别）。

请你停下注意"有界"不 是拓扑性质这件事的分量——这意味着你不能用"X 有界" 区分 \(X\) 与 \(Y\)。\((0, 1)\) 与 \(\mathbb{R}\) 拓扑上"同一个"——拓扑学家眼里没区别。这件事让人不舒服很正常，因为日常几何里"有界 / 无界" 看起来截然不同。但拓扑就是把这种距离上的差别 抹掉，留下纯结构 上的差别。

7.5 拓扑性质清单（前瞻）

下面这张表列这本书后面会专章讲 的拓扑性质。每条都"同胚下保持"——"X 满足 P 且 X \cong Y \Rightarrow Y 满足 P"。

性质含义（粗略）本书章
紧致（compact）每个开覆盖有有限子覆盖ch16
连通（connected）不能写成两非空开集的不交并ch18
道路连通（path-connected）任两点有连续路径相连ch18
Hausdorff（T2）任两不同点有不交开邻域ch12
第一可数（first countable）每点有可数邻域基ch15
第二可数（second countable）整个拓扑有可数基ch15
可分（separable）有可数稠密子集ch15
正规（normal, T4）任两不交闭集有不交开邻域分ch12
度量化（metrizable）存在度量诱导该拓扑ch21
局部紧致（locally compact）每点有紧致邻域ch16

这十条几乎覆盖了"拓扑空间分类" 的所有粗略工具。具体证"每条都是拓扑性质" 是各章的主要任务之一——但大部分 证明是同一个套路：

  1. 把性质 P 用"开集 / 连续映射" 翻译；
  2. 注意同胚 \(h : X \to Y\) 把 \(X\) 的开集精确双射到 \(Y\) 的开集；
  3. 若 P 在 \(X\) 成立，沿 \(h\) 把见证（覆盖、稠密集、邻域基等）搬到 \(Y\)，得到 P 在 \(Y\) 的见证。

这个套路普遍到没什么新意——但每个性质都得验一遍，因为"用开集表述" 这一步对每个性质长得不一样。

注意。有些性质是"拓扑性质" 但不 "遗传"——不能传给子空间。比如"紧致" 不遗传到任意子空间（紧致空间的子集不一定紧致——不过闭 子集会）；"连通" 不遗传（圆连通，但拿掉一个开弧得两段不连通片）；"第一 / 第二可数" 遗传。"可分" 一般不遗传到任意子空间——但遗传到开子空间。这些细节是各章主题。

7.6 区分技巧

"X 与 Y 不同胚" 怎么证？最常用的策略：

\[ \boxed{\ \exists \text{ 拓扑性质 } P, \ X \text{ 满足 } P, \ Y \text{ 不满足 } P \ \Longrightarrow \ X \not\cong Y.\ } \]

逆否命题——若 \(X \cong Y\)，由 P 是拓扑性质，X 满足 \(\Rightarrow Y\) 满足，矛盾。

下面几个具体例。

例 7.6.1：\(\mathbb{R}_d\) 与 \(\mathbb{R}\) 不同胚。取 P = "每个单点集开"。\(\mathbb{R}_d\) 满足（离散）、\(\mathbb{R}\) 不满足（\(\{0\}\) 不开）。是不是拓扑性质？验证：若 \(X\) 每个单点集开、\(h : X \to Y\) 同胚，对 \(y \in Y\)，\(h^{-1}(y) = \{x\}\)（\(h\) 双射），那 \(\{x\}\) 在 \(X\) 中开，所以 \(h(\{x\}) = \{y\}\) 在 \(Y\) 中开（\(h\) 是开映射）。\(\checkmark\)

更直接版的 P = "X 是离散拓扑"——也是拓扑性质（同胚保持整个 开集格）。\(\mathbb{R}_d\) 离散、\(\mathbb{R}\) 不离散。

不过 P 不必这么强；任何能区分的性质都行。比如 P = "X 不可数且每点都孤立"——\(\mathbb{R}_d\) 满足、\(\mathbb{R}\) 不满足。也能区分。

例 7.6.2：\([0, 1]\) 与 \([0, 1)\) 不同胚。P = "X 紧致"（ch16）。\([0, 1]\) 紧致（Heine-Borel）、\([0, 1)\) 不紧致（开覆盖 \(\{[0, 1 - 1/n) : n \geq 2\}\) 无有限子覆盖）。同胚保紧致 \(\Rightarrow\) 不同胚。

另一种方法：拿掉一点看连通性变化。\([0, 1] \setminus \{0\} = (0, 1]\) 连通；\([0, 1) \setminus \{1/2\} = [0, 1/2) \cup (1/2, 1)\) 不连通。但 \([0, 1] \setminus \{1/2\} = [0, 1/2) \cup (1/2, 1]\) 也不连通。所以不能简单 用"拿一点连通性" 区分两者——得拿"边界点"。\([0, 1]\) 有两个边界点（0 与 1，每个拿掉后剩下连通），\([0, 1)\) 只有一个边界点（0，拿掉后剩下连通；但 \([0, 1)\) 中拿任意内点剩下不连通）。"边界点个数" 是拓扑不变量（属于"局部" 性质，ch24 流形带边讨论）。

例 7.6.3：\(\mathbb{Q}\) 与 \(\mathbb{R}\) 不同胚。P = "X 连通"。\(\mathbb{R}\) 连通（实数线是单段）、\(\mathbb{Q}\) 不 连通——任何 \(\mathbb{Q}\) 的子集 \(\{q < \pi\}\) 与 \(\{q > \pi\}\) 都开（\(\pi\) 不在 \(\mathbb{Q}\) 里），它俩不交、并起来 \(= \mathbb{Q}\)。同胚保连通 \(\Rightarrow\) 不同胚。

例 7.6.4：\(S^1\) 与 \(\infty\)（8 字）不同胚。P = "X 拿掉某一点后剩下的连通片数 \(\leq 2\)"。\(S^1\) 拿掉任 一点后剩下一段开弧 \(\cong (0, 1)\) 连通——1 片。\(\infty\) 8 字有一个交点 \(p\)，拿掉它后剩下两个开弧——3 片（一个开弧"双向" 算一段；具体看几何模型——8 字 \(= S^1 \vee S^1\) 两个圆共点，拿掉共点后剩两个开弧）。3 \(>\) 2，矛盾。

但等等。"拿掉一点后" 这条作为拓扑性质，正确表述是"对任意 \(p\)，\(X \setminus \{p\}\) 的连通片数"。这不是 一个简单数——是"p 在 X 中的局部连通性见证"。这是 拓扑不变量（同胚 \(h\) 把 \(p\) 送到 \(h(p)\)，\(X \setminus \{p\} \cong Y \setminus \{h(p)\}\)，连通片数同胚保持）。

但作为整空间 的性质 P，要表述成"存在点 \(p\) 使 \(X \setminus \{p\}\) 不连通" 或类似。\(\infty\) 满足（取交点 \(p\)，\(\infty \setminus \{p\}\) 三段不连通）。\(S^1\) 不满足（拿任一点剩下都连通）。所以 \(\infty\) 与 \(S^1\) 不同胚。

这种"拿点看变化" 的技巧叫 cut points 论证，是低维拓扑里常用的强工具。

7.7 \(\mathbb{R}\) 与 \(\mathbb{R}^2\) 不同胚

这是最经典 的"不同胚" 证明，用的是"cut points"。

断言。\(\mathbb{R} \not\cong \mathbb{R}^2\)。

证。假设有同胚 \(h : \mathbb{R} \to \mathbb{R}^2\)。取 \(x = 0 \in \mathbb{R}\)。\(\mathbb{R} \setminus \{0\} = (-\infty, 0) \cup (0, \infty)\)不 连通。同胚把 \(\mathbb{R} \setminus \{0\}\) 双连续映到 \(\mathbb{R}^2 \setminus \{h(0)\}\)。但 \(\mathbb{R}^2 \setminus \{h(0)\}\) 连通——任何两点 \(p, q \in \mathbb{R}^2 \setminus \{h(0)\}\) 可以用"绕开 \(h(0)\)" 的折线相连，故是道路连通故连通。

"连通" 是拓扑性质（ch18）——\(\mathbb{R} \setminus \{0\}\) 不连通而 \(\mathbb{R}^2 \setminus \{h(0)\}\) 连通，矛盾。\(\square\)

请你停下盯一下这个证明的简洁性——只用"拿掉一点连通性怎么变" 就把"1 维 vs 2 维" 区分开了。这是拓扑学第一个真正"反直觉地强" 的成果——你没用 任何"维数" 的概念，但给出了 维数的拓扑刻画。

推广。同样思路证 \(\mathbb{R}^n \not\cong \mathbb{R}^m\)（\(n \neq m\)）不容易——"拿掉一点" 把 \(\mathbb{R}^n\) 变成 \(\mathbb{R}^n \setminus \{0\}\)，对 \(n \geq 2\) 都连通。区分 \(\mathbb{R}^n\) 与 \(\mathbb{R}^m\)（\(n, m \geq 2\)）需要更深的不变量——同伦群、同调群。

具体：\(\pi_{n-1}(\mathbb{R}^n \setminus \{0\}) = \mathbb{Z}\)（"n−1 维球面是 \(\mathbb{R}^n \setminus \{0\}\) 的形变收缩"），所以 \(\pi_{n-1}\) 给"维数" 的真正拓扑刻画。这是ch23 基本群 + 代数拓扑的入口。

一个重要历史脚注——"\(\mathbb{R}^n \cong \mathbb{R}^m \Rightarrow n = m\)" 这条最早由 Brouwer 1912 年证明（不变域定理 + 不变维度定理），其前 Cantor 已在 1877 年发现 \(\mathbb{R}\) 与 \(\mathbb{R}^2\) 之间存在双射（用 Hilbert 曲线 / 截位插入）——但那个双射不连续。Brouwer 的贡献正是证明连续 双射不存在。这是拓扑学早期最重要的定理之一。

7.8 嵌入与浸入

"同胚到子空间" 是同胚的近亲——但更宽松：

\[ \boxed{\ f : X \to Y \text{ 是嵌入（embedding）} \iff f(X) \subseteq Y \text{ 配子空间拓扑后, } f : X \to f(X) \text{ 是同胚.}\ } \]

展开——\(f\) 单射、连续，且 \(f^{-1} : f(X) \to X\) 连续（在子空间拓扑下）。\(f\) 不必满，但限制到像 \(f(X)\) 后双连续。

例：包含映射 \(\iota : A \hookrightarrow X\)（\(A\) 配子空间拓扑）总是嵌入——\(\iota : A \to A\)（恒等）平凡同胚。

例：\(f : (0, 1) \to \mathbb{R}^2\)，\(f(t) = (t, 0)\)。把开区间嵌成 \(\mathbb{R}^2\) 中 x 轴的一段。是嵌入——\(f\) 单、连续，\(f^{-1}(x, 0) = x\) 在 \(f((0, 1)) = (0, 1) \times \{0\}\) 上连续。

反例：\(f : [0, 2\pi) \to S^1 \subseteq \mathbb{R}^2\)，\(f(t) = (\cos t, \sin t)\)。双射（每个圆上点对应唯一 \(t \in [0, 2\pi)\)）、连续，但 \(f^{-1}\) 不连续——点 \((1, 0) = f(0)\) 的小邻域是圆上一段小开弧 \(\{(\cos t, \sin t) : t \in (-\delta, \delta) \cup (2\pi - \delta, 2\pi)\}\)，对应到 \([0, 2\pi)\) 上是 \([0, \delta) \cup (2\pi - \delta, 2\pi)\)——不连通的两段——而 \(f^{-1}((1, 0)) = 0\) 的每个 \([0, 2\pi)\) 邻域 \([0, \epsilon)\) 是连通的一段。\(f^{-1}\) 把"不连通邻域" 拉回"连通邻域" ——不可能连续。

这个反例说明：连续单射不 自动是嵌入。"把 \([0, 2\pi)\) 卷成圆" 这件事在拓扑上 不是把 \([0, 2\pi)\) 嵌进 \(\mathbb{R}^2\) ——因为头尾被粘在了一起，造成 \(f^{-1}\) 在粘点不连续。

正确的"圆作为 \([0, 2\pi]\) 的商" 应该用商拓扑（ch10），把端点粘起来——这样可以 给同胚 \(S^1 \cong [0, 2\pi] / (0 \sim 2\pi)\)。

浸入 vs 嵌入。在微分拓扑（ch24）里还有"浸入"（immersion）——局部是嵌入，但全局不必。比如 \([0, 2\pi) \to S^1\) 是 浸入但不是 嵌入。8 字曲线作为某条 \(\mathbb{R} \to \mathbb{R}^2\) 的像也是浸入但不是嵌入（自交点处违反单射）。"嵌入" 与"浸入" 的区别在低维拓扑里至关重要。

7.9 拓扑分类 vs 度量分类

把这一章和你以前学的度量空间分类 对比。两层等价关系：

  • 等距同构（isometry）：\(d_X(x, x') = d_Y(h(x), h(x'))\) 的双射。保距离。
  • 同胚（homeomorphism）：双连续双射。保拓扑结构。

等距同构远比同胚严格。每对等距同构的空间都同胚（等距 \(\Rightarrow\) 双连续），但反过来错——同胚的两个度量空间往往不 等距。

具体：圆 \(S^1\) 半径 1 与圆 \(S^1\) 半径 2 是同胚（双连续双射 \(x \mapsto 2x\)）但不 等距（一个周长 \(2\pi\)、一个 \(4\pi\)，距离不一样）。

具体：\((0, 1)\) 与 \(\mathbb{R}\)（欧氏度量）同胚（\(\tan(\pi x - \pi/2)\)）但不 等距（一个有界、一个无界）。

"拓扑等价类" 远比"度量等价类" 粗——很多度量上不同的空间在拓扑上同胚。这是为什么"拓扑学" 才是真正研究"形状" 的学科——它把度量上的细节抹掉，留下纯结构。

另一方面，"拓扑等价类" 也比"同伦等价类" 严格。同伦等价（homotopy equivalence）——存在 \(f : X \to Y\)、\(g : Y \to X\) 使 \(g \circ f \simeq \mathrm{id}_X\)、\(f \circ g \simeq \mathrm{id}_Y\)（\(\simeq\) 是同伦关系，ch23）——是比同胚更粗 的等价。

具体：\(\mathbb{R}^n\) 与一点 \(\{*\}\) 同伦等价（\(\mathbb{R}^n\) 收缩到一点）但不 同胚（一个无限、一个有限）。圆环 \(D^2 \setminus \{0\} = \{x \in \mathbb{R}^2 : 0 < |x| \leq 1\}\) 与圆 \(S^1\) 同伦等价（圆环形变收缩到中心圆）但不 同胚（圆环是 2 维曲面，圆是 1 维曲线）。

三层粗细递进：

\[ \text{等距} \quad \Rightarrow \quad \text{同胚} \quad \Rightarrow \quad \text{同伦等价}. \]

每层放宽都让"等价类" 变大。代数拓扑（同伦群、同调群）研究同伦等价类，点集拓扑（这本书）研究同胚类，度量几何 研究等距类。三种几何观点，三种"形状"。

7.10 临别话

这一章给"什么算同一个拓扑空间" 一个精确答案：同胚。然后给"怎么证两个空间不同胚" 一个武器库：拓扑性质 + cut points + 后续章节的紧致 / 连通 / Hausdorff / 可数等。

三件事请你带走：

  • "双连续双射" 中的"双" 不可省——\(\mathbb{R}_d \to \mathbb{R}\) 反例钉死。
  • "拓扑性质" 是同胚下不变的属性，区分空间的标准武器。
  • 三层等价关系——等距 / 同胚 / 同伦等价——粗细递进。本书定位在"同胚" 这一层。

下一章我们开始把"从已知拓扑空间造新拓扑空间" 这件事系统化——子空间、积空间、商空间、任意积空间。这四种构造（ch8-11）是后面所有几何讨论的基础工具。每一种构造都把"拓扑学家做事的方式" 推进一步：从分析现有空间 到主动构造新空间。
