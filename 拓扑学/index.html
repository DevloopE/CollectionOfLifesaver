普林斯顿拓扑学读本

把"距离" 抽掉, 留下"形状" · 点集拓扑 + 通向代数拓扑的桥

前言

欢迎你翻开这本书。拓扑学是数学里最奇怪 的一门——它把"距离" 这个最基本的几何概念抽掉, 只研究"开集"。听起来像放弃了几何, 实际上是把几何提炼到本质。"连续""紧致""连通" 这些你以前用多元微积分和实分析反复用过的词, 在这本书里第一次 摆脱距离的依赖, 成为纯粹 的几何概念。

这种抽离让"什么算同一个空间" 这件事变了。一个咖啡杯和一个甜甜圈, 在度量上完全不同, 但拓扑上"相同"——你能把杯子连续变形成甜甜圈, 不撕不粘。这种"同" 叫同胚。整本书很大块在研究"什么是拓扑性质"——即在同胚下保持不变的性质。

这本书的角色

系列第 12 卷。它填的是一个明确的缺口——你前面读过多元微积分(用 \(\mathbb{R}^n\) 距离)、实分析(把度量空间公理化)、泛函分析(无穷维空间), 都用过"开集""紧致""连续" 这些词, 但没人 正面问"没有距离的开集是什么"。本书正面攻击。

它也是代数拓扑(归档在 archive/tier4/, 待重写) 的前置。基本群、同调、覆盖空间这些东西需要拓扑空间作为对象——本书最后两章 ch23-24 把基本群和流形的入口给你, 让你能开始 读 Hatcher 那种代数拓扑书。

这本书不 是什么

它不 是 Hatcher 那种代数拓扑——同伦群 / 同调 / 上同调 / 覆盖空间 / 谱序列等不是本书话题。本书止于点集(general topology) + 一点代数拓扑入口。

它不 是微分拓扑——流形、切丛、微分形式、Sard 定理等也不是本书话题。本书 ch24 简介流形概念但不做微分结构。

它不 是 Munkres 那种"所有点集结果列表"。本书每章只挑核心的 4-6 个结果讲透。

读这本书之前你需要什么

你需要实分析第 4-9 章——度量空间、收敛、紧致、连通在 \(\mathbb{R}^n\) 的版本。本书 ch2 简短复习, 但背景不能完全空。

你不 需要泛函分析——不过如果读过, ch11 Tychonoff、ch22 Baire 你会更熟。

你需要集合论基本素养——等价关系、商集、笛卡尔积(任意 index 集)、选择公理。附录 A 速查。

"我下周就要考期中, 从哪里开始?"

看你课重点。本科点集拓扑 标准考试看：

  • 拓扑空间 + 基 + 子基（ch3-4）
  • 连续 + 同胚（ch6-7）
  • 子空间 / 积 / 商拓扑（ch8-11）
  • 分离公理 + Hausdorff（ch12）
  • 紧致 + Tychonoff（ch14-16）
  • 连通（ch18-19）

研究生点集 + 入门代数拓扑 加上：

  • Urysohn 引理 + Tietze + 度量化（ch20-21）
  • Baire 范畴定理 + 应用（ch22）
  • 基本群（ch23）

例题 + 反例 + 严格度

例题埋在正文, 没"例 4.2.1" 标签。每个新概念都有反例伴随——拓扑学反例特别重要, 因为很多"看似显然" 的事其实需要额外假设(Hausdorff、可分、第二可数等)。

关键反例库：Sierpiński 空间、双箭头空间、Sorgenfrey 直线、长直线、Cantor 集、范例数 \(\omega_1\) 上的序拓扑。附录 A 索引常用反例。
