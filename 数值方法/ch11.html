第 11 章 分段多项式与三次样条

上一章 Runge 现象的 lesson 是：用一个高次多项式覆盖整个区间是个坏主意。Chebyshev 节点修了一部分（在你能选节点时），但很多实际场景里你不能选节点——数据是按时间均匀采样的、是按物理传感器位置摆好的、是已经预存的等距网格。在这些情况下，修法不是"更高次"，是"分段低次"。

这一章给最常用的"分段低次" 方法：三次样条。它在每段上是 3 次多项式，段与段在节点处光滑接合（值、一阶、二阶都连续）。三次样条几乎是工程实战里"插值" 这个词的默认含义——MATLAB、Mathematica、SciPy、各 CAD 软件、各动画软件，"插值" 的默认实现都是某种形式的三次样条。

11.1 分段思路

把 \([a, b]\) 切成 \(n\) 段：\(a = x_0 < x_1 < \cdots < x_n = b\)。在每段 \([x_i, x_{i+1}]\) 上用一个低次多项式 \(s_i(x)\)。整体函数 \(S(x)\) 由各段拼起来：\(S|_{[x_i, x_{i+1}]} = s_i\)。

核心好处：

  • 每段次数低（1 次、2 次或 3 次），没有 Runge 现象的高次震荡。
  • "局部行为" 由局部多项式决定——一段的 \(s_i\) 不影响远处。
  • 容易控制。

核心权衡：节点 \(x_i\) 处段与段接合时多光滑。这是设计选择：

  • 只值连续：\(C^0\)，可能有"折角"。
  • 值 + 一阶导连续：\(C^1\)，平滑但二阶导可能跳。
  • 值 + 一阶 + 二阶导连续：\(C^2\)，看上去和"原函数" 没差别。三次样条就是这个。

更高的光滑度（\(C^3\) 等）可以做但通常不需要——三次样条已经满足绝大多数应用。

11.2 分段线性插值

最朴素：每段是直线。点 \((x_i, y_i)\) 和 \((x_{i+1}, y_{i+1})\) 用线段连：

\[ s_i(x) = y_i + \frac{y_{i+1} - y_i}{x_{i+1} - x_i}(x - x_i), \quad x \in [x_i, x_{i+1}]. \]

整体 \(S\) 是折线——\(C^0\) 但不 \(C^1\)。

误差：如果 \(f \in C^2\)，

\[ \max_{x \in [x_i, x_{i+1}]} |f(x) - s_i(x)| \le \frac{(x_{i+1} - x_i)^2}{8} \max_x |f''(x)|. \]

所以全区间最大误差是 \(O(h^2)\)，其中 \(h = \max_i (x_{i+1} - x_i)\) 是网格尺度。

"\(O(h^2)\)" 翻译成实用：把节点数加倍（\(h\) 减半），误差减小四倍。要 16 位精度（误差 \(10^{-15}\)），\(h \approx 10^{-8}\)——节点数得 \(10^8\)。不可接受。

分段线性的好处是简单 + 局部，但收敛太慢。我们需要更高阶的分段方法——同时保留"分段" 的好处和"高阶" 的快收敛。三次样条是答案。

11.3 三次样条：定义

函数 \(S : [a, b] \to \mathbb{R}\) 叫三次样条，如果：

  • (a) 在每段 \([x_i, x_{i+1}]\) 上 \(S\) 是 3 次多项式；
  • (b) \(S(x_i) = y_i\) 对所有 \(i\)（插值条件）；
  • (c) \(S, S', S''\) 在所有内部节点 \(x_1, \ldots, x_{n-1}\) 处连续。

"\(C^2\) 光滑" 是关键——\(S''\) 连续意味着曲率没跳变，看起来像"真实函数"。这是三次样条比折线显著好的原因之一。

计数自由度。每段 4 个系数（\(a_i + b_i x + c_i x^2 + d_i x^3\)），\(n\) 段共 \(4n\) 个未知数。

条件：

  • \(2n\) 个：每段两端值匹配数据 (\(s_i(x_i) = y_i, s_i(x_{i+1}) = y_{i+1}\))；
  • \(n - 1\) 个：内部节点 \(s_i'(x_{i+1}) = s_{i+1}'(x_{i+1})\)（一阶导连续）；
  • \(n - 1\) 个：内部节点 \(s_i''(x_{i+1}) = s_{i+1}''(x_{i+1})\)（二阶导连续）。

合计 \(4n - 2\) 个方程。比未知数少 2 个——所以三次样条不唯一，需要补 2 个边界条件。

11.4 边界条件四选一

常见 4 种补法：

(BC1) 自然样条（natural）。\(S''(x_0) = S''(x_n) = 0\)。即两端"无曲率"，曲线在端点附近接近线性。物理直觉：钢条只在内部节点固定，两端自由——它会松弛到无曲率。这是数学上最"对称" 的选择。

(BC2) 夹紧样条（clamped）。\(S'(x_0) = y_0', S'(x_n) = y_n'\) 给定。即已知两端的斜率。需要额外信息（端点导数），但精度比自然样条高一阶。

(BC3) 周期样条（periodic）。\(S^{(k)}(x_0) = S^{(k)}(x_n)\) for \(k = 0, 1, 2\)。前提：\(y_0 = y_n\)。用于周期数据（信号、轨道）。

(BC4) 非节样条（not-a-knot）。\(S'''\) 在 \(x_1\) 和 \(x_{n-1}\) 处也连续。等价：\(s_0\) 和 \(s_1\) 是同一个 3 次多项式（"\(x_1\) 不再是真节点"），\(s_{n-2}\) 和 \(s_{n-1}\) 同理。这是 MATLAB / SciPy 的默认选择，因为它不需要额外信息又比自然样条精度高。

实战经验：

  • 不知道选哪个 → 用 not-a-knot（默认）。
  • 明知端点导数 → 用 clamped（精度最高）。
  • 数据是周期 → 用 periodic。
  • "自然样条" 听上去最简单，但精度不好（端点附近误差比内部高一阶）。日常工作里用得不多。

11.5 怎么算样条

常见的实现是"二阶导版本"——把节点处的二阶导 \(M_i = S''(x_i)\) 当未知数，导出一个三对角线性方程组。

推导（草图）。在 \([x_i, x_{i+1}]\) 上 \(S''\) 是线性的（因为 \(S\) 是三次的），由两端值 \(M_i, M_{i+1}\) 决定。两次积分加上端点条件 \(S(x_i) = y_i, S(x_{i+1}) = y_{i+1}\) 给出 \(s_i\) 显式形式。

用一阶导连续条件 \(s_i'(x_{i+1}) = s_{i+1}'(x_{i+1})\) 得到

\[ \frac{h_{i-1}}{6} M_{i-1} + \frac{h_{i-1} + h_i}{3} M_i + \frac{h_i}{6} M_{i+1} = \frac{y_{i+1} - y_i}{h_i} - \frac{y_i - y_{i-1}}{h_{i-1}}, \]

其中 \(h_i = x_{i+1} - x_i\)。这条方程对每个内部节点 \(i = 1, \ldots, n-1\) 成立——共 \(n - 1\) 个方程，\(n + 1\) 个未知数 \(M_0, M_1, \ldots, M_n\)。补两个边界条件给 \(M_0\) 和 \(M_n\)（自然样条直接给 \(M_0 = M_n = 0\)）。

得到的线性方程组是三对角——每行只 3 个非零元素（\(M_{i-1}, M_i, M_{i+1}\)）。三对角系统用Thomas 算法（高斯消元的特殊化）\(O(n)\) 解出。所以整个三次样条构造是 \(O(n)\) 时间。

有了 \(M_i\)，每段的 \(s_i\) 就有显式公式：

\[ s_i(x) = M_i \frac{(x_{i+1} - x)^3}{6 h_i} + M_{i+1} \frac{(x - x_i)^3}{6 h_i} + \left(\frac{y_i}{h_i} - \frac{M_i h_i}{6}\right)(x_{i+1} - x) + \left(\frac{y_{i+1}}{h_i} - \frac{M_{i+1} h_i}{6}\right)(x - x_i). \]

看着复杂但都是直接展开。SciPy 的 CubicSpline 内部就是这条流程。

11.6 收敛、好处、局限

收敛速率。对 \(f \in C^4\)，

\[ \boxed{\ \max_x |f(x) - S(x)| = O(h^4),\ } \]

其中 \(h = \max_i h_i\) 是网格最大尺度。

对比分段线性的 \(O(h^2)\)：节点数加倍，三次样条误差减小16 倍而不是 4 倍。要 16 位精度只需 \(h \approx 10^{-4}\)——节点数 \(\sim 10^4\)，远比线性的 \(10^8\) 实用。

更妙的：不受 Runge 现象影响。无论你的节点等距还是不等距，无论 \(f\) 在哪个区域更"难"（变化快），三次样条都温和地拟合。

好处清单：

  • \(O(n)\) 构造，\(O(\log n)\) 求值（找正确段 + 评估）。
  • \(O(h^4)\) 收敛——比分段线性高两阶。
  • 对节点不挑——等距、不等距、聚集都行。
  • 对数据形状鲁棒——不出 Runge。
  • \(C^2\) 光滑——视觉和物理上都"自然"。

局限：

  • 非局部。三对角方程组的解 \(M_i\) 全局耦合——动一个 \(y_i\) 会改变远处所有 \(M_j\)（虽然影响指数衰减）。如果你需要"真局部" 修改（比如 CAD 里拖一个控制点只影响附近曲线），用 B-spline（下一章末尾会提）。
  • 边界条件影响。端点附近精度依赖于边界条件选择。处理不当会有可见的端点伪影。
  • 不能处理跳点 / 折角。如果 \(f\) 真的有跳变，强迫 \(C^2\) 光滑会让样条在跳点附近过冲。
  • 过冲（overshoot）。即使 \(f\) 只是陡峭而非真跳变，三次样条可能在陡变区域产生轻微过冲（典型 5-10%）。修法用单调样条（如 PCHIP—— Piecewise Cubic Hermite Interpolating Polynomial），它牺牲一点光滑性换严格单调。

11.7 实战与生态

SciPy 的插值族：

  • scipy.interpolate.CubicSpline：经典三次样条。
  • scipy.interpolate.PchipInterpolator：单调样条，保形。
  • scipy.interpolate.Akima1DInterpolator：Akima 样条，对离群点更稳健。
  • scipy.interpolate.UnivariateSpline：带平滑参数的样条——可以在严格插值与平滑拟合之间调节，对噪声数据有用（连接到下一章的最小二乘思想）。

对二维 / 高维数据：

  • scipy.interpolate.RectBivariateSpline：规则二维网格。
  • scipy.interpolate.griddata：散点数据。
  • scipy.interpolate.RBFInterpolator：基于径向基函数的高维插值。

实际工程里 99% 的插值任务直接用其中一个即可。不要从头实现样条（有大量边界 case 容易踩坑）；要理解它们的数学，知道哪里会出问题、如何诊断。

下一章我们退一步问：如果数据有噪声呢？"插值" 强迫曲线穿过每个数据点——但如果数据点本来就是"真值 + 噪声"，强迫穿过它们意味着穿过噪声，曲线被噪声污染。这种情况要的是拟合，不是插值。最经典的拟合方法是最小二乘——找一个低维参数族里的"最佳" 曲线，不要求穿过每个点，只要求总平方误差最小。下一章把它和数值线性代数（QR 分解、正交多项式）结合，给一套既稳定又高效的算法。
