第 24 章 PDE 有限差分入门 + FFT 一瞥

这是这本书的最后一章。我会做两件事：第一，把偏微分方程 用最朴素的有限差分变成一个大型 ODE / 线性代数问题——这一步把前面 23 章几乎所有工具都串起来用一遍。第二，看一眼快速 Fourier 变换（FFT）——一个把"\(O(n^2)\) 的笨办法变 \(O(n\log n)\) 的聪明办法" 的算法奇迹，本身值一本书，本章只能给你一个具体的窥见。

这一章不奢望覆盖 PDE 数值方法（光这一题就够再写一本书）。目标更朴素：让你看到本书前面的求根、插值、积分、线性系统、ODE 是怎么合体解决一个真实科学计算任务的；让你不害怕下次见到 PDE 时去打开 SciPy 或 PETSc。

24.1 三种经典 PDE

整个 PDE 数值方法的训练，95% 的力气花在三种"原型"方程上。它们不是因为简单——是因为所有更复杂的物理模型在数值算法的层面都还原成它们的某种组合：

抛物型——热方程：

\[ \frac{\partial u}{\partial t} = \alpha\, \Delta u, \qquad u(\mathbf x, 0) = u_0(\mathbf x). \]

\(\alpha > 0\) 扩散系数。物理：温度分布、化学浓度扩散、Black-Scholes 期权定价（本质也是扩散）。时间是单向的——解越来越光滑，朝平衡态扩散。

双曲型——波动方程：

\[ \frac{\partial^2 u}{\partial t^2} = c^2 \Delta u. \]

\(c > 0\) 波速。物理：声波、电磁波、弹性振动。时间是可逆的——能量守恒，解中的高频结构永远在传播，不会"抹平"。

椭圆型——Laplace / Poisson 方程：

\[ -\Delta u = f(\mathbf x). \]

没有时间。物理：稳态温度分布、静电势、流体的稳态压力。本质是边值问题——给区域边界值，求内部分布。

这三类 PDE 数值上的"味道" 完全不同：抛物型刚性强（用隐式 + BDF），双曲型不刚性但要小心数值耗散（CFL 条件 + 高阶迎风），椭圆型是大型稀疏线性方程组（CG / 多重网格 / FFT 各显神通）。这一章主要看抛物型和椭圆型，双曲型留给 PDE 专门教材。

24.2 离散化：把连续问题变成离散问题

"有限差分" 的核心思想就一行：用第 13 章的差商代替导数。

选定一组离散点——在空间上 \(x_j = j \cdot \Delta x\)（\(j = 0, 1, \ldots, M\)），在时间上 \(t_n = n \cdot \Delta t\)（\(n = 0, 1, \ldots, N\)）——把真解 \(u(x_j, t_n)\) 用近似 \(U_j^n\) 代替。每个 \(U_j^n\) 是一个数。整个解就是一张 \((M+1) \times (N+1)\) 的数表。

对一阶导数最常见的差商：

\[ \partial_t u(x_j, t_n) \approx \frac{U_j^{n+1} - U_j^n}{\Delta t} \quad (\text{前向，}\, O(\Delta t)). \]

对二阶空间导数（中心差分）：

\[ \partial_x^2 u(x_j, t_n) \approx \frac{U_{j+1}^n - 2 U_j^n + U_{j-1}^n}{\Delta x^2} \quad (O(\Delta x^2)). \]

把这些代到 PDE 里，就把 PDE 变成了一组关于 \(\{U_j^n\}\) 的代数 / 递推方程。下面看具体例子。

24.3 1D 热方程的显式格式

取 1D 热方程 \(u_t = \alpha u_{xx}\) 在 \(x \in [0, 1], t \in [0, T]\)，边界 \(u(0, t) = u(1, t) = 0\)，初值 \(u(x, 0) = u_0(x)\)。

把 \(\partial_t\) 用前向差分、\(\partial_x^2\) 用中心差分代入：

\[ \frac{U_j^{n+1} - U_j^n}{\Delta t} = \alpha \cdot \frac{U_{j+1}^n - 2 U_j^n + U_{j-1}^n}{\Delta x^2}. \]

整理（记 \(r := \alpha \Delta t / \Delta x^2\)）：

\[ \boxed{\ U_j^{n+1} = U_j^n + r\, (U_{j+1}^n - 2 U_j^n + U_{j-1}^n).\ } \]

这是一条显式 递推：右边只依赖第 \(n\) 层的 \(U\)；可以从 \(U^0\)（初值）一层一层推到 \(U^N\)。每个时间步 \(O(M)\) 个加法乘法——便宜。

试一下。取 \(\alpha = 1\), \(M = 50\), \(\Delta x = 0.02\), \(\Delta t = 10^{-4}\)（所以 \(r = 0.25\)），初值 \(u_0(x) = \sin(\pi x)\)。

真解 \(u(x, t) = e^{-\pi^2 t} \sin(\pi x)\)（这道方程的本征函数解）。在 \(t = 0.1\) 时 \(u \approx e^{-0.987} \sin(\pi x) \approx 0.373 \sin(\pi x)\)。

跑显式格式，\(t = 0.1\) 时 \(U_{25}^{1000} \approx 0.373\)——几乎和真值一致。这事行。

现在把 \(\Delta t\) 加大到 \(2 \times 10^{-4}\)（\(r = 0.5\)）：仍然行。再加大到 \(3 \times 10^{-4}\)（\(r = 0.75\)）：第几十步开始，\(U_j^n\) 出现高频锯齿震荡，振幅每步翻倍，迅速 NaN。显式格式爆了。

24.4 CFL 条件：热方程版

"r = 0.5 行、r = 0.75 不行" 这件事不是巧合——它有一个具体的边界。把上面的递推视为离散傅里叶模式上的递推（每个空间频率独立演化），可以严格证明：

\[ \boxed{\ \text{显式 1D 热方程稳定} \iff r = \frac{\alpha \Delta t}{\Delta x^2} \le \frac{1}{2}.\ } \]

这是热方程的CFL 条件（Courant-Friedrichs-Lewy，1928）。CFL 是 PDE 数值方法的"代际" 概念，所有显式格式都有它的某个版本。

CFL 在热方程下读起来令人绝望：\(\Delta t \le \Delta x^2 / (2\alpha)\)。时间步必须随空间步的平方 缩小。把 \(\Delta x\) 减半（让空间分辨率翻倍），\(\Delta t\) 必须缩到 \(1/4\)；总工作量 = 时间步数 × 每步空间运算 = \(4 \times 2 = 8\) 倍。把 \(\Delta x\) 缩 10 倍 → 工作量 1000 倍。显式热方程在细网格上几乎不可用。

把这件事翻译回上一章语言：1D 热方程的有限差分把 PDE 变成了大型 ODE 系统 \(\dot{\mathbf U} = A \mathbf U\)，其中 \(A\) 是三对角矩阵（每行 \(\frac{\alpha}{\Delta x^2}(1, -2, 1)\)）。\(A\) 的特征值范围 \([-4\alpha/\Delta x^2, 0]\)——最大 \(|\lambda| \sim 4\alpha/\Delta x^2\)，这就是显式 Euler 的稳定性步长被 \(\Delta x^{-2}\) 拖死的源头。

"把 PDE 变成 ODE 然后套显式 Euler" 等于把上一章的"刚性问题" 自动召唤出来。CFL 条件其实就是刚性 ODE 对显式方法步长上限 的一个版本——只是这次刚性是由空间离散化制造出来的，而不是物理本来有的。

修法：用隐式。和上一章完全一样的思路。

24.5 Crank-Nicolson：每步解三对角系统

Crank-Nicolson 格式（1947）把空间二阶差分在 \(t_n\) 和 \(t_{n+1}\) 上各占一半权重：

\[ \frac{U_j^{n+1} - U_j^n}{\Delta t} = \frac{\alpha}{2 \Delta x^2}\Big[(U_{j+1}^{n+1} - 2 U_j^{n+1} + U_{j-1}^{n+1}) + (U_{j+1}^n - 2 U_j^n + U_{j-1}^n)\Big]. \]

整理出隐式格式：左边只剩 \(U^{n+1}\)，右边只剩 \(U^n\)：

\[ - \tfrac{r}{2} U_{j-1}^{n+1} + (1 + r) U_j^{n+1} - \tfrac{r}{2} U_{j+1}^{n+1} = \tfrac{r}{2} U_{j-1}^n + (1 - r) U_j^n + \tfrac{r}{2} U_{j+1}^n. \]

每个时间步要解一个三对角线性方程组——左边的系数矩阵是固定的三对角。这正好是第 16 章 16.3 节末尾提到的"三对角系统专用算法"（Thomas 算法）能 \(O(M)\) 解的情形。

Crank-Nicolson 是无条件稳定的——任何 \(r > 0\) 都行。\(\Delta t\) 不再被 \(\Delta x^2\) 卡死。它还是二阶时间精度（比 backward Euler 的一阶强）。对扩散方程几乎是默认选择。

请你停下来回头看：本章 24.3 → 24.4 → 24.5 重演了上一章 23.1 → 23.4 → 23.5 的剧情——显式 → CFL/刚性卡死 → 隐式 + 解线性系统脱困。这是所有时间相关 PDE 数值方法的同一首歌。每次只换乐器（不同的空间离散化、不同的隐式算子）。

24.6 FFT：一个算法奇迹

切换主题。离散 Fourier 变换（DFT）把长度 \(n\) 序列 \(\{x_0, x_1, \ldots, x_{n-1}\}\) 映成另一长度 \(n\) 序列：

\[ X_k = \sum_{j=0}^{n-1} x_j\, \omega^{jk}, \qquad \omega = e^{-2\pi i / n}, \quad k = 0, 1, \ldots, n-1. \]

这是一个 \(n \times n\) 的矩阵-向量乘——朴素直接做 \(O(n^2)\) 次乘法。\(n = 10^6\) 时是 \(10^{12}\) 操作——几小时。

1965 年 Cooley 和 Tukey 发表的快速 Fourier 变换（FFT）把这件事降到 \(O(n\log n)\)。\(n = 10^6\) 时变成 \(2 \times 10^7\) 操作——半秒。同样的数学结果，速度差 50000 倍。

核心想法（"分治" 在傅里叶上）：把长度 \(n\) 的 DFT 分成两个长度 \(n/2\) 的 DFT——一个对偶下标 \(\{x_0, x_2, x_4, \ldots\}\)，一个对奇下标 \(\{x_1, x_3, x_5, \ldots\}\)。具体推导：

\[ X_k = \sum_{\text{偶 } j} x_j \omega^{jk} + \sum_{\text{奇 } j} x_j \omega^{jk}. \]

设 \(j = 2m\) 和 \(j = 2m + 1\)，注意 \(\omega^{2mk} = (\omega^2)^{mk} = e^{-2\pi i \cdot mk / (n/2)}\)——这是长度 \(n/2\) 的根！

所以

\[ X_k = \underbrace{\sum_{m=0}^{n/2 - 1} x_{2m} (\omega^2)^{mk}}_{=: E_k} + \omega^k \cdot \underbrace{\sum_{m=0}^{n/2 - 1} x_{2m+1} (\omega^2)^{mk}}_{=: O_k}. \]

\(E_k\) 和 \(O_k\) 各自是长度 \(n/2\) 的 DFT。所以一个长度 \(n\) DFT = 两个长度 \(n/2\) DFT + \(n\) 次合并乘加。

设 \(T(n)\) 是长度 \(n\) DFT 的乘法次数。递推 \(T(n) = 2 T(n/2) + O(n)\)。Master 定理给 \(T(n) = O(n \log n)\)。

这个分治算法本身从 Gauss 1805 的笔记里就有（解决另一个具体计算问题），但被忽略了 160 年——因为 1805 年没有计算机，没人在乎"同一计算的效率差"。直到电子计算机普及，"同一答案的两种算法在硬件上速度差 5 万倍" 才成了能改变世界的事。Cooley-Tukey 1965 的论文被认为是 20 世纪十大算法之一。

24.7 FFT 应用一：周期 Laplace 在频域是除法

现在 FFT 怎么解 PDE？最干净的例子：周期边界条件下的 1D Poisson 方程

\[ -u''(x) = f(x), \quad x \in [0, 2\pi], \quad u(0) = u(2\pi),\ u'(0) = u'(2\pi). \]

把 \(u, f\) 都展成 Fourier 级数 \(u(x) = \sum_k \hat u_k e^{i k x}\)、\(f(x) = \sum_k \hat f_k e^{i k x}\)。代到方程：\(-(ik)^2 \hat u_k = \hat f_k\)，即

\[ \hat u_k = \frac{\hat f_k}{k^2} \quad (k \ne 0). \]

\(k = 0\) 模式（常数项）需要相容性条件 \(\hat f_0 = 0\)（积分平均为 0），且 \(\hat u_0\) 由附加条件确定。

翻译成离散：在 \(M\) 个均匀网格点采样 \(f_j = f(x_j)\)；FFT 算 \(\hat f_k\)；逐点除以 \(k^2\) 得 \(\hat u_k\)；逆 FFT 算回 \(u_j\)。整道 Poisson 方程的求解 = 一次 FFT + 一次除法 + 一次逆 FFT，\(O(M \log M)\)。

对比：用第 16 章 LU 直接解\(M \times M\) 三对角系统是 \(O(M)\)（看似更快）；但2D Poisson 在 \(M \times M\) 网格用 LU 是 \(O(M^4)\)（带状）甚至更糟，FFT 解法是 \(O(M^2 \log M)\)——后者快得多。

更进一步：FFT 方法对常系数 + 周期边界 的椭圆问题是最快 已知方法。代价是对边界条件挑食——非周期边界要换成离散余弦 / 正弦变换（DCT / DST），仍然 \(O(n \log n)\) 但要小心。一般边界条件（Dirichlet / Neumann 混合）就只能回到 LU 或迭代法。

24.8 FFT 应用二：卷积的 O(n log n) 算法

更广为人知的 FFT 应用是卷积加速。两个长度 \(n\) 序列的卷积

\[ (x * y)_k = \sum_j x_j\, y_{k-j} \]

朴素算 \(O(n^2)\)。但卷积定理 说：

\[ \widehat{x * y} = \hat x \cdot \hat y \quad (\text{逐点乘法}). \]

所以 \(x * y\) = 逆 FFT(FFT \(x\) 逐点乘 FFT \(y\))。三次 FFT + \(n\) 次乘 = \(O(n \log n)\)。

实战意义：

  • 大整数乘法：把整数当系数序列，乘法 = 卷积 → FFT。这是 GMP 等大数库背后的算法。
  • 多项式乘法：同上，多项式 ≡ 系数序列。
  • 卷积神经网络里大卷积核：用 FFT 比直接滑窗快。
  • 信号处理：滤波器 = 输入 \(*\) 滤波核——大滤波器用 FFT。
  • 偏微分方程的谱方法：用 FFT 在频域算非线性项的乘积。

这种"把卷积变乘法" 的能力是为什么 FFT 渗透到几乎所有计算密集应用。NumPy 的 np.fft.fft、SciPy 的 scipy.fft、专业库 FFTW 都给你高度优化的实现——你只管调用。理解 24.6 节的分治原理是为了让你知道它能这么快不是黑魔法，而是个具体的代数恒等式。

24.9 终章：你现在能做什么

这本书到此结束。回头看你走过的路：

  • 第 1-5 章：浮点数、误差、条件数、稳定性。基础语法。
  • 第 6-8 章：求根。一维 + 多维，二分 + 牛顿 + 拟牛顿。
  • 第 9-11 章：插值。Lagrange、Chebyshev 节点、样条。
  • 第 12 章：最小二乘 + 正交多项式。
  • 第 13-15 章：数值微分 + 积分（Newton-Cotes、Gauss、Romberg、自适应）。
  • 第 16-21 章：数值线性代数。LU / QR / 迭代 / Krylov / 特征值 / SVD。
  • 第 22-23 章：ODE。Euler、RK4、多步法、刚性、隐式。
  • 第 24 章（你正在读的）：PDE 入门 + FFT。

每一块单独看都不大——一个具体算法，一个稳定性分析，一两道实例。但合起来你学的是一种姿势：把连续数学翻译成离散计算的姿势。这种姿势在每个数学领域里都换形式重现：求根 = 把"\(f(x) = 0\) 这道连续方程的解"翻译成"一个迭代序列的极限"；积分 = 把"\(\int f\) 这个连续量"翻译成"有限个 \(f\) 取值的加权和"；ODE = 把"\(y(t)\) 这条连续曲线"翻译成"离散时刻的数表"；PDE = 同上，再加上空间离散化。

每次翻译都伴随两个问题：这道翻译有多准（精度、收敛阶），这道翻译有多稳（对扰动放大多少、对舍入放大多少）。这两个问题就是数值方法的整个智力工作。

你已经看过很多次"同一道数学题，两个代数等价的算法，结果天差地别"的演示。第 1 章求根公式抵消、第 4 章病态线性系统、第 12 章正规方程 vs QR、第 23 章显式 vs 隐式 ODE——这些反复出现的对比不是个别例子，是规律：代数等价不蕴含数值等价。这条话是这本书最重要的一句。

你下一步该做什么？这本书是地图，不是使用手册。要让它真正变成你的工具，你需要：

  • 动手做一个真问题。挑一个你自己感兴趣的：拟合一组实验数据；解你专业里某个微分方程；分析一段音频；训练一个简单回归模型。把本书的工具用到那个问题上，碰真壁，再回来查相应章节。这本书的章节都是可以独立读的，每次复习一节就够。
  • 看一两本进阶专著。Trefethen-Bau《Numerical Linear Algebra》是数值线性代数最好的入门；Trefethen《Approximation Theory and Approximation Practice》把插值/逼近讲到艺术品级；Hairer-Norsett-Wanner《Solving ODEs I/II》是 ODE 的圣经；LeVeque《Finite Difference Methods for ODEs and PDEs》是 PDE 入门首选。
  • 读真实的库源码。LAPACK / SciPy / SUNDIALS / FFTW 都是开源的。打开 scipy/integrate/_ivp/bdf.py 一类文件，你会看到本书第 23 章讲的多步法 + 牛顿 + 步长控制全在那里——但被工程化打磨过几十年。读这种代码会教你"书上的算法" 和"能在 GitHub 上跑的代码" 之间的距离。

最后一段话留给我自己想说的：数值方法常被当成"工程师才学的东西"——纯数学家不屑，应用领域学生当成必经之恶。这两种态度都让人错过它的真价值。数值方法是连续数学的具身：是把抽象概念按到一台真物理机器上的过程。在这个过程中，连续数学的每个定理都被重新审问"它到底意味着什么、能算到什么程度"。这种审问会让你对数学的理解比单纯证明定理更深一层——你会知道为什么 这条定理重要，因为你看见过没有它时算法怎么爆炸。

这就是"把数学翻译成能跑"——副标题的意思。希望这本书让你具备了那种翻译能力，也希望它让你对数学的看法变得更具身 一点。

下一步在你手上。去做一个真问题。
