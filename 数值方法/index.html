普林斯顿数值方法读本

浮点、误差、解方程、插值、数值线性代数、ODE 与 PDE · 把"数学"翻译成"能跑"

前言

欢迎你翻开这本书。和系列前面九本不同，这一本要讲的不是"数学是什么"，是"当你拿一台真实的电脑去做数学时，世界变成什么样"。两边的差距没你以为的小。第 1 章你会看到三个例子，让你立即感受到这种差距是结构性的，不是工程瑕疵。

这本书有一个很简单的主张：每一个连续数学定理背后，都站着一个数值算法；而那个数值算法常常比定理本身更需要思考。纯数学家说"\(\int_0^1 e^{-x^2} \, dx\) 的值存在"——这是真的。数值人要做的是：把它算到 16 位精度，在 1 微秒内，并且能证明这 16 位都是对的。这件事不容易；这件事就是这本书。

这本书的角色

系列前面九本（多元微积分、线性代数、微分方程、复分析、概率统计、偏微分方程、实分析、泛函分析、抽象代数）讲的是连续数学。它们都假定你能"算"——梯度、积分、矩阵逆、解 ODE——但从不正面问"算"是什么意思。这本书把"算"打开。

核心命题：算 ≠ 写公式。给一个数学对象（一个积分、一个方程、一个矩阵），有许多种算法能把它变成一个数。这些算法在纸上常常等价；在电脑上不等价。它们各有各的速度、稳定性、存储代价、并行度。挑哪个不是品味问题——是技术问题，有诊断、有定理。

这本书不是什么

它不是编程书。我会用 Python / MATLAB / NumPy 的语义说话，但不会教你"怎么用 NumPy 的 API"。你想学具体语言，去看官方文档或专门的库教程。

它不是软件工程书。我不讲版本管理、性能调优、GPU 编程。这些都重要，但不是本书的范围。

它不是数值分析的研究专著。本书覆盖的是本科 + 研究生预备阶段的标准数值方法——求根、插值、积分、线性代数、ODE、入门级 PDE 与 FFT。深一点的话题（自适应网格、辛积分、谱方法、稀疏直接法）只一句话提及。

读这本书之前你需要什么

你需要多元微积分的基本词汇——梯度、雅可比、Taylor 展开。第 7、8、20 章会大量用。

你需要线性代数，特别是矩阵分解、特征值、范数。第 16-21 章基本就是"线性代数的数值版"。

你最好读过微分方程的初阶部分（Euler 方法、解曲线的稳定性）。第 22-23 章会接着讲。

你不需要先学过实分析或泛函分析。本书会在用到的地方说清。如果你后面想深入误差分析的"为什么"——比如 backward error 的几何刻画——可以回头看泛函分析第 8-9、第 17-18 章。

你需要的更像一种态度：愿意开一个解释器一边读一边动手。这本书的几乎每一节都有可以五行代码验证的例子。把这些例子真敲一遍，比读十遍证明都管用。数值方法是一门实验科学——你看到的每一个结论，都可以也应当被你自己跑一次。

"我下周就要考期中，从哪里开始？"

看你考什么。绝大多数本科数值课考的是这几块：

  • 误差分析与条件数（第 3-5 章）；
  • 求根：二分 + 牛顿（第 6-7 章）；
  • 插值与最小二乘（第 9-12 章）；
  • 数值积分：梯形 + Simpson + Gauss（第 14-15 章）；
  • 数值线性代数：LU、QR、迭代法（第 16-19 章）；
  • ODE 数值解：RK4 + 稳定性（第 22-23 章）。

挑你课重的那几块直接看，每节里的例子都自己敲一遍，绝大多数考试 80% 的题型就稳了。

例题与代码片段

例题埋在正文里，没有"例 4.2.1"这种标签。每个例题都先于一般化结论出现——你先看到一个具体数会怎么爆炸，然后才看到"我们把这种现象叫做 X"。

代码片段用伪代码 + Python-like 语法。我会偶尔写真实的 Python 一行（特别在前几章演示浮点行为），但本书不依赖任何具体语言。你能写出 for 循环 + if 判断 + 函数调用就够了。

  5. 第 5 章 算法稳定性：实现的"后天好坏" — 同一个问题，算法可以稳或不稳。
  6. 第 6 章 一维方程求根：二分与不动点 — 最朴素的求根算法。
  7. 第 7 章 牛顿法与割线法 — 二次收敛的代价与好处。
  8. 第 8 章 多变量牛顿与拟牛顿法 — 把 1D 牛顿推到 \(\mathbb{R}^n\)。

  13. 第 13 章 数值微分 — 为什么数值导数总比数值积分难。

  16. 第 16 章 直接法解线性系统：LU 与 Cholesky — 高斯消元的工程化。
  17. 第 17 章 QR 分解与最小二乘的稳定算法 — Householder 反射。
  18. 第 18 章 迭代法：Jacobi、Gauss-Seidel、SOR — 当矩阵太大不能直接解。
  19. 第 19 章 Krylov 子空间与共轭梯度 — 现代大规模求解的工作马。

  21. 第 21 章 奇异值分解 — 为什么 SVD 是数值线性代数最有用的工具。
  22. 第 22 章 ODE 数值解：Euler 与 Runge-Kutta — 离散化时间的入门。
  23. 第 23 章 ODE：多步法、稳定性、刚性方程 — 为什么 RK4 不总能用。
  24. 第 24 章 PDE 有限差分入门 + FFT 一瞥 — 把工具汇合。
