B.1 复函数解析（全纯）

设 \(U \subseteq \mathbb C\) 是开集、\(f : U \to \mathbb C\)。\(f\) 在点 \(z_0 \in U\) 处全纯（holomorphic at \(z_0\)） \(\iff\) 复导数

\[ f'(z_0) := \lim_{h \to 0, h \in \mathbb C} \frac{f(z_0 + h) - f(z_0)}{h} \]

存在并不依赖 \(h \to 0\) 的方向。这条"不依赖方向" 是实可微 与复可微 的关键差别——\(f(z) = \bar z\) 在每个点实可微但不复可微。

\(f\) 在 \(U\) 上全纯 \(\iff\) \(f\) 在 \(U\) 每点全纯。本书全纯、解析、complex differentiable 这些词同义。

等价表征。\(f : U \to \mathbb C\) 全纯 \(\iff\) 满足以下任一（实际四个等价）：

  1. 复可微。
  2. 满足 Cauchy-Riemann 方程：\(f = u + iv\)，则 \(\partial u/\partial x = \partial v/\partial y\)、\(\partial u/\partial y = -\partial v/\partial x\)。
  3. 每点局部展开为收敛幂级数：\(f(z) = \sum_{n=0}^\infty a_n (z - z_0)^n\) 在 \(z_0\) 的某邻域。
  4. 满足 Cauchy 积分公式（B.2 节）：\(f(z_0) = \frac{1}{2\pi i} \oint_\gamma \frac{f(z)}{z - z_0}\,dz\) 对包围 \(z_0\) 的简单闭曲线 \(\gamma\)。

"幂级数 = 全纯" 是实分析与复分析最深的差别之一——实数上 \(f \in C^\infty\) 不 蕴含 \(f\) 是解析（如 \(f(x) = e^{-1/x^2}\) for \(x \ne 0\) 是 \(C^\infty\) 但 Taylor 级数 \(0\)，不等于 \(f\)）。复数上"一阶可微" 自动给"无穷阶可微" + "解析"。

B.2 Cauchy 积分公式 + 留数定理

Cauchy 积分定理。设 \(f\) 在简单连通开集 \(U \subseteq \mathbb C\) 上全纯，\(\gamma\) 是 \(U\) 中分段光滑闭曲线。则

\[ \oint_\gamma f(z) \, dz = 0. \]

这是复分析最基础的"积分恒等于零" 公式。直觉：全纯函数没有奇点 让闭路径环绕——所以闭路径积分回到原点。

Cauchy 积分公式。同上设 + \(z_0 \in U\) 在 \(\gamma\) 内部。则

\[ \boxed{\ f(z_0) = \frac{1}{2\pi i} \oint_\gamma \frac{f(z)}{z - z_0} \, dz.\ } \]

这条公式让你从 \(\gamma\) 上的值 算出\(\gamma\) 内部任意点 的值。复全纯函数有"极强的全局刚性"——边界数据完全决定内部数据。

留数。设 \(f\) 在 \(z_0\) 处有孤立奇点（即 \(z_0\) 周围去掉 \(z_0\) 的某邻域 \(f\) 全纯）。\(f\) 在 \(z_0\) 周围有 Laurent 展开：

\[ f(z) = \sum_{n = -\infty}^{\infty} c_n (z - z_0)^n. \]

\(z_0\) 处的留数（residue）= Laurent 展开里 \((z - z_0)^{-1}\) 的系数：

\[ \boxed{\ \mathrm{Res}_{z_0} f := c_{-1}.\ } \]

留数测量 \(f\) 在 \(z_0\) 处"第一阶奇异性" 的大小。

留数定理。设 \(f\) 在 \(U\) 上除有限个孤立奇点 \(z_1, \ldots, z_k\) 外全纯，\(\gamma\) 是 \(U\) 中分段光滑简单闭曲线，包围所有 \(z_j\)（按正向）。则

\[ \boxed{\ \oint_\gamma f(z) \, dz = 2\pi i \sum_{j = 1}^{k} \mathrm{Res}_{z_j} f.\ } \]

留数定理是整个解析数论 的工程引擎——\(\zeta\) 函数与 \(L\)-函数的"奇点贡献" 通过这条公式翻译成数论信息。ch16素数定理证明的核心是研究 \(\zeta(s)\) 在临界线 \(\Re s = 1\) 上的留数行为。

计算简单极点的留数：若 \(z_0\) 是 \(f\) 的简单极点（即 Laurent 展开 \(c_{-1}/(z - z_0) +\) 全纯部分），则

\[ \mathrm{Res}_{z_0} f = \lim_{z \to z_0} (z - z_0) f(z). \]

例：\(f(z) = \zeta(s)/s\) 在 \(s = 1\) 有简单极点（\(\zeta\) 在 \(s = 1\) 极点），\(\mathrm{Res}_{s = 1} \zeta(s)/s = 1\)。这是素数定理证明的关键残留留数之一。

B.3 解析延拓

解析延拓的唯一性。设 \(f, g\) 在连通开集 \(U\) 上都全纯，\(f = g\) 在 \(U\) 的某有非孤立点的子集（如某条曲线、某连通开集、或某带极限点的序列）。则 \(f \equiv g\) 在整个 \(U\)。

这条"唯一性" 让"解析延拓" 的概念合法：如果你把 \(f\) 从某小区域延拓到大区域，只要全纯，延拓就是唯一的——不存在两个不同的延拓。

具体延拓技术。给定 \(f\) 在某区域 \(U\) 全纯，怎么把它延拓到更大区域 \(\tilde U \supset U\)？

  • 幂级数延拓：在 \(U\) 内取一点 \(z_0\)，把 \(f\) 在 \(z_0\) 展开为幂级数 \(\sum a_n (z - z_0)^n\)。这个级数的收敛半径 \(R\) 给一个圆盘 \(|z - z_0| < R\)，可能超出 \(U\)。在新圆盘内，幂级数定义 \(f\) 的延拓。
  • 函数方程：找一条等式 \(f(z) = G(f(\sigma(z)))\)，其中 \(\sigma\) 把 "未定义区域" 映到"已定义区域"。例：Riemann 函数方程 \(\zeta(s) = 2^s \pi^{s-1} \sin(\pi s/2) \Gamma(1-s) \zeta(1-s)\) 把 \(\Re s < 0\) 区域的 \(\zeta\) 用 \(\Re s > 1\) 区域的 \(\zeta\) 表达——延拓到所有 \(\mathbb C \setminus \{1\}\)。
  • 积分变换：用如 \(\Gamma(s) = \int_0^\infty t^{s-1} e^{-t} dt\) 这种积分定义函数。这种积分在某些 \(s\) 区域绝对收敛、定义全纯函数；用积分的代数恒等式（部分积分等）把定义延伸。

例 1：Riemann zeta 函数。最初定义：

\[ \zeta(s) := \sum_{n=1}^{\infty} \frac{1}{n^s}. \]

右边在 \(\Re s > 1\) 绝对收敛、定义 \(\zeta(s)\) 为全纯 函数。但真 \(\zeta\) 是延拓到整个 \(\mathbb C\)（除 \(s = 1\) 外，\(s = 1\) 是简单极点）的全纯函数。延拓技术：先用 Euler-Maclaurin 求和把 \(\zeta\) 的定义延拓到 \(\Re s > 0\)，再用 Riemann 函数方程延拓到 \(\Re s < 0\)。

例 2：Gamma 函数。最初定义：

\[ \Gamma(s) := \int_0^\infty t^{s-1} e^{-t} \, dt. \]

右边在 \(\Re s > 0\) 绝对收敛。用部分积分得 \(\Gamma(s + 1) = s \Gamma(s)\)——这条函数方程用来延拓：\(\Gamma(s) = \Gamma(s + 1)/s\) 把 \(\Gamma\) 延拓到 \(\Re s > -1\)（除 \(s = 0\) 外），再迭代延拓到全 \(\mathbb C \setminus \{0, -1, -2, \ldots\}\)。在 \(s = 0, -1, -2, \ldots\) 处 \(\Gamma\) 有简单极点。

这两个例子是解析数论里最重要的两个全纯函数——它们的延拓性质决定了"素数分布" 与"L-函数零点" 的精细结构。

B.4 全纯函数的零点

设 \(f\) 在连通开集 \(U\) 全纯、不恒为零。则 \(f\) 的零点集是离散 的——每个零点 \(z_0 \in U\) 都有一个邻域，里面除 \(z_0\) 外没有其他零点。

这条"零点孤立" 是 B.3 节"延拓唯一性" 的特殊情况——若 \(f\) 在某个有极限点的集上为零，则恒为零。

零点的阶。设 \(f(z_0) = 0\)、\(f \not\equiv 0\)。\(f\) 在 \(z_0\) 处的幂级数

\[ f(z) = \sum_{n \ge 0} a_n (z - z_0)^n \]

有 \(a_0 = 0\)。\(f\) 在 \(z_0\) 的零点阶（order of zero） = 满足 \(a_m \ne 0\) 的最小 \(m \ge 1\)。零点阶 \(m \iff f(z) = (z - z_0)^m g(z)\)，\(g(z_0) \ne 0\) 全纯。

幅角原理。设 \(f\) 在 \(\overline{\Omega}\)（\(\Omega\) 是简单连通区域）全纯且无零点在边界 \(\partial \Omega\)。则

\[ \frac{1}{2\pi i} \oint_{\partial \Omega} \frac{f'(z)}{f(z)} \, dz = N, \]

其中 \(N\) = \(f\) 在 \(\Omega\) 内零点的带阶 数（每个零点按它的阶计数）。

这条"积分计零点" 公式是ch16"临界条带 \(0 < \Re s < 1\) 内 \(\zeta\) 的零点" 计数与ch21"模形式空间维数" 计数的工具。Riemann 假设的精确陈述："\(\zeta\) 在临界条带的所有零点都在临界线 \(\Re s = 1/2\) 上。"

Hadamard 因子分解。一个有限阶 整函数（即 \(|f(z)| \le C e^{|z|^\rho}\) 某 \(\rho < \infty\)）可以写成"无穷乘积" 形式：

\[ f(z) = z^m e^{P(z)} \prod_{n \ge 1} E_p\left(\frac{z}{a_n}\right), \]

其中 \(\{a_n\}\) 是非零零点、\(P\) 是 \(\rho\) 次以下多项式、\(E_p\) 是收敛因子。

这条把"整函数" 类似多项式那样分解成"线性因子的乘积"——只是有无穷多线性因子加上一个收敛修正。Riemann 用它表达 \(\xi(s) := \frac{1}{2} s(s-1) \pi^{-s/2} \Gamma(s/2) \zeta(s)\)（"Riemann xi 函数"，是 \(\zeta\) 的整函数补充），把 \(\zeta\) 的所有非平凡零点显式作为"线性因子" 列出。

B.5 Tauberian 定理

"Tauberian 定理"（Tauberian theorem）= 一类把"渐近 \(L\)-函数行为" 翻译成"渐近系数和" 的定理。它们是素数定理证明 的"分析机"。

简化的Wiener-Ikehara Tauberian 定理（1932）：

定理。设 \(F(s) = \sum_{n \ge 1} a_n n^{-s}\) 是非负 系数的 Dirichlet 级数，在 \(\Re s > 1\) 收敛，并且 \((s - 1) F(s)\) 在 \(\Re s \ge 1\) 上延拓为连续函数，且

\[ \lim_{s \to 1^+} (s - 1) F(s) = c. \]

则

\[ \boxed{\ \sum_{n \le x} a_n \sim c x \quad \text{as} \quad x \to \infty.\ } \]

翻译："\(F(s)\) 在 \(s = 1\) 的留数 是 \(c\)" \(\implies\) "\(\sum a_n\) 增长是 \(\sim cx\)"。

这条定理把"解析事实"（\(F\) 在 \(s = 1\) 的留数）翻译成"组合事实"（系数和的渐近）。

素数定理的证法。取 \(F(s) = -\zeta'(s)/\zeta(s)\)（zeta 的对数导数）。它的 Dirichlet 展开

\[ -\frac{\zeta'(s)}{\zeta(s)} = \sum_{n \ge 1} \Lambda(n) n^{-s}, \]

其中 \(\Lambda(n) = \log p\) 当 \(n = p^k\)、否则 \(0\)（"von Mangoldt 函数"）。\(\zeta(s)\) 在 \(s = 1\) 简单极点 \(\Rightarrow\) \(F(s) = -\zeta'/\zeta\) 在 \(s = 1\) 简单极点，留数 \(1\)。

关键事实——"\(\zeta(1 + it) \ne 0\) 对实 \(t \ne 0\)"（这是 1896 Hadamard 与 de la Vallée-Poussin 的核心证明步骤）——保证 Wiener-Ikehara 的"\((s-1) F(s)\) 在 \(\Re s = 1\) 连续" 假设满足。套定理得

\[ \sum_{n \le x} \Lambda(n) \sim x. \]

由部分求和（Abel 求和）从 \(\Lambda\) 渐近推出 \(\pi(x) \sim x/\log x\)——素数定理。

这条证法是解析数论 的基本范式：

  1. 把数论问题翻译成 Dirichlet 级数 \(F(s)\)。
  2. 研究 \(F(s)\) 的解析性质（极点、留数、零点）。
  3. 用 Tauberian 定理把解析行为翻译回数论渐近。

本书ch16整章在做这件事的具体细节。

B.6 Dirichlet 级数收敛半平面

Dirichlet 级数：

\[ F(s) = \sum_{n=1}^{\infty} \frac{a_n}{n^s}, \quad a_n \in \mathbb C, \ s \in \mathbb C. \]

这与 \(\zeta\)（\(a_n \equiv 1\)）、L-级数 \(L(s, \chi) = \sum \chi(n)/n^s\)（\(\chi\) 是 Dirichlet 特征，链回 ch15）、模形式 \(L\)-函数 \(L(f, s) = \sum a_n(f)/n^s\)（链回ch21）—— 都是 Dirichlet 级数。

定理（绝对收敛半平面）。\(\sum |a_n|/n^s\) 收敛 \(\iff \Re s > \sigma_a\)（"绝对收敛横坐标"）。

具体的：\(\sigma_a = \limsup_{N \to \infty} \log\left(\sum_{n=1}^N |a_n|\right)/\log N\)。

定理（条件收敛半平面）。\(\sum a_n/n^s\) 收敛 \(\iff \Re s > \sigma_c \le \sigma_a\)。

例：\(\zeta(s) = \sum 1/n^s\)。\(\sigma_a = \sigma_c = 1\)。在 \(\Re s > 1\) 绝对收敛。

例：交错 \(\eta(s) := \sum (-1)^{n-1}/n^s\)。\(\sigma_a = 1\)（\(|a_n| = 1\)），但 \(\sigma_c = 0\)（部分和有界）。所以 \(\eta\) 在 \(\Re s > 0\) 收敛但只 \(\Re s > 1\) 绝对收敛。\(\eta\) 与 \(\zeta\) 通过 \(\eta(s) = (1 - 2^{1-s}) \zeta(s)\) 相关——这给 \(\zeta\) 的另一种延拓 到 \(\Re s > 0\)。

例：\(L(s, \chi)\)、\(\chi\) 非主特征。\(\sum_{n \le N} \chi(n)\) 有界（"Pólya-Vinogradov 不等式"），所以 \(\sigma_c = 0\)——\(L(s, \chi)\) 在 \(\Re s > 0\) 收敛。这是ch15"Dirichlet 算术级数素定理" 的关键事实。

Euler 乘积。如果 \(a_n\) 是积性 的（\(a_{mn} = a_m a_n\) 当 \(\gcd(m, n) = 1\)），则 \(F(s)\) 在绝对收敛区域有 Euler 乘积：

\[ F(s) = \prod_p \left(\sum_{k=0}^\infty \frac{a_{p^k}}{p^{ks}}\right). \]

如果还完全积性（\(a_{mn} = a_m a_n\) 总成立），简化为

\[ F(s) = \prod_p \frac{1}{1 - a_p p^{-s}}. \]

例：

  • \(\zeta(s) = \prod_p \frac{1}{1 - p^{-s}}\)（\(a_n \equiv 1\) 完全积性）。这是 Euler 1737 的发现——把"素数" 这种离散对象与"解析函数" 联系起来的桥。
  • \(L(s, \chi) = \prod_p \frac{1}{1 - \chi(p) p^{-s}}\)（Dirichlet 1837）。
  • \(L(E, s) = \prod_p \frac{1}{1 - a_p p^{-s} + p^{1-2s}}\)（椭圆曲线 \(L\)-函数，链回ch21）。\(a_n\) 积性（这是 \(E\) 的2 维 Galois 表示 的反映）但不 完全积性，所以局部因子是 \(\frac{1}{1 - a_p p^{-s} + p^{1-2s}}\) 而非 \(\frac{1}{1 - a_p p^{-s}}\)——次数 2 而非 1。
  • \(L(f, s)\) 模形式：同上结构。这是模性定理 \(L(E, s) = L(f_E, s)\) 的解析侧。

Euler 乘积是解析数论的核心工具——它把"所有正整数的求和" 转换为"所有素数的乘积"，让素数 这种离散对象成为解析函数 \(F(s)\) 的"基本构造单元"。"每个 Euler 乘积都是一种"素数到分析函数" 的字典"——这是ch16素数定理证明、ch21模形式 \(L\)-函数、ch22Langlands 函子性都依赖的根本机制。

B.7 一句话指南

这本书用到的复分析按"抽象程度" 排：

  • 最具体：幂级数、简单极点、函数方程延拓。ch16素数定理证明里反复出现。
  • 第二层：留数定理、幅角原理、Tauberian 定理。这些是把"解析行为" 翻译成"组合渐近" 的标准工具。
  • 第三层：解析延拓、Dirichlet 级数、Euler 乘积、函数方程。这些是L-函数研究的语言。
  • 第四层：整函数的 Hadamard 因子分解、Selberg 类（"满足函数方程 + Euler 乘积 + 多项式增长" 的 \(L\)-函数族）。这是 21 世纪 \(L\)-函数研究前沿，超出本书。

解析数论与代数数论的分工：

  • 代数数论（抽象代数 + 附录 A）：研究具体数域 \(K\) 的素分解、类群、Galois 群。回答"哪些素分裂" 的精确判别。
  • 解析数论（复分析 + 本附录）：研究所有素数的渐近 行为。回答"素数有多稠密" 的统计结果。
  • 两者通过L-函数 桥接——\(L\)-函数的分析 信息（极点、零点、解析延拓）来自代数 信息（Galois 表示、自守表示、群论数据）。

21 世纪数论的核心问题（Riemann 假设、BSD、Langlands 函子性）都 在这条桥上——它们要求把代数侧的猜想性质翻译成分析侧的可验证陈述（或反之）。这本书前 24 章给的是这条桥的两个端口；走到桥的中间 需要研究生级训练 + 多年专门研究——这是本书读完后的下一段旅途。
