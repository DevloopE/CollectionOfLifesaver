第 16 章 素数定理证明思路

第 1 章给了一句话："素数定理 \(\pi(N) \sim N/\ln N\)，1896 年 Hadamard 和 de la Vallée-Poussin 同年独立证完。" 当时我让你接受这条结果。这一章我兑现承诺——给出证明的主线。

"主线" 意思是关键想法 + 关键引理，不是逐步的精确证明。完整证明（带所有解析估计）容易在哲学上把人迷晕。我们做的事是：让你看到为什么"素数分布" 和"复函数 $\zeta$ 的零点位置" 这两件看似无关的事是一回事。这条联系是 19 世纪数论最深的发现，也是上一章"解析数论" 这条路线最早的胜利果实。

大致路线图：

  • 16.1：把 \(\pi(N)\) 重新写成两个等价的求和 \(\theta, \psi\)。
  • 16.2：\(\zeta(s)\) 在 \(\Re s > 1\) 收敛、解析延拓到 \(\mathbb{C} \setminus \{1\}\)、在 \(s = 1\) 处单极点。
  • 16.3：核心引理：\(\zeta(s)\) 在 \(\Re s = 1\) 上没有零点。
  • 16.4–16.5：从 16.3 推 \(\psi(N) \sim N\)，再推 \(\pi(N) \sim N/\ln N\)。
  • 16.6：Riemann 假设——零点位置如何决定误差项。
  • 16.7：现代视角——\(\zeta\) 是 GL_1 的自守 \(L\)-函数；Langlands 哲学。

16.1 三个等价的素数定理

素数定理有三种等价陈述。以后我们证最方便的那种 (\(\psi\))，再翻译回 \(\pi\)。

原始版（Gauss）：

\[ \pi(N) = \#\{p \le N : p \text{ 素}\} \sim \frac{N}{\log N}, \quad N \to \infty. \]

"\(\sim\)" 表示 \(\lim_{N \to \infty} \pi(N) / (N / \log N) = 1\)。

Chebyshev \(\theta\) 版：

\[ \theta(N) = \sum_{p \le N} \log p \sim N. \]

每个素数贡献 \(\log p\) 而非 1。

von Mangoldt \(\psi\) 版：

\[ \psi(N) = \sum_{p^k \le N} \log p = \sum_{n \le N} \Lambda(n) \sim N. \]

这里 \(\Lambda(n) = \log p\) if \(n = p^k\) for some \(k \ge 1\)，否则 0（14.8 节引入）。\(\psi\) 把素数和素幂一起算。

三者等价：

\(\theta \sim N \Leftrightarrow \psi \sim N\)：\(\psi(N) - \theta(N) = \sum_{p^k \le N, k \ge 2} \log p\)。素方根贡献 \(\sum_{p \le \sqrt N} \log p \le \theta(\sqrt N) \le \sqrt N \cdot \log N\)（用粗估），更高次幂贡献更少。所以 \(\psi(N) - \theta(N) = O(\sqrt N \log N) = o(N)\)。差是次主项，所以两者主项一致。

\(\psi \sim N \Rightarrow \pi \sim N/\log N\)：用 Abel 求和把 \(\theta(N) = \sum_{p \le N} \log p\) 变换成 \(\pi\)：

\[ \theta(N) = \sum_{p \le N} \log p = \pi(N) \log N - \int_2^N \frac{\pi(t)}{t} dt. \]

（标准 Abel 求和。）反过来 \(\pi(N) = \theta(N)/\log N + (\text{次项})\)，所以 \(\theta(N) \sim N \Rightarrow \pi(N) \sim N/\log N\)。

三种陈述等价。我们之后只需证 \(\psi(N) \sim N\)。

为什么 \(\psi\) 最方便？因为 \(\psi\) 直接通过 \(\zeta\) 表达——14.8 节证明 \(\Lambda * \mathbf{1} = \log\)，等价 \(\sum_n \Lambda(n)/n^s = -\zeta'(s)/\zeta(s)\)（15.8 节）。这条公式让 "\(\psi\) 的渐近" 直接映到 "\(\zeta\) 的解析性质"。

16.2 \(\zeta(s)\) 的解析延拓

本节列出 \(\zeta\) 必需的复变事实，不证。完整证明属于复分析课程。

事实 1（收敛）：\(\zeta(s) = \sum_{n \ge 1} 1/n^s\) 在半平面 \(\Re s > 1\) 绝对收敛，是个全纯函数。

证：\(|1/n^s| = 1/n^{\Re s}\)；\(\sum 1/n^{\Re s}\) 在 \(\Re s > 1\) 收敛（\(p\) 级数）。所以原级数绝对收敛，每个分量是 \(s\) 的全纯函数（\(1/n^s = e^{-s \log n}\)），一致收敛保住全纯。

事实 2（解析延拓）：\(\zeta(s)\) 能延拓到整个 \(\mathbb{C} \setminus \{1\}\)，是亚纯函数。

构造方法之一：函数方程 + 反射。Riemann 1859 给的恒等式

\[ \zeta(s) = 2^s \pi^{s - 1} \sin(\pi s / 2) \Gamma(1 - s) \zeta(1 - s). \]

右边在 \(\Re s < 0\) 时由 \(\zeta(1 - s)\)（在 \(\Re(1 - s) > 1\) 即 \(\Re s < 0\) 时收敛）+ 已知 \(\Gamma\) 给出。这条函数方程让 \(\zeta\) 从 \(\Re s > 1\) 延到 \(\Re s < 0\)，中间的"临界带" \(0 \le \Re s \le 1\) 用其它方法（积分表示）填上。

事实 3（极点）：\(\zeta(s)\) 在 \(s = 1\) 处有单极点，留数 = 1。即 \(\zeta(s) \sim 1/(s - 1)\) 当 \(s \to 1\)。

这跟调和级数 \(\sum 1/n\) 在 \(s = 1\) 处的发散对应——延拓后这个发散被装成"极点"。具体：

\[ \zeta(s) = \frac{1}{s - 1} + \gamma + O(s - 1), \]

其中 \(\gamma \approx 0.5772\) 是 Euler-Mascheroni 常数。这个极点是$\zeta$ 整个故事的引擎——它对应"素数无穷多" 这条最朴素事实。

事实 4（平凡零点）：函数方程让 \(\zeta(s) = 0\) 在 \(s = -2, -4, -6, \ldots\) 处——这些叫"平凡零点"。它们由 \(\sin(\pi s / 2)\) 因子带来：\(s = -2k\) (\(k \ge 1\)) 处 \(\sin(\pi s / 2) = 0\)。

事实 5（非平凡零点）：除平凡零点外，\(\zeta\) 的所有零点都在临界带 \(0 < \Re s < 1\) 内。这些叫"非平凡零点"。

函数方程让非平凡零点关于 \(\Re s = 1/2\) 对称：\(\zeta(\rho) = 0 \Rightarrow \zeta(1 - \rho) = 0\)。

已知前几个非平凡零点：\(s = 1/2 + 14.134\ldots i\)，\(s = 1/2 + 21.022\ldots i\)，\(s = 1/2 + 25.011\ldots i\)，等等。它们已知都在临界线 $\Re s = 1/2$ 上——但"全部非平凡零点都在 \(\Re s = 1/2\) 上"未证，那是 Riemann 假设。

本章只用临界带这件事，加上下一节的"$\Re s = 1$ 上无零点"。后者是素数定理的关键。

16.3 关键引理：\(\zeta(s)\) 在 \(\Re s = 1\) 上无零点

这是整章的钉子。1896 年 Hadamard 和 de la Vallée-Poussin 同年独立给的——素数定理 的核心证明步骤。

定理：对所有实数 \(t \ne 0\)，\(\zeta(1 + it) \ne 0\)。

等价：临界带的右边界 \(\Re s = 1\) 上 \(\zeta\) 无零点（除了 \(s = 1\) 那个极点）。

证明思路（朴素三步）。

步 1：考虑 \(\Re s > 1\) 时

\[ \log \zeta(s) = -\sum_p \log(1 - p^{-s}) = \sum_p \sum_{k \ge 1} \frac{p^{-ks}}{k}. \]

（用 Euler 乘积 + \(-\log(1 - x) = \sum_{k \ge 1} x^k / k\)。）

所以

\[ \Re \log \zeta(s) = \log |\zeta(s)| = \sum_p \sum_k \frac{\cos(k t \log p)}{k p^{k \sigma}} \]

当 \(s = \sigma + it\) (\(\sigma > 1\))。$\log |\zeta|$ 是关于 cos 的求和。

步 2：神奇不等式 \(3 + 4 \cos \theta + \cos 2\theta \ge 0\)。

验证：\(3 + 4 \cos \theta + \cos 2\theta = 3 + 4 \cos \theta + 2 \cos^2 \theta - 1 = 2(1 + \cos \theta)^2 \ge 0\)。\(\checkmark\)

这条恒等式来自一个完全平方。它的关键是系数 $3, 4, 1$ 让组合成完全平方。其它系数比如 \(1, 4, 1\) 不行。

步 3：把不等式套到 \(\zeta\) 上。设 \(s = \sigma + it\)，\(\theta = -t \log p\)：

\[ 3 \log |\zeta(\sigma)| + 4 \log |\zeta(\sigma + it)| + \log |\zeta(\sigma + 2it)| = \sum_p \sum_k \frac{3 + 4 \cos(k t \log p) + \cos(2 k t \log p)}{k p^{k \sigma}} \ge 0. \]

取指数：

\[\boxed{\ |\zeta(\sigma)|^3 \cdot |\zeta(\sigma + it)|^4 \cdot |\zeta(\sigma + 2it)| \ge 1, \quad \sigma > 1, t \in \mathbb{R}.\ }\]

这就是那条神奇不等式——是整段证明的钥匙。

反证假设 \(\zeta(1 + it_0) = 0\) 对某 \(t_0 \ne 0\)。由 \(\zeta\) 全纯，靠近 \(1 + it_0\) 时 \(\zeta(s)\) 像 \((s - (1 + it_0))^m\) 衰减到 0（\(m\) 是零点阶 \(\ge 1\)）。

用 \(s = \sigma + it_0\) (\(\sigma \to 1^+\))：\(|\zeta(\sigma + it_0)|^4 \to 0\) 像 \((\sigma - 1)^{4m}\)。

同时 \(|\zeta(\sigma)|^3 \to \infty\) 像 \(1/(\sigma - 1)^3\)（极点 \(s = 1\)）。

\(|\zeta(\sigma + 2it_0)|\) 在 \(\sigma = 1\) 处有限（不在零点 / 极点处）。

组合：左边 \(\to (\sigma - 1)^{4m - 3} \cdot \text{有限}\)。如果 \(m \ge 1\)，\(4m - 3 \ge 1 > 0\)，左边 \(\to 0\)，矛盾下界 \(\ge 1\)。\(\checkmark\)

请你停下来感受一下这条证明的精神。它把"$\zeta$ 在 $\Re s = 1$ 无零点" 这个深刻事实压成一个三角恒等式 + 一条朴素 cos 不等式。"\(2(1 + \cos \theta)^2 \ge 0\)" 听起来像高中题——但它就是 19 世纪解析数论第一座大山的钥匙。

这个证法叫Mertens 不等式（虽然 Mertens 用不同形式，Hadamard 整理成现代形）。它的稳健性 让相同手法对一般 \(L(s, \chi)\) 也奏效——为 Dirichlet 等差数列素数定理 + 非主特征 \(L(1, \chi) \ne 0\) 提供一致框架。

16.4 从 16.3 推 \(\psi(N) \sim N\)

这一步是Tauberian 定理 的应用。完整证明技术性强，我们给主线。

关键工具：Wiener-Ikehara Tauberian 定理（1932）。陈述：

设 $f(t)$ 是 $[0, \infty)$ 上非负不减函数，

\[ F(s) = \int_0^\infty f(t) e^{-st} dt \]

在 $\Re s > 1$ 收敛。如果 $F(s) - 1/(s - 1)$ 在 $\Re s = 1$ 上有连续延拓（即 $\zeta$-类型函数减掉极点剩有限），那 $f(t) \sim e^t$。

把它套到我们的设置。

步 1（Mellin 变换）：

\[ -\frac{\zeta'(s)}{\zeta(s)} = \sum_n \frac{\Lambda(n)}{n^s} = s \int_1^\infty \frac{\psi(x)}{x^{s + 1}} dx. \]

（用 Abel 求和将 \(\sum \Lambda(n)/n^s\) 转成 \(\psi\) 的积分。）

令 \(x = e^t\)，\(\sigma = s\)：

\[ -\frac{\zeta'(s)}{s \zeta(s)} = \int_0^\infty \frac{\psi(e^t)}{e^{st}} dt. \]

这是 \(\psi(e^t) e^{-t}\) 的 Laplace 变换的某变种。

步 2（边界行为）：要 Tauberian 适用，得保证 "\(F(s) - 1/(s - 1)\) 在 \(\Re s = 1\) 连续延拓"。即 \(-\zeta'/\zeta \cdot 1/s\) 减掉它在 \(s = 1\) 的极点 (留数 1) 之后，其它处不能爆炸。

"其它处" 意思是 \(\Re s = 1, s \ne 1\)。\(-\zeta'/\zeta\) 在哪里有极点？答：在 \(\zeta\) 零点处。所以要 \(\Re s = 1\) 上 \(\zeta\) 无零点（\(s = 1\) 极点除外）。这就是 16.3 引理的角色！

有了 16.3，"$F(s) - 1/(s - 1)$ 在 $\Re s = 1$ 连续" 成立（细节略）。Wiener-Ikehara 给：

\[ \psi(e^t) e^{-t} \to 1 \quad (t \to \infty), \]

即 \(\psi(x) \sim x\)，\(x \to \infty\)。换 \(x = N\)：\(\psi(N) \sim N\)。\(\checkmark\)

整段证明的精神：

  • "素数分布" 翻译成"$\psi(N)$ 渐近"（16.1）。
  • "$\psi(N)$ 渐近" 翻译成"$-\zeta'(s)/\zeta(s)$ 在 $\Re s = 1$ 上无奇异点"（Mellin 变换）。
  • "$-\zeta'/\zeta$ 在 $\Re s = 1$ 无奇异（除 $s = 1$）" 翻译成"$\zeta$ 在 $\Re s = 1$ 无零点"（16.3）。
  • "$\zeta$ 无零点" 用 \(2(1 + \cos\theta)^2 \ge 0\) 这条朴素不等式证。

四步翻译，把"素数分布" 这个朴素的整数论问题，化简成一个朴素三角恒等式。这是 19 世纪解析数论奇迹的全部内容。

16.5 从 \(\psi(N) \sim N\) 推 \(\pi(N) \sim N/\log N\)

16.1 已经做了 \(\psi \to \theta \to \pi\) 的转换。重写要点：

\(\psi \sim N \Rightarrow \theta \sim N\)：\(\psi(N) - \theta(N) = O(\sqrt N \log N) = o(N)\)，所以 \(\theta(N) = \psi(N) + o(N) \sim N\)。

\(\theta \sim N \Rightarrow \pi \sim N/\log N\)：Abel 求和给

\[ \theta(N) = \sum_{p \le N} \log p = \pi(N) \log N - \int_2^N \frac{\pi(t)}{t} dt. \]

反解：\(\pi(N) = \theta(N)/\log N + \frac{1}{\log N} \int_2^N \pi(t)/t\, dt\)。

主项 \(\theta(N)/\log N \sim N/\log N\)。次项需估计 \(\int \pi(t)/t \, dt\)；用粗界 \(\pi(t) = O(t/\log t)\)，积分 \(\le \int_2^N \frac{1}{\log t} dt = O(N/\log N)\)（标准 Li(N) 估计）。所以次项 \(O(N/\log^2 N) = o(N/\log N)\)，主项支配，\(\pi(N) \sim N/\log N\)。\(\checkmark\)

素数定理证完了。

请你停下来回顾整条证明的朴素度。最深的两步是：(1) 解析延拓 + 函数方程让 \(\zeta\) 成为 \(\mathbb{C}\) 上的复函数；(2) 那条 \(2(1 + \cos\theta)^2 \ge 0\) 神奇不等式压制零点。剩下的都是技术翻译——Tauberian 定理、Abel 求和、积分估计。

这种"深核心 + 一堆技术翻译" 是 19 世纪解析数论的典型证明结构。你能 跳过技术细节、记住核心，就抓住了 80% 的精神。

16.6 Riemann 假设——零点位置决定误差

素数定理给 \(\pi(N) \sim N/\log N\)。但误差项 多大？也就是 \(\pi(N) - \mathrm{Li}(N)\)（\(\mathrm{Li}(N) = \int_2^N dt/\log t\) 是更准的主项）的大小？

已知（用 16.3 引理 + de la Vallée-Poussin 1899 改进）：

\[ \pi(N) = \mathrm{Li}(N) + O(N e^{-c \sqrt{\log N}}), \]

误差比 \(N\) 小但不像 \(\sqrt N\) 那么小。

Riemann 假设 (RH, 1859)：\(\zeta(s)\) 所有非平凡零点都在临界线 \(\Re s = 1/2\) 上。

等价表述（一种）：

\[ \pi(N) = \mathrm{Li}(N) + O(\sqrt N \log N). \]

"误差项 $O(\sqrt N \log N)$" 跟 "所有非平凡零点 $\Re s = 1/2$" 是同一件事——这是 Riemann 1859 论文里的核心观察。

为什么？粗略：\(\zeta\) 的对数导数 \(-\zeta'/\zeta\) 的极点在 \(\zeta\) 的零点处。零点位置控制 \(-\zeta'/\zeta\) 的"奇点谱"。Tauberian 定理（更精细版本）让奇点谱直接 翻译成 \(\psi(N)\) 的展开，进而 \(\pi(N)\) 的展开。零点 \(\rho = 1/2 + i \gamma\) 给 \(\psi(N)\) 一项 \(N^\rho / \rho \approx \sqrt N\) 量级。所有非平凡零点都在 \(\Re s = 1/2\) iff 误差被压到 \(O(\sqrt N \log N)\)。

RH 已被验证到前 \(10^{13}\) 个非平凡零点（Gourdon 等大型计算）——全在 \(\Re s = 1/2\) 上。但证 仍然没有。

RH 是ch1 提到的 Clay 七大千禧年问题之一，悬赏 100 万美元。

它的蕴含 之深：除了素数分布，几乎所有解析数论里的"误差项最优" 结果都依赖 RH。例：相邻素数间隙 \(p_{n + 1} - p_n = O(\sqrt{p_n} \log p_n)\)（条件 RH），最强无条件结果 \(O(p_n^{0.525})\)（Baker–Harman–Pintz 2001）。两者差距巨大，RH 的强度可见。

16.7 现代视角：\(\zeta\) 是 GL_1 自守 \(L\)-函数

20 世纪后半，\(\zeta\) 故事被极大推广。本节给一句话视角，不展开（属于ch24 类域论 + 自守表示）。

视角：\(\zeta(s)\) 是群 GL_1（即 $\mathbb{Q}^\times$）上的自守表示关联的 \(L\)-函数。Dirichlet \(L(s, \chi)\) 也是 GL_1 自守。一般地，对任何代数群 GL_n，存在一类自守表示（Hecke、Maass、Selberg、Langlands 给的概念），每个自守表示关联一个 \(L\)-函数。

Langlands 哲学（1967，未证）：

  • 所有"自然出现" 的 \(L\)-函数（来自 Galois 表示、椭圆曲线、模形式、Shimura 簇）都是 自守 \(L\)-函数。
  • 反过来，自守 \(L\)-函数都满足 (a) 解析延拓 (b) 函数方程 (c) Euler 乘积 (d) 适当推广的 Riemann 假设。

这条哲学把 \(\zeta\) 的"解析延拓 + 函数方程 + 临界线" 三个事实推广到所有数论 $L$-函数。它是 20 世纪后半数论的主线，跟非交换调和分析、自守形式、$p$-adic 表示 这些工具紧密咬合。

已经证的部分：

  • 椭圆曲线模性 (Wiles 1995, Breuil–Conrad–Diamond–Taylor 2001)：每条 \(\mathbb{Q}\) 上椭圆曲线的 \(L\)-函数 = 某权 2 模形式的 \(L\)-函数。这条让Fermat 大定理 成立。下书 ch24。
  • Sato-Tate 猜想 (Clozel-Harris-Shepherd-Barron-Taylor 2008)：椭圆曲线 \(a_p / \sqrt p\) 的分布是 \([-2, 2]\) 上的 sin² 分布。

未证：

  • BSD 猜想（椭圆曲线 \(L\)-函数零点 \(\Leftrightarrow\) Mordell-Weil 秩）——Clay 千禧年问题之一。
  • Riemann 假设 + 它的所有自守推广。
  • Langlands 函子性的大部分实例。

简单来说：本章看到的 "$\zeta$ 决定素数分布"，在现代视角下是"GL_1 自守 $L$-函数决定 $\mathbb{Q}^\times$ 上的算术" 的最朴素特例。一般地，"GL_n 自守 $L$-函数决定 GL_n 关联代数对象的算术"。$L$-函数零点 = 算术分布 这种联系是数论现代框架的中心承诺。

16.8 一句临别话

这一章没给完整证明——给的是骨架。要点：

(1) 素数定理 $\pi(N) \sim N/\log N$ = $\zeta$ 在 $\Re s = 1$ 上无零点。这条等价是整段故事最深的事实。它把"朴素整数论问题" 翻译成"复函数零点位置问题"。

(2) "$\zeta$ 在 $\Re s = 1$ 无零点" 用一条初等 不等式 \(2(1 + \cos\theta)^2 \ge 0\) 证。Hadamard、de la Vallée-Poussin 1896 同年独立完成。

(3) RH（所有非平凡零点 \(\Re s = 1/2\)）是更深一层猜想——它给"素数分布" 的最优误差。已计算验证 \(10^{13}\) 个零点，未证。

(4) 现代视角下，\(\zeta\) 只是 GL_1 自守 \(L\)-函数。Langlands 哲学把整套思路推广到所有自守 \(L\)-函数。

请你停下来欣赏这条故事的整体形态。"素数" 是数学里最朴素的对象——只能被 1 和自己整除的数。"素数定理" 是关于素数最朴素的统计陈述。可这条陈述需要整个复分析 + 函数论 才能证。证明的核心步骤是个三角恒等式。证明的扩展（RH）是未解 的、被悬赏 100 万美元的数学最难问题之一。整套故事的现代延伸（Langlands）是 20-21 世纪数学最大的猜想框架。

这种"朴素表面 vs 深层结构" 是数论的本质诱惑——\(\zeta\) 不过是其中最朴素的一个例子。下一章 ch17 我们换条赛道，看 Gauss 在 1801 年之前已经搞出来的另一个深结构——二元二次型 和类数，那是代数数论的另一支线，不依赖复分析。
