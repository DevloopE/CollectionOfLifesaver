第 17 章 二元二次型与类数

请你先回想一个老问题：哪些整数能写成两个平方的和 \(n = a^2 + b^2\)？第 13 章用高斯整数 \(\mathbb{Z}[i]\) 给了一个干净答案——\(n\) 能写成两平方和当且仅当 \(n\) 的每个 \(p \equiv 3 \pmod 4\) 类素因子都偶次出现。这条结果背后是 \(\mathbb{Z}[i]\) 是 UFD（唯一分解整环）。

这一章我们换一道更刁的题：哪些整数能写成 \(n = a^2 + 5 b^2\)？这道看似只把 1 换成 5 的小改动会把"唯一分解" 这条工具直接打破——\(\mathbb{Z}[\sqrt{-5}]\) 不是 UFD。整套表方法跟着崩。但崩的方式自己是一段非常深的故事——它催生了 Gauss 的类数理论，催生了 Dedekind 的理想概念，最终催生了整个代数数论。这一章我们沿这条 \(a^2 + 5 b^2\) 的窄路走进去。

规律——能表的 \(p\) 似乎集中在 \(p \equiv 1, 9 \pmod{20}\)。验证更多——\(p = 101 = 81 + 20 = 9^2 + 5 \cdot 2^2\)，\(101 \equiv 1\)；\(p = 109 = 4 + 105\) 不行，\(= 64 + 45 = 8^2 + 5 \cdot 3^2 \checkmark\)，\(109 \equiv 9\)。

请你停下来感受这件事的怪。判别式相同（都是 \(-20\)）的两种"二次型" \(x^2 + 5 y^2\) 和 \(2 x^2 + 2 xy + 3 y^2\) 各自表示一半素数——按 \(p \bmod 20\) 分类：\(\{1, 9\}\) 那一半归第一型，\(\{3, 7\}\) 那一半归第二型。这种"判别式给一族形式" + "每族里有几个不同的形式" 正是这一章要建立的语言。

而有几个不同的形式 这个数有名字——叫类数 \(h(D)\)，是这章核心不变量。\(h(-4) = 1\)（只有 \(x^2 + y^2\)），\(h(-8) = 1\)（只有 \(x^2 + 2 y^2\)），但 \(h(-20) = 2\)（既要 \(x^2 + 5 y^2\) 又要 \(2 x^2 + 2 xy + 3 y^2\)）。

这"从 1 跳到 2" 不是数据的偶然——它是 \(\mathbb{Z}[\sqrt{-5}]\) 不是 UFD 的定量体现。我们慢慢挖。

17.2 二元二次型：定义与判别式

把上面遇到的对象正式命名。一个二元二次型是关于两个变量 \(x, y\) 的齐次二次多项式：

\[ f(x, y) = a x^2 + b xy + c y^2, \quad a, b, c \in \mathbb{Z}. \]

它的判别式定义为

\[ \boxed{\ D := b^2 - 4 ac.\ } \]

判别式来自配方的副产物——把 \(f\) 完成平方：

\[ 4 a f(x, y) = (2 a x + b y)^2 - (b^2 - 4 ac) y^2 = (2 a x + b y)^2 - D y^2. \]

所以 \(f\) 的正负性完全由 \(D\) 和 \(a\) 决定：

  • \(D < 0\) 且 \(a > 0\)：\(f(x, y) > 0\) 对所有 \((x, y) \neq (0, 0)\)。叫正定。本章只讨论这种。
  • \(D < 0\) 且 \(a < 0\)：负定，乘 \(-1\) 化归正定。
  • \(D > 0\)：\(f\) 既取正值又取负值，叫不定。处理方式不同（链回连分数 + Pell 方程，ch12）。
  • \(D = 0\)：退化（\(f\) 是完全平方）。

具体例子：

  • \(x^2 + y^2\)：\(a = c = 1, b = 0\)，\(D = -4\)。
  • \(x^2 + 2 y^2\)：\(D = -8\)。
  • \(x^2 + 5 y^2\)：\(D = -20\)。
  • \(2 x^2 + 2 xy + 3 y^2\)：\(a = 2, b = 2, c = 3\)，\(D = 4 - 24 = -20\)。

注意：\(D = -20\) 对应两个不同的形式。这是 17.1 节看到的现象的代数表达。

判别式还有一个重要约束——\(D = b^2 - 4 ac\) 总满足 \(D \equiv 0\) 或 \(1 \pmod 4\)（因为 \(b^2 \bmod 4 \in \{0, 1\}\)）。这条限制告诉你哪些 \(D\) 能成为二次型的判别式：\(D \equiv 0, 1 \pmod 4\)。其他 \(D\) 不出现。

17.3 等价性：把"换变量"算成同一型

问题：\(x^2 + y^2\) 和 \((x + y)^2 + y^2 = x^2 + 2 xy + 2 y^2\) 是不是"同一" 二次型？

把 \((x, y)\) 换成 \((x + y, y)\) 是个可逆整数变量替换：

\[ \begin{pmatrix} x' \\ y' \end{pmatrix} = \begin{pmatrix} 1 & 1 \\ 0 & 1 \end{pmatrix} \begin{pmatrix} x \\ y \end{pmatrix}. \]

这个矩阵 \(M = \begin{pmatrix} 1 & 1 \\ 0 & 1 \end{pmatrix}\) 满足 \(\det M = 1\)，且 \(M^{-1} = \begin{pmatrix} 1 & -1 \\ 0 & 1 \end{pmatrix}\) 也是整数矩阵——所以替换是双向的。两个表达式表示的整数集合完全一样：\(n\) 能用 \(x^2 + y^2\) 表 \(\iff\) \(n\) 能用 \(x^2 + 2 xy + 2 y^2\) 表。这两个二次型应该被算成"同一个"。

正式定义：两个二次型 \(f, g\) 等价（写 \(f \sim g\)），如果存在 \(M = \begin{pmatrix} \alpha & \beta \\ \gamma & \delta \end{pmatrix} \in \mathrm{SL}_2(\mathbb{Z})\)（即整数矩阵，\(\det M = \alpha \delta - \beta \gamma = 1\)）使

\[ g(x, y) = f(\alpha x + \beta y, \ \gamma x + \delta y). \]

选 \(\det = +1\)（不允许 \(-1\)）让等价类带"方向"，叫狭义等价。Gauss 用这个版本。也有宽义等价（\(\det = \pm 1\)，即 \(\mathrm{GL}_2(\mathbb{Z})\) 作用），后者把 \(f(x, y)\) 和 \(f(x, -y)\) 算同。本章用狭义。

关键性质：等价的二次型有相同的判别式 + 相同的"被表示集"。判别式相同因为 \(D = -4 \det A_f\)，\(A_f\) 是 \(f\) 的对称矩阵 \(\begin{pmatrix} a & b/2 \\ b/2 & c \end{pmatrix}\)（容许半整数项是因为 \(b\) 不一定偶——但 \(2 A_f\) 是整数矩阵）；变换后矩阵变成 \(M^T A_f M\)，行列式不变（\(\det M = 1\)）。被表示集相同因为变量替换是双射。

所以"哪些 \(n\) 能被 \(f\) 表" 这件事只依赖 \(f\) 的等价类，不依赖具体的 \((a, b, c)\)。我们要研究的是等价类，不是单个二次型。

17.4 既约形式：每个等价类里挑一个标准代表

一个等价类有无穷个二次型——\(\mathrm{SL}_2(\mathbb{Z})\) 是无穷群。我们想每类里挑一个"最简" 代表。这就是既约形式。

对正定二次型 \(f = a x^2 + b xy + c y^2\)（\(D < 0\), \(a > 0\)），既约定义为：

\[ \boxed{\ |b| \le a \le c, \quad \text{且若 } |b| = a \text{ 或 } a = c \text{ 则 } b \ge 0.\ } \]

第一个条件让系数最小，第二个解决边界处的二选一歧义（避免重复计数）。

定理（Gauss）。每个正定二次型等价于恰好一个既约形式。

证明思路（不严格、给直觉）：用两个基本变换——

  • \(T = \begin{pmatrix} 0 & -1 \\ 1 & 0 \end{pmatrix}\)：把 \(f(x, y)\) 换成 \(f(-y, x) = c x^2 - b xy + a y^2\)。即交换 \(a \leftrightarrow c\) 同时翻 \(b \to -b\)。
  • \(S_n = \begin{pmatrix} 1 & n \\ 0 & 1 \end{pmatrix}\)：把 \(f(x, y)\) 换成 \(f(x + n y, y) = a x^2 + (b + 2 a n) x y + (\text{某个 } c')\)。即把 \(b\) 加上 \(2 a n\)（同时 \(c\) 跟着调）。

策略：先用 \(S_n\) 把 \(b\) 调到 \(|b| \le a\)（即 \(b\) 落在 \([-a, a]\) 内）。如果此时 \(a > c\)，用 \(T\) 交换 \(a, c\)。重复直到稳定。每次 \(T\) 操作让 \(a\) 严格减小（因为 \(c < a\) 后交换让新 \(a\) 更小）；\(a\) 是正整数，不能无限减——必终止。

这是个具体的归约算法，叫"Gauss 归约"。它在二次型里的作用类似 Euclid 算法在整数里的作用。

例子：归约 \(f = 7 x^2 + 14 xy + 9 y^2\)，\(D = 196 - 252 = -56\)。

第 1 步：\(|b| = 14 > a = 7\)，先用 \(S_n\) 调。\(b' = b + 2 a n = 14 + 14 n\)。取 \(n = -1\)：\(b' = 0\)，\(c' = c + b n + a n^2 = 9 - 14 + 7 = 2\)。新型 \(7 x^2 + 0 \cdot xy + 2 y^2 = 7 x^2 + 2 y^2\)。

第 2 步：\(a = 7, c = 2\)，\(a > c\)。用 \(T\) 交换：新型 \(2 x^2 + 0 \cdot xy + 7 y^2 = 2 x^2 + 7 y^2\)。

第 3 步：\(|b| = 0 \le a = 2 \le c = 7\)，且 \(b = 0 \ge 0\)。既约。

所以 \(7 x^2 + 14 xy + 9 y^2 \sim 2 x^2 + 7 y^2\)。验证判别式：\(0 - 4 \cdot 2 \cdot 7 = -56\)。\(\checkmark\)

定理（既约形式的有限性）。对任何固定的 \(D < 0\)，判别式为 \(D\) 的既约正定二次型只有限多。

证明：既约条件给 \(|b| \le a \le c\)。所以 \(|D| = 4 ac - b^2 \ge 4 a^2 - a^2 = 3 a^2\)，即 \(a \le \sqrt{|D|/3}\)。\(a\) 的可能值有限。对每个 \(a\)，\(b\) 满足 \(|b| \le a\)，可能值有限。\(c\) 由 \(a, b, D\) 唯一定（\(c = (b^2 - D) / (4 a)\)，且要求是正整数）。所以三元组 \((a, b, c)\) 有限。

这个有限数有名字。

17.5 类数 \(h(D)\)

定义。判别式为 \(D\) 的既约正定二次型个数叫类数，记 \(h(D)\)。

等价地：\(h(D)\) 是判别式为 \(D\) 的正定二次型在 \(\mathrm{SL}_2(\mathbb{Z})\) 等价下的等价类个数。

具体计算几个：

\(D = -4\)。\(a \le \sqrt{4/3} \approx 1.15\)，所以 \(a = 1\)。\(|b| \le 1\)，\(b \in \{-1, 0, 1\}\)。\(c = (b^2 + 4) / 4\) 要是正整数：

  • \(b = 0\)：\(c = 1\)。形式 \(x^2 + y^2\)。\(\checkmark\)
  • \(b = \pm 1\)：\(c = 5/4\)，非整数。\(\times\)

所以 \(h(-4) = 1\)，唯一既约形式 \(x^2 + y^2\)。

\(D = -8\)。\(a \le \sqrt{8/3} \approx 1.63\)，所以 \(a = 1\)。\(b \in \{-1, 0, 1\}\)，\(c = (b^2 + 8)/4\)：

  • \(b = 0\)：\(c = 2\)。形式 \(x^2 + 2 y^2\)。\(\checkmark\)
  • \(b = \pm 1\)：\(c = 9/4\)，非整数。\(\times\)

\(h(-8) = 1\)，唯一形式 \(x^2 + 2 y^2\)。

\(D = -20\)。\(a \le \sqrt{20/3} \approx 2.58\)，所以 \(a \in \{1, 2\}\)。

\(a = 1\)：\(|b| \le 1\)，\(c = (b^2 + 20)/4\)。\(b = 0 \Rightarrow c = 5\)。形式 \(x^2 + 5 y^2\)。\(\checkmark\) \(b = \pm 1 \Rightarrow c = 21/4\)，不行。

\(a = 2\)：\(|b| \le 2\)，\(c = (b^2 + 20)/8\)。\(b = 0 \Rightarrow c = 5/2\)，不行。\(b = \pm 1 \Rightarrow c = 21/8\)，不行。\(b = \pm 2 \Rightarrow c = 24/8 = 3\)。需要 \(a \le c\)（满足）+ 边界条件。\(|b| = a = 2\) 时要 \(b \ge 0\)，所以选 \(b = 2\)。形式 \(2 x^2 + 2 xy + 3 y^2\)。\(\checkmark\)

所以 \(h(-20) = 2\)，两个既约形式 \(x^2 + 5 y^2\) 和 \(2 x^2 + 2 xy + 3 y^2\)——正是 17.1 节看到的两族。类数 = 2 解释了为什么 \(p \equiv 1, 9 \pmod{20}\) 走第一型而 \(p \equiv 3, 7 \pmod{20}\) 走第二型——一族不够用，必须两族分担。

"什么时候 \(h(D) = 1\)" 这个问题特别神秘——后面 17.8 节单独讨论。

17.6 类数 = 类群的大小（链回抽象代数）

到这里"类数" 是个数。但它的真意义是它在数一个群的元素数——这个群叫类群，记 \(\mathrm{Cl}(D)\)。

类群从哪儿来？回到 \(\mathbb{Z}[\sqrt{-5}]\)（更一般地：判别式 \(D\) 对应的二次整数环 \(\mathcal{O}_D\)）。这环不是 UFD——典型例子

\[ 6 = 2 \cdot 3 = (1 + \sqrt{-5})(1 - \sqrt{-5}). \]

左右两种分解都不可再约（要严格证 \(2, 3, 1 \pm \sqrt{-5}\) 在 \(\mathbb{Z}[\sqrt{-5}]\) 里是不可约元，但容易），且两边的因子互不相伴（差一个单位倍）。所以 \(6\) 有两种本质不同的不可约分解——UFD 失败。

Dedekind 的天才解决方案（1871）：把"数" 换成"理想"。在 \(\mathbb{Z}[\sqrt{-5}]\) 中：

\[ (2) = \mathfrak{p}_2^2, \quad (3) = \mathfrak{p}_3 \mathfrak{p}_3', \quad (1 + \sqrt{-5}) = \mathfrak{p}_2 \mathfrak{p}_3, \quad (1 - \sqrt{-5}) = \mathfrak{p}_2 \mathfrak{p}_3'. \]

这里 \(\mathfrak{p}_2 = (2, 1 + \sqrt{-5})\)、\(\mathfrak{p}_3 = (3, 1 + \sqrt{-5})\)、\(\mathfrak{p}_3' = (3, 1 - \sqrt{-5})\) 是素理想（不一定主理想）。两边算 \((6) = \mathfrak{p}_2^2 \cdot \mathfrak{p}_3 \mathfrak{p}_3' = (2) \cdot (3)\)，也 \(= \mathfrak{p}_2 \mathfrak{p}_3 \cdot \mathfrak{p}_2 \mathfrak{p}_3' = (1 + \sqrt{-5})(1 - \sqrt{-5})\)。理想分解唯一（即使元素分解不唯一）。

这个"理想唯一分解" 是 Dedekind 域的定义。代价是出现了"非主理想" \(\mathfrak{p}_2\)（不存在 \(\alpha \in \mathbb{Z}[\sqrt{-5}]\) 使 \(\mathfrak{p}_2 = (\alpha)\)）。

把"分式理想模主理想" 商出来，得到的群就是类群：

\[ \mathrm{Cl}(\mathcal{O}_D) := \{\text{分式理想}\} / \{\text{主理想}\}. \]

\(\mathrm{Cl}\) 的大小衡量"有多少非主理想" / "UFD 失败的程度"。

  • \(\mathrm{Cl} = \{1\}\)（平凡）⟺ 每个理想都是主理想 ⟺ \(\mathcal{O}_D\) 是 PID ⟺ \(\mathcal{O}_D\) 是 UFD（对 Dedekind 域两条等价，链回抽象代数 ch14）。
  • \(\mathrm{Cl}\) 越大，"非主理想越多"，UFD 越失败。

定理（Gauss / 后人现代化）。\(h(D) = |\mathrm{Cl}(\mathcal{O}_D)|\)。

"既约二次型个数" = "理想类群的大小"。两个看起来八竿子打不着的对象（一个是配方系数的归约结果，一个是抽象代数里的商群）是同一个数。这是 Gauss-Dedekind-Dedekind 跨 70 年才完全建立的桥。

这条对应不仅是数相等——它是同一群结构。两个二次型的"合成"（Gauss composition，技术细节略）对应两个理想类相乘——既约二次型的等价类本身构成一个群（同构于 \(\mathrm{Cl}(\mathcal{O}_D)\)）。这个发现让二次型理论从"分类问题" 升级为"群论问题"，是 19 世纪代数数论起飞的关键。

17.7 \(h = 1\)：唯一分解 + 简单表示准则

当 \(h(D) = 1\)，类群平凡，\(\mathcal{O}_D\) 是 UFD。这时"哪些 \(p\) 能用 \(D\) 的（唯一）形式表" 有简单准则——把"能表" 翻译为"素数 \(p\) 在 \(\mathcal{O}_D\) 中分裂"。

对 \(D = -4\)（\(\mathcal{O} = \mathbb{Z}[i]\)，\(h = 1\)）：

\[ p = a^2 + b^2 \iff p \text{ 在 } \mathbb{Z}[i] \text{ 中分裂} \iff p \equiv 1 \pmod 4 \text{ 或 } p = 2. \]

这是 ch13 已证。

对 \(D = -8\)（\(\mathcal{O} = \mathbb{Z}[\sqrt{-2}]\)，\(h = 1\)）：

\[ p = a^2 + 2 b^2 \iff p \equiv 1, 3 \pmod 8 \text{ 或 } p = 2. \]

对 \(D = -7\)（\(\mathcal{O} = \mathbb{Z}[(1 + \sqrt{-7})/2]\)，\(h = 1\)）：

\[ p = x^2 + xy + 2 y^2 \iff p \equiv 1, 2, 4 \pmod 7. \]

统一模式：\(h = 1\) 时，"哪些 \(p\) 能表" 由 \(p\) 在 \((\mathbb{Z}/|D|)^*\) 里某个特定子群里决定——即纯模条件。这就是 ch13 那种"\(p \equiv 1 \pmod 4\)" 类规则的一般化。

但当 \(h(D) > 1\)，模条件不够了。\(D = -20\) 时 \(p \equiv 1 \pmod{20}\) 和 \(p \equiv 9 \pmod{20}\) 走第一型，\(p \equiv 3, 7\) 走第二型——两类都满足"是 mod 20 二次剩余" 这个模条件，但只有一半属于 \(x^2 + 5 y^2\) 的范围。模条件能告诉你 \(p\) 是否被某个判别式 \(D\) 的二次型表，但不能告诉你具体走哪一型——这后者需要类群信息。

用现代术语：判断"\(p\) 由哪一型表" 的完整规则需要类域论（ch22）——\(p\) 在 \(D\) 的Hilbert 类域 里如何分裂决定它走哪一型。这是把"二次型表示" 推到极致的现代版本，也是 Gauss-Dedekind-Hilbert-Takagi 一百年发展的最终形态。

17.8 几个具体值 + Gauss 类数问题

所有判别式 \(D < 0\) 中 \(h(D) = 1\) 的有几个？

Gauss 在《算术研究》（1801）里把虚二次域的类数表算到 \(D\) 几千，注意到一个奇特规律：\(h(D) = 1\) 的 \(D\) 极少。他列出了 9 个：

\[ D \in \{-3, -4, -7, -8, -11, -19, -43, -67, -163\}. \]

（如果用"基本判别式" 这个稍精细的概念，列表正好 9 个；其他 \(h = 1\) 的形态都是这 9 个的"非基本" 版本。这个技术细节我们略。）

Gauss 猜：这 9 个就是全部。他没法证。这是Gauss 类数 1 问题。

整个 19 世纪没人能证。1934 年 Heilbronn 证明了"除了上述 9 个之外至多还有有限多个" + Heilbronn-Linfoot 把"有限多" 缩到"至多 1 个"。但这"第 10 个" 的存在性 / 不存在性还悬着。

1952 年 Heegner（一个德国高中老师，业余数学家）发表论文宣称证明了"第 10 个不存在"。但他用了一些争议步骤，主流数学家不认可。

1966 年 Stark 独立给了一个干净证明，1967 年发表。同年 Baker 用线性形式对数下界（这套理论让他获 Fields 奖）也独立给出证明。

1968 年 Stark 重新审视 Heegner 的论文，确认 Heegner 的证明本质正确——只是当年的呈现方式让审稿人误判。Heegner 平反——他真是第一个证完的人，比公认证明早 14 年。这是数学史上"圈外人被忽视" 的著名案例。

Stark-Heegner 定理最终陈述：虚二次域 \(\mathbb{Q}(\sqrt{D})\)（\(D < 0\)）类数为 1 的恰好是 \(D \in \{-3, -4, -7, -8, -11, -19, -43, -67, -163\}\)。Gauss 250 年的猜测被证。

类数 \(h(D) = 2\) 的 \(D\) 同样有限多（Gauss 给 18 个，后人证完）。\(h = 3\) 也有限。一般地：

定理（Heilbronn, Siegel）。\(\lim_{D \to -\infty} h(D) = \infty\)。

即"判别式越大，类数趋向越大"——所以每个固定的 \(h\) 只对应有限多 \(D\)。但 Siegel 的证明不有效——它告诉你"有限多" 但不告诉你"具体哪些"。每个固定 \(h\) 的完整列表需要单独的有效定理。\(h \le 100\) 的完整列表已经搞定（Watkins 2004），更高需要更多工作。

这个问题为什么这么难？根本原因是类数和椭圆曲线 / 模形式的零点紧密相关——具体地，\(h(D)\) 由 Dirichlet 类数公式

\[ h(D) = \frac{w \sqrt{|D|}}{2 \pi} L(1, \chi_D) \]

给出（\(w\) 是 \(\mathcal{O}_D\) 的单位根数，\(\chi_D\) 是 \(D\) 对应的二次特征，\(L(s, \chi)\) 是 Dirichlet \(L\)-函数）。所以"类数 = 1" 等价于"\(L(1, \chi_D)\) 取特定小值"——而 \(L\) 函数的零点 / 值估计是解析数论里最难的一类问题（链回 ch16 \(\zeta\) 函数 + Riemann 假设）。Stark/Heegner/Baker 的证明本质上都是"用 \(L\) 函数性质排除 \(D\) 的可能"。

请你最后停下来想一件事。本章开头的钩子是"哪些 \(p\) 能写成 \(a^2 + 5 b^2\)"——一个看似中学水平的小问题。可它的完整解答（"\(p \equiv 1, 9 \pmod{20}\)"）背后是判别式 + 等价 + 类数 + 类群——一个完整的代数结构层。再问"什么时候 \(h(D) = 1\)"——这个问题让 Gauss 250 年没解，让 19 世纪整个代数数论发展，让 1967 年 Stark / Heegner 用 \(L\) 函数才证完。

整数表面平凡，骨子里复杂得超乎想象——这是 ch1 里说过的话。这一章是这句话最具体的证据。下一章我们换工具——把"距离" 重新定义，让"整数" 在一种全新的几何里出现。这就是 \(p\)-adic 数。
