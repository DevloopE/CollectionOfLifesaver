第 19 章 椭圆曲线引论

请你看这个方程：

\[ y^2 = x^3 - x. \]

问：它在整数 上有几个解？

朴素枚举。\(x = 0\)：\(y^2 = 0 \Rightarrow y = 0\)，解 \((0, 0)\)。\(x = 1\)：\(y^2 = 0 \Rightarrow y = 0\)，解 \((1, 0)\)。\(x = -1\)：\(y^2 = 0 \Rightarrow y = 0\)，解 \((-1, 0)\)。\(x = 2\)：\(y^2 = 6\)，不是平方。\(x = 3\)：\(y^2 = 24\)，不是平方。\(x = -2\)：\(y^2 = -6 < 0\)，无解。\(x = 5\)：\(y^2 = 120\)，无。试到 \(x = 100\) 都没新解。

所以整数解就是 \(\{(0, 0), (1, 0), (-1, 0)\}\)——只 3 个。这看起来是个无聊问题。

但请你换一个问题：有理 解呢？还是只 3 个吗？

不一定。我们没法穷举所有有理 \((x, y)\)。需要更巧的工具。

这一章的钩子：方程 \(y^2 = x^3 - x\)（更一般 \(y^2 = x^3 + a x + b\)）的有理点集合 上有一个自然的群结构——你可以"加" 两个有理解得到第三个有理解。这种"群结构" 是椭圆曲线最神奇的特征，把"找有理解" 这个 Diophantine 问题翻译成"研究群 \(E(\mathbb{Q})\) 的结构"。这套路径是 20 世纪数论最重大的发现之一，也是 Wiles 1995 证 Fermat 大定理的工具。

19.1 几个具体例子

椭圆曲线的标准定义形式是Weierstrass 方程：

\[ y^2 = x^3 + a x + b, \quad a, b \in \mathbb{Q} \ (\text{或 } \mathbb{Z}, \text{或 } \mathbb{F}_p, \ldots). \]

下面是几个例子，我们之后会反复回到。

例 1：\(E_1 : y^2 = x^3 - x\)（即 \(a = -1, b = 0\)）。右边因式 \(x(x-1)(x+1)\)，三实根 \(\{-1, 0, 1\}\)。实图像是"一团圆 + 一条向右无穷延伸的曲线"——一个有限的椭圆形分量在 \(-1 \le x \le 0\)，加上一个右半分量从 \(x = 1\) 出发。

例 2：\(E_2 : y^2 = x^3 + 1\)。右边一实根 \(x = -1\)。实图像只有一个分量（"一条曲线"），形状像个倒过来的字母 C。

例 3：\(E_3 : y^2 = x^3 - 2\)。无理实根 \(x = 2^{1/3} \approx 1.26\)。一分量，形状类似 \(E_2\)。

这三条曲线"样子" 不同（实图像分量数不同——\(E_1\) 两块，\(E_2, E_3\) 一块），但它们都在同一族里。区分它们的工具是判别式。

19.2 Weierstrass 形式 + 判别式 + 光滑条件

对 \(E : y^2 = x^3 + a x + b\)，定义

\[ \boxed{\ \Delta := -16 (4 a^3 + 27 b^2).\ } \]

这是 Weierstrass 椭圆曲线的判别式。它和右边三次 \(x^3 + ax + b\) 是否有重根紧密相连——\(x^3 + ax + b\) 有重根 \(\iff 4 a^3 + 27 b^2 = 0\)（这是三次方程的判别式标准条件）。所以

\[ \Delta = 0 \iff x^3 + a x + b \text{ 有重根}. \]

当 \(\Delta = 0\)，曲线 \(y^2 = x^3 + a x + b\) 有奇点（自交点 / 尖点）——即某点 \((x_0, 0)\) 处 \(y\) 对 \(x\) 的局部导数无定义，曲线在那点"不光滑"。我们要求这种情况排除，因为下面要建立的群律在奇点处会失败。

正式定义：椭圆曲线 是 Weierstrass 方程 \(y^2 = x^3 + a x + b\) 加上条件 \(\Delta \neq 0\)。这条件保证曲线光滑。

验证三例：

  • \(E_1 : a = -1, b = 0\)。\(\Delta = -16 \cdot (-4) = 64 \neq 0\)。\(\checkmark\) 光滑。
  • \(E_2 : a = 0, b = 1\)。\(\Delta = -16 \cdot 27 = -432 \neq 0\)。\(\checkmark\)
  • \(E_3 : a = 0, b = -2\)。\(\Delta = -16 \cdot 108 = -1728 \neq 0\)。\(\checkmark\)

反例：\(y^2 = x^3\)（即 \(a = b = 0\)）：\(\Delta = 0\)。这条曲线在 \((0, 0)\) 有尖点——它不是椭圆曲线。

\(y^2 = x^3 - 3 x + 2\)：\(4 a^3 + 27 b^2 = 4(-27) + 27 \cdot 4 = -108 + 108 = 0\)，\(\Delta = 0\)。检查 \(x^3 - 3 x + 2 = (x - 1)^2 (x + 2)\) 确实有重根 \(x = 1\)。这条曲线在 \((1, 0)\) 有结点（自交）——也不是椭圆曲线。

请你停下来思考这个"光滑性" 要求的分量。它把椭圆曲线限制在一个开稠 集合上（参数空间 \((a, b)\) 几乎所有点都满足 \(\Delta \neq 0\)，例外是个低维"判别式轨迹"），让每条 椭圆曲线都"形状好"——没尖点、没自交，每点处可以做局部坐标。这条简单条件让下面的群结构成立。

19.3 几何：实图像 + 复 torus

实图像。把 \(y^2 = x^3 + a x + b\) 在 \(\mathbb{R}^2\) 平面上画出来。形状由 \(x^3 + a x + b\) 的实根数决定：

  • 三实根：图像两个分量——一个有限"椭圆" 在两个最小根之间，一个无限"分支" 从最大根向右无穷延伸。
  • 一实根（两共轭复根）：图像一个分量，从实根向右延伸。

名字"椭圆曲线" 的来源：实分量看起来像椭圆。但椭圆曲线不是椭圆——它是三次曲线，名字是历史误传。词源：椭圆曲线最早出现在椭圆周长积分里（\(\int \sqrt{1 - k^2 \sin^2 \theta} d\theta\)，无法用初等函数表达），那时候人们叫它"椭圆积分"——后来对应的代数曲线就叫"椭圆曲线"。这是个词义化石。

复图像（torus 解释）。把曲线放进 \(\mathbb{C}^2\)。这时的"解集" 是一个二维复曲面（即四维实流形）的二维（实 二维）子集——其实是个一维复流形。

定理：复椭圆曲线 \(E(\mathbb{C})\)（加上无穷远点 \(O\)）拓扑同胚于复 torus \(\mathbb{C} / \Lambda\)，其中 \(\Lambda \subset \mathbb{C}\) 是某个秩 2 的格（即 \(\Lambda = \omega_1 \mathbb{Z} + \omega_2 \mathbb{Z}\)，\(\omega_1, \omega_2\) 是 \(\mathbb{C}\) 中线性无关的两个数）。

"complex torus" 直观：把 \(\mathbb{C}\)（一个无穷大平面）按格 \(\Lambda\) 折成"平行四边形 + 把对边粘合"——结果是甜甜圈 形状。一个椭圆曲线在 \(\mathbb{C}\) 上就是一个甜甜圈。

这个"甜甜圈" 解释让椭圆曲线的代数 + 几何 + 分析三方面瞬间统一。甜甜圈是个阿贝尔群（\(\mathbb{C} / \Lambda\) 显然是商群），所以椭圆曲线自然 有群结构——这就是下一节要建的群律的来源。

从 \(\mathbb{C} / \Lambda\) 到 \(y^2 = 4 x^3 + a x + b\) 的具体变换由Weierstrass \(\wp\) 函数给：\(x = \wp(z), y = \wp'(z)\) 把 \(z \in \mathbb{C}/\Lambda\) 送到 \(E\) 上的点。这是 19 世纪 Weierstrass 给的解析机器，链回复分析椭圆函数那一节。

19.4 群律：怎么"加" 两个点

这是这一章最重要的构造。给椭圆曲线 \(E\) 上两个点 \(P, Q\)，定义"和" \(P + Q\) 如下。

第一步：画过 \(P, Q\) 的直线。这直线在 \(E\) 上还会交一个点（除了 \(P, Q\) 之外的第三个）。原因：\(E\) 的方程 \(y^2 = x^3 + a x + b\) 是三次，把直线 \(y = m x + c\) 代入得三次方程 \(x^3 - m^2 x^2 + (a - 2 m c) x + (b - c^2) = 0\)。三次有三根（计重数）；其中两根 = \(P, Q\) 的 \(x\) 坐标；存在第三根。两点确定一条直线，直线交三次曲线有三点——所以总能算出第三点（让我们叫它 \(R'\)）。

用 Vieta：三根之和 = 二次项系数取负 = \(m^2\)。所以 \(R'\) 的 \(x\) 坐标 = \(m^2 - x_P - x_Q\)。

\(R'\) 的 \(y\) 坐标 = \(m \cdot x_{R'} + c\)（在直线上），其中 \(c\) 是直线截距。

第二步：把 \(R'\) 沿 \(x\)-轴反过来。定义 \(R = -R'\)，即把 \(R'\) 的 \(y\) 坐标取负。这一步是为了让群律满足结合 性 + 让单位元干净。

结果：\(P + Q := R\)。

用公式：

\[ \boxed{\ x_R = m^2 - x_P - x_Q, \quad y_R = -(m \cdot x_R + c) = m (x_P - x_R) - y_P,\ } \]

其中

\[ m = \frac{y_Q - y_P}{x_Q - x_P} \quad (\text{若 } P \neq Q), \]

或如果 \(P = Q\)（"两点重合"），用切线斜率代直线：

\[ m = \frac{3 x_P^2 + a}{2 y_P}. \]

（这后者来自隐函数 \(2 y \, dy = (3 x^2 + a) dx\)。\(m\) 是 \(E\) 在 \(P\) 处的切线斜率。）

处理"无穷远点" \(O\)。以上构造在大部分情况成立，但有几个边界。比如 \(P = (x_0, y_0)\)，\(Q = (x_0, -y_0)\)（\(x\) 同 \(y\) 反）——它们的"直线" 是垂直线 \(x = x_0\)，三次曲线和垂直线在 \(\mathbb{R}^2\) 中只交两点（\(P, Q\) 自己），没有"第三点"。

解决：人为加一个无穷远点 \(O\)（在射影平面上自然存在），让所有垂直线都"交在 \(O\)"。这样 \(P + Q = -O\)。我们要求 \(O\) 自反（\(-O = O\)），所以 \(P + Q = O\)——即 \(P\) 和 \(Q\) 互为逆元。

定义 \(O\) 是群单位元：\(P + O = P\) 对所有 \(P\)。整个群结构现在如下：

  • 单位元 \(O\)。
  • 逆元 \(-P = (x_P, -y_P)\)。
  • 加法 \(P + Q\) 用上面的弦 / 切线 + \(x\)-轴反公式。

请你停下来想这套群律的朴素美。完全是代数 + 几何 的混合——画一根直线、找它和曲线的第三交点、沿 \(x\)-轴反——三步组合出一个群结构。所有人的第一反应是"这居然行"。

19.5 验证群公理

群公理：单位元、逆元、结合律。前两个由定义直接给。结合律 \((P + Q) + R = P + (Q + R)\) 是真正不平凡的。

有两种证法。

证法 1（代数）。展开 \((P + Q) + R\) 和 \(P + (Q + R)\) 的所有 \(x, y\) 坐标公式，验证它们相等。这个展开痛苦——涉及多项式恒等式 + 大量代数。可以做完，但乏味。

证法 2（几何 / 复分析）。用 \(E(\mathbb{C}) \cong \mathbb{C} / \Lambda\) 的同构。\(\mathbb{C} / \Lambda\) 显然 是阿贝尔群（\(\mathbb{C}\) 是阿贝尔，\(\Lambda\) 是它的子群，商群继承结构）。下一步：证椭圆曲线的弦切群律 \(\Leftrightarrow\) \(\mathbb{C}/\Lambda\) 的加法，通过 \(\wp\) 函数对应。\(\wp\) 函数有恒等式 \(\wp(z_1) + \wp(z_2) + \wp(z_1 + z_2) = (\text{某斜率公式})^2 / 4\) 之类，正好对应弦切公式。

这条路径让"结合律" 从一个代数巧合 变成底层结构的反映——结合律之所以成立，是因为 \(E\) 实际上是个商群。代数公式只是这个事实在 Weierstrass 坐标下的表象。

总结：椭圆曲线 \(E\) 加上 \(O\)（无穷远点）+ 弦切群律是一个阿贝尔群。这群在不同基域下：

  • \(E(\mathbb{C}) \cong \mathbb{C}/\Lambda\)（torus）。
  • \(E(\mathbb{R})\)：实分量数（1 或 2）决定它是 \(S^1\) 或 \(S^1 \times \mathbb{Z}/2\)（拓扑 + 群结构）。
  • \(E(\mathbb{Q})\)：未知一般结构，但 ch20 的 Mordell-Weil 给：有限生成阿贝尔群。
  • \(E(\mathbb{F}_p)\)：有限阿贝尔群，大小约 \(p + 1\)（Hasse 估计）。

每种基域给同一抽象方程不同的群结构，是椭圆曲线富裕的来源。

19.6 \(j\)-不变量与同构分类

问题：什么时候两条椭圆曲线 \(E : y^2 = x^3 + a x + b\) 和 \(E' : y^2 = x^3 + a' x + b'\) "是同一条"？

具体地：什么时候存在变量替换 \((x, y) \mapsto (u^2 x', u^3 y')\)（\(u \neq 0\)）把 \(E\) 变成 \(E'\)？这种替换保 Weierstrass 形式（替换后 \(y^2 = x^3 + a x + b\) 变成 \(u^6 y'^2 = u^6 x'^3 + a u^2 x' + b\)，整理得 \(y'^2 = x'^3 + (a/u^4) x' + (b/u^6)\)——形式不变，只 \((a, b) \to (a/u^4, b/u^6)\)）。

所以同构分类等价于"差一个 \((a, b) \to (a/u^4, b/u^6)\) 替换"。寻找一个"不被这个替换改变" 的量——即在 \((a, b) \to (a/u^4, b/u^6)\) 下不变的函数。

注意 \(a^3 \to a^3/u^{12}\)，\(b^2 \to b^2/u^{12}\)——同步。所以 \(a^3/b^2\) 不 变。但还有 \(\Delta = -16 (4 a^3 + 27 b^2)\)；\(\Delta \to \Delta/u^{12}\)。所以 \(a^3 / \Delta\)、\(b^2 / \Delta\) 都不变。

历史标准选择：

\[ \boxed{\ j(E) := -1728 \cdot \frac{(4 a)^3}{\Delta} = \frac{1728 \cdot 4 a^3}{4 a^3 + 27 b^2}.\ } \]

（因子 1728 + 4 是历史规约，让 \(j\) 有特定整数值在重要曲线上。）

定理。两条椭圆曲线 \(E, E'\) 在 \(\mathbb{C}\) 上同构 \(\iff j(E) = j(E')\)。

所以 \(j\) 是同构的完全不变量——只一个数就完全分类椭圆曲线（在 \(\mathbb{C}\) 上）。这条结果让 \(j\)-不变量成为椭圆曲线最基本的标签。

例：

  • \(E_1 : y^2 = x^3 - x\)：\(a = -1, b = 0\)，\(j = 1728\)。这值有"特殊" 含义——\(j = 1728\) 的曲线有额外对称性（CM by \(\mathbb{Z}[i]\)）。
  • \(E_2 : y^2 = x^3 + 1\)：\(a = 0, b = 1\)，\(j = 0\)。也有 CM（by \(\mathbb{Z}[\zeta_3]\)）。
  • \(j = 0\) 和 \(j = 1728\) 是椭圆曲线"族" 中的两个特殊点；其他 \(j\) 一般。

\(j\) 是个有理函数（系数有理时）。它在 \(\mathbb{Q}\) 上取值在 \(\mathbb{Q}\)；不同有理 \(j\) 给不同同构类（在 \(\mathbb{C}\) 同构的意义下）。

注意——同 \(j\) 但不同 \(\mathbb{Q}\) 同构类 也可能。两个 \(\mathbb{Q}\)-曲线 \(E, E'\) 在 \(\mathbb{C}\) 同构（即 \(j\) 同）但没 \(\mathbb{Q}\) 同构（找不到 \(\mathbb{Q}\) 系数的变换）。这种"同 \(j\) 不同 \(\mathbb{Q}\)-曲线" 叫twist。它们的有理点群 \(E(\mathbb{Q})\) 和 \(E'(\mathbb{Q})\) 可以非常不同——对秩问题有重要影响。

19.7 跑一遍：\(E_1 : y^2 = x^3 - x\) 上的群运算

\(E_1\) 上几个明显有理点：\(O\)（无穷远）、\(P_1 = (0, 0)\)、\(P_2 = (1, 0)\)、\(P_3 = (-1, 0)\)。所有 \(P_i\) 都满足 \(2 P_i = O\)（因为 \(-P_i = (x_i, -0) = P_i\)，所以 \(P_i = -P_i\)，即 \(2 P_i = O\)）。这些是"2-挠点"——满足 \(2 P = O\) 的点。

计算 \(P_1 + P_2\)。\(P_1 = (0, 0)\)，\(P_2 = (1, 0)\)。直线过两者：水平线 \(y = 0\)。代入 \(E_1\)：\(0 = x^3 - x = x(x-1)(x+1)\)，三根 \(\{0, 1, -1\}\)。前两根 \(0, 1\) 给 \(P_1, P_2\)；第三根 \(-1\) 给 \(R' = (-1, 0)\)。沿 \(x\)-轴反 \(R = (-1, 0) = P_3\)。所以 \(P_1 + P_2 = P_3\)。

验证类似关系：\(P_1 + P_3 = P_2\)，\(P_2 + P_3 = P_1\)。所以 \(\{O, P_1, P_2, P_3\}\) 形成 \(E_1\) 的一个子群，结构是 \(\mathbb{Z}/2 \times \mathbb{Z}/2\)（Klein 4 群）。

有非挠的点吗？尝试 \(x = 2\)：\(y^2 = 8 - 2 = 6\)，无理。\(x = -2\)：\(y^2 = -8 + 2 = -6 < 0\)。\(x = 1/4\)：\(y^2 = 1/64 - 1/4 = (1 - 16)/64 = -15/64 < 0\)。\(x = 5\)：\(y^2 = 125 - 5 = 120\)，无理。\(x = 9\)：\(y^2 = 729 - 9 = 720\)，无理。\(x = 25\)：\(y^2 = 25^3 - 25 = 15600\)，\(\sqrt{15600} = 20 \sqrt{39}\)，无理。

事实上 \(E_1(\mathbb{Q}) = \{O, P_1, P_2, P_3\}\) 整体——这条曲线没 非平凡的有理点。这是 Fermat 1640 年用无穷下降 法证的结果（链回 ch20 那个证明的早期形态）。

所以 \(E_1(\mathbb{Q}) \cong (\mathbb{Z}/2)^2\)——有限 群，大小 4。下一章 Mordell-Weil 会告诉你这种"\(E(\mathbb{Q})\) 有限" 是挠子群非空但秩 0 的特殊情况；其他椭圆曲线 \(E(\mathbb{Q})\) 可能无穷（秩 \(\ge 1\)）。

对比例：\(E : y^2 = x^3 - 2\)。明显有理点 \((3, \pm 5)\)（\(27 - 2 = 25 = 5^2\)）。问 \(2 \cdot (3, 5) = ?\)（用切线公式）。\(m = (3 \cdot 9) / (2 \cdot 5) = 27/10\)。\(x_R = m^2 - 6 = 729/100 - 6 = 129/100\)。\(y_R = m(3 - 129/100) - 5 = (27/10)(171/100) - 5 = 4617/1000 - 5 = -383/1000\)。

所以 \(2 P = (129/100, -383/1000)\)——一个新有理点，但分母变大。继续算 \(3 P, 4 P, \ldots\) 分母会指数级增长。这条曲线 \(E\) 的 \(E(\mathbb{Q})\) 包含 \((3, \pm 5)\) 生成的无穷循环 子群——秩 \(\ge 1\)。

请你停下来感受这两个例子的反差：

  • \(E_1 : y^2 = x^3 - x\)：有理点群有限（4 个）。
  • \(E_2' : y^2 = x^3 - 2\)：有理点群无限。

仅仅 \(b\) 系数从 \(0\) 改到 \(-2\) 让群结构从"有限" 跳到"无限"。哪条曲线给有限群、哪条给无限群——这是椭圆曲线秩问题，是 21 世纪数论最神秘的开放问题。BSD 猜想给一个理论答案，但还没证（链回 ch20 + 千禧年问题）。

19.8 整数点 vs 有理点

椭圆曲线上有两种"解" 的概念，一般差距巨大。

整数点：\(E(\mathbb{Z}) := \{P \in E(\mathbb{Q}) : x_P, y_P \in \mathbb{Z}\}\)。

定理（Siegel, 1929）。设 \(E\) 是 \(\mathbb{Q}\) 上椭圆曲线（\(\Delta \neq 0\)）。则 \(E(\mathbb{Z})\) 有限。

证明用Diophantine 逼近——相当深的工具，不展开。这条结果对一般"三次方程整数解有限" 给一个统一原因。如果你想"枚举所有整数点"，原则上有算法（界从 Baker 等人的有效估计来），实际计算可行。

例：\(E : y^2 = x^3 - 2\) 的整数点是 \(\{(3, \pm 5)\}\)（加上无穷远点）。对应有理点群是无穷——但整数 点只这两个。

有理点：\(E(\mathbb{Q})\)。Mordell 1922 给：

定理（Mordell, 1922）。\(E(\mathbb{Q})\) 是有限生成阿贝尔群。

结合抽象代数 ch8有限生成阿贝尔群分类：

\[ E(\mathbb{Q}) \cong \mathbb{Z}^r \oplus T, \]

\(T\) 是有限挠子群，\(r \ge 0\) 是秩。

  • \(T\) 容易计算（Mazur 定理给所有可能 \(T\)）。
  • \(r\) 难计算——没有已证的算法保证终止。这是椭圆曲线最神秘的不变量。

下一章 ch20 是这一定理的证明 + \(r\) 问题的现代讨论 + BSD 猜想。Mordell-Weil 是椭圆曲线理论的主定理，标志椭圆曲线从一个特殊几何对象上升为现代数论的核心研究对象。

这一章的 take-away：椭圆曲线是有群结构的 Diophantine 方程。"找有理点" 这个问题被翻译为"研究群 \(E(\mathbb{Q})\)"——一个具体的代数对象。下一章我们看这个群有多大。
