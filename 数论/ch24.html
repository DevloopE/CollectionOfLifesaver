第 24 章 Fermat 大定理与 Wiles 路径

这是这本书最后一章——给现代数论的最高峰一个轮廓。1637 年某天 Fermat 在他的 Diophantus 译本（II 卷 Q8 旁）写下：

"方程 \(x^n + y^n = z^n\) 对 \(n \ge 3\) 没有正整数解。我有一个奇妙的证明，可惜书边太窄写不下。"

358 年后，1995 年 5 月，Andrew Wiles 在 Annals of Mathematics 发表 130 页论文 Modular Elliptic Curves and Fermat's Last Theorem，证完。证明动用了几乎 20 世纪整个现代数论：椭圆曲线、模形式、Galois 表示、类域论、变形理论（Mazur）、Hecke 代数。Fermat 那"奇妙的证明" 几乎可以肯定不存在——他可能想到的方法对小 \(n\) 有效但对一般 \(n\) 失败。

这一章不 证细节——细节需要一年研究生课。这一章给战略：从 Fermat 的方程到 Wiles 的核心引理，整条逻辑链怎么扣起来，每一环用本书前几章哪一块的工具。

24.1 FLT 陈述 + 358 年部分进展

定理（Fermat 大定理 / FLT）。方程

\[ \boxed{\ x^n + y^n = z^n\ } \]

对整数 \(n \ge 3\) 没有正整数解 \((x, y, z)\)。

显然 \(n = 1\) 平凡（任何 \(x + y = z\) 都解）；\(n = 2\) 是毕达哥拉斯三元组，有无穷解（\((3, 4, 5), (5, 12, 13), \ldots\)）。FLT 说从 \(n = 3\) 起情况完全相反。

"缩归一" 观察：若 FLT 对某个 \(n\) 成立，则对所有 \(n\) 的倍数成立。原因——若 \(x^{kn} + y^{kn} = z^{kn}\) 即 \((x^k)^n + (y^k)^n = (z^k)^n\)，所以 \(n\) 上有解给 \(kn\) 上有解（取 \(k\) 次方）。所以只需证 FLT 对 \(n = 4\) 与所有奇素数 \(n = p\) 成立。

358 年部分进展（节选）。

  • \(n = 4\)：Fermat 自己证（用无穷下降法）。这是他给出过证明 的唯一情况。
  • \(n = 3\)：Euler 1770 给。证明用\(\mathbb Z[\sqrt{-3}]\) 的算术——但 Euler 没意识到这个环不是 UFD，证明有一个缝，后由 Gauss 补上。这是第一次 "整数 + 一个新虚数" 的代数数论手法应用——直接催生了ch13的高斯整数研究。
  • \(n = 5\)：Dirichlet + Legendre 1825 独立证。
  • \(n = 7\)：Lamé 1839 证（极其复杂的初等手法）。
  • Sophie Germain 1820 年代：证 FLT "第一情形" 对所有满足"\(2p + 1\) 也素" 的Sophie Germain 素数 \(p\) 成立。第一情形 = 解中 \(p \nmid xyz\) 的情况。她是少数能在 19 世纪学术圈做出这种水平工作的女性数学家。
  • Kummer 1847 年：证 FLT 对所有正则素数（regular primes）成立——\(p\) 是正则的 \(\iff p\) 不整除圆分域 \(\mathbb Q(\zeta_p)\) 的类数。这是类域论 的源头——Kummer 通过研究分歧素的整数环结构 引入理想数（ideal number），这是 Dedekind 后来"理想"的雏形。Kummer 估计有"无穷多"正则素数，但这条至今未证（是的，正则素数无穷性比 FLT 还难）。然而所有 \(p < 100\) 中除 \(37, 59, 67\) 都正则——所以 Kummer 的方法实际证 FLT 对几乎所有"中小" 素数。
  • 1980 年代之前：用计算机验证 FLT 对 \(n \le 4 \times 10^6\) 成立。但这只是"大量验证"，没有证明所有 \(n\)。

到 1985 年，FLT 是数学最古老的已陈述未解问题 之一（仅次于 Goldbach 1742 与孪生素数猜想 1849）。它有"每个十年都被声称证明，每次都被找出错" 的悲剧史——19 世纪 Lamé 以为自己证完，半小时被 Liouville 指出"你假设了 \(\mathbb Z[\zeta_p]\) 是 UFD"。这个错正好是 Kummer 看到的类数 问题，反讽地推动了数论的现代化。

24.2 1985 Frey 关键想法

故事真正改变是 1985 年。Gerhard Frey 在德国 Saarbrücken 大学的会议上提出一个意外的归约：

Frey 的关键观察。假设 FLT 反例存在：有素数 \(p \ge 5\) 与互素的非零整数 \(a, b, c\) 使

\[ a^p + b^p + c^p = 0. \]

（这里我把 \(z^p\) 移到左边，等价于原 FLT。）从这个反例构造椭圆曲线：

\[ \boxed{\ E_{a, b, c} : \ y^2 = x(x - a^p)(x + b^p).\ } \]

这条曲线（基于 \(a, b, c\) 反例）叫Frey 曲线。

它是普通的椭圆曲线（链回ch19）——形式 \(y^2 = x^3 + Ax + B\) 经一次仿射变换后是 \(y^2 = x(x - a^p)(x + b^p)\)。具体的：右边是 \(x\) 的 3 次首一多项式，三个根 \(0, a^p, -b^p\) 互不相同（因 \(a, b \ne 0\)、\(p\) 奇、\(a, b\) 互素），所以是非奇异 椭圆曲线。

Frey 的观察：这条曲线"太奇怪" 以至于"不能存在"——具体的"奇怪" 性质下节给。

请你停下来感受这个跳：

  • FLT 是关于整数的陈述：\(x^p + y^p = z^p\) 没整数解。
  • Frey 把它转换 成关于椭圆曲线的陈述：某种特殊椭圆曲线不能存在。

这条转换没有解决 FLT；它把 FLT 重写成另一个等价问题。但新形式 可以用 20 世纪发达的椭圆曲线理论 工具攻击——这是 1985 年前不可能的。

24.3 Frey 曲线的"奇怪" 性质

椭圆曲线 \(E\) 的几个数论数据：

  • 判别式 \(\Delta_E\)：从方程系数算的整数；它告诉你 \(E\) "在哪些素 \(p\) 处有坏化简"——\(p \mid \Delta\) 时 \(E\) 模 \(p\) 退化为奇异曲线。
  • 导子 \(N_E\)：与 \(\Delta\) 同样素因子，但每个素的指数减小——准确测量"坏化简的复杂程度"。

对 Frey 曲线 \(E_{a,b,c}\) 算判别式（这步是工程级技术，不展开）：

\[ \boxed{\ \Delta_{E_{a,b,c}} = 16 (abc)^{2p}.\ } \]

"2p 次完全幂"——这是 Frey 曲线最奇特的性质。

正常椭圆曲线的判别式不会是整体的高次幂——\(\Delta_E\) 一般是各种素的不同指数的乘积。对 Frey 曲线每个素都恰好是 \(2p\) 次方（除少数有 \(2\) 的额外因子）。Frey 称这是"巨大的、不自然的对称性"——它意味着 \(E_{a,b,c}\) 在所有 模 \(\ell\) 处的"表示" 都有异常 简化的结构。

这件事的精确技术含义：椭圆曲线 \(E\) 有 \(\ell\)-adic Galois 表示 \(\rho_{E, \ell} : \mathrm{Gal}(\overline{\mathbb Q}/\mathbb Q) \to \mathrm{GL}_2(\mathbb Z_\ell)\)（即 Galois 群通过 \(E[\ell^\infty]\) "扭点" 的作用）。对一般椭圆曲线 \(E\)，\(\rho_{E, \ell}\) 在 \(\ell \nmid \Delta_E\) 的素 \(\ell\) 处"无分歧"（unramified），对 \(\ell \mid \Delta_E\) 处有分歧。

对 Frey 曲线，由判别式 \(2p\) 次幂的事实，可证：\(\rho_{E_{a,b,c}, p}\)（\(\ell = p\) 的 Galois 表示）"几乎不分歧"——只在 \(\ell = 2\) 处分歧，所有奇素处都无分歧。这种 Galois 表示叫"极小分歧" 表示，它满足非常严格的代数约束。

24.4 Serre + Ribet 1986：Frey 曲线"不模"

1985-1986 年 Jean-Pierre Serre 在 Frey 的观察基础上提出更精确的猜想（叫"\(\varepsilon\) 猜想" 或level lowering 猜想）：如果 Frey 曲线 \(E_{a,b,c}\) 是模的，则它对应的模形式必须是权 2、level 2——但 \(S_2(\Gamma_0(2)) = 0\) 是零空间，所以这种模形式不存在。

翻译：

  • 假设 Frey 曲线 \(E_{a,b,c}\) 模——即对应权 2 模形式 \(f_{E_{a,b,c}} \in S_2(\Gamma_0(N))\)，\(N = N_{E_{a,b,c}}\)。
  • 由 24.3 的"极小分歧" 性质 + Serre 的 level lowering 思想：可以"剥" 掉 \(N\) 中所有奇素因子，把模形式 \(f\) 替换成 \(\Gamma_0(2)\)（即 \(N\) 缩到 2）的同阶模形式 \(g\)。
  • 但 \(S_2(\Gamma_0(2))\) 是零空间（用ch21的维数公式 + \(\Gamma_0(2)\) 的具体计算可验证）。所以 \(g = 0\)——矛盾。

所以"Frey 曲线模" 是不可能的——结论：Frey 曲线不 模。

这是 Ribet 1986 在 \(\varepsilon\) 猜想方向上的具体证明（用Mazur 的模化曲线 / Hecke 代数 工具）。这步本身已经是极重 的论文，但比 Wiles 的工作小一个数量级。Ribet 1986 完成后，逻辑链就被搭好：

\[ \boxed{\ \textbf{Taniyama-Shimura 真} \ \implies \ \textbf{所有椭圆曲线都模} \ \implies \ \textbf{Frey 曲线模} \ \implies \ \textbf{矛盾} \ \implies \ \textbf{FLT 真}.\ } \]

所以 1986 年起，FLT 等价于 Taniyama-Shimura 猜想（至少对一类含 Frey 曲线的椭圆曲线类）。

24.5 Taniyama-Shimura 猜想

ch21 8 节做过预告：

Taniyama-Shimura-Weil 猜想（模性猜想）。每个有理椭圆曲线 \(E\) 都模——存在权 2 Hecke 特征形式 \(f_E \in S_2(\Gamma_0(N_E))\) 使 \(L(E, s) = L(f_E, s)\)。

1955 年 Taniyama 在东京会议上以"问题列表" 形式提出。Shimura 1960 年代精细化。Weil 1967 证明了一些必要条件并推动了它在西方的接受。但没有人 知道怎么证明 它——1980 年代时它仍是一条几乎无人下手的猜想。

直观上为什么它"应该 真"：

  • 椭圆曲线给的 \(L(E, s) = \prod_p (1 - a_p p^{-s} + p^{1-2s})^{-1}\)（\(a_p = p + 1 - \#E(\mathbb F_p)\)）有Euler 乘积，结构上像模形式 \(L\) 函数。
  • 椭圆曲线在 \(\mathbb F_p\) 上的"表示论"（即 Galois 群在 \(E[\ell^\infty]\) 上的作用）与模形式的 Hecke 代数有相同的抽象代数结构——两边都是某种2 维 Galois 表示。
  • 实验：每条具体椭圆曲线都能找到对应的模形式——但只对已计算的有限多条，理论上仍需证。

Taniyama-Shimura 是 Langlands 程序"非阿贝尔类域论"（链回ch22）在 \(n = 2\) 时的具体实例。它说"2 维 Galois 表示\(\leftrightarrow\) 模形式"——这是 Langlands 函子性的一个 instance。

24.6 Wiles 1995：证 Taniyama-Shimura 对半稳定椭圆曲线

1986 年 Frey-Ribet 把 FLT 归约到 Taniyama-Shimura（对一类椭圆曲线）。同年，Andrew Wiles 在普林斯顿听到 Ribet 工作后下决心 证 Taniyama-Shimura——他不 公开做这件事，独自 在阁楼工作 7 年（1986-1993）。

Wiles 的策略：不证完整 Taniyama-Shimura（太难），只证它对"半稳定" 椭圆曲线——这是包含 Frey 曲线的子类，足以证 FLT。具体的：

半稳定椭圆曲线（semistable elliptic curve）：导子 \(N_E\) 是无平方 的（每个素因子只出现 1 次）。Frey 曲线满足这条。Wiles 要证：所有半稳定椭圆曲线模。

核心策略 — R = T 定理（Wiles 的发明）。这条定理是 Wiles 工作的代数核心。它的陈述：

\[ \boxed{\ R \cong T,\ } \]

其中：

  • \(R\) 是变形环（deformation ring）—— Mazur 1989 引入：参数化某种"满足约束的 Galois 表示族"。\(R\) 的具体构造用 变形理论（deformation theory of Galois representations），是抽象代数里"形式群方案" 的工具。
  • \(T\) 是Hecke 代数（Hecke algebra）—— 由 Hecke 算子 \(T_p\)（链回ch21 6 节）作用在某固定模形式空间上生成的代数。具体的：取权 2、level \(N\)、character 一致的模形式空间 \(S_2(\Gamma_0(N))\)，让所有 \(T_p\) 作用在它上面，所有这些算子的多项式组合构成 \(T\)。

"\(R \cong T\)" 的含义：每个"满足约束的 Galois 表示" 都来自 一个模形式（Hecke 代数的元素 = 模形式特征值）。换句话说，所有满足约束的 Galois 表示都模——也就是 Taniyama-Shimura 的精化版本。

Wiles 1993 年 6 月在剑桥 Newton 研究所宣布证明（"这或许是最后一次发声了"）。报告震动数学界——但接下来审稿过程发现一个缝：在论证 \(R \cong T\) 的某一步用到一条还没证完的引理。

1993 年秋到 1994 年秋，Wiles 与他的学生 Richard Taylor 一起工作，发明了一种不同的"欧拉系"（Euler system）方法绕开缝。1994 年 9 月 Wiles 给 Taylor 看新的稿件——这次缝补好了。1995 年 5 月《Annals of Mathematics》发表两篇论文：Wiles 单独的 130 页《Modular Elliptic Curves and Fermat's Last Theorem》+ Wiles-Taylor 合作的 30 页《Ring-Theoretic Properties of Certain Hecke Algebras》（补缝部分）。

FLT 358 年来悬案 终于结束。Fermat 的 marginal note 至此被证明——但用的工具是 Fermat 1637 年完全无法想象的现代数论：椭圆曲线、模形式、Galois 表示、变形理论、Iwasawa 理论、Hecke 代数。

完整 Taniyama-Shimura 猜想（不限于半稳定椭圆曲线）由 Breuil-Conrad-Diamond-Taylor（BCDT）2001 年在 \(J. Amer. Math. Soc.\) 完成。从此每条有理椭圆曲线都已知 模——这条事实今天叫模性定理（不再是猜想）。

24.7 现代数论的全景：Galois 表示 + 自守形式 + Langlands

Wiles 工作之后 30 年（1995-2025），数论的核心研究问题有三大方向：

方向 1：扩展模性。Taniyama-Shimura 处理 \(\mathbb Q\) 上的椭圆曲线。把它推广到：

  • 全实域（totally real field）上的椭圆曲线。Khare-Wintenberger 2009 证 Serre 的"模性猜想" 完整版，作为这方向的基础。
  • 更高维的阿贝尔簇（abelian variety）。这是 21 世纪研究最活跃的方向之一。
  • 更广泛的"motives"（Grothendieck 1960 年代提出的 \(L\)-函数的"动机" 来源）。Fontaine-Mazur 猜想是这方向的纲领。

方向 2：Langlands 程序。把"2 维 Galois 表示\(\leftrightarrow\) 模形式" 推广到 \(n\) 维：

  • \(n\) 维 Galois 表示 \(\rho : \mathrm{Gal}(\overline{\mathbb Q}/\mathbb Q) \to \mathrm{GL}_n(\mathbb C)\) 应该对应\(\mathrm{GL}_n\) 上的自守表示——给出 \(L(\rho, s) = L(\pi, s)\) 的相同 \(L\)-函数。
  • Arthur 2013 完成\(\mathrm{GL}_n\) 在数域上的内禀稳定迹公式，把 \(\mathrm{GL}_n\) 范畴里 Langlands 函子性的"主体" 大体上证明了——但具体某些非阿贝尔扩张的对应仍未完成。
  • \(p\)-adic Langlands 程序（Breuil 2003 起）：\(p\)-adic Galois 表示↔\(p\)-adic 自守表示。这是 \(p\)-adic 解析（链回ch18）与代数几何的现代综合。

方向 3：BSD 猜想。Birch-Swinnerton-Dyer 猜想（1965 提出，Clay 千禧年问题之一）：

\[ \mathrm{rank}\, E(\mathbb Q) = \mathrm{ord}_{s = 1} L(E, s). \]

椭圆曲线的有理点的秩（链回ch20Mordell-Weil）等于 它的 \(L\)-函数在 \(s = 1\) 的零点阶。这是连接几何（点的群结构）与解析（\(L\)-函数）的"中央猜想"。已知部分：Coates-Wiles 1977 对 \(\mathrm{ord} = 0\) 时的"弱 BSD"、Gross-Zagier-Kolyvagin 1989 对 \(\mathrm{ord} \le 1\) 时。\(\mathrm{ord} \ge 2\) 仍未解。

这三方向互相交织——每一个的进展推进其他两个。它们共同的工具：Galois 表示、自守形式、模形式空间的 Hecke 代数、变形理论、\(p\)-adic 分析、代数几何（motivic cohomology）。这些工具的每一种 都需要本科数学之外 2-3 年的研究生学习。它们是 21 世纪数论的语言。

24.8 这本书结束

这本书 24 章把"整数" 这个最朴素的数学对象从最初的"\(7^{100}\) 末位是几" 一路带到 Wiles 路径的全图景。中间走过：

  • 初等数论（ch1-12）：模算术、素数、二次互反、Pell 方程。这一段你高中代数功底就能读，是其余所有内容的地基。
  • 代数数论（ch13-17）：高斯整数、算术函数、Dirichlet 卷积、二元二次型、类数。这一段你需要抽象代数知识。它把"整数" 嵌入更大的代数结构里，让你看到素分解 在普通环里失败、又怎么用理想 与类群 修复。
  • 解析数论（ch16, 18）：素数定理思路、Riemann zeta、\(p\)-adic 数。\(\zeta\) 函数把"整数" 联系到复分析——一个直到 19 世纪中叶都没人想到的桥。
  • 椭圆曲线 + 模形式（ch19-21）：从"三次方程" 到"群律" 到"有限生成群" 到"上半平面对称函数"。每一步都是 19-20 世纪某位伟大数学家的发现。
  • 类域论 + Langlands（ch22）：把所有数论 看作一个对应——"Galois 群\(\leftrightarrow\) 自守表示"。这条对应阿贝尔部分（即类域论）已完成；非阿贝尔部分（即一般 Langlands）是 21 世纪研究的核心。
  • 应用 + 终章（ch23-24）：密码学是数论今天的用途；Wiles 路径是数论今天的最高峰。

请你停下来感受"整数" 这个对象的分量。

  • 它是你最早接触的数学对象——在你学加法之前你已经会数 1, 2, 3。
  • 它有三千年 的研究史——从 Euclid（公元前 300 年）到 Wiles（公元 1995 年）。
  • 它需要20 世纪几乎所有现代数学——抽象代数、复分析、\(p\)-adic 分析、代数几何、表示论——才能被深刻理解。
  • 它的21 世纪研究 仍在进行——Langlands 程序、BSD、Riemann 假设——每一个都是数学最大问题之一。

"朴素表面 vs 深层结构" 这条反差第 1 章就讲了——20 多章后你应该看见它的真正分量。整数不是"初等" 数学；它是数学里最深的对象之一，因为它把所有现代数学工具吸引来描述自己。

1801 年 Gauss 19 岁在日记本上写："数学是科学的女王，数论是数学的女王。"

2025 年这条话还成立。

3000 年来，每一代数学家都觉得"整数终于被理解了"，下一代又发现新一层。这条规律不会停——你今天读完这本书，几十年后某位学生会读一本"普林斯顿数论读本 V20"，把 Wiles 之后的 Langlands 完成、BSD 解决、Riemann 假设证完都讲清楚——而那本书的第 25 章仍会写"这条线还在前进"。整数的故事没有结束，因为结束的从来不是数学，结束的是这一代人对数学的理解。

Wiles 1995 年关于 Fermat 大定理的访谈最后一句话："有些问题陪伴你一辈子。它们不一定是最重要的；但它们是属于你的——你和它们之间有一种私人的关系。Fermat 大定理是这种问题之一。"

希望你读完这本书后，"整数" 在你心里也变成这样一个对象——你的一辈子的伙伴，永远比你想象深，永远比上一次理解的更深。

这本书结束了。下面是附录 A 群环域速查与附录 B 复分析与解析数论速查，把全书用到的代数 + 复分析背景做一个集中的复习；最后是主题索引。如果你想深入哪一块，《抽象代数读本》、《复分析读本》、Silverman The Arithmetic of Elliptic Curves、Diamond-Shurman A First Course in Modular Forms、Cox Primes of the Form \(x^2 + ny^2\) 这几本书是最好的下一站。

祝你下一段旅途——比这本书走得更远。
