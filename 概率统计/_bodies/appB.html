附录 B 常用分布速查

本附录列出本书涉及的 16 种最常见概率分布的"速查表"——分布名、参数、PMF/PDF、期望、方差、矩生成函数（MGF）、典型应用场景。复习考试或写代码时翻一翻即可。

请你注意：

  • 表中"\(B(\alpha, \beta)\)" 是 Beta 函数，"\(\Gamma(\alpha)\)" 是 Gamma 函数。
  • "MGF" 列：若 MGF 不存在或表达式过于复杂，标 "—"。
  • 对多元/矩阵分布，期望和方差栏给出向量/矩阵形式。

B.1 离散分布

分布 参数 PMF 期望 方差 MGF 典型应用
Bernoulli \(p \in [0, 1]\) \(P(X = 1) = p\)，\(P(X = 0) = 1 - p\) \(p\) \(p(1 - p)\) \(1 - p + p e^t\) 抛硬币、单次试验是否成功
二项 Binomial \(n \in \mathbb{N}\)，\(p \in [0, 1]\) \(\binom{n}{k} p^k (1 - p)^{n - k}\)，\(k = 0, 1, \ldots, n\) \(np\) \(np(1 - p)\) \((1 - p + p e^t)^n\) \(n\) 次独立同分布试验中成功次数；质控；A/B 测试
几何 Geometric \(p \in (0, 1]\) \(P(X = k) = (1 - p)^{k - 1} p\)，\(k = 1, 2, \ldots\) \(\dfrac{1}{p}\) \(\dfrac{1 - p}{p^2}\) \(\dfrac{p e^t}{1 - (1 - p) e^t}\)，\(t < -\ln(1 - p)\) 第一次成功所需试验次数；等待时间
负二项 NegBin \(r \in \mathbb{N}\)，\(p \in (0, 1]\) \(\binom{k - 1}{r - 1} p^r (1 - p)^{k - r}\)，\(k = r, r + 1, \ldots\) \(\dfrac{r}{p}\) \(\dfrac{r(1 - p)}{p^2}\) \(\left(\dfrac{p e^t}{1 - (1 - p) e^t}\right)^r\) 第 \(r\) 次成功所需试验次数；过度离散计数数据建模
Poisson \(\lambda > 0\) \(P(X = k) = \dfrac{\lambda^k e^{-\lambda}}{k!}\)，\(k = 0, 1, 2, \ldots\) \(\lambda\) \(\lambda\) \(\exp(\lambda(e^t - 1))\) 稀有事件计数；电话呼叫数；放射衰变；车流量
超几何 Hypergeometric \(N, K, n \in \mathbb{N}\)，\(K \leq N\)，\(n \leq N\) \(\dfrac{\binom{K}{k}\binom{N - K}{n - k}}{\binom{N}{n}}\) \(\dfrac{nK}{N}\) \(\dfrac{nK(N - K)(N - n)}{N^2(N - 1)}\) — 不放回抽样；扑克牌问题；总体小时的质控
多项 Multinomial \(n\)，\(\mathbf{p} = (p_1, \ldots, p_k)\)，\(\sum p_i = 1\) \(\dfrac{n!}{x_1! \cdots x_k!} \prod_i p_i^{x_i}\)，\(\sum x_i = n\) \((np_1, \ldots, np_k)\) \(\Sigma_{ij} = -np_i p_j\)（\(i \ne j\)），\(\Sigma_{ii} = np_i(1 - p_i)\) — 分类计数；卡方拟合优度；语料库词频

B.2 连续分布

分布 参数 PDF 期望 方差 MGF 典型应用
均匀 Uniform \(a < b\) \(\dfrac{1}{b - a}\)，\(x \in [a, b]\) \(\dfrac{a + b}{2}\) \(\dfrac{(b - a)^2}{12}\) \(\dfrac{e^{tb} - e^{ta}}{t(b - a)}\) 无信息先验；随机数生成；输入采样
指数 Exponential \(\lambda > 0\) \(\lambda e^{-\lambda x}\)，\(x \geq 0\) \(\dfrac{1}{\lambda}\) \(\dfrac{1}{\lambda^2}\) \(\dfrac{\lambda}{\lambda - t}\)，\(t < \lambda\) 无记忆等待时间；可靠性；Poisson 过程间隔时间
Gamma \(\alpha > 0\)，\(\beta > 0\)（率） \(\dfrac{\beta^\alpha}{\Gamma(\alpha)} x^{\alpha - 1} e^{-\beta x}\)，\(x > 0\) \(\dfrac{\alpha}{\beta}\) \(\dfrac{\alpha}{\beta^2}\) \(\left(\dfrac{\beta}{\beta - t}\right)^\alpha\)，\(t < \beta\) \(\alpha\) 次指数等待时间和；Poisson 率的 Bayes 共轭先验；保险理赔
正态 Normal \(\mu \in \mathbb{R}\)，\(\sigma^2 > 0\) \(\dfrac{1}{\sigma\sqrt{2\pi}} e^{-(x - \mu)^2 / (2\sigma^2)}\) \(\mu\) \(\sigma^2\) \(\exp(\mu t + \sigma^2 t^2 / 2)\) 测量误差；样本均值的极限分布（CLT）；几乎一切回归的前提
Beta \(\alpha, \beta > 0\) \(\dfrac{x^{\alpha - 1}(1 - x)^{\beta - 1}}{B(\alpha, \beta)}\)，\(x \in [0, 1]\) \(\dfrac{\alpha}{\alpha + \beta}\) \(\dfrac{\alpha\beta}{(\alpha + \beta)^2(\alpha + \beta + 1)}\) —（复杂超几何） Bernoulli/二项的共轭先验；比率/百分比建模
卡方 \(\chi^2\) \(k \in \mathbb{N}\)（自由度） \(\dfrac{1}{2^{k/2}\Gamma(k/2)} x^{k/2 - 1} e^{-x/2}\)，\(x > 0\) \(k\) \(2k\) \((1 - 2t)^{-k/2}\)，\(t < 1/2\) 样本方差的分布；卡方拟合优度检验；ANOVA
\(t\) 分布 Student \(\nu > 0\)（自由度） \(\dfrac{\Gamma((\nu + 1)/2)}{\sqrt{\nu\pi}\Gamma(\nu/2)} \left(1 + \dfrac{x^2}{\nu}\right)^{-(\nu + 1)/2}\) \(0\)（\(\nu > 1\)） \(\dfrac{\nu}{\nu - 2}\)（\(\nu > 2\)） — 样本均值的标准化分布（小样本，未知方差）；\(t\) 检验
\(F\) 分布 \(d_1, d_2 \in \mathbb{N}\)（两个自由度） \(\dfrac{1}{B(d_1/2, d_2/2)} \cdot (d_1/d_2)^{d_1/2} \cdot \dfrac{x^{d_1/2 - 1}}{(1 + d_1 x / d_2)^{(d_1 + d_2)/2}}\)，\(x > 0\) \(\dfrac{d_2}{d_2 - 2}\)（\(d_2 > 2\)） \(\dfrac{2 d_2^2 (d_1 + d_2 - 2)}{d_1 (d_2 - 2)^2 (d_2 - 4)}\)（\(d_2 > 4\)） — 两组样本方差比；ANOVA；回归整体显著性
多元正态 MVN \(\boldsymbol\mu \in \mathbb{R}^k\)，\(\Sigma\) 正定 \(\dfrac{1}{\sqrt{(2\pi)^k |\Sigma|}} \exp\left(-\tfrac{1}{2}(\mathbf{x} - \boldsymbol\mu)^T \Sigma^{-1}(\mathbf{x} - \boldsymbol\mu)\right)\) \(\boldsymbol\mu\) \(\Sigma\)（协方差矩阵） \(\exp(\boldsymbol\mu^T \mathbf{t} + \tfrac{1}{2}\mathbf{t}^T \Sigma \mathbf{t})\) 多元数据；线性回归误差；Kalman 滤波；高斯过程

B.3 关键关系与速记

下面是几条经常用到的"分布之间的关系"，记住它们能帮你少推导很多。

  • Bernoulli \(\to\) 二项：\(n\) 个 iid Bernoulli\((p)\) 之和服从 Binomial\((n, p)\)。
  • 几何 \(\to\) 负二项：\(r\) 个 iid Geometric\((p)\) 之和服从 NegBin\((r, p)\)。
  • 指数 \(\to\) Gamma：\(\alpha\) 个 iid Exp\((\beta)\) 之和服从 Gamma\((\alpha, \beta)\)。
  • Poisson 极限定理：当 \(n \to \infty\)、\(p \to 0\)、\(np \to \lambda\) 时，Binomial\((n, p) \to\) Poisson\((\lambda)\)。
  • 正态 \(\to\) 卡方：若 \(Z_1, \ldots, Z_k\) iid \(N(0, 1)\)，则 \(\sum_{i=1}^k Z_i^2 \sim \chi^2_k\)。
  • 正态 + 卡方 \(\to t\)：\(\dfrac{Z}{\sqrt{V/\nu}}\) 服从 \(t_\nu\)，其中 \(Z \sim N(0, 1)\)、\(V \sim \chi^2_\nu\) 独立。
  • 两个卡方 \(\to F\)：\(\dfrac{V_1/d_1}{V_2/d_2}\) 服从 \(F_{d_1, d_2}\)，其中 \(V_i \sim \chi^2_{d_i}\) 独立。
  • Beta \(\to\) 均匀：Beta\((1, 1) =\) Uniform\([0, 1]\)。
  • Gamma \(\to\) 卡方：Gamma\((k/2, 1/2)\) = \(\chi^2_k\)（形状-率参数化下）。
  • 多项 \(\to\) 二项：Multinomial 的边际分量是二项。
  • MVN \(\to\) 正态：MVN 的任一线性组合 \(\mathbf{a}^T \mathbf{X}\) 是一元正态。

B.5 临界值速记（常用）

做假设检验和置信区间时常需要的几个数（保留四位有效数字）：

  • 标准正态分位数：\(z_{0.05} = 1.645\)，\(z_{0.025} = 1.960\)，\(z_{0.01} = 2.326\)，\(z_{0.005} = 2.576\)。
  • \(t_{n-1}\) 分位数（双尾 95%，即 \(t_{0.025}\)）：\(n = 5\) 时 2.776；\(n = 10\) 时 2.262；\(n = 30\) 时 2.045；\(n = 100\) 时 1.984；\(n \to \infty\) 时 1.960（与 \(z\) 一致）。
  • \(\chi^2_k\) 分位数（95%）：\(k = 1\) 时 3.841；\(k = 5\) 时 11.07；\(k = 10\) 时 18.31；\(k = 20\) 时 31.41。
  • \(F\) 分布临界值（95%）：依赖两个自由度，请查 \(F\) 表或软件函数（R 的 qf(0.95, d1, d2)）。
