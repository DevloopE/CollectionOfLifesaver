第 4 章 离散随机变量与分布

到这里你已经熟悉了样本空间 \(\Omega\)、事件 \(A \subset \Omega\)、概率 \(P(A)\) 三件东西。它们是概率论的"骨架"，但是骨架不会动——你必须给它装上"肉"，才能拿来描述真实问题里的数量。比方说掷两枚骰子，你关心的常常不是"出现 (3, 5) 这一具体结果"，而是"两枚骰子的点数之和等于 8"或者"出现的最大点数是几"。这些"我关心的数"，本质上都是把样本空间里的每个结果映射成一个实数——这就是随机变量。

本章要做这几件事：

  • 给"随机变量"一个干净的定义；
  • 区分离散和连续，本章只讲离散——下一组章节再讲连续；
  • 定义两个最常用的描述工具：概率质量函数 PMF 和累积分布函数 CDF；
  • 定义两个最常用的"汇总数字"：期望（均值）和方差（标准差）；
  • 把这些定义在抛硬币、掷骰、抽签三类经典问题里跑一遍。

这一章本质上是语言课——你正在学一套今后要天天用的"概率语言"。后面第 5 章把这套语言用在六七种最重要的具体分布上；第 7 章以后就把它推广到连续情形。所以请你慢一点，把这章的每个定义都自己用纸笔验证一遍——这是后面所有内容的"地基"。

4.1 随机变量：从样本空间到实数线的函数

先举一个最朴素的例子。掷一枚硬币三次，样本空间是

\[ \Omega = \{HHH, HHT, HTH, THH, HTT, THT, TTH, TTT\}, \]

八个等可能结果。现在你关心什么呢？也许你关心"正面出现的总次数"，记作 \(X\)。这个 \(X\) 显然不是一个固定数字——它取决于实际抛出来的是哪一种结果：如果掷出 \(HHH\)，\(X = 3\)；如果掷出 \(HTT\)，\(X = 1\)；如果掷出 \(TTT\)，\(X = 0\)。也就是说，\(X\) 是一条规则，它给样本空间里每个 \(\omega \in \Omega\) 指派了一个实数：

\[ X(HHH) = 3, \quad X(HHT) = 2, \quad X(HTH) = 2, \quad X(THH) = 2, \] \[ X(HTT) = 1, \quad X(THT) = 1, \quad X(TTH) = 1, \quad X(TTT) = 0. \]

说穿了——\(X\) 是一个函数，定义域是 \(\Omega\)，值域在实数 \(\mathbb{R}\) 里。这就是随机变量的全部秘密。

定义：随机变量是一个函数 \(X : \Omega \to \mathbb{R}\)。

请你注意这件事——随机变量的"随机"二字，不是说函数本身有什么诡异。函数 \(X\) 一旦定义好了，就是一条确定的映射规则；它"随机"的真正含义是：你事先不知道样本空间中的哪个 \(\omega\) 会被实验"挑出来"，所以你也就事先不知道 \(X(\omega)\) 是几。等实验做完，"骰子已经落地"，\(X\) 的值也就确定了。随机的不是函数，是函数的输入。

有一个语言习惯要立刻熟悉：人们写 "\(X = k\)" 这种表达式时，它实际是一个事件——指 \(\Omega\) 中所有使 \(X(\omega) = k\) 的那些 \(\omega\) 组成的子集：

\[ \{X = k\} = \{\omega \in \Omega : X(\omega) = k\}. \]

所以 "\(P(X = k)\)" 这个写法，意思是"事件 \(\{X = k\}\) 的概率"——也就是，所有让 \(X\) 取到 \(k\) 的那些原始结果的概率之和。回到上面三次抛硬币的例子：事件 \(\{X = 2\}\) 等于 \(\{HHT, HTH, THH\}\)，三个等可能结果，所以 \(P(X = 2) = 3/8\)。同样，\(P(X = 0) = 1/8\)，\(P(X = 1) = 3/8\)，\(P(X = 3) = 1/8\)。检查：四个概率加起来 \(1/8 + 3/8 + 3/8 + 1/8 = 1\)，OK。

同样可以问"\(X \leq 2\)"——这是事件 \(\{X = 0\} \cup \{X = 1\} \cup \{X = 2\}\)，概率 \(7/8\)。或者"\(X > 1\)"——事件 \(\{X = 2\} \cup \{X = 3\}\)，概率 \(4/8 = 1/2\)。任何关于 \(X\) 的"问句"，都自动翻译成一个关于 \(\Omega\) 子集的事件，从而拿到一个概率。这就是为什么有了 \(X\) 之后，我们几乎不再去单独操作 \(\Omega\)——所有问题都在 \(X\) 的"实数版样本空间"上做。这一点是后面所有便利的源头。

用第二个例子加深印象。掷两枚骰子，样本空间 \(\Omega = \{(i, j) : i, j \in \{1, \ldots, 6\}\}\) 共 36 个等可能结果。考虑：

  • \(S(\omega) = i + j\)：两枚点数之和。\(S\) 取值在 \(\{2, 3, 4, \ldots, 12\}\)。
  • \(M(\omega) = \max(i, j)\)：较大点数。\(M\) 取值在 \(\{1, 2, \ldots, 6\}\)。
  • \(D(\omega) = |i - j|\)：点数之差的绝对值。\(D\) 取值在 \(\{0, 1, 2, 3, 4, 5\}\)。
  • \(\mathbf{1}_A(\omega) = 1\) 若 \(i = j\)，否则 \(=0\)：这是事件"两枚相同"的指示函数。

这四个全都是同一个 \(\Omega\) 上的随机变量。同一个实验，可以同时定义任意多个随机变量——它们是从同一份原始数据里抽取出来的不同"侧写"。这个观察，到第 6 章讨论"联合分布"和"独立性"时会再次出现。

4.2 离散随机变量

随机变量按"取值的样子"大致分两类。如果 \(X\) 的取值集合是有限或可数无限的——比如 \(X \in \{0, 1, 2, 3\}\)，或者 \(X \in \{1, 2, 3, \ldots\} = \mathbb{N}\)——那么 \(X\) 叫做离散随机变量。"可数"的意思是能用自然数把它的取值一个一个数清楚；中间不会有"连续的一段"。

如果 \(X\) 取值是某个实数区间，比如 \(X \in [0, 1]\)、\(X \in (0, \infty)\) 或 \(X \in \mathbb{R}\)——那 \(X\) 叫做连续随机变量。这种情形我们留到第 7 章细讲。

本章和下章只讲离散。请你认清以下都是离散：

  • 抛 5 次硬币的正面数 \(X \in \{0, 1, 2, 3, 4, 5\}\)；
  • 掷骰直到首次出现 6 所需的次数 \(X \in \{1, 2, 3, \ldots\}\)；
  • 一小时内电话总机接到的呼叫数 \(X \in \{0, 1, 2, \ldots\}\)；
  • 从一袋 100 个零件里抽 10 个，抽到的次品数 \(X \in \{0, 1, \ldots, 10\}\)。

而下面这些是连续——本章不处理：身高、体重、温度、等候时间、灯泡寿命……都是"取一个实数区间里的值"的随机变量，需要不同的工具（概率密度而不是概率质量）。

你可能会问："我抛 100 次硬币的正面数，明明取值 \(\{0, 1, \ldots, 100\}\) 也不少，为什么算'离散'呢？" 答案是：只要这个集合是可数的——能一个一个数清楚——它就是离散，无论"多"或"少"都不影响。极端例子：泊松分布的取值是 \(\{0, 1, 2, \ldots\}\)，无穷多个数，仍然是离散。区分离散和连续的，不是数目多少，是结构形态。这点请你认准。

4.3 概率质量函数 PMF

对一个离散随机变量 \(X\)，要把"\(X\) 在概率上长什么样"完全描述清楚，只需要列出"它取每个可能值的概率"。这张"取值-概率"对应表，就叫做 \(X\) 的概率质量函数（probability mass function, PMF），记作

\[ \boxed{p_X(k) = P(X = k).} \]

下标 \(X\) 用来标明是哪个随机变量的 PMF（场合明确时常省去）。它是一个普通的实值函数，定义域是 \(X\) 的取值集合，值域在 \([0, 1]\) 里。

PMF 必须满足两条合法性条件，这两条到处会用：

  • 非负：每个 \(p_X(k) \geq 0\)；
  • 归一：\(\sum_{k} p_X(k) = 1\)，求和遍历所有可能取值。

反过来，任何一组满足这两条的数 \(\{p_k\}_k\) 都可以当作某个离散随机变量的 PMF——离散概率论里"分布"和"PMF"事实上是同义词，给一个就给了另一个。

例 4.1（三次抛硬币的正面数）　继续 4.1 节的例子。已经算过 \(P(X = 0) = 1/8\)、\(P(X = 1) = 3/8\)、\(P(X = 2) = 3/8\)、\(P(X = 3) = 1/8\)，所以

\[ p_X(0) = \tfrac{1}{8}, \quad p_X(1) = \tfrac{3}{8}, \quad p_X(2) = \tfrac{3}{8}, \quad p_X(3) = \tfrac{1}{8}. \]

验证：四个非负，加起来 \(1\)，OK。这其实是一个 \(\mathrm{Bin}(3, 1/2)\)——下一章会正式讲二项分布。

例 4.2（两枚骰子点数之和）　设 \(S = i + j\)。每个 \((i, j)\) 等可能、概率 \(1/36\)。要算 \(p_S(k)\)，只需数有几对 \((i, j)\) 使 \(i + j = k\)：

  • \(k = 2\)：只有 \((1, 1)\)，1 对。\(p_S(2) = 1/36\)。
  • \(k = 3\)：\((1, 2), (2, 1)\)，2 对。\(p_S(3) = 2/36\)。
  • \(k = 4\)：\((1, 3), (2, 2), (3, 1)\)，3 对。\(p_S(4) = 3/36\)。
  • \(k = 5\)：4 对，\(p_S(5) = 4/36\)。\(k = 6\)：5 对，\(p_S(6) = 5/36\)。\(k = 7\)：6 对，\(p_S(7) = 6/36\)。
  • \(k = 8\)：5 对，\(p_S(8) = 5/36\)。\(k = 9\)：4 对。\(k = 10\)：3 对。\(k = 11\)：2 对。\(k = 12\)：1 对。

把它们画成柱状图，会得到一个对称的"小山"，山顶在 \(k = 7\)。这就是为什么"两骰之和"赌局里 7 是最常出的——它对应的对子最多。

检查归一：\(1 + 2 + 3 + 4 + 5 + 6 + 5 + 4 + 3 + 2 + 1 = 36\)，所以 \(\sum_k p_S(k) = 36/36 = 1\)，OK。

友情提示：日后看到"PMF"三个字母，请直接把它在脑子里翻译成"取值 \(k\) 概率多少的对应表"。这个直觉一旦稳固，后面所有公式都只是这张表上的求和。

4.4 累积分布函数 CDF

有时你不想问"\(X = k\) 的概率"，而想问"\(X \leq x\) 的概率"——比如你下注时关心"两骰之和不超过 7"。这种"小于等于某个数"的概率，叫做累积分布函数（cumulative distribution function, CDF），记作

\[ \boxed{F_X(x) = P(X \leq x), \qquad x \in \mathbb{R}.} \]

请你注意三个细节：

  1. CDF 是定义在整条实数轴上的——\(x\) 不必恰好是 \(X\) 的某个取值。比如对例 4.2 的 \(S\)，问 \(F_S(7.5)\) 是合法的，意思是 \(P(S \leq 7.5) = P(S \leq 7) = 21/36\)（因为 \(S\) 取整数，"小于等于 7.5"和"小于等于 7"是同一件事）。
  2. 是 "\(\leq\)" 而不是 "\(<\)"。这一字之差对离散随机变量很关键：\(F_X(2)\) 包含 "\(X = 2\)" 那部分概率，而 "\(P(X < 2) = F_X(2) - p_X(2)\)" 不包含。请你养成时刻盯紧不等号方向的习惯。
  3. CDF 把所有"分布信息"都塞进一个单一函数里——给了 \(F_X\) 就等价于给了 PMF（PMF 可以从 CDF 在跳跃点的"跳高"里读出，见下文）。

对离散 \(X\)，CDF 长什么样？我们在例 4.1 上做出来：

\(X\) 只取 \(0, 1, 2, 3\) 四个值，所以 \(F_X(x)\) 是分段常数——一个阶梯函数：

\[ F_X(x) = \begin{cases} 0, & x < 0, \\ 1/8, & 0 \leq x < 1, \\ 4/8, & 1 \leq x < 2, \\ 7/8, & 2 \leq x < 3, \\ 1, & x \geq 3. \end{cases} \]

怎么得到？\(F_X(0)\) 把"\(X \leq 0\)"内的概率全加起来——只有 \(X = 0\) 一个值，所以 \(1/8\)。\(F_X(1)\) 加上 \(X = 1\) 部分，\(1/8 + 3/8 = 4/8\)。\(F_X(2) = 4/8 + 3/8 = 7/8\)。\(F_X(3) = 7/8 + 1/8 = 1\)。在每个整数处，函数"跳一下"——跳的高度恰好就是那个点的 PMF 值。

这条观察很重要，请你正式记下：

\[ \boxed{p_X(k) = F_X(k) - F_X(k^-), \qquad \text{即 \(F_X\) 在 \(k\) 处的"跳跃高度"}.} \]

其中 \(F_X(k^-) = \lim_{x \to k^-} F_X(x)\) 是左极限。所以 PMF 就是 CDF "跳台阶"的高度，这就是离散情形 PMF 与 CDF 的等价关系。

所有 CDF 都享有以下三条万用性质，无论离散还是连续：

  • 单调不减：若 \(x_1 \leq x_2\)，则 \(F_X(x_1) \leq F_X(x_2)\)。直观——区间 \((-\infty, x_1]\) 包含在 \((-\infty, x_2]\) 里，所以前者概率不大于后者。
  • 右连续：\(\lim_{x \to a^+} F_X(x) = F_X(a)\)。这是约定 "\(\leq\)" 不是 "\(<\)" 的直接后果。
  • 边界值：\(\lim_{x \to -\infty} F_X(x) = 0\)，\(\lim_{x \to +\infty} F_X(x) = 1\)。直观——"小于负无穷"是不可能的事件（概率 0），"小于正无穷"是必然事件（概率 1）。

反过来，任何满足这三条的函数都是某个随机变量的 CDF。这件事到第 7 章引入连续分布时会更有用——那时 CDF 反而比 PMF 更基础。

例 4.3（用 CDF 算"区间概率"）　设 \(X\) 是例 4.2 的两骰之和。问 \(P(5 < X \leq 9)\) 是多少？

方法一：直接相加。"\(5 < X \leq 9\)" 即 \(X \in \{6, 7, 8, 9\}\)，从 PMF 表读出 \(p_S(6) + p_S(7) + p_S(8) + p_S(9) = (5 + 6 + 5 + 4)/36 = 20/36 = 5/9\)。

方法二：差出 CDF。\(F_S(9) - F_S(5)\)。先算 \(F_S(5) = P(S \leq 5) = (1 + 2 + 3 + 4)/36 = 10/36\)；再算 \(F_S(9) = P(S \leq 9) = (1+2+3+4+5+6+5+4)/36 = 30/36\)。差为 \(20/36 = 5/9\)。两个方法答案一致——好。

顺便记住一条万用公式：对任意 \(a < b\)，

\[ P(a < X \leq b) = F_X(b) - F_X(a). \]

不要对这条等式中"开闭"位置感到神经质——只要严格按照"开 vs. 闭"对应"减 vs. 不减"的规则去配对，从来不会出错。

4.5 期望（均值）

给定离散随机变量 \(X\)，最经常被问的"汇总数字"就是它的期望（expectation）或叫均值（mean），记作 \(\mathbb{E}[X]\) 或 \(\mu_X\)。它的定义朴素到让人觉得没什么可说：

\[ \boxed{\mathbb{E}[X] = \sum_k k \cdot p_X(k).} \]

求和遍历 \(X\) 所有可能取值。意思就是——把每个取值 \(k\) 用它的"概率权重" \(p_X(k)\) 加权平均。"加权平均"四个字，几乎就是这个公式的全部含义。

例 4.4（一枚骰子的期望）　\(X \in \{1, 2, 3, 4, 5, 6\}\)，每个 \(p_X(k) = 1/6\)。代入：

\[ \mathbb{E}[X] = 1 \cdot \tfrac{1}{6} + 2 \cdot \tfrac{1}{6} + 3 \cdot \tfrac{1}{6} + 4 \cdot \tfrac{1}{6} + 5 \cdot \tfrac{1}{6} + 6 \cdot \tfrac{1}{6} = \tfrac{1 + 2 + 3 + 4 + 5 + 6}{6} = \tfrac{21}{6} = 3.5. \]

3.5——这是一个骰子永远掷不出来的值。这并不是"期望"概念的缺陷，而是它的本质：期望是很多次重复实验后的平均值，不是"最可能掷到的值"。掷一万次骰子，把所有点数加起来除以一万，结果会非常接近 3.5；但单次掷出 3.5 这个值是不可能的。请你分清这两件事。后面的大数定律（第 12 章）会把"很多次的平均接近期望"这件事严格说清楚。

例 4.5（不公正骰子的期望）　假设有人做了一颗"灌铅"骰子，PMF 如下表：

\[ p_X(1) = 0.05, \quad p_X(2) = 0.10, \quad p_X(3) = 0.15, \quad p_X(4) = 0.20, \quad p_X(5) = 0.20, \quad p_X(6) = 0.30. \]

检查归一：\(0.05 + 0.10 + 0.15 + 0.20 + 0.20 + 0.30 = 1.00\)，OK。期望：

\[ \mathbb{E}[X] = 1 \cdot 0.05 + 2 \cdot 0.10 + 3 \cdot 0.15 + 4 \cdot 0.20 + 5 \cdot 0.20 + 6 \cdot 0.30 = 4.20. \]

比公正骰子的 3.5 大——直观上对，因为这颗骰子"偏向大的点数"。

4.5.1 期望可能不存在

等等！如果 \(X\) 的取值集合是无穷的——比如 \(X \in \{1, 2, 3, \ldots\}\)——那 \(\mathbb{E}[X] = \sum_{k=1}^\infty k \cdot p_X(k)\) 是个无穷级数，不一定收敛。如果级数发散到 \(+\infty\)，我们就说"期望是 \(+\infty\)"或者"期望不存在"（取决于约定）。下一章的几何分布、泊松分布的期望都是有限的，但有时候期望真的会是无穷——比如所谓"圣彼得堡悖论"里的赌博收益。这件事请你记一笔账，本章不深入。

4.5.2 线性与单调（直观先记，公式见 4.7）

期望有两条最常用的"全局"性质：

  • 线性：\(\mathbb{E}[X + Y] = \mathbb{E}[X] + \mathbb{E}[Y]\)，\(\mathbb{E}[a X] = a \mathbb{E}[X]\)。这条对任意两个随机变量都成立——不需要它们独立。这是期望最珍贵的性质，下一章用它的次数会让你印象深刻。
  • 单调：若 \(X \leq Y\)（即对每个 \(\omega\) 都有 \(X(\omega) \leq Y(\omega)\)），则 \(\mathbb{E}[X] \leq \mathbb{E}[Y]\)。从加权平均的直观看，这是显然的。

这两条的正式陈述和证明放在 4.7 节，但你先把"线性"这条牢牢记住——后面到处要用。

4.6 期望的几何意义："质量中心"

4.5 节的公式 \(\mathbb{E}[X] = \sum_k k \cdot p_X(k)\) 看上去只是个加权平均，未必让你立刻"看见"它的意思。这里给一个物理直观，请你认真记住，今后碰到任何分布都用得上。

把每个取值 \(k\) 想象成实数轴上的一个点，把概率 \(p_X(k)\) 想象成放在这个点上的质量。这样你就有了一根"加权杠杆"——质量分布在数轴上几个离散的位置。问题是：这根杠杆的质心（重心）在哪里？

答案就是 \(\mathbb{E}[X]\)。质心在物理里就是"质量加权位置的加权平均"，公式 \(\bar x = \sum m_k x_k / \sum m_k\)；当总质量归一（\(\sum m_k = 1\)，正是 PMF 的归一条件）时，分母消失，\(\bar x = \sum x_k m_k\)——这正好就是期望的定义。

这个直观立刻给你三条免费推论：

  • 对称分布的期望在对称中心。如果 PMF 关于某个点 \(c\) 对称（即 \(p_X(c + d) = p_X(c - d)\) 对所有 \(d\) 成立），那 \(\mathbb{E}[X] = c\)。例如公正骰子关于 \(c = 3.5\) 对称，所以 \(\mathbb{E}[X] = 3.5\)（前面算过）；两骰之和关于 \(c = 7\) 对称，所以 \(\mathbb{E}[S] = 7\)。免费——没做求和。
  • 把分布"整体平移"\(b\)，期望也跟着平移 \(b\)。\(\mathbb{E}[X + b] = \mathbb{E}[X] + b\)。质心跟着所有质量一起平移。
  • 把分布"整体放大"\(a\) 倍，期望也放大 \(a\) 倍。\(\mathbb{E}[a X] = a \mathbb{E}[X]\)。在杠杆上把所有位置坐标都乘以 \(a\)，质心位置坐标也乘以 \(a\)。

后两条合起来正是下一节要正式陈述的"期望的运算规则"。

请你记住："期望 = 质心"这个图像。它比代数公式好用得多——好多分布的期望，你看一眼对称性就能写出来，不需要做任何求和。

4.7 期望的运算：仿射变换与线性

把 4.6 节的几何直观正式翻译成代数。设 \(a, b\) 是常数（不是随机变量），\(X\) 是离散随机变量。考虑新的随机变量 \(Y = a X + b\)。它的期望是多少？

\[ \mathbb{E}[a X + b] = \sum_k (a k + b) p_X(k) = a \sum_k k \, p_X(k) + b \sum_k p_X(k) = a \mathbb{E}[X] + b. \]

这里第二步用到了 \(\sum_k p_X(k) = 1\)（PMF 归一）。所以

\[ \boxed{\mathbb{E}[a X + b] = a \mathbb{E}[X] + b.} \]

这条公式叫"仿射变换下期望的变换规则"，今后随手就要用。两个特殊情形：

  • \(b = 0\)：\(\mathbb{E}[a X] = a \mathbb{E}[X]\)，"乘常数可以拿到 \(\mathbb{E}\) 外面"。
  • \(a = 0\)：\(\mathbb{E}[b] = b\)，"常数的期望就是它自己"——好理解：常数本质上是只取一个值的"退化"随机变量。

更一般的线性性是：

\[ \boxed{\mathbb{E}[a X + b Y] = a \mathbb{E}[X] + b \mathbb{E}[Y].} \]

这条对任意两个随机变量 \(X, Y\)都成立，无论它们独立与否。证明在第 6 章定义联合分布之后给——你需要"联合 PMF"的语言才能把它写干净。但请你立刻把它当作"日用工具"记住。

例 4.6（线性性的免费胜利）　考虑掷一颗骰子三次，记三次点数之和为 \(T = X_1 + X_2 + X_3\)。问 \(\mathbb{E}[T]\)。

解：用线性性，\(\mathbb{E}[T] = \mathbb{E}[X_1] + \mathbb{E}[X_2] + \mathbb{E}[X_3] = 3.5 + 3.5 + 3.5 = 10.5\)。完。

请你比较：如果按定义直接算——你要先列出 \(T\) 的 PMF（取值在 \(\{3, 4, \ldots, 18\}\) 共 16 个值，每个对应若干掷法），再做加权和。这个过程要消耗你不少时间。线性性把它压缩成了一行加法。这就是线性性的威力——把一个复杂随机变量的期望，分解为它的"组成部分"的期望之和。

4.8 函数的期望：LOTUS 公式

有时候你想算的不是 \(\mathbb{E}[X]\)，而是 \(X\) 的某个函数 \(g(X)\) 的期望——比如 \(\mathbb{E}[X^2]\)、\(\mathbb{E}[\cos X]\)、\(\mathbb{E}[1/X]\)。一个朴素的办法是先算出新随机变量 \(Y = g(X)\) 的 PMF，再用期望定义。但有一条更直接的公式，把这一步省掉了：

\[ \boxed{\mathbb{E}[g(X)] = \sum_k g(k) \cdot p_X(k).} \]

意思是：用 \(X\) 自己的 PMF 加权函数值 \(g(k)\) 即可，不需要先求 \(g(X)\) 的 PMF。这条公式有个被翻译得很有趣的英文名字——无意识统计学家公式（Law of the Unconscious Statistician, LOTUS）。叫这个名字，是因为很多人不假思索地就这样算，事后才发现"这其实需要证明！"。LOTUS 的证明是：把 \(\sum_k g(k) p_X(k)\) 按"\(g(k) = y\) 的那些 \(k\) 们"重新分组求和，就回到了 \(\sum_y y \cdot P(g(X) = y) = \mathbb{E}[g(X)]\)。详细的"分组重排"留给你做。

例 4.7（骰子点数的平方期望）　\(X\) 是公正骰子，\(\mathbb{E}[X^2] = ?\)

\[ \mathbb{E}[X^2] = \sum_{k=1}^6 k^2 \cdot \tfrac{1}{6} = \tfrac{1}{6}(1 + 4 + 9 + 16 + 25 + 36) = \tfrac{91}{6} \approx 15.17. \]

请你注意：\(\mathbb{E}[X^2] \neq (\mathbb{E}[X])^2\)。这里 \((\mathbb{E}[X])^2 = 3.5^2 = 12.25\)，比 \(\mathbb{E}[X^2] = 91/6 \approx 15.17\) 小。差额恰好是方差——下一节就讲。这条不等式 \(\mathbb{E}[X^2] \geq (\mathbb{E}[X])^2\) 永远成立，等号当且仅当 \(X\) 是常数（即 \(X\) 没有"真正随机性"）。这是一种叫"凸函数 Jensen 不等式"的东西的特殊情形，第 11 章会专门讲。

例 4.8（线性函数的特殊情形）　对 \(g(x) = a x + b\)，LOTUS 给出

\[ \mathbb{E}[a X + b] = \sum_k (a k + b) p_X(k) = a \mathbb{E}[X] + b, \]

正好就是 4.7 节那条公式——LOTUS 的一个特殊情形。这件事说明 4.7 节的"线性性 + 常数加项"全都是 LOTUS 的"线性"特例。

4.9 方差与标准差

期望告诉你"分布的中心在哪里"。但中心没说完整故事——一个分布可能"很集中"（比如几乎确定取 5 这个值），也可能"很分散"（比如有时取 0、有时取 10）。要把"分散程度"度量出来，需要新的概念——方差。

朴素想法：度量"\(X\) 与它的均值 \(\mu = \mathbb{E}[X]\) 之间的偏离"。直接的偏离量是 \(X - \mu\)，但这个偏离可正可负，"取期望"会互相抵消（事实上 \(\mathbb{E}[X - \mu] = \mathbb{E}[X] - \mu = 0\)，永远抵消干净）。所以我们要把它的"符号去掉"再求期望。两种自然办法：

  • 取绝对值——\(\mathbb{E}[|X - \mu|]\)，叫"平均绝对偏差"。这个量从直觉看也合理，但它涉及绝对值，不好做代数运算（不可微等问题）。
  • 取平方——\(\mathbb{E}[(X - \mu)^2]\)，叫"方差"，记作 \(\mathrm{Var}(X)\) 或 \(\sigma_X^2\)。平方后是非负的，且对代数运算非常友好——这是大家最终选择的"分散程度"度量方式。

形式定义：

\[ \boxed{\mathrm{Var}(X) = \mathbb{E}\left[(X - \mu)^2\right], \qquad \mu = \mathbb{E}[X].} \]

方差的单位是 \(X\) 单位的平方——比如 \(X\) 是身高（厘米），方差就是平方厘米。这有时不直观，所以人们也定义

\[ \boxed{\mathrm{SD}(X) = \sigma_X = \sqrt{\mathrm{Var}(X)},} \]

叫做标准差（standard deviation），它和 \(X\) 同单位，更便于和均值放在一起谈。"\(X\) 的均值是 5、标准差是 0.7"是日常统计报告里最常见的一种描述。

请你记住：方差和标准差永远 \(\geq 0\)；它们等于 0 当且仅当 \(X\) 退化为一个常数（即不再"随机"）。所以方差 = 0 是"完全确定"的代名词。

4.10 关键计算公式

用方差的定义 \(\mathrm{Var}(X) = \mathbb{E}[(X - \mu)^2]\) 直接算，需要先算 \(\mu\)，然后再算 \((X - \mu)^2\) 的期望——两次求和。有一条等价公式可以省一道工序：

\[ \boxed{\mathrm{Var}(X) = \mathbb{E}[X^2] - (\mathbb{E}[X])^2.} \]

推导很短，请你跟一遍。设 \(\mu = \mathbb{E}[X]\)。展开平方：

\[ (X - \mu)^2 = X^2 - 2 \mu X + \mu^2. \]

对两边取期望，用 4.7 节的线性性（\(\mu\) 是常数，可以拿到 \(\mathbb{E}\) 外面）：

\[ \mathbb{E}[(X - \mu)^2] = \mathbb{E}[X^2] - 2 \mu \mathbb{E}[X] + \mu^2 = \mathbb{E}[X^2] - 2\mu^2 + \mu^2 = \mathbb{E}[X^2] - \mu^2. \]

把 \(\mu^2\) 换回 \((\mathbb{E}[X])^2\)，就是公式。这条公式的实用价值是：算方差时，你只要算两个数—— \(\mathbb{E}[X]\) 和 \(\mathbb{E}[X^2]\)，然后做减法。比按定义两次求和省事。

例 4.9（公正骰子的方差）　已经算过 \(\mathbb{E}[X] = 3.5\)，\(\mathbb{E}[X^2] = 91/6\)。所以

\[ \mathrm{Var}(X) = \tfrac{91}{6} - 3.5^2 = \tfrac{91}{6} - \tfrac{49}{4} = \tfrac{182}{12} - \tfrac{147}{12} = \tfrac{35}{12} \approx 2.92. \]

标准差 \(\sigma_X = \sqrt{35/12} \approx 1.71\)。所以一颗骰子的点数"以 3.5 为中心、典型偏离量约 1.71"——直觉上对得起这个分布的样子（点数在 \(\{1, \ldots, 6\}\) 范围内）。

这条公式的另一个副产品，是"\(\mathbb{E}[X^2] \geq (\mathbb{E}[X])^2\)" 的一个新看法：因为 \(\mathrm{Var}(X) \geq 0\)，所以 \(\mathbb{E}[X^2] - (\mathbb{E}[X])^2 \geq 0\)。等号成立当且仅当方差 = 0，即 \(X\) 是常数。

4.11 仿射变换下的方差

上一节我们看到 \(\mathbb{E}[a X + b] = a \mathbb{E}[X] + b\)。那么 \(\mathrm{Var}(a X + b)\) 是多少？请你猜一下，再往下看。

把 \(Y = a X + b\) 代入方差定义。新均值 \(\mathbb{E}[Y] = a \mu + b\)（其中 \(\mu = \mathbb{E}[X]\)）。所以

\[ Y - \mathbb{E}[Y] = (a X + b) - (a \mu + b) = a (X - \mu). \]

"加常数 \(b\)" 完全消失了——这是关键。然后

\[ \mathrm{Var}(Y) = \mathbb{E}\left[ \big(a (X - \mu)\big)^2 \right] = \mathbb{E}[a^2 (X - \mu)^2] = a^2 \mathbb{E}[(X - \mu)^2] = a^2 \mathrm{Var}(X). \]

所以

\[ \boxed{\mathrm{Var}(a X + b) = a^2 \mathrm{Var}(X).} \]

注意是 \(a^2\) 不是 \(a\)（因为方差是"平方单位"），而 \(b\) 完全消失。直观上很合理：

  • 给 \(X\) 整体加个常数 \(b\)，所有质量一起平移，形状不变，所以分散程度不变。
  • 给 \(X\) 整体乘以 \(a\)，所有质量在数轴上"按 \(a\) 倍拉伸"，分散程度按 \(a^2\) 倍变化（按 \(|a|\) 倍变化的是标准差）。

对应到标准差：

\[ \mathrm{SD}(a X + b) = |a| \cdot \mathrm{SD}(X). \]

这里出现 \(|a|\) 是因为标准差是非负的开方——\(\sqrt{a^2} = |a|\)。

例 4.10（华氏-摄氏换算的方差）　某地温度 \(X\)（摄氏度）的均值 \(\mathbb{E}[X] = 20\)、方差 \(\mathrm{Var}(X) = 9\)（即标准差 3）。把温度换成华氏度 \(Y = 1.8 X + 32\)，问 \(\mathbb{E}[Y]\) 与 \(\mathrm{Var}(Y)\)。

\(\mathbb{E}[Y] = 1.8 \cdot 20 + 32 = 68\)（华氏度）。\(\mathrm{Var}(Y) = 1.8^2 \cdot 9 = 3.24 \cdot 9 = 29.16\)。标准差 \(\mathrm{SD}(Y) = 1.8 \cdot 3 = 5.4\)。

请注意这个数对得起直觉：温度从 20 摆到 23（标准差 3）的"温度感受"，相当于从 68 摆到 73.4（华氏标准差 5.4）。

4.12 详细例题：抛硬币 / 掷骰 / 抽签

本章工具到齐了。现在我们用它们处理几个"完整问题"——既练习公式、也培养"看到题目知道用什么公式"的本能。

例 4.11（抛硬币：成功次数的分布与汇总数字）　抛一枚正面概率为 \(p = 0.4\) 的硬币 4 次，每次独立。记 \(X\) 为正面出现的总次数。求：(a) \(X\) 的 PMF；(b) \(\mathbb{E}[X]\)；(c) \(\mathrm{Var}(X)\)。

解：(a) 出现"\(k\) 次正面、\(4 - k\) 次反面"这种特定模式的概率是 \(p^k (1 - p)^{4 - k}\)（独立性！）。但是"哪几次正面、哪几次反面"有 \(\binom{4}{k}\) 种顺序——所以

\[ p_X(k) = \binom{4}{k} p^k (1 - p)^{4 - k}, \qquad k = 0, 1, 2, 3, 4. \]

具体代入 \(p = 0.4\)、\(1 - p = 0.6\)：

\[ p_X(0) = 1 \cdot 1 \cdot 0.6^4 = 0.1296, \] \[ p_X(1) = 4 \cdot 0.4 \cdot 0.6^3 = 4 \cdot 0.4 \cdot 0.216 = 0.3456, \] \[ p_X(2) = 6 \cdot 0.4^2 \cdot 0.6^2 = 6 \cdot 0.16 \cdot 0.36 = 0.3456, \] \[ p_X(3) = 4 \cdot 0.4^3 \cdot 0.6 = 4 \cdot 0.064 \cdot 0.6 = 0.1536, \] \[ p_X(4) = 1 \cdot 0.4^4 \cdot 1 = 0.0256. \]

检查归一：\(0.1296 + 0.3456 + 0.3456 + 0.1536 + 0.0256 = 1.0000\)，OK。

这是 \(X \sim \mathrm{Bin}(4, 0.4)\)——下一章正式认识它。

(b) 直接算：\(\mathbb{E}[X] = 0 \cdot 0.1296 + 1 \cdot 0.3456 + 2 \cdot 0.3456 + 3 \cdot 0.1536 + 4 \cdot 0.0256 = 0 + 0.3456 + 0.6912 + 0.4608 + 0.1024 = 1.6\)。

注意到 \(1.6 = 4 \cdot 0.4 = n p\)。这并非巧合——下一章会证 \(\mathrm{Bin}(n, p)\) 的期望永远是 \(n p\)。从直觉上讲：抛 4 次、每次 \(0.4\) 的成功概率，平均得 \(1.6\) 次成功。

(c) 算 \(\mathbb{E}[X^2]\)：\(0 + 1 \cdot 0.3456 + 4 \cdot 0.3456 + 9 \cdot 0.1536 + 16 \cdot 0.0256 = 0.3456 + 1.3824 + 1.3824 + 0.4096 = 3.52\)。

\(\mathrm{Var}(X) = 3.52 - 1.6^2 = 3.52 - 2.56 = 0.96\)。注意 \(0.96 = 4 \cdot 0.4 \cdot 0.6 = n p (1 - p)\)——又是一条以后会证的二项方差公式。

例 4.12（一颗骰子，三种"汇总函数"的期望）　设 \(X\) 是公正骰子，求 \(\mathbb{E}[X]\)、\(\mathbb{E}[(X - 3)^2]\)、\(\mathbb{E}[1/X]\)。

解：\(\mathbb{E}[X] = 3.5\)（前面已算）。

\(\mathbb{E}[(X - 3)^2]\)：用 LOTUS。\((X - 3)^2\) 在 \(X = 1, 2, 3, 4, 5, 6\) 处的值是 \(4, 1, 0, 1, 4, 9\)。所以

\[ \mathbb{E}[(X - 3)^2] = \tfrac{1}{6}(4 + 1 + 0 + 1 + 4 + 9) = \tfrac{19}{6} \approx 3.17. \]

请你注意：这不是方差！方差是 \(\mathbb{E}[(X - \mathbb{E}[X])^2] = \mathbb{E}[(X - 3.5)^2]\)，不是 \(\mathbb{E}[(X - 3)^2]\)。两者差在"用的是均值 3.5 还是别的数 3"。事实上 \(\mathbb{E}[(X - c)^2]\) 作为 \(c\) 的函数，在 \(c = \mathbb{E}[X] = 3.5\) 处取最小值——这就是为什么方差专门用均值做"中心"——它是"把 \(X\) 用一个常数预测、误差平方期望最小的那个常数"。这件事到第 8 章讲条件期望时还会回来。

\(\mathbb{E}[1/X]\)：

\[ \mathbb{E}[1/X] = \tfrac{1}{6}\left(\tfrac{1}{1} + \tfrac{1}{2} + \tfrac{1}{3} + \tfrac{1}{4} + \tfrac{1}{5} + \tfrac{1}{6}\right) = \tfrac{1}{6} \cdot \tfrac{147}{60} = \tfrac{147}{360} \approx 0.408. \]

请你注意：\(\mathbb{E}[1/X] \neq 1/\mathbb{E}[X] = 1/3.5 \approx 0.286\)。这两件事一般不相等。"取期望"和"取倒数"不可交换；推广开来，\(\mathbb{E}[g(X)] \neq g(\mathbb{E}[X])\)，除非 \(g\) 是仿射函数。把 \(X\) 换成 \(g(X)\) 然后取期望，必须用 LOTUS——这是 LOTUS 公式真正不可替代的场合。

例 4.13（抽签问题：放回 vs. 不放回的期望）　从一袋 5 张签（编号 1 至 5）里抽 2 张。让 \(S\) 表示抽到的两个号码之和。分两种情形：

  • (i) 放回抽样——抽出第一张、记下号码、放回袋中、抽第二张。两次独立。
  • (ii) 不放回抽样——抽两张签，不放回。

问 \(\mathbb{E}[S]\) 在两种情形下分别是多少。

解：用线性性。\(S = X_1 + X_2\)，其中 \(X_i\) 是第 \(i\) 张签的号码（\(i = 1, 2\)）。

关键观察：不论放回还是不放回，每张单独看，\(X_i\) 都等可能取 \(\{1, 2, 3, 4, 5\}\) 中任一——所以 \(\mathbb{E}[X_i] = (1+2+3+4+5)/5 = 3\)。这一点在不放回情形需要小心想：第二次抽到号码 \(k\) 的概率是 \(1/5\) 吗？是的——按对称性，"第二张签是 1 号"和"第二张签是 5 号"应该一样可能（你可以想象先洗牌再依次拿，每个号码出现在第二位的可能性相同）。所以两种情形下，单个 \(X_i\) 的 PMF 完全相同。

由线性性：\(\mathbb{E}[S] = \mathbb{E}[X_1] + \mathbb{E}[X_2] = 3 + 3 = 6\)。放回与不放回的期望相同。

这是个值得记住的现象：线性性不要求独立。无论 \(X_1, X_2\) 是否独立，期望可以照样相加——这就是为什么期望工具如此强大。

但是！方差就不一样了：放回时 \(\mathrm{Var}(S) = \mathrm{Var}(X_1) + \mathrm{Var}(X_2) = 2 \mathrm{Var}(X_1)\)（独立）；不放回时 \(\mathrm{Var}(S) < 2 \mathrm{Var}(X_1)\)，因为 \(X_1\) 抽得大值时，剩下的池子会偏小，\(X_2\) 也偏小——存在负相关。算清楚需要"协方差"的概念，留到第 6 章。这里你记住一句话：期望对所有线性组合都简单；方差只对独立和才简单。

例 4.14（圣诞节交换礼物 / 配对问题）　\(n\) 个人各带一份礼物到聚会，把所有礼物随机分发给所有人（每人恰好得一份），假设是一个均匀随机的排列。记 \(N\) 为"恰好拿回自己带的那份礼物"的人数。问 \(\mathbb{E}[N]\)。

解：直接列 \(N\) 的 PMF 是个出名的烦人事（涉及"错排公式"），但是 \(\mathbb{E}[N]\) 用线性性几乎是免费的。引入指示函数：

\[ I_i = \begin{cases} 1, & \text{第 \(i\) 个人拿回自己的礼物}, \\ 0, & \text{否则}. \end{cases} \]

那 \(N = I_1 + I_2 + \cdots + I_n\)。每个 \(I_i\) 取值 \(\{0, 1\}\)，且 \(P(I_i = 1) =\) "第 \(i\) 人恰好拿到第 \(i\) 份"的概率 \(= 1/n\)（在均匀随机排列中，第 \(i\) 人拿到任何一份具体礼物都是等可能的）。所以 \(\mathbb{E}[I_i] = 1 \cdot 1/n + 0 \cdot (1 - 1/n) = 1/n\)。

由线性性：

\[ \mathbb{E}[N] = \sum_{i=1}^n \mathbb{E}[I_i] = n \cdot \tfrac{1}{n} = 1. \]

无论 \(n\) 是 5、50 还是 500，平均都是恰好 1 个人拿回自己的礼物——是不是有点反直觉？请你品味一下。"指示函数 + 线性性"是后面会大量复用的一对组合拳，请记住这个套路。

例 4.15（生日匹配的最简化问题）　某班 30 个学生，假设每人生日独立、均匀分布在 365 天。考虑随机变量 \(M\) =班里"生日恰好和班长生日（指定的某一天）相同的人数（不含班长本人）"。问 \(\mathbb{E}[M]\) 与 \(\mathrm{Var}(M)\)。

解：除班长外有 29 个学生。引入 \(I_j = \mathbf{1}\{\text{第 \(j\) 个学生与班长同生日}\}\)，\(j = 1, \ldots, 29\)。每个 \(I_j\) 是伯努利试验（取值 \(\{0, 1\}\)），成功概率 \(p = 1/365\)。\(M = I_1 + \cdots + I_{29}\)。

\(\mathbb{E}[M] = 29 \cdot 1/365 \approx 0.0795\)。

因为各学生的生日独立，所有 \(I_j\) 互相独立，所以方差也可以加（这是独立和的方差性质，第 6 章正式证）：

\[ \mathrm{Var}(M) = \sum_{j=1}^{29} \mathrm{Var}(I_j) = 29 \cdot p(1 - p) = 29 \cdot \tfrac{1}{365} \cdot \tfrac{364}{365} \approx 0.0793. \]

请你注意这件事："期望和方差的数值非常接近"——下一章泊松分布会告诉你为什么这是一种"稀有事件"信号。事实上当 \(n\) 大、\(p\) 小、\(np = \lambda\) 适中时，\(\mathrm{Bin}(n, p) \approx \mathrm{Pois}(\lambda)\)，而泊松的期望恰等于方差。

例 4.16（最大点数的期望——用 CDF 反推 PMF）　\(M\) = 同时掷两枚骰子的较大点数。求 \(\mathbb{E}[M]\)。

解：先算 CDF。事件 "\(M \leq k\)" 等价于"两枚骰子的点数都 \(\leq k\)"。每枚 \(\leq k\) 的概率 \(k/6\)，独立，所以

\[ F_M(k) = P(M \leq k) = \left(\tfrac{k}{6}\right)^2 = \tfrac{k^2}{36}, \qquad k = 1, 2, 3, 4, 5, 6. \]

从 CDF 反推 PMF：\(p_M(k) = F_M(k) - F_M(k - 1) = \tfrac{k^2 - (k-1)^2}{36} = \tfrac{2k - 1}{36}\)。具体：

\[ p_M(1) = \tfrac{1}{36}, \quad p_M(2) = \tfrac{3}{36}, \quad p_M(3) = \tfrac{5}{36}, \quad p_M(4) = \tfrac{7}{36}, \quad p_M(5) = \tfrac{9}{36}, \quad p_M(6) = \tfrac{11}{36}. \]

检查归一：\(1 + 3 + 5 + 7 + 9 + 11 = 36\)，OK。\(\mathbb{E}[M] = \sum_{k=1}^6 k \cdot \tfrac{2k - 1}{36} = \tfrac{1}{36}\sum_{k=1}^6 (2k^2 - k) = \tfrac{1}{36}(2 \cdot 91 - 21) = \tfrac{161}{36} \approx 4.47\)。

所以"两骰最大点数"的均值 \(\approx 4.47\)，明显比单骰的 3.5 大——直观对，因为"取最大"会偏向大数。

这条解题思路还有一个一般版："从 CDF 倒推 PMF"——当事件"\(X \leq k\)"比"\(X = k\)"好直接计算时（如这里"两枚都 \(\leq k\)"），先算 CDF 再差出 PMF。这是个相当常见的技巧，请你存一份在工具箱里。
