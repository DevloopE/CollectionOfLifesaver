第 6 章 连续随机变量与密度

到第 5 章为止，我们见过的随机变量都是离散的——掷骰子的点数、抽签的次数、一小时内打来的电话数、抓到红球的颗数。它们的取值要么是有限多个，要么是可数无限多个，反正可以"一个一个数"。每一个具体的取值 \(x\)，都对应一个正概率 \(P(X = x)\)；把这些质量加起来等于 \(1\)。这套语言简单、干净，对很多问题足够用。

可是世界上的随机量并不都长这样。请你想想下面几件事——它们是日常的、但偏偏没有办法用"一个值对应一个正概率"这种离散方式刻画：你测一个人的身高时，得到 \(173.218\) 厘米还是 \(173.219\) 厘米？你早上去公交站等下一班车，等的时间是 \(2.314\) 分钟还是 \(2.315\) 分钟？你拿万用表量电池的电压，读出 \(1.4982\) 伏还是 \(1.4983\) 伏？这些"测量出来的"量，本质上落在一段实数上，不是一个一个分立的点；它们的可能值之"多"，和整条实数轴一样多。

第 6 章就是要把概率论的语言扩展到这种情形。我们要做的事是这一章的核心：发明一个叫做概率密度函数的对象 \(f_X(x)\)——它本身不是概率，而是"概率的密度"，乘上一段长度才变成概率。围绕它，我们重新定义期望、方差、累积分布函数；并讲一个新的、第 5 章没有的功夫——函数变换：如果 \(X\) 服从某个连续分布，那 \(Y = g(X)\) 服从什么分布？

本章后半段花大量篇幅做例题。如果你只想抓住一句话：对连续随机变量，概率 = 密度对区间的积分。这一句记牢，本章 80% 的事情都能自己摸索出来。

6.1 为什么需要连续随机变量

先把动机讲透。下面三个具体场景，每一个都让"离散"语言显得局促：

场景一：身高。 随机抽一个成年男子，记他的身高为 \(X\) 厘米。\(X\) 的可能取值是什么？理论上是 \([0, \infty)\) 上某段——比如 \([140, 220]\) 之内大致没问题。你可能会反驳："但实际测量只能精确到毫米啊，所以 \(X\) 实际上只取离散值，比如 \(173.0, 173.1, 173.2, \ldots\)" 这话有道理，但这是测量仪器的限制，不是身高本身的性质。理想情况下，身高是一个真实的实数，可以取 \([140, 220]\) 内的任意值。我们建模的对象应当是"理想的真实身高"，而不是"读数后被四舍五入的身高"。

场景二：等公交。 早上你去公交站，假定下一班车在某个未知时刻到达。你到站时正好开始等，记等候时间为 \(T\) 分钟。\(T\) 可以是 \(0\)（你刚到车就来了），可以是 \(15.7\)，可以是 \(0.001\)。它在 \([0, 30]\) 这段实数里取任意值（假定每 30 分钟一班）。这又是一个落在区间上的随机量。

场景三：电压。 一节标称 \(1.5\) 伏的电池，实际电压因制造误差略有波动。你随机抓一节出厂电池量它的电压 \(V\)，得到的是某个落在 \([1.45, 1.55]\) 附近的实数——具体多少，事先不知道。\(V\) 是连续随机变量。

这三个例子有一个共同特征：取值是"测量出来"的实数，而不是"数出来"的整数。处理这种量，我们需要一套全新的工具。但请你注意——新的工具不是把旧的扔掉重来，而是用积分替代求和，用密度替代质量。表面看起来变了，骨子里"概率论那一套"完全不变。

6.2 关键差异：\(P(X = x) = 0\)

连续随机变量与离散最深刻的差别，是每个具体取值的概率都是 0。这件事乍听违反直觉，是初学者最大的绊脚石之一，必须当场把它讲清楚。

设 \(X\) 是某连续随机变量，比如刚才的等公交时间 \(T \in [0, 30]\)。问：\(P(T = 5.000000\ldots)\) 等于多少？也就是说，"你正好等了恰好 5 分钟（一秒不多、一毫秒不少、一纳秒都不偏）"的概率有多大？

答案是 \(0\)。理由可以用一个朴素的极限论证：

把区间 \([0, 30]\) 分成 \(n\) 个等长小段，每段长 \(30/n\)。如果"等候时间"在区间里"均匀地"分布（这个说法第 7 章会精确化），那么 \(T\) 落在任一小段的概率应当是 \(1/n\)。现在 \(T = 5\) 对应"\(T\) 落在长度为 0 的一个点上"，可以看成"落在 \(n \to \infty\) 时长度趋于 0 的小段里"，其概率 \(1/n \to 0\)。所以

\[ P(T = 5) = 0. \]

同样的论证适用于任何具体的值 \(x\)：

\[ \boxed{P(X = x) = 0 \quad \text{对每一个具体值 } x \text{ 都成立}.} \]

请你注意这件事的不寻常之处。在离散情形下，概率全靠"每个点对应一个正概率"撑起来；现在每个点的概率都是 \(0\)，那概率是不是全没了？没有——它跑到区间上去了。具体地：\(P(T \in [4, 6]) > 0\)（比如等了 4 到 6 分钟之间的概率），但 \(P(T = 5)\) 偏偏 \(= 0\)。这件事提示我们：

对连续变量，概率不附着于点上，而附着于"区间（或更一般的可测集）"上。

你可能会立刻问："那如果 \(P(T = 5) = 0\)，是不是说 '\(T\) 等于 5' 这件事不可能发生？" 这是个好问题。答案是不。注意"概率为 0"和"不可能"在连续情形下不是一回事——概率 0 的事件仍然可能发生（或者，更精确地说，"\(T\) 取某个具体值"这件事必定有某个具体值发生，但事先指定的那个值碰上的概率是 0）。这是测度论里"几乎处处"概念的雏形，本章不展开，但请你接受这条事实：在连续世界里，"概率 = 0" ≠ "不可能"。

这条性质有一个非常漂亮的副产品：在连续情形下，区间端点用闭还是用开，没有任何区别。

\[ P(a \leq X \leq b) = P(a < X \leq b) = P(a \leq X < b) = P(a < X < b). \]

因为 \(P(X = a) = P(X = b) = 0\)，两个端点加不加进去都不影响。这件事让连续情形下的不等号写法非常宽松，跟离散正好相反（离散下端点是要紧的）。

6.3 概率密度函数 (PDF)

既然概率不再"住"在点上而住在"区间"上，我们需要一个新对象：能告诉我们"在某个区间上有多少概率"的工具。这个工具就是概率密度函数（probability density function，简写 PDF）。

给定一个连续随机变量 \(X\)，它的 PDF 是一个非负函数 \(f_X(x)\)（或简写 \(f(x)\)），满足：对任意 \(a \leq b\)，

\[ \boxed{P(a \leq X \leq b) = \int_a^b f_X(x) \, dx.} \]

把这条公式读出来：要算"\(X\) 落在 \([a, b]\) 中"的概率，就把 PDF 在这段区间上积分。这就是 PDF 的定义性质——它通过积分给出概率。

让我们对照一下离散情形以加深印象：

离散连续
概率质量函数 \(p(x) = P(X = x)\)概率密度函数 \(f(x)\)
\(P(X \in A) = \sum_{x \in A} p(x)\)\(P(X \in A) = \int_A f(x) \, dx\)
\(\sum_x p(x) = 1\)\(\int_{-\infty}^{\infty} f(x) \, dx = 1\)
\(p(x) \geq 0\) 且 \(\leq 1\)\(f(x) \geq 0\)（但可以 \(> 1\)）

这张表已经把这一章的"代数结构"全说完了——把求和换成积分，把质量换成密度。其他一切几乎是机械翻译。

有一种几何直观非常有助于理解 PDF。把 \(f(x)\) 画成 \(x\) 轴上的一条曲线，\(P(a \leq X \leq b)\) 就是这条曲线下、从 \(x = a\) 到 \(x = b\) 之间的面积。整个曲线下面积是 \(1\)（因为总概率为 \(1\)）。"概率落在哪个区间"被翻译成了"曲线下哪一块面积"。

6.4 PDF 的性质

从定义出发，任何合法的 PDF 必须满足两条性质：

性质 1（非负性）： \(f(x) \geq 0\) 对所有 \(x \in \mathbb{R}\) 成立。

为什么？因为如果某处 \(f(x_0) < 0\)，那对包含 \(x_0\) 的小区间 \([x_0 - \varepsilon, x_0 + \varepsilon]\)，积分将得到一个负值——但概率不能是负的。所以 PDF 处处非负。

性质 2（归一性）： \(\int_{-\infty}^{\infty} f(x) \, dx = 1\)。

这等价于说"\(X\) 一定取某个实数值"，所以总概率是 \(1\)。这是离散下 \(\sum p(x) = 1\) 的连续版。

反过来：任何满足这两条性质的函数 \(f\)，都是某个连续随机变量的 PDF。所以"构造一个连续分布"在数学上等价于"构造一个非负、积分为 1 的函数"。这点听起来抽象，但下一章会反复用——每当我们看到一个像样的非负函数，问一下"它积起来是不是 1？" 如果是，它就定义了一个分布。

例 6.1（验证一个候选 PDF）　设 \(f(x) = \begin{cases} c x & \text{若 } 0 \leq x \leq 1, \\ 0 & \text{其他}, \end{cases}\) 求常数 \(c\)，使 \(f\) 成为合法 PDF。

解：先看非负性——只要 \(c \geq 0\)，函数在 \([0, 1]\) 上是 \(c x \geq 0\)，其他地方为 0，处处非负。再看归一性：

\[ \int_{-\infty}^{\infty} f(x) \, dx = \int_0^1 c x \, dx = c \cdot \frac{x^2}{2} \bigg|_0^1 = \frac{c}{2}. \]

令 \(c/2 = 1\)，得 \(c = 2\)。所以 \(f(x) = 2x\)（在 \([0, 1]\) 上，其他为 0）是合法 PDF。

注意一件好玩的事：\(f(1) = 2 \cdot 1 = 2\)。密度可以大于 1。这件事下一节专门讨论，因为它是初学者最容易翻车的地方。

6.5 注意：\(f(x)\) 不是概率！

这是本章最要紧的"防错条款"。请务必看清这件事：密度函数 \(f(x)\) 不是概率。它的值可以大于 1，可以是任意大的数，只要积起来等于 1 就行。

你可能会想：但是 \(p(x) = P(X = x)\) 在离散情形下是概率啊，作为对应物 \(f(x)\) 不也该是概率吗？这是初学者第一个错觉。

正确的对应关系不是 \(f(x) \leftrightarrow p(x)\)，而是 \(f(x) \, dx \leftrightarrow p(x)\)。换句话说——

\[ \boxed{P(x \leq X \leq x + dx) \approx f(x) \cdot dx.} \]

密度乘"一小段长度"才是概率。所以"密度"两字很有道理：它度量"单位长度上有多少概率"。如果某处密度是 \(2\)，意思是"该处每单位长度上有 \(2\) 单位的概率"——那么长度为 \(0.3\) 的小段上的概率约为 \(2 \cdot 0.3 = 0.6\)，仍然 \(< 1\)，不矛盾。

这套思路在物理里你大概熟悉。线密度（每米多少千克）可以是 \(50 \text{ kg/m}\)，这并不"奇怪"——你只是把它对长度积分得到质量；同样地，PDF 可以取很大的值，你只需把它对区间积分得到概率。在 6.4 节那个例子里 \(f(1) = 2\)，并不"违法"——因为 \(P(X = 1) = 0 \neq 2\)，密度 2 没有任何概率含义；而 \(P(0.95 \leq X \leq 1) = \int_{0.95}^1 2x \, dx = (1 - 0.9025) = 0.0975 < 1\)，毫无问题。

更极端的例子：考虑 \(f(x) = \dfrac{1}{2\sqrt{x}}\) 在 \(x \in (0, 1]\) 上（其他为 0）。检查归一性：\(\int_0^1 \frac{1}{2\sqrt{x}} dx = \sqrt{x} \big|_0^1 = 1\) ✓。这是合法 PDF。可它在 \(x \to 0^+\) 时趋于无穷！密度可以无界。这并不阻止它给出有限概率：\(P(X \leq 0.01) = \int_0^{0.01} \frac{1}{2\sqrt x} dx = \sqrt{0.01} = 0.1\)，仍然 \(< 1\)，仍然合法。

请你把这件事记牢——它是连续概率与离散概率最容易混淆的差别：

密度 \(f(x)\) 是函数值；概率是 \(f(x) \cdot dx\) 或 \(\int_a^b f(x) dx\)。

6.6 累积分布函数 (CDF)

有了 PDF 之后，我们可以定义另一个非常有用的函数——累积分布函数（cumulative distribution function，简写 CDF）。它的定义对离散和连续都一样：

\[ \boxed{F_X(x) = P(X \leq x).} \]

"累积"是因为它把"\(X\) 所有不超过 \(x\) 的概率全加起来"。对连续情形，这就是从 \(-\infty\) 到 \(x\) 的密度积分：

\[ F_X(x) = P(X \leq x) = \int_{-\infty}^x f_X(t) \, dt. \]

注意我把积分变量从 \(x\) 改成 \(t\)——因为 \(x\) 已经被用作"积分上限"（CDF 的自变量）了，再用 \(x\) 做被积变量会引起混乱。这种"换名字"的小习惯请记住，将来会反复出现。

CDF 有四个基本性质（对连续与离散都成立）：

  • 单调不减：\(x_1 \leq x_2 \Rightarrow F(x_1) \leq F(x_2)\)。这是因为 \(\{X \leq x_1\} \subset \{X \leq x_2\}\)，加上更多事件的概率不会减少。
  • 极限：\(\lim_{x \to -\infty} F(x) = 0\)，\(\lim_{x \to +\infty} F(x) = 1\)。"不大于很小的数" 是不可能事件，"不大于很大的数" 是必然事件。
  • 右连续：\(\lim_{x \to x_0^+} F(x) = F(x_0)\)。这条对离散下"跳跃点"特别要紧，但对连续随机变量自动满足，且 CDF 实际上是处处连续的。
  • 区间概率：\(P(a < X \leq b) = F(b) - F(a)\)。这是 CDF 最常用的一招——把"落在区间"翻译成"两个 CDF 值相减"。

对连续随机变量，由于 \(P(X = a) = P(X = b) = 0\)，区间概率公式可以更宽松地写成 \(P(a \leq X \leq b) = F(b) - F(a)\)，端点开闭无所谓。

例 6.2（求 CDF）　续例 6.1，\(f(x) = 2x\)（在 \([0, 1]\) 上，其他为 0）。求 \(F(x)\)。

解：分段算。

当 \(x < 0\) 时，被积函数 \(f(t)\) 在 \(t \leq x < 0\) 上恒为 0，所以 \(F(x) = 0\)。

当 \(0 \leq x \leq 1\) 时，

\[ F(x) = \int_{-\infty}^x f(t) \, dt = \int_0^x 2t \, dt = t^2 \bigg|_0^x = x^2. \]

当 \(x > 1\) 时，所有概率早已"用完"，

\[ F(x) = \int_0^1 2t \, dt = 1. \]

合起来：

\[ F(x) = \begin{cases} 0 & x < 0, \\ x^2 & 0 \leq x \leq 1, \\ 1 & x > 1. \end{cases} \]

请你画一下这个函数：在 \([0, 1]\) 上是抛物线由 0 升到 1，两端水平。它满足前述四条性质（连续、单调、两端取 0 和 1）。一个方便的检查：\(P(0.5 \leq X \leq 0.8) = F(0.8) - F(0.5) = 0.64 - 0.25 = 0.39\)。再用 PDF 直接积一遍：\(\int_{0.5}^{0.8} 2x \, dx = x^2 \big|_{0.5}^{0.8} = 0.39\) ✓。

6.7 PDF 与 CDF 的关系

上一节的 \(F(x) = \int_{-\infty}^x f(t) \, dt\) 这条公式，把它反过来读，就是微积分基本定理：

\[ \boxed{f(x) = F'(x).} \]

密度是 CDF 的导数。这是 PDF 与 CDF 之间最简洁的桥梁。

用这条桥梁，PDF 与 CDF 之间可以自由互相恢复：

  • 已知 PDF \(f\)，对它积分得 CDF \(F\)。
  • 已知 CDF \(F\)，对它求导得 PDF \(f\)。

实践中两者常常都用得上：CDF 用来快速计算"落在某区间"的概率（一减一就行）；PDF 用来直观看分布在哪里"密"在哪里"稀"，并参与许多其他计算（期望、方差、变换）。

顺便核对例 6.2：\(F(x) = x^2\) 在 \((0, 1)\) 上，求导得 \(F'(x) = 2x = f(x)\) ✓。

请你注意一件事：在 \(F\) 的连接点（如这里的 \(x = 0\) 和 \(x = 1\)），\(F\) 仍然连续，但 \(F'\) 可能不存在（或左右导数不等）。这种"少数点上不可导"是允许的——只要总体上 \(\int f = 1\) 成立，PDF 在有限多个点取什么值无所谓。所以一般约定 \(f(0) = 0\)、\(f(1) = 0\)（或者 \(2\)，怎么定义都不影响积分）。

6.8 期望与 LOTUS

把求和换成积分这一招继续用下去——离散情形下 \(\mathbb{E}[X] = \sum x \cdot p(x)\)，连续情形下：

\[ \boxed{\mathbb{E}[X] = \int_{-\infty}^{\infty} x \cdot f(x) \, dx.} \]

"期望" = "取值"的"加权平均"，权重是密度（前者用密度，后者用质量；逻辑完全相同）。

同样地，对函数 \(g\) 的期望——也就是无意识统计学家公式（Law of the Unconscious Statistician，简称 LOTUS）——在连续情形下变成：

\[ \boxed{\mathbb{E}[g(X)] = \int_{-\infty}^{\infty} g(x) \cdot f(x) \, dx.} \]

这条公式让你不必先去求 \(Y = g(X)\) 的密度——直接拿 \(X\) 的密度配上 \(g(x)\) 积一下就行。这是 LOTUS 的全部威力。请你记住：要算 \(\mathbb{E}[X^2]\)，不必先去求 \(X^2\) 的分布；直接 \(\int x^2 f(x) dx\) 即可。

例 6.3（期望的计算）　续例 6.1，\(f(x) = 2x\) 在 \([0, 1]\) 上。求 \(\mathbb{E}[X]\)。

\[ \mathbb{E}[X] = \int_0^1 x \cdot 2x \, dx = \int_0^1 2 x^2 \, dx = \frac{2 x^3}{3} \bigg|_0^1 = \frac{2}{3}. \]

检查直觉：密度 \(2x\) 在 \([0, 1]\) 上向右倾斜（\(x\) 越大密度越大），所以期望应当偏右——大于中点 \(0.5\)。算出来 \(2/3 \approx 0.67\)，确实偏右，方向对。

例 6.4（用 LOTUS 算 \(\mathbb{E}[X^2]\)）　同上分布，求 \(\mathbb{E}[X^2]\)。

\[ \mathbb{E}[X^2] = \int_0^1 x^2 \cdot 2x \, dx = \int_0^1 2 x^3 \, dx = \frac{x^4}{2} \bigg|_0^1 = \frac{1}{2}. \]

这就是 LOTUS——把 \(g(x) = x^2\) 直接代进 \(\int g(x) f(x) dx\)，连"\(X^2\) 服从什么分布"都不用管。

6.9 方差与标准差

方差的定义和离散完全一样：

\[ \boxed{\mathrm{Var}(X) = \mathbb{E}[(X - \mu)^2] = \mathbb{E}[X^2] - \mu^2,} \]

其中 \(\mu = \mathbb{E}[X]\)。两条公式都对连续成立，求时仍然推荐"先算 \(\mathbb{E}[X^2]\)、再减 \(\mu^2\)"这条捷径——比直接算 \(\mathbb{E}[(X - \mu)^2]\) 通常少一些代数。

标准差仍然是方差的算术平方根：\(\sigma_X = \sqrt{\mathrm{Var}(X)}\)。它有"和 \(X\) 同量纲"的好处——如果 \(X\) 是身高（厘米），\(\mathrm{Var}(X)\) 的量纲是平方厘米，\(\sigma_X\) 才回到厘米，更便于解释。

例 6.5（方差的计算）　续例 6.3、6.4。\(\mathbb{E}[X] = 2/3\)，\(\mathbb{E}[X^2] = 1/2\)。所以

\[ \mathrm{Var}(X) = \frac{1}{2} - \left(\frac{2}{3}\right)^2 = \frac{1}{2} - \frac{4}{9} = \frac{9 - 8}{18} = \frac{1}{18}. \]

标准差 \(\sigma = 1/\sqrt{18} \approx 0.236\)。直观核对：分布支撑在 \([0, 1]\) 上、长度为 1，标准差应当不会大于 \(1/2\)（极端的两点分布才有 \(\sigma = 0.5\)），\(0.236\) 完全在合理范围内。

方差的几个常用线性性质——\(\mathrm{Var}(aX + b) = a^2 \mathrm{Var}(X)\)，\(\mathrm{Var}(X + Y) = \mathrm{Var}(X) + \mathrm{Var}(Y)\)（独立时）——在连续情形下与离散完全一致，证明套路也一样（用 LOTUS + 期望线性性），这里不重复。

6.10 函数变换：从 \(X\) 的密度求 \(Y = g(X)\) 的密度

这是本章最有技术含量的一节，也是连续情形特有的难点（离散情形下，要求 \(Y = g(X)\) 的分布只需逐点列举即可，没有"密度变换"的麻烦）。

问题的提法：已知 \(X\) 的 PDF \(f_X\)，给定一个函数 \(g\)，问 \(Y = g(X)\) 的 PDF \(f_Y\) 长什么样？

有两种主要方法。先讲适用面最广的"先 CDF 后求导"方法，再讲单调情形下更快的"变量替换公式"。

6.10.1 方法一：先 CDF 后求导

这种方法步步可执行，只要你能写出"\(Y \leq y\)"等价于"\(X \in ?\)"，剩下的就是积分和求导。

具体步骤：

  1. 写出 \(F_Y(y) = P(Y \leq y) = P(g(X) \leq y)\)。
  2. 把 "\(g(X) \leq y\)" 翻译成 "\(X \in A_y\)"，其中 \(A_y\) 是某个依赖 \(y\) 的子集（通常是一个或几个区间）。
  3. 用 \(X\) 的密度算 \(P(X \in A_y) = \int_{A_y} f_X(x) \, dx\)，得到 \(F_Y(y)\)。
  4. 对 \(y\) 求导：\(f_Y(y) = F_Y'(y)\)。

这套方法朴素、机械、不挑函数 \(g\)（它可以不单调，可以分段，都能做）。下面用一个小例子演示一遍。

例 6.6（平方变换：先 CDF 后求导）　设 \(X\) 服从 \([0, 1]\) 上的均匀分布，PDF 为 \(f_X(x) = 1\) 在 \([0, 1]\) 上，0 在其他。令 \(Y = X^2\)，求 \(Y\) 的 PDF。

解：先确定 \(Y\) 的支撑——\(X \in [0, 1]\) 蕴含 \(Y = X^2 \in [0, 1]\)。所以 \(F_Y(y)\) 在 \(y < 0\) 时为 0、在 \(y > 1\) 时为 1。重点是 \(0 \leq y \leq 1\) 这段。

\[ F_Y(y) = P(Y \leq y) = P(X^2 \leq y) = P(-\sqrt{y} \leq X \leq \sqrt{y}). \]

因为 \(X\) 只取 \([0, 1]\) 中的值，\(X \geq 0\) 自动满足，所以条件简化为 \(P(0 \leq X \leq \sqrt{y}) = \sqrt{y}\)（均匀分布的概率等于区间长度）。所以

\[ F_Y(y) = \sqrt{y}, \qquad 0 \leq y \leq 1. \]

对 \(y\) 求导：

\[ f_Y(y) = \frac{d}{dy} \sqrt{y} = \frac{1}{2\sqrt{y}}, \qquad 0 < y \leq 1. \]

验证：\(\int_0^1 \frac{1}{2\sqrt{y}} dy = \sqrt{y} \big|_0^1 = 1\) ✓。注意 \(f_Y\) 在 \(y \to 0^+\) 时趋于无穷——这就是 6.5 节预告过的"密度可以无界"。它对应一个直觉：\(X \mapsto X^2\) 这个变换"压缩"了靠近 0 的区域（\(X\) 在 \([0, 0.1]\) 内被映到 \(Y\) 在 \([0, 0.01]\) 内），所以 \(Y\) 的概率在 0 附近变得"非常密集"。

6.10.2 方法二：单调情形下的变量替换公式

当 \(g\) 是严格单调（且足够光滑）的，存在一条更直接的公式——它本质上是"先 CDF 后求导"那条路径的简洁包装，但用起来快很多。我们推一下。

设 \(g\) 在 \(X\) 的支撑上严格递增。令 \(Y = g(X)\)，则

\[ F_Y(y) = P(Y \leq y) = P(g(X) \leq y) = P(X \leq g^{-1}(y)) = F_X(g^{-1}(y)). \]

对 \(y\) 求导（链式法则）：

\[ f_Y(y) = F_Y'(y) = f_X(g^{-1}(y)) \cdot \frac{d}{dy} g^{-1}(y). \]

因为 \(g\) 递增，\(g^{-1}\) 也递增，导数 \(\frac{d}{dy} g^{-1}(y) > 0\)。

如果 \(g\) 是严格递减的，则 \(g(X) \leq y \Leftrightarrow X \geq g^{-1}(y)\)，所以

\[ F_Y(y) = P(X \geq g^{-1}(y)) = 1 - F_X(g^{-1}(y)), \]

对 \(y\) 求导：

\[ f_Y(y) = -f_X(g^{-1}(y)) \cdot \frac{d}{dy} g^{-1}(y). \]

这次 \(\frac{d}{dy} g^{-1}(y) < 0\)，所以最后整体是正的。两种情形可以合并写成一条公式：

\[ \boxed{f_Y(y) = f_X\big(g^{-1}(y)\big) \cdot \left| \frac{d}{dy} g^{-1}(y) \right|.} \]

这就是变量替换公式。它的几何意义是"换元积分公式"反过来用：在区间 \([y, y + dy]\) 内的概率 \(f_Y(y) dy\)，等于在区间 \([x, x + dx]\)（其中 \(x = g^{-1}(y)\)、\(dx\) 与 \(dy\) 通过 \(g\) 联系）内的概率 \(f_X(x) dx\)。两边相等给出 \(f_Y(y) = f_X(x) \cdot |dx/dy| = f_X(g^{-1}(y)) \cdot |\frac{d g^{-1}}{dy}|\)。

这条公式的"绝对值"是关键——它确保密度 \(f_Y \geq 0\)，无论 \(g\) 是增还是减都成立。实际操作中只需记两个步骤：(1) 把 \(x\) 用 \(y\) 表出（即 \(x = g^{-1}(y)\)），代入 \(f_X\)；(2) 乘上 \(|dx/dy|\)。绝对值随手补。

注意 \(g\) 必须严格单调这条公式才适用。如果 \(g\) 不单调（比如 \(g(x) = x^2\) 在 \(\mathbb{R}\) 上），就要回到方法一，或者把 \(g\) 拆成单调段分段处理（每段贡献一项）。

例 6.7（重做平方变换：变量替换法）　复习例 6.6，\(X \sim \mathrm{Unif}(0, 1)\)、\(Y = X^2\)。在 \(X\) 的支撑 \([0, 1]\) 上，\(g(x) = x^2\) 严格递增（这点重要：如果支撑是 \([-1, 1]\)，\(g\) 就不单调，要用方法一）。

逆函数：\(g^{-1}(y) = \sqrt{y}\)。其导数 \(\frac{d}{dy}\sqrt{y} = \frac{1}{2\sqrt{y}}\)。代入公式：

\[ f_Y(y) = f_X(\sqrt{y}) \cdot \left| \frac{1}{2\sqrt{y}} \right| = 1 \cdot \frac{1}{2\sqrt{y}} = \frac{1}{2\sqrt{y}}, \qquad 0 < y \leq 1. \]

与例 6.6 答案一致 ✓。这一遍只用了三行——这就是变量替换公式的便利。

6.11 综合例题

例 6.8（线性变换：标准化）　设 \(X\) 是连续随机变量，PDF 为 \(f_X(x)\)。考虑 \(Y = a X + b\)（\(a \neq 0\)，是常数）。求 \(f_Y\)。

解：\(g(x) = a x + b\)，严格单调（\(a > 0\) 时增，\(a < 0\) 时减），逆函数 \(x = g^{-1}(y) = \frac{y - b}{a}\)。其导数 \(\frac{d g^{-1}}{dy} = \frac{1}{a}\)，绝对值 \(\frac{1}{|a|}\)。所以

\[ \boxed{f_Y(y) = \frac{1}{|a|} f_X\left(\frac{y - b}{a}\right).} \]

这条公式将在第 7 章正态分布的"标准化"中频繁出现：把 \(X \sim \mathcal{N}(\mu, \sigma^2)\) 通过 \(Z = (X - \mu)/\sigma\) 变成 \(\mathcal{N}(0, 1)\)，背后用的就是这条公式。

例 6.9（对数变换）　设 \(X \sim \mathrm{Unif}(0, 1)\)，定义 \(Y = -\ln X\)。求 \(Y\) 的 PDF，确认 \(Y\) 服从一个我们将在第 7 章正式遇见的分布。

解：\(X \in (0, 1)\)（端点的概率为 0，可以忽略），所以 \(\ln X < 0\)，\(Y = -\ln X > 0\)。\(Y\) 的支撑是 \((0, \infty)\)。

\(g(x) = -\ln x\) 在 \((0, 1)\) 上严格递减（\(g'(x) = -1/x < 0\)）。逆函数：\(y = -\ln x \Rightarrow x = e^{-y}\)。所以 \(g^{-1}(y) = e^{-y}\)，其导数 \(\frac{d g^{-1}}{dy} = -e^{-y}\)，绝对值 \(e^{-y}\)。代入：

\[ f_Y(y) = f_X(e^{-y}) \cdot e^{-y}. \]

而 \(X \sim \mathrm{Unif}(0, 1)\) 蕴含 \(f_X = 1\) 在 \((0, 1)\) 上；只要 \(y > 0\)，\(e^{-y} \in (0, 1)\)，\(f_X(e^{-y}) = 1\)。所以

\[ f_Y(y) = e^{-y}, \qquad y > 0. \]

这正是第 7 章要讲的指数分布 \(\mathrm{Exp}(1)\) 的密度！这件事在仿真里非常有用——只要你有"均匀随机数生成器"（很容易），就能通过 \(Y = -\ln U\) 生成"指数随机数"（其中 \(U \sim \mathrm{Unif}(0, 1)\)）。这是逆变换抽样法的最简单实例。

例 6.10（不单调变换：分段处理）　设 \(X \sim \mathrm{Unif}(-1, 1)\)，定义 \(Y = X^2\)。求 \(f_Y\)。

解：注意 \(g(x) = x^2\) 在 \([-1, 1]\) 上不单调——它在 \([-1, 0]\) 上递减、在 \([0, 1]\) 上递增。所以单调情形的公式不能直接用，回到方法一。

\(X\) 的密度：\(f_X(x) = 1/2\) 在 \([-1, 1]\) 上（区间长度为 2，所以归一化后高度为 \(1/2\)）。\(Y\) 的支撑是 \([0, 1]\)。

对 \(0 \leq y \leq 1\)：

\[ F_Y(y) = P(X^2 \leq y) = P(-\sqrt{y} \leq X \leq \sqrt{y}) = \int_{-\sqrt{y}}^{\sqrt{y}} \frac{1}{2} dx = \sqrt{y}. \]

求导：\(f_Y(y) = \frac{1}{2\sqrt{y}}\)，\(0 < y \leq 1\)。

验证：\(\int_0^1 \frac{1}{2\sqrt{y}} dy = 1\) ✓。和例 6.6 形式一样——巧合？不，因为这里 \(X\) 的负值与正值"对称地"折回 \(Y \geq 0\) 这一边，密度被对称地"加倍"，但 \(X\) 的密度本身在 \([-1, 1]\) 上是 \(1/2\)（不是例 6.6 中的 1），抵消之后仍然是 \(\frac{1}{2\sqrt{y}}\)。

有一种更"加法"的写法：当 \(g\) 不单调但在有限多个段上单调时，可以把每一段的贡献分别用变量替换公式，然后相加。这里 \(g\) 在 \([-1, 0]\) 和 \([0, 1]\) 上分别有逆 \(x = -\sqrt{y}\) 和 \(x = +\sqrt{y}\)，每段对 \(f_Y\) 贡献 \(f_X(\pm \sqrt{y}) \cdot \frac{1}{2\sqrt{y}} = \frac{1}{2} \cdot \frac{1}{2\sqrt{y}} = \frac{1}{4\sqrt{y}}\)。两段加起来 \(\frac{1}{2\sqrt{y}}\) ✓。这种"分段相加"的思路在更复杂的不单调变换里很有用。

例 6.11（求中位数）　对例 6.1 的分布 \(f(x) = 2x\) 在 \([0, 1]\) 上，求中位数 \(m\)（即满足 \(P(X \leq m) = 1/2\) 的那个值）。

解：用 CDF。\(F(m) = m^2 = 1/2\)，所以 \(m = 1/\sqrt{2} \approx 0.707\)。

对照期望 \(\mathbb{E}[X] = 2/3 \approx 0.667\)：中位数 \(>\) 期望——这件事是"右倾分布的左偏"还是"右倾分布的右偏"？分布 \(2x\) 是右重（往右越来越密），所以中位数应当偏右、超过对称的 0.5。期望也偏右，但中位数偏得更厉害一些。一般来说，分布的"形状偏向"会使中位数和期望大小关系不同；这里中位数 \(>\) 期望，是因为这个 PDF 在 \([0, 1]\) 上线性增长，左尾"长而薄"对期望的拉拽弱于对中位数的拉拽。这种细致的直觉留在你自己做几个例子之后慢慢体会。

例 6.12（综合应用：对带常数的密度求期望与方差）　设 \(X\) 的 PDF 为 \(f(x) = c (1 - x^2)\) 在 \([-1, 1]\) 上，其他为 0。(a) 求 \(c\)；(b) 求 \(\mathbb{E}[X]\) 与 \(\mathrm{Var}(X)\)。

解：(a) 归一化：

\[ \int_{-1}^1 c(1 - x^2) dx = c \left[x - \frac{x^3}{3}\right]_{-1}^1 = c\left(\frac{2}{3} - \left(-\frac{2}{3}\right)\right) = \frac{4c}{3}. \]

令 \(\frac{4c}{3} = 1\)，得 \(c = 3/4\)。所以 \(f(x) = \frac{3}{4}(1 - x^2)\) 在 \([-1, 1]\) 上。

(b) \(\mathbb{E}[X] = \int_{-1}^1 x \cdot \frac{3}{4}(1 - x^2) dx\)。被积函数 \(x(1 - x^2)\) 是奇函数，在对称区间 \([-1, 1]\) 上积分为 0。所以 \(\mathbb{E}[X] = 0\)。

这个结果由对称性也能直接看出来——密度关于 \(x = 0\) 对称，期望必为 0。

\[ \mathbb{E}[X^2] = \int_{-1}^1 x^2 \cdot \frac{3}{4}(1 - x^2) dx = \frac{3}{4}\int_{-1}^1 (x^2 - x^4) dx = \frac{3}{4} \left[\frac{x^3}{3} - \frac{x^5}{5}\right]_{-1}^1. \]

计算括号：\(\left(\frac{1}{3} - \frac{1}{5}\right) - \left(-\frac{1}{3} + \frac{1}{5}\right) = 2\left(\frac{1}{3} - \frac{1}{5}\right) = 2 \cdot \frac{2}{15} = \frac{4}{15}\)。所以 \(\mathbb{E}[X^2] = \frac{3}{4} \cdot \frac{4}{15} = \frac{1}{5}\)。

方差：\(\mathrm{Var}(X) = \mathbb{E}[X^2] - 0^2 = 1/5 = 0.2\)。标准差 \(\sigma = 1/\sqrt{5} \approx 0.447\)。这个分布支撑在 \([-1, 1]\)（长 2），标准差约 \(0.45\)，看起来合理——介于"均匀分布在 \([-1, 1]\)" 的 \(\sigma = 1/\sqrt{3} \approx 0.577\) 和"集中在 0 附近"的更小值之间。

本章到此结束。我们建立了连续随机变量的整套语言：PDF 与 CDF 的相互关系、期望与方差的积分公式、函数变换的两种方法。下一章我们把这套语言用到几个最重要的具体分布上——均匀、指数、Gamma、正态、Beta、卡方——并对每一种讲清"它的密度长什么样"、"它的期望与方差是多少"、"它在什么场合出现"。这些是统计学整个上层建筑的基石，请你在第 7 章里逐个把它们认熟。
