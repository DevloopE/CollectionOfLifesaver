第 11 章 不等式

到这里为止，每当我们想知道"某个事件的概率"，我们都老老实实地算它——把概率密度函数积分、把概率质量函数求和、把组合数挨个数清。可现实中我们经常碰到这种情况：

  • 我们不知道这个随机变量的精确分布，只知道它的均值和方差。
  • 分布太复杂，没有解析表达式可算。
  • 我们只想要一个"过得去"的上界来宽松估算"超出这个范围的概率不超过多少"。

本章给三件这样的工具——Markov、Chebyshev、Jensen不等式。它们的共同特征是：它们用非常少的信息（一个均值，一个方差，或者一个凸函数性质）就能给出对概率或期望的定量控制。这听起来玄乎——居然只用均值就能限制"超大概率"？事实上确实如此，下面就看。

这一章的主线很短：

  • Markov 不等式：只用均值控制 \(P(X \geq a)\)。
  • Chebyshev 不等式（由 Markov 推得）：用方差控制 \(P(|X - \mu| \geq k\sigma)\)。
  • Jensen 不等式：用凸性控制 \(g(\mathbb{E}X)\) 与 \(\mathbb{E}[g(X)]\) 的大小关系。
  • Chernoff 界：把 Markov 用到 \(e^{tX}\) 上得到的指数型尾界。
  • Cauchy-Schwarz 不等式：协方差 \(\to\) 相关系数 \(\in [-1, 1]\) 这条事实的来源。

第 12 章的大数定律将直接用 Chebyshev 推出。所以请先把这一章吞下。

11.1 Markov 不等式：只用均值控制尾巴

开门见山。第一条不等式叫做Markov 不等式，由 19 世纪末俄国数学家 Markov 给出。陈述如下：

设 \(X \geq 0\) 是非负随机变量，\(a > 0\)。则 \(P(X \geq a) \leq \dfrac{\mathbb{E}[X]}{a}\)。

\[ \boxed{P(X \geq a) \leq \frac{\mathbb{E}[X]}{a}, \qquad X \geq 0,\ a > 0.} \]

请你注意两个使用前提：(1) \(X\) 是非负的；(2) \(a\) 是正的。这两条少不得——下面证明里你立刻看到为什么。

这条不等式的证明只有一行，且优雅得让人喷饭。我们写离散情形：

\[ \mathbb{E}[X] = \sum_{x} x\, p(x) \geq \sum_{x \geq a} x\, p(x) \geq \sum_{x \geq a} a\, p(x) = a \cdot P(X \geq a). \]

第一步：把求和限到 \(x \geq a\)，少加几项；因为所有 \(x \geq 0\)，少加只可能让总和变小，所以是 \(\geq\)。第二步：在 \(x \geq a\) 那一段，把每个 \(x\) 换成 \(a\)（更小），所以又是 \(\geq\)。最后把 \(a\) 提到求和外。把首尾连起来：\(\mathbb{E}[X] \geq a P(X \geq a)\)，除以 \(a\) 即得。

连续情形完全平行（把求和换成积分）。这就是为什么我们说 Markov 是"一行证明"。

11.1.1 几何理解

把 \(X\) 想成一根杆子，\(\mathbb{E}[X]\) 是它的重心。把分布想成"质量分布"。Markov 在说："如果重心在 \(\mu\) 处，那么过 \(a\) 处的质量比例不超过 \(\mu/a\)。"

直觉性的论证：要使 \(X \geq a\) 的概率"很大"——比如 \(p\)——就要把 \(p\) 比例的质量都放到 \(\geq a\) 处，这单这一部分就给重心贡献至少 \(p \cdot a\)。所以 \(\mu \geq p \cdot a\)，即 \(p \leq \mu / a\)。这就是 Markov。

这条不等式的好处是普适——不需要知道分布形状、不需要方差、什么都不需要，只需要均值。坏处是它非常松——\(P(X \geq 100\mu) \leq 1/100 = 1\%\)，听起来不错，但很多分布实际上 \(P(X \geq 100\mu)\) 远小于 \(10^{-10}\)。所以 Markov 是底线保证，不是精细估计。

例 11.1（Markov 的初等练习）　某网店每天的销售额 \(X\)（万元）平均为 5。试问 \(P(X \geq 50)\) 不超过多少？

解：直接套 Markov（\(X \geq 0\)，\(a = 50\)）：

\[ P(X \geq 50) \leq \frac{\mathbb{E}[X]}{50} = \frac{5}{50} = \frac{1}{10}. \]

"销售额超过日均 10 倍的概率不超过 10%"。注意我们对销售额的分布一无所知——它可能是正态、可能是指数、可能是奇形怪状的混合。Markov 仍然给出这个保证。

11.1.2 Markov 紧吗？

能不能让 \(P(X \geq a) = \mathbb{E}[X]/a\) 恰好成立？能。让 \(X\) 取两个值：\(P(X = 0) = 1 - p\)、\(P(X = a) = p\)。那 \(\mathbb{E}[X] = pa\)，\(P(X \geq a) = p = \mathbb{E}[X]/a\)。等式取到。

所以 Markov 在原则上是紧的——存在分布使等号成立。这告诉我们如果只知道均值，不可能有比 Markov 更紧的尾界。要想更紧的界，必须用更多信息——比如方差，那就引出 Chebyshev。

11.2 Chebyshev 不等式：用方差控制偏离

Chebyshev 不等式说：随机变量超出均值若干倍标准差以外的概率有上界，且与具体分布无关。准确陈述：

\[ \boxed{P(|X - \mu| \geq k\sigma) \leq \frac{1}{k^2},} \]

其中 \(\mu = \mathbb{E}[X]\)，\(\sigma = \sqrt{\mathrm{Var}(X)} > 0\)，\(k > 0\) 是任意正数。

等价的"非倍数"形式：

\[ P(|X - \mu| \geq \varepsilon) \leq \frac{\mathrm{Var}(X)}{\varepsilon^2}. \]

两种形式实质一样——令 \(\varepsilon = k\sigma\) 互换。

11.2.1 由 Markov 推 Chebyshev

这条不等式的证明是 Markov 的巧妙搬移。诀窍：对非负变量 \((X - \mu)^2\) 用 Markov。

注意 \((X - \mu)^2 \geq 0\) 总成立，且 \(\mathbb{E}[(X - \mu)^2] = \mathrm{Var}(X) = \sigma^2\)。事件"\(|X - \mu| \geq \varepsilon\)" 等价于 "\((X - \mu)^2 \geq \varepsilon^2\)"。所以

\[ P(|X - \mu| \geq \varepsilon) = P((X - \mu)^2 \geq \varepsilon^2) \stackrel{\text{Markov}}{\leq} \frac{\mathbb{E}[(X - \mu)^2]}{\varepsilon^2} = \frac{\sigma^2}{\varepsilon^2}. \]

令 \(\varepsilon = k\sigma\)：

\[ P(|X - \mu| \geq k\sigma) \leq \frac{\sigma^2}{k^2 \sigma^2} = \frac{1}{k^2}. \quad \square \]

这就是为什么前面强调 Markov 一行证完很重要——它是 Chebyshev 的脊梁。

11.2.2 解读：两倍标准差外不超过 1/4

取 \(k = 2\)：\(P(|X - \mu| \geq 2\sigma) \leq 1/4\)。也就是说 "偏离均值两倍标准差以上"的概率不超过 25%，对任何分布。

取 \(k = 3\)：\(P(|X - \mu| \geq 3\sigma) \leq 1/9 \approx 11\%\)。

取 \(k = 10\)：\(P(|X - \mu| \geq 10\sigma) \leq 1\%\)。

请你注意这条结果跟分布形状毫无关系——你不需要知道是正态、是指数、是离散还是连续。只要知道均值和方差，就有这个保证。当然这又是个底线——对正态分布而言，\(P(|X - \mu| \geq 2\sigma) \approx 4.6\%\)，比 25% 紧得多。但 Chebyshev 的用途是"我连分布都不知道，能保多少？"

Chebyshev 不等式在数学界叫"最弱的有用不等式"——它弱（远不如对正态分布的精确尾界），但在不知道分布的情况下又强到能直接证大数定律。下章我们就看 Chebyshev 是怎么把"\(\mathrm{Var}(\bar X_n) = \sigma^2/n \to 0\)"翻译成"\(\bar X_n \to \mu\) 依概率"。

例 11.2（用 Chebyshev 估硬币正面比例）　抛 \(n = 10000\) 次公平硬币，记 \(\bar X_n\) 为正面比例。求 "\(\bar X_n\) 偏离 0.5 至少 0.01" 的概率上界。

解：每次抛硬币 \(X_i \in \{0, 1\}\)，\(\mu = 1/2\)，\(\sigma^2 = 1/4\)。样本均值的方差 \(\mathrm{Var}(\bar X_n) = (1/4)/10000 = 1/40000\)。Chebyshev：

\[ P(|\bar X_n - 1/2| \geq 0.01) \leq \frac{1/40000}{(0.01)^2} = \frac{1/40000}{1/10000} = \frac{1}{4} = 25\%. \]

"抛一万次，正面比例偏离 50% 一个百分点以上"的概率不超过 25%。这非常松——实际值约 0.05%。但 Chebyshev 给的是无脑保证。

把样本量提到 \(n = 10^6\)：

\[ P(|\bar X_n - 1/2| \geq 0.01) \leq \frac{(1/4)/10^6}{10^{-4}} = \frac{1}{400} = 0.25\%. \]

样本越多，偏离概率越小——Chebyshev 给我们一个具体的速率。这就是大数定律的雏形。

11.3 Jensen 不等式：凸函数与期望

Markov 和 Chebyshev 是关于概率的不等式。本节的 Jensen 不等式是关于期望的不等式——它说凸函数和期望的顺序怎样换。

先复习一下"凸"的概念。一个函数 \(g : \mathbb{R} \to \mathbb{R}\) 叫凸函数（convex），如果对任意 \(x_1, x_2\) 和 \(\lambda \in [0, 1]\)，

\[ g(\lambda x_1 + (1 - \lambda) x_2) \leq \lambda g(x_1) + (1 - \lambda) g(x_2). \]

几何含义：函数图像两点之间的弦位于图像上方。等价地，函数曲线"向上弯"。常见的凸函数：\(g(x) = x^2\)、\(g(x) = e^x\)、\(g(x) = -\ln x\)（\(x > 0\)）、\(g(x) = |x|\)、\(g(x) = x^p\)（\(p \geq 1\)，\(x \geq 0\)）。

"凹"是凸的反面——\(g\) 凹当 \(-g\) 凸；典型凹函数：\(\ln x\)、\(\sqrt x\)。

Jensen 不等式：

\[ \boxed{g \text{ 凸} \Longrightarrow g(\mathbb{E}[X]) \leq \mathbb{E}[g(X)].} \]

反过来，\(g\) 凹 \(\Rightarrow g(\mathbb{E}[X]) \geq \mathbb{E}[g(X)]\)。

11.3.1 证明：用切线

设 \(\mu = \mathbb{E}[X]\)。\(g\) 凸时，过点 \((\mu, g(\mu))\) 总能画一条支撑直线（切线或一般的支撑线），使整条曲线在它的上方。设这条支撑直线为 \(L(x) = a + b(x - \mu)\)，满足 \(L(\mu) = g(\mu)\) 且 \(L(x) \leq g(x)\) 对所有 \(x\)。

对所有 \(x\) 取期望：

\[ \mathbb{E}[L(X)] = a + b(\mathbb{E}[X] - \mu) = a + 0 = g(\mu). \]

而由 \(L(x) \leq g(x)\) 且期望保单调性，\(\mathbb{E}[L(X)] \leq \mathbb{E}[g(X)]\)。所以

\[ g(\mu) = \mathbb{E}[L(X)] \leq \mathbb{E}[g(X)]. \quad \square \]

等号成立的条件是 \(X\) 几乎处处取常数（或 \(g\) 在 \(X\) 取值范围内恰为线性）。

11.3.2 应用一：方差永远非负

取 \(g(x) = x^2\)（凸）。Jensen 给

\[ (\mathbb{E}[X])^2 \leq \mathbb{E}[X^2]. \]

移项：\(\mathbb{E}[X^2] - (\mathbb{E}[X])^2 \geq 0\)。但这左边正是 \(\mathrm{Var}(X)\)！所以 Jensen 直接给出：方差非负。这条性质我们一直用，但这是它最干净的来源。

11.3.3 应用二：算术-几何均值不等式（AM-GM）

取 \(g(x) = -\ln x\)（凸；因为 \((-\ln x)'' = 1/x^2 > 0\)）。让 \(X\) 在 \(\{x_1, \ldots, x_n\}\)（正数）上均匀取值（每个概率 \(1/n\)）。Jensen：

\[ -\ln\!\left(\frac{x_1 + \cdots + x_n}{n}\right) \leq \frac{1}{n}\sum_{i=1}^n (-\ln x_i). \]

两边乘 \(-1\)（变号反向）：

\[ \ln\!\left(\frac{x_1 + \cdots + x_n}{n}\right) \geq \frac{1}{n}\sum \ln x_i = \ln\!\left(\sqrt[n]{x_1 \cdots x_n}\right). \]

取指数（保单调）：

\[ \boxed{\frac{x_1 + x_2 + \cdots + x_n}{n} \geq \sqrt[n]{x_1 x_2 \cdots x_n}.} \]

这就是算术-几何均值不等式——左边算术均值，右边几何均值。中学竞赛里要费力证明的事情，用 Jensen 三行就完。

11.3.4 应用三：信息论的非负性

另一个 Jensen 经典应用：信息论里的 KL 散度（Kullback-Leibler divergence）非负。设两个离散概率分布 \(p, q\) 在同一支集上，定义

\[ D(p \| q) = \sum_x p(x) \ln \frac{p(x)}{q(x)} = \mathbb{E}_p\!\left[\ln \frac{p(X)}{q(X)}\right]. \]

由 \(-\ln\) 凸性（Jensen，反向方向）：

\[ D(p \| q) = \mathbb{E}_p\!\left[-\ln \frac{q(X)}{p(X)}\right] \geq -\ln\, \mathbb{E}_p\!\left[\frac{q(X)}{p(X)}\right] = -\ln\, \sum_x p(x) \cdot \frac{q(x)}{p(x)} = -\ln\, 1 = 0. \]

所以 \(D(p \| q) \geq 0\)，且等号当且仅当 \(p = q\)。这条事实是整个信息论与现代机器学习里"交叉熵损失"的原理性根据。一切由 Jensen 一招推出。

例 11.3（Jensen 的初等示意）　一只股票每年随机增长 +50% 或 -50%，等概率独立。"平均"年增长率是 0。请问 10 年后你预期剩下多少本金？又"平均下来"剩多少？

解：设初始 \(\$1\)。10 年后本金 \(W = (1 + R_1)(1 + R_2)\cdots(1 + R_{10})\)，其中每个 \(1 + R_i \in \{1.5, 0.5\}\) 各 \(1/2\)。

"预期剩多少"——求 \(\mathbb{E}[W]\)：

\[ \mathbb{E}[W] = \prod_{i=1}^{10} \mathbb{E}[1 + R_i] = (1.0)^{10} = 1. \]

"平均下来"——更接近"几何均值"，即 \(W^{1/10}\)。其期望或典型值：

\[ \mathbb{E}[\ln W] = 10 \cdot \mathbb{E}[\ln(1+R)] = 10 \cdot \frac{1}{2}(\ln 1.5 + \ln 0.5) = 10 \cdot \frac{1}{2} \cdot \ln 0.75 \approx -1.44. \]

所以"中位"本金 \(\approx e^{-1.44} \approx 0.24\)——只剩 24%！

这就是Jensen 在金融里的著名结论："期望增长率"和"长期典型增长率"不一样。期望被极少数大胜利拉高了，绝大多数路径其实在亏。这就是为什么"上涨 50%、下跌 50%"轮替不会让你回到原点——而是亏。Jensen 不等式给出严格的方向：\(\mathbb{E}[\ln W] \leq \ln \mathbb{E}[W]\)（\(\ln\) 凹），所以"对数期望"小于"期望对数"，长期路径"压制"住了平均收益。

11.4 Chernoff 界：指数型 Markov

Markov 不等式只用一阶矩，给出的是 \(O(1/a)\) 形式的衰减，对很多场合都太松。一个标准的加强办法是把 Markov 用到 \(e^{tX}\) 上——这能利用 \(X\) 的所有矩信息（通过 MGF），从而给出指数衰减的尾界。这就是 Chernoff 界。

对任意 \(t > 0\)：事件 "\(X \geq a\)" 等价于事件 "\(e^{tX} \geq e^{ta}\)"（指数函数严增）。\(e^{tX}\) 是非负的，所以可以用 Markov：

\[ P(X \geq a) = P(e^{tX} \geq e^{ta}) \leq \frac{\mathbb{E}[e^{tX}]}{e^{ta}} = e^{-ta}\, M_X(t). \]

这条对每个 \(t > 0\) 都成立，所以可以选最优的 \(t\) 让上界最小：

\[ \boxed{P(X \geq a) \leq \inf_{t > 0}\, e^{-ta}\, M_X(t).} \]

这是Chernoff 界。它的强大之处在于：当 \(M_X(t)\) 是某个"良性"形式（比如指数函数），对 \(t\) 优化后通常能给出像 \(e^{-c \cdot a^2}\) 这种 指数级衰减 的界——比 Markov 的 \(1/a\) 强得多。

例 11.4（Chernoff 用在 \(n\) 次硬币）　设 \(S_n\) 是 \(n\) 次独立公平硬币的正面数（\(\mathrm{Bin}(n, 1/2)\)）。求 \(P(S_n \geq n/2 + a)\) 的 Chernoff 界。

解：MGF \(M_{S_n}(t) = \big(\frac{1+e^t}{2}\big)^n\)。Chernoff：

\[ P(S_n \geq n/2 + a) \leq e^{-t(n/2 + a)}\, \left(\frac{1 + e^t}{2}\right)^n. \]

把 \(t\) 取小（具体优化要算导数为零，结果 \(t \approx 4a/n\) 当 \(a \ll n\)）。最终（标准结果）：

\[ P(S_n \geq n/2 + a) \leq e^{-2 a^2 / n}. \]

这是著名的 Hoeffding 不等式——它告诉你"超过均值 \(a\)"的概率指数式衰减，速度由 \(2a^2/n\) 控制。

对比一下：

  • Chebyshev 给 \(P(S_n \geq n/2 + a) \leq \frac{n/4}{a^2}\)（用 \(\mathrm{Var}(S_n) = n/4\)），即 \(O(n/a^2)\)。
  • Chernoff 给 \(O(e^{-2 a^2/n})\)。

取 \(n = 100\)，\(a = 30\)：Chebyshev 给 \(\leq 25/900 \approx 2.8\%\)；Chernoff 给 \(\leq e^{-1800/100} = e^{-18} \approx 1.5 \times 10^{-8}\)。差了 6 个数量级！这就是为什么 Chernoff 界在现代算法、机器学习、随机化算法里被频繁使用。

本节我们只取一个例子的味道，更系统的 Chernoff 界处理留给概率论进阶教材。

11.5 Cauchy-Schwarz 不等式：协方差与相关系数

本章最后一条不等式叫 Cauchy-Schwarz 不等式。它原本是关于向量内积的，在期望版本里说的是：

\[ \boxed{|\mathbb{E}[XY]|^2 \leq \mathbb{E}[X^2] \cdot \mathbb{E}[Y^2].} \]

等价形式（取平方根）：\(|\mathbb{E}[XY]| \leq \sqrt{\mathbb{E}[X^2]} \cdot \sqrt{\mathbb{E}[Y^2]}\)。

11.5.1 证明：判别式法

对任意实数 \(t\)，考虑非负量

\[ \mathbb{E}[(X - tY)^2] \geq 0. \]

展开：

\[ \mathbb{E}[X^2] - 2t \mathbb{E}[XY] + t^2 \mathbb{E}[Y^2] \geq 0. \]

这是 \(t\) 的二次多项式，恒非负。所以判别式 \(\leq 0\)：

\[ 4 (\mathbb{E}[XY])^2 - 4 \mathbb{E}[X^2] \mathbb{E}[Y^2] \leq 0, \]

整理即得 Cauchy-Schwarz。

11.5.2 用在协方差上：相关系数 \(\in [-1, 1]\)

把 Cauchy-Schwarz 用到 "中心化" 变量 \(X - \mu_X\)、\(Y - \mu_Y\) 上：

\[ |\mathrm{Cov}(X, Y)|^2 = |\mathbb{E}[(X - \mu_X)(Y - \mu_Y)]|^2 \leq \mathbb{E}[(X-\mu_X)^2] \mathbb{E}[(Y-\mu_Y)^2] = \mathrm{Var}(X) \mathrm{Var}(Y). \]

开方：

\[ |\mathrm{Cov}(X, Y)| \leq \sigma_X \cdot \sigma_Y. \]

这条不等式让我们能定义一个标准化的"线性相关度量"：

\[ \boxed{\rho(X, Y) = \frac{\mathrm{Cov}(X, Y)}{\sigma_X\, \sigma_Y} \in [-1, +1].} \]

这就是相关系数（correlation coefficient），中文也叫皮尔逊相关系数。

  • \(\rho = +1\)：\(X\) 和 \(Y\) 完全正线性相关，即存在 \(a > 0, b\) 使 \(Y = aX + b\)。
  • \(\rho = -1\)：完全负线性相关，\(Y = aX + b\)，\(a < 0\)。
  • \(\rho = 0\)：线性无关；但不意味着独立（10.4 节那个 \(Y = X^2\) 的反例 \(\rho = 0\) 但有强非线性依赖）。

等号 \(|\rho| = 1\) 在 Cauchy-Schwarz 中，恰对应"\(X - tY\) 退化为常数"——即 \(X\) 是 \(Y\) 的仿射函数。所以相关系数确实只反映线性关系。

例 11.5（计算相关系数）　设 \(X \sim \mathrm{Uniform}(-1, 1)\)，\(Y = X^2\)。求 \(\rho(X, Y)\)。

解：\(\mathbb{E}[X] = 0\)，\(\mathbb{E}[X^3] = 0\)（奇）。所以

\[ \mathrm{Cov}(X, Y) = \mathbb{E}[XY] - \mathbb{E}[X]\mathbb{E}[Y] = \mathbb{E}[X^3] - 0 = 0. \]

从而 \(\rho = 0\)。但 \(X\) 与 \(Y = X^2\) 显然不独立。这又一次提醒：\(\rho = 0\) 仅意味着线性无关，不意味着独立。

例 11.6（线性回归的"最佳"系数）　设 \(X, Y\) 为随机变量，参数 \(\mu_X, \mu_Y, \sigma_X, \sigma_Y, \rho\)。要找常数 \(a, b\) 使均方误差 \(\mathbb{E}[(Y - a - bX)^2]\) 最小。求 \(a, b\)。

解：对 \(a, b\) 求偏导为零。\(\partial/\partial a = -2 \mathbb{E}[Y - a - bX] = 0\) 给 \(a = \mu_Y - b\mu_X\)。\(\partial/\partial b = -2\mathbb{E}[X(Y - a - bX)] = 0\) 给 \(\mathbb{E}[XY] = a\mu_X + b \mathbb{E}[X^2]\)。代入 \(a\) 解出

\[ b = \frac{\mathrm{Cov}(X, Y)}{\mathrm{Var}(X)} = \rho \cdot \frac{\sigma_Y}{\sigma_X}. \]

这就是著名的简单线性回归系数公式。请你注意：相关系数 \(\rho\) 直接给出"最佳线性预测斜率"——\(\rho = 0\) 时最佳斜率为 0（\(X\) 对预测 \(Y\) 没有线性贡献）；\(|\rho| = 1\) 时预测完美。

例 11.7（Markov 用到非线性变换）　设 \(X \geq 0\)。试证 \(P(X \geq a) \leq e^{-ta}\, \mathbb{E}[e^{tX}]\) 对任意 \(t > 0\)。这条不等式比 Markov 紧吗？

解：注意 \(X \geq a \iff e^{tX} \geq e^{ta}\)（\(t > 0\)）。\(e^{tX}\) 非负，对它用 Markov：

\[ P(e^{tX} \geq e^{ta}) \leq \frac{\mathbb{E}[e^{tX}]}{e^{ta}} = e^{-ta}\, M_X(t). \]

这就是 11.4 节的 Chernoff 界推导。它更紧当且仅当 \(M_X\) 长得 "够好"（在某个 \(t\) 处使 \(e^{-ta} M_X(t) < \mathbb{E}[X]/a\)）。对正态、伯努利等良性分布通常都更紧。

例 11.8（Jensen 在金融的另一个例子）　投资 \(X\) 的"对数收益率" \(\ln(1 + R)\) 服从某个分布（不必正态）。证明：复利预期收益率 \(\mathbb{E}[\ln(1 + R)]\) 不会 超过简单预期收益率的对数 \(\ln\, \mathbb{E}[1 + R]\)。

解：\(\ln\) 是凹函数。Jensen：\(\ln \mathbb{E}[1 + R] \geq \mathbb{E}[\ln(1 + R)]\)。证毕。

意思：长期复利的"几何均值"不会超过短期算术均值的对数。这条事实在长期投资文献里反复出现——它在数学上是 Jensen 一行，金融上则解释了"为什么主动管理基金长期跑不赢指数"等无数现象。

这一章我们攒下了几把"小刀"——Markov、Chebyshev、Jensen、Chernoff、Cauchy-Schwarz——它们用极少信息控制概率与期望。下一章的大数定律，将把 Chebyshev 当作主刀：用一行计算把 \(\mathrm{Var}(\bar X_n) = \sigma^2/n\) 翻成 "\(\bar X_n\) 收敛到 \(\mu\)"。请你准备好——大数定律是统计学全部基础的"基础的基础"。
