第 13 章 中心极限定理

第 12 章的大数定律告诉我们："样本均值 \(\bar X_n\) 趋于真实均值 \(\mu\)"。本章要讲的中心极限定理（Central Limit Theorem，简称 CLT）告诉我们一件深得多的事情：

偏差 \(\bar X_n - \mu\) 用 \(\sigma/\sqrt n\) 标准化后，分布总是收敛到标准正态——无论原始 \(X_i\) 服从什么分布。

这条事实震撼到几乎像魔法。无论你的随机变量是均匀的、二项的、指数的、几何的，或者长得奇形怪状的某个混合分布——只要它们独立同分布且方差有限，许多个加起来再标准化，分布就会趋近正态分布 \(\mathcal N(0, 1)\)。这就解释了为什么正态分布在自然界里反复出现——它是一切独立小因素聚合的极限形状。身高、考试分数、测量误差、某些金融收益、噪声电压……凡是"许多个独立小贡献求和"的量，都倾向于正态。这是"宇宙级"的现象。

本章会讲：

  • CLT 的直觉与陈述。
  • 用 MGF 给出的非严格证明梗概——你可以看出"为什么是正态"。
  • 几何直觉——用模拟展示分布"收紧 + 正态化"的过程。
  • 实用经验：\(n\) 多大才"够"？"民调误差为什么 \(\propto 1/\sqrt n\)"。
  • 应用：质量控制的 \(3\sigma\) 法、二项分布的正态近似、大公司财务方差。
  • CLT 为什么在自然界里"宇宙地"出现。
  • 多元 CLT 简介。

13.1 直觉：把许多独立同分布加起来

先做一个可手算的小实验。

设 \(X \sim \mathrm{Uniform}(0, 1)\)。它的密度是矩形——在 \([0, 1]\) 上 \(=1\)，外面 \(=0\)。这毫不像正态。

现在让 \(Y = X_1 + X_2\)（两个独立 \(\mathrm{Uniform}(0, 1)\) 之和）。可以求得 \(Y\) 的密度是三角形——\(0\) 处密度 0，\(1\) 处密度 1（峰），\(2\) 处密度 0。已经"开始变弯"了。

取 \(n = 12\) 个独立 \(\mathrm{Uniform}(0,1)\) 之和：\(S_{12} = \sum_{i=1}^{12} X_i\)。每个 \(X_i\) 期望 \(1/2\)、方差 \(1/12\)，所以 \(\mathbb{E}[S_{12}] = 6\)、\(\mathrm{Var}(S_{12}) = 1\)。

把 \(S_{12}\) 的密度画出来——会发现它非常像 \(\mathcal N(6, 1)\) 的钟形曲线。事实上 \(S_{12} - 6\) 已经被"老一辈"用作生成 "近似标准正态" 随机数的快速方法（早期计算机时代曾流行）。

这就是 CLT 的精髓——从矩形开始，加几次就钟形了。无论起点多奇怪，求和都会"洗"成正态。

13.2 形式陈述

把上面的直觉变成定理。

中心极限定理（Lindeberg-Lévy 形式）　设 \(X_1, X_2, \ldots\) 独立同分布，\(\mathbb{E}[X_i] = \mu\)、\(\mathrm{Var}(X_i) = \sigma^2 \in (0, \infty)\)。令

\[ Z_n = \frac{\bar X_n - \mu}{\sigma / \sqrt n} = \frac{S_n - n\mu}{\sigma \sqrt n}, \]

其中 \(S_n = \sum_{i=1}^n X_i\)。则当 \(n \to \infty\)，

\[ \boxed{Z_n \xrightarrow{d} \mathcal N(0, 1).} \]

等价地写：对任意实数 \(a\)，

\[ \lim_{n \to \infty} P(Z_n \leq a) = \Phi(a) = \int_{-\infty}^a \frac{1}{\sqrt{2\pi}} e^{-z^2/2}\, dz. \]

请注意这条定理的"标准化"步骤——分子是"偏差" \(S_n - n\mu\)，分母是"标准差" \(\sigma\sqrt n\)。这两个数让 \(Z_n\) 的均值 \(=0\)、方差 \(=1\)，与目标 \(\mathcal N(0, 1)\) 匹配。如果不标准化，"偏差" 本身按 \(\sqrt n\) 膨胀（标准差是 \(\sigma\sqrt n\)），不会收敛到固定分布；如果除以 \(n\)，"偏差比" 按 \(\sigma/\sqrt n\) 收敛到 0（这就是大数定律）。CLT 是"恰到好处"的标准化——除以 \(\sqrt n\) 让它落到一个非平凡分布。

所以请把 CLT 看成"大数定律的精化"：

  • 大数定律：\(\bar X_n \to \mu\)，速率 \(O(1/\sqrt n)\)（即 \(\bar X_n - \mu\) 标准差按 \(1/\sqrt n\) 缩）。
  • 中心极限定理：\(\bar X_n - \mu\) 标准化后，分布趋近 \(\mathcal N(0, \sigma^2/n)\)。不仅仅说"趋于 0"，还具体描述 偏差的分布形状。

用 CLT 我们可以做一件大数定律做不到的事情：计算偏差落在某个范围的概率。比如 \(P(\bar X_n \in [\mu - 1.96\sigma/\sqrt n, \mu + 1.96\sigma/\sqrt n]) \approx 95\%\)（用 \(\Phi(1.96) - \Phi(-1.96) \approx 0.95\)）。这是统计推断里所有"95% 置信区间"的来源。

13.3 用 MGF 证明梗概

本节给一个非严格的证明思路——足以让你看到"为什么是正态"。完整证明要细化"MGF 收敛 \(\Rightarrow\) 分布收敛"这一步（属于 Lévy 连续性定理），我们略过。

设 \(Y_i = (X_i - \mu)/\sigma\)——把每个 \(X_i\) 标准化。则 \(\mathbb{E}[Y_i] = 0\)、\(\mathrm{Var}(Y_i) = 1\)、所有 \(Y_i\) 独立同分布。注意

\[ Z_n = \frac{1}{\sqrt n}\sum_{i=1}^n Y_i. \]

设 \(M(t) = \mathbb{E}[e^{tY_1}]\) 是 \(Y_1\) 的 MGF（假定它在 0 的某邻域有限）。\(M(0) = 1\)，\(M'(0) = \mathbb{E}[Y_1] = 0\)，\(M''(0) = \mathbb{E}[Y_1^2] = 1\)。

由独立性（10.9 节），\(\sum Y_i\) 的 MGF 是 \(M(t)^n\)。除以 \(\sqrt n\)：\(Z_n = \frac{1}{\sqrt n}\sum Y_i\) 的 MGF 是

\[ M_{Z_n}(t) = \mathbb{E}\!\left[e^{t \cdot \frac{1}{\sqrt n}\sum Y_i}\right] = \prod_{i=1}^n \mathbb{E}\!\left[e^{(t/\sqrt n) Y_i}\right] = \left[M\!\left(\frac{t}{\sqrt n}\right)\right]^n. \]

把 \(M(t/\sqrt n)\) 在 0 处泰勒展开：

\[ M(t/\sqrt n) = 1 + 0 \cdot \frac{t}{\sqrt n} + \frac{1}{2}\cdot \frac{t^2}{n} + O(n^{-3/2}) = 1 + \frac{t^2}{2n} + O(n^{-3/2}). \]

所以

\[ M_{Z_n}(t) = \left(1 + \frac{t^2}{2n} + O(n^{-3/2})\right)^n. \]

用 \(\lim_{n\to\infty}(1 + a_n/n)^n = e^a\)（当 \(a_n \to a\) 时）：右边的对数 \(= n \ln(1 + t^2/(2n) + \ldots) \approx n \cdot t^2/(2n) = t^2/2\)，所以

\[ \lim_{n \to \infty} M_{Z_n}(t) = e^{t^2/2}. \]

而 \(e^{t^2/2}\) 正是 \(\mathcal N(0, 1)\) 的 MGF（10.7 节）！由 MGF 唯一性，\(Z_n \xrightarrow{d} \mathcal N(0, 1)\)。证毕（梗概）。

这条证明的核心是泰勒展开——对 MGF 在 0 处展开到二阶，第三阶以上由 \(\sqrt n\) 缩放自动稀释为 0。也就是说只有"前两阶矩" \(\mathbb{E}[Y]\) 和 \(\mathbb{E}[Y^2]\) 决定极限分布，更高阶矩"被洗掉"。这就是为什么无论原始分布长什么样，极限都是同一个正态——前两阶矩之外的细节统统被极限抹平。

这种"高阶细节被稀释"的现象是 CLT 强大与"普适"的根本——正态分布既不偏爱二项也不偏爱均匀，它只是"二阶矩决定一切" 这件事的化身。

13.4 几何直觉：用模拟看 CLT

本节用伪代码展示三个不同分布的样本均值如何"长成正态"。建议你打开任何编程环境（Python/R/Julia）跑一遍，亲眼看到这件事。

实验 1：均匀分布的样本均值　

\[ \begin{aligned} &\text{for trial} = 1, \ldots, 10000: \\ &\quad X_1, \ldots, X_n \sim \mathrm{Uniform}(0, 1) \\ &\quad \text{record } \bar X_n \end{aligned} \]

对 \(n = 1, 2, 5, 30\) 分别画 \(\bar X_n\) 的直方图。\(n = 1\) 是矩形（明显非正态）；\(n = 2\) 是三角形；\(n = 5\) 已经像钟形；\(n = 30\) 几乎完美的正态。

实验 2：极偏分布的样本均值　设每个 \(X_i \sim \mathrm{Exp}(1)\)（指数分布，密度 \(e^{-x}\)，单边、尾巴重）。

画 \(\bar X_n\) 的直方图：\(n = 1\) 是单边指数衰减；\(n = 5\) 仍偏右；\(n = 30\) 已对称为钟形；\(n = 100\) 几乎正态。

结论：偏分布需要更大的 \(n\) 才能逼近正态。指数分布大概要 \(n \geq 30\)；如果是更偏的（比如对数正态），可能要 \(n \geq 100\) 甚至更多。

实验 3：二项分布　设 \(S_n \sim \mathrm{Bin}(n, p)\)，\(p = 0.3\)。\(\bar X_n = S_n/n\)。直方图：\(n = 5\) 仍偏（小 \(n\) 二项偏左）；\(n = 50\) 几乎正态；\(n = 1000\) 完全正态。

所以 CLT 的"收敛速率" 与分布的偏度相关——越偏越慢，越对称越快。但极限都是同一个正态。

13.5 \(n\) 多大才"够"？

实际应用里我们老问"\(n\) 取多大才能用 CLT 近似？" 没有铁律，但有几条经验：

  • "温和"分布（接近对称、单峰、尾巴不太重）：\(n \geq 30\) 通常足够。这是几乎所有教科书引用的"经验法则"。
  • 偏分布（指数、对数正态、伽马）：\(n \geq 50\) 至 \(100\)。
  • 极偏 / 重尾：可能要 \(n \geq 1000\) 甚至更多。某些"边界"分布（柯西）永远不收敛——不满足 CLT 假设。
  • 二项 \(\mathrm{Bin}(n, p)\)：经验法则 \(np \geq 5\) 且 \(n(1-p) \geq 5\)（同时满足）。\(p\) 越接近 \(0.5\)，越快趋近正态；\(p\) 极端（接近 0 或 1）时偏度大，要 \(n\) 更大。

这些都是经验法则——不是数学定理。具体情形要画图或做模拟看。

请注意：CLT 是个极限定理，对有限 \(n\) 永远只是近似。真要严格控制误差，可以用Berry-Esseen 定理给出"近似误差按 \(1/\sqrt n\) 衰减"，但常数依赖三阶绝对矩。我们这本入门书不细谈。

13.6 应用：抽样误差按 \(1/\sqrt n\) 缩——民调公式

本节看 CLT 的最广为流传的应用：民调与抽样调查。

设候选人真实支持率 \(p\)，民调员独立访问 \(n\) 个人，\(\hat p_n\) 是样本支持率。已知 \(\mathbb{E}[\hat p_n] = p\)、\(\mathrm{Var}(\hat p_n) = p(1-p)/n\)。

由 CLT，

\[ \frac{\hat p_n - p}{\sqrt{p(1-p)/n}} \xrightarrow{d} \mathcal N(0, 1). \]

所以对大 \(n\)，

\[ \hat p_n \approx \mathcal N(p,\, p(1-p)/n). \]

"95% 区间"——\(\hat p_n\) 落在 \(p \pm 1.96 \sqrt{p(1-p)/n}\) 内的概率约 95%（因为 \(P(-1.96 \leq Z \leq 1.96) = 0.95\) 对 \(Z \sim \mathcal N(0, 1)\)）。

实际中 \(p\) 未知，常用最坏情况 \(p(1-p) \leq 1/4\)（在 \(p = 1/2\) 取大），得

\[ \text{误差幅度} \leq 1.96 \cdot \sqrt{\frac{1/4}{n}} = \frac{0.98}{\sqrt n} \approx \frac{1}{\sqrt n}. \]

这就是民调结果的"误差为 \(\pm 1/\sqrt n\)"。具体：

  • \(n = 1000\)：误差 \(\approx 1/\sqrt{1000} \approx 3.16\%\)。这就是新闻里"误差 ±3 个百分点"的来源。
  • \(n = 1500\)：误差 \(\approx 2.58\%\)，常用的"误差 ±3%" 公式严格写是 \(0.98/\sqrt{1500} \approx 0.025 = 2.5\%\)。新闻里"\(\pm 3\%\) 误差，\(n = 1500\) 受访者" 就是这条 \(0.03 \approx 1/\sqrt{1500}\) 的近似。
  • \(n = 10000\)：误差 \(\approx 1\%\)。要让误差再缩一倍到 \(0.5\%\)，需 \(n = 40000\)（\(1/\sqrt n\) 缩一倍要 \(n\) 翻 4 倍）。

这条 \(1/\sqrt n\) 速率非常重要——它告诉你样本量四倍才换一倍精度。这就是为什么民调经济上停在 \(n \approx 1000\)——再加样本，精度提升缓慢，不划算。

请注意：民调真实误差除了"抽样误差"还有"非抽样误差"——选择偏倚、回答不诚实、样本不代表总体等等。CLT 给的 \(\pm 3\%\) 只对"抽样误差"有效，不覆盖那些。这是民调常常"误差范围 \(\pm 3\%\) 但实际偏离 5%" 的根本原因。

13.7 应用：质量控制中的 \(3\sigma\) 法

制造业里大量产品的质量指标（重量、长度、电阻）是许多小变量之和——机器误差、原料波动、操作员手势之差等等。由 CLT，质量指标 \(X\) 的分布近似正态 \(\mathcal N(\mu, \sigma^2)\)。

正态分布的"集中性"：

\[ P(|X - \mu| \leq \sigma) \approx 68.3\%, \qquad P(|X - \mu| \leq 2\sigma) \approx 95.4\%, \qquad P(|X - \mu| \leq 3\sigma) \approx 99.7\%. \]

第一条对应"68 - 95 - 99.7 法则"。第三条特别有名：正态分布约 99.7% 的质量在 \(\mu \pm 3\sigma\) 内。即"超出 \(3\sigma\) 的概率约 0.3%"。

质量控制的 \(3\sigma\) 法：

  1. 测量长期"在控制" 状态下的均值 \(\mu\) 与标准差 \(\sigma\)。
  2. 设上下控制线 UCL \(= \mu + 3\sigma\)、LCL \(= \mu - 3\sigma\)。
  3. 正常每件产品测得 \(X\)。如果 \(X\) 落在 \([\mathrm{LCL}, \mathrm{UCL}]\) 内，认为生产线 "在控"；如果出界，报警——很可能不是随机波动，而是某个系统性变化（机器故障、原料异常等）。

因为正常情形下"出界"的概率仅 0.3%，所以"经常出界" 是强烈的"非正常" 信号。这就是休哈特控制图（Shewhart control chart）的逻辑，是现代统计过程控制（SPC）的核心。

"六西格玛"（Six Sigma）质量管理体系的命名也来源于这件事——目标是把工艺控制到"\(\mu\) 距离规格上下界 \(\geq 6\sigma\)"，即不合格率降到几百万分之几（\(2 \cdot \Phi(-6) \approx 2 \cdot 10^{-9}\)）。这是制造业极致质量目标的代名词。

13.8 二项 \(\to\) 正态近似（连续性修正）

设 \(S_n \sim \mathrm{Bin}(n, p)\)，\(np\) 与 \(n(1-p)\) 都不太小。由 CLT，

\[ \frac{S_n - np}{\sqrt{np(1-p)}} \xrightarrow{d} \mathcal N(0, 1), \]

所以对中等大的 \(n\)，

\[ S_n \approx \mathcal N(np,\, np(1-p)). \]

这就是著名的De Moivre-Laplace 定理，是 CLT 的"二项特例"——18 世纪棣莫弗就发现了 \(\mathrm{Bin}(n, 1/2)\) 趋近正态，是 CLT 历史上的第一例。

用这条近似算二项概率：要算 \(P(S_n \leq k)\)，把它"标准化"用 \(\Phi\)：

\[ P(S_n \leq k) \approx \Phi\!\left(\frac{k - np}{\sqrt{np(1-p)}}\right). \]

但这里有一个细节，叫做连续性修正（continuity correction）。\(S_n\) 是整数值，正态是连续。把整数事件"\(S_n \leq k\)"近似成 "\(X \leq k + 0.5\)"（\(X\) 连续）通常更准——把整数边界向"包含 \(k\)"的方向延 0.5。

\[ \boxed{P(S_n \leq k) \approx \Phi\!\left(\frac{k + 0.5 - np}{\sqrt{np(1-p)}}\right).} \]

类似地：

\[ P(a \leq S_n \leq b) \approx \Phi\!\left(\frac{b + 0.5 - np}{\sqrt{np(1-p)}}\right) - \Phi\!\left(\frac{a - 0.5 - np}{\sqrt{np(1-p)}}\right). \]

"\(\pm 0.5\)" 这种小补偿对中等 \(n\) 的精度提升明显；\(n\) 巨大时差别可以忽略。

例 13.1（用正态近似算二项）　抛 \(n = 100\) 次公平硬币，求正面数 \(\geq 60\) 的概率。

解：\(\mathrm{Bin}(100, 0.5)\)，\(np = 50\)、\(\sqrt{np(1-p)} = 5\)。带连续修正：

\[ P(S_{100} \geq 60) = P(S_{100} \geq 60) \approx P(X \geq 59.5) = 1 - \Phi\!\left(\frac{59.5 - 50}{5}\right) = 1 - \Phi(1.9) \approx 1 - 0.9713 = 2.87\%. \]

真实二项概率（精确算）\(\approx 2.84\%\)——近似几乎完美。如果不用连续修正：\(1 - \Phi(60 - 50)/5) = 1 - \Phi(2.0) \approx 2.28\%\)，偏差较大。所以连续修正在中等 \(n\) 时很值钱。

13.9 中心极限定理为什么"宇宙地"出现

本节回到本章开头的"普适"主题。为什么自然界里这么多东西看起来正态分布？答：因为它们都是"许多独立小因素之和"。

身高：成年人身高是上千个基因 \(\times\) 童年营养 \(\times\) 数千次睡眠质量 \(\times\) 激素分泌 ... 等等的"复杂叠加"。任何单个因素影响都很小，但它们独立累积。CLT \(\Rightarrow\) 身高分布近似正态。看真实数据：人类身高确实接近正态（轻微偏度由于性别等次级因素）。

测量误差：科学实验中"测量值 = 真值 + 误差"。误差又是"仪器误差 + 操作误差 + 环境误差 + ..." 之和。CLT \(\Rightarrow\) 测量误差近似正态。这就是为什么 18-19 世纪的"误差理论" 几乎完全建立在正态分布上——高斯本人正是因为研究天文测量误差而引入了正态分布。

金融收益：短时段（日、周）股票收益常被建模为正态——由"许多独立小冲击"组成。但请注意这只是粗近似——真实金融收益常有"重尾"（黑天鹅事件），不完全正态。但短期且"安静"市场下，正态近似有效。

考试分数：一份考试的总分是"对每道题的得分"之和。每道题的对错独立程度高，CLT \(\Rightarrow\) 总分近似正态。这就是为什么标准化考试（SAT、GRE 等）的分数分布看起来都是钟形——它本质上是 CLT 在起作用。

气体分子速度：理想气体中每个分子的速度是"过去无数次碰撞冲击"的累积。CLT \(\Rightarrow\) 速度分量近似正态——这就是麦克斯韦-玻尔兹曼分布的起源（速度向量的每个分量正态，所以速率服从某种特定分布）。

这些例子有共同模式：许多个独立小贡献相加。CLT 说："不管单个贡献长什么样，加起来都是正态"。所以正态分布不是大自然偏爱的"特殊形状"，而是"无数加法的极限形状"——它是数学上的不动点，加法运算把一切都吸引到它。

反过来：当一个量是"许多独立小因素相乘"时，取对数变加法，所以对数近似正态，而原量近似对数正态。这就是为什么收入、城市人口等"乘法叠加"的量常服从对数正态。

13.10 多元中心极限定理简介

CLT 还有一个矢量值版本，叫多元中心极限定理。

设 \(\mathbf X_1, \mathbf X_2, \ldots\) 是 \(d\) 维随机向量，独立同分布，\(\mathbb{E}[\mathbf X_i] = \boldsymbol{\mu}\)、\(\mathrm{Cov}(\mathbf X_i) = \boldsymbol{\Sigma}\)（\(d \times d\) 协方差矩阵，正定）。则

\[ \sqrt n (\bar{\mathbf X}_n - \boldsymbol{\mu}) \xrightarrow{d} \mathcal N(\mathbf 0, \boldsymbol{\Sigma}), \]

其中 \(\mathcal N(\mathbf 0, \boldsymbol{\Sigma})\) 是均值 0、协方差 \(\boldsymbol{\Sigma}\) 的多元正态分布，密度是

\[ f(\mathbf z) = \frac{1}{(2\pi)^{d/2} \sqrt{\det \boldsymbol{\Sigma}}}\, \exp\!\left(-\frac{1}{2}\mathbf z^\top \boldsymbol{\Sigma}^{-1} \mathbf z\right). \]

这条多元 CLT 是统计推断的多元基础——比如 OLS 线性回归的渐近正态性，"\(\sqrt n (\hat\beta - \beta) \to \mathcal N(0, V)\)"，就是多元 CLT 的应用。

本入门书不深入推导多元 CLT，只让你知道它存在且形式与一维平行。

13.11 详细例题

例 13.2（保险公司亏损概率）　保险公司有 \(n = 10000\) 个客户，每位独立理赔金额 \(L_i\) 期望 \(\mu = 1000\) 元，标准差 \(\sigma = 500\) 元。每位收保费 \(\pi = 1050\) 元。求"年内总亏损" 的概率。

解：总盈利 \(= n\pi - \sum L_i = 10500000 - S_n\)，其中 \(S_n = \sum L_i\)。亏损 \(\iff S_n > n\pi = 10500000\)。

\(\mathbb{E}[S_n] = n\mu = 10^7\)、\(\mathrm{Var}(S_n) = n\sigma^2 = 10000 \cdot 250000 = 2.5 \times 10^9\)、\(\mathrm{SD}(S_n) = 50000\)。

由 CLT，\(S_n \approx \mathcal N(10^7, 50000^2)\)。亏损概率：

\[ P(S_n > 10500000) = P\!\left(\frac{S_n - 10^7}{50000} > \frac{500000}{50000}\right) = P(Z > 10) \approx \Phi(-10) \approx 7.6 \times 10^{-24}. \]

实际上几乎不可能亏损。这就是大数定律 + CLT 给保险公司的"定量稳赚保证"。如果只用 Chebyshev，上界 \(\leq \mathrm{Var}/(500000)^2 = 0.01\)——保守得多。CLT 利用了"分布近似正态"这条额外信息，给出指数级强的尾界。

例 13.3（CLT 与样本量决定）　某品牌饮料每瓶净含量目标 500 毫升。质检随机抽 \(n\) 瓶，记 \(\bar X_n\)。若 \(\bar X_n\) 偏离 500 超过 1 毫升就召回。设单瓶标准差 \(\sigma = 5\) 毫升。要让"误判"（过程其实在控但仍超 1 毫升）概率不超过 5%，需 \(n\) 多大？

解：在控时 \(\mathbb{E}[\bar X_n] = 500\)。由 CLT，\(\bar X_n \approx \mathcal N(500, 25/n)\)。"超 1" 概率：

\[ P(|\bar X_n - 500| > 1) = 2(1 - \Phi(\sqrt n / 5)) \leq 0.05. \]

查表 \(\Phi(1.96) \approx 0.975\)，所以要 \(\sqrt n / 5 \geq 1.96\)，即 \(n \geq (1.96 \cdot 5)^2 \approx 96.04\)。取 \(n = 97\) 就够。

对比 Chebyshev：要 \(\frac{25/n}{1} \leq 0.05\)，即 \(n \geq 500\)——5 倍多。又一次 CLT 比 Chebyshev 紧很多。

例 13.4（赌场盈利精算）　某赌场每天 \(n = 10^6\) 局赌博，每局对赌场期望盈利 \(\mu = 0.05\)（5 分）、标准差 \(\sigma = 1\)（1 元）。求"日盈利低于 4 万元" 的概率。

解：日总盈利 \(W = \sum X_i\)，期望 \(n\mu = 50000\)、方差 \(n\sigma^2 = 10^6\)、标准差 \(1000\)。由 CLT，\(W \approx \mathcal N(50000, 10^6)\)。

\[ P(W < 40000) = \Phi\!\left(\frac{40000 - 50000}{1000}\right) = \Phi(-10) \approx 7.6 \times 10^{-24}. \]

赌场日盈利低于 4 万的概率几乎为 0——它每天都赚约 5 万元，标准差只有 1000 元。这就是大数定律 + CLT 让赌场旱涝保收的精确机制。规模越大（\(n\) 越大），盈利越稳定。

例 13.5（民调样本量重算）　12.6 节用 Chebyshev 估"民调误差 \(\pm 1\%\) 概率 \(\leq 5\%\)"需要 \(n \geq 50000\)。请用 CLT 重算。

解：\(\hat p_n \approx \mathcal N(p, p(1-p)/n)\)。"\(|\hat p_n - p| \geq 0.01\)" 概率：

\[ P(|Z| \geq 0.01 / \sqrt{p(1-p)/n}) \leq 0.05 \iff \frac{0.01 \sqrt n}{\sqrt{p(1-p)}} \geq 1.96. \]

取最坏 \(p = 1/2\)，\(\sqrt{p(1-p)} = 1/2\)：

\[ \frac{0.01 \sqrt n}{1/2} \geq 1.96 \iff \sqrt n \geq 98 \iff n \geq 9604. \]

实际民调 \(n = 9604\) 就够保证 95% 的把握把误差控制在 \(\pm 1\%\)。比 Chebyshev 的 \(n = 50000\) 紧 5 倍多。这又一次表明 CLT 是更强的工具——但前提是"\(\hat p_n\) 已经足够近正态"，对 \(n = 10^4\) 量级肯定满足。

例 13.6（多元 CLT 简单实例：两维样本均值）　双变量 \((U, V)\) i.i.d. 抽样 \(n\) 次，\(\bar U_n, \bar V_n\)。\(U\) 与 \(V\) 不独立，相关系数 \(\rho\)。由二维 CLT，

\[ \sqrt n \!\left(\!\begin{matrix}\bar U_n - \mu_U \\ \bar V_n - \mu_V\end{matrix}\!\right) \xrightarrow{d} \mathcal N\!\left(\!\begin{matrix}0 \\ 0\end{matrix},\, \begin{matrix}\sigma_U^2 & \rho \sigma_U\sigma_V \\ \rho\sigma_U\sigma_V & \sigma_V^2\end{matrix}\!\right). \]

这条联合正态分布告诉我们：样本均值的"两维分布" 是个椭圆形钟形——主轴方向、椭圆扁度都由 \(\rho\) 决定。这是多元统计的渐近基础：所有"\(\sqrt n\) 倍偏差" 都收敛到多元正态，分析起来标准化、整齐。

例 13.7（CLT 失效：柯西分布）　设 \(X_i\) i.i.d. 柯西分布（密度 \(\frac{1}{\pi(1+x^2)}\)）。求 \(\bar X_n\) 的分布。

解：柯西分布的方差不存在（\(\int x^2 / (1 + x^2)\, dx = \infty\)），所以 CLT 不适用。事实上 \(\bar X_n\) 的分布仍然是柯西（与 \(n\) 无关）！它不趋向正态，不趋向任何点；它就是个不变的柯西。

这条反例提醒：CLT 需要方差有限。重尾分布（柯西、巴雷托大尾、α-稳定分布）需要不同的极限定理（"广义中心极限定理"，极限是 α-稳定分布而非正态）。但绝大多数自然/工程应用里方差有限，CLT 适用。

例 13.8（投票结果置信区间构造）　民调 \(n = 1000\) 人，得 \(\hat p = 0.52\)（即 520 人支持 A）。求 95% 置信区间。

解：标准误差 \(\hat{\mathrm{SE}} = \sqrt{0.52 \cdot 0.48 / 1000} \approx 0.0158\)。95% 置信区间：

\[ 0.52 \pm 1.96 \cdot 0.0158 \approx 0.52 \pm 0.031 = [0.489, 0.551]. \]

结论："以 95% 把握，A 真实支持率在 48.9% 到 55.1% 之间"。注意区间包含 50%——所以不能从这个民调"显著"地说 A 领先。

这是统计推断里"区间估计"的最基本套路——CLT \(\to\) 标准化 \(\to\) 用 \(\Phi^{-1}(0.975) = 1.96\) 定区间。实际的统计学课程里这条逻辑会被拓展、精化，但核心永远是 CLT。

例 13.9（连续修正的影响）　设 \(S \sim \mathrm{Bin}(20, 0.5)\)，求 \(P(S = 10)\)。

解：精确二项 \(P(S = 10) = \binom{20}{10}/2^{20} \approx 0.1762\)。

不带连续修正的"正态密度近似"：\(f_{\mathcal N(10, 5)}(10) = \frac{1}{\sqrt{10\pi}} \approx 0.1784\)——已经接近。

带连续修正：\(P(S = 10) \approx P(9.5 \leq X \leq 10.5) = \Phi(0.5/\sqrt 5) - \Phi(-0.5/\sqrt 5) \approx 2 \cdot 0.0888 = 0.1776\)——更准。

这就是为什么对"等于某个具体整数" 的概率，用区间近似 \([k - 0.5, k + 0.5]\) 比用"在那一点的密度乘 1" 更准——前者真实考虑了"整数被周围 \(\pm 0.5\) 包夹的连续区间"。

到此为止，我们已经把概率论里最重要的两条极限定理——大数定律与中心极限定理——讲完。这两条定理一起回答了"许多个随机变量加起来会怎样" 这一中心问题：

  • 大数定律：平均趋于期望（"求和洗刷个体随机性"）。
  • 中心极限定理：偏差分布趋于正态（"求和把任何形状洗成钟形"）。

从这一对定理出发，统计推断的全部基础——区间估计、假设检验、回归分析——一一展开。下一章我们正式进入统计学部分：从"知道分布算事件" 翻转到"看到数据估分布"，这就是统计推断的世界。
