附录 A 完备度量空间 + Baire 速查

本书第 2 章把"完备" 当成度量 / 范数空间的基础属性引入；第 8 章 BLT 用了一次完备性"稠密扩张"；第 10-12 章三大基础定理（开映射、闭图像、一致有界）以 Baire 范畴为无名背景；第 13-14 章 Hahn-Banach 用 Zorn 引理顺过——但"这些事在哪里完备性变成关键" 的全景图，正文里没有集中处理。这个附录把它们集中处理。

  • 完备化定理（Cauchy 序列等价类）。
  • Banach 不动点定理 + 完整证。

A.1 有限维赋范空间任意范数等价

这条是第 1 章 1.4 节的对照——"有限维上所有范数等价，无穷维上不" 区别的有限维半。

定理。设 \(V\) 是有限维赋范空间，\(\dim V = n < \infty\)。则 \(V\) 上任意两个范数 \(\|\cdot\|_a, \|\cdot\|_b\) 等价——存在 \(c, C > 0\) 使

\[ c \|x\|_a \le \|x\|_b \le C \|x\|_a \quad \forall x \in V. \]

证明。选 \(V\) 的基 \(\{e_1, \ldots, e_n\}\)。给定向量 \(x = \sum x_k e_k\)。定义比较范数

\[ \|x\|_0 := \max_k |x_k|. \]

这是个范数（直接验证）。我们证每个 范数 \(\|\cdot\|_a\) 与 \(\|\cdot\|_0\) 等价——再用传递性给两条等价。

步 1：上界 \(\|x\|_a \le M \|x\|_0\)。

\[ \|x\|_a = \|\sum x_k e_k\|_a \le \sum |x_k| \|e_k\|_a \le \big(\sum \|e_k\|_a\big) \cdot \|x\|_0 = M \|x\|_0. \]

所以 \(\|\cdot\|_a\) 是 \((V, \|\cdot\|_0)\) 上的连续函数。

步 2：下界 \(\|x\|_a \ge c \|x\|_0\)。

看单位 \(\|\cdot\|_0\)-球面 \(S = \{x : \|x\|_0 = 1\}\)。把 \(V\) 视为 \(\mathbb{R}^n\)（用基），\(\|\cdot\|_0\) 是 \(\ell^\infty\) 范数，\(S\) 是有界闭集（\(\ell^\infty\) 球面），由 Heine-Borel 紧。

\(\|\cdot\|_a\) 在 \(S\) 上连续（步 1），所以达到正最小值 \(c := \min_{x \in S} \|x\|_a > 0\)（\(c > 0\) 因为 \(0 \notin S\)，且 \(\|\cdot\|_a > 0\) 在非零向量上）。

对任意 \(x \ne 0\)：\(x / \|x\|_0 \in S \implies \|x / \|x\|_0\|_a \ge c \implies \|x\|_a \ge c \|x\|_0\)。

结合：\(c \|x\|_0 \le \|x\|_a \le M \|x\|_0\)。所以 \(\|\cdot\|_a \sim \|\cdot\|_0\)。\(\square\)

请你停下来欣赏这条证明的关键——它在第 2 步用了 Heine-Borel（有限维"有界 + 闭 ⟹ 紧"）。在无穷维这条不成立，所以"任意范数等价" 这件事在无穷维就不能这样证，事实上反例存在（第 1 章 \(C[0,1]\) 上 \(L^1, L^\infty\) 不等价）。

三个推论。

推论 1. 有限维赋范空间自动完备。证明：由等价 + \(\mathbb{R}^n\) 完备性。

推论 2. 有限维子空间在 Banach 空间中自动闭。证明：用推论 1 + 完备 \(\implies\) 闭于嵌入空间。

推论 3. 有限维之间的每个 线性映射自动连续。证明：取基写矩阵 + 矩阵给 \(\ell^\infty\) 上 Lipschitz。

这三条让"有限维" 在泛函分析里是 trivial 部分——所有崩塌都在无穷维出现。

A.2 完备化定理

不完备的度量空间可以"填满" 成完备的——通过 Cauchy 序列的等价类。这是泛函分析里对"引入新对象" 最常用的策略，比如从有理数 \(\mathbb{Q}\) 完备化得到实数 \(\mathbb{R}\)、从 Riemann 可积函数完备化得到 \(L^2\)。

定义。设 \((X, d)\) 度量空间。完备化 是指一对 \((\widehat X, j)\)，其中 \(\widehat X\) 是完备度量空间，\(j : X \to \widehat X\) 是等距嵌入（\(d_{\widehat X}(j x, j y) = d(x, y)\)），且 \(j(X)\) 在 \(\widehat X\) 中稠密。

定理。每个度量空间 \((X, d)\) 都有完备化，且本质唯一（差一个等距同构）。

构造。定义 \(C(X) := \{X \text{ 上的 Cauchy 序列}\}\)。在 \(C(X)\) 上等价关系：\(\{x_n\} \sim \{y_n\} \iff d(x_n, y_n) \to 0\)。设 \(\widehat X := C(X) / \sim\)。

距离：\(\widehat d([\{x_n\}], [\{y_n\}]) := \lim_n d(x_n, y_n)\)（极限存在因 Cauchy + 三角不等式）。

嵌入 \(j(x) := [\{x, x, x, \ldots\}]\)（常序列等价类）。直接验证 \(j\) 等距，\(j(X)\) 稠密（每个 Cauchy 序列由其前 \(n\) 项常序列逼近）。

完备性的证。设 \(\{[\{x_n^{(k)}\}]\}_{k}\) 是 \(\widehat X\) 中 Cauchy 序列。对角线选 \(y_k := x_k^{(k)}\)（足够后的项），则 \(\{y_k\}\) 是 \(X\) 中 Cauchy（用三角不等式 + Cauchy 性两次），其等价类 \([\{y_k\}]\) 是序列极限。\(\square\)

赋范情形：如果 \(X\) 是不完备赋范空间，完备化 \(\widehat X\) 自动是 Banach 空间——加法 + 数乘 + 范数都按等价类的代表逐项极限定义。

典型例：

  • 有理数 \(\mathbb{Q}\)（\(d(x, y) = |x - y|\)）的完备化 = \(\mathbb{R}\)。
  • \(C[a, b]\) 配 \(L^p\) 范数 \(\|f\|_p = (\int |f|^p)^{1/p}\) 不完备（例：\(f_n = \) sgn 函数的连续逼近，Cauchy 但极限是不连续 sgn）。完备化 = \(L^p[a, b]\)。

实际工作中你很少 直接构造完备化——而是预先在已经完备 的空间（\(L^p\)、\(C[a,b]\) 配 sup 范数、Hilbert）里工作。但完备化定理保证了"需要时总能完备化"。

A.3 Banach 不动点定理

这条是分析里"用迭代构造解" 的工作马。

定理（Banach 不动点）。设 \((X, d)\) 完备度量空间，\(T : X \to X\) 压缩（contraction）——存在 \(0 \le k < 1\) 使

\[ d(T x, T y) \le k \cdot d(x, y) \quad \forall x, y \in X. \]

则 \(T\) 有唯一 不动点 \(x^* \in X\)（即 \(T x^* = x^*\)），且对任意 \(x_0 \in X\) 迭代序列 \(x_{n+1} := T x_n\) 收敛到 \(x^*\)，速度

\[ d(x_n, x^*) \le \frac{k^n}{1 - k} d(x_0, x_1). \]

证明。

(1) 唯一性。设 \(x^*, y^*\) 都是不动点。则 \(d(x^*, y^*) = d(T x^*, T y^*) \le k \cdot d(x^*, y^*) \implies (1 - k) d(x^*, y^*) \le 0 \implies d = 0 \implies x^* = y^*\)。

(2) 存在性。取任意 \(x_0\)，定义 \(x_{n+1} = T x_n\)。计算

\[ d(x_{n+1}, x_n) = d(T x_n, T x_{n-1}) \le k d(x_n, x_{n-1}) \le k^n d(x_1, x_0). \]

所以对 \(m > n\)：

\[ d(x_m, x_n) \le \sum_{j = n}^{m - 1} d(x_{j+1}, x_j) \le \sum_{j=n}^{m-1} k^j d(x_1, x_0) \le \frac{k^n}{1 - k} d(x_1, x_0). \]

\(k < 1 \implies\) 右边 \(\to 0\) 当 \(n \to \infty\) \(\implies \{x_n\}\) Cauchy。\(X\) 完备 \(\implies\) 有极限 \(x^*\)。\(T\) 压缩 \(\implies T\) 连续 \(\implies T x_n \to T x^*\)。但 \(T x_n = x_{n+1} \to x^*\)。所以 \(T x^* = x^*\)。

(3) 收敛速率。令上面 \(m \to \infty\)：\(d(x^*, x_n) \le k^n / (1 - k) \cdot d(x_1, x_0)\)。\(\square\)

这条定理的应用范围超广：

  • 常微分方程局部存在唯一性（Picard 定理）：把 ODE \(\dot u = f(t, u)\) 转为积分方程 \(u(t) = u_0 + \int_0^t f(s, u(s)) ds\)，右边是 \(C[0, T]\) 上的算子。在 \(T\) 小时它是压缩 \(\implies\) 唯一解。
  • 隐函数定理：把"解 \(F(x, y) = 0\) 关于 \(y\) 隐式表为 \(y = g(x)\)" 转为 Banach 空间上的不动点。
  • Newton 法收敛性：在 \(f'(x^*) \ne 0\) 附近 Newton 迭代是压缩，二次收敛是压缩常数 \(\to 0\) 的体现。
  • Markov 链平稳分布存在唯一性（在合适度量下转移算子是压缩）。

"把存在性问题转为不动点问题，再用 Banach" 是分析的标准战术。需要的两件事是 (a) 选完备空间，(b) 把"解的方程" 写成 \(T x = x\) 形态，再验证 \(T\) 是压缩。

A.4 Fekete 子可加引理

在动力系统、组合学、数论里反复出现的"子可加序列除以 \(n\) 收敛" 引理。

Fekete 引理（1923）。设 \(\{a_n\}_{n \ge 1}\) 是子可加实数列：\(a_{m + n} \le a_m + a_n\) 对所有 \(m, n \ge 1\)。则

\[ \lim_{n \to \infty} \frac{a_n}{n} = \inf_{n \ge 1} \frac{a_n}{n} \in [-\infty, +\infty). \]

极限必存在（可能 \(-\infty\)），且等于下确界。

证明。记 \(L := \inf_n a_n / n\)。要证 \(\limsup a_n/n \le L\)。

给 \(\varepsilon > 0\)，选 \(m\) 使 \(a_m/m < L + \varepsilon\)。对任意 \(n\)，写 \(n = q m + r, 0 \le r < m\)。子可加：

\[ a_n \le q \cdot a_m + a_r. \]

所以

\[ \frac{a_n}{n} \le \frac{q a_m + a_r}{n} = \frac{q m}{n} \cdot \frac{a_m}{m} + \frac{a_r}{n} \to \frac{a_m}{m} \quad (n \to \infty) \]

（\(qm/n \to 1\), \(a_r/n \to 0\) 因为 \(r\) 有限多个值）。所以 \(\limsup a_n/n \le a_m/m < L + \varepsilon\)。\(\varepsilon\) 任意 \(\implies \limsup \le L\)。结合 \(\liminf \ge L\)（下确界）：极限 = \(L\)。\(\square\)

典型应用：

  • 谱半径公式 \(r(T) = \lim \|T^n\|^{1/n}\)（用 \(a_n = \log \|T^n\|\) 子可加由 \(\|T^{m+n}\| \le \|T^m\| \|T^n\|\)）。
  • 动力系统拓扑熵（Bowen-Walters 定义里覆盖数 \(N(n, \varepsilon)\) 子可加）。
  • 组合学：自避走道路数 \(c_n\) 满足 \(c_{m+n} \le c_m c_n\)（取 log 子可加）\(\implies\) 增长率 \(\mu := \lim c_n^{1/n}\) 存在。

这条引理"子可加 \(\implies\) 比率收敛" 是个非常有力但易被忽视的工具。

A.5 Krein-Milman 定理

"紧凸集 = 端点的凸闭包"——在无穷维拓扑向量空间里仍然成立。

定义。设 \(K\) 凸集，\(x \in K\) 是端点（extreme point），如果不存在 \(y, z \in K, y \ne z, t \in (0, 1)\) 使 \(x = t y + (1 - t) z\)。即 \(x\) 不能写成 \(K\) 中两点的真凸组合。

记 \(\mathrm{ext}(K) :=\) \(K\) 的端点集。

Krein-Milman 定理（1940）。设 \(X\) 是局部凸 Hausdorff 拓扑向量空间（如赋范 / 弱拓扑），\(K \subset X\) 非空 紧凸。则

\[ K = \overline{\mathrm{conv}(\mathrm{ext}(K))}. \]

即 \(K\) 是其端点凸组合的闭包。

具体的：

  • 有限维：\(K \subset \mathbb{R}^n\) 紧凸，\(\mathrm{ext}(K)\) 是 \(K\) 的顶点集。Krein-Milman 给"多面体 = 顶点凸包"——朴素事实。
  • \(\mathbb{R}^n\) 单位球面：每个边界点都是端点（球面无"平直部分"），\(\mathrm{ext}(\overline{B}) = S^{n-1}\)。
  • \(\ell^p\) 单位球，\(1 < p < \infty\)：每个 \(\|x\|_p = 1\) 的 \(x\) 都是端点（严格凸性）。
  • \(\ell^1\) 单位球：端点 = 标准基 \(\pm e_n\)（"角点"）。\(\ell^1\) 不严格凸，所以非角点不是端点。
  • \(\ell^\infty\) 单位球：端点 = 极端序列 \(\{x : x_n = \pm 1\}\)（每位都达到 ±1）——超大集合。

证明思路（一页粗略）：用 Hahn-Banach 找支撑超平面，沿该平面切下"面"（face），递归到一维，最后得端点。完整证明 2-3 页技术，请见 Conway 第 4 章。

这条定理在概率 / 统计 / 优化里超有用——说"找紧凸集上某线性泛函的极值" 等价于"在端点上找"，因为端点凸包的线性泛函最值就是端点上最值。这是线性规划 单纯形法的无穷维版背景。

A.6 Eberlein-Šmulian 与 Goldstine

这两条是弱拓扑 的精细工具，本书第 17-19 章用过若干次。

Eberlein-Šmulian 定理。设 \(X\) Banach。一个集 \(K \subset X\) 在弱拓扑 下相对紧 \(\iff\) 它序列 弱相对紧（即每个序列有弱收敛子列）。

这条不显然，因为弱拓扑一般不第一可数（无穷维 + 不可分时）——而紧 \(\iff\) 序列紧" 在非第一可数空间一般失败。Eberlein-Šmulian 说在 Banach 弱拓扑下二者恰好 等价。证明用 Banach-Alaoglu + 双对偶嵌入 + 局部凸性，6-8 页技术。

实际后果：弱紧致论证可以"抽序列子列" 而不需要"抽网"。这是第 20 章"弱抽 + 紧升 + 强传" 三步法用序列 形式正确的根基。

Goldstine 定理。设 \(X\) Banach，\(B_X = \{x : \|x\| \le 1\}\) 单位球，\(B_{X^{**}} = \{\xi \in X^{**} : \|\xi\| \le 1\}\) 双对偶单位球。在典范嵌入 \(X \hookrightarrow X^{**}\) 下，

\[ B_X \text{ 在 } X^{**} \text{ 弱*拓扑下稠密于 } B_{X^{**}}. \]

即：每个 \(\xi \in X^{**}\) 单位向量都是 \(X\) 中单位向量的弱*极限。

这条定理"双对偶不大于单对偶的弱*闭包"。它是\(X\) 自反 (\(X = X^{**}\)) 的等价刻画的工具——\(X\) 自反 \(\iff B_X\) 弱紧 \(\iff\) Goldstine 给的弱*稠密性实际是 \(B_X = B_{X^{**}}\)。

证明用 Hahn-Banach 分离 + 凸闭包等于弱闭包（对凸集）+ 双对偶范数关系，2 页技术。

A.7 实战准则

  • "完备" 是 Banach / Hilbert 定义的核心——它是反复有用的"底层资源"，让 BLT、Banach-Steinhaus、开映射、闭图像都成立。每次见到"完备空间" 这条假设，知道它是不能省的——本书第 12-14 章三大基础定理全部依赖它。
  • 有限维的"任意范数等价" + "子空间自动闭" + "线性映射自动连续" 三条让有限维成为泛函分析中的 trivial 部分——所有努力都在无穷维。
  • Banach 不动点是构造 工具——只要能把问题写成 \(T x = x\) 且 \(T\) 压缩，存在唯一解就是免费 的。配套的 Picard 定理是 ODE 标准存在唯一性。
  • Krein-Milman 在凸优化、概率（极端测度）、控制论里反复用。它的"对偶" 用法（找端点 → 推出整个集合的性质）值得记住。
  • Eberlein-Šmulian 让你在弱紧致论证里放心地用序列——这条在第 23 章 PDE 直接方法里默默用了无数次。
  • Goldstine 是"自反性" 内部刻画的核心工具，在第 16 章对偶论中反复出现。
