附录 B Lebesgue + \(L^p\) 速查

本书第 4 章引 \(L^p[a, b]\) 当 Banach 空间例子，第 5 章用 \(L^2\) 当 Hilbert 标准例，第 15 章 Riesz 表示在 \(L^p\)、第 16 章对偶论 \((L^p)^* = L^q\)、第 23 章 Sobolev 嵌入——都默默假定读者熟悉 Lebesgue 测度论 + \(L^p\) 空间。这个附录把这些被假定的事实集中处理，作为不熟悉读者的"速查 + 备忘"。

  • 测度空间、可测函数、Lebesgue 积分定义。
  • Lebesgue vs Riemann（哪些"区别" 关键）。
  • 三大收敛定理：MCT、Fatou、DCT。
  • \(L^p\) 空间：Riesz-Fischer 完备性、Hölder、Minkowski、对偶。
  • 稠密性（光滑紧支撑函数稠于 \(L^p\)）。
  • Fubini-Tonelli。
  • Radon-Nikodym 定理。

B.1 测度空间与 Lebesgue 积分

定义。设 \(X\) 集合。\(\sigma\)-代数 \(\mathcal{M} \subset 2^X\) 是闭于可数并、可数交、补 的子集族，且包含 \(X\) 与 \(\emptyset\)。

\(\sigma\)-代数 + 集合 = 可测空间 \((X, \mathcal{M})\)。\(\mathcal{M}\) 中元素叫可测集。

定义（测度）。测度 \(\mu : \mathcal{M} \to [0, +\infty]\) 满足：

  • \(\mu(\emptyset) = 0\)。
  • \(\sigma\)-加性：对不交可数族 \(\{A_n\}\)，\(\mu(\bigsqcup A_n) = \sum \mu(A_n)\)。

三元组 \((X, \mathcal{M}, \mu)\) 是测度空间。最重要的例子：

  • \((\mathbb{R}, \mathcal{L}, m)\)——Lebesgue 测度，\(m([a, b]) = b - a\)。
  • \((\mathbb{R}^n, \mathcal{L}^n, m^n)\)——\(n\) 维 Lebesgue。
  • \((X, 2^X, \#)\)——计数测度（每个集 = 元素个数）。
  • \((X, \mathcal{B}, \delta_x)\)——Dirac 测度（\(\delta_x(A) = 1\) if \(x \in A\), 0 否）。
  • 概率空间 \((\Omega, \mathcal{F}, P)\)，\(P(\Omega) = 1\)。

定义（可测函数）。\(f : X \to [-\infty, \infty]\) 可测（关于 \(\mathcal{M}\)），如果 \(f^{-1}((a, \infty]) \in \mathcal{M}\) 对所有 \(a \in \mathbb{R}\)。

等价：\(f^{-1}\) 把 \(\mathbb{R}\) 上每个 Borel 集映到 \(\mathcal{M}\) 中。可测函数族闭 于加 / 减 / 乘 / 除 / 上下确界 / 极限——是个非常稳定的族。

积分构造（三步）。

  1. 简单函数 \(\varphi = \sum_{k=1}^N c_k \mathbf{1}_{A_k}\)（\(A_k\) 不交可测、\(c_k \ge 0\)）：\(\int \varphi d\mu := \sum c_k \mu(A_k)\)。
  2. 非负可测 \(f\)：\(\int f d\mu := \sup_{\varphi \le f, \varphi \text{ 简单}} \int \varphi d\mu\)。
  3. 一般可测 \(f\)：\(f = f^+ - f^-\)（正负部分）。如果 \(\int f^+ < \infty\) 且 \(\int f^- < \infty\)，定义 \(\int f := \int f^+ - \int f^-\)。这种 \(f\) 叫可积。

\(L^1(X, \mu) := \{f \text{ 可测} : \int |f| d\mu < \infty\}\)。配范数 \(\|f\|_1 = \int |f| d\mu\)。

B.2 Lebesgue vs Riemann

Riemann 积分（高中那个）通过"水平切割定义域" 定义——把 \([a, b]\) 分成小段、每段上取函数的上下确界、取上下和、加细分割时上下和收敛到同一值就Riemann 可积。

Lebesgue 通过"水平切割值域" 定义——按函数取值大小 把定义域分类，再加权（按测度）。

这条区别带来三个关键改进：

(1) 更多函数可积。\([0, 1]\) 上 Dirichlet 函数 \(\chi_{\mathbb{Q}}\)（有理点 1，无理点 0）不Riemann 可积（任何分割上下和差 = 1）；但Lebesgue 可积，\(\int \chi_{\mathbb{Q}} dm = m(\mathbb{Q} \cap [0, 1]) = 0\)（有理数集是零测）。一般"有限或可数无穷多间断" 的有界函数 Lebesgue 可积。

(2) 收敛定理强得多。"\(f_n \to f\) 逐点 + \(\int f_n \to \int f\)" 这种交换在 Riemann 框架要求一致收敛（很强）；Lebesgue 框架只要求逐点收敛 + 控制函数（DCT，下节）——弱得多。

(3) 完备空间。\(L^p\) 空间在 Lebesgue 积分下完备（Riesz-Fischer），但 Riemann 积分意义下"\(L^p\)" 不完备——Cauchy 列可能极限是不 Riemann 可积的函数（如 Dirichlet）。这条让 Lebesgue 积分成为泛函分析的唯一选择——你不能在 Riemann 框架下做无穷维分析。

同时：每个 Riemann 可积函数 Lebesgue 可积，且两个积分相等（在两者都定义的范围内）。所以 Lebesgue 是 Riemann 的真扩张，没有冲突。

B.3 三大收敛定理

这三条是 Lebesgue 积分论的工作马，本书里用了几十次。

单调收敛定理（MCT, Beppo Levi）。设 \(\{f_n\}\) 是 \(X\) 上的非负可测函数列，\(f_n \nearrow f\) 逐点（递增收敛）。则

\[ \int f_n d\mu \to \int f d\mu \quad (n \to \infty). \]

"递增 \(\implies\) 极限 + 积分可换序"。注意不需要 \(\int f < \infty\)——可以是 \(+\infty\)（两边都 \(\infty\)）。

Fatou 引理。设 \(\{f_n\}\) 非负可测。则

\[ \int (\liminf_n f_n) d\mu \le \liminf_n \int f_n d\mu. \]

"\(\liminf\) 和 \(\int\) 不能换序，但 \(\le\) 仍然成立"。这条是反向不等式版的 MCT——非常宽松，几乎总能用。

具体例：\(f_n(x) = n \mathbf{1}_{[0, 1/n]}(x)\) 在 \([0, 1]\) 上。\(\int f_n = n \cdot 1/n = 1\)。\(f_n \to 0\) 几乎处处。Fatou：\(\int 0 = 0 \le \liminf \int f_n = 1\)。严格不等——质量"逃到无穷高"，积分没掉但函数几乎处处为 0。

控制收敛定理（DCT, Lebesgue）。设 \(\{f_n\}\) 可测，\(f_n \to f\) 几乎处处，且存在可积 控制函数 \(g \in L^1\) 使 \(|f_n| \le g\) 对所有 \(n\) 几乎处处。则

\[ f \in L^1, \quad \int f_n d\mu \to \int f d\mu, \quad \int |f_n - f| d\mu \to 0. \]

"有控制 + 逐点收敛 \(\implies\) 积分可换序"。这是最常用 的收敛定理。

例：\(\int_0^1 x^n dx = 1/(n+1) \to 0\)。验证：\(x^n \to 0\) 在 \([0, 1)\)，\(|x^n| \le 1\) 可积控制 \(\implies\) DCT 给极限 0。

这三条定理之间关系：MCT \(\subset\) Fatou \(\subset\) DCT 是常见误解——实际上每条都独立，它们涵盖不同情形。MCT 要求递增（强结构），DCT 要求控制函数（不要求单调）。Fatou 是最弱的（什么都不要求）但只给不等式。

B.4 \(L^p\) 空间

定义。对 \(1 \le p < \infty\)：

\[ L^p(X, \mu) := \{f \text{ 可测} : \int |f|^p d\mu < \infty\} / \sim, \]

其中 \(f \sim g \iff f = g\) 几乎处处。配范数

\[ \|f\|_p := \big(\int |f|^p d\mu\big)^{1/p}. \]

对 \(p = \infty\)：

\[ L^\infty(X, \mu) := \{f \text{ 可测} : \mathrm{ess\,sup} |f| < \infty\} / \sim, \quad \|f\|_\infty := \mathrm{ess\,sup} |f|. \]

"本质上确界" \(\mathrm{ess\,sup} f = \inf\{M : f \le M \text{ a.e.}\}\)——忽略零测集上的"峰值"。

Riesz-Fischer 定理。对 \(1 \le p \le \infty\)，\(L^p(X, \mu)\) 是 Banach 空间——完备。

\(p = 2\) 时配内积 \(\langle f, g \rangle = \int f \overline g d\mu\)，\(L^2\) 是 Hilbert 空间。

这条完备性极其重要——它让"Cauchy 序列有极限" 成立，所有泛函分析的基础定理（开映射、闭图像、一致有界）都在 \(L^p\) 上适用。证明用 Cauchy 序列 + 三大收敛定理（DCT 关键）。

Hölder 不等式。设 \(1 \le p, q \le \infty\), \(1/p + 1/q = 1\)（"共轭指数"）。则

\[ \boxed{\ \big| \int f g d\mu \big| \le \|f\|_p \cdot \|g\|_q.\ } \]

具体的 \(p = q = 2\)：Cauchy-Schwarz \(|\int f g| \le \|f\|_2 \|g\|_2\)。\(p = 1, q = \infty\)：\(|\int f g| \le \|g\|_\infty \int |f|\)。

证明思路：用 Young 不等式 \(a b \le a^p / p + b^q / q\)（\(a, b \ge 0\)）逐点积分。等号成立 \(\iff |f|^p \propto |g|^q\)。

Hölder 是所有 泛函分析估计的基础——比 Cauchy-Schwarz 更一般。

Minkowski 不等式（\(L^p\) 三角不等式）。对 \(1 \le p \le \infty\)：

\[ \|f + g\|_p \le \|f\|_p + \|g\|_p. \]

这就是 \(\|\cdot\|_p\) 满足三角不等式 \(\implies\) 范数公理满足。证明用 Hölder 配 \((p, q)\)：写 \(|f+g|^p = |f+g| \cdot |f+g|^{p-1}\)，三角 + Hölder。

注：对 \(0 < p < 1\)，\(L^p\) 仍是完备度量 空间（用 \(d(f, g) = \int |f - g|^p\)），但不赋范——三角不等式失败（反向：\(\|f + g\|_p^p \le \|f\|_p^p + \|g\|_p^p\)）。本书几乎只用 \(p \ge 1\)。

对偶定理。对 \(1 \le p < \infty\)，\(1/p + 1/q = 1\)：

\[ \boxed{\ (L^p(X, \mu))^* = L^q(X, \mu).\ } \]

具体的：每个有界线性泛函 \(\Lambda \in (L^p)^*\) 唯一表示为 \(\Lambda(f) = \int f g d\mu\) 某 \(g \in L^q\)，且 \(\|\Lambda\| = \|g\|_q\)。

这是 Riesz 表示定理的 \(L^p\) 版本（详第 15 章）。

例外：\(p = \infty\) 时，\((L^\infty)^* \supsetneq L^1\)——双对偶不闭合。具体的 \((L^\infty)^*\) 是个非常大的空间（含奇怪的"有限可加"测度），实际工作中很少使用。所以"对偶" 在 \(L^p\) 框架下只对 \(1 \le p < \infty\) 干净。

另一边：\((L^1)^* = L^\infty\)（\(p = 1, q = \infty\) 是上面对偶定理特例）。

自反性：\(L^p\) 自反 \(\iff 1 < p < \infty\)。\(L^1\) 与 \(L^\infty\) 不自反——这是它们行为"怪" 的原因（缺弱紧、缺好的对偶等）。详第 16 章。

B.5 稠密性定理

定理（紧支撑光滑函数稠于 \(L^p\)）。对 \(1 \le p < \infty\) 和 \(\Omega \subset \mathbb{R}^d\) 开：

\[ C_c^\infty(\Omega) \text{ 在 } L^p(\Omega) \text{ 中稠密}. \]

即：每个 \(f \in L^p(\Omega)\) 是 \(C_c^\infty\) 函数序列的 \(L^p\) 极限。

证明思路：

  1. 简单函数稠 \(L^p\)（直接定义）。
  2. 简单函数 = 可测集的指示函数线性组合 \(\implies\) 用截断 + 内部正则性 \(\implies\) 紧支撑函数稠。
  3. 紧支撑可测函数 \(\implies\) 用磨光器（mollifier）\(\rho_\varepsilon\) 卷积\(f * \rho_\varepsilon \in C^\infty\) + 紧支撑（如果 \(f\) 紧支撑）+ \(L^p\) 收敛到 \(f\)。

这条稠密性让"用光滑函数验证 \(L^p\) 上的等式" 成为标准技术——只要在光滑紧支撑函数上成立、再两边稠密延拓。这条策略在第 23 章弱解理论里反复用。

例外：\(p = \infty\)。\(C_c^\infty\) 不稠于 \(L^\infty\)——\(L^\infty\) 函数可以有"本质间断"，无法被光滑函数 \(L^\infty\) 逼近。这是 \(L^\infty\) 怪行为之一。

B.6 Fubini-Tonelli 定理

积分换序。设 \((X, \mathcal{M}, \mu)\), \((Y, \mathcal{N}, \nu)\) 是 \(\sigma\)-有限测度空间，\(\mu \otimes \nu\) 是积测度（在乘积 \(\sigma\)-代数 \(\mathcal{M} \otimes \mathcal{N}\) 上）。

Tonelli 定理。设 \(f : X \times Y \to [0, \infty]\) 可测。则

\[ \int_{X \times Y} f \, d(\mu \otimes \nu) = \int_X \big(\int_Y f(x, y) d\nu(y)\big) d\mu(x) = \int_Y \big(\int_X f(x, y) d\mu(x)\big) d\nu(y). \]

三个积分都相等（可以是 \(+\infty\)）。"非负可测 \(\implies\) 换序自由"——不需要可积。

Fubini 定理。设 \(f : X \times Y \to \mathbb{R}\) 可测，且 \(\int |f| d(\mu \otimes \nu) < \infty\)（即 \(f \in L^1(X \times Y)\)）。则上面三个积分都有限且相等，且

  • 对几乎所有 \(x \in X\)，\(y \mapsto f(x, y) \in L^1(Y)\)。
  • 函数 \(x \mapsto \int_Y f(x, y) d\nu(y) \in L^1(X)\)。

"可积 \(\implies\) 换序自由"。

实战流程：

  1. 先用 Tonelli 验证 \(\int |f|\) 有限（用任意一个换序计算）。
  2. 如果有限，再用 Fubini 换序计算 \(\int f\)。

这条"先 Tonelli 后 Fubini" 流程是积分换序的标准做法。

典型反例（不可积换序失败）：

\[ f(x, y) = \frac{x^2 - y^2}{(x^2 + y^2)^2}, \quad (x, y) \in (0, 1)^2. \]

\(\int_0^1 \int_0^1 f \, dx \, dy = \pi/4\)，\(\int_0^1 \int_0^1 f \, dy \, dx = -\pi/4\)。两个换序的积分不等！这是因为 \(\int |f| = \infty\)，Fubini 不适用。

B.7 Radon-Nikodym 定理

"测度对测度的导数"。

定义。测度 \(\nu\) 绝对连续于 \(\mu\)（记 \(\nu \ll \mu\)），如果 \(\mu(A) = 0 \implies \nu(A) = 0\)。即：\(\mu\)-零测集也 \(\nu\)-零测。

Radon-Nikodym 定理。设 \(\mu, \nu\) 是 \(\sigma\)-有限测度，\(\nu \ll \mu\)。则存在唯一（差零测）的非负可测函数 \(f \in L^1_{\mathrm{loc}}(\mu)\) 使

\[ \nu(A) = \int_A f d\mu \quad \forall A \in \mathcal{M}. \]

\(f\) 称为 \(\nu\) 关于 \(\mu\) 的 Radon-Nikodym 导数，记 \(f = d\nu / d\mu\)。

这条定理把"一个测度" 转化为"另一个测度上的密度函数"——是测度论与积分论之间的桥梁。具体应用：

  • 概率论：条件期望。\(E[X | \mathcal{G}]\) 用 Radon-Nikodym 导数严格定义——把"条件分布" 当成 \(\mu(\cdot \cap G)\) 关于 \(\mu(G)\) 的 RN 导数。
  • 测度变换（统计学的"importance sampling"）：从 \(P\) 下样本算 \(Q\) 下期望，权重 = \(d Q / d P\)。
  • 对偶 \((L^1)^* = L^\infty\) 的证明就是 RN 应用——把"每个 \(L^1\) 上的有界泛函" 表为"测度"，再 RN 表为 \(L^\infty\) 函数。
  • Lebesgue 微分定理（\(\frac{d}{dx} \int_a^x f = f\)）的抽象推广。

证明用 Hilbert 空间投影或单调类（约 4-5 页技术）。请见 Folland "Real Analysis" 第 3 章。

"绝对连续" 假设关键——如果 \(\nu\) 含集中"singular" 部分（如 Dirac \(\delta\) 关于 Lebesgue），定理不直接适用。Lebesgue 分解定理把任意测度 \(\nu\) 分解为 \(\nu = \nu_{\mathrm{ac}} + \nu_{\mathrm{sing}}\)（绝对连续 + 奇异），\(\nu_{\mathrm{ac}}\) 才有 RN 密度。

B.8 实战准则

  • 本书几乎只用 \(L^p\)（\(1 \le p \le \infty\)）。\(p = 1, 2, \infty\) 是三个主要主角，\(p = 2\) 是唯一 Hilbert，\(p = 1, \infty\) 不自反所以行为微妙。
  • "几乎处处" 这条修饰符要时刻警觉——\(L^p\) 元素是等价类，\(f = g\) a.e. 即同一个 \(L^p\) 元素。逐点性质（如"\(f\) 在某点取值多少"）一般没有意义。
  • 三大收敛定理：DCT 最常用、Fatou 最宽松、MCT 最强结构。能用 DCT 时用 DCT；不行试 MCT；都不行用 Fatou 给 \(\le\) 估计。
  • Hölder 不等式 + Minkowski 是 \(L^p\) 估计的基础。所有 PDE / 概率 / 调和分析的具体估计基本都从这两条出发。
  • Fubini 用前 必须验证 \(\int |f| < \infty\)（用 Tonelli）。这条疏忽是初学者最常见错误。
  • 稠密性 \(C_c^\infty \subset L^p\)（\(p < \infty\)）让"光滑测试函数" 策略合法——这条在 PDE 弱解理论 + 分布论里是基础。
  • RN 定理给"测度对测度的导数" 严格定义——是概率论里"条件期望" + "importance sampling" 的数学基础。
