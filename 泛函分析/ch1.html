第 1 章 引论：从有限维到无穷维

如果你之前已经读过《线性代数读本》，那么"向量空间 + 内积 + 线性变换 + 谱定理"对你来说应该已经是一套相当熟的工具了。在 \(\mathbb{R}^n\) 或 \(\mathbb{C}^n\) 这种有限维世界里，这套工具几乎是无敌的：任何线性变换都能写成矩阵，任何范数之间都等价，任何子空间都是闭的，任何对称矩阵都能正交对角化，任何有界闭集都是紧的。你做几道题之后会忍不住觉得"线性代数已经够了"。这本书想做的第一件事，就是温柔地告诉你——只要把"维数"这两个字从有限改成无穷，上面这一长串"任何"几乎全部要重新审一遍。

这并不是说前面的工具坏了。它们本身没有错，错的是我们以为它们会"自动"在无穷维成立。事实上，泛函分析里最值钱的几个定理——三大原理、Riesz 表示、Banach–Alaoglu、谱定理——背后的故事都可以一句话说穿：它们在告诉你，哪一些有限维里白送的事，到了无穷维里需要一个非平凡的论证才能保住。这就是为什么这本书的第 11、12、13、16、18、21 章会被反复称为"原理"或"定理"——因为有限维里它们根本不算定理，到了无穷维里，它们才忽然变得需要证明。

所以这一章不急着堆定义。我们先做几件具体的事，把维数变化带来的"撕裂感"摊开来给你看。等你亲眼看见有限维直觉是怎么一条一条失败的，后面整本书的方向就清楚了：泛函分析就是一门"重新建造无穷维直觉"的课。

  • 先用一个简单到几乎不像数学题的例子（坐标向量列）说明：在 \(\ell^2\) 里"有界闭集是紧的"已经不再成立。
  • 然后看一个非常自然的线性变换（求导算子）为什么不连续，从而引出"在哪个空间上看"才是泛函分析的真正问题。
  • 接着看一个自然的子空间（多项式集合）为什么不闭，让你第一次体会到"闭"在无穷维里要专门考虑。
  • 把"两个范数等价"这件事拆开：为什么有限维一定等价、无穷维一定不等价；为什么这件事重要。
  • 说清"算子 = 矩阵"这种翻译为什么在无穷维里失效，以及代之而起的新对象（算子范数、紧算子、谱）。
  • 最后把整本书的主线亮出来：完备性、几何、算子、对偶、谱。

1.1 第一处崩塌：单位球不再紧

从 \(\mathbb{R}^n\) 里你早就习惯了一句话：有界闭集就是紧致的。这是 Heine–Borel 定理（详见实分析读本第 6 章），它支撑了几乎所有"任何序列都有收敛子序列"的论证：极值定理（连续函数在闭区间上取到最值）、Bolzano–Weierstrass、紧致性方法证明微分方程解的存在性等等。这条结论对几乎所有的应用来说就像氧气一样自然。

现在我们走到一个非常温和的无穷维空间里看看会发生什么。考虑

\[ \ell^2 = \left\{ x = (x_1, x_2, x_3, \ldots) : x_n \in \mathbb{R}, \ \sum_{n=1}^{\infty} x_n^2 < \infty \right\}, \]

它就是把"实数列的平方和有限"这件事当成一个空间。范数定义为

\[ \|x\|_2 = \left( \sum_{n=1}^{\infty} x_n^2 \right)^{1/2}. \]

这个空间和 \(\mathbb{R}^n\) 长得真的非常像：你完全可以把 \(\mathbb{R}^n\) 看成 \(\ell^2\) 里"只用前 \(n\) 个坐标"的子集。它有内积、有范数、有"标准基" \(e_n = (0, \ldots, 0, 1, 0, \ldots)\)（第 \(n\) 位是 1，其余为 0）。乍看之下，把 \(\mathbb{R}^n\) 的所有几何直觉照搬过来似乎不会有任何问题。

现在做一件最朴素的事：把这些 \(e_n\) 一个个排起来，看看这个序列。

它显然在 \(\ell^2\) 的闭单位球里——每一个 \(e_n\) 的范数都是 1，所以它们都落在 \(\overline{B}_{\ell^2}(0, 1) = \{x : \|x\|_2 \le 1\}\) 里。这个闭球是有界的（半径就是 1），是闭的（取范数函数是连续的，所以 \(\{\|x\|_2 \le 1\}\) 是闭集）。在 \(\mathbb{R}^n\) 里这种集合一定紧。所以照 \(\mathbb{R}^n\) 的逻辑，这个序列 \(e_1, e_2, e_3, \ldots\) 一定要有一个收敛子序列。

但它没有。任意两个不同的 \(e_n\) 与 \(e_m\)（\(n \neq m\)）之间的距离是

\[ \|e_n - e_m\|_2 = \sqrt{1^2 + (-1)^2} = \sqrt{2}. \]

也就是说，这个序列里任意两个元素之间的距离都恰好等于 \(\sqrt{2}\)。这种序列绝不可能有 Cauchy 子序列，自然也不可能有收敛子序列（在度量空间里收敛 \(\Rightarrow\) Cauchy）。

\[ \boxed{\ \ell^2 \text{ 中闭单位球 } \overline{B}(0,1) \text{ 是有界闭集，但不紧致。}\ } \]

这个例子简单到让人不舒服。我们没用任何花招，没构造什么病态对象，只是把"标准基"列出来——结果就把 Heine–Borel 撞碎了。这是泛函分析里第一个、也是最有教育意义的反例："有界 + 闭 \(\Rightarrow\) 紧"在无穷维里不再自动成立。

这件事到底意味着什么？意味着所有依赖 Heine–Borel 的论证都得重新审一遍。比如在实分析读本第 6 章里我们用过这种论证：从一个有界数列里抽收敛子序列，然后取极限。这种步骤在无穷维空间里不能盲用。这就是为什么后面我们要专门发明弱收敛（第 17 章）和Banach–Alaoglu 定理（第 18 章）——它们是为了在无穷维里找回某种"紧致感"而存在的工具。

顺便注意一件事：上面我们其实证明的是更一般的结论。在任何无穷维赋范空间里，闭单位球都不紧。后面（第 18 章前后）会用 Riesz 引理把这件事正式说出来。这意味着"单位球紧"几乎可以当作"空间是有限维的"的等价刻画——这条等价是泛函分析里最早出现也最深的几何分水岭之一。

1.2 第二处崩塌：线性映射不再自动连续

在《线性代数读本》里我们写"线性变换"的时候，连续性这件事完全不需要提。原因是：在有限维赋范空间上，每个线性映射都自动连续。这是个看起来很无聊的事实，但它支撑了你以前所有的几何直觉——"线性变换把收敛序列送到收敛序列""线性变换在原点附近大致是局部线性的"——这些都建立在自动连续性之上。

换到无穷维你就要小心。一个最自然的反例是求导算子。

取空间 \(C^1[0,1] = \{f : [0,1] \to \mathbb{R}, f \text{ 连续可微}\}\)，并赋以最自然的范数 \(\|f\|_\infty = \sup_{x \in [0,1]} |f(x)|\)。考虑映射

\[ D : C^1[0,1] \to C[0,1], \qquad D(f) = f'. \]

这是个完全合法的线性映射：\(D(\alpha f + \beta g) = \alpha f' + \beta g' = \alpha D(f) + \beta D(g)\)。问题是它连续吗？连续在赋范空间里就是第 8 章会反复用到的判别条件：是否存在常数 \(C\) 使

\[ \|D(f)\|_\infty \le C \|f\|_\infty \quad \text{对所有 } f \in C^1[0,1] ? \]

看起来很可能。可惜不是。考虑函数列

\[ f_n(x) = \frac{\sin(n^2 x)}{n}. \]

这些函数光滑、可微，且

\[ \|f_n\|_\infty = \frac{1}{n} \to 0. \]

也就是说在 \(C[0,1]\) 的 sup-范数下，\(f_n\) 趋向零函数。但

\[ f_n'(x) = n \cos(n^2 x), \qquad \|f_n'\|_\infty = n \to \infty. \]

所以输入"越来越小"，输出"越来越大"——这个比例可以无界，求导算子完全不可能"被一个固定的 \(C\) 控制住"。

\[ \boxed{\ D : (C^1[0,1], \|\cdot\|_\infty) \to (C[0,1], \|\cdot\|_\infty) \text{ 是线性的，但不连续（不有界）。}\ } \]

这个例子要细品。求导是世界上最自然的线性算子之一。它在 \(\mathbb{R}^n\) 上的所有有限维近似（差分算子）都是连续的——一旦把维数固定为 \(n\)，差分矩阵就只是一个 \(n \times n\) 矩阵，自动连续。但是无穷维里这个"自然"的算子不连续。

这件事的教训非常深。在无穷维里，"我用什么范数"和"我研究的算子是不是连续"不能分开问。同一个表达式 \(D(f) = f'\)，在 \((C^1[0,1], \|\cdot\|_\infty)\) 上不连续，但是如果你换一个范数——比如

\[ \|f\|_{C^1} = \|f\|_\infty + \|f'\|_\infty, \]

把"函数本身的大小"和"导数的大小"打包在一起，那么 \(D\) 立刻变成连续的（事实上 \(\|D(f)\|_\infty \le \|f\|_{C^1}\) 是定义内嵌的）。同一个算子，换一个范数就换一个命运。

这就是为什么 Sobolev 空间（第 23 章）会被发明出来。\(H^1\)、\(H^2\)、\(W^{1,p}\) 这些空间的存在意义就是：把"\(f\) 加上若干阶导数"全部装进同一个范数，于是相应阶的微分算子就变成了天然连续的算子，PDE 弱解理论才有地方落脚。

你也许已经感受到了：在泛函分析里，"在哪个空间上看"取代了"算子是什么形式"，成为更核心的问题。

1.3 第三处崩塌：自然子空间不再自动闭

有限维里你不会问"子空间是不是闭的"——它一定是。证明也很简单：有限维子空间是它的某组基张成的，而基有限，等价于一个有限维赋范空间到 \(\mathbb{R}^n\) 的等距嵌入，所以是闭的。

无穷维里这件事要专门确认。最经典的反例是：

取 \(C[0,1]\) 配上 sup-范数。它的子空间

\[ \mathcal{P}[0,1] = \{ p : [0,1] \to \mathbb{R}, p \text{ 是多项式}\} \]

是一个真子空间（包含所有多项式但不包含一般的连续函数）。这个子空间在 \(C[0,1]\) 中不闭。事实上由 Weierstrass 逼近定理（你大概已经在实分析读本第 15 章里见过它），任意连续函数都可以被多项式列一致逼近。这等于说

\[ \overline{\mathcal{P}[0,1]} = C[0,1]. \]

所以多项式集合的闭包就是整个 \(C[0,1]\)，而 \(\mathcal{P}[0,1]\) 显然不等于 \(C[0,1]\)（比如 \(|x - 1/2|\) 不是多项式）。这说明 \(\mathcal{P}[0,1]\) 是个非闭的真子空间。

\[ \boxed{\ \mathcal{P}[0,1] \subsetneq C[0,1] \text{ 是线性子空间，但其闭包是整个 } C[0,1] \text{——子空间不闭。}\ } \]

这是个非常有教育意义的崩塌。多项式空间是你能想到的最"代数自然"的子空间——加法封闭、数乘封闭、函数运算友好——可它就是不闭。在无穷维里，代数自然 ≠ 拓扑封闭。从此以后，每当我们要谈"投影""正交补""商空间""对偶嵌入"，都必须先问一句：这个子空间是不是闭？

这件事在第 6 章会成为正交分解定理的一个关键前提（"闭子空间才有正交补"），在第 22 章会成为 Fredholm 选择性的关键前提（"算子的像是闭的，才有交替性"）。如果你以前一直没把"闭子空间"当回事，从现在开始要养成专门问一句的习惯。

1.4 第四处崩塌：范数之间不再等价

有限维里有一个非常"任性"的定理：在一个有限维向量空间上，任意两个范数都等价——意思是存在常数 \(c, C > 0\)，使

\[ c \|x\|_a \le \|x\|_b \le C \|x\|_a, \qquad \forall x. \]

这个等价关系的实际含义是：所有有限维范数定义的拓扑相同。"收敛"、"开集"、"连续函数"这些概念不依赖于你选什么范数。这就是为什么你在线性代数里从来没认真问过"我用的是 \(\ell^1\) 范数还是 \(\ell^2\) 范数还是 \(\ell^\infty\) 范数"——在 \(\mathbb{R}^n\) 上选哪个都不影响"收敛"是什么意思。

无穷维里这件事彻底完蛋。在 \(C[0,1]\) 上比较两种范数：

\[ \|f\|_\infty = \sup_{x \in [0,1]} |f(x)|, \qquad \|f\|_1 = \int_0^1 |f(x)| \, dx. \]

显然 \(\|f\|_1 \le \|f\|_\infty\)（因为长度 1 的区间上积分被上确界控制）。但反过来的不等式不成立。考虑函数列 \(f_n(x) = x^n\)。这些函数在 \([0,1]\) 上满足

\[ \|f_n\|_\infty = 1, \qquad \|f_n\|_1 = \int_0^1 x^n \, dx = \frac{1}{n+1} \to 0. \]

所以一个"在 \(\|\cdot\|_1\) 意义下趋于零"的序列，在 \(\|\cdot\|_\infty\) 意义下根本不收敛。两个范数定义的拓扑不一样——它们各自有不同的"收敛"概念。

这就是为什么我们在写 \(L^p(\Omega)\) 的时候必须把指标 \(p\) 写出来。\(L^1\)、\(L^2\)、\(L^\infty\) 是三个不同的空间，它们的"收敛"互不相同，它们的对偶空间也互不相同（第 15 章），它们能装下的算子也互不相同。"用哪个 \(p\)" 是分析里最常见但也最重要的选择。

你以后看 PDE 的论文，发现作者一会儿用 \(L^2\) 一会儿用 \(L^\infty\) 一会儿用 \(W^{1,p}\)，不要把这看成"用什么都行的口味问题"——他每写一次就是在选一个不同的拓扑、不同的几何、不同的对偶。这个选择往往就是论文最难也最关键的一步。

1.5 "算子 = 矩阵"为什么不够用了

在线性代数里，线性变换和矩阵几乎是同一件事。你给我一个线性映射 \(T : \mathbb{R}^n \to \mathbb{R}^m\)，我给你一个 \(m \times n\) 矩阵；反之亦然。这个"线性映射 \(\leftrightarrow\) 矩阵"的字典之所以好用，是因为有限维空间有标准基，线性映射的全部信息都能装进一张表里。

到了无穷维，事情没那么简单。你也可以试着把算子写成"无穷大的矩阵"，比如算子 \(T : \ell^2 \to \ell^2\) 在标准基 \(\{e_n\}\) 下的"矩阵元" \(T_{ij} = \langle T e_j, e_i \rangle\)。但这个无穷矩阵自己并不能告诉你 \(T\) 是不是连续、是不是紧、谱长什么样——你必须把它放回某个范数下、某种收敛下，配合分析的语言才能说话。

这就是泛函分析比线性代数多出来的一层：算子不仅是代数对象，也是分析对象。我们要谈算子的范数（第 8、9 章）、算子的谱（第 21 章）、算子的紧性（第 20 章）、算子序列的收敛（强收敛、弱收敛、算子范数收敛——这三种到第 17 章会全部出现）。每个这样的概念在有限维里要么自动成立、要么根本不存在。

举一个特别说明问题的例子。考虑 \(\ell^2\) 上的右移算子

\[ S(x_1, x_2, x_3, \ldots) = (0, x_1, x_2, x_3, \ldots). \]

它显然是线性的、连续的（保范数：\(\|Sx\|_2 = \|x\|_2\)）。在有限维里它甚至会有"逆"—— \(\mathbb{R}^n\) 上的右移配上"丢掉最后一位"就是它的左逆。但在无穷维 \(\ell^2\) 上，\(S\) 是单射但不是满射。它的像是 \(\{(0, y_2, y_3, \ldots)\}\)，不包含 \((1, 0, 0, \ldots)\)。

\[ \boxed{\ \ell^2 \text{ 上的右移算子 } S \text{：单射，不满射，} \|S\| = 1，但不可逆。\ } \]

在有限维里，"单射"和"满射"对方阵而言互相等价（这是线性代数读本第 7 章的核心结论之一）。在无穷维里这种等价彻底失效。这件事不是反常，而是一种新结构——它的"剩余"会在第 21 章里以谱的形式重新出现，作为右移算子的残留谱（residual spectrum）。

1.6 这本书的主线：完备、几何、算子、对偶、谱

把上面五个崩塌串起来，你已经能看出泛函分析的整张地图。我们要建造的，是一套能在无穷维里继续工作的"线性代数 + 分析"框架。它由五条主线组成。

第一条：完备性。无穷维里"取极限"是日常操作，所以你必须有一个不会让 Cauchy 序列"漏出去"的空间。这就是 Banach 空间（赋范 + 完备）和 Hilbert 空间（内积 + 完备）的来历。第 2、3 章建立这两个概念。

第二条：几何。有了内积才有"角度"，有了"角度"才能谈正交、投影、最佳逼近。第 6 章会把"投影到闭子空间"的几何讲到能让你把最小二乘、回归、条件期望、Fourier 部分和都统一成同一件事；第 7 章把它推广到"正交基 + Fourier 展开"。

第三条：算子。无穷维线性映射要先解决"连不连续"的问题（第 8 章），然后才能谈算子范数（第 9 章）。算子代数 \(\mathcal{B}(X)\) 不是空中楼阁——你做 PDE、做积分方程、做量子力学，写下的每一个"线性算子"都是 \(\mathcal{B}(X)\) 里的元素。

第四条：对偶。这是泛函分析最迷人也最有用的一面。每个赋范空间 \(X\) 都自带一个"线性泛函的空间" \(X^*\)（第 15 章），它不只是技术工具，而是另一种几何视角——很多在 \(X\) 里很难处理的问题，到了 \(X^*\) 里就立刻变成一个"看角度"的问题。Hahn–Banach 定理（第 13、14 章）保证了这个 \(X^*\) 一定足够大；Riesz 表示定理（第 16 章）告诉你某些重要空间的 \(X^*\) 长什么样；Banach–Alaoglu 定理（第 18 章）告诉你 \(X^*\) 的单位球居然在弱*拓扑下是紧的——把第 1.1 节里崩塌掉的"紧致性"用一种新方式找了回来。

第五条：谱。谱定理是有限维线性代数里"对称矩阵能被正交对角化"的无穷维版本（第 21 章）。它需要紧算子（第 20 章）来让"特征值离散"重新成立，需要自伴算子（第 21、24 章）来让特征向量正交。这是为什么量子力学的语言天生就是泛函分析的语言——可观测量是 \(L^2\) 上的自伴算子，它的谱就是物理量的可能取值。

把这五条主线交织在一起，你就得到一门看待无穷维的方式。这门方式之所以值得花一整本书来学，是因为它一旦学会，就会像线性代数那样，反过来组织你以前做过的几乎所有分析——Fourier 级数变成"展开成正交基"，最小二乘变成"投影到闭子空间"，PDE 弱解变成"线性泛函在 Sobolev 空间上零化某个子集"，量子力学变成"自伴算子的谱测度"。所有这些，都将在后面 23 章里一项一项落地。

下一章我们就从第一步开始：在赋范空间上把"完备性"建立起来。这一步会顺便把"度量空间"和"赋范空间"的关系说清楚，把"Banach 空间"这个名字配上正确的画面，并把后面会反复用到的几个例子（\(\ell^p\)、\(L^p\)、\(C[a,b]\)）从一开始就摆上桌。
