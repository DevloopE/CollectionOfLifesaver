第 1 章 引论：从有限维到无穷维

请你做一道朴素的事。在 \(\ell^2\) 里取这样一列向量：

\[ e_1 = (1, 0, 0, 0, \ldots), \quad e_2 = (0, 1, 0, 0, \ldots), \quad e_3 = (0, 0, 1, 0, \ldots), \quad \ldots \]

每一个 \(e_n\) 都是"第 \(n\) 位 1，其余 0"。每个的范数 \(\|e_n\|_2 = 1\)。所以这整列都坐落在 \(\ell^2\) 的闭单位球 \(\overline{B}(0, 1) = \{x : \|x\| \le 1\}\) 里——一个有界、闭的集合。

在 \(\mathbb{R}^n\) 里，"有界 + 闭 ⟹ 紧致" 是 Heine-Borel 定理（详见实分析读本第 6 章）。紧致意味着每个序列都有收敛子序列。所以这列 \(e_1, e_2, e_3, \ldots\) 必须 有收敛子序列。

请你算一下任意两个 \(e_n, e_m\)（\(n \neq m\)）之间的距离：

\[ \|e_n - e_m\|_2 = \sqrt{1^2 + (-1)^2} = \sqrt{2}. \]

任意两个不同元素相距 \(\sqrt{2}\)。这不是个会变小的距离——是个固定的常数。所以这列没有任何子序列是 Cauchy 的，更别说收敛。

到了这里你应该停下来想几秒。Heine-Borel 在 \(\mathbb{R}^n\) 是定理。在 \(\ell^2\) 它是错的。\(\ell^2\) 不是什么病态空间——它是无穷维欧氏空间最自然的对应物，有内积、有范数、有标准基。整套有限维直觉都应该搬过来。可是 Heine-Borel 没搬过来。

这一章我让你看四个这样的"有限维直觉到了无穷维就垮" 的例子。每一个看似不显然的例子背后，都站着一条贯穿全书的对应工具——这本书后面 23 章基本就是把这些"垮掉" 一一修 回来。

1.1 第一处崩塌：单位球不再紧

上面那个 \(\{e_n\}\) 例子已经把这件事钉死了。\(\ell^2\) 闭单位球有界、闭，但不紧。

\[\boxed{\ \ell^2 \text{ 闭单位球 } \overline{B}(0,1) \text{ 有界 + 闭，但不紧。}\ }\]

这个例子轻得让人不安——我们没构造任何病态对象，没用什么花招。只是把"标准基" 列出来，Heine-Borel 就崩了。

事实上更普遍：任何无穷维赋范空间的闭单位球都不紧。这条定理（Riesz 引理 + 紧致性）是第 22 章的内容，结论是"单位球紧 ⟺ 空间有限维"——单位球紧致几乎等于"维数有限" 的同义词。

这件事意味着所有依赖 Heine-Borel 的论证都要重新审一遍。从一个有界数列里抽收敛子序列，然后取极限——这种你在实分析里反复用过的标准动作，在无穷维里不能盲用。这是为什么我们需要发明弱收敛（第 17 章）和 Banach–Alaoglu 定理（第 18 章）—— 它们是为了在无穷维里把"紧致感" 用一种更弱的形式找回来。

1.2 第二处崩塌：线性算子不再自动连续

来一个让你更不舒服的反例。求导算子是数学里最自然的东西之一。设

\[ D : C^1[0, 1] \to C[0, 1], \quad D(f) = f'. \]

"\(C^1[0, 1]\)" 是连续可微函数空间，"\(C[0, 1]\)" 是连续函数空间。两边都配上最自然的范数 \(\|f\|_\infty = \sup_{x \in [0,1]} |f(x)|\)。

\(D\) 显然线性：\(D(\alpha f + \beta g) = \alpha f' + \beta g' = \alpha D(f) + \beta D(g)\)。问题：它连续 吗？

在赋范空间里，"线性 + 连续" 等价于"线性 + 有界"——存在常数 \(C\) 使 \(\|D(f)\|_\infty \le C \|f\|_\infty\) 对所有 \(f\)（这条等价性是第 8 章的核心）。所以问题改为：是否存在 \(C\) 使

\[ \|f'\|_\infty \le C \|f\|_\infty \text{ 对所有 } f \in C^1[0, 1]? \]

朴素直觉说"应该有吧"。让我们试一组反例。考虑

\[ f_n(x) = \frac{\sin(n^2 x)}{n}, \quad n = 1, 2, 3, \ldots \]

这些函数光滑、可微。算它们的范数：

\[ \|f_n\|_\infty = \sup_x \left| \frac{\sin(n^2 x)}{n} \right| \le \frac{1}{n} \to 0. \]

所以在 \(C[0, 1]\) 的 sup 范数下，\(f_n\) 趋向零函数。如果 \(D\) 连续，\(D(f_n) = f_n'\) 应该趋向 \(D(0) = 0\)。

可是

\[ f_n'(x) = n \cos(n^2 x), \quad \|f_n'\|_\infty = n \to \infty. \]

\(f_n\) 趋向 0，而 \(f_n' = D(f_n)\) 跑向无穷。\(D\) 不连续。

\[\boxed{\ D : (C^1[0,1], \|\cdot\|_\infty) \to (C[0,1], \|\cdot\|_\infty) \text{ 线性, 但不连续。}\ }\]

让你停下来想清楚这件事的分量。求导算子是最自然 的线性算子之一——你高中就在用。它在所有有限维 子空间上（比如限制到次数 \(\le n\) 的多项式空间）都是连续的——因为有限维上每个线性映射都自动连续（第 8 章会证）。但放到整个 \(C^1[0, 1]\) 上，它就不连续。

这件事告诉你一个深刻的视角转换：在无穷维里，"我用什么范数" 和"我研究的算子是不是连续" 不能分开问。同一个表达式 \(D(f) = f'\)，在 \((C^1, \|\cdot\|_\infty)\) 上不连续，但在 \((C^1, \|f\|_{C^1} := \|f\|_\infty + \|f'\|_\infty)\) 上立刻变成连续的——因为新范数把"导数大小" 也算进了输入度量。

同一个算子，换一个范数就换一个命运。这条认识是为什么 Sobolev 空间（第 23 章）会被发明出来——它们的设计目的就是让微分算子变成有界的，于是 PDE 的弱解理论才有地方落脚。

1.3 第三处崩塌：自然子空间不再自动闭

再来。在 \(C[0, 1]\) 配上 sup 范数。考虑它的子集

\[ \mathcal{P}[0, 1] = \{p : p \text{ 是 } [0, 1] \text{ 上的多项式}\}. \]

它显然是个线性子空间（多项式的和、数乘还是多项式）。问题：它闭 吗？

朴素直觉说"应该闭吧——多项式那么自然"。但Weierstrass 逼近定理（你在实分析读本第 15 章已见过）说：每个 \(C[0, 1]\) 函数都是某个多项式列的一致极限。也就是说

\[ \overline{\mathcal{P}[0, 1]} = C[0, 1]. \]

多项式集合的闭包就是整个 \(C[0, 1]\)。可显然 \(\mathcal{P}[0, 1] \neq C[0, 1]\)（比如 \(|x - 1/2|\) 是连续的但不是多项式）。所以 \(\mathcal{P}\) 是不闭的 真子空间。

\[\boxed{\ \mathcal{P}[0,1] \subsetneq C[0,1] \text{ 是线性子空间, 但闭包是整个 } C[0,1]——\textbf{不闭}.\ }\]

让你看清这一点的颠覆性。多项式空间是你能想象到的最代数自然 的子空间——加法封闭、数乘封闭、连乘法都封闭。可它就是不闭。在无穷维里：代数自然 ≠ 拓扑闭。

从此以后，每当我们要谈"投影""正交补""商空间""对偶嵌入"——所有这些需要拓扑前提 的代数构造——你都必须先问一句"这个子空间闭吗"。这条习惯在第 6 章正交分解定理（"闭 子空间才有正交补"）和第 22 章 Fredholm 选择性（"算子的像是闭 的，才有选择性"）里反复出现。

1.4 第四处崩塌：范数之间不再等价

有限维有一个让人懒得思考的小定理：在有限维 向量空间上，任意两个范数等价——存在常数 \(c, C > 0\) 使 \(c\|x\|_a \le \|x\|_b \le C\|x\|_a\) 对所有 \(x\)。

实际意义：有限维空间上"收敛""开集""连续函数" 这些概念不依赖于你选什么范数。这就是为什么你在线性代数里从来没 认真问过"我用的是 \(\ell^1\) 范数还是 \(\ell^2\) 范数还是 \(\ell^\infty\) 范数"——在 \(\mathbb{R}^n\) 上选哪个都不影响"收敛" 是什么意思。

在无穷维这件事彻底完蛋。

看 \(C[0, 1]\) 上的两种范数：

\[ \|f\|_\infty = \sup_{x \in [0,1]} |f(x)|, \qquad \|f\|_1 = \int_0^1 |f(x)| \, dx. \]

显然 \(\|f\|_1 \le \|f\|_\infty\)（在长度 1 的区间上积分 \(\le\) 上确界）。但反方向不 成立。看序列 \(f_n(x) = x^n\)。算：

\[ \|f_n\|_\infty = 1, \qquad \|f_n\|_1 = \int_0^1 x^n \, dx = \frac{1}{n+1} \to 0. \]

这个序列在 \(\|\cdot\|_1\) 意义下趋零，但在 \(\|\cdot\|_\infty\) 意义下根本不动。两个范数定义的"收敛" 不一样。它们各自定义不同的拓扑、不同的开集、不同的"连续函数"。

\[\boxed{\ \text{在无穷维空间上, 不同范数定义不同拓扑——它们不等价。}\ }\]

这就是为什么我们写 \(L^p(\Omega)\) 时必须把指标 \(p\) 写出来。\(L^1\)、\(L^2\)、\(L^\infty\) 是三个不同的空间——它们的"收敛" 互不相同，对偶空间也互不相同（第 15 章），算子的"有界" 在它们上也意思不同。

你以后看 PDE 论文，发现作者一会儿用 \(L^2\) 一会儿用 \(L^\infty\) 一会儿用 \(W^{1, p}\)——不要 把这看成口味问题。他每写一次就在选一种不同的拓扑、不同的几何、不同的对偶。这个选择往往就是论文最难也最关键的一步。

1.5 第五处崩塌："算子 = 矩阵" 失效

在线性代数里，线性变换 \(T : \mathbb{R}^n \to \mathbb{R}^m\) 和 \(m \times n\) 矩阵几乎是同一件事。给我矩阵我给你变换；给我变换我给你矩阵。这个字典之所以好用，是因为有限维空间有标准基，线性变换的全部信息能装进一张表里。

无穷维你也可以试试。设算子 \(T : \ell^2 \to \ell^2\)，在标准基 \(\{e_n\}\) 下定义"无穷大矩阵元" \(T_{ij} = \langle T e_j, e_i \rangle\)。这给一张无穷大 矩阵。

问题：这张矩阵能告诉你 \(T\) 是不是连续吗？是不是紧的吗？谱长什么样吗？不能。无穷矩阵自己不带任何"分析" 信息——它必须和某个范数 + 某种收敛配起来才说话。

更具体的失效。考虑 \(\ell^2\) 上的右移算子

\[ S(x_1, x_2, x_3, \ldots) = (0, x_1, x_2, x_3, \ldots). \]

它显然线性。它保范数（\(\|S x\|_2 = \|x\|_2\)），所以连续，\(\|S\| = 1\)。

问：\(S\) 可逆吗？在有限维方阵世界里，"保范数 + 线性" 的方阵是正交矩阵——可逆，逆是它的转置。

但在 \(\ell^2\) 上，\(S\) 不可逆。它的像是 \(\{(0, y_2, y_3, \ldots)\}\)——不含 \((1, 0, 0, \ldots)\)。\(S\) 是单射但不满射。

\[\boxed{\ \ell^2 \text{ 上右移算子 } S: \text{ 单射, 保范数, 但不满射, 不可逆。}\ }\]

在有限维 里，"方阵单射 ⟺ 满射 ⟺ 可逆"——这是线性代数读本第 7 章的核心结论之一。在无穷维里这套等价彻底失效。"单不满" 这种残缺方阵在无穷维里随处可见——它会在第 21 章 谱论里以"残留谱" 的名字重新出现。

1.6 这本书要修的五件事

把上面五个崩塌串起来，你看出泛函分析的整张地图。这门课要做的，就是修 这五件事——给每个崩塌一套替代工具，让你在无穷维里继续做分析。

具体的修法：

  • 1.1 单位球不紧 → 弱收敛 + Banach–Alaoglu 把"紧" 在弱拓扑下找回来（ch17, 18）
  • 1.2 算子不连续 → 把空间换得更"丰富"（Sobolev 空间），让自然算子重新有界（ch23）
  • 1.3 子空间不闭 → 强制只研究闭 子空间，所有"投影 / 正交补 / 商" 都需 closedness 前提（ch6, 22）
  • 1.4 范数不等价 → 接受每个 \(p\) 是不同空间，按问题需要选 \(L^p\)；研究空间之间的嵌入与对偶（ch4, 15）
  • 1.5 算子 ≠ 矩阵 → 把算子当分析 对象——研究算子范数（ch9）、紧算子（ch20）、谱（ch21）、对偶算子（ch15）

这本书 24 章基本就是按这条修复地图组织的。

更高一层：泛函分析的所有"高定理" 都可以理解成给某个有限维奇迹一个无穷维证明版。Hilbert 空间正交分解（ch6）= 有限维"任意子空间有正交补" 的修复版（要求闭）。Riesz 表示（ch16）= 有限维"对偶等于自身" 的无穷维条件版。Banach–Alaoglu（ch18）= 有限维"球紧" 的弱*版。谱定理（ch21）= 有限维"对称矩阵正交对角化" 的无穷维（要求紧 + 自伴）版。Hahn–Banach（ch13, 14）= 让"线性泛函足够多" 这件事成为定理而不是有限维白送的事实。

你读到 ch24 那章，就会发现整本书是同一件事——把有限维线性代数 + 紧致性几何系统翻译 到无穷维，每次翻译都需要付一点税（"闭" "紧" "可分" "自伴"），那些税正是定理的假设条件。

下一章我们从最朴素的工具开始：度量、范数、Banach 空间——把"无穷维分析的入场券" 备齐。完备性会成为反复出现的关键词——它是上面五个崩塌的修法里都用到的"底层资源"。
