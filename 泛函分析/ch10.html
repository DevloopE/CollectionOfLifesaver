第 10 章 Baire 范畴定理

请你做一道朴素的事。考虑实数 \(\mathbb{R}\)——你最熟悉的完备度量空间。问：\(\mathbb{R}\) 能不能写成可数个瘦 集合的并？

"瘦" 还没有定义，让我先用一个直觉版的：一个集合"瘦" 是指它"没占什么地方"——比如有限集、可数集、闭区间的边界（端点）、单点集、低维流形。一个不"瘦" 的集合占据一些"胖" 的部分——比如开区间、有正测度的集合。

朴素直觉：可数个"瘦" 集合并起来还是"瘦" 的——\(\mathbb{R}\) 这么"胖" 不可能用可数个"瘦" 集填满。

这个直觉对，但它出乎意料地难证——直到 1899 年 Baire 给出了正确的"瘦" 定义和正确的证明。这条定理的形式不漂亮——它说"完备度量空间不能写成可数个无处稠密闭集的并"。它本身没什么直接的几何画面。但它一旦套到 Banach 空间上就孵化出三大原理（一致有界、开映射、闭图像）——这是泛函分析整个上层建筑的脊柱。

这一章我把定理 + 完整证明 + 三个非泛函应用给你——让你看见这条朴素定理的杠杆力。后面两章再把它的泛函 用法展开。

10.1 无处稠密 / 第一第二范畴

先把"瘦" 这个直觉名词变成定义。

定义（无处稠密）。设 \(X\) 是度量空间，\(A \subset X\)。\(A\) 无处稠密（nowhere dense）是说：\(\overline{A}\) 的内部 为空，即 \(\mathrm{int}(\overline{A}) = \emptyset\)。

展开成"用人话"：把 \(A\) 取闭包，那个闭包不包含任何开球。等价地说，\(A\) " 不能控制任何开球"——任何 \(X\) 中的开球 \(B\) 里，都 能找到一个更小的开球 \(B' \subset B\) 使 \(B' \cap A = \emptyset\)。

例子（在 \(\mathbb{R}\) 中）：

  • 有限集 \(\{x_1, \ldots, x_n\}\)：闭包就是自己，没有内部，无处稠密。
  • 整数集 \(\mathbb{Z}\)：闭包仍是 \(\mathbb{Z}\)，没有内部（整数之间总有非整数），无处稠密。
  • Cantor 集：闭集，但它"洞洞洞"，没有任何区间，无处稠密。
  • 有理数集 \(\mathbb{Q}\)：不 是无处稠密！\(\overline{\mathbb{Q}} = \mathbb{R}\)，内部就是整个 \(\mathbb{R}\)，非空。

请你停下来注意第四条——\(\mathbb{Q}\) 虽然很瘦（可数）但不无处稠密！这告诉你"瘦" 在数学上有两种不同的方式：拓扑的（无处稠密 = "闭包没内部"）和测度的（零测度 = "体积为零"）。\(\mathbb{Q}\) 测度为零但拓扑上稠——它从测度看"瘦"，从拓扑看"胖"。这两种"瘦" 不一致，是同一个集合可以从一种意义看大、从另一种意义看小 的源头。Baire 范畴用的是拓扑意义 的"瘦"。

有了"无处稠密" 这个原子，定义"第一范畴" 集：

定义（第一 / 第二范畴）。集合 \(A \subset X\) 第一范畴（meager / first category）是说：\(A\) 可以写成可数个 无处稠密集的并，即 \(A = \bigcup_{n = 1}^\infty A_n\)，每个 \(A_n\) 无处稠密。

第二范畴（non-meager / second category）就是"不是第一范畴"——不能写成可数个无处稠密集的并。

(" 范畴" 这个名字可能让你联想到 category theory，无关——这个 category 是 Baire 1899 年用的术语，比 category theory 早 50 年。我会尽量说"瘦" "胖" 而不重复"第一范畴"。)

例子。\(\mathbb{Q} \subset \mathbb{R}\) 是第一范畴——把 \(\mathbb{Q} = \{q_1, q_2, \ldots\}\)，每个单点 \(\{q_n\}\) 无处稠密，可数并起来给 \(\mathbb{Q}\)。所以可数集都是第一范畴的。

但是！我刚才说 \(\mathbb{Q}\) 在 \(\mathbb{R}\) 中不 是无处稠密。这两件事没矛盾——"第一范畴" 不要求这个集合自己 无处稠密，只要求它是可数个 无处稠密集的并。\(\mathbb{Q}\) 不无处稠密，但它是可数个无处稠密单点集的并——所以是第一范畴。

10.2 Baire 定理两条等价陈述

核心命题终于可以写出来。

定理（Baire 范畴定理）。设 \(X\) 是完备 度量空间。则 \(X\) 不是 第一范畴的。等价地说：\(X\) 不能写成可数个无处稠密闭集的并。

这条陈述有一个对偶 形式（用补集翻一下），更常被使用：

定理（Baire 范畴定理，开稠版本）。设 \(X\) 是完备度量空间，\(\{U_n\}_{n = 1}^\infty\) 是 \(X\) 中可数族开 且稠 子集。则它们的交 \(\bigcap_n U_n\) 稠 于 \(X\)。

两个版本的等价性：取 \(A_n = X \setminus U_n\)。"\(U_n\) 开稠" 等价于"\(A_n\) 闭且无处稠密"。"\(\bigcap U_n\) 稠" 等价于"\(X \setminus \bigcap U_n = \bigcup A_n\) 不能填满任何开球"——也就是"\(X\) 不能写成 \(\bigcup A_n\)"。

第二条陈述更 常用，因为"稠" 这个性质在使用时给一个构造方向——你想找某个性质的元素，你知道它落在稠 集 \(\bigcap U_n\) 里。

请你先停一下，把这两条陈述读两遍。它们听起来就像"完备空间不会被瘦集填满 / 完备空间里稠开集的可数交还是稠的"。听起来温和。但下一节我会让你看到它的证明——那个证明告诉你这条定理真正用了什么资源。

10.3 完整证明：嵌套闭球 + 完备性

我用第二个（开稠）版本。证明的灵魂是嵌套闭球 + Cauchy 完备性。

设 \(X\) 完备度量，\(U_1, U_2, \ldots\) 都是开稠子集。要证 \(\bigcap U_n\) 稠——也就是每个 非空开集 \(V \subset X\) 与 \(\bigcap U_n\) 相交。

固定一个非空开集 \(V\)。我们要找一个点 \(x \in V \cap \bigcap_n U_n\)。

归纳构造嵌套闭球 \(\overline{B_n}\)。

第一步。\(U_1\) 稠 ⟹ \(V \cap U_1 \neq \emptyset\)。又 \(V \cap U_1\) 是开（两个开集的交），非空，所以包含某个开球 \(B(x_1, r_1) \subset V \cap U_1\)。把半径减半：取闭球 \(\overline{B_1} := \overline{B(x_1, r_1 / 2)} \subset B(x_1, r_1) \subset V \cap U_1\)。注意 \(\overline{B_1}\) 是闭 球，但它整个落在开 球 \(B(x_1, r_1)\) 里，所以 \(\overline{B_1} \subset V \cap U_1\)。

这一步关键：闭球 \(\overline{B_1}\) 半径 \(r_1 / 2 \le 1/2\)（同时取 \(r_1 \le 1\)，调好初始）。

归纳步。假设已经构造好 \(\overline{B_n} = \overline{B(x_n, r_n / 2)}\)，半径 \(\le 1/2^n\)，且 \(\overline{B_n} \subset U_n\)。

看 \(B(x_n, r_n / 2) \cap U_{n+1}\)。\(U_{n+1}\) 稠 ⟹ 这个交非空（开球与稠集必相交）。它是开集，所以含一个开球 \(B(x_{n+1}, r_{n+1})\)，而且我们把 \(r_{n+1}\) 选得小一点：\(r_{n+1} \le 1 / 2^{n+1}\) 且 \(B(x_{n+1}, r_{n+1}) \subset B(x_n, r_n / 2) \cap U_{n+1}\)。同样取闭球 \(\overline{B_{n+1}} := \overline{B(x_{n+1}, r_{n+1} / 2)}\)，整个落在 \(B(x_{n+1}, r_{n+1})\) 里，所以

\[ \overline{B_{n+1}} \subset \overline{B(x_n, r_n / 2)} = \overline{B_n}, \quad \overline{B_{n+1}} \subset U_{n+1}. \]

嵌套关系成立，半径几何减半。

用 Cauchy 完备性。这一系列闭球 \(\overline{B_1} \supset \overline{B_2} \supset \overline{B_3} \supset \ldots\)，半径 \(r_n / 2 \to 0\)。中心序列 \(\{x_n\}\) 是 Cauchy 列：对 \(m > n\)，\(x_m \in \overline{B_n}\)，所以 \(d(x_n, x_m) \le r_n / 2 \to 0\)。

这里用到 \(X\) 完备——它给我们 \(\{x_n\}\) 收敛到某点 \(x \in X\)。由 \(\overline{B_n}\) 是闭，\(x_m \in \overline{B_n}\) 对所有 \(m \ge n\)，所以极限 \(x \in \overline{B_n}\)。这对所有 \(n\) 成立，所以

\[ x \in \bigcap_{n} \overline{B_n} \subset \bigcap_n U_n. \]

又 \(x \in \overline{B_1} \subset V\)。所以 \(x \in V \cap \bigcap_n U_n\)——存在性证完。\(\bigcap U_n\) 与每个非空开集 \(V\) 相交，稠。

请你停下来欣赏这个证明的骨架：

  1. 稠 ⟹ "每一步都能找到下一个开球"。
  2. 开 ⟹ "开球留有空间放下一个嵌套闭球"。
  3. 完备 ⟹ "嵌套闭球的交非空"（中心 Cauchy 收敛）。
  4. 闭球嵌套 ⟹ "极限点落在每个 \(U_n\) 里"。

四个零件缺一不可。完备性是这条定理的真正资源——你不 能在不完备的度量空间上奢望这条定理。

反例：\(X = \mathbb{Q}\)（有理数）显然不 完备。你可以把 \(\mathbb{Q} = \bigcup_q \{q\}\)——可数个无处稠密单点集的并——所以 \(\mathbb{Q}\) 是 第一范畴的。Baire 定理在不完备空间上失效。

这条证明的风格 在分析里反复出现：" 嵌套闭球 / 嵌套闭集 + Cauchy 完备 = 极限存在"。你已经在实分析读本第 6 章 Heine-Borel 的"嵌套区间引理"、在第 7 章 Banach 不动点定理里见过同一个骨架。Baire 定理是这套骨架的一般化。

10.4 三个非泛函应用

Baire 定理的"杠杆力" 在它能给各种 看起来无关的存在性结果统一 的证明。下面三个例子都 不是泛函分析问题——但都用 Baire 一招制胜。这些例子让你看见 这条定理的真正威力。

应用 1：\(\mathbb{R}\) 不可数。这个事实你可能已经知道（Cantor 对角线论证）。但用 Baire 给一个替代 证明：

假设 \(\mathbb{R} = \{x_1, x_2, x_3, \ldots\}\) 可数。每个单点集 \(\{x_n\}\) 在 \(\mathbb{R}\) 中无处稠密（闭集，没内部）。所以 \(\mathbb{R} = \bigcup \{x_n\}\) 是可数个无处稠密集的并——第一范畴。但 \(\mathbb{R}\) 完备，由 Baire 不能第一范畴。矛盾——\(\mathbb{R}\) 不可数。

这个证明只用一行 Baire，比 Cantor 对角线更朴素。它告诉你 Baire 是" 反构造" 的强力工具——"如果空间可以填满，则瘦不可能太多"。

应用 2：几乎所有连续函数处处不可微。这个结论你听了肯定不舒服。"几乎所有" 在这里是拓扑意义 上的：在 \(C[0, 1]\)（带 sup 范数）这个 Banach 空间里，"处处不可微的连续函数" 的集合是" 第二范畴的"——也就是它不能 被一些"瘦" 集合避开，反而是主体；"至少在一点可微的连续函数" 是第一范畴 的——它"瘦"。

这是个让人不安的事实：你脑子里能想到的"典型" 连续函数——多项式、\(\sin x\)、\(\cos x\)——都到处可微。但在 \(C[0, 1]\) 这个度量空间的拓扑意义 下，这些"典型" 函数构成的"到处可微" 集合反而 是稀有的，处处不可微才是常态。

证明草图（不展开技术细节）。对每个 \(n, M\)，定义

\[ E_{n, M} := \{ f \in C[0, 1] : \exists x_0,\ |f(x_0 + h) - f(x_0)| \le M |h| \text{ 对 } |h| < 1/n \}. \]

"在某点 \(x_0\) 可微" 的函数必定 落在某个 \(E_{n, M}\) 里（取 \(M = 2 |f'(x_0)|\) 等）。所以"至少一点可微" 的集合 \(\subset \bigcup_{n, M} E_{n, M}\)。

关键引理：每个 \(E_{n, M}\) 在 \(C[0, 1]\) 中是闭（用一致收敛保持局部 Lipschitz 常数）和无处稠密（任何连续函数附近都能放一个"很尖锐的折线" 跑出 \(E_{n, M}\)）。

所以 \(\bigcup_{n, M} E_{n, M}\) 是可数个无处稠密闭集的并，第一范畴。\(C[0, 1]\) 完备，由 Baire 它的补集稠——"处处不可微" 的连续函数稠在 \(C[0, 1]\) 中，第二范畴。

这条定理把"Weierstrass 处处不可微函数" 这种听起来奇异的对象从"例外" 改写成"常态"。这是 Baire 给的最有冲击力的应用之一——它颠覆了"连续函数应该大部分可微" 这种朴素想象。

应用 3：Liouville 数稠。Liouville 数 是指能被有理数极快 逼近的实数：\(x\) 是 Liouville 是说对每个 \(n\) 存在有理数 \(p/q\)（\(q > 1\)）使

\[ 0 < \left| x - \frac{p}{q} \right| < \frac{1}{q^n}. \]

Liouville 数都是超越数（不是任何整系数多项式的根）——这是 Liouville 1844 年构造第一个超越数的工具。

用 Baire 证：Liouville 数集在 \(\mathbb{R}\) 中稠。

对每个 \(n\)，定义

\[ U_n := \bigcup_{q \ge 2} \bigcup_{p} \left( \frac{p}{q} - \frac{1}{q^n}, \frac{p}{q} + \frac{1}{q^n} \right) \setminus \mathbb{Q}. \]

这是个开集（开区间的并去掉一个可数集仍开），并且稠（每个有理数附近 \(1/q^n\) 的区间都包含），不 包含有理数。

Liouville 数 \(= \bigcap_n U_n\)——每个 \(n\) 的逼近条件就是落在 \(U_n\) 里。

由 Baire（\(\mathbb{R}\) 完备），可数稠开集的交稠。Liouville 数稠在 \(\mathbb{R}\) 中。

这件事让人吃惊：超越数 / 极致逼近性这种听起来"稀有" 的性质，在 \(\mathbb{R}\) 中其实稠。它再次说明" 拓扑意义的稠 ≠ 测度意义的大"——Liouville 数稠（拓扑大），但 Lebesgue 测度为零（测度小）。两种"大" 的概念交叉时往往出乎意料。

三个应用看完，你应该看清 Baire 的使用模式：

  1. 把你想避开的坏性质 写成可数个闭集的并。
  2. 证每个闭集"无处稠密"——通常是"不能控制任何开球" 的局部摆动论证。
  3. Baire 给：补集稠 / 全空间不能被这些填满。
  4. 所以" 不坏" 的元素稠 / 存在 / 是常态。

10.5 模板：Baire 在 Banach 空间上怎么用

到这里你已经看到 Baire 在朴素度量空间上的力量。下面三章它将以泛函分析 模式登场。我先把"模板" 给你，让你看到下一章一致有界原理是怎么用 Baire 的。

模板：

  1. 有一族算子 / 函数 \(\{T_\alpha\}\)，逐点信息已知（比如"每个 \(x\) 都有 \(\sup_\alpha \|T_\alpha x\| < \infty\)"）。
  2. 把空间按"逐点信息的强度" 分层：\(E_n = \{x : \sup_\alpha \|T_\alpha x\| \le n\}\)。注意这些 \(E_n\) 是闭（\(\|T_\alpha x\|\) 在 \(x\) 上连续，\(\sup\) 是闭条件的交）。
  3. 逐点信息 ⟺ \(X = \bigcup_n E_n\)。
  4. 由 Baire：\(X\) 完备且不能由可数个无处稠密闭集填满，所以某个 \(E_{n_0}\) 必有非空内部——某个开球 \(\overline{B(x_0, r)} \subset E_{n_0}\)。
  5. 用线性性平移 + 这个"实有内部的 \(E_{n_0}\)" 得到整个 单位球上的一致控制——升级到全局一致信息。

下一章第 11 章就是这个模板的精确应用——一致有界原理："逐点有界 ⟹ 一致有界"。开映射定理 / 闭图像定理也走类似路线，区别在第 5 步的"线性平移" 怎么具体操作。

让你先看清这条" 升级" 的本质：Baire 把"有些点上信息好"（每个 \(x\) 有自己的有限上界）免费升级 到"整个单位球上信息好"（一个统一的有限上界）。这个免费升级专属于 Banach 空间——它的"买路钱" 就是完备性。所有"逐点 ⟹ 一致" 类型的定理（不只一致有界——还有 Lebesgue 微分定理的某些版本、Fubini 定理的某些版本）都吃这条 Baire 的红利。

10.6 完备的"必要" 与"充分"

这一章的核心结论用一句话写完：

\[\boxed{\quad \text{完备度量空间} \implies \text{第二范畴}. \quad}\]

但完备性是" 充分" 还是"必要"？

"充分" 显然成立——这就是 Baire 定理本身。"必要" 不成立——存在不完备但第二范畴 的度量空间。比如 \(\mathbb{R} \setminus \{0\}\) 不完备，但仍然第二范畴（任何瘦集填不满）。所以真实关系是：完备 ⟹ 第二范畴；反之不必。

这件事在使用上重要：当你想用 Baire，验证完备就够，不必更多。Banach 空间定义 就是"完备的赋范空间"，所以所有 Banach 空间都满足 Baire 定理。这就是为什么后面三大原理在 Banach 空间上"免费" 成立。

另一个微妙处。Baire 定理还有第二个 版本——对局部紧 Hausdorff 空间 也成立（不需要完备性，用紧致性代替）。这对实数 / Euclidean 拓扑这种本来就既完备又局部紧的空间没区别，但对某些拓扑群 / 流形会有用。我们这本书不 用这个版本——只用度量 + 完备的版本。

到这里 Baire 定理本身的工作已经做完。让我把这一章的关键结论汇总：

\[\boxed{\quad \begin{aligned} &\textbf{1. 朴素定义:}\quad && \text{无处稠密} = \overline{A} \text{ 没内部};\ \text{ 第一范畴} = \text{ 可数无处稠密的并}.\\ &\textbf{2. Baire 定理:}\quad && X \text{ 完备度量} \implies X \text{ 第二范畴};\ \text{ 等价: 可数稠开集的交稠}.\\ &\textbf{3. 证明骨架:}\quad && \text{嵌套闭球} + \text{Cauchy 完备} = \text{ 极限点存在于全部 } U_n.\\ &\textbf{4. 三个非泛函应用:}\quad && \mathbb{R} \text{ 不可数};\ \text{ 处处不可微连续函数稠};\ \text{ Liouville 数稠}.\\ &\textbf{5. 泛函模板:}\quad && \text{逐点信息} \xrightarrow{\text{Baire}} \text{ 全局一致信息}\ (\text{下章兑现}). \end{aligned} \quad}\]

下一章我们把这条模板第一次真用 在算子上——一致有界原理（Banach-Steinhaus）：一族算子在每个点上"逐点有界"，自动升级为"算子范数一致有界"。这条定理的应用极广——从 Fourier 级数发散的"典型连续函数" 论证、到弱收敛序列必有界、到二元数值积分的稳定性，全部基于这条 Baire 的孵化结果。
