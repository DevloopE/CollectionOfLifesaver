第 11 章 一致有界原理 (Banach-Steinhaus)

请你设想这样一个具体处境。我手里有一族算子 \(\{T_\alpha\}_{\alpha \in A}\)——索引集 \(A\) 任意大，可数不可数都行——它们都是从一个 Banach 空间 \(X\) 到一个赋范空间 \(Y\) 的有界线性算子。我对你没承诺它们的范数 \(\|T_\alpha\|\) 一致受控；我只承诺一件软得多的事：

\[ \text{对每个固定的 } x \in X, \quad \sup_{\alpha \in A} \|T_\alpha x\| < \infty. \]

请你停下来读一遍这条假设。它说"逐点 有界"——你给我一个 \(x\)，我能告诉你这族算子作用在这个 \(x\) 上的值不会跑到无穷。但是这个"上确界" 可能跟 \(x\) 有关——\(x\) 一变, 上确界就可以变。当 \(x\) 跑遍整个 \(X\) 时, 这些 \(\sup_\alpha \|T_\alpha x\|\) 完全可以越来越大, 没有统一上界。

朴素直觉到这一步通常会得到这个结论："那当然啦, 没承诺一致就别指望一致。"

这一章我要让你看见一个出乎意料的现象——这个朴素直觉错了。只要 \(X\) 完备, 上述"逐点有界" 假设自动升级为"一致有界":

\[ \sup_{\alpha \in A} \|T_\alpha\| < \infty. \]

注意这里左边是算子范数 的上确界——它控制了所有 \(x\) 一起的行为, 不是逐点的。从"对每个 \(x\) 各自有界" 到"对所有 \(x\) 一起有界", 这是一次自动升级。它是算子论里 Baire 纲定理（第 10 章）第一次大显身手——也是三大原理 中的第一条。

这一现象的名字叫 一致有界原理 (uniform boundedness principle, UBP), 也叫 Banach-Steinhaus 定理。它的证明短得让人不安——只用 Baire + 一点平移 + 线性。可它的影响极广：du Bois-Reymond 那个有名的 Fourier 级数发散反例（19 世纪让所有人困惑了几十年）, 用 UBP 三行就出来。

11.1 定理陈述

我们先把要证的事写干净。

定理 11.1 (一致有界原理 / Banach-Steinhaus, UBP). 设 \(X\) 是 Banach 空间, \(Y\) 是赋范空间, \(\{T_\alpha\}_{\alpha \in A}\) 是 \(\mathcal{B}(X, Y)\) (即 \(X \to Y\) 的有界线性算子全体）中的一族算子。如果

\[ \forall x \in X : \quad \sup_{\alpha \in A} \|T_\alpha x\| < \infty, \]

则

\[ \sup_{\alpha \in A} \|T_\alpha\| < \infty. \]

请你注意三个细节, 以便看清这条定理的"分量"。

第一, \(X\) 必须完备。这是 Baire 进来的前提。如果 \(X\) 不完备, UBP 一般不成立——课后我们会构造一个反例（取 \(X\) 是某个不完备空间, 让一族算子逐点有界但范数无界）。完备性是 UBP 的"底层资源", 跟第 1 章讲的 5 个修复一样, 都靠它。

第二, \(Y\) 不必完备。这条定理只用到 \(X\) 的 Baire 性。\(Y\) 只是装算子的目标空间, 它可以是任何赋范空间。

第三, 索引集 \(A\) 任意大——可数、不可数, 甚至连续族都行。这条定理对一族算子的"势" 没有要求, 这一点很关键, 因为后面的应用经常对一个连续参数 \(\alpha \in [0, \infty)\) 用它。

把假设和结论摆在一起:

\[\boxed{\ \text{逐点有界} \ \stackrel{X \text{ 完备}}{\Longrightarrow}\ \text{一致有界 (即算子范数有界)}.\ }\]

"逐点 \(\Rightarrow\) 一致" 这件事本身就反直觉——一般你期望需要"对所有 \(x\) 用相同常数" 这种强假设, 才能推"对所有 \(x\) 用相同范数界"。但 Baire 让你不用——它把"对每个 \(x\)" 自动黏合成"对所有 \(x\)"。下面我们看怎么做到的。

11.2 证明：Baire 锁定 \(F_n\) 含开球, 平移升级

请跟我把整个证明做一遍, 它分两步, 思路在两个点上"撞墙" 然后 reframe。

第一步：构造 \(F_n\), 用 Baire 找含开球的 \(F_N\)。

定义

\[ F_n = \{x \in X : \|T_\alpha x\| \le n \text{ 对所有 } \alpha \in A\} = \bigcap_{\alpha \in A} \{x : \|T_\alpha x\| \le n\}, \quad n = 1, 2, 3, \ldots \]

每个 \(F_n\) 是闭的——因为它是闭集 \(\{x : \|T_\alpha x\| \le n\} = T_\alpha^{-1}(\overline{B_Y(0, n)})\) 的交, 而 \(T_\alpha\) 连续 (这就是为什么用闭球 \(\le n\) 而不是开球 \(< n\): 闭集的交还是闭集)。

逐点有界假设给的是: 对每个 \(x \in X\), 存在 \(N\) (依赖 \(x\)) 使 \(\|T_\alpha x\| \le N\) 对所有 \(\alpha\)——也就是 \(x \in F_N\)。所以

\[ X = \bigcup_{n=1}^{\infty} F_n. \]

请你停下来看一眼——这是 Baire 想要的形式：一个 Banach 空间被写成可数个闭集的并。Baire 纲定理（第 10 章）说: 完备空间不能写成可数个无内点 的闭集的并; 反过来说, 如果它是可数个闭集的并, 至少有一个闭集有内点。

所以存在某个 \(N\) 使 \(F_N\) 有内点——即存在 \(x_0 \in F_N\) 与 \(r > 0\) 使

\[ B(x_0, r) := \{x \in X : \|x - x_0\| < r\} \subset F_N. \]

翻译成算子语言, 这一步说: 在 \(x_0\) 附近半径 \(r\) 的整个开球里, 所有 \(T_\alpha\) 都受 \(N\) 控制。

第二步：平移到原点附近, 升级到算子范数。

朴素地想"找到了开球, 然后呢"——这一步要 reframe。开球不在原点, 我们要把"\(x_0\) 附近的控制" 翻译成"原点附近的控制"——也就是用线性 把它平移过去。

取任何 \(z \in X\) 满足 \(\|z\| < r\)。则 \(x_0 + z \in B(x_0, r) \subset F_N\), 所以 \(\|T_\alpha(x_0 + z)\| \le N\) 对所有 \(\alpha\)。又 \(x_0 \in F_N\) (因为它是球心, 球本身在 \(F_N\) 里), 所以 \(\|T_\alpha x_0\| \le N\)。线性给

\[ \|T_\alpha z\| = \|T_\alpha(x_0 + z) - T_\alpha x_0\| \le \|T_\alpha(x_0 + z)\| + \|T_\alpha x_0\| \le N + N = 2N. \]

所以对所有 \(\|z\| < r\) 与所有 \(\alpha\), \(\|T_\alpha z\| \le 2N\)。

最后用算子范数的缩放定义。任意 \(x \in X\) 非零, 设 \(z = \frac{r}{2 \|x\|} x\), 则 \(\|z\| = r/2 < r\), 故 \(\|T_\alpha z\| \le 2N\), 即 \(\frac{r}{2 \|x\|} \|T_\alpha x\| \le 2N\), 故

\[ \|T_\alpha x\| \le \frac{4N}{r} \|x\|. \]

这条对所有 \(\alpha, x\) 成立, 即

\[ \|T_\alpha\| = \sup_{\|x\| = 1} \|T_\alpha x\| \le \frac{4N}{r} \quad \text{对所有 } \alpha \in A. \]

也就是 \(\sup_\alpha \|T_\alpha\| \le 4N/r < \infty\)。证毕 \(\square\)。

请你回头看一下整个证明的形状。第一步是把分析问题改写成"集合的并 = 全空间", 让 Baire 出场, 拿到一个含开球 的闭集。第二步是用线性结构——平移 + 缩放——把"\(x_0\) 附近的有界" 升级到"全空间的算子范数有界"。Baire 给"局部的几何资源", 线性给"从局部推到全局" 的代数升级。这两件事的合成就是三大原理共有的骨架——你在下一章开映射定理里会看到一模一样的两步, 形状几乎一字不差。

另外, 请注意 \(X\) 完备 的关键作用就在那一句"Baire 给出一个 \(F_N\) 有内点"。如果 \(X\) 不完备, Baire 失效, 你拿不到这个含开球的闭集, 整个论证立刻崩。完备性不可省。

11.3 三种等价表述

UBP 有几个等价的写法, 用的时候按情境取最方便的。

UB1 (原型, 即 11.1): 逐点有界 \(\Rightarrow\) 一致有界。

UB2 (反方向): 设 \(X\) 是 Banach, \(\{T_\alpha\}\) 是 \(X \to Y\) 的有界线性算子族。如果 \(\sup_\alpha \|T_\alpha\| = \infty\), 则存在某个 \(x \in X\) 使 \(\sup_\alpha \|T_\alpha x\| = \infty\)。

UB2 是 UB1 的逆否——直接换写。它的解读是："要破坏一致有界, 至少要有一个 \(x\) 单独被破坏掉——你不能靠'每个 \(x\) 都受控但总和无界' 来破坏。" 这条版本在反例构造里特别好用 (你下一节就会看到 Lebesgue 常数怎么用它)。

UB3 (强收敛版, 也叫 Banach-Steinhaus 收敛定理): 设 \(X\) Banach, \(\{T_n\}\) 是 \(X \to Y\) 的有界线性算子序列, 满足对每个 \(x \in X\) 极限

\[ T x := \lim_{n \to \infty} T_n x \]

都存在 (\(Y\) 中)。则:

  • (a) \(\sup_n \|T_n\| < \infty\) (一致有界);
  • (b) \(T\) 是有界线性算子, 且 \(\|T\| \le \liminf_n \|T_n\|\)。

证明 UB3。逐点极限存在 \(\Rightarrow\) 逐点 Cauchy \(\Rightarrow\) 逐点有界 (\(\sup_n \|T_n x\| < \infty\) 对每个 \(x\)), 由 UB1 得 \(M := \sup_n \|T_n\| < \infty\), 这就是 (a)。

对于 (b), \(T\) 显然线性 (极限保持线性运算)。范数: 对任何 \(x\) 与 \(\|x\| \le 1\),

\[ \|T x\| = \lim_n \|T_n x\| = \liminf_n \|T_n x\| \le \liminf_n \|T_n\| \cdot \|x\| \le \liminf_n \|T_n\|. \]

取 \(\sup\) 给 \(\|T\| \le \liminf_n \|T_n\|\), 故 \(T\) 有界 (\(\|T\| \le M < \infty\))。\(\square\)

UB3 是日常做分析时最常用 的版本。它说：逐点极限算子 \(T\) 自动有界, 还给你一个范数估计 \(\|T\| \le \liminf \|T_n\|\)。这一句话你在量子力学（一族投影算子的弱极限是投影）、PDE 弱解 (一列近似解的极限是弱解)、Fourier 分析 (部分和算子 \(S_N\) 的极限) 里反复用。

请你停一下意识到 UB3 多么有用。在没有 UBP 之前, 你只能说"逐点极限存在", 但对极限算子的连续性 一无所知——它甚至可能不是有界算子。UBP 把这件事免费给你。它告诉你: 在 Banach 空间里, 一族算子的逐点收敛, 自动给你一个有界的极限算子。这是一条非常硬的结论。

11.4 反向 (共鸣定理): 发散点稠密

UBP 还有一个更强的"反方向" 版本, 它告诉你: 如果一族算子的范数无界, 那么逐点发散 的点不是少数, 而是稠密 (甚至是稠密 \(G_\delta\), 即"几乎处处")。

定理 11.2 (共鸣定理, condensation of singularities). 设 \(X\) 是 Banach, \(\{T_n\}\) 是 \(X \to Y\) 的有界线性算子序列, 且 \(\sup_n \|T_n\| = \infty\)。则集合

\[ D = \{x \in X : \sup_n \|T_n x\| = \infty\} \]

是 \(X\) 中的稠密 \(G_\delta\) 集——特别地, \(D \neq \emptyset\), 而且 \(X \setminus D\) 是第一纲集。

证明思路。考虑

\[ U_k = \{x \in X : \sup_n \|T_n x\| > k\} = \bigcup_n \{x : \|T_n x\| > k\}. \]

每个 \(U_k\) 是开的（开集的并）, 而且稠密——因为如果 \(U_k\) 在某个开球 \(B(x_0, r)\) 上不稠密, 即整个 \(B(x_0, r) \cap U_k = \emptyset\), 那么 \(\sup_n \|T_n x\| \le k\) 对 \(B(x_0, r)\) 中所有 \(x\) 成立, 用上面 11.2 的"平移升级" 论证就得 \(\sup_n \|T_n\| < \infty\), 与假设矛盾。所以 \(U_k\) 在每个开球里都有点, 即稠密。

\(D = \bigcap_k U_k\), 是稠密开集的可数交。Baire 纲定理告诉你: Banach 空间中可数个稠密开集的交是稠密 \(G_\delta\)。所以 \(D\) 稠密。\(\square\)

这个版本的解读："不一致有界" 是个非常严重 的事——它不仅意味着存在 某个 \(x\) 让算子族在它上发散, 而且意味着这种 \(x\) 到处都是 (稠密)。下面 11.5 节的 du Bois-Reymond 反例就会用这个版本：不仅存在一个连续函数让 Fourier 部分和发散, 而且这种"病态" 函数的集合 是稠密 \(G_\delta\)——病态是常态, 不是个例。

11.5 应用: du Bois-Reymond 反例

这一节做一个具体而重要的应用——它是 19 世纪一个让人困惑的难题, 在 UBP 框架下三行就解。

问题。给一个连续周期函数 \(f \in C(\mathbb{T})\) (这里 \(\mathbb{T} = \mathbb{R}/2\pi \mathbb{Z}\) 是单位圆周, \(f\) 视为 \(2\pi\) 周期的连续函数), 它的 Fourier 级数

\[ S_N f(x) = \sum_{n = -N}^{N} \hat{f}(n) e^{i n x}, \quad \hat{f}(n) = \frac{1}{2\pi} \int_0^{2\pi} f(t) e^{-i n t} \, dt \]

是不是对每个 \(x\) 都收敛到 \(f(x)\)?

朴素直觉 (19 世纪人也是这样想的): 既然 \(f\) 连续而且 Fourier 系数 \(\hat{f}(n) \to 0\) (Riemann-Lebesgue), 直觉上 \(S_N f\) 应该趋向 \(f\)。事实上 Fejér 已经证明 Cesàro 平均 \(\sigma_N f \to f\) 一致, 这给了人很强的信心。

du Bois-Reymond 在 1873 年构造了一个具体的连续函数, 它的 Fourier 级数在 \(0\) 点发散。这件事让人困惑了很久——构造很复杂, 不直观。直到 Banach-Steinhaus 出来, 它变成三步:

步骤 1: 把 \(S_N\) 写成卷积。回忆 Dirichlet 核

\[ D_N(t) = \sum_{n = -N}^{N} e^{i n t} = \frac{\sin((N + 1/2) t)}{\sin(t/2)}, \]

则 \(S_N f(0) = \frac{1}{2\pi} \int_0^{2\pi} f(t) D_N(t) \, dt\) (取 \(x = 0\) 看在原点的部分和值)。

步骤 2: 把 \(f \mapsto S_N f(0)\) 看成 \(C(\mathbb{T}) \to \mathbb{C}\) 的有界线性泛函

\[ T_N : C(\mathbb{T}) \to \mathbb{C}, \quad T_N f = S_N f(0) = \frac{1}{2\pi} \int_0^{2\pi} f(t) D_N(t) \, dt. \]

这是有界线性泛函 (和测度积分一样的论证), 它的范数是 \(C(\mathbb{T})\) 上的算子范数, 等于

\[ \|T_N\| = \frac{1}{2\pi} \int_0^{2\pi} |D_N(t)| \, dt =: L_N \]

叫做 Lebesgue 常数。这条等式右边是范数的标准计算 (见第 9 章关于 \(C(\mathbb{T})^* = M(\mathbb{T})\) 的对偶) ——给你 \(D_N\) 的符号 (取 \(f = \mathrm{sgn}(D_N)\) 的连续逼近), 上确界正好是 \(\int |D_N|\)。

步骤 3: 估 \(L_N\) 的下界, 表明 \(L_N \to \infty\)。这是经典估计:

\[ L_N = \frac{1}{2\pi} \int_0^{2\pi} \left| \frac{\sin((N + 1/2) t)}{\sin(t/2)} \right| dt \ge \frac{1}{2\pi} \int_0^{\pi} \frac{|\sin((N + 1/2) t)|}{t/2} \, dt \]

(用 \(\sin(t/2) \le t/2\) 在 \([0, \pi]\) 上)。换元 \(s = (N + 1/2) t\),

\[ L_N \ge \frac{1}{\pi} \int_0^{(N + 1/2) \pi} \frac{|\sin s|}{s} \, ds \ge \frac{1}{\pi} \sum_{k=1}^{N} \int_{(k-1)\pi}^{k \pi} \frac{|\sin s|}{s} \, ds \ge \frac{1}{\pi} \sum_{k=1}^{N} \frac{1}{k \pi} \int_0^\pi |\sin s| \, ds = \frac{2}{\pi^2} \sum_{k=1}^N \frac{1}{k}. \]

(中间用 \(s \le k \pi\) 和 \(\int_0^\pi |\sin s| ds = 2\)。) 所以

\[ L_N \ge \frac{2}{\pi^2} H_N \to \infty, \quad H_N = 1 + 1/2 + \cdots + 1/N. \]

结论: \(\|T_N\| = L_N \to \infty\), 即算子族 \(\{T_N\}\) 不一致有界。由 UB2 (或共鸣定理 11.2), 存在 \(f \in C(\mathbb{T})\) 使

\[ \sup_N |T_N f| = \sup_N |S_N f(0)| = \infty. \]

也就是说 \(S_N f(0)\) 发散。事实上由 11.4 共鸣定理, 这种 \(f\) 的集合是 \(C(\mathbb{T})\) 中的稠密 \(G_\delta\)——绝大多数连续函数 (在 Baire 纲意义下) 的 Fourier 级数在 \(0\) 点都发散。

\[\boxed{\ \exists f \in C(\mathbb{T}) : S_N f(0) \text{ 发散; 这种 } f \text{ 是 } C(\mathbb{T}) \text{ 中稠密 } G_\delta.\ }\]

请你停下来感受一下这件事。19 世纪人花了几十年争论"连续函数的 Fourier 级数到底收不收敛"。du Bois-Reymond 给出反例, 但构造是手工的、复杂的、看不出"为什么"。Banach-Steinhaus (1927) 出来后,三步: 写卷积, 算 Lebesgue 常数, 用 UBP。它解释了为什么反例必然存在——因为 Lebesgue 常数 \(L_N\) 趋向无穷。

这个例子是泛函分析"用一般原理统一具体现象" 力量的最佳样本。Lebesgue 常数 \(L_N\) 的发散是事实, UBP 把它直接翻译成"必有反例"——你不需要构造反例, 你只要算 \(L_N\)。

(顺便: 上确界为什么是 \(L_N\)? 因为 \(C(\mathbb{T})\) 的对偶是有限测度空间, 算子范数 = 测度的全变差; 这里测度是 \(\frac{1}{2\pi} D_N(t) \, dt\), 全变差 = \(\frac{1}{2\pi} \int |D_N|\)。详见第 15 章。)

11.6 应用: 双线性形式分别有界 \(\Rightarrow\) 联合有界

第二个应用形状不同, 但同样核心。

问题。设 \(X, Y\) 是 Banach 空间, \(B : X \times Y \to \mathbb{C}\) (或 \(\mathbb{R}\)) 是双线性 的——即对每个 \(y\) 固定, \(x \mapsto B(x, y)\) 线性; 对每个 \(x\) 固定, \(y \mapsto B(x, y)\) 线性。问: 如果 \(B\) 分别 有界——即对每个固定的 \(y\), \(x \mapsto B(x, y)\) 是 \(X \to \mathbb{C}\) 的有界线性泛函; 对每个固定的 \(x\) 同样——那么 \(B\) 是不是联合 有界, 即存在 \(C\) 使

\[ |B(x, y)| \le C \|x\| \|y\| \text{ 对所有 } x, y? \]

朴素直觉又一次说"不一定吧, 分别和联合是两回事"。但 UBP 又一次反直觉地给出"是", 只要 \(X, Y\) 完备。

定理 11.3. 设 \(X, Y\) Banach, \(B : X \times Y \to \mathbb{C}\) 双线性且分别有界 (即对每个 \(x\), \(y \mapsto B(x, y)\) 在 \(Y\) 上有界; 对每个 \(y\), \(x \mapsto B(x, y)\) 在 \(X\) 上有界)。则 \(B\) 联合 有界:

\[ \exists C > 0 : |B(x, y)| \le C \|x\| \|y\| \text{ 对所有 } x \in X, y \in Y. \]

证明。对每个 \(y \in Y\), 定义 \(T_y : X \to \mathbb{C}\) 为 \(T_y(x) = B(x, y)\)。由分别有界假设, \(T_y\) 是有界线性泛函, 即 \(T_y \in X^*\), \(\|T_y\| < \infty\)。

把 \(\{T_y\}_{y \in B_Y(0,1)}\) (即 \(\|y\| \le 1\)) 看成一族 \(X \to \mathbb{C}\) 的有界算子。要用 UBP, 需要逐点有界——即对每个固定的 \(x\), \(\sup_{\|y\| \le 1} |T_y(x)| = \sup_{\|y\| \le 1} |B(x, y)| < \infty\)。这正是另一边分别有界的内容: 固定 \(x\) 后, \(y \mapsto B(x, y)\) 在 \(Y\) 上有界, 即存在 \(C_x\) 使 \(|B(x, y)| \le C_x \|y\| \le C_x\) 对 \(\|y\| \le 1\)。所以逐点有界。

由 UBP, \(\sup_{\|y\| \le 1} \|T_y\| =: M < \infty\)。这意味着对所有 \(\|y\| \le 1\) 和所有 \(x\), \(|B(x, y)| \le M \|x\|\)。对一般 \(y\) 用线性: \(|B(x, y)| = \|y\| \cdot |B(x, y/\|y\|)| \le M \|x\| \|y\|\)。取 \(C = M\) 即可。\(\square\)

请你品一下这条结论。在有限维 里, "双线性 + 分别连续 = 联合连续" 是个 trivial 事实——因为有限维上线性自动连续。在一般 拓扑空间里, 这个升级不成立——需要 Banach 完备。这正是 Lax-Milgram 定理 (下一章之后会用到) 的前置: 我们经常处理的双线性形式 (比如 \(B(u, v) = \int \nabla u \cdot \nabla v\)) 我们只能验证它分别 有界 (固定一个变量, 验证另一个变量的连续性, 通过积分估计就能做)。UBP 让我们一句话从这个验证升级到联合有界, 后面才能谈 Lax-Milgram、谈弱解。

11.7 总结：UBP 的"形状"

这一章你看到了 UBP 的三个面孔——原型 (UB1)、强收敛 (UB3)、共鸣 (11.2)——和两个应用 (Fourier 级数发散; 双线性自动联合有界)。我希望你带走的形状 有两件事:

第一, UBP 的本质是"逐点 \(\Rightarrow\) 一致, 当 \(X\) 完备"。它是 Baire 纲在算子论里的化身——Baire 给"含开球", 线性把"局部" 升级到"全局"。这两件事的合成形状, 在下一章开映射定理里你会看到几乎一字不差的重复。

第二, UBP 的应用形状是"换边" 论证——你想证某个一致估计 (例如 \(\|T_N\|\) 一致界), 你反过来 假设它不一致, 然后用 UBP 反方向得"存在 \(x\) 使 \(\sup \|T_N x\| = \infty\)"。du Bois-Reymond 反例就是这个形状的最佳样本: 我们想问"是否所有 \(f\) 都收敛", 转化成"是否 \(\|T_N\|\) 一致有界", 算 \(L_N \to \infty\), 立刻反向得反例。

下一章我们看三大原理的第二条 + 第三条: 开映射定理与闭图像定理。两条都是 Baire 的化身, 形状和 UBP 同源, 但它们解决的问题不同——它们处理的是"代数等式自动升级为拓扑等式" 的现象。准备好。
