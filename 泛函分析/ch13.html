第 13 章 Hahn-Banach 解析形式

请你先做一道朴素的问。一个非零向量 \(x \in X\) 摆在你面前 (\(X\) 是赋范空间)。你想"看见" 它——也就是, 找一个连续线性泛函 \(\varphi \in X^*\) 使 \(\varphi(x) \neq 0\)。

有限维你瞬间能做: 在 \(\mathbb{R}^n\) 里, \(x = (x_1, \ldots, x_n) \neq 0\), 取 \(x_k \neq 0\), 让 \(\varphi(y) = y_k\) 是第 \(k\) 个坐标投影, \(\varphi(x) = x_k \neq 0\)。这个泛函显然连续 (有限维线性映射都连续)。

无穷维呢? 设 \(x \neq 0\) 是 \(\ell^\infty\) 中某个序列, 你想找连续线性泛函看见它。最自然的尝试是"取第 \(k\) 位坐标"——可这是有限维直觉的迁移, 在无穷维它有时不够 (比如 \(x = (1, 1, 1, \ldots)\) 在 \(\ell^\infty\)/\(c_0\) 商空间里非零, 你需要更精细的"极限型" 泛函如 Banach 极限才能看见它)。

这一章我要让你看清: "无穷维空间总能找到泛函看见每一个非零向量" 这件事, 在一般 Banach 空间里既不显然也并不自动。它需要一条强大的延拓定理来保证。这条定理就是Hahn-Banach 定理——三大原理的最后一条。它的风味跟 UBP/OMT/CGT 完全不同:

  • UBP/OMT/CGT 用 Baire 纲, 需要 Banach (完备), 把"软" 假设升级为"硬" 结论。
  • Hahn-Banach 用 Zorn 引理 (即选择公理), 不需要完备, 解决的是"子空间上的泛函能否延拓到全空间, 同时保范数"。

这一章我们做解析形式——直接给"延拓 + 保范数"。下一章 (第 14 章) 我们做几何形式——把"延拓" 翻译成"超平面分离"。两形式等价, 角度不同。

13.1 为什么需要 Hahn-Banach: 让 \(X^*\) 足够大

我们先把动机 讲清楚。下面三件事都需要"对偶空间 \(X^* = \mathcal{B}(X, \mathbb{C})\) 有足够多的元素", 而 Hahn-Banach 是这件事的根源。

事件 1 (识别一个向量)。两个向量 \(x, y \in X\), 我们想知道 \(x = y\) 是否成立。等价 写法: 是否对每个 \(\varphi \in X^*\) 有 \(\varphi(x) = \varphi(y)\)? 等价的"是" 需要对偶空间足够大: 如果 \(x \neq y\) 但所有 泛函都给同样的值, 那就是说 \(X^*\) "看不见" \(x - y\), \(X^*\) 太小 了。Hahn-Banach 排除这种情况: 对每个非零 \(z \in X\), 总有 \(\varphi \in X^*\) 使 \(\varphi(z) \neq 0\)。

事件 2 (计算范数)。给一个 \(x\), 我们想用泛函算它的范数。一个 trivial 估计: \(\|x\| \ge |\varphi(x)| / \|\varphi\|\) 对每个非零 \(\varphi\) (因为 \(|\varphi(x)| \le \|\varphi\| \|x\|\))。理想情形: 这条估计是紧 的, 即 \(\|x\| = \sup_{\|\varphi\| \le 1} |\varphi(x)|\)。这条等式叫"对偶式范数公式", 它要求"有 \(\varphi\) 真把 \(x\) 看到极致"——具体是 \(\|\varphi\| = 1\) 且 \(\varphi(x) = \|x\|\)。Hahn-Banach 给你这个 \(\varphi\) 的存在。

事件 3 (闭子空间识别)。给一个子空间 \(M \subset X\), 我们想知道某个 \(x \notin \overline{M}\) 是否真的不在闭包里。等价的形式 (后面证): \(x \notin \overline{M}\) \(\Leftrightarrow\) 存在 \(\varphi \in X^*\) 使 \(\varphi|_M = 0\) 而 \(\varphi(x) \neq 0\)——即一个"湮没 \(M\) 但看见 \(x\)" 的泛函。Hahn-Banach 给这个 \(\varphi\) 的存在。它把"不在闭包里" 这种几何条件翻译成"有泛函分离", 这正是 ch14 几何形式的内容。

三件事的共同需求: "对偶空间足够大"。在有限维这件事白送 (\(X^* \cong X\) 维数相等, 自然有"足够多" 泛函)。在无穷维, "足够大" 不是 trivial——需要 Hahn-Banach 才能做到。

Hahn-Banach 定理把"足够大" 翻译成一种具体的延拓技术: 给我子空间上一个泛函, 我能延拓到全空间不增范数。这是一切上面"事件" 的根源。

13.2 实数情形定理陈述

我们先在实 数域 \(\mathbb{R}\) 上写陈述, 之后再处理复数。

定理 13.1 (Hahn-Banach 解析形式, 实). 设 \(X\) 是实 向量空间 (不必赋范! 不必完备!), \(p : X \to \mathbb{R}\) 是次线性 的——即满足:

  • (i) 正齐: \(p(\lambda x) = \lambda p(x)\) 对所有 \(\lambda \ge 0\) 与 \(x \in X\);
  • (ii) 次加: \(p(x + y) \le p(x) + p(y)\) 对所有 \(x, y \in X\)。

设 \(M \subset X\) 是线性子空间, \(f : M \to \mathbb{R}\) 是线性 泛函, 满足

\[ f(x) \le p(x) \text{ 对所有 } x \in M \]

(称 \(f\) 被 \(p\) 支配)。则存在线性 泛函 \(F : X \to \mathbb{R}\) 使

  • (a) \(F|_M = f\) (延拓);
  • (b) \(F(x) \le p(x)\) 对所有 \(x \in X\) (保支配)。

请你停下来体会一下这条陈述的普适性: 它不假设 \(X\) 是赋范、不假设完备; 它只用一个次线性 \(p\) 当"控制函数"。当 \(p(x) = \|x\|\) (这显然是次线性: 范数自动正齐 + 三角不等式), 同时 \(f\) 是 \(M\) 上的有界线性泛函, "支配" 即 \(|f(x)| \le \|x\|\) 即 \(\|f\| \le 1\)。这种特殊化给出最常用的"赋范情形", 见 13.6 节。

核心是一维延拓引理——把 \(f\) 从 \(M\) 延拓到 \(M \oplus \mathbb{R} v\) (一个一维扩张), 同时保支配。这是整个 Hahn-Banach 的"核心 lemma"; 一旦做到, 用 Zorn 把它一直推到全空间。下面我们一步一步做。

13.3 一维延拓引理 (核心 lemma)

引理 13.2 (一维延拓). 设 \(X\) 实向量空间, \(p : X \to \mathbb{R}\) 次线性, \(M \subset X\) 线性子空间, \(f : M \to \mathbb{R}\) 线性且 \(f(x) \le p(x)\) 对所有 \(x \in M\)。设 \(v \in X \setminus M\) 是任一不在 \(M\) 中的向量。则存在 \(F : M \oplus \mathbb{R} v \to \mathbb{R}\) 线性, \(F|_M = f\), 且 \(F(z) \le p(z)\) 对所有 \(z \in M \oplus \mathbb{R} v\)。

证明。\(M \oplus \mathbb{R} v\) 中元素都唯一写成 \(z = x + t v\), \(x \in M\), \(t \in \mathbb{R}\)。要定义 \(F\) 线性, 只需选一个数 \(\alpha = F(v)\), 然后令

\[ F(x + t v) = f(x) + t \alpha. \]

这显然延拓 \(f\) (\(t = 0\) 时 \(F = f\)) 且线性。问题: 是否存在 \(\alpha\) 使保支配 \(F(x + t v) \le p(x + t v)\)?

把这条条件按 \(t\) 的符号拆。\(t > 0\) 时 \(F(x + t v) = f(x) + t \alpha \le p(x + t v)\) 即

\[ \alpha \le \frac{p(x + t v) - f(x)}{t} = p(x/t + v) - f(x/t) \]

(用次线性: 把 \(t\) 提出来; 用 \(f\) 线性: 把 \(t\) 提出来)。改写 \(y = x/t \in M\): 即

\[ \alpha \le p(y + v) - f(y) \text{ 对所有 } y \in M. \]

\(t < 0\) 时 \(F(x + t v) = f(x) + t \alpha \le p(x + t v)\), 写 \(t = -s\), \(s > 0\):

\[ f(x) - s \alpha \le p(x - s v), \quad \alpha \ge \frac{f(x) - p(x - s v)}{s} = f(x/s) - p(x/s - v). \]

改写 \(y' = x/s \in M\): 即

\[ \alpha \ge f(y') - p(y' - v) \text{ 对所有 } y' \in M. \]

\(t = 0\) 时条件 \(f(x) \le p(x)\) 已假设。

所以"存在 \(\alpha\) 使保支配" 等价于

\[ \sup_{y' \in M} [f(y') - p(y' - v)] \le \inf_{y \in M} [p(y + v) - f(y)]. \]

这就是要证的核心不等式。如何证? 任取 \(y, y' \in M\), 我们要

\[ f(y') - p(y' - v) \le p(y + v) - f(y), \quad \text{即} \quad f(y) + f(y') \le p(y + v) + p(y' - v). \]

左边 \(f(y) + f(y') = f(y + y')\) (线性) \(\le p(y + y')\) (假设)。又 \(p(y + y') = p((y + v) + (y' - v)) \le p(y + v) + p(y' - v)\) (次加)。串起来:

\[ f(y) + f(y') = f(y + y') \le p(y + y') \le p(y + v) + p(y' - v). \]

正是要证的。所以 \(\sup \le \inf\), 故存在 \(\alpha\) (例如 \(\alpha = \sup_{y'} [f(y') - p(y' - v)]\)) 满足两边夹的条件。\(\square\)

请你停下来感受这条一维延拓 的核心思路。我们要选一个数 \(\alpha\), 使得新泛函在所有 方向 (\(t > 0\) 和 \(t < 0\)) 都被 \(p\) 支配。这给 \(\alpha\) 的"下界" 和"上界"。证存在 \(\alpha\) 满足两边即证"下界 \(\le\) 上界", 而这条不等式来自 \(p\) 的次加性 和 \(f \le p\) 的支配。整个 Hahn-Banach 的全部"魔术" 都在这一步: 一行不等式 \(f(y + y') \le p(y + y') \le p(y + v) + p(y' - v)\)。

下面把这条一维延拓"叠" 起来——用 Zorn 引理推到全空间。

13.4 用 Zorn 推到全空间

这一步需要 Zorn 引理 (即选择公理)。Zorn 引理说: 偏序集中, 如果每条链都有上界, 则存在极大元。我们在"所有保支配延拓" 的集合上用它。

定义

\[ \mathcal{P} = \{(N, g) : N \supset M \text{ 是线性子空间}, g : N \to \mathbb{R} \text{ 线性}, g|_M = f, g \le p \text{ 在 } N \text{ 上}\}. \]

这是"从 \(f\) 出发的所有保支配延拓" 的集合。它非空 (\((M, f) \in \mathcal{P}\) 显然)。

偏序: \((N_1, g_1) \le (N_2, g_2)\) 当且仅当 \(N_1 \subset N_2\) 且 \(g_2|_{N_1} = g_1\)。

要用 Zorn, 验证每条链都有上界。设 \(\{(N_\alpha, g_\alpha)\}_{\alpha \in A}\) 是 \(\mathcal{P}\) 中的链 (即任两元素可比)。令 \(N = \bigcup_\alpha N_\alpha\)。由链性质 \(N\) 是线性子空间 (任两 \(x, y \in N\) 在某个 \(N_\alpha\) 里都在, 那里就是子空间)。定义 \(g : N \to \mathbb{R}\) 为 \(g(x) = g_\alpha(x)\) 当 \(x \in N_\alpha\)——由链性质这定义良好 (\(x \in N_\alpha \cap N_\beta\) 时 \(g_\alpha(x) = g_\beta(x)\))。\(g\) 显然线性, 延拓 \(f\), 被 \(p\) 支配。所以 \((N, g) \in \mathcal{P}\) 且是这条链的上界。

由 Zorn, 存在极大元 \((N^*, F) \in \mathcal{P}\)。我们要证 \(N^* = X\)。如果不, 取 \(v \in X \setminus N^*\), 由一维延拓引理 (13.2), \(F\) 可延拓到 \(N^* \oplus \mathbb{R} v\) 保支配, 这给出严格更大的元素, 与极大性矛盾。所以 \(N^* = X\), \(F\) 是要找的延拓。\(\square\)

请你停一下意识到 Zorn 的角色。它把"一维一维延拓" 整合成"一次性延拓到全空间"。不可数 维的空间也通用——这是 Zorn 的力量。代价: Zorn 等价于选择公理, 有"非构造" 的味道——你证明了 \(F\) 存在, 但没有给出 \(F\) 的显式构造。这是 Hahn-Banach 跟 UBP/OMT/CGT 不同的地方: 那三条原理给的"存在性" 都是 Baire 给的, Baire 本身在 ZF 里也是"半构造" 的, 但 Hahn-Banach 用 Zorn 更"非构造"。

历史上 Hahn (1927) 和 Banach (1929) 独立证了这条定理, 都用了某种形式的 Zorn (当时叫"超穷归纳")。他们在意的是对偶空间足够大 这件事——这是他们要让 \(X^*\) 真正成为有用对象的根源。

13.5 复数情形 (实部 / 虚部技巧)

很多应用 (Fourier 分析、量子力学、复 Banach 代数) 用复数域。Hahn-Banach 在复数情形怎么写? 答案是同样的, 但要做一点"实部 / 虚部" 的技巧。

定理 13.3 (Hahn-Banach, 复). 设 \(X\) 是复 向量空间, \(p : X \to \mathbb{R}\) 是半范——即满足:

  • (i) \(p(\lambda x) = |\lambda| p(x)\) 对所有 \(\lambda \in \mathbb{C}\) 与 \(x\) (注意是 \(|\lambda|\), 不是 \(\lambda\));
  • (ii) \(p(x + y) \le p(x) + p(y)\)。

(半范比次线性强一点——它要求任何 复标量都用 \(|\cdot|\) 提出, 包括负数和复数。范数是半范的特例。) 设 \(M\) 复线性子空间, \(f : M \to \mathbb{C}\) 复线性, \(|f(x)| \le p(x)\) 对所有 \(x \in M\)。则存在复线性 \(F : X \to \mathbb{C}\), \(F|_M = f\), \(|F(x)| \le p(x)\) 对所有 \(x\)。

证明 (Bohnenblust-Sobczyk 1938)。把复数情形归约 为实数情形, 用"实部 / 虚部技巧":

步骤 1: 把 \(f\) 拆成实部 + 虚部。\(f(x) = u(x) + i \cdot u'(x)\), 其中 \(u, u' : M \to \mathbb{R}\) 都是实线性 (复线性 \(\Rightarrow\) 实部、虚部各自实线性)。

关键观察: \(f\) 复线性 \(\Rightarrow\) \(f(i x) = i f(x) = i (u(x) + i u'(x)) = -u'(x) + i u(x)\)。同时 \(f(i x) = u(i x) + i u'(i x)\), 取实部得 \(u(i x) = -u'(x)\), 即 \(u'(x) = -u(i x)\)。所以

\[ f(x) = u(x) - i u(i x). \]

\(f\) 完全 由它的实部 \(u\) 决定。这是复 \(\to\) 实归约的关键。

步骤 2: 把 \(X\) 视为实 向量空间 (忘记复结构, 只保留实标量)。\(M\) 也是实子空间。\(u : M \to \mathbb{R}\) 是实线性, 而 \(|u(x)| \le |f(x)| \le p(x)\) (实部不超过模, \(p\) 是半范故实)。所以 \(u \le p\)。由实 Hahn-Banach (13.1, 这里 \(p\) 半范 \(\Rightarrow\) 次线性), 存在实线性 \(U : X \to \mathbb{R}\) 延拓 \(u\), 且 \(U \le p\) 在全 \(X\) 上。

步骤 3: 反向构造复线性延拓。令 \(F(x) = U(x) - i U(i x)\)。验证 \(F\) 复线性:

  • 实加性: \(F(x + y) = U(x + y) - i U(i x + i y) = U(x) + U(y) - i (U(i x) + U(i y)) = F(x) + F(y)\)。
  • 实数乘: 显然。
  • 乘 \(i\): \(F(i x) = U(i x) - i U(-x) = U(i x) + i U(x) = i \cdot (U(x) - i U(i x)) = i F(x)\) (用 \(U\) 实线性, \(U(-x) = -U(x)\))。复线性 \(\Leftrightarrow\) 实线性 + \(F(i x) = i F(x)\), 验证完。

\(F\) 延拓 \(f\): \(F|_M(x) = U(x) - i U(i x) = u(x) - i u(i x) = f(x)\) (因为 \(U\) 在 \(M\) 上等于 \(u\), \(M\) 复子空间故 \(i x \in M\))。

最后保支配 \(|F(x)| \le p(x)\)。这是最 tricky 的一步——\(U \le p\) 给的是实部 的支配, 怎么推模 的支配?

设 \(F(x) = r e^{i \theta}\), \(r = |F(x)| \ge 0\)。则 \(e^{-i \theta} F(x) = r\) 是实数。复线性给 \(F(e^{-i \theta} x) = e^{-i \theta} F(x) = r\)。所以 \(F(e^{-i \theta} x) = r\) 是实数, 即 \(U(e^{-i \theta} x) = r\) (因为 \(F\) 实部 = \(U\), \(F\) 虚部 = \(-U(i \cdot)\), 但 \(F(e^{-i\theta} x)\) 是实数所以虚部 \(= 0\), 即 \(F\) 实部 \(= U(e^{-i\theta} x) = r\))。

由 \(U \le p\) 与 \(p\) 半范:

\[ |F(x)| = r = U(e^{-i \theta} x) \le p(e^{-i \theta} x) = |e^{-i \theta}| p(x) = p(x). \]

\(\square\)

请你停下来体会"实部 / 虚部" 这个把戏: 复 Hahn-Banach 不是单独证, 而是把"看成实空间" 用实 Hahn-Banach 给个实部延拓, 再"反推" 复线性。整个论证用到复结构的地方只有一处: 步骤 3 里 \(F(i x) = i F(x)\) 的验证, 还有最后那个 \(e^{-i \theta}\) 旋转技巧。"乘 \(i\) 旋转出实数, 用 \(U\) 估出实数, 旋回去得模"——这个手法叫"极坐标技巧", 是复线性泛函估计的标准动作。

13.6 三条核心推论 (在赋范空间里)

把 Hahn-Banach 应用到赋范空间, 取 \(p(x) = \|x\|\), 立刻给三条具体推论, 它们是日常使用 Hahn-Banach 的 90% 场景。

推论 13.4 (赋范延拓)。设 \(X\) 赋范空间, \(M \subset X\) 线性子空间, \(f \in M^* = \mathcal{B}(M, \mathbb{C})\) (\(M\) 上有界线性泛函)。则存在 \(F \in X^*\), \(F|_M = f\), 且 \(\|F\|_{X^*} = \|f\|_{M^*}\)。

证明。取 \(p(x) = \|f\|_{M^*} \cdot \|x\|\), 这是半范 (复) / 次线性 (实)。在 \(M\) 上 \(|f(x)| \le \|f\|_{M^*} \cdot \|x\| = p(x)\)。由 Hahn-Banach (实/复对应版本) 延拓 \(F\), \(|F(x)| \le p(x) = \|f\|_{M^*} \|x\|\) 即 \(\|F\|_{X^*} \le \|f\|_{M^*}\)。反向 \(\|F\|_{X^*} \ge \|f\|_{M^*}\) 显然 (\(F\) 延拓 \(f\) 不能使范数变小, 因为 \(M\) 上的上确界 \(\le\) \(X\) 上的上确界)。所以 \(\|F\| = \|f\|\)。\(\square\)

这条推论是 Hahn-Banach 在赋范情形最常用 的形式: 子空间上的有界泛函可以保范数延拓到全空间。下面两条都是它的特殊化。

推论 13.5 (看见每个 \(x\), 即"识别")。设 \(X\) 赋范空间, \(x_0 \in X\), \(x_0 \neq 0\)。则存在 \(\varphi \in X^*\) 使 \(\|\varphi\| = 1\) 且 \(\varphi(x_0) = \|x_0\|\)。

证明。在一维子空间 \(M = \mathbb{C} x_0\) 上 (复情形, 实情形类似) 定义 \(f(\lambda x_0) = \lambda \|x_0\|\)。这是线性, 且

\[ |f(\lambda x_0)| = |\lambda| \|x_0\| = \|\lambda x_0\|, \]

所以 \(\|f\|_{M^*} = 1\), 同时 \(f(x_0) = \|x_0\|\)。由 13.4, 延拓到 \(\varphi \in X^*\), \(\|\varphi\| = 1\), \(\varphi(x_0) = \|x_0\|\)。\(\square\)

这条推论给出"对偶式范数公式":

\[ \|x\| = \sup_{\|\varphi\| \le 1} |\varphi(x)| = \max_{\|\varphi\| \le 1} |\varphi(x)|. \]

(\(\sup\) 由 \(|\varphi(x)| \le \|\varphi\| \|x\| \le \|x\|\); \(\sup\) 达到由 13.5 给的特定 \(\varphi\), 故 \(\sup = \max\)。) 这个公式让我们能把"计算 \(x\) 的范数" 转化为"对所有 \(\varphi\) 取上确界"——这种转化在估计、对偶、弱收敛 (第 17 章) 里反复用到。

另外, 13.5 也给出"分点" 的能力: 两个不同向量 \(x, y\), 取 \(z = x - y \neq 0\), 由 13.5 存在 \(\varphi\) 使 \(\varphi(z) = \|z\| > 0\), 即 \(\varphi(x) - \varphi(y) > 0\), 特别 \(\varphi(x) \neq \varphi(y)\)。"对偶空间足够大, 能区分任意两个不同向量"。

推论 13.6 (闭子空间识别). 设 \(X\) 赋范, \(M \subset X\) 线性子空间, \(x_0 \in X\) 满足

\[ d := \mathrm{dist}(x_0, M) = \inf_{m \in M} \|x_0 - m\| > 0 \]

(即 \(x_0 \notin \overline{M}\))。则存在 \(\varphi \in X^*\) 使

  • (a) \(\varphi|_M = 0\) (湮没 \(M\));
  • (b) \(\varphi(x_0) = d\);
  • (c) \(\|\varphi\| = 1\)。

证明。在子空间 \(M' = M \oplus \mathbb{C} x_0\) 上 (注意 \(x_0 \notin M\) 由 \(d > 0\) 保证, 所以这是真"加" 一维) 定义

\[ f(m + \lambda x_0) = \lambda d, \quad m \in M, \lambda \in \mathbb{C}. \]

这条定义良好 (因为分解 \(m + \lambda x_0\) 是唯一的, 由 \(M' = M \oplus \mathbb{C} x_0\) 的直和性) 且线性。\(f|_M = 0\) (取 \(\lambda = 0\)), \(f(x_0) = d\) (取 \(m = 0, \lambda = 1\))。

估范数: 对 \(\lambda \neq 0\),

\[ \frac{|f(m + \lambda x_0)|}{\|m + \lambda x_0\|} = \frac{|\lambda| d}{\|m + \lambda x_0\|} = \frac{d}{\|m/\lambda + x_0\|} \le \frac{d}{d} = 1 \]

(用 \(\|m/\lambda + x_0\| = \|x_0 - (-m/\lambda)\| \ge \mathrm{dist}(x_0, M) = d\), 因为 \(-m/\lambda \in M\))。所以 \(\|f\|_{M'^*} \le 1\)。

反向: 取 \(m_n \in M\) 使 \(\|x_0 - m_n\| \to d\), 则

\[ \frac{|f(-m_n + x_0)|}{\|-m_n + x_0\|} = \frac{d}{\|x_0 - m_n\|} \to 1. \]

所以 \(\|f\|_{M'^*} = 1\)。

由 13.4 延拓 \(f\) 到 \(\varphi \in X^*\), \(\|\varphi\| = 1\), 且 \(\varphi|_M = 0, \varphi(x_0) = d\)。\(\square\)

这条推论是"闭子空间识别" 的核心: \(x_0 \notin \overline{M}\) 等价于存在湮没 \(M\) 但不湮没 \(x_0\) 的连续泛函。它的逆否更直观: \(x_0 \in \overline{M}\) 等价于"任何湮没 \(M\) 的泛函都湮没 \(x_0\)"——这是经典的"湮没条件"。

这条推论在多个场合用到:

  • 稠密性测试: \(M\) 在 \(X\) 中稠密 \(\Leftrightarrow\) \(\overline{M} = X\) \(\Leftrightarrow\) 唯一湮没 \(M\) 的连续泛函是 \(0\)。Weierstrass 逼近 (\(\mathcal{P}\) 在 \(C[0,1]\) 稠密) 的另一个证 是: 假设 \(\varphi \in C[0,1]^*\) 湮没所有多项式, 表示 \(\varphi\) 为 Riesz-Markov 测度 \(\mu\), 由矩问题 论证 \(\mu = 0\)。
  • 核 / 像的对偶: 算子 \(T : X \to Y\) 的像 \(T(X)\) 在 \(Y\) 中稠密 \(\Leftrightarrow\) 对偶算子 \(T^* : Y^* \to X^*\) 是单射 (\(T^* \varphi = 0 \Leftrightarrow \varphi \circ T = 0 \Leftrightarrow \varphi\) 湮没 \(T(X)\) \(\Leftrightarrow\) \(\varphi = 0\) 由 13.6)。这是第 15 章对偶论的基础。
  • 商空间嵌入: \(X / M\) 的元素 \([x_0]\) 非零 \(\Leftrightarrow\) \(x_0 \notin M\), 在 \(M\) 闭时 \(\Leftrightarrow\) 存在湮没 \(M\) 的连续泛函在 \(x_0\) 上非零。这是商空间对偶 \((X/M)^* \cong M^\perp\) (湮没 \(M\) 的泛函集合) 的根源。

13.7 总结：Hahn-Banach 的"形状" + 三大原理总览

这一章你看到了 Hahn-Banach 解析形式的形状:

核心机制: 一维延拓 + Zorn 升级。一维引理是真正"难" 的步骤 (用 \(p\) 的次加性闭合一条不等式), Zorn 把它"叠" 到全空间。

风味跟前两章不同: 不用 Baire, 不需要完备 (赋范空间够, 不必 Banach)。它用选择公理。

核心后果: 对偶空间 \(X^*\) 足够大——能"看见" 每一个非零向量、能"识别" 每一对不同向量、能"分离" 闭子空间外的点。这三件事是对偶论 的全部根基。

到这里, 我们已经看完了三大原理的全部:

  • 第 11 章 UBP: 逐点有界 \(\Rightarrow\) 一致有界 (Baire 给"含开球", 线性升级)。
  • 第 12 章 OMT/CGT: 满射 \(\Rightarrow\) 开映射; 图像闭 \(\Rightarrow\) 算子有界 (Baire + 几何级数)。
  • 第 13 章 Hahn-Banach (解析): 子空间泛函 \(\Rightarrow\) 全空间延拓 (Zorn + 一维核心引理)。

它们一起构成 Banach 空间论的"三块基石"。前两条让"代数等式自动升级为拓扑等式"; 第三条让"对偶论可用"。这本书后面所有的"高定理"——Riesz 表示 (ch16)、Banach-Alaoglu (ch18)、谱定理 (ch21)、Sobolev 嵌入 (ch23)——都不可避免地 用到这三大原理之一。

下一章 (第 14 章) 我们做 Hahn-Banach 的几何形式: 把"延拓泛函" 翻译成"分离凸集 / 分离超平面"。这是同一条定理的几何视角, 它是凸分析、最优化、变分方法的语言基础。准备好。
