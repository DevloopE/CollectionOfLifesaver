第 14 章 Hahn–Banach 几何形式与凸集分离

第 13 章把 Hahn–Banach 当成一条延拓定理。这一章把同一条定理翻成几何语言：两个不相交的凸集之间总能找到一个"超平面"把它们分开。这个画面是凸分析、凸优化、博弈论、经济学最优化、PDE 变分法的共同根——所有"对偶""支撑""拉格朗日"故事的几何源头都在这里。

核心直觉：超平面 \(= \{x : f(x) = c\}\)（某个线性泛函的水平集）。"分离两个集" \(=\) "存在 \(f\) 与 \(c\) 使一个集里所有点 \(f \le c\)，另一个集里所有点 \(f \ge c\)"。"分离" \(\iff\) "找到 \(f\)"，所以 Hahn–Banach 提供的"线性泛函丰富性"立即翻译成"分离丰富性"。

  • 把"超平面""半空间""支撑超平面"几何化。
  • Minkowski 泛函 \(p_C\)：把任意凸吸收集变成次线性控制函数，连接 13 章的解析形式与本章的几何形式。
  • 第一分离定理：凸开集与凸集分离。
  • 第二分离定理（强分离）：凸闭集与凸紧集严格分离。
  • 支撑超平面定理：凸集的边界点都有支撑超平面。
  • 三个直接应用：Lagrange 对偶（凸优化）、Hahn–Banach 间接证存在性（变分法）、Krein–Milman 定理（凸紧集是其极点的闭凸包）的预告。

14.1 超平面、半空间、支撑超平面

设 \(X\) 实赋范空间，\(f \in X^*\) 非零，\(c \in \mathbb{R}\)。集合

\[ H = \{x : f(x) = c\} \]

是 \(X\) 的超平面（"维数比 \(X\) 少 1" 的仿射子空间）。它把 \(X\) 分成两个开半空间

\[ H^+ = \{x : f(x) > c\}, \qquad H^- = \{x : f(x) < c\}, \]

以及它们的闭包（\(\ge\) 与 \(\le\)）。

在欧氏 \(\mathbb{R}^n\) 中超平面就是普通"直线 / 平面"的一般化。在 Hilbert 空间里 Riesz 表示让每个 \(f\) 对应一个向量 \(v\)，超平面就是 \(\{x : \langle x, v \rangle = c\}\)，几何上它是垂直于 \(v\) 的"切片"。

一个集合 \(C\) 在超平面 \(H = \{f = c\}\) 的一侧，是指 \(C \subset H^+ \cup H = \{f \ge c\}\) 或 \(C \subset H^- \cup H\)。

两个集合 \(A, B\)被超平面 \(H\) 分离，是指它们在 \(H\) 的不同侧。当不等号严格时（\(A \subset H^+\)、\(B \subset H^-\)），叫严格分离。

14.2 Minkowski 泛函：把凸集变次线性

怎样把"凸"翻译成"次线性"——这是连接几何与解析 Hahn–Banach 的关键。答案是 Minkowski 泛函。

设 \(C \subset X\) 是吸收（每个 \(x\) 都存在 \(\lambda > 0\) 使 \(\lambda x \in C\)）的凸集，包含原点。Minkowski 泛函：

\[ \boxed{\ p_C(x) := \inf \{t > 0 : x \in t C\}.\ } \]

几何意义：\(p_C(x) = 1/\) "把 \(x\) 缩到落入 \(C\) 边界所需的最小倍数"。对吸收集这个 \(\inf\) 总是有限。

关键性质：

  • (MK1) 次线性：\(p_C(\alpha x) = \alpha p_C(x)\) 对 \(\alpha > 0\)，\(p_C(x + y) \le p_C(x) + p_C(y)\)；
  • (MK2) 内点刻画：若 \(C\) 还是开集，则 \(C = \{x : p_C(x) < 1\}\)；
  • (MK3) 范数情形：若 \(C = B(0, 1)\) 是单位球，则 \(p_C(x) = \|x\|\)；范数本身就是单位球的 Minkowski 泛函。

这条 Minkowski 泛函的次线性 \((MK1)\) 让我们能用 13 章解析 Hahn–Banach。它就是从凸到次线性的"翻译器"。

14.3 第一分离定理

定理（第一分离 / Hahn–Banach 几何形式 I）。设 \(X\) 是实赋范空间，\(A, B \subset X\) 是非空、凸、不相交的子集，且 \(A\) 开。则存在 \(f \in X^*\) 与 \(c \in \mathbb{R}\) 使

\[ \boxed{\ f(a) < c \le f(b), \quad \forall a \in A, b \in B.\ } \]

注意：分离是严格的左侧但不要求严格右侧（\(B\) 可能在超平面上）。要求 \(A\) 开，是为了 Minkowski 泛函可用。

证明思路：考虑 \(C = A - B = \{a - b : a \in A, b \in B\}\)。\(C\) 是凸开集（凸开 \(\pm\) 凸开 = 凸开），且 \(0 \notin C\)（因为 \(A \cap B = \emptyset\)）。要分离 \(A\) 与 \(B\) \(\Leftrightarrow\) 找 \(f\) 使 \(f|_C < 0\) \(\Leftrightarrow\) \(f|_C \le 0\) 且 \(f \neq 0\)。

取 \(x_0 \in C\)。令 \(C' = C - x_0\)（凸开，含原点）。Minkowski 泛函 \(p_{C'}\) 次线性，\(C' = \{p_{C'} < 1\}\)。\(0 \notin C \Rightarrow -x_0 \notin C' \Rightarrow p_{C'}(-x_0) \ge 1\)。

在一维子空间 \(M = \mathbb{R} x_0\) 上定义 \(f_0(t x_0) = -t \cdot p_{C'}(-x_0)\)。验证 \(f_0 \le p_{C'}\)：对 \(t > 0\)，\(f_0(t x_0) = -t p_{C'}(-x_0) < 0 \le p_{C'}(t x_0)\)；对 \(t < 0\)，\(f_0(t x_0) = -t p_{C'}(-x_0) = |t| p_{C'}(-x_0) = p_{C'}(-t \cdot (-x_0))\)，由次线性 \(\le p_{C'}((-t)(-x_0)) = p_{C'}(t x_0)\)。

解析 Hahn–Banach 给延拓 \(f \in X^*\)，\(f \le p_{C'}\) 在全空间。所以 \(f|_{C'} \le p_{C'}|_{C'} < 1\)，且 \(f(-x_0) \ge p_{C'}(-x_0) \ge 1\)，即 \(f(x_0) \le -1\)。

把 \(C' = C - x_0\) 还回去：\(f|_C = f|_{C'} + f(x_0) < 1 + (-1) = 0\)。即 \(f(a) - f(b) < 0\) 对 \(a \in A, b \in B\)，得分离。结束。

(这条证明的细节很多——重点不是逐步背下，而是看见 Minkowski 泛函如何把"凸不相交"翻译成"次线性 + 解析 Hahn–Banach 适用"。)

14.4 第二分离定理（强分离）

第一分离允许 \(B\) 在超平面上。下面这个版本要求严格分离：

定理（第二分离 / 强分离）。设 \(X\) 实 Banach 空间，\(A, B\) 非空凸不相交子集，\(A\) 闭，\(B\) 紧。则存在 \(f \in X^*\)、\(c_1 < c_2\) 使

\[ \boxed{\ f(a) \le c_1 < c_2 \le f(b), \quad \forall a \in A, b \in B.\ } \]

证明思路：\(A - B\) 闭（紧 + 闭 = 闭，由有限维或 Banach 空间论证）；\(0 \notin A - B\) \(\Rightarrow\) 存在 \(\varepsilon > 0\) 使 \(B(0, \varepsilon) \cap (A - B) = \emptyset\)。把第一分离定理用在 \(A' = (A - B) + B(0, \varepsilon/2)\) 与 \(\{0\}\) 上。结束。

这条强分离常被叫做"Mazur 分离定理"。它的几何画面非常清楚：闭凸集 \(A\) 和它外面的紧凸集 \(B\) 之间一定有一道严格的"墙"——左边一切 \(\le c_1\)，右边一切 \(\ge c_2\)。

14.5 支撑超平面定理

这是凸集"边界几何"的基本定理：

定理（支撑超平面）。设 \(C \subset X\) 凸开非空，\(x_0 \in \partial C\)（边界点）。则存在 \(f \in X^*\)、\(c \in \mathbb{R}\) 使 \(f(x_0) = c\) 且 \(f(x) < c\) 对 \(x \in C\)。"\(\{f = c\}\)" 叫 \(C\) 在 \(x_0\) 的支撑超平面。

证明：把第一分离用到 \(A = C\) 与 \(B = \{x_0\}\)。\(B\) 是单点 \(\Rightarrow\) 凸；\(A\) 开凸；\(x_0 \in \partial C \Rightarrow x_0 \notin C \Rightarrow A \cap B = \emptyset\)。第一分离给 \(f, c\) 使 \(f(a) < c \le f(x_0)\)。把 \(c = f(x_0)\) 取到边界即可。

支撑超平面在凸优化、PDE 变分法里反复出现。它是 \(C\) 在每个边界点处"切向 + 法向"的几何替代物——在不要求光滑的凸集上仍然给"支撑方向"。

14.6 三个应用

应用 1：凸优化中的 Lagrange 对偶。考虑约束最小化

\[ \min_x f(x) \quad \text{s.t.} \quad g(x) \le 0, \]

其中 \(f, g\) 凸。可行集 \(\{x : g(x) \le 0\}\) 凸。在 \((f, g)\)-平面上看"可达像"

\[ A = \{(t, s) : \exists x, f(x) \le t, g(x) \le s\} \]

是凸集；目标是看 \(A\) 在 \(s = 0\) 截面的最小 \(t\)——这就是 \(\min f\) 对约束 \(g \le 0\) 的值。如果 \(A\) 与 \(\{(t^*, 0) : t < p^*\}\) 不相交（\(p^*\) 是真值），第一分离给一个超平面 \(\alpha t + \beta s = c\)。这条超平面就是 Lagrange 函数 \(L(x, \beta) = f(x) + \beta g(x)\) 的对偶变量解释。整个 Lagrange 对偶理论 = 凸集分离的代数包装。

这就是凸优化课里"为什么 Lagrange 乘数是真定理而不是巧合"的根本原因。Hahn–Banach 几何形式提供了它的存在性。

应用 2：变分法中的存在性。设 \(X\) 是 Banach 空间，\(F : X \to \mathbb{R}\) 凸下半连续，要证某个变分问题有解。常用策略：把"最小化序列" \(\{x_n\}\) 看成 \(X^*\) 上某种 Hahn–Banach 几何的"试探函数集"，用第二分离定理在某个紧致环境里找到极小元素。详见第 23 章变分法入门。

应用 3：Krein–Milman 定理（预告）。这是凸分析的高峰之一：

定理（Krein–Milman）。Banach 空间中的凸紧集 \(K\) 等于其极点（不能写成 \(K\) 中两点的非平凡凸组合的点）的闭凸包。

证明完全靠 Hahn–Banach 几何形式 + Zorn 引理。直观上：每个支撑超平面会切下一片"凸面"，这片凸面的极点也是 \(K\) 的极点；逐次切下去得到所有极点。详细在附录 A。

Krein–Milman 是为什么"线性规划的最优解一定在顶点上"在无穷维变体里依然真——这是博弈论 von Neumann minimax 定理的几何根。

本章把 Hahn–Banach 的几何面挑明。第 13、14 两章合起来给了泛函分析最深刻的对偶性原则：一个空间 \(X\) 和它的对偶 \(X^*\) 通过 Hahn–Banach 互相照镜子。任何 \(X\) 上的几何问题（点、子空间、凸集、距离、紧致）都对应一个 \(X^*\) 上的代数问题（线性泛函、零化、分离、保范延拓）。

下一章我们正式开始 \(X^*\) 这个"镜中世界"——研究对偶空间的结构、它和 \(X\) 的关系、以及它在分析里反复出现的若干代数等距同构。
