第 14 章 Hahn-Banach 几何形式与凸集分离

上一章我们证完了 Hahn-Banach 延拓定理：子空间上的有界线性泛函可以延拓到全空间，且范数不变。那一章里 HB 是个"代数" 工具——给一个泛函加几个变量。这一章我把同一个定理翻译成几何 语言。翻译完之后你会发现：HB 真正的故事不是延拓，是"两个不相交的凸集中间总有一面墙"。

让我从一个具体场景开始。

设 \(C \subset \mathbb{R}^2\) 是个凸的开圆盘——比方说 \(\{(x, y) : x^2 + y^2 < 1\}\)。再取一个不在 \(C\) 里的点 \(p = (2, 0)\)。请你画一下。你直观就知道：可以画一条直线，让 \(C\) 完全在直线左边、\(p\) 在直线右边。比如 \(x = 1.5\) 这条竖线就行。

现在把这件事抽象一下。

  • "\(C\) 在直线左边" 翻译成：存在线性泛函 \(f(x, y) = x\) 和实数 \(\alpha = 1.5\)，使 \(f \le \alpha\) 在 \(C\) 上。
  • "\(p\) 在直线右边" 翻译成：\(f(p) > \alpha\)。
  • 那条直线本身就是 \(\{(x, y) : f(x, y) = \alpha\}\)——一个超平面。

这就是分离定理的几何雏形：不相交凸集 → 中间夹一个超平面，使一边的所有点 \(f \le \alpha\)、另一边 \(f \ge \alpha\)。

这个画面在 \(\mathbb{R}^2\) 完全显然。本章要做的事是把它推到无穷维 Banach 空间。在无穷维"画一条线" 不再可视，但线性泛函 + 实数 这套代数描述照搬。证明的关键是 HB——延拓定理在这里以"把一个一维上构造好的线性形扩到全空间" 的姿态出场。整套故事自然衔接到下一系列：凸优化的 Lagrange 对偶、变分法的能量极小化、博弈论的极小极大、经济学里的均衡价格。它们的共同根都在这章。

14.1 超平面、半空间、支撑超平面

先把语言备齐。设 \(X\) 是实赋范空间，\(f \in X^*\) 是非零有界线性泛函，\(\alpha \in \mathbb{R}\)。

超平面：

\[ H = H_{f, \alpha} := \{x \in X : f(x) = \alpha\}. \]

一个非零线性泛函的"水平集"。在 \(\mathbb{R}^n\) 里这就是一张 \((n-1)\) 维子流形（直线、平面、…）；在 Banach 空间里它仍是个余维 1 的仿射 子集——是 \(\ker f\) 平移 \(\alpha\) 的一份。

引理（HB 推论一遍）：\(H_{f, \alpha}\) 是闭的当且仅当 \(f\) 连续。证明短得几乎透明：\(H = f^{-1}(\{\alpha\})\)，连续函数的闭集原像是闭。反过来如果 \(H\) 闭，可以证 \(\ker f\) 闭，进而 \(f\) 连续（因为线性泛函只有"连续 / 处处不连续" 两种命运——这条结论就是 13.4 节的内容，源于不连续线性泛函的核稠密）。

我们以后只关心 \(f \in X^*\) 给的超平面——它们是闭 超平面。

开半空间 / 闭半空间：

\[ H_{f, \alpha}^{<} := \{x : f(x) < \alpha\}, \qquad H_{f, \alpha}^{\le} := \{x : f(x) \le \alpha\}. \]

类似地有 \(H^>, H^\ge\)。开半空间是凸的、开的；闭半空间是凸的、闭的。

"超平面分离" 的两个变体。设 \(A, B \subset X\) 不相交。

  • 分离：存在 \(f \in X^*, f \neq 0\) 与 \(\alpha\) 使 \(f \le \alpha\) 在 \(A\) 上、\(f \ge \alpha\) 在 \(B\) 上。
  • 严格分离：存在 \(f, \alpha, \varepsilon > 0\) 使 \(f \le \alpha - \varepsilon\) 在 \(A\) 上、\(f \ge \alpha + \varepsilon\) 在 \(B\) 上。

严格分离强一档——它要求超平面不 触碰 \(A \cup B\)。第一分离（14.3）只给"分离"；第二分离（14.4）给"严格分离"，代价是更强假设。

支撑超平面。设 \(C \subset X\) 凸、\(x_0 \in \partial C\)（边界点）。一个超平面 \(H_{f, \alpha}\) 支撑 \(C\) 于 \(x_0\) 当且仅当：

  1. \(x_0 \in H\)（即 \(f(x_0) = \alpha\)），
  2. \(C \subset H^{\le}\)（整个 \(C\) 在 \(H\) 一侧）。

几何上：超平面"恰好顶住" 凸集 \(C\) 在 \(x_0\) 这点。支撑超平面是凸分析里到处出现的对象——次微分（subdifferential）、Lagrange 乘子、约束方向、KKT 条件——全是它的代数化。

14.2 Minkowski 泛函：从凸集到次线性

分离定理的证明要把"凸吸收集" 和"次线性泛函" 这两件事连起来。这一节做这个连接。这件事 Minkowski 1911 年就在用——他想给凸体一个"规范度量"，从这个度量自动复活 HB 的延拓机器。

设 \(C \subset X\) 凸、且吸收（absorbing）——即 \(0 \in \operatorname{int} C\)，等价地 \(\forall x \in X, \exists t > 0\) 使 \(t x \in C\)。

Minkowski 泛函 / 度量函数定义为

\[ \boxed{\ p_C(x) := \inf \{ t > 0 : x / t \in C \}.\ } \]

朴素直觉：\(p_C(x)\) 是把 \(x\) 缩到 \(C\) 里需要的最小缩放比。如果 \(C\) 是单位球，\(p_C(x) = \|x\|\)；如果 \(C\) 比单位球大，\(p_C(x) < \|x\|\)；如果 \(C\) 是个椭球或多面体，\(p_C\) 给一个"非各向同性的范数"。

关键性质。设 \(C\) 凸吸收。则

  1. \(p_C(x) \ge 0\) 总成立（定义里 \(t > 0\)）。
  2. 正齐次：\(p_C(\lambda x) = \lambda p_C(x)\) 对所有 \(\lambda \ge 0\)。
  3. 次可加：\(p_C(x + y) \le p_C(x) + p_C(y)\)。
  4. \(\{p_C < 1\} \subset C \subset \{p_C \le 1\}\)。如果 \(C\) 还是开的，\(C = \{p_C < 1\}\)。

第 (2)(3) 条把 \(p_C\) 钉成一个次线性 泛函——正是 HB 延拓定理（13.2 节）需要的那种"control"。这是为什么我们花一节写 Minkowski 泛函：它把"凸集" 翻译成了 HB 能吃的"次线性泛函"。

(1)(2) 直接验。(4) 来自吸收性 + 凸性的组合算（如果 \(p_C(x) < 1\) 则存在 \(t < 1\) 使 \(x/t \in C\)，再用凸组合 \(x = t \cdot (x/t) + (1-t) \cdot 0\) 得 \(x \in C\)）。我们重点证 (3)：

设 \(p_C(x) = a\)、\(p_C(y) = b\)。给任意 \(\varepsilon > 0\)，存在 \(s < a + \varepsilon, t < b + \varepsilon\) 使 \(x/s, y/t \in C\)。考虑 \((x + y)/(s + t)\)：

\[ \frac{x + y}{s + t} = \frac{s}{s + t} \cdot \frac{x}{s} + \frac{t}{s + t} \cdot \frac{y}{t}. \]

这是 \(x/s\) 和 \(y/t\)（都属于 \(C\)）的凸组合——\(C\) 凸所以仍在 \(C\) 里。于是 \(p_C(x + y) \le s + t < a + b + 2\varepsilon\)。\(\varepsilon\) 任意，结束。

请你停下来欣赏这步："\(C\) 凸" 这条假设在哪儿用了？正是上面那个凸组合。如果 \(C\) 不凸，这一步就崩——\(p_C\) 也不再次可加，HB 就用不上。分离定理只对凸集成立，凸性的代数化身就是这步凸组合。

顺便：(1)~(3) 没用 \(C\) 是开的或闭的，只要凸吸收。但 (4) 在 \(C\) 开和闭的版本里有微妙差别。后面证第一分离我们会用"\(C\) 开 ⟹ \(C = \{p_C < 1\}\)" 这一面，让"\(p_C(x) < 1 \Leftrightarrow x \in C\)" 成为干净的等价。

14.3 第一分离定理：凸开 + 凸不相交

现在证本章第一个主定理。

定理（第一分离定理 / Hahn-Banach 几何形式 I）。设 \(X\) 实赋范空间，\(A, B \subset X\) 非空、凸、不相交，且 \(A\) 开。则存在 \(f \in X^*, f \neq 0\) 与 \(\alpha \in \mathbb{R}\) 使

\[ f(a) < \alpha \le f(b), \quad \forall a \in A, b \in B. \]

等价地说：超平面 \(H_{f, \alpha}\) 把 \(A, B\) 分到两边，\(A\) 整个在开半空间 \(\{f < \alpha\}\) 里、\(B\) 整个在闭半空间 \(\{f \ge \alpha\}\) 里。

证明。分四步。

步骤 1：归约到"原点不在 \(C\) 里" 的情形。考虑 \(C := A - B = \{a - b : a \in A, b \in B\}\)。性质：

  • \(C\) 凸（两个凸集的 Minkowski 差仍凸——直接验），
  • \(C\) 开（\(C = \bigcup_{b \in B} (A - b)\)，开集的并仍开），
  • \(0 \notin C\)（因为 \(A \cap B = \emptyset\)）。

如果我们能找到 \(f \neq 0\) 使 \(f(c) < 0\) 对所有 \(c \in C\)（即 \(f\) 在 \(C\) 上严格负），那么 \(f(a) < f(b)\) 对所有 \(a \in A, b \in B\)。设 \(\alpha = \inf_{b \in B} f(b)\)，得到 \(f(a) < \alpha \le f(b)\) 的格式。所以问题归约为：给一个凸开集 \(C\)、\(0 \notin C\)，找 \(f \in X^*\) 使 \(f \le 0\) 在 \(C\) 上、\(f \neq 0\)。

（在这一归约里我们还需要 \(\alpha < \infty\)，即 \(f\) 在 \(B\) 上有下界。这条最后会自动给出来。）

步骤 2：把 \(C\) 平移成"含原点的吸收开凸"。\(C\) 是开凸但不含 0。任取一点 \(x_0 \in C\)。令 \(D := C - x_0\)。\(D\) 仍开凸，且含 0（因为 \(x_0 - x_0 = 0\)）。又 \(0 \in \operatorname{int} D\) 因 \(D\) 开，所以 \(D\) 吸收。

注意原 \(C\) 不含 0 翻译成：\(-x_0 \notin D\)。

步骤 3：在一维子空间上构造一个线性形。对 \(D\) 取 Minkowski 泛函 \(p := p_D\)。由 14.2，\(p\) 次线性。又 \(D\) 开 ⟹ \(D = \{p < 1\}\)。因 \(-x_0 \notin D\)，\(p(-x_0) \ge 1\)。

设 \(L = \mathbb{R} \cdot (-x_0)\)，定义 \(g : L \to \mathbb{R}\) 为

\[ g(t \cdot (-x_0)) := t \cdot p(-x_0). \]

\(g\) 在 \(L\) 上线性，且：

  • \(t \ge 0\) 时 \(g(t \cdot (-x_0)) = t p(-x_0) = p(t \cdot (-x_0))\) 正齐次使等号。
  • \(t < 0\) 时 \(g(t \cdot (-x_0)) = t p(-x_0) \le 0 \le p(t \cdot (-x_0))\)。

所以 \(g \le p\) 在 \(L\) 上。

步骤 4：用解析 HB 把 \(g\) 延拓到全空间。HB 延拓定理（13.2）说：在次线性 \(p\) 控制下的子空间线性形可延拓到全空间，仍受 \(p\) 控制。设延拓为 \(f : X \to \mathbb{R}\) 线性、\(f \le p\)、\(f|_L = g\)。

检验 \(f\) 满足要求：

  • \(f(-x_0) = g(-x_0) = p(-x_0) \ge 1 > 0\)，所以 \(f \neq 0\)。
  • 对 \(d \in D\)：\(p(d) < 1\)（因 \(D = \{p < 1\}\)），所以 \(f(d) \le p(d) < 1\)。即 \(f < 1\) 在 \(D\) 上。
  • 翻译回 \(C = D + x_0\)：\(f(c) = f(d) + f(x_0) < 1 + f(x_0)\) 对所有 \(c \in C\)。设 \(\beta = 1 + f(x_0)\)，则 \(f < \beta\) 在 \(C\) 上。
  • 而我们要的是 \(f \le 0\) 在 \(C\) 上。这步还得再调一下——其实更干脆是直接代回 \(C = A - B\)。

清算一下。\(C = A - B\)，\(f < \beta\) 在 \(C\) 上 ⟺ \(f(a) - f(b) < \beta\) 对 \(a \in A, b \in B\) ⟺ \(f(a) < f(b) + \beta\)。又我们前面平移用了 \(D = C - x_0\)，等价于把 \(\beta\) 选成 \(\sup_C f\)（取 \(\beta = \sup_C f\)，则 \(f \le \beta\) 在 \(C\) 上，事实上 \(f < \beta\) 因 \(C\) 开）。这样 \(\sup_{a \in A} f(a) - \inf_{b \in B} f(b) \le \beta < \infty\)，于是 \(\sup_{a \in A} f(a) \le \inf_{b \in B} f(b)\) 处可以取一个 \(\alpha\) 夹在中间，再用 \(A\) 开 + \(f\) 连续推出 \(f(a) < \alpha\)（开集上线性泛函上确界不达到），得到

\[ f(a) < \alpha \le f(b), \quad \forall a \in A, b \in B. \]

剩一件事：\(f\) 是否连续？HB 延拓出来的 \(f\) 一开始只是线性的——但 \(f \le p\) 加 \(p\) 在 0 邻域有界（因 \(0 \in \operatorname{int} D\)）合起来给 \(f\) 在 0 邻域有界，所以 \(f\) 连续——\(f \in X^*\)。这步是次线性 control 自动给"连续" 的标准论证（详细推 13.3 节里有）。

证毕。请你停下来回看证明的脉络：凸开不相交（几何条件） → \(A - B\) 凸开不含 0 → 平移到含 0 → Minkowski 泛函（凸 → 次线性）→ 解析 HB 延拓 → 得分离泛函。整个证明就是把"几何分离" 翻译成"代数延拓" 的链子。Minkowski 泛函是这条链子里的关键齿轮——没有它，HB 不知道该控制谁。

14.4 第二分离定理：凸闭 + 凸紧 → 严格分离

第一分离给的是 \(<\) 和 \(\le\) 的不对称分隔。如果两个集都闭，能不能得到严格 分离 \(f(a) < \alpha < \beta < f(b)\)？答案是：要付一点代价——其中一个集必须紧。

定理（第二分离定理 / 强分离）。设 \(X\) 实 Banach 空间，\(A, B \subset X\) 非空凸不相交，\(A\) 闭、\(B\) 紧。则存在 \(f \in X^*, f \neq 0\) 与 \(\alpha < \beta\) 使

\[ f(a) \le \alpha < \beta \le f(b), \quad \forall a \in A, b \in B. \]

等价地说：超平面 \(H_{f, (\alpha + \beta)/2}\) 严格 把两集分到两边——超平面本身不与任何一集相交。

证明。考虑 \(C = A - B\)。两条性质：

  1. \(C\) 凸。
  2. \(C\) 闭。这步用到 \(B\) 紧——任取 \(c_n = a_n - b_n \to c\)，因 \(B\) 紧，\(b_n\) 有收敛子列 \(b_{n_k} \to b \in B\)，于是 \(a_{n_k} = c_{n_k} + b_{n_k} \to c + b \in A\)（因 \(A\) 闭），故 \(c = (c + b) - b \in A - B = C\)。如果 \(B\) 只是闭不紧，这步会失败——闭 - 闭未必闭，需要紧来保证子列收敛。
  3. \(0 \notin C\)（\(A \cap B = \emptyset\)）。

所以 \(C\) 是闭凸不含 0 的集合。"\(0 \notin C\) + \(C\) 闭" 给一个具体距离 \(\delta := d(0, C) > 0\)（因 \(C\) 闭，\(\inf_{c \in C} \|c\|\) 在闭集上严格正——这是闭集到点距离不为零的标准事实）。

令 \(U := \{x \in X : \|x\| < \delta\}\)，开球。\(U\) 凸开，且 \(U \cap C = \emptyset\)（任何 \(c \in C\) 满足 \(\|c\| \ge \delta\)）。

对 \(U, C\) 用第一分离定理（\(U\) 开、\(C\) 凸、二者凸不相交）：存在 \(f \in X^*, f \neq 0\) 与 \(\gamma\) 使

\[ f(u) < \gamma \le f(c), \quad \forall u \in U, c \in C. \]

关键：\(U\) 是关于 0 对称的开球（含 0），所以 \(f(U) = (-\|f\| \delta, \|f\| \delta)\)（线性泛函在球上是个对称区间）。\(f < \gamma\) 在 \(U\) 上 ⟹ \(\|f\| \delta \le \gamma\)，即 \(\gamma \ge \|f\| \delta > 0\)。所以

\[ f(c) \ge \|f\| \delta > 0, \quad \forall c \in C = A - B. \]

翻译：\(f(a) - f(b) \ge \|f\| \delta > 0\) 对所有 \(a \in A, b \in B\)。即

\[ \inf_{a \in A} f(a) \ge \sup_{b \in B} f(b) + \|f\| \delta. \]

取 \(\alpha = \sup_{b \in B} f(b) + \|f\| \delta / 3\)、\(\beta = \alpha + \|f\| \delta / 3\)。等等——做反了，应该交换：注意我们记号让 \(f(a) > f(b)\)，所以严格说 \(f\) 在 \(B\) 上小、在 \(A\) 上大。换成

\[ f(b) \le \sup_B f =: s, \qquad f(a) \ge s + \|f\| \delta. \]

取 \(\alpha = s + \|f\| \delta / 3, \beta = s + 2 \|f\| \delta / 3\)，则

\[ f(b) \le s < \alpha < \beta < s + \|f\| \delta \le f(a). \]

这是严格分离。证毕。

请你停下来对比第一分离和第二分离的假设 与结论：

  • 第一分离：\(A\) 开 + 凸不相交 → \(f(a) < \alpha \le f(b)\)（不等号是非对称的 \(<\) 与 \(\le\)）。
  • 第二分离：\(A\) 闭 + \(B\) 紧 + 凸不相交 → \(f(a) \le \alpha < \beta \le f(b)\)（严格分离，超平面不碰任何一集）。

"开 vs 闭+紧" 的取舍贯穿凸分析：开集让 Minkowski 泛函好用，闭+紧让差集闭、距离严格正。两个版本各有用场——第一个适合原子的存在性论证（例如证支撑超平面必存在），第二个适合数值意义下的 separation（例如证最小范数解存在唯一）。

顺带：如果两集都只是闭（不要求紧），严格分离可能失败。经典反例：在 \(\mathbb{R}^2\) 里 \(A = \{(x, y) : y \ge e^x\}\)（指数曲线上方）、\(B = \{(x, y) : y \le 0\}\)（下半平面）。两集都闭、凸、不相交，但没有任何超平面严格分离它们——它们沿 \(x \to -\infty\) 越来越接近。这是为什么"\(B\) 紧" 这条假设不能去掉。

14.5 支撑超平面定理

第二分离的一个直接推论是支撑超平面的存在性。

定理（支撑超平面）。设 \(X\) 实 Banach 空间，\(C \subset X\) 凸闭，\(x_0 \in \partial C\)（\(C\) 的边界）。则存在 \(f \in X^*, f \neq 0\) 使

\[ f(x_0) = \sup_{c \in C} f(c). \]

即超平面 \(H_{f, f(x_0)}\) 在 \(x_0\) 处支撑 \(C\)。

证明。\(x_0 \in \partial C\) 意味着 \(x_0 \notin \operatorname{int} C\)。设 \(U := \operatorname{int} C\)。两件事：\(U\) 凸开（凸集内部凸），\(\{x_0\}\) 凸紧，\(U \cap \{x_0\} = \emptyset\)。用第一分离（更适合开集这边的版本）得 \(f \in X^*, f \neq 0\) 与 \(\alpha\) 使 \(f(u) < \alpha \le f(x_0)\) 对所有 \(u \in U\)。

把不等号紧化到 \(C\) 整体：\(\overline{U} = \overline{C}\)（凸集内部的闭包等于其闭包，对内部非空的凸集成立）= \(C\)（\(C\) 闭）。所以 \(f \le \alpha\) 在 \(C\) 上（连续性）；又 \(f(x_0) \ge \alpha\)；这两条合起来 \(f(x_0) = \alpha = \sup_C f\)。证毕。

支撑超平面定理 是凸分析的主轴定理 之一。它给凸集"每个边界点都有一面支撑墙"——这是凸集"光滑性" 的代数表达。它直接给两个分支：

  • 次微分（subdifferential）：凸函数 \(f\) 在某点 \(x_0\) 的次微分 \(\partial f(x_0)\) 是"支撑 epigraph 的所有线性形" 的集合。支撑超平面定理保证这集合非空——这是凸优化里 Lagrange 乘子存在性的原始形式。
  • 对偶锥与极锥：凸锥 \(K\) 的极锥 \(K^\circ = \{f : f \le 0 \text{ 在 } K\}\)。支撑定理保证它足够丰富，足以表达 \(K\) 的所有"约束方向"。

14.6 三个应用

这一节给三条本章工具的"应用样例"，让你看到分离定理在数学的哪些角落落地。

应用 1：Lagrange 对偶 = 凸集分离的代数化。考虑约束极小化问题

\[ \min_{x} f(x), \qquad \text{s.t. } g_i(x) \le 0, \ i = 1, \ldots, m, \]

其中 \(f, g_i\) 凸。Lagrange 函数

\[ L(x, \lambda) = f(x) + \sum_i \lambda_i g_i(x), \qquad \lambda \ge 0. \]

对偶问题 \(\max_{\lambda \ge 0} \inf_x L(x, \lambda)\)。著名的强对偶性（在 Slater 条件下）说原问题最优值等于对偶问题最优值。

这条强对偶本质 是分离定理。考虑可达集

\[ A := \{(g_1(x), \ldots, g_m(x), f(x)) + (\mathbb{R}^m_{\ge 0} \times \mathbb{R}_{\ge 0}) : x \in X\} \subset \mathbb{R}^{m+1}. \]

\(A\) 凸（\(f, g_i\) 凸；加非负锥仍凸）。设原问题最优值 \(p^*\)。"原问题最优值大于 \(p^*\)" 翻译成 \(\{(0, \ldots, 0, p^* - \varepsilon)\} \cap A = \emptyset\) 对小 \(\varepsilon > 0\)。用第二分离得一个 \((m+1)\)-维线性形 \(\lambda \cdot u + \mu \cdot v\) 把这两个凸集分开。该线性形的系数就是 Lagrange 乘子。强对偶性 = 分离超平面的代数化。

这条对应让你以后看任何"约束优化 + 对偶" 的故事时，都能在脑子里把它翻译成"哪两个凸集，被哪面墙分开"。从 SVM（支持向量机里 Lagrange 乘子是支撑超平面的法向量）到经济学一般均衡（价格向量是支撑超平面的法向）到 PDE 变分约束（接触问题里 Lagrange 乘子是接触力），全是同一条几何直觉。

应用 2：变分法的存在性。设 \(I : X \to \mathbb{R} \cup \{+\infty\}\) 是个凸下半连续泛函（典型例子：弹性能量、Dirichlet 能量），\(K \subset X\) 凸闭。问 \(\inf_K I\) 是否达到。

朴素方法：取一极小化序列 \(x_n \in K, I(x_n) \to \inf_K I\)。希望 \(x_n\) 有收敛子列、子列极限取到 inf。无穷维里 \(x_n\) 不一定有强收敛子列（ch1.1 节"单位球不紧"），但弱收敛子列有（ch18 Banach-Alaoglu）。问题剩下：弱极限是否仍在 \(K\) 里、\(I\) 在弱极限处是否达到 \(\inf\)。

"\(K\) 弱闭" 这条是分离定理的直接推论：凸闭集自动弱闭。证明就是支撑超平面：\(K\) 凸闭，则任何 \(x_0 \notin K\) 可被超平面与 \(K\) 严格分离（第二分离）；这条性质自动给"\(K\) 在弱拓扑下也是闭的"。详见 ch17 关于强弱拓扑的讨论。

这一观察让整个变分方法 落脚：凸 + 闭 自动给弱闭，弱闭 + 弱紧（Banach-Alaoglu 在自反空间 + 强制性）自动给最小元存在。这是 Sobolev 空间上 PDE 变分理论的基础结构。

应用 3：Krein-Milman 预告。\(\mathbb{R}^n\) 里凸紧多面体是其顶点的凸包。Krein-Milman 把这件事推广到无穷维：Banach 空间里凸紧集是其极点（extreme points）的闭凸包。证明的核心步骤就是支撑超平面——每个边界点处取支撑超平面，超平面与凸紧集的交是更低维的凸紧面，归纳下去得到极点。

这是泛函分析里的另一条主线（ch19 / ch24 会有更详细的版本）。它让你能在抽象空间里"找出顶点"——这是在概率论（极点分解 \(\Leftrightarrow\) 测度的分解）、量子力学（纯态 = 密度算子的极点）、优化（线性规划最优在顶点取得）等地方都基础的工具。

14.7 一句临别话

HB 延拓定理（ch13）说"线性泛函足够多"。本章把这件事翻成几何："不相交凸集中间总有一面墙"。请你把ch13 的 HB 和这一章并起来看：同一个定理，两副面孔。

  • ch13 的 HB：把一根定义在子空间上的线段往全空间方向接长，长度不变。
  • ch14 的 HB：把两块不相交的凸集分开，中间夹一面超平面。

它们等价。从延拓证分离我们刚做完；反过来从分离证延拓也只需要把"延拓 \(g\) 到 \(f\)" 翻译成"分离 \(g\) 的图像与某个 epigraph"。所以"延拓 = 分离" 是同一件事的两种讲法。

这条观察是泛函分析的美——一个定理同时是代数的也是几何的，应用到一个领域就成代数构造（ch15 对偶空间），应用到另一个领域就成几何工具（凸优化、变分法）。

下一章我们把 HB 推向另一个方向：研究 \(X^*\) 这个对偶空间本身的结构。HB 保证了 \(X^*\) 大；现在我们正式研究它。8 条标准对偶字典 + 二次对偶 + 对偶算子，为 Banach 空间理论 + 谱论铺路。
