第 15 章 对偶空间 \(X^*\)

请你做一道朴素的事。在 \(\ell^2\) 里取一个线性泛函——比如

\[ \varphi(x) = \sum_{n=1}^\infty \frac{x_n}{n}. \]

这是个把序列 \(x = (x_1, x_2, \ldots)\) 映成一个数的规则。它显然线性。它有界 吗？算一下：用 Cauchy-Schwarz，

\[ |\varphi(x)| \le \left( \sum \frac{1}{n^2} \right)^{1/2} \left( \sum x_n^2 \right)^{1/2} = \frac{\pi}{\sqrt{6}} \|x\|_2. \]

有界。所以 \(\varphi \in (\ell^2)^*\)。

请你换一种眼光看这件事。\(\varphi\) 这个泛函由系数列 \((1, 1/2, 1/3, \ldots) \in \ell^2\) 完全决定（因 \(\sum 1/n^2 < \infty\)）。它是一个 \(\ell^2\) 元素。所以 \((\ell^2)^* \ni \varphi \leftrightarrow (1, 1/2, 1/3, \ldots) \in \ell^2\)——对偶元素 = 一个 \(\ell^2\) 序列。这条对应不是巧合：后面我们要证 \((\ell^2)^* \cong \ell^2\)。

这一章正式研究对偶空间 \(X^*\)。HB 定理（ch13）已经保证 \(X^*\) "足够大"——能区分任意两点 + 算出范数 + 做几何分离。本章问下一个问题：\(X^*\) 长什么样？它有什么结构？怎么把抽象的"全部连续线性泛函" 算成具体的对象（序列、函数、测度、或 \(X\) 本身的元素）？

这件事远不止"分类的乐趣"。它是泛函分析的第二只手。研究 \(X\) 时你只能从内部看（取元素、做加法、看范数）；研究 \(X^*\) 是从"所有线性观察" 看 \(X\)。两只手合起来，你可以做以前做不了的事——弱收敛（ch17）、Banach-Alaoglu 紧化（ch18）、对偶算子的谱（ch21）、变分法的 Lagrange 乘子（ch24）。整套高阶泛函分析的语法 是双向的——同时操纵 \(X\) 和 \(X^*\)。

这一章把这副语法建立。六节内容：(1) \(X^*\) 永远是 Banach；(2) 二次对偶与典范嵌入；(3) 8 条标准对偶字典；(4) 零化子；(5) 对偶算子；(6) ker / range 与对偶的关系。

15.1 \(X^*\) 永远是 Banach 空间

设 \(X\) 是赋范空间（不必完备），\(\mathbb{K} \in \{\mathbb{R}, \mathbb{C}\}\)。对偶空间是

\[ X^* := \mathcal{B}(X, \mathbb{K}) = \{f : X \to \mathbb{K} \text{ 线性 + 有界}\}, \]

配上算子范数 \(\|f\| = \sup_{\|x\| \le 1} |f(x)|\)。

定理。\(X^*\) 总是 Banach 空间——无论 \(X\) 是否完备。

这条结论的"无论 X 是否完备" 一句你应该停下来想一想。X 自己可能是个坏空间（不完备、有 Cauchy 列发散），但它的对偶 一定是好空间。原因：完备性是值域那一边的事——\(X^*\) 的元素值在 \(\mathbb{K}\) 里，\(\mathbb{K}\) 完备，所以 \(X^*\) 完备。这条直觉是 \(\mathcal{B}(X, Y)\) 完备性定理（ch9）的一个特例。

证明。设 \(\{f_n\} \subset X^*\) 是 Cauchy 列。即 \(\|f_n - f_m\| \to 0\)。要证 \(\{f_n\}\) 收敛到某 \(f \in X^*\)。

步骤 1：逐点定义极限。对每个固定 \(x \in X\)，

\[ |f_n(x) - f_m(x)| \le \|f_n - f_m\| \cdot \|x\| \to 0. \]

所以 \(\{f_n(x)\}\) 是 \(\mathbb{K}\) 里的 Cauchy 列，\(\mathbb{K}\) 完备 ⟹ 有极限，记 \(f(x) := \lim_n f_n(x)\)。

步骤 2：\(f\) 线性。直接验：\(f(\alpha x + \beta y) = \lim f_n(\alpha x + \beta y) = \lim (\alpha f_n(x) + \beta f_n(y)) = \alpha f(x) + \beta f(y)\)。

步骤 3：\(f\) 有界。\(\{f_n\}\) Cauchy ⟹ 范数有界，设 \(\|f_n\| \le M\)。则 \(|f(x)| = \lim |f_n(x)| \le M \|x\|\)，所以 \(\|f\| \le M\)，\(f \in X^*\)。

步骤 4：\(f_n \to f\) 在 \(X^*\) 范数下。给 \(\varepsilon > 0\)，取 \(N\) 使 \(\|f_n - f_m\| < \varepsilon\) 对 \(n, m \ge N\)。固定 \(n \ge N\)，对任意 \(x\) with \(\|x\| \le 1\)：\(|f_n(x) - f(x)| = \lim_m |f_n(x) - f_m(x)| \le \varepsilon\)。取 sup 得 \(\|f_n - f\| \le \varepsilon\) 对 \(n \ge N\)，证毕。

这个证明是个泛函分析最干净的小论证。请你停下来欣赏它的形状：逐点收敛 + Cauchy 一致控制 → 收敛到一个连续元素。这条结构后面在每个完备性证明里都会重复——\(L^p\) 完备（ch4）、\(C(K)\) 完备、\(\mathcal{B}(X, Y)\) 完备（ch9），全是同一种"逐点 + 一致" 的舞步。

推论：\(X^*\) 既是赋范空间，也是 Banach。所以本章后续我们可以放心地谈"\(X^*\) 上的 Cauchy 列收敛"、"\(X^*\) 的闭子空间"、"对 \(X^*\) 做 ch6 内积投影" 之类的事——所有 Banach 空间机器都对它有效。

15.2 二次对偶 \(X^{**}\) 与典范嵌入

\(X^*\) 是 Banach，所以可以再取它的对偶：

\[ X^{**} := (X^*)^* = \mathcal{B}(X^*, \mathbb{K}). \]

\(X^{**}\) 是"X 的对偶的对偶"——一个 second order 对象。它和 \(X\) 有什么关系？

关键观察：每个 \(x \in X\) 自然给一个 \(X^*\) 上的线性形——"在 x 处求值"。具体地

\[ \boxed{\ \widehat{x}(f) := f(x), \quad f \in X^*.\ } \]

固定 \(x\)、把 \(f\) 当变量，\(\widehat{x} : X^* \to \mathbb{K}\) 是线性（线性在 \(f\) 里）。又

\[ |\widehat{x}(f)| = |f(x)| \le \|f\| \cdot \|x\|, \]

所以 \(\widehat{x} \in X^{**}\) 且 \(\|\widehat{x}\| \le \|x\|\)。

这给一个映射

\[ J : X \to X^{**}, \quad J(x) := \widehat{x}. \]

\(J\) 称为典范嵌入（canonical embedding）或自然嵌入。它显然线性。

关键定理。\(J\) 是等距——\(\|J(x)\| = \|x\|\) 对所有 \(x \in X\)。

"\(\le\)" 我们刚证。"\(\ge\)" 这一面靠的是 HB 推论 1（ch13.4 节）：

给任意 \(x_0 \neq 0\)，存在 \(f_0 \in X^*\) 使 \(\|f_0\| = 1\) 且 \(f_0(x_0) = \|x_0\|\)。

用这个 \(f_0\)：\(\|\widehat{x_0}\| = \sup_{\|f\| \le 1} |\widehat{x_0}(f)| = \sup |f(x_0)| \ge |f_0(x_0)| = \|x_0\|\)。等号取到。证毕。

请你停下来想一件事：等距嵌入 是个非常强的性质。它说

\[ \boxed{\ X \hookrightarrow X^{**} \text{ 等距}.\ } \]

X 永远是它的二次对偶 \(X^{**}\) 的一个等距子空间——你可以把 \(X\) "装进" \(X^{**}\) 里，距离不失真。这条嵌入是 HB 的最重要的几何后果之一。它告诉你 \(X^{**}\) 至少和 \(X\) 一样大。

但 \(J\) 是否满射？这才是真问题。如果 \(J\) 满，\(X \cong X^{**}\)（等距同构），\(X\) 称为自反（reflexive）。如果 \(J\) 不满，\(X \subsetneq X^{**}\)（等距），\(X\) 不自反。

例：

  • 所有 Hilbert 空间自反（\(H^* = H\)，所以 \(H^{**} = H\)，\(J\) 自然就是恒等映射）。
  • \(L^p, \ell^p\) 对 \(1 < p < \infty\) 都自反。\(L^2 = H\) 是个特例。
  • \(L^1, \ell^1, L^\infty, \ell^\infty, c_0, C(K)\) 不自反。例如 \(c_0^{**} = \ell^\infty \neq c_0\)。

"自反" 是无穷维空间里非常珍贵的性质。它直接给：(a) 弱拓扑下闭单位球紧（ch18 的强化版），(b) 弱序列紧致定理（每个有界列有弱收敛子列），(c) 凸优化的极小元存在性（变分法基础）。是否自反 在分析无穷维问题时通常是第一个问的问题。

15.3 八条标准对偶字典

抽象 \(X^*\) 长什么样取决于 \(X\) 是谁。下面是泛函分析的对偶字典——八条标准恒等式，把抽象 \(X^*\) 翻译成具体对象。这些字典每条都是一条独立的表示定理，几乎都靠 ch16 的 Riesz 表示证。这一节我先列出来 + 讲意义，证明在下章。

空间 \(X\)对偶 \(X^*\)配对方式
\(\ell^p\) (\(1 \le p < \infty\))\(\ell^q\), \(1/p + 1/q = 1\)\(\langle x, y \rangle = \sum x_n \overline{y_n}\)
\(L^p(\Omega)\) (\(1 \le p < \infty\))\(L^q(\Omega)\)\(\int f \overline{g} \, d\mu\)
\(c_0\) (零序列)\(\ell^1\)\(\sum x_n y_n\)
\(C(K)\) (\(K\) 紧 Hausdorff)\(M(K)\) (Radon 测度)\(\int f \, d\mu\)
Hilbert \(H\)\(H\) (反线性同构)\(\langle \cdot, y \rangle_H\)
有限维 \(\mathbb{K}^n\)\(\mathbb{K}^n\)\(\sum x_i \overline{y_i}\)
Sobolev \(H_0^1(\Omega)\)\(H^{-1}(\Omega)\)分布配对
\(\ell^\infty\) (有界序列)\(\neq \ell^1\)（更大）无简单表示

请你停下来仔细看这张表。三条观察：

观察 1：对偶往往"复活" 共轭指数。\(\ell^p\) 和 \(L^p\) 的对偶都是 \(\ell^q, L^q\)，\(p, q\) 满足 \(1/p + 1/q = 1\)。这条共轭关系来自 Hölder 不等式 \(|\sum x_n y_n| \le \|x\|_p \|y\|_q\)——它是 \(L^q\) 元素能给 \(L^p\) 上有界泛函的根本原因。\(p = q = 2\) 那条对角线就是 Hilbert 自对偶。

观察 2：\(p = 1\) 与 \(p = \infty\) 不对称。\((\ell^1)^* = \ell^\infty\)（这条是上面表格暗含的），但反方向 \((\ell^\infty)^* \neq \ell^1\)——\((\ell^\infty)^*\) 比 \(\ell^1\) 严格大。这是为什么"自反" 对 \(1 < p < \infty\) 成立、对 \(p = 1, \infty\) 失败。物理直觉：\(\ell^1\) 元素"都集中在前几位"，\(\ell^\infty\) 元素"可以处处不衰减"——后者更"大"，所以它的对偶含一些非 \(\ell^1\) 的"幽灵" 元素（用 HB 构造的、不来自任何具体序列的"广义极限")。

观察 3：\(C(K)^* = M(K)\) 是测度论的桥。这条把"泛函分析对偶" 与"测度论积分" 等同起来——一个连续函数空间的对偶就是 Radon 测度空间。这条等同是 Riesz-Markov-Kakutani 定理（ch16），它告诉你为什么"线性泛函" 和"积分" 在数学里几乎是同义词。

这八条对偶字典是泛函分析的常用词典——你以后做计算时不停查它。例如 ch22 谱论里"对偶算子的谱"、ch24 PDE 里"\(H^{-1}\) 配对"、ch20 紧算子里"对偶算子是紧算子"——所有这些都依赖把 \(X^*\) 用具体对象表示出来。

这一章不证表里的等式（那是下章 Riesz 的工作）。这一章只把"这件事大概对、为什么" 给个直觉。两个直觉性论证：

(1) \((\ell^p)^* = \ell^q\) 的方向 "\(\supset\)"。给任意 \(y \in \ell^q\)，定义 \(\varphi_y(x) := \sum x_n \overline{y_n}\)。Hölder：\(|\varphi_y(x)| \le \|x\|_p \|y\|_q\)，所以 \(\varphi_y \in (\ell^p)^*, \|\varphi_y\| \le \|y\|_q\)。这给一个嵌入 \(\ell^q \hookrightarrow (\ell^p)^*\)。Riesz 表示要做的是反方向"每个 \((\ell^p)^*\) 元素来自某个 \(\ell^q\) 元素"——这一面要 HB / 选择公理般的存在性论证（详见 ch16）。

(2) \((c_0)^* = \ell^1\)。\(c_0\) 是"零序列空间"——\(x = (x_1, x_2, \ldots)\) 满足 \(x_n \to 0\)，配 sup 范数。给 \(y = (y_1, y_2, \ldots) \in \ell^1\)，定义 \(\varphi_y(x) = \sum x_n y_n\)。\(|\varphi_y(x)| \le \|x\|_\infty \cdot \|y\|_1\)，所以 \(\varphi_y \in c_0^*\)。反过来 \(c_0^* \to \ell^1\) 的存在用 HB 构造。等距同构。

注意这里的微妙：\(c_0\) 是 \(\ell^\infty\) 的子空间。它们的对偶却不一样——\(c_0^* = \ell^1\) 但 \((\ell^\infty)^* \supsetneq \ell^1\)。这告诉你对偶不是子空间继承的——子空间的对偶可能比"父空间的对偶往子空间限制" 更小或更大。

15.4 零化子（annihilator）与预零化子

有了 \(X\) 与 \(X^*\) 这对儿，可以双向构造"正交补" 般的对象。

设 \(M \subset X\) 子集（不必子空间）。零化子（annihilator）定义为

\[ \boxed{\ M^\circ := \{f \in X^* : f(x) = 0 \text{ 对所有 } x \in M\} \subset X^*.\ } \]

"把 \(M\) 全部映到 0 的所有泛函"。

设 \(N \subset X^*\) 子集。预零化子（pre-annihilator）定义为

\[ \boxed{\ {}^\circ N := \{x \in X : f(x) = 0 \text{ 对所有 } f \in N\} \subset X.\ } \]

"被 \(N\) 全部映到 0 的所有 X 元素"。

这俩是对偶语境下的"正交补" 概念——在 Hilbert 上 \(M^\circ\) 退化成 \(M^\perp\)（用 Riesz 表示把 \(X^*\) 等同于 \(X\)），但在一般 Banach 空间 \(\perp\) 没意义（没有内积），\(^\circ\) 是它的正确推广。

性质：

  1. \(M^\circ\) 总是 \(X^*\) 的闭 线性子空间（即使 \(M\) 不闭、不线性——\(M^\circ\) 是"零集" 的交，所以闭线性）。
  2. 同样 \({}^\circ N\) 总是 \(X\) 的闭 线性子空间。
  3. \(M \subset M_1 \implies M^\circ \supset M_1^\circ\)（反向单调）。
  4. \(M^\circ = (\overline{\operatorname{span}} M)^\circ\)——零化子只看 \(M\) 的闭线性扩张。

双重零化子定理：设 \(M \subset X\) 是子空间。则

\[ {}^\circ (M^\circ) = \overline{M}. \]

这条是凸集分离的直接应用。证明：

"\(\supset\)"：\(\overline{M} \subset {}^\circ(M^\circ)\) 因为 \(M^\circ\) 在 \(M\) 上为零、连续性传到 \(\overline{M}\)。

"\(\subset\)"：要证 \(x \notin \overline{M} \implies x \notin {}^\circ(M^\circ)\)。\(\{x\}\) 凸紧、\(\overline{M}\) 凸闭、二者不相交，由 ch14 的第二分离定理 得 \(f \in X^*\) 使 \(f|_{\overline{M}} \le \alpha < \beta \le f(x)\)。\(M\) 是子空间 意味着 \(0 \in \overline{M}\)、\(f(0) = 0\)，所以 \(\alpha \ge 0\)。但 \(M\) 是线性 子空间——如果 \(f|_M \neq 0\)，乘任意常数会让 \(f|_M\) 跑遍 \(\mathbb{K}\)，与 \(f|_M \le \alpha\) 矛盾。所以 \(f|_M = 0 \implies f \in M^\circ\)。但 \(f(x) > 0\)，所以 \(x \notin {}^\circ(M^\circ)\)。证毕。

这条定理是"对偶 + 闭包" 几何 的基础。请你停下来注意这条用 ch14 的什么："凸点和闭凸子空间不相交 → 严格分离" 这一面被用来"找一个零化 M 但不零化 x 的泛函"。这是 HB 的几何形式在子空间情景的具体落地。

双重零化子定理一条立刻的推论：

\[ \boxed{\ M \subset X \text{ 闭子空间} \iff M = {}^\circ(M^\circ).\ } \]

"闭子空间和它的双重零化子相等"——这是 Banach 空间版的"双重正交补 = 自身"（Hilbert 正交分解的对应物）。

15.5 对偶算子 \(T^*\)

设 \(X, Y\) 赋范空间，\(T : X \to Y\) 有界线性算子。对偶算子（adjoint / transpose） \(T^* : Y^* \to X^*\) 定义如下：

对 \(g \in Y^*\)（一个 \(Y\) 上的泛函），定义

\[ \boxed{\ T^*(g) := g \circ T \in X^*.\ } \]

显式地：\((T^* g)(x) = g(T x)\) 对所有 \(x \in X\)。

"把 \(g\) 拉回到 \(X\) 上"。这就是对偶算子的定义。

请你画一下对偶图：

\[ X \xrightarrow{\ T\ } Y, \qquad X^* \xleftarrow{\ T^*\ } Y^*. \]

箭头方向翻过来。这是"取对偶反转所有箭头" 的范畴论现象——对偶是个反变函子（contravariant functor）。

线性 + 有界 + 范数：

定理。\(T^*\) 线性、有界，且 \(\|T^*\| = \|T\|\)。

线性显然（\(g \mapsto g \circ T\) 在 \(g\) 里线性）。有界：\(\|T^* g\| = \sup_{\|x\| \le 1} |g(T x)| \le \sup_{\|x\| \le 1} \|g\| \|T x\| = \|g\| \|T\|\)，所以 \(\|T^*\| \le \|T\|\)。

"\(\ge\)" 这一面又靠 HB 推论 1：给 \(\varepsilon > 0\)，取 \(x_\varepsilon\) 使 \(\|x_\varepsilon\| = 1\) 且 \(\|T x_\varepsilon\| > \|T\| - \varepsilon\)。再取 \(g_\varepsilon \in Y^*\) 使 \(\|g_\varepsilon\| = 1\) 且 \(g_\varepsilon(T x_\varepsilon) = \|T x_\varepsilon\|\)（HB 推论 1 应用在 \(T x_\varepsilon \in Y\)）。则

\[ \|T^* g_\varepsilon\| \ge |T^* g_\varepsilon (x_\varepsilon)| = |g_\varepsilon(T x_\varepsilon)| = \|T x_\varepsilon\| > \|T\| - \varepsilon. \]

取 \(\varepsilon \to 0\) 得 \(\|T^*\| \ge \|T\|\)。等号取到。

这条 \(\|T^*\| = \|T\|\) 是 对偶算子最重要的"规范守恒律"。它告诉你对偶是等距的——把算子取对偶、范数不变。这条在 ch20 紧算子（紧 ⟺ 对偶紧 Schauder 定理）和 ch21 谱论（自伴算子的谱实）那里反复用。

函子性："对偶反转一切"：

  • \((I_X)^* = I_{X^*}\)。
  • \((S \circ T)^* = T^* \circ S^*\)（注意顺序翻转）。
  • \((\alpha S + \beta T)^* = \alpha S^* + \beta T^*\)（线性）。
  • \((T^{-1})^* = (T^*)^{-1}\) 当 \(T\) 可逆。

双重对偶：\(T^{**} : X^{**} \to Y^{**}\)。在典范嵌入下 \(T^{**}|_{J(X)} = J \circ T\)——即 \(T^{**}\) 限制到 \(X\) 等于 \(T\) 自己。所以"取两次对偶 = 自己（在嵌入意义下）"。

15.6 ker, range 与对偶

对偶算子的 ker / range 与原算子的 ker / range 通过零化子精确对应。这一节列四条公式（"对偶字典之零化子版"），是 ch22 Fredholm 理论的前置工具。

设 \(T : X \to Y\) 有界线性算子，\(T^* : Y^* \to X^*\) 对偶。

四条公式：

\[ \boxed{\ \ker T^* = (\operatorname{Range} T)^\circ, \qquad \overline{\operatorname{Range} T} = {}^\circ (\ker T^*).\ } \]

\[ \boxed{\ \ker T = {}^\circ (\operatorname{Range} T^*), \qquad \overline{\operatorname{Range} T^*} \subset (\ker T)^\circ.\ } \]

第二行第二条 \(\overline{\operatorname{Range} T^*} = (\ker T)^\circ\) 当且仅当 \(\operatorname{Range} T^*\) 闭。这条精细差别在 ch22 Fredholm 替换里关键。

证四条之首：\(\ker T^* = (\operatorname{Range} T)^\circ\)。逐链等价：

\[ g \in \ker T^* \iff T^* g = 0 \iff g(T x) = 0 \ \forall x \in X \iff g \in (\operatorname{Range} T)^\circ. \]

第二条 \(\overline{\operatorname{Range} T} = {}^\circ(\ker T^*) = {}^\circ ((\operatorname{Range} T)^\circ)\) 由双重零化子定理（15.4）。其余两条同构。

这四条公式有大堆实际后果。我列三条：

后果 1：稠密性 ⟺ 对偶单射。

\[ \overline{\operatorname{Range} T} = Y \iff (\operatorname{Range} T)^\circ = \{0\} \iff \ker T^* = \{0\} \iff T^* \text{ 单射}. \]

"原算子像稠密 ⟺ 对偶算子单射"。这是数值线性代数和 PDE 都用到的事实。例如 ch22 里 Fredholm 选择性的"解的存在性 ⟺ 对偶问题无非平凡解" 就是这条。

后果 2：单射 ⟺ 对偶像稠密。

\[ \ker T = \{0\} \iff {}^\circ (\operatorname{Range} T^*) = \{0\} \iff \overline{\operatorname{Range} T^*} = X^*. \]

"原算子单射 ⟺ 对偶算子像稠密"。和后果 1 对称。

后果 3：闭值域定理（Banach）。\(\operatorname{Range} T\) 闭 ⟺ \(\operatorname{Range} T^*\) 闭 ⟺ \(\operatorname{Range} T = {}^\circ (\ker T^*)\) ⟺ \(\operatorname{Range} T^* = (\ker T)^\circ\)。这条很深的等价不证（属 ch22）——但你看公式上的对称就能猜对。

这四条公式 + 三条后果是"原算子-对偶算子" 字典。和 15.3 的"空间字典" 配合，构成研究 Banach 空间算子时的基本词汇——每个关于 \(T\) 的代数 / 拓扑事实都有一个关于 \(T^*\) 的对应版本。这种"双手作业" 的灵活性是泛函分析比线性代数强大的关键之一。

15.7 一句临别话

这一章把对偶机器装齐：

  • \(X^*\) 永远 Banach（15.1）。
  • \(X \hookrightarrow X^{**}\) 等距嵌入（15.2，靠 HB 推论 1）。
  • 八条对偶字典——具体计算 \(X^*\) 的"查表手册"（15.3）。
  • 零化子 + 双重零化子定理（15.4，靠 ch14 几何分离）。
  • 对偶算子 \(T^*, \|T^*\| = \|T\|\)（15.5）。
  • 四条 ker / range 公式（15.6）。

请你停下来回想 ch1.1 节列的"五件崩塌" 之第五件——"算子 ≠ 矩阵"。我们答应"把算子当分析对象" 的修法。本章就是修法的关键一步：算子有对偶，对偶是一种独立的分析工具，让你能从"泛函测量" 的方向研究算子。这是矩阵理论没有 的工具——\(\mathbb{R}^n\) 上算子的对偶就是它的转置矩阵，没新鲜事。但 \(\ell^2 \to \ell^2\) 上的右移算子和它的对偶（左移算子）是不同的算子，对偶视角揭示出"右移单射不满"、"左移满不单" 这种有限维不可能的现象（ch1.5 节那个例子的对偶解释）。

下一章我们直面对偶字典——三种 Riesz 表示定理（Hilbert / \(L^p\) / \(C(K)\)）合在一起讲，你会看到为什么所有对偶都长成"积分配对" 的样子。这是泛函分析最美的一节——对偶 = 积分。
