第 19 章 自反空间与一致凸

上一章把 Banach–Alaoglu 装配好之后，我反复说一句话："在自反空间里，弱紧 = 弱*紧"。整个变分法 5 步的合法性都依赖这一条——"有界 ⟹ 有弱收敛子列"——而这条只对自反空间成立。

所以一个直接的实战问题：哪些 Banach 空间是自反的？这一章给完整清单 + 一个非常实用的几何充分条件（"一致凸 ⟹ 自反"，Milman–Pettis 定理）。后者特别重要——它把"抽象的自反性" 转化为"看一眼单位球长什么形状就能判断" 的几何条件。\(L^p\)（\(1 < p < \infty\)）自反这件事最终会落在Clarkson 不等式 上，这是这一章的技术高潮。

19.1 自反性的等价刻画

先把"自反" 这个词的多种说法摆全。设 \(X\) 是 Banach 空间，\(J : X \to X^{**}\) 是第 15 章典范嵌入，\(J(x)(f) := f(x)\)。\(J\) 总是等距——\(\|J(x)\|_{X^{**}} = \|x\|_X\)（这条来自 Hahn–Banach）。但 \(J\) 不一定满射。

定义。\(X\) 是自反的 \(\iff J\) 满射 \(\iff X \cong X^{**}\)（通过 \(J\)）。

注意"自反" 不是说"\(X\) 同构于 \(X^{**}\)"——而是说"典范嵌入 \(J\) 是同构"。James 1951 年构造了一个空间 \(X\) 使 \(X \cong X^{**}\) 但 \(J\) 不满射——这种"非典范同构" 的存在让我们必须把"通过典范嵌入" 写在定义里。

定理（自反性的三条等价刻画）。下面三条等价：

  1. \(J : X \to X^{**}\) 满射。
  2. \(\overline{B}_X\) 弱紧（Kakutani 定理）。
  3. 每个有界序列都有弱收敛子列（Eberlein–Šmulian 定理）。

第 1 条是定义。第 2 条是上一章说的"弱紧 = 弱*紧 in \(X\)"——把 \(X\) 当成 \(X^{**}\) 起 "原空间" 角色，应用 Banach–Alaoglu。第 3 条是把"拓扑紧" 升级为"序列紧"——这条等价是 Eberlein 1947 与 Šmulian 1940 的深定理。

这三条等价在使用上分工不同：

  • 第 1 条用来判断 一个抽象空间是否自反（"算 \(X^{**}\)，看 \(J\) 是不是满射"）。
  • 第 2 条是几何画面（"单位球弱紧"）。
  • 第 3 条是实战工具（"抽弱收敛子列" 直接合法）。

变分法的"直接方法"（17.7 节+ 18.6 节）始终用第 3 条——"有界 ⟹ 抽弱收敛子列"。这是为什么"自反性" 是 PDE 弱解理论的硬性要求。

自反性的几个基本性质（不证，作清单使用）：

  • 自反空间的闭子空间 自反。
  • 自反空间的商空间 自反。
  • \(X\) 自反 \(\iff X^*\) 自反。"自反性" 在取对偶下保持。
  • 有限维 ⟹ 自反（\(X = X^{**}\) 显然，因维数相同）。
  • Hilbert 空间 ⟹ 自反（Riesz 表示给 \(X = X^* = X^{**}\)）。

19.2 自反空间分类清单

把常见 空间按"是否自反" 分类。这张表你以后会反复回头查。

自反空间：

  • 有限维 \(\mathbb R^n, \mathbb C^n\)。
  • Hilbert 空间 \(H\)（包括 \(L^2(\Omega), \ell^2, \) 任何 Sobolev \(H^k(\Omega)\)）。
  • \(L^p(\Omega), \ell^p\) 当 \(\boxed{1 < p < \infty}\)。这是 19.5 节 Clarkson 不等式给的。
  • Sobolev 空间 \(W^{k, p}(\Omega)\) 当 \(1 < p < \infty\)。"\(L^p\) 自反 + 嵌入论证" 给的。

非自反空间（"不便" 空间，需要小心）：

  • \(L^1(\Omega), \ell^1\)。\(L^1\) 的对偶是 \(L^\infty\)；\(L^\infty\) 的对偶严格大于 \(L^1\)（包含"有限可加测度" 这种奇怪东西）。所以 \(J : L^1 \to (L^\infty)^* = L^{**}\) 不满。
  • \(L^\infty(\Omega), \ell^\infty\)。同上理由——\((L^\infty)^* \supsetneq L^1\)。
  • \(c_0\)（趋零序列空间）。\(c_0^* = \ell^1\)，\((c_0)^{**} = \ell^\infty \supsetneq c_0\)。
  • \(C(K)\)（紧 Hausdorff \(K\) 上连续函数）。\(C(K)^* = M(K)\)（有限 Borel 测度，第 16 章），\(C(K)^{**} \supsetneq C(K)\)。

请你停下来看这两份名单。"中间" 的 \(L^p\)（\(1 < p < \infty\)）都自反；"边缘" 的 \(L^1, L^\infty\) 不自反；离散版 \(\ell^p\) 同样规律。这条规律不是巧合——\(p = 1\) 是 \(L^p\) 范数"退化为绝对值之和" 的极限，\(p = \infty\) 是 \(L^p\) 范数"退化为最大值" 的极限。两个极限点上单位球的"角" 太尖（\(\ell^1\) 单位球是菱形、\(\ell^\infty\) 单位球是方块），失去了"弱紧" 必需的"圆滑" 性。中间 \(p\) 的单位球是"圆滚滚" 的——这就是 19.3 节"一致凸" 概念要正式表达的几何。

在 PDE / 变分法实战中，这张表的指导意义：

  • 能用 \(L^p\)（\(1 < p < \infty\)）就用——直接套自反性 + 弱紧致工具。
  • 必须用 \(L^1\) 时（比如某些守恒律、某些测度论问题），放弃 自反性，改用"测度紧致"（\(L^1\) 嵌入到测度空间，用弱*紧；这是 Young 测度、缓和分布等理论的入口）。
  • 必须用 \(L^\infty\) 时（比如 Hamilton–Jacobi 方程的"粘性解" 框架），放弃 自反性，改用弱* 紧（\((L^\infty)^* \supsetneq L^1\)，但 \(L^1\) 的对偶 是 \(L^\infty\)——所以 \(L^\infty\) 是 \(L^1\) 的对偶，弱*紧致仍可用）。

这张"自反 vs 非自反" 清单某种意义上是 PDE 现代理论"选择哪个空间做研究" 的最初指南。

19.3 一致凸：单位球"圆滚滚"

"自反" 是一个抽象 的拓扑性质（"典范嵌入是满射"），看不见摸不着。下面给一个具体 的几何充分条件——一致凸——它把"看一眼单位球长什么形状" 变成判断自反性的工具。

定义（一致凸，Clarkson 1936）。Banach 空间 \(X\) 称为一致凸，如果对每个 \(\varepsilon > 0\) 存在 \(\delta(\varepsilon) > 0\) 使

\[ \boxed{\ \|x\|, \|y\| \le 1 \text{ 且 } \|x - y\| \ge \varepsilon \implies \left\| \frac{x + y}{2} \right\| \le 1 - \delta(\varepsilon).\ } \]

用人话：如果 \(x, y\) 都在闭单位球里且相距 \(\ge \varepsilon\)，则它们的中点严格在球内一段固定距离。

几何画面：考虑单位球 \(\overline{B}\)。取球面上两点 \(x, y\)（\(\|x\| = \|y\| = 1\)）。中点 \((x+y)/2\) 一定在闭球内。它有多深？

  • 如果球面是圆形（如 Hilbert 空间），中点严格在内——除非 \(x = y\)。深度随 \(x, y\) 的"分离度" \(\|x - y\|\) 单调增。
  • 如果球面有平边（如 \(\ell^\infty\) 的方块、\(\ell^1\) 的菱形），平边上两点的中点仍在 平边上——深度 0。这种空间不一致凸。

"一致凸" 强调的是"凹陷深度"对任意 \(\varepsilon\) 都有一致下界 \(\delta(\varepsilon)\)。

例子。

  • Hilbert 空间。平行四边形等式 \(\|x + y\|^2 + \|x - y\|^2 = 2(\|x\|^2 + \|y\|^2)\) 给：若 \(\|x\|, \|y\| \le 1, \|x - y\| \ge \varepsilon\)，则 \(\|x + y\|^2 \le 4 - \varepsilon^2\)，所以 \(\|(x+y)/2\| \le \sqrt{1 - \varepsilon^2/4}\)。取 \(\delta(\varepsilon) = 1 - \sqrt{1 - \varepsilon^2/4} \approx \varepsilon^2/8\)。Hilbert 一致凸，凹陷深度 \(\sim \varepsilon^2\)。
  • \(L^p, \ell^p\)（\(1 < p < \infty\)）：一致凸（19.5 节用 Clarkson 证明）。\(p = 2\) 是 Hilbert，\(p \neq 2\) 时仍一致凸但 \(\delta(\varepsilon)\) 的形式不同。
  • \(L^1, L^\infty\)：不 一致凸（19.6 节会展示具体反例）。
  • \(c_0, C(K)\)：不 一致凸。

请你看这个分类——和 19.2 节自反清单完全一致。这不是巧合：Milman–Pettis 定理说一致凸 ⟹ 自反。下一节证。

19.4 Milman–Pettis 定理

定理（Milman 1938，Pettis 1939）。一致凸的 Banach 空间是自反的。

证明。要证 \(J : X \to X^{**}\) 满射。等价地，对每个 \(\Phi \in X^{**}\)（不妨 \(\|\Phi\| = 1\)），找 \(x \in X\) 使 \(J(x) = \Phi\)（即 \(x \in X\) 满足 \(\|x\| = 1\) 且 \(\Phi(f) = f(x)\) 对所有 \(f \in X^*\)）。

第 1 步：找逼近。由 \(\|\Phi\| = 1\) 和上确界定义，对每个 \(\varepsilon > 0\) 存在 \(f_\varepsilon \in X^*\) 使 \(\|f_\varepsilon\| = 1\) 且 \(|\Phi(f_\varepsilon)| > 1 - \varepsilon/2\)。

反过来，由 \(f_\varepsilon\) 的范数定义和 \(\|f_\varepsilon\| = 1\)，存在 \(x_\varepsilon \in X\) 使 \(\|x_\varepsilon\| \le 1\) 且 \(|f_\varepsilon(x_\varepsilon)| > 1 - \varepsilon/2\)。

这些 \(x_\varepsilon\) 是 "逼近 \(\Phi\) 的具体向量"。

第 2 步：证 \(\{x_{1/n}\}\) 是 Cauchy。取 \(\varepsilon, \varepsilon' > 0\) 小。考虑两个对应的向量 \(x_\varepsilon, x_{\varepsilon'}\)。中点 \(z := (x_\varepsilon + x_{\varepsilon'})/2\)。

选 \(g \in X^*\) 使 \(\|g\| = 1\) 且 \(\Phi(g) > 1 - \min(\varepsilon, \varepsilon')/4\)（同样的逼近论证）。

关键算式。把 \(g\) 应用到 \(z\)：

\[ g(z) = \frac{g(x_\varepsilon) + g(x_{\varepsilon'})}{2}. \]

但更巧妙的：考虑 \(\Phi(g)\) 与 \(g(z)\) 的差。我用另一种 关系——设 \(h \in X^*\) 是 \(z\) 范数的"对偶" 元素（Hahn–Banach 给一个 \(h \in X^*\) 使 \(\|h\| = 1\) 且 \(h(z) = \|z\|\)）。则 \(|\Phi(h)| \le \|\Phi\| \|h\| = 1\)，且

\[ \|z\| = h(z) = \frac{h(x_\varepsilon) + h(x_{\varepsilon'})}{2}. \]

但每个 \(|h(x_i)| \le \|h\| \|x_i\| \le 1\)，所以 \(\|z\| \le 1\)。问题是 \(\|z\|\) 有多 接近 1。

(标准证明做的是另一条路——这里给一种偏轻的版本。)考虑 \(\Phi\) 应用到 \(f_\varepsilon, f_{\varepsilon'}\) 的平均：

\[ \Phi\left(\frac{f_\varepsilon + f_{\varepsilon'}}{2}\right) = \frac{\Phi(f_\varepsilon) + \Phi(f_{\varepsilon'})}{2} > 1 - \frac{\max(\varepsilon, \varepsilon')}{2}. \]

所以 \(\|(f_\varepsilon + f_{\varepsilon'})/2\| \ge 1 - \max(\varepsilon, \varepsilon')/2\)（因为 \(\Phi\) 对偶范数 \(\le 1\)）。

对偶到 \(X^*\) 的一致凸（这一步用到"X 一致凸 ⟹ X* 一致凸" 的对偶引理，详细见 Brezis 第 3 章；本质是把 \(X\) 上凹陷反映到 \(X^*\) 上的"光滑性"，再再对偶回 \(X^{**}\)）：

\[ \|f_\varepsilon - f_{\varepsilon'}\| \le \tilde\delta^{-1}(\max(\varepsilon, \varepsilon')) \to 0. \]

这一步配合 \(X\) 自身的一致凸，给 \(\|x_\varepsilon - x_{\varepsilon'}\| \to 0\)。

(完整严格证明涉及"对偶空间一致凸性 = 原空间光滑性"等较深引理，标准教科书如 Brezis "Functional Analysis" 第 3.7 节、Conway "A Course in Functional Analysis" 第 V.10 节给完整版。这里给的是几何骨架——证明的动力 来自一致凸的"中点必落进去" 性质。)

第 3 步：取极限。\(\{x_{1/n}\}\) Cauchy + \(X\) 完备 ⟹ \(x_{1/n} \to x^*\) 强收敛。验证 \(J(x^*) = \Phi\)：对任意 \(f \in X^*\)，\(f(x^*) = \lim f(x_{1/n})\)；可证 \(\lim f(x_{1/n}) = \Phi(f)\)（用 \(x_{1/n}\) 是 \(\Phi\) 的"逼近向量" 的性质）。结束。

关键想法的一句话 总结：一致凸把"对偶上的逼近向量序列" 强迫成 Cauchy；完备性让它收敛；极限就是 \(\Phi\) 的代表元。

反向不真。James 1951 构造了一个自反但不一致凸的 Banach 空间（"James 空间"，比较著名）。所以"一致凸 ⟹ 自反" 是单向的——一致凸是自反的充分 但不必要 条件。

实战上 Milman–Pettis 极有用。它把"自反性" 这种抽象拓扑性质归结为"看单位球长什么样" 的具体几何。下一节用它证 \(L^p\)（\(1 < p < \infty\)）自反。

19.5 Clarkson 不等式

怎么证 \(L^p\)（\(1 < p < \infty\)）一致凸？需要一个具体 的"中点深度" 估计。这就是 Clarkson 1936 给的不等式。

Clarkson 不等式（\(2 \le p < \infty\)）。对任意 \(f, g \in L^p\)：

\[ \boxed{\ \left\| \frac{f + g}{2} \right\|_p^p + \left\| \frac{f - g}{2} \right\|_p^p \le \frac{1}{2}\left( \|f\|_p^p + \|g\|_p^p \right).\ } \]

Clarkson 不等式（\(1 < p \le 2\)）。对任意 \(f, g \in L^p\)：

\[ \left\| \frac{f + g}{2} \right\|_p^q + \left\| \frac{f - g}{2} \right\|_p^q \le \left( \frac{1}{2} (\|f\|_p^p + \|g\|_p^p) \right)^{q/p}, \]

其中 \(q = p/(p-1)\) 是共轭指数。

这两条都把"中点 + 半差" 用"原始两点的范数" 控制——本质是 Hilbert 平行四边形 \(\|f+g\|^2 + \|f-g\|^2 = 2(\|f\|^2 + \|g\|^2)\) 在 \(p \neq 2\) 上的一种"不等式版本"。

证明（\(p \ge 2\) 版本，草图）。靠两条逐点不等式：对任意 \(a, b \in \mathbb R\) 和 \(p \ge 2\)，

\[ \left| \frac{a+b}{2} \right|^p + \left| \frac{a-b}{2} \right|^p \le \frac{1}{2}\left(|a|^p + |b|^p\right). \]

这条逐点不等式可由凸性 + 单变量分析证明。把它对 \(a = f(x), b = g(x)\) 取 + 在 \(\Omega\) 上积分，得 Clarkson。

(\(1 < p < 2\) 的版本更微妙，用对偶 + Hölder。)

从 Clarkson 推一致凸（\(p \ge 2\) 情形）：设 \(\|f\|_p, \|g\|_p \le 1\) 且 \(\|f - g\|_p \ge \varepsilon\)。则

\[ \left\| \frac{f+g}{2} \right\|_p^p \le \frac{1}{2}(\|f\|_p^p + \|g\|_p^p) - \left\| \frac{f-g}{2} \right\|_p^p \le 1 - (\varepsilon/2)^p. \]

所以 \(\|(f+g)/2\|_p \le (1 - (\varepsilon/2)^p)^{1/p}\)。取 \(\delta(\varepsilon) = 1 - (1 - (\varepsilon/2)^p)^{1/p}\)——一致凸条件满足。结束。

所以 \(L^p\)（\(p \ge 2\)）一致凸 ⟹ 自反（Milman–Pettis）。\(1 < p < 2\) 的版本类似（用第二组 Clarkson 不等式）。

对偶视角：对偶范数 \((L^p)^* = L^q\)（\(1/p + 1/q = 1\)，\(p, q \in (1, \infty)\)，第 15 章），自反给 \((L^p)^{**} = (L^q)^* = L^p\)——典范嵌入就是 同构。所以 "\(L^p\) 自反" 这条不只是抽象事实——它告诉你 \(L^p\) 与 \(L^q\) 之间的对偶配对是对称 的（每边互为对方的对偶）。

整个 \(L^p\)（\(1 < p < \infty\)）几何就这样：单位球"圆滚滚"（一致凸）⟹ 自反 ⟹ 弱拓扑下闭单位球紧 ⟹ 变分法畅通。

19.6 非自反空间的"不便"

把上面的好故事翻过来。在非 自反空间里，弱紧致失败——以下三个具体不便会反复出现。

不便 1：\(L^1\) 弱紧致失败。取 \(\Omega = (0, 1)\)。考虑序列 \(f_n(x) = n \mathbf 1_{(0, 1/n)}(x)\)。每个 \(f_n \in L^1\)，\(\|f_n\|_1 = 1\)——序列一致有界。

问：\(f_n\) 有弱收敛子列吗？

假设 \(f_{n_k} \rightharpoonup f\) 在 \(L^1(0, 1)\) 弱意义下。则对每个 \(g \in L^\infty(0, 1)\)：\(\int f_{n_k} g \to \int f g\)。取 \(g = \mathbf 1_{(0, 1)}\)（恒 1）：\(\int f_{n_k} = 1 \to \int f = 1\)。但同时取 \(g_\delta = \mathbf 1_{(\delta, 1)}\) 对小 \(\delta\)：\(\int_{n_k > 1/\delta} f_{n_k} g_\delta = 0\)（因 \(f_{n_k}\) 集中在 \((0, 1/n_k) \subset (0, \delta)\)）。所以 \(\int f \cdot g_\delta = 0\)，对所有 \(\delta > 0\)——意味着 \(f = 0\) 几乎处处。

但 \(\int f = 1\) 与 \(f = 0\) a.e. 矛盾。所以没有 弱收敛子列。\(L^1\) 不弱紧。

背后的物理图像：质量"集中到一点"（\(f_n\) 接近 Dirac \(\delta_0\)）。极限不在 \(L^1\) 里——\(\delta_0\) 是测度，不是函数。\(L^1\) "逃逸" 到了它的对偶 \((L^\infty)^*\) 里去了。

修法（不修 \(L^1\) 自反，因不可能）：把问题搬到测度空间 \(M(\Omega)\)（有限 Borel 测度）。\(M(\Omega) = C_0(\Omega)^*\)，所以闭球弱*紧（Banach–Alaoglu）。测度的弱*紧致 是流体力学、调和分析、概率论里常用的工具——正是 \(L^1\) 弱紧致失败的"修补"。

不便 2：\(L^\infty\) 不可分 + 弱*拓扑不可度量。\(L^\infty\) 是 \(L^1\) 的对偶——所以 \(L^\infty\) 闭单位球弱*紧（Banach–Alaoglu）。但 \(L^1\) 可分（多项式 / 阶梯函数稠密），所以 \(L^\infty\) 闭单位球弱*序列紧 + 度量化（18.3-4 节）。

但 \(L^\infty\) 自身的弱 拓扑就麻烦——\(L^\infty\) 不可分，它的弱*拓扑不可度量。所以"\(L^\infty\) 上的有界序列有弱收敛子列" 这种话在弱 意义下不对——必须用弱*。

实战意义：处理 \(L^\infty\) 数据时（如 Hamilton–Jacobi 方程的"粘性解"、调和分析里的 BMO 空间），始终 用弱*——把 \(L^\infty\) 当成 \(L^1\) 的对偶。

不便 3：\(L^1\) 单位球缺少强极值点。"极值点" 是一个集合的"角"——闭凸集里不能写为内部两点中点的点。Hilbert 单位球的极值点是整个球面。\(L^p\)（\(1 < p < \infty\)）类似。

但 \(L^1\) 单位球的极值点很少——对 \(L^1(0, 1)\)，极值点是正 函数 \(f \ge 0\) with \(\|f\|_1 = 1\) 而没 任何"分裂" 余地的。具体的极值点几乎不存在（"每个 \(L^1\) 函数都可以拆成两个不等的 \(L^1\) 单位球元素的中点"——除非函数已经是 Dirac 测度）。

这条"极值点稀少" 让 Krein–Milman 定理（"紧凸集 = 其极值点闭凸包"）在 \(L^1\) 上给不出有用的结构。要在 \(L^1\) 上做"极值点分析"，必须扩张到测度空间或用"分解定理"（如 Choquet 表示）——这些都是高度技术性的工具。

这一章三件不便共同说一句话：非自反空间是泛函分析的"边缘" 案例。它们存在、它们重要（\(L^1, L^\infty\) 是日常空间），但变分法、PDE 弱解、紧致论证 在它们上要么失败要么需要更复杂的工具。如果你能选择空间——选自反的；如果不能——准备多写几页论证。

这一章和上两章合起来构成"无穷维紧致" 这个主题的完整故事：有限维 Heine–Borel 在无穷维崩塌（ch1）→ 弱收敛把 \(\ell^2\) 单位球的紧致用新形式找回（ch17）→ Banach–Alaoglu 把"对偶单位球弱*紧" 升为定理（ch18）→ 自反空间让"原空间弱紧" 也成立 + 一致凸给可操作的几何判据（本章）。这条主线是泛函分析"无穷维代数 + 几何 + 分析" 三位一体的脊骨——下面 5 章谈紧算子、谱定理、Sobolev 空间、量子力学时反复回头用它。
