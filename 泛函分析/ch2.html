第 2 章 度量、范数、Banach 空间

上一章我反复用了一个词："\(\ell^2\) 闭单位球"。我把它当作大家都懂的对象拿来举反例——可你要是停下来认真问一句"\(\ell^2\) 到底是什么"，或者"什么叫一个空间'完备'"，或者"'Banach 空间'三个字加起来到底指什么"，会发现上一章其实欠了一笔账。

这一章把账还掉。我们要把整套语言装齐——度量、范数、收敛、Cauchy、完备、Banach 空间——这些词在后面 22 章里几乎每页都出现，必须先搞清。

请你不要把这一章当作"复习记号"。复习的部分（度量空间）我会压到极简；但"完备" 这个词我要花最多的时间——它是上一章五处崩塌的修法里反复用到的底层资源，不把它讲透，后面的定理都站不稳。

2.1 度量空间：极简复习

开始最朴素的层面。一个度量空间 是一对 \((X, d)\)，其中 \(X\) 是个集合，\(d : X \times X \to [0, \infty)\) 满足三条：

  • \(d(x, y) = 0 \iff x = y\)（正定）
  • \(d(x, y) = d(y, x)\)（对称）
  • \(d(x, z) \le d(x, y) + d(y, z)\)（三角不等式）

"\(d(x, y)\)" 你想成"\(x\) 到 \(y\) 的距离"，这三条公理就是距离这个词的最低要求。

例子之多让人乏味——\(\mathbb{R}^n\) 配欧氏距离、离散度量（\(d(x, y) = 1\) 若 \(x \neq y\)）、字符串编辑距离、图上的最短路径距离——都是度量空间。这本书里出现的全部度量空间几乎都是赋范空间（下面 2.2 节）的副产品，所以"度量空间" 这个一般概念我们就到这里。

真正只用到度量结构（不用线性结构）的地方有三件事：

  • 开球：\(B(x, r) = \{y : d(x, y) < r\}\)。开集就是"每个点都被某个开球包住" 的集合。
  • 收敛：\(x_n \to x\) 意味着 \(d(x_n, x) \to 0\)。
  • 连续：\(f : X \to Y\) 在 \(x\) 处连续 \(\iff\) 对每列 \(x_n \to x\) 有 \(f(x_n) \to f(x)\)。

这三件事在实分析读本第 2 章里详细讲过。这本书我们直接拿来用。

2.2 范数：度量 + 线性结构

泛函分析研究的对象不是任意度量空间，而是带线性结构的度量空间——你能加向量、能数乘。一个集合若能加、能数乘（满足通常的 8 条公理），就叫向量空间。我们已经在线性代数里熟透。

把"距离" 和"线性" 兼容起来的办法是范数。设 \(X\) 是 \(\mathbb{R}\) 或 \(\mathbb{C}\) 上的向量空间。一个范数 \(\|\cdot\| : X \to [0, \infty)\) 满足：

  • \(\|x\| = 0 \iff x = 0\)（正定）
  • \(\|\alpha x\| = |\alpha| \cdot \|x\|\)（齐次）
  • \(\|x + y\| \le \|x\| + \|y\|\)（三角）

有了范数，自然得到度量 \(d(x, y) := \|x - y\|\)。所以赋范空间是度量空间的特例——但额外要求度量"和加法、数乘兼容"。这层兼容把上面 2.1 节那些一般度量空间的"散" 收编成可以做线性代数的"整"。

请你看清这一步的代价：朴素度量空间允许任意距离函数；赋范空间只允许"和线性结构相容" 的那部分。代价换来的好处是：你以后可以同时用分析（极限）和代数（加法、子空间）两套工具——这正是泛函分析能成事的根本。

下面四个范数贯穿全书。请你记牢每一个的形状。

例 1：\(\mathbb{R}^n\)（或 \(\mathbb{C}^n\)）。对 \(x = (x_1, \ldots, x_n)\) 配以下任一范数：

\[ \|x\|_1 = \sum_{i=1}^n |x_i|, \qquad \|x\|_2 = \left(\sum_{i=1}^n |x_i|^2\right)^{1/2}, \qquad \|x\|_\infty = \max_{1 \le i \le n} |x_i|. \]

这三个有限维范数等价（上一章 1.4 节说过）——它们定义同一种收敛、同一种开集。所以在 \(\mathbb{R}^n\) 上选哪个不影响"分析" 结论，只影响常数。

例 2：\(\ell^p\) 序列空间（\(1 \le p < \infty\)）。考虑无穷数列 \(x = (x_1, x_2, x_3, \ldots)\) 满足 \(\sum_{i=1}^\infty |x_i|^p < \infty\)。这个集合叫 \(\ell^p\)，配范数

\[ \|x\|_p = \left(\sum_{i=1}^\infty |x_i|^p\right)^{1/p}. \]

\(p = 2\) 是最重要的特例：\(\ell^2\) 由"平方可和" 数列组成，是上一章反复用的对象。\(p = \infty\) 时定义

\[ \ell^\infty = \{(x_i) : \sup_i |x_i| < \infty\}, \quad \|x\|_\infty = \sup_i |x_i|. \]

"这些真是范数" 不是显然的——三角不等式（在 \(p > 1\) 时）就是 Minkowski 不等式，证明要用到 Hölder 不等式，详细在第 4 章 4.1 节。这里你先接受它。

例 3：\(C[a, b]\) 连续函数空间。区间 \([a, b]\) 上所有连续函数构成向量空间。配 sup 范数

\[ \|f\|_\infty = \sup_{x \in [a, b]} |f(x)|. \]

因 \([a, b]\) 紧、\(f\) 连续，sup 实际上是 max（取得到）。这个范数也叫"一致范数"，因为 \(\|f_n - f\|_\infty \to 0\) 等价于 \(f_n\) 一致收敛到 \(f\)。

\(C[a, b]\) 还可以配其他范数，比如 \(L^1\) 范数 \(\|f\|_1 = \int_a^b |f| \, dx\)。但请记着上一章 1.4 节提醒过的：不同范数定义不同空间。下面 2.4 节会看到 \((C[0, 1], \|\cdot\|_\infty)\) 完备，而 \((C[0, 1], \|\cdot\|_1)\) 不完备——这是同一个 集合配两种范数得到的截然不同 的对象。

例 4：\(L^p(\Omega, \mu)\) 函数空间。设 \((\Omega, \mu)\) 是测度空间。集合

\[ L^p(\Omega) = \{f : \Omega \to \mathbb{R} \text{ 可测, } \int_\Omega |f|^p \, d\mu < \infty\} \big/ \sim \]

（这里 \(\sim\) 表示"几乎处处相等" 的等价关系），配范数

\[ \|f\|_p = \left(\int_\Omega |f|^p \, d\mu\right)^{1/p}. \]

\(p = \infty\) 时 \(\|f\|_\infty = \mathrm{ess\,sup}_\Omega |f|\)。

"为什么要商掉"几乎处处相等"" 是个细节但要紧——若不商掉，"\(\|f\|_p = 0\)" 不蕴含 \(f = 0\)（只蕴含 \(f = 0\) 几乎处处），范数公理失效。商掉后 \(L^p\) 的元素严格说不是函数而是函数等价类——这是实分析读本第 11 章详细处理过的微妙点，我们这里默认。

把这四个例子记下来。后面 22 章里 90% 的具体空间都来自它们，或者它们的稍许变形。

2.3 范数诱导拓扑：开集、收敛、连续

有了范数，自然得到度量；有了度量，自然得到拓扑。用人话：开集是什么、什么叫收敛、什么叫连续——这些问题范数全部已经回答。

具体地：

  • 开球：\(B(x, r) = \{y \in X : \|y - x\| < r\}\)。
  • 开集：\(U \subseteq X\) 开，当且仅当 \(\forall x \in U, \exists r > 0, B(x, r) \subseteq U\)。
  • 闭集：\(F\) 闭 \(\iff X \setminus F\) 开 \(\iff\) "\(F\) 中收敛序列的极限仍在 \(F\) 中"。
  • 收敛：\(x_n \to x\) 意味着 \(\|x_n - x\| \to 0\)。
  • 连续：\(f : X \to Y\)（两边都是赋范空间）在 \(x\) 处连续 \(\iff\) \(\forall \varepsilon > 0, \exists \delta > 0\) 使 \(\|y - x\| < \delta \implies \|f(y) - f(x)\|_Y < \varepsilon\)。

这五条全部和你在实分析里学过的一致——只是把绝对值 \(|\cdot|\) 换成范数 \(\|\cdot\|\)。

有几条专属赋范空间的小事值得停一停。

第一，加法、数乘是连续的。即映射 \((x, y) \mapsto x + y\) 和 \((\alpha, x) \mapsto \alpha x\) 都连续。证明就是三角不等式：\(\|(x_n + y_n) - (x + y)\| \le \|x_n - x\| + \|y_n - y\| \to 0\)。这条听起来理所当然，但它正是赋范空间比一般度量空间多出来的好处——线性运算和拓扑兼容。

第二，范数本身是连续的。即 \(x \mapsto \|x\|\) 是 \(X \to \mathbb{R}\) 的连续函数。证明用反三角不等式：\(|\|x\| - \|y\|| \le \|x - y\|\)。这条让你以后可以放心写"取极限再取范数 = 取范数再取极限"——\(\|x_n\| \to \|x\|\) 当 \(x_n \to x\)。

第三，闭包 = 收敛子序列极限的全体。即 \(\overline{S} = \{x : \exists x_n \in S, x_n \to x\}\)。这条让你以后查一个集合"闭吗" 时只要看它"对取极限封闭吗"——这是上一章 1.3 节"多项式空间不闭" 反例的核心机制。

2.4 Cauchy 序列与完备性

到这里我们已经搭好了"什么叫极限"。下一个问题——也是这一章的中心问题——是："极限存在不存在" 怎么事先知道？

朴素一点的问法：你拿到一列 \(x_1, x_2, x_3, \ldots\)，相邻项之间越来越靠拢，你想说"它们正在收敛到某个东西"。但那个东西 是谁？你没法在不知道极限的情况下说"\(x_n \to x\)"——\(x\) 还没出现呢。

这个尴尬有个标准的回避法。Cauchy 序列的定义只用序列内部的距离信息：

\[ \boxed{\ \{x_n\} \text{ 是 Cauchy} \iff \forall \varepsilon > 0, \exists N, \forall m, n \ge N: \|x_m - x_n\| < \varepsilon.\ } \]

用人话："序列里的元素彼此之间" 越来越靠拢——不需要"极限是谁"。

容易证：每个收敛序列都是 Cauchy。\(x_n \to x\) 蕴含 \(\|x_m - x_n\| \le \|x_m - x\| + \|x - x_n\| < 2\varepsilon\)。

反方向呢？每个 Cauchy 序列都收敛吗？这是个让人不舒服的问题。在 \(\mathbb{R}\) 上答案是是——这正是实数完备性，是构造实数的核心目的之一。但在更一般的赋范空间里，答案可能否。

看一个反例。考虑 \(C[0, 1]\) 配 \(L^1\) 范数 \(\|f\|_1 = \int_0^1 |f| \, dx\)。设

\[ f_n(x) = \begin{cases} 0 & 0 \le x \le \tfrac{1}{2} - \tfrac{1}{n} \\ \text{线性插值} & \tfrac{1}{2} - \tfrac{1}{n} \le x \le \tfrac{1}{2} \\ 1 & \tfrac{1}{2} \le x \le 1 \end{cases} \]

每个 \(f_n\) 都是连续函数（在过渡段线性插值衔接两段常值）。

算一下 \(\|f_m - f_n\|_1\)。两个函数只在中间过渡段不一致，过渡段长度 \(\le \max(1/m, 1/n)\)，差值 \(\le 1\)。所以

\[ \|f_m - f_n\|_1 \le \max(1/m, 1/n) \to 0. \]

这是 Cauchy 列。

那它收敛到谁？随 \(n \to \infty\)，过渡段缩到一点，\(f_n\) "在 \(L^1\) 意义下" 趋向

\[ f(x) = \begin{cases} 0 & 0 \le x < 1/2 \\ 1 & 1/2 \le x \le 1 \end{cases}. \]

这是个阶跃函数——不连续。\(f \notin C[0, 1]\)。

所以这列 \(\{f_n\}\) 是 \(C[0, 1]\) 里的 Cauchy 序列，但没有 \(C[0, 1]\) 中的极限——本来"该是" 极限的那个东西不在空间里。

\[\boxed{\ (C[0, 1], \|\cdot\|_1) \text{ 中存在 Cauchy 序列, 但无极限——空间不完备}.\ }\]

请你停下来感受一下这件事的严重性。\(C[0, 1]\) 是你能想到的最自然 的函数空间——连续函数加 \(L^1\) 距离，没什么花招。可它有缺口。Cauchy 序列在它里面"朝某个东西收敛"，那个东西却不属于它。

这个缺口意味着什么？意味着你不能在这个空间里安心做"取极限" 的论证——你证了一个序列 Cauchy，得出"所以收敛" 的结论，结果极限可能跑出空间。所有依赖"从 Cauchy 到极限" 的标准论证（迭代法、不动点、级数收敛、Cauchy 准则等等）都不能用。

这个性质——"每个 Cauchy 序列都在空间内收敛"——重要到值得被冠名：

\[ \boxed{\ \text{赋范空间 } X \text{ 完备} \iff \text{每个 Cauchy 序列在 } X \text{ 中收敛}.\ } \]

2.5 Banach 空间

把"赋范" 和"完备" 拼起来，就得到泛函分析整本书的中央对象。

定义。一个Banach 空间是完备的赋范空间。

这个定义短到可疑——但你应该能感觉它的分量。Banach 空间是"能放心做分析的最低配置"——加法、数乘、范数、距离全有，而且取极限不会跑出去。这五条加在一起就给你做泛函分析的基本资格。

Banach 空间以波兰数学家 Stefan Banach（1892-1945）命名。他在 1920 年代奠定这门学科的现代框架——他 1932 年的《Théorie des opérations linéaires》是泛函分析的奠基之作，里面 80% 以上的定理至今写在每本教科书的前几章里。

下面是五 类标准 Banach 空间。请你把它们记下来——后面 22 章里出现的 Banach 空间几乎都是这五类的特例或变形。

类 1：有限维 \(\mathbb{R}^n\)（或 \(\mathbb{C}^n\)）。配任何范数都完备。证明用 \(\mathbb{R}\) 完备 + 范数等价（任何有限维范数都和 \(\|\cdot\|_2\) 等价）：Cauchy 列每个分量 Cauchy，每个分量在 \(\mathbb{R}\) 中收敛，于是整个向量收敛。

这条已经告诉你有限维空间永远是 Banach 空间。所以"不完备" 这个毛病只在无穷维 出现——有限维拿不出反例。这呼应上一章 1.4 节"有限维范数都等价" 的同类性质：有限维空间太"结实"，分析层面的病基本都不出现。

类 2：\(\ell^p\)（\(1 \le p \le \infty\)）。序列空间。完备性证明的标准模板：

  • 取 Cauchy 列 \(\{x^{(n)}\}\)，每个 \(x^{(n)} = (x^{(n)}_1, x^{(n)}_2, \ldots)\)。
  • 对每个固定 \(i\)，\(\{x^{(n)}_i\}_{n}\) 是 \(\mathbb{R}\) 中的 Cauchy 列（因 \(|x^{(n)}_i - x^{(m)}_i| \le \|x^{(n)} - x^{(m)}\|_p\)）。
  • 所以每个分量 \(x^{(n)}_i \to x_i\)。把候选极限拼起来：\(x = (x_1, x_2, \ldots)\)。
  • 验证 \(x \in \ell^p\)（用 Cauchy 性 + Fatou 引理或类似的极限工具）。
  • 验证 \(\|x^{(n)} - x\|_p \to 0\)。

这个"逐分量收敛 → 拼极限 → 验证在空间里 → 验证范数收敛" 的四步套路是大多数函数 / 序列空间完备性证明的模板。具体细节在第 4 章 4.1 节。

类 3：\((C[a, b], \|\cdot\|_\infty)\)。连续函数配 sup 范数。完备性的核心是：

  • \(\|f_n - f_m\|_\infty \to 0\) 等价于 \(\{f_n\}\) 一致 Cauchy。
  • 一致 Cauchy ⟹ 一致收敛到某极限函数 \(f\)。
  • 一致极限保连续性（这是实分析里的标准定理）——所以 \(f\) 仍连续，即 \(f \in C[a, b]\)。

对比上面 2.4 节的反例：同一个集合 \(C[0, 1]\)，配 sup 范数完备，配 \(L^1\) 范数不完备。"完备" 是范数和集合合起来 的性质，单看哪一个都判不了。

类 4：\(L^p(\Omega, \mu)\)（\(1 \le p \le \infty\)）。这是 Riesz-Fischer 定理（1907）的内容——\(L^p\) 完备。证明用 Lebesgue 控制收敛定理 + 单调收敛定理，详细在实分析读本第 12 章。

请你把这条结论记牢：\(L^p\) 完备正是 Lebesgue 测度比 Riemann 测度优越的核心理由。Riemann 可积函数空间配 \(L^p\) 范数不完备——你用 Riemann 积分搭出的 Cauchy 序列经常收敛到非 Riemann 可积的极限。Lebesgue 测度的发明（Lebesgue 1902）部分动机就是把这个缺口填上。"\(L^p\) 完备" 这条结论是 20 世纪分析学的基石之一。

类 5：\(C^k[a, b]\) 配合适范数。\(k\) 次连续可微函数空间，配范数

\[ \|f\|_{C^k} = \sum_{j=0}^k \|f^{(j)}\|_\infty. \]

完备性源于：在 \(C^k\) 中 \(\{f_n\}\) Cauchy ⟺ 每个 \(\{f_n^{(j)}\}\)（\(j = 0, 1, \ldots, k\)）在 sup 范数下 Cauchy ⟹ 每个一致收敛 ⟹ 用"一致收敛 + 导数序列一致收敛 ⟹ 极限函数可微" 的标准定理推出极限 \(f\) 是 \(C^k\)。

请你注意这五类例子的共性：每一个的完备性都不是免费的。要么靠 \(\mathbb{R}\) 的完备性"逐分量传上去"（类 1, 2），要么靠"一致收敛保连续 / 可微" 这种实分析定理（类 3, 5），要么靠 Lebesgue 积分理论（类 4）。这是为什么我说"完备性是底层资源"——它的存在背后总有某条更深的实分析定理 在支撑。

反过来：找一个赋范空间不完备很容易——上面 2.4 节那个 \((C[0, 1], \|\cdot\|_1)\) 的反例就够日常使用。证明一个赋范空间完备很难——通常要专门写一节证明。这种"构造容易, 验证难" 的不对称是为什么 Banach 空间这个概念值得单独命名——"这是个 Banach 空间" 是个非平凡 的断言。

2.6 完备化：把缺口填上

如果你手上有一个不完备 的赋范空间 \(X\)，比如上面 \((C[0, 1], \|\cdot\|_1)\)，能不能"把它的缺口补满" 得到一个 Banach 空间？答案是能——这个过程叫完备化。

定理（完备化存在唯一）。设 \(X\) 是赋范空间。则存在一个 Banach 空间 \(\hat X\) 和一个保范单射 \(\iota : X \hookrightarrow \hat X\)，使 \(\iota(X)\) 在 \(\hat X\) 中稠密。这样的 \(\hat X\) 在保范同构意义下唯一。

构造的思路（详细在附录 A）：取 \(X\) 中 Cauchy 序列的全体，对"差为零序列" 等价化，得到的等价类集合就是 \(\hat X\)。这个构造和 Cantor 用 Cauchy 序列构造实数 一模一样——\(\mathbb{R}\) 就是 \(\mathbb{Q}\) 的完备化。

对 \((C[0, 1], \|\cdot\|_1)\) 完备化，你猜得到结果是谁吗？答案是 \(L^1[0, 1]\)——Lebesgue 可积函数空间。

这件事让人停一停。\(L^1[0, 1]\) 不是抽象天上掉下来的对象——它就是把"连续函数 + \(L^1\) 距离" 这个朴素空间的缺口补满的结果。Lebesgue 积分的发明部分就是为了显式构造这个补满，给"\(L^1\) 函数" 一个看得见摸得着的解释。

同理：

  • \(L^p[0, 1]\) 是 \((C[0, 1], \|\cdot\|_p)\) 的完备化。
  • Sobolev 空间 \(W^{k, p}\)（第 23 章）是 \(C^\infty\) 配适当 Sobolev 范数的完备化。
  • 更一般的 Bochner 空间、抽象 \(L^p\) 都按同一模式构造。

这告诉你"完备化" 不是个奇技淫巧，是整个现代分析创造新空间的标准方法。每次你写下一个新的范数 \(\|\cdot\|_*\)，你立刻得到一个新的 Banach 空间——它是某个朴素函数类（比如 \(C^\infty\) 或 \(C[a, b]\)）在这范数下的完备化。Sobolev 空间、Besov 空间、Triebel-Lizorkin 空间、Hardy 空间、BMO——所有这些 20 世纪发明的"新空间" 都是这个套路的产物。

这一章我们到这里。下一章我们追问一个被有意搁下的问题：Banach 空间够不够？

答案是不够——它装下了"距离"，但没装下"角度"。在 \(\mathbb{R}^n\) 里你能问"这两个向量正交吗"，因为有内积。要在无穷维继续问这个问题，必须有内积——这就是 Hilbert 空间。

完备性会作为后台资源继续在场——Hilbert 空间是完备 的内积空间，Banach 是完备 的赋范空间。"完备" 这个词从这一章开始永远不再脱手。
