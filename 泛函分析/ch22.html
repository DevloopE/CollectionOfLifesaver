第 22 章 Fredholm 选择性

请你回想线性代数里"方阵的选择性" 这条铁律。给 \(n \times n\) 方阵 \(A\)，要解 \(A x = y\)。结论很干脆：

下面三条等价：

  1. \(A x = 0\) 只有平凡解（即 \(\ker A = 0\)）。
  2. \(A x = y\) 对每个 \(y\) 都有唯一解。
  3. \(A\) 满秩 / 可逆。

这是有限维线性代数的选择性——要么齐次方程只零解 + 非齐次方程总可解 + 解唯一，要么齐次有非零解 + 非齐次只对某些 \(y\) 可解 + 非齐次解空间是仿射子空间。两种情形二选一。

有限维这条来自维数论："\(\dim \ker A + \dim \mathrm{range}\, A = n\)"——所以核大 \(\iff\) 像小，反之亦然。在无穷维，维数论失效（一边无穷一边无穷不能减），所以这条选择性一般不成立。比如第 1 章 1.5 的右移算子：单射但不满射——既不属于"齐次唯一" 也不属于"非齐次总有解"，根本破了选择性。

但 1903 年 Fredholm 在研究积分方程时观察到一件惊人的事：对方程 \(T x - \lambda x = y\)，其中 \(T\) 是紧算子且 \(\lambda \ne 0\)，有限维选择性原样恢复。这条恢复让 PDE 弱解理论 + 积分方程 + 边值问题都被同一套抽象工具统一处理——所以 Fredholm 选择性常被称为"20 世纪上半叶分析最优雅定理"。

本章把这条选择性正式陈述、给完整证明（六步：Riesz 引理 → 核有限维 → 像闭 → 像有限余维 → 指数 0 → 选择性）、再给三个标准应用。中心结果：

\[ \boxed{\ T \in \mathcal{K}(X), \lambda \ne 0 \implies \ker(T - \lambda I) \cong \mathrm{coker}(T - \lambda I) \text{ 维数有限且相等。}\ } \]

22.1 Fredholm 选择性陈述

设 \(X\) 是 Banach，\(T \in \mathcal{K}(X)\) 紧，\(\lambda \in \mathbb{C} \setminus \{0\}\)。记 \(A := T - \lambda I\)（注意符号——是 \(T\) 减 \(\lambda I\)，不是 \(\lambda I - T\)；两者只差一个符号，不影响结论但要一致）。

用有限维语言 描述选择性，下面六条等价：

  1. 齐次唯一：\(A x = 0 \implies x = 0\)，即 \(\ker A = 0\)。
  2. 非齐次有解：\(A x = y\) 对每个 \(y \in X\) 有解。
  3. 双射：\(A\) 是双射。
  4. 可逆：\(A^{-1}\) 存在且有界。
  5. \(\lambda \notin \sigma(T)\)。
  6. \(\dim \ker A = \dim \mathrm{coker}\, A\) 等于 0（\(\mathrm{coker}\, A := X / \mathrm{range}\, A\)）。

注：(1) ⟺ (2) ⟺ ... ⟺ (5) 是典型 Fredholm 形态——"齐次唯一" 这件本来不一定蕴含"非齐次有解" 的事，在紧扰动下被迫等价。

更强的陈述（"定量 形式"）：

定理（Fredholm 选择性 + 指数 0）。设 \(T \in \mathcal{K}(X), \lambda \ne 0\)，\(A = T - \lambda I\)。则：

  1. \(\dim \ker A < \infty\)。
  2. \(\mathrm{range}\, A\) 在 \(X\) 中闭。
  3. \(\dim \mathrm{coker}\, A < \infty\)。
  4. \(\dim \ker A = \dim \mathrm{coker}\, A\)（即 \(A\) 是指数 0 的 Fredholm 算子）。

从 (4) 立即得"(1) ⟺ (2)"——核为 0 \(\iff\) cokernel 为 0 \(\iff\) range 满。

这条定理的定性意义：紧扰动 \(T\) 把恒等 \(I\) 变成 \(I + (T - I) \cdot \) 系数，但不改变 它的"指数"。指数是个拓扑不变量——衡量"算子离可逆有多远"——紧扰动是它的"不动操作"。

本章余下五节就是上面四步的证明 + 应用。

22.2 Riesz 引理与"无穷维单位球不紧"

证明 Fredholm 需要一个细节工具：在赋范空间里，如果有一个真闭子空间，可以找到"差不多远" 单位向量与它隔开。这就是Riesz 引理。

Riesz 引理。设 \(X\) 赋范，\(Y \subsetneq X\) 是闭真 子空间。则对任意 \(0 < \delta < 1\)，存在 \(z \in X\) 满足 \(\|z\| = 1\) 且 \(\mathrm{dist}(z, Y) := \inf_{y \in Y} \|z - y\| \ge 1 - \delta\)。

"差不多远"——你想从 \(Y\) 隔开 \(1 - \delta\)，可以；隔开满 \(1\) 一般做不到（除非 \(X\) 是 Hilbert，第 6 章正交分解给）。Banach 上只能差一点点。

证明。取 \(x \in X \setminus Y\)（存在因 \(Y \neq X\)）。设 \(d := \mathrm{dist}(x, Y) > 0\)（\(Y\) 闭 + \(x \notin Y\) \(\implies\) \(d > 0\)）。

由下确界定义，存在 \(y_0 \in Y\) 使 \(d \le \|x - y_0\| \le d / (1 - \delta)\)。设

\[ z := \frac{x - y_0}{\|x - y_0\|}. \]

\(\|z\| = 1\)。对任意 \(y \in Y\)：

\[ \|z - y\| = \frac{\|x - y_0 - \|x - y_0\| y\|}{\|x - y_0\|} = \frac{\|x - (y_0 + \|x - y_0\| y)\|}{\|x - y_0\|}. \]

\(y_0 + \|x - y_0\| y \in Y\)（\(Y\) 是子空间）\(\implies\) 分子 \(\ge d\)。所以

\[ \|z - y\| \ge \frac{d}{\|x - y_0\|} \ge \frac{d}{d / (1 - \delta)} = 1 - \delta. \]

对所有 \(y\) 取下确界：\(\mathrm{dist}(z, Y) \ge 1 - \delta\)。\(\square\)

推论（无穷维单位球不紧）。设 \(X\) 是无穷维赋范。则 \(\overline{B}_X(0, 1)\) 不紧。等价地：赋范空间单位球紧 \(\iff\) 空间有限维。

证明。"\(\Leftarrow\)" 是 Heine-Borel。"\(\Rightarrow\)" 用 Riesz 反证。设 \(X\) 无穷维。任取 \(x_1\) 单位向量，\(Y_1 := \mathrm{span}\{x_1\}\)（一维 \(\implies\) 闭真子空间——任何有限维子空间在 Banach 上自动闭）。Riesz 给 \(x_2\) 单位、\(\mathrm{dist}(x_2, Y_1) \ge 1/2\)。

设 \(Y_2 := \mathrm{span}\{x_1, x_2\}\)（二维闭真）。Riesz 给 \(x_3\) 单位、\(\mathrm{dist}(x_3, Y_2) \ge 1/2\)。

...递归。得无穷序列 \(\{x_n\}\) 单位向量、两两距离 \(\|x_n - x_m\| \ge 1/2\)（\(n < m\) 时 \(x_m\) 距 \(Y_{m-1} \supset \{x_n\}\) 远 \(1/2\)）。无 Cauchy 子列 \(\implies\) 单位球不紧。\(\square\)

这条推论第 1 章 1.1 节用过——\(\ell^2\) 标准基直接验证。Riesz 引理把这条推广到任何 无穷维赋范空间，包括 Banach 不带内积的情况。

22.3 \(\ker(T - \lambda I)\) 有限维

第一步证明。\(N := \ker A = \ker(T - \lambda I) = \{x : T x = \lambda x\}\)。即 \(\lambda\) 的特征空间。

断言：\(N\) 有限维。

证明。看 \(T\) 限制到 \(N\) 上。对 \(x \in N\)，\(T x = \lambda x \implies T|_N = \lambda I_N\)。即 \(T\) 在 \(N\) 上是纯量乘子。

\(T\) 紧 \(\implies T|_N\) 紧（紧算子的限制紧——序列论证立得）。但 \(T|_N = \lambda I_N\) 是 \(N\) 上的恒等算子的 \(\lambda\) 倍——它紧 \(\iff\) 恒等算子 \(I_N\) 紧 \(\iff N\) 单位球紧 \(\iff N\) 有限维（22.2 节推论）。

\(\lambda \ne 0\) \(\implies\) \(\lambda I_N\) 紧 \(\implies I_N\) 紧 \(\implies N\) 有限维。\(\square\)

这条推理的核心："紧算子的特征空间必有限维"。证明走了"限制紧 + 限制是非零纯量乘 + 单位球紧 \(\iff\) 有限维" 的链。每一步都用上了我们之前 22 章里建好的工具，串起来非常优雅。

\[\boxed{\ T \text{ 紧}, \lambda \ne 0 \implies \dim \ker(T - \lambda I) < \infty.\ }\]

22.4 \(\mathrm{range}(T - \lambda I)\) 闭

第二步——比第一步深得多。设 \(R := \mathrm{range}\, A = (T - \lambda I) X\)。

断言：\(R\) 闭。

这件事不显然——一般来说有界算子的像不必闭（举反例：在 \(\ell^2\) 上 \(T(x_n) = (x_n / n)\) 是有界、像是 \(\ell^2\) 的稠密真子空间，不闭）。但 \(T - \lambda I\)（\(T\) 紧 \(\lambda \ne 0\)）例外。

证明。设 \(y_n \in R, y_n \to y\)。要证 \(y \in R\)。

每个 \(y_n = (T - \lambda I) x_n\) 某 \(x_n\)。技巧：可以选 \(x_n\) 使 \(\|x_n\|\) 受控——具体的，

\[ \|x_n\| \le 2 \cdot \mathrm{dist}(x_n, \ker A). \]

这需要对每个 \(x_n\) 做商空间替换：替换 \(x_n\) 为 \(x_n - z\) 某 \(z \in \ker A\)（不改 \(A x_n\)），使 \(\|x_n - z\| \le 2 \mathrm{dist}(x_n, \ker A)\)（取近似最近点）。重命名替换后的仍叫 \(x_n\)。

断言：\(\{x_n\}\) 有界。

反证。假设 \(\|x_n\| \to \infty\)（取子列）。设 \(u_n := x_n / \|x_n\|\)（单位向量）。\(A u_n = y_n / \|x_n\| \to 0 / \infty = 0\)。即 \((T - \lambda I) u_n \to 0\)，等价 \(T u_n - \lambda u_n \to 0\)。

\(T\) 紧 + \(\{u_n\}\) 有界 \(\implies\) 抽子列 \(T u_{n_k} \to z\) 强（某 \(z\)）。结合：\(\lambda u_{n_k} = T u_{n_k} - (T u_{n_k} - \lambda u_{n_k}) \to z - 0 = z\)，所以 \(u_{n_k} \to z / \lambda\)。

\(\|u_{n_k}\| = 1 \implies \|z / \lambda\| = 1 \implies z \ne 0\)。

对 \(u_{n_k} \to z / \lambda\) 用 \(T\) 连续传递：\(T u_{n_k} \to T(z / \lambda)\)。又 \(T u_{n_k} \to z \implies z = T(z / \lambda) \implies T z = \lambda z \implies z \in \ker A\)。

但 \(u_{n_k} \to z / \lambda\) 又要求 \(\mathrm{dist}(u_{n_k}, \ker A) \to 0\)（因为 \(z / \lambda \in \ker A\)）。这与 \(\|x_n\| \le 2 \mathrm{dist}(x_n, \ker A)\) 即 \(\mathrm{dist}(u_n, \ker A) \ge 1/2\) 矛盾。

所以 \(\{x_n\}\) 有界。

断言完成。\(\{x_n\}\) 有界 + \(T\) 紧 \(\implies\) 抽子列 \(T x_{n_k} \to w\) 强。\(\lambda x_{n_k} = T x_{n_k} - y_{n_k} \to w - y\)。所以 \(x_{n_k} \to (w - y) / \lambda =: x\)。\(T\) 连续 \(\implies T x_{n_k} \to T x = w\)。所以 \(A x = T x - \lambda x = w - \lambda \cdot (w - y) / \lambda = w - w + y = y\)。即 \(y \in R\)。\(\square\)

请你停下来欣赏这条证明的结构。它两次用紧性抽子列：

  1. 第一次（反证里）：证 \(\{x_n\}\) 有界。技巧是把 \(\|x_n\| \to \infty\) 假设转为 \(\|u_n\| = 1\) 的归一化，然后让"\(A u_n \to 0\) + \(u_n \to z/\lambda\) + \(z \in \ker A\)" 撞上 "\(\mathrm{dist}(u_n, \ker A) \ge 1/2\)" 矛盾。
  2. 第二次（正面）：从 \(\{x_n\}\) 有界抽子列得 \(T x_{n_k}\) 收敛，从而 \(x_{n_k}\) 收敛，从而 \(y \in R\)。

商空间技巧（"替换 \(x_n\) 让 \(\mathrm{dist}(x_n, \ker A)\) 占主导"）非常典型。它让 \(\ker A\) 不阻碍我们的论证——把 \(x_n\) 投影到 \(X / \ker A\) 后再回到 \(X\)。

22.5 完成 Fredholm + 指数 = 0

22.3 + 22.4 给了 (1)、(2) 两条。剩下 (3) \(\dim \mathrm{coker}\, A < \infty\) 与 (4) \(\dim \ker A = \dim \mathrm{coker}\, A\)。

这两条用对偶 论证。要点：\(A^* = T^* - \lambda I\)（在 \(X^*\) 上），\(T^*\) 也紧（紧算子的对偶紧——Schauder 定理，第 20 章不证但用）。所以对 \(A^*\) 同样可应用 (1) 和 (2)。

证明 (3)。由 Hahn-Banach 一致刻画：

\[ \mathrm{coker}\, A = X / \mathrm{range}\, A \cong (\ker A^*)^* \quad (\text{对偶身份}) \]

更具体的：商空间 \(X / R\) 的对偶是 \(R^\perp = \{\varphi \in X^* : \varphi|_R = 0\}\)，而 \(R^\perp = \ker A^*\)（因为 \(\varphi \in R^\perp \iff \varphi(A x) = 0 \forall x \iff (A^* \varphi)(x) = 0 \forall x \iff A^* \varphi = 0\)）。所以

\[ \dim X / R = \dim (X / R)^* = \dim R^\perp = \dim \ker A^*. \]

由 22.3 应用到 \(A^* = T^* - \lambda I\)（\(T^*\) 紧、\(\lambda \ne 0\)）：\(\dim \ker A^* < \infty\)。所以 \(\dim \mathrm{coker}\, A < \infty\)。

证明 (4)。设 \(n := \dim \ker A, m := \dim \mathrm{coker}\, A = \dim \ker A^*\)。要证 \(n = m\)。

这步需要更精细的论证（标准证明走升序核 链 \(\ker A \subset \ker A^2 \subset \ldots\) 必有限稳定，给"广义核" 维数与 cokernel 维数相等）。完整证明 2-3 页技术，思路是：

  • 构造从 \(\ker A\) 到 \(\mathrm{coker}\, A\) 的线性映射，并证它是同构。这个映射不是平凡的——典型的是用 \(A\) 的 Riesz-Schauder 分解：\(X = N \oplus M\)，其中 \(N\) 是广义核（有限维），\(M = R\) 是 range（闭）。\(A|_M : M \to M\) 是双射，\(A|_N : N \to N\) 是幂零。
  • 从这个分解直接读出 \(\dim N = \dim X / R\)（在广义 意义）+ 进一步精化到原始核 vs. cokernel 维数。

详细技术请见 Brezis "Functional Analysis"（2011）第 6 章 Theorem 6.6 或 Conway "A Course in Functional Analysis" Chap. XI Sec. 4。本章默认这条结论。

组合 22.3 + 22.4 + (3) + (4)：

\[\boxed{\ T \text{ 紧}, \lambda \ne 0 \implies T - \lambda I \text{ 是指数 } 0 \text{ Fredholm 算子。}\ }\]

"指数 \(0\)" 自动给"选择性"——核为 0 \(\iff\) cokernel 为 0 \(\iff\) range 满 \(\iff\) 双射 \(\iff \lambda \notin \sigma(T)\)。

注意 (4) 在有限维 是显然的（来自维数论 \(\dim \ker A + \dim R = \dim X\)）。无穷维不能用维数论但仍然成立——这条事实就是 Fredholm 的"奇迹"。它告诉你紧扰动 \(T\) 不改变恒等的"指数 0" 性质。这是更普遍的"指数定理"（Atiyah-Singer）的最基础例子，下面 22.6 节末端会再提。

22.6 三个标准应用

应用 1：Fredholm 积分方程

这是 Fredholm 1903 原始动机。设 \(K \in C([a, b]^2)\)，方程

\[ \varphi(x) - \int_a^b K(x, y) \varphi(y) dy = f(x) \]

是著名的第二类 Fredholm 积分方程。改写：\((I - T_K) \varphi = f\)，\(T_K\) 是 Fredholm 积分算子（第 20 章 20.4 例 2 紧）。

选择性立即给：

  • "齐次唯一"：\(\varphi - T_K \varphi = 0\) 只零解。
  • "非齐次有解"：每个 \(f \in C[a, b]\) 给唯一 \(\varphi\)。

等价。所以你只需检查齐次方程——若它只零解，立即得每个非齐次方程有唯一解。这是 Fredholm 选择性最直接的实用价值——把"存在性" 转化为"唯一性"，唯一性常常容易得多（用能量估计、最大原理等）。

应用 2：椭圆 PDE 选择性

设 \(\Omega \subset \mathbb{R}^d\) 有界，研究椭圆方程

\[ -\Delta u + c(x) u = f \text{ in } \Omega, \quad u = 0 \text{ on } \partial \Omega. \]

其中 \(c \in L^\infty(\Omega)\)。改写：\((I + c \cdot (-\Delta)^{-1}) v = (-\Delta)^{-1} f\)（设 \(v = (-\Delta)^{-1/2}\) 之类——实际重写技术省略，思路是把 PDE 转为 \((I - K) v = g\) 形式）。

\(c \cdot (-\Delta)^{-1}\) 是紧算子（\(L^\infty\) 乘是有界，\((-\Delta)^{-1}\) 紧——第 21 章 21.5 例 2）。所以 \(I + c (-\Delta)^{-1} = I - (-c (-\Delta)^{-1})\) 适用 Fredholm 选择性。

结论：椭圆方程要么齐次解空间是有限维（共振情形），要么非齐次方程对每个 \(f\) 有唯一解（非共振情形）。这是椭圆 PDE 标准存在唯一性定理的源泉，所有 Lax-Milgram 和共振分析都从这里长出。

应用 3：Toeplitz 算子 + 指标定理预告

"指数 0" 是个非常特殊的事——一般 Fredholm 算子可以有非零 指数。最经典的例子是 Toeplitz 算子。

设 \(\mathbb{T}\) 是单位圆，\(L^2(\mathbb{T})\) 是它上的平方可积函数。\(H^2(\mathbb{T})\) 是 Hardy 空间——傅里叶系数 \(\hat f(n) = 0\) 对 \(n < 0\) 的子空间。设 \(P : L^2 \to H^2\) 是正交投影。

给 \(\varphi \in L^\infty(\mathbb{T})\)（连续就行）。Toeplitz 算子 \(T_\varphi : H^2 \to H^2\) 定义为

\[ T_\varphi f := P (\varphi \cdot f). \]

\(T_\varphi\) 一般不紧——但 Atkinson 定理告诉你 \(T_\varphi\) 是Fredholm（即 \(\dim \ker, \dim \mathrm{coker}\) 都有限），当且仅当 \(\varphi\) 在圆上不为 0。

更深的结论（Gohberg-Krein 1957，Atiyah-Singer 1963 的最简例）：

\[ \boxed{\ \mathrm{ind}(T_\varphi) := \dim \ker T_\varphi - \dim \mathrm{coker}\, T_\varphi = -\mathrm{wind}(\varphi).\ } \]

"\(\mathrm{wind}(\varphi)\)" 是 \(\varphi : \mathbb{T} \to \mathbb{C}^*\) 的缠绕数（绕原点几圈）。

这条公式是指标定理（Atiyah-Singer 1963 的一维版本）的最简单形态——它把分析量（核与 cokernel 维数差）与拓扑量（缠绕数）等同。这种"分析 = 拓扑" 的桥梁是 20 世纪后半叶数学最深的故事之一——延伸到流形上的椭圆算子（Atiyah-Singer 完整版）、到代数几何的 Riemann-Roch、到弦论的 anomaly 计算。Fredholm 选择性是这条故事链的第一环——它给"指数" 这个概念在最简单设置（紧扰动）下的取值（恒等的指数 0）。

下一章把上面这些抽象工具组合起来，给经典椭圆 PDE \(-\Delta u + u = f\) 一个干净的弱解理论——会把 Sobolev 空间、Lax-Milgram、Sobolev 嵌入、Rellich 紧嵌入这一整套机器拼装起来。"紧致论证" 三步法（弱抽 + 紧升 + 强传）将在那里得到最完整的应用。
