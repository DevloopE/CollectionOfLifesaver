第 3 章 内积与 Hilbert 空间

上一章我们把 Banach 空间装好了。它已经够装很多分析——加法、数乘、距离、极限、闭子空间、连续算子，全在场。可你拿着一个 Banach 空间想做几何，会发现它缺一样东西。

请你先在 \(\mathbb{R}^2\) 上做一道朴素的事。给两个向量 \(u = (1, 0)\) 和 \(v = (0, 1)\)。问：它们正交 吗？

你心里立刻答"正交"。你怎么知道的？因为你算了内积 \(u \cdot v = 1 \cdot 0 + 0 \cdot 1 = 0\)。"正交" 这个词在你脑子里就是"内积为零" 的同义词。

请你停下来问：没有内积，这个问题怎么问？

你看一个朴素的 Banach 空间——比如 \(\ell^\infty\)，配 sup 范数。给两个序列 \(u = (1, 0, 0, \ldots)\) 和 \(v = (0, 1, 0, \ldots)\)。它们正交 吗？

这个问题在 \(\ell^\infty\) 里没法问。\(\|u\|_\infty = 1\)、\(\|v\|_\infty = 1\)、\(\|u + v\|_\infty = 1\)——勾股定理 \(\|u + v\|^2 = \|u\|^2 + \|v\|^2\) 这种关系不成立（\(1 \neq 1 + 1\)）。"角度" 这个概念在 \(\ell^\infty\) 上根本不存在。

这是个让人不舒服的发现。Banach 空间的所有线性算子、对偶、谱、紧致性——这些"分析" 工具都还在；但"几何" 工具——正交、投影、勾股、Fourier 展开——都不在。要把它们要回来，必须给空间装一个内积。

这一章建立这层结构。最终的对象——内积空间 + 完备——就是 Hilbert 空间。它是泛函分析里"最贵最好用" 的几何对象。

3.1 内积公理

设 \(X\) 是 \(\mathbb{R}\)（实情形）或 \(\mathbb{C}\)（复情形）上的向量空间。一个内积是映射 \(\langle \cdot, \cdot \rangle : X \times X \to \mathbb{F}\)（\(\mathbb{F}\) 是底域），满足：

  • \(\langle x, x \rangle \ge 0\)，且 \(\langle x, x \rangle = 0 \iff x = 0\)（正定）
  • \(\langle x, y \rangle = \overline{\langle y, x \rangle}\)（共轭对称——实情形就是对称）
  • \(\langle \alpha x + \beta y, z \rangle = \alpha \langle x, z \rangle + \beta \langle y, z \rangle\)（第一变量线性）

第二条加第三条蕴含第二变量共轭线性：\(\langle x, \alpha y + \beta z \rangle = \bar\alpha \langle x, y \rangle + \bar\beta \langle x, z \rangle\)。在复情形这个共轭不能省——省了正定性会塌（你能验证 \(\langle x, x \rangle = \overline{\langle x, x \rangle}\) 蕴含它实，所以 \(\ge 0\) 才有意义）。

从内积自然得到范数：

\[ \|x\| := \sqrt{\langle x, x \rangle}. \]

"这真是范数" 不是显然的——三角不等式需要 Cauchy-Schwarz，下面 3.2 节证。先把这条记号约定下来。

三个标准例子。

例 1：\(\mathbb{R}^n\)（或 \(\mathbb{C}^n\)）。标准内积

\[ \langle x, y \rangle = \sum_{i=1}^n x_i \overline{y_i}. \]

诱导范数 \(\|x\|_2 = \sqrt{\sum |x_i|^2}\)，就是欧氏范数。这是内积空间的"原型"——你高中学的"点乘" 就是这个。

例 2：\(\ell^2\)。对 \(x, y \in \ell^2\)，定义

\[ \langle x, y \rangle = \sum_{i=1}^\infty x_i \overline{y_i}. \]

这个级数为什么收敛？由 Cauchy-Schwarz（在 \(\mathbb{R}^N\) 的有限部分和上用），\(|\sum_{i=1}^N x_i \overline{y_i}| \le \|x\|_2 \cdot \|y\|_2 < \infty\)，部分和绝对值有界。事实上 \(|x_i \overline{y_i}| \le (|x_i|^2 + |y_i|^2)/2\)，所以级数绝对收敛。

这个内积诱导的范数就是 \(\ell^2\) 范数 \(\|x\|_2 = \sqrt{\sum |x_i|^2}\)——和上一章 2.2 节例 2 中 \(p = 2\) 的范数一致。"\(\ell^p\) 配 \(p\)-范数" 中只有 \(p = 2\) 的范数来自内积——这是 \(\ell^2\) 比其他 \(\ell^p\) 享受特殊地位的根源（详见 3.3 节）。

例 3：\(L^2(\Omega, \mu)\)。对可测函数 \(f, g\) 满足 \(\int |f|^2, \int |g|^2 < \infty\)，定义

\[ \langle f, g \rangle = \int_\Omega f \overline{g} \, d\mu. \]

诱导 \(\|f\|_2 = (\int |f|^2)^{1/2}\)。这又是上一章 2.2 节例 4 中 \(p = 2\) 时的范数。

同样的现象：\(L^2\) 是唯一来自内积的 \(L^p\)——其他 \(p\) 的范数你怎么也凑不出一个内积。这是 \(L^2\) 在 PDE、量子力学、信号处理里被反复偏爱的根本原因。

3.2 Cauchy-Schwarz 不等式

有了内积，第一条 大不等式就来了：

定理（Cauchy-Schwarz）。对内积空间中任意 \(x, y\)，

\[ \boxed{\ |\langle x, y \rangle| \le \|x\| \cdot \|y\|.\ } \]

等号成立当且仅当 \(x, y\) 线性相关。

证明。设 \(y \neq 0\)（否则两边都 0，平凡）。对任意标量 \(\lambda\)：

\[ 0 \le \|x - \lambda y\|^2 = \langle x - \lambda y, x - \lambda y \rangle = \|x\|^2 - \lambda \overline{\langle x, y \rangle} - \bar\lambda \langle x, y \rangle + |\lambda|^2 \|y\|^2. \]

选 \(\lambda = \langle x, y \rangle / \|y\|^2\)（让二次表达式取最小）。代入得

\[ 0 \le \|x\|^2 - \frac{|\langle x, y \rangle|^2}{\|y\|^2}. \]

整理：\(|\langle x, y \rangle|^2 \le \|x\|^2 \|y\|^2\)。开方完事。

这条证明的几何意思你应该看出来：\(\lambda y\) 是"\(x\) 在 \(y\) 方向上的投影系数"，\(x - \lambda y\) 是"\(x\) 减去投影后的残差"。"残差范数 \(\ge 0\)" 这个朴素事实就给 Cauchy-Schwarz——它本质是投影误差非负 的代数化。

Cauchy-Schwarz 的第一个推论：诱导范数真的 是范数。三角不等式：

\[ \|x + y\|^2 = \|x\|^2 + \langle x, y \rangle + \langle y, x \rangle + \|y\|^2 \le \|x\|^2 + 2 \|x\| \|y\| + \|y\|^2 = (\|x\| + \|y\|)^2. \]

开方得 \(\|x + y\| \le \|x\| + \|y\|\)。Cauchy-Schwarz 是这一步的关键——上一章我们对 \(\ell^p, L^p\) 的三角不等式（Minkowski）也类似要用 Hölder（Cauchy-Schwarz 的推广）。

第二个推论：内积本身连续。即映射 \((x, y) \mapsto \langle x, y \rangle\) 连续。证：

\[ |\langle x_n, y_n \rangle - \langle x, y \rangle| \le |\langle x_n - x, y_n \rangle| + |\langle x, y_n - y \rangle| \le \|x_n - x\| \|y_n\| + \|x\| \|y_n - y\|. \]

当 \(x_n \to x, y_n \to y\)，\(\|y_n\|\) 有界，所以右边 \(\to 0\)。这条让你后面可以放心写"取极限可以钻进内积里"——\(\langle \lim x_n, y \rangle = \lim \langle x_n, y \rangle\)。

3.3 平行四边形恒等式：内积的"指纹"

给一个赋范空间 \((X, \|\cdot\|)\)，怎么知道它的范数能不能 来自某个内积？

这个问题非常具体——而且有非常优雅 的回答。先看一条几何观察。

在 \(\mathbb{R}^2\) 里画两个向量 \(u, v\)，画平行四边形（边是 \(u, v\) 的两条邻边）。两条对角线分别是 \(u + v\) 和 \(u - v\)。中学几何告诉你"对角线平方和等于四条边平方和"：

\[ \|u + v\|^2 + \|u - v\|^2 = 2 \|u\|^2 + 2 \|v\|^2. \]

这条叫平行四边形恒等式。它在任何 内积空间里都成立——证明就是把两个范数平方展成内积然后相加：

\[ \|u + v\|^2 + \|u - v\|^2 = (\|u\|^2 + 2\Re\langle u, v\rangle + \|v\|^2) + (\|u\|^2 - 2\Re\langle u, v\rangle + \|v\|^2) = 2\|u\|^2 + 2\|v\|^2. \]

所以"能从内积来的范数" 必须满足平行四边形恒等式。问题是：反方向呢？

定理（Jordan-von Neumann, 1935）。设 \((X, \|\cdot\|)\) 是赋范空间。则范数 \(\|\cdot\|\) 来自某个内积，当且仅当 \(\|\cdot\|\) 满足平行四边形恒等式。

" \(\Leftarrow\)" 部分的内积可以由极化恒等式 显式重构：

\[ \langle u, v \rangle = \frac{1}{4}\big(\|u + v\|^2 - \|u - v\|^2\big) \quad \text{(实情形)} \]

\[ \langle u, v \rangle = \frac{1}{4}\sum_{k=0}^{3} i^k \|u + i^k v\|^2 \quad \text{(复情形)} \]

这两个公式说明：范数已经决定了内积。一旦平行四边形恒等式成立，内积可以从范数反算出来，不存在"找不到内积" 的可能。

这条定理给你一个判别工具。给一个具体范数，验证或否定平行四边形恒等式就能判它"是不是来自内积"。

用它判一下 \(\ell^p\)（\(p \neq 2\)）。取 \(u = (1, 0, 0, \ldots), v = (0, 1, 0, \ldots) \in \ell^p\)。算：

\[ \|u + v\|_p = (1^p + 1^p)^{1/p} = 2^{1/p}, \qquad \|u - v\|_p = 2^{1/p}, \]

\[ \|u\|_p = \|v\|_p = 1. \]

平行四边形恒等式要求 \(2 \cdot 2^{2/p} = 4\)，即 \(2^{2/p} = 2\)，即 \(p = 2\)。

所以只有 \(\ell^2\) 的范数来自内积。\(\ell^1, \ell^3, \ell^\infty\) 都不是。同理（用 \([0, 1]\) 上两个支集分离的指示函数检验）只有 \(L^2\) 的范数来自内积。

\[\boxed{\ \text{在 } \ell^p \text{ 与 } L^p \text{ 家族中, 只有 } p = 2 \text{ 的范数来自内积}.\ }\]

请你停下来想想这条结论的规整。"什么 \(p\) 让 \(\ell^p\) 是 Hilbert 空间" 这个问题没有"看心情" 的余地——恰好 \(p = 2\)。这种"整数 2 的特殊地位" 不是巧合，是平行四边形恒等式（一条二次关系）天然要求"二次范数"。

3.4 Hilbert 空间

把"内积" 和"完备" 拼起来——这是上一章 Banach 的剧本再上一层。

定义。一个Hilbert 空间是完备的内积空间。换句话说：内积诱导范数，那个范数让空间成为 Banach 空间。

Hilbert 空间以 David Hilbert（1862-1943）命名。他在 1900 年代研究积分方程时引入"无穷维欧氏几何" 的雏形——\(\ell^2\) 上的几何。后来 von Neumann 在 1930 年代为量子力学奠基时，把这套东西公理化为"Hilbert 空间"，并在他 1932 年的《Mathematische Grundlagen der Quantenmechanik》里把量子力学整个改写为 Hilbert 空间上的算子代数。从此 Hilbert 空间成了 20 世纪数学物理的中央对象。

四个标准例子。

例 1：\(\mathbb{R}^n, \mathbb{C}^n\)。有限维欧氏空间——内积空间且自动完备。这是"原型" Hilbert 空间。

例 2：\(\ell^2\)。无穷维 Hilbert 空间的原型。完备性证明在上一章 2.5 节类 2。它有一组明显的"标准基" \(\{e_n\}\)（每个 \(e_n\) 是"第 \(n\) 位 1 其余 0" 的序列），这组基在 Hilbert 空间意义下完备（详见第 5 章正交基理论）——任意 \(x \in \ell^2\) 可写为 \(x = \sum \langle x, e_n \rangle e_n\)。

例 3：\(L^2(\Omega, \mu)\)。"函数版的 \(\ell^2\)"。完备性是 Riesz-Fischer 定理。当 \(\Omega\) 是 \([0, 2\pi]\)、\(\mu\) 是 Lebesgue 测度，\(L^2[0, 2\pi]\) 的标准正交基是 Fourier 基 \(\{e^{i n x} / \sqrt{2\pi} : n \in \mathbb{Z}\}\)——任意 \(L^2\) 函数可写为 Fourier 级数在 \(L^2\) 范数下收敛。这个论断是经典 Fourier 分析的现代陈述（详见第 10 章）。

例 4：Sobolev 空间 \(H^k(\Omega) := W^{k, 2}(\Omega)\)。是 \(L^2\) 函数中"导数到 \(k\) 阶都还在 \(L^2\)" 的子空间，配内积

\[ \langle f, g \rangle_{H^k} = \sum_{|\alpha| \le k} \langle \partial^\alpha f, \partial^\alpha g \rangle_{L^2}. \]

诱导的 \(H^k\) 范数让它成为 Hilbert 空间（详见第 23 章）。\(H^1, H^2\) 是 PDE 弱解理论的舞台——椭圆方程、Sobolev 嵌入、迹定理都活在这个圈子里。

请你看清楚这四个例子：有限维 Hilbert（原型）、序列 Hilbert（\(\ell^2\)）、函数 Hilbert（\(L^2\)）、Sobolev Hilbert（\(H^k\)）。它们覆盖了大部分实际场景：你做信号处理用 \(L^2\)；你做 PDE 用 \(H^k\)；你做矩阵论用 \(\mathbb{R}^n\)；你做小波 / 序列分析用 \(\ell^2\)。再花式的 Hilbert 空间（再生核 Hilbert 空间 RKHS、Bergman 空间等）也都基于这些。

3.5 正交：定义、Pythagoras、正交补

到这里我们终于可以兑现章首的承诺——把"角度" 装回去。

定义。设 \(H\) 是 Hilbert 空间，\(x, y \in H\)。称 \(x \perp y\)（"\(x\) 与 \(y\) 正交") 当且仅当 \(\langle x, y \rangle = 0\)。

同样地，对集合 \(S \subseteq H\)，\(x \perp S\) 意味着 \(\langle x, s \rangle = 0\) 对所有 \(s \in S\)。\(S \perp T\)（两个集合正交）意味着 \(s \perp t\) 对所有 \(s \in S, t \in T\)。

Pythagoras 定理。若 \(x \perp y\)，则

\[ \|x + y\|^2 = \|x\|^2 + \|y\|^2. \]

证明一行：\(\|x + y\|^2 = \|x\|^2 + \langle x, y \rangle + \langle y, x \rangle + \|y\|^2 = \|x\|^2 + \|y\|^2\)。两个交叉项 0。

更一般：若 \(\{x_1, \ldots, x_n\}\) 两两正交，则 \(\|\sum x_i\|^2 = \sum \|x_i\|^2\)。这是无穷级数版的 Bessel 不等式 / Parseval 等式（第 5 章）的直接前身。

正交集与正交单位集。集合 \(\{e_\alpha\}_{\alpha \in I} \subseteq H\) 叫正交集，若 \(\langle e_\alpha, e_\beta \rangle = 0\) 当 \(\alpha \neq \beta\)。再加 \(\|e_\alpha\| = 1\)，叫正交单位集（orthonormal set）。

有限维线性代数里你已经熟悉这些——\(\mathbb{R}^n\) 的标准基 \(\{e_1, \ldots, e_n\}\) 就是正交单位集。Hilbert 空间里我们要把这套搬到无穷维。"无穷" 这一步带来微妙之处（一个正交单位集"张成" 空间是什么意思？要用闭包），全部留给第 5 章。

正交补 \(M^\perp\)。设 \(M \subseteq H\) 任意子集。定义

\[ M^\perp := \{x \in H : x \perp m \text{ 对所有 } m \in M\}. \]

\(M^\perp\) 叫 \(M\) 的正交补。

下面这个性质看似平凡，但它是 Hilbert 空间整本书的钩子：

\[\boxed{\ \text{对任意子集 } M, \ M^\perp \text{ 是 } H \text{ 的闭线性子空间}.\ }\]

证明三步。

第一，线性。若 \(x, y \in M^\perp\)，对每个 \(m \in M\)，\(\langle \alpha x + \beta y, m \rangle = \alpha \langle x, m \rangle + \beta \langle y, m \rangle = 0\)，所以 \(\alpha x + \beta y \in M^\perp\)。

第二，闭。若 \(x_n \in M^\perp\) 且 \(x_n \to x\)，对每个 \(m \in M\)，由 3.2 节末尾的"内积连续"，\(\langle x, m \rangle = \lim \langle x_n, m \rangle = 0\)。所以 \(x \in M^\perp\)。

第三，所以 \(M^\perp\) 是闭子空间。

注意一件特别的事：这条结论不要求 \(M\) 自己是子空间。\(M\) 可以是任意子集——一个点、一条曲线、一个不闭子空间——而 \(M^\perp\) 永远是闭子空间。这是为什么"正交补" 在 Hilbert 空间几何里这么有威力——它把任何东西"升级" 成闭子空间。

这一点和上一章 1.3 节的崩塌（"子空间不自动闭"）形成对照——在 Banach 空间里，闭子空间是稀有资源，要专门构造；在 Hilbert 空间里，"取正交补" 这个操作免费 给你闭子空间。这是 Hilbert 比 Banach 几何上"富" 得多的具体体现。

正交补还有一条让人停一下的代数性质：\(M \subseteq (M^\perp)^\perp\)（即任何 \(x \in M\) 都和"所有和 \(M\) 正交的东西" 正交）。这条显然。但反方向：什么时候 \((M^\perp)^\perp = \overline{\mathrm{span}\, M}\)？答：总是——这就是第 6 章正交分解定理的核心结论之一。这条结论是"线性代数中 \((W^\perp)^\perp = W\)" 在 Hilbert 空间里的版本——但需要 \(W\) 闭。

3.6 为什么 Hilbert 比 Banach"贵"

这一章我们装好了内积、Cauchy-Schwarz、Pythagoras、正交补。这些东西在 Banach 空间里全部缺席——\(\ell^p\)（\(p \neq 2\)）和 \(C[0, 1]\) 这种 Banach 空间没有内积，自然没有"正交"，没有"正交补总是闭子空间"，没有"勾股定理"。

所以"是 Hilbert" 这个性质比"是 Banach" 严格地更强。换句话说，"Hilbert 假设" 是个比"Banach 假设" 更苛刻 的前提。每个定理多一个 Hilbert 前提，就少一些适用空间——但作为补偿，能证更多结论。这是泛函分析里的典型权衡：结构越多, 适用空间越窄, 但能下的结论越深。

下面这张预告表列出后面 22 章里"专属 Hilbert" 的几个王牌定理，让你提前知道这一章装的语言会去哪里。

  • 第 6 章 正交投影定理。在 Hilbert 空间，对每个闭 子空间 \(M\)，\(H = M \oplus M^\perp\)——正交分解。每个 \(x \in H\) 唯一写成 \(x = x_M + x_{M^\perp}\)。在 Banach 空间这条整体不成立（一般 Banach 空间不一定有补空间，更别说正交 补）。这条定理是"最佳逼近" 的一切根据——计算 \(x\) 到子空间的最近点 = 投影到 \(M\) 上。
  • 第 16 章 Riesz 表示定理。Hilbert 空间 \(H\) 上每个连续线性泛函 \(\varphi : H \to \mathbb{F}\) 都能写为 \(\varphi(x) = \langle x, y_\varphi \rangle\)，其中 \(y_\varphi \in H\) 唯一。这条意味着\(H^* \cong H\)——Hilbert 空间和它的对偶天然同构。在 Banach 空间这条不真——\(L^p\) 和 \((L^p)^*\) 不同构（除非 \(p = 2\)），\(\ell^\infty\) 的对偶比它自己还大。Hilbert 空间这种"自对偶" 让它在变分问题里特别好用。
  • 第 21 章 紧自伴算子的谱定理。Hilbert 空间上紧自伴算子有"正交单位特征向量基"——和有限维"对称矩阵正交对角化" 一模一样的画面。这条定理无 Hilbert 不行——离开内积就没"正交单位"，没"对角化" 的标准。它是积分方程、量子力学、PDE 谱方法的核心工具。

这三条加上 Hilbert 几何（正交、Pythagoras、Bessel、Parseval、正交基）构成 Hilbert 空间论的"豪华套餐"。要在 Banach 空间也享受这些，得用非常多 的额外工具——Hahn-Banach、对偶配对、共轭算子、自反性——而且很多结论根本拿不回来（比如紧 Banach 算子没有对应的"正交对角化"）。

最后请你停下来想一件事：上面这些专属 Hilbert 的好处，都依赖正交补的两条关键性质——"\(M^\perp\) 永远是闭子空间"（3.5 节）+ "\(H = M \oplus M^\perp\) 当 \(M\) 闭"（第 6 章）。这两条是其他 一切结构的根。一旦你接受"有内积 = 有正交"，剩下整个 Hilbert 理论就是"把这把钥匙开过的门列举一遍"。

下一章我们暂时离开抽象——把后续 22 章频繁出现的具体空间（\(\ell^p, L^p, C(K), C^k, c_0\)）讲到"看名字立刻有图像" 的程度。这一章的内积语言、上一章的范数语言会在那里和具体例子真正对上。
