第 4 章 经典空间例子库

前两章我们建了两层抽象：Banach 空间（第 2 章）、Hilbert 空间（第 3 章）。读完那两章，你已经能讲 整个泛函分析的语言了——但还没 见到很多具体例子。

泛函分析不是抽象自娱——它所有的定理最后都要落到具体空间上跑。一个定理"对所有 Banach 空间成立" 是好听的话；可你真用它的时候，问的总是"对 \(L^3\) 成立吗"、"\(C[0, 1]\) 算 Banach 吗"、"\(\ell^\infty\) 是 Hilbert 还是 Banach"。这些问题不能现场猜。

这一章我把后续 20 章里出现频率最高 的几个空间——\(\ell^p\)、\(L^p\)、\(C(K)\)、\(C^k\)、\(c_0\)——讲到"看见名字立刻有图像" 的程度。这是个例子库章，不冲新理论；你以后写论文 / 听报告碰到这些名字，要能立刻答出"它是 Banach 还是 Hilbert"、"它对偶是谁"、"它和谁有嵌入"、"它可分吗自反吗"。

这一章的最终产物在 4.5、4.6 两节——一张判别表 和一张嵌入 / 对偶速查表。看完这一章，把那两张表打印贴在书桌前——以后查空间的"身份证" 就用它们。

4.1 \(\ell^p\) 序列空间

第 2 章 2.2 节例 2 我们已经写过 \(\ell^p\) 的范数。这一节给完整的身份证。

定义。对 \(1 \le p < \infty\)，

\[ \ell^p = \left\{ x = (x_n)_{n=1}^\infty \in \mathbb{F}^\mathbb{N} : \|x\|_p := \left(\sum_{n=1}^\infty |x_n|^p\right)^{1/p} < \infty \right\}. \]

对 \(p = \infty\)，

\[ \ell^\infty = \{x : \|x\|_\infty := \sup_n |x_n| < \infty\}. \]

"\(\|\cdot\|_p\) 是范数" 不是显然的——核心是三角不等式，即 Minkowski 不等式：

\[ \left(\sum |x_n + y_n|^p\right)^{1/p} \le \left(\sum |x_n|^p\right)^{1/p} + \left(\sum |y_n|^p\right)^{1/p}. \]

证明要 Hölder 不等式：对 \(1/p + 1/q = 1\)（共轭指标），

\[ \sum |x_n y_n| \le \left(\sum |x_n|^p\right)^{1/p} \left(\sum |y_n|^q\right)^{1/q}. \]

Hölder 是 Cauchy-Schwarz 的推广：\(p = q = 2\) 就回到 Cauchy-Schwarz。

这两条不等式（Hölder + Minkowski）是整个 \(L^p, \ell^p\) 理论的基础。证明在实分析读本第 12 章详述，这里我们直接用。

完备性。\(\ell^p\) 完备，标准模板见上一章 2.5 节类 2。结论：所有 \(\ell^p\)（\(1 \le p \le \infty\)）都是 Banach 空间。

Hilbert？由第 3 章3.3 节，只有 \(\ell^2\) 是 Hilbert。其他 \(\ell^p\) 是 Banach 但不是 Hilbert——它们没有内积。

嵌入：注意方向！对 \(1 \le p < q \le \infty\)，

\[ \boxed{\ \ell^p \subseteq \ell^q, \qquad \|x\|_q \le \|x\|_p.\ } \]

朴素证明：若 \(x \in \ell^p\)，则 \(|x_n| \to 0\)（级数 \(\sum|x_n|^p\) 收敛蕴含项趋零）。所以 \(|x_n| \le 1\) 终究成立。对终究成立的部分，\(|x_n|^q \le |x_n|^p\)（因 \(|x_n| \le 1, q \ge p\)），所以级数 \(\sum |x_n|^q\) 收敛。

请你盯着这个方向看一会：\(p\) 越小, \(\ell^p\) 越小——\(\ell^1 \subsetneq \ell^2 \subsetneq \ell^3 \subsetneq \cdots \subsetneq \ell^\infty\)。直观：\(\ell^1\) 要"绝对可和"——条件最苛刻，所以最少元素；\(\ell^\infty\) 只要"有界"——条件最松，所以最多元素。

下面 4.2 节会说，\(L^p\) 的嵌入方向反过来（在有限测度空间上）——这是个你必须记住的"对照陷阱"。

对偶。对 \(1 \le p < \infty\) 和共轭指标 \(1/p + 1/q = 1\)，

\[ \boxed{\ (\ell^p)^* \cong \ell^q.\ } \]

具体地：每个连续线性泛函 \(\varphi : \ell^p \to \mathbb{F}\) 唯一表示为 \(\varphi(x) = \sum x_n \overline{y_n}\)，其中 \(y \in \ell^q\)。\(p = 1\) 时 \(q = \infty\)：\((\ell^1)^* = \ell^\infty\)。\(p = 2\) 时 \(q = 2\)：\((\ell^2)^* = \ell^2\)（自对偶——Hilbert 空间的特征）。\(p = \infty\) 时例外——\((\ell^\infty)^* \neq \ell^1\)，\(\ell^\infty\) 的对偶比 \(\ell^1\) 大得多，包含像 Banach 极限这种非平凡对象。

对偶定理的完整陈述与证明在第 15 章。这里你先记结论。

子空间 \(c_0\)。定义

\[ c_0 = \{x \in \ell^\infty : x_n \to 0\}. \]

"趋零数列" 的全体。它配 \(\|\cdot\|_\infty\) 是 \(\ell^\infty\) 的闭 子空间（极限的极限还是 0），所以是 Banach 空间。但不是 Hilbert（同样用 \(p \neq 2\) 的论证）。

\(c_0\) 在对偶理论里有特殊地位：\((c_0)^* \cong \ell^1\)，而 \((\ell^1)^* \cong \ell^\infty\)。所以 \((c_0)^{**} \cong \ell^\infty \neq c_0\)——\(c_0\) 不是自反 的（详见 4.5 节）。这条让 \(c_0\) 成为非自反 Banach 空间 的原型，反例库里反复出现。

可分性。\(\ell^p\)（\(1 \le p < \infty\)）和 \(c_0\) 可分——"有理坐标 + 有限非零" 的元素稠密。\(\ell^\infty\) 不可分——它包含一个不可数离散子集（取 \(\{0, 1\}^\mathbb{N}\) 中"不同的两个序列\(\|\cdot\|_\infty\) 距离 \(\ge 1\)"）。

4.2 \(L^p(\Omega, \mu)\) 函数空间

序列空间换成函数空间，故事几乎平移过来。

定义。设 \((\Omega, \Sigma, \mu)\) 是测度空间。对 \(1 \le p < \infty\)，

\[ L^p(\Omega) = \{f : \Omega \to \mathbb{F} \text{ 可测, } \int_\Omega |f|^p \, d\mu < \infty\} \big/ \sim, \]

配 \(\|f\|_p = (\int |f|^p)^{1/p}\)。\(\sim\) 是"几乎处处相等" 的等价。

\(p = \infty\) 时

\[ L^\infty(\Omega) = \{f : \mathrm{ess\,sup}_\Omega |f| < \infty\} \big/ \sim, \quad \|f\|_\infty = \mathrm{ess\,sup} |f|. \]

这里 ess sup（本性上确界）是去掉零测集后的 sup——这是 Lebesgue 框架特有的细节，详见实分析读本第 11 章。

完备性。\(L^p\) 完备（Riesz-Fischer 1907），见第 2 章 2.5 节类 4。结论：所有 \(L^p\)（\(1 \le p \le \infty\)）都是 Banach 空间。

Hilbert？同 \(\ell^p\)：只有 \(L^2\) 是 Hilbert。

嵌入：方向反！这是要划重点的地方。设 \(\mu(\Omega) < \infty\)（有限测度）。则对 \(1 \le p < q \le \infty\)，

\[ \boxed{\ L^q(\Omega) \subseteq L^p(\Omega), \qquad \|f\|_p \le \mu(\Omega)^{1/p - 1/q} \|f\|_q.\ } \]

证明用 Hölder 不等式（取常数函数 1 与 \(|f|^p\) 配对）。

请你停下来对照：\(\ell^p\) 是 \(\ell^p \subseteq \ell^q\)（\(p < q\)）；\(L^p\) 是 \(L^q \subseteq L^p\)（\(p < q\)，有限测度）。方向相反。

为什么？直觉：在无穷支集（序列空间 \(\mathbb{N}\) 测度无穷）上，"大 \(p\)" 容忍尾部"慢衰减"——所以小 \(p\) 空间 \(\subset\) 大 \(p\) 空间。在有限测度空间上，"大 \(p\)" 才约束你"不能在小集合上爆掉"——所以大 \(p\) 空间 \(\subset\) 小 \(p\) 空间。两种"嵌入方向" 由测度的总质量决定。

当测度是无穷（比如 \(L^p(\mathbb{R})\) 配 Lebesgue 测度），两个方向都不成立——不存在 \(L^p \subseteq L^q\) 或 \(L^q \subseteq L^p\) 的简单包含。\(L^1(\mathbb{R})\) 包含可积但不平方可积的函数（如 \(1/\sqrt x\) 在 \([0, 1]\) 外的延拓），\(L^2(\mathbb{R})\) 包含平方可积但不可积的函数（如 \(1/(1+|x|)\)），两边互不含。

这个"嵌入方向取决于测度大小" 的现象是数值人 / 物理人最容易踩坑的地方——你在 \(\mathbb{R}\) 上写"取一个 \(L^2 \cap L^1\) 函数" 不算多余, 因为 \(L^2\) 和 \(L^1\) 真不互含。

对偶。对 \(1 \le p < \infty\)，

\[ \boxed{\ (L^p)^* \cong L^q, \quad \frac{1}{p} + \frac{1}{q} = 1.\ } \]

每个 \(\varphi \in (L^p)^*\) 唯一表示为 \(\varphi(f) = \int f \overline{g} \, d\mu\)，其中 \(g \in L^q\)。

同 \(\ell^\infty\)：\((L^\infty)^* \neq L^1\)——\(L^\infty\) 的对偶包含像"有限可加测度" 这类非 \(L^1\) 的东西。

这条对偶定理（\(L^p\)-\(L^q\) 对偶）是第 15 章的中心。它和 Riesz 表示（第 16 章）一起构成"具体空间的对偶字典"——是日常工作里你要查最频繁的内容之一。

可分性。\(L^p\)（\(1 \le p < \infty\)，\(\sigma\) 有限测度）可分。\(L^\infty\) 一般不可分。

4.3 \(C[a, b]\) 与 \(C(K)\)

"连续函数空间" 这个家族同样要熟。

定义。设 \(K\) 是紧 Hausdorff 空间。\(C(K)\) 是 \(K\) 上连续 \(\mathbb{F}\)-值函数的全体，配

\[ \|f\|_\infty = \sup_{x \in K} |f(x)| = \max_{x \in K} |f(x)|. \]

（紧 Hausdorff + 连续保证 sup 取得到。）

最常用的特例是 \(K = [a, b]\)（实区间），得 \(C[a, b]\)。也常用 \(K = \mathbb{T}\)（圆周——周期函数空间）、\(K = S^n\)（球面）、\(K = \overline\Omega\)（紧化的开域）。

完备性。\(C(K)\) 配 sup 范数完备——见第 2 章 2.5 节类 3。结论：所有 \(C(K)\) 都是 Banach 空间。

Hilbert？不是——sup 范数不来自内积。验证：取 \(K = [0, 1]\)，\(u(x) = x\)，\(v(x) = 1 - x\)。算 \(\|u\|_\infty = \|v\|_\infty = 1\)、\(\|u + v\|_\infty = 1\)、\(\|u - v\|_\infty = 1\)。平行四边形恒等式要 \(2 + 2 = 1 + 1\)，即 \(4 = 2\)。塌。

对偶 = 测度。这是 \(C(K)\) 故事最深的部分。

定理（Riesz-Markov-Kakutani）。设 \(K\) 紧 Hausdorff。则

\[ C(K)^* \cong M(K) := \{\text{有限正则 Borel 复测度}\}, \]

同构由"积分配对" 给出：\(\mu \mapsto \varphi_\mu\)，\(\varphi_\mu(f) = \int_K f \, d\mu\)。范数 \(\|\varphi_\mu\| = |\mu|(K)\)（总变差）。

这条定理把"测度" 和"连续函数对偶" 等同起来——你想知道 \(C(K)\) 上有什么连续线性泛函，等价地问"\(K\) 上有什么测度"。这是测度论和泛函分析之间的桥，详细在第 16 章。

历史上 Riesz 1909 在 \(K = [0, 1]\) 时证了这条；Markov 和 Kakutani 1940 年代推广到一般紧 Hausdorff 空间。这条定理直接催生了"测度的弱收敛"（即概率论中常说的弱收敛），是概率论与泛函分析之间的根本桥梁。

稠密性：Stone-Weierstrass。这条经典定理说多项式（更一般地，任何包含常数、分离点、对乘法封闭的子代数）在 \(C(K)\) 中稠密。具体在 \(K = [a, b]\)：

\[ \overline{\mathcal{P}[a, b]} = C[a, b]. \]

多项式空间的闭包 就是整个 \(C[a, b]\)——这正是上一章 1.3 节"多项式空间不闭" 的反面陈述。

Stone-Weierstrass 还有三角多项式版：圆周 \(\mathbb{T}\) 上三角多项式 \(\sum_{|n| \le N} c_n e^{i n \theta}\) 在 \(C(\mathbb{T})\) 中稠密——这条是经典 Fourier 分析的稠密性根据，详见第 10 章。

可分性。当 \(K\) 紧度量空间，\(C(K)\) 可分——多项式（或类似稠密集）的有理系数版本可数稠密。一般紧 Hausdorff（不可度量化）的 \(C(K)\) 可能不可分。

4.4 \(C^k\) 空间

把"连续" 换成"\(k\) 次连续可微"。

定义。对 \(k \in \mathbb{N}_0\)，

\[ C^k[a, b] = \{f : [a, b] \to \mathbb{F}, \, f \text{ 在 } [a, b] \text{ 上 } k \text{ 次连续可微}\}, \]

配范数

\[ \|f\|_{C^k} = \sum_{j=0}^k \|f^{(j)}\|_\infty = \sum_{j=0}^k \sup_{x \in [a, b]} |f^{(j)}(x)|. \]

"\(k\) 次连续可微" 意思是 \(f, f', f'', \ldots, f^{(k)}\) 都存在并连续。\(C^0 = C\)（无导数要求，最弱）。\(C^k\) 越大 \(k\) 越严格——\(C^0 \supsetneq C^1 \supsetneq C^2 \supsetneq \cdots\)。

\(k = \infty\) 时 \(C^\infty\) 表示"任意阶可微" 函数。\(C^\infty\) 不能配上面这种有限 范数——\(\sum_{j=0}^\infty\) 不收敛。\(C^\infty\) 上自然的拓扑是Fréchet 拓扑（用一族半范数 \(\|\cdot\|_{C^k}\) 而不是一个范数），它是局部凸空间，不 是 Banach。这是第 24 章稍后讨论的对象。

完备性。有限 \(k\) 时 \(C^k[a, b]\) 完备——见第 2 章 2.5 节类 5。结论：所有 \(C^k\)（\(k \in \mathbb{N}_0\)）都是 Banach 空间。

Hilbert？不是——\(C^k\) 范数是 sup 型，不来自内积。但是注意：把 \(C^k\) 换成 \(H^k = W^{k, 2}\)（Sobolev 空间，"\(L^2\) 意义下 \(k\) 次可导"），就是 Hilbert（见第 3 章 3.4 节例 4）。

这两个家族（\(C^k\) 与 \(H^k\)）是 PDE 理论里两套并行的语言——古典解的世界用 \(C^k\)，弱解的世界用 \(H^k\)。它们都"带导数"，但前者要求处处可微（强）、后者要求积分意义可微（弱）。

嵌入。\(C^k[a, b] \subsetneq C^{k-1}[a, b] \subsetneq \cdots \subsetneq C[a, b]\) 严格嵌入，且嵌入连续——\(\|f\|_{C^{k-1}} \le \|f\|_{C^k}\) 显然（少加一项）。

多变量版本。对开域 \(\Omega \subseteq \mathbb{R}^n\)，定义 \(C^k(\overline\Omega)\) 是"\(\Omega\) 上 \(k\) 次可微且所有偏导可延拓到 \(\overline\Omega\) 上连续" 的函数全体，配范数

\[ \|f\|_{C^k} = \sum_{|\alpha| \le k} \|\partial^\alpha f\|_\infty. \]

这是 PDE 教科书里"古典 \(C^k\) 解" 的栖息地。完备性同上。

4.5 谁是 Hilbert 谁是 Banach 谁不自反

把上面四节的结论归一张表。"自反" 这个词我们还没正式定义——简短说：Banach 空间 \(X\) 自反，当且仅当自然嵌入 \(X \hookrightarrow X^{**}\) 是同构（即 \(X^{**} \cong X\)）。Hilbert 空间永远自反；\(\ell^p, L^p\)（\(1 < p < \infty\)）自反；\(\ell^1, L^1, \ell^\infty, L^\infty, c_0, C(K)\) 不自反。详细在第 15 章 15.5 节。

\(\ell^p\)，\(1 < p < \infty\)是仅 \(p = 2\)是是
\(\ell^1\)是否否是
\(\ell^\infty\)是否否否
\(c_0\)是否否是
\(L^p(\Omega, \mu)\), \(1 < p < \infty\)是仅 \(p = 2\)是是 (\(\sigma\) 有限)
\(L^1(\Omega, \mu)\)是否否是 (\(\sigma\) 有限)
\(L^\infty(\Omega, \mu)\)是否否否(一般)

读这张表时请注意几条规律：

  • Hilbert ⟹ 自反。所有 Hilbert 行都是自反。
  • "端点" \(p = 1, \infty\) 是病态点。\(\ell^1, \ell^\infty, L^1, L^\infty\) 全部不自反——它们的对偶不和它们自己同构（\(\ell^1\) 的对偶是 \(\ell^\infty\)，但 \(\ell^\infty\) 的对偶比 \(\ell^1\) 大；类似 \(L^1\)）。"非自反" 在弱收敛理论（第 17 章）里意味着失去弱紧性——所以"极小化序列" 这种标准 PDE 工具在 \(L^1\) 上要小心。
  • "\(\infty\)-范数" 通常不可分。\(\ell^\infty, L^\infty\)（一般 \(\Omega\)）、\(C(K)\)（\(K\) 不可度量化时）都不可分。可分性影响是否有可数 Schauder 基（详见第 5 章）和很多近似论证。
  • \(C^k\) 是 Banach 但不是 Hilbert。要"带导数 + Hilbert" 必须用 Sobolev \(H^k\)。

背这张表的最快方法：记住"三类危险空间"——\(L^1 / \ell^1\)（不自反）、\(L^\infty / \ell^\infty\)（不自反 + 不可分 + 对偶有怪东西）、\(C(K)\)（对偶 = 测度）。其他都正常。

读这张表的关键观察：

  • 共轭指标 \(1/p + 1/q = 1\) 是 \(L^p, \ell^p\) 对偶理论的"语法"。\(p = 2 \Rightarrow q = 2\)（自对偶）。\(p \to 1 \Rightarrow q \to \infty\)（端点）。\(p \to \infty\) 没有有限的 \(q\)——这就是为什么 \((L^\infty)^* \neq L^1\) 的部分原因。
  • "\(c_0 \to \ell^1 \to \ell^\infty\)" 三连：\((c_0)^* = \ell^1\)，\((\ell^1)^* = \ell^\infty\)，\((\ell^\infty)^* \supsetneq \ell^1\)。所以 \(c_0\) 二次对偶 \((c_0)^{**} = \ell^\infty\)，比 \(c_0\) 大——\(c_0\) 不自反。
  • \(C(K)\) 对偶是测度空间 \(M(K)\)——这是测度论与连续函数论之间的深刻对应。在概率论里"测度的弱*收敛" 等价于"对所有连续函数积分收敛"——这正是这个对偶的直接应用。

这两张表你以后查得最频繁。建议抄一遍贴书桌前。
