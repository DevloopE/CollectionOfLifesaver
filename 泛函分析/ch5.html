第 5 章 经典不等式

请你先做一道你高中就会的事。给两个非负实数 \(a, b\)。比较

\[ ab \quad \text{和} \quad \frac{a^2 + b^2}{2}. \]

左边是几何均值平方那一类，右边是算术均值平方那一类。两边相减

\[ \frac{a^2 + b^2}{2} - ab = \frac{(a - b)^2}{2} \ge 0. \]

所以 \(ab \le \frac{a^2 + b^2}{2}\)。等号在 \(a = b\) 时成立。这是最朴素 的"两个数的乘积被两个数的平方平均控制" 的不等式。

请你停下来想：这条小结果跟分析有什么关系？答：跟 Cauchy–Schwarz 不等式有关系。把它写在向量上：对 \(\mathbb{R}^n\) 里两个向量 \(x = (x_1, \ldots, x_n)\), \(y = (y_1, \ldots, y_n)\)，把上面对每个分量做一遍：

\[ |x_i y_i| \le \frac{x_i^2 + y_i^2}{2}. \]

求和：\(\sum |x_i y_i| \le \frac{\|x\|_2^2 + \|y\|_2^2}{2}\)。这条本身已经是个有用的估计。要把它"打磨" 成 Cauchy–Schwarz \(|\sum x_i y_i| \le \|x\|_2 \cdot \|y\|_2\)，只差一步——把 \(x, y\) 各自换成"归一化" 版本（除以自己的范数），让 \(\|x\|_2 = \|y\|_2 = 1\)，再倒推。

整本泛函分析里几乎所有"用范数估计内积 / 用一类积分估计另一类积分" 的不等式，背后都是同一种思路：先证一个两个数 的不等式 \(ab \le \cdots\)（分项不等式），再"归一化 + 求和" 把它升级到向量 / 函数。这一章我把这条思路串成一条链：

\[ \boxed{\ \text{Young} \xrightarrow{\text{归一化求和}} \text{Hölder} \xrightarrow{\text{重新分组}} \text{Minkowski}.\ } \]

三步走完，\(L^p\) 空间是 Banach 空间这件事就被钉牢了——这是第 4 章承诺要还的债。链子之外再加两件常用工具：Bessel（投影几何里出来的）+ Jensen（凸函数版本的"分项不等式")。

5.1 Young 不等式：链子的种子

从前面那条 \(ab \le (a^2 + b^2)/2\) 出发，你能把指数非对称地 推广。Young 不等式说：

定理（Young 不等式）。设 \(a, b \ge 0\)，\(p, q \in (1, \infty)\) 满足共轭关系

\[ \frac{1}{p} + \frac{1}{q} = 1. \]

则

\[ \boxed{\ ab \le \frac{a^p}{p} + \frac{b^q}{q}.\ } \]

等号在 \(a^p = b^q\) 时成立。

证明只用一行——\(\log\) 的凹性。设 \(a, b > 0\)（\(a = 0\) 或 \(b = 0\) 时不等式平凡）。\(\log\) 在 \((0, \infty)\) 上凹，所以对凸组合系数 \(1/p, 1/q\)（这两个数加起来是 1）有

\[ \log\left(\frac{1}{p} \cdot a^p + \frac{1}{q} \cdot b^q\right) \ge \frac{1}{p} \log(a^p) + \frac{1}{q} \log(b^q) = \log a + \log b = \log(ab). \]

对两边取指数（指数函数单增）：\(\frac{a^p}{p} + \frac{b^q}{q} \ge ab\)。证毕。

这个证明的精神："两个数的乘积 \(\le\) 它们的某种'幂平均'"。\(p = q = 2\) 时退化到本章开头那条 \(ab \le (a^2 + b^2)/2\)（共轭关系 \(1/2 + 1/2 = 1\) 成立）。

请你停下来感受 \(p\) 与 \(q\) 的共轭 关系。\(1/p + 1/q = 1\) 意味着 \(q = p/(p-1)\)。当 \(p = 1.1\) 时 \(q = 11\)；\(p = 2\) 时 \(q = 2\)；\(p = 3\) 时 \(q = 3/2\)；\(p \to \infty\) 时 \(q \to 1\)。这条关系不是装饰——它就是 \(L^p\) 与 \(L^q\) 配对的几何根，下一节立刻看到。

5.2 Hölder 不等式：把 Young 升级到向量

现在把 5.1 的"分项不等式" 套到向量上去。设 \(x = (x_1, \ldots, x_n) \in \mathbb{R}^n\), \(y = (y_1, \ldots, y_n) \in \mathbb{R}^n\)，\(p, q \in (1, \infty)\) 共轭。

朴素试法：把 Young 直接对每个 \((|x_i|, |y_i|)\) 用一遍：

\[ |x_i y_i| \le \frac{|x_i|^p}{p} + \frac{|y_i|^q}{q}. \]

对 \(i\) 求和：

\[ \sum_i |x_i y_i| \le \frac{1}{p} \sum_i |x_i|^p + \frac{1}{q} \sum_i |y_i|^q = \frac{\|x\|_p^p}{p} + \frac{\|y\|_q^q}{q}. \]

这条已经 是个估计，但形式不漂亮——左边一个内积式量，右边是范数的幂 之和。理想的形状应该是"内积 \(\le\) 范数 × 范数"。

撞墙了。怎么办？重新归一化。

关键技巧：把 \(x, y\) 各自除以自己的范数。设 \(\|x\|_p > 0, \|y\|_q > 0\)（否则平凡），令

\[ \tilde x_i = \frac{x_i}{\|x\|_p}, \quad \tilde y_i = \frac{y_i}{\|y\|_q}. \]

则 \(\|\tilde x\|_p = 1, \|\tilde y\|_q = 1\)。把上面那条不等式套到归一化版本：

\[ \sum_i |\tilde x_i \tilde y_i| \le \frac{\|\tilde x\|_p^p}{p} + \frac{\|\tilde y\|_q^q}{q} = \frac{1}{p} + \frac{1}{q} = 1. \]

左边代回原变量：\(\sum |x_i y_i| / (\|x\|_p \|y\|_q) \le 1\)。两边乘 \(\|x\|_p \|y\|_q\)：

\[ \boxed{\ \sum_{i=1}^n |x_i y_i| \le \|x\|_p \cdot \|y\|_q. \ } \]

这就是Hölder 不等式（离散版本）。漂亮形状达成。

请你看清这条证明的力学：朴素套 Young 给的是"幂相加" 形状；这个形状不是要的；解决办法不是改算法，是改输入——把 \(x, y\) 归一化让 \(\|x\|_p = \|y\|_q = 1\)，于是右边的 \(\frac{\|x\|_p^p}{p} + \frac{\|y\|_q^q}{q}\) 自动塌缩成 \(\frac{1}{p} + \frac{1}{q} = 1\)。共轭关系就是 让这步塌缩发生的条件。

积分版本完全平行：对可测函数 \(f, g\)，

\[ \int |f g| \, d\mu \le \left(\int |f|^p \, d\mu\right)^{1/p} \cdot \left(\int |g|^q \, d\mu\right)^{1/q} = \|f\|_p \cdot \|g\|_q. \]

证明一字不差——把求和换成积分。这就是第 4 章定义的 \(L^p\) 范数与 \(L^q\) 范数的对偶配对。

\[\boxed{\ \text{Hölder：} L^p \text{ 与 } L^q \text{ 是天生的'对偶配对'，配对形状是积分。}\ }\]

这件事的分量。\(L^p\) 与 \(L^q\)（\(1/p + 1/q = 1\)）不是两个互不相干的空间——它们通过 Hölder 不等式互为对方的对偶。这是第 15 章"\(L^p\) 的对偶就是 \(L^q\)" 的几何根。Hölder 是这个对偶关系能存在 的不等式条件——没有 Hölder，"\(L^q\) 函数定义 \(L^p\) 上的连续线性泛函" 这件事根本说不出口。

三个特例你必须记住：

  • \(p = q = 2\)：Hölder 退化为Cauchy–Schwarz（5.4 节）。
  • \(p = 1, q = \infty\)：\(\int |fg| \le \|f\|_1 \|g\|_\infty\)（积分被一个一致界 × 总质量控制）。
  • \(p = \infty, q = 1\)：对称的另一版。

5.3 Minkowski 不等式：\(L^p\) 三角不等式

现在到第 4 章遗留的那笔账：\(L^p\) 范数是 范数吗？范数三条要求是：(i) 正定；(ii) 齐次；(iii) 三角不等式。前两条 \(L^p\) 显然满足。第三条——三角不等式 \(\|f + g\|_p \le \|f\|_p + \|g\|_p\)——不显然，因为求和发生在幂 里面。

这就是 Minkowski 不等式的内容。

定理（Minkowski 不等式）。设 \(p \in [1, \infty]\)，\(f, g \in L^p(\mu)\)。则

\[ \boxed{\ \|f + g\|_p \le \|f\|_p + \|g\|_p.\ } \]

证明思路：把 \(\|f + g\|_p^p = \int |f+g|^p\) 拆成两块，每块用 Hölder。

\(p = 1\) 与 \(p = \infty\) 平凡（\(p = 1\) 是积分 + 三角不等式；\(p = \infty\) 是 sup 加性）。看 \(1 < p < \infty\)，设 \(q\) 是 \(p\) 的共轭。计算：

\[ |f + g|^p = |f + g| \cdot |f + g|^{p-1} \le |f| \cdot |f + g|^{p-1} + |g| \cdot |f + g|^{p-1}. \]

积分两边：

\[ \int |f + g|^p \le \int |f| \cdot |f + g|^{p-1} + \int |g| \cdot |f + g|^{p-1}. \]

对每一项用 Hölder（\(p\) 与 \(q\) 共轭）：\(\int |f| \cdot |f + g|^{p-1} \le \|f\|_p \cdot \||f+g|^{p-1}\|_q\)。算后一个范数：

\[ \||f + g|^{p-1}\|_q = \left(\int |f+g|^{(p-1) q}\right)^{1/q} = \left(\int |f+g|^p\right)^{1/q} = \|f+g\|_p^{p/q}. \]

（用了共轭关系 \((p-1)q = p\)：从 \(1/p + 1/q = 1\) 推 \(q(p-1) = p\)。）

合起来：

\[ \|f+g\|_p^p \le (\|f\|_p + \|g\|_p) \cdot \|f+g\|_p^{p/q}. \]

两边除以 \(\|f+g\|_p^{p/q}\)（设 \(\|f+g\|_p > 0\)，否则平凡）。剩下的指数 \(p - p/q = p(1 - 1/q) = p \cdot 1/p = 1\)。所以

\[ \|f+g\|_p^1 \le \|f\|_p + \|g\|_p. \]

证毕。

这条证明的味道：朴素的 \(|f + g|^p \le 2^{p-1}(|f|^p + |g|^p)\)（这条由凸性给）给 不出三角不等式——它的常数 \(2^{p-1}\) 太大。Minkowski 的关键技巧：把 \(|f + g|^p\) 拆成 \(|f + g| \cdot |f+g|^{p-1}\)，把那个 \(|f+g|^{p-1}\) 当成 \(L^q\) 函数，用 Hölder 把它 "消化" 到右边——共轭关系让指数算账正好平衡。

请你回头看链子：Young → Hölder → Minkowski。

  • Young：两个数 的不等式，1 行 \(\log\) 凹性。
  • Hölder：归一化求和，把 Young 升级成向量 / 函数 的不等式。
  • Minkowski：拆 \(|f+g|^p\)，对两块各用一次 Hölder——升级成三角 不等式。

三步全都不靠运气，每步靠的是上一步的成果。这种"从一个具体不等式逐步搭出整套" 的写法，是泛函分析估计技术的标准模板——你后面看 Sobolev 嵌入定理（第 23 章）和插值定理时会一再撞见。

5.4 Cauchy–Schwarz：Hölder 在 \(p = q = 2\) 的特例

Hölder 取 \(p = q = 2\)（共轭关系 \(1/2 + 1/2 = 1\) 成立），得

\[ \boxed{\ \left|\int f g \, d\mu\right| \le \int |fg| \, d\mu \le \|f\|_2 \cdot \|g\|_2.\ } \]

这就是Cauchy–Schwarz 不等式（积分版）。离散版：\(|\sum x_i y_i| \le \|x\|_2 \|y\|_2\)（你在线性代数里见过的形式）。

请你停下来反省。Cauchy–Schwarz 是整套分析里你用过最多的不等式之一——它在实分析、线性代数、概率、微分方程里反复出现。可它原来只是 Hölder 的一个特例。

这个视角的颠覆性。Cauchy–Schwarz 在以前的课里看起来像个独立 的事实，需要单独证（要么用判别式法 \(\|x - t y\|^2 \ge 0\)，要么用平行四边形）。但放到 Hölder 链子里它不是孤立 的——它是 \(p = 2\) 的对称特例，恰好是因为 \(2\) 是它自己的共轭。

更深的视角。\(L^2\) 是唯一 的"自共轭"（\(p = q\)）的 \(L^p\) 空间。这件事直接对应着：\(L^2\) 是唯一 的有内积 的 \(L^p\) 空间——Cauchy–Schwarz 在 \(L^2\) 上不只是不等式，它定义 了内积。其他 \(L^p\)（\(p \neq 2\)）只是赋范空间，没有内积，所以你在 \(L^p\)（\(p \neq 2\)）里没有"垂直"、"投影到子空间" 这些几何概念。

这是为什么下一章——Hilbert 几何——会专门挑出 \(L^2\) 这一类 Hilbert 空间。整本书"有内积的" 和"只有范数的" 故事在这里分家。

5.5 Bessel 不等式：投影几何里出来的

换一个角度。前面四节的不等式都是从 Young 链生出来——本质是凸性。Bessel 不等式从另一处 长出来——本质是Pythagoras。

设 \(H\) 是 Hilbert 空间，\(\{e_1, e_2, \ldots\}\) 是 \(H\) 里一组正交规范 元素（\(\langle e_i, e_j \rangle = \delta_{ij}\)）。给一个 \(x \in H\)。试着把 \(x\) "投影" 到这组元素张成的子空间上。前 \(N\) 项的投影系数：

\[ c_n = \langle x, e_n \rangle, \quad n = 1, 2, \ldots, N. \]

设 \(s_N = \sum_{n=1}^N c_n e_n\)。从几何上看，\(s_N\) 是 \(x\) 在 \(\{e_1, \ldots, e_N\}\) 张成子空间上的投影；\(x - s_N\) 应该和 \(\{e_n\}\) 都垂直。检查：对 \(j \le N\)，

\[ \langle x - s_N, e_j \rangle = \langle x, e_j \rangle - \sum_{n=1}^N c_n \langle e_n, e_j \rangle = c_j - c_j = 0. \ \checkmark \]

所以 \(x - s_N \perp s_N\)。Pythagoras 给

\[ \|x\|^2 = \|x - s_N\|^2 + \|s_N\|^2. \]

算 \(\|s_N\|^2\)：因为 \(\{e_n\}\) 正交规范，\(\|s_N\|^2 = \sum_{n=1}^N |c_n|^2\)。

所以

\[ \sum_{n=1}^N |c_n|^2 = \|s_N\|^2 = \|x\|^2 - \|x - s_N\|^2 \le \|x\|^2. \]

对 \(N\) 取极限：

\[ \boxed{\ \sum_{n=1}^\infty |\langle x, e_n \rangle|^2 \le \|x\|^2. \ } \]

这就是Bessel 不等式。

请你看清这条证明里发生了什么。Pythagoras 一行就给出来了——其余都是"正交规范" 的代数自动展开。这是为什么"凸性 vs 正交" 是泛函分析里两套独立 的几何工具：Young 链来自凸性，Bessel 来自正交。它们各管各的。

Bessel 不等式的分量：它说 \(x\) 在任意 一组正交规范元素上的"展开系数" 平方和被 \(\|x\|^2\) 控制——所以这些系数可以平方求和。这件事有两个直接后果：

  1. 抽象 Fourier 级数 \(\sum c_n e_n\) 在 \(H\) 中总是收敛 的（部分和是 Cauchy 列，第 7 章会用）。
  2. "等号" 何时成立？答：当 \(\{e_n\}\) 不只是正交规范，还张成稠密集——也就是正交规范基。这条等号叫Parseval 等式，第 7 章专门讲。

Bessel 是"不等式"，Parseval 是"等式"——它们之间的差距 \(\|x\|^2 - \sum|c_n|^2\) 就是 \(x\) 落在 \(\{e_n\}\) 张成子空间正交补里那部分的范数平方。这条几何故事是下两章的全部内容。

5.6 Jensen 不等式：凸性的最后一击

把 Young 那条"\(\log\) 凹" 的论证再推一步——直接对任意 凸函数说话。

定理（Jensen 不等式，离散版）。设 \(\varphi : \mathbb{R} \to \mathbb{R}\) 凸，\(\lambda_1, \ldots, \lambda_n \ge 0\) 满足 \(\sum \lambda_i = 1\)，\(x_1, \ldots, x_n \in \mathbb{R}\)。则

\[ \boxed{\ \varphi\left(\sum_{i=1}^n \lambda_i x_i\right) \le \sum_{i=1}^n \lambda_i \varphi(x_i).\ } \]

证明：对 \(n\) 归纳；\(n = 2\) 是凸函数定义；一般 \(n\) 把前 \(n-1\) 项的凸组合归到一个数，再用 \(n = 2\) 一次。

积分版（更常用）：设 \(\mu\) 是概率测度（\(\mu(\Omega) = 1\)），\(f \in L^1(\mu)\)，\(\varphi : \mathbb{R} \to \mathbb{R}\) 凸。则

\[ \varphi\left(\int f \, d\mu\right) \le \int \varphi(f) \, d\mu. \]

用人话："先平均再凸函数 \(\le\) 先凸函数再平均"。这是对前面"两个数" 不等式的无穷个数 推广。

三个标志性应用：

应用 1（AM–GM 不等式）。取 \(\varphi(x) = -\log x\)（\(\log\) 凹 ⟺ \(-\log\) 凸）。设 \(x_i > 0\)，\(\lambda_i = 1/n\)。Jensen 给

\[ -\log\left(\frac{1}{n} \sum x_i\right) \le \frac{1}{n} \sum (-\log x_i) = -\log\left(\prod x_i\right)^{1/n}. \]

翻过来：

\[ \frac{x_1 + \cdots + x_n}{n} \ge \sqrt[n]{x_1 \cdots x_n}. \]

算术均值 \(\ge\) 几何均值（AM–GM）。这条结果你可能在中学就背过——它的根 是 \(\log\) 函数的凹性 + Jensen。

应用 2（矩生成函数的凸性）。设 \(X\) 是随机变量，\(M_X(t) = \mathbb{E}[e^{tX}]\) 是它的矩生成函数。\(\varphi(x) = e^x\) 凸。Jensen 给

\[ e^{t \cdot \mathbb{E}[X]} = e^{\mathbb{E}[tX]} \le \mathbb{E}[e^{tX}] = M_X(t). \]

翻成：\(M_X(t) \ge e^{t \mu}\)（\(\mu = \mathbb{E}[X]\)）——矩生成函数总在它的"中心指数" 之上。这条不等式是 Chernoff 界（概率统计里讲的"大偏差" 不等式）的起点。

应用 3（条件期望的 Jensen）。对凸 \(\varphi\) 与子 \(\sigma\)-代数 \(\mathcal{G}\)，

\[ \varphi(\mathbb{E}[X | \mathcal{G}]) \le \mathbb{E}[\varphi(X) | \mathcal{G}] \quad \text{a.s.} \]

这条是鞅论里子鞅 / 上鞅 概念的支柱——它说"凸函数把鞅变成子鞅"。这条事实在概率统计里有详细讨论；这里只想让你看到 Jensen 多么无所不在。

5.7 一张全景图

把这一章五条不等式按"长在哪里" 摆一张图：

  • 凸性家族：Young → Hölder → Minkowski → Cauchy–Schwarz（最后一条是 Hölder 的 \(p=2\) 特例）。这一支控制 \(L^p\) 空间的范数与对偶配对。
  • 正交家族：Bessel（来自 Pythagoras + 正交规范）。这一支控制 Hilbert 空间的展开 与投影。
  • 凸函数家族：Jensen（凸函数与积分平均的交换不等式）。这一支控制概率不等式与对数 / 指数估计。

它们之间的关系并不是一层平行，是一张层次图：Jensen \(\Rightarrow\) Young（\(\varphi = \exp\) 与 \(\lambda_1 = 1/p, \lambda_2 = 1/q\)）\(\Rightarrow\) Hölder \(\Rightarrow\) Minkowski；Cauchy–Schwarz 是 Hölder 的特例；Bessel 是另外 一棵树（来自正交几何，与凸性无关）。

这张图你以后会一再回头查。每次写一个估计——

  • "积分 \(\int |f|^p\) 大不大" → 用 Hölder / Minkowski。
  • "内积 \(\langle x, y \rangle\) 大不大" → 用 Cauchy–Schwarz。
  • "系数 \(\sum |c_n|^2\) 收不收敛" → 用 Bessel。
  • "期望 / 平均通过凸函数后变多大" → 用 Jensen。

泛函分析的全部"硬估计" 工程，差不多 就是把这五条不等式以不同顺序、不同代换、不同维度组合起来。它们是这门学科的四则运算。

下一章我们换一个完全不同的方向——把 \(L^2\) 那条"自共轭" 特性独立出来研究。整套Hilbert 空间几何 从那里出发。第一件事：给一个闭凸集和一个外面的点，里面哪个点离它最近。回答这个具体问题，整章就长出来了。
