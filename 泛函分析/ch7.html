第 7 章 正交基与 Fourier 展开

请你回到上一章我们做完的那件事——给 Hilbert 空间一个闭子空间 \(M\)、一个外面的点 \(x\)，最佳逼近 \(P_M(x)\) 是唯一 的，残差 \(x - P_M(x)\) 与 \(M\) 垂直。

现在请你把 \(M\) 推到极端。固定一组正交规范的元素 \(\{e_n\}_{n=1}^\infty \subseteq H\)（\(\langle e_i, e_j \rangle = \delta_{ij}\)）。把 \(M_N := \mathrm{span}\{e_1, \ldots, e_N\}\) 看成"越来越大" 的闭子空间，\(M_1 \subseteq M_2 \subseteq \cdots\)。每个 \(M_N\) 上有最佳逼近

\[ P_{M_N}(x) = \sum_{n=1}^N \langle x, e_n \rangle \, e_n. \]

让 \(N \to \infty\)。问：

\[ P_{M_N}(x) \to \ ? \]

朴素直觉说"应该收敛到 \(x\) 吧——子空间越来越大、最近点越来越好"。可是这个朴素直觉有个漏洞：\(P_{M_N}(x) \to x\) 当且仅当 \(\bigcup M_N\) 的闭包就是整个 \(H\)。如果 \(\{e_n\}\) 张成的子空间不稠密在 \(H\) 里，\(P_{M_N}(x)\) 只会收敛到 \(x\) 在那个子空间闭包里的投影——剩下的"正交补部分" 永远丢掉。

所以"\(\{e_n\}\) 张成稠密" 是把"子空间投影" 推成"恢复全部 \(x\)" 的关键条件。这种"足够大" 的正交规范集叫正交规范基。它把整个 \(H\) 完全分解：每个 \(x\) 都能写成绝对收敛级数 \(x = \sum \langle x, e_n \rangle e_n\)——抽象 Fourier 展开。

这一章把"正交规范基" 这个概念建立成定理，给四条等价刻画，然后看四个标准例子（\(\ell^2\)、三角函数、Legendre、Hermite）。最后两节告诉你两件事：(1) 所有可分无穷维 Hilbert 空间"本质上" 都是 \(\ell^2\)（没有第二种长相）；(2) \(L^2\) 收敛是无穷维上"天然" 的收敛——逐点和一致都是更苛刻的概念。

7.1 正交规范基：定义与四条等价

设 \(H\) 是 Hilbert 空间。一组元素 \(\{e_n\}_{n \in I} \subseteq H\) 叫正交规范集（ONS）若 \(\langle e_i, e_j \rangle = \delta_{ij}\)（即两两垂直，每个范数 1）。

定义：正交规范集 \(\{e_n\}\) 叫正交规范基（ONB，orthonormal basis）若它满足以下任一等价 条件。

定理（正交规范基的四条等价刻画）。设 \(\{e_n\}_{n=1}^\infty \subseteq H\) 是正交规范集。以下四条等价：

(a) "张成稠密"：\(\overline{\mathrm{span}\{e_n\}} = H\)。

(b) "唯一垂直 0"：若 \(x \in H\) 满足 \(\langle x, e_n \rangle = 0\) 对所有 \(n\)，则 \(x = 0\)。

(c) "Parseval 等式"：\(\|x\|^2 = \sum_{n=1}^\infty |\langle x, e_n \rangle|^2\) 对所有 \(x \in H\)。

(d) "展开收敛"：\(x = \sum_{n=1}^\infty \langle x, e_n \rangle e_n\)（在 \(H\) 范数收敛）对所有 \(x \in H\)。

请你停下来端详这四条。它们说的是同一件事 的四个角度：

  • (a) 拓扑视角："这组 \(e_n\) 张得够大，闭包覆盖整个 \(H\)"。
  • (b) 几何视角："没有非零向量同时垂直所有 \(e_n\)"——\(\{e_n\}^\perp = \{0\}\)。
  • (c) 能量视角："\(x\) 的范数平方 = 系数平方和"——Bessel 不等式（第 5 章 5.5）的等号情形。
  • (d) 展开视角："每个 \(x\) 都能写成 \(\{e_n\}\) 的绝对收敛级数"——抽象 Fourier 展开。

四条里你以前接触过的可能是 (d)（Fourier 级数收敛）；最常用的是 (b)（"验证一组 \(\{e_n\}\) 是基" 的最方便办法是查"谁同时垂直所有 \(e_n\)？只有 0"）。

7.2 等价证明：四条互推

证明等价的"三角链" 走法：(a) ⟺ (b)，(a) ⟹ (d) ⟹ (c) ⟹ (a)。

(a) ⟺ (b)：把 \(M = \overline{\mathrm{span}\{e_n\}}\) 当作闭子空间。第 6 章 6.3 节"双重正交补" 给 \((M^\perp)^\perp = M\)。

  • (a) ⟹ (b)：\(M = H\)，所以 \(M^\perp = \{0\}\)。"同时垂直所有 \(e_n\) ⟺ 在 \(M^\perp\) 里 ⟺ 是 0"。
  • (b) ⟹ (a)：(b) 说 \(M^\perp = \{0\}\)，所以 \(M = (M^\perp)^\perp = \{0\}^\perp = H\)。

(a) ⟹ (d)：设 \(M_N = \mathrm{span}\{e_1, \ldots, e_N\}\)。第 6 章 6.2 给 \(P_{M_N}(x) = \sum_{n=1}^N \langle x, e_n \rangle e_n\)。要证 \(\sum_{n=1}^N \langle x, e_n \rangle e_n \to x\)。

设 \(s_N := P_{M_N}(x)\)。第一步：\(\{s_N\}\) 是 Cauchy 列。算 \(\|s_M - s_N\|^2\)（\(M > N\)）：\(s_M - s_N = \sum_{n=N+1}^M \langle x, e_n \rangle e_n\)，正交规范元素的线性组合，所以

\[ \|s_M - s_N\|^2 = \sum_{n=N+1}^M |\langle x, e_n \rangle|^2. \]

Bessel 不等式（5.5）给 \(\sum_n |\langle x, e_n \rangle|^2 \le \|x\|^2 < \infty\)，所以这个尾巴 \(\to 0\)。\(\{s_N\}\) Cauchy。\(H\) 完备，所以 \(s_N \to s\) 对某个 \(s \in H\)。

第二步：要证 \(s = x\)。看 \(x - s\)：对每个 \(j\)，

\[ \langle x - s, e_j \rangle = \langle x, e_j \rangle - \lim_N \langle s_N, e_j \rangle = \langle x, e_j \rangle - \langle x, e_j \rangle = 0 \]

（最后一步：当 \(N \ge j\) 时 \(\langle s_N, e_j \rangle = \langle x, e_j \rangle\)，因为 \(e_j\) 与其它 \(e_n\) 垂直。）

所以 \(x - s\) 同时垂直所有 \(e_n\)。由 (b)（已知 (a) ⟺ (b)，(a) 假设之），\(x - s = 0\)。即 \(s = x\)。

(d) ⟹ (c)：由展开收敛，\(\|x\|^2 = \lim_N \|s_N\|^2\)（范数连续）。\(\|s_N\|^2 = \sum_{n=1}^N |\langle x, e_n \rangle|^2\)（正交规范），所以 \(\|x\|^2 = \sum_{n=1}^\infty |\langle x, e_n \rangle|^2\)。

(c) ⟹ (a)：反证。设 \(M = \overline{\mathrm{span}\{e_n\}} \neq H\)，则 \(M^\perp \neq \{0\}\)。取 \(0 \neq x \in M^\perp\)，那么 \(\langle x, e_n \rangle = 0\) 对所有 \(n\)，所以 (c) 给 \(\|x\|^2 = \sum 0 = 0\)，矛盾。

四条等价证毕。

请你看清这条证明里谁起的什么作用：

  • (a) ⟺ (b)：纯几何（双重正交补）。
  • (a) ⟹ (d)：Bessel 给 Cauchy 性 + \(H\) 完备升级到收敛 + 取极限识别极限。
  • (d) ⟹ (c)：范数连续。
  • (c) ⟹ (a)：反证 + Bessel 不等式的"等号 ⟺ 张成稠密" 几何意义。

这个证明示范了 Hilbert 几何最经典的"四样工具一齐发力"：内积、正交、Bessel、完备性。少一个都不行。

7.3 Parseval 等式：能量守恒的物理意义

把 7.1 (c) 单独提出来。

\[ \boxed{\ \|x\|^2 = \sum_{n=1}^\infty |\langle x, e_n \rangle|^2.\ } \]

左边是 \(x\) 的"能量"（范数平方），右边是 \(x\) 在每个"方向" \(e_n\) 上的分量平方和。Parseval 等式说："总能量 = 分量能量之和"。

这个等式在物理 / 信号处理里叫能量守恒。具体场景：

  • 声音信号 \(f(t)\)：把它做 Fourier 展开，\(f(t) = \sum c_n e^{i 2\pi n t / T}\)。Parseval 给 \(\int |f(t)|^2 dt = \sum |c_n|^2\)：波形的总能量 = 各频率分量能量之和。这是为什么"把信号在频域里分析等价于在时域里分析"——它们的总能量一样多。
  • 量子力学：波函数 \(\psi\) 在能量本征态 \(\{e_n\}\) 上展开，\(\psi = \sum c_n e_n\)。\(|c_n|^2\) 是测得能量为 \(E_n\) 的概率。Parseval 说概率之和 = 1（如果 \(\|\psi\| = 1\)）——概率守恒。这是第 24 章 量子力学应用的核心。
  • 统计：\(L^2\) 函数的方差能被分解到正交基上的"方差贡献" 之和。这是 ANOVA、PCA、谱估计的几何根。

请你停下来感受一件事：Parseval 等式看起来像 个深刻的数学事实。但从这一章的视角看，它就是 Pythagoras 的无穷维版——\(s_N = \sum_{n=1}^N c_n e_n\) 是正交 元素的有限和，所以 \(\|s_N\|^2 = \sum |c_n|^2\)（标准 Pythagoras）；让 \(N \to \infty\)，左右各取极限就是 Parseval。整套"无穷维波动力学" 的能量守恒，本质就是直角三角形勾股定理。

另一个视角：Parseval 让"正交规范基展开" 成为一种同构——稍后 7.5 节我们会看到，它把 \(H\) 等同于 \(\ell^2\)，把 \(x\) 等同于其系数列 \((c_n)\)，且这个等同保范数 + 保内积。

7.4 四个标准例子

抽象定理需要看具体例子才有血肉。这一节给四组正交规范基。

例 1：\(\ell^2\) 的标准基

在 \(\ell^2 = \{(x_n) : \sum |x_n|^2 < \infty\}\) 里取

\[ e_k = (0, \ldots, 0, \underbrace{1}_{\text{第 } k \text{ 位}}, 0, 0, \ldots). \]

\(\langle e_i, e_j \rangle = \delta_{ij}\) 显然。验证 7.1 (b)："同时垂直所有 \(e_n\)" ⟺ "每一位都是 0" ⟺ \(x = 0\)。所以 \(\{e_n\}\) 是 \(\ell^2\) 的正交规范基。

展开 \(x = (x_1, x_2, \ldots) = \sum x_n e_n\)（这就是"列向量是分量之和" 的无穷版）。Parseval：\(\|x\|^2 = \sum |x_n|^2\)（这就是 \(\ell^2\) 范数的定义）。

这是最朴素 的 ONB 例子——它把 7.1 的所有概念在最简单的空间上具体化。

例 2：\(L^2[-\pi, \pi]\) 的三角基

在 \(L^2[-\pi, \pi]\) 里取

\[ e_n(x) = \frac{1}{\sqrt{2\pi}} e^{i n x}, \quad n \in \mathbb{Z}. \]

正交规范：\(\int_{-\pi}^\pi e^{inx} \overline{e^{imx}} \, dx = \int e^{i(n-m)x} dx = 2\pi \delta_{nm}\)。所以这组（除以 \(\sqrt{2\pi}\)）是正交规范集。

"是基" 的证明走 7.1 (a)——这组的张成在 \(L^2[-\pi, \pi]\) 中稠密。这是Stone–Weierstrass 定理 的一个推论：三角多项式在 \(C[-\pi, \pi]\)（连续函数空间）上一致稠密；连续函数在 \(L^2\) 中稠密；所以三角多项式在 \(L^2\) 中稠密。

展开：\(f \in L^2[-\pi, \pi]\) 有

\[ f = \sum_{n \in \mathbb{Z}} \hat f(n) \, \frac{e^{i n x}}{\sqrt{2\pi}}, \quad \hat f(n) = \frac{1}{\sqrt{2\pi}} \int_{-\pi}^\pi f(x) e^{-i n x} \, dx. \]

这就是Fourier 级数。Parseval：\(\int |f|^2 = \sum |\hat f(n)|^2\)（"波形能量 = 频谱能量"）。

实数版：用 \(\{1/\sqrt{2\pi}, \cos(nx)/\sqrt{\pi}, \sin(nx)/\sqrt{\pi}\}_{n \ge 1}\)，正交规范。展开就是你以前学过的 \(f(x) = a_0/2 + \sum (a_n \cos nx + b_n \sin nx)\)（系数差一个标量调整）。

请你回头想：你在微分方程课和偏微分方程课里反复用 Fourier 级数解 PDE——分离变量、热方程、波动方程——背后的"为什么这样的展开收敛" 你以前可能没问过。现在的回答：因为三角函数是 \(L^2[-\pi, \pi]\) 的正交规范基，由 7.1 (d) 展开必然在 \(L^2\) 中收敛。

例 3：\(L^2[-1, 1]\) 的 Legendre 多项式

在 \(L^2[-1, 1]\) 上做Gram–Schmidt：从单项式列 \(\{1, x, x^2, x^3, \ldots\}\) 出发正交化。前几个：

\[ P_0(x) = 1, \quad P_1(x) = x, \quad P_2(x) = \frac{1}{2}(3x^2 - 1), \quad P_3(x) = \frac{1}{2}(5x^3 - 3x), \ldots \]

这组叫Legendre 多项式。归一化后（\(P_n\) 范数是 \(\sqrt{2/(2n+1)}\)）就成正交规范集。

"是基" 的证明：Weierstrass 逼近定理（连续函数在 \([-1, 1]\) 上能被多项式一致逼近）+ 连续函数在 \(L^2\) 中稠密——所以多项式在 \(L^2[-1, 1]\) 中稠密，Legendre 多项式张成稠密。

展开：\(f \in L^2[-1, 1]\) 有 \(f = \sum c_n P_n / \|P_n\|\)。这种展开在数值积分（Gauss–Legendre 求积公式）和 PDE（球面调和函数的径向部分）里反复出现。

例 4：\(L^2(\mathbb{R}, e^{-x^2} dx)\) 的 Hermite 多项式

在加权 \(L^2(\mathbb{R}, e^{-x^2} dx)\)（内积 \(\langle f, g \rangle = \int f \bar g \, e^{-x^2} dx\)）上对 \(\{1, x, x^2, \ldots\}\) Gram–Schmidt。前几个：

\[ H_0(x) = 1, \quad H_1(x) = 2x, \quad H_2(x) = 4x^2 - 2, \quad H_3(x) = 8x^3 - 12 x, \ldots \]

这是Hermite 多项式。归一化后（\(\|H_n\|^2 = \sqrt{\pi} \cdot 2^n n!\)）成正交规范集。

"是基" 涉及"多项式 \(\cdot\) Gauss 衰减" 在加权 \(L^2\) 中稠密——这条证明用 Stone–Weierstrass 的一种变种。

把权 \(e^{-x^2}\) "吸到" 函数里去：定义 \(\tilde H_n(x) = H_n(x) e^{-x^2/2}\)。这组 \(\{\tilde H_n\}\) 是普通 \(L^2(\mathbb{R})\)（Lebesgue 测度）的正交规范基——Hermite 函数。它们是量子谐振子 的能量本征态（第 24 章 应用）。

这四个例子的统一性：每个 ONB 都来自某个"自然" 的正交化过程。\(\ell^2\) 的标准基天然正交。三角函数是平移 / 周期对称性的本征函数。Legendre 是单项式在 \([-1, 1]\) 上的 Gram–Schmidt 结果，并且是 Sturm–Liouville 算子的本征函数。Hermite 同样是某 Sturm–Liouville 算子（量子谐振子）的本征函数。"本征函数 = ONB" 这条线索在第 21 章 紧自伴算子谱定理里被推到极限——每个 紧自伴算子都给出一组 ONB（它的本征函数）。

7.5 可分 Hilbert 空间 ≅ \(\ell^2\)

到现在我们看了几个具体的 Hilbert 空间——\(\ell^2\)、\(L^2[-\pi, \pi]\)、\(L^2[-1, 1]\)、\(L^2(\mathbb{R})\)——它们看起来很不一样：一个是数列空间，三个是函数空间，函数空间又有不同的定义域和测度。可这一节告诉你：它们本质上都是同一个空间——都和 \(\ell^2\) 同构。

先回顾：度量空间叫可分 的若它含可数稠密子集。所有 7.4 的例子都可分（多项式 / 三角多项式 / 等等是它们各自的可数稠密集）。事实上在实际遇到 的 Hilbert 空间几乎全是可分的——非可分 Hilbert 空间是相当特殊的对象。

定理（可分 Hilbert 空间结构定理）。设 \(H\) 是可分无穷维 Hilbert 空间。则 \(H\) 与 \(\ell^2\) 等距同构，即存在 双射 \(U : H \to \ell^2\) 满足

\[ \langle U x, U y \rangle_{\ell^2} = \langle x, y \rangle_H \quad \text{对所有 } x, y \in H. \]

等距同构意味着 \(U\) 不只是双射，还保内积（特别保范数 + 保正交）——\(H\) 与 \(\ell^2\) 在 Hilbert 几何意义下不可区分。

证明草图。两步。

第一步：可分 \(H\) 有可数 ONB。

取 \(\{x_n\}_{n=1}^\infty\) 是 \(H\) 的可数稠密集。对它做Gram–Schmidt 正交化——按顺序取每个 \(x_n\)，减掉它在前面已构造好的 ONS 上的投影分量，再归一化。如果减后是 0（即 \(x_n\) 落在前面的张成里），跳过它。这给一组正交规范集 \(\{e_n\}\)（数量可能少于 \(x_n\)，但因为 \(H\) 无穷维所以仍是无穷的）。

检查 7.1 (a)：\(\{e_n\}\) 张成稠密。理由：\(\mathrm{span}\{e_1, \ldots, e_N\} = \mathrm{span}\{x_1, \ldots, x_M\}\)（某个 \(M \ge N\)），所以 \(\overline{\mathrm{span}\{e_n\}} \supseteq \overline{\{x_n\}} = H\)。所以 \(\{e_n\}\) 是 ONB。

第二步：构造同构 \(U\)。

对 \(x \in H\)，\(U x := (\langle x, e_n \rangle)_{n=1}^\infty\)。Bessel 给 \(\sum |\langle x, e_n \rangle|^2 \le \|x\|^2 < \infty\)，所以 \(U x \in \ell^2\)。

线性是显然的（内积线性）。保范数 / 保内积：Parseval 给 \(\|U x\|_{\ell^2}^2 = \sum |\langle x, e_n \rangle|^2 = \|x\|^2\)；类似地（极化恒等式）保内积。

满射：给任意 \((c_n) \in \ell^2\)，\(\sum c_n e_n\) 在 \(H\) 中收敛（部分和是 Cauchy 列：\(\|s_M - s_N\|^2 = \sum_{n=N+1}^M |c_n|^2 \to 0\)），极限 \(x \in H\)，且 \(\langle x, e_k \rangle = c_k\)（对每个 \(k\) 验证）。

单射：\(U x = 0\) ⟹ \(\langle x, e_n \rangle = 0\) 所有 \(n\) ⟹ \(x = 0\)（7.1 (b)，因为 \(\{e_n\}\) 是 ONB）。

证毕。

\[\boxed{\ \text{所有可分无穷维 Hilbert 空间} \cong \ell^2.\ }\]

请你停下来感受这条结论的结构性。它说的是——抽象的可分 Hilbert 空间只有一种长相。\(L^2[-\pi, \pi]\)、\(L^2[-1, 1]\)、\(L^2(\mathbb{R})\)、抽象的任何 可分无穷维 Hilbert 空间，只要选好一组 ONB 就能"翻译" 成 \(\ell^2\)，每条 Hilbert 几何性质都原样保留。

有限维有平行的事实：所有 \(n\) 维实内积空间都等距同构 于 \(\mathbb{R}^n\)（标准内积）。无穷维的 \(\ell^2\) 在可分 Hilbert 空间这个范畴里扮演的角色就是 \(\mathbb{R}^n\) 在有限维内积空间范畴里的角色——"标准模型"。

这件事的意义。当你在抽象 \(L^2(\Omega)\) 里证明一个 Hilbert 空间结论时——比如"紧自伴算子的本征函数能张成 \(L^2(\Omega)\)"——你本质 在证 \(\ell^2\) 上的对应结论。所有"可分 Hilbert 空间" 故事都能化简到 \(\ell^2\)。这是泛函分析巨大的简化——一个无穷维空间能完全用列向量描述。

注意"可分" 的限制。非可分 Hilbert 空间存在（如 \(\ell^2(I)\)，\(I\) 不可数）；它们的 ONB 也存在（用 Zorn 引理），但大小等于 \(I\)。所有"同维数" 的 Hilbert 空间是同构的（"维数" 在这里是 ONB 的基数）；可分对应可数维。

7.6 \(L^2\) 收敛 vs 逐点 vs 一致

正交规范基展开的"收敛" 是\(L^2\) 范数收敛——\(\|s_N - x\|_{L^2} \to 0\)。请你停下来反省一件事：以前你在微分方程里看到 Fourier 级数 \(\sum (a_n \cos nx + b_n \sin nx)\) 时，可能默默以为它逐点 收敛到 \(f\)，或者更强的一致 收敛。这两种期望都不对。Fourier 级数的收敛在\(L^2\) 意义 下总是 成立（如果 \(f \in L^2\)），逐点 和一致 都不 自动给。

具体看：

  • \(L^2\) 收敛：\(\int |s_N - f|^2 \to 0\)。由 7.1 (d) "展开收敛"，对每个 \(L^2\) 函数自动 给。
  • 逐点收敛：\(s_N(x) \to f(x)\) 对几乎每个 \(x\)。这是 Carleson 1966 的著名定理——成立但极难证（不是这本书的内容）。
  • 一致收敛：\(\sup_x |s_N(x) - f(x)| \to 0\)。需要 \(f\) 满足相当强的光滑条件（如 \(f\) 连续 + \(f' \in L^2\)）。

更不安的——du Bois-Reymond 1873 构造了连续函数 \(f\) 使其 Fourier 级数在某个点不收敛。所以"连续 + Fourier 级数 ⟹ 逐点收敛" 也错。要保证逐点收敛，需要更强的条件（Dirichlet 判别法、Lipschitz 条件、或 Carleson 那种深刻分析）。

请你看清楚这件事的颠覆性。"逐点" 和"一致" 是你以前在实分析里的核心收敛模式——你做了几百页极限交换 / 函数列收敛的练习，全都用这两种。到了 \(L^2\) 里，这两种收敛都"太苛刻"。

反过来，"\(L^2\) 收敛" 是更松 的概念——它只看"积分意义下的差"，不要求每个点都对得上。这种放松正好 配合 Hilbert 几何——它对应"距离 = 范数"。

从泛函分析视角，"\(L^2\) 收敛是无穷维上天然的收敛" 是因为：

  1. 它由内积导出范数，所以你能用平行四边形 / Pythagoras / Bessel / Parseval 等几何工具。
  2. 它等价于"投影 / 展开有限项 \(\to\) 无限项"——这就是 7.1 (d)。
  3. 它让 \(L^2\) 是完备 的，所以 Cauchy 列 \(\to\) 极限——这是 Hilbert 空间结构的基础。

"逐点" 和"一致" 在这套几何里不自然——它们不来自任何范数 / 内积，是另一种（非线性）收敛。一致收敛可以由 \(\|\cdot\|_\infty\) 范数刻画，但 \(L^\infty\) 不是 Hilbert 空间（无内积），所以投影几何完全失效。这是第 5 章结尾我说"\(L^2\) 是唯一有内积的 \(L^p\)" 的延续——只有 \(L^2\) 有"内积带来的几何"，所以只有 \(L^2\) 收敛能配 Hilbert 投影定理。

这条认识在以后的应用里反复重要：

  • PDE 弱解（第 23 章）：用 \(H^1\)（Sobolev 版的 \(L^2\)）收敛 / 范数。逐点意义下"解方程" 太苛刻，\(L^2 / H^1\) 意义下足够。
  • 统计 / ML：均方误差 \(\mathbb{E}[(\hat f - f)^2]\) 就是 \(L^2\) 误差。"逐点最优" 通常做不出来，"\(L^2\) 平均最优" 做得出来——这是为什么回归 / 投影 / 条件期望全用 \(L^2\)。
  • 信号处理：能量度量自然是 \(L^2\)。逐点理想滤波很难（吉布斯现象就是逐点失败的例子），但 \(L^2\) 意义下任何滤波都能"能量近似"。

7.7 一张全景图

这一章把 Hilbert 几何推到了最干净的形式。三件主要事：

  • 正交规范基（7.1 + 7.2）：四条等价刻画——稠密 / 唯一垂直 0 / Parseval / 展开收敛。它把"有内积 + 完备" 加上"足够多的方向" 升级为"每个向量都有展开"。
  • 结构定理（7.5）：所有可分无穷维 Hilbert 空间都同构于 \(\ell^2\)。一种长相，无第二种。
  • 收敛模式（7.6）：\(L^2\) 收敛是 Hilbert 几何原生 的；逐点和一致是更苛刻 的、和这套几何不直接配套的概念。

四个标准例子（7.4）让抽象具体化——\(\ell^2\) 标准基、三角函数、Legendre、Hermite，每一组都来自一个"自然" 的正交化或本征展开。

请你回头看一下这一章和上一章合起来做的事：

  • 第 6 章：投影到一个 闭子空间——给唯一最佳逼近。
  • 本章：投影到所有 ONB 张成的子空间——每个 \(x\) 都被完全展开。

从一个子空间到"所有方向"——这是有限维线性代数"基展开" 思想的无穷维实现。在线性代数里，\(\mathbb{R}^n\) 的标准基让任意 \(x\) 写成 \(x = \sum x_i e_i\)；在 Hilbert 空间里，ONB 让任意 \(x\) 写成 \(x = \sum \langle x, e_n \rangle e_n\)（绝对收敛，用 Bessel + 完备）。同一种代数 / 几何，搬到了无穷维。要付的税：基要正交规范，空间要完备，基要张成稠密——这些"税" 是有限维白送的事实，无穷维必须明示。

下一章我们离开 Hilbert 空间的"静态" 几何，进入算子 的世界——把"线性映射 \(T : X \to Y\)" 当成研究对象，研究"什么时候连续"、"什么时候有界"、"怎么定义算子范数"。这是另一个层级的抽象，也是泛函分析后半本（算子论 + 谱论）的开端。
