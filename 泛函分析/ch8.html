第 8 章 有界线性算子

请你回到第 1 章 1.2 节那个让你不舒服的反例。求导算子

\[ D : (C^1[0, 1], \|\cdot\|_\infty) \to (C[0, 1], \|\cdot\|_\infty), \quad D(f) = f' \]

是数学里最自然的线性算子之一——可它在 sup 范数下不连续。我们当时找到的反例是 \(f_n(x) = \sin(n^2 x)/n\)：\(\|f_n\|_\infty \to 0\)，但 \(\|D f_n\|_\infty = n \to \infty\)。

那一章我把这个事实放在那里，警告你"线性不再自动连续"。但我没告诉你两件关键的事：

  1. 当一个线性算子是连续的，能用什么等价条件判断它？只能去查 \(\varepsilon\)-\(\delta\) 吗？
  2. "连续算子" 这个词太弱——能不能给"连续到什么程度" 一个数，让我们能比较两个算子的"温和度"？

这一章把两个问题彻底解决。第一个问题的答案是这一章的核心定理——一条四等价 链：

\[ \text{连续 } \Longleftrightarrow \text{ 有界 } \Longleftrightarrow \text{ Lipschitz } \Longleftrightarrow \text{ 仅在原点连续}. \]

这条等价让"证算子连续" 这件事彻底机械化——四种不同形式的语句，挑一个最容易验证的就行。第二个问题的答案是算子范数 \(\|T\| := \sup_{\|x\| = 1} \|Tx\|\)——一个把"算子的放大能力" 压成单一数字的不变量。这一章把它定义清楚，下一章把所有这些算子打包成一个 Banach 空间。

8.1 复习：求导算子的崩塌再看一次

让我们把 1.2 节那个反例从一个新角度再读一遍。设 \(T : X \to Y\) 是赋范空间间的线性算子。"线性" 已经给我们一件好事：

\[ T(\alpha x + \beta y) = \alpha T(x) + \beta T(y). \]

特别 \(T(0) = 0\)。所以问"\(T\) 在 \(0\) 是不是连续" 等价于问"\(x_n \to 0\) 时 \(Tx_n \to 0\) 吗"。

对求导算子 \(D\)，我们找到的反例正是 \(f_n \to 0\) 但 \(D f_n\) 不仅不趋零、还跑到无穷。整个失败发生在原点——\(D\) 在 0 这一点都不连续。

更细看：考虑 \(D\) 在另一个点 \(g \in C^1[0, 1]\) 是不是连续。设 \(g_n \to g\)，问 \(D g_n \to D g\) 吗？由线性，\(D g_n - D g = D(g_n - g)\)。让 \(h_n = g_n - g \to 0\)，那么"\(D g_n \to D g\)" 等价于"\(D h_n \to 0\)"。也就是" \(D\) 在 \(g\) 连续 ⟺ \(D\) 在 \(0\) 连续"。

这是个一般现象：线性算子的连续性是个"全或无" 的事——要么处处连续，要么连原点都不连续，没有"在某些点连续在另一些点不连续" 这种事。这个观察就是定理 C4 的核心（"仅在原点连续 ⟹ 处处连续"）——下一节会把它收进四等价里。

现在我们正式开问：到底有什么 等价方式判断"\(T\) 连续"？

8.2 四条等价（C1-C4）

这是这一章的核心定理。设 \(T : X \to Y\) 是赋范空间间的线性 算子。下面四条互相等价：

\[\boxed{\quad \begin{aligned} &\textbf{C1 (连续)：} && T \text{ 在 } X \text{ 上每点连续}.\\ &\textbf{C2 (有界)：} && \exists\, M \ge 0,\ \forall x \in X,\ \|Tx\|_Y \le M \|x\|_X.\\ &\textbf{C3 (Lipschitz)：} && \exists\, M \ge 0,\ \forall x, y,\ \|Tx - Ty\|_Y \le M \|x - y\|_X.\\ &\textbf{C4 (仅原点)：} && T \text{ 在 } x = 0 \text{ 连续}. \end{aligned} \quad}\]

让我把每一条的"意思" 先给你画清楚，再做证明。

  • C1 是定义——\(\varepsilon\)-\(\delta\) 在每一 点都验证一次。
  • C2 把"\(T\) 不会把'有界集' 拉伸到'无界'" 这件事抓出来。如果有这样一个 \(M\)，单位球在 \(T\) 下的像装在半径 \(M\) 的球里——很形象。
  • C3 是Lipschitz 连续——比"连续" 更强的局部条件，意味着"距离的放大率有上界"。
  • C4 是个看似最弱的条件——只在一 个点验证连续。

定理说：这四条对线性算子完全等价。一旦验证其中任何一条，剩下三条自动成立。

证明。我们做一个循环：C1 ⟹ C4 ⟹ C2 ⟹ C3 ⟹ C1。

(C1 ⟹ C4)。显然——"每点连续" 推出"原点连续"。

(C4 ⟹ C2)。这是最关键的一步。设 \(T\) 在 \(0\) 连续。对 \(\varepsilon = 1\)，存在 \(\delta > 0\) 使

\[ \|x\|_X \le \delta \implies \|Tx\|_Y \le 1. \]

现在对任意 \(x \neq 0\)，把它"缩回 \(\delta\) 球内"——令 \(\tilde x = \delta x / \|x\|_X\)。则 \(\|\tilde x\|_X = \delta\)，于是 \(\|T \tilde x\|_Y \le 1\)。利用线性，\(T \tilde x = (\delta / \|x\|_X) T x\)，所以

\[ \|T x\|_Y = \frac{\|x\|_X}{\delta} \|T \tilde x\|_Y \le \frac{1}{\delta} \|x\|_X. \]

取 \(M = 1/\delta\)，C2 成立。\(x = 0\) 时两边都 0，照样成立。

请你停下来欣赏这一步。它展示了线性 在这场证明里的真实分量——线性让我们能把任意点缩放到 \(\delta\) 球内，所以原点附近的连续性自动放大 成全空间的有界性。这种"局部 ⟹ 全局" 的杠杆专属于线性算子。一般非线性映射在 0 连续完全推不出在远处怎样。

(C2 ⟹ C3)。设 \(\|Tx\| \le M \|x\|\)。对任意 \(x, y\)，由线性 \(Tx - Ty = T(x - y)\)，所以

\[ \|Tx - Ty\| = \|T(x - y)\| \le M \|x - y\|. \]

这正是 Lipschitz 的定义。

(C3 ⟹ C1)。设 \(\|Tx - Ty\| \le M \|x - y\|\)。对任意 \(x_0\) 和 \(\varepsilon > 0\)，取 \(\delta = \varepsilon / M\)（若 \(M = 0\) 则 \(T \equiv 0\) 显然连续）。\(\|x - x_0\| < \delta\) 时 \(\|Tx - T x_0\| \le M \delta = \varepsilon\)。连续性成立。

四个等价闭环。

这条定理的使用方法 在以后这本书里会出现几百次：要证一个线性算子连续，找最容易验证的那一条。绝大多数时候是 C2——直接给出常数 \(M\) 满足 \(\|Tx\| \le M \|x\|\)。比如积分算子、乘子算子、有界域上的 Fourier 变换——都是 C2 一行带过。

另外这条定理给"有界算子" 这个名词正名。从此我们说"有界线性算子" 时等价 地是说"连续线性算子" / "Lipschitz 线性算子"——三个名字一个对象。本书后面默认混用，不再区分。

8.3 算子范数

C2 给了一个常数 \(M\) 满足 \(\|Tx\| \le M \|x\|\)。但不止一个 这样的 \(M\)——任何更大的也行。最朴素的问题：有没有一个最小的 \(M\)？

有。它就是算子范数：

\[\boxed{\quad \|T\| := \inf \{ M \ge 0 : \|Tx\| \le M \|x\| \text{ 对所有 } x \in X \}. \quad}\]

朴素地想——这个 inf 应该等于"放大率的上确界"：

\[ \|T\| = \sup_{x \neq 0} \frac{\|Tx\|}{\|x\|}. \]

下面来验证这等式，并给出另外两种等价表达。

命题。对有界线性算子 \(T : X \to Y\)，下列三式相等：

\[\boxed{\quad \|T\| = \sup_{x \neq 0} \frac{\|Tx\|}{\|x\|} = \sup_{\|x\| = 1} \|Tx\| = \sup_{\|x\| \le 1} \|Tx\|. \quad}\]

证明。设三式右边分别为 \(A, B, C\)。由线性 \(Tx / \|x\| = T(x / \|x\|)\)，把 \(x \neq 0\) 归一化为 \(\|x\| = 1\)，得 \(A = B\)。又 \(\{\|x\| = 1\} \subset \{\|x\| \le 1\}\)，所以 \(B \le C\)。反向：对 \(\|x\| \le 1\)，\(\|Tx\| \le M\|x\| \le M\)，取 \(M = B\) 得 \(C \le B\)。所以 \(B = C\)。

剩下要证 \(A = \|T\|\)。"\(\|Tx\| \le M\|x\|\) 对所有 \(x\)" 等价于"\(\|Tx\| / \|x\| \le M\) 对所有 \(x \neq 0\)"，等价于"\(M \ge A\)"。所以最小的 \(M\) 就是 \(A\)。

这三个等价表达各有用场：

  • \(\sup \|Tx\| / \|x\|\)——最直接的"放大率" 视角。算 \(\|T\|\) 时找一个让比值最大的 \(x\)。
  • \(\sup_{\|x\| = 1} \|Tx\|\)——单位球面上的最大像。几何上最干净。
  • \(\sup_{\|x\| \le 1} \|Tx\|\)——闭单位球上的最大像。在做"凸集 / 紧致" 论证时常用。

用算子范数立刻得到一条关键不等式：

\[\boxed{\quad \|Tx\|_Y \le \|T\| \cdot \|x\|_X \quad \text{对所有 } x \in X. \quad}\]

这条不等式将在这本书每一处算估计的地方出现。它是"算子范数控制误差" 这句话的精确表达——如果你的输入误差是 \(\|x\|\)，输出误差不会超过 \(\|T\|\) 倍。这就是为什么算子范数在数值分析里直接就是条件数 的成分（参见数值方法读本第 4 章）。

请你也注意一个微妙处：如果 \(T\) 是从空间 \(X\) 到自身——\(T : X \to X\)——那定义里"两个范数" 其实是同一个范数。在这种情况算 \(\|T\|\) 才有意义，否则你还得说清"我用的是哪一边的范数"。后面 9.4 节的 Banach 代数结构就完全建立在 \(\mathcal{B}(X) = \mathcal{B}(X, X)\) 这种"自映自" 的算子上。

8.4 六个标准例子

抽象定义到这里说完了。光看定义没感觉——我们要看一组具体 算子，把范数算出来。下面六个例子是这本书后面会反复回访的"标准库"。

例 1（恒等）。\(I : X \to X\)，\(I(x) = x\)。显然 \(\|Ix\| = \|x\|\) 对所有 \(x\)。所以 \(\|I\| = 1\)。这条最朴素，但你以后看 \(\mathcal{B}(X)\) 作为代数（下一章）就会发现 \(I\) 是乘法单位元——它的范数刚好 1，是这个代数"归一化" 的来源。

例 2（数乘）。固定 \(\lambda \in \mathbb{R}\)（或 \(\mathbb{C}\)），定义 \(T_\lambda : X \to X\)，\(T_\lambda x = \lambda x\)。\(\|T_\lambda x\| = |\lambda| \|x\|\)，所以 \(\|T_\lambda\| = |\lambda|\)。"算子范数" 退化到"标量绝对值"——这给你一个有限维直觉的一致检查：1×1 矩阵 \([\lambda]\) 的算子范数确实就是 \(|\lambda|\)。

例 3（乘子算子）。这是上一例在函数空间的推广。在 \(L^2[0, 1]\) 上固定一个有界可测函数 \(g \in L^\infty[0, 1]\)，定义

\[ M_g : L^2[0, 1] \to L^2[0, 1], \quad (M_g f)(x) = g(x) f(x). \]

线性显然。算范数：

\[ \|M_g f\|_2^2 = \int_0^1 |g(x)|^2 |f(x)|^2 dx \le \|g\|_\infty^2 \int_0^1 |f|^2 dx = \|g\|_\infty^2 \|f\|_2^2. \]

取根号 \(\|M_g f\|_2 \le \|g\|_\infty \|f\|_2\)，所以 \(\|M_g\| \le \|g\|_\infty\)。反方向（给定下界）需要构造让比值接近 \(\|g\|_\infty\) 的 \(f\)：取 \(g\) 取值接近 \(\|g\|_\infty\) 的小集合 \(E\) 上的特征函数 \(f = \chi_E / \sqrt{|E|}\)，验算给

\[ \boxed{\ \|M_g\| = \|g\|_\infty.\ } \]

让你停下来感受这条等式：作为算子 的"放大率" 等于 \(g\) 作为函数 的"上确界"。它把"函数 的几何" 直接翻译成"算子 的几何"。这个对应在第 21 章谱论里就是"乘子算子的谱 = \(g\) 的本质值域" 的源头。

例 4（积分算子）。设 \(K \in C([0,1] \times [0,1])\) 是连续核函数。定义

\[ T : C[0, 1] \to C[0, 1], \quad (Tf)(x) = \int_0^1 K(x, y) f(y) \, dy. \]

线性显然。算范数（用 sup 范数）：

\[ |(Tf)(x)| \le \int_0^1 |K(x, y)| |f(y)| dy \le \|f\|_\infty \int_0^1 |K(x, y)| dy. \]

对 \(x\) 取 sup：

\[ \|Tf\|_\infty \le \|f\|_\infty \cdot \sup_{x} \int_0^1 |K(x, y)| dy. \]

所以 \(\|T\| \le \sup_x \int |K(x, y)| dy\)。反向也成立（取 \(f(y) = \mathrm{sgn}\, K(x_0, y)\) 的连续逼近），最终

\[ \boxed{\ \|T\| = \sup_{x \in [0, 1]} \int_0^1 |K(x, y)| dy.\ } \]

这个公式叫积分算子的核范数。当 \(K\) 是平方可积时再做一遍同样的算式（用 Cauchy-Schwarz 而不是绝对值）会给出 \(L^2\) 上的Hilbert-Schmidt 范数 \(\|T\|_{HS}^2 = \int \int |K|^2\)——后面第 20 章紧算子里会用。

例 5（移位）。在 \(\ell^2\) 上定义右移

\[ R(x_1, x_2, x_3, \ldots) = (0, x_1, x_2, x_3, \ldots). \]

线性显然。算范数：\(\|R x\|_2^2 = 0^2 + x_1^2 + x_2^2 + \ldots = \|x\|_2^2\)。所以 \(\|R x\|_2 = \|x\|_2\)，\(\|R\| = 1\)。

类似定义左移 \(L(x_1, x_2, x_3, \ldots) = (x_2, x_3, \ldots)\)。\(\|L x\|_2^2 = x_2^2 + x_3^2 + \ldots \le \|x\|_2^2\)，所以 \(\|L\| \le 1\)。取 \(x = e_2\) 给 \(L e_2 = e_1\)、\(\|L e_2\|_2 = 1 = \|e_2\|_2\)，所以 \(\|L\| = 1\)。

这两个算子虽然范数都是 1，性质却差很多——\(R\) 是单射但不满射；\(L\) 是满射但不单射。它们俩在第 1 章 1.5 节的"算子 ≠ 矩阵" 故事里出场过，将在第 21 章谱论里以"残留谱" 和"近似点谱" 的不同形态再次出现。

例 6（求导，重伤）。回到这一章的开头：

\[ D : (C^1[0, 1], \|\cdot\|_\infty) \to (C[0, 1], \|\cdot\|_\infty), \quad Df = f'. \]

反例 \(f_n(x) = \sin(n^2 x) / n\) 给 \(\|f_n\|_\infty \to 0\) 但 \(\|D f_n\|_\infty = n \to \infty\)。比值 \(\|D f_n\| / \|f_n\| = n \cdot n = n^2 \to \infty\)。

所以\(\sup \|Df\| / \|f\|\) 是无穷。按定义 \(\|D\| = \infty\)——不 是有界算子。

修法已经在第 1 章预告过：换输入 范数。如果在 \(C^1[0, 1]\) 上换用 \(\|f\|_{C^1} := \|f\|_\infty + \|f'\|_\infty\)，那么

\[ \|Df\|_\infty = \|f'\|_\infty \le \|f\|_\infty + \|f'\|_\infty = \|f\|_{C^1}. \]

所以在 \((C^1, \|\cdot\|_{C^1}) \to (C, \|\cdot\|_\infty)\) 这对范数下 \(\|D\| \le 1\)。同一个表达式 \(D(f) = f'\)，换一对范数就从无界变有界——这条认识在第 23 章 Sobolev 空间里被推到极致：Sobolev 空间的设计目的 就是让微分算子重新有界。

这六个例子涵盖了这本书后面要遇到的全部"算子原型"——恒等 / 数乘 / 乘子 / 积分 / 移位 / 求导。每一种都有它的几何故事和数值故事，建议你把这一节当作"速查表" 收藏。

8.5 有限维的奇迹：所有线性都自动有界

到了无穷维你才看到"线性 ≠ 连续" 这种事。但有限维呢？让我把那条让你不必 担心的定理写出来：

定理（有限维线性自动有界）。设 \(X, Y\) 是赋范空间，\(\dim X < \infty\)。则任何线性算子 \(T : X \to Y\) 都 是有界的。

证明草图。设 \(\dim X = n\)，取 \(X\) 的基 \(\{v_1, \ldots, v_n\}\)。在 \(X\) 上定义辅助范数 \(|x|_1 := \sum |a_i|\)，其中 \(x = \sum a_i v_i\)。对线性 \(T\)，由三角不等式

\[ \|Tx\|_Y = \left\| \sum a_i T v_i \right\|_Y \le \sum |a_i| \|T v_i\|_Y \le \left( \max_i \|T v_i\|_Y \right) \cdot |x|_1. \]

所以 \(T\) 关于 \(|\cdot|_1\) 范数有界。关键的一步：在有限维 上任意两个范数等价（这条定理在第 1 章 1.4 节讲过、在第 4 章正式证）——所以 \(|\cdot|_1\) 与原来的 \(\|\cdot\|_X\) 等价：存在 \(c > 0\) 使 \(|x|_1 \le c \|x\|_X\)。代入：

\[ \|Tx\|_Y \le \left( \max \|T v_i\| \right) \cdot c \cdot \|x\|_X. \]

有界。证毕。

这条定理告诉你线性代数为什么从来不担心连续性——你在 \(\mathbb{R}^n\) 上写下一个矩阵 \(A\)，它自动 有限范数 \(\|A\|\)。你从不需要专门说"这个矩阵是连续的" 那句话——所有矩阵都是。这是有限维白送的奇迹。

无穷维这件事消失。求导算子 \(D\) 在有限维子空间（比如 \(n\) 次以下多项式）上完全连续；放到整个 \(C^1\) 上立刻无界。失去这条奇迹是从有限维到无穷维要付的" 第一笔大税"——这条税恰恰是定义"有界算子" 这个概念的动机。

等价的一句话：在有限维上"线性算子的研究" = "矩阵的研究"，因为矩阵自动连续。在无穷维上"线性算子的研究" 必须先筛选——只有一部分线性算子值得研究（=有界的那些），剩下的（求导、乘以 \(x\) 的算子等）需要换语境（更强的范数或 Sobolev 空间）才能驯服。

8.6 BLT 延拓定理

这一章最后一条工具——也是后面建构里最常用的一条实际工具。它的名字叫 BLT（Bounded Linear Transformation）延拓定理：在稠子空间上有定义的有界算子，可以唯一地延拓到全空间，不增加范数。

定理（BLT 延拓）。设 \(X\) 是赋范空间，\(D \subset X\) 是稠 子空间（即 \(\overline{D} = X\)）。设 \(Y\) 是Banach 空间。设 \(T_0 : D \to Y\) 是有界线性算子，范数 \(\|T_0\|\)。则存在唯一 的有界线性算子 \(T : X \to Y\) 使

  • \(T|_D = T_0\)（在 \(D\) 上和原来一样）；
  • \(\|T\| = \|T_0\|\)（范数不增）。

证明。对 \(x \in X\)，由 \(D\) 稠，找 \(D\) 中序列 \(x_n \to x\)。\(\{x_n\}\) 是 Cauchy 列，由 Lipschitz \(\|T_0 x_n - T_0 x_m\| \le \|T_0\| \|x_n - x_m\| \to 0\)，所以 \(\{T_0 x_n\}\) 在 \(Y\) 中 Cauchy。这里用到 \(Y\) 完备——它给我们 \(T_0 x_n\) 收敛到某 \(y \in Y\)。

定义 \(T x := y\)。要验证良定义：若 \(x_n', x_n \to x\) 是两个序列，混合成 \(x_1, x_1', x_2, x_2', \ldots\) 仍收敛到 \(x\)，由刚才 \(T_0\)-像收敛到唯一极限——所以两个序列给出同样 \(y\)。

线性继承自 \(T_0\)（极限保线性运算）。范数：

\[ \|Tx\| = \lim \|T_0 x_n\| \le \|T_0\| \cdot \lim \|x_n\| = \|T_0\| \|x\|, \]

所以 \(\|T\| \le \|T_0\|\)。又 \(T\) 是 \(T_0\) 的延拓，"更大空间上的 sup" \(\ge\) "子空间上的 sup"，给 \(\|T\| \ge \|T_0\|\)。等号。

唯一性：若另一个延拓 \(T'\) 也连续，则 \(T'\) 和 \(T\) 在稠 子空间 \(D\) 上相等。由连续性，两个连续函数在稠集上相等就处处相等。\(T = T'\)。

这条定理的用法 在这本书里到处可见。三个典型场景：

  • 定义 Fourier 变换在 \(L^2\) 上。原始 Fourier 变换在 Schwartz 类（光滑速降函数）上有定义；Schwartz 类在 \(L^2\) 中稠。Plancherel 定理给在 Schwartz 上的等距 \(\|\hat f\|_2 = \|f\|_2\)（即 \(\|T_0\| = 1\)）。BLT 把它延拓到全 \(L^2\) 上的等距算子——这就是\(L^2\) Fourier 变换 的诞生方式。完整故事在第 7 章。
  • 定义 Hilbert 空间到对偶的同构（Riesz）。详见第 16 章。
  • 定义紧算子的延拓与谱。详见第 20 章。

请你停下来欣赏这条定理的哲学含义：在无穷维里，你从不 需要一开始就在整个空间上构造算子。"先在稠子空间上定义、再延拓" 是标准 工作流。这意味着复杂构造（比如 Fourier 变换）总能从最朴素的"多项式 / 指数 / 阶梯函数" 开始，再借助 BLT 推到全空间。延拓时唯一付的"税" 是 Y 完备——这正是第 2 章反复强调"完备性是入场券" 的具体兑现。

8.7 从此默认"算子 = 有界线性"

这一章我们把"哪些线性算子值得研究" 这件事彻底解决——四等价定理给我们机械化的判定，算子范数给我们度量"算子温和度" 的数。

从此以后，本书里说"算子" 时默认 指"有界线性算子"。无界线性算子（比如求导算子在 \(C^1\) 配 sup 范数下）这本书不 系统讨论——它们要么换空间（Sobolev）后变有界，要么换语境（无界自伴算子在 Hilbert 空间上有自己的理论）需要更专门的工具，留给后续读物。

我把这一章四条核心结论汇总：

\[\boxed{\quad \begin{aligned} &\textbf{1. 等价链:}\quad && \text{连续 } \Leftrightarrow \text{ 有界 } \Leftrightarrow \text{ Lipschitz } \Leftrightarrow \text{ 仅原点连续}.\\ &\textbf{2. 算子范数:}\quad && \|T\| = \sup_{\|x\| = 1} \|Tx\|, \quad \|Tx\| \le \|T\| \|x\|.\\ &\textbf{3. 有限维奇迹:}\quad && \dim X < \infty \implies \text{每个线性 } T : X \to Y \text{ 自动有界}.\\ &\textbf{4. BLT 延拓:}\quad && D \subset X \text{ 稠}, Y \text{ 完备} \implies \text{ 稠子空间有界算子唯一延拓到 } X. \end{aligned} \quad}\]

带着这四条结论你就有资格进入下一章。第 9 章把所有 有界线性算子打包成一个空间 \(\mathcal{B}(X, Y)\)——它本身是一个赋范空间，并且当 \(Y\) 完备时自动 是 Banach 空间。算子的"放大率" 与新空间的范数是同一件事。这条新结构再加上算子的乘法（"复合"），让 \(\mathcal{B}(X)\) 变成一个Banach 代数——谱论 的全部技术基础。
