第 9 章 算子范数与算子代数

上一章把"哪些 算子值得研究" 这件事筛选清楚——有界线性算子。我们也学会了用 \(\|T\|\) 度量它们的"放大率"。

朴素的下一问：所有有界线性算子 加起来构成的对象长什么样？

具体的：固定两个赋范空间 \(X, Y\)，把所有有界线性算子 \(T : X \to Y\) 收集起来——记作

\[ \mathcal{B}(X, Y) := \{T : X \to Y \mid T \text{ 是有界线性算子}\}. \]

这个集合本身 是个空间。"算子的加法 / 数乘" 是显然的（按值定义），算子范数 \(\|T\|\) 是个范数，于是 \(\mathcal{B}(X, Y)\) 自动成为一个赋范空间。

这一章就是研究这个新空间。三个关键结论：

  1. 当 \(Y\) 完备时，\(\mathcal{B}(X, Y)\) 自动 完备——是 Banach 空间。
  2. 当 \(X = Y\) 时，复合 给 \(\mathcal{B}(X)\) 一个乘法，与算子范数相容（\(\|ST\| \le \|S\| \|T\|\)）——这是个 Banach 代数。
  3. 这个代数里，\(I - T\)（\(\|T\| < 1\)）总可逆，逆是Neumann 级数 \(\sum T^n\)——这条几何级数论证是后面谱论的整个工程基础。

这层"算子也是空间 / 也是代数" 的视角换一次本体论：算子不再是"作用在向量上的东西"，它自己就是个分析对象，有范数、有收敛、有乘法、有可逆性。后面紧算子（ch20）、谱（ch21）、Riesz 表示（ch16）全部以这层视角为前提。

9.1 \(\mathcal{B}(X, Y)\) 是赋范空间

先把"算子的加法 / 数乘" 写清楚。两个有界算子 \(S, T : X \to Y\)，定义

\[ (S + T)(x) := S(x) + T(x), \quad (\alpha T)(x) := \alpha \cdot T(x). \]

线性显然。有界性：

\[ \|(S + T) x\| = \|Sx + Tx\| \le \|Sx\| + \|Tx\| \le (\|S\| + \|T\|) \|x\|. \]

所以 \(S + T\) 有界，且

\[\boxed{\ \|S + T\| \le \|S\| + \|T\|.\ }\]

同样 \(\alpha T\) 有界，\(\|\alpha T\| = |\alpha| \|T\|\)。

验证 \(\|\cdot\|\) 是范数：

  • 正定：\(\|T\| = \sup \|Tx\| / \|x\| \ge 0\)；\(\|T\| = 0 \Leftrightarrow Tx = 0\) 对所有 \(x\) \(\Leftrightarrow T = 0\)。
  • 齐次：\(\|\alpha T\| = |\alpha| \|T\|\)。
  • 三角：上面刚证。

三条全过——\(\mathcal{B}(X, Y)\) 在算子范数下是赋范空间。

这件事其实有点神奇。\(\mathcal{B}(X, Y)\) 的元素是函数（从 \(X\) 到 \(Y\) 的映射），可它们用算子范数聚成一个赋范空间，和你之前看到的"点的空间"（\(\ell^2\), \(C[0, 1]\) 等）地位完全平行。算子也是几何对象。

9.2 \(Y\) 完备 ⟹ \(\mathcal{B}(X, Y)\) 完备

这是这一章第一 个真正定理。它的结论 强（Banach 空间），它的条件 简单（只需 \(Y\) 完备，不要求 \(X\) 完备）。

定理（\(\mathcal{B}(X, Y)\) 是 Banach 空间）。设 \(X\) 是赋范空间，\(Y\) 是 Banach 空间。则 \(\mathcal{B}(X, Y)\) 在算子范数下是 Banach 空间。

证明分四步：取一列 Cauchy 算子 \(\{T_n\}\)；逐点构造极限算子 \(T\)；验证 \(T\) 线性 + 有界；验证 \(T_n \to T\) 在算子范数下。

第一步：构造 \(T\)。设 \(\{T_n\} \subset \mathcal{B}(X, Y)\) 是 Cauchy 列：

\[ \forall \varepsilon > 0,\ \exists N,\ \forall n, m \ge N: \|T_n - T_m\| < \varepsilon. \]

对固定 \(x \in X\)，看 \(\{T_n x\}\) 在 \(Y\) 中。用算子范数控制：

\[ \|T_n x - T_m x\| = \|(T_n - T_m) x\| \le \|T_n - T_m\| \cdot \|x\| < \varepsilon \|x\|. \]

所以 \(\{T_n x\}\) 是 \(Y\) 中 Cauchy 列。用 \(Y\) 完备（这是这条定理唯一一处真正用到的"分析资源"）：\(T_n x\) 收敛到某 \(y \in Y\)。定义 \(T x := y\)。

第二步：\(T\) 线性。由极限保线性运算：

\[ T(\alpha x + \beta x') = \lim T_n(\alpha x + \beta x') = \lim (\alpha T_n x + \beta T_n x') = \alpha Tx + \beta Tx'. \]

第三步：\(T\) 有界。Cauchy 列必有界，所以存在 \(M\) 使所有 \(\|T_n\| \le M\)。对每个 \(x\)：

\[ \|Tx\| = \lim \|T_n x\| \le \limsup \|T_n\| \|x\| \le M \|x\|. \]

所以 \(\|T\| \le M < \infty\)。

第四步：\(T_n \to T\) 在算子范数下。这一步最微妙，要求"对所有 \(x\) 同时控制"。固定 \(\varepsilon > 0\)，取 \(N\) 使 \(n, m \ge N\) 时 \(\|T_n - T_m\| < \varepsilon\)。对任意 \(x\) 和这样的 \(n, m\)：

\[ \|T_n x - T_m x\| < \varepsilon \|x\|. \]

固定 \(n \ge N\)，让 \(m \to \infty\)。由 \(T_m x \to Tx\)，

\[ \|T_n x - T x\| = \lim_{m \to \infty} \|T_n x - T_m x\| \le \varepsilon \|x\|. \]

这条对所有 \(x\) 同时成立，所以

\[ \|T_n - T\| = \sup_{\|x\| = 1} \|T_n x - T x\| \le \varepsilon. \]

对所有 \(n \ge N\) 成立——\(T_n \to T\) 在算子范数下。\(\mathcal{B}(X, Y)\) 完备。

请你停下来欣赏第四步的顺序。我们先固定 \(n\) 让 \(m \to \infty\)（用 \(Y\) 完备得到逐点极限），然后在对所有 \(x\) 同时 取 sup 之前，先把对 \(\varepsilon\) 的一致控制 已经攒好。这种"逐点极限 + 一致控制 = 算子范数收敛" 的论证会反复出现 在这本书每一处证 \(\mathcal{B}(X, Y)\) 元素列收敛的地方。

这条定理一个意外的 推论：对偶空间 \(X^* := \mathcal{B}(X, \mathbb{R})\) 总是 Banach 空间——因为 \(\mathbb{R}\)（或 \(\mathbb{C}\)）完备。无论 \(X\) 自己完不完备，它的对偶都完备。这是第 15 章对偶论的开门事实。

9.3 算子序列三种收敛

\(\mathcal{B}(X, Y)\) 既然是赋范空间，"收敛" 已经有意义——按算子范数 \(\|T_n - T\| \to 0\)。但这不是唯一一种自然的"算子收敛"。下面三种都自然，强弱依次：

\[\boxed{\quad \begin{aligned} &\textbf{一致收敛 (uniform):}\quad && \|T_n - T\| \to 0.\\ &\textbf{强收敛 (strong):}\quad && T_n x \to T x \text{ 对每个 } x \in X.\\ &\textbf{弱收敛 (weak):}\quad && \langle T_n x, y^* \rangle \to \langle T x, y^* \rangle \text{ 对每个 } x, y^*. \end{aligned} \quad}\]

(弱收敛 用到对偶空间 \(Y^*\)，里面的 \(y^* \in Y^*\)。如果 \(Y\) 是 Hilbert 空间，可以把 \(\langle \cdot, y^*\rangle\) 想成与某 \(y \in Y\) 的内积。)

"一致" 比"强" 强；"强" 比"弱" 强——这点你可以一行验：算子范数收敛 ⟹ 算子在每个固定 \(x\) 上收敛（取 \(x\) 后 \(\|T_n x - Tx\| \le \|T_n - T\| \|x\| \to 0\)）；强收敛 ⟹ 弱收敛（按对偶配对的连续性）。

但反向 怎样？让我们看一个让你不舒服的反例：有强收敛但不一致收敛的算子列。

反例（移位算子）。设 \(X = \ell^2\)，\(R\) 是右移算子（上一章例 5）。看其幂 \(R^n\)：

\[ R^n (x_1, x_2, x_3, \ldots) = (\underbrace{0, \ldots, 0}_{n \text{ 个}}, x_1, x_2, \ldots). \]

问 \(R^n\) 收敛吗？算 \(\|R^n e_1\|_2 = 1\)（\(R^n e_1 = e_{n+1}\) 仍是单位长度），所以 \(R^n e_1\) 不趋零。强收敛不到 0。

但算 弱收敛。对每个 \(y \in \ell^2\)：

\[ \langle R^n x, y \rangle = \sum_k (R^n x)_k y_k = \sum_{k > n} x_{k - n} y_k. \]

由 Cauchy-Schwarz \(|\langle R^n x, y \rangle| \le \|x\|_2 \cdot \left(\sum_{k > n} y_k^2\right)^{1/2}\)。后面那个尾项 \(\sum_{k > n} y_k^2 \to 0\)（因为 \(\sum y_k^2 < \infty\)）。所以 \(\langle R^n x, y \rangle \to 0\) 对每个 \(x, y\)——\(R^n\) 弱 收敛到 0。

合起来：

\[\boxed{\ R^n \xrightarrow{\text{弱}} 0, \quad R^n \not\xrightarrow{\text{强}} 0, \quad \text{自然 } R^n \not\xrightarrow{\text{一致}} 0.\ }\]

这是一个标准反例——它告诉你"三种收敛" 在无穷维真的不同。

请你停下来想：右移 \(R^n\) 把每个 \(\ell^2\) 向量"越推越远"——非零分量被推到第 \(n+1, n+2, \ldots\) 位。在每个固定的 投影下（"看第 \(k\) 位"）总会被推到那个位置之外，所以投影变成 0。但整体的 \(\ell^2\) 长度 一直是 1——不可能消失。这是"质量逃到无穷" 的现象，弱收敛到 0 但不强收敛——它将在第 17 章弱收敛和第 18 章 Banach-Alaoglu 里被反复回访。

有限维不会有这种事。\(\mathbb{R}^n\) 上有限维算子的三种收敛等价——这又是有限维的"白送奇迹"。无穷维一裂为三。

9.4 \(\mathcal{B}(X)\) 是 Banach 代数

把定义域 = 值域 限制到一种特殊情况——\(X = Y\)。记 \(\mathcal{B}(X) := \mathcal{B}(X, X)\)。这是X 上的有界算子的全体。

新结构：复合给一个乘法。两个算子 \(S, T \in \mathcal{B}(X)\)，定义 \((S T)(x) := S(T(x))\)。

线性：\(ST(\alpha x + \beta x') = S(\alpha T x + \beta T x') = \alpha STx + \beta STx'\)。

有界：

\[ \|(ST) x\| = \|S(Tx)\| \le \|S\| \|Tx\| \le \|S\| \|T\| \|x\|. \]

所以 \(ST \in \mathcal{B}(X)\)，且

\[\boxed{\quad \|ST\| \le \|S\| \cdot \|T\|. \quad}\]

这条不等式叫子乘法性（submultiplicativity）。它和"恒等元 \(I\) 范数 1" 一起，让 \(\mathcal{B}(X)\) 成为赋范代数。当 \(X\) 完备（即 \(\mathcal{B}(X)\) 完备），它就是Banach 代数——一个完备的赋范代数。

验证三个代数公理（结合性 / 分配性 / 单位元）：

  • 结合性：\((RS)T = R(ST)\)（因为函数复合本就结合）。
  • 分配性：\(R(S + T) = RS + RT\)（每点验证）。
  • 单位元：\(I T = T I = T\)，且 \(\|I\| = 1\)。

但请注意——不交换。\(ST\) 一般不等于 \(TS\)。最简单的反例：在 \(\mathbb{R}^2\) 上取两个旋转矩阵，绕不同轴或不同角度——它们一般不交换。在 \(\ell^2\) 上取右移 \(R\) 和左移 \(L\)：\(L R = I\)（"向右移再向左移" 还原），但 \(R L \neq I\)（"向左移把 \(x_1\) 丢了，再向右移补 0\)\(\)\(\)，结果是 \((0, x_2, x_3, \ldots) \neq (x_1, x_2, \ldots)\)）。所以 \(L R \neq R L\)。这正是第 1 章 1.5 节"算子 ≠ 矩阵" 故事的精确表达——\(R\) 单不满，所以它有左逆但无右逆。

这层"算子是非交换 Banach 代数" 的视角是谱论（第 21 章）的整个工作语言。在那一章你会问：什么样的 \(T \in \mathcal{B}(X)\) 在这个代数里"可逆"？答案不是"单射 + 满射"（无穷维这两个不再等价于可逆），而是更精细的"谱不含 0"。

9.5 Neumann 级数：\(\|T\| < 1 \Rightarrow I - T\) 可逆

谱论的第一 个关键工具——也是在 \(\mathcal{B}(X)\) 里"用代数解决可逆问题" 的最朴素例子。

朴素直觉。在实数 上，\(|t| < 1\) 时几何级数 \(\sum t^n = 1/(1 - t)\)。也就是 \(1 - t\) 可逆，逆等于级数和。能不能直接抄 到 \(\mathcal{B}(X)\)？

答案：可以，只要 \(X\) 完备。

定理（Neumann 级数）。设 \(X\) 是 Banach 空间，\(T \in \mathcal{B}(X)\)，\(\|T\| < 1\)。则 \(I - T\) 在 \(\mathcal{B}(X)\) 中可逆，并且

\[\boxed{\quad (I - T)^{-1} = \sum_{n = 0}^\infty T^n. \quad}\]

该级数在算子范数下绝对收敛，并 \(\|(I - T)^{-1}\| \le 1 / (1 - \|T\|)\)。

证明。先证级数收敛。由子乘法性 \(\|T^n\| \le \|T\|^n\)（按归纳：\(\|T^{n+1}\| = \|T \cdot T^n\| \le \|T\| \|T^n\| \le \|T\|^{n+1}\)）。所以

\[ \sum_{n = 0}^\infty \|T^n\| \le \sum \|T\|^n = \frac{1}{1 - \|T\|} < \infty. \]

这是个绝对收敛 的级数。用 \(\mathcal{B}(X)\) 完备（=用 \(X\) 完备 + 上一节定理）：绝对收敛级数收敛。设和为 \(S := \sum T^n\)。由三角不等式 \(\|S\| \le \sum \|T^n\| \le 1 / (1 - \|T\|)\)。

验证 \(S\) 是 \(I - T\) 的逆。先算

\[ (I - T) S = (I - T) \sum_{n = 0}^\infty T^n = \sum_{n = 0}^\infty (T^n - T^{n+1}) = I - \lim_{N \to \infty} T^{N+1}. \]

由 \(\|T^N\| \le \|T\|^N \to 0\)，最后那项是 0。所以 \((I - T) S = I\)。同样 \(S (I - T) = I\)（"左乘 / 右乘" 都用一遍同样的伸缩级数）。所以 \(S\) 就是 \(I - T\) 的逆。

请你停下来感受这条定理的分量。它把"逆算子的存在性" 这件听起来抽象的事，具体 写成一个级数。这不是"存在某个逆"，是"逆 = 这个级数和"。在数值线性代数里这条定理直接给一个迭代算法：要算 \((I - T)^{-1} b\)，就累加 \(b + Tb + T^2 b + T^3 b + \ldots\) 直到误差够小——这正是数值方法里"Jacobi 迭代" "Gauss-Seidel 迭代" 的精神祖先。

条件 \(\|T\| < 1\) 意味着 \(T\) 是" 弱" 的扰动——\(I - T\) 是"恒等加小扰动"。Neumann 定理说：" 恒等加小扰动" 总可逆，逆有具体级数。

这件事还有一条直接推论：可逆算子的集合在 \(\mathcal{B}(X)\) 中是开集。设 \(A\) 可逆，\(\|A^{-1}\|^{-1}\) 是开球的半径。当 \(B\) 满足 \(\|A - B\| < \|A^{-1}\|^{-1}\)，写 \(B = A - (A - B) = A (I - A^{-1}(A - B))\)。设 \(T = A^{-1}(A - B)\)，则 \(\|T\| \le \|A^{-1}\| \|A - B\| < 1\)，所以 \(I - T\) 由 Neumann 可逆，从而 \(B = A(I - T)\) 也可逆。可逆性是开条件——这条事实在谱论里就成了"谱是闭集" 的来源。

9.6 谱半径公式预告

Neumann 定理给一个充分条件 "\(\|T\| < 1 \Rightarrow I - T\) 可逆"。这个条件能不能放宽？

具体的：考虑 \(I - \lambda T\)，其中 \(\lambda \in \mathbb{C}\) 是数。Neumann 给"\(|\lambda| \|T\| < 1\)" 时可逆。但有时即便 \(|\lambda| \|T\| > 1\)，\(I - \lambda T\) 也可逆——比如 \(T\) 的"真实 大小" 比它的范数小（在某些方向算子被 \(T\) 拉伸，但在大部分方向不会）。

正确的"真实大小" 不是 \(\|T\|\)——是谱半径：

\[ \rho(T) := \sup \{|\lambda| : \lambda \in \sigma(T)\}, \]

其中 \(\sigma(T)\) 是 \(T\) 的谱（让 \(\lambda I - T\) 不可逆的所有 \(\lambda\)）。

关键是谱半径公式（Gelfand）：

\[\boxed{\quad \rho(T) = \lim_{n \to \infty} \|T^n\|^{1/n}. \quad}\]

(这条极限永远存在，详见附录 A。)

这条公式意义重大：

  • 右边是纯算子范数 量，可算可估；左边是谱 量，在复平面上的几何信息。两者相等——这是个"分析-代数" 之间的桥梁。
  • 由不等式 \(\|T^n\|^{1/n} \le \|T\|\)，谱半径 \(\rho(T) \le \|T\|\)。谱半径不超过算子范数——所以 Neumann 的"\(\|T\| < 1\)" 是个偏保守的条件，真正的临界是"\(\rho(T) < 1\)"。
  • 当 \(T\) 自伴或正规（在 Hilbert 空间上），\(\rho(T) = \|T\|\)——两者重合。一般情况严格不等。例：在 \(\ell^2\) 上的 Volterra 算子 \((Vf)(x) = \int_0^x f\)，\(\|V\| = 1\) 但 \(\rho(V) = 0\)（它的所有迭代 \(V^n\) 范数 \(\to 0\)）。

在数值方法里这条公式直接是" 迭代法收敛性" 的判据。要用迭代 \(x_{n+1} = T x_n + b\) 解方程组 \((I - T) x = b\)，迭代收敛 \(\Leftrightarrow \rho(T) < 1\)——参见数值方法读本第 18 章关于 Jacobi / Gauss-Seidel 收敛性的判据。

完整的谱论留到 第 21 章，那里会用 Banach 代数 + Neumann + 谱半径公式 + Cauchy 积分（解析函数演算）一起搭复变函数版的算子理论。这一章先把工具备齐：

\[\boxed{\quad \begin{aligned} &\textbf{1. 空间结构:}\quad && \mathcal{B}(X, Y) \text{ 是赋范空间}; Y \text{ 完备} \Rightarrow \text{ Banach 空间}.\\ &\textbf{2. 三种收敛:}\quad && \text{一致 } \Rightarrow \text{ 强 } \Rightarrow \text{ 弱};\ \text{反向不成立}(R^n \text{ 反例}).\\ &\textbf{3. 代数结构:}\quad && \mathcal{B}(X) \text{ 是 Banach 代数};\ \|ST\| \le \|S\|\|T\|;\ \text{ 不交换}.\\ &\textbf{4. Neumann:}\quad && \|T\| < 1 \Rightarrow (I - T)^{-1} = \sum T^n;\ \text{ 可逆性是开条件}.\\ &\textbf{5. 谱半径:}\quad && \rho(T) = \lim \|T^n\|^{1/n} \le \|T\|;\ \text{ 完整理论见 ch21}. \end{aligned} \quad}\]

下一章我们离开"算子代数" 的纯线性代数面，回到完备空间的纯拓扑——证一个看起来朴素到无聊的定理：完备度量空间不能写成可数个"瘦" 闭集的并（Baire 范畴定理）。这个定理本身的形式 不漂亮——但它一旦套到 Banach 空间上就孵化 出三大原理（一致有界 / 开映射 / 闭图像）。这是泛函分析整本书的脊柱，让所有"逐点信息升级到一致信息" "代数等式自动成为拓扑等式" 的论证有了底层引擎。
