第 1 章 引论：从最速降线到变分原理

1696 年 6 月, Johann Bernoulli 在莱比锡《Acta Eruditorum》上发布一道公开挑战题, 给"欧洲所有最杰出的数学家"。题目是这样的:

"给空间中两点 A 和 B（A 高 B 低, 但不 在同一垂线上）。求一条曲线连接 A 到 B, 使得一颗只受重力作用的小球从 A 出发沿这条曲线滑到 B 所用时间最短。"

请你停下来想一秒——你第一反应 是什么曲线？

大多数人答直线——AB 之间最短路径。这是空间最短, 但时间 不一定最短。第二反应可能是"先垂直下降, 然后水平到 B"——这样开始时获得最大速度。也错。

真答案——一条cycloid（摆线）。具体: 一个圆沿水平直线滚动时, 圆周上一点划出的轨迹。把这条曲线放置成"从 A 出发向下滚, 到 B 结束", 球沿这条线滑下时时间最短。Bernoulli 1696 给挑战, Newton 看到题目当晚 就解出来寄回答案（Newton 当时已 53 岁, 是英国造币厂厂长, 不再做学术）。Newton 匿名投稿, Bernoulli 看到答案直接说: "I recognize the lion by his claw."（"我从爪印认出狮子"）。

这条问题——最速降线问题（brachistochrone）——是历史上第一道严肃的变分问题。它的解法是经典力学一次根本视角变化的开端: 从"瞬间动力学"（Newton 的 F = ma）到"整体路径优化"（Lagrange 的 \(\delta S = 0\)）。这一章用这道历史题作钩子, 把整本书的主线 立起来。

1.1 这道题的两种解法

朴素 Newton 风格解法: 设小球沿曲线 \(y(x)\) 下降。在 \((x, y)\) 处, 由能量守恒 \(\frac{1}{2} m v^2 = m g y\), 得速度 \(v = \sqrt{2gy}\)。线元 \(ds = \sqrt{dx^2 + dy^2}\)。时间

\[ T = \int_A^B \frac{ds}{v} = \int_0^X \frac{\sqrt{1 + y'(x)^2}}{\sqrt{2 g y(x)}} \, dx. \]

问题——找让 \(T\) 取最小的曲线 \(y(x)\)。这不是 一道普通的"找最值" 题——你不是找一个数, 是找一个函数。函数最小化的工具叫变分法。

变分法解法（Euler-Lagrange 方程, 1744）: 设泛函 \(T[y] = \int_0^X F(y, y') dx\) where \(F(y, y') = \sqrt{(1 + y'^2)/(2gy)}\)。极小条件是Euler-Lagrange 方程

\[ \frac{d}{dx} \frac{\partial F}{\partial y'} - \frac{\partial F}{\partial y} = 0. \]

对我们这个 \(F\) 算偏导（一些代数), 得

\[ y(1 + y'^2) = C \quad (\text{C 是常数}). \]

这条 ODE 的解就是 cycloid 的参数化方程

\[ x(\theta) = \frac{C}{2}(\theta - \sin \theta), \quad y(\theta) = \frac{C}{2}(1 - \cos \theta). \]

看到了吗——同一道题两种解法。Newton 的: 写出 \(T\) 表达式直接试各种 \(y\), 难。Lagrange 的（用变分法）: 写出 \(T\) 后套 EL 方程, 一行 ODE 解完。变分法 是工具, 变分原理（"真实路径让某个泛函取极值"）是哲学。

1744 年 Euler 把变分法系统化。1788 年 Lagrange 在《Mécanique analytique》中把整个 经典力学重写成变分形式。这就是 Lagrange 力学的诞生。

1.2 把变分法搬到力学

Bernoulli 的"最短时间" 是几何 / 光学 问题。Lagrange 的洞察——整个力学 都能写成"极小化某个积分"。

具体说: 给一个力学系统（粒子在势 \(V(x)\) 中, 或多粒子, 或刚体, 或带约束）。定义

\[ \boxed{\ L = T - V \quad (\text{动能减势能}),\ } \]

叫Lagrangian。给一条从 \(t_1\) 到 \(t_2\) 的路径 \(q(t)\), 定义

\[ \boxed{\ S[q] = \int_{t_1}^{t_2} L\big(q(t), \dot q(t), t\big) \, dt,\ } \]

叫作用量。

Hamilton 原理（最小作用量原理）: 真实物理路径让 \(S\) 取极值。

这条原则替代 Newton 的 F = ma。给定 Lagrangian, 求 EL 方程

\[ \frac{d}{dt} \frac{\partial L}{\partial \dot q} - \frac{\partial L}{\partial q} = 0 \]

就给出运动方程。对单粒子在 \(V(x)\) 中, \(L = \frac{1}{2} m \dot x^2 - V(x)\), EL 方程算完正好是 \(m \ddot x = -V'(x)\) — 即 F = ma。同样物理结果, 不同语言。

请你停下来感受这件事的分量。Lagrange 不是给一个新理论——他给同样的牛顿物理一个不同 的写法。这种"换写法" 看似无聊（既然结果一样, 何必）。但它的价值 在四个方面: 简化、对称、推广、桥梁。

1.3 为什么换语言: 简化

第一个理由: 处理约束系统。

具体: 单摆。一根绳长 \(\ell\) 一端固定一端挂质量 \(m\)。绳张力 \(T\) 是未知 的（取决于运动状态）。Newton 风格: 写 \(\boldsymbol F = m \boldsymbol a\), 把 \(\boldsymbol F\) 分解成重力 + 张力, 张力分量需要靠约束方程 \(|\boldsymbol r| = \ell\) 求解。两条独立方程 + 一条约束 = 三未知（两个位置 + 张力）。复杂。

Lagrange 风格: 选广义坐标 \(\theta\)（摆角）。约束自动 满足——\(\theta\) 是 1 维的, 摆球总在圆上。Lagrangian:

\[ L = \frac{1}{2} m \ell^2 \dot \theta^2 - m g \ell (1 - \cos \theta). \]

EL 方程: \(\frac{d}{dt}(m \ell^2 \dot \theta) + m g \ell \sin \theta = 0\), 即 \(\ddot \theta + (g/\ell) \sin \theta = 0\)。单摆方程, 一行字, 不 出现张力。

张力哪去了？Lagrangian 里没出现, 因为它是约束力—— 不做功（垂直于运动）。Lagrange 公式自动消去所有约束力, 只留下"真物理" 的力。这种"选广义坐标 + 约束自动消失" 的便利在多体复杂系统（机械臂、行星系统、分子振动）里是必须。Newton F = ma 在这些系统下做不动。

1.4 为什么换语言: 对称 ↔ 守恒律

第二个理由: Noether 定理。

1918 年 Emmy Noether 证: Lagrangian 的每一个连续对称性, 都对应一条守恒律。具体:

  • 时间平移对称（\(L\) 不显含 \(t\)）⟹ 能量守恒
  • 空间平移对称（\(L\) 在 \(\boldsymbol r \to \boldsymbol r + \boldsymbol c\) 下不变）⟹ 动量守恒
  • 旋转对称（\(L\) 在旋转下不变）⟹ 角动量守恒
  • 规范对称（电磁场 \(A \to A + d\chi\) 下不变）⟹ 电荷守恒
  • ……

这条定理把"守恒律" 从"幸运观察" 升级成"对称性的代数推论"。Newton 风格里这些守恒律靠特定例子各自证, 没有统一原因。Lagrange 风格里它们是同一定理 的具体化。

这条 Noether 定理在 20 世纪物理变成圣经——粒子物理把"新粒子 / 新相互作用" 等价为"新对称性", 用 Noether 定理推守恒律（重子数 / 轻子数 / 电荷 / 色…）。第 7 章给完整证明 + 多个例子。

1.5 为什么换语言: 推广性 + 量子桥

第三个理由: Lagrangian 推到任何"几何空间"。

Newton F = ma 写在欧氏空间 里——\(F\) 与 \(a\) 都是向量, 加减需要欧氏结构。怎么把这套搬到弯曲 空间（GR 时空）？硬搬不通。

Lagrangian 写法: \(L = T - V\), \(T = \frac{1}{2} g_{ij}(q) \dot q^i \dot q^j\)（\(g_{ij}\) 是度量张量）。这种写法自动 适应任何流形, 任何度量。所以 Einstein 写广义相对论就用 Lagrangian 公式: 时空作用量 \(S = \int R \sqrt{|g|} \, d^4 x\)（Einstein-Hilbert 作用量）, 极值条件给 Einstein 方程（链微分几何 ch23）。Newton F = ma 完全做不到这一步。

第四个理由: 量子桥。Feynman 1948 路径积分把量子力学写成"所有路径加权 \(e^{i S/\hbar}\)"。经典极限 \(\hbar \to 0\) 时主导贡献 来自 \(S\) 极值的路径——也就是 Lagrange 力学的真路径。所以经典力学是量子力学的稳态相位 极限。Newton F=ma 与量子力学不直接连; Lagrange 力学是量子力学的经典极限。本书ch21 讨论"经典 → 量子" 的桥梁。

请你回头看这四个理由——简化、对称、推广、桥梁。任何一个都足够让 Lagrange 力学值得学。合在一起, Lagrange / Hamilton 力学是物理学不可避免的第二语言。这本书就是教这门语言。

1.6 这本书的路线

24 章五段式：

  • Newton 复习 (ch2-3): 速度回顾, 不重写。中心力 + 开普勒作为"最朴素的两体问题"。
  • Lagrange 力学 (ch4-9): 广义坐标 + 约束 → 变分原理 → EL 方程 → Noether → 小振动 → 刚体。读完这一段你能用 Lagrange 解题。
  • Hamilton 力学 (ch10-15): Legendre 变换 → 相空间 → Poisson 括号 → 正则变换 → Hamilton-Jacobi → 作用-角变量 + 积分系统。这一段是经典力学的几何 化身。
  • 高级 (ch16-21): 微扰 + KAM + 混沌 + 弹性 / 流体 / 经典场论 + 量子前夜。
  • 应用 (ch22-24): 相对论力学 + 航天 + 几何力学（symplectic）。

1.7 临别话

经典力学在物理学课程里常被排在第一年（中学/大一）, 听起来"简单 + 古老"。这是个误导。初等 经典力学（高中级别）确实简单——抛物运动、单摆、碰撞。分析 力学（Lagrange / Hamilton）不 简单——它涉及变分法 / 流形 / symplectic 几何 / 变换群, 数学深度足以让一个数学博士专研多年。本书走的是分析力学 路线。

Arnold 在他的经典教材里写: "经典力学不仅是物理课, 还是一门数学学科; 它的语言是 symplectic 几何 + Lie 群"。这本书的目标就是带你看见这一面——经典力学是几何, 不是公式堆砌。
