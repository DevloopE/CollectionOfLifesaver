第 12 章 Poisson 括号 + 守恒律

上一章我们建立了 Hamilton 框架的几何: 相空间 \((q, p)\)、辛形式 \(\omega\)、Liouville 体积守恒。这一章我们引入它的代数。

具体的引子——请你停下来盯着上一章末尾 Liouville 方程那个表达式:

\[ \frac{\partial \rho}{\partial t} + \sum_i \Big(\frac{\partial H}{\partial p_i} \frac{\partial \rho}{\partial q^i} - \frac{\partial H}{\partial q^i} \frac{\partial \rho}{\partial p_i}\Big) = 0. \]

那个奇怪的"对 \(p\) 微分乘对 \(q\) 微分减对 \(q\) 微分乘对 \(p\) 微分" 的双线性结构出现在哪里？答案——它出现在每一个 与 Hamilton 流有关的计算里。任何函数 \(f(q, p)\) 沿 Hamilton 流的演化, 任何守恒条件, 任何正则变换的不变量, 全都用这个结构。

这就是写一个代数 对象的契机——给它一个名字, 研究它的代数性质, 让"沿 Hamilton 流的演化" 变成一行代数。这就是Poisson 括号。

12.1 Poisson 括号: 定义与性质

朴素。给两个相空间上的光滑函数 \(f(q, p), g(q, p)\)。模仿 Liouville 方程那个表达式, 定义

\[\boxed{\ \{f, g\} := \sum_{i=1}^n \Big(\frac{\partial f}{\partial q^i} \frac{\partial g}{\partial p_i} - \frac{\partial f}{\partial p_i} \frac{\partial g}{\partial q^i}\Big).\ }\]

这就是 Poisson 括号。它把"两个函数" 映射到"另一个函数"——双线性运算 \(\{,\}: C^\infty(\mathbb R^{2n}) \times C^\infty(\mathbb R^{2n}) \to C^\infty(\mathbb R^{2n})\)。

定义看起来像凭空降下来的——为什么是这个特定的组合？两个原因。第一——它就是 Liouville 方程里出现的那个组合; 第二——它是辛形式 \(\omega = \sum dp_i \wedge dq^i\) 在两个函数的微分上的自然 配对（见 12.7）。我们先验证它的代数性质, 再揭示它的几何意义。

性质 1: 双线性。\(\{a f + b g, h\} = a \{f, h\} + b \{g, h\}\), 同理对第二个槽双线性。直接从定义代入验证（每个偏导都是线性算子）。

性质 2: 反对称。\(\{f, g\} = -\{g, f\}\)。从定义

\[ \{f, g\} = \sum_i \big(\partial_{q^i} f \cdot \partial_{p_i} g - \partial_{p_i} f \cdot \partial_{q^i} g\big), \]

交换 \(f \leftrightarrow g\) 得

\[ \{g, f\} = \sum_i \big(\partial_{q^i} g \cdot \partial_{p_i} f - \partial_{p_i} g \cdot \partial_{q^i} f\big) = -\{f, g\}. \]

反对称的立即 推论: \(\{f, f\} = 0\)。

性质 3: Leibniz 法则。\(\{f g, h\} = f \{g, h\} + g \{f, h\}\)。直接验证（链式法则）。这条性质说 \(\{,\}\) 在第一个槽上像"导数" 一样表现。

性质 4: Jacobi 恒等式。这是最深的一条:

\[\boxed{\ \{f, \{g, h\}\} + \{g, \{h, f\}\} + \{h, \{f, g\}\} = 0.\ }\]

三个嵌套括号循环置换 \(f \to g \to h \to f\) 求和等于零。

验证 Jacobi 恒等式是一段标准的代数练习——展开三组嵌套括号, 每个嵌套是四阶 偏导（外层对内层求两次微分）。最终所有四阶偏导项两两抵消。这是因为四阶混合偏导 \(\partial^4/(\partial q^a \partial p_b \partial q^c \partial p_d)\) 等都是对称的（Clairaut）, 而 Poisson 括号的反对称结构让奇置换贡献为零, 偶置换三个的和正好抵消。

简短论证（不展开所有指标）: 注意 \(\{f, \cdot\}\) 是一个一阶微分算子（对每个 \(\cdot\) 取一次偏导）, 所以 \(\{f, \{g, h\}\}\) 包含 \(g, h\) 各最多一次偏导加一次组合, 总共二阶偏导 in \((g, h)\), 一阶 in \(f\)。把三项 cyclic 加起来, 含两次微分作用同一函数的项（如 \(\partial^2 g/\partial q \partial p\) 出现于 \(\{f, \{g, h\}\}\) 又出现于其它项）抵消, 最终结果恒为零。完整代数推导见 Goldstein 第 9 章。

这四条——双线性 + 反对称 + Leibniz + Jacobi——把 \(C^\infty(\mathbb R^{2n})\) 变成一个Poisson 代数。或者说, 函数空间在 Poisson 括号下是一个Lie 代数（双线性 + 反对称 + Jacobi）, 同时配上 Leibniz 法则让它和函数乘法相容。这种"Lie 代数 + 兼容乘法" 的双重结构让 Poisson 括号是经典力学的真正代数。

12.2 Hamilton 方程的代数化

请你看这一行:

\[ \dot q^i = \frac{\partial H}{\partial p_i}, \qquad \dot p_i = -\frac{\partial H}{\partial q^i}. \]

用 Poisson 括号包装。算 \(\{q^i, H\}\):

\[ \{q^i, H\} = \sum_j \Big(\frac{\partial q^i}{\partial q^j} \frac{\partial H}{\partial p_j} - \frac{\partial q^i}{\partial p_j} \frac{\partial H}{\partial q^j}\Big) = \sum_j \delta^i_j \frac{\partial H}{\partial p_j} - 0 = \frac{\partial H}{\partial p_i}. \]

(用 \(\partial q^i/\partial q^j = \delta^i_j\), \(\partial q^i/\partial p_j = 0\)). 同理

\[ \{p_i, H\} = -\frac{\partial H}{\partial q^i}. \]

所以 Hamilton 方程等价 于

\[\boxed{\ \dot q^i = \{q^i, H\}, \qquad \dot p_i = \{p_i, H\}.\ }\]

更一般——任何函数 \(f(q, p, t)\) 沿 Hamilton 流的演化:

\[ \frac{df}{dt} = \sum_i \Big(\frac{\partial f}{\partial q^i} \dot q^i + \frac{\partial f}{\partial p_i} \dot p_i\Big) + \frac{\partial f}{\partial t} = \sum_i \Big(\frac{\partial f}{\partial q^i} \frac{\partial H}{\partial p_i} - \frac{\partial f}{\partial p_i} \frac{\partial H}{\partial q^i}\Big) + \frac{\partial f}{\partial t}, \]

即

\[\boxed{\ \frac{df}{dt} = \{f, H\} + \frac{\partial f}{\partial t}.\ }\]

这一公式是 Hamilton 力学的主方程。它包含了:

  • Hamilton 方程（取 \(f = q^i\) 或 \(p_i\), 这两个不显含 \(t\), \(\partial f/\partial t = 0\)）。
  • 能量守恒（取 \(f = H\), 不显含 \(t\): \(\dot H = \{H, H\} + 0 = 0\)）, 因 \(\{H, H\} = 0\)（反对称）。
  • 守恒律刻画（下节）。

12.3 基本对易关系 \(\{q^i, p_j\} = \delta^i_j\)

把 Poisson 括号施于坐标 自身:

\[ \{q^i, q^j\} = \sum_k \big(\partial_{q^k} q^i \cdot \partial_{p_k} q^j - \partial_{p_k} q^i \cdot \partial_{q^k} q^j\big) = \sum_k \big(\delta^i_k \cdot 0 - 0 \cdot \delta^j_k\big) = 0, \]

同理 \(\{p_i, p_j\} = 0\)。但

\[ \{q^i, p_j\} = \sum_k \big(\delta^i_k \cdot \delta^k_j - 0 \cdot 0\big) = \delta^i_j. \]

整理:

\[\boxed{\ \{q^i, q^j\} = 0, \qquad \{p_i, p_j\} = 0, \qquad \{q^i, p_j\} = \delta^i_j.\ }\]

这是基本 Poisson 关系（fundamental Poisson brackets）。它们刻画了相空间坐标 \((q, p)\) 之间最基础的代数结构。

请你停下来对照量子力学(链V14 量子力学):

\[ [\hat q^i, \hat q^j] = 0, \qquad [\hat p_i, \hat p_j] = 0, \qquad [\hat q^i, \hat p_j] = i \hbar \delta^i_j. \]

对应关系明显—— Poisson 括号 \(\{,\}\) ↔ 量子对易子 \([,]/(i \hbar)\)。这条对应不是巧合, 而是正则量子化 的本质（Dirac 1925）。我们在 12.8 详谈。

12.4 守恒律 ⟺ 与 \(H\) 的 Poisson 括号为零

用主方程 \(\dot f = \{f, H\} + \partial f/\partial t\)。如果 \(f\) 不显含 \(t\):

\[ \dot f = 0 \iff \{f, H\} = 0. \]

所以——函数 \(f\) 沿 Hamilton 流守恒, 当且仅当 \(\{f, H\} = 0\)。

这把"守恒" 这个动力学概念变成了一个纯代数 条件。Newton 框架下检验"这量是否守恒" 需要解 ODE; Lagrange 框架下需要 Noether 定理 + 找对称; Hamilton 框架下你只需要算一个 Poisson 括号。

例 1: \(\{H, H\} = 0\) 由反对称, 所以 \(H\) 永远是它自己的守恒量（前提不显含时间）, 即能量守恒。

例 2: 一维中心力, \(H = p_r^2/(2m) + p_\phi^2/(2 m r^2) + V(r)\) (用极坐标), \(\phi\) 不出现, 所以 \(\{p_\phi, H\} = -\partial H/\partial \phi = 0\)——角动量 \(p_\phi\) 守恒。

例 3: 自由粒子 \(H = p^2/(2 m)\), \(\{p, H\} = -\partial H/\partial q = 0\)——动量 守恒。

这些例子在 Lagrangian / Newton 框架下也容易, 但 Hamilton + Poisson 框架的优势在系统化: 守恒律检测 变成一个机械计算。

12.5 Poisson 定理: 守恒量的 Poisson 括号仍然是守恒量

请你想象——已经知道两个守恒量 \(f, g\)（\(\{f, H\} = 0, \{g, H\} = 0\)）, 它们的 Poisson 括号 \(\{f, g\}\) 是个新函数。这个新函数是不是也守恒？

用 Jacobi 恒等式:

\[ \{f, \{g, H\}\} + \{g, \{H, f\}\} + \{H, \{f, g\}\} = 0. \]

用 \(\{g, H\} = 0\) (前提) 和 \(\{H, f\} = -\{f, H\} = 0\), 前两项都是 0, 剩下 \(\{H, \{f, g\}\} = 0\), 即 \(\{\{f, g\}, H\} = 0\)。即:

\[\boxed{\ f, g \text{ 守恒} \implies \{f, g\} \text{ 守恒}.\ }\]

这叫Poisson 定理（Poisson 1809）。它告诉你——守恒量构成 Lie 代数: Poisson 括号在守恒量的集合里封闭。

这条定理的实用价值——已知两条守恒律, 算它们的括号有时候给新的 守恒律, 而且新守恒律对应的对称可能不 是显然的。例如三维中心力问题中, 角动量分量 \(L_x, L_y, L_z\) 都守恒, 它们的 Poisson 括号给 \(\{L_x, L_y\} = L_z\) 等（见下节）, 这不 给新独立守恒量, 但说明 \(\{L_i\}\) 在 Poisson 括号下闭合, 形成 \(\mathfrak{so}(3)\) Lie 代数。这是经典力学和 Lie 理论之间的第一个深桥。

实战中很多时候 \(\{f, g\}\) 退化为 0 或已知 守恒量的函数（不给新东西）, 但这本身也是信息——它告诉你你已经掌握的守恒量"已经够多" 了, 系统是可积的（链 ch15）。

12.6 角动量的 Poisson 括号

三维粒子, 角动量分量 \(L_i = \epsilon_{ijk} q^j p_k\)（用 Einstein 求和约定 + Levi-Civita 符号 \(\epsilon_{ijk}\)）。

计算 \(\{L_i, L_j\}\):

\[ \{L_i, L_j\} = \{\epsilon_{iab} q^a p_b, \epsilon_{jcd} q^c p_d\}. \]

用双线性 + Leibniz + 基本关系:

\[ \{q^a p_b, q^c p_d\} = q^a \{p_b, q^c p_d\} + \{q^a, q^c p_d\} p_b \]

展开:

\[ \{p_b, q^c p_d\} = \{p_b, q^c\} p_d + q^c \{p_b, p_d\} = -\delta^c_b p_d, \]

\[ \{q^a, q^c p_d\} = \{q^a, q^c\} p_d + q^c \{q^a, p_d\} = q^c \delta^a_d. \]

所以

\[ \{q^a p_b, q^c p_d\} = -q^a \delta^c_b p_d + q^c \delta^a_d p_b. \]

带回去:

\[ \{L_i, L_j\} = \epsilon_{iab} \epsilon_{jcd} \big(-q^a \delta^c_b p_d + q^c \delta^a_d p_b\big). \]

第一项: \(-\epsilon_{iab} \epsilon_{jbd} q^a p_d\)（用 \(\delta^c_b\)）。用 \(\epsilon\)-恒等式 \(\epsilon_{iab} \epsilon_{jbd} = \delta_{id} \delta_{aj} - \delta_{ij} \delta_{ad}\)（经典恒等式: 两 \(\epsilon\) 缩并一个指标）, 得 \(-(\delta_{id} \delta_{aj} - \delta_{ij} \delta_{ad}) q^a p_d = -q^j p_i + \delta_{ij} (q \cdot p)\)。

第二项: \(\epsilon_{iab} \epsilon_{jcb} q^c p_b\)。同样 \(\epsilon\) 恒等式给 \(\delta_{ij} \delta_{ac} - \delta_{ic} \delta_{aj}\), 得 \(\delta_{ij} (q \cdot p) - q^i p_j\)。

两项相加:

\[ \{L_i, L_j\} = -q^j p_i + \delta_{ij}(q \cdot p) + \delta_{ij}(q \cdot p) - q^i p_j = -q^j p_i + 2 \delta_{ij}(q \cdot p) - q^i p_j. \]

嗯——这看起来不太对。让我换一个更干净的算法——直接用分量 写。

具体: \(L_x = q^y p_z - q^z p_y, L_y = q^z p_x - q^x p_z, L_z = q^x p_y - q^y p_x\). 算 \(\{L_x, L_y\}\):

\[ \{L_x, L_y\} = \{q^y p_z - q^z p_y, q^z p_x - q^x p_z\}. \]

四项分开:

\[ \{q^y p_z, q^z p_x\} = q^y \{p_z, q^z\} p_x + q^z \{q^y, p_x\} p_z + (\ldots) = -q^y p_x + 0 = -q^y p_x, \]

(用 \(\{p_z, q^z\} = -1, \{q^y, p_x\} = 0, \{q^y, q^z\} = 0, \{p_z, p_x\} = 0\)).

\[ \{q^y p_z, -q^x p_z\} = -q^y q^x \{p_z, p_z\} - \{q^y, q^x\} p_z p_z + \ldots = 0, \]

\[ \{-q^z p_y, q^z p_x\} = -q^z \{p_y, q^z p_x\} - \{q^z, q^z p_x\} p_y = 0 - 0 = 0, \]

\[ \{-q^z p_y, -q^x p_z\} = q^z \{p_y, q^x p_z\} + \{q^z, q^x p_z\} p_y = q^z \cdot q^x \{p_y, p_z\} + q^z \{p_y, q^x\} p_z + \{q^z, q^x\} p_z p_y + q^x \{q^z, p_z\} p_y = 0 + 0 + 0 + q^x p_y. \]

(因为 \(\{q^z, p_z\} = 1, \{p_y, q^x\} = 0\) 等)

合: \(\{L_x, L_y\} = -q^y p_x + 0 + 0 + q^x p_y = q^x p_y - q^y p_x = L_z\).

同理可得 \(\{L_y, L_z\} = L_x, \{L_z, L_x\} = L_y\), 即

\[\boxed{\ \{L_i, L_j\} = \epsilon_{ijk} L_k.\ }\]

这就是经典角动量代数。它和量子角动量代数

\[ [\hat L_i, \hat L_j] = i \hbar \epsilon_{ijk} \hat L_k \]

(链 V14 ch14) 完全对应——把 \(\{,\}\) 换成 \([,]/(i\hbar)\) 就得到。

这个对应不是表面相似——它揭示转动群 \(SO(3)\) 的 Lie 代数 \(\mathfrak{so}(3)\) 同时支配经典和量子的角动量。两边都说"三个分量在自身括号下封闭, 结构常数是 \(\epsilon_{ijk}\)"。Lie 代数是数学对象, 经典和量子是它在不同语境的表现。

这条平行也实战有用——量子角动量本征值 \(L^2 = \hbar^2 \ell(\ell + 1), L_z = \hbar m\) 的推导纯代数（用 \([L_+, L_-] = 2 \hbar L_z\) 等）, 它的经典翻版是 Casimir 不变量 \(L^2 = L_x^2 + L_y^2 + L_z^2\) 在 Poisson 括号下与所有 \(L_i\) 交换 (\(\{L^2, L_i\} = 0\), 直接用 Jacobi + 自身括号验证)。

12.7 Poisson 括号的几何意义

现在回头解释 Poisson 括号几何上 是什么。

设 \(f, g \in C^\infty(\mathbb R^{2n})\), \(df, dg\) 是它们的微分（1-形式）。辛形式 \(\omega = \sum dp_i \wedge dq^i\) 给一个从 1-形式到向量场的映射（"反度量提升"）: 给 \(df\), 定义 Hamilton 向量场 \(\boldsymbol X_f\) 满足

\[ \iota_{\boldsymbol X_f} \omega = -df, \quad \text{即} \quad \omega(\boldsymbol X_f, \cdot) = -df. \]

这条方程在 \(\omega\) 非退化时唯一确定 \(\boldsymbol X_f\)。具体计算给

\[ \boldsymbol X_f = \sum_i \big(\partial_{p_i} f \cdot \partial_{q^i} - \partial_{q^i} f \cdot \partial_{p_i}\big). \]

请验证: \(\omega(\boldsymbol X_f, \boldsymbol X_g) = \sum_i (\partial_{p_i} f \cdot \partial_{q^i} g - \partial_{q^i} f \cdot \partial_{p_i} g) = \{g, f\} = -\{f, g\}\)。或者:

\[\boxed{\ \{f, g\} = \omega(\boldsymbol X_f, \boldsymbol X_g).\ }\]

(部分文献符号约定不同, 可能差一个负号; 这里取 \(\{f, g\} = \boldsymbol X_g(f) = -\boldsymbol X_f(g)\) 的约定)

这就是 Poisson 括号的几何 意义——它是辛形式 \(\omega\) 作用在两个函数对应的 Hamilton 向量场上的值。所以"Poisson 括号" = "辛形式诱导的双线性"。

这种几何观点使 Poisson 括号独立 于坐标——任何坐标系下计算给同样答案。事实上这是正则变换（ch13）的本质——保持 Poisson 括号的变换 = 保持辛形式的变换 = 保 Hamilton 方程的变换。三种刻画等价。

这段几何也说明 为什么 Poisson 括号有那四条性质——它们是辛结构的代数后果。Lie 代数 \(\mathfrak{X}(M)\) 的向量场括号 \([\boldsymbol X_f, \boldsymbol X_g] = \boldsymbol X_{\{f, g\}}\) 把"函数空间 Poisson 括号" 翻译成 "向量场李括号"——两边的 Jacobi 恒等式互相推导。

12.8 量子化原则: \(\{,\} \to \frac{1}{i \hbar}[,]\)

Dirac 1925 发现的对应原则——经典 Poisson 括号 在量子化下变成量子对易子除以 \(i \hbar\):

\[\boxed{\ \{f, g\}_{\text{经典}} \quad \longleftrightarrow \quad \frac{1}{i \hbar} [\hat f, \hat g]_{\text{量子}}.\ }\]

具体:

  • 基本关系 \(\{q, p\} = 1\) ↔ \([\hat q, \hat p] = i \hbar\)（Heisenberg 关系）。
  • Hamilton 方程 \(\dot f = \{f, H\}\) ↔ Heisenberg 方程 \(\dot{\hat f} = \frac{1}{i\hbar} [\hat f, \hat H]\)（量子时间演化）。
  • 角动量代数 \(\{L_i, L_j\} = \epsilon_{ijk} L_k\) ↔ \([\hat L_i, \hat L_j] = i \hbar \epsilon_{ijk} \hat L_k\)。
  • 守恒律 \(\{f, H\} = 0\) ↔ \([\hat f, \hat H] = 0\)（量子守恒律）。

这条对应是 1925-1926 年量子力学诞生的导航星。Heisenberg 在 1925 给出"矩阵力学" 时, Dirac 看到 Heisenberg 的对易子有"非交换乘法" 的代数结构, 立刻想起 Poisson 括号——两者都是反对称双线性 + Jacobi 恒等式。Dirac 在 1925 年 Cambridge 做了正确的猜想: 量子代数是把经典 Poisson 括号用 \([\cdot, \cdot]/(i\hbar)\) 替换得到的。

这条对应不是定理（数学上"量子化" 这个映射并非唯一, 有排序歧义等技术问题, 数学家 Groenewold-van Hove 1946 给出的不可能定理 说严格"整体" 量子化不存在）, 但它在物理上足够好——大多数实际系统的量子化用 Dirac 法则就行, 偶尔有排序问题用对称化或 Weyl 量子化处理。

这条对应揭示了一个深刻事实——量子力学不是经典力学的彻底替代, 而是经典力学的"变形": 把 Poisson 括号变形成对易子, \(\hbar\) 是变形参数。\(\hbar \to 0\) 极限下两者应该收敛, 这是对应原理（Bohr 1913）。详细见 V14 量子力学 + 即将的 V21 量子场论。

从 ch13 开始我们用这个代数视角研究正则变换——保 Poisson 括号的坐标变换, 量子对应是幺正变换。两边的对应在那一章会展现得更深。

12.9 临别话

这一章把 Hamilton 力学的动力学 提炼成代数。一个简单运算 \(\{f, g\}\) 包含了:

  • Hamilton 方程: \(\dot q^i = \{q^i, H\}, \dot p_i = \{p_i, H\}\)。
  • 沿流演化: \(\dot f = \{f, H\} + \partial_t f\)。
  • 守恒律刻画: \(\{f, H\} = 0\)。
  • Poisson 定理: 守恒量空间在括号下闭合。
  • 角动量 Lie 代数 \(\mathfrak{so}(3)\): \(\{L_i, L_j\} = \epsilon_{ijk} L_k\)。
  • 正则变换刻画: 保 \(\{,\}\) ⟺ 保 Hamilton 方程（下章）。
  • 量子化: \(\{,\} \to \frac{1}{i \hbar} [,]\)。

这是物理学典型的"代数化收益" 案例——把动力学问题化成代数问题, 大量原本需要"解 ODE" 的事变成"算括号"。

下一章我们用 Poisson 括号刻画正则变换——相空间上保 Hamilton 方程形式的坐标变换。这是相空间的"等距"（保辛形式）变换, 形成无穷维群。生成函数 \(F(q, Q)\) 把变换参数化, 让"找好坐标" 这件事系统化。再下一章 ch14 我们用 Hamilton-Jacobi 方程把"找最好坐标" 变成解一个 PDE。再下一章 ch15 给出作用-角变量——可积系统的最优坐标。这一段共五章是经典力学的核心几何代数。
