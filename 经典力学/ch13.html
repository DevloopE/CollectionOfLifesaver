第 13 章 正则变换

请你回想 Lagrange 力学里换坐标。给一个变量替换 \(q^i \to Q^i(q, t)\), Lagrangian 自动变成 \(L(Q, \dot Q, t)\)（用链式法则把 \(\dot q\) 变成 \(\dot Q\)）, EL 方程在新坐标下形式不变。这件事很方便——你可以为不同问题选不同的广义坐标, EL 方程的形式不变。

请你停下来想 Hamilton 力学里能不能也这样。Hamilton 方程涉及两组变量 \((q, p)\), 而且 \(p\) 是共轭动量, 不是独立的——它是 \(p = \partial L/\partial \dot q\) 由 \(L\) 决定。所以"换坐标" 在相空间是个更微妙的事。

具体地——能不能做一个变换

\[ (q, p) \to (Q(q, p, t), P(q, p, t)) \]

使得 Hamilton 方程在新坐标下还是 Hamilton 方程的形式？也就是说存在一个新 \(K(Q, P, t)\) 使

\[ \dot Q^i = \frac{\partial K}{\partial P_i}, \qquad \dot P_i = -\frac{\partial K}{\partial Q^i}. \]

这种"保 Hamilton 方程" 的变换称为正则变换（canonical transformation）。它们形成一个无穷维群, 是相空间的结构保持 变换——经典力学几何里"等距" 变换的对应物。

这一章我们做四件事——给正则变换三种等价定义、用生成函数 描述它们、做两个具体例子（谐振子简化、相空间旋转）、揭示时间演化本身 是正则变换。最后一点深刻得离谱——它把"动力学" 解构成"对称变换", 是 ch14 Hamilton-Jacobi 方程的种子, 也是 V14 量子力学"幺正演化" 的经典对应。

13.1 正则变换的三种等价定义

朴素定义。变换 \((q, p) \to (Q, P)\) 是正则的, 如果存在 \(K(Q, P, t)\) 使 Hamilton 方程在新坐标下成立。

这定义对——但不可操作: 给一个变换, 你怎么验证它"正则"？暴力验证 Hamilton 方程在新坐标下成立太麻烦。需要更结构化 的等价刻画。

定义 1: 保 Hamilton 方程。如刚才说的——存在 \(K\) 使新坐标下 Hamilton 方程仍成立。

定义 2: 保辛形式。变换保辛形式 \(\omega = \sum dp_i \wedge dq^i\) 不变:

\[ \sum_i dP_i \wedge dQ^i = \sum_i dp_i \wedge dq^i. \]

(等号在外微分代数里; 即新坐标下计算的 \(\omega\) 等于旧坐标下的 \(\omega\)).

定义 3: 保 Poisson 括号。新坐标下计算的 Poisson 括号和旧坐标下相同:

\[ \{f, g\}_{Q, P} = \{f, g\}_{q, p} \quad \forall f, g. \]

等价地——基本 Poisson 关系在新坐标下保持:

\[ \{Q^i, Q^j\}_{q, p} = 0, \qquad \{P_i, P_j\}_{q, p} = 0, \qquad \{Q^i, P_j\}_{q, p} = \delta^i_j. \]

三种定义等价。这一节剩下证一段, 把三者串起来。

13.2 三种定义的等价性证明

定义 2 ⟺ 定义 3。先看这一对。Poisson 括号几何上 \(\{f, g\} = \omega(\boldsymbol X_f, \boldsymbol X_g)\)（上一章 12.7）。所以变换保 \(\omega\) 等价于保 Poisson 括号。具体——若变换保 \(\omega\), 任两个函数微分 \(df, dg\) 在新旧坐标下"同样的" \(\omega\) 配对; 反之同样。

更代数地: 把 \(2n\) 维相空间坐标排成 \(\xi = (q^1, \ldots, q^n, p_1, \ldots, p_n)\)。辛形式矩阵

\[ J = \begin{pmatrix} 0 & I_n \\ -I_n & 0 \end{pmatrix}. \]

(\(I_n\) 是 \(n\) 阶单位矩阵)。Poisson 括号

\[ \{f, g\} = \sum_{a, b} \frac{\partial f}{\partial \xi^a} J^{ab} \frac{\partial g}{\partial \xi^b}. \]

变换 \(\xi \to \eta(\xi)\), Jacobian \(M^a_b = \partial \eta^a/\partial \xi^b\)。新 Poisson 括号 \(\{f, g\}_\eta = \sum_{a, b} (\partial f/\partial \eta^a) J^{ab} (\partial g/\partial \eta^b)\)。变量代换让旧括号变形:

\[ \{f, g\}_\xi = \sum_{a, b, c, d} \frac{\partial f}{\partial \eta^c} M^c_a J^{ab} M^d_b \frac{\partial g}{\partial \eta^d} = \sum_{c, d} \frac{\partial f}{\partial \eta^c} (M J M^T)^{cd} \frac{\partial g}{\partial \eta^d}. \]

所以 \(\{f, g\}_\xi = \{f, g\}_\eta\) 对所有 \(f, g\) ⟺ \(M J M^T = J\)。这条矩阵 方程就是 \(M\) 是辛矩阵的定义。它等价于变换保辛形式。

定义 1 ⟺ 定义 3。这一对稍复杂。一边——保 Hamilton 方程的变换必保 Poisson 括号。设 \(K(Q, P, t)\) 是新 Hamiltonian, 取 \(f = Q^i, g = Q^j\), 由主方程 \(\dot Q^i = \{Q^i, K\}_{Q, P} + \partial Q^i/\partial t\)。但同时 \(\dot Q^i = \{Q^i, H\}_{q, p} + \partial Q^i/\partial t\)（在原坐标下用主方程）。两个比较 + 任意 \(H\), 推出 \(\{Q^i, Q^j\}_{q, p} = \{Q^i, Q^j\}_{Q, P} = 0\) 等基本关系——具体推导用变换的每个 函数都遵守主方程。

反方向——保 Poisson 括号 ⟹ 保 Hamilton 方程。设 \(\{Q^i, P_j\}_{q, p} = \delta^i_j\) 等。算 \(\dot Q^i\):

\[ \dot Q^i = \frac{\partial Q^i}{\partial q^j} \dot q^j + \frac{\partial Q^i}{\partial p_j} \dot p_j + \frac{\partial Q^i}{\partial t}. \]

用 \(\dot q^j = \partial H/\partial p_j, \dot p_j = -\partial H/\partial q^j\):

\[ \dot Q^i = \sum_j \Big(\frac{\partial Q^i}{\partial q^j} \frac{\partial H}{\partial p_j} - \frac{\partial Q^i}{\partial p_j} \frac{\partial H}{\partial q^j}\Big) + \frac{\partial Q^i}{\partial t} = \{Q^i, H\}_{q, p} + \frac{\partial Q^i}{\partial t}. \]

把 \(H\) 看成 \((Q, P)\) 的函数 \(K(Q, P, t) = H(q(Q, P, t), p(Q, P, t), t)\)（如果变换不显含时间, \(K = H\); 否则需要修正项, 见 13.3 末尾）。算 \(\{Q^i, K\}_{q, p}\):

\[ \{Q^i, K\}_{q, p} = \sum_j \Big(\frac{\partial Q^i}{\partial q^j} \frac{\partial K}{\partial p_j} - \frac{\partial Q^i}{\partial p_j} \frac{\partial K}{\partial q^j}\Big). \]

但 \(\partial K/\partial p_j = \sum_k (\partial K/\partial Q^k) (\partial Q^k/\partial p_j) + (\partial K/\partial P_k)(\partial P_k/\partial p_j)\), 类似 \(\partial K/\partial q^j\)。代回去, 用 \(\{Q^i, Q^k\}_{q, p} = 0, \{Q^i, P_k\}_{q, p} = \delta^i_k\), 化简得

\[ \{Q^i, K\}_{q, p} = \sum_k \delta^i_k \frac{\partial K}{\partial P_k} = \frac{\partial K}{\partial Q^i}\,\text{的对应}\;\text{即}\;\frac{\partial K}{\partial P_i}. \]

(具体是 \(\partial K/\partial P_i\), 用基本 Poisson 关系筛出)。所以 \(\dot Q^i = \partial K/\partial P_i\)——新坐标下 Hamilton 方程形式正确。同理 \(\dot P_i = -\partial K/\partial Q^i\)。完成。

结论——三种定义等价。任意一条都可以用作"正则变换" 的判据。在实战中:

  • 验证一个具体变换是否正则 → 用定义 3（计算基本 Poisson 括号）。
  • 构造正则变换 → 用生成函数（下节）。
  • 理解几何 → 用定义 2（保辛形式）。

13.3 生成函数 \(F(q, Q, t)\)

正则变换系统化 的方式——用生成函数。

核心想法: 在 \((q, p)\) 变换到 \((Q, P)\) 时, 作用量原理 \(\delta \int (\sum p \dot q - H) dt = 0\) 在新坐标下应该也成立 \(\delta \int (\sum P \dot Q - K) dt = 0\)。两者都给同样的物理, 所以 Lagrangian 形式 \(\sum p \dot q - H\) 与 \(\sum P \dot Q - K\) 之差应该是全微分:

\[ \sum_i p_i \dot q^i - H = \sum_i P_i \dot Q^i - K + \frac{dF}{dt}, \]

其中 \(F\) 是某个函数（叫生成函数）。整理:

\[ \sum_i p_i dq^i - \sum_i P_i dQ^i + (K - H) dt = dF. \]

(把 \(\dot q dt = dq\) 等式代换)

所以 \(F\) 应该看成 \((q, Q, t)\) 的函数:

\[ F = F_1(q, Q, t), \quad dF_1 = \sum_i \frac{\partial F_1}{\partial q^i} dq^i + \sum_i \frac{\partial F_1}{\partial Q^i} dQ^i + \frac{\partial F_1}{\partial t} dt. \]

对照系数:

\[\boxed{\ p_i = \frac{\partial F_1}{\partial q^i}, \qquad P_i = -\frac{\partial F_1}{\partial Q^i}, \qquad K = H + \frac{\partial F_1}{\partial t}.\ }\]

这就是第一类生成函数。它给一种构造 正则变换的办法——选任何 \(F_1(q, Q, t)\), 上面三条公式定义一个正则变换。

但 \(F_1(q, Q)\) 假设 \(q\) 和 \(Q\) 都是独立变量——有时候一个变换里 \(q\) 与 \(Q\) 不 独立（例如恒等变换 \(Q = q, P = p\), \(q\) 与 \(Q\) 完全相同）。这种情况下用其它 三类生成函数:

第二类: \(F_2(q, P, t)\)（用 \(q, P\) 当独立变量）。Legendre 变换 \(F_2 = F_1 + \sum P_i Q^i\), 给

\[ p_i = \frac{\partial F_2}{\partial q^i}, \qquad Q^i = \frac{\partial F_2}{\partial P_i}, \qquad K = H + \frac{\partial F_2}{\partial t}. \]

第三类: \(F_3(p, Q, t)\)。\(F_3 = F_1 - \sum p_i q^i\), 给

\[ q^i = -\frac{\partial F_3}{\partial p_i}, \qquad P_i = -\frac{\partial F_3}{\partial Q^i}, \qquad K = H + \frac{\partial F_3}{\partial t}. \]

第四类: \(F_4(p, P, t)\)。\(F_4 = F_1 - \sum p_i q^i + \sum P_i Q^i\), 给

\[ q^i = -\frac{\partial F_4}{\partial p_i}, \qquad Q^i = \frac{\partial F_4}{\partial P_i}, \qquad K = H + \frac{\partial F_4}{\partial t}. \]

四类对应"独立变量选择" 的四种组合。每个具体变换问题选哪类视情况而定——选让计算最简单的那类。

例: 恒等变换 \(Q = q, P = p\)。\(F_1\) 不行（\(q, Q\) 不独立）。\(F_2(q, P) = \sum_i q^i P_i\) 给 \(\partial F_2/\partial q^i = P_i = p_i\)（正确, 因 \(P = p\)）, \(\partial F_2/\partial P_i = q^i = Q^i\)（正确）。所以 \(F_2 = q \cdot P\) 是恒等变换的生成函数。这把 \(F_2\) 标榜为"最常用" 的一类——它包含恒等变换附近的所有正则变换。

13.4 例 1: 谐振子的"超级坐标"

考虑一维谐振子 \(H = p^2/(2m) + (1/2) m \omega^2 q^2\)。请你停下来想——能不能找一个正则变换 \((q, p) \to (Q, P)\), 让新 \(K(Q, P)\) 看起来极简, 比如 \(K = \omega P\)？

这个目标的吸引力——若 \(K = \omega P\), Hamilton 方程给

\[ \dot Q = \frac{\partial K}{\partial P} = \omega, \qquad \dot P = -\frac{\partial K}{\partial Q} = 0. \]

解出来 \(Q(t) = \omega t + Q_0, P(t) = P_0\)——最 简单的解, 一条直线 + 一个常数。原振子的复杂运动在这个新坐标下退化 成"匀速直线"。

怎么找这个变换。直觉——谐振子相轨迹是椭圆 \(p^2/(2m) + (1/2) m \omega^2 q^2 = E\)。把它参数化:

\[ q = \sqrt{\frac{2 E}{m \omega^2}} \sin Q, \qquad p = \sqrt{2 m E} \cos Q. \]

(\(Q\) 是椭圆上的"角度"). 想让 \(K = \omega P\) 等于 \(E\), 即 \(P = E/\omega\)。所以猜:

\[ q = \sqrt{\frac{2 P}{m \omega}} \sin Q, \qquad p = \sqrt{2 m \omega P} \cos Q. \]

验证这是正则变换——计算 \(\{Q, P\}_{q, p}\) 应该等于 1。逆变换:

\[ P = \frac{1}{2 \omega} (p^2/m + m \omega^2 q^2) = \frac{H}{\omega}, \qquad Q = \arctan\Big(\frac{m \omega q}{p}\Big). \]

计算 \(\{Q, P\}_{q, p}\):

\[ \{Q, P\} = \frac{\partial Q}{\partial q} \frac{\partial P}{\partial p} - \frac{\partial Q}{\partial p} \frac{\partial P}{\partial q}. \]

\(\partial P/\partial p = p/(m \omega), \partial P/\partial q = m \omega q/\omega \cdot \omega = m \omega q\), 嗯让我重做——\(P = H/\omega = p^2/(2 m \omega) + (1/2) m \omega q^2\), 所以 \(\partial P/\partial p = p/(m \omega), \partial P/\partial q = m \omega q\).

\(Q = \arctan(m \omega q/p)\), \(\partial Q/\partial q = (m \omega/p) \cdot 1/(1 + (m \omega q/p)^2) = m \omega p/(p^2 + m^2 \omega^2 q^2) = m \omega p/(2 m \omega P) = p/(2 P)\)。

类似 \(\partial Q/\partial p = -m \omega q/p^2 \cdot 1/(1 + (m\omega q/p)^2) = -m \omega q/(p^2 + m^2 \omega^2 q^2) = -m \omega q/(2 m \omega P) = -q/(2 P)\)。

合:

\[ \{Q, P\} = \frac{p}{2 P} \cdot \frac{p}{m \omega} - \Big(-\frac{q}{2 P}\Big) \cdot m \omega q = \frac{p^2}{2 m \omega P} + \frac{m \omega q^2}{2 P} = \frac{1}{P} \cdot \frac{p^2/(2 m \omega) + (1/2) m \omega q^2}{1} = \frac{P}{P} = 1. \]

(用 \(P = p^2/(2 m \omega) + (1/2) m \omega q^2\))

验证: \(\{Q, P\}_{q, p} = 1\). 所以这是正则变换。新 Hamiltonian \(K = H = \omega P\)。Hamilton 方程: \(\dot Q = \omega, \dot P = 0\). 解 \(Q(t) = \omega t + Q_0, P(t) = E/\omega\)。回到原坐标:

\[ q(t) = \sqrt{\frac{2 E}{m \omega^2}} \sin(\omega t + Q_0), \qquad p(t) = \sqrt{2 m E} \cos(\omega t + Q_0). \]

这就是谐振子的标准解——但不 通过解 ODE 得到, 而是通过选好坐标。在新坐标里运动是平凡的, 难度全部转移到"找变换" 这一步。

这种 \((Q, P)\) 叫作用-角变量 (action-angle variables): \(Q\) 是相角（每周期增 \(2\pi\)）, \(P\) 是作用量（积分 \(\oint p \, dq/(2\pi)\)）。\(P = E/\omega\) 在量子力学里直接对应能量量子化 \(E_n = \hbar \omega (n + 1/2)\)（旧量子论 Bohr-Sommerfeld 量子化 \(\oint p \, dq = n h\)）, 所以这里 \(P\) 量子化 \(P = n \hbar\) 给 \(E = n \hbar \omega\)（差 \((1/2) \hbar \omega\) 是零点能, 旧量子论看不到）。这种"作用-角变量" 在 ch15 系统讨论。

13.5 例 2: 简单变换 \(q \to p, p \to -q\)

请你考虑一个非常具体、非常奇怪的变换:

\[ Q = p, \qquad P = -q. \]

"把动量当新坐标, 把负坐标当新动量"。这是不是正则变换？

验证: \(\{Q, P\}_{q, p} = \{p, -q\} = -\{p, q\} = -(-1) = 1\)。是。

新 Hamiltonian: 把原 \(H(q, p)\) 中 \(q \to -P, p \to Q\) 替换:

\[ K(Q, P) = H(-P, Q). \]

例如自由粒子 \(H = p^2/(2m)\) 变成 \(K = Q^2/(2 m)\)。Hamilton 方程: \(\dot Q = \partial K/\partial P = 0, \dot P = -\partial K/\partial Q = -Q/m\). 解出 \(Q = \) 常数, \(P = -Q t/m + P_0\). 在原变量: \(p = \) 常数（动量守恒）, \(-q = -p t/m + P_0\), 即 \(q = (p/m) t - P_0\)（自由粒子做匀速直线）。和原解一致。

请你停下来欣赏这一变换的几何意义。\((q, p) \to (p, -q)\) 是相空间上的旋转 \(\pi/2\)（顺时针）。在 \((q, p)\) 平面把 \(q\) 轴转到 \(p\) 轴位置, \(p\) 轴转到 \(-q\) 轴位置。这种"互换 \(q\) 与 \(p\)" 的变换在 Lagrange 框架是不可思议 的——\(q\) 是位置, \(p\) 是动量, 物理意义不同。但 Hamilton 框架下 \(q\) 与 \(p\) 在方程中地位平等, 这种互换是正则变换的合法 操作。

这就是 Hamilton 框架的几何解放——相空间的对称性比位形空间大, 包含了"位置和动量互换" 这种 Lagrange 想都不敢想的变换。这种对称在量子力学里以"Fourier 变换互换 \(\hat q\) 与 \(\hat p\) 表象" 复活——位置表象和动量表象是同一 量子态的两种描述, 通过 Fourier 变换互换。Hamilton 力学的正则变换 群是这一切的经典基础。

13.6 无穷小正则变换 + 生成元

请你考虑一个非常接近恒等 的正则变换

\[ Q^i = q^i + \epsilon \, \xi^i(q, p), \qquad P_i = p_i + \epsilon \, \eta_i(q, p), \]

其中 \(\epsilon\) 是无穷小参数, \(\xi^i, \eta_i\) 是某个向量场。问: 什么条件下这是正则变换？

计算 \(\{Q^i, P_j\}_{q, p}\) 到 \(\epsilon\) 的一阶:

\[ \{Q^i, P_j\} = \{q^i + \epsilon \xi^i, p_j + \epsilon \eta_j\} = \delta^i_j + \epsilon(\{q^i, \eta_j\} + \{\xi^i, p_j\}) + O(\epsilon^2). \]

正则条件 \(\{Q^i, P_j\} = \delta^i_j\), 所以 \(\epsilon\) 一阶项

\[ \{q^i, \eta_j\} + \{\xi^i, p_j\} = \frac{\partial \eta_j}{\partial p_i} - \frac{\partial \xi^i}{\partial q^j} = 0. \]

类似 \(\{Q^i, Q^j\} = 0\) 给 \(\partial \xi^j/\partial p_i - \partial \xi^i/\partial p_j = 0\), 即 \(\xi\) 在 \(p\) 上的对称。\(\{P_i, P_j\} = 0\) 给 \(\partial \eta_j/\partial q^i - \partial \eta_i/\partial q^j = 0\), \(\eta\) 在 \(q\) 上对称。这些条件等价于存在一个函数 \(G(q, p)\), 使

\[ \xi^i = \frac{\partial G}{\partial p_i}, \qquad \eta_i = -\frac{\partial G}{\partial q^i}. \]

(标准向量场积分性条件——\(\xi, \eta\) 是某个标量函数 \(G\) 通过 Hamilton 向量场公式生成)

所以无穷小正则变换

\[\boxed{\ Q^i = q^i + \epsilon \frac{\partial G}{\partial p_i}, \qquad P_i = p_i - \epsilon \frac{\partial G}{\partial q^i}.\ }\]

函数 \(G\) 叫生成元（infinitesimal generator）。任何标量函数 \(G(q, p)\) 都生成一个无穷小正则变换。

用 Poisson 括号写得更紧凑:

\[ \delta f := f(Q, P) - f(q, p) = \epsilon \{f, G\} + O(\epsilon^2), \]

对任何函数 \(f\)。换言之——无穷小正则变换 = "用 \(G\) 通过 Poisson 括号作用"。这就是 Hamilton 框架的对称性 的代数表达。

13.7 时间演化是正则变换

请你看一个特殊选择的生成元: 取 \(G = H\), \(\epsilon = dt\). 无穷小变换

\[ \delta q^i = dt \cdot \frac{\partial H}{\partial p_i} = \dot q^i \, dt, \qquad \delta p_i = -dt \cdot \frac{\partial H}{\partial q^i} = \dot p_i \, dt. \]

这就是 Hamilton 流动一步的位移——时间演化本身 是无穷小正则变换, 生成元就是 \(H\) 自己。

或者积分起来——从时刻 \(t_0\) 到时刻 \(t_0 + t\) 的时间演化映射 \(\Phi_t: (q_0, p_0) \mapsto (q(t), p(t))\) 是正则变换。

请你停下来感受这件事的份量——动力学不是某种"运动", 它是相空间上一族正则变换。Hamilton 流是相空间的"对称变形", 时刻 \(t\) 不同则正则变换不同, 形成一族单参数子群（\(\Phi_{s + t} = \Phi_s \circ \Phi_t\)）。

这条洞察的三重后果:

  • Liouville 定理是"正则变换保相空间体积" 的必然——上一章 11.6 可以由此立刻看出。
  • Hamilton 是时间演化的生成元——这条事实在量子力学翻译为"\(\hat H\) 是幺正演化的生成元", \(U(t) = e^{-i \hat H t/\hbar}\)。经典 Hamilton 流 ↔ 量子幺正演化, 都由 \(H\) 生成。
  • 动力学和"对称变换" 在 Hamilton 框架下语法上不可区分——一切都是"某个函数 \(G\) 通过 Poisson 括号生成的变换"; 时间演化 (\(G = H\))、空间平移 (\(G = p\))、旋转 (\(G = L\)) 都是这族变换的例子。"动力学" 退化成一种特殊的对称变换。

13.8 守恒量是正则变换的生成元（相空间 Noether）

Noether 定理（ch7）说"对称 ↔ 守恒律"。在 Lagrange 框架它说"位形空间对称（\(L\) 不变变换）↔ 共轭动量守恒"。在 Hamilton 框架可以更清晰地表述。

命题: 设 \(G(q, p)\) 不显含时间。则 \(G\) 是 Hamilton 流的守恒量 (\(\dot G = 0\)) 当且仅当 \(G\) 生成的正则变换保 Hamiltonian \(H\)。

证明。\(G\) 生成的无穷小变换让任何函数 \(f\) 变为 \(f + \epsilon \{f, G\}\)。\(H\) 在变换下的变化:

\[ \delta H = \epsilon \{H, G\} = -\epsilon \{G, H\}. \]

"变换保 \(H\)" 即 \(\delta H = 0\), 即 \(\{G, H\} = 0\)。但 \(\{G, H\} = 0\) 也是 "\(G\) 守恒" 的条件（ch12 主方程给 \(\dot G = \{G, H\}\)）。两条等价。Q.E.D.

这就是 Noether 定理在 Hamilton 框架的极简 版本——一行 Poisson 括号验证。具体例子:

  • \(G = p_x\) 生成沿 \(x\) 平移: \(\delta q^x = \{q^x, p_x\} = 1, \delta q^y = 0, \delta q^z = 0, \delta p = 0\)。即 \(q^x \to q^x + \epsilon\), 是 \(x\) 方向的平移。\(G\) 守恒 ⟺ \(\{p_x, H\} = -\partial H/\partial x = 0\) ⟺ \(H\) 在 \(x\) 平移下不变。动量守恒 ↔ 平移对称。
  • \(G = L_z\) 生成绕 \(z\) 轴旋转: \(\delta q^x = \{q^x, L_z\} = -q^y\) 等, 给绕 \(z\) 轴的无穷小旋转。\(L_z\) 守恒 ⟺ \(\{L_z, H\} = 0\) ⟺ \(H\) 在绕 \(z\) 转下不变。角动量守恒 ↔ 旋转对称。
  • \(G = H\) 生成时间演化。\(H\) 守恒 ⟺ \(\{H, H\} = 0\)（恒成立）⟺ \(H\) 在时间演化下不变。即\(H\) 不显含时间（注意自动满足）⟺ 能量守恒 ↔ 时间平移对称。

Hamilton 框架下 Noether 定理变成一句话: 守恒量 \(G\) 与它生成的正则变换的对称性是同一件事。这是对称性物理的统一 视角——粒子物理里的"规范对称 ↔ 守恒荷" 是这一原则在场论里的扩展。

13.9 临别话

这一章的中心命题——正则变换是相空间的"等距"。它们形成无穷维群, 由生成函数 \(F\) 或生成元 \(G\) 参数化, 保持辛形式 / Poisson 括号 / Hamilton 方程形式。它们让"找好坐标" 这件事在 Hamilton 框架下有一个系统 的工具。

本章的三个深刻结论:

  • 谐振子可以通过正则变换"退化" 成 \(K = \omega P\)。这是作用-角变量 的雏形——任何"可积系统" 都能通过正则变换化简成 \(K = K(P)\)（不依赖 \(Q\)）的形式, 解出来全是直线运动。ch15 系统化。
  • 时间演化本身是正则变换, 生成元是 \(H\)。这把"动力学" 解构成"对称变换"。Hamilton 流不是某种"运动", 是相空间的结构变形。这条洞察直接通往 ch14 Hamilton-Jacobi 方程: 找一个特殊正则变换, 让 \(K \equiv 0\) (新 Hamiltonian 全消)。
  • 守恒量是正则变换的生成元。Noether 定理在 Hamilton 框架退化成 \(\{G, H\} = 0\)。这是粒子物理"对称 ↔ 守恒" 范式的原型。

下一章 Hamilton-Jacobi 方程把这套技术推到极限——找一个使 \(K \equiv 0\) 的正则变换, 当代入 Hamilton 方程时新坐标完全静止 (\(\dot Q = 0, \dot P = 0\)), 即 \(Q, P\) 都是常数。这听起来不可能——但只要解一个 PDE \(\partial S/\partial t + H(q, \partial S/\partial q, t) = 0\), 就能找到这种变换。这条 PDE 是经典力学的压轴成就, 也是量子力学波函数 \(\psi = e^{i S/\hbar}\) 的种子（短波极限下）。从 Bernoulli 1696 一直走到 Hamilton-Jacobi PDE, 经典力学的从粒子轨道到波动方程 这一弧线在 ch14 闭合。
