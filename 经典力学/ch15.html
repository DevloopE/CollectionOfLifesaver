第 15 章 作用-角变量 + 积分系统

上一章末尾 Jacobi 给了一个深刻的诠释——找正则变换让 \(H = 0\)，所有运动停止。这是极限情形。这一章问一个相邻问题——能不能找到正则变换让 Hamilton 方程最简单可能，但不要求 \(H = 0\)？

具体——能不能找一对新坐标 \((I, \theta)\)，让

\[ H = H(I), \quad \text{即 } H \text{ 不依赖 } \theta? \]

如果可以，Hamilton 方程平凡解：\(\dot I = -\partial H / \partial \theta = 0\)（\(I\) 守恒）；\(\dot \theta = \partial H / \partial I = \omega(I)\)（\(\theta\) 线性增长）。一切运动变成"常速绕一个圈"。

这种 \((I, \theta)\) 在周期系统（谐振子、单摆、Kepler 椭圆轨道）中存在并称为作用-角变量（action-angle variables）。能这么做的系统叫积分系统（integrable systems）。

15.1 周期运动的"作用"

从最朴素的开始：一维周期系统。\(H(q, p)\) 是给定能量 \(E\) 时\((q, p)\) 平面上的一条闭曲线（能量等高线 \(H = E\)）。每个周期粒子绕这条曲线跑一圈。具体例：

谐振子 \(H = p^2 / (2m) + \frac{1}{2} m \omega^2 q^2\)——能量等高线是椭圆。粒子在 \((q, p)\) 平面绕椭圆跑，周期 \(T = 2\pi / \omega\)。

单摆 \(H = p^2 / (2m\ell^2) + m g \ell (1 - \cos q)\)——小振幅时近似椭圆，大振幅时变形（甚至完全打开成"翻滚" 状态）。

Kepler 椭圆——粒子在 \((r, p_r)\) 截面上的运动是"近日点 \(r_{\min}\) 到远日点 \(r_{\max}\) 来回"，闭曲线。

给定一个周期闭曲线，定义

\[ \boxed{\ I := \frac{1}{2 \pi} \oint p\, dq.\ } \]

这个量叫作用变量（action variable）——在 \((q, p)\) 平面上沿闭曲线一圈对 \(p\, dq\) 积分，再除以 \(2\pi\)。注意它不是作用量 \(S = \int L\, dt\)——它是别的东西。两个量都叫"action" 是术语撞车（Hamilton 主函数 + 作用变量），上下文区分。

请你停下来想 \(I\) 的几何含义。\(\oint p\, dq\) 是闭曲线包围的相空间面积（Stokes 定理：\(\oint p\, dq = \iint dp \wedge dq\)）。除以 \(2\pi\) 是为了让 \(\theta\) 在标准 \([0, 2\pi)\) 取值。所以

\[ I = \frac{1}{2 \pi} \cdot \text{周期闭曲线包围的面积}. \]

这是一个纯几何量——和参数化方式无关。同一条闭曲线无论你怎么走（顺时针、逆时针、从哪个点起步），\(I\) 是同一个数。

15.2 谐振子的作用-角

跑一个具体例子。一维谐振子 \(H = p^2 / (2m) + \frac{1}{2} m \omega^2 q^2 = E\)。能量等高线

\[ \frac{p^2}{2 m E} + \frac{q^2}{2 E / (m \omega^2)} = 1, \]

这是一个椭圆，半轴 \(\sqrt{2 m E}\)（沿 \(p\) 方向）和 \(\sqrt{2 E / (m \omega^2)}\)（沿 \(q\) 方向）。面积 \(= \pi \cdot \sqrt{2 m E} \cdot \sqrt{2 E / (m \omega^2)} = 2\pi E / \omega\)。所以

\[ I = \frac{1}{2\pi} \cdot \frac{2 \pi E}{\omega} = \frac{E}{\omega}. \]

\(I = E / \omega\)。把它倒过来：\(E = \omega I\)，即

\[ \boxed{\ H(I) = \omega I.\ } \]

纯洁。\(H\) 是 \(I\) 的线性 函数。\(\theta\) 满足 \(\dot \theta = \partial H / \partial I = \omega\)，所以 \(\theta = \omega t + \theta_0\)——线性增长。

反推 \((q, p)\)：标准定义里 \(\theta\) 是椭圆上的相位角。把 \(q = \sqrt{2 I / (m \omega)} \sin \theta\)、\(p = \sqrt{2 m \omega I} \cos \theta\) 代回去验证：\(H = (2 m \omega I / 2 m) \cos^2 \theta + \frac{1}{2} m \omega^2 \cdot 2 I / (m \omega) \sin^2 \theta = \omega I (\cos^2 \theta + \sin^2 \theta) = \omega I\)。✓

这条 \(q = A \sin(\omega t + \theta_0)\)（\(A = \sqrt{2I / (m \omega)}\)）是谐振子的标准解，但你第一次从"找正则变换让 \(H = H(I)\)" 的角度推出来。这条视角的好处——它不依赖谐振子是线性。任何周期系统都有作用-角变量，结构同样。

请你看 \(I = E / \omega\) 这条关系。Bohr 1913 的量子化条件 \(E = n \hbar \omega\)（谐振子）就等价于 \(I = n \hbar\)。"作用变量在量子论里被量子化为 \(\hbar\) 的整数倍"——15.7 节回到这件事。

15.3 单摆与 Kepler

15.3.1 单摆

\(H = p^2 / (2 m \ell^2) + m g \ell (1 - \cos q)\)。小振幅 \(q \ll 1\) 时 \(1 - \cos q \approx q^2 / 2\)，回到谐振子 \(I = E / \omega\)，\(\omega = \sqrt{g/\ell}\)。

大振幅时，\(I = (1/2\pi) \oint p\, dq\) 是个椭圆积分，没有初等表达。频率 \(\omega(I) = dH/dI\) 不再常数——振幅依赖。这就是单摆的"非线性"：大振幅周期变长，小振幅周期 \(2\pi \sqrt{\ell/g}\)。这个非线性反映在 \(H(I)\) 不是线性函数。

临界能量 \(E = 2 m g \ell\) 时单摆"勉强翻顶"——这是分界轨道（separatrix），\(I\) 在这里有跳变（从"振荡" 区到"翻滚" 区）。分界轨道是非积分系统 的种子——下一章混沌从这里长出来。

15.3.2 Kepler

三维 Kepler 是3 自由度系统——需要三个 作用变量 \((I_r, I_\theta, I_\phi)\)。HJ 方程的完全分离（14.4.3 节）给出

  • \(I_\phi = L_z\)（z 角动量）
  • \(I_\theta + I_\phi = L\)（总角动量）
  • \(I_r + I_\theta + I_\phi = k \sqrt{m / (-2 E)}\)（"主作用"，由 Kepler 第三定律编码）

合起来 \(H = -m k^2 / (2 (I_r + I_\theta + I_\phi)^2)\)——\(H\) 只依赖三个作用之和。这是 Kepler 的退化性（degeneracy）——三个频率 \(\omega_r, \omega_\theta, \omega_\phi\) 全部相等，所以椭圆轨道闭合（不退转）。Kepler 是maximally degenerate 系统——这种特别现象的来源是 Kepler 问题有额外 守恒量（Laplace-Runge-Lenz 矢量），让相空间的对称群比一般中心力还大。

太阳系不是严格 Kepler——其它行星扰动让退化打破，地球轨道近日点缓慢转动（牛顿验证过这条）。这种"perihelion precession" 是非退化扰动 的标志，下一章主题。

15.4 Liouville-Arnold 定理：积分系统的精确刻画

把上面"找作用-角" 的故事推到一般 \(n\) 自由度系统。问题：什么时候存在作用-角？答案是Liouville-Arnold 定理——经典力学最美的结构定理之一。

定理（Liouville-Arnold，约 1840 / 1963）. 设 \(H\) 是一个 \(n\) 自由度 Hamilton 系统。如果存在 \(n\) 个独立的守恒量 \(F_1, F_2, \ldots, F_n\)（其中 \(F_1 = H\)）满足:

  1. 它们独立：\(dF_1, \ldots, dF_n\) 在 phase space 上几乎处处线性独立；
  2. 它们两两Poisson 对易：\(\{F_i, F_j\} = 0\)（"in involution"）；
  3. 能量等值面 \(M_c := \{F = c\}\) 是紧致连通 的；

则:

  1. \(M_c\) 微分同胚于 n 维环面 \(T^n = S^1 \times S^1 \times \cdots \times S^1\)；
  2. 存在作用-角变量 \((I_1, \ldots, I_n, \theta_1, \ldots, \theta_n)\) 使 \(H = H(I)\) 不依赖 \(\theta\)；
  3. 解为 \(\theta_i(t) = \omega_i(I) t + \theta_i^0\)，频率 \(\omega_i = \partial H / \partial I_i\)，\(I_i\) 守恒。

系统称完全可积（completely integrable）。

请你停下来想这条定理的力量。它说"n 个对易守恒量 ⟹ 相空间叶层化为 n 维 torus，每个 torus 上动力学是常频准周期"。

三个条件每一条都关键：

  • 独立——少一个守恒量你只压缩到 \(2n - k + 1\) 维子流形（\(k\) 是独立守恒量数），不到 \(n\) 维。
  • 对易——非对易的守恒量给"非交换" 的 Lie 群结构（如 SU(2) 角动量），结果是球面 不是 torus。
  • 紧——能量面无界时（如自由粒子）拓扑不是 torus 而是 \(\mathbb R^n\)。

定理的构造性 部分——给三条件就构造出 \((I, \theta)\)——通过 HJ 方程的完全分离实现。\(I_i\) 是各分离方向的"每周期 \(\oint p\, dq\) 除以 \(2\pi\)"。\(\theta_i\) 由生成函数 \(W(q, I)\) 通过 \(\theta_i = \partial W / \partial I_i\) 给出。

15.5 完全可积系统是稀有

读完上一节你可能想"n 个对易守恒量？听起来不难，每个 1 自由度系统都有 \(H\) 守恒，多自由度系统应该也总能找到"。

不。完全可积系统是相空间里的零测集。Poincaré 1890 年证明三体问题不可积——除了能量、动量、角动量这"朴素" 守恒量（共 7 条，3 自由度三体系统需要 9 条），不存在其它解析守恒量。

这条结果引发20 世纪动力系统革命。Birkhoff 1927、Kolmogorov 1954、Arnold 1963、Moser 1962 把"不可积系统的行为" 系统研究——发现混沌、奇异吸引子、KAM tori——这些下一章主题。

"大多数 Hamilton 系统不可积" 这条直觉的几何画面——相空间里 \(n\) 个对易守恒量定义一个n 维拉格朗日子流形（symplectic 几何术语）。在 \(2n\) 维相空间里这是测度零 子集。典型 系统的轨道不躺在 torus 上，而是在更大维数的子集上"填满"——这就是遍历性（ergodic）。

请你停下来体会"积分系统是稀有" 这件事的份量。我们物理课本上能解的几乎所有"经典力学题"——谐振子、Kepler、对称陀螺、Toda 链、KdV 方程——都是积分系统。这给你一种错觉，"力学问题原则上能解析解"。错。能解的是很少数系统，刚好因为它们具有特殊对称性 / 守恒量丰富。大部分 真实物理系统（多体、湍流、行星动力学的精确版本）不积分，必须 数值或微扰处理。

15.6 已知积分系统的类目

把"少数能解的" 系统列一遍，让你看它们的覆盖面：

有限自由度积分系统:

  • 谐振子（任意维数）——\(H = \sum \omega_i I_i\)，trivially 积分。
  • Kepler（中心力 \(V = -k/r\)）——三作用 + 退化。
  • 各向同性谐振子 + Coulomb（同时 SO(4) 对称）——这两个例外的对称性来自动量空间几何，第 3 章中心力问题里讨论过。
  • 对称陀螺（Lagrange 陀螺、Euler 陀螺、Kovalevskaya 陀螺）——19 世纪三个被发现的可积刚体，第四个不存在（Husson 1906 证明）。
  • Toda 链（\(N\) 个粒子在指数相互作用势下）——1967 年 Toda 发现可积。
  • Calogero-Moser 系统（\(N\) 粒子 \(1/(q_i - q_j)^2\) 相互作用）——量子和经典都可积。
  • 磁单极散射——SU(2) 对称给可积。

无限自由度积分 PDE（也叫"孤立子方程"）:

  • KdV 方程 \(u_t + 6 u u_x + u_{xxx} = 0\)——浅水波。1967 年通过逆散射变换证可积，无穷多 守恒量。
  • 非线性 Schrödinger 方程 \(i \psi_t + \psi_{xx} + |\psi|^2 \psi = 0\)——光纤通信中 soliton。
  • Sine-Gordon \(\phi_{tt} - \phi_{xx} + \sin \phi = 0\)——Josephson 结、相对论场论玩具。
  • Camassa-Holm——浅水波改进版。

这些可积 PDE 的发现 1967 年起开启了逆散射变换（inverse scattering transform）整个学科。每个可积 PDE 都对应一种"无限维 Hamilton 系统"，相空间是函数空间，作用-角变量是谱数据。这是 20 世纪应用数学的一大主线。

请你看这个列表的共同特征——它们都有特殊代数结构（Lie 群、量子群、Lax 对、双 Hamilton 结构）。可积性的根本原因总是代数对称性。这条洞察让"找新可积系统" 变成"找带新对称性的代数结构"——这是数学物理的一支显学。

15.7 量子积分性 + Bohr-Sommerfeld 量子化

这一节看作用-角在量子力学的化身。量子力学初期 Bohr 1913、Sommerfeld 1916 给"old quantum theory"——粒子绕周期轨道的作用变量被量子化:

\[ \boxed{\ I_i = n_i \hbar, \quad n_i \in \mathbb Z_{\ge 0}.\ } \]

叫 Bohr-Sommerfeld 量子化条件。来源是"电子绕原子核的轨道周长是 de Broglie 波长的整数倍"——驻波条件。

对谐振子 \(I = E / \omega\)，BS 量子化给 \(E_n = n \hbar \omega\)——精确得到谐振子能级（最低 \(n = 0\) 给 \(E = 0\)；正确量子答案是 \(E_n = (n + 1/2) \hbar \omega\)，差一个零点能 \(\hbar \omega / 2\)，是 BS 的一阶修正）。

对氢原子 Kepler \(H = -m k^2 / (2 (I_r + I_\theta + I_\phi)^2)\)，BS 量子化给 \(E_n = -m k^2 / (2 \hbar^2 n^2)\) where \(n = n_r + n_\theta + n_\phi\)——精确给出 Bohr 能级 \(E_n = -13.6 \text{eV}/n^2\)！

这就是 1913-1925 旧量子论的方法：每个积分系统的作用-角变量量子化为 \(\hbar\) 整数倍，给出能级。这套方法对积分系统精确，但对不可积系统（如氦原子两电子）不知道怎么用——这正是促使 Heisenberg 1925 发明矩阵力学、Schrödinger 1926 发明波动力学的历史动机。

1972 Maslov 完善 BS 公式，加入"Maslov 指标" 修正零点能：

\[ I_i = \left(n_i + \frac{\mu_i}{4}\right) \hbar, \]

\(\mu_i\) 是Maslov 指标（每个 caustic 加 1）。这条修正让谐振子的 \(\mu = 2\) 给出 \(E = (n + 1/2) \hbar \omega\) 正确量子答案。

请你在这里看到经典 → 量子 的桥梁——作用变量是经典物理的"面积"，量子化为 \(\hbar\) 整数倍。\(\hbar\) 是相空间的"最小面积单位"。这条几何诠释让"量子化" 不再是神秘的"从经典到量子的跳跃"——它变成"经典相空间被划成 \(\hbar\) 大小的格子"。

15.8 通往 KAM

这一章我们证明了"积分系统" 的存在——\(n\) 守恒量、torus 叶层化、作用-角变量、平凡 Hamilton 方程。但 15.5 节说积分系统稀有，典型 Hamilton 系统不可积。

1954 Kolmogorov 提出一个介于两极之间的问题：取一个积分系统 \(H_0(I)\)，加一个小扰动 \(\epsilon V(I, \theta)\)：

\[ H = H_0(I) + \epsilon V(I, \theta). \]

新系统不再积分（\(V\) 让 \(\theta\) 在 \(H\) 里出现，破坏 \(I\) 的守恒性）。但整体行为有多远离原积分系统？\(I\) 还近似守恒？torus 还存在吗？

朴素回答："用经典微扰论展开"——但展开发散，因为分母上出现"共振" \(\boldsymbol n \cdot \boldsymbol \omega(I) = 0\) 让分母变零（"small denominators" 问题）。这条困难 Poincaré 1890 年就识别，但没有解决方法。

1954 Kolmogorov、1962 Arnold、1963 Moser 给出 KAM 定理——对"非退化" 积分系统加"小" 解析扰动，大多数 torus（满足 diophantine 条件）仍然存在，只是有点变形。这条定理用Newton 迭代 巧妙绕过 small denominators 问题，是 20 世纪数学物理一座丰碑。

它的物理意义就是回答"太阳系稳定吗" 这条 200 年问题。"太阳系" = "Kepler 主导 + 行星互扰小" = "积分系统加小扰动"，KAM 定理保证"大多数初值的轨道在长时间内仍像 Kepler 椭圆"——一种受限 的稳定性。下一章我们把这条定理铺开来讲。

你这一章带走的东西:

  • 作用-角变量——周期系统的"最简单可能形式"。
  • 积分系统——存在 \(n\) 个对易守恒量、torus 叶层化、作用-角；这是经典力学的美 的极限。
  • Liouville-Arnold 定理——积分系统的精确刻画。
  • 但积分系统稀有——绝大多数 Hamilton 系统不可积。这就给"近积分 系统怎么办" 留下了下一章 KAM 的舞台。
