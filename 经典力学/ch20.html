第 20 章 经典场论与 Lagrangian 密度

请你回想前面 19 章我们一直在讨论什么样的系统。粒子（一个 / 几个 / 许多）, 刚体（连续物体但只有 6 自由度）, 弹性体 + 流体（连续介质, 出现了无穷自由度的影子）。除了最后两章之外, 我们的系统都是有限自由度 的——\(N\) 个粒子总共 \(3N\) 个广义坐标, 一个数。

现在请你把视野再拉一档。看你周围的世界, 真正的"物理对象" 不止粒子。电磁场——空间每点有一个电场矢量 \(\boldsymbol E(\boldsymbol x, t)\) + 一个磁场矢量 \(\boldsymbol B(\boldsymbol x, t)\)。引力场——空间每点有一个时空度规 \(g_{\mu\nu}(\boldsymbol x, t)\)。Higgs 场——空间每点有一个标量 \(\phi(\boldsymbol x, t)\)。这些不是粒子, 是场。

场和粒子的根本差别: 自由度数。\(N\) 个粒子有 \(3N\) 个自由度——大但有限。一个场——空间无穷多点, 每点上有一个或几个分量, 总共"每点 \(\times\) 无穷多点" = 不可数无穷个自由度。这是真正 的无穷自由度系统。

Lagrange / Hamilton 力学能不能推广到这种无穷自由度？答案: 干净地推广。架构是Lagrangian 密度 + 作用量积分。这一章把这套架构搭起来, 看两个标志性例子（Klein-Gordon 标量场 + Maxwell 电磁场）, 谈对称 + Noether 在场论里的样子, 最后指向QFT: 把场量子化, 你得到现代基本物理（粒子物理标准模型, 凝聚态量子场论）。这一章是经典力学课的极限 与 QFT 的入口。

20.1 从粒子到场: \(N \to \infty\) 极限

要把粒子力学搬到场, 最直接的桥——1D 弹簧链。

设 \(N\) 个相同质量 \(m\) 沿 \(x\) 轴等间距排列, 间距 \(a\), 相邻两质量用劲度 \(K\) 弹簧连接（链 ch8 小振动）。第 \(n\) 个质量的位移是 \(q_n(t)\)。Lagrangian:

\[ L = \sum_n \frac{1}{2} m \dot q_n^2 - \sum_n \frac{1}{2} K (q_{n+1} - q_n)^2. \]

EL 方程: \(m \ddot q_n = K(q_{n+1} - 2 q_n + q_{n-1})\)。这是离散波动方程; 解是声子 模式 \(q_n \sim e^{i k n a - i \omega t}\), 色散 \(\omega = 2\sqrt{K/m} |\sin(ka/2)|\)。\(N \to \infty\) + \(a \to 0\) 极限怎么变？

把 \(q_n(t)\) 看成光滑场 \(\phi(x, t)\) 在离散点 \(x = n a\) 的值: \(q_n(t) = \phi(n a, t)\)。差分变导数:

\[ q_{n+1} - q_n \approx a \, \partial_x \phi. \]

求和变积分: \(\sum_n \to \int dx/a\)。Lagrangian 化为

\[ L = \int dx \left[ \frac{m}{2 a} (\partial_t \phi)^2 - \frac{K a}{2} (\partial_x \phi)^2 \right] = \int dx \, \mathcal L, \]

其中

\[ \mathcal L = \frac{1}{2} \mu (\partial_t \phi)^2 - \frac{1}{2} \tau (\partial_x \phi)^2, \quad \mu = m/a, \tau = K a. \]

这就叫Lagrangian 密度——单位长度的 Lagrangian。\(\mu\) 是单位长度质量, \(\tau\) 是单位长度的"张力"（弹簧链替代量）。运动方程: 1D 波动方程

\[ \mu \partial_t^2 \phi - \tau \partial_x^2 \phi = 0, \quad c = \sqrt{\tau/\mu}. \]

请你停下来欣赏这一推广: 离散弹簧链的 Lagrangian 是个求和; 连续场的 Lagrangian 是个积分; 结构完全平行。求和号变积分号, 差分变偏导, 离散变量变场, 离散指标变连续坐标。这是 19 世纪从离散介质（弹性体声波）到连续介质（声学）极限的标准操作。20 世纪 Maxwell + Einstein 把它推到真正的场（电磁场, 引力场), 这些场的"底层离散" 不知道是什么, 但 Lagrangian 密度形式同样适用。

20.2 Lagrangian 密度 + 作用量

一般场论的设置:

  • 时空: 4D Minkowski 空间, 坐标 \(x^\mu = (t, \boldsymbol x)\), \(\mu = 0, 1, 2, 3\)。
  • 场: 一个或多个函数 \(\phi^a(x^\mu)\), \(a\) 是场的指标（标量场 \(a\) 单一; 矢量场 \(a = 0, 1, 2, 3\); 张量场更高）。
  • Lagrangian 密度 \(\mathcal L\): 一个函数, 依赖场及其一阶时空导数

\[ \mathcal L = \mathcal L(\phi^a, \partial_\mu \phi^a, x^\mu). \]

注意只到一阶导——因为我们要 EL 是二阶 PDE（高阶动力学有奥斯特罗格拉茨基不稳定性）。

  • 作用量:

\[ \boxed{\ S[\phi] = \int d^4 x \, \mathcal L(\phi, \partial \phi).\ } \]

积分是4D 时空积分——区别于粒子 \(S = \int dt \, L\) 是1D 时间积分。

  • Hamilton 原理: 真实场满足 \(\delta S = 0\), 其中变分对场 \(\phi\) 求, 在时空区域边界上固定 \(\phi\)。

变分演算给场版 EL 方程。具体: 变分 \(\phi \to \phi + \delta \phi\)（\(\delta \phi\) 在边界上为零）, \(\delta(\partial_\mu \phi) = \partial_\mu(\delta \phi)\)。\(\delta \mathcal L = (\partial \mathcal L/\partial \phi) \delta \phi + (\partial \mathcal L/\partial(\partial_\mu \phi)) \partial_\mu(\delta \phi)\)。第二项分部积分（边界为零）:

\[ \delta S = \int d^4 x \left[\frac{\partial \mathcal L}{\partial \phi} - \partial_\mu \frac{\partial \mathcal L}{\partial(\partial_\mu \phi)}\right] \delta \phi. \]

对任意 \(\delta \phi\) 都为零, 给

\[ \boxed{\ \partial_\mu \frac{\partial \mathcal L}{\partial(\partial_\mu \phi)} - \frac{\partial \mathcal L}{\partial \phi} = 0 \quad (\text{场版 EL}).\ } \]

这是粒子版 \(\frac{d}{dt} (\partial L/\partial \dot q) - \partial L/\partial q = 0\) 的直接 推广: 把 \(d/dt\) 换成 \(\partial_\mu\)（4 个时空导数都参加）。多个场时每个 \(\phi^a\) 各一条 EL。

请你停下来确认: 从粒子到场的整个 Lagrange 架构平行得让人吃惊。Hamilton 原理换坐标系不变, EL 方程形式不变, 只是积分维数 1D → 4D。这种"拓朴学般稳健" 是 Lagrange 力学的最大优点之一——它推广到几乎任何系统。

20.3 例 1: Klein-Gordon 标量场

最简单非平凡场: 实标量场 \(\phi(x^\mu)\)（一个分量, Lorentz 标量）。最简单的相对论性 Lagrangian 密度:

\[ \boxed{\ \mathcal L_{\mathrm{KG}} = \frac{1}{2} \partial_\mu \phi \, \partial^\mu \phi - \frac{1}{2} m^2 \phi^2.\ } \]

(用 metric \(\eta_{\mu\nu} = \mathrm{diag}(+, -, -, -)\); \(\partial_\mu \phi \partial^\mu \phi = (\partial_t \phi)^2 - (\nabla \phi)^2\)。) 这个 Lagrangian 长得像"动能 - 势能": \(\frac{1}{2}(\partial_t \phi)^2\) 是动能密度, \(\frac{1}{2}(\nabla \phi)^2 + \frac{1}{2} m^2 \phi^2\) 是势能密度（弹性 + 自势）。

EL 方程:

\[ \partial_\mu \frac{\partial \mathcal L}{\partial(\partial_\mu \phi)} - \frac{\partial \mathcal L}{\partial \phi} = \partial_\mu \partial^\mu \phi + m^2 \phi = 0. \]

记 \(\Box = \partial_\mu \partial^\mu = \partial_t^2 - \nabla^2\)（达朗贝尔算符）, 得

\[ \boxed{\ (\Box + m^2) \phi = 0 \quad (\text{Klein-Gordon 方程}).\ } \]

这是相对论性 标量波方程。\(m = 0\) 时退化为波动方程 \(\Box \phi = 0\)（光速传播的标量波）。\(m \ne 0\) 时, 平面波解 \(\phi = e^{-i E t + i \boldsymbol p \cdot \boldsymbol x}\) 给色散关系

\[ E^2 = p^2 + m^2 \quad (\text{相对论能量-动量关系}). \]

这正是质量 \(m\) 的相对论粒子! Klein-Gordon 1926 年最初就是 想用它做相对论 Schrödinger 方程的——但发现它有"负能量" + "负概率" 问题, 不适合做单粒子量子力学。直到 Dirac 1928（电子的旋量场）+ Pauli-Weisskopf 1934（场量子化解释 KG）, 大家才理解: KG 是场 方程, 不是粒子方程; 它的"负能量" 解对应反粒子, 它的量子化给标量介子（如 \(\pi\) 介子）。这是 QFT 的开端。

从经典力学角度, Klein-Gordon 是最简单 的相对论场。它的 Lagrangian 是所有场论 Lagrangian 的"HelloWorld"——之后所有更复杂场（Dirac 旋量, Maxwell, Yang-Mills, ...）的 Lagrangian 都借用这种"动能 + 质量项" 结构。

20.4 例 2: Maxwell 电磁场

第二个标志性例子: 电磁场。这次场是个4-向量 \(A^\mu(x) = (\phi, \boldsymbol A)\)（电势 + 矢势）。电场 + 磁场用 \(A^\mu\) 写:

\[ \boldsymbol E = -\nabla \phi - \partial_t \boldsymbol A, \quad \boldsymbol B = \nabla \times \boldsymbol A. \]

更优美的形式: 引入场强张量

\[ F_{\mu\nu} = \partial_\mu A_\nu - \partial_\nu A_\mu. \]

这是个反对称 4×4 张量, 6 个独立分量, 对应 \(\boldsymbol E\) 的 3 个 + \(\boldsymbol B\) 的 3 个。具体 \(F_{0i} = E_i\), \(F_{ij} = -\epsilon_{ijk} B_k\)。

Maxwell Lagrangian 密度:

\[ \boxed{\ \mathcal L_{\mathrm{Maxwell}} = -\frac{1}{4} F_{\mu\nu} F^{\mu\nu} - J^\mu A_\mu,\ } \]

其中 \(J^\mu = (\rho, \boldsymbol j)\) 是 4-电流密度（外源）。展开 \(F_{\mu\nu} F^{\mu\nu} = -2 (\boldsymbol E^2 - \boldsymbol B^2)\), 所以 \(-\frac{1}{4} F_{\mu\nu} F^{\mu\nu} = \frac{1}{2}(\boldsymbol E^2 - \boldsymbol B^2)\)。这就是电场 - 磁场 的能量差形式 Lagrangian。

EL 方程对 \(A_\mu\):

\[ \partial_\mu \frac{\partial \mathcal L}{\partial(\partial_\mu A_\nu)} - \frac{\partial \mathcal L}{\partial A_\nu} = 0. \]

算偏导（\(\partial \mathcal L/\partial(\partial_\mu A_\nu) = -F^{\mu\nu}\), \(\partial \mathcal L/\partial A_\nu = -J^\nu\)）, 得

\[ \boxed{\ \partial_\mu F^{\mu\nu} = J^\nu \quad (\text{Maxwell 方程, 含源}).\ } \]

展开成 \(\boldsymbol E, \boldsymbol B\) 给

  • \(\nabla \cdot \boldsymbol E = \rho\)（Gauss 电）
  • \(\nabla \times \boldsymbol B - \partial_t \boldsymbol E = \boldsymbol j\)（Ampère-Maxwell）

另两条 Maxwell（Faraday + Gauss 磁）自动 满足——它们是 \(F_{\mu\nu} = \partial_\mu A_\nu - \partial_\nu A_\mu\) 这条恒等式的Bianchi 恒等式 \(\partial_{[\rho} F_{\mu\nu]} = 0\)。所以从 \(A^\mu\) + Maxwell Lagrangian 出发, 四条 Maxwell 方程都得到了——两条来自 EL, 两条来自定义。

请你停下来再次惊叹: 一行 Lagrangian \(-\frac{1}{4} F^2 - J A\) 包含了整个经典电动力学。这是 Lagrangian 力学最美的胜利之一——19 世纪 Maxwell 用 20 条方程描述电磁, 20 世纪 Lagrange / Lorentz / Einstein 把它压成一行。

这个 Lagrangian 还有个对称性: 规范对称（gauge symmetry）。变换 \(A_\mu \to A_\mu + \partial_\mu \chi\)（\(\chi\) 任意函数）不改变 \(F_{\mu\nu}\), 也不改变 \(\mathcal L\) 中的 \(F^2\) 项。这条对称是电磁场的核心特征; 它在 QFT 里推广到规范理论 框架, 给标准模型的 SU(3) × SU(2) × U(1)。

20.5 Noether 在场论: 能量-动量张量

ch7 我们看了粒子版 Noether: 每个连续对称对应一条守恒量。场论里 Noether 升级——每个对称对应一条守恒流:

\[ \partial_\mu j^\mu = 0, \quad \text{即 } \partial_t \rho + \nabla \cdot \boldsymbol j = 0. \]

这是连续性方程形式（ch19 里见过）, 它的总积分 \(Q = \int d^3 x \, j^0\) 是守恒荷。

具体: 时空平移对称（Lagrangian 不显含 \(x^\mu\)）⟹ 能量-动量张量 \(T^{\mu\nu}\) 守恒, \(\partial_\mu T^{\mu\nu} = 0\)。\(T^{\mu\nu}\) 的具体形式（Noether 给）

\[ T^{\mu\nu} = \frac{\partial \mathcal L}{\partial(\partial_\mu \phi^a)} \partial^\nu \phi^a - \eta^{\mu\nu} \mathcal L. \]

分量解释:

  • \(T^{00}\)——能量密度。
  • \(T^{0i}\)——能量流密度（即 \(c\) 倍动量密度）。
  • \(T^{ij}\)——应力张量（动量 \(i\) 在 \(j\) 方向流速）。

守恒律 \(\partial_\mu T^{\mu 0} = 0\) 是能量守恒, \(\partial_\mu T^{\mu i} = 0\) 是动量守恒。两条粒子时是独立 的守恒律（ch2）, 场论里被打包成一个张量方程。

对 KG 标量场, 算出

\[ T^{\mu\nu} = \partial^\mu \phi \partial^\nu \phi - \eta^{\mu\nu} \mathcal L, \]

能量密度 \(T^{00} = \frac{1}{2}(\partial_t \phi)^2 + \frac{1}{2}|\nabla \phi|^2 + \frac{1}{2} m^2 \phi^2\)——正定, 像粒子动能 + 势能。这是 KG 的良性: 它的能量是有界下方的（不像 Dirac 的负能问题）。

对 Maxwell, \(T^{\mu\nu}\) 算出

\[ T^{\mu\nu}_{\mathrm{EM}} = F^{\mu\rho} F^\nu{}_\rho - \frac{1}{4} \eta^{\mu\nu} F^{\rho\sigma} F_{\rho\sigma}. \]

能量密度 \(T^{00}_{\mathrm{EM}} = \frac{1}{2}(\boldsymbol E^2 + \boldsymbol B^2)\)——这正是大学电磁课本里的"电磁能量密度"。这条结果以前是凭直觉 / 实验得到的; Noether + 场版 Lagrangian 让它从原理推出。

20.6 局部规范对称: 场论里最深的洞见

电磁场的对称变换 \(A_\mu \to A_\mu + \partial_\mu \chi(x)\) 中的 \(\chi\) 是时空函数——每点可以独立选不同的 \(\chi\)。这种"局部" 对称（每点独立）和"全局" 对称（所有点同样变换）是不同物种。

请你想清楚: 全局对称（如全空间平移 \(\boldsymbol x \to \boldsymbol x + \boldsymbol c\)）是有限维 的对称变换。局部对称是无穷维 的——每点选一个变换参数, 总共 \(\infty\) 个自由度。这种自由度的"无穷" 是规范理论的核心。

规范对称的意义: 它是物理冗余, 不是真物理变化。\(A_\mu\) 不是直接观测量; 真观测量是 \(F_{\mu\nu}\), 它在规范变换下不变。\(A_\mu\) 像"方位坐标"——你可以选不同的, 但物理同样。这种"多个数学描述对应一个物理" 的冗余在 GR（坐标自由度）+ Yang-Mills（非阿贝尔规范）+ 弦论里反复出现, 成为现代物理的组织原则。

1954 年 Yang + Mills 把 U(1) 局部对称推广到 SU(N) 局部对称——给非阿贝尔规范理论。20 世纪后半叶: SU(2) × U(1) 给电弱（Glashow-Weinberg-Salam, 1961-67）; SU(3) 给强相互作用（QCD, 1973）。标准模型 = SU(3) × SU(2) × U(1) 局部规范。这条线索完全长在场论 Lagrangian + 局部对称的土壤上, 是 20 世纪粒子物理的全部架构。

20.7 Hamilton 形式 + 量子化预备

Lagrangian 场论可以转 Hamilton 形式。Legendre 变换

  • 共轭场 \(\pi^a(x) = \partial \mathcal L/\partial(\partial_t \phi^a)\)
  • Hamiltonian 密度 \(\mathcal H = \pi^a \partial_t \phi^a - \mathcal L\)
  • Hamilton 方程 \(\partial_t \phi^a = \delta H/\delta \pi^a\), \(\partial_t \pi^a = -\delta H/\delta \phi^a\)（变分导数）

对 Klein-Gordon: \(\pi = \partial_t \phi\), \(\mathcal H = \frac{1}{2} \pi^2 + \frac{1}{2}(\nabla \phi)^2 + \frac{1}{2} m^2 \phi^2\)。

Poisson 括号在场论:

\[ \{\phi(\boldsymbol x, t), \pi(\boldsymbol y, t)\} = \delta^3(\boldsymbol x - \boldsymbol y). \]

注意是等时 括号, 给的是 \(\delta\) 函数（不是 Kronecker \(\delta\)）。这是连续指标版本。

量子化: 把 Poisson 括号升成对易子:

\[ [\hat \phi(\boldsymbol x, t), \hat \pi(\boldsymbol y, t)] = i \hbar \delta^3(\boldsymbol x - \boldsymbol y). \]

这就是正则量子化。\(\hat \phi(\boldsymbol x)\) 现在是算符（每点一个, 对易关系给定）, 作用在 Hilbert 空间的态上。Fock 空间分解: \(\hat \phi\) 用产生 / 湮灭算符 \(a_k^\dagger, a_k\) 展开, 真空 \(|0\rangle\) 满足 \(a_k |0\rangle = 0\), \(a_k^\dagger |0\rangle\) 是单粒子态。这就是 QFT。

请你停下来意识到: 经典场论 + 正则量子化 = 量子场论（QFT）。从 Lagrangian 密度出发, 通过 Hamilton 形式, 通过对易关系, 直接到量子。这条路径 Heisenberg + Pauli + Dirac 1929 年就走通了——它是经典力学到量子力学的第二条桥（第一条是 ch21 即将讨论的 Schrödinger / 粒子量子化）。两条桥是同一山顶（量子力学）的两条上山路。

20.8 通向 QFT: 场量子化是物理学的基础

场量子化是不 是哲学游戏——它是物理学描述基本粒子的唯一方式。具体:

  • 粒子物理标准模型：六种夸克, 六种轻子, 四种规范玻色子（\(\gamma, W^\pm, Z, \) 胶子\(\times 8\)）, 一个 Higgs——所有这些是场, 量子激发就是粒子。\(\pi\) 介子是 KG 标量场（pseudoscalar 实际上）的量子激发; 光子是 Maxwell 场的量子激发; 电子是 Dirac 旋量场的量子激发。
  • 凝聚态: 声子（晶格振动场的量子）、磁子（自旋波场的量子）、超导库珀对（电子场的"双激发"）、超流相位场（Goldstone 模）等等——都是场量子化的产物。
  • 宇宙学早期: 暴涨场（一个标量场）的量子涨落给宇宙微波背景的"原始扰动"——这套用 KG-类场 Lagrangian 写下, 量子化, 算扰动谱, 与观测对照。Planck 卫星 2013-2018 测出这条谱, 完美符合暴涨预言。

这一章我们在经典 场论结束。量子化是量子力学（特别 ch24 量子场论入门）+ 量子场论专门课程的内容。但请你记住: 经典场论的 Lagrangian / Hamilton 框架是 QFT 的"骨架"; 量子化只是把骨架"充电"——把场变成算符。骨架本身——\(\mathcal L, S = \int \mathcal L d^4 x, \delta S = 0\)——是经典物理给的礼物。

这条线把"苹果落地" 和"夸克 + 胶子" 用同一架构——Lagrangian / Hamilton——连起来。这是 18 世纪 Lagrange / Hamilton 给我们的礼物: 一种能从单粒子推到量子场的统一语言。

下一章 ch21 我们走另一条桥: 从经典走向量子。Hamilton-Jacobi 方程的"波振面" 解释——光学短波极限 → 几何光学, 量子力学短波极限 → 经典力学。这是量子诞生时 Schrödinger / Dirac / Feynman 的视角, 也是本书结尾的高潮。

本书剩下: ch22 相对论力学 + ch23 航天力学应用 + ch24 几何力学（symplectic 流形）。这是经典力学的"专题" 部分, 不是必修序列。

请你在合本之前停下来想清楚一件事。经典力学 是 17-18 世纪的物理, 已经 350 年; 但它的语言（Lagrange, Hamilton, Lagrangian 密度, 作用量, 对称, 守恒）仍然 是 21 世纪基本物理的工作语言。Newton 公式 \(F = ma\) 在大学物理课结束就基本不再写了, 但 Lagrangian / 作用量出现在每一篇粒子物理 / 凝聚态 / 弦论 paper 的开头。这不是历史巧合, 是因为变分原理 是物理本身的语言——大自然似乎"真的" 在最小化什么东西, 而我们把它写成 \(S = \int \mathcal L \, dt\) 或 \(S = \int \mathcal L \, d^4 x\)。这条信仰是经典力学留给现代物理的最大遗产。
