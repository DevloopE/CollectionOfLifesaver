第 24 章 几何力学（symplectic + moment map）

本书最后一章。我们走了 23 章, 从 1696 Bernoulli 最速降线开始, 经过 Newton + Lagrange + Hamilton + Poisson 四种力学语言, 经过 KAM + 混沌 + 弹性 + 流体 + 经典场论 + 量子前夜 + 相对论 + 航天, 走到最后一站——几何力学。

这是 1970-1980 年代由 Vladimir Arnold（莫斯科）+ Jerrold Marsden（Berkeley/Caltech）+ Alan Weinstein（Berkeley）+ Tudor Ratiu（EPFL Lausanne）等人系统化的现代 经典力学。它的核心命题是: 把经典力学完全几何化——相空间是辛流形（symplectic manifold）, Hamilton 流是辛同胚, 守恒量是moment map, 对称约化是商映射。所有 \((q, p)\) 坐标本质消失, 留下纯几何对象。

为什么这种"纯几何" 的视角值得最后一章？三个理由:

  • 普适: 几何力学的语言适用任何偶数 维流形 + 闭非退化 2-形式, 不仅仅是 \(T^* Q\)。这覆盖所有 经典力学系统——刚体（旋转群上）、Toda 链、Calogero-Moser 系统、流体（无穷维 Lie 群）、规范理论。
  • 对称约化: Marsden-Weinstein 定理 (1974) 给"有对称的力学系统怎么"商掉"对称性" 一个干净几何答案。这是 ch7 Noether 定理的几何完成。
  • 通向现代物理: 几何力学的语言渗透量子化（几何量子化, 链 ch21.9）+ 数学物理（可积系统, 链 ch15）+ 表示论（Kirillov 轨道）+ 量子计算的几何相位。

本章是导论——给概念 + 主要定理, 不深证明。深入 see Marsden-Ratiu《Introduction to Mechanics and Symmetry》（标准教材）+ Arnold《Mathematical Methods of Classical Mechanics》（更老但更优雅）。

24.1 复习相空间 + 辛形式

从第 11 章到第 15 章, 我们用相空间 \((q, p)\) 描述 Hamilton 力学。Poisson 括号

\[ \{f, g\} = \sum_i \left( \frac{\partial f}{\partial q^i} \frac{\partial g}{\partial p_i} - \frac{\partial f}{\partial p_i} \frac{\partial g}{\partial q^i} \right) \]

给观测量代数。Hamilton 流 \(\dot q^i = \partial H/\partial p_i, \dot p_i = -\partial H/\partial q^i\) 由辛 2-形式

\[ \omega = \sum_i dp_i \wedge dq^i \]

统一描述 (链第 13 章 13.7 节正则变换 = 保 \(\omega\) 的变换)。

这种"用 \(\omega\) 来组织一切" 的视角让坐标变得不重要。任何保 \(\omega\) 的坐标变换都给同样的物理。真正的对象是 \(\omega\) 与流形, 不是坐标。这就是几何力学的哲学。

本节正式定义辛流形——抽象到不依赖坐标 的层面。

24.2 辛流形定义

设 \(M\) 是 \(2n\) 维光滑流形。辛 2-形式 \(\omega\) 是 \(M\) 上的2-阶微分形式（链 V13 微分几何 ch12-15）, 满足:

  1. 闭: \(d \omega = 0\)。
  2. 非退化: 对每点 \(x \in M\) 与每非零切向量 \(v \in T_x M\), 存在 \(w \in T_x M\) 使 \(\omega(v, w) \neq 0\)。等价说, 矩阵 \(\omega_{ij}\)（\(\omega\) 在某基下分量）非奇异——这要求 \(\dim M\) 为偶数。

满足两条的 \((M, \omega)\) 叫辛流形。

例 1: \(M = \mathbb R^{2n}\), \(\omega = \sum_i dp_i \wedge dq^i\)。这是"标准辛空间", 经典 \((q, p)\) 相空间。

例 2: 球面 \(S^2\) (二维) 带面积形式 \(\omega = \sin \theta \, d\theta \wedge d\varphi\) (球面坐标)。\(d\omega = 0\), 非退化。\((S^2, \omega)\) 是辛流形——这给一个不是"位置 + 动量" 的辛例子。

例 3: 复射影空间 \(\mathbb C P^n\) 带 Fubini-Study 形式。\(2n\) 实维。这是量子态空间的辛结构（量子力学的纯态构成 \(\mathbb C P^{n-1}\)）。

关键定理（Darboux 定理, 1882）: 所有辛流形局部上等价于标准 \(\mathbb R^{2n}\)——存在局部坐标 \((q^1, \ldots, q^n, p_1, \ldots, p_n)\) 使 \(\omega = \sum dp_i \wedge dq^i\)。

这说: 辛几何没有"局部不变量"——所有辛流形局部都长一样 (与 Riemann 几何不 同, 后者有曲率作局部不变量)。整体上辛流形可以非常不同 (球面 vs 平面 vs 环面), 但局部完全相同。

24.3 例: \(T^* Q\) 自然辛

给任意流形 \(Q\)（"配置空间", 比如 N 粒子系统的 \(\mathbb R^{3N}\), 或单摆的圆 \(S^1\), 或刚体的 \(SO(3)\)）, 它的余切丛 \(T^* Q\) 自然带辛结构。

构造: \(T^* Q\) 上点是 \((q, p)\), 其中 \(q \in Q\), \(p \in T_q^* Q\)（\(q\) 处余切向量）。定义"同调 1-形式" \(\theta\) 在 \((q, p)\) 处:

\[ \theta_{(q, p)}(v) = p(\pi_*(v)), \]

其中 \(\pi: T^* Q \to Q\) 是投影, \(\pi_*\) 是它在切丛的微分。然后

\[ \boxed{\ \omega = -d \theta.\ } \]

证明 \(\omega\) 是辛: \(d \omega = -d d \theta = 0\)（外微分平方零, 链 V13 ch14）; 非退化由 \(Q\) 上局部坐标 \(q^i\) + 自然对偶 \(dq^i\) 给的 \(p = p_i \, dq^i\) 验证, 得 \(\omega = dp_i \wedge dq^i\) 标准形式。

所以任意配置空间 \(Q\) 的余切丛 \(T^* Q\) 自然带辛结构——这就是"位置-动量相空间" 在几何上的本质。

例: \(Q = \mathbb R^n\)（自由粒子）, \(T^* Q = \mathbb R^{2n}\), 标准辛。

例: \(Q = S^1\)（单摆配置）, \(T^* S^1 = S^1 \times \mathbb R\)（圆柱）。辛形式 \(\omega = dp \wedge d\theta\)。Hamilton 流的整体几何非常具体——能量小给小圆轨道, 能量大给翻越的轨道, 临界能量分隔两区。这就是 ch15 作用-角变量在单摆上的几何。

例: \(Q = SO(3)\)（刚体配置 = 旋转矩阵）, \(T^* SO(3) = SO(3) \times \mathfrak{so}(3)^*\)。这是刚体力学的相空间。

但不是所有辛流形都是 \(T^* Q\) 形式——比如球面 \(S^2\) 上的面积形式不能写成 \(T^* Q\)。一般辛流形比余切丛多。这种"辛但不是余切丛" 的例子在几何力学中重要——它们对应"没有自然位置 / 动量分离" 的力学系统。

24.4 Hamilton 向量场 \(X_H\)

给辛流形 \((M, \omega)\) + 光滑函数 \(H: M \to \mathbb R\)（"Hamiltonian"）, 怎么得 Hamilton 流？答案: Hamilton 向量场 \(X_H\) 由

\[ \boxed{\ \iota_{X_H} \omega = dH\ } \]

唯一定（用 \(\omega\) 非退化）。\(\iota\) 是内积（contraction）——\(\iota_X \omega\) 是 1-形式 \(v \mapsto \omega(X, v)\)。

方程 \(\iota_{X_H} \omega = dH\) 说: \(X_H\) 是 \(\omega\) 把 \(dH\) "翻译" 成的向量场。它给"沿 \(H\) 的等位面切向量"——确实, \(X_H \cdot H = \omega(X_H, X_H) = 0\)（\(\omega\) 反对称）, 即 \(H\) 沿 \(X_H\) 不变, 能量守恒。

在标准坐标 \((q, p)\), \(\omega = dp \wedge dq\), 解 \(\iota_X \omega = dH\) 直接给

\[ X_H = \frac{\partial H}{\partial p} \frac{\partial}{\partial q} - \frac{\partial H}{\partial q} \frac{\partial}{\partial p}. \]

这正是 Hamilton 流——\(\dot q = \partial H/\partial p, \dot p = -\partial H/\partial q\)。所以"抽象 \(\iota_X \omega = dH\)" 在标准坐标下退化成熟悉的 Hamilton 方程。但抽象形式在任意辛流形 上工作, 不需要全局 \((q, p)\) 坐标。

Poisson 括号也几何重写:

\[ \{f, g\} = \omega(X_f, X_g) = X_g \cdot f = -X_f \cdot g. \]

(各种等价表达; 用 \(\iota_{X_g} \omega = dg\) 推导。)

所以Poisson 括号 = 用 \(\omega\) 把 \(df\) 与 \(dg\) 配成的标量。整套 Poisson 代数就是辛形式的内禀产物。

请你停下感受这件事: 在几何力学语言里, "Hamilton 力学" 是 "选个辛流形 \((M, \omega)\) + 选个 \(H", 然后 \(X_H\) 由 \(\omega\) 自动构造。\((q, p)\) 不是对象——它们只是局部坐标. 真对象是 \((M, \omega, H)\)。

24.5 辛同胚 = 保 \(\omega\) 微分同胚

辛流形之间"力学等价" 的映射是辛同胚: 微分同胚 \(\phi: (M_1, \omega_1) \to (M_2, \omega_2)\) 满足

\[ \phi^* \omega_2 = \omega_1. \]

即 \(\phi\) "把 \(\omega_2\) 拉回 \(M_1\) 等于 \(\omega_1\)"。

关键性质:

  • 辛同胚保 Hamilton 流——\(\phi(\)\(M_1\) 中 Hamilton 流的轨道\() = M_2\) 中的对应 Hamilton 流轨道。
  • 辛同胚保 Poisson 括号——\(\phi^* \{f, g\}_2 = \{\phi^* f, \phi^* g\}_1\)。
  • 辛同胚保体积——\(\omega^n / n!\) 是 \(2n\) 维"Liouville 体积", 辛同胚保它（链第 11 章 11.7 Liouville 定理）。

对自映射 \(\phi: M \to M\) (即 \(M_1 = M_2 = M\)), 辛同胚就是正则变换（链第 13 章）的现代版。所以本书 ch13 的"正则变换" = "辛同胚", 只是名字不同。

例: Hamilton 流本身是辛同胚族。给 \(H\), 时间 \(t\) 的演化 \(\phi_t: M \to M\) 是辛同胚（每个 \(t\)）, 且 \(\phi_0 = \text{id}, \phi_{t+s} = \phi_t \circ \phi_s\)（一参数群）。这是"Hamilton 流" 的几何描述: 每个 Hamiltonian 给一族辛同胚。

例: 量子力学里的幺正算子 \(\hat U(t) = e^{-i \hat H t/\hbar}\)。它保 Hilbert 内积, 是经典辛同胚的"量子升级"。这也是 ch21 的"经典 → 量子桥" 的几何提示——辛同胚 \(\to\) 幺正算子。

24.6 Lie 群作用 + moment map

现在到几何力学最深的概念——moment map。

给辛流形 \((M, \omega)\) + Lie 群 \(G\) 在 \(M\) 上的辛作用 (即 \(G\) 的每个元素 \(g \in G\) 给辛同胚 \(\phi_g: M \to M\), 且 \(\phi_{gh} = \phi_g \circ \phi_h\))。

这种"对称作用" 给一族无穷小变换。每个 \(\xi \in \mathfrak g\)（\(G\) 的 Lie 代数, 即"无穷小生成元"）给 \(M\) 上的"基本向量场" \(\xi_M\)——它是 \(\phi_{\exp(t \xi)}\) 在 \(t = 0\) 的导数。

Hamilton 化: 对每个 \(\xi \in \mathfrak g\), 是否存在函数 \(H_\xi: M \to \mathbb R\) 使 \(\xi_M = X_{H_\xi}\)（即 \(\xi_M\) 的"Hamilton 函数"）？答案——若 \(H^1(M, \mathbb R) = 0\)（\(M\) 一维上同调消失, 大多数物理空间满足), 则是。

把所有 \(H_\xi\) 集成moment map:

\[ \boxed{\ \mu: M \to \mathfrak g^*, \quad \langle \mu(x), \xi \rangle = H_\xi(x).\ } \]

(\(\mathfrak g^*\) 是 \(\mathfrak g\) 的对偶空间。)

\(\mu\) 把相空间点 映到对偶 Lie 代数, 每个 \(\xi \in \mathfrak g\) 配上一个 Hamilton 函数。直观: \(\mu(x)\) 是"对称群作用 \(G\) 在 \(x\) 处的"守恒量""。具体例子:

例 1: 平移对称。\(M = T^* \mathbb R^3\), \(G = \mathbb R^3\) 平移群作用 \((q, p) \mapsto (q + a, p)\)。\(\mathfrak g = \mathbb R^3\), 基 \(e_i\) 对应 \(i\) 方向平移。基本向量场 \(\xi_M = \partial/\partial q\) (沿 \(q\) 方向)。Hamilton 函数 \(H_{e_i} = p_i\) (因为 \(\iota_{\partial/\partial q^i} \omega = -dp_i\), 所以 \(X_{p_i} = -\partial/\partial q^i\)... 符号 cross-check 略)。所以 \(\mu(q, p) = p\)——moment map = 动量!

例 2: 旋转对称。\(M = T^* \mathbb R^3\), \(G = SO(3)\) 旋转群。\(\mathfrak{so}(3) = \mathbb R^3\) (反对称矩阵 \(\leftrightarrow\) 向量, 链 ch9 刚体)。Moment map \(\mu(q, p) = q \times p = \boldsymbol L\)——角动量!

例 3: 时间平移。\(G = \mathbb R\), 作用是 Hamilton 流。Moment map \(\mu = H\)——能量。

请你看到这件事的普适: 所有 Noether 守恒量都是 moment map。Noether 定理（链第 7 章）的几何完成: 守恒量 \(=\) 对称作用的 moment map 分量。这给"对称 ↔ 守恒" 一个结构性 表达——不再是"每例分别证", 而是"由 moment map 普适给"。

关键性质 (\(G\) 作用 + \(\mu\)):

  • 若 \(H\) G 不变（即对所有 \(g \in G\), \(H \circ \phi_g = H\)）, 则 \(\mu\) 沿 Hamilton 流守恒——\(d\mu/dt = 0\)。这是 Noether 的几何陈述。
  • \(\mu\) 是等变 的: \(\mu(\phi_g(x)) = \text{Ad}^*_g \mu(x)\), 其中 \(\text{Ad}^*\) 是 \(G\) 在 \(\mathfrak g^*\) 上的余伴随作用。这条等变性使得 \(\mu\) 行为良好。

24.7 Marsden-Weinstein 辛约化

有了 moment map, 怎么"商掉" 对称性？1974 年 Jerry Marsden + Alan Weinstein 给出干净答案——辛约化（symplectic reduction）。

setup: \((M, \omega)\) 辛流形, \(G\) Lie 群辛作用, \(\mu: M \to \mathfrak g^*\) moment map。固定 moment 值 \(\eta \in \mathfrak g^*\)（"固定守恒量"）。考虑 level set \(\mu^{-1}(\eta) \subset M\)——所有 \(x\) 满足 \(\mu(x) = \eta\)。

这个 level set 上 \(G_\eta = \{g \in G : \text{Ad}^*_g \eta = \eta\}\)（\(\eta\) 的稳定子）作用。商空间

\[ \boxed{\ M_\eta = \mu^{-1}(\eta) / G_\eta.\ } \]

是约化相空间。

Marsden-Weinstein 定理: 若 \(G_\eta\) 在 \(\mu^{-1}(\eta)\) 上的作用自由 + 真 (常成立), 则 \(M_\eta\) 是辛流形, 维数 \(\dim M_\eta = \dim M - 2 \dim G\)。

具体: 把对称商掉 减 \(\dim G\) 维（每个对称 1 个自由度）, 又减 \(\dim G\) 维（固定 moment 值用掉守恒量）, 总共减 \(2 \dim G\)。

这是"对称约化" 的几何形式——用对称把相空间维数缩小, 留下的空间仍是辛流形, 仍可做 Hamilton 力学。

例 1: 中心力问题。\(M = T^* \mathbb R^3\)（6 维）, \(G = SO(3)\)（3 维）, \(\mu(q, p) = q \times p = \boldsymbol L\)。固定 \(\boldsymbol L = \boldsymbol L_0\)（角动量），\(G_{\boldsymbol L_0} = SO(2)\)（绕 \(\boldsymbol L_0\) 轴旋转, 1 维）。约化空间 \(M_{\boldsymbol L_0}\) 维数 \(6 - 2 \cdot 3 + 2 \cdot 1 = 2\)（因为 \(G_\eta = SO(2)\) 不是全 \(G\), 标准定理给 \(\dim M - \dim G - \dim G_\eta = 6 - 3 - 1 = 2\)）。2 维相空间——这正是第 3 章把中心力 reduced 到一维径向问题（一维位置 + 一维径向动量）。约化让"3D 矢量问题" 变成"1D 径向问题", Marsden-Weinstein 给几何理由。

例 2: 刚体动力学。\(M = T^* SO(3)\)（6 维）, \(G = SO(3)\)（左作用, 3 维）。约化 \(M_\eta\) 是 2 维。具体: 这 2 维是 \(\mathbb R^3\) 上的 sphere \(|\boldsymbol L| = \text{const}\)（角动量大小不变）。Hamilton 流在 \(S^2\) 上是 Euler 方程的轨迹（链第 9 章）。Euler 方程的几何来源 = 刚体相空间 \(T^* SO(3)\) 的 Marsden-Weinstein 约化。这种"从全相空间 → 约化空间" 视角让 Euler 方程的"椭圆轨道 / 双曲鞍点" 几何有干净来源。

例 3: \(N\) 体引力。\(M\) 是 \(N\) 粒子的 \(6 N\) 维相空间, \(G = E(3)\)（欧氏运动群, 6 维）。约化掉平移对称（3 维）+ 旋转对称（3 维）= 6 维, 给\(6N - 12\) 维约化空间。这是天体力学的"Jacobi 坐标" 后台机制。

请你欣赏 Marsden-Weinstein 的普适: 任何带 Lie 群对称的力学系统都可几何约化。这给"用对称简化问题" 一个统一框架, 不再是"每例独立设计技巧"。

24.8 应用: 刚体, Toda 链, Calogero-Moser

本节给几何力学的3 个经典例子, 让你看到这套语言怎么用。

1. 刚体 (\(SO(3)\) 作用)。前节提过。完整刚体力学的相空间 \(T^* SO(3)\) (6 维), Hamiltonian \(H = \frac{1}{2} I^{-1}_{ij} L^i L^j\) (惯性张量倒 + 角动量平方)。约化到 \(\boldsymbol L\) 不变球面 \(S^2\), Hamilton 流给 Euler 方程

\[ \dot {\boldsymbol L} = \boldsymbol L \times (I^{-1} \boldsymbol L). \]

解析积分（用守恒量 \(|\boldsymbol L|^2\) + 能量 \(H\)）, Jacobi 1839 用椭圆函数给精确解。这是经典力学最优雅的解析可积 例子之一。

2. Toda 链。Morikazu Toda 1967 提出: \(N\) 个粒子在线上, 相邻粒子用指数 势耦合 \(V(q_i - q_{i+1}) = e^{-(q_{i+1} - q_i)}\)。Hamiltonian \(H = \sum p_i^2/2 + \sum e^{-(q_{i+1} - q_i)}\)。

看似无对称, 但 Flaschka 1974 + Manakov 1974 发现: Toda 链有 \(N\) 个独立守恒量, 完全可积！具体: 用 Lax 对 \((L, A)\) 形式 \(\dot L = [A, L]\), \(L\) 的特征多项式系数全是守恒量。

几何力学解释: Toda 链是某个 Lie 代数 \(sl(N, \mathbb R)\) 的余伴随轨道 上的 Hamilton 系统。Marsden-Weinstein 约化 + Lie-Poisson 结构给完全积分性的结构性 来源。这是"几何力学解释可积性" 的范例。

Toda 链应用: 数值离散化 KdV 方程 + 量子可积链 + 随机矩阵理论（Toda 链的连续极限是 KdV; KdV 是 \(N \to \infty\) Toda）。

3. Calogero-Moser 系统。Calogero 1971 + Moser 1975 提: \(N\) 个粒子在线上, 相互作用 \(V(q_i - q_j) = 1/(q_i - q_j)^2\)（反平方）。Hamiltonian \(H = \sum p_i^2/2 + \sum_{i < j} 1/(q_i - q_j)^2\)。

同样有 \(N\) 个独立守恒量, 完全可积。Lax 表示 + 散射理论给精确解。

几何力学解释: Calogero-Moser 是 \(T^* GL(N) \times T^* \mathbb C^N\) 经过 \(GL(N)\) 对称约化的结果。这种"更大空间的辛约化给可积链" 模式叫Hamiltonian 约化构造可积系统, 是 Marsden-Weinstein-Atiyah-Hitchin 1980 年代的重大发现。

Calogero-Moser 应用: 扩展到量子（量子 Calogero-Moser）, 与 random matrix theory 紧密相连; 连续极限给 Benjamin-Ono 方程; 复化版本（"elliptic Calogero-Moser"）与超对称 Yang-Mills 理论有深联系（Donagi-Witten 1995）。

这三个例子让你看到几何力学的丰收——它不是抽象游戏, 而是产生具体新结果 的工具。可积系统的整个 1980-2000 黄金时代靠这套语言驱动。

24.9 收尾段: 1696-2026 力学进化

这是本书最后一节。让我们回顾整个旅程, 看这 330 年力学经过了什么。

1696 Bernoulli 最速降线。一道挑战题——"滑下时间最短的曲线是什么"。Newton 当晚解出, 揭开变分法 历史。这是本书 ch1 的钩子, 也是经典力学的"第一次革命"——从"物体每瞬怎么动" 转向"整条路径如何决定"。

1788 Lagrange 《Mécanique analytique》。把整个力学重写成变分形式。\(L = T - V\), \(\delta S = 0\), EL 方程。这是第一次系统化的"力学 = 变分"——经典力学第二次革命。

1833 Hamilton 正则方程。Legendre 变换把 \(L\) 变 \(H\), 二阶 ODE 变一阶 ODE 系统, 引入相空间。这给经典力学一个对称, 几何 的形式——为后续 Poisson 括号、量子力学、辛几何打底。

1858-1872 Liouville + Sophus Lie。Liouville 完全可积性 (\(N\) 个互对合守恒量给 \(N\) 自由度系统的解析可解); Lie 创立 Lie 群理论 (连续对称 \(\to\) Lie 代数), 1893 出版《Theorie der Transformationsgruppen》。这两套理论给后续几何力学的群论基础。

1887 Poincaré + 三体不可解。证明三体一般不可积; 引入Poincaré 截面 + 稳定 / 不稳定流形 + 同宿轨道。混沌理论的种子。

1905-1916 Einstein 相对论。狭义 + 广义。把 Newton 引力换成时空几何, Lagrangian 公式自然推广。本书 ch22 + V13 ch23-24。

1918 Noether 定理。对称 ↔ 守恒。这条定理后来被一切现代物理 (粒子物理、规范理论、量子场论) 当成圣经。

1925-1928 Heisenberg-Schrödinger-Dirac 量子力学。从经典 Hamilton-Poisson 框架"正则量子化" 出量子力学。本书 ch21。

1948 Feynman 路径积分。经典 Lagrangian 直接通向量子振幅。给量子力学第二种描述形式。

1954-1963 KAM 定理（Kolmogorov + Arnold + Moser）。"几乎可积" 系统的 invariant tori 在小扰动下保留——澄清了"太阳系稳定吗" 的 200 年问题。本书 ch16。

1960-1980 混沌（Lorenz + Smale + Feigenbaum + Mandelbrot）。蝴蝶效应 + 分形 + 周期倍增。打破"确定性 = 可预测" 神话。本书 ch17。

1970-1980 几何力学（Arnold + Marsden + Weinstein + Ratiu）。本章。把经典力学几何化——辛流形 + moment map + 对称约化。

1968-1990 可积系统。KdV + Toda + Calogero-Moser + 反散射变换 + Lax 对。让"可积" 从孤立例子变成结构性现象。

1985-2000 量子可积 + 量子群（Drinfeld + Faddeev）。把可积系统量子化, 引入量子群。与统计力学、共形场论紧密相连。

2000-2026 现代延伸:

  • 非交换辛几何 + 量子化（Kontsevich 形式量子化定理 1997）。
  • Floer 同调 + 辛拓扑: 辛流形的拓扑不变量, Gromov 1985 + Floer 1988+。
  • 镜对称 + Calabi-Yau（弦论 + 几何力学的交汇）。
  • 积分系统 + Geometric Langlands（Beilinson-Drinfeld + Frenkel）。
  • 量子计算的几何相位（Berry 1984 + 拓扑量子计算 Kitaev 2003）——经典几何相空间 \(\to\) 量子 Berry 相 \(\to\) 拓扑量子门。
  • 航天工程: JWST 在 \(L_2\), Voyager 出太阳系, SpaceX 计划火星, Lucy 任务到 Trojan 小行星。本书 ch23。

从 1696 Bernoulli 一道挑战题, 到 2026 几何力学 + 量子计算 + 星际探测——330 年的力学进化让人类:

  • 知道天体怎么运动（Kepler + Newton + Lagrange）。
  • 知道为什么能量 / 动量守恒 (Noether)。
  • 知道经典如何通向量子 (Dirac + Feynman)。
  • 知道力学如何成为几何（Arnold + Marsden + Weinstein）。
  • 能把卫星送到 Lagrange 点 (Voyager + JWST)。
  • 能用经典几何相位做量子计算（Berry + Kitaev）。

这就是经典力学——不是 高中物理的"抛物运动 + 单摆" 旧课, 而是物理学最古老 / 最现代的双面: 1687 Newton 原理 + 2026 几何量子计算并存的活生生学科。

24.10 临别话: 看待力学的方式

本书结束。我希望你建立的不是 一堆方程, 而是看待力学的方式。

具体三层:

  1. 瞬间动力学（Newton, ch2-3）: 物体在每一瞬怎么动。\(F = ma\)。直接简单, 但处理约束 + 多坐标系不方便。
  2. 全局变分（Lagrange, ch4-9）: 整条路径如何决定。\(\delta S = 0\)。处理约束自动, 给 Noether 守恒律, 推广到任何流形 + 弯曲时空。
  3. 几何（Hamilton-Poisson + symplectic, ch10-15 + ch24）: 相空间是辛流形, Hamilton 流是辛同胚, 守恒量是 moment map。最抽象, 最对称, 通向量子化 + 可积系统 + 现代物理。

这三个层次不是"谁取代谁", 是同样物理的三种语言。看一道题, 学会问"用哪种语言最方便": Atwood 机用 Newton; 多自由度耦合用 Lagrange; 找守恒律 + 对称性用 Hamilton-Poisson; 处理一般辛流形 / 约化用几何力学。

第二个收获——对称性是物理学的灵魂。Noether 定理（ch7）+ moment map（本章）+ 量子化（ch21）三处反复出现: 物理学的守恒律 / 量子数 / 选择规则 都是对称的几何后果。20 世纪粒子物理把这条想法发挥到极致——夸克色 + 弱同位旋 + 超对称 + 弦理论的轮廓全是对称群论。本书前 23 章为这个视角搭舞台。

第三个收获——数学预言物理。Lagrange 1788 + Hamilton 1833 没想过量子力学; 但他们给的形式正好是 1925 量子力学需要的。Riemann 1854 没想过广义相对论; 但他给的弯曲几何正好是 1915 Einstein 需要的。Lie 1880-1893 没想过粒子物理; 但他给的 Lie 群正好是 20 世纪规范理论需要的。纯数学家在物理学家用它之前几十年甚至上百年就已经准备好工具。这是数学和物理彼此滋养的最美一面。

你读完本书, 看见了 1696-2026 的力学进化, 看见了 Newton-Lagrange-Hamilton-Poisson-Symplectic 五种语言, 看见了对称性怎么变守恒律, 看见了经典怎么通向量子, 看见了航天工程怎么用 18 世纪 Lagrange 点. 现在你有语言 谈论力学的本质, 不再被困在"F = ma" 的初等表述。这就是分析力学的真正价值。
