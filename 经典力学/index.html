普林斯顿经典力学读本

从最速降线到 Hamilton 流 · Lagrangian + Hamiltonian + Poisson + 几何力学

前言

欢迎你翻开这本书。经典力学是物理学最古老的分支——从 Newton 1687 年《自然哲学的数学原理》算起 339 年。可它有一个奇怪的事实——大部分 高级物理（电磁、相对论、量子力学、量子场论、统计力学）都不 直接用 Newton F=ma, 而是用 1788 Lagrange + 1833 Hamilton 重写过的语言。这本书就是教你从 Newton 到 Lagrange 到 Hamilton 这次语言升级——以及为什么它如此重要。

核心命题: Newton 的力学告诉你"物体在每一瞬怎么动"。Lagrange-Hamilton 的力学告诉你"物体的整条路径 由什么决定"。从瞬间动力学到全局变分，是物理学最深的视角转换之一。它让对称性 = 守恒律（Noether 定理）成为定理而不是巧合，让相对论 / 量子力学 / 场论的写法必然如此，让混沌 / 积分系统 / 几何力学有共同语言。

这本书的角色

系列第 15 卷。前置：多元微积分（梯度 / 散度 / 偏导）+ 微分方程（ODE 求解）。线性代数（矩阵 / 特征值）在 ch8 振动 + ch12 Poisson 用到。微分几何（流形 / 切丛）让 ch10-15 Hamiltonian 力学的几何视角更深, 但不必要。

下一步：量子力学（V14, 已上线）—— 量子力学的"正则量子化" 直接把经典 Hamilton 翻译成量子算子。也通向场论（连续 Lagrangian）+ 统计力学（相空间几何）。

这本书不是什么

它不是高中物理——不重讲牛顿三定律 + 抛物运动。本书 ch2 简短复习, 然后 ch3 起切到 Lagrange。

它不是连续介质力学专书——本书 ch18-19 简介弹性 + 流体, 但不深入。专书参考 Landau-Lifshitz Theory of Elasticity + Fluid Mechanics。

它不是经典场论专书——本书 ch20 给入口（Lagrangian 密度 + Klein-Gordon）, 但完整 QFT 不在范围。

读这本书之前你需要什么

你需要多元微积分（链式法则 + 偏导 + 多重积分）。本书几乎每章都用。

你需要微分方程到二阶 ODE（齐次 + 非齐次, 线性系统）。Lagrange / Hamilton 方程都是 ODE 系统。

你最好懂高中物理——动量、动能、势能。本书会 复习这些, 但用过会更顺。

你不 需要相对论 + 量子力学——本书 ch22 讲狭义相对论（自包含）, ch24 简介量子前夜。

"我下周就要考期中, 从哪里开始?"

本科分析力学 标准考试看：

  • Lagrange 方程 + 例子（ch5-7）
  • Noether 定理 + 守恒律（ch7）
  • 小振动 + 简正模式（ch8）
  • 刚体 + 欧拉角（ch9）
  • Hamilton 方程（ch10-11）
  • Poisson 括号 + 正则变换（ch12-13）

研究生高级 加上：

  • Hamilton-Jacobi（ch14）
  • 作用-角变量 + 积分系统（ch15）
  • KAM + 混沌（ch16-17）
  • 经典场论（ch20）
