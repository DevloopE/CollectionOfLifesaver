附录 A 复变函数速查

本书ch3 Schrödinger 方程的 \(i\hbar\)、ch5 高斯波包 Fourier 变换、ch11 不确定性原理 Cauchy-Schwarz、ch20 Fermi 金律的 \(\delta\) 函数极限、ch21 散射 Green 函数 + 留数计算——每一处都暗中用了"复变函数" 工具。复 在 QM 不是装饰, 是结构: 波函数本质复值, 算子本征值实数 \(\Leftrightarrow\) 算子自伴; 演化算子 \(e^{-iHt/\hbar}\) 是 Hilbert 空间的旋转——所有这些都依赖复函数代数。这个附录把本书用到的具体复变工具集中处理。

  • 复函数 + 解析性。
  • Cauchy-Riemann 方程。
  • Cauchy 积分定理 + 公式。
  • 留数定理 + 几个积分实例。
  • Fourier 变换 + 复路径技巧（量子散射常用）。

这里没有新概念, 只是把《复分析》读本里和量子力学相关的部分浓缩。要更深的请回去查那本。

A.1 复函数 + 解析性

复数: \(z = x + iy\), \(x, y \in \mathbb R\), \(i^2 = -1\). 复平面 \(\mathbb C\) 与 \(\mathbb R^2\) 等同。极坐标 \(z = r e^{i\theta}\), \(r = |z| = \sqrt{x^2 + y^2}\), \(\theta = \arg z\)。Euler 公式

\[ \boxed{\ e^{i\theta} = \cos\theta + i\sin\theta.\ } \]

这条公式是 QM 演化的核心——\(e^{-iEt/\hbar}\) 是个旋转, 把模 1 的复数旋转。

复函数: \(f: \mathbb C \to \mathbb C\), 写 \(f(z) = u(x, y) + i v(x, y)\), \(u, v\) 是实值。

复可微 (解析): \(f\) 在 \(z_0\) 处复可微 ⟺ 极限 \(\lim_{h \to 0} (f(z_0 + h) - f(z_0))/h\) 对任何 复路径 \(h \to 0\) 都存在且相等。这条比实可微强得多——实平面上 \(h\) 只能从 \(\pm\) 方向, 复平面上 \(h\) 可以从所有方向, 极限都得相同。

"解析"（analytic, holomorphic）= 在某区域每一点 都复可微。

例:

  • \(f(z) = z^2\): 解析（处处）。
  • \(f(z) = e^z = e^x(\cos y + i \sin y)\): 解析。
  • \(f(z) = \bar z = x - iy\): 不 解析（沿 \(h\) 实正得 1, 沿 \(h\) 虚正得 \(-1\), 不一致）。所以"取共轭" 在复分析里是个非平凡操作。
  • \(f(z) = 1/z\): 解析（\(z \neq 0\)）。\(z = 0\) 是极点。

A.2 Cauchy-Riemann 方程

\(f(z) = u(x, y) + i v(x, y)\) 解析等价于: 偏导存在 + 满足

\[ \boxed{\ \frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}, \quad \frac{\partial u}{\partial y} = -\frac{\partial v}{\partial x}.\ } \]

这是Cauchy-Riemann 方程(CR 方程)。证明思路: 沿实方向 \(h \to 0\) 给 \(f' = u_x + i v_x\); 沿虚方向 \(h = i k, k \to 0\) 给 \(f' = -i u_y + v_y\)。两边等给 CR。

推论 1（Laplace 方程）: 把 CR 第一式 \(u_x = v_y\) 对 \(x\) 求导 + 第二式 \(u_y = -v_x\) 对 \(y\) 求导, 加和得 \(u_{xx} + u_{yy} = 0\). 同样 \(v_{xx} + v_{yy} = 0\)。解析函数的实虚部都是调和函数。这条事实让复分析与 2D 静电学（Laplace 方程的解）紧密联系。

推论 2（共轭调和）: 给一个调和 \(u\), 求 \(v\) 满足 CR——\(v\) 由 \(u\) 决定到一个常数。\(v\) 叫 \(u\) 的"共轭调和"。

例验证: \(f(z) = z^2 = (x + iy)^2 = (x^2 - y^2) + i(2xy)\)。\(u = x^2 - y^2, v = 2xy\)。\(u_x = 2x, v_y = 2x\) ✓; \(u_y = -2y, v_x = 2y, -v_x = -2y\) ✓。Laplace: \(u_{xx} + u_{yy} = 2 + (-2) = 0\) ✓。

A.3 Cauchy 积分定理 + 公式

核心定理 1: Cauchy 积分定理。设 \(f\) 在简单封闭曲线 \(\gamma\) 内部及上面解析, 则

\[ \boxed{\ \oint_\gamma f(z) dz = 0.\ } \]

证明思路: 用 Green 定理 + CR 方程, 把复积分翻译成实平面 2D 积分, CR 让两个 2D 积分项都为零。直观: 解析函数沿封闭路径"没有累积"——保守"力场"。

推论: 不同从 \(z_0\) 到 \(z_1\) 的路径给同样积分（差是封闭路径 = 0）。所以 \(\int_{z_0}^{z_1} f dz\) 只与端点有关, 不与路径有关——只要 \(f\) 在路径间区域处处解析。

核心定理 2: Cauchy 积分公式。设 \(f\) 在 \(\gamma\) 内及其上解析, \(z_0\) 在 \(\gamma\) 内, 则

\[ \boxed{\ f(z_0) = \frac{1}{2\pi i} \oint_\gamma \frac{f(z)}{z - z_0} dz.\ } \]

这条公式说——解析函数在区域内任一点的值, 完全由它在边界上的值决定。这种"边界决定内部" 是解析函数的极强性质, 实函数没有类似性质（\(C^\infty\) 函数可以在不同点完全独立, 解析函数不能）。

推广（高阶 Cauchy 公式）: 解析函数无穷可微, 且

\[ f^{(n)}(z_0) = \frac{n!}{2\pi i} \oint_\gamma \frac{f(z)}{(z - z_0)^{n+1}} dz. \]

所以"解析 ⟹ 无穷可微"——这又是复函数比实函数强的地方（实函数 \(C^1\) 不蕴含 \(C^2\)）。

A.4 留数定理

函数 \(f\) 在 \(z_0\) 有极点(简单极点 = 1 阶极点) 如果 \(f(z) = a_{-1}/(z - z_0) + a_0 + a_1(z - z_0) + \ldots\)（在 \(z_0\) 附近 Laurent 展开）。留数:

\[ \mathrm{Res}(f, z_0) = a_{-1} = \lim_{z \to z_0} (z - z_0) f(z) \quad (\text{1 阶极点}). \]

\(n\) 阶极点（\(f \sim a_{-n}/(z - z_0)^n + \ldots\)）的留数:

\[ \mathrm{Res}(f, z_0) = \frac{1}{(n-1)!} \lim_{z \to z_0} \frac{d^{n-1}}{dz^{n-1}} \big[(z - z_0)^n f(z)\big]. \]

留数定理: 设 \(f\) 在 \(\gamma\) 内除有限个极点 \(\{z_k\}\) 处解析, 则

\[ \boxed{\ \oint_\gamma f(z) dz = 2\pi i \sum_k \mathrm{Res}(f, z_k).\ } \]

这条定理是计算实积分 最强大的工具——许多形式难的实积分通过把它扩到复平面 + 找留数瞬间 算出来。

例 1: \(\int_{-\infty}^\infty \frac{dx}{1 + x^2} = ?\)。把 \(1/(1 + z^2)\) 在复平面上闭合积分（实轴 + 上半圆 \(R \to \infty\), 上半圆贡献趋零）。极点 \(z = i\) 在上半平面, 留数 \(\lim_{z \to i}(z - i)/((z - i)(z + i)) = 1/(2i)\). 所以积分 \(= 2\pi i \cdot 1/(2i) = \pi\)。

例 2（量子散射常用）: Green 函数

\[ G(\boldsymbol r) = -\frac{1}{(2\pi)^3} \int \frac{e^{i\boldsymbol k \cdot \boldsymbol r}}{k^2 - k_0^2 - i\varepsilon} d^3 k = -\frac{1}{4\pi r} e^{i k_0 r}. \]

计算: 先做角度积分 → 留下 \(\int_0^\infty \frac{k \sin(kr)}{k^2 - k_0^2 - i\varepsilon} dk\). 把 \(\sin = (e^{ikr} - e^{-ikr})/(2i)\), 把积分扩到 \((-\infty, \infty)\), 用Jordan 引理(指数因子让大半圆贡献趋零)闭合上半 / 下半。\(-i\varepsilon\) 让极点 \(k = k_0 \pm i\varepsilon/(2k_0)\) 偏一点, 选取相应半平面给出射 边界条件 \(e^{ik_0 r}\)。这条 \(-i\varepsilon\) 处方就是21.4 Lippmann-Schwinger 方程里 Green 函数的"选择出射波" 边界条件的来源。

A.5 Fourier 变换 + 复路径技巧

Fourier 变换在 QM 里到处出现——ch5 高斯波包从 \(x\) 空间到 \(k\) 空间, ch21 Born 振幅是 \(V\) 的 Fourier 变换。基本定义:

\[ \tilde f(k) = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^\infty f(x) e^{-ikx} dx, \quad f(x) = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^\infty \tilde f(k) e^{ikx} dk. \]

关键性质（QM 中常用）:

  • 线性: \(\widetilde{a f + b g} = a \tilde f + b \tilde g\).
  • 导数 ↔ 乘 \(k\): \(\widetilde{f'}(k) = ik \tilde f(k)\). 这是 ch9 动量算子 \(\hat p = -i\hbar \partial/\partial x\) 在 \(k\) 表象成乘 \(\hbar k\) 的根源。
  • Parseval: \(\int |f|^2 dx = \int |\tilde f|^2 dk\). 两表象归一化等价。
  • 卷积 ↔ 乘积: \(\widetilde{f * g} = \sqrt{2\pi} \tilde f \tilde g\). \(\delta\) 函数: \(\tilde \delta(k) = 1/\sqrt{2\pi}\)（常数）。

具体例子（QM 用得多）:

\(f(x)\)\(\tilde f(k)\)
\(e^{-x^2/(2\sigma^2)}\)\(\sigma e^{-\sigma^2 k^2/2}\)（差归一化）
\(e^{-a|x|}\)\(\sqrt{2/\pi} \cdot a/(a^2 + k^2)\)
\(1/(x^2 + a^2)\)\(\sqrt{\pi/2} e^{-a|k|}/a\)
\(\delta(x)\)\(1/\sqrt{2\pi}\)
\(\theta(x) e^{-ax}\)（\(a > 0\)）\(1/(\sqrt{2\pi}(a + ik))\)

第二、三行用留数定理 算: \(\int e^{-ikx}/(x^2 + a^2) dx\), 极点 \(x = \pm ia\)。\(k > 0\) 时 \(e^{-ikx} = e^{-ik(\text{Re}x) + k \text{Im}x}\), 让下半 平面（\(\text{Im}x < 0\)）的指数趋零, 闭合下半圆, 极点 \(-ia\); \(k < 0\) 闭合上半, 极点 \(+ia\)。两情况合一得 \(\sqrt{\pi/2} e^{-a|k|}/a\)。

第三行Lorentzian → 指数 是20 章 Fermi 金律里 \(\delta\) 函数极限和指数衰减的连接——共振线宽 \(\Gamma\) ↔ 寿命 \(1/\Gamma\) 是这条 Fourier 配对的物理体现。

A.6 实战准则

  • 遇到积分 \(\int_{-\infty}^\infty\) 形式的有理函数 / 三角 / 指数衰减——先试把它扩到复平面, 找极点, 用留数定理。
  • 含 \(e^{ikx}\) 的积分用 Jordan 引理决定闭合上半还是下半圆——\(k > 0\) 闭合上半（让 \(\text{Im}z > 0\) 时 \(e^{ikz}\) 衰减）, \(k < 0\) 闭合下半。
  • 含奇异点在实轴上 的积分（散射 Green 函数 \(1/(k^2 - k_0^2)\) 实根 \(k = \pm k_0\)）需要"\(-i\varepsilon\) 处方" 把极点偏到一侧——这种偏离选择决定边界条件（"出射 vs 入射" 散射波）。
  • Fourier 变换的对称性: \(f\) 实 ⟹ \(\tilde f^*(-k) = \tilde f(k)\); \(f\) 偶 ⟹ \(\tilde f\) 偶; \(f\) 奇 ⟹ \(\tilde f\) 奇。这条让你预知 \(\tilde f\) 的某些特征不必算。
  • 波函数的位置(\(x\)) 与动量(\(p = \hbar k\)) 表象之间用 Fourier 变换链接 (ch9-11)——所以 \(\Delta x \Delta p \ge \hbar/2\) 本质是Fourier 不确定性: 函数在 \(x\) 空间窄 ⟺ 在 \(k\) 空间宽。
  • 代码: NumPy np.fft.fft 给离散 Fourier 变换; scipy.special 含许多解析函数（Bessel、Hermite、Laguerre 等, 散射 + 谐振子 + 氢原子用）。
