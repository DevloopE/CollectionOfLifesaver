第 12 章 测量与塌缩

请你想象做下面这个实验。准备一个自旋-1/2 粒子（电子）在状态 \(|\psi\rangle = (|\uparrow\rangle + |\downarrow\rangle)/\sqrt 2\)——"上自旋和下自旋的等权叠加"。让它通过一个 Stern-Gerlach 磁铁（沿 \(z\) 轴方向梯度场）。

磁铁会把"上" 自旋偏向上方, "下" 自旋偏向下方。理论预测——经典物理说粒子带某种角动量沿 \(z\) 的某个方向, 通过磁场会被偏到某个固定角度。一束在 \(|\psi\rangle\) 状态的粒子进入磁场, 你预期看到什么？

实验结果: 屏幕上恰好两个亮点——一个上方, 一个下方, 各占一半粒子, 没有任何粒子落在中间。换句话说, 每个粒子要么 完全"上" 要么 完全"下", 没有"半上半下"。

但我们准备的态明明是 \(|\uparrow\rangle\) 和 \(|\downarrow\rangle\) 的叠加。叠加去哪了？——它"塌缩" 了。每个粒子在通过磁铁的瞬间, 被强制选择"上" 或"下", 概率各 1/2。叠加态的叠加 性质消失。

这就是量子测量 的基本现象。它之所以是量子力学最哲学的问题, 是因为它和ch4 学过的 Schrödinger 演化看起来完全冲突:

  • SE 演化 \(i\hbar \partial_t |\psi\rangle = \hat H |\psi\rangle\)——确定性（同样初态给同样末态）+ 可逆（\(U(t)\) 是 unitary, \(U^{-1} = U^\dagger\)）+ 线性（叠加保留: \(U(t)(\alpha|\psi_1\rangle + \beta|\psi_2\rangle) = \alpha U|\psi_1\rangle + \beta U|\psi_2\rangle\)）。
  • 测量 \(|\psi\rangle \to |\uparrow\rangle\) 或 \(|\downarrow\rangle\)——随机（同样初态给不同末态, 概率分布）+ 不可逆（你不能从 \(|\uparrow\rangle\) 反推回原叠加）+ 非线性（叠加被破坏）。

这两条动力学怎么共存？这就是测量问题(measurement problem), 量子力学最大的诠释难题。这一章我把测量的标准框架（Born + 投影假设）讲完, 然后看测量问题的几种回答（哥本哈根 / 多世界 / 隐变量 / decoherence), 不过分深入但要让你看见它的轮廓。

12.1 两种动力学: SE 演化 vs 测量

把上面的对照写得更细。

SE 演化是ch9-ch10 我们写过的:

\[ i\hbar \frac{d|\psi\rangle}{dt} = \hat H |\psi\rangle. \]

解形式 \(|\psi(t)\rangle = U(t) |\psi(0)\rangle\) where \(U(t) = e^{-i\hat H t/\hbar}\)。\(U(t)\) 是 unitary（保模、保内积）, 因此可逆。同一初态在两次同样的实验里给同一末态, 确定。两个初态的叠加给两个末态的叠加, 线性。

这条动力学在你"不看" 系统的时候一直 在跑。粒子在磁铁里走, 波函数按 SE 演化, 整个过程都没有 "塌缩" 出现。

然后你看。具体说: 在屏幕上看粒子落在哪。瞬间结果 是个具体值——某个确定的位置——而不 是个概率分布。这个"看 → 得到具体结果" 的瞬间, 波函数被假设塌缩 到与那个结果对应的本征态。

请你看这两条动力学的"性格冲突":

性质SE 演化测量
确定性是否（随机）
可逆是 (\(U^{-1}=U^\dagger\))否（信息丢失）
线性是否（投影非线性）
连续是 (\(U(t)\) 强连续)否（瞬时跳）

每一条对照都反。但实验明明告诉我们两种行为都存在。怎么调和？

这条问题的哲学含义——它问"什么是测量"。如果"测量" 就是粒子和某个仪器的相互作用, 那仪器本身也是个量子系统（仪器是原子组成的）, 它和粒子应该按 SE 演化(unitary, 保叠加), 不应该出现"塌缩"。但宏观仪器读数实际上 给出确定结果, 不 是叠加。这条"从微观叠加到宏观确定" 的转折在哪发生？这是测量问题的真正硬核。

下面几节先把"标准" 框架写完, 12.6 后再看几种诠释。

12.2 哥本哈根诠释 + Born 概率

1927 年 Bohr + Heisenberg + Born 在哥本哈根形成的"哥本哈根诠释" 是量子力学的标准 操作手册。它有三条:

  • (C1) 状态。系统由 Hilbert 空间向量 \(|\psi\rangle\) 描述（归一化, \(\langle \psi|\psi\rangle = 1\)）。两条动力学并存:
    • "不看" 时, \(|\psi\rangle\) 按 SE 演化。
    • "看"（测量）时, \(|\psi\rangle\) 按下面 (C3) 塌缩。
  • (C2) Born 概率。测量自伴算子 \(\hat A\)（带本征值 \(a_n\), 本征态 \(|n\rangle\)）, 得到结果 \(a_n\) 的概率 \[ P(a_n) = |\langle n | \psi\rangle|^2. \]
  • (C3) 投影假设。测得 \(a_n\) 后, 系统瞬时塌缩到对应本征态 \(|n\rangle\): \[ |\psi\rangle \xrightarrow{\text{测得 } a_n} |n\rangle. \]

这三条加上 SE 演化, 量子力学就完全 操作可计算了。它告诉你:

  • (怎么算) 给定初态 \(|\psi(0)\rangle\), 用 SE 演化到测量时刻 \(|\psi(t)\rangle\); 用 Born 给出测量结果的概率分布; 测得某个结果后用投影假设给出新态。
  • (不告诉你) "塌缩" 在物理上是什么; 哪种相互作用算"测量"; 宏观仪器为什么"选边"。

哥本哈根诠释的态度——"闭嘴算就行"（"shut up and calculate"）。它把测量当作一条原始公设, 不问它的内部机制。这条态度让量子力学在 1930 年代立即 可用; 但它把哲学问题推迟 到 1960 年代之后才被认真对待。

请你先接受 (C1)-(C3) 当工作框架, 后面 12.6 起再看其它诠释怎么不同。

12.3 投影算子和投影假设的几何

投影假设 (C3) 用投影算子 写得更整洁。设 \(\hat A\) 有谱分解

\[ \hat A = \sum_n a_n \hat P_n, \qquad \hat P_n = |n\rangle\langle n|, \]

其中 \(\hat P_n\) 是到第 \(n\) 个本征子空间的投影算子（\(\hat P_n^2 = \hat P_n\), \(\hat P_n^\dagger = \hat P_n\), \(\sum_n \hat P_n = \hat I\)）。

投影假设可以写成

\[ \boxed{\ |\psi\rangle \xrightarrow{\text{测得 } a_n} \frac{\hat P_n |\psi\rangle}{\|\hat P_n |\psi\rangle\|}.\ } \]

分子是 \(|\psi\rangle\) 投到第 \(n\) 个本征子空间的部分; 分母重新归一。Born 概率自然给

\[ P(a_n) = \|\hat P_n |\psi\rangle\|^2 = \langle\psi | \hat P_n | \psi\rangle. \]

这条公式更一般——它对退化 谱（多个本征向量对应同一本征值, 即 \(\hat P_n\) 投到一个 \(>1\) 维子空间）也对。退化情形下塌缩后的态不一定是单本征向量, 而是子空间里的某个具体向量, 由 \(\hat P_n |\psi\rangle\) 自然挑出。

几何上请你画一画——\(|\psi\rangle\) 是 Hilbert 空间里的单位向量。本征子空间 \(\{n\}\) 给一组互相正交的"方向" / "子空间"。投影到第 \(n\) 个就是把 \(|\psi\rangle\) 在那个方向 / 子空间上的分量 抽出来。Born 概率就是这个分量长度的平方。勾股定理 给 \(\sum_n P(a_n) = \sum_n \|\hat P_n\psi\|^2 = \|\psi\|^2 = 1\)——概率自动归一。

这种"测量 = 投影 + 归一 + Born 概率" 的几何描述常被称为 von Neumann 投影测量(projective measurement, 1932 年 von Neumann 在《量子力学的数学基础》里给出)。它是测量公设的最常见形式。

12.4 顺序测量 (i): 重复性

测量假设有个立即 推论——"同一观测连测两次给同样结果"。具体地: 测 \(\hat A\) 得 \(a_n\), 立刻 再测 \(\hat A\), 必得 \(a_n\)。

证明: 第一次测后, 态变 \(|n\rangle = \hat P_n|\psi\rangle / \|\cdot\|\)（设非退化, 是 \(\hat A\) 的第 \(n\) 个本征向量）。"立刻" 表示中间没经过 SE 演化, 态仍是 \(|n\rangle\)。第二次测 \(\hat A\) 得 \(a_m\) 的概率

\[ P(a_m | \text{首测得 } a_n) = |\langle m|n\rangle|^2 = \delta_{mn}. \]

所以必得 \(a_n\), 概率 1。\(\blacksquare\)

这条"重复性" 听起来朴素但非常 重要。它是测量结果在物理上有意义 的基础——如果你测出"电子动量是 \(p_0\)", 你立刻测一次还是 \(p_0\), 这个值就是电子动量, 不是个偶然抽签。这条性质把 \(a_n\) 这个数字称为"电子在测后状态下的动量" 是合理的。

请你想象——如果没有 重复性, 量子力学的"测量" 这个概念就基本没意义了。每次测都给个随机数, 你完全不能说"这个粒子的动量是某某"。重复性赋予测量语义。

12.5 顺序测量 (ii): 不相容观测的互补性

对照——测两个不相容 观测会怎样？

设 \([\hat A, \hat B] \ne 0\), 它们没有共同的完整本征基。本征态 \(|n\rangle_A, |m\rangle_B\) 之间一般 \(\langle m_B | n_A\rangle \ne 0\) 但 \(\ne \delta_{mn}\)。

从初态 \(|\psi\rangle\) 出发——

  • 测 \(\hat A\), 概率 \(P(a_n) = |\langle n_A|\psi\rangle|^2\) 得 \(a_n\), 态变 \(|n_A\rangle\)。
  • 立刻测 \(\hat B\), 概率 \(P(b_m | a_n) = |\langle m_B|n_A\rangle|^2\) 得 \(b_m\), 态变 \(|m_B\rangle\)。
  • 立刻再测 \(\hat A\), 概率 \(P(a_k | b_m) = |\langle k_A|m_B\rangle|^2\) 得 \(a_k\)。

关键: 第三次测 \(\hat A\) 的结果不一定 是 \(a_n\)。第二次测 \(\hat B\) 把态从 \(|n_A\rangle\) 推到 \(|m_B\rangle\), 后者一般不是 \(\hat A\) 的本征态, 所以再测 \(\hat A\) 的结果是个新 概率分布。第一次测 \(\hat A\) 的"记忆" 被擦了。

这就是 Bohr 强调的互补性(complementarity)——位置和动量是"互补" 的, 知道一个就不能同时知道另一个; 知道得越准越互相干扰。测量动作 改变状态, 不仅仅"读取" 它。

用 Stern-Gerlach 链具体看。粒子在 \(|\uparrow_z\rangle\) 通过 \(S_x\) 磁铁——\(|\uparrow_z\rangle = (|\uparrow_x\rangle + |\downarrow_x\rangle)/\sqrt 2\), 测 \(S_x\) 给 \(\pm \hbar/2\) 各 1/2。设得 \(+\hbar/2\), 态变 \(|\uparrow_x\rangle = (|\uparrow_z\rangle + |\downarrow_z\rangle)/\sqrt 2\)。再测 \(S_z\)——给 \(\pm \hbar/2\) 各 1/2。原本知道的"沿 \(z\) 是上"完全丢了。中间夹一个 \(S_x\) 测量, 把 \(z\) 信息抹掉。

这条互补性的实验在 Stern-Gerlach 链里反复演示, 是量子测量"非被动" 的清楚证据。"测量改变态" 不是哲学修辞, 是可重复的实验事实。

12.6 Schrödinger 的猫

1935 年 Schrödinger 在一封批评哥本哈根诠释的信里给出"猫" 的思想实验:

把一只猫关在密封盒子里, 配上: 一个原子（半衰期 1 小时）+ 一个 Geiger 计数器 + 一个毒气瓶。原子衰变 → 计数器响 → 毒气放出 → 猫死。1 小时后原子有 50% 概率衰变。

按哥本哈根诠释——你不打开盒子时, 没有"测量", 整个系统按 SE 演化。原子 + 计数器 + 毒气 + 猫合在一起的波函数

\[ |\Psi\rangle = \frac{1}{\sqrt 2}\big( |\text{未衰}\rangle|\text{无毒}\rangle|\text{活猫}\rangle + |\text{已衰}\rangle|\text{有毒}\rangle|\text{死猫}\rangle\big). \]

叠加。猫处于"半活半死" 的叠加态。

这显然 反直觉。猫这个宏观物体怎么可能"叠加"？打开盒子瞬间塌缩到"活" 或"死", 这条"测量" 的瞬间 / 边界在哪？打开盒子的人是不是"观察者"？如果观察者也是个量子系统, 谁来观察观察者？

这条思想实验把"测量问题" 从微观（电子叠加）放大到宏观（猫叠加）。哥本哈根诠释回答得不彻底——它承认 微观叠加, 但说宏观仪器 / 观察者"不" 处于叠加, 边界在哪没说。这条边界问题——"Heisenberg cut" / "von Neumann 切口"——是测量问题的核心。

请你停下来想: 这不 是 Schrödinger 在反对量子力学算法（他自己写的方程）。他在指出哥本哈根诠释把测量当原始公设的哲学不彻底。70 年后这条困惑仍然没有完全解决。

12.7 退相干: 实际中"看起来塌缩"

20 世纪 70-80 年代 Zurek、Joos、Zeh 等人发展了退相干(decoherence) 理论, 给"测量问题" 一个部分 答案——不是诠释问题的彻底解决, 但是在"为什么宏观看不到叠加" 这个具体问题上很有说服力。

核心想法: 没有完全孤立的量子系统。任何"系统" \(S\) 都和"环境" \(E\)（空气分子、热辐射、宇宙背景辐射、…）相互作用。系统 + 环境合起来按 SE 演化（unitary, 保叠加）。但你只能观测系统 \(S\), 看不到环境 \(E\)——所以你算的是约化密度矩阵(reduced density matrix)

\[ \rho_S = \text{Tr}_E |\Psi_{SE}\rangle\langle \Psi_{SE}|. \]

(密度矩阵 是 \(|\psi\rangle\langle\psi|\) 的推广, 用于描述混合态; ch22 详谈纠缠时正式引入。)

关键计算: 当系统和环境相互作用足够长时间, \(\rho_S\) 的非对角 矩阵元（"相干性"）指数衰减到 0:

\[ \rho_S = \sum_n |c_n|^2 |n\rangle\langle n| + \underbrace{\sum_{n\ne m} c_n c_m^* |n\rangle\langle m| \cdot e^{-t/\tau_d}}_{\to 0}. \]

对宏观系统, \(\tau_d\) 极小（猫和空气相互作用的退相干时间 \(\sim 10^{-23}\) 秒）。剩下的 \(\rho_S\) 是对角 形式——经典概率分布的样子。实际看起来 系统已"选定" 某个本征态, 概率 \(|c_n|^2\)（恰好就是 Born）。

这个结果不需要 测量公设, 它纯 是 SE + "不看环境" 的推论。它解释 了为什么宏观叠加在实验上看不到——退相干太快, 你来不及看到 \(\rho_S\) 还有非对角元。

但请你注意——退相干不 完全解决测量问题。它给了"看起来塌缩"（\(\rho_S\) 对角化, 像经典概率混合）, 但 没解释"为什么实际只观察到某一个特定结果"（"为什么是猫死了, 不是猫活着"）。后者是选择问题(definite outcome problem), 退相干无能为力。

退相干是哥本哈根诠释的动力学化——它把"塌缩" 这条非动力学公设替换为"系统 + 环境的快速纠缠"。但选择哪个结果, 仍需要别的诠释来回答。

12.8 别的诠释一句话

测量问题催生了几种非 哥本哈根诠释。每种试图把"塌缩" 这个非动力学公设要么消除要么解释。我各给一句话, 不深入:

  • 多世界诠释(Everett 1957)。没有塌缩。整个宇宙波函数按 SE 演化。每次"测量" 时, 宇宙"分支" 成多个世界, 每个世界对应一个可能结果。"你" 也分支——在一个分支里看到猫死, 在另一个分支里看到猫活, 两个分支都同时存在 但互不通信。Born 概率被解释为"分支权重"（这条解释技术上仍有争议）。
  • 隐变量诠释(de Broglie 1927 / Bohm 1952)。有塌缩, 但塌缩只是"无知"。粒子有真实 位置和动量, 我们只是不知道。波函数是个"导航波" 引导粒子走, Born 概率反映我们对真实变量的统计无知, 不是物理本身的随机。Bell 1964 证明非定域 隐变量可以兼容量子预测, 定域 隐变量不行（ch22 Bell 不等式详谈）。
  • 退相干 + 一致历史(Griffiths / Omnès / Gell-Mann-Hartle)。在"退相干已发生" 的语境里, 经典逻辑近似有效, 概率论按经典方式应用。哥本哈根诠释的精细化版。
  • 客观塌缩理论(GRW 1986 / Penrose 1996)。修改 Schrödinger 方程, 加一项真随机 塌缩项, 使宏观系统自发塌缩, 微观不变。实验可证伪。目前最严格实验仍兼容标准 QM, 但 GRW 给的塌缩率上限被压缩。

哪种正确？目前没有共识。所有诠释都给出相同的实验预测（这是它们被认真对待的前提）。区别在哲学——什么是真实、什么是观察者、什么是测量。

请你保持开放——别让任何诠释把"这就是真相" 卖给你。当代物理学家大多务实: 算的时候用哥本哈根（最简单 + 操作明确）, 哲学反思时用多世界 / Bohm / 退相干（看个人偏好）。不 信任何一种诠释是说自己"是物理上正确" 的。

12.9 这一章的命名 + reframe

把核心观念命名:

  • 测量问题: SE 演化（unitary, 线性, 可逆）vs 测量塌缩（随机, 非线性, 不可逆）的不一致。
  • Born 概率 \(P(a_n) = |\langle n|\psi\rangle|^2\): 测量结果的统计预测。
  • 投影假设 \(|\psi\rangle \to \hat P_n|\psi\rangle/\|\cdot\|\): 测量后状态的更新规则。
  • 重复性: 同观测连测两次给同结果, 测量赋予语义。
  • 互补性: 不相容观测的测量顺序不可 交换, 后测会擦前测的"记忆"。
  • 退相干: 系统 + 环境纠缠 → 约化密度矩阵对角化 → "看起来" 塌缩。

这一章的 Banner 弧线: 朴素想法——"测量就是看一下, 不会动到系统"——撞墙于 Stern-Gerlach 链 + 不相容观测的实验。Reframe: 测量是个主动 的、把状态推到本征子空间的投影 操作, 概率服从 Born。一般化: 投影测量推广到 POVM（正算子值测量, ch22）, 退相干理论给"测量" 一个动力学解释, 多种诠释回答"塌缩本质"。

测量问题没有 在这一章被解决——它没有在 90 年的物理学里被完全解决。但你已经把它的形状 看清楚了——它不是一个"哥本哈根诠释错了" 的简单问题, 它是"从微观叠加到宏观确定的过渡机制" 这个深刻的物理 / 哲学问题。本书ch22 引入纠缠和 Bell 不等式后, 我们会再回来这条话题——Bell 实验把"定域隐变量" 这条诠释路径切死, 但仍未把测量问题彻底解决。

下一章我们换回数学骨架——量子力学有两种等价但风格不同的"表象"——Schrödinger（波函数随时间, 算子不变）vs Heisenberg（算子随时间, 状态不变）。理解两种表象的等价性 + 切换方法是ch20 微扰论的入场券。ch13 详谈。
