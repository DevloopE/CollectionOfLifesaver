第 14 章 角动量算子与对易关系

请你先想象一颗经典的行星。它绕太阳转, 在某瞬时它有位置 \(\boldsymbol r\) 和动量 \(\boldsymbol p\)。它的角动量 是叉乘

\[ \boldsymbol L = \boldsymbol r \times \boldsymbol p, \qquad L_x = y p_z - z p_y, \quad L_y = z p_x - x p_z, \quad L_z = x p_y - y p_x. \]

三个分量都是普通实数, 同时 测它们没问题——你拍一张行星照片, 角动量三个分量同时确定。地球角动量沿 \(z\)（黄道法线）大约 \(2.66 \times 10^{40}\) kg·m²/s, \(x, y\) 分量小到可忽略——三个数字, 一起读出来。

这一章你会看到, 量子 角动量做不到这件事。把 \(x\) 换成 \(\hat x\), \(p_x\) 换成 \(\hat p_x = -i\hbar \partial_x\)（链回ch9）, 写下 \(\hat L_x = \hat y \hat p_z - \hat z \hat p_y\) 等等。这三个算子不互相对易——\([\hat L_x, \hat L_y] \neq 0\)。具体是 \([\hat L_x, \hat L_y] = i\hbar \hat L_z\)。这条非零对易子的物理后果是三个分量不能同时被精确测量——量子角动量从一开始就和经典分家了。

但分家的代价不是混乱, 是新的结构。\(\hat L^2 = \hat L_x^2 + \hat L_y^2 + \hat L_z^2\)（角动量大小的平方）对每一个 \(\hat L_i\) 都对易, 所以 \(\hat L^2, \hat L_z\) 可以同时本征化。这一对的本征值给出量子角动量的全貌——\(\hat L^2\) 取离散值 \(\hbar^2 l(l+1)\) for \(l = 0, 1, 2, \ldots\), \(\hat L_z\) 取 \(\hbar m\) for \(m = -l, -l+1, \ldots, l-1, l\)。角动量的"大小"和"沿一个轴的分量"被量子化, 而剩下两个分量保持模糊。

这一章先从对易关系算起——只用基本的位置-动量对易子, 纯代数 推出整个角动量谱。完全不解 PDE。这是量子力学最优雅的一段——结构本身决定结果。

14.1 角动量算子的定义 + 第一个对易子

把经典 \(\boldsymbol L = \boldsymbol r \times \boldsymbol p\) 直接替换成算子。三分量是

\[ \hat L_x = \hat y \hat p_z - \hat z \hat p_y, \quad \hat L_y = \hat z \hat p_x - \hat x \hat p_z, \quad \hat L_z = \hat x \hat p_y - \hat y \hat p_x. \]

简记为 \(\hat L_i = \epsilon_{ijk} \hat r_j \hat p_k\)（重复指标求和, \(\epsilon_{ijk}\) 是 Levi-Civita 反对称张量）。这三个算子在位置表象里是

\[ \hat L_x = -i\hbar \left( y \frac{\partial}{\partial z} - z \frac{\partial}{\partial y} \right), \quad \hat L_y = -i\hbar \left( z \frac{\partial}{\partial x} - x \frac{\partial}{\partial z} \right), \quad \hat L_z = -i\hbar \left( x \frac{\partial}{\partial y} - y \frac{\partial}{\partial x} \right). \]

请你先注意一件事——\(\hat L_x\) 里出现的位置 (\(y, z\)) 和动量 (\(\hat p_z, \hat p_y\)) 不在同一个方向上。\(\hat y\) 和 \(\hat p_z\) 对易（不同方向, \([\hat y, \hat p_z] = 0\)）, \(\hat z\) 和 \(\hat p_y\) 也对易。所以 \(\hat L_x\) 内部各项次序无所谓, 不需要"对称化"。这件事对所有 \(\hat L_i\) 都成立。

现在算 \([\hat L_x, \hat L_y]\)。展开：

\[ [\hat L_x, \hat L_y] = [\hat y \hat p_z - \hat z \hat p_y, \, \hat z \hat p_x - \hat x \hat p_z]. \]

对易子是双线性, 拆成四项

\[ = [\hat y \hat p_z, \hat z \hat p_x] - [\hat y \hat p_z, \hat x \hat p_z] - [\hat z \hat p_y, \hat z \hat p_x] + [\hat z \hat p_y, \hat x \hat p_z]. \]

第二、第三项里两边都没"同方向 的位置-动量"配对——\(\hat y \hat p_z\) 和 \(\hat x \hat p_z\) 中, \(\hat y, \hat x, \hat p_z\) 三两两对易, 整体对易子 0。同理 \([\hat z \hat p_y, \hat z \hat p_x] = 0\)。剩下第一、第四项。

第一项 \([\hat y \hat p_z, \hat z \hat p_x]\)。用恒等式 \([AB, CD] = A[B, C]D + [A, C]BD + AC[B, D] + C[A, D]B\)（不等价于 \([A, C]BD + ...\), 严格做要小心, 简便的做法是用 \([AB, C] = A[B, C] + [A, C]B\) 嵌套）。这里 \(\hat y\) 和 \(\hat z, \hat p_x\) 都对易, \(\hat p_x\) 和 \(\hat p_z\) 对易, 所以唯一非零的对易子来自 \([\hat p_z, \hat z]\)：

\[ [\hat y \hat p_z, \hat z \hat p_x] = \hat y \cdot [\hat p_z, \hat z] \cdot \hat p_x = \hat y \cdot (-i\hbar) \cdot \hat p_x = -i\hbar \hat y \hat p_x. \]

第四项 \([\hat z \hat p_y, \hat x \hat p_z]\)。同理唯一非零来自 \([\hat z, \hat p_z] = i\hbar\)：

\[ [\hat z \hat p_y, \hat x \hat p_z] = \hat x \cdot [\hat z, \hat p_z] \cdot \hat p_y = \hat x \cdot i\hbar \cdot \hat p_y = i\hbar \hat x \hat p_y. \]

合起来

\[ [\hat L_x, \hat L_y] = -i\hbar \hat y \hat p_x + i\hbar \hat x \hat p_y = i\hbar (\hat x \hat p_y - \hat y \hat p_x) = i\hbar \hat L_z. \]

这就是第一个核心对易关系:

\[ \boxed{\ [\hat L_x, \hat L_y] = i\hbar \hat L_z.\ } \]

由对称性（绕轴循环）立即得

\[ [\hat L_y, \hat L_z] = i\hbar \hat L_x, \qquad [\hat L_z, \hat L_x] = i\hbar \hat L_y. \]

三式合一: \([\hat L_i, \hat L_j] = i\hbar \epsilon_{ijk} \hat L_k\)。

请你停下来盯住这个结果。它看起来简单——三个对易子, 每个等于 \(i\hbar\) 乘第三个分量。但它的物理后果是核弹级: 三个角动量分量不能同时本征化, 也就是不能同时精确测量。这条与不确定性原理（链ch11）直接挂钩。

具体说：广义不确定性关系（ch11）

\[ \Delta A \cdot \Delta B \ge \frac{1}{2} | \langle [A, B] \rangle |. \]

代 \(A = \hat L_x, B = \hat L_y\)：

\[ \Delta L_x \cdot \Delta L_y \ge \frac{\hbar}{2} | \langle \hat L_z \rangle |. \]

如果 \(\langle \hat L_z \rangle \neq 0\), \(L_x, L_y\) 必然两边都有不确定性——你无法 让它们同时精确为某个值。

14.2 \(\hat L^2\) 与每个分量对易

三分量两两不对易, 那"角动量大小" \(\hat L^2 = \hat L_x^2 + \hat L_y^2 + \hat L_z^2\) 呢？

声明: \(\hat L^2\) 与每一个 \(\hat L_i\) 都对易——\([\hat L^2, \hat L_x] = [\hat L^2, \hat L_y] = [\hat L^2, \hat L_z] = 0\)。

来证 \([\hat L^2, \hat L_z] = 0\)（其它分量同理）。展开 \(\hat L^2 = \hat L_x^2 + \hat L_y^2 + \hat L_z^2\)：

\[ [\hat L^2, \hat L_z] = [\hat L_x^2, \hat L_z] + [\hat L_y^2, \hat L_z] + [\hat L_z^2, \hat L_z]. \]

最后一项 0（任何算子和自己对易）。前两项用 \([A^2, B] = A[A, B] + [A, B] A\)：

\[ [\hat L_x^2, \hat L_z] = \hat L_x [\hat L_x, \hat L_z] + [\hat L_x, \hat L_z] \hat L_x = \hat L_x (-i\hbar \hat L_y) + (-i\hbar \hat L_y) \hat L_x = -i\hbar (\hat L_x \hat L_y + \hat L_y \hat L_x). \]

\[ [\hat L_y^2, \hat L_z] = \hat L_y [\hat L_y, \hat L_z] + [\hat L_y, \hat L_z] \hat L_y = \hat L_y (i\hbar \hat L_x) + (i\hbar \hat L_x) \hat L_y = i\hbar (\hat L_y \hat L_x + \hat L_x \hat L_y). \]

两个加起来——括号里的反对称组合互相抵消：

\[ [\hat L^2, \hat L_z] = -i\hbar (\hat L_x \hat L_y + \hat L_y \hat L_x) + i\hbar (\hat L_y \hat L_x + \hat L_x \hat L_y) = 0. \]

完美抵消, 不需要任何近似。结论：

\[ \boxed{\ [\hat L^2, \hat L_i] = 0 \quad \text{for } i = x, y, z.\ } \]

这条事实救命。它告诉你, 虽然 \(\hat L_x, \hat L_y, \hat L_z\) 之间互相不让位（不能同时本征化）, 但大小 \(\hat L^2\) 是中立的——它和每个分量都对易, 所以可以和任意一个分量同时本征化。习惯上选 \(\hat L_z\) 当那个"同时本征化" 的分量。这选择是约定俗成（z 轴是球坐标的极轴, 数学上方便）, 物理上 x, y, z 完全等价。

14.3 共同本征态 \(|l, m\rangle\)

设 \(|l, m\rangle\) 是 \(\hat L^2\) 与 \(\hat L_z\) 的共同本征态:

\[ \hat L^2 |l, m\rangle = \lambda |l, m\rangle, \qquad \hat L_z |l, m\rangle = \mu |l, m\rangle. \]

本征值 \(\lambda, \mu\) 我们暂时不知道, 等会儿用代数定。这一节我先用直觉给一个写法：

\[ \hat L^2 |l, m\rangle = \hbar^2 l(l+1) |l, m\rangle, \qquad \hat L_z |l, m\rangle = \hbar m |l, m\rangle. \]

\(l, m\) 是无量纲数（\(\hbar\) 已经把量纲分离）。\(\hbar^2\) 出在 \(\hat L^2\) 是因为 \(\hat L\) 本身有量纲 \(\hbar\)。\(l(l+1)\) 看起来不自然, 但 14.5-14.6 节会让你看到它必然如此——是阶梯结构强制的。

当下接受这个写法。\(l\) 标记哪一个 \(\hat L^2\) 本征值（哪个"角动量大小"）, \(m\) 标记该 \(l\) 内 \(\hat L_z\) 哪个本征值。两个一起 = 完整标识一个本征态。

请你停下来比较一下这件事和 ch7 谐振子。在谐振子里 \(\hat H\) 一个算子就够了——本征态用一个量子数 \(n\) 标记, \(E_n = \hbar\omega(n + 1/2)\)。这里我们要两个 量子数 \(l, m\) 才能完整标记角动量本征态——因为我们的可观测量是两个互相对易的算子 \(\hat L^2, \hat L_z\) 而非一个。这是 QM 的一般情形：完整描述一个量子系统需要"极大互相对易算子集"（CSCO）。本章里这个集是 \(\{\hat L^2, \hat L_z\}\)。

14.4 升降算子 \(\hat L_\pm\)

角动量代数最漂亮的一步——和谐振子完全平行。定义

\[ \hat L_+ := \hat L_x + i \hat L_y, \qquad \hat L_- := \hat L_x - i \hat L_y. \]

注意 \(\hat L_\pm\) 不是 自伴算子（\(\hat L_+\) 的伴是 \(\hat L_-\)）。它们的对易关系：

\[ [\hat L_z, \hat L_+] = [\hat L_z, \hat L_x] + i [\hat L_z, \hat L_y] = i\hbar \hat L_y + i (-i\hbar \hat L_x) = i\hbar \hat L_y + \hbar \hat L_x = \hbar (\hat L_x + i \hat L_y) = \hbar \hat L_+. \]

同样 \([\hat L_z, \hat L_-] = -\hbar \hat L_-\)。合起来:

\[ \boxed{\ [\hat L_z, \hat L_\pm] = \pm \hbar \hat L_\pm.\ } \]

另一条：

\[ [\hat L^2, \hat L_\pm] = [\hat L^2, \hat L_x] \pm i [\hat L^2, \hat L_y] = 0 \pm 0 = 0. \]

所以 \(\hat L_\pm\) 与 \(\hat L^2\) 对易。

第三条乘积：

\[ \hat L_+ \hat L_- = (\hat L_x + i \hat L_y)(\hat L_x - i \hat L_y) = \hat L_x^2 + \hat L_y^2 + i(\hat L_y \hat L_x - \hat L_x \hat L_y) = \hat L_x^2 + \hat L_y^2 + i [\hat L_y, \hat L_x] = \hat L_x^2 + \hat L_y^2 + \hbar \hat L_z. \]

所以 \(\hat L_x^2 + \hat L_y^2 = \hat L^2 - \hat L_z^2\), 整理

\[ \hat L_+ \hat L_- = \hat L^2 - \hat L_z^2 + \hbar \hat L_z, \qquad \hat L_- \hat L_+ = \hat L^2 - \hat L_z^2 - \hbar \hat L_z. \]

这两式 14.6 节用来卡 \(m\) 的范围。

14.5 阶梯结构

设 \(|l, m\rangle\) 是共同本征态。看 \(\hat L_+ |l, m\rangle\) 是什么。

第一, \(\hat L^2 (\hat L_+ |l, m\rangle) = \hat L_+ \hat L^2 |l, m\rangle = \hbar^2 l(l+1) (\hat L_+ |l, m\rangle)\)（因 \([\hat L^2, \hat L_+] = 0\)）。所以 \(\hat L_+ |l, m\rangle\) 还是 \(\hat L^2\) 的本征态, 本征值不变。\(\hat L_+\) 不改 \(l\)。

第二, \(\hat L_z (\hat L_+ |l, m\rangle)\)。用 \(\hat L_z \hat L_+ = \hat L_+ \hat L_z + [\hat L_z, \hat L_+] = \hat L_+ \hat L_z + \hbar \hat L_+\):

\[ \hat L_z (\hat L_+ |l, m\rangle) = \hat L_+ (\hat L_z + \hbar) |l, m\rangle = \hat L_+ (\hbar m + \hbar) |l, m\rangle = \hbar (m + 1) (\hat L_+ |l, m\rangle). \]

结论: \(\hat L_+ |l, m\rangle\) 是 \(\hat L_z\) 的本征态, 本征值 \(\hbar(m+1)\)——\(m\) 加 1。

所以 \(\hat L_+\) 把 \(|l, m\rangle\) 送到（正比于）\(|l, m+1\rangle\)。同理 \(\hat L_-\) 把 \(|l, m\rangle\) 送到 \(|l, m-1\rangle\)。这正是它们的名字"升降算子" 的来源——升降的是 \(m\)。

把图画出来：固定 \(l\), \(m\) 在某条水平线上整数排开（\(\ldots, -1, 0, 1, 2, \ldots\)）, \(\hat L_+\) 把状态向右移一格, \(\hat L_-\) 向左移一格。这就是阶梯。

这条阶梯不能无限。原因：\(\hat L_x^2 + \hat L_y^2 = \hat L^2 - \hat L_z^2\)。两边在 \(|l, m\rangle\) 上取期望:

\[ \langle \hat L_x^2 + \hat L_y^2 \rangle_{l, m} = \hbar^2 l(l+1) - \hbar^2 m^2 = \hbar^2 [l(l+1) - m^2]. \]

左边是非负（每项是自伴算子的平方, \(\langle \hat L_x^2 \rangle = \|\hat L_x |l,m\rangle\|^2 \ge 0\)）。所以

\[ m^2 \le l(l+1). \]

记 \(m_{\max}, m_{\min}\) 是 \(m\) 的最大、最小允许值。在 \(|l, m_{\max}\rangle\) 上 \(\hat L_+\) 必须打成 0（否则会得 \(m_{\max}+1\), 违反上界）：

\[ \hat L_+ |l, m_{\max}\rangle = 0. \]

同理

\[ \hat L_- |l, m_{\min}\rangle = 0. \]

14.6 整数 \(l\) + \(-l \le m \le l\)

用 14.4 节的乘积式 \(\hat L_- \hat L_+ = \hat L^2 - \hat L_z^2 - \hbar \hat L_z\)。在 \(|l, m_{\max}\rangle\) 上：

\[ 0 = \hat L_- \hat L_+ |l, m_{\max}\rangle = (\hat L^2 - \hat L_z^2 - \hbar \hat L_z) |l, m_{\max}\rangle = [\hbar^2 l(l+1) - \hbar^2 m_{\max}^2 - \hbar^2 m_{\max}] |l, m_{\max}\rangle. \]

所以

\[ l(l+1) = m_{\max}^2 + m_{\max} = m_{\max}(m_{\max} + 1). \]

这在 \(m_{\max} \ge 0\) 时唯一解 \(m_{\max} = l\)。

同样 \(\hat L_+ \hat L_- = \hat L^2 - \hat L_z^2 + \hbar \hat L_z\) 作用在 \(|l, m_{\min}\rangle\)：

\[ l(l+1) = m_{\min}^2 - m_{\min} = m_{\min}(m_{\min} - 1). \]

这在 \(m_{\min} \le 0\) 时唯一解 \(m_{\min} = -l\)。

所以 \(m\) 范围是 \(\{-l, -l+1, \ldots, l-1, l\}\)——一共 \(2l + 1\) 个值。\(m_{\max}, m_{\min}\) 之间用 \(\hat L_+\) 一步步爬, 步长 1, 总共爬 \(m_{\max} - m_{\min} = 2l\) 步。但 \(2l\) 必须是整数（步数嘛）, 所以 \(l\) 是半整数: \(l = 0, 1/2, 1, 3/2, 2, \ldots\)。

对纯轨道角动量 \(\boldsymbol L = \boldsymbol r \times \boldsymbol p\), 还要求波函数单值（绕 \(\phi\) 一圈回原, \(\psi(\phi + 2\pi) = \psi(\phi)\)）。这条边界条件强制 \(m\) 是整数, 从而 \(l\) 也是整数: \(l = 0, 1, 2, 3, \ldots\)。ch15 解球谐函数时会显式看到这条。

但 QM 还有一种内部 角动量——自旋（链ch17）, 它没有 经典对应, 没有 \(\boldsymbol r \times \boldsymbol p\) 的波函数限制, 可以取半整数 \(l = 1/2, 3/2, \ldots\)。电子自旋 1/2 就是这样。本章 + 下两章只讨论轨道角动量, 所以 \(l\) 是整数。

结果汇总:

\[ \boxed{\ \hat L^2 |l, m\rangle = \hbar^2 l(l+1) |l, m\rangle, \quad \hat L_z |l, m\rangle = \hbar m |l, m\rangle, \quad l = 0, 1, 2, \ldots; \ m = -l, -l+1, \ldots, l.\ } \]

请你停下来欣赏一下这条结果——它完全是代数得出。我们没解任何 PDE, 没找任何具体波函数, 只用对易子 + Hilbert 空间的非负性。但每个角动量本征值必然形如 \(\hbar^2 l(l+1)\), 每个 \(\hat L_z\) 本征值必然形如 \(\hbar m\)（\(|m| \le l\)）。这就是 Lie 代数表示论的力量——结构本身决定谱。

14.7 不确定性的具体形态

把 14.1 节的不等式

\[ \Delta L_x \cdot \Delta L_y \ge \frac{\hbar}{2} |\langle \hat L_z \rangle| \]

在 \(|l, m\rangle\) 上展开。这态上 \(\langle \hat L_z \rangle = \hbar m\)。所以

\[ \Delta L_x \cdot \Delta L_y \ge \frac{\hbar^2 |m|}{2}. \]

具体计算 \(\Delta L_x, \Delta L_y\)。由对称（这态对 \(L_x, L_y\) 对称——做绕 \(z\) 轴 \(\pi/2\) 旋转把 \(L_x \leftrightarrow L_y\)）, \(\langle \hat L_x \rangle = \langle \hat L_y \rangle = 0\), \(\langle \hat L_x^2 \rangle = \langle \hat L_y^2 \rangle\)。由 \(\hat L_x^2 + \hat L_y^2 = \hat L^2 - \hat L_z^2\):

\[ \langle \hat L_x^2 \rangle = \langle \hat L_y^2 \rangle = \frac{1}{2} \hbar^2 [l(l+1) - m^2]. \]

\(\Delta L_x = \Delta L_y = \hbar \sqrt{[l(l+1) - m^2]/2}\)。乘积

\[ \Delta L_x \cdot \Delta L_y = \frac{\hbar^2}{2} [l(l+1) - m^2] \ge \frac{\hbar^2 |m|}{2}. \]

等价于 \(l(l+1) - m^2 \ge |m|\), 即 \(l(l+1) \ge m^2 + |m| = |m|(|m|+1)\), 这就是 \(l \ge |m|\)（已成立）。等号在 \(|m| = l\) 时取到——极限 状态 \(|l, l\rangle\) 让 \(\hat L_x, \hat L_y\) 不确定性最小（但仍非零）。

请你想象这件事的几何画面: 取 \(l = 1\), 三个 \(m\) 值 \(\pm 1, 0\)。经典 角动量矢量长度 \(\sqrt{l(l+1)} \hbar = \sqrt 2 \hbar\), 沿 \(z\) 投影 \(\hbar m\) = \(\hbar, 0, -\hbar\)。把大小 \(\sqrt 2 \hbar\) 与z 投影 0, \(\pm \hbar\) 配对——经典上一根定向矢量配 \(z\) 投影 \(\hbar\) 应该让 \(L_x^2 + L_y^2 = (\sqrt 2 \hbar)^2 - \hbar^2 = \hbar^2\), 所以 \(L_\perp = \hbar\), 矢量倾斜 \(\arctan(\hbar/\hbar) = 45°\)。但 QM 里 \(L_x, L_y\) 不能同时定——只能说"它在 \((L_x, L_y)\) 平面里转", 模糊地。这就是"角动量在锥面上" 的标准画法（Sakurai, Griffiths 都画过）: 矢量大小固定 \(\sqrt 2 \hbar\), 沿 \(z\) 投影 \(\hbar\), 但绕 \(z\) 的方向角任意——画成一个锥。

14.8 通向球谐函数 \(Y_l^m(\theta, \phi)\)

本章纯代数推完了——\(l, m\) 量子数 + 谱结构。但具体的本征函数是什么样？\(|l, m\rangle\) 在位置表象 \(\langle \theta, \phi | l, m\rangle\)（球坐标角部分）= ?

这是ch15 的内容。简要预告：在球坐标 \((r, \theta, \phi)\) 里 \(\hat L^2, \hat L_z\) 的微分形式是

\[ \hat L_z = -i\hbar \frac{\partial}{\partial \phi}, \qquad \hat L^2 = -\hbar^2 \left[ \frac{1}{\sin\theta} \frac{\partial}{\partial \theta}\left( \sin\theta \frac{\partial}{\partial \theta}\right) + \frac{1}{\sin^2\theta} \frac{\partial^2}{\partial \phi^2}\right]. \]

这两个的共同本征函数就是球谐函数 \(Y_l^m(\theta, \phi)\)：

\[ \hat L^2 Y_l^m = \hbar^2 l(l+1) Y_l^m, \qquad \hat L_z Y_l^m = \hbar m Y_l^m. \]

\(Y_l^m\) 是定义 在 2-球面 \(S^2\) 上的复值函数, 形如 \(Y_l^m(\theta, \phi) = N_{lm} P_l^m(\cos\theta) e^{im\phi}\), \(P_l^m\) 是缔合 Legendre 函数。前几个：

\[ Y_0^0 = \frac{1}{\sqrt{4\pi}}, \quad Y_1^0 = \sqrt{\frac{3}{4\pi}} \cos\theta, \quad Y_1^{\pm 1} = \mp \sqrt{\frac{3}{8\pi}} \sin\theta \, e^{\pm i\phi}. \]

等等。\(Y_l^m\) 在 \(L^2(S^2)\) 上构成完备正交基（链回泛函分析 Hilbert 空间）。ch15 把这些球谐函数显式写出来 + 解释它们在化学里的意义（s/p/d 轨道）。

14.9 临别话

这一章你看到一件事: 对易关系决定谱。三条 \([\hat L_i, \hat L_j] = i\hbar \epsilon_{ijk} \hat L_k\) + Hilbert 空间的非负性, 一起强迫角动量量子数 \(l, m\) 取离散值。完全没用具体波函数。

这种"纯代数" 风格不是孤立——它是整个 QM 的范式。谐振子（ch7）的能级 \(E_n = \hbar\omega(n + 1/2)\) 也是用 \([\hat a, \hat a^\dagger] = 1\) 一条对易关系加非负性推出的。ch17 自旋 1/2 也是用 \([\hat S_i, \hat S_j] = i\hbar \epsilon_{ijk} \hat S_k\) 一条推出 Pauli 矩阵。这种"结构 → 谱" 的逻辑在量子力学中无处不在, 是 Lie 代数表示论在物理上的具体化。

下一章正式打开球对称势。3D Schrödinger 方程在 \(V = V(r)\) 时分离变量——\(\psi(r, \theta, \phi) = R(r) Y_l^m(\theta, \phi)\)。\(Y_l^m\) 是这一章代数推出的本征函数, 与势无关; 径向 \(R(r)\) 满足"径向 SE", 是 1D 问题。这种分离让 3D 问题归约到 1D, 是后两章氢原子完整解析解的桥梁。
