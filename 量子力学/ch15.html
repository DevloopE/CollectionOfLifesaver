第 15 章 球对称势 + 球谐函数

请你想一个具体问题。一个粒子在 3D 空间, 受到一个力。力的大小只跟距离原点的远近 \(r\) 有关——离得越近力越强（或越弱）, 与方向无关。这种势叫球对称势 \(V = V(r)\)。

例子充满物理学：地球绕太阳——\(V = -GMm/r\)（Newton 引力）, 球对称。氢原子电子绕质子——\(V = -e^2/(4\pi\epsilon_0 r)\)（Coulomb 引力）, 球对称。3D 谐振子 \(V = \frac{1}{2} k r^2\), 球对称。3D 无限球势阱 \(V = 0\) for \(r < a\), \(\infty\) for \(r > a\), 球对称。整个 原子物理 + 行星天文都建立在球对称势上——因为粒子之间的基本相互作用（引力、Coulomb 力）都是球对称的。

3D Schrödinger 方程

\[ -\frac{\hbar^2}{2m} \nabla^2 \psi + V(\boldsymbol r) \psi = E \psi \]

对一般 \(V(\boldsymbol r)\) 是 PDE in 三个变量 (\(x, y, z\)), 解起来恶心。但当 \(V = V(r)\) 球对称时, 我们能分离变量: 假设 \(\psi(r, \theta, \phi) = R(r) Y(\theta, \phi)\), Schrödinger 方程裂成 一个径向常微分方程 + 一个角向"球谐" 方程。3D 问题归约到 1D。

而且角向部分已经被解过了——就是ch14 的 \(Y_l^m(\theta, \phi)\), 与势 \(V(r)\) 无关。所以解球对称势的 SE 永远先把球谐函数"套上", 剩下径向 \(R(r)\) 满足一个看起来像 1D SE 的方程, 多了一个离心势项 \(\hbar^2 l(l+1)/(2 m r^2)\)。本章要走完这条分离变量, 把球谐函数显式写出来, 然后用两个例子（无限球势阱 + 3D 谐振子）兑现"归约到 1D"。

15.1 球坐标下的 \(\nabla^2\)

3D Laplacian 在直角坐标 \(\nabla^2 = \partial_x^2 + \partial_y^2 + \partial_z^2\)。换到球坐标 \((r, \theta, \phi)\):

  • \(r\)：到原点距离, \(r \ge 0\)
  • \(\theta\)：与 \(z\) 轴夹角, \(0 \le \theta \le \pi\)
  • \(\phi\)：在 \(xy\) 平面绕 \(z\) 轴方位角, \(0 \le \phi < 2\pi\)

变换关系 \(x = r\sin\theta\cos\phi, y = r\sin\theta\sin\phi, z = r\cos\theta\)。

用链式法则把 \(\nabla^2\) 化到 \((r, \theta, \phi)\)（推导烦琐, 此处引用 多元微积分课的标准结果）：

\[ \boxed{\ \nabla^2 = \frac{1}{r^2} \frac{\partial}{\partial r}\left( r^2 \frac{\partial}{\partial r}\right) + \frac{1}{r^2 \sin\theta} \frac{\partial}{\partial \theta}\left( \sin\theta \frac{\partial}{\partial \theta}\right) + \frac{1}{r^2 \sin^2\theta} \frac{\partial^2}{\partial \phi^2}.\ } \]

请你停下来盯住这个表达式的结构。径向部分 \(r^{-2} \partial_r (r^2 \partial_r)\) 只含 \(r\)。角向部分（中间 + 后面两项）只含 \(\theta, \phi\), 而且都带 \(1/r^2\)。把角向部分括起来写

\[ \nabla^2 = \frac{1}{r^2} \frac{\partial}{\partial r}\left( r^2 \frac{\partial}{\partial r}\right) - \frac{1}{r^2 \hbar^2} \hat L^2, \]

其中 \(\hat L^2\) 是ch14 14.8 节给出的角动量平方算子的微分形式

\[ \hat L^2 = -\hbar^2 \left[ \frac{1}{\sin\theta} \frac{\partial}{\partial \theta}\left( \sin\theta \frac{\partial}{\partial \theta}\right) + \frac{1}{\sin^2\theta} \frac{\partial^2}{\partial \phi^2}\right]. \]

这条等式不是巧合——\(\hat L^2\) 本来就是位置空间的 "角动量大小平方" 算子, 它就是 \(\nabla^2\) 的角向部分（乘个 \(-\hbar^2\)）。这说明 3D 动能算子 \(-\frac{\hbar^2}{2m} \nabla^2\) 自然分成"径向动能 + 离心势"两部分, 其中离心势就是 \(\hat L^2/(2 m r^2)\)。

15.2 分离变量假设

3D 时间无关 SE

\[ \hat H \psi = \left[-\frac{\hbar^2}{2m} \nabla^2 + V(r)\right] \psi = E \psi. \]

代入分离形式 \(\psi(r, \theta, \phi) = R(r) Y(\theta, \phi)\):

\[ -\frac{\hbar^2}{2m} \left[\frac{Y}{r^2} \frac{d}{dr}\left(r^2 \frac{dR}{dr}\right) - \frac{R}{r^2 \hbar^2} \hat L^2 Y\right] + V(r) R Y = E R Y. \]

两边除以 \(R Y / r^2\), 并乘 \(-2m/\hbar^2\):

\[ \frac{1}{R} \frac{d}{dr}\left(r^2 \frac{dR}{dr}\right) - \frac{2 m r^2}{\hbar^2} [V(r) - E] = \frac{1}{\hbar^2 Y} \hat L^2 Y. \]

左边只含 \(r\), 右边只含 \(\theta, \phi\)。两边相等只能都等于一个常数。约定写这常数为 \(l(l+1)\)（事先知道答案——14.6 节的 \(l\)）。所以分成两条:

角向方程：

\[ \hat L^2 Y(\theta, \phi) = \hbar^2 l(l+1) Y(\theta, \phi). \]

径向方程：

\[ \frac{1}{r^2} \frac{d}{dr}\left(r^2 \frac{dR}{dr}\right) + \left[\frac{2m}{\hbar^2} (E - V(r)) - \frac{l(l+1)}{r^2}\right] R = 0. \]

两条方程互相独立: 角向方程不 含 \(V\)（与势无关）, 径向方程不 含 \(\theta, \phi\)（与方向无关）。这是球对称势的结构性礼物——一旦 \(V\) 球对称, 角向部分永远长一样。

15.3 角向解 = 球谐函数 \(Y_l^m\)

角向方程 \(\hat L^2 Y = \hbar^2 l(l+1) Y\) 已经在ch14 用对易代数 解过——本征值 \(\hbar^2 l(l+1)\) for \(l = 0, 1, 2, \ldots\), 每个 \(l\) 对应 \(2l+1\) 个本征态, 标记 \(m = -l, \ldots, l\)。

本章告诉你这些本征函数的显式形式。继续分离 \(Y(\theta, \phi) = \Theta(\theta) \Phi(\phi)\), 代入 \(\hat L^2 Y = \hbar^2 l(l+1) Y\), 类似拆开得：

\(\Phi\) 满足 \(\Phi'' = -m^2 \Phi\), 解 \(\Phi(\phi) = e^{i m \phi}\)。\(2\pi\)-周期性强迫 \(m\) 整数。

\(\Theta\) 满足缔合 Legendre 方程 \(\sin\theta (\sin\theta \Theta')' + [l(l+1) \sin^2\theta - m^2] \Theta = 0\)。换 \(u = \cos\theta\), 标准缔合 Legendre 方程, 解是缔合 Legendre 函数 \(P_l^m(u)\)（链微分方程第 13 章特殊函数）。\(P_l^m\) 是 \((1 - u^2)^{m/2}\) 乘以 \(l\) 次多项式, 在 \(u = \pm 1\) （即 \(\theta = 0, \pi\)）有限。

合起来:

\[ \boxed{\ Y_l^m(\theta, \phi) = N_{lm} \, P_l^m(\cos\theta) \, e^{i m \phi},\ } \]

其中 \(N_{lm}\) 是归一化常数

\[ N_{lm} = (-1)^m \sqrt{\frac{(2l + 1)}{4\pi} \cdot \frac{(l - m)!}{(l + m)!}}, \]

使得 \(\int_0^{2\pi} \int_0^\pi |Y_l^m|^2 \sin\theta \, d\theta \, d\phi = 1\)。

这是球谐函数。它们在 \(L^2(S^2)\) 上构成正交完备基:

\[ \int Y_l^{m \, *} Y_{l'}^{m'} \, d\Omega = \delta_{ll'} \delta_{mm'}, \]

\(d\Omega = \sin\theta \, d\theta \, d\phi\) 是球面立体角元素。

15.4 球谐函数前几个 + 物理可视化

\(l = 0\)（s 轨道）：

\[ Y_0^0 = \frac{1}{\sqrt{4\pi}}. \]

常数, 球对称——s 轨道没有 角向偏好, 像一个均匀球。氢原子 1s, 2s, 3s 轨道角向部分都是它。

\(l = 1\)（p 轨道）：

\[ Y_1^0 = \sqrt{\frac{3}{4\pi}} \cos\theta, \quad Y_1^{\pm 1} = \mp \sqrt{\frac{3}{8\pi}} \sin\theta \, e^{\pm i\phi}. \]

\(Y_1^0\) 沿 \(z\) 轴有"双瓣"——上半 \(\cos\theta > 0\), 下半 \(\cos\theta < 0\), 在赤道是节面。化学家叫它 \(p_z\) 轨道。

\(Y_1^{\pm 1}\) 是复值。化学里常把它们组合成实值:

\[ p_x = \frac{1}{\sqrt 2}(Y_1^{-1} - Y_1^1) \propto \sin\theta \cos\phi = \frac{x}{r}, \quad p_y = \frac{i}{\sqrt 2}(Y_1^{-1} + Y_1^1) \propto \sin\theta \sin\phi = \frac{y}{r}. \]

\(p_x, p_y, p_z\) 在 x, y, z 轴上分别有"双瓣"，三两正交。三个 \(l = 1\) 状态 = 三个互相垂直的 p 轨道。

\(l = 2\)（d 轨道）：5 个状态, 标准化学化为 \(d_{z^2}, d_{xz}, d_{yz}, d_{x^2-y^2}, d_{xy}\)。形状更复杂——四瓣（\(d_{xy}, d_{x^2-y^2}\) 等）或者"甜甜圈 + 哑铃"（\(d_{z^2}\)）。这些是过渡金属化学的核心。

请你画一遍这些图（任何 QM 教材都有）。\(l\) 越大, 角向越复杂——\(l\) 表示"有多少节面"（\(l\) 个）。物理直觉: \(l\) 越大, 离心势 \(\hbar^2 l(l+1)/(2 m r^2)\) 越强, 粒子被推离原点更厉害, 角向"波纹" 越多。

15.5 径向 SE

回到分离出来的径向方程:

\[ \frac{1}{r^2} \frac{d}{dr}\left(r^2 \frac{dR}{dr}\right) + \left[\frac{2m}{\hbar^2} (E - V(r)) - \frac{l(l+1)}{r^2}\right] R = 0. \]

这看起来还不像 1D SE。技巧——换变量 \(u(r) := r R(r)\), 即 \(R = u/r\)。算 \((1/r^2)(r^2 R')' = (R + r R')'/r = (R + r R')'/r\), 慢慢推一推（或者直接验证）:

\[ \frac{1}{r^2} \frac{d}{dr}\left(r^2 \frac{dR}{dr}\right) = \frac{1}{r} \frac{d^2 (rR)}{d r^2} = \frac{1}{r} \frac{d^2 u}{dr^2}. \]

代回, 两边乘 \(r\):

\[ \frac{d^2 u}{dr^2} + \left[\frac{2m}{\hbar^2}(E - V(r)) - \frac{l(l+1)}{r^2}\right] u = 0. \]

整理

\[ \boxed{\ -\frac{\hbar^2}{2 m} \frac{d^2 u}{dr^2} + V_{\text{eff}}(r) \, u = E u, \quad V_{\text{eff}}(r) := V(r) + \frac{\hbar^2 l(l+1)}{2 m r^2}.\ } \]

这就是 1D 时间无关 Schrödinger 方程! 形式与ch6-8 完全一样, 只是势变成有效势 \(V_{\text{eff}}\)。径向问题归约到熟悉的 1D 谱问题。

归一化条件: \(\int |\psi|^2 d^3 r = \int_0^\infty |R|^2 r^2 \, dr \cdot \int |Y|^2 d\Omega = \int_0^\infty |R|^2 r^2 \, dr = \int_0^\infty |u|^2 \, dr = 1\)。所以在 \(u\) 这一面上归一化和 1D SE 完全一样。

边界条件:

  • \(r \to \infty\): \(u \to 0\)（束缚态）
  • \(r \to 0\): \(u(0) = 0\)（否则 \(R = u/r \to \infty\), 物理上波函数发散; 严格来说 \(\nabla^2 (1/r) = -4\pi \delta^3(\boldsymbol r)\), 在原点产生 \(\delta\) 源, 除非 \(V\) 也有 \(\delta\), 否则 SE 在原点不成立）

\(u(0) = 0\) 这条边界条件让径向问题等价于 一个"定义在 \(r > 0\) 的 1D SE, 在 \(r = 0\) 处反射壁"。ch6 1D 无限深势阱的所有技术直接搬过来。

15.6 离心势的物理意义

有效势

\[ V_{\text{eff}} = V(r) + \frac{\hbar^2 l(l+1)}{2 m r^2}. \]

多出来的项 \(\hbar^2 l(l+1)/(2 m r^2)\) 叫离心势（centrifugal barrier）。它正, 在 \(r \to 0\) 时发散到 \(+\infty\), 在 \(r \to \infty\) 时趋 0。

经典对应: 一颗有角动量 \(L\) 的粒子绕中心运动, 角动量守恒强迫粒子不能径直撞向中心——它必须保持一个最小距离（"近心点"）。在径向问题里这表现为有效势 \(L^2 / (2 m r^2)\)（写过经典力学的人马上就认出来——这就是有效势能 法处理中心力问题的标准技术）。量子里 \(L^2 \to \hbar^2 l(l+1)\), 完全平行。

物理后果：

  • \(l = 0\)（s 态）: 没有离心势, 波函数可以 在原点取非零值（事实上 \(R(0) \neq 0\), \(u(0) = 0\) 但 \(u'(0) \neq 0\)）。\(|\psi|^2\) 在 \(r = 0\) 处有限非零——电子有一定概率"在 核里"。这条对超精细分裂、Mössbauer 效应都重要。
  • \(l \ge 1\)（p, d, ... 态）: 离心势把波函数推离 原点。在小 \(r\) 处 \(V_{\text{eff}} \approx \hbar^2 l(l+1)/(2 m r^2)\) 主导, 解 \(u \sim r^{l+1}\)（推导：把 \(u = r^\alpha\) 代进, 平衡 \(u'' \sim \alpha(\alpha-1) r^{\alpha-2}\) 与 \(l(l+1) u/r^2 \sim l(l+1) r^{\alpha-2}\), 得 \(\alpha(\alpha-1) = l(l+1)\), 即 \(\alpha = l+1\) 或 \(\alpha = -l\)；后者发散舍去）。

所以 \(R(r) = u/r \sim r^l\) 在原点附近——\(l\) 越大, 波函数被"挤走" 越厉害。p 电子在原点附近有 \(R \sim r\), d 电子有 \(R \sim r^2\), 等等。

15.7 应用一: 3D 无限球势阱

具体物理问题——粒子被困在半径 \(a\) 的球内

\[ V(r) = \begin{cases} 0, & r < a, \\ +\infty, & r \ge a. \end{cases} \]

这就像ch6 的 1D 无限深势阱的 3D 版。

径向方程在 \(r < a\): \(V = 0\), 所以

\[ \frac{d^2 u}{dr^2} + \left[k^2 - \frac{l(l+1)}{r^2}\right] u = 0, \qquad k = \sqrt{2 m E}/\hbar. \]

这是球 Bessel 方程。解 \(u(r) = A r j_l(kr) + B r y_l(kr)\), \(j_l, y_l\) 是球 Bessel 函数。\(y_l\) 在 \(r = 0\) 处发散, 边界 \(u(0) = 0\) 强迫 \(B = 0\)。所以

\[ u(r) = A r j_l(kr), \quad R(r) = A j_l(kr). \]

边界 \(R(a) = 0\)（壁外波函数 0, 连续性）：\(j_l(ka) = 0\)。设 \(\beta_{nl}\) 是 \(j_l\) 的第 \(n\) 个正零点（\(n = 1, 2, 3, \ldots\)）, 则 \(k a = \beta_{nl}\), 即

\[ \boxed{\ E_{nl} = \frac{\hbar^2 \beta_{nl}^2}{2 m a^2}.\ } \]

具体数：

  • \(l = 0\): \(j_0(x) = \sin x / x\), 零点 \(\beta_{n0} = n\pi\), 所以 \(E_{n0} = \hbar^2 n^2 \pi^2 / (2 m a^2)\)。这就是 1D 无限深势阱的能级——意外吗？不意外: \(j_0\) 是 \(\sin/x\), 径向 \(R = \sin(k r)/r\), 等价于 1D 上一个反射壁的 \(\sin\)。
  • \(l = 1\): \(j_1(x) = \sin x / x^2 - \cos x / x\), 第一个零点 \(\beta_{11} \approx 4.49\)。\(E_{11} \approx 4.49^2/\pi^2 \approx 2.05\) 倍 \(l = 0\) 基态能量。
  • \(l = 2\): \(\beta_{12} \approx 5.76\), \(E_{12}/E_{10} \approx 5.76^2/\pi^2 \approx 3.36\)。

这给出的能级谱不是简单的 \(n^2\)（不像 1D 势阱）, 也不是简单的 \(n + l\)（不像 3D 谐振子）, 而是不规则的——每个 \(l\) 一组零点, 互相穿插。

简并度: 每个 \((n, l)\) 能级有 \(2l + 1\) 个 \(m\) 子态——简并度 \(2l + 1\)（不计自旋）。

15.8 应用二: 3D 各向同性谐振子

\(V(r) = \frac{1}{2} m\omega^2 r^2 = \frac{1}{2} m\omega^2 (x^2 + y^2 + z^2)\)。球对称, 所以本章方法可以做。但同样 这个势在直角坐标下分离变量 也很简单——\(\hat H = \hat H_x + \hat H_y + \hat H_z\), 三个独立 1D 谐振子（链ch7）, 能级

\[ E_{n_x n_y n_z} = \hbar\omega(n_x + n_y + n_z + 3/2), \quad n_x, n_y, n_z = 0, 1, 2, \ldots \]

记 \(N := n_x + n_y + n_z\), 总能 \(E_N = \hbar\omega(N + 3/2)\), \(N = 0, 1, 2, \ldots\)。

简并度（多少种 \((n_x, n_y, n_z)\) 让 \(n_x + n_y + n_z = N\)）= \(\binom{N+2}{2} = (N+1)(N+2)/2\)。

用球对称分离: 径向方程

\[ -\frac{\hbar^2}{2 m} \frac{d^2 u}{dr^2} + \left[\frac{1}{2} m \omega^2 r^2 + \frac{\hbar^2 l(l+1)}{2 m r^2}\right] u = E u. \]

解（用 Hermite + Laguerre 多项式技术, 类似 ch7）：

\[ E_{n_r, l} = \hbar \omega (2 n_r + l + 3/2), \quad n_r = 0, 1, 2, \ldots \]

\(n_r\) 是径向量子数（径向波函数节点数）。设 \(N := 2 n_r + l\), 则 \(E = \hbar\omega(N + 3/2)\)——和直角坐标的 \(N = n_x + n_y + n_z\) 完全吻合! 但\(N\) 的分解方式不同：

\(N\)直角分解 \((n_x, n_y, n_z)\)球分解 \((n_r, l)\)简并
0(0,0,0): 1 种(0,0): 1×(2·0+1)=11
1(1,0,0)+排列: 3 种(0,1): 1×(2·1+1)=33
2(2,0,0)+(1,1,0)+排列: 6 种(0,2)+(1,0): 5+1=66
3(3,0,0)+(2,1,0)+(1,1,1)+...: 10 种(0,3)+(1,1): 7+3=1010

两套基都张成同一个 \(N\) 能级的本征空间, 但用不同的对易算子集。直角坐标用 \(\{\hat H_x, \hat H_y, \hat H_z\}\)（互相对易）, 球坐标用 \(\{\hat H, \hat L^2, \hat L_z\}\)（互相对易）。同一个空间的两组正交基。

请你停下来欣赏一下这件事。3D 谐振子同时 有直角对称（\(SO(3)\) 之外还有更多对称——实际上是 \(SU(3)\)）和球对称。这种"意外" 简并是 \(SO(3)\) 之上还有更大对称群的标志。氢原子也有类似的意外 简并（链ch16）, 那里是 \(SO(4)\) 对称。这条话题超出本书范围, 但它告诉你: 简并度的"意外多" 永远暗示一个更大的隐藏对称群。

15.9 临别话

这一章你看到三件事的合奏:

  1. 球对称势的 SE 自动分离变量——\(\psi = R(r) Y(\theta, \phi)\)。
  2. 角向部分与势无关——永远是球谐函数 \(Y_l^m\)。这条让所有"原子物理" 共享同一组角向波函数。
  3. 径向问题归约到 1D SE——加了一个有效离心势 \(\hbar^2 l(l+1)/(2 m r^2)\)。1D 的全部技术（势阱、谐振子、Schrödinger 解析技巧）直接搬过来。

下一章把这套机器开到最重要的 球对称势—— Coulomb \(V = -e^2/(4\pi\epsilon_0 r)\), 即氢原子。径向方程会变成 Laguerre 方程, 解给 Laguerre 多项式 \(L_n^l\), 离散能级 \(E_n = -13.6 \text{ eV}/n^2\)（解释 ch1 Bohr 公式的来源）, 简并度 \(2 n^2\)。这是 QM 最漂亮的解析结果, 也是元素周期表的第一原理基础。
