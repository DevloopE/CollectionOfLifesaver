第 20 章 时间依赖微扰与 Fermi 金律

上一章微扰论处理静态 修正——电场恒定, 磁场不变, 加进 Hamiltonian 后能级位移。这条不 涉及"时间"——\(\hat H\) 与 \(t\) 无关, 系统总能量守恒, 粒子停在能量本征态里不动。

但物理界的大部分有趣过程都涉及 时间。原子吸收一个光子, 从基态跃迁 到激发态; 激发态自发 发射光子回基态; 加速器把质子撞到靶上, 产生新粒子; 半导体里电子从价带散射 到导带; 核衰变 \({}^{14}\text{C} \to {}^{14}\text{N} + e^- + \bar\nu_e\); 化学反应 \(\text{H}_2 + \text{Cl} \to \text{HCl} + \text{H}\)——所有这些都是"从一个态变成另一个态" 的动力学 过程。

本章的中心问题: 给一个时变扰动 \(\hat H(t) = \hat H_0 + \hat V(t)\), 系统初始在 \(\hat H_0\) 的本征态 \(|i\rangle\)（\(t = 0\)）, 问过了时间 \(t\) 之后, 系统在另一个本征态 \(|f\rangle\) 的概率是多少？特别地, 如果 \(\hat V(t)\) 是周期性的（比如电磁波 \(\hat V \propto e^{-i\omega t}\)）, 跃迁速率（probability per unit time）是多少？

1932 年 Fermi 在他的核物理 讲义里把这个问题的答案写成金律:

\[ \boxed{\ \Gamma_{i \to f} = \frac{2\pi}{\hbar}|\langle f|\hat V|i\rangle|^2 \rho(E_f).\ } \]

跃迁速率 = (\(2\pi/\hbar\)) × (矩阵元模平方) × (末态密度)。Fermi 自己叫它第二金律（"第一金律" 是另一条相关公式, 现在不常用）。这条公式遍布原子物理 / 凝聚态物理 / 核物理 / 高能物理——是真的"金"。

本章从时变 SE 出发, 一步步推到金律, 然后给三个应用: 原子吸收 + 自发发射 + 核衰变。

20.1 时变微扰: 设定

Hamiltonian

\[ \hat H(t) = \hat H_0 + \hat V(t). \]

\(\hat H_0\) 时间无关, 已知本征态 \(\{|n\rangle\}\), 能级 \(\{E_n\}\): \(\hat H_0 |n\rangle = E_n |n\rangle\)。\(\hat V(t)\) 是时间依赖 的微扰, 假设"小"。

系统状态 \(|\Psi(t)\rangle\) 满足时变 Schrödinger 方程

\[ i\hbar \frac{d}{dt}|\Psi(t)\rangle = [\hat H_0 + \hat V(t)]|\Psi(t)\rangle. \]

把 \(|\Psi(t)\rangle\) 在 \(\hat H_0\) 本征基展开, 但抽出 自由演化相位

\[ |\Psi(t)\rangle = \sum_n c_n(t) e^{-i E_n t/\hbar}|n\rangle. \]

这里 \(c_n(t)\) 是时间依赖振幅, 与 \(\hat V\) 演化挂钩; 没有 \(\hat V\) 时 \(c_n\) 是常数（系统停在一个本征态）。

代入 SE:

\[ i\hbar \sum_n \dot c_n e^{-i E_n t/\hbar}|n\rangle + \sum_n c_n E_n e^{-i E_n t/\hbar}|n\rangle = \sum_n c_n e^{-i E_n t/\hbar}(\hat H_0 + \hat V)|n\rangle. \]

右边 \(\hat H_0 |n\rangle = E_n |n\rangle\) 那部分与左边第二项消, 剩下

\[ i\hbar \sum_n \dot c_n e^{-i E_n t/\hbar}|n\rangle = \sum_n c_n e^{-i E_n t/\hbar} \hat V|n\rangle. \]

左乘 \(\langle f|\):

\[ i\hbar \dot c_f e^{-i E_f t/\hbar} = \sum_n c_n e^{-i E_n t/\hbar}\langle f|\hat V|n\rangle. \]

两边乘 \(e^{i E_f t/\hbar}\), 引入跃迁频率 \(\omega_{fn} = (E_f - E_n)/\hbar\):

\[ \boxed{\ i\hbar \dot c_f(t) = \sum_n c_n(t) e^{i\omega_{fn} t}\langle f|\hat V(t)|n\rangle.\ } \]

这是精确 方程组——把无限维 SE 写成 \(c_n(t)\) 的耦合 ODE。但它是等价 的, 没有简化。微扰来自下一步——把 \(c_n\) 按 \(\hat V\) 阶展开。

20.2 一阶概率振幅

设 \(t = 0\) 时系统在初态 \(|i\rangle\):

\[ c_n(0) = \delta_{ni}. \]

零阶: 假设 \(\hat V = 0\), 则 \(c_n(t) = \delta_{ni}\) 不变。

一阶: 把 \(c_n(t) \approx \delta_{ni}\) 代入 ODE 右边（"一阶 Born 近似"——右边只用零阶值）, 得

\[ i\hbar \dot c_f(t) \approx e^{i\omega_{fi} t}\langle f|\hat V(t)|i\rangle, \qquad f \neq i. \]

积分

\[ \boxed{\ c_f^{(1)}(t) = -\frac{i}{\hbar}\int_0^t e^{i\omega_{fi}t'}\langle f|\hat V(t')|i\rangle\, dt'.\ } \]

这条公式是时变微扰论的核心。从初态 \(|i\rangle\) 到末态 \(|f\rangle\) 的概率振幅; 模平方给概率

\[ P_{i \to f}(t) = |c_f^{(1)}(t)|^2. \]

20.3 周期微扰

最重要的特例——周期性 微扰

\[ \hat V(t) = \hat V_0 e^{-i\omega t} + \hat V_0^\dagger e^{+i\omega t}. \]

（实数化: 通常物理微扰是实 函数, 比如 \(\hat V \propto \cos\omega t\), 把它写成两个共轭指数的和。\(\hat V_0\) 时间无关。）

对 \(|i\rangle \to |f\rangle\) 跃迁, 取 \(E_f > E_i\)（吸收能量）, 关心 \(e^{-i\omega t}\) 项（"吸收"; \(e^{+i\omega t}\) 项对应"受激发射", 即末态比初态低）。代入 20.2:

\[ c_f^{(1)}(t) = -\frac{i}{\hbar}\langle f|\hat V_0|i\rangle \int_0^t e^{i(\omega_{fi} - \omega)t'}\, dt' = -\frac{i}{\hbar}\langle f|\hat V_0|i\rangle \cdot \frac{e^{i(\omega_{fi}-\omega)t} - 1}{i(\omega_{fi}-\omega)}. \]

整理

\[ c_f^{(1)}(t) = -\frac{1}{\hbar}\langle f|\hat V_0|i\rangle \cdot e^{i(\omega_{fi}-\omega)t/2}\cdot \frac{2\sin[(\omega_{fi}-\omega)t/2]}{\omega_{fi}-\omega}. \]

模平方

\[ \boxed{\ P_{i\to f}(t) = \frac{|\langle f|\hat V_0|i\rangle|^2}{\hbar^2}\cdot\frac{4 \sin^2[(\omega_{fi}-\omega)t/2]}{(\omega_{fi}-\omega)^2}.\ } \]

分母 \((\omega_{fi}-\omega)^2\)——共振峰。当 \(\omega = \omega_{fi}\), 表面看分母 = 0 公式发散; 但分子 \(\sin^2[...]\) 也 = 0, 用 \(\lim_{x \to 0} \sin^2 x / x^2 = 1\), 得

\[ P_{i\to f}^{\text{res}}(t) = \frac{|\langle f|\hat V_0|i\rangle|^2}{\hbar^2}\cdot t^2. \]

共振时概率随 \(t^2\) 增长。非共振时概率振荡, 不增长。这条"共振 突出, 非共振 振荡" 的形状是 NMR / 激光物理的核心。

20.4 共振峰: \(\sin^2(x)/x^2\) 的极限

把 \(P_{i\to f}(t)\) 看作 \((\omega_{fi}-\omega)\) 的函数。形状是

\[ \frac{\sin^2(\Delta\omega \cdot t/2)}{(\Delta\omega/2)^2}, \qquad \Delta\omega = \omega_{fi} - \omega. \]

峰宽 \(\sim 2\pi/t\)（一阶零点位置）, 峰高 \(\sim t^2\)。峰下面积 与 \(t\) 成正比——这是 \(\delta\) 函数极限的雏形。

具体: 用恒等式

\[ \lim_{t \to \infty} \frac{\sin^2(\alpha t)}{\alpha^2 t} = \pi \delta(\alpha), \]

所以

\[ \lim_{t \to \infty}\frac{1}{t}\cdot\frac{4\sin^2[(\omega_{fi}-\omega)t/2]}{(\omega_{fi}-\omega)^2} = 2\pi \delta(\omega_{fi}-\omega). \]

定义跃迁速率（概率每单位时间）

\[ W_{i\to f} \equiv \lim_{t\to\infty}\frac{P_{i\to f}(t)}{t}, \]

得

\[ W_{i\to f} = \frac{2\pi}{\hbar^2}|\langle f|\hat V_0|i\rangle|^2 \delta(\omega_{fi}-\omega) = \frac{2\pi}{\hbar}|\langle f|\hat V_0|i\rangle|^2 \delta(E_f - E_i - \hbar\omega). \]

\(\delta\) 函数说明了能量守恒: 跃迁只在 \(E_f = E_i + \hbar\omega\)（吸收一个能量为 \(\hbar\omega\) 的"量子"）时发生。这条"必须共振" 的精确叙述。

20.5 连续末态: Fermi 金律

上面 \(\delta\) 函数对离散 末态有点尴尬——它要求精确 共振, 任何小失谐都给 0 概率。实际中末态从不 是绝对离散的——总有一些连续谱（比如自由出射粒子的连续动量）, 或者带宽（实际能级有自然宽度）。

设末态在能量 \(E_f\) 附近连续, 用态密度 \(\rho(E_f)\)（单位 1/能量）描述: 在 \([E_f, E_f + dE]\) 范围内有 \(\rho(E_f) dE\) 个末态。把对单一末态求 \(W_{i\to f}\), 乘以态密度积分:

\[ \Gamma_{i\to f} = \int W_{i\to f'}\, \rho(E_{f'})\, dE_{f'} = \frac{2\pi}{\hbar}|\langle f|\hat V_0|i\rangle|^2 \rho(E_f), \]

用 \(\delta\) 函数把积分挑出 \(E_{f'} = E_i + \hbar\omega\) 处。这就是Fermi 金律:

\[ \boxed{\ \Gamma_{i\to f} = \frac{2\pi}{\hbar}|\langle f|\hat V|i\rangle|^2 \rho(E_f).\ } \]

请你停下来理解这条公式的结构:

  • \(\Gamma_{i\to f}\) 是跃迁速率（概率每单位时间, 单位 \(1/\)秒）。
  • \(|\langle f|\hat V|i\rangle|^2\) 是矩阵元模平方——量子力学的"耦合强度"; 它由微扰 \(\hat V\) 与初末态的重叠 决定, 是动力学的 QM 部分。
  • \(\rho(E_f)\) 是末态密度——可达末态有多少。这是统计 部分; 自由粒子 \(\rho \propto p^2 \propto \sqrt E\)（3D 动量空间球壳）。
  • \(2\pi/\hbar\) 是定值因子。

金律的应用范围极广——只要"初态固定 + 末态可数（连续）+ 微扰小" 三条满足。下面三个应用展示。

20.6 应用 1: 原子吸收 + 受激发射 + 自发发射 (Einstein A, B 系数)

原子在电磁场里。电磁波微扰 (电偶极近似)

\[ \hat V(t) = -\hat{\vec d}\cdot \vec E(t), \qquad \vec E(t) = E_0 \hat\varepsilon \cos\omega t, \]

\(\hat{\vec d} = -e\hat{\vec r}\) 是电偶极算子, \(\hat\varepsilon\) 是偏振方向。

(a) 吸收: 原子从 \(|i\rangle\) 跃迁到 \(|f\rangle\), 吸收一个能量 \(\hbar\omega = E_f - E_i\) 的光子。Fermi 金律给

\[ \Gamma_{i\to f}^{\text{abs}} = \frac{\pi e^2 E_0^2}{2\hbar^2}|\langle f|\hat\varepsilon\cdot\hat{\vec r}|i\rangle|^2 \delta(\omega - \omega_{fi}). \]

对辐射场各向同性 取平均, 用能量密度 \(u(\omega) = \varepsilon_0 E_0^2/2\) 替换 \(E_0^2\), 得

\[ \Gamma_{i\to f}^{\text{abs}} = B_{if} \, u(\omega_{fi}), \]

\(B_{if} = \pi e^2 |\langle f|\hat{\vec r}|i\rangle|^2 / (3\varepsilon_0 \hbar^2)\) 是Einstein \(B\) 系数（吸收）。

(b) 受激发射: 原子从 \(|f\rangle\) 受电磁场刺激跃迁回 \(|i\rangle\), 发射一个光子（与入射光相干, 同频同相同方向）。同样金律, 矩阵元模平方相同, 给同样的 \(B\) 系数 \(B_{fi} = B_{if}\) (Einstein 1916)。

这条对称性 是激光（LASER = Light Amplification by Stimulated Emission of Radiation）的物理基础。如果某个能级 \(|f\rangle\) 比 \(|i\rangle\) 占据多（"粒子数反转"）, 入射光会被放大（受激发射 > 吸收）。1960 Maiman 第一台红宝石激光器实现; 现在激光在通信、医学、加工无处不在。

(c) 自发发射: 即使没有 入射光, 激发态原子也会自发发光跃迁到低态。这个不能 用半经典理论（电磁场为经典）解释——必须把电磁场量子化（QED）, 让真空中存在零点涨落, 起到"诱发" 跃迁的作用。Einstein 1916 用统计平衡（黑体辐射 + Boltzmann 分布）逆推 出

\[ \boxed{\ A_{fi} = \frac{\hbar\omega_{fi}^3}{\pi^2 c^3}B_{fi} = \frac{e^2 \omega_{fi}^3}{3\pi\varepsilon_0 \hbar c^3}|\langle i|\hat{\vec r}|f\rangle|^2.\ } \]

\(A_{fi}\) 是Einstein \(A\) 系数（自发发射速率）。注意 \(A \propto \omega^3\)——高频跃迁自发 发射快得多。\(2p \to 1s\)（Lyman-\(\alpha\), 121.6 nm）寿命约 1.6 ns; 微波跃迁（21 cm 氢线）寿命 \(10^7\) 年（极慢, 让 21 cm 线在天文学里能"跨越宇宙" 不被淹没）。

1948-49 QED 给出 \(A\) 的第一原理 推导, 不需要 Einstein 的统计配平。这条公式在原子物理 / 天体物理 / 激光物理无所不在。

20.7 应用 2: 选择规则

金律里的矩阵元 \(\langle f|\hat{\vec d}|i\rangle\) 决定哪些跃迁允许（matrix element \(\neq 0\)）哪些禁戒（=0）。这给选择规则。

电偶极矩阵元 \(\langle n'\ell' m'|\hat{\vec r}|n\ell m\rangle\) 由角部分 球谐函数 \(Y_{\ell}^m\) 的积分决定。利用球谐函数的乘法公式（Wigner-Eckart 定理）, 电偶极算子在角动量基下表现为 \(\ell = 1\) 张量, 给

\[ \boxed{\ \Delta\ell = \pm 1, \quad \Delta m = 0, \pm 1.\ } \]

满足这两条的跃迁"允许"; 否则"禁戒"（电偶极矩阵元 = 0, 但更高阶——磁偶极、电四极——可能给非零, 但速率小很多）。

例子: 氢原子 \(2p \to 1s\) (\(\Delta\ell = -1, \Delta m = 0\)) 允许, 速率正常。\(2s \to 1s\) (\(\Delta\ell = 0\)) 电偶极禁戒; \(2s\) 寿命 \(\sim 0.12\) 秒（比 \(2p\) 长 8 个量级！）。这条"亚稳" \(2s\) 在天文 / 激光物理有重要应用。

选择规则是金律的立即 推论, 一行公式扫除大量可能跃迁——大大简化原子光谱分析。

20.8 应用 3: 核衰变 + 半导体散射

核 \(\beta\) 衰变: \({}^{14}\text{C} \to {}^{14}\text{N} + e^- + \bar\nu_e\)。Fermi 1934 把它用金律处理: 弱相互作用矩阵元 \(\langle f|\hat V_F|i\rangle = G_F \cdot M_{nuc}\), 其中 \(G_F\) 是 Fermi 常数（弱耦合强度）, \(M_{nuc}\) 是核矩阵元。末态有三体（核 + 电子 + 反中微子）, 末态密度由动量空间相空间体积给

\[ \rho(E) = \frac{V^2}{(2\pi\hbar)^6}\int p_e^2 \, dp_e \int p_\nu^2\, dp_\nu \, \delta(E - E_e - E_\nu). \]

积分给"Fermi 函数", 形状决定 \(\beta\) 谱（电子能量分布）。这是 1934 Fermi 弱相互作用理论的第一 应用, 直接验证了中微子的存在（β 谱形状必须有第三粒子吸收能量, 不然为单能尖峰）。1956 Reines + Cowan 实验直接探测到反中微子。

更广义: \(\alpha\) 衰变（链回ch8 隧穿）的速率也用金律加 WKB 矩阵元算; 重核裂变 / 聚变速率同。整个核反应理论 是金律 + 相空间因子。

半导体载流子散射: 半导体里电子从一个 Bloch 态 \(|\vec k\rangle\) 散射到 \(|\vec k'\rangle\), 由声子（晶格振动）、杂质、电子-电子相互作用驱动。每条散射机制给金律

\[ \Gamma_{\vec k \to \vec k'} = \frac{2\pi}{\hbar}|\langle \vec k'|\hat V_{\text{scatt}}|\vec k\rangle|^2 \delta(E_{\vec k'} - E_{\vec k}). \]

把对 \(\vec k'\) 积分（用末态密度），给总散射率 \(1/\tau(\vec k)\)。这条弛豫时间 \(\tau\) 进入 Boltzmann 输运方程, 决定半导体的电导率 \(\sigma = n e^2 \tau / m^*\)（Drude 公式 + 量子修正）。

简而言之: 你手机里 CPU 的速度限制, 最终被金律决定的电子-声子散射率所控制。

更多金律应用:

  • 分子振动跃迁（红外光谱）速率。
  • 核磁共振 NMR 在 RF 脉冲下的跃迁概率（链回 ch17.8）。
  • 原子 / 分子碰撞 截面（Born 近似 → 微分散射截面）, 与ch21 散射理论相通。
  • 粒子物理中的衰变率: \(\mu^- \to e^- + \bar\nu_e + \nu_\mu\) 寿命 2.2 \(\mu\)s——金律 + 电弱矩阵元给。

20.9 临别话

Fermi 金律是 QM 给动力学 的最重要公式。它把"跃迁速率" 写成"矩阵元 × 末态密度"——一边纯量子（耦合强度）, 一边纯统计（可达末态计数）。这条简洁分解让它在物理学里几乎到处出现。

本章主线:

  • 时变 SE → 一阶振幅 \(c_f^{(1)}(t) = -i/\hbar \int \langle f|\hat V(t')|i\rangle e^{i\omega_{fi}t'} dt'\)。
  • 周期微扰 → 共振峰 \(\sin^2(\Delta\omega t/2)/(\Delta\omega/2)^2\)。
  • 长时极限 + 连续末态 → \(\Gamma = (2\pi/\hbar)|\langle f|\hat V|i\rangle|^2 \rho(E_f)\)。
  • 应用: Einstein A/B 系数, 选择规则 \(\Delta\ell = \pm 1\), 核衰变, 半导体散射。

使用警告:

  • 金律是一阶 Born 近似——\(\hat V\) 必须"小" + 跃迁概率 \(\ll 1\)。强光、强场需要更高阶或非微扰处理（数值解时变 SE）。
  • "长时间" 极限是关键——\(t\) 必须远大于 \(1/\omega_{fi}\) 但远小于 \(1/\Gamma\)（不然跃迁出去的概率反向影响初态, 单向假设失效）。
  • 简并末态需要小心: 可能多条独立跃迁通道, 每条加金律然后求和。

下一章ch21 我们做散射理论——粒子从远处入射, 与靶相互作用, 散射出去。这是金律的"空间" 版本（碰撞 = 一种跃迁过程）, 给微分截面 \(d\sigma/d\Omega\), 是高能物理 + 凝聚态实验最直接的可观测量。
