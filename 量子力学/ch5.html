第 5 章 自由粒子 + 波包

前面四章我们建好了量子力学的语言: 波函数 \(\psi(x, t)\)、Schrödinger 方程 \(i\hbar \partial_t \psi = \hat H \psi\)、Born 规则 \(|\psi|^2\) 是概率密度。现在到了"真去解一个具体系统" 的时刻。最简单的具体系统是什么外力都没有——自由粒子, \(V(x) \equiv 0\)。请你停下来想一秒：经典上自由粒子最无聊（直线匀速运动）, QM 上呢？

答案——QM 自由粒子是本书第一处真正撞墙 的地方。最朴素的解（平面波）不归一, 不是合法波函数。修补它逼出一整套新工具——Fourier 叠加构造波包。这一章我把这条路从撞墙到修补到命名（"波包""群速度""波包扩散"）一步步走完。结果是 QM 一维粒子的第一个完整动力学场景, 也是后面所有"位置 vs 动量" 直觉的源头。

5.1 第一招: 套 SE, 解平面波——但不归一

设 \(V(x) \equiv 0\)。一维含时 Schrödinger 方程:

\[ i \hbar \frac{\partial \psi}{\partial t} = -\frac{\hbar^2}{2m} \frac{\partial^2 \psi}{\partial x^2}. \]

这条 PDE 长得像扩散方程（多了 \(i\)）。请你回想 ch3 的"分离变量" 招——令 \(\psi(x, t) = \varphi(x) T(t)\), 代入两边除以 \(\varphi T\):

\[ \frac{i \hbar \dot T}{T} = -\frac{\hbar^2}{2m} \frac{\varphi''}{\varphi}. \]

左边只与 \(t\) 有关, 右边只与 \(x\) 有关。两边都得是常数, 记作 \(E\)（量纲是能量, 物理上就是能量本征值）。两条 ODE:

\[ i\hbar \dot T = E T \quad \Rightarrow \quad T(t) = e^{-i E t / \hbar}. \]

\[ -\frac{\hbar^2}{2m} \varphi'' = E \varphi \quad \Rightarrow \quad \varphi'' = -\frac{2 m E}{\hbar^2} \varphi. \]

设 \(E > 0\), 令 \(k = \sqrt{2 m E}/\hbar\), 则 \(\varphi'' = -k^2 \varphi\), 解为 \(\varphi(x) = A e^{i k x} + B e^{-i k x}\)。每一项都是平面波。取 \(B = 0, A = 1\), 整个解写成

\[ \boxed{\ \psi_k(x, t) = e^{i k x} e^{-i E t / \hbar} = e^{i (k x - \omega t)}, \quad \omega = E/\hbar = \frac{\hbar k^2}{2 m}.\ } \]

这条函数在每一点 模平方都等于 \(|\psi_k|^2 = 1\)。看起来不错——但请你停下来检查归一化:

\[ \int_{-\infty}^{\infty} |\psi_k(x, t)|^2 \, dx = \int_{-\infty}^\infty 1 \, dx = \infty. \]

归一化积分发散。Born 规则要求 \(|\psi|^2\) 是概率密度, 总概率得是 1。平面波违反归一化。这是数学上的硬伤——平面波不是合法的物理波函数。

更糟的物理含义: \(|\psi_k|^2 = 1\) 在所有 \(x\) 都一样大, 意味着粒子同等概率 出现在宇宙任何位置。一个"位置完全不确定" 的态。这跟"有动量 \(p = \hbar k\) 的粒子" 这种直觉完全不像——经典上粒子是个点, 应该有 位置。

5.2 撞墙后的 reframe: 叠加

请你停下来仔细想——平面波 \(\psi_k\) 不归一, 但 它满足 SE。SE 是线性 方程: 任两个解 \(\psi_1, \psi_2\) 的线性组合 \(c_1 \psi_1 + c_2 \psi_2\) 也是解。而且 \(k\) 取任何 实数都给一个解 \(\psi_k\)。

所以"叠加无穷多个不同 \(k\) 的平面波" 是个允许操作。具体地——把离散 求和换成 \(k\) 上的积分:

\[ \psi(x, t) = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^\infty \tilde\psi(k) \, e^{i (k x - \omega(k) t)} \, dk, \qquad \omega(k) = \frac{\hbar k^2}{2 m}. \]

权重 \(\tilde\psi(k)\) 决定每个 \(k\) 的"多少份"。如果 \(\tilde\psi(k)\) 在某个中心 \(k_0\) 附近窄峰（高斯之类）, 那这个积分就把不同 \(k\) 的平面波相干叠加——干涉相消使得 \(\psi(x, t)\) 在大部分位置接近 0, 只在某个区间内大——一个局域 的波。这种局域的波叫波包。

这条 reframe 改变了游戏规则: 不再问"动量精确为 \(p\) 的态长什么样"（答案: 不归一的平面波, 不存在）, 改问"动量在 \(p_0 \pm \Delta p\) 范围、位置在 \(x_0 \pm \Delta x\) 范围的态长什么样"——答案: 波包, 完全合法的归一波函数。

请你欣赏这件事——QM 第一次逼 你接受"位置和动量不能同时精确"。这不是哲学姿态, 是数学事实——动量精确 ⟹ 平面波 ⟹ 不归一 ⟹ 不合法; 位置精确 ⟹ \(\delta\) 函数 ⟹ 也是 \(\tilde\psi(k)\) 上的常数 ⟹ 同样不归一。合法 的物理态必然在两边都"有点不确定"。这是不确定性原理的雏形, ch11 完整化。

5.3 高斯波包: 一个能算清的例子

选最容易的一种 \(\tilde\psi(k)\)——高斯。理由: (1) 高斯的 Fourier 变换还是高斯, 算得动; (2) 高斯在物理直觉上是"典型" 的局域分布。

\(t = 0\) 时设

\[ \boxed{\ \psi(x, 0) = \left( \frac{1}{2\pi \sigma^2} \right)^{1/4} \exp\!\left( -\frac{x^2}{4 \sigma^2} \right) e^{i k_0 x}.\ } \]

三块: (a) 归一化常数 \((2\pi\sigma^2)^{-1/4}\); (b) 高斯包络 \(e^{-x^2/(4\sigma^2)}\) 让 \(|\psi|^2\) 在 \(x = 0\) 附近宽 \(\sigma\) 的钟形; (c) 平面波因子 \(e^{i k_0 x}\) 给波包"携带" 中心动量 \(p_0 = \hbar k_0\)。

验证归一化:

\[ |\psi(x, 0)|^2 = \frac{1}{\sqrt{2\pi \sigma^2}} e^{-x^2 / (2\sigma^2)}, \]

这是均值 0、方差 \(\sigma^2\) 的标准高斯密度——\(\int |\psi|^2 dx = 1\)。

位置期望、不确定性:

\[ \langle x \rangle (0) = \int x |\psi|^2 dx = 0, \quad \langle x^2 \rangle(0) = \sigma^2, \quad \Delta x(0) = \sigma. \]

动量空间表示——做 Fourier 变换:

\[ \tilde\psi(k) = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^\infty \psi(x, 0) e^{-i k x} dx. \]

把 \(\psi(x, 0)\) 代入, 凑指数完全平方（标准技巧, 微积分册做过——把 \(-x^2/(4\sigma^2) - i(k - k_0) x\) 写成 \(-(x + 2i\sigma^2 (k - k_0))^2/(4\sigma^2) - \sigma^2 (k - k_0)^2\)）, 得

\[ \tilde\psi(k) = \left( \frac{2 \sigma^2}{\pi} \right)^{1/4} \exp\!\left( -\sigma^2 (k - k_0)^2 \right). \]

请你比较: 位置空间宽 \(\sigma\), 动量空间宽 \(1/(2\sigma)\)（因为高斯方差 \(1/(2\sigma^2)\) 对应标准差 \(1/(\sqrt{2}\cdot \sqrt{2}\sigma)\), 先不严格, 5.6 算清）。一边窄 ⟹ 另一边宽——这就是 Fourier 不确定性, 在物理上变成位置-动量不确定性。

5.4 时间演化: Fourier 解

把高斯波包推到一般 \(t\)。SE 在 Fourier 表示里很简单——每个 \(k\) 模式独立按 \(e^{-i\omega(k) t}\) 演化。所以

\[ \psi(x, t) = \frac{1}{\sqrt{2\pi}} \int \tilde\psi(k) e^{i(kx - \omega(k) t)} dk \]

把 \(\tilde\psi(k) = (2\sigma^2/\pi)^{1/4} e^{-\sigma^2 (k - k_0)^2}\) 与 \(\omega(k) = \hbar k^2/(2m)\) 代入。指数:

\[ -\sigma^2 (k - k_0)^2 + i k x - \frac{i \hbar k^2 t}{2 m}. \]

这是 \(k\) 的二次, 完全平方再积分（细节是高斯积分 \(\int e^{-a k^2 + b k} dk = \sqrt{\pi/a} e^{b^2/(4a)}\)）。结果繁但解析:

\[ \psi(x, t) = \left( \frac{1}{2\pi \sigma_t^2} \right)^{1/4} \frac{1}{\sqrt{1 + i\hbar t/(2 m \sigma^2)}} \exp\!\left( -\frac{(x - v_g t)^2}{4 \sigma^2 (1 + i\hbar t/(2m\sigma^2))} + i k_0 (x - v_g t/2) - i\omega_0 t \right), \]

其中 \(v_g = \hbar k_0/m\)（下面命名）, \(\omega_0 = \hbar k_0^2/(2m)\)。先不要被这堆指数吓到——重要的不是这条公式的全貌, 是模平方:

\[ \boxed{\ |\psi(x, t)|^2 = \frac{1}{\sqrt{2\pi \sigma_t^2}} \exp\!\left( -\frac{(x - v_g t)^2}{2 \sigma_t^2} \right), \quad \sigma_t^2 = \sigma^2 + \frac{\hbar^2 t^2}{4 m^2 \sigma^2}.\ } \]

这就是含时间高斯波包的概率密度——仍是高斯, 但中心 平移到 \(x = v_g t\)（以速度 \(v_g\) 漂移）, 宽度 从 \(\sigma\) 涨到 \(\sigma_t\)（随时间变宽）。两件事——漂移 + 扩散——分别是 5.5 和 5.6 的主题。

5.5 群速度 vs 相速度

波包中心 以什么速度漂移？从上面 \(|\psi|^2\) 看, 中心 \(x_c(t) = v_g t\), 所以速度 \(\dot x_c = v_g = \hbar k_0/m\)。代入 \(p_0 = \hbar k_0\):

\[ v_g = \frac{p_0}{m}. \]

请你停下来——这正是经典力学里"动量 \(p_0\) 的粒子的速度"。波包以经典速度漂移。这是 QM 与经典物理的第一处真正接合点。

这个速度有个名字——群速度:

\[ \boxed{\ v_g := \frac{d\omega}{dk} \bigg|_{k_0}.\ } \]

验证: \(\omega(k) = \hbar k^2/(2m)\), \(d\omega/dk = \hbar k/m\), 代 \(k = k_0\) 得 \(v_g = \hbar k_0/m\)——和上面一致。"群" 来自"波群" = 波包: 一群相近 \(k\) 的平面波相干叠加, 它们的"包" 以 \(d\omega/dk\) 速度移动。

对比相速度——单个平面波 \(e^{i(kx - \omega t)}\) 的相位 \(kx - \omega t\) 等值线沿 \(x = (\omega/k) t\) 移动, 所以

\[ v_p := \frac{\omega}{k} = \frac{\hbar k}{2 m} = \frac{p}{2m}. \]

注意 \(v_p = v_g/2\)——相速度是群速度的一半！这看起来矛盾——粒子不是按 \(p/m\) 跑吗？答案: 粒子的位置概率密度 跟着波包 走（群速度）, 跟相位 没关系（相速度）。相速度只描述每一根平面波的相位演化, 不是物理可观测。

请你欣赏这一点——"粒子速度 = 群速度" 是 QM 与经典力学的 Ehrenfest 定理（ch10 给一般版本）的第一个具体例子: 波函数中心按经典轨迹走。这把"粒子" 这个经典直觉嵌入 QM——粒子不是点, 是波包; 波包不是死的, 它的中心遵守经典动量公式。

5.6 波包扩散: 位置不确定性变大

5.4 给的 \(|\psi|^2\) 宽度

\[ \sigma_t^2 = \sigma^2 + \frac{\hbar^2 t^2}{4 m^2 \sigma^2}. \]

大 \(t\) 时, \(\sigma_t \approx \hbar t/(2 m \sigma)\)——线性 增长。所以 \(\Delta x(t) = \sigma_t \to \infty\)。自由粒子的位置不确定性随时间无限增大。

这是个让人停下来的现象。经典上 \(t = 0\) 时位置 \(x = 0 \pm \sigma\)、动量 \(p = p_0 \pm \Delta p\) 的粒子集合, \(t\) 时刻位置在 \(x = (p_0/m) t \pm (\Delta p/m) t \pm \sigma\), 也线性扩散——但因为 初始动量本来就有不确定。QM 里同样: 高斯波包里有一族 \(k\), 速度 \(\hbar k/m\), 不同 \(k\) 速度不同 ⟹ 跑得快慢不同 ⟹ 越往后位置越分散。

但 QM 多了一层——初始时位置 \(\sigma\) 越小, 后期扩散反而越快。看公式: \(\sigma_t^2\) 对小 \(\sigma\) 来说第二项 \(\hbar^2 t^2/(4 m^2 \sigma^2)\) 反比 \(\sigma^2\) 增大。物理直觉: 初始位置越准 ⟹ Fourier 变换的动量越宽 ⟹ 不同 \(k\) 散得越开 ⟹ 跑得越散。这就是不确定性原理的动力学后果。

这一现象的"守恒量" 是动量分布——\(|\tilde\psi(k, t)|^2 = |\tilde\psi(k, 0)|^2\)（只多了一个相位 \(e^{-i\omega(k) t}\), 模平方不变）。所以 \(\Delta p\) 不 随时间变。变的只是 \(\Delta x\), 而且总是变大。

对比束缚态（下一章无限深势阱）——束缚态的概率密度不 扩散, 因为势阱壁挡住。自由粒子才有这种"无限扩散"。

5.7 \(\Delta x \, \Delta p \ge \hbar/2\) 在波包上具体验证

算 \(\Delta x \cdot \Delta p\) 看它能多小。

位置标准差。\(t = 0\) 时高斯密度 \(|\psi|^2 = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-x^2/(2\sigma^2)}\), 标准差 \(\Delta x(0) = \sigma\)。

动量标准差。\(|\tilde\psi(k)|^2 = \sqrt{2\sigma^2/\pi} e^{-2\sigma^2 (k - k_0)^2}\), 这是 \(k\) 的高斯, 标准差 \(\Delta k = 1/(2\sigma)\)。代 \(p = \hbar k\), \(\Delta p = \hbar \Delta k = \hbar/(2\sigma)\)。

积:

\[ \Delta x \cdot \Delta p = \sigma \cdot \frac{\hbar}{2 \sigma} = \frac{\hbar}{2}. \]

请你停下来欣赏: 恰好 \(\hbar/2\)。高斯波包饱和 不确定性下界。没有 任何归一波函数能让 \(\Delta x \Delta p\) 严格小于 \(\hbar/2\)（ch11 证一般情形——Cauchy-Schwarz + \([\hat x, \hat p] = i\hbar\)）, 高斯是唯一能取等的。这件事让高斯波包成为"最经典的" 量子态——它在 \((x, p)\) 相空间占的面积最小。物理学家叫它最小不确定性态。

大 \(t\) 时:

\[ \Delta x(t) = \sigma_t = \sigma \sqrt{1 + \hbar^2 t^2/(4 m^2 \sigma^4)}, \quad \Delta p = \hbar/(2\sigma) \text{ (不变)}. \]

积 \(\Delta x(t) \Delta p = (\hbar/2) \sqrt{1 + \hbar^2 t^2/(4 m^2 \sigma^4)} \ge \hbar/2\), 等号只在 \(t = 0\) 取。波包随时间从最小不确定性走开——位置变模糊, 动量保持不变, 积往上走。

5.8 物理含义 + 经典极限

把 5.1-5.7 串起来。一句话总结自由粒子 QM:

  • 不存在"动量精确的粒子"——平面波不归一。合法物理态是波包。
  • 波包中心 按经典动量公式漂移（\(v_g = p_0/m\)）。这就是经典粒子轨迹的 QM 起源。
  • 波包宽度 随时间增大（\(\sigma_t\) 线性增长）——位置不确定性单调增加。
  • 动量分布不变——\(\Delta p\) 是常数。
  • 高斯波包初始饱和 不确定性 \(\Delta x \Delta p = \hbar/2\), 之后向上走。

那经典极限呢——为什么我们看不到桌球扩散？算个数。1 g 的小球 \(m = 10^{-3}\) kg, 初始 \(\sigma = 1\) mm\(= 10^{-3}\) m。1 秒后扩散

\[ \Delta x(1) \approx \frac{\hbar t}{2 m \sigma} = \frac{10^{-34}}{2 \cdot 10^{-3} \cdot 10^{-3}} = 5 \times 10^{-29} \text{ m}. \]

\(10^{-29}\) 米——比一个原子还小 18 个数量级。完全不可观测。所以宏观物体看不到波包扩散——\(\hbar\) 太小, 大质量更压缩这个效应。

电子呢？\(m = 9 \times 10^{-31}\) kg, 同样 \(\sigma = 10^{-9}\) m（1 nm, 原子尺度）, 1 ps 后:

\[ \Delta x \approx \frac{10^{-34} \cdot 10^{-12}}{2 \cdot 10^{-30} \cdot 10^{-9}} = 5 \times 10^{-8} \text{ m}, \]

50 nm——已经远远超出原始 1 nm。电子在 ps 尺度必然 扩散到看不见的程度。这就是为什么电子显微镜的电子要"聚焦"——不聚焦它就散开。

更深的——上面的"群速度 = 经典速度" 是Ehrenfest 定理的特例: 算符 \(\hat x\) 的期望按"经典 \(\dot x = p/m\)" 演化。一般形式（ch10）:

\[ \frac{d}{dt} \langle \hat x \rangle = \frac{\langle \hat p \rangle}{m}, \quad \frac{d}{dt} \langle \hat p \rangle = -\langle \nabla V \rangle. \]

对自由粒子 \(V = 0\), 第二式给 \(\langle \hat p \rangle\) 守恒, 第一式给 \(\langle \hat x \rangle\) 线性增长——和波包中心轨迹完全一致。Ehrenfest 把"QM 期望按经典轨迹走" 一般化, 自由粒子是它最干净的例子。

5.9 一般化: 任何 \(\omega(k)\) 都给波包

这一章用的所有工具——Fourier 叠加、群速度 \(d\omega/dk\)、波包扩散——并不 局限于 SE 自由粒子 \(\omega = \hbar k^2/(2m)\)。它对任何"色散关系" \(\omega = \omega(k)\) 都成立。

例子:

  • 电磁波在真空: \(\omega = c k\) ⟹ \(v_p = v_g = c\), 不扩散——光波包以光速传播且形状保持。
  • 电磁波在介质: \(\omega = c k/n(\omega)\), \(n\) 频率依赖 ⟹ \(v_g \neq v_p\), 还有扩散——这就是棱镜分光的原理（也叫"色散" 由此得名）。
  • 水波: \(\omega = \sqrt{g k}\) ⟹ \(v_g = (1/2)\sqrt{g/k}\), 比相速度小, 不同 \(k\) 散得开——海面上看到一群水波"展开" 就是这个。

"波包扩散" 是所有色散波 的普适现象。SE 自由粒子（\(\omega = \hbar k^2/2m\)）只是其中一个具体例子——但它的物理意义最大: 把"粒子" 这个经典概念嵌入 量子波动力学。

本书后续每一章, 每一次你看到"位置分布"或"动量分布", 都可以回头想这一章——QM 里没有"同时精确的位置和动量", 只有分布; 分布的中心按经典走, 分布的宽随时间动。这是 QM 的基本运动学画面。

5.10 临别话

这一章你看到 QM 的第一次"具体动力学": 自由粒子怎么在时间里演化。撞墙在 5.1（平面波不归一）, reframe 在 5.2（叠加成波包）, 命名在 5.5、5.7（群速度、波包扩散、最小不确定性态）, 一般化在 5.9（任何色散关系）。请你把这条路当成一个模板——后面 ch6（势阱）、ch7（谐振子）、ch8（势垒）每一个都会重走类似流程: 朴素 SE 解 → 撞某种墙 → reframe → 命名新对象。

下一章我们换一个完全不同的物理设定——粒子不 自由, 而是被困在一段长度 \(L\) 的"盒子" 里, 边界外势能无穷大。这种束缚态 第一次给出离散能级——QM 的招牌特征。Bohr 1913 用 ad hoc 假设硬塞的"能级离散", 在这里会从 SE 解的边界条件自然 长出来。这就是 QM 真正"解释" 而不只是"假设" 离散能级的第一处。
