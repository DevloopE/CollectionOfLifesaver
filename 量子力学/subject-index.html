查找：按"我现在想做什么"

  1. 解一个一维 SE: 自由 ch5、势阱 ch6、谐振子 ch7、势垒 ch8。先看势的形式选合适章节。
  2. 算氢原子能级 / 波函数: 第 16 章给完整解析解; 球谐 ch15 + 径向 ch16。
  3. 处理含自旋系统: ch17 Pauli 矩阵; 多自旋用张量积 ch18 + 附录 B.4。
  4. 多电子分子 / 原子计算: HF 24.2, 高精度 CCSD(T) 24.2, 实用 DFT 24.3-24.4。
  5. "小扰动" 修正能级: 时间无关微扰 ch19; 时间依赖 + 跃迁速率 ch20 (Fermi 金律)。
  6. 给散射数据反推作用力: 高能 / 弱势 用 Born 21.4; 低能 / 强势 用部分波 + 相移 21.6-21.7。共振找新粒子 21.7。
  7. 判断"这个态是否纠缠": 看能否分解为乘积态 ch22; 算约化密度矩阵 附录 B.4, \(\mathrm{Tr}\rho^2 < 1\) 即纠缠。
  8. 设计量子算法: 子例程库——Hadamard 制备叠加 → oracle 编码问题 → 干涉提取答案。Deutsch-Jozsa / Grover / Shor 是模板 23.5-23.7。
  9. 验证 Bell 不等式: CHSH \(|S| \le 2\) vs QM 预言 \(2\sqrt 2\) 22.4; 选 0°/45°/90°/135° 方向。
  10. 计算复积分（量子散射 Green 函数等）: 留数定理 A.4; 含 \(e^{ikx}\) 用 Jordan 引理选半圆; 实轴极点用 \(\pm i\varepsilon\) 处方决定边界条件。
  11. 诊断"我的 QM 计算结果不对": 检查（1）波函数归一化; （2）算子是否自伴（本征值实数）; （3）对易关系是否一致; （4）数值上调用对角化 scipy.linalg.eigh 而不是普通 eig; （5）多体问题张量积维数是否正确。
  12. 开始读 QFT / 量子化学 / 量子信息: 见 ch24 临别话 推荐书单。本书是它们的共同入口。
